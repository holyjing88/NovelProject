## 二、战力公式

```
有效战力 = 境界基础 × 道具阶 × 符阵加成 × 丹药状态 × 遁逃准备 × 道侣同心(≤1.15)
```

| 越级上限 | 条件 |
|----------|------|
| 炼气越筑基 | 符箓+阵盘+地利，不可硬扛 |
| 筑基越结丹 | 真火+二品丹+队友拖时间 |
| 结丹越元婴 | 沧溟海地利+断浪舟+刑煞门内乱 |
| 元婴以上 | 每境最多越一小阶，需代价 |

## 三、陈寻三策 + 报仇三则

**三策**：不浪突破、坊市不留真名、不欠命债。  
**生死一策**（全书硬规）：**活着最先**；禁止自刎/自尽；实在无法活，**拉仇人垫背**。  
**报仇三则**：记仇不忘、实力到了才清算、不牵连恩人。

## 九、系统一致性规则

1. 境界不可跳  
2. 道具不可越两阶使用（古宝以上除外）  
3. 真火融火顺序 1→9  
4. 青冥露子时满、午时漏半  
5. 法宝换人用损威三成  
6. 道侣合击不计入道具阶，乘数 ≤1.15  
7. 因果阳因不足，古宝以上无法认主  
8. **符录**符品不可越两阶；420前无契符；528和合符需420后  
9. **丹道**不可越两品；破境日只炼丹不绘符（`02` §16.2）  
10. **阵法**瞬发用符、占点用阵；780前四绶合阵仅预演（`02` §17.3）

## 十五、符录系统（符品九阶）

> **符录** = 修士所习符法之总册（纸册记符名、符品、灵材、施法条件），与恩施册**同匣异页**，叙事不出现 UI 面板。  
> 符箓、阵盘、契符统一按 **符品** 定级；与 **道具九阶** 对齐但命名独立。

### 15.1 符品九阶

| 5 | **天符** | 结丹～元婴 | 古宝 | 半持久 | 需天时 |
| 6 | **契符** | 元婴～化神 | 灵宝 | 契约绑定 | 需血/名/誓 |

### 15.2 符录规则

| 规则 | 说明 |
|------|------|
| 越品绘制 | **不可越两品**（同道具九阶） |
| 越境使用 | 高一境威减三成；低一境耗灵加倍 |
| 一次性符 | 燃尽即废，不可馈缘倒卖 |
| 阵盘 | 地符以上，可重复充灵 ≤3 次 |
| 契符 | 绑名分/道侣/誓约；毁约反噬阴因 |
| 陈寻三策延伸 | **每大境藏一张遁符**（黄→玄→地→天） |
| 馈缘 | 赠**成品符**可计档；赠符录**页**需对方立誓不外泄 |
| 与丹道 | 绘符耗灵同炼丹；破境日不绘符 |
