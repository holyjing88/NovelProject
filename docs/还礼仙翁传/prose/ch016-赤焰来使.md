# 第十六章 赤焰来使

孙福事毕第三日，山门外赤焰旗至。

旗红如血，绣黑焰纹，风一过，像把东脉的灵田都舔了一口。弟子列阵，剑未出，气先紧。

韩铁山跨马至，甲硬声硬，抱拳礼数足，眼神不足：「青岚宗主何在？赤焰谷韩铁山，奉少谷主令，来谈东脉灵田。」

柳青鸢出殿，青衫素，威在眸：「东脉灵田，青岚门民共耕，非一谷可索。」

韩铁山笑，笑里刀：「宗主言重。少谷主只说，七日之内，划界立碑，东脉三成归赤焰，换七年不动兵戈。若不然——」

他顿顿，像故意让风把后半句送进每个人耳朵：「妖潮与赤焰，同至。」

殿前一寂。

周德海即出，捧折子，像早备好的主和稿：「宗主，赤焰势大，东脉民散则全宗失血。割三成，换七年，可议。」

霍镇山怒目：「割地换安，安在谁枕上？民散根拔，七年和平，是葬歌！」

柳青鸢沉脸，未拍板。

莫长春袖空，杖在塔中，立殿阶侧，不语，只看识海里塔壁竹纹——竹纹比往日游得快，像因果将涨。

韩铁山目光扫过他，像扫一块将熄炭：「这位，便是传说中的莫长老？听说……九月寿元？」

莫长春笑：「九月够听你把话说完。说。」

韩铁山一愣，未料将死之人接话接得稳，便敛笑：「少谷主敬长老曾是化神，故留七日。七日后，厉少谷主亲至，谈不成，便不必谈。」

他甩袖，赤焰旗转，马蹄去，像把血旗插在东脉边上，等七日后的雨。

长老会当夜急开。

周德海主和：「东脉三成，可保核心灵脉。」

霍镇山主战：「妖潮频仍，让地则民散，民散则门散。」

柳青鸢指节发白，终看向莫长春：「师兄，你怎么看？」

莫长春一念出塔，杖点地：「地可以给，根不能拔。」

不表态站队，却令柳知他心向战——根在民，在田，在青岚门东脉那条命。

周德海冷笑退席半步：「莫长老说得轻巧，九月之人，不必守田。」

莫长春看他：「正因为我九月，才懂田比命短。命短的人，最懂根。」

殿外，弟子议论未歇，议论里怕，也燃——怕七日之期，燃的是青岚门终于不再只守东脉。

苏念慈在丹堂听见「厉无殇」三字，炉火一跳。她低声对弟子：「赤焰不赤焰，别传。传多了，传成惧。」

赵玄机在执法堂抄门规，抄到「外使索地」一条，停住，墨滴成字，像莫长春那句「根不能拔」又浮上来。

夜，塔壁浮字，淡金：

**外患已至，七日为期。**

莫长春收念，望东脉方向，像望一场将来的宴——宴上有血旗，有少谷主，有全场寂然。

他念：「厉无殇……名字倒凶。希望人别比名更凶。」

莫长春笑：「不等了。七日，够备一份礼。」

袖空，一念出塔，素帕在指间一绕，帕角干净，像从未擦过汗，却像等了很多年。

使节离山前，韩铁山回头看了一眼莫长春，一眼深，深里像记仇，也像记礼。他对副将道：「九月老头，接话接得稳。稳的人，别轻敌。」

东脉民户闻讯，田埂上停锄。老农问传讯弟子：「赤焰要三成，是要田，还是要命？」

弟子答不出，答不出里怕，怕里燃——燃的是七年和平，还是七年葬歌。

莫长春过东脉驿道，未入田，只立道旁碑，碑上刻「青岚东脉，民耕三百年」。他杖点碑角，声轻，却清晰：「碑在，根在。根在，三成索不去。」

路过的民户听见，跪一半，跪里掺敬，敬里掺怕——怕妖潮，怕赤焰，怕九月老头走得太快。

柳青鸢夜召莫长春，递东脉图：「师兄，韩铁山眼神不足，礼数足。足里藏刀。」

莫长春看图，笑：「刀藏七日里。七日里，先备席，再备帕。」

柳青鸢一怔：「帕？」

「礼轻，意重。」他不解释，只指塔意，「塔在，帕在。在不在袖，在不在识海。」

她不再问，像二十年同门，懂「不问」也是信。

周德海夜会心腹，折子摊开：「七日之内，宗主必犹豫。犹豫，就是我们的缝。」

心腹低声：「莫长老不犹豫。」

周德海笑，笑冷：「他不犹豫，九月犹豫。九月将尽，谁肯把命押在东脉？」

魏无涯在远处檐角，扇开一线，观赤焰旗残影，低笑：「七日，够一场戏。」

霍镇山遣人传话：「民不散，先听宗主。听宗主，再听阵。」

莫长春立院中，袖空，一念出塔，杖在掌，杖点地，像给七日敲节拍。

他念：「韩铁山走了，厉无殇要来。来之前，人心先裂。」

东脉驿道，传讯马蹄急，急得像妖潮与赤焰两字，已先到了田埂。

山门列阵撤后，弟子仍窃语，窃语潮头在「妖潮与赤焰，同至」七字。有人怕，有人燃，燃里像终于有人把青岚门从「只守」推成「要争」。

柳青鸢夜嘱执事：「东脉驿一贴裁断，一贴使帖抄本。贴给民看，也贴给敢诬的人看。」

周德海密室里，心腹问：「使走了，还拱吗？」周冷笑：「使走了，界还在。界在，咱们的缝还在。」

顾小满在杂役堂记：赤焰旗，七日，东脉三成。记完，声平：「旗在纸上红，田在人心里青。」

莫长春袖空，一念出塔，杖点阶，像给七日敲最后一记节拍。他念：「使走了，裂要来。人心比旗快。」

赵玄机验使帖，帖烫金，抄「外使索地」至第三遍，忽然停——停处是莫长春那句「根不能拔」，拔不动，像根在字上生了。

霍镇山按刀回营，刀稳，稳里战意未消。他对亲兵道：「七日，不是等，是磨。磨盾，磨胆，磨到少谷主进门那日，咱们先硬一寸。」

苏念慈闭炉，闭得像怕旗红进丹堂，却隔窗望东脉方向，望得像望一场将来的火——火不在炉，在旗。

柳青鸢夜召霍镇山，未谈和战，只指东脉图一角：「此处水渠，战时须保。保不住，民散。」霍镇山抱拳：「水渠在，人在；人在，盾在！」

魏无涯檐角观，扇开一线，低笑：「七日，够一场戏。戏不在旗，在脸——脸到了，礼才出。」

民户田埂上，有人把「三成」写成「三命」，写错者被老农打手：「写错字，田就真没了。」

殿前，弟子议论未歇，有人低声：「妖潮与赤焰，同至？」另一人答：「同至之前，先备礼。」

莫长春院中，塔壁浮字淡金未干，像七日后那场宴，已在格子里温着。他抚杖，笑：「不等了。七日，够备一份礼。」

韩铁山离山十里，驻马回望，低声：「九月老头，接话接得稳。稳的人，别轻敌。」

莫长春立院中，素帕在塔中压着，低声：「刀藏七日里。七日里，先备席，再备帕。」

周德海赔笑散场，笑洒，洒得像怕旗风也洒到自己。回府路上，他嘱心腹：「七日里，莫长春必赠礼。赠礼若成，和字更丑——丑了，咱们的缝更大。」

霍镇山遣人传话：「民不散，先听宗主。听宗主，再听阵。」路过的民户听见，跪一半，跪里掺敬，敬里掺怕。

殿前，弟子议论未歇，议论里怕，也燃——怕七日之期，燃的是青岚门终于不再只守东脉。

莫长春院中，塔壁浮字淡金未干，像七日后那场宴，已在格子里温着。他抚杖，笑：「不等了。七日，够备一份礼。使走了，裂要来。人心比旗快。」

赤焰旗去，血纹犹在门外土上，像七日之期已先咬了一口东脉。

弟子渐散，散里仍回头瞧一眼灰袍袖空——瞧得像瞧一眼九月还能走多久。
