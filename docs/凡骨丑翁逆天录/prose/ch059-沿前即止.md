# 第五十九章 沿前即止

冬更深，丙九静。静里，坛温却醒半分。醒半分，不是醒全，像瓮翻身，翻完又眠。

韩泥编筐末藤，藤刺再扎，扎得深，血珠再饱满。他仍甩向破席。席旧，吸血，血渗至席边，沿前止。

止，像一道看不见的线。线内是席，线外是沿。沿上无尘，沿上无血。

邻床梦呓：「……红……」

韩泥盖席，声低：「你看错了。睡。」

血到了席，没到沿。沿在眠，血不主动碰，这是铁规。

符怀暖一丝，护手不僵。僵，下一刺便偏。不偏，缘才正。

老耿门外咳：「沿净，好。净，才接得住赠。」

韩泥「嗯」一声，不追问。问急，像邪。等，像守瓮人。耿脚步远。

褐袍僧影已散，像凡符使命已了。余下瓮与血缘，余下席与沿。

午后，清西栏，粪锹沉，掌茧裂口渗潮。潮不是血，血不碰沿。

管事盯梢：「沿净，记在心里。脏沿，像拜邪。」

韩泥「嗯」一声：「不拜邪。拜邪，路断。」

赖福讽：「守坛守出仙缘？」

管事声冷：「再嗓，西栏你一人清到年尾。」

赖福哑了。哑，韩泥肩上的粪锹却捆得更实。

刘婆端粥来，稀，热：「冬深，别空肚。」

「不空。」韩泥喝尽，「嗯」一声。热在，像第七笔又实。

铁无言巡栏边，声低：「沿净？」

「净。」韩泥说，「血在席，不在沿。」

铁无言「嗯」一声，灯远。

他低声对坛：「你再眠一夜。眠着，我席盖严。严，不主动滴你。滴，要等扎那一刺，误那一甩。」

坛不应字，只温，温得像「嗯」。外头风息，柱上无新条。

明日仍搬柴，仍清栏，仍末排。末排不要紧，要紧是沿净，手稳，心记着七笔。

他按席，席厚，厚像守了六十日的席。命跟坛，坛跟缘，缘在沿前那一止。

血是活计扎出的，不是主动滴坛的。滴坛，要等误那一甩。

秦霜远廊一句：「手稳，心别邪。邪，符灭。」

韩泥怀符暖一丝，心记着七笔，不邪。

同批少年碰他袖：「你席盖那么严，藏啥？」

「坛。」韩泥说，「坛在眠，不在醒。」

「醒了呢？」

「醒了，缘满。」韩泥说，「缘满，不在急。」

少年不懂，哼一声，走了。

他独眼平：「沿在眠，血不主动碰。碰，要等活计误那一甩。」

外头更鼓三响，响沉。沉里，瓮温醒半分，又眠回去。眠回去不要紧，要紧是沿净。

他低声道：「沿前即止，不接沿。不接，才接得住瓮醒。」

夜尽，外头更鼓又两响。响沉，坛温浮一线，又沉回去。

邻床又梦呓「红」。韩泥盖席，声低：「你看错了。红在席，不在沿。」

血止在席边，沿上仍干。仍干，像缘还差半寸。半寸，靠等，不靠催。

卯时前，他搬柴半刻。柴沉，肩勒旧印。印深，像丙九五年磨出来的。磨出来的肩，才扛得住冬深。

编筐午后，藤刺浅半分再扎。浅半分，血珠仍饱满。饱满，甩向破席，席厚，吸得尽。

清栏时，赖福故意撞他肩。撞，像试沿净不净。韩泥让半步，不让席。席在，缘在。

管事瞧见，声冷：「撞也没用。沿净的人，撞不动。」

赖福气青，走了。走了，沿更净。

铁无言巡廊，灯远前一句：「沿前即止，是止给瓮看，不是止给牙看。」

韩泥「嗯」一声。嗯，像懂「牙」字指谁。

沿滑，像祖父韩老栓的手。手不在，滑还在。滑不要紧，沿上无血，无血才净。

刘婆门外咳：「沿净，好。净，才少病。」

韩泥应：「净。净，像守瓮人。」

他擦沿，沿净，沿滑。净，像六十日擦出来的规矩。规矩在沿，不在舌。

外头风息，风息了，席却更严。严，像后头那一夜预演。

他低声对坛：「你再眠。眠着，我守沿，守席，守不主动滴血。」

次日卯时，编筐末藤，藤刺偏深半分。深半分，血珠更饱满。饱满，仍甩向破席，席吸尽，沿仍净。

第二日午后，清西栏第三趟，粪锹沉得肩骨吱呀一声。吱呀不要紧，沿要紧。

赖福路过，故意踢起沿尘。尘落，韩泥不辩，只拿旧席角按住尘，再擦沿一圈。擦净，管事瞧见，声冷：「赖福，再搅，你清栏加倍。」

赖福退，退得像不甘心，却不敢再嗓。

管事验沿第二日：「沿净，记心里。心里记着，舌上不辩。」

韩泥「嗯」一声，不辩。辩，像赖福。

铁无言夜里提灯来，灯照席边湿痕，照出一线止于前三指。他低声：「止得好。止，不是怕，是等。」

韩泥应一声，像把「等」字按进掌上。

铁无言巡栏，灯晃席角，声低：「血在席，不在沿。沿上若有痕，像主动滴。」

韩泥答：「痕在席边，止沿前三指。三指外，沿尘。」

同批少年蹭袖角，声小：「沿前三指，啥意思？」

「三指外是沿。」韩泥说，「沿不接血，缘才等得久。」

少年不懂，却走了。

黄昏第二日，他掀席看湿痕。湿痕像一线墨，墨止于沿前三指，不洇沿。

老耿挑水路过，水洒稳：「沿净，好。」

韩泥应一声，不追问。问急，像邪。

他对坛低声：「血不渗沿。渗沿，是主动滴。主动滴，像借腥。」

坛温一线，不语，像应「嗯」。

第三日清晨，编筐藤刺浅半分。浅半分，血仍来。血找席，席挡沿，沿仍净。

秦霜远廊木鱼不敲，只一句：「沿前止，舌也止。舌止，符才暖。」

韩泥怀符，符暖一丝。暖，像第七笔在冬里添柴。

第二日清晨，霜白贴棚檐。檐下沿净，净得像能照见瓮眠。

他编筐、清栏、搬柴，三件活计一桩不落。落齐，沿才更净。

管事午前验沿，烛照一线：「脏一指，像拜邪。」

韩泥用旧布再擦，擦得沿滑，沿尘，沿上无血。

赖福远处啐一口，啐不到沿。沿净，啐也脏不了缘。

铁无言夜巡第三趟，灯停坛角半息：「止得好。止，是给后头留半步。」

韩泥不答，只盖席。盖严，像把半步留给瓮醒。

第三日，冬更深，丙九静一寸。静一寸，像沿前那一止。

他盖席，盖严。严，像瓮醒前夜预演。预演，不主动滴血。

外头风息，风息了，席却更严。更严，像给沿前那一止加了一道锁。锁在席，不在舌。

他低声：「席边湿，沿不接。不接，才接得住瓮醒。」

沿前那一止，止住了，像定局。定局在沿，不在急。急，像拜邪。拜邪，路断。路断，粥就凉。粥凉，恩就虚。
