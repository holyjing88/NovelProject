# 第一百一十五章 尾摊潜买

公示第二日，韩泥要去坊市一行。

不去挂牌，只拣漏。挂牌惹眼；惹眼，赖福就多一个除名借口。他要的是货，不是名。名在远，货在近——近的先抓。

铁无言立十步外，声低：「潜，别贪。贪，赖福记过。」

「不贪。」韩泥说，「沿净。」

「沿净就行。」铁无言顿了顿，「药山废渣多了半袋，换三文铜。三文够买捆旧藤——藤是幌子，漏才是货。」

「晓得。」韩泥应一声。幌子要像，像了，夜行方不算偷境。

「丑时末走，卯时前回。」铁无言又说，「回晚了，点名出空，空了难补。」

「回得及。」韩泥说。

「及了也别慌。」铁无言补，「慌了步乱，乱了被人记。」

「不慌。」韩泥道。

丑时末，他披破袄，揣玉牌，贴侧门过。守卫打盹，呼噜浅，他过得像一阵风。风不响，靴底也不响——响了，公示第三日就成真滚。

出侧门那一刻，夜风扑脸，扑得清醒。清醒好——清醒才闻得出货，闻得出温。

坊市尾摊，天未亮，摊主张灯如豆。豆灯下堆残器、残页、药槽边铁，像垃圾。垃圾里有时有温。他蹲下，鼻下轻嗅，铁片锈厚，厚里一点正温，像炉边角。

摊主睡眼惺忪：「一文，拿走。」

「温正？」韩泥问。

「正不正你闻。」摊主不耐烦，「爱要不要。尾摊货，不问出身。」

韩泥摸出一文，铁入怀，怀沉半寸。他又翻残页，页碎，碎里三字可辨：「焚火诀」。诀残，够稳窑温；温稳，培元散才正。

摊主哼：「两文。爱要不要。」

「要。」他再出两文，铁加页一并收袖。剩零藏深——藏得像藏路：路向八层，路向还芽儿那半块石。

「买藤不？」摊主忽然醒半分，「角落那捆，破，不贵。」

「要。」韩泥再丢铜，「藤编筐，人问得出。」

摊主愣了愣，像没料到丑杂役还会圆谎。他不问，只收钱。不问最好——问多了，尾巴就长。

「你不像买花的。」摊主随口一句。

「买活路。」韩泥说，「花不顶饿。」

天边白一线，早市人稠起来。白织月带婢过，婢指他：「丑杂役也逛坊？」

白织月袖风冷：「末等捡垃圾，像配。」

韩泥不辩。辩费舌；舌要留着藏铁藏页，贴墙赶回宗门。路窄，人渐多，他贴墙走，走得像影子。影子不抬头——抬头，就多一张嘴记得他。

卯时前到漏舍，赖福才不知夜行。铁页藏窑缝，铁片嵌窑口，缝窄得像席严那夜。他洗了手，坐进队里。手洗得发白——白了，像从没碰过坊市的锈。

邻床少年迷糊问：「你一早手怎么这么白？」

「洗碗。」韩泥说，「碗油。」

「哦。」少年翻身又睡。睡好——睡了就不问。

卯时散队，赖福点人，点到他：「在。晦气还在。」

「在。」韩泥只一字。

一字，手稳。赖福盯他袖角半息，找不着夜行由头——由头在腹，气不露；露了，公示第三日就成真滚。

「昨夜去哪？」赖福忽然问。

「编筐。」韩泥声平，「藤湿，晾了一夜。」

「晾一夜？」赖福冷笑，「谁证？」

「筐在。」韩泥说，「管事验沿。」

「验沿！」赖福对管事弟子扬下巴，「验！」

管事弟子过来摸筐沿，藤屑新，潮意还在：「筐实。藤是新浸过的。」

赖福噎住。噎住也好——噎住了，鞭子落不到实处。

午后废窑，他展残页。「焚火」二字像钩，钩进息，息进温，温升半分。半分之间，炼气八层门槛开一线。开一线，不声张。声张，赖福指「偷境」。

铁无言过窑外：「坊市回来的？」

「买藤。」韩泥说，「藤编筐。」

「筐实，手稳。」铁无言说，「别漏。漏了，尾摊这条路废。」

「不漏。」

「不漏就好。」铁无言眯眼，「页呢？」

「在。」韩泥不取出，「窑缝。」

「缝严。」铁无言道，「严了，八层才算你的。」

刘婆送粥至窑道，粥稀，热：「夜去哪？」

「买藤。」韩泥答。

「买藤别漏名。」刘婆哼，「漏名，赖福克。克了，饭更稀。」

他点头，喝尽。谢轻，恩重——夜行险换一文铁两文诀，值在窑温，值在八层。

「老婆子粥烫。」韩泥低声，「烫着，手不僵。」

「贫。」刘婆却又笑了半寸，「烫就多喝。喝完别晕——晕了像偷。」

「不晕。」韩泥说，「记着。」

公示第二日尽，坡下仍指他末席。指不要紧，腹里气又实半寸。八层仍立末；立末，手稳，就不滚。不滚，三年约才接得住。

他按玉牌，牌凉：「三年约，记着。坊市一行，也记着。」

记着，捡漏才不白捡。丑时练鼻下静，粟壳香淡，混一丝铁锈味——味淡，香一字不落便成。

外头赖福骂：「公示二日，明日仍末！」

「在。」韩泥低声。

在，就不滚。坛腹微温，应「捡」——捡是丑翁本事：渣里淘，尾摊淘，五年杂役里淘。淘出来了，才配九层，才配徒恩首还。

次日卯时，管事验筐，藤屑新，筐实。赖福阴笑：「实也晦气。」

韩泥捧筐交差，不辩。管事又记一笔：「筐实。」实了，夜行那一寸就盖住了。

午后，铁无言送边角渣：「掺一炉。效正，别多。」

「正。」韩泥说。

「正了别急着亮。」铁无言压声，「亮早了，人抢页。」

「不亮。」韩泥说，「先稳温。」

白织月药山道再嘲：「垃圾配你。」

「垃圾里有时有温。」韩泥声平，不多解释。温在铁片，温在残页；不露，露了成盗。

婢子嘀咕：「他居然还嘴？」

「嘴少。」白织月道，「少的人，倒难掐。」

夜，八层气走周，周周实。实像捡来的命、捡来的诀、捡来的路。路向稳窑，路向试炼，路向还芽儿。

他低声道：「尾摊潜行，记着。八层，记着。」

记着，就不白险。白险，辱白挨；不白挨，才接得住下一炉——炉里那点灵砂若剔得出来，残页才算真正落地。

更鼓沉。韩泥守窑缝，铁页不示，铁片不露。摸玉牌，牌凉贴胸：「手稳，粒正，试台见。」

见字落，坛温一线，像应。应了，路又明一寸——明向捡漏得诀，明向九层压顶，明向沈枯芽手里那半块石。

风从窑口进，带一点冷。冷里有钩：明日剔渣，渣里若真有未化灵砂，这趟尾摊就不算白潜。

他独眼半阖，掌按铁片，片温沉沉——沉着，像还没讲完的后半句。句未完，路就悬着；悬着，才够他把丑时再熬成一寸进境。进境不亮，亮的留到试台——试台那日，铁和页才算真正咬进命里。

回窑他不敢立刻展页，先编半扇筐。筐响，响像普通人。普通人夜行不被疑。

铁片贴窑口，温渗掌心。温正，火路能接。接上了，培元散才不瘸。

「门槛开了？」铁无言窗外只问一句。

「开了。」韩泥说，「不亮。」

「别亮。」铁无言道，「亮给鼻子闻，页就废。」

他把残页边角按平，平了才好藏。藏好了，明日剔渣才有底。

邻床迷糊喊：「你翻页声太响。」

「藤响。」韩泥回，「不是页。」

夜里他把三文铜的来路过一遍：废渣换铜，铜换藤，藤换口实。口实圆了，夜行才不算偷。不算偷，八层才站得住。

公示收榜时，赖福还阴阳：「逛坊的，明天仍末。」

韩泥应「在」。应的是人名，不是末命。命在窑缝、铁片、残页里——页在，路就在。

他把袖里灵石碎摸了一遍，碎仍少。少不要紧——页在、铁在、温在，石会跟着来。来了先还芽儿。芽儿石在远，钩在近：近钩是明日炉边那粒砂。

入夜铁片嵌更深，指腹发烫。烫像提醒：捡来的要守。守住了，才配把「焚火」炼进息里。息稳，下一炉才敢起。起炉之日，钩落到渣里那粒未化灵砂——砂出，这趟尾摊才算把公示的辱翻成进路。
