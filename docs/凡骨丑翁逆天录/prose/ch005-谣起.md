# 第五章 谣起

脏水的事，半日传遍泥岗村。韩泥再去井台，人散得比往日快——躲的不是他，是「晦气」二字。赵绣春那盆水，泼成了由头，落在周嫂嘴里，最响。周嫂洗衣，棒槌砸石台，咚咚，像砸人心。「叶家寡妇往晦气翁门缝送东西……」

「药铺生意淡，沾上了嘛……」

「绝户翁克邻克亲，谁近谁倒霉……」

韩泥拎桶走过，水不洒。洒了，更像晦气。童谣丫墙根排石子，念得脆：

「晦气翁，脏水翁，叶家寡妇不要命，送汤送粥晦气重——」

新词。毒在把叶青禾扯进来。韩泥脚步没停。谣是风，风无姓，不入心；人有姓，叶青禾的恩，他记了，不能让谣记脏。井台边，周嫂棒槌正欢，见韩泥，故意敲得更响：「晦气翁，你敢打水？」

韩泥放绳，提桶，水起，浑，稳稳落地。「敢。」他说，「井是村的，不是赵家的。」

周嫂棒槌悬半空，像没料到他会回。回得平，不尖，像石头落地，砸不出火星。「你……你顶嘴？」

「打水。」韩泥拎桶转身，「谣烫不了心，棒槌也烫不了。」

周嫂脸青一阵白一阵，棒槌落下，砸衣上，水花溅自己一脚。旁人窃笑，笑很轻，像怕周嫂听见。马春花凑过来，嗓尖：「你还敢回嘴？叶家寡妇都晦气了，你更晦气！」

韩泥拎桶，水不洒：「晦气在嘴，不在井。井是村的。」

马春花啐一口，啐远，像没讨着好。没讨着，也是小胜——胜在路还能走，井还能打。韩泥走了。不笑，不得意，只把桶拎直——谣还在嘴上，不在他手里。午后，赵绣春家门口聚妇人，传她的话：「见一次泼一次。」

韩泥屋里编筐，藤条勒指。门缝外飘过一句：「叶青禾也不嫌丢人！」

韩泥指尖顿了顿，低声道：「牵扯的是汤，不是人。」

他不说给外人听，说给自己听——汤是情，不是奸；情脏不了，脏的是嚼舌根的嘴。编到一半，里正巡巷，咳：「谣少嚼。嚼多了，连坐。」

赵绣春门内一滞，声哑半寸。里正不看她，只看全村：「井是村的，铺是叶家的。谁再聚众辱人，记名。」

记名二字，妇人噤了。噤了，韩泥指节松半分——松不是喜，是路还能走。傍晚，倒泔水，娃们嚷：「晦气翁出来啦！叶家寡妇也要倒霉啦！」

韩泥直腰，嗓哑，不高：「倒霉的是嘴贱的。娃小，别学。」

娃们一散。散不全，散一半——够他回屋。夜里雨意来，茅草沙沙响。门墩一响。韩泥开门，无人。地上两片姜，切得薄，像叶家药铺刀工。姜旁一张粗纸，二字，丑，却正：

趁热。不是汤，是姜。汤喝过，姜仍来——谣起，她没退。韩泥拾姜，掌心热，热得像反驳全村嘴。他进屋，姜搁窗台，把「趁热」那张纸折好，塞入袄内贴胸——不是血证，是证：叶青禾没怕谣。门外叶青禾声轻，轻得像怕人听见：「姜别省。省是穷，不是怕。」

韩泥「嗯」一声：「不怕。记着。」

脚步远去，远得像把谣甩在巷口。巷口谣还在，屋里姜在，坛在——在三，人就散不了。贴胸一息，袄内微烫，坛角跟着温一线。温应姜，不应谣。蒙尘里水汽爬半寸，爬完即敛，像瓮在睡，却听见了「趁热」。他低声道：「谣在街里，姜在心里。」
叶家药铺灯亮到夜深。帘内秤声不绝，一声接一声，稳得像钉：铺子没关，谣就没赢。韩泥在门缝外听一息，不近，近像牵扯。听，就够——够知叶丫头没退，够知自己该活长些，好还。赵绣春门内骂：「关什么铺！晦气！」

骂声尖，尖不过秤杆落盘的脆。脆响又起，赵声哑半寸。韩泥回屋，姜切片，投进热水碗。姜辣冲鼻，他喝一口，辣得眼热，热得像把全村谣烫退半寸。里正巡夜，咳：「谣少嚼。」

赵绣春门内一滞，声哑。哑，也是小胜——胜在还能喝一口姜汤，胜在叶家秤还在。退半寸够。够他明日还去井台，还去集尾，还编筐。雨又落一阵，停一阵。停时，他摸坛沿，沿温不退，像瓮替他守着「趁热」——守着，就不怕谣一夜吞村。韩泥低声道：「谣在嘴上，姜在心里，坛在角上。三样都在，人就散不了。」

贴胸「趁热」印痕浮，浮得像字在呼吸。呼吸轻，恩重。重，便活长些，好还。叶铺最后一声秤落，落得像给全村盖章——章盖「没退」，章盖他该活，该还，该走到宗门那日。沈枯芽隔门问，声小：「韩爷爷，姜辣不辣？」

「辣。」韩泥说，「辣好。辣，谣退半寸。」

芽儿笑一下，笑浅，像恩。恩浅，也真。他摸坛沿，沿温不退。不退，像瓮替他守着「趁热」——守着，就不怕谣一夜吞村。巷口周嫂还洗衣，见韩泥倒泔，故意嚷：「晦气翁，叶寡妇还给你送姜呐？」

韩泥盆稳，泔水不洒：「送的是姜，不是谣。谣你洗不掉，姜我能收。」

周嫂棒槌一滞，脸青半寸，没再接。赵绣春门内接，接得短：「关什么关！」

叶铺秤声落，落脆，赵声又哑。韩泥回屋，对门缝不应，只对窗台上姜——不对姜，对自己说：「她没退，我不散。」

陈姑路过，领拢紧，低声：「嘴贱的，烫不死心。心热，才行。」

韩泥「嗯」一声：「心热。靠姜，靠秤，靠活。」

里正巡夜又咳，咳过巷口：「叶家铺子开着，谁再关，记名。」

赵绣春门内一滞，骂声咽半寸。咽半寸，够他贴胸那张「趁热」再暖一息。远处秤杆又落一声，落脆，像替谣摁了章——摁「没退」，摁得赵家骂声又哑半寸。叶青禾帘内一句：「秤在，谣进不来。」

韩泥「嗯」一声，没抬头：「进不来，也得防。」

他低声道：「谣在舌上，汤在心里。心里不凉，路就不断。」

谣在巷尾拖长，他编筐不辍。新词借旧调，调在井台——台边嘴多，理少，少理就多记。记谁推谣，记叶帘还动，动像秤还在，秤在，铺就不关。谣在巷尾拖长，姜在心口发辣；他编筐不辍：街上有新词，心里有旧秤——秤在叶铺，铺不关，公审前锤未落，汤就还能递进门缝。**状态**：大境·凡人 · 资质·丑骨末席（伪灵根） · 鸿蒙九劫瓮·眠 · 洞府·漏舍凡舍
