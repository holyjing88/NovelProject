# 第五十一章 先验商队

三日后，卯时，廊下霜白。霜白，像柱上三日条落了地——条在耳，条在柱，今日要验。末排先验，韩泥立最前。前，不是荣，是险——险在「滚」字，滚了，麸皮停，记功抹，七笔恩断。丙九列队，赖福在尾，尾里窃语。语轻，像等晦气滚。管事敲锣两声，声落，廊静。静，像辨香半刻前那口气——气在鼻，不在嗓。秦霜立廊角，木鱼不敲，只托：「辨香半刻，报香一字不落。漏一字，记过；记过两次，滚。」

韩泥木牌在怀，帖在袖。袖里三日条已作废，作废，像昨日数完。数完，今日只验鼻。香囊三只，韩泥闭息，鼻先静。静里，粟壳淡，丹烟苦底，第三囊甜腻藏腥——腥在底，不在面，像栖香谣散了，散成规矩。「粟壳淡，丹烟苦底，甜腻藏腥。」他声哑，声平。秦霜「嗯」一声，不夸，只记簿。记簿，像过第一关。铁无言立柱边，验他鼻下静气，声低：「静，别贪快。贪快，香飘。」

韩泥「嗯」一声。嗯，像心口七笔压着——压着，手就不抖。半刻尽，韩泥一字未漏。未漏，末排先验过——过了，不是灵根好，是鼻下稳。秦霜目光扫过队尾，扫到赖福，声冷：「再嗓，记名。」

赖福哑了。哑，像秤空一格——一格空，韩泥肩上的路却实一寸。管事贴新条：「辨香过，候凝丹试手。试手在三日后。这三日，仍搬柴。」

韩泥接条，条薄，真。真在「仍末」，真在路未断。---

午后，宗门外驿道，尘起。尘起，像冬里商线活了——活了，不在门内，在门外。西驿商队过，驼铃远，车辙深。深里，五脉旗号杂：佛门灰、密宗赭、西教白、密院黑、沉丹青——青是主场，杂是客。五脉齐，齐不要紧，要紧是界。界在门内门外，门内末排，门外商线。韩泥搬柴经侧门，侧门窄，窄得只容半眼。半眼，够看商线。驼队里有人咳，咳得寒，寒像北风贴骨。骨寒，商队却不停——停不得，停，货腐，命也腐。佛门灰旗前，有人施粥，粥稀，热。热在客路，不在丙九棚——棚里粥，刘婆端。密宗赭旗后，褐袍人坐驿亭，坐得不急。急便邪，不急，像等冬寒够深。西教白旗风里猎猎，猎猎不是谣，是界。界远了，谣才借不到腥。密院黑旗压车辕，车沉，沉得像克扣的麸皮——克的是粮，克不走商线。沉丹青旗最末，末不要紧，要紧是五脉都过这一道门。老耿挑水路过，水洒半勺，洒得稳。稳，像抬棺绳不抖。韩泥「嗯」一声招呼。招呼轻，耿不回话，只抬下巴指商队：「五脉齐，冬里货紧。紧，别凑秤。」

「不凑。」韩泥说，「凑秤，手飘。」

老耿又指宗门外驿亭，亭矮，亭里褐袍人裹袍，袍角绣极淡密宗纹——纹丑，真，真像老耿赠经片那日那片经片。韩泥心口一动，动，不像邪，像路：密宗线不在村庙了，线在宗门外，线等着冬寒落地。商队过去，尘落。落尽，驿亭褐袍人未走，只坐，坐得像等一个搬柴的丑杂役路过第二趟。韩泥不凑。凑，像贪。贪，恩易断。他第二趟搬柴，仍不凑亭。不凑，像廊下讲规矩那日廊下避腥——腥沾袖，不贴胸。褐袍人抬眼，眼淡，淡得像凡物看人。看人，不像看仙缘，像看骨寒。韩泥独眼平，平过去。平，是守。黄昏，丹房外，铁无言翻簿，见他一眼：「鼻下稳，三日后再验手。手抖，仍滚。」

韩泥「嗯」一声：「手记着七笔，不抖。」

秦霜声从远廊来：「五脉商队在，宗门戒躁。躁者，借商线生谣——谣借腥，你别沾袖。」

韩泥独眼平：「腥沾袖，不贴胸。贴胸的，记在心里。」

她点头，像廊下讲规矩那日那日。那日规矩在耳，今日规矩在商——商在门外，门内仍末排。同批少年碰他袖：「你真过了鼻子？」

「过了。」韩泥说，「过鼻子，不过命。命在坛。」

「坛？」

「坛在，才验得过第二关。关过，才凝丹。凝丹，先还热的。」

少年不懂「热」，闭嘴。闭嘴，韩泥不解释。解释轻，恩重。夜，韩泥对坛，声低：

「第一关过了。过了，才等试手。试手前，冬来了。」

坛腹温一线，应「香」字，不应慌。他是记心翁，恩仇都记在心里。心里七笔，分列，不混。混了，手就飘；不混，手就稳。他低声道：「辨香过，商线现。现，不乱心。心乱，恩断；恩断，汤凉。」

外头驼铃余音散尽，散尽，褐袍人影还在驿亭——在，像密宗线从村庙挪到宗门，挪得不急，急便邪。柱上旧条作废，新条写「三日后试手」。试手在，商队在，坛温在——在三，命还在末排窄路上。窄路也是路。路通向凡符，通向寒不侵，通向瓮醒前那几夜。刘婆门外咳：「试手前，别空肚。空肚，手抖。」

韩泥应：「不空。粥在，姜在记里。」

丑时，坛腹又温一线，温短，像瓮听辨香过耳。过耳，不应灵，应规矩。外头风硬，风硬了，柱上试手条却更真——真在末排，真在验手，真在「滚」字后那一条窄路。他低声道：「三日，试手。试得过，关才开第二道。关开了，才学凝丹。学了，先还烫。」

坛壁水汽爬半寸，敛。敛着，像替他守着：香已过，手别飘；手不飘，恩不断。恩不断，叶汤就还在前头。木牌在枕下，与试手帖并放。并放，像命与规矩挨着——挨着，才验得过。邻床梦呓：「香……」

韩泥不答。答轻，恩重。重，记在心里，不在舌。商队过后，驿亭空半刻。空，像留一道界——界内末排搬柴，界外五脉换货。褐袍人袖里黄纸角露一线，露，像凡物藏锋。韩泥瞥见，不凑，凑像贪。老耿挑水第二趟，水洒稳，声哑：「僧不入门，符不入丹房。入门，像邪。」韩泥点头。点头，像把界记住——记住，才不借商线腥。黄昏回棚，坛腹温比晨多一丝。多一丝，像瓮听辨香过耳，听商铃远，听末排仍活。<!-- v42-topup -->

商队验讫，铃远去。他袖洗三遍，三遍净了，胸才贴坛；坛应一线温，不应慌。西驿商队铃轻，铃轻像界——界内末排，界外五脉。五脉在，心不乱；乱，腥借不到手。商队验讫，骡铃远去。韩泥仍绕驿亭三步，像界——界内杂役，界外五脉。管事嘱：「商线腥，袖洗三遍。洗不净，别近坛。」他近坛，不近亭。午后第二趟搬柴，肩勒旧印，印深，像五年杂役磨出的实。褐袍人袖里黄纸边毛，毛得像凡物。凡物不急，急像邪；等冬深，才赠。赖福讽：「看商队能看出仙缘？」韩泥不答。夜，柱上商队条在，条像倒计时。倒计时在柱，七笔在心，木牌在怀。他低声对坛：「线在界外，心在界内。界内，手别飘。」坛不应字，只温一线。风息了，符未至，怀仍空。空不要紧，沿净，席严，才接得住后头缘。商队领队验秤，秤杆稳，稳得像叶家那杆。韩泥旁站，不凑，只记：商线在，心不乱。铁无言低声：「五脉商线，腥在袖，不在心。心在，符才接得住。」

夜雪粒细，落肩即化。化不要紧，肩沉，沉得住，冬深才赠。<!-- v42-topup -->

商队铃远去，五脉线在界外；韩泥不借腥，袖洗三遍——三遍净了，冬深凡符才接得住，接得住，才守瓮醒那一误。。<!-- v38-end -->
