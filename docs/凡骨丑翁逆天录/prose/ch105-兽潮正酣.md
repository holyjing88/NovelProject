# 第一百零五章 兽潮正酣

潮中没有「中间」好看。好看是说书人的话。这地方只有第二更、第三更、第四更——更尽，障破了补，补完再破。破在障上可以，破在心里不行。

心要稳，稳在角。角在，韩泥就在。

铁无言臂伤又重一线，重了仍挡。挡时韩泥拿灰香残渣清路，清路像清栏——清栏他熟，熟了手才不飘。「左漏！」铁无言猛喊。

韩泥左挪三步，三步撒灰。灰尽，尽了还有诀。诀出雾起，雾里甜毒退半寸。退半寸，栏里少倒两只小兽。两只不算功，只算活。

赖福被派来运石灰，箱子撞着肩，路过东角时瞥了韩泥一眼。瞥像不认——不认也行，瞥里终究记下：丑杂役没逃。记在眼里，比记在嘴上阴，也比记在嘴上真。

管事弟子传令：「杂役退二步！内门上！」

韩泥退。退到脚尖仍能碰到障角。碰着，角就不散。散了才是真退，真退他就不会干。

白织月从内墙递过来一包伤药，药品阶低，低得像临时起意，声音却硬：

「不是谢你。是别死在内墙前，晦气。」

韩泥收下，不谢。谢轻了像交易。他把药拆开，直接裹上铁无言臂上那道口子。铁无言抽气，又笑骂：

「药正。你嗅的？」

「嗅。」韩泥说，「嗅完，手稳。」

「稳就包。」铁无言道，「包好了继续干。干完再疼。」

臂包好，好了才配继续并肩。并肩不是口号，是一个人喊漏、一个人补漏。

兽影再贴上来，爪风带毒。毒一近，韩泥含下清瘴丸第二粒。粒尽，还有第三粒——第三粒他留着，留给还得续扛的更鼓。留不是抠，是怕后更突然空袖。

第三更他避到角背。背着障，障挡住别人的眼，他才敢掌炉再炼。火起半寸，丸成两粒：一粒自服，一粒掷给邻角同袍。同袍接住，吞下，嗓子颤：

「……谢。」

「别谢。」韩泥道，「手稳。稳了接着站。」

同袍眼红，点头。点头像链子多扣一环。环还不圆，圆要等到后头真正并肩清窝的时候；此刻只求站着。

「还能炼吗？」铁无言问。

「能一点。」韩泥抹去掌灰，「不能多。多了炉飘，飘了比兽可怕。」

「那就一点。」铁无言矮身挡爪，「我挡你炼。」

第四更障又开缝。韩泥糊缝糊得满手泥灰，泥灰里混着血。血糊住指节，指节反而更稳——稳得像记着：破的是障，不是心。

管事弟子二次传令：「再退！退到标线后！」

韩泥退半脚，脚后跟仍蹭着角石：「退到这儿。再后，角就没人了。」

「你抗令？」

「抗角。」韩泥说，「角没了，令也没用。」

那弟子愣住，竟没再逼。逼也逼不动一根认死理的桩。

潮中的夜特别长。长得人听见自己牙齿打颤。韩泥不颤嘴，只颤袖里空囊——空了提醒他：后手在丸，在诀，在铁无言那条还在挡的臂。臂上白织月的药味淡淡的，淡里有干净，干净就好用。

「丑翁。」铁无言忽然叫他，「要是我倒了，你别扶太久。扶久了你也倒。」

「你倒不了。」韩泥道，「倒了谁挡左漏？」

「……嘴硬。」

「硬才能顶。」韩泥说完，自己也觉得这句土。土就土，潮中用不上漂亮话。

卯时将近，兽吼忽然矮下来，矮得像退潮。退潮不能信。信了，人一散，回头就是血。韩泥仍立着，立得像桩。桩裂了皮，裂是活计。

回棚时，刘婆端着稀粥堵在道上：「还活着？」

「活着。」韩泥说，「手稳。」

「饭稳，恩不断。」刘婆把碗塞他手里，「喝。喝完再续命。」

韩泥「嗯」一声，粥烫唇，烫得心里那根弦松半寸。松半寸不是松懈，是还能感到热。热着，就还没被潮腌透。

他坐到席边按坛，低声道：「潮中过了。还没清。」

坛腹温着，像应「中」字。中不是终。终得把余瘴钩净，把丸炼真，才能算这一段合上。丑时香落齐了，掌纹又烫一线——烫像五层实到头，头上仍不破境。不破也好，破在这种夜里最容易邪。

邻床不知何时睡实了。睡实像信他的角还稳。稳是给人偷来的一点觉，他不怨。

更鼓尽，界外腥味仍沉。沉像还要续。续不是吓他，是功课还没写完。他低声对自己说：潮中。下一程，续。续的时候手要在，铁无言要在，清瘴那条路要在。

铁无言靠在门框上包布，布渗淡红：「你丸还剩吗？」

「一粒。」韩泥道，「留给真压死的时候。」

「留给你自己。」铁无言哼，「我有伤药。」

「伤药外抹。」韩泥摇头，「毒往里走时，还得丸。」

两人对望一息，没再争。争输赢不如争口气。

门外有人传话：「东角换人？换不换？」

「不换。」韩泥先应，「人换角不换。角黑了，黑是汗，是爪，是毒。毒在，人就还在这儿顶。」

传话的走了。走时嘀咕「疯子」。疯子也好，疯子才肯在潮中把「还」字咬着不放——还刘婆的粥，还铁无言并肩那一眼，还那包说「不是谢」的伤药。药不是谢，他偏记成恩。记成恩，才知道往后怎么还。

夜风把柱上旧条刮得哗哗响。旧条墨淡了，「备战」俩字仍能认。认着，他就知道规矩没散；规矩没散，下一更续上去才不算乱跑。他摸了摸怀中坛沿，沿温，像催他歇半息再出去——歇得，但别睡死。睡死了像忘恩；潮中的人忘不起。明日界外若还腥，腥就是钩，钩着他把「续」字往实里做。

潮中第五更，兽影稀了，人反而更乏。乏得眼皮打晃。韩泥用舌尖顶住上颚，顶醒自己。铁无言靠着障木喘：「稀了别追。追出去，死得难看。」

「不追。」韩泥说，「清路。路清了，人才能后撤整队。」

「你还懂阵？」

「懂活命。」韩泥把石灰薄薄洒一线，「活命的阵，最土，也最管用。」

赖福又晃过来，簿子夹腋下：「东角还在？在就画圈。画完，赏不赏另说。」

「别画赏。」韩泥道，「画『在』。在比赏实。」

「你可真不会做人。」

「会做角。」韩泥不看他，「角做人，我做活。」

白织月第二次派人问伤：「那包药够不够？不够……我再说，不是谢。」

韩泥对外门跑腿的点一下头：「够。臂肿退了。退了就能挡。」

「她让你亲口应。」

「应了。」韩泥抬声，刚好让内墙听得见，「药正。人还在。」

内墙那边沉默一息，像把「晦气」俩字又咽回去。咽回去也好，潮中用不着干净脸。

沈枯芽不知钻到多后，缩着脖子递来半块饼：「刘婆逼我送来。说吃了才有力气还……还她粥钱。」

「粥钱日后算。」韩泥咬一口，「你现在滚回去。滚慢了，赖福记你。」

「韩叔你……」

「滚。」韩泥眼一横。横完又补软半句，「告诉刘婆：饭稳。」

第六更末，邻角同袍忽然跪下去。跪不是跪神，是腿软。韩泥把最后半撮灰塞他鼻下：「闻！苦的。苦了就醒。」

同袍猛咳，咳完眼泪出来：「醒……醒了。」

「醒了就站。」韩泥拉他一把，「站着等比躺着强。躺着，潮会踩。」

铁无言瞧见，嗓子哑里带笑：「你还收徒？」

「不收。」韩泥道，「只借一眼。借着眼，大家都少死一个。」

更鼓再沉时，他知道「中」字快写完了。写完不是散场，是把笔递给「续」。续更脏，更磨人，也更要丸、要诀、要并肩。他按着坛，对铁无言说：

「下一程若续，你左我右。右若漏，你喊。」

「喊。」铁无言点头，「你也别死在中段。死了，我少个会闻药的鼻子。」

「不死。」韩泥把木牌按回胸口，「牌在，人在。人在，明日的条才看得下去。」

界外腥风又拱一下。拱得像余潮回头探。探就探，他钉子还在。钉子不拔，续字就有地方落。
