# 第四十章 赤焰逼近

赤焰使者二次上门，旗红，马嘶，像把七日之期又插回山门。

韩铁山抱拳，礼数足，刀意不足：「宗主，三日之内，东脉划界。否则夜袭，妖骑先行。」

柳青鸢沉脸：「东脉不让。」

「那便战。」韩铁山笑，笑里刀，「莫长老九月……哦，听说还剩四月？将死之人，也配谈战？」

全场一寂，寂里有人偷瞧莫长春。

弟子围山门三层，像等一场戏——上回手帕定身，这回韩铁山当众辱九月，看长老如何接。

莫长春拄杖，声轻却清：「配不配，试过才知。礼还没送完，你们倒急了。」

韩铁山扬眉，未料九月老头接话接得稳，稳得像不接辱，只接刀柄。他驱妖骑耀武，蹄声震阶，像给三日之期敲鼓。妖骑在阶下踏圈，腥风扑面，弟子后退，后退里又有人咬牙往前，像半块灵石撑着不退。

霍镇山盾光微亮，骂声起：「韩铁山，耀武给谁看？给田看，还是给死人看？」

柳青鸢指节发白，未拔剑，剑意已冷。莫长春却只侧脸对柳，低声：「宗主，该送的礼，还差几件。」

「什么礼？」

「给丹堂的，给杂役的，给赵小子的。」他笑，「还有给赤焰的——他们急，我们不必急。」

韩铁山听见，冷笑：「莫长春，夜袭子时，看你还急不急。」

莫长春点头，像谢提醒：「子时好。省得老夫敲门。」

全场哗然，哗然里，韩铁山笑意一僵，像被反刺了一刀，刀不深，却痒，痒在弟子眼里，痒在「将死之人竟敢约子时」的反差里。

魏无涯在檐角，扇合，低笑一声，像看戏看到高潮前一刻。

周德海立后阵棚，令旗在指间，迟迟不落，落得像犹豫，犹豫得像奸。

柳青鸢剑意冷，未拔剑，剑在鞘，威在眸。莫长春侧脸对柳，声轻，却全场能听：「他们急，我们不必急。」

韩铁山拂袖去，旗转，像血拖过山门。

寿元在塔壁跳：四月。莫长春瞧见，不慌，像瞧见季节换牌。廊下弟子炸开议论，议论里怕，也燃——怕夜袭，燃的是长老敢接子时。

顾小满在杂役堂分班，听「子时」二字，眼里亮，亮得像雨夜那道光还在等。她不知自己将走暗线，只知东脉不让，杂役也不能退。

林挽月罚役途中，听山门哗然，唇白，像终于看懂：韩铁山辱九月，辱不出莫长春退；退的只能是赤焰旗，不是青岚根。

苏念慈在丹堂听讯，炉火旺，旺得像替山门也担。她低声：「子时丹堂最危，炉不能熄。」

厉无殇在帐内听韩铁山回报，笑冷：「将死之人约子时？好。本少谷主也省得敲门。」

魏无涯传讯，只八字：「莫长春，不急，急的是你们。」韩铁山阅讯，指节白，像被反刺一刀，刀不深，却痒在「赤焰竟先急」的反差里。

三日之期缩短成一夜，一夜缩短成子时——子时之前，全宗都在等，等韩铁山笑还能挂多久。

赤焰逼近那日，旗红映云，云像也被染了三分。

柳青鸢佩旧剑，声洪：「逼近不为好看，为活着。」霍镇山按刀：「不让，就战！」

赵玄机铁牌在腰，笑浅：「赤焰势大，门里该统。」顾小满声平：「势大记下了。人不退。」

苏念慈开炉，炉旺，旺得像把逼近炼成苦甘。她低声：「逼近别传成慌。」

莫长春不辩，只巡，巡像把逼近走成礼——礼不让，阵才稳。

柳青鸢声洪：「活着，才谈礼。」

外门弟子比划定身三息，内门弟子摇头：「你们没瞧见，韩铁山这句话，比妖骑还腥。」「像捅刀。」「看长老接不接。」

山门风腥，旗红拖血。弟子窃语：「四月够吗？」「够不够，子时见。」莫长春塔壁无新铭，塔影却沉，像又浮半句金铭：辱九月者，子时还。

子时将至前，霍镇山布盾于东脉隘口，盾光连成片，片里照着赤焰旗远去的血痕。年轻弟子问「莫长老真要守子时」，莫长春答：「不急的，是他们；先急的，往往先露怯。」

厉无殇帐内，韩铁山呈回报，笑冷催刀：「三日变一夜，一夜变子时。看九月老头还稳不稳。」厉无殇只道：「稳不稳，刀上见。」

廊下议论未止，却多了一层静：辱九月未成，赤焰反而被约到了子时——约到了，全宗都在等那一声蹄响。

山门比试另行日，外门弟子仍拿「子时」当梗：「死人约活人刀，像不像戏？」内门嗤笑：「戏？你上去接韩铁山一句试试。」

赵玄机在思过崖听风声，风像把「辱九月」吹到他耳里。他指节叩石，忽然笑了一笑，笑得冷：「辱不出退，倒把全门激成一贯——赵氏若此时统，统的是乱，不是势。」

苏念慈遣丹童往各脉送预防烟火咒，丹童跑到山门又折回，回得像怕丹香惊了赤焰旗。苏念慈只道：「炉在，就不慌。」

顾小满在分班册上添「子时」二字，又画圈，圈像暗线的第一笔。她低声：「四月也好，子时也好，人不退。」

柳青鸢令各脉清点烟火咒与巡哨，声稳：「子时若至，先护田，再护名。」霍镇山在隘口试盾，盾光映旗血，映得像把三日之期也映短了一寸。

魏无涯传讯厉无殇，只六字：「急者先露怯。」厉无殇阅罢，笑冷，笑里却多了一丝疑——疑在九月老头为何不接辱，反接刀。

东脉隘口夜风硬，硬里夹着腥。霍镇山令弟子轮值，值得像把「子时」二字先刻在土上——刻不深，够醒。

莫长春巡至丹堂外，声轻：「炉在，阵在，礼在。子时到了，看谁先敲门。」

山门比试场边，弟子仍拿「子时」当梗，梗里却藏怕——怕夜袭，也怕九月老头真敢接。有人低声：「上回手帕定身，这回定的是刀。」

顾小满将分班册呈霍镇山，册上「子时」二字圈了三遍，圈像暗线将起的第一笔。她不知自己将走暗线，只知杂役不退，线才直。

韩铁山旗远后，东脉土上多了一排新蹄印，印深，像三日之期已先踩进泥里。

寿元在塔壁跳：四月。莫长春瞧见，不慌，像瞧见季节换牌，也像瞧见礼还没送完——送完了，才轮到刀。

赤焰驿馆送来第二场挑衅：一头妖禽掷在山门阶下，掷得像把「子时」也掷到众人眼前。霍镇山一脚踢开，骂：「畜生耀武，人也学着耀武？」

厉无殇帐内阅讯，笑冷，却多了一丝疑——疑在九月老头为何不接辱，反接刀。他令韩铁山：「子时前，再掷一旗于东脉田埂，看青岚护田，还是护名。」

莫长春巡至东脉隘口，对霍镇山低声：「旗掷田上，田在，人就在。人散了，名也护不住。」霍镇山骂声起：「好！田在，盾在！」

韩铁山三日之期，像帖在东脉土上，土还温，温得像血旗先舔了一口，一口腥，腥在子时前。

莫长春巡至东脉隘口，对霍镇山低声：「旗掷田上，田在，人就在。」

魏无涯传讯，只八字：「莫长春，不急，急的是你们。」像把辱九月的刀，反递回赤焰手里。

柳青鸢令各脉清点烟火咒，声稳：「子时若至，先护田，再护名。」像把「不急」二字，也写进了宗主令里。

韩铁山旗远后，东脉土上多了一排新蹄印，印深，像三日之期已先踩进泥里。寿元在塔壁跳：四月，像季节换牌，也像礼还没送完。
