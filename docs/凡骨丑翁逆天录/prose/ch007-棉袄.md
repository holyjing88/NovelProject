# 第七章 棉袄

入冬前夜，泥岗村冷得早。韩泥屋漏，风钻缝，像针，一针一针扎麻脸。藤条散了，编不成，指僵，僵得握不住针。棉袄若不来，这一夜难挨。他呵气暖手，气白，白得像短命的热。门蹭了一下。蹭得轻，像怕惊动晦气。陈姑七十出头，背驼，眼浊，抱一件棉袄，旧，洗得发白，絮从袖口钻出，钻得像白发。袄上有皂角味，皂角味旧，旧得像洗过许多冬。「穿上。」嗓沙，像砂纸磨锅。韩泥没接。陈姑瞪他：「嫌旧？你身上补丁摞补丁，算新的？」

「不是嫌。」韩泥嗓哑，「是怕。」

「怕什么？」

「怕连累你。」

陈姑哼一声，把袄往他怀里一塞，塞得实：「连累？我七十了，还怕连累？你绝户，我孤寡，凑个伴。你死了谁给我烧纸，我死了谁给我摔盆？」

她没说完后半句，韩泥懂。他接了。棉袄沉，暖，像一捧实在棉花。指尖碰她手背，皮糙骨硬，硬得像老树根。「陈姑……」

「别谢。」她摆手像赶鸡，「我不是为你，为我自己。你穿活了，我死后才有人记得给我点一炷。」

「棉袄我收。记在心里。」

陈姑哼一声：「记什么，穿活了再说。」

她停一步，又回头，眼浊里亮一下：「别冻死。冻死了，我白送。」

「不冻。」韩泥说，「穿活了，心里记着。」

陈姑哼一声，转身走，风一刮，晃一晃，像一片旧布在村道上飘。韩泥穿上，棉絮味旧，暖进骨，骨缝里那阵僵，松了半分。袄袖长，长出一截，他卷了卷，卷得像把寒挡在门外。屋里暗，暗里他抬臂，臂弯棉絮窸窣，窸窣得像有人陪。心里又添一撇——陈姑这一件袄，短，像袖口絮，暖得真。叶汤、赵辱、芽粥、陈袄。四恩里的第三笔，未到齐，心已齐一截。他出门打水——冷天夜里去井台，头一遭。棉袄领子竖着，竖得像一层薄盾。村人见棉袄，让了半步。半步够走，够打，够让陈姑那袄在村道上有用。有人低声：「晦气翁也配穿袄？」

另一人哼：「你配嚼？」

声息散了，散得像风咬不动这半步。井台边周嫂在，嘴动，想说什么，见棉袄，闭了。闭得紧，像把话咽回肚里。「晦气翁也穿衣了？」周嫂旁有人嘀咕。韩泥拎绳，放桶，水起，浑，稳稳落地：「穿衣，也打水。井是村的。」

周嫂棒槌悬着，终是没砸响。旁边妇人低声：「陈姑送的吧？」

周嫂脸一沉：「少嚼。」

韩泥打水，桶满，拎回。水晃，晃在桶沿，不洒。洒了，更像晦气。他不辩，不炫，只穿。穿本身就是答：晦气翁没冻死，有人把他当人。周嫂终是开口，声硬：「陈姑糊涂，你也糊涂？」

韩泥停半息：「糊涂的人，不打井。」

「你打井，你穿衣，你还真当自己是村人？」

「井是村的。」韩泥说，「衣是陈姑的。我打水，不偷不抢。」

周嫂棒槌一落，落回盆里，溅起水花：「算你识相。」

说罢走，水在桶里晃，晃出稳声。稳声像小胜：井还能打，路还能走，袄还在肩。井台边有人窃语：「陈姑老糊涂了。」

另一人哼：「糊涂？人家清醒。绝户没人管，她管。」

韩泥听见了，不回头。回头像认。认不得，认恩就行。后半夜雨漏。雨打棉袄肩，重些，仍比单衣强。灶边坐，姜与粥的余温似还在胃，棉袄加一层，像把恩捂实。漏雨滴在灶沿，滴成细线，线断，断在坛边。坛沿微凉，凉得手指一触，沉。他低声道：「冬来了，人没冻死。」

外头陈姑家咳两声，像确认她还活着，他还活着。活着，恩就能还；能还，就不白收。他忽然想：陈姑七十，还肯送袄，送的不是袄，是「还有人把你当人」。泥岗村少这个，少，才贵，贵，就得记。记下了，就不怕冬。回屋路上，冷风刮脸，麻脸疼，疼得清醒。清醒里，他想起张麻婆骂街尾那句「筐编得结实」——结实，就能卖，能卖，就能活。活路一条一条叠，叠成心里几笔恩，笔丑，真。雨歇时，村狗叫一声，又哑。韩泥在门内，棉袄肩还潮，潮得暖。暖进骨，骨里像多了一根撑。他按了按心口，陈姑这一撇，短，真。真恩压虚风，压得住。明日还要去集尾，还要去井台。去的时候，袄在，路在，人不冻。他合眼，合得浅。浅梦里没有空白页，只有棉袄肩上一片潮，潮得像陈姑没走远。没走远好，走远也记着——记着，冬就不白来。梦里陈姑嗓沙，沙一声：「穿活了。」

韩泥在梦里应：「穿活了。」

醒时雨已歇，歇得像冬退一步。退一步，够人活一夜。韩泥低声道：「陈姑，我记着。袄在，冬就不绝。」

外头风又起，起得短，像冬在试门。门内棉袄还暖，暖得像有人替他把住这夜。次日，韩泥担水过陈姑门口。门开一线，陈姑探半张脸，眼浊，却亮：「还暖？」

「暖。」韩泥停一息，「袄在，人没冻。」

「别谢。」陈姑摆手，「谢了，马春花又有一嘴。」

韩泥应声：「不谢。记着。」

陈姑哼一声，门又合。合得轻，轻得像怕人看见她管绝户翁。韩泥担水回屋，棉袄肩被桶绳勒出一道浅印，印浅，暖还在——暖在絮里，絮在恩里。路过赵绣春墙根，有人窃语：「陈姑老糊涂，绝户也送袄？」

赵绣春嗓尖，尖不过桶底碰石台的闷响：「送袄？送晦气！」

韩泥不辩，只把桶放稳，水不洒。洒了，更像晦气。里正巡巷，咳一声：「井是村的，袄是人送的。谁再嚼，记名。」

赵绣春门内一滞，声哑半寸。哑半寸，够他肩上的暖再实一分——实了，第三笔恩就不白收。井台边，张麻婆骂街尾瞥他一眼：「穿活了？别冻死，冻死抬棺累我。」

韩泥只答：「活。袄在肩。」

张麻婆哼更大，却没再骂。骂是邻，邻还在，冬就不算绝。马春花巷口嘟囔：「送袄？迟早扔沟。」

韩泥肩不动：「袄在身，沟甩不掉。」

陈姑门外咳一声，不高：「穿活了，别糟蹋。糟蹋，老婆子白送。」

韩泥应：「穿活了。袄在，冬就不绝。」

坛沿一线温，贴掌不退。不退，像替他守着陈姑这一件袄，守到还得上的那日。

陈姑棉袄贴背，风硬短半寸。第三笔恩落了，冬来人不冻——不冻，才走得到井台，才接得住下一碗「趁热」。叶家药铺灯弱着，弱也是亮；亮着，叶汤就不倒。
