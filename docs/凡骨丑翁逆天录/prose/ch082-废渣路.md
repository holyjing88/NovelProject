# 第八十二章 废渣路

暗袋七包后，铁无言指了一条路。那路不在册，在口。口轻，记重：「药山废窑，半塌，杂役可过，不可占。过，捡渣试温；占，除名。」

废窑门槛外，他不入窑深。入深，像占；占，除名。只立门槛外，掌贴门槛石，石凉，凉进骨。凉不要紧，炉温应掌，应得像坛腹温一线。

试温半刻，渣灰里亮多一丝，多不要紧，多也是路。路从药山起，经暗袋，指废窑，指95匿丹，指110首丹——远不要紧，路头在十包渣里。

臾墟残念弱一息：「……渣里活，路在渣。路在渣，别叫丹。」韩泥不答，答在心里——心里记废渣路，路在渣，不在舌。

过，不占。韩泥熟这规矩——熟，像界碑，像沿净，像坛不露。卯时前，他绕药山侧径。径窄，霜白，霜下碎石响半声。响不要紧，脚轻才要紧；脚重了，赖福就嗅「偷占」。废窑在坡背，窑门塌半扇，门内灰热一线——热不是火，是余炉温。余炉温淡，淡不要紧；淡也要记。他不入窑深，只立门槛外。外头风硬，硬里带苦辛。

鼻下有根，沉腐浮三字不落舌，指捻一撮暗袋渣，渣灰，灰里一点亮。亮像未烬引火，引火轻，记重。记重了，废渣路才走得通。他低声道：「试温，不占。」

不在舌上，在掌里。掌贴门槛石，石凉，凉进骨。凉不要紧，炉温应掌，应得像坛腹温一线——温不应字，应「路」。二层气走指梢，指梢温半分，温进渣灰。渣灰里亮多一丝，多不要紧，多也是路。铁无言立径外，声低：「温到了别贪。贪占窑，路断。」

韩泥应声：「不占。试温。」

试半刻，收指，收得像盖席。盖严，渣仍回暗袋，袋沉，沉才像攒。回坡下，管事验足迹。足迹不过窑深，深内一步无印——无印，是小胜。「丙九韩泥，过窑试温，未占。」管事记簿，笔丑，真。赖福讽远：「废窑也过？过出邪炉，连坐丙九。」

「韩泥不接茬。接茬，全丙九饿。沿净。」午后，药渣场。他把暗袋七包并成十包——三包新收，新收不要紧，路才长。路长，像废渣路：从坡下捡，到坡腰捡，到废窑试温，到暗袋攒，到将来开炉。同批少年嗤：「丑杂役也过废窑？过出丹了？」韩泥不辩。辩，费舌；舌要留来记路，留来守坛。他只答：「试温。

」少年不懂「试温」，却懂「不辩」，闭嘴了。刘婆端粥，仍多半勺：「废窑去了？」「去了。」韩泥说，「试温，不占。」「不占怕什么。」刘婆哼，「手稳，饭稳。饭稳，恩不断。」

韩泥点头。恩在粥里，也在「不占」二字里——二字像规，像界碑，像坛沿净。

黄昏，他编筐半刻，藤刺扎掌，偏手，刺浅些。编筐钱买米，米在，恩才还得上——还得上，才对得起废渣路这一步。

筐编完，他廊下立，任风硬，鼻下仍静。静里，废渣路像线，线从药山起，经暗袋，指废窑，指95匿丹，指110首丹。线远，不要紧；线头在近。近，才走得长。丑时前，他盘坐床沿，席严，坛不露。掌贴沿，沿温一线，远灯不熄。不熄，像四层在等——等不要紧，先走路。气转一周，二层稳，稳里暗袋那十包渣像十粒种。

种轻，记重；记重了，丹路才开。坛底轻颤，臾墟残念弱一息：「……渣里活，路在渣。」

韩泥不答，答在心里——心里记一笔：废渣路，路在渣，不在舌。邻床问：「你床角袋越来越沉？」

「渣沉。」韩泥说，「沉不要紧，沉才像路。」

少年不懂「路沉」，闭嘴睡了。秦霜木鱼远，一声：「路在心，心别邪。」

韩泥不答，答在心里——心里恩辱分列。路是活，活通向温听雨洒扫，通向嗅药小成。他独眼平，手不抖：「废渣路走了。窑试温，袋十包。温听雨在远廊。」

韩泥只答：「不恋。试温，记着。」

坛腹应温，像废窑门槛那一点炉回。回轻，记重；记重了，才接得住丹路开。他低声对坛：「废渣路走了。坛在，袋在。袋在，就不滚。」

韩泥只答：「试温。管事记了。」

在，就不滚。不滚，路才走得长——走得长，才开得了丹路，才还得了叶汤。柱上条角白一线，三年约在怀。怀凉，袋沉，掌里远灯不熄——灯在，路就不绝。他低声道：「废渣路在渣里。舌不留，手要稳。稳了，丹路自来开。」

丹路自来开，在远，在89章；远不要紧，路头在十包渣里。渣在，就不滚。废渣路第二日，巷尽炉基前，韩泥驻足。驻足，不为贪砖，为记焦回——回在鼻，不在舌。斗第三趟满，过磅，记「废渣路净」。记净，是小胜。铁无言验斗沿，沿无活草，无整药：「路尽了，别回头贪。」

韩泥应声：「不回头。界在。」

温听雨风口立半息，裙素，袖香温，她没近，只远看一眼斗沿。看一眼，也像验净。赖福讽远：「走路也走，末等也末等。」

韩泥不接茬。舌要留来鼻，留来守坛。刘婆端粥，仍多半勺：「路尽了腰还直？」

「直。」韩泥说，「腰直，槽才稳。」

午后角柜样增至十包，包实，锁柜。锁丑，锁像规。邻床问：「炉基为啥不扒？」

「扒了，记名。」韩泥说，「记名，恩断。」

丑时前，十包并息，并出一线清。清里，掌纹影颤，颤像四层在等三层破。他低声对坛：「路尽了。焦回记着。温氏明日识，不叫。温，像废渣路尽那线白。更鼓沉，独眼平：「废渣路尽了。洒扫初识在后，嗅药小成在后。」更鼓沉，沉里，韩泥按木牌，牌凉，掌温。废窑试温后第三日，药山侧径霜白。

他绕径上山，不为占，为记路——路从坡下起，经暗袋，指废窑，指95匿丹。径角有杂役弃牌，牌裂，字浅：「占窑除名」。韩泥瞥一眼，不近——近，像贪。贪，路断。路从药山起，经暗袋，指废窑，指110首丹——远不要紧，路头在十包渣里。管事验足迹，足迹不过窑深，深内一步无印——无印，是小胜。」

编筐钱买米，米在，恩才还得上。

「」廊下风硬，他把木牌系紧一线，系得像把仍末摁进掌心。同批少年远避，避像避晦气，晦气遮瓮，遮住了才稳。编筐角藤刺仍在，他偏手，刺浅些，沿才净，坛才藏得住。管事夜巡验棚，灯照席角，沿净，像腌菜坛，不像邪。老耿挑水过，声哑，境别拜邪，拜了路断，他只记着。

秦霜木鱼远，一声，心别邪，邪了鼻下香飘，香飘手就滚。

柱上条角白一线，三年约在怀，愧字在怀，手仍不抖。

白织月远廊过，袖风冷，没停，胜在鼻，不在嘴。

铁无言脚步远，停半息，声低，别漏香，别叫境，只记。

刘婆粥稀，勺沉，饭稳恩不断，他喝尽，不谢，只记。

废渣路第三日，巷口风硬，硬里焦回。韩泥斗车稳，稳不要紧，界碑影仍在雾里，影斜，像在提醒：再一步，除名。

他收最后一斗，斗满，过磅，记「路尽净」。记净，是小胜。温听雨远廊一眼，一眼也像验斗沿净。

午后角柜十包，包实，锁柜。锁丑，锁像规。规在，坛在；坛在，根在。根在，才还得了叶汤。

刘婆端粥，仍多半勺：「路尽了别贪砖。」韩泥点头，谢轻，恩重。恩在粥里，也在「别贪」里。

废窑试温半刻，温稳一线，稳像嗅诀「沉」字落地。落地不要紧，记火路，记投序，记83洒扫初识。初识，各做各的；各做各的，是界。界在，缘苗才不邪。

更鼓沉，沉里，韩泥按木牌，牌凉贴胸。82章手在，名在册，坛沿净。舌不留，鼻下静。恩辱分列，不混。活长了，才还得了叶汤，才接得住门缝灯。

更鼓沉，他摸暗袋，袋凉，坛温一线。温应路，不应慌。

「渣沉。」

「「温听雨在远廊。」温听雨在远廊。」
更鼓沉，他摸暗袋，袋凉，坛温一线。嗅渣场风硬，风不要紧；要紧是鼻清，手稳——稳了，才藏得住再试炉温。
