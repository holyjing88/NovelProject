# 第十五章 定审

村议辱叶后两日，里正敲锣。「叶家事，不能糊。开大会，辨诬，还清白。」

村人聚祠堂，赵绣春党不散，散的是胆——胆怕里正报官，怕宗门查连坐。马春花嗓还尖：「辨什么？寡妇送汤，事实！」

里正板脸：「事实，当众说。说不服，再罚。」

韩泥坐末席，臂布未拆，独眼低。低不是怂，是等——等叶家开口，等公审落锤。日子定了，锤有期。议到午后，争出三条：

一，叶家关铺谢罪，七日；  
二，韩泥搬离村西，换破庙；  
三，开公审大会，村人作证，辨「送汤是恩是邪」。赵绣春要一、二要，不要三：「三有什么好辨？辨了给晦气翁脸？」

叶青禾在帘后，声出，稳：「要三。不要一、二。」

「不要关铺？」周嫂叫，「铺子晦气，谁抓药？」

「铺子开。」叶青禾道，「抓药的，是病。不是嘴。」

里正拍板：「三留。一、二缓。公审大会，要开。」

「何日？」有人问。里正望天，天阴，像旱前兆：「霜降后第三日，卯时，祠堂公审。帖今日贴。」

里正扬声补一句：「日子定了。谁再辱叶、砸秤，先记名，再报县。」

定了日，定的是锤——锤有期，赵绣春退半步，退得不甘，像牙咬空。周嫂还想嚷，被里正一眼压住：「嚷到公审上嚷。今日散。」

散时，村人三三两两，嘴仍动，动的是「辨诬」二字——二字新，新得像门缝光，光小，却照见：叶家未必输，谣未必赢。祠堂阶下，有人扯里正袖，低声：「公审，赖管事来不来？」

里正沉声：「来不来，村都要辨。辨不清，连坐先落在嘴上。」

扯袖者咽，咽里退——退，像怕，也像盼：盼有一锤，锤住谣。赵绣春阶上不肯下，嗓还尖：「辨诬？辨了，寡妇脸面往哪搁？」

叶青禾帘后声稳：「脸面在汤里。汤热，脸就不寒。」

赵绣春一窒，想再尖，里正铜盆一敲：「再闹，记名！」

尖收半寸，后力没了——没了，公审的缝又开一线。韩泥最后一个出祠堂，独眼扫过空阶，阶上还留着叶青禾站过的印，浅，真。韩泥出祠堂，叶青禾在侧巷等，没近，只隔三步。「韩翁。」

「嗯。」

「公审……」她停，「会连你。」

「该连。」韩泥独眼平，「汤是你送的。恩是我记的。连，正好辨。」

叶青禾唇动，想再说，他抬手止：「别谢。谢字轻。等辨完，我还。」

叶青禾低声：「赵绣春说，辨前还要闹。闹到关铺。」

「闹。」韩泥道，「铺开着，闹空。空的是嘴，不是秤。」

叶青禾「嗯」一声，布鞋踩泥，轻，稳。韩泥摸心口，叶汤那笔恩深，深得像公审的钉——钉未落，洞先凿好。傍晚，赖扒皮短签又贴告示桩：「公审前，沈家罚银再延五日。再牵扯，连坐全名。」

韩泥路过，不近桩。近，成靶。他只记：仇苗在，公审在，还字也在。路人窃语：「延五日，芽儿娘还能喘。」

「喘着，公审就来了。来了，晦气翁逃不逃？」

韩泥担筐不停，只道：「不逃。站。站给锤看。」

窃语者静，静里让开半步——让，像钩住的心，也松半分。回屋，坛角微温，坛腹微温。温里他低声道：「大会要开。开，叶丫头的汤，得立住。」

坛壁蒙尘里，极淡一道影游过，像半个符，像半个字，一闪，灭。灭得他以为眼花——眼花也罢，坛应了「辨诬」，应得他心定半分。村狗叫一声，又哑。哑，像全村等日——日定了，钩更紧。里正锣余音还在巷里荡，荡得人心发紧：紧不是怕，是知道有一锤要来。韩泥编筐，藤条绕膝，绕得像把「辨诬」编进去——编进去，等那日，叶汤能立，芽儿能活，叶家能开门。赵绣春过巷，瞥他一眼，想尖，尖到一半，见里正锣还在，咽回去。咽回去，也是钩：日子定了，赵党也要借锤，人心已定一半。韩泥不望她，只望筐：筐圆不圆不重要，结实才重要。结实，才能撑到辨诬那日。叶青禾药铺灯，夜里仍亮。亮弱，像公审前的烛——烛不救全村嘴，救抓药的人。子夜，药铺外「哐」一声。秤杆被人砸中，杆弯，未断。叶青禾扶杆，指白，声仍稳：「砸断了，全村没秤。你们砸谁？」

巷口人影退，退得快，像怕里正记名。韩泥隔巷听见，没近。近，成牵扯。他只低声道：「秤在，铺就在。砸秤的，公审上见。」

叶青禾把秤杆扶正，秤砣再落一声——落给全村听：铺还在，脊还直。里正闻声赶来，举锣未敲，先问：「谁砸的？」

巷静。静里，赵绣春门内啐一声，啐碎，像心虚。里正板脸：「再砸秤，先记名。公审日子定了，砸秤也是辱。」

叶青禾扶杆，指节白，声仍平：「秤杆弯，弯不断。断的是谣，不是铺。」

韩泥隔巷，没近，只记：记砸秤这一辱，候帖上见。夜，里正叩门，三下，沉。韩泥开门一尺：「里正。」

「公审要开。」里正嗓低，「你少往前冲。冲，害叶家。」

「我不冲。」韩泥道，「我站。站末席，等锤。」

里正顿一息：「锤落谁身上，未定。定的是，谣不能糊。」

「糊不了。」韩泥独眼平，「汤烫着，糊不了。」

里正「嗯」一声，步远，远里锣未再敲，心已钉——钉名「辨诬」，日后再钉。韩泥低声道：「日定了，理也定了。定的是辨诬，不是关铺。」

里正再敲一下锣，锣声短，短而清，像给全村钉一颗钉：钉名「辨诬」，钉日「霜降后第三日卯时」。韩泥收筐，藤条在手，手稳。稳，像等锤——锤有期，翁先立。坛角微温不退，沿口水汽一线，像瓮在睡，却听见了「辨诬」二字。沈家豁口咳一声，咳轻，像证：日子定了，芽儿娘还能喘，公审就能辨。公审日子定了，村里嘴却先分流。一派跟赵绣春，嚷「关铺谢罪」；一派跟里正，嚷「等粮归仓」；还有一派不说话，只夜里绕叶铺走，走慢，像怕铺真关。韩泥编筐，听得见脚步，不开门。开门，成「牵扯」；不开，只把藤条编实——实，才能撑到辨诬那日。叶青禾隔巷低声问：「韩翁，公审上……我怎么说？」

「说汤。」韩泥道，「热，送，该送。别辩奸，奸在谣嘴里。」

「你呢？」

「我认恩。」韩泥独眼平，「恩我担。担了，正好连。」

叶青禾「嗯」一声，步远，远里药铺灯仍亮——亮弱，像公审前的烛，烛不救嘴，救抓药的人。赖扒皮短签在桩，风撕角，撕不撕名。名在仇苗里，苗在，公审前还要攒十文、还要立汤、还要让芽儿活。里正锣余音又荡一回，荡得短，短而清：「辨诬」二字，像钉，钉进全村心——心紧，不是怕，是知道锤要来。韩泥摸坛，符影不再现，腹温却不退。不退，像瓮睡了，理醒了——醒半分，够他等日。公审日子定了，霜降后第三日卯时；韩泥擦坛沿三遍：秤杆被砸，杆弯未断，铺还在，汤就不凉。**状态**：大境·凡人 · 资质·丑骨末席（伪灵根） · 鸿蒙九劫瓮·眠 · 洞府·漏舍凡舍
