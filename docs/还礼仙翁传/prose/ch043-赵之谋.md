# 第四十三章 赵之谋

赵玄机妒火至峰，峰不在莫长春，在林挽月看他的眼神——那眼神里多了犹豫，少了慕，像一根线松了。

外门林苑，夜，竹影碎，月白得像一张纸，铺在赵玄机脚边，纸上该写局，不该写恩。

「挽月，」赵玄机声低，「莫长老浴房外徘徊，你瞧见了吧？」

林挽月一颤：「赵师兄……那是谣言……」

「谣言要有人传。」赵玄机截话，「长老辱你、辱宗门，你忍得，执法堂忍不得。」

「可……」

「可什么？」他目光冷，「你慕我，还是慕他装的神仙样？」

林挽月脸白，唇抖，终未答。竹影晃，像有人也在听，听在廊角，听在远处，听在将来要翻的堂上。

赵玄机递她一枚传讯符：「明日戌时，浴房外径。你只需『偶然』路过，看见『偶然』徘徊的长老。余下，执法堂会办。」

林挽月捏符，像捏一块烫手铁：「若错了呢？」

「错了，」赵玄机淡道，「也是长老九月将死，急出来的丑。与你何干？」

他走了，袖风冷。林挽月立月下，立成一根将断的竹。

远处，莫长春院中，簿无字，却沉，像知奸谋将起，不急，只念：「年轻人，急什么。」

顾小满在杂役堂浆洗，抬头望林苑方向，无端心口一紧，像半块灵石在掌里跳，跳成警。

她不知局将起，只知赵玄机白日路过执法堂时，眼神太稳，稳得像已写好明日嚼舌的稿。

孙福旧党在角隅嘀咕：「莫长老老不正经，终于有人敢办。」

顾小满冷眼：「办错了，堂上见。」

嘀咕者噤声，像谣言碰上半石，又磕碎牙。

---

夜更深，林挽月捏符欲碎，又止，止得像怕碎的不是符，是自己最后一点慕。

她想起赵师兄还库，想起莫长老止步不入的传闻尚未起，忽然觉得两条路都在脚下，一条慕，一条证，证字更重，重得像恩。

赵玄机在执法堂偏厅，指节叩铁牌，一声比一声轻，轻得像妒火终于找到出口，出口却是脏的。

他低声，像对自己说：

「明日，全宗都要看。」

魏无涯在檐角，扇开一线，观气运，低笑：「局将脏，脏里也有人要证。」

林挽月欲碎符又止，止得像慕将断。风过竹，沙沙，像答，又像不答。

她低声，像钩，说给月：

「若真是局……明日戌时，该有人止步。」

执法堂弟子问：「赵执事，明日若长老不入浴房……」赵玄机淡道：「不入，也是徘徊。徘徊，够嚼。」弟子笑，笑里脏，像已看见明日浪。

苏念慈在丹堂值夜，炉火旺，旺得像替林苑也担。她不知局将起，只知浴房二字若成浪，丹堂也要被溅脏。

柳青鸢在宗主殿批文书，笔尖一顿，像觉出竹影里的局，却不点破，只落笔：「戌时，执法堂巡。」——巡是真的，局也是真的，真假要等莫长春止步才算。

月白，符烫，像两条路都在脚下——一条脏，一条证。证字还无人说，脏字已在赵玄机袖里写好。

魏无涯在檐角，扇开一线，观气运，低笑：「局将脏，脏里也有人要证。」风过竹，沙沙，像答，又像不答——答在明日戌时，答在止步三息，答在证字还无人说、脏字已在袖里写好那一瞬。

夜更深，林挽月捏符欲碎，又止，止得像怕碎的不是符，是自己最后一点慕。她想起赵师兄还库，想起两条路都在脚下，一条慕，一条证，证字更重，重得像恩。赵玄机在执法堂偏厅，指节叩铁牌，一声比一声轻，轻得像妒火终于找到出口，出口却是脏的。

远处，莫长春院中，簿无字，却沉，像知奸谋将起，不急，只念：「年轻人，急什么。」她低声，像钩，说给月：「若真是局……明日戌时，该有人止步。」

他低声，像对自己说：「明日，全宗都要看。」魏无涯在檐角，扇开一线，观气运，低笑：「局将脏，脏里也有人要证。」

林挽月脸白，唇抖，终未答。竹影晃，像有人也在听，听在廊角，听在远处，听在将来要翻的堂上。月白，符烫，像两条路都在脚下——一条脏，一条证。

嘀咕者噤声，像谣言碰上半石，又磕碎牙。林挽月欲碎符又止，止得像慕将断。风过竹，沙沙，像答，又像不答。

魏无涯檐角观，扇合，低笑，像看戏将脏，脏里也有人要证。远处莫长春院中，簿无字，却沉，像知奸谋将起，不急，只念：「年轻人，急什么。」

赵玄机偏厅叩铁牌，一声比一声轻，轻得像妒火找脏出口。明日，全宗都要看——看止步，看嚼舌，看证字何时落。
