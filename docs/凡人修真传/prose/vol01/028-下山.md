# 第 28 章 下山

> 纲：青牛入宗衔接专纲 · 011～035 块 B

---

陈寻背老耿下山，夜路滑，一步一喘。

血渗他颈，温，腥，像又一笔账贴在皮肤上。林风在身后追，追不及，像狼未散的魂。魂不散，村就不宁；宁不了，守夜还得续——续之前，先把这条命背到灯下去。

老耿半昏，嘟：「瓮……修好了……别淹……」

「不淹。」陈寻说，「你活。」

村口有人举灯，喊声杂，像潮。潮里，有人躲，有人迎，迎的是热闹，躲的是血。陈寻不歇，直往沈家药铺——药铺灯常亮，亮得像一条能接命的路。路窄，窄在肩，肩肿，肿也得走；走不动，就停一息，一息，再一步。

---

沈振东出，不接话，先药，先缝，先止血。

针过，线拉，老耿臂肉翻，他眉不皱，手稳，像缝一件寻常的衣，也像缝一条命。命细，细在针下；针一歪，臂就废；臂废了，瓮还在，人却少半力——少半力，北岭下一夜谁守？

沈清弦在帘后递纱，手白，不发一语，只把药递得准。准，像沈家规矩：先活，后账；先缝，后问。

萧惊鸿赶到，要骂猎户逃，陈寻拦：「骂无用。命先要回来。」

沈振东缝罢，低声：「臂废半力，命在。养月，勿动刀。」

老耿睁眼，独眼看陈寻，喉动，想说什么，陈寻摇头：「不用说。瓮我守着。」

老耿闭眼，像睡，也像把命暂时交出去。

陈寻坐阶，满手血，取册，蘸血：**老耿，命。**

字浮，沉，阳温顶格，像瓮，像臂，像一条命的分量——比瓮沉，比名烫，烫里稳，稳里沉。沉得他指腹发麻，麻像提醒：命记了，不是恩结，是债起；债起，就得还，还长，还重，还到沈铺门前这一步，只是开头。

册子微颤，像认得这一笔的分量。陈寻指腹按住，不让颤——颤是情，情乱账。

里老赶来，问狼、问羊、问猎户。陈寻一句句答，答得像报账：狼逃，羊损二，猎户散，守夜续。里老叹：「北岭不平，村无宁日。」

陈寻不辩，只指阶上老耿：「他平了一夜，够。」

张麻婆提热水来，骂：「赵家看热闹，沈家救命。这村，还得靠手，不靠嘴。」

嘴能赢一时，手才能赢一夜。一夜赢下来，老耿的臂还在，瓮还在，村的胆还在——胆小，也得长；长，靠一针一线，靠一桶热水，靠一个少年满手血仍坐阶上记账。

陈寻接水，洗手，血淡，淡不尽，像阳面又添一笔，笔重，不是恩，是命，命也记。

程不二在巷角，无人见，只留一枚石，石上刻「三」。

陈寻拾石，袖藏。石凉，字浅，像一句未说全的话——三，是三月，是第三夜，还是第三条路，他不知，先藏。藏了，心不乱；乱了，就看不见沈振东递来的净布，看不见里老眼里那一瞬复杂——复杂里，像终于承认：这个被泼水的少年，背的不只是耿，是北岭那一夜的账。

沈振东递净布，陈寻擦手，不谢口，只谢工：「药钱，我记。」

沈振东摇头：「先活。账，后算。」

萧惊鸿蹲阶下，低声：「你背他下来，肩都肿了。」

陈寻说：「肿，消。命，不能消。」

天将明，鸡鸣，村议暂平，北岭未平。陈寻担水，换布，像还一笔放长线还的工——命记了，工才始。

他看怀册，看袖石，看瓮由萧惊鸿搁在廊角，泥缝干，像碑。心里落数：八笔恩，瓮已还，命刚记；测灵，两月余；路，还长。石「三」，像程不二在数他还能活几个夜——数得准不准，不由石，由步。

沈清弦帘后收纱，指节白，像也沾了血，却未近血。陈寻不看帘，只谢工：「今夜，算我欠。」

沈振东摇头：「欠不欠，不是你说了算。先把他养住。」

陈寻起身，肩肿，步稳，往家去。巷空，灯稀，像村议暂平后的另一种静——静里，有药香，有血腥，有未还的工。

到家，他洗手，洗不尽腥。册在怀，石在袖，命字阳温，像一团不肯凉的火。他对着灯看「老耿，命」二字，不看泪，只看分量——分量够了，才配谈还。

鸡鸣又起，北岭告示又补：守夜续，猎户缺二。陈寻把柴刀别好，像把昨夜背下来的人，换成今夜要守的村。

他回院，先睡半个时辰，肩肿醒，肿里热，热像血未散。萧惊鸿在墙外守，声低：「沈家那边，老耿醒了，骂你呢。」

陈寻说：「骂好。骂，是活。」

他摸册，「老耿，命」仍在，阳温未退。退不了，也不想退——这一笔，比瓮重，比名沉，沉到他把袖中石「三」都压得更凉。

天光满院，他提水欲往沈铺，路过告示，见「守夜续」三字，心不动——动也没用，动，得先活。活，才配再上山。

告示纸边卷，卷得像北岭的风；风在，狼就在；狼在，守夜就在。守夜在，他就还得去——去之前，先把沈铺的工还了，把老耿的命养住，把册上那一笔「命」按稳。

他担水去，步稳，肩仍肿，肿是昨夜，稳是今日——今日，要还养伤的工。

路过巷口，童谣丫在，不唱，只冷眼看他满袖血痕。陈寻不停，丫也不问。问，是孩；不问，像也懂：有些夜，背的是命，不是话。

沈铺门内，针声将起。陈寻提水，心里已落一字：命。

他未进门，先听针声——针声稳，像沈振东的手，也像册上刚落的那一笔，笔在，人就得活。活，才算下山。下山，才算还瓮之后的第一笔工。工，才始。始，在针下。

针下若稳，肩上肿就会消；消了，守夜还得去。去，不是逞，是村还在，牙还在，账还在。

沈铺门内，药香混血腥，混成一种洗不净的味。味在鼻，命在背，背空了，才松。松了，还得提水。
