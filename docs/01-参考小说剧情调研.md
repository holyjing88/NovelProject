> 本文档仅作创作参考与题材分析，不直接引用原文内容。最终游戏使用原创剧情《万古守灯人》。

## 一、《我得长生后》（驿城长豇豆 · 起点中文网）

### 核心设定

| 维度 | 内容 |
|------|------|
| 主角 | 陈可观，十八岁贫家子，为活命净身入宫 |
| 金手指 | 《造化长春功》——补缺长生，寿元无限（仍可被杀） |
| 力量体系 | 精神力九品至神品；兼修武道、儒道、佛门、剑修 |
| 世界观 | 架空大乾王朝，东莱-大乾-北魏-南疆四极并立 |
| 主题 | 人间烟火气，最抚凡人心；长生不是逃离人间 |

### 剧情脉络（两卷结构）

**第一卷：皇宫里的老太监（1-105章）**
- 开元元年入宫，净身濒死，旧书楼获功法
- 旧书楼擦灰读书，跟李公公学生存之道
- 结识小太监高欢，为其"撑伞"成为初心
- 鬼市搏命，初窥九品精神力感知
- 与太平公主市井同游，卷入宫廷权谋
- 栖凤殿前截停公主出嫁，情感高潮

**第二卷：军队里的定心丸（106-211章）**
- 出宫从军，成为军中精神支柱
- 赵凌月女帝改革：减田赋、开商路、设监察使
- 北伐布局十三年，对抗定国郡王霍无敌
- 南疆煞气洞苦修（主观三千万年，客观三年）
- 擎天峰突破超品造物主境
- 结局：携妻归故里，敲响童年家门

### 可借鉴的叙事元素（非照搬）

1. **时间变量**：长生者用三十年练一招，百姓一日三餐不可断
2. **底层视角**：太监、青楼清倌、伙夫、酒楼打工剑仙
3. **力量代价**：精神力反噬需付出代价（如"用吾须发，换尔性命"）
4. **群像塑造**：高欢、齐浩然、霍千机各有独立弧光
5. **喜剧外壳+严肃内核**：荒诞对话解构权谋，笑着流泪

### 重要配角原型

- 李公公：旧书楼管事，以懒散为铠甲的清醒守护者
- 高欢：底层小太监，务实热望，主角人性温度参照
- 赵凌月：女帝，制度革新者，与主角理念共鸣
- 霍无敌：功高震主的悲剧武将，非脸谱恶人
- 兰雅/景梦：青楼女子自我成长，拒绝猎奇叙事

## 二、《老人修仙记》（李大开版 · 凡人流）

### 核心设定

| 维度 | 内容 |
|------|------|
| 主角 | 李大开，五十岁开始修仙 |
| 风格 | 凡人流、无系统、杀伐果断、慢热、搞笑 |
| 特点 | 体内阳气澎湃，需提防仙子觊觎 |
| 力量体系 | 传统修仙：炼气→筑基→金丹→元婴→化神 |

### 剧情脉络（卷结构）

**第一卷：杂役弟子**
- 开局成为"老爷爷"（被误认为老者）
- 青楼舌战、讹银子进城
- 七星宗选拔，成为杂役弟子
- 黑风谷采药，获修士尸体与储物袋
- 坊市倒卖丹药，宗门内斗

**中期：宗门风云**
- 七香院修行，师姐秦茹、师兄乾青柳
- 天尸门威胁，五行炼尸、七星宗大战
- 冲击筑基，千机塔探险
- 上古炼气士遗迹，获千机塔和炼火诀

**后期：跨界冒险**
- 化神中期，穿越法则封锁
- 千机塔九九八十一难
- 界面漩涡，多界穿梭
- 金色飞剑、仙元石、芥子空间

### 可借鉴的叙事元素（非照搬）

1. **高龄起步**：打破"少年天才"惯例，老年主角的逆袭
2. **资源博弈**：坊市交易、丹药倒卖、储物袋争夺
3. **宗门生态**：杂役弟子、内门斗争、派系倾轧
4. **杀伐果断**：吃里扒外者杀、借刀杀人、金蝉脱壳
5. **幽默调剂**："老李你要老婆不要"、左眼皮跳财

## 三、其他同名作品辨析

| 作品 | 作者/平台 | 核心差异 |
|------|-----------|----------|
| 《老头要修仙》 | 起点中文网 | 六十八岁采药人张进，万灵图录，多界穿越 |
| 《老头修仙记》 | 番茄小说 | 徐大爷渡劫失败，万倍返还系统 |
| 《老头修仙记》 | 江怀瑾毛利兰 | 名柯同人，二次穿越 |

**结论**：用户所指的"老头修仙记"最可能为李大开版《老人修仙记》，因其在凡人流老年修仙题材中影响力最大、章节最完整（1200+章）。

## 四、两部作品的共性主题

1. **逆龄/长生起点**：一个从残缺中获得长生，一个从老年开始修仙
2. **凡俗扎根**：力量体系与社会肌理深度绑定，修仙要银子
3. **时间哲学**：漫长生命中的选择、孤独与坚守
4. **底层逆袭**：从社会最边缘处向上攀爬
5. **人间温情**：在杀伐与权谋中保留对普通人的关怀

## 五、原创化方向（版权安全）

### 必须替换的元素

- 所有角色姓名、地名、功法名称
- 具体情节桥段（如净身入宫、栖凤殿截婚）
- 力量体系数值与命名
- 经典台词与梗

### 可保留的题材方向

- 老年/长生主角视角
- 凡人流慢热成长
- 人间烟火主题
- 力量有代价的修行观
- 市井+宗门+禁地三段式地图
- 选择驱动的叙事结构

### 原创作品命名

## 六、《斗破苍穹》（天蚕土豆 · 玄幻经典）

### 核心设定

| 维度 | 内容 |
|------|------|
| 主角 | 少年天才跌落，隐忍三年后逆袭 |
| 金手指 | 戒指中的远古灵魂导师 + 异火 |
| 力量体系 | 斗之气 → 斗者 → 斗师 → … → 斗帝 |
| 世界观 | 加玛帝国 · 云岚宗 · 塔戈尔沙漠 · 中州 |
| 主题 | 莫欺少年穷；尊严与成长 |

### 剧情脉络（结构特征）

- **开篇跌落**：天才变废柴，当众受辱，立下「三年之约」
- **导师引路**：隐藏强者传授技艺，主角暗中苦修
- **炼药立身**：以丹道积累资源，坊市与宗门双线博弈
- **宗门大比**：公开战证明实力，打脸与正名
- **异火寻宝**：闯塔、夺火、秘境升级
- **势力升级**：从地方宗门 → 帝国 → 大陆

### 可借鉴的叙事元素（非照搬）

1. **公开受辱 + 延时复仇**：用「约定」制造长线悬念
2. **导师在暗处**：强者残魂/封印者引导，但主角仍靠自己
3. **技艺立身**：炼药/炼器作为弱者的突破口
4. **宗门大比**：阶段性实力验收与打脸高潮
5. **秘境夺宝**：塔/火/遗迹作为突破契机

## 七、《凡人修仙传》（忘语 · 凡人流鼻祖）

### 核心设定

| 维度 | 内容 |
|------|------|
| 主角 | 韩立，资质平庸，谨慎到近乎冷酷 |
| 金手指 | 小绿瓶（催熟灵药）—— 但前期几乎不用 |
| 力量体系 | 炼气 → 筑基 → 金丹 → 元婴 → 化神 |
| 世界观 | 越国七派、乱星海、天南、灵界、仙界 |
| 主题 | 修仙界无人情，唯有谨慎与资源 |

### 剧情脉络（结构特征）

- **入门艰难**：杂役、测试、最低等身份起步
- **资源至上**：捡漏、坊市、杀人夺宝（有底线地变通）
- **丹道立身**：炼丹换灵石，以技补根骨
- **秘境求生**：古修士洞府、禁地，步步为营
- **宗门政治**：派系、联姻、利益交换，不站队则死
- **跨界飞升**：每个地图都是更大棋盘

### 可借鉴的叙事元素（非照搬）

1. **极致谨慎**：表面温吞，实则留后手、藏底牌
2. **资源博弈**：每一块灵石、每一颗丹都有来路
3. **捡漏与洞府**：死人身上找机缘，但要有代价
4. **宗门生态**：杂役—外门—内门—真传的残酷阶梯
5. **慢热升级**：一次大突破背后往往是十年积累

## 八、《斗罗大陆》（唐家三少 · 武魂体系）

### 核心设定

| 维度 | 内容 |
|------|------|
| 主角 | 唐三，双武魂，穿越者 |
| 金手指 | 玄天功 + 唐门暗器 + 双武魂 |
| 力量体系 | 魂师等级 + 魂环（颜色/年限）+ 魂技 |
| 世界观 | 斗罗大陆 · 武魂殿 · 史莱克学院 |
| 主题 | 团队、传承、师徒、大赛正名 |

### 剧情脉络（结构特征）

- **觉醒武魂**：六岁时测定天赋，决定命运
- **拜师传承**：隐藏强者收徒，传授独门心法
- **学院修行**：团队组建，各有专精（控制/强攻/辅助）
- **魂环猎取**：按等级猎兽，获取技能与突破
- **全大陆大赛**：学院联赛作为中期高潮
- **身世揭秘**：父辈恩怨与终极反派挂钩

### 可借鉴的叙事元素（非照搬）

1. **觉醒仪式**：公开测试天赋，制造起点落差
2. **师徒传承**：临终传功、遗志托付
3. **固定队**：三到五人各有分工，互补作战
4. **等级猎取**：每次突破需「外物仪式」（本作可改为灯油/记忆）
5. **大赛正名**：团队赛证明「不被看好的组合」

## 十、2025–2026 流行网文趋势（行业调研）

> 数据来源：2025 中国网络文学发展研究报告、起点/番茄 2025–2026 榜单、短剧改编案例。  
> 仅提炼**公共趋势与结构**，不引用任何作品的专有设定与情节。

### 十大流行特征

| 趋势 | 代表方向（题材标签） | 读者为什么买单 |
|------|----------------------|----------------|
| **类型复合** | 仙侠+探案、民俗+悬疑、都市+规则 | 打破「纯升级打怪」疲劳 |
| **反套路修仙** | 修仙要资源、苟道流、无系统凡人流 | 真实感与代入感 |
| **规则铁律** | 序列/职阶、规则怪谈、禁忌行规 | 智斗 suspense，可二创 |
| **单元案件** | 3–5 章一案，伏笔汇主线 | 碎片化阅读、短剧切片 |
| **精神内核** | 不只爽，要有「守什么」 | 精品化、IP 长尾 |
| **慢哲思长生** | 长生之后、时间重量、孤独 | 差异化情感 |
| **人间烟火锚点** | 职业属性（捞尸、打更、书吏） | 文化亲近、半真实沉浸 |
| **群像小队** | 值夜者式固定队、各有牺牲 | 情感共鸣破圈 |
| **IP 系列化** | 单元延伸、多季短剧、一鱼多吃 | 长线运营 |
| **读者参与** | 分支投票、名场面共创 | 社区粘性 |

### 2025–2026 平台爆款类型对照（标签级）

| 平台倾向 | 热门标签组合 | 结构共性 |
|----------|--------------|----------|
| 起点精品 | 东方神话、民俗悬疑、凡人流、权谋 | 世界观厚、伏笔长、群像牺牲 |
| 番茄流量 | 规则怪谈、脑洞仙侠、老年反差、短剧向 | 强钩子、单元快、名场面密 |
| 短剧改编 | 凡人流、单元修仙、慢孤独、土甜/权谋 | 每集一个爽点/泪点，可续季 |

### 与《万古守灯人》的天然契合

本作已有 **老年主角、人间烟火、记忆代价、群像守灯**，与 2025–2026「反套路 + 慢热 + 内核」方向高度吻合。  
待强化方向：**规则铁律可视化、单元探案、民俗灯祭、短剧分季、名场面清单**。

## 十一、2025 趋势 × 《万古守灯人》融合方向

| 流行趋势 | 本作原创映射 |
|----------|--------------|
| 大奉式·探案+修仙 | → **「一盏灯一案」**：灯影照真相，书吏断案 |
| 捞尸人·民俗悬疑 | → **走灯节、灯禁十则、水灯案**（江南水乡民俗） |
| 诡秘/十日·规则铁律 | → **守灯十诫**（触则反噬，可设计副本） |
| 没钱修什么仙·反套路 | → **灯油=记忆=寿元**，修仙先攒人间烟火 |
| 不死帝师·慢孤独 | → **越亮越不能飞升**，长生是「为他人续灯」 |
| 苟道流 | → **留灯三策**升级：不冒头、不亮底、不空油 |
| 值夜者群像 | → **青萝五灯队** + 每季一人牺牲/失忆 |
| 短剧单元延伸 | → 五卷可拆 **20 单元案** + 每卷独立季 |
| 读者参与 | → 游戏/互动版「万灯抉择」影响支线 |
