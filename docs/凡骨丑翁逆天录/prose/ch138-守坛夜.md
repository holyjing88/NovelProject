# 第一百三十八章 守坛夜

守坛夜，坛在角，角在席下。

席严，坛不露——不露，才像丙九末席该藏的。

韩泥按坛沿，沿净，净像62席盖。盖严，赖福眼进不来；进不来，嗅诀才练得稳。

鼻下静半刻，沉腐浮三字在息里——浮不要紧，沉才准；准了，142才谢得老耿。

邻床翻身，眯眼：「你丑时还不睡？」

「守坛。」韩泥声平，「坛在，人在。」

人问半句，没再问。问，像沾晦气。

瓮腹应温，温一线，像130还刘婆那夜——那夜还恩，这夜守路。

路在，就不滚；不滚，大比帖才压得住。

铁无言窗外过，脚步停半息，声低：「守夜，别出门。」

「不出。」韩泥说，「出门，手飘。」

脚步远，远像134修栏那日——日续，夜守，兄弟线在，不在嘴。

刘婆柴门外咳：「夜里冷，别冻手。」

「不冻。」韩泥说，「冻，手飘；飘了，渣不正。」

更鼓沉，沉里他再验嗅诀——诀在鼻，不在舌；舌要留来数息，留来守坛。

秦霜木鱼轻敲：「守夜，心别邪。」

韩泥不答，答在心里。心里只静，静到只闻辛淡。

丑时末，他望坛，低声道：「守过了。记着。」

坛不应字，只温。温像应「夜」——夜守，日活，就不绝。

天边白一线，白里药峰派活影将至。将至不要紧，手在，路在。

席下坛腹应温，温一线，像95匿丹那夜——夜不同，温同。

管事夜巡，灯照席角，照见厚席不见坛。坛不露不要紧，沿要净。

坡下石阶，他数息，数粒，数债序。债序在，步不飘。

赖福火折照栏，照见韩泥立像桩——桩稳，栏稳，就不滚。

刘婆端姜汤，汤烫，仍多半勺：「守夜别空肚。」

「不空。」韩泥说，「空肚，手飘。」

更鼓沉，沉里他再验嗅诀——诀在鼻，不在舌。

他望坛，独眼平：「守坛夜过了。记着。温像应「守」——守完，才有142。守夜至丑时末，粟壳香淡，香一字不落。瓮腹应温，温不应喜，应「稳」——稳，是守坛回响。」

铁无言窗外脚步远，远像134那日——日修栏，夜守坛。

韩泥独眼半阖，内视里阶影稳在十五。根在，守坛就不白守。

他低声道：「守过了。记着。」

天边白一线，白里，142影将至——将至不要紧，手在，路在。

守坛夜，坛在角，角在席下——席严，坛不露；不露，才像丙九末席该藏的。藏，不是怯，是活；活了，139钱戾衡影才接得住；接在台，不在坊尾骂。骂不要紧，守夜静，静到只闻辛淡。辛淡在鼻，不在舌；舌要留来数息，留来守坛。坛腹应温，温一线，像130还刘婆那夜——那夜还恩，这夜守路。

路在，就不滚；不滚，140积势才收束。收束在远，138只守。守夜，铁无言窗外脚步停半息，声低：「别出门。」不出，手才不飘；飘了，142谢不出。谢在142，不在138；138只嗅诀，只守坛，只记。记什么？记沉腐浮落齐，记席严沿净，记兄弟线在窗外那半步。半步不进门，进门像情；情在还，不在舌。

刘婆姜汤烫，烫像叶汤那碗还没还尽的——还尽在省亲前，省亲在188前不还；不还不要紧，138只守。守完，丑时末，粟壳香淡，香一字不落。落齐，138这一夜记进心里。心里不骄，只静；静了，139风声紧才像活，不像吓。

守坛夜，坛在，人在，就不滚。不滚，142才谢得出。

守夜至丑时，粟壳香淡，香落齐。落齐，138才像守，不像熬。

守完，天边白一线，白里，他低声道：「守过了。记着。」

铁无言窗外脚步远，远像134修栏那日——日修栏，夜守坛；坛在，人在，就不滚。不滚，139仇风才接得住；接在台，不在骂。丑时末，他低声道：「守过了。记着。」

更鼓沉，天边白一线。白里，他按木牌，牌凉贴胸。

坊市尾巷风硬，他收摊如石，石在，就不滚。

验息后，他理怀内册页，页黄，字丑，丑而真。

铁无言声低：「别飘。」韩泥应声：「不飘。」

刘婆端粥，仍多半勺。韩泥点头，谢轻，恩重。

药峰分拣照旧，指稳，鼻下静一息。

帖在怀，怀沉，沉的是序，不是惧。

白织月袖风过，韩泥不辩。辩，费舌；舌要留来守坛。

赖福廊下过，韩泥只答：「在。」在，就不滚。

柱上条角白一线，三年约在怀。怀凉，掌温。

坡下石阶湿，他数息，数帖，数债序。

同棚杂役笑，笑在耳，不在心。

废渣巷日斜，他验渣，渣正，手才静。

侧门驿亭风硬，他听风，风过，记在心里。

散队后，他过外堂阶下，分拣半刻，指稳——指稳，路就不飘。

编筐半刻，藤刺扎掌，刺浅，浅些沿才净。沿净，坛才藏得住。

丑时，练鼻下静。粟壳香淡，香一字不落。

管事记簿，记细，是小胜。胜在忍，不在辩。

他望坛，低声道：「手在，路在。」坛不应字，只温。
