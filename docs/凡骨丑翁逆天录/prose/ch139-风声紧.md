# 第一百三十九章 风声紧

风声紧，紧在坊尾，紧在帖背。

钱戾衡的人，不近身——近身，像挑衅；挑衅不要紧，韩泥记影。

影在条角，条丑，字真：「预选丑骨，台上方出丑。」

韩泥不辩。辩，费舌；舌要留来守坛，留来候142。

玉牌贴胸，凉，烫着三年约——约说站他台前，才不滚。

滚了，恩断；不滚，刘婆那笔才不算白还。

铁无言立三步外，声低：「风声紧，别接骂。」

「不接。」韩泥说，「接骂，像滚。」

坊市收摊，他立末，像石。石在，风过，风过不留痕。

白织月袖风过，声冷：「钱戾衡的人，你接得住？」

「接。」韩泥声平，「接在台，不在坊。」

她袖过，没停。没停，小胜。

刘婆端姜汤，汤烫，仍多半勺：「风硬，别吹歪路。」

「不歪。」韩泥说，「歪了，约白接。」

回杂役屋，他按坛。坛腹温一线，像应「风」字。

丑时，练鼻下静。粟壳香淡，香一字不落。

他低声对坛：「钱戾衡，记着。台见。」

坛不应字，只温。温像应「约」——约在，手就稳。

更鼓沉，他望帖，帖角掀半分：「风声紧。下一回，积势收。」

邻床悄问：「你怕不怕？」

「怕。」韩泥说，「怕也守。守完，才有路。」

玉牌贴胸，凉，烫着三年约——约说站钱戾衡台前，才不滚。

坊尾条角，字丑，真在「预选」。真不要紧，真在台，不在坊。

管事弟子传话，话硬：「风声紧，别接骂。」韩泥只答：「不接。」

药山半腰，风硬，他听风半刻——风里有仇，仇在远，手在近。

编筐棚借半刻，藤刺扎掌，刺浅，浅些沿才净。

铁无言立三步外，抱臂，像108那日——那日挡，这日站。

「刘婆哼：「风硬，别吹歪路。」「不歪。」韩泥说。台见。」坛腹温，贴掌。更鼓沉，他按心口：「怕也守。守完，才有路。」坊尾条角，韩泥不近，不近像忍——忍不是怂，是活。

钱戾衡影在字后，像仇在路后。路后不要紧，台在前。

玉牌凉，凉里烫约——约说炼气十三，已十四十五；约还说站他台前。

韩泥不辩，辩，费舌；舌要留来数息，留来守坛。

铁无言声低：「台见，别飘。」「不飘。」韩泥应声。

更鼓沉，他望坛：「风声紧。记着。台见。」

「风声紧，紧在坊尾，紧在帖背，紧在钱戾衡那眼风——眼风在字后，像仇在路后。路后不要紧，台在前；在前，手就稳。稳了，140积势才收束；收束像册页翻尽，尽而不越142。142在远，139只紧——紧不要紧，紧才试忍不忍。忍五年赖福，忍108并肩，忍130还刘婆；还了，139才配接仇风。」

仇风接，不接骂；骂，像滚。滚了，约白接；约说炼气十三，已十四十五；约还说站他台前，才不滚。不滚，是活；活了，140才像第二部首簇收束。收束不越175，不越188，不越260筑基；越界，手飘。飘了，丹不正；正，是五年淘的。淘在渣，淘在忍，淘在还。还了刘婆，叶汤在前；前在142，142谢老耿，经叶家。

「家在远，139只记影。影在坊尾条角，条丑，字真；真，才像仇线活着。活着不要紧，活着，台才见。见在掌，不在坊；掌稳，帖才压得住。压得住，140积势记齐——齐，像弓满，箭在142。守完，才有路。」路通向大比，通向筑基，通向省亲；省亲在260后，260前，139只紧。钱戾衡，记着。台见。见在掌，不在坊尾骂。

坊尾影远，远不要紧；远，台才在前。在前，手稳，就不飘。

「影在钱戾衡，衡在台，台在前。守完，才有路。」邻床悄问：「钱戾衡的人，你怕不怕？」「怕。守完，才有140收束；收束不越142，142谢老耿，经叶家。」更鼓沉，他望帖：「台见。记着。」

坊市尾巷风硬，他收摊如石，石在，就不滚。

验息后，他理怀内册页，页黄，字丑，丑而真。

铁无言声低：「别飘。」韩泥应声：「不飘。」

刘婆端粥，仍多半勺。韩泥点头，谢轻，恩重。

药峰分拣照旧，指稳，鼻下静一息。

帖在怀，怀沉，沉的是序，不是惧。

白织月袖风过，韩泥不辩。辩，费舌；舌要留来守坛。

赖福廊下过，韩泥只答：「在。」在，就不滚。

柱上条角白一线，三年约在怀。怀凉，掌温。

坡下石阶湿，他数息，数帖，数债序。

同棚杂役笑，笑在耳，不在心。

废渣巷日斜，他验渣，渣正，手才静。

侧门驿亭风硬，他听风，风过，记在心里。

散队后，他过外堂阶下，分拣半刻，指稳——指稳，路就不飘。

编筐半刻，藤刺扎掌，刺浅，浅些沿才净。沿净，坛才藏得住。

管事记簿，记细，是小胜。胜在忍，不在辩。

他望坛，低声道：「手在，路在。」坛不应字，只温。

更鼓沉，天边白一线。白里，他按木牌，牌凉贴胸。
