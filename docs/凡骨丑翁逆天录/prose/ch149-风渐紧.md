# 第一百四十九章 风渐紧

风声紧，紧在周家——周家逼嫁风声，风在远，在舌，在耳。

坊市有人议：「周家要嫁女，嫁丑杂役？」议在舌，韩泥不辩。

辩，费舌；舌要留来守坛，留来候150收束。

玉牌贴胸，凉，烫着三年约——约说站钱戾衡台前，才不滚。

钱戾衡影在远，影不要紧；周家风在，手更要稳。

铁无言声低：「风声紧，别接骂。」「不接。」韩泥说。

刘婆声低：「周家风硬，别吹歪路。」「不歪。」

疤鬃驴喷鼻，鼻响短，短不要紧；短，像忍。

他低声对坛：「风声紧。记着。不滚。」

坊市尾巷，风里有新词——词是「周家」，是「逼嫁」，是「债」，是149该听的铺垫，不是149该接的架。

韩泥不近巷口，近巷口像沾是非；是非不要紧，听全了，才避；避，才活。

刘婆端菜过，菜粗，声更低：「周家那风声，别入耳？」「入。」韩泥说，「入，才避；避，不在辩，在记。」

记什么？记151才近，记188才远，记149只铺风，不铺架——架接了，手飘；飘了，150收不住。

铁无言抱臂，声低：「周家线，别沾。」「不沾。」韩泥说，「沾，像滚。」

沈枯芽口信到，信仍枯叶，叶上多一字，字小：「周家别逼叶姐。」

字小不要紧，字真；真，才像148叶线续到149——续在远，不在今；今，只听。

管事弟子传话，话硬：「外门少议家事。议了，记簿。」韩泥只答：「不议。」

赖福幸灾乐祸，声尖：「丑骨也怕周家？」韩泥不辩。辩，费舌；舌要留来数息，留来守坛。

疤鬃驴在柱后，鼻喷白气，气稳，稳像149——稳听，稳记，稳不滚。

他低声对坛：「周家风起了。151在前。记着。」坛不应字，只温。

更鼓沉，他收摊，收得像石。石在，风在，风在远，手在近。

风紧，不要紧；紧，才试忍不忍。忍五年赖福，忍142送出，忍149只听不架。

韩泥收摊，收得像石。石在，风在，风在远。

回丙九，他按坛。坛腹温一线，像应「听」——听，才避；避，才活。

邻床悄问：「周家真逼嫁？」「风在。」韩泥说，「风在，151见。」

刘婆端菜过，声更低：「别接架。」「不接。」韩泥说，「接，手飘。」

更鼓沉，他望帖：「150在前。165在远。记着。」

铺周家逼嫁风声，不铺架。架在151～188，188雨夜握腕才解；解在远，149只听。听坊尾新词：周家，逼嫁，债——词在舌，不在手；手若接，像滚。滚了，150收不住，165火探不稳。稳，靠忍；忍五年赖福，忍142送出，忍148叶线不分。分好了，149才配听风。

听风不是怂，是活；活了，才像丑骨末席在大比前该存的劲——劲不花在架，花在记。记151才近，记188才远，记周家线不混叶汤，不混耿恩，不混刘婆。混，七笔乱；乱，手飘。飘了，丹不正；正，是149该守的界。界内，刘婆低声说别入耳，韩泥说入才避；避，不在辩，在记。

记沈枯芽枯叶多一字：周家别逼叶姐——字小，字真，真像148叶线续到149。续在远，149只铺。铺法还有管事硬话：外门少议家事；议了，记簿。韩泥答不议，赖福嘲，韩泥不辩——辩，费舌；舌要留来数息，留来守坛。坛在，149就不滚；不滚，150才收束，165才像钩。

钩青木药火，火在废窑，窑在远；远不要紧，149只风。风紧，试忍；忍过，更鼓沉，收摊如石，低声对坛：「151在前。记着。」记着，是149该落的钉——钉住听，不钉住架。

周家风在远，151才近；近不要紧，149只听，不架。架接了，150收不住，165火探不稳。

周家二字在远，远不要紧；远，150才收得住。收得住，165火才探得不躁。他望帖：「只听，不架。151见。」风在舌，手在近；近，就不滚。

同棚杂役笑，笑在耳，不在心。

废渣巷日斜，他验渣，渣正，手才静。

侧门驿亭风硬，他听风，风过，记在心里。

散队后，他过外堂阶下，分拣半刻，指稳——指稳，路就不飘。

编筐半刻，藤刺扎掌，刺浅，浅些沿才净。沿净，坛才藏得住。

丑时，练鼻下静。粟壳香淡，香一字不落。

管事记簿，记细，是小胜。胜在忍，不在辩。

他望坛，低声道：「手在，路在。更鼓沉，天边白一线。白里，他按木牌，牌凉贴胸。坊市尾巷风硬，他收摊如石，石在，就不滚。」

验息后，他理怀内册页，页黄，字丑，丑而真。

铁无言声低：「别飘。」韩泥应声：「不飘。」

刘婆端粥，仍多半勺。韩泥点头，谢轻，恩重。

药峰分拣照旧，指稳，鼻下静一息。

帖在怀，怀沉，沉的是序，不是惧。

白织月袖风过，韩泥不辩。辩，费舌；舌要留来守坛。

赖福廊下过，韩泥只答：「在。」在，就不滚。

柱上条角白一线，三年约在怀。怀凉，掌温。

坡下石阶湿，他数息，数帖，数债序。
