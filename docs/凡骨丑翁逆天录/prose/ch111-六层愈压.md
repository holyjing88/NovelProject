# 第一百一十一章 六层愈压

春末，药山废窑温一线。

六层愈压，气涨半寸，半寸像首炼余韵。余韵不要紧，要紧是气又实一分——实了，才接得住赖福下一秤。下一秤，是克扣，是折麸，是忍。

赖福立队首，木牌晃：「三折？两折？丑杂役末席，两折。」两折，仍克。克不要紧，韩泥记——记在心里，不进舌。

铁无言声低：「六层，别漏。漏了，白织月来嗅。」韩泥只答：「手稳。」手稳，坛在角，角稳，气才涨。

丙九队风硬，赖福立丙九队侧，声低：「六层愈压，别飘。飘了，折麸册就洒。」韩泥应声：「不飘。折麸册在，手在。」手在，就不滚；不滚，查棚席才接得住。

韩泥盘坐窑角，掌按坛，坛腹热，热进丹田，热得像首炼那日炉心未散。首炼培元散成了，成在渣里，成在丑手里；成不要紧，要紧是气又涨半寸——半寸，炼气六层。六层破时，他独眼半阖，鼻下嗅诀轻动，嗅到渣灰里一点辛，辛正，像培元散余韵。余韵不要紧，要紧是气实了；实了，才接得住下一秤。

他睁眼，独眼平，不声张。声张，赖福来；赖福来，克扣又来。克扣像秤，秤麸皮，秤灵石折换，秤你配不配活。配不配，看手；手稳，六层也稳。稳了，他才把席角压石又按实一分——实一分，坛不露；坛不露，赖福查不出暗修由头。

卯时散队，丙九列队领饭。赖福立队首，木牌晃，晃得像新规矩：「丑杂役，灵石折麸，三折。三折，省宗门粮。」

丙九嗡一声。嗡里，有人看韩泥——看，也像看晦气传染。韩泥不接嗡，端碗，碗糙，粥稀，稀里多半勺是刘婆舀的。多半勺，够顶一口饥。他喝粥不急，急像心虚；不急，才像凡人杂役，才不像偷气。

赖福眼细，细得像秤针，针尖对着他腕：「韩泥，你暗修？」

「管事看手。」韩泥声平，「手抖才滚。」

「手不抖，灵根末。」赖福笑，笑薄，「末等也偷气？偷气，除名。」

韩泥不辩。辩，费舌；舌要留来守坛，留来按窑温。他把玉牌摸怀，牌凉——凉是三年约，凉是钱戾衡那一脚。脚在远，克扣在近；近，先忍。忍不是怂，是活——活长了，才还得了芽儿半块灵石，才接得住外门复测那一帖。

管事弟子路过，瞥赖福册：「三折？执事批了？」

赖福脸一青，青完改口：「两折。舌响，香飘；香飘，手就滚。记了，刃才磨亮半分；亮半分，够撑过查席，够撑过下一压。午后药渣场，他分拣渣灰。渣灰里一点辛，辛正，像培元散余韵。余韵不要紧，要紧是鼻下静——静一字不落，落齐了，六层才实。实了，才接得住赖福下一秤。秤沉，手不抖；抖了，滚字就真。

铁无言抱臂立侧，声低：「六层？」「稳，别漏。」铁无言说，「漏了，白织月来嗅。」韩泥应声。应声，像应坛。坛在角，角稳，气才涨。涨不要紧，不露——露了，赖福查席，查坛，查你配不配留。不配留，叶汤恩断，芽儿半块灵石也还不上。赖福傍晚来渣场，靴底声沉：「韩泥，明日搬粪加倍。加倍，赎你偷气。」」

「没偷。」韩泥铲下稳，「气在坛，不在偷。」

「坛？」赖福眼一眯，「什么坛？」

韩泥独眼平：「腌菜坛。凡人坛。管事看粪清不清，不看坛。」

赖福哼，哼得像没逮住：「凡人坛，也晦气。晦气，多搬一日粪。」

多搬，仍手稳。手稳，是小胜——胜在活，不在嘴。散队后，刘婆端姜汤，姜薄，热：「两折？」

「两折。」韩泥说。

「折什么。」刘婆哼，「手稳，饭稳。饭稳，恩不断。」

韩泥点头。谢轻，恩重。恩在姜汤里，也在忍里——忍里藏着刃，刃对着赖福，对着钱戾衡，对着玉牌上那三字。刃不亮不要紧，先磨；磨在活里，磨在忍里，磨在每一勺稀粥里。

他回漏舍，席角按实，坛不露。坛腹温一线，贴掌，像爹的手又搭上来。爹说过：「坛别扔。」扔不得，扔了就无根；无根，六层白涨。无根，韩家就散了；散了，还恩路也断。

邻床少年悄问：「你真六层了？」

「层在腹，不在嘴。」韩泥说，「嘴响，赖福来。」

少年闭嘴。闭嘴，韩泥摸玉牌——牌凉，仇烫。烫在三年约前头，不在这一日。这一日，只手稳，只坛在，只六层实。

外头风硬，柱上条角白一线。白，像仇苗又深半寸。深半寸，不拔，先活。活长了，才还得了芽儿半块灵石，才接得住外门复测那一帖。帖在远，克扣在近；近，忍一日是一日。

他低声对坛：「六层。记着。记着，刃才磨得亮。」

坛不应字，只温。温不应辱，应「进」——进境一寸，忍一寸；忍一寸，活一日。活一日，刘婆粥多顶一口，叶汤那笔头恩又近一步。近一步，还恩序不乱；序不乱，丹才正。

赖福脚步在廊外停半息：「丑杂役，明日两折，别怨。」

韩泥只答：「在。」

在，就不滚。不滚，六层才实——实了，才配等坊市尾摊，才配还芽儿那一半石。玉牌在怀，坛在角，手在，路在。路在，三年约就不绝。不绝，钱戾衡「让你滚，你才有资格不滚」才接得住。散队后，韩泥仍去药渣场分拣第二趟。渣灰里苦辛交错，他鼻下静，静得像把六层气一点一点压实。

铁无言翻昨夜账，朱印新，声低：「三折变两折，赖福记仇。记仇不要紧，你手别飘。」韩泥只答：「不飘。」不飘，账才清，清才接得住下一秤。

傍晚编筐，藤条浸软，绕两圈勒紧，指节敲底，声闷而实。筐紧，管事弟子无话；无话，赖福脸青半分。青半分，也是小胜——胜在活，不在嘴。编筐时肘新针脚不挂袋角，不挂，风不进；风不进，臂能抬粪，能抬筐，能守窑温。

夜，漏舍更鼓沉。他按席角，席严，坛不露。坛腹温一线，贴掌，像爹说「坛别扔」。扔不得，六层白涨。邻床少年梦呓嫌粪臭，韩泥不辩——辩，费舌；舌要留来记仇苗，记玉牌，记钱戾衡「记住今日」。

丑时二更，药山风硬，硬得像赖福下一秤。他不凑廊下笑，笑在耳，不在心。心在坛，坛醒了，刃才磨得亮。亮在还恩那日，不在舌上。外头柱上条角白一线，白得像坊市尾摊在远——远不要紧，先熬过两折麸皮，先熬过查筐。

秦霜远廊下过，木鱼不敲，声钝：「进境急，心别邪。邪了，鼻下香飘。邪了，手滚；不滚，六层才实。实了，才配沈枯芽半块灵石前那一还，才配刘婆将来新衣养老钱。」

他低声对坛：「六层记着。赖福两折，记着。钱戾衡三年约，记着。」记着，就不白忍。白忍，辱就白挨；不白挨，仇苗才扎得实——扎实了，才接得住112查席，才接得住赖福诬告。

散队后，他过药渣场第二趟，掌按木牌，牌凉贴胸。胸稳，手稳，六层才不虚。虚了，像怕；怕了，赖福克扣成真。真成不要紧，记着；记着，刃磨亮，试台在远，芽儿石在远，刘婆新衣在更远。远不要紧，先六层实，先熬过查筐诬告公示，先坊市捡漏。捡漏了，九层才破，芽儿才还，复测才报。报了，备战，夺魁，还刘婆，大比帖，钱戾衡台前不滚。

丑时末，韩泥廊下练半刻鼻下静。粟壳香淡，香一字不落。落齐了，两折克扣这一日才记进心里。记进心里，不进舌——舌响，赖福记「晦气」；晦气不要紧，手稳，六层才实。实了，才配112查席，才配赖福下一压。

他摸玉牌，牌凉，低声道：「三年约，记着。手稳，粒正，试台见。」见字落，坛温一线，像应。应了，路又明一寸——明向还恩，明向不滚，明向刘婆新衣养老钱。

六层愈压，压不要紧——压实了，才接得住试台验丹，才配备战帖在远。
