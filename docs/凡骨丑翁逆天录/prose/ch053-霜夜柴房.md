# 第五十三章 霜夜柴房

霜夜，柴房风硬。风从板缝钻进来，贴骨，像克扣的麸皮贴肩——克粮，克不走手正。

韩泥搬柴到三更。柴湿，沉，肩勒旧印，印深，像五年杂役磨出来的。丙九末排，柴房不能摆坛：坛在棚角，心却近着坛。近着，才像韩家擦坛的人日夜都挂着心。柴垛越摞越高，高得像把「仍末」两个字也摁进肩里。

铁无言提灯巡夜，灯远，声近：「柴湿，少劈，别闪腰。腰闪了，试手滚。」

「腰记着，不闪。」韩泥应。

铁无言停半息，灯照他掌：「驿亭褐袍人，又候三日。冬深了才赠，赠不白受。」

韩泥独眼平：「不借。」

「袖腥，洗得掉。」铁无言说，「心邪，洗不掉。鼻下静着，别凑亭。」

「不凑。」韩泥道。

铁无言不再言，灯远了。远，像门缝又开一线——线窄，够走。赖福路过，讽一句：「丑杂役守柴，守出仙缘？」

韩泥不答。答轻了，恩倒重了；重了，别给人当笑话嚼。管事贴条于柱：「五日后试手凝丹。误者，滚。」

五日，比辨香又近一步。韩泥摸条，条薄，真：真在仍是末排，真在窄路还在。同批少年碰他袖：「凝丹你也敢想？」

「不想。」韩泥说，「手想正。正了，才不滚。」

「滚了咋办？」

「滚了，恩断。」韩泥说，「恩断了，汤就凉。」

少年懂「凉」，闭嘴。韩泥又补半句：「凉了，还不起。还不起，我守这坛也没意思。」少年听不懂全部，却听懂「没意思」，再不吱声。

三更后，柴房只剩他一人。斧钝，他劈浅些。浅些，腰不闪。闪腰，试手滚；不滚，路还在。柴湿，劈浅，斧声闷，闷像更鼓。更鼓里他歇半刻，掌茧裂口渗汗，汗咸，不是血。血往席上甩，席挡脏。他把斧柄握紧又松开，松开不是懒，是怕腕子僵。僵了，藤条不听；藤条不听，编筐就歪。

他数更鼓，更鼓沉。沉里，柱上五日条像活了半分——活了，试手在望；望不要紧，手要紧。四更，最后一担柴入库。库门落锁，锁声沉得像更鼓落底。铁无言灯照柴垛，垛纹直，直像手正留下的印。他低声：「沉得住，记功在。」

韩泥只点头。记功在，麸皮不克。克粮，克不走手正。他回棚，席角潮，席仍严。邻床鼾雷，雷里他按席，席贴坛角。角温一线，温短，像应「等」字——等冬深，等五日试手。

五更将尽，天边白一线。白像柱上五日条活了。活了，试手在望；末排仍末，路仍窄。他对坛低声：「柴沉，手正。正了，才验得过凝丹。凝丹之前，先把烫的还上。」

坛不应字，只温一线，又凉。凉，像瓮仍眠。卯时前，他又去柴房半刻。霜厚，板缝响，响像牙咬。他摸怀，尚无符，骨寒贴掌。寒不要紧，心记着七笔，便不易飘。

刘婆端粥来，稀，热：「霜夜别空肚。空肚，手飘。」

「不空。」韩泥喝尽。热在，像第七笔又实。

「腰还疼不疼？」刘婆问。

「不疼。」韩泥说，「记着，就不闪。」

管事路过，验柴垛：「湿柴沉，沉得住，记功。」记功，像窄路又宽半分。宽不要紧，要紧是五日条还在柱上。赖福远处骂：「晦气也配记功？」管事声冷：「再嗓，试手名单划你名。」赖福哑了。哑了，韩泥肩上的柴却捆得更实。

午后搬柴过半，肩正。正，像坡下站了一上午辨香，站出了实。编筐时藤运到丙九，他领末捆。藤硬，刺扎掌，掌茧裂口渗潮——潮不是血。领完藤，他回柴房补半担，斧痕深浅尽量一齐，腰才不拧。管事验韧：「韧，编得出筐。筐圆，手才不抖。」

韩泥不偷懒。偷懒，麸皮克；克的是粮，克不走手正。铁无言廊下见他：「五日条在柱，别贪快。贪快，丹歪。」

「不贪。」韩泥应，「贪了，恩断。」

铁无言退回廊角。韩泥第二趟搬柴，侧门窄，窄得只容半眼。半眼够看驿亭。亭里褐袍人仍坐，坐得不急——急便邪，不急像等冬寒够深。他不凑。凑像贪，贪了恩易断。褐袍人袖里黄纸角露一线，露了即收，像试他凑不凑。他只绕远三步：界内末排，界外赠缘。

黄昏雪粒起，细像粉。粉落肩，肩沉，沉得住。他回棚按席，席严，严像祖父韩老栓嘱过的腌菜坛。手不在，嘱还在。席下藏实——藏实了，才少惹查。邻床笑：「拜坛拜出邪脉。」

管事路过，声冷：「再嗓，滚去西栏清粪。」

邻床闭嘴。韩泥不辩，只把辱记在心里，与恩分开摆。摆清了，心就不乱。夜里他对坛低声：「霜夜柴沉，手要正。正了，才接后头五日。」

坛腹温一线，又凉。凉，像应「等」。外头风息，柱上五日条更真——真在试手，真在末排，真在窄路。路通向凝丹，凝丹通向还。还，先还烫。丑时坛腹又温一线，温短，像应「柴」字，不应慌。他指腹蹭牌角，蹭出一点实。实了，五日后才站得住。

刘婆门外咳：「试手前夜，别空肚。空肚，鼻下香飘。」

「不空。」韩泥应，「粥在，姜在记里。」

他又低声道：「五日，试手。试得过，关才开第二道。关开了，先还烫。」木牌在枕下，与五日帖并放——命与规矩挨着，才验得过。老耿门外挑水，水洒实，声哑：「褐袍人候冬。冬深了才赠。」韩泥应声，不凑亭。四更前他又劈半担，劈浅，斧钝不要紧，腰记着就行。腰记着，试手才不滚。

铁无言灯远之前，又嘱：「手正，别拜邪。拜邪，恩断。」

「不拜。」韩泥只答这一句。

「肩还正？」

「正。」韩泥说，「正了，才验得过。」

卯时管事验腰，看斧痕深浅：「深浅一致，腰没闪。」

「闪腰，试手滚。」韩泥说，「不滚，窄路才实。」

秦霜远廊经过，停半息：「霜夜别空肚。」她不再言，木鱼在指间，不敲，只托着——托着，像规矩还在耳边。刘婆又端半碗姜汤来，姜薄，热：「霜夜腰寒，姜压一压。压住了，手才不僵。」

韩泥喝尽，只点头。姜热像灶下还着的一星火——火在记里，不在舌头上。

柴房库门边，管事又叫他数垛：「数清，别给赖福抓空子。空子一抓，记功变记过。」

「数清了。」韩泥说，「七垛半，半垛在门左。」

「记住数。」管事道，「试手前这几日，腰、手、数，一样都不能糊。」

「不糊。」韩泥应。

他回身把斧入架，架上斧痕深浅仍齐。齐了，像给腰再钉一颗钉。钉在腰里，不在嘴上——嘴上响，手就飘；手飘了，五日条上的「滚」字就会活过来咬人。晨光里，柱上条字边细冰化尽，像试手从柱上落到掌上。掌还正，坡下才站得住。

霜夜将尽，斧声歇了，手还正——正着，才配接后头凝丹那一关。柱上五日条仍在，明日验的不是灵色，是腰里那一口气还沉不沉。沉着，窄路就还在；路在，汤就不凉。凉不得：凉了，霜夜里劈出的那点实，就白费了。

他临走又看了一眼柴垛纹路，纹直，直像「五日」二字压进肩胛——压住了，腰才不闪，手才不飘。棚外更鼓最后一响沉下去，他才把霜从眉上抹掉：抹掉的是霜，抹不掉的是柱上那条试手期。期在，肩就得沉；肩沉着，坡下才站得住。站得住，才谈得上还。还字一响，霜夜就不白挨。不白挨，斧才算没白抡过。
