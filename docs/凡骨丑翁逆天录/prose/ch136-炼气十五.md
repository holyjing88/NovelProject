# 第一百三十六章 炼气十五

验息这天，天没亮透，药峰坡上已经冷得扎手。

韩泥站在队末，独眼半阖，内视里那道阶影稳稳停在十五。稳了不是高兴，是活着——活着，大比那条路才多出一寸来走。

管事弟子挨个摸脉，摸到他，停了半息，才落笔：「炼气十五层，登记。」

「登记。」韩泥应得短。短才像平常过日子，不像显摆。

同批少年里有人嘀咕：「丑骨也能十五？」

旁边人怼回去：「嘘！管事听见，罚你扫半个月渣。」

韩泥不辩。辩像逞能；逞能不要紧，层在手里、手在筐上，比嘴硬管用。

药峰派活照旧。分拣半刻，他鼻下静着，只闻辛淡，不闻耳边那点辱。辱在耳可以过，进了鼻子就坏丹；鼻子准了，往后谢老耿那一笔才送得出。

棚里编筐，藤刺扎掌，刺浅。刺浅些，席下坛沿才干净。筐卖了钱买米，米在，恩才还得上。

铁无言送栏灰过来，灰细得像筛过，声更低：「十五层了，别飘。」

「不飘。」韩泥说，「飘了，预选帖白贴怀。」

「帖白贴，台白上。」铁无言抬了抬下巴，「你懂就好。」

刘婆端粥进棚，仍多半勺，热气糊住眼：「层高了，饭也要稳。空肚手飘，飘了嗅不准。」

「嗯。」韩泥接碗，「记着。」

「记着啥？」刘婆哼，「记着别瞎跟少年比跑。你跑不动，站得住就赢。」

赖福火折照席角，亮得刺眼：「席下啥味儿？贼腥？」

「渣味。」韩泥声平，「药山活，活人身上有味。」

「渣味？」赖福冷笑，「渣味也能登记十五？造假了吧？」

「验息册上有字。」韩泥说，「字在，假不了。」

赖福噎半息，火折一晃就退。退像没查着——查不着不要紧，席严着，坛就不露。

丑时，人睡了，他练鼻下静。粟壳香淡，香落得齐，齐到心里只剩一层实。

他望坛，掌贴坛腹，低声道：「十五实了。记着。」

坛腹温一线，像应「层」字。层在，路在，人就不滚。

邻床少年翻身，悄问：「韩叔，层高了，怕不怕上大比？」

「怕。」韩泥说，「怕也守。守完，才有路。」

「怕还守？」少年讷住，「你们老杂役都这么拧？」

「拧着才活到今儿。」韩泥道，「顺顺当当的，早被骂散了。」

更鼓沉，他按心口：「汤还是热的。」

六字落地，像路标——标向谢老耿，标向省亲，标向叶青禾那碗还没还尽的汤。汤热着，这层才不算白涨。

白织月远廊过，袖香冷，脚步不停。不停，是小胜——胜在层实，不在跟她争嘴。

药廊杂役挤眉弄眼：「丑翁升层，白姑娘看一眼都不肯？」

「看一眼又怎样。」另一人撇嘴，「他手稳，比看一眼值钱。」

韩泥听见，当没听见。听见了去辩，像把层当喇叭吹。

管事弟子午后再验，册上添一行：炼气十五。字丑，真在。真了，才像积势，不像运气。

铁无言又丢来一把炭渣：「层高了，别骄。骄了，手飘；手飘，丹不正。」

「不骄。」韩泥说，「骄像滚。滚了，刘婆那笔白还。」

「还清了还怕滚？」

「怕。」韩泥独眼平，「还了一笔，后面还有叶汤。滚了，汤凉。」

废渣巷日斜，他验三包渣。两包正，一包涩。涩的剔开，里头果真掺了石屑。

递渣的少年脸白：「你……你真闻得出来？」

「闻得出来。」韩泥把石屑弹回他掌心，「掺石，想坑我，也想坑挂牌的名声。」

「我……我被人逼的……」

「逼不逼是你的事。」韩泥说，「渣到我手，就得正。不正，续不了摊。」

少年抱头去，他收渣入囊。囊净，手稳——十五层不是雷，是这一线实。

赖福夜里又查，火折照见厚席不见坛。坛不露，沿才净；沿净了，嗅诀才练得稳。

同棚有人压低嗓子：「听说钱戾衡那边盯上预选丑骨了。」

「盯又怎样。」另一人回，「盯着皮，盯不进骨。骨丑，路还得他自己走。」

韩泥听着，把「钱戾衡」三个字按进心里，不接嘴。接嘴像接骂；接骂像滚。

侧门驿亭风硬，他过亭时收了收怀中帖。帖在，序在——药峰、坊市、守坛、风声，一寸寸来，不抢。

坡下石阶湿，他一步一步数。数帖，数债：刘婆还了，叶汤在前，老耿在前。前在远，今儿只把十五实牢。

午后药峰加了一趟分拣。管事把筐扔他脚边：「十五层也得弯腰。弯不下去，层给你拿回去。」

「弯得下。」韩泥蹲了，指尖翻叶，鼻下一息：「这包潮，得晒。」

「潮你也闻得清？」少年挤过来，不信。

「潮进炉，火不匀。」韩泥把包推向日头，「晒透再炼，省得炸锅。」

管事鼻子哼一声，却没再嘲。簿角又添半笔，笔迹歪：分拣稳。歪不要紧，真在。

晚饭档口，同棚杂役拿竹筷敲碗：「韩叔，十五了，请喝口酒吧？」

「不喝。」韩泥摇头，「喝了鼻钝。鼻钝，明日验息账翻脸。」

「翻脸你也登记了啊！」

「登记是今儿。」他声平，「明儿还得活。活着比酒香。」

人哄笑散了。笑在耳，不进心。心里只留粟壳那点淡香——香落齐，人就不飘。

秦霜远远敲了两下木鱼，声轻：「层升，心动。心动，魔贴边。」

韩泥对空应了半句：「心不动。动的是手，手要稳。」

夜里赖福又转一圈，靴底特意踩重：「十五层杂腿子，夜里还装勤？勤给谁看？」

「给坛看。」韩泥说，「坛看得见，您看不见，正好。」

「油嘴！」赖福抬火折，照了照席角，又照了照他袖口，「袖里夹丹没有？」

「没有。」韩泥抖袖，「袖空，手不空。手不空，活才在。」

火折熄了半截，赖福骂骂咧咧走。门一响，棚里才松口气。

邻床少年悄悄竖大拇指：「韩叔，你今天硬。」

「硬没用。」韩泥躺下，独眼望梁，「硬过火，明天就被掐。掐之前，先把筐编完。」

他摸黑编了半扇筐沿。藤条涩，涩得掌热。热提醒他：层是实的，饭是实的，恩也是实的——实了，才配把「谢」字往后排。

管事扬声收队：「十五层已登记者，明日照常派活。别以为升层就能歇。」

「照常。」韩泥应。应了，才像杂役该有的活法。

散队后他回丙九，按坛沿。沿净，指稳。稳像弓拉满——满不要紧，箭不在今日喜，在往后谢那一粒培元散。

铁无言隔窗丢进一小块炭：「夜里别瞎试高招。高招不稳，掉层丢人。」

「不试。」韩泥捏炭，「试在渣上，不在气上。气稳就留着。」

「留着对。」窗外人走了，脚步像修栏，稳。

天边白一线，药峰影将至。将至不要紧，手在，路在。

刘婆清晨又探头：「升层了脸别肿。肿了，瞧不见路。」

「不肿。」韩泥应，「肿的是别人眼。我眼只一只，看债够了。」

「油。」刘婆哼着走，又回头，「粥在灶边，自己盛。多盛半勺也行。」

他盛了。半勺热进胃，胃暖，掌才不抖。

药廊角落，沈枯芽远远鞠了半躬，不敢近：「韩爷爷……层……层高了，叶姐让我问一句，您还记不记得汤。」

「记得。」韩泥说，「烫的。辣的。趁热才记得住。」

「那就好。」芽儿眼眶一红，又忍住，「我走了。走慢点，省得惹眼。」

「走。」韩泥挥手，「别替我传闲话。闲话一响，汤就凉。」

芽儿去后，他把「汤」字又按进心口一分。按实了，十五层才不像孤立的数字——数字要会走路，走到还得起恩的地方。

柱下有少年故意抬高嗓：「丑骨十五？怕是管事眼花！」

「眼花验得过脉？」旁人怼，「你验一个给我瞧瞧。」

嘲的人噎住。韩泥从他们身边过，不停，只把筐绳勒紧。勒紧了，人就不飘。

他低声对坛：「手在，路在。下一回，坊市续。」

坛不应字，只温。温里像压着一声闷鼓——鼓还没响，街已经开始有人说大比。说大比的人多，真听风的人少；他得去末摊坐着听，听清谁是谣、谁是真，才不把十五层晃散。
