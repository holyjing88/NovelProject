# 第三十二章 一盒茶饼

秦商言将离那日，莫长春留她半刻。

客厢茶桌，无珍馐，只有一壶粗茶，两只杯，杯口缺一角，像执法堂寻常待客——不隆重，反而让秦商言这种算惯了灵石的人，更摸不准深浅。

她坐得直，像随时准备讨价还价。莫长春从袖中取出一盒茶饼，纸包旧，边起毛，角磨圆，像揣了很多年，揣在袖里与簿挨着，挨着挨着，就成了礼。

「家乡茶。」他推过去，「不值钱。请你尝。」

秦商言一愣，目光在纸包与莫长春杖尖之间游移：「莫长老……不求价？」

「不求。」

「商道讲公平。」她目光微顿，第一次像被「无价」绊住，「无价的礼，最难还。商言做了十年买卖，最怕难还。」

莫长春笑：「难还就好。还得起的，就不叫礼。」

秦商言打开纸包，茶香扑出，苦后回甘，像凡间日子，不像百宝阁货架上的灵茶。她拈起一块，细嚼，细嚼出涩，也细嚼出暖——暖得像有人把「赠出不计价」五个字，放在她舌上。

袖中什么一跳。她无簿，却像站近这老头时，能觉出「记一笔」的重量。

下一瞬，案上多一卷薄简，墨迹新干，像茶气里突然长出字来：

**赤焰东脉调动：三日后佯攻，七日后夜袭。**

秦商言抬眼，声低：「这笔买卖，商言还没算清。」

「茶饼算清了。」莫长春摆手，「简是回赠。你问赤焰在哪花钱，老夫答在妖与田之间——茶，是谢你肯答。」

她沉默两息，忽然问：「若商言把简卖给别人呢？」

「卖就卖。」莫长春道，「情报值钱，人情更值钱。你卖简，是商道；你记恩，是礼。各算各的账。」

秦商言起身，郑重揖手：「秦商言记下了。」

出厢时，柳青鸢在廊下路过，目光相触，各自点头，像宗门与商会之间，多了一根看不见的线，线那头拴着东脉，这头拴着茶饼。

山下驿站，秦商言对下属道：「莫长老送的茶，比灵石贵。记入恩账，不记入买卖账。」

下属问：「若赤焰夜袭成真，商言赚不赚？」

秦商言望青岚方向，笑，笑里第一次像亏本也乐意：「赚。赚在青岚若赢，百宝阁有路走；若输，商言也有恩可还——恩，是另一条商路。」

夜，莫长春独坐廊柱旁，不坐主位，簿上浮字：

**茶轻意重，情报入账。**

他念：「秦氏记下了。下一阵，东脉账本。」

杖中竹纹一游，像应。风大，像把茶香送回丹堂，送回东脉，送回三月后的天试。

赵玄机在执法堂听闻「情报入账」，指节叩铁牌，一声，像把妒意敲实，又敲松一寸。

他念，像章末钩：「茶轻，简重。大因果还在赤焰夜袭那夜，像妖骑，像杖剑，像全场寂然的斩。」

大因果还在赤焰夜袭那夜，像妖骑，像杖剑，像全场寂然的斩。

山下驿站，秦商言对下属道：「莫长老送的茶，比灵石贵。记入恩账，不记入买卖账。」下属问：「若赤焰夜袭成真，商言赚不赚？」秦商言望青岚方向，笑里第一次像亏本也乐意。

赵玄机在执法堂听闻「情报入账」，指节叩铁牌，一声，像把妒意敲实，又敲松一寸。弟子围问：「长老又送礼？」莫长春笑：「礼送了，简收了。接下来看宗主怎么战。」

风大，像把茶香送回丹堂，送回东脉，送回三月后的天试。茶轻，简重——大因果还在赤焰夜袭那夜。百宝阁车马下山，山下弟子踮脚，踮得像看一场亏本却乐意的买卖，买卖里敬，也疑——疑茶饼真能换情报。

秦商言入厢，青衣利落，笑挂在唇。她看向莫长春，目光停得久一分：「听闻莫长老将死之人，手却松——坊市送棉衣，东脉定魔少，风声都传到百宝阁了。」莫长春拄杖笑：「手松？老夫手紧。紧在只紧该紧的地方。」「赤焰最近在哪花钱。」「在妖与田之间。买路，买命，买你们青岚门犹豫的那七日。」

秦商言打开纸包，茶香扑出，苦后回甘。案上多一卷薄简，墨迹新干：**赤焰东脉调动：三日后佯攻，七日后夜袭。**「茶饼算清了。」莫长春摆手，「简是回赠。」秦商言起身，郑重揖手：「秦商言记下了。」

围观弟子在客厢外踮脚，踮得像等秦商言真吃茶饼——吃了，简才长出来。有人低声：「无价礼，最难还。」另一人答：「难还，才记恩。」
