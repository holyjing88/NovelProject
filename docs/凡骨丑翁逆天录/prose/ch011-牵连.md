# 第十一章 牵连

谣起半月，叶家药铺门先冷了。

霜后风硬，巷里人绕铺走，绕得像绕瘟。韩泥早起编筐，藤条浸了一夜，皮软，绕两圈勒紧——结在，筐不散。他担筐过巷，肩上旧印又深一层，深得住，就不慌。

赵绣春站在叶铺前，嗓子尖得像刀背刮铁：「晦气沾了铺，谁还敢抓药？寡妇不要脸，连绝户翁都喂！」

帘后秤砣落盘，一声，沉。叶青禾没出，那一声够了——沉着，铺还在。

韩泥脚步不停。停，坐实「牵扯」；不停，耳已入耳。入耳就得记：记的不是谣，是叶家被人往脏水里按。

村童绕远，绕得直白。绕的不是他，是「叶家也沾了」的新词。童谣丫墙根改词，念得脆：「叶家铺，晦气铺，寡妇送汤全家倒——」

念到「倒」字，自己卡壳，像怕叶家秤真砸下来。韩泥路过，停半步，只道：「倒字烫嘴。烫了，就别硬念。」

丫缩脖子，跑了。跑了也好——嘴停一息，叶铺就能喘一口气。

晌午，他赴集尾卖筐。有人拒收：「晦气筐，压价。」

韩泥「嗯」一声，换摊。半个时辰，仍卖十二文。买主捏筐沿，又看他手，手疤旧，不像邪。韩泥道：「晦气在嘴上。筐里只有藤。」

买主笑短，仍付十二文。十二文落掌，烫一息——烫着，像村还没把活路说死。

集尾另一摊，有人窃语：「叶家秤砣，夜里也落。落给谁听？」

「给寡妇听。听她还不还汤。」

韩泥独眼不抬：「秤砣落，药还抓。抓药的人，不怕秤。」

两摊静一息。静，不是服他，是服病——病一封口，谁都得掂一掂那十二文的分量。

午后，周嫂聚妇人在井台，棒槌砸得响：

「叶青禾送姜送汤，药铺生意淡，活该！」

「沈家芽儿被罚，也是她带坏风气！」

「赖管事说了，再牵扯，叶家迁出村！」

韩泥打水，桶满，拎稳。「井是村的。」他嗓哑，「药铺也是村的。村人生病，不抓药，抓谁？」

周嫂棒槌悬半空：「你替寡妇说话？」

「我替活路说话。」韩泥转身，「谣在嘴上，病在身上。嘴能停，病不能停。」

周嫂脸青，棒槌落下，砸衣上，水花溅自己脚。旁人窃笑，笑轻——笑的是周嫂噎，不是笑韩泥。

马春花远处帮腔：「韩泥，你少装。装善人，叶家就能开？」

韩泥拎桶走：「开不开，病说了算。你嗓子尖，尖不治咳。」

马春花噎半息，没再追。追，像怕井边人多；人多，理还沾一点边。这是小胜：井还能打，话还能落一句。落一句够，不多落——落多了成靶。

周嫂棒槌又悬，悬在衣角上，像还想砸，砸不出声。韩泥不回头，只把桶担直。担直了，叶铺方向的灯弱，仍亮；亮着，牵连就不算死局。

傍晚，叶家药铺门板半掩。韩泥不进门，把编筐剩的三根细藤搁门阶，转身走。走三步，门内出一声，低：

「韩翁。」

叶青禾。「藤补筐用。」韩泥没回头，「我不欠藤。欠汤。」

「汤……」她停一息，「村人说，再送，铺子开不下去。」

韩泥停步，侧半脸，独眼平：「铺子开不开，是病的事。送不送，是你心的事。心的事，别听嘴。」

帘后又一声秤砣，轻，像应他。叶青禾低声：「赵绣春说，再牵扯，要报官。」

「报。」韩泥道，「官来，也抓药。药铺关，全村咳。」

就五句。叶青禾没再追。她背影仍直，直得像那碗汤还热着。

韩泥回屋编筐，藤条绕膝，勒指。指疼，疼得清醒——清醒里，叶汤那笔恩，像一根梁。梁若被谣咬断，整间泥屋都塌。

他按了按心口，摸到贴胸那张「趁热」纸角。纸角旧，边毛，像叶青禾没退的证明。

韩泥低声道：「牵连你，是我的债。债我担，不推你。」

说给自己，也说给帘后那声秤砣响——响在夜里，像叶家还开着门。

夜里，村东忽起狗吠，吠急，像有人砸门。韩泥推门望，叶家药铺方向，灯乱晃，妇人声尖：「滚！再闹，报官！」

赵绣春嗓更高：「报啊！看谁晦气！」

里正声插进来，沉：「再闹，记名！记名，连坐！」

赵绣春一窒，骂声收半寸，尖还在，后力没了。赵党有人还想嚷，嚷到一半，见里正灯笼照过来，声吞进喉咙。

灯笼晃三下，照见叶铺门阶干净——干净，像没人敢踩晦气进门。灯灭一息，又亮。亮得弱，像叶家撑。

韩泥合袄，塞怀。他不往前冲——冲是害，害叶家更深。

「要还。」他摸心口，「得活到还得上的那日。」

巷口，陈姑咳着路过，停门缝外，没进。「韩翁。」她嗓沙，「叶家的事，你别往前凑。凑，害她。」

「我不凑。」韩泥没开门，「我记。记着，等公审。」

陈姑顿一息：「公审……真开？」

「里正锣未敲，心先悬。」韩泥道，「悬着，叶家就不算死。」

陈姑「嗯」一声，脚步远，远里像叹——叹轻，像也给叶家留一口气。

布贩子挑担过巷，担绳吱一声，停半息，像想说什么，终究把脸别开，走开。韩泥懂：躲不难，难的是躲了还记。

沈枯芽门缝问：「韩爷爷，叶姑姑会关铺吗？」

韩泥应声：「秤在，就不关。」

沈家豁口灯芯跳一下，像有人把娃按回被里。娃不问第二遍——问多了，娘咳。

叶铺帘静，谣又近一步。他不近阶前——近是害，远是护。护到公审，护到叶丫头还能开秤；秤在，这恩才还得了烫，烫不能凉。

夜里，韩泥编筐至丑时，藤勒指，指疼，疼得清醒。坛腹微温一丝，像应他那句「记心，不记逃」。逃了，叶汤就浅；浅了，公审前心就空。

村狗再吠两声，吠向叶铺，吠完又哑。哑，像全村悬着一口气——气在叶家灯上，灯弱，未灭。

一袋粗盐搁门阶边，无人认领。盐白，白得像不该沾晦气的手才敢搁。韩泥不收，只起扣鞋泥，泥里一缕咸，像村还留着一句没说出的话。

盐担老汉在巷口咳嗽，声低：「韩翁，盐我搁的。叶家抓药，我娃还活着……别嚷，嚷就被赵家听见。」

韩泥没接盐：「盐你收回去。收在心上就够。」

老汉愣半息，把盐袋又掂回肩上，走时肩膀松了半分——松着，像听见了「够」。

漏雨泥屋又漏一阵，雨丝细，细得像针。针扎肩，他不躲——躲像亏心。心不虚，雨扎就扎。他摸坛，坛凉，心热，中间夹着五十八年的命。命薄，还没断；断不了，就因为叶汤还在，叶家灯还弱着。

丑时过半，叶帘内秤又落一声，落脆。脆响压过巷尾马春花那点余尖——尖不过秤，秤在，铺就不关。

张麻婆隔墙骂童谣丫：「少学那改词！念到叶家门口，秤砸你脚！」骂完往自己门缝塞半块饼，塞完又骂：「吃你的！少凑热闹！」

韩泥听见，唇角没动。动像谢。他只把藤条绕得更紧。

侧巷里，叶青禾似站了一息。没近，声低，像隔墙落进泥：「韩翁，别替我认晦气。」

韩泥应了一声：「我不认晦气。认债。」

「债……」她顿，「公审上再说。」

「嗯。说。把汤说热。」

就这几句。她脚步走远，秤砣又落一声——落给全村听：铺还开着。

韩泥藤条绕膝，心里那笔债更实。实了，手就不抖；手不抖，将来还的时候才不寒酸。他合上眼，低声把三样按住：叶灯要在，赵辱要清，牵连是债——债我担。

风从门缝过，带一点姜的辛味。他按坛沿，沿在，根就不散。瓮还没醒，他不喊醒——明天村里还要嚼舌头，舌头热不过公审的锤子。

明日还要去井台打水，还要去集尾卖筐。叶家灯弱，弱也是亮；亮着，牵连就不算死局。心不空，只候锤——锤来，证白才爽。爽是后话，话先在「我记着」里。牵连是债，我担；等公审，把汤是热的说清。
