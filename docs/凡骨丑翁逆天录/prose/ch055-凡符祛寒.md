# 第五十五章 凡符祛寒

冬深，霜厚，宗门外驿亭终于有人起身。

韩泥搬柴过侧门，老耿在旁挑水，水洒半勺，声哑：「去。不白拿。拿了，记耿恩，记僧恩，不拜血祠。」

褐袍僧不诵长经，袖里取出黄符，符边毛，毛得像凡物。凡物，才入凡人手。

「寒侵骨，骨寒侵手。」僧声低，「符不入丹房，不越阶。阶在远，寒在近。近，先祛。」

韩泥不接，先问：「为何是我？」

僧指他掌茧裂口：「你鼻下静，手下稳，坛边席严。严者，符不辱主。」

老耿哼一声：「僧是我引的。引，不是邪，是冬里少死人。」

韩泥这才接符。符薄，贴掌，掌暖一丝，暖得像经片贴怀。

「凡符祛寒。」僧道，「燃则一日暖，不燃则贴怀。贴怀，护骨寒，不护心邪。心邪，符自灭。」

韩泥「嗯」一声：「心记着七笔，不邪。」

僧又嘱：「符凡品，祛寒，防。勿示人，示人，谣又来。」

僧合十，不拜坛，只拜冬：「瓮在眠，人别睡死。睡死，是拜邪。眠，是等。」

僧离亭，脚步远。远，像凡符使命落了地。

回棚，符藏怀，贴身，不燃。燃，烟起，管事查。

赖福嗅他袖，嗅像找由头：「藏啥？」

「寒。」韩泥说，「寒在骨，不在袖。」

管事路过，声冷：「宗门外的事，少议论。议论，像借商线腥。」

赖福哑了。管事又道：「符凡物，别傲。傲，邪。」

刘婆端姜汤，姜薄，热：「怀里有符，别傲。」

韩泥喝尽。姜热，像远村灶火——远火在，还路在。

邻床问：「啥符？」

「凡符。」韩泥说，「祛寒，防。不燃，贴怀。」

「贴怀干啥？」

「贴怀，手不僵。」韩泥说，「手僵，藤刺偏。偏刺，血找错路。」

午后，西栏清粪，风硬，符暖护骨，手不僵。粪锹沉，沉得像搬柴。

搬柴手稳，清粪手也要稳。稳，试手才不滚。

铁无言巡栏边，声低：「符在怀，别示。示，谣来。」

韩泥「嗯」一声：「不示。示，像借腥。」

小兽咳，咳得细。他多铺一层干草，草旧，旧能暖。

管事盯梢：「清粪加半刻。半刻偷懒，克扣麸皮。」

韩泥不偷懒。克的是粮，克不走还路。

夜，他对坛低声：「凡符起，仍凡。凡，够活命。活命，才还恩。」

坛腹温一线，不与符抢。符护人，瓮眠等缘。

秦霜远廊经过，一句：「心别邪。邪，符灭。」

韩泥怀符暖一丝，心记着七笔，不邪。

丑时，他按席，席严，沿净。净，才接后头血近沿，接瓮醒那一误。

外头更鼓沉，柱上五日条在。五日，试手凝丹。凝丹前，符不燃，沿不血，席盖严。

同批少年碰他袖：「你真得了符？」

「得了。」韩泥说，「凡符，祛寒。不示，不傲。」

「示了咋办？」

「示了，谣来。」韩泥说，「谣来，路易断。」

少年懂「断」，闭嘴。

第三日，搬柴经侧门，风更硬。韩泥不凑亭，只瞥一眼。褐袍亭已空，空，像赠毕。

铁无言巡栏边，声低：「符在怀，手别僵。僵，藤刺偏。」

韩泥「嗯」一声：「不僵。僵，血找错路。」

刘婆端粥，稀，热：「符在怀，别空肚。」

韩泥喝尽。热在，像第七笔又实。

符贴怀第三日，他不燃不示。示了，谣来；谣来，路易断，粥就凉。

他独眼平，手不抖：「符护骨，沿在净。净了，才接后头缘。」

符录起了，仍凡，不越阶。凡，够活命；活命，才还恩。

僧赠符时，风硬，风从驿亭过，过宗门侧门，过丙九棚角。角潮，席严，沿净。净，符才肯入怀。

符贴掌那息，掌暖一丝，暖得像经片贴怀。经片在泥岗，符在宗门。宗门符，仍凡，不越阶。

回棚第一夜，他不燃符，不示符。示，谣来。谣来，路易断。断，粥就凉。凉，还路就窄。

四更，邻床翻身，袖风扫过他怀。怀暖一丝，符不灭。灭，心邪。心不邪，符自护。

卯时搬柴，风更硬，硬像西栏粪锹。粪锹沉，柴也沉。沉不要紧，符暖护骨，骨不寒，手不僵。

管事验垛，声沉：「冬里沉得住，记功在。」记功在，麸皮不克。克粮，克不走还路。

午后，铁无言廊下见他：「符凡品，别傲。傲，邪。」韩泥「嗯」一声：「不傲。傲，符灭。」

小兽栏角，小兽眼湿，咳得细。他多铺干草，草旧，旧能暖。暖兽，像冬里少死一人。

黄昏，坛腹温一线，不与符抢。缘在沿前，不在符烟。

夜，他对坛低声：「凡符在怀，仍凡。凡，够扛丑时寒。寒扛住，手才不僵。僵，藤刺偏。」

第二日清栏，粪锹沉，符暖护骨。骨暖，掌裂不僵。不僵，缘才正。正缘，才接后头试手。

刘婆又端半勺姜汤，热：「符在怀，别空肚。空肚，鼻下香飘。」韩泥喝尽：「不空。姜在记里。」

僧离后，驿亭空，空得像赠毕。毕了，怀才暖。暖一丝，够扛丑时寒。

回棚，邻床问符味。味淡，淡得像凡物。人手，才贴怀。

第一夜丑时，他不燃符。查，像借腥。腥，恩易断。

第二日，搬柴风硬，符暖护骨。骨不寒，掌裂不僵。不僵，藤刺才正。

西栏第二趟，粪锹沉，符仍暖。暖，不抢瓮温。瓮眠，符护人，各守各的。

管事第二趟路过：「符凡物，别傲。傲，符灭。」

小兽第二日咳得细，他多铺草。草旧，旧能暖。

第三日，褐袍亭尘落，尘落像僧影散尽。散尽，符使命落了地。

铁无言第三趟嘱：「符在怀，手别僵。」韩泥「嗯」一声：「不僵。僵，血找错路。」

夜第三日，坛腹温一线，不与符抢。抢，像拜邪。不抢，像各守各的缘。

他对坛低声：「凡符录起，仍凡。凡，够活命。命在，才还恩。」

刘婆第三日端粥，热：「符在怀，别空肚。」韩泥喝尽，「嗯」一声。

符贴怀第四日，他不示人不燃符。燃，管事查。示，谣来。

搬柴第四趟，风硬，符暖护骨。骨暖，手不僵。不僵，缘才正。

西栏第四趟，粪锹沉，符仍暖。暖一丝，够扛冬寒。

秦霜第四日远廊一句：「心别邪。」韩泥「嗯」一声，心记着七笔。

夜第四日，坛与符各守各的。

他对坛低声：「符录起了。起了，仍凡。凡，够活命。」

刘婆第四日粥，热。韩泥喝尽。

符贴怀第五日，韩泥仍不燃不示。怀暖一丝，护丑时骨寒。骨不寒，掌裂不僵，藤刺才不偏。

管事第五日嘱：「宗门外的事，少议论。」韩泥不议。不议，恩在，仇也在，都在心里，不在舌。

铁无言第五日廊下见他，声低：「符凡品，祛寒，防。勿越阶。」韩泥「嗯」一声：「不越。越，像邪。」

夜第五日，他对坛低声：「凡符在怀，瓮仍眠。眠，不争。争，像拜邪。」坛不应字，只温一线。

符贴怀第六日，他不燃不示。示，谣来；谣来，路易断。韩泥独眼平，手不抖，沿在净。

第六日搬柴，侧门风硬如刀。刀贴骨，符暖护之。护骨不寒，手才不僵，缘才正。

管事第六日嘱：「符在怀，少声张。」韩泥「嗯」一声。声张，谣来；谣来，路易断。

夜第六日，符与瓮各守各线。

符贴怀第六日，沿在净。净，才接后头试手。

僧影散尽后，驿亭空。空，像凡符使命落了地。
