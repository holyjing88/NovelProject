# 第十五章 村议

村议辱叶后两日，里正敲锣。

锣声沉，沉得像把全村嘴压进泥里。韩泥臂布未拆，早起编筐，藤条绕膝，绕得像把明日要说的话先编实——实了，才撑得住锤。

野藤先拿刀刮去老皮，再入水浸软，浸软了才编。藤条在膝上绕一圈，勒紧，再绕一圈——绕两圈，勒紧，留半寸尾，尾塞结里，跑不了。指节起皮，起皮不慌；慌了结松，筐就塌。

他编筐至晌午，筐底圆了，沿齐了。用指节敲了敲底，声闷，闷而实——实了，才配装药，配卖钱。里正锣又响，响得他搁下藤条，往祠堂去。

「叶家事，不能糊。」里正扬声，「明日巳时，祠堂开村会，全村到场，辨诬还清白。缺席，记名。」

村人聚祠堂，赵绣春党不散，散的是胆——胆怕里正报官，怕宗门查连坐。

马春花嗓还尖：「辨什么？寡妇送汤，事实！」

里正板脸：「事实，当众说。说不服，再罚。」

韩泥坐末席，独眼低。低不是怂，是等——等叶家开口，等锤有期。

议到午后，争出三条：

一，叶家关铺谢罪，七日；  
二，韩泥搬离村西，换破庙；  
三，开村会大会，村人作证，辨「送汤是恩是邪」。

赵绣春要一、二要，不要三：「三有什么好辨？辨了给晦气翁脸？」

叶青禾在帘后，声出，平：「要三。不要一、二。」

「不要关铺？」周嫂叫，「铺子晦气，谁抓药？」

「铺子开。」叶青禾道，「抓药的，是病。不是嘴。」

里正拍板：「三留。一、二缓。村会要开。」

祠堂檐角风铃轻响，响得像替里正锣补半声。韩泥担筐过，听见，不停——停，像等锤；不等，像怕锤。他都不选，只选活。

「何日？」有人问。

里正望天，天阴，像旱前兆：「明日巳时，祠堂。帖今日贴。」

里正扬声补一句：「日子定了。谁再辱叶、砸秤，先记名，再报县。」

里正贴帖，浆糊沾指，指红，像血却无害。村人围帖，念到「巳时」，有人拱手，像在拜日子——拜，也是怕。

韩泥不拜。他抹坛沿，沿冷，心热。

马春花瞥他，啐：「晦气翁，明日你也去！去听听寡妇怎么辩！」

韩泥「嗯」一声：「去。听真话。」

马春花一愣，想再尖，赵绣春拽她袖，低声却亮：「让他去！去了，坐实一对！」

韩泥不辩。辩，成嘴。嘴不如心。心里叶汤已深，深得够扛明日村会。

定了日，定的是锤——锤有期，赵绣春退半步，退得不甘，像牙咬空。

周嫂还想嚷，被里正一眼压住：「嚷到村会上嚷。今日散。」

散时，村人三三两两，嘴仍动，动的是「辨诬」二字——二字新，新得像门缝光，光小，却照见：叶家未必输，谣未必赢。

祠堂阶下，有人扯里正袖，低声：「村会，赖管事来不来？」

里正沉声：「来不来，村都要辨。辨不清，连坐先落在嘴上。」

叶青禾在侧巷等，没近，只隔三步。「韩翁。」

「嗯。」

「村会……」她停，「会连你。」

「该连。」韩泥独眼平，「汤是你送的。恩是我记的。连，正好辨。」

叶青禾唇动，想再说，他抬手止：「别谢。谢字轻。等辨完，我还。」

叶青禾低声：「赵绣春说，会前还要闹。闹到关铺。」

「闹。」韩泥道，「铺开着，闹空。空的是嘴，不是秤。」

叶青禾「嗯」一声，布鞋踩泥，轻，平。韩泥摸心口，叶汤那笔恩深，深得像村会的钉——钉未落，洞先凿好。

傍晚，韩泥担筐回村。筐结实，藤条勒肩，勒出印。路过告示桩，短签角又撕一分，撕不撕名。名在仇苗里，苗在，村会前还要立汤。

药铺外「哐」一声。秤杆被人砸中，杆弯，未断。

叶青禾扶杆，指白，声仍平：「砸断了，全村没秤。你们砸谁？」

巷口人影退，退得快，像怕里正记名。

韩泥隔巷听见，没近。近，成牵扯。他只低声道：「秤在，铺就在。砸秤的，村会上见。」

叶青禾把秤杆扶正，秤砣再落一声——落给全村听：铺还在，脊还直。

里正闻声赶来，举锣未敲，先问：「谁砸的？」

巷静。静里，赵绣春门内啐一声，啐碎，像心虚。

里正板脸：「再砸秤，先记名。村会日子定了，砸秤也是辱。」

叶青禾扶杆，指节白，声仍平：「秤杆弯，弯不断。断的是谣，不是铺。」

夜里，里正叩门，三下，沉。韩泥开门一尺：「里正。」

「村会要开。」里正嗓低，「你少往前冲。冲，害叶家。」

「我不冲。」韩泥道，「我站。站末席，等锤。」

里正顿一息：「锤落谁身上，未定。定的是，谣不能糊。」

「糊不了。」韩泥独眼平，「汤烫着，糊不了。」

里正「嗯」一声，步远。韩泥低声道：「日定了，理也定了。定的是辨诬，不是关铺。」

韩泥回屋编筐，藤条绕膝，绕得像把「辨诬」编进去——编进去，等那日，叶汤能立，芽儿能活，叶家能开门。

坛角微温不退，沿口水汽一线，像瓮在睡，却听见了「辨诬」二字。坛壁蒙尘里，极淡一道影游过，像半个符，像半个字，一闪，灭。灭得他以为眼花——眼花也罢，坛应了「辨诬」，应得他心定半分。

沈枯芽门缝塞一片姜，姜薄，薄得像娃也懂「村会」二字重。韩泥拾起，不尝——尝，像收谢；不收，只记。记在心里，娃的胆，也算一证。

夜里，叶青禾隔巷低声问：「韩翁，村会上……我怎么说？」

「说汤。」韩泥道，「热，送，该送。别辩奸，奸在谣嘴里。」

「你呢？」

「我认恩。」韩泥独眼平，「恩我担。担了，正好连。」

叶青禾「嗯」一声，步远，远里药铺灯仍亮——亮弱，像村会前的烛，烛不救嘴，救抓药的人。

赖扒皮短签在桩，风撕角，撕不撕名。名在仇苗里，苗在，村会前还要攒十文、还要立汤、还要让芽儿活。

韩泥擦坛沿三遍，沿净，心定。赵绣春过巷，瞥他一眼，想尖，尖到一半，见里正锣还在，咽回去。咽回去，也是钩：日子定了，赵党也要借锤，人心已定一半。

里正锣余音又荡一回，荡得短，短而清：「明日巳时，祠堂村会。」

韩泥收筐，藤条在手，手平。明日村会，叶汤得立——立住了，才走得向公审那日。

他合眼，手仍按坛沿。五十八岁，绝户，晦气翁，还在泥岗村西头这间最漏的屋里。明日巳时，祠堂见。

村狗叫一声，又哑。哑，像全村等日——日定了，钩更紧。韩泥不拜日，只拜手——手平，筐实，汤才能立。

丑时前，叶家药铺灯弱着，弱也是亮。亮着，叶汤就不倒。韩泥举目西望，药铺灯一线，像恩还在远处守着。

他低声道：「明日村会。叶汤得立。立住了，才走得向公审那日。」

韩泥摸坛，腹温不退，像瓮睡了，理醒了半分。沈家豁口咳一声，咳轻，像证：日子定了，芽儿娘还能喘，村会就能辨。

韩泥隔巷听见，不开门。开门，成牵扯；不开，只把藤条编实——实，才能撑到辨诬那日。叶青禾药铺灯，夜里仍亮。亮弱，像村会前的烛——烛不救全村嘴，救抓药的人。

里正又敲一下锣，锣声短，短而清，像给全村钉一颗钉：钉名「辨诬」，钉日「明日巳时」。

韩泥编筐，藤条绕膝，绕得像把「辨诬」编进去——编进去，等那日，叶汤能立，芽儿能活，叶家能开门。秤杆被砸，杆弯未断，铺还在，汤就不凉。沿口一线温，贴住虎口，脆，短，够撑明日村会。瓮未醒，他不喊醒；醒在缘，不在这一更。明日巳时，祠堂村会，叶汤得立给全村看。活路在，坛在。

坡下石阶湿，他数息，数帖，数债序。
