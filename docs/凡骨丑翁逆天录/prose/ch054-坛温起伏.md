# 第五十四章 坛温起伏

连雨一日，棚漏风。风进，坛席潮。

韩泥擦沿，沿滑，滑像祖父韩老栓的手。手不在，滑还在。坛腹午温，夜凉，晨又温一线——起伏，像瓮在呼吸。吸的不应人，吐的不应灵。他不催：催了，路断。潮气爬过席角，他只把席按严，按实了，温才不外泄。

邻床笑：「晦气翁拜坛，拜出邪脉。」

管事路过，声冷：「再嗓，滚去西栏清粪。」

邻床闭嘴。韩泥不辩，辱记心里，与恩分开摆。摆清了，心就不乱。雨歇，驿道泥浅。小商队过，两驼，铃轻。褐袍人起身半寸，又坐下，像等雨停后冬寒才够深。韩泥搬柴，不凑亭——凑了，像贪符。

老耿挑水，水洒实：「那人等冬。冬深了才赠。赠，不白受。」

「白不受。」韩泥应，「受了，记恩。」

「记就行。」老耿道，「别当着人盯着看。盯久了，赖福又要造谣。」

「不盯。」韩泥说。

午后藤运到丙九，藤硬，硬像冬规矩。韩泥领末捆，管事验韧：「韧，编得出筐。筐圆，手才不抖。」

同批少年偷懒，管事记名。韩泥不偷懒。刘婆端姜汤，姜薄，热：「雨潮，别空肚。」

他喝尽。赖福讽：「丑手编筐，筐也晦气。」

刘婆哼一声：「手正，筐就圆。圆了，能装柴，也能装恩。」

赖福脸青，退半步。藤刺扎掌，扎旧茧，茧裂，裂口潮——潮不是血。血不碰沿，沿才安。韩泥把裂口摁了摁，不哼。哼了像示弱；示弱像给人口实。

黄昏，铁无言廊下见他：「坛温起伏，别当醒。醒要等缘，不等人催。」

韩泥独眼平：「不催。催了，路断。」

「沿净？」

「净。」韩泥说，「血不碰沿，沿才接得住后头缘。」

同批少年碰他袖：「坛温起伏，你怕不怕醒？」

「不怕。」韩泥说，「怕也守。守着，沿才净。」

「醒了呢？」少年压低嗓，「醒了给你仙缘？」

「醒不醒由它。」韩泥道，「由我不由它的，是席盖严不严。」

夜雨又细。细敲棚顶，顶响像更鼓。他对坛低声：「温凉起伏，像你眠。眠着，不急醒。醒，要等血缘碰沿——碰，不是今日。」

他摸掌茧，茧裂，裂口无血。无血，沿便安。坛腹温一线，又凉。凉，像应「等」。外头更鼓三响，沉沉的。沉里，褐袍人影仍在驿亭——在着，像密宗线候冬，不急；急便邪。丑时前邻床梦呓：「……温……」

韩泥盖席，声低：「你看错了。睡。」

他把席按厚。雨里小商队铃轻，像界。界内末排，界外五脉。符还没到，怀里空着——空不要紧，褐袍人袖里黄纸边又露一线，露得像凡物将出。他绕远，避腥沾袖：袖腥洗得掉，心邪洗不掉。

雨歇后泥路仍滑。搬柴经侧门，他瞥驿亭一眼——亭里人仍坐，袖角黄纸毛边在风里微动。动，像凡符将出；不出急，急像邪。管事盯梢：「坛边别拜邪。拜邪，滚。」

韩泥不拜。守沿，缘在。同批少年问：「那人还不走？」

「不走。」韩泥说，「走了，冬未深。深了，才赠。」

管事验柴：「沉得住，记功在。」

刘婆端粥，稀，热：「雨停了，别空肚。」

韩泥喝尽，只点头。热在，像第七笔又实。霜白那日侧门仍窄，驼铃远，尘落。褐袍人仍坐，坐得像等冬寒再深半分——半分，像凡符出袖前的最后一试。他肩勒旧印，印深，像丙九五年磨出来的实。不凑亭，只记黄纸边毛，毛得像凡物。

铁无言廊角声低：「坛温起伏，别当醒。醒在缘，不在舌。」

韩泥只点头。点头，像把「等」字按进掌上。夜坛腹仍起伏像呼吸——呼吸不应人，不应灵。他低声：「你再眠。眠着，我守席，守沿，守不主动滴血。」

凉，像答应「等」。外头更鼓沉，柱上五日条还在。五日，试手在望——望不要紧，要紧是沿净、手正。「赠来仍凡，不越阶。」他独眼平，「越阶，像拜邪。」

雨又起那夜，棚顶响密，密像算盘落子。韩泥披旧袄，袄薄，薄能挡半分潮。潮进席角，他不挪席——挪了沿要再验。邻床问：「坛温像醒吗？」

「像眠。」韩泥说，「眠着，不急醒。醒，要等缘。」

五更将尽，天边白一线。白里驿亭灯灭，褐袍人影仍在。仍在，像冬寒还差半分才够赠。半分，他等得起。卯时管事验棚：「坛角杂物，收干净。」韩泥只擦沿。沿滑像祖父的手——手不在，滑还在，还在，像血脉日常。秦霜远廊经过，木鱼不敲，只托着。她停半息：「坛温起伏，别拜。拜了，像借腥。」

「不拜。」韩泥答。

午后西栏风硬，粪锹沉得像湿柴。沉不要紧，沿要净。管事验沿，烛照一线：「脏一指，像拜邪。」他用旧布再擦，擦得沿上无尘、无血。夜雨第三夜，邻床翻身碰他席角——席角潮，沿仍净。净，像六十日擦出来的规矩。老耿夜过侧门，桶沿轻碰栏边，声闷：「褐袍人候冬，冬深才动。动，是赠，不是邪。」韩泥不凑亭：凑像贪符，贪符像拜邪。

丑时前木牌贴胸，胸口吃住七笔。七笔分着，不混——混了，手就飘。驿道干了半寸，驼铃又轻。铃轻，亭里人仍坐，坐得像等霜再厚一层。厚了，黄纸才肯出袖。赖福路过侧门，嗓尖：「褐袍人还不走，专候丑杂役？」管事冷声：「再嗓，克半月麸皮。」赖福退，退得像吞了苍蝇。韩泥不接话——接话像邀功；功在沿净，不在舌。

管事验垛：「冬里沉得住，记功。」他不谢，只把柴摞齐——齐了，像给后头试手留一条正路。编筐末藤硬，他仍偏手，刺浅些。浅些，茧仍裂，裂口潮不是血：血要等活计误那一甩，甩在沿前，不在舌尖。铁无言夜巡嘱半句：「沿净，才接得住赠。」

「净。」韩泥应，「净了，才像守瓮人。」

同批少年蹭袖角，声小：「五日条上写试手，你怕不怕滚？」

「滚看手。」韩泥说，「手在，就不滚。不滚，才接得住后头凡符。」

少年似懂，收声走了。柱上五日条角起霜，霜化字更真——真在沿净，真在冬深将够。够深了，褐袍僧才肯离亭。雨歇了，轮换稳了，亭里人还在候冬。风硬一寸，硬像凡符出袖前的最后一试。

刘婆第二趟端姜汤，姜薄，热得钝：「雨潮，手别飘。飘了，试手滚。」

韩泥喝尽：「不空。手不飘，汤才还得热。」热在胃里，像第七笔在冬里又实一分。

连雨停透那晚，他把坛席掀起一角晾半息，又迅速盖回。晾不是松手，是赶潮。潮气重了，温凉起伏更容易被人看成「醒」。看成醒，谣就来；谣来了，管事的眼就尖。他不允许谣踩到坛沿上——沿上只能有尘、有滑，不能有热闹。

同棚少年缩着脖子问：「你天天擦，手不酸？」

「酸。」韩泥说，「酸也擦。擦了，心才安。」

「安了就能试手？」

「安了手才不抖。」韩泥道，「手不抖，试手才有门。」

侧门驼铃再轻一声，袖里黄纸角又翻半分，翻完贴回——像有人试这冬够不够冷。冷够了，凡符才肯落怀；落怀前，沿得净，手得正，人得睡着也别睡死。睡死了是拜邪；眠着，才是等。

等得起，窄路才还在。还在，褐袍人才肯离亭；离亭之时，他怀里不能是空的，心上也不能是邪的。坛温再起落一夜，他也不追问醒不醒——问起的人多，守沿的人少；他选少的那条。少的那条窄，窄才够真；真了，冬深一赠才接得住。接得住，试手那五日才不白挨。不白挨，窄路才值钱。
