# 第三十六章 丹方残页

莫长春从旧藏取出一页残纸，纸黄，边焦，像劫火里逃出来的命。

丹堂，苏念慈正在理药材，见他入，手一顿：「师兄？」

「给你。」他把残页递过去，「上古丹方残页。能不能破金丹，看丹堂，看你。」

苏念慈接过，指尖抖，抖得像捧的不是纸，是一炉命：「师兄……此物可破金丹。宗门谱上，标为禁品残卷……」

「禁品，」莫长春道，「禁的是乱用。你炼，是稳丹堂，稳青岚。」

她抬眼，目光里不止师姐看师兄，还有丹师看火，还有女人看一个把重礼推给她的人：「你……你又把重的推给我。」

「重的才还得起。」他笑，像说过千百遍的话，「轻的，才送得出去。」

苏念慈把残页贴胸，一瞬，像贴炉心：「我炼。」

「七夜。」莫长春道，「老夫送渣，你守火。渣是心意，不是废料。」

她点头，点头像立约。

廊外弟子窃语：「莫长老又把好东西塞丹堂？」「禁品残卷也敢给？」「苏首座敢接？」

苏念慈冷声：「闭嘴。开炉。」

火起，药香起，像一条线，从残页接到炉盖。弟子不敢噪，噪也被药香压住，像全宗都在等七夜后的那一声「成」。

柳青鸢遣人来问：「宗主问，残页可有谱？」

莫长春回：「有。在丹堂，不在嘴上。」

来人退。苏念慈听见，眸光微动，不谢，只把火拨旺，旺得像替宗主也担一分。

赵玄机路过丹堂，停半息，望门内火光，未讽，只拂袖走，走前低声对守门弟子：「执法堂轮值，闲人退三十步。」

弟子窃语：「赵执事替丹堂清场？」「还库之后，果然不一样。」

夜，莫长春立丹堂门外廊柱旁，不进门，像守门，也像陪炉。

第一夜，炉鸣细；第二夜，药烟苦。弟子轮值，顾小满送水，柳青鸢遣人送阵材，像全宗都知道丹堂七夜，不是苏一人之事。

苏念慈在门内扬声：「师兄，为何不进门？」

「进门，」莫长春道，「像抢你的戏。老夫在门口坐着，像等回赠。」

苏念慈一窒，继而笑，笑里倔：「回赠不是还债。」

「对。」他应，「所以老夫等。」

簿无新字，袖却暖，像残页的重量已进丹堂，进苏念慈掌心，进七夜后的那一颗金丹。

第三夜炸炉险象，莫长春门外递渣，渣稳火，稳得像心意不是废料。

魏无涯观丹堂火，低笑：「重的推给她，轻的才送得出。」

霍镇山闻讯，骂声洪：「丹堂七夜，谁噪谁滚！」骂声像鼓，鼓点催火，催得全宗都不敢噪。

顾小满送水，水温，温得像杂役也押七夜。她抬头看门外长老，长老也看她，目光一触，像说：记着，比跪有用。

第四夜药烟苦，第五夜炉鸣细，第六夜腥风一线——腥得像夜袭在提前嗅，嗅得阶下弟子怕，怕里仍不肯退。

杖点廊石，一声，像给第一夜敲鼓。阶下弟子屏息，屏息里，全宗都在等延迟的那一声「成」。

苏念慈在门内拨火，火舌舔炉壁，舔得像把禁品残卷炼成胆。她忽然懂：师兄不进门，不是不信她，是把成丹的戏，全留给她唱。

他低声，像钩：

「七夜。成不成，都在这一炉。成之前，谁也别替丹堂喊体统。」

大因果还在远处，像天试，像赤焰——残页只是小赠，丹堂大炼才是大还礼的前奏。

杖点廊石，一声，像给第一夜敲鼓。阶下弟子屏息，屏息里，全宗都在等延迟的那一声「成」。

莫长春立门外，像七夜戏台，台口坐着，不抢戏，只等回赠。他低声，像钩：「七夜。成不成，都在这一炉。成之前，谁也别替丹堂喊体统。」

第一夜，炉鸣细；第二夜，药烟苦。弟子轮值，顾小满送水，柳青鸢遣人送阵材，像全宗都知道丹堂七夜，不是苏一人之事。苏念慈在门内扬声：「师兄，为何不进门？」

「进门，」莫长春道，「像抢你的戏。老夫在门口坐着，像等回赠。」苏念慈一窒，继而笑，笑里倔。

围观弟子在丹堂阶下踮脚，踮得像等成丹那一声——声未出，先出的是药香，香压过噪。有人低声：「禁品残卷也敢给？」另一人答：「给的是戏，不是丹。」

第三夜炸炉险象，莫长春门外递渣，渣稳火，稳得像心意不是废料。霍镇山闻讯，骂声洪：「丹堂七夜，谁噪谁滚！」第四夜药烟苦，第五夜炉鸣细，第六夜腥风一线——腥得像夜袭在提前嗅。

杖点廊石，一声，像给第一夜敲鼓。阶下弟子屏息，全宗都在等延迟的那一声「成」。苏念慈在门内拨火，忽然懂：师兄不进门，是把成丹的戏，全留给她唱。
