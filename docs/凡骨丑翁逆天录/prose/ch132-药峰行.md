# 第一百三十二章 药峰行

药峰晨雾未散，雾里药香硬。

硬不要紧，硬才像峰。峰高，活苦；苦，才像外门积势，不像一部爽完就散。

管事弟子分活：「丙九韩泥，东垄分拣，午前清栏，午后编筐。」

韩泥只答：「在。」在，就不滚。

东垄渣灰，灰里正。正不要紧，正像嗅诀回响——回响在鼻下，不在舌。

同批少年掩鼻：「你夺魁了还分拣？」

「分拣。」韩泥声平，「魁在牌，活在手。手不停，牌才不虚。」

少年退半步，没再问。问，像沾晦气。

刘婆送线过，线粗，声低：「药峰活重，别空肚。」

「不空。」韩泥说，「空肚，手飘。」

刘婆哼，没再多话。话在130那笔还，还了，线才送得心安。

铁无言挑水路过，水洒稳：「药峰别拜邪。拜邪，路断。」

韩泥只答：「不拜。坛在，人在。」

午后编筐半刻，藤刺扎掌，刺浅，浅些沿才净。筐勒匀，匀像火路——勒不匀，渣散，路断。

赖福秤筐，秤杆偏：「仍少二斤。」

「看错了。」韩泥说，「渣在筐底，底沉。」

赖福眉一蹙，没再称。称，像找茬；找茬不要紧，底沉，秤平。

日暮收工，他回丙九，不近廊下笑。笑在耳，不在心——心在坛，坛腹温一线。

丑时，练鼻下静。粟壳香混药香，药香正，正不要紧；要紧是静一字不落。

管事夜巡记簿，记「丙九韩泥，分拣清栏」，记细，是小胜。胜在忍，不在辩。

他望药山半腰，腰上雾散，散出一线白——白像130还刘婆那晨。晨远，路在；在，就不绝。

邻床悄问：「你天天这么活，不累？」

「累。」韩泥说，「累，才像活。活，才配帖在怀。」

更鼓沉，他按木牌，牌凉贴胸。胸贴紧，像贴债序——序里写药峰，写坊市，写大比。

他低声对坛：「药峰过了一日。下一回，挂牌。」

坛不应字，只温。温像应「分拣」——分拣净，嗅才准；准了，丑丹才配试。

药山半腰，雾散一线，线白，白像130那晨。晨里他分拣半刻，指稳，稳像捧不洒的热汤。

管事弟子再验袋，袋灰，灰里白细，细像药。药不要紧，管事不验名——名在远，在大比。

坡下石阶湿，湿里他数息，数债序。债序在，步不飘；步不飘，才接得住133挂牌。

赖福余党掷石过垄，石落筐边，筐未翻。翻才要紧——未翻，手稳，活像常杂役。

沈枯芽再递口信，信仍无字，只有一片枯叶——叶黄，叶上无字，字在记里。

更鼓沉，他望药峰，独眼平：「药峰过了一日。记着。」记着，是丑翁本事。

邻床问：「你天天分拣，魁名不虚？」「不虚。」韩泥说，「魁在牌，活在手。」

午后清栏，栏臭，臭不要紧。臭，药山活；活有味，味在角，不在席。

更鼓沉，韩泥按木牌，牌凉贴胸。胸贴紧，像贴药峰这一日。

同批少年掩鼻过，问：「你咋天天上山？」韩泥只答：「倒渣。」

倒渣里藏分拣，分拣里藏嗅——嗅在鼻，不在舌。

他望药峰雾，雾散，散出一线路——路通向133坊市，通向帖，通向大比。

药峰东垄，垄窄，窄里藏真——真在渣，真在嗅，真在外门积势那口气。气要匀，匀了，手才不飘；飘了，133挂牌就虚。韩泥分拣半刻，指稳，稳像130捧丹——捧得稳，才还刘婆；还得稳，才配帖在怀。同批少年笑他「魁名捡渣」，笑在舌，不在手；手不停，笑就远。

远不要紧，远，才听得全——听得全药峰管事夜巡脚步，听得全赖福余党使坏，听得全142在远。142在远，132只分拣；分拣净，嗅才准；准了，老耿那笔才谢得出。谢在142，不在132；132只积势，不越界。刘婆送线，线粗，声低，声里带130还恩的余温——温在，线才送得心安。

铁无言挑水，水洒稳，稳像108后仍来134修栏那半步。半步续，续在132还不显；不显，才像兄弟线。线不断，580从绶在远；远不要紧，今在药峰，今在分拣，今在编筐。日暮收工，他回丙九，不近廊下笑；笑在耳，不在心——心在坛，坛腹温一线，像应「药峰」二字。二字落，132过了，133在前。

散队后，他编筐半刻，藤刺扎掌，刺浅些。编筐钱买米，米在，恩才还得上。

药渣场角，风硬，他摊渣半刻，渣灰，灰里正。正不要紧，正像132回响——回响在掌，不在舌。

日暮编筐半刻，藤刺扎掌，刺浅，浅些沿才净。沿净，坛沿才藏得住。更鼓沉，他望药峰：「药峰过了。记着。」

胸贴紧，像贴药峰这一日——日苦，日真；真了，133挂牌才不虚。虚了，风就硬；硬不要紧，手稳，风就改一半。

侧门驿亭风硬，他听风，风过，记在心里。

散队后，他过外堂阶下，分拣半刻，指稳——指稳，路就不飘。

编筐半刻，藤刺扎掌，刺浅，浅些沿才净。沿净，坛才藏得住。

粟壳香淡，香一字不落。

管事记簿，记细，是小胜。

他望坛，低声道：「手在，路在。」坛不应字，只温。

更鼓沉，天边白一线。白里，他按木牌，牌凉贴胸。

坊市尾巷风硬，他收摊如石，石在，就不滚。

验息后，他理怀内册页，页黄，字丑，丑而真。

铁无言声低：「别飘。」韩泥应声：「不飘。」

刘婆端粥，仍多半勺。韩泥点头，谢轻，恩重。

药峰分拣照旧，指稳，鼻下静一息。

帖在怀，怀沉，沉的是序，不是惧。

白织月袖风过，韩泥不辩。辩，费舌；舌要留来守坛。

赖福廊下过，韩泥只答：「在。」在，就不滚。

柱上条角白一线，三年约在怀。怀凉，掌温。

坡下石阶湿，他数息，数帖，数债序。

同棚杂役笑，笑在耳，不在心。

废渣巷日斜，他验渣，渣正，手才静。
