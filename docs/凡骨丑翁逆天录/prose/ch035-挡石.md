# 第三十五章 挡石

入门第五日，山门外驿亭又见人影。韩泥领麸皮过侧门，袋沉，沉得像赖福的牙。牙咬得紧，紧在克扣，克扣在，路还在。驿亭下，布鞋稳，蓝布袄补过——叶青禾。韩泥脚停，停得像钉。钉在宗门地，也钉在五十八年。五十八年，她送汤不进门；今日，她追到山门外，仍不闯侧门，只站亭下。「叶丫头。」他嗓哑。「嗯。」她声平，「我爹药，你收全了？」

「收全了。」

「别死半路。」她顿一息，「死了，汤白送。」

韩泥「嗯」一声：「汤是热的，我记着。」

就三句。不多，稳，稳得像药铺秤。亭外坡道，马蹄乱。赵绣春带周嫂、马春花，从泥岗一路追到驿——追的不是韩泥，是叶。叶敢送，她们敢追，追成最后一辱。赵绣春嗓尖：「叶青禾！追到宗门门口送汉子！你还要脸不！」

叶青禾看她，不躲：「送药，不送人。人已在宗门，药是爹的。」

「送药送情！全村谣都听着——」

马春花拾道旁石子，石卵大，扬手便掷，掷向韩泥，掷得像要砸碎「晦气翁进宗」这最后一丝活路。叶青禾侧身，一步，挡在飞石前。石落她肩，闷响，响得短，响得韩泥独眼一紧。血不出，衣裂一线，裂在袄肩，裂得像门缝那日递汤——汤没洒，石却落了。落她身上，不落他脸上。山门侧门，杂役静了一瞬。赖福想笑，笑不出。管事沉脸：「驿亭闹事，记名！」

赵绣春见管事，嗓收半寸，仍尖：「她自找！寡妇追男人——」

叶青禾捂肩，声仍不高：「药是叶振东的。情是恩。恩不脏，脏的是嘴。」

韩泥放下麸皮袋，独眼抬，抬向叶肩，又落赵绣春，不落泪，不落怒，落一句：

「这恩，必还。」

四字，平，像钉，钉进五十八年，钉进丙九杂役，钉进省亲那日之前所有忍。叶青禾看他，眼不媚，不怯，像看一碗还温着的汤：「我不图你还。图你活着。」

「活着，就还。」韩泥说，「等我能炼丹了，先还叶丫头。」

叶青禾唇动，想再说，他抬手止——止得像公审那日：「别谢。谢字轻。」
赵绣春见势不妙，拽周嫂走，马春花啐远，啐像败，败得像公审余韵——余韵在，赵仍不服，服不服，石已落，落的是叶肩，立的是韩泥誓。管事记赵名，记完，对叶青禾：「叶氏，伤不重，去医。再闹，报县。」

叶青禾「嗯」一声，退半步，半步像公审后第一次近，近而不缠：「药在袋。肩在痛。痛换你一句实誓，值。」

韩泥不接「值」字，值在心里。心里，叶汤那笔恩，从深变成钉——钉名「必还」，钉死，不混辱，不混仇。管事对韩泥：「你，搬麸皮回房。别杵亭下惹眼。」

韩泥应声，袋沉，沉得像誓。叶青禾退至亭外十步，仍不闯侧门，只扬声：「药袋在栏边石下。别忘。」

晌午搬麸皮回栏，他在栏边石下揭药袋。袋口绳松，药皮干，仍带叶振东药铺那股苦辛。苦辛入口鼻，像辞村那篓药债又压回肩上——债要还，先还肩上一石，再还袋中药。

「不忘。」韩泥说，「药是热的，肩也是热的。」

叶青禾声仍不高：「追药，不追人。人已在册，药是爹的。」

骂声远，远像泥岗谣——谣在，腿在宗门，谣追不上侧门。回丙九房，夜。韩泥摸坛沿，沿烫一线，烫得像应「必还」。坛壁水汽爬，爬半寸，蒙尘里符影半闪，像15章那日，像公审前夜，像爹遗言在底。他低声道，深：

「必还。先还汤。先还肩上一石。」

肩上一石，不是血债，是情债。情债用丹还，用命还，用省亲那日还，用一生还。邻床少年听半句，问：「你跟那寡妇……」

韩泥独眼平：「恩。不是奸。奸在嘴，恩在心。」

少年闭嘴，闭嘴，像山门侧门那刻——刻进丙九，刻进丑杂役，刻进凡骨试录。外头更鼓三响，响得像给「必还」盖章。章盖完，瓮未醒，人心先醒半分——醒半分，够活，够忍，够走到瓮醒。韩泥按心口，心沉，沉得像恩从深变成钉。他低声道：「肩上一石，将来用丹还，用命还。还，先还热的。」

坛壁水汽爬，符影半闪，闪得像公审前夜那光——光应誓，不应名。名未醒，誓先实。刘婆夜里送粥，粥稀，热：「喝了。别空肚立誓，空肚誓轻。」

韩泥喝尽，不谢。誓不轻，粥也不轻——轻的是谢字，重的是还。邻床少年又问：「必还，还什么？」

「还汤。」韩泥说，「还肩上一石。还叶丫头爹的药。」

「你还不会炼丹。」

「不会，学。」韩泥独眼平，「学一日，记一日。记满了，就还。」

少年闭嘴，像被「记」字堵住。外头更鼓四响，响沉，沉得像省亲那日在远等——等活，等忍，等醒。管事夜巡过，敲柱：「驿亭闹事，记过了。别再生事。」

韩泥只答：「不生。生事，饭少。」

管事瞥他一眼，走了。走，像放一条窄路——窄路也是路，路通向还。誓在，瓮在。刘婆门外咳：「肩上一石，别扛在肘上。肘漏风，粥白送。」

韩泥应：「不扛肘。扛心。心在，誓不轻。」

管事门外又贴条，条压柱边，字利：「入门满旬，凡骨测骨。五日后卯时，丙九末排到。」

五日测骨——比还汤远，比滚近。近，手就不能抖。他低声道：「五日，学一日，记一日。记满了，才还得上热的。」

未时杂役分班，有人嚼「寡妇挡石」，嚼得短，短不过管事记名那一笔。韩泥铲下仍稳——稳，像告诉整条驿道：石落了，誓更实；誓实，脸不脏。坛壁一线水汽，应「誓」，不应名。

韩泥按木牌，牌薄，真在记名。五日测骨帖在柱，誓落心口——还先用丹，还先烫；烫在叶汤，石在她肩，誓在他心，三者分列不混，混则还乱。

丙九房里有人梦呓骂飞石，韩泥不辩。辩费舌，舌要留来记誓，留来活，留来还。坛壁水汽应誓，符影一闪，敛。敛像证：不求名，求还；还，先还烫。

天将亮时，叶肩那一线裂仍疼在眼里——疼在心里，不在嘴上。嘴上轻，心重；心重，才配得上叶丫头那句「我不图你还」。图不图，他都要还；还，先活五日，再活冬后，再活到能炼丹。

卯时前，他在柱下把木牌又按实一遍，按得像把「必还」钉进掌纹。掌纹丑，丑得真；真掌，才端得住测骨帖。帖在，脸不脏；脸脏在赵家嘴，不在他肩——肩疼在心里，疼得他更记得叶丫头那一句「我不图你还」。

搬麸皮过侧门，袋沉，他肩不塌，石印那一线裂在袄里疼着，疼而不喊。喊像少年，他是记心翁。刘婆远处瞥一眼，没多问，只把粥勺在灶边顿了顿——顿得像说：肩疼不要紧，肘别漏风。韩泥点头，点得像盖誓的印。印盖了，五日测骨才站得住；站得住，才学得到还肩上一石的那味丹。

驿亭柱下，测骨帖在风里不动。不动，像钉；钉住了，必还誓就不飘。飘了，恩断，汤就凉。凉不得。丑时前，他早起清粪，铲下稳。栏边药袋仍干，苦辛在鼻，像敦促：先活五日，再还肩上一石。五日测骨在卯时，卯时前，手不能抖；抖了，恩断，汤就凉，必还誓就白立。坛壁水汽应「五日」，符影半闪，敛。

名未醒，誓先实；实了，才撑得到学丹那日。肩上一石在叶，石在她肩，誓在他心，三者分列不混，混则还乱。五日测骨在柱，他按木牌，只道：先活五日，再还肩上一石。
