# 第二章 瓮眠

雨停在后半夜。

韩泥天不亮就出门。筐没编完，米缸已见底，他得上后山采药——苦蒿、蛇藤、能换铜板的都要。泥岗村后山土薄，药贱，可贱也比空手强。

他一只眼睛好，一只蒙着层白翳，看远无碍，辨药足够。村里人说他「独眼晦气」，韩泥当听不见。独眼是幼时染病留下的，治不好，也不连累谁，连累自己的，只有这张脸和这条跛腿。
山路滑，露水重。

韩泥深一脚浅一脚走，麻脸被草叶划了两道，也不擦。他认得哪片坡长苦蒿，哪处石缝藏蛇藤，五十八年，靠这些活。活得不体面，却活得久——村里爱嚼舌根的，好几个比他年轻，先入了土。

他蹲下身，用竹铲起一株苦蒿，根须带泥，气味冲鼻。苦蒿能换两个铜板，两个铜板能买一把粗盐。韩泥把药放进篓里，篓绳勒肩，勒出旧印。

坛留在屋里。

出门前他照惯例擦了坛沿。这坛在韩家屋角，祖辈在时就在，祖父在时也在，爹在时也在，传到他，仍是一只蒙尘的腌菜坛。坛旧，满身泥垢，釉色黯淡，像腌菜用完就搁在角上的凡物。村里人笑：「绝户翁连坛都蒙尘，命也灰。」韩泥不辩，擦惯了，像擦一件家当。穷人家，坛也是家当——爹说过，坛在，根在；祖父说过，那是祖辈传下来的根。

日头爬上来时，药篓已半满。

石缝深处，他铲到一株老蛇藤，根粗须长，味冲。老蛇藤价高，比苦蒿多换四文——四文，够买一把盐。

韩泥把藤缠好，篓沉了一截。沉，踏实。

他坐在石上歇脚，啃了口干饼。远处村烟直。手心忽然一热，热得莫名——像韩家血脉里有什么，隔着蒙尘的坛，应了一下。

他搓了搓手，热气散了，只剩茧。不知缘由，便不当回事。

后山风紧，吹得草伏倒一片。韩泥独眼眯了眯，看见远处有野兔窜过，不追。追了也不够一锅，追了还费腿。他歇够了，背起篓，继续下山。
回村已是午后。

泥屋比早晨更潮。韩泥第一件事去看坛。

腌菜坛仍在角落，封口的破布还在，坛里空，坛壁蒙尘，内息尽敛，瞧不出半点仙气——本该如此，腌菜坛能有什么仙气？韩泥伸手一摸，微凉，沉，像一块死陶。他缩回手，没声张。

村里若知晦气翁屋里有古怪，明日唾沫就能淹死他。韩泥不怕死，怕麻烦。麻烦来了，筐编不成，药卖不出，活路就断。

夜里，韩泥就着油灯坐坛边。

坛不语，人不问。他只低声道：「人活着，恩辱得记着。记在心里，不乱。」

他不会写字，也不懂滴血认主那套——老农民哪懂这些？懂的是欠了要还，辱了要忍，仇了要候。候在心上，比写在纸上实在。

他编筐，藤条在膝上绕，绕得指节红。油灯焰抖，抖得像命还薄。编一半，停，独眼瞥坛角——坛仍沉，仍蒙尘，像腌菜坛该有的样。

「爹。」他低声，「坛在，根在不在，我说不清。」

坛不应。不应好，应了才吓人。

他又绕半圈，藤条勒紧，勒出闷响：「人活着，恩辱得记着。记在心里，不乱。」

这话说过，再说一遍，像把绳编实。
丑时。

村狗不叫，鸡未鸣。泥岗村静得像一口扣着的瓮。韩泥没睡实，迷迷糊糊，掌心又热。

他摸向坛沿。

坛壁竟比白日温了一线，温得极轻，像有人在里面翻身，又立刻睡回去。韩泥睁眼，屋黑，油灯灭，坛沿一线温光，一闪，又没了。

坛壁蒙尘里，凝出一丝水汽，沿水汽爬半寸，爬得像梦里朱红那一线，爬完即敛。

瓮没醒。

韩泥知道没醒——若醒了，村里不会还叫他晦气翁。可坛比昨多了一丝活气，活气藏在蒙尘里，像至宝沉睡，不肯露脸。

他坐起，按了按心口。外头风紧了一下，茅草屋顶沙沙响，像有人在瓦上踩，踩了一下就走。韩泥没追，只等。打棺人听过太多「瓦上脚步」，十回有九回是风，剩一回是贼，贼也不偷他这间漏屋。

等不来人，等来更漏的雨意。

天边泛白时，他出门前又看了坛一眼。坛仍蒙尘，仍像凡物，仍沉在角上。沉得好——至宝不露相，露相惹祸。

今日还要去集上卖药，药贱也卖，卖一文是一文。

集尾，老蛇藤先出手，四文。苦蒿六株，十二文。合计十六文，比往日多一倍。买药的问：「晦气翁，今日走运？」

韩泥摇头：「手稳，药老。」

买主笑，笑短，却真：「实诚。」

另一买主凑近，掂蛇藤根，鼻嗅：「味冲。你这独眼，辨药倒准。」

「眼一只好，够用。」韩泥说，「药贱，人不贱。」

买主一愣，多摸两文：「拿去买盐。别饿着手。」

韩泥收铜钱，布沉。十六文，沉，像活路又宽一指——辱在村里，钱在掌心，不混。多两文他没收，推回去：「说好了十六，不少赚，不多欠。」

买主更愣，终是收回，却道：「下回还来。」

「还来，药还老。」韩泥说。

出村时，周嫂在井台边洗衣，见他背篓，欲言又止：「……今日多？」

韩泥「嗯」一声：「药老。」

周嫂棒槌悬着，终是没砸：「少沾晦气。」

韩泥不辩，绕远半步。半步够活，够采药，够把恩辱养在心里，等第一笔汤落肚，好记。

回屋，他把十六文数了三遍，三遍数得一样，心里才踏实。踏实了，才配睡。睡浅，掌心又热一线，热得像坛在远处应——应一文是一文，不应多。

他低声道：「十六文，活路宽一指。宽一指，够。恩辱记着，不乱。」

坛角仍沉，沉得像凡物。凡物好，露相惹祸。

门外张麻婆骂：「老东西，数钱数到天亮？」

韩泥「嗯」一声：「数三遍，三遍一样，心里才踏实。」

张麻婆哼：「踏实了就睡。明儿罐子还得卖药，别数没了手。」

韩泥：「手在，药在。数不在碗里，在掌心。」

丑时前，韩泥睡浅。梦里爹的手搭坛沿，铁山嘱：「坛别扔。韩家就剩你，坛在，根在。」醒时掌心干，心口却硌——硌像欠字，欠字没形，有重。他摸坛壁，壁凉转温，温极细，细得像地底爬上来的一线活。瓮未醒，他不喊醒。喊醒要血，血要还，还不了就莫惊它。采药归来，十六文落袋。钱少，够买粗盐，盐能腌，腌能存，存到开春。开春还远，远也要活。活法是擦沿、编筐、不辩——辩无用，记有用。外头风硬，吹坛角布包挪半寸。布包是爹留的，旧，塞不紧。塞不紧不怕，怕的是沿口蒙尘。尘在，像命灰；擦过，灰下还有韩家。丑时未到，坛壁先温。温像应步，不应名。名未醒，步先稳——稳到明日还能去庙前扫叶，换一顿斋饭。斋饭不暖，碗在；碗在，人就在泥岗村西头最漏的屋里。采药时他避深林，避不是怯，是命薄——命薄的人，深林里死得无声。无声的死，还不了恩。还不了恩，爹那句「坛别扔」就白嘱。十六文他买粗盐，盐粒在指间，像数日子。日子数到丑时，坛壁又温一线。温一线，够记：瓮眠，未醒；未醒，莫惊；惊了，沿上血将来还不起。丑时里他数呼吸，数到三，坛壁温一线。温一线，像爹嘱还在沿上——沿在，韩家不散，散不了，就还得走还路。

丑时将至，坛壁微温如息；瓮未醒，他不喊醒，只记擦沿是家法——家法在，泥岗根在，根在才配梦见爹嘱「坛别扔」，别扔，才还有路。
