# 第五十六章 编筐季末

试手前二日，丙九编筐季末。韩泥接末单，藤条硬，硬得像冬里的规矩。管事盯梢：「末单交不出，试手名单划掉。」韩泥不辩，只编。编筐是本行，深秋末筐也是本行。筐成，手稳，稳比美重要。藤刺扎掌，扎旧茧，茧裂，裂口潮，潮不是血。血不碰沿，沿才净。赖福讽：「丑手编筐，筐也晦气。」刘婆路过，端粥，稀，热：「手稳，筐就圆。圆，能装柴，能装恩。」韩泥喝粥，「嗯」一声。热在，像第七笔又实。筐沿齐，齐像秤杆落盘。管事验筐，指筐底：「底不塌，记功。」记功，像窄路又宽半分。---

午后，符怀暖一丝，护骨寒，手不僵。僵，藤刺偏。不偏，缘才正。搬柴过半，肩稳，像筐成后的余韵。余韵不在夸，在手还在。铁无言路过，声低：「季末了，手还在。在，才接后头几日。」韩泥只答：「在。」「试手验啥？」铁无言问。「验手。」韩泥说，「手抖，丹歪，滚。丹歪不要紧，手不能歪。」铁无言「嗯」一声：「静带到手上。手上静，丹才正。」同批少年碰他筐：「末单也编得齐？」「齐不齐，管事验。」韩泥说，「我验手。手稳，筐就圆。」少年碰筐沿，沿不塌：「晦气筐，倒结实。」韩泥不辩。结实，像小胜。胜在实，不在嘴。黄昏，坡下风硬，柱上条在：「后日卯时，试手凝丹。」后日，比五日条又近一步。近不要紧，要紧是手还在。管事验末捆，声沉：「季末了，藤刺还会扎。扎了，别硬扛。扛裂，手抖。」韩泥「嗯」一声：「不硬扛。扛，恩断。」赖福偷懒，管事记名。记名，像秤上空一格。韩泥不偷懒。偷懒，麸皮克。克粮，克不走还路。末藤刺深时，韩泥停半息，停，像听藤条说话。落盘稳，谣压半寸。管事验筐底：「底不塌，装得住柴，装得住药。」韩泥不邀功。功在心里，不在舌。舌轻，恩重。赖福讽筐晦气，讽完，筐仍被领走。领走，像小胜，胜在实，不在嘴。夜，坛腹温一线，像应筐成。成，不急醒。醒在缘，不在筐。他低声道：「季末了，手还在。在，才接血近沿那几日。近，不碰沿。」坛不应字，只温，温得像「嗯」。外头更鼓沉，沉里，试手条在柱，柱上「后日卯时」。后日，凝丹试手。试手不是醒瓮。醒瓮在更后，在误触那一滴。邻床问：「你怕不怕滚？」「怕。」韩泥说，「怕也记。记着，手就不飘。」「飘了咋办？」「飘了，恩断。」韩泥说，「恩断，汤就凉。」邻床懂「凉」，闭嘴。他是记心翁，恩仇都记在心里。筐是活计，活计扎掌，掌裂在冬，不在沿。丑时前，裂口渗潮，潮不是血。无血，沿便安。他擦沿，沿滑，沿净。净，像祖父手还在。还在，像守瓮血脉的日常。刘婆门外咳：「后日试手，别空肚。」韩泥应：「不空。粥在，姜在记里。」符怀暖一丝，护手不僵。僵，后日藤刺便偏。他低声对坛：「筐成了，手还在。还在，才接后头试手，接血近沿，接瓮醒。」坛腹又温一线，温短，像瓮听筐成，仍眠。外头风息，柱上「后日卯时」更真。真在试手，真在末排，真在窄路。窄路也是路。路通向凝丹，凝丹通向还。还叶汤，还振东药。还，先还烫。秦霜远廊经过，一句：「手别抖。抖，丹歪，滚。」韩泥「嗯」一声，独眼平。平，像辨香那日。坛壁水汽爬半寸，敛。敛着，像替他守着。手别飘，恩不断。第二日，末单仍编。编，手在，缘才正。管事验筐沿：「沿齐，底不塌，记功。」铁无言路过，声低：「后日试手，静带到手上。」韩泥「嗯」一声：「带上。带上，丹才正。」刘婆端粥，稀，热：「后日试手，别空肚。」「不空。」韩泥喝尽，「嗯」一声。邻床问：「编筐也试手？」「编筐练手。」韩泥说，「手稳，筐圆。筐圆，丹才不歪。」午后搬柴，肩稳，符怀暖，掌裂不僵。管事盯梢，盯赖福偷懒。偷懒，记名。记名，秤上空一格。坡下风硬，柱上「后日卯时」在。在，像窄路也近了半分。还在，才接后头试手。」第三日，末藤仍编。管事验筐：「沿齐，底不塌，记功。」肩稳，像七笔压着。铁无言路过，声低：「后日试手，仍末。误者，滚。」韩泥「嗯」一声：「仍末。末不要紧，手要紧。」同批少年碰他筐：「后日你真去？」「去。」韩泥说，「去，验手，不验灵。」刘婆端姜汤，姜薄，热：「后日试手，别傲。」「不傲。」韩泥说，「傲，符自灭。」末藤刺深时，他停半息，停，像听藤条说话。裂口渗潮，潮不是血。还在，才接后头试手，接血近沿。」管事又嘱：「后日试手，仍搬柴。搬柴手抖，试手也滚。」韩泥「嗯」一声，肩稳。稳，像辨香那日站出来的。同批少年问：「编筐也验手？」铁无言廊角，声低：「沿净，别拜邪。拜邪，恩断。」韩泥「嗯」一声：「不拜。守沿，等缘。」刘婆又端半勺粥，热：「抖的是心，不是手。心记着，手就稳。半勺，热在，像第七笔粥又实一分。坡下霜白，柱上「后日卯时」在。在，像窄路也亮了半分。他摸木牌，与试手帖并放。并放，像命与规矩挨着。挨着，才验得过。第四日，末单收尾。收尾，手还在。管事验末捆：「季末了，藤刺还会扎。」编筐补刺，藤刺浅些。浅些，裂口不深太快。」他低声对坛：「季末了，手还在。在，才接后头试手。」第五日清晨，筐末扎紧。扎紧，像秤杆落盘。管事验筐沿：「沿齐，底不塌，季末记功。」同批少年碰筐：「末单也齐？」搬柴过半，肩稳。符怀暖一丝，掌裂不僵。铁无言路过：「后日试手，静带到手上。」刘婆端粥，稀，热：「后日别空肚。空肚，手飘。」坛腹又温一线，温短，仍眠。」肩稳。季末第五日，筐末扎紧，扎紧像秤杆落盘，落盘手还在。管事验筐底：「底不塌，季末记功。藤硬，手更稳，稳比快重要。筐沿齐，齐像秤杆落盘——落盘稳，谣压半寸。管事验筐，指筐底：「底不塌，装得住柴，装得住药。」韩泥「嗯」一声，不邀功。功在簿，不在舌；舌轻，恩重。领走，像小胜——胜在实，不在嘴。午后搬柴，肩稳，符怀暖，掌裂不僵。僵，血线便偏；不偏，沿才净。」韩泥不答「几日」，只答：「在。」夜，坛腹温一线，像应筐成。成，不急醒——醒在缘，不在筐。
