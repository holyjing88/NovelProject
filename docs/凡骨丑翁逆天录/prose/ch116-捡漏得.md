# 第一百一十六章 捡漏得

尾摊换来的残页贴在窑壁上，字碎得可怜，可「焚火」二字还认得。韩泥按页行息，息走三遍，窑温往上顶——顶得像炉心没冷透。

冷不要紧。温才要紧。温稳了，渣里那点香才正；香正了，培元散才配再炼。再炼不是图看，是图复测；复测连着三年约那句「末席滚」——滚要手抖，他不抖。

他把铁片嵌进窑口。嵌稳了，像捡回一截炉角。炉角露不得；露了，捡漏就成「盗器」。盗器除名，除名一出，叶汤那笔恩还不上，芽儿那半块灵石也悬着。

赖福公示第三日，仍让他吃饭最后。粥薄得能照影，刘婆勺子沉，照样舀满：「吃。空肚手飘，飘了窑就砸。」

「嗯。」韩泥吃尽，不辩。辩费舌头；舌头得留着按诀、守温。

邻座有人小声：「他那页从哪来的？坊市？」

旁人立刻扯袖：「少问。问多，赖管事记你名。」

韩泥当没听见。听见了也不接——接了像认赃。他把碗底刮净，起身走窑道，步子稳得像粪铲落栏。

午后他用铁片剔渣，剔出一粒未化灵砂。砂细得像漏网鱼——鱼小，够掺一炉。炉不必多，多惹眼；一炉够验手，够给复测垫底。

白织月过药山道，袖风带香，笑得薄：「丑杂役也玩火？」

韩泥按着铁片，不抬头：「火在窑，不在嘴。」

「玩火玩死。」她走近半步，「末等也配焚火？」

「配不配，温知道。」韩泥声平，「温稳，散正。正了，你鼻子闻得出。」

她唇线一绷，袖风一扫走了。走了也好——香一飘，赖福就有借口说「偷火诀」。

傍晚铁无言来，掷一小袋边角渣：「外门剩的。你用。」

「谢。」韩泥只一字。

「谢轻。」铁无言道，「手稳，别漏。漏了，赖福那笑就成真。」

「不漏。」韩泥应，「掺砂炼。炼完藏怀，不示人。」

他把灵砂掺进渣灰。灰里一点辛香升起时，鼻下静得像兽潮那夜——那夜并肩扛，今夜独炼。独炼不怕，怕丹不正。

正了就藏。示人等于送把柄：赖福克，白织月嘲。嘲不要紧，试台见；见了，「运气」二字尖不过「正」。

炉心没有，窑温代炉。代炉，是丑翁的路——路在渣，路在捡，路在五年一点点淘。淘出来了，才配冲九层，才配先还芽儿那半块石。

赖福带人闯窑道，嗓尖得刺耳：「偷诀偷火，在这！」

管事弟子跟来，瞥一眼残页，页碎得可怜：「坊市散页，非禁物。韩泥，炼什么？」

「培元散试手。」韩泥把掌张开给人看香气，「渣里炼，不耗宗门灵材。」

管事凑近嗅了嗅，香淡，却正：「正，记一笔。赖管事，捡漏非盗。」

赖福脸青转阴：「正也晦气。晦气，明日除名试炼——手抖就滚！」

「试炼看手。」韩泥抬独眼，「手在，就不滚。」

那两个字落下去，像把钩子扔向坡下石坪。石坪在明日，八层在腹里；腹里温一线，坛应着「捡」——捡来的铁，捡来的诀，捡来的命。命得先忍住，忍住才还得上恩。

邻床少年夜里凑过来，小声：「你真炼成了？」

「粒在怀，不在嘴。」韩泥道，「你闭嘴睡。睡实了，才不像喊告示。」

「我……我只是好奇。」少年嗓子发干，「试炼怕不怕？」

「怕手凉。」韩泥说，「凉进骨不要紧，凉进掌才要命。」

少年缩回去。韩泥摸玉牌，牌凉，三年约烫——烫在钱戾衡那句「你才有资格不滚」，也烫在赖福压顶最后一关。关过了，芽儿的石才在前头。

他低声对窑缝记：「捡漏得诀，得铁，得砂。得了，才配九层；九层过了，才还沈枯芽半块灵石。」

坛腹温一线，像应「正」。正一粒，够备复测；复测够了，才配往后夺魁，才配谈刘婆新衣养老钱。刘婆恩在远，芽儿笔在近——近的先还，远的先记着；混了，粒就飘。

丑时他廊下练鼻下静。香落齐，这一炉捡漏记进心里。记进心里，刃亮一分——亮在还芽儿，亮在试炼台，不亮在白织月袖风上。袖风冷，忍过去；过去了，试台见正。

公示榜仍挂他名，名末，真末。末不要紧。腹里八层实，手里一粒正——正是五年淘出来的，不是运气。运气白织月会说；她说她的，他炼他的。

赖福廊外又骂：「明日试炼，看你滚不滚！」

韩泥系紧木牌，声低：「手在，就不滚。」

同批少年在棚口嘀咕：「真敢跟赖管事硬？」

「硬什么。」韩泥擦手，「手稳就行。稳了，木牌晃不死人。」

「你怀里那粒……」

「没有。」韩泥截话，「有也不给你闻。闻了嘴就痒，痒了告示就出。」

少年退开。韩泥回窑，把罐泥又按实一分。按实像把「捡」字按进灰里——灰里辛正，鼻下有根，根细才辨得出尾摊那撮漏灰值不值。值了，才配撑过坡下那锤。

管事弟子午间路过废窑，看他铲渣，随口问：「残页还贴着？」

「贴着。」韩泥说，「碎页非禁。您闻过，香正。」

「正就别浪。」管事道，「明日试炼石凉。凉进骨，手别跟着凉。」

「不凉。」韩泥应，「凉在石，不在掌。」

管事嗯一声走了。走了，韩泥才吐半口气——气不能全吐，全吐像松；松了，窑温落，粒就杂。

第二日他又剔窑口余渣，多出两粒细砂。砂进罐，罐封泥，封得像腌菜坛——坛相凡，蔽眼；蔽眼了，手才稳得住。午后再炼一炉，成三粒，粒丑，泥纹浅，嗅之却正。正，仍藏怀。

刘婆端姜汤到窑道，姜薄，热：「窑里闷？」

「闷。」韩泥喝尽。

「闷出正丹。」刘婆哼，「老婆子等你试台说话。话说不清，粥就白给你多舀。」

「不白。」韩泥道，「试台见。」

「见了也别臭美。」刘婆又道，「臭美，手就飘。飘了，石凉进骨，你白炼。」

「不飘。」韩泥应，「炼在温，不在脸。」

夜里他抚铁片，片温，温得像炉角又归了位。归位露不得。淘了五年，淘出正，淘出还芽儿的资格；资格在，步就不飘。飘了粒洒，洒了除名，除名一出，玉牌白凉，钱戾衡就笑真。

他摸牌贴胸：「三年约，记着。手稳，粒正，明日试台见。」

见字落下，坛温一线。温像推他——推向九层，推向坡下那块试炼石。石凉不要紧；手不抖，就不滚。不滚，芽儿那半块旧石才接得住，徒恩那一笔才落得实。

更鼓沉。韩泥按木牌，牌凉贴胸。名仍在册末，坛沿仍净。舌不响，鼻子静。活再长一日，叶汤才还得上，门缝灯才接得住。刘婆粥仍多半勺——勺暖，恩不断；恩不断，手才稳得过明日那关。

廊外木牌又晃。晃不要紧。他不辩。辩了费舌；舌头留给窑温，留给「正」字。正字立住了，除名试炼那锤才砸不穿他掌心——掌心里八层气正走周，周周实，像等着隘口一裂。

铁无言丑时前又过窑外，扔半句：「试炼看手，不看脸。脸丑你认，手稳你守。」

「守。」韩泥应。

「还有。」铁无言压低，「石光短别慌。短是丑骨，稳是你。稳过了，九层才接得住芽儿那笔。」

「接得住。」韩泥道，「接不住，这五年淘了个空。」

空不得。空了，叶汤凉，刘婆勺轻，玉牌白。他坐回窑角，把焚火二字再过一遍息。息稳，温升半分；半分够明日掌按石时，不让灰线晃成滚字。

棚下起了细风。风从窑缝进，带着一点冷。冷里有钩：明日坡下那锤落不落，全看掌心暖不暖——暖在气，不在嘴。

他独眼半阖，掌按铁片，片温沉沉。沉着，像还没讲完的后半句：捡漏得了；得了，就要把坡下那关站住，才配谈九层，才配谈还石。
