# 第九十一章 废窑试炉

药山风硬。

废窑矮，他驼背，头几乎顶梁。梁上蛛网，网旧，网里落半片药渣。渣灰，灰里一点青。青芯，鼻下一动，香淡，淡而正——正，才配试头一炉。

第四炉，他不求成，只求记。记火路，记投序，记渣响——响在耳，不在舌。耳里，窑壁回一点闷声，闷像坛腹「嗡」的余音。

下山绕远路，远不要紧；远，才避赖福眼。眼在坊边，在秤上，在「晦气杂役怎么还不滚」那一句里。滚字在册，册在管事手，不在这一日的窑温里。

冬后第三月，韩泥清栏归来，掌茧裂处渗潮。潮不要紧，要紧是鼻下嗅诀已成——成在渣，成在废，成在赖扒皮克扣里淘出的那一点香。

铁无言抱臂立栏边，声低：「药山废渣场，午后有人倒炉灰。灰里若有青芯，别当泥。」

韩泥应声：「记。」

记，不为捡便宜，为试炉。试炉在废窑，不在舌。舌响，手就飘；飘了，坛也藏不住。

午后，他借送药渣之名上山。药山半腰，旧窑塌了半边，窑口红，红像烧过又凉。凉不要紧，灵脉浅一线，浅才没人占。占，要手稳，要丑，要末席——末席占穴，管事少问。

韩泥入窑，窑矮，他驼背，头几乎顶梁。梁上挂蛛网，网旧，网里落半片药渣。青芯，他鼻下一动，香淡，淡而正。

他从怀里取出瓮底泥炉——炉小，像碗，像爹留的陶片拼的。拼不要紧，拼得稳。稳，才配试头一炉。

废渣分拣，指稳。青芯三撮，褐渣一把，灵泉水半葫芦——水少，少不要紧；少，才试得出火路。火从掌纹那路引出，引到炉底，炉温起半寸，起得像坛腹应温。

第一炉，火猛。猛了，渣焦，焦味苦，苦里带腥。腥不要紧，试炉本就要焦几次。他熄半息，掌按炉壁，壁烫，烫进茧里——烫，像记。

第二炉，火弱。弱了，青芯不发，发不出正香。正香在鼻下，不在眼。眼昏，像赖扒皮盯梢；盯不要紧，窑深，盯不到。

第三炉，火匀半寸。匀了，渣里浮一点白气，白气散，散前他嗅到一丝培元路——路细，细像叶汤那日碗沿的热。热在记里，不在炉上；炉上只要稳。

管事弟子巡山，声远：「杂役别乱钻，钻了记过！」

韩泥应声：「送渣。」

管事瞥他筐，筐满，满像常活。常活，才藏得住试炉。管事走了，走了也像没瞧见窑口红那一瞬亮。

韩泥收炉，炉温退，退得像坛腹敛温。敛不要紧，试过了，路开了。路开，不在嘴，在掌——掌纹那路，又多一线细脉。

黄昏回兽栏，赖福堵侧门：「上山了？上山干什么？」

「倒渣。」韩泥声平，「筐满，手稳。」

赖福眼细，细得像秤：「秤过，少二斤青芯。」

「看错了。」韩泥说，「青芯在灰里，灰在筐底。」

赖福掀筐，掀见灰，灰里无青——青在袖袋，袖袋扎紧。扎紧，沿才净；净了，坛才藏得住。

夜，席盖严。韩泥按坛，低声道：「试过了。窑在，路在。」

坛腹温一线，贴掌，像应「试」字。试，不是成；成在远，远不要紧，手在。

邻床问：「你上山，怕不怕记过？」

「怕。」韩泥说，「怕也试。试过了，才还得了恩。」

少年不懂「恩」，却懂「记过」二字重。重，像麸皮停。韩泥不辩，只编筐半刻。藤刺扎掌，刺浅，浅些，沿才净。

赖福脚步在廊外停半息，停，像盯。盯不要紧，窑红已凉，凉得像没亮过。韩泥独眼平，手不抖，低声道：「试炉一回，不叫。」

不叫，才守得住。守得住，才接得住下一回——下一回，火要更匀，渣要更细，路要更稳。

编筐半刻，藤刺扎掌，刺浅不要紧——浅些，沿才净。筐勒匀，匀像火路；勒不匀，渣散，路断。韩泥独眼平，末藤硬，硬得像冬里规矩。规矩在，手才不飘；飘了，正测白来，白来不要紧，坛在，根在。

管事夜巡，灯照席角，照见厚席不见坛。坛不露不要紧，沿要净。沿净，席严，坛才藏得住。

药渣场角，铁无言验渣，渣灰，灰里正。正不要紧，正像试炉回响。回响在鼻下，不在舌。舌要留来清栏，留来守坛。老耿挑水路过，水洒稳：「试炉别拜邪。拜，路断。」韩泥只答：「不拜。坛在，人在。」

同批少年掩鼻过兽栏，问：「你怎天天上山？」韩泥只答：「倒渣。」答短，短才像常活。常活里藏试炉，试炉里藏还路——还路通向叶汤，通向门缝，通向省亲那日。那日远，远不要紧，手在，路就不绝。

丑时前，廊下鼻下静半刻。粟壳香淡，香一字不落。落齐，废窑才站得住第二日。瓮腹应温时，韩泥独眼半阖，内视里阶影稳在五层。嗅诀小成，沉腐浮三字在鼻下。根在，试炉就不白试。下一回，火要更匀，渣要更细。

沈枯芽在药田边探头，探头又缩，缩像怕赖扒皮。韩泥不近，只记——记叶家线，线在远，线在还。还，不在这一日，在试炉开了路。

小杂役借他编筐，借完还半根藤，藤硬，硬像规矩。规矩在，筐才匀；筐匀，才卖钱；钱买米，米买静。静，在席下坛腹温一线。

管事另记一栏「上山频次」，笔尖停：「丙九韩泥，倒渣勤。」勤，像晦气，也像活。活记在册，滚字才不好随便抹。赖福瞥见，脸青，没嗓。

夜雨前，药山云低，低不要紧，低才藏窑红。窑红一瞬，一瞬像试炉第三回火匀的回响。回响在掌茧，不在舌。舌响，手飘；手飘，坛露。

韩泥按木牌，牌凉贴胸，低声对邻床：「睡。明早清栏。」邻床哼，又睡实。睡实不要紧，醒的是沿——沿净，席严，坛在。坛在，92再试才接得住。再试，火要更匀，渣要更细，细得像编筐末藤勒到第四匝。

冬后第三月，丙九名册又添一行「倒渣勤」。勤，赖福嫌晦气，嫌不要紧，勤才像活。活长在册上，冬后正测那日，滚字才不好随便抹。韩泥按木牌，牌凉，凉进掌纹，凉不要紧，凉的是名，温的是坛。

编筐半刻，他专挑末席剩的硬藤。硬藤勒指，指裂，裂不要紧，裂像坛沿朱红——朱红在记里，不在席。席严，坛不露；露了，赖扒皮掀盖，掀盖路断。路不断，92再试才接得住。

药渣场角，小杂役问：「废窑能试？」「能。」韩泥说，「试在渣，不在名。」名在远，在130丑丹师，在260更远。远不要紧，91只开炉——开炉，火要匀，渣要细，细得像筛过三遍的灰。

铁无言验渣，灰里正，正像试炉回响。回响在鼻下，沉腐浮三字落齐。落齐，手才不飘；飘了，正测白来。白来不要紧，根在五层，层稳，坛醒，瓮穴灵穴在沿。沿净，席严，试炉就不白试。

管事弟子另记「上山倒渣勤」，勤字小，小不要紧，小也是活。活长在册，冬后正测那日，滚字才不好随便抹。韩泥按木牌，牌凉贴胸，独眼平——平，像坡下按石那一掌。掌在，路在，坛在。试炉一回，不叫；叫了，赖扒皮掀席，掀席路断。路不断，92再试在前。

匝匀，筐才卖钱；钱买米，米买静；静在席下，坛腹温一线，像应「试」字。根在五层，层稳，嗅诀小成，沉腐浮落齐——落齐，废窑就不白占。占穴占的是还路，还路通向叶汤，通向门缝，通向省亲。省亲远，远不要紧，手在，路就不绝。

同批少年赌他窑红是偷火，赌不要紧，赌输的在赖福嗓里——嗓尖，尖也真。真在众耳，韩泥只答「倒渣」，答完清栏第二趟，铲下稳，稳像没试炉。没试不要紧，试在掌，掌纹那路多一线细脉，细脉通向92，通向93，通向还路。还路长，长不要紧，手在，路就不绝。
