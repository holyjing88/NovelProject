# 第三十章 赠药

辞村前夜，霜厚。

韩泥早起编筐，藤条绕膝，指节糙。筐未成，他改捆药篓，捆实，像把泥岗恩辱都捆进肩里——肩沉，沉的不是逃，是带。带得动，才还得清。药篓角药须缠实，叶脉还湿，湿得像叶家秤还在。

门墩一响。无人喊，只有一只布包，包沉，沉得像命。包里草药，根须缠实，叶脉还湿，湿得像刚挖。挖得深，情也深。布角墨字丑，却正：

振东

叶振东。叶青禾的爹，少言，恩在暗处。暗处恩，韩泥一直知道，知道在心里留着空，空等这一包。包到，空填。

他拈一根药须，须湿，湿得像刚离土。土腥里带苦，苦得像叶家药铺秤杆落下去那声——实，不飘。他不开门追，追像谢。谢轻，还重。他把包收进篓，收实，像收省亲那日第一笔预告。告在先，还随后。

药须缠了几根细藤，藤韧，韧得像叶家父女少言的劲——劲在暗处，不在舌。舌响，谣跟；暗处恩，才走得远。

天未亮，叶家药铺门开一线。叶振东在帘后，没露脸，只嗓低：「路上用。别死半路。」

韩泥在门外：「我不谢。我记。」

「记下了，就活长些。」叶振东说，「青禾的汤，别白喝。」

韩泥应：「汤烫着记的。药也记着。」

帘后静一息，像人点头，没再出声。出声惹谣，不出声，恩也在。

叶青禾在秤后，没抬头：「我爹送的，别谢。谢轻。」她声平，「沉丹宗路远，别死半路。死了，汤白送。路远，也得走。」

叶青禾停秤，停一息：「去吧。别回头。回头，谣又跟上。」

韩泥不回头。回头，像欠。欠，他将来还，不在今夜。

陈姑送来一件补过的旧袄，袄厚，厚得像冬恩再送一程。一程，够走到宗门山脚。张麻婆塞馒，沈枯芽塞一小包干药，包上仍是「别病」二字。别病，别弱，别死在半路——半路死，恩成空。韩泥都收，收进篓，收进心里，不入腹先记在心里。记在心里，不是贪，是证。

证恩在，路才直。里正敲锣，锣声哑：「韩泥明日离村赴宗，各家少嚼舌。嚼多了，连坐。」

村人散，散得别扭，像送晦气，又像送一条活路。活路细，细得像旱里一线水。水细，也解渴。马春花远站，远站，像退到底。退到底，也是公审余韵——韵在，不用再辩。

韩泥夜坐，数恩。沈枯芽扒门缝，声怯：「韩爷爷，你明儿真走？」

韩泥停手，没开门，只答：「走。」

「走了……粥还热吗？」

「热。」韩泥说，「叶家秤在，汤就不凉。你长大些，自己也会称。」

娃在门外静一息，又问：「六笔恩……是哪六笔？」

韩泥低声，像教，不像炫：「叶汤，芽粥，陈袄，张馒，耿经，叶父药。记全了，才走得直。你将来若有人问，你也能答。」

门外点头声轻，轻得像米粒落秤盘。落定了，韩泥才继续捆篓。

他低声道：「泥岗恩，未竟。宗门债，新起。」

省亲那日，心里已划：筑基后，必还——未筑基，先记全。全，才走得直。坛角经片微温，坛腹微温，像应。应，不是醒，是送。送一程，送命入宗，送恩入心，送仇入候。

韩泥合袄，塞怀，低声道：「走。」

一字，钩向瓮醒，钩向还刘婆，钩向诛戾衡，钩向证道。钩长，命更长。外头霜白，白得像新纸。新纸待写，写的是宗门篇，写的是凡骨试，写的是瓮醒前所有忍。

忍，不是怂，是心记定了。心记定了，人便定。定字落，辞村在即。在即，他不回头。回头，谣在；向前，路在。路在，恩在，仇在，坛在。在，就够。

五更，村犬稀吠，吠得像送。送晦气翁，也送活路。韩泥最后看坛，坛底经片微温，温得像耿恩送行。他蹲低，掌心贴沿，沿凉后一线温——温得像爹临终那夜。他低声复述爹的话，像复述祖父的话：「这是你爷爷传我的，我也这么传你。」又补一句：「坛在，韩家不散。」

封坛，封实，像把根留在泥岗，把命带出村口。封实了，才数篓——药须缠实，叶脉还湿；馒布硬，陈袄补过，经片在底，干药小包上「别病」二字。他一根根拈药须，拈得慢，慢得像怕弄断情。弄断了，路也断；不断，西驿才接得住。

六笔齐，篓沉，沉的不是晦气，是恩。恩带着走，才不算逃。逃了，叶家药就白送。

村口井台还结着夜霜，霜在沿，沿净。他路过，捧一掌冷水泼面，面凉，心不退温。泼罢，把篓带紧一线，带实了，肩才不滑。

里正带人点到村口，点韩泥名：「去了，别惹宗门祸。祸回来，连坐。」

韩泥背篓：「我不惹祸。祸若惹我，我记。」

里正又补一句，像钉：「西驿验骨，丑骨末席也得站住。站不住，别回村丢人。」

韩泥只答：「站。站住了，才还得了恩。」

沈枯芽躲在娘身后，声小：「韩爷爷……回来还喝粥吗？」

韩泥停一息，独眼平：「回来，粥还热。热在叶家，热在心里。」

娃点头，像懂「还」字。还字重，重过送。

陈姑远站，怀里抱着补过的旧袄，没上前，只嗓沙：「路上冷，再披一层。」

韩泥停一息：「够了，陈姑。」

「不够。」陈姑把袄塞他怀里，「穿活了，回来还我。」

「回来，还你。」韩泥说，「不谢，记着。」

陈姑哼一声，转身，背驼，像一片旧布飘回村道。

张麻婆从巷口探半个头，扔来布包，布包里仍是馒：「吃！别饿死半路，晦气！」

韩泥接稳，掰一口，面实，实得像一辈子泥岗最后一口粮。粮入腹，恩记在心里。他低声道：「麻婆，记着。」

沈枯芽娘拽娃站远，娃声小：「韩爷爷，药别洒……」

韩泥蹲半息，独眼平：「不洒。回来，粥还热。」

娃点头。点头，像懂还字比送字重。他低声道：「六笔齐，路在篓里。篓在肩，肩在宗门。」

老耿在巷口远站，没近，只咳一声：「坛封好了？」

韩泥应声：「封好了。片在底，人不拜邪。」

老耿「嗯」一声，脚步远，远里像抬棺绳不抖——抖了，恩也歪；不抖，路才直。

韩泥迈第一步，不回头。回头，谣在；向前，沉丹宗在，凡骨试在，瓮醒那日在远。槐影短，路长——长，他瘸足紧，紧得像抬棺不抖。抖的是仇，仇另记，分立，才走得直。

天光一线，够凡骨钻。风过槐枝，极轻一声，像有人贴耳说了半句听不清的话——不是名，只是风。韩泥不追问，只按了按心口，继续走。

泥岗村在身后矮下去，矮得像恩辱都留在坛里。坛在，根在；根在，人便不虚。

老槐下，叶青禾帘内秤又落一声，落脆，像给辞村摁了章——章盖「还在」，章盖他该走，该活，该还。篓角药须仍湿，湿得像叶家秤还在。秤在，六笔在，路在肩。西驿道远，肩篓沉，步紧——步紧不像逃，像赴；赴，是先验骨，先站末席，先活长。活长了，才还得了热的。

他低声复述六笔，像清点篓里物：「叶汤，芽粥，陈袄，张馒，耿经，叶父药——六笔齐，路才直。辱线另算，仇线另记，不混。」混了，还乱；乱了，西驿白去。坛角经片微温，温得像耿恩送行——行一程，是一程，一程一程，才到还叶丫头那碗烫。泥岗在身后，宗门在前。肩沉，步紧，路直。

丑时，沈枯芽扒门缝，声怯：「韩爷爷，六笔恩……我记住了。」韩泥停手，没开门，只答：「记住就好。记住，将来有人问，你也能答。」娃在门外静一息，步远。步远，像把辞村前的最后一粒米又送一遍。韩泥合袄，独眼闭半息，再开——开，明日迈第一步，不回头。回头，谣在；向前，路在。
