# 第二十一章 新谣

榜末第二日，村道更静。

韩泥早起编筐，藤条绕着膝盖，绕得紧，像要把新谣也绕住，不让它尖。野藤浸软，绕两圈，勒紧，留半寸尾，尾塞结里，跑不了。筐底圆了，他用指节敲了敲底，声闷，闷而实——实了，才配装药，配卖钱。

静不是安，是怕。怕旱，怕罚，怕「血祠」二字。二字贴在墙根，贴得歪，像娃的手笔，也像大人借刀——刀借童口，口最干净，干净就能最毒。

童谣丫蹲在墙根，石子排成圈，不念。韩泥担筐过，脚步放轻。轻，怕惊她。往日她念「晦气翁，脏水翁」，念得脆，像刀。今日不念，像刀收了刃，刃还在，只是不敢亮。

他停半息，筐不晃。童谣丫抬头，眼清清地看他，像看一块石头——石头不凶，也不暖。暖过了，会惹娘骂；凶过了，会惹自己怕。

「不念了？」韩泥问。

「娘说，念了晦气。」她声小，「念了，旱更大。旱大，没粥喝。」

韩泥独眼平着：「旱在天。嘴在人。人闭嘴，天也不下雨。」

童谣丫嘴唇动了动，动半句，终是没出声：「韩翁……我娘不让念了。」

韩泥道：「不念好。念了，嘴脏。」

童谣丫又道：「不念，赵婶说我不懂事。」

韩泥道：「懂事不是学骂。懂事是活着。」

童谣丫抱紧膝盖，把石子踢散，散得像谣碎了一地，碎到拼不回旧词。韩泥走了。走两步，又放下一根细藤，藤韧，娃可编着玩。玩不玩无妨，无妨也像人还把她当娃——不当晦气翁的靶，当泥岗村的娃。

午后，新谣出了街。不是她念，是周嫂教，教得齐，齐得像排练：

「晦气翁，血祠翁，寡妇送汤雨不来——」

「叶家铺，血坛开，绝户进门全村灾——」

「榜末名，克亲命，补袄缝衣引旱星——」

第三句是新的。新得像刀又磨快了一面。面冲着叶，冲着袄，冲着昨夜针脚。韩泥在集尾听过半句，半句就够记——记不全，记下「补袄引旱」四字。四字是钩，钩明日公审，钩叶家秤。

赵绣春在巷口听，笑给妇人看：「听，连娃都懂！懂还做，就是存心！」

叶青禾药铺门闭得实。实里像有人在内称重，秤砣落一声，沉。沉得像脊背。韩泥在巷尾听见了。听见，心紧，不紧在谣，紧在秤——秤还在，铺还在，人就还在。人在，谣就是空风。

赵绣春瞥他，嗓子尖：「晦气翁，听见没？补袄引旱！」

韩泥不辩袄，只道：「我听见秤还在。」

赵绣春一噎：「秤在，人也邪！」

韩泥道：「邪不邪，明日公审。今日，我编筐。」

就三句。不辩邪，只辩活。赵绣春想再尖，周嫂拽她袖，低声：「里正说了，公审前少生事。」

赵绣春收半寸，仍笑，笑给风听。风过，秤声又落，落得像一口气松下来——谣再毒，毒不过一碗药的重。重是真，真压过假。

傍晚，韩泥回屋，门缝塞着一片粗纸，纸角压着姜印，像旧证。无人名，只两个字：

别念。

叶青禾的字，丑，正。不是给他，是给全村——给童谣丫，给周嫂，给赵绣春，给明日公审前还乱飞的嘴。韩泥不送还。还成对话；不还，成守。他把纸压在心口，压在叶汤旁，像又多一层布。布挡风，不挡刀。刀明日来。

门外，童谣丫又蹲得远，远得刚好看见他门。她嘴唇动半句，终是没声。冷漠比骂更冷。冷让人醒：谣不只伤人，还伤童。童冷了，村就老了。

韩泥开开门缝一尺，在筐边搁了两枚野果，小，涩，娃爱吃不爱吃都无妨。「果是山的。」他嗓子哑，「山不晦气。」

童谣丫愣了愣，拾起果就跑。跑远了，远处回头一眼——一眼不算暖，算人还肯把她当娃。当娃，就比当刀好。

夜里，心口又多记一辱。辱的是新谣。谣不记全，记下「血祠」二字，二字像钉，钉在旱上。他低声道：「记着。封不住村嘴，封得住自己嘴——不辩，不怒，不害叶。害叶，就真成谣里那个血祠翁。」

他坐在灶前，掌贴坛腹，腹里温着一线。温里他候明日，候后日，候叶家能把秤摆实的那一日。秤实，天旱也实人心。

赵绣春在巷尾骂一句，骂给风听：「别念？寡妇还立字了！」

风过，字不在风里，在心里。他把「别念」纸角再压实，像压一条会咬人的谣。童谣丫远处啃野果，啃得涩，涩得皱眉，却没扔。没扔，就是冷里一点活——活比谣长。

新谣第三句「补袄缝衣引旱星」，韩泥听见了，听见也不辩。辩袄，就是辩叶；辩叶，明日公审前再害她一口。他只把那纸按得更深。

周嫂教谣，教到自家娃咳。咳了，也得抓药——抓药还得找叶家。找叶家，嘴就得哑半寸。哑半寸，就是钩：公审前，理还在。理在，谣就还没赢。

夜里，叶家窗缝亮一线，像有人写信。韩泥不近，只隔巷低问：「还实？」

叶青禾声平：「实。秤在，话在。」

韩泥道：「别念那纸，我收着。」

叶青禾道：「不是给你。是给娃。娃不念，谣就少一口。」

韩泥顿了半息：「嗯。明日在。」

叶青禾道：「你坐着。别害我名。」

韩泥道：「不害。害你，恩白记。」

叶青禾又道：「公审前，别穿我补的袄出门。出门，他们又要说引旱。」

韩泥道：「不出门。袄在里，肘不漏。」

叶青禾道：「嗯。漏风的事我管，漏名的事你管。」

窗缝灯实，实得像喘过气来——谣在街，秤在铺，别念在心，心就不跟着谣跑。他心里低念「先还叶」，念不出声；声惹祸，念在心。心紧，紧像绳。绳将来捆实还，还先还烫——烫在叶汤，在秤声里。

丑时近，里正带人把告示贴在村柱上，墨迹歪，字正：「明日子时公审，不见者记名。」韩泥隔巷望柱影，不近——近了，赵绣春要笑「晦气翁也贴眼」。贴眼不要紧，眼在秤，不在谣。

远处陈姑咳着路过，停一息，对里正道：「公审辨诬，不辨生意。辨对了，秤才实。」

里正「嗯」一声，像记了这句话。

他低声道：「先还叶。还定了，就不怕新谣第三句。」坛角微凉，坛腹微温。温里他候公审，候锤落，候叶家能把秤摆实的那日——秤实，谣再新，也烫不了心。

天将亮，柱上「明日子时公审」六字墨迹未干，丑而真。真在，脚就站得住末席；站得住，还字就不飘。谣在街，别念在心；心压着，心就不跟着谣跑。

韩泥收筐，藤条在手，手实。明日公审，叶汤得立——立住了，才走得向还字那日。

野果还在娃手里。果是山的，山不晦气。不晦气，就是一口气。布条「先还叶」钉在心口，钉实，像还路的契。契在，公审锤落，也只备一句「我记着」——词多了像求；不求，只记。

张麻婆隔巷喊了一声：「别念那纸，比念谣强！公审前少挨刀！」

韩泥应：「记着了。」

应完，他举目西望，叶家药铺灯一线，像恩还在远处守着。明日公审，锤未落，脚已站住末席。谣出街，别念在心——心压着，还字就不飘。他把「别念」纸再折一道，折进袄里，靠着心跳。心跳稳，谣就尖不破皮。

沿口一线温贴住虎口，够撑明日公审。瓮未醒，他不喊醒；醒在缘，不在这一更。柱上帖的墨未干，干了就是日子；日子到了，秤要实，汤要热，还字才站得住脚下。
