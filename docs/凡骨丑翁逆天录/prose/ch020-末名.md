# 第二十章 末名

旱第五日，里正贴了榜。

榜贴在祠堂门上，墨还新，字却丑，却硬得像钉：「十月采药额，按斤计；末名罚月例，连坐户口。」

「连坐」两个字写得小，小得像故意让人先看见名字，后才看见罚。罚在后面，像刀背；名在前头，像刀口。韩泥挤进人群，独眼一格一格认字。名字从上往下排，像刀排队：赵家亲戚、沈家药账、周家汉子——每个名字旁都有斤两，斤两像肉，割的是活路。

最末三个字，浅得像被人故意磨过：

韩泥。斤两：七两。倒数第一。

人群让开一条缝，缝里全是看戏的眼睛。戏免费，刀却真。马春花笑得短：「绝户翁采药，采到榜尾，晦气连山都嫌！山都枯了，你活啥？」

赵绣春扇着风，扇得慢，像体面人：「赖管事早说了，晦气翁沾谁，谁倒霉。山倒霉，榜也倒霉。」

周嫂把榜尾念得响，专门念给全祠堂听：「七两！够买几文咸菜？药铺寡妇还送他姜，送姜送成榜末！」

韩泥不辩。辩下去只会变成嘴仗，嘴仗救不了榜。榜是村规，规后面有人——赖扒皮那支短签还搁在村头柜里，像一颗牙，等着咬人。他看着自己的名落在末尾。末，不是手笨。后山旱，药枯；臂伤没好全，坡滑；更有人传，跟他同坡的会沾晦气，村娃不肯指路，指路要挨娘打。压得实。实里他反而静下来，静得像井绳——绳上没水，绳还在。

里正念榜嗓子发哑，像自己也怕这旱：「末名罚月例，不得推诿。」

有人插嘴：「韩泥臂伤，坡滑，七两……也算数？」

里正顿了顿：「榜按斤，不按伤。伤，自己熬。」

韩泥转身，担着筐去集尾。筐里还有昨夜编剩的两捆细藤，藤韧，能卖。藤不晦气，晦气的是人嘴。嘴能贴榜，贴不了藤的价。半个时辰，卖出九文——比旱前少，少也进了袋。九文够买盐，够熬稀粥，够让命再撑几日。几日就够，够到还，够到公审后。

回村路上，他路过沈家豁口，停了半息，没叩门——叩门成牵扯。隔巷听见芽儿咳，咳得轻，像药叶还管用。管用，榜末也不绝恩。恩在，手就还实。

张麻婆路过，瞥他一眼，嗓子劈着骂：「榜末咋了？筐实，手就实！手实的人，老天爷瞎眼！」

她又啐一口：「山不实，扒皮实！实吃人！吃沈家，吃芽儿，吃你，吃全村！」

骂里带着暖，暖里带着刀。刀向外，不向他。韩泥记着，不谢。谢字欠着，欠到能还。张麻婆又道：「榜末怕啥？筐在，手在，命在。命在，扒皮吃不着你骨头。」

韩泥道：「骨头硬，给他硌牙。」

张麻婆笑了笑，笑里没嘲：「对，硌牙好。硌牙，他就不爱咬你。」

她压低嗓子补一句：「榜末咋了？末席也能坐。坐着听真话，比站着骂强。」

韩泥道：「明日去。坐着听。」

张麻婆「嗯」一声，担着筐继续走。走远了，像把一口气松给了他——松不在榜，松在九文，松在那句「命在」。

回村他又路过榜，用指尖蘸了唾沫，不抹名，只抹「罚」字。抹不净，净的是心：心不被末名压弯。弯了，就真成晦气。榜前有人偷偷指他的名给娃看，像指鬼。娃躲，躲得像躲榜末那几两斤。

韩泥路过，不近，只把担筐放直——筐直，人直。直不是傲，是手还实。实，就是眼下的小胜：榜能压名，压不住藤，压不住九文，压不住叶汤那点热。热在，人就还不是绝户——绝户是户籍，不是命。

门外有人敲，敲得轻：「韩翁，沈家让捎句话。」

韩泥把门开一条缝：「说。」

来人低声：「芽儿咳，药叶不够。你榜末，别往心里去。沈家还记着你狼口那回。」

韩泥道：「记着了。药叶我收过。咳，让娃多喝热水。」

来人又道：「里正说明日公审，沈家不站赵家。」

韩泥顿了半息：「不站，别惹。惹了，连坐。」

来人点头，走了。走远了，韩泥合上门，心里又沉一分——沉不是怕，是压里多了个暖：末名在榜，沈家还肯捎话，话就是活路。

屋里，心口四恩在，仇苗也在。他在末名旁又记下一辱——辱候着，不急着报。报要活，活要粮。粮在缸里，米是九文换来的，少，够三口。三口不是一家三口，是一口命、一碗粥、一条还。

他舀米，慢慢洗。水浑，仍洗。洗米像洗榜——洗不掉末，洗得掉慌。慌洗了，手就实；手实就能编筐，编筐就能再有九文。榜末是压，洗米是喘口气——气不在名，在手还实。

「末名在榜。」他低声道，「恩不在榜。恩在心里。恩在叶汤，在衣捺，在门墩的布包，在叶家没灭的灯。」

门外祠堂方向又聚起声，声里提着「明日公审叶家」。韩泥合上袄，塞进怀。腰上硌一下，硌得清醒。末名是压，清醒是活。活不够大，够撑。撑到明日巳时，撑到叶青禾把秤摆实。

赵绣春派人路过，瞥一眼榜末，又瞥一眼他屋，瞥完啐道：「末名配绝户，绝配！」

韩泥在门内「嗯」一声，不出门。出门就成戏。戏让赵绣春赢嘴，他不让嘴赢，让手赢——手实，筐实，命实。命实，明日公审才坐得住末席。

编筐时他摸了摸袄肘针脚，针脚密，密得像叶青禾的手还在。手在，汤就还能热；汤热，榜末也不冷命。九文在掌心沉着，像告诉他自己：末不是绝，是候——候公审那一锤，候叶汤还热。他用指节敲了敲筐底，声闷，闷而实——实了，才配装药，配卖钱。

夜里他还编筐。筐卖得出，榜末也能活。活，就是压里那口气。独眼瞥一眼榜上墨，墨新，名旧——这名跟了五十八年，末惯了，不怕再末一笔。

丑时，榜前墨香散进漏屋，像催他早睡。早睡睡不实，实的是手。手还在编，藤条勒指，勒出红印。红印浅，却像末名也能勒出一道活路。坛角微凉，腹里一线温，像瓮听见了「末席」——听见，不醒，只记。

他低声道：「榜末七两，手字重。公审照样去。」

编到后来，藤条断了一根，断得像把「七两」也编进筐里——编进，背着走，不辩。缸里米少，仍煮粥。粥稀，稀得像榜末的命——命稀，还在。

明日巳时，祠堂锣还没敲，心先悬着。悬着，就不算绝户绝尽。绝尽的是户籍，不是手。手在，末席在，锤来也坐得住。

粥落碗，碗烫，烫得像叶汤还在。他举目西望，叶家药铺灯一线，像恩还在远处守着。榜能压名，压不住叶汤那点热。叶铺灯终夜弱亮，弱亮像汤的影——影在，榜末也不冷命。

赵绣春门内又骂半句，骂到「末名」便住——住，像也怕明日那一锤。韩泥合上门，漏雨泥屋又漏一阵。雨丝细，他不躲——躲像亏心。手实，就能带坛上路；路还长，长不怕，怕的是不走。

盐担老汉路过门口，停了停，嗓子粗却低：「榜末七两，不当命。公审那天，我还帮周嫂一句。」

韩泥道：「帮真话。别帮我。」

老汉「嗯」一声走了。韩泥合眼，手仍按在坛沿上。末名在榜，恩在心里。心里热，手就还实。怕的是叶汤被谣咬断；叶汤在，末也在末席，末席也能听真话。公审明日见。沿口一线温贴住虎口，够撑明日公审。瓮未醒，他不喊醒——醒在缘，不在这一更。
