# 第五十九章 赵死守

丹堂战报汇总，墨迹新干，血味未干。

堂内案上堆着断刀、裂甲、染血绷带，堆得像一夜浓缩成一张榜。执事堂长老唱名，声平，平里却压不住外头弟子的窃语——窃语里，莫长老两连斩还在传，传得赵玄机守门一夜，像被挤到第二行。

赵玄机守门一夜，刀缺三口，肩甲塌，人仍立，立得像执法堂铁牌终于亮在该亮处。他立门洞，背对堂内，面对廊下人群，像把门也当作一张要还的脸。

战报写得细：子时赤焰先锋扑门，赵玄机率二十执法弟子列阵，斩先锋三人；丑时侧翼溃，他援阵眼两次，袖风接苏首座抛丹，接得不浪费一颗；寅时韩铁山远程施压，他仍不退，退半步即违令。血透甲，不是演，是恩。

柳青鸢升座，目光扫过战报，扫过赵玄机背影，宣功：「赵执事戴罪立功，守丹堂门，斩先锋三人，援阵眼两次。思过缩短，功记在心里。」

赵玄机跪谢，膝磕地砖，声哑：「分内之事。」

「分内之外，还有命。」柳青鸢补一句，补得像给全宗听：守门不是演，是还。

莫长春出列，拱手，当众，像当众还他一份体面：

「赵执事，人情还了。以后，继续还。」

赵玄机抬头，愣，眼眶红，不跪谢，只重重抱拳，拳硬，像把妒火敲进铁里，敲成守门。他想起私丹还库那夜，想起思过崖半月改十日，想起门洞血流仍不退——原来九月老头不是演，是礼；礼到了，门才站得住。

弟子窃语反转，反转成浪：

「莫长老一剑斩金丹……」

「赵师兄血战丹堂门……」

「九月老头，深藏不露？」

「深藏的是你们的眼，不是长老的剑。」有人喝止，喝止里敬，敬赵，也敬莫——敬的是恩能还，恨能收，门能守。

林挽月立人群后，唇白，白得像终于看懂：慕错人，还能改；恩对路，不能改。她望赵玄机背影，目光一触，一触即分，像把「继续还」三字，也刻进自己心里。

苏念慈在丹堂廊下，听见「深藏不露」，冷笑，笑里酸，酸里傲：「师兄从不藏，你们眼盲。」她炉仍温，温得像给赵师兄也留了一格火：守门的人，该有人看见。

霍镇山抱拳，声洪：「赵小子，今夜像个人！」

弟子笑，笑里敬，也疑——疑的是赵还能不能守到帖来，敬的是赵今夜真守住了。

孙福旧党面色白，像听见「记恩」二字，也听见自己的旧恩仇将被翻。有人缩进人群，缩得像怕铁牌也记自己。

赵玄机立门洞，刀缺，甲塌，人直，直得像执法堂终于有一块铁，亮在该亮处。他望莫长春，目光一触，一触即分，像把「继续还」三字，刻进刀鞘。

莫长春不辩，杖点地，像点住全宗嚼舌的余音。他低声，仅赵听见：

「守门比妒人难。你守住了，比还库更难。」

赵玄机喉紧，嗯了一声，重，像剑入鞘。

「妒还在，不要紧。」莫长春又补，补得像给九月将尽的夜听，「妒在，证明你还活着。活着，就继续还。」

赵玄机眼眶更红，红得像甲上的血，血干，意还在。

顾小满在人群后，眼里亮，亮得像雨夜那道光还在。她低声：「赵师兄……也记恩了。」册子入怀，入怀像又多了一行：守门，记在心里。

魏无涯在檐角，扇合，观气运，观赵玄机背影，低笑一声，像看一枚子落对了格。他不近，像懂：当众还恩，比私下赏更难收，难收，才真记在心里。

周德海在长老院听闻「赵功记在心里」，抚须不语，不语像一盆浇了一半的油，等下一阵风。风若来，是盟会，是侧门，是闸纹。

莫长春望宗主殿方向，天试风声像帖在云上。他扬声，像给全宗听：「守门功记在心里了。下一帖，该天试了。」

殿外一静，静后是一层低低的应——不是欢呼，是信，像东脉灵田雨后，根还在土里。

风过丹堂，药香与血腥缠在一起，像战收束，帖将至。刀缺甲塌人仍立——立的是门，立的是恩，立的是下一程各自还。

浪声里，敬与疑交织，像全宗都在改口——改口赵，改口莫，改口九月。改口慢，慢得像恩到心才肯改；改口到了，帖还在云上写名。

执事堂贴榜，榜文粗黑：赵玄机守门功第一，莫长春护法功不列榜——不列，像更高；更高里，弟子反而更信：信九月不是演，信赵不是妒，信青岚还有还礼的路。

林挽月自榜下过，停半息，问顾小满：「册子还记吗？」

「记。」顾小满答，「记闸，记门，记恩。」

林挽月唇白，白里一线定，定得像终于把慕错人，改到恩对路上。

苏念慈在丹堂廊下，听榜文，听「赵第一」，听「莫不列榜」，冷笑浅，浅里傲：「师兄从不争榜，榜争不来师兄。」

霍镇山再抱拳，声洪：「赵小子，下一夜还守，镇妖司给你补二十人！」

「十人够。」赵玄机答，答得像守门只要门，不要阵。

莫长春至赵身侧，仅二句：「帖来之前，门在你在。」

「在。」

赵玄机换刀，刀新，甲补，补得像把下一夜也提前押在门上。补甲时，他望莫长春背影，背影慢，慢得像老人散步，散步里袖空，塔意却沉——沉得像把「继续还」也沉进全宗眼里。

夜，丹堂门洞仍亮，亮得像给东脉守最后一寸根。赵玄机立门，莫长春不陪，陪是抢戏；不陪，是信。信赵，信门，信帖将至。

翌日，执法堂开赏，赏钱不多，赏名很硬：赵玄机，守门第一。赵玄机抱拳，不谢钱，谢令，令像路，路是莫长春留的，路是柳青鸢准的，路是他自己血守出来的。

林挽月罚役途中，听赏名，停步，问执事：「莫长老呢？」

执事答：「护法不列榜。」

林挽月唇白，点头，点得像懂：不列榜，才是榜。

孙福旧党有人嘀咕「赵也翻身」，被霍镇山一眼瞪回：「翻身？是还恩！恩不对路，翻什么身？」

莫长春经廊下，听见，不辩，杖点地，像点住余音。他望天，云薄，云薄里帖影近，近得像给赵的铁牌，也给全宗的眼。

全宗改口，改口慢，慢得像恩到心；到心这一日，赵玄机仍立门洞，刀新，甲补，人直——直得像执法堂终于有一块铁，亮在该亮处，也亮在帖将至的云上。

苏念慈送药至门洞，药温，温像炉伴战。赵玄机接，接得像接恩，不接妒。她未言莫，言只二字：「守。」

「守。」赵答，答完，门更直。

莫长春廊下过，未停，停像抢戏；过，像信。信赵能守，信帖会来，信部二这一夜，终于收在铁牌上，不收在嘴上。

魏无涯远城听全宗改口，听「赵第一」，听「莫不列榜」，扇合，低笑：「守门记在心里，比斩妖更难嚼。难嚼，才真记在心里。」

霍镇山至门洞，拍赵肩，拍得像把镇妖司也押在门上：「帖来之前，你守，我补。」

赵玄机抱拳，拳硬，硬里红，红像恩，不像妒。

林挽月罚役归途，望丹堂门洞亮，亮像也懂：恩对路，不能改。她未近，近像打扰，打扰赵守，也打扰自己尚未还清的慕错。

执事堂再贴细榜：莫长春，护法长老，不列武榜；柳青鸢，宗主，天试武试主将。榜贴，贴像把「你赠，我打」也贴进全宗眼。

识海塔影一荡，一念出塔，杖点地砖，声轻：「守稳，帖要来。天试帖至，远里见五宗。」一念收塔，杖落掌，像把全宗改口也收进塔壁。

赵守丹堂，丹在，人在，规也在——规立了，天试帖才肯到。
