# 第五十四章 坛温起伏

连雨一日，棚漏风。风进，坛席潮。

韩泥擦沿，沿滑，滑像祖父韩老栓的手。手不在，滑还在。坛腹午温，夜凉，晨又温一线。起伏，像瓮呼吸。吸，不应人；吐，不应灵。

邻床笑：「晦气翁拜坛，拜出邪脉。」

管事路过，声冷：「再嗓，滚去西栏清粪。」

邻床闭嘴。韩泥不辩，辱记心里，与恩分列。列清，心不乱。

雨歇，驿道泥浅。小商队过，两驼，铃轻。褐袍人起身，像等雨停后冬寒才够深。

韩泥搬柴，不凑亭。凑，像贪符。

老耿挑水，水洒稳：「那人等冬。冬深，才赠。赠，不白受。」

韩泥应声：「白不受。受，记恩。」

午后，藤运到丙九，藤硬，硬像冬规矩。韩泥领末捆，管事验韧：「韧，编得出筐。筐圆，手才不抖。」

同批少年偷懒，管事记名。韩泥不偷懒。

刘婆端姜汤，姜薄，热：「雨潮，别空肚。」

韩泥喝尽。赖福讽：「丑手编筐，筐也晦气。」

刘婆哼一声：「手稳，筐就圆。圆，能装柴，能装恩。」

赖福脸青，退半步。藤刺扎掌，扎旧茧，茧裂，裂口潮，潮不是血。

黄昏，铁无言廊下见他：「坛温起伏，别当醒。醒，要等缘，不等人催。」

韩泥独眼平：「不催。催，路断。」

「沿净？」

「净。」韩泥说，「血不碰沿，沿才接得住后头缘。」

同批少年碰他袖：「坛温起伏，你怕不怕醒？」

「不怕。」韩泥说，「怕也守。守着，沿才净。」

夜，雨又细。细敲棚顶，顶响像更鼓。

他低声对坛：「温凉起伏，像你眠。眠着，不急醒。醒，要等血缘碰沿。碰，不是今日。」

他摸掌茧，茧裂，裂口无血。无血，沿便安。

坛腹温一线，又凉。凉，像应「等」。

外头更鼓三响，响沉。沉里，褐袍人影仍在驿亭。在，像密宗线候冬，不急，急便邪。

丑时前，邻床梦呓：「……温……」

韩泥盖席，声低：「你看错了。睡。」

他按席，席厚。雨里小商队铃轻，像界。界内末排，界外五脉。

他低声道：「起伏不要紧，沿净要紧。净了，才接后头赠。」

符未至，怀尚空。空不要紧，褐袍人袖里黄纸边露一线，露，像凡物将出。

韩泥绕远，绕，像避腥沾袖。袖腥，洗得掉。心邪，洗不掉。

老耿门外咳：「沿净，好。净，才接得住赠。」

韩泥只答「好」，不追问。问急，像邪。等，像守瓮人。

雨歇后，泥路仍滑。搬柴经侧门，驿亭褐袍人袖里黄纸边又露一线。露，像凡符将出，不出急，急像邪。

管事盯梢：「坛边别拜邪。拜邪，滚。」

韩泥不拜。守沿，缘在。

同批少年问：「那人还不走？」

「不走。」韩泥说，「走，冬未深。深，才赠。」

管事验柴：「沉得住，记功在。」

刘婆端粥，稀，热：「雨停了，别空肚。」

韩泥喝尽。热在，像第七笔又实。

霜白那日，侧门窄。驼铃远，尘落。褐袍人仍坐，坐得像等冬寒再深半分。半分，像凡符出袖前的最后一试。

韩泥搬柴，肩勒旧印，印深，像丙九五年磨出来的实。他不凑亭，只瞥一眼。一眼，黄纸边毛，毛得像凡物。

铁无言廊角，声低：「坛温起伏，别当醒。醒在缘，不在舌。」

韩泥只点头，不催。

夜，坛腹午温，夜凉，起伏像瓮呼吸。呼吸不应人，不应灵。

他低声对坛：「你再眠。眠着，我守席，守沿，守不主动滴血。」

凉，像答应「等」。

外头更鼓沉，柱上五日条在。五日，试手在望。望不要紧，要紧是沿净，手稳。

他独眼平，手不抖：「沿净，才接得住赠。赠来，仍凡，不越阶。」

雨又起那夜，棚顶响密，密像算盘落子。韩泥披旧袄，袄薄，薄能挡半分潮。潮进席角，他不挪席，挪了沿要再验。验，手就忙。他默数坛腹起伏：午温一息，夜凉两息，丑时再温半息——起伏像瓮呼吸，呼吸不应人，不应灵。

邻床问：「坛温像醒吗？」

「像眠。」韩泥说，「眠着，不急醒。醒，要等缘。」

五更将尽，天边白一线。白里，驿亭灯灭，褐袍人影仍在。仍在，像冬寒还差半分才够赠。半分，他等得起。等，像守瓮人。

卯时，管事验棚：「坛角杂物，收干净。」韩泥只擦沿。沿滑像祖父韩老栓的手——手不在，滑还在，还在，像血脉日常。

秦霜远廊经过，木鱼不敲，只托着。托着，像规矩还在风里。她停半息：「坛温起伏，别拜。拜，像借腥。」韩泥只答：「不拜。」

午后西栏风硬，粪锹沉，沉得像湿柴。湿柴沉，肩也沉。沉不要紧，沿要净。净，才接得住后头凡符。

管事验沿，烛照一线：「脏一指，像拜邪。」韩泥用旧布再擦，擦得沿滑，沿尘，沿上无血。

夜雨第三夜，邻床翻身，碰他席角。席角潮，沿仍净。净，像六十日擦出来的。擦出来的，才配接凡物。

老耿挑水，水洒稳，声哑：「褐袍人候冬，冬深才动。动，是赠，不是邪。」韩泥不凑亭。

丑时前，他摸木牌，牌边磨毛。毛得像心口七笔。七笔分列，不混。混了，手就飘。不飘，才守得住坛温起伏。

驿道干了半寸，驼铃又轻。铃轻，褐袍人仍在。仍在，候冬。

管事验垛：「冬里沉得住，记功。」记功，麸皮不克。

同批少年编筐偷懒，管事记名。韩泥不偷懒，藤刺扎掌，茧裂无血。无血，沿便安。

坛腹起伏，呼吸不应人。不应人，像瓮仍眠。

他低声道：「褐袍人候冬，凡符将出。出，仍凡，不越阶。」

霜白铺驿道。铺了，褐袍人影仍在亭里。仍在，像冬寒还差一线才够深。深了，凡物才肯出袖。

柱上五日条与坛温起伏并放着。并放，像命与规矩挨着。挨着，才接得住后头试手，接得住凡符将出。

铁无言夜巡，灯远前嘱：「沿净，别拜邪。拜邪，恩断。」韩泥应声：「恩不断。」

轮换稳了，雨歇了，褐袍人还在。沿净，才接得住赠。

小兽栏角，小兽咳得细。他多铺干草，草旧，旧能暖。暖兽，像冬里少死一人。少死一人，恩便多记一笔。

雨停那夜，驿亭灯又亮半息。亮完即灭，灭处，黄纸边在风里翻了一角，又贴回袖里——像凡物在试冬够不够深。试完，仍不肯出。不肯出不要紧，沿净了，才接得住。接得住，凡符才肯入怀。

坛腹起伏最后一夜，午温夜凉各一线，像给后头凡符入怀又垫实半寸——垫实，不在舌，在沿。他低声道：「轮换稳了，雨歇了，褐袍人还在。沿净，才接得住赠。」

雨歇第九日，棚顶板缝仍潮。潮进席角，他不挪席——挪了，沿要再验。验，手就忙；忙，编筐藤刺便偏。偏刺，血找错路。

铁无言验沿，烛照一线：「脏一指，像拜邪。」韩泥用旧布再擦，擦得沿滑，沿尘，沿上无血。无血，褐袍人袖里黄纸才肯露那一角。
