# 第六十五章 兽栏嗅渣

赖福盯梢第二日。

兽栏腥更重，重像冬风灌进栏角，灌得人鼻酸。韩泥铲下稳，稳得像备检那日按灵石——石凉，手不抖；粪沉，手也不抖。

管事弟子巡栏，灯照角里：「丙九，湿粪换干草。换不净，记名。」

「换净。」韩泥应声。

换净，是小胜。胜在活，不在嘴。同批少年掩鼻退远：「丑杂役，你怎不怕腥？」

「怕。」韩泥说，「怕也清。清完，才有饭。」

少年愣半息，没再问——问，像沾晦气。韩泥多清半刻，小兽咳轻。咳轻，管事记一笔：「丙九韩泥，清细。」

赖福立栏外，眼细：「清细？清细也末等灵根。」

韩泥不辩。辩费舌；舌要留给嗅渣，留给守坛。第二趟清栏，肩勒旧印，印深，沉得住。沉得住，掌纹那路才不飘——飘了，粪铲就偏；偏了，管事记「抖」。

午后，药渣场。灰堆高，高里偶有一点香——香淡，淡得像将来炼丹的命。风过，灰起一线，韩泥鼻下静，静到只闻辛淡，不闻辱。

铁无言抱臂立栏边，声低：「渣里别漏香。漏香，像求。不求，只记。」

韩泥指稳分拣，分到一撮灰，灰里一点辛。辛正，他鼻下静一息——静不是闭气，是辨。辨邪正，辨药性，辨这灰里有没有「活」。

铁无言走近半步：「鼻下静，渣才说话。说话，手才稳。稳了，冬后石前才不抖。」

「静在鼻。」韩泥应声。

铁无言又指三堆灰：「这堆辛，这堆腐，这堆假香。假香浮，别当正。正香沉，沉在底。」

韩泥分拣，指稳，分出三堆。分出，铁无言才点头：「对一次，记一次。记多了，鼻下才有根。」

铁无言又取半片枯叶，叶上沾灰，灰里一点苦：「苦浅，像叶家药。药在远，恩在胸；胸有恩，手才不飘。」

韩泥鼻下一静，静里辨出苦与辛——苦沉，辛浮。浮的少，少不要紧，少也是路。铁无言把枯叶收回袖里：「别求。求，像拜。只记。」

「再辨。」铁无言指另一堆，「这堆偏腐，别当正香。」

韩泥分拣，指稳，稳里分出腐与辛——腐沉，辛浮。浮少也要记，记错了，鼻下就飘。铁无言点头走了，走了也像认可：认可不重，重的是渣场仍收他。

药渣场角有半块旧砖，砖下鼠道潮，潮里辛淡——淡得像夜里才醒的药。韩泥不扒砖，只记：记深处，不记表面；记深处，鼻下才有根。根在，冬后石前才验得过手，验得过「仍末」二字。

另一角，管事弟子验桶：「药渣水别倒错渠。错渠，全丙九受。」韩泥指桶沿，沿净，水浊而稳。嗅在渣，不在舌；舌响，香飘。管事瞥他一眼，没记名——没记名，也是小胜。

午后末，刘婆舀粥，仍多半勺：「嗅渣别空肚。空肚，鼻下飘。」

韩泥喝尽：「不空。粥在，静在记里。」

白织月经场角，袖风冷：「丙九也分拣？浪费香。」

韩泥不抬头：「香在灰里。灰在活里。」

白织月唇线绷一息，走了。少一句嘲，小胜。赖福晃秤：「分拣也分拣，末等也末等，正测仍滚。」

韩泥不接茬。接茬，全丙九饿。他低声道：「滚字在柱上，不在掌上。」

掌上有路，路是瓮醒留下的。路在，就不怕末等——末等是秤上的，不是心上的。心上有七笔，分列，不混。

第三趟清栏，雪混粪，铲下仍稳。管事弟子验角：「丙九韩泥，冬角仍你。正测前，别抖。」

「不抖。」韩泥应声。

清栏末，小兽角里咳一声，咳深半分。深不要紧，深才要换草。换草时他鼻下一静，静里闻见粪里一点药回——回得像白日那撮辛从渣里爬回来。爬回来，手才信「嗅」不是邪，是活。

不抖，是小胜。胜在肩，不在舌。回棚前，他在药渣场多留半刻，把白日那撮辛样用纸包好——纸丑，包得实。实了，才记得住；记住了，鼻下才有根。样纸塞怀，怀暖半分，暖不应符，应「记」——记牢了，夜里再辨一遍；辨对了，鼻下才生根。

回棚路上，风硬，吹得席角掀一线。他按席角，席严，坛不露。按实，掌纹那路微温——温不应风，应「嗅」字。

他揭席半寸，不揭坛，只让鼻下嗅息贴近沿边一息。沿边无香，有旧日的酸，有泥腥，有腌菜坛壳里的凡气。凡气里，却有一点药香回涌——回涌，像渣场那撮灰应了他。

坛腹温一线，贴掌，贴到鼻下也暖半分。暖不像火，像认。认他鼻下静，认他渣里不漏，认他仍末却不滚。

「嗅到了。」他对坛低声，「嗅到，手才稳。手稳，正测才站得住。」

坛不应字，只温。邻床问：「你闻什么？」

「坛味。」韩泥说，「睡吧。」

邻床哼一声，睡了。睡了，秘密才藏得住。夜，他再分拣一遍白日留的渣样——样少，少不要紧，要紧是记。记辛，记淡，记腐沉辛浮。样纸摊开，灰里一点辛又浮半分，浮不要紧，浮也要记——记错了，鼻下就飘；记对了，渣才说话。

他又在廊下练半刻闭息，息在鼻，不在嗓。嗓响，香飘；香飘，手就滚。滚了，正测白来。白来不要紧，手在，名在，路在。

铁无言路过廊下，脚步停半息，声更低：「鼻下静，符才暖。暖，护骨寒。」

韩泥不答，答在心里——心里七笔不混，嗅在渣，暖在怀。怀符仍凡，不越阶；阶在远，寒在近，近先祛。

赖福在门外咳一声：「还不睡？睡实了，也好滚。」

「在。」韩泥只答。

在，就不滚。不滚，才接得住冬后那一验。验手，不验灵色；灵色末，手不能末。

他按心口，七笔分列。嗅渣是活，活是路；路通向还恩，通向省亲，通向叶丫头那碗还没还尽的汤。坛腹温一线，应「嗅」，不应慌。

慌像少年，他是记心翁——翁慢，恩不凉。更鼓沉，沉里，他合眼不睡实，鼻下仍静一息——静，像把白日那撮辛，摁进掌纹那路里。

路在，嗅诀起了；起了，不在舌，在鼻下。鼻下静，香才不飘；香不飘，手才稳。

「渣里一点辛，辛在活里。」他对坛低声，「活在，才嗅得到。」

编筐季未尽，他接末单，藤条硬得像冬里的规矩。编到一半，藤刺扎掌，他偏手，刺浅些——浅些，血仍不来；不来不要紧，手要在。管事弟子路过，瞥筐沿：「丙九也编筐？」

「编。」韩泥说，「筐成，手稳。手稳，石前不抖。」

管事哼一声，走了。走，也像记下一笔「手稳」。赖福讽远：「末等编筐，筐也晦气。」

韩泥只按木牌，不接。接茬，全丙九饿；舌要留给鼻下静，静了，渣才说话。

更鼓二响，响沉。他按坛沿，沿温不退，坛底极轻一颤，像应「嗅」字最后一笔。应完，他盖席半寸，不揭坛，只让鼻下息贴近沿边一息——沿边无香，有凡气；凡气里药香回涌，回涌像爹说过：「鼻下静，路才直。」

夜禁脚步远，柱上条角白一线。白，像正测已在路上走。「嗅到了。」他低声道，「坛应了。三日后验手，鼻下仍静。」

坛腹应温，不应字。温，像把「嗅」字摁进掌纹那路里——路在，就不滚。

赖福脚步在门外停半息，停，像嗅。嗅不要紧，坛不露，席严——严了，才像腌菜坛，不像邪。赖福终是没掀席，脚步远，远里仍有一句：「丑杂役，我盯着你。」

「鼻下静，手在。」韩泥声平。静在记里，验手在冬后，不在这一夜。坛腹应温，像把「嗅」字摁进掌纹那路里——路在，就不滚。嗅在渣，不在舌。

更鼓沉，沉里，他摸怀玉牌，凉贴胸——三年约还远，明日试手验腰，坡下才站得住。坛应了嗅，明日若有人再来推「滚」字，他只拿手稳顶回去。
