# 第五十二章 夜袭备战

子时前两时辰，东脉像一张未张满的弓。

柳青鸢立灵田高台，令旗起落，镇妖司列阵，外门内门各守一线。霍镇山盾阵顶前，骂声比鼓响：「赤焰要来，老子先骂醒你们！」

弟子列阵，腿抖，抖里仍咬牙，像半块灵石撑着。民户缩在阵后，孩子哭被捂住，哭里有人低声念青岚门，念莫长老，念三品金丹——念成胆。

周德海立后阵指挥棚，斗篷遮面，令旗在指间，迟迟不落：「后阵稳固，侧门慎开，防中伏。」

霍镇山回头吼：「周老头，侧门不开，丹堂腹背受敌，你防谁的伏？」

周德海笑，笑里冷：「防你们的莽。」

柳青鸢不辩，只记，像记一笔将来才还的账。顾小满在侧径望后阵，望周德海令旗，令旗干净，干净得像从未举起过。

围观弟子在阵后窃语：「周长老又迟疑？」「侧门不开，丹堂危险。」「莫长老在丹堂外坐着，像七夜前一样。」

丹堂，苏念慈立阵眼石，地火阵纹淡红，炉中火舌细稳。弟子搬阵材，搬药匣，搬得急，却不敢噪。

莫长春未进门，立门外石阶，杖点地，一声，两声，像给夜战敲节拍：「阵眼你守，老夫门外坐着。炉开着，别熄。」

苏念慈扬声：「守。」

执法堂，赵玄机率二十人赴丹堂侧翼，路过莫长春，抱拳，礼深：「长老，赵玄机今夜守门。」

「守。」莫长春道，「门比剑难守。守住了，恩就还了一半。」

赵玄机颔首，刀光出鞘半寸，像把妒火也出鞘，换成守门的硬。

林挽月罚役途中听见，唇白，白得像终于看懂：恩对路，不能改。

杂役堂，顾小满分班，路线图在怀，像第二张半石。莫长春递她三张隐匿符，符温：「活着把消息送到。送到，比杀一人有用。」

顾小满接符，指节白：「小满记恩。」

她率队没入侧径，夜吞人，像吞一根细线，线那头拴着青岚门的耳。

莫长春望线消失处，望东脉，望后阵，低声，像钩：

「子时。弓弦满了。缺一环，田就失。」

风腥，田稻似浪，浪里杀意已至门前。

「七夜前守炉，今夜守什么？」「守门。守田。守恩。」窃语轻，轻不过田稻似浪，浪里杀意已至门前。

韩铁山在远山布阵，笑里刀，刀向田，也向丹堂。厉无殇在帐内听报，声哑：「莫长春，青霜入袖？」他砸杯，杯碎，像砸不碎袖中记忆：手帕三息，杖扫妖骑，如今霜入袖——礼还没送完，恨还没记够。

魏无涯观调度，低笑，像看因果满到不得不兑。他传讯秦商言，只四字：「子时动。」

弓张满，三线布，侧门闭——闭的是伏，还是忠？全宗尚不知，尚要子时称一称。

莫长春望线消失处，望东脉，望后阵，低声，像钩：「子时。弓弦满了。缺一环，田就失。」风腥，田稻似浪，浪里杀意已至门前——门前是丹堂，丹堂是炉，炉边坐着九月老头，像七夜前一样。

杂役堂，顾小满分班，路线图在怀，像第二张半石。莫长春递她三张隐匿符，符温：「活着把消息送到。送到，比杀一人有用。」顾小满接符，指节白：「小满记恩。」她率队没入侧径，夜吞人，像吞一根细线，线那头拴着青岚门的耳。赵玄机率二十人赴丹堂侧翼，路过莫长春，抱拳，礼深：「长老，赵玄机今夜守门。」

弓张满，三线布，侧门闭——闭的是伏，还是忠？全宗尚不知，尚要子时称一称。风腥，田稻似浪，浪里杀意已至门前。

霍镇山回头吼：「周老头，侧门不开，丹堂腹背受敌，你防谁的伏？」周德海笑，笑里冷：「防你们的莽。」柳青鸢不辩，只记，像记一笔将来才还的账。

「守。」莫长春道，「门比剑难守。守住了，恩就还了一半。」赵玄机颔首，刀光出鞘半寸，像把妒火也出鞘，换成守门的硬。
