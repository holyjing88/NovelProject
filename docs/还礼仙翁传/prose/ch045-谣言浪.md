# 第四十五章 谣言浪

翌日，全宗议论如潮。

「莫长老浴房外徘徊！」

「老不正经达峰了！」

传得比除魔符还快，脏得比药渣还腥。弟子添油加醋，添到莫长春成了九月将死急色，添到苏念慈成了被辱的丹堂首座，添到柳青鸢成了压不住的宗主。

丹堂，苏念慈摔杯，瓷碎，声脆：「师兄岂是这种人！」

弟子噤声，噤里仍有人偷换眼神，像爱听脏的，脏的比三品金丹还香。

执法堂，柳青鸢令禁谣，铁牌光冷，却难禁人心——人心爱听脏的，脏的比除魔符传得快。赵玄机立堂上，面色平，平里烧，烧得像妒火终于点着纸，却烧不出预想中的轰响，因为莫长春昨夜止步不入，戏缺了高潮，谣言只好靠嚼舌补。

林挽月躲廊角，不敢看莫长春方向，只敢看自己的鞋尖，尖上泥，像罪。

顾小满在杂役堂浆洗，低头，指节白，像半块灵石又被人嚼成脏字。她抬头，冷眼扫过传谣者：「宗主裁过，再传，去执法堂。」

传话者噤声，却嘟囔：「杂役也管门风？」

顾小满把册子往案上一放：「凭记。你也能记。」

莫长春小院，门闭，茶仍煎，煎得淡。

弟子门外嚼舌，故意让他听见：「九月老头，浴房外……」

他开门，捧茶出，坐廊凳——不坐主位，借门外风声，笑：「他们说得对，老夫靠的就是这些人——靠他们传话快。」

弟子一愣，骂也不是，退也不是，像一拳打在棉上，棉里却有杖，杖不点人，点得人心虚。

苏念慈闻讯赶来，眼眶红，未哭，只道：「师兄……」

「不急。」莫长春道，「浪要起，也得起完。起完了，才好还。」

柳青鸢在远处听见，眸光冷，像把「还」字记进宗主令里。霍镇山骂：「脏嘴！夜袭在前，还嚼这个！」

魏无涯在檐角，扇开一线，观气运，观谣言浪，低笑一声，像看戏看到最脏处，反而更静。

弟子围观廊下，围观里反转尚未起，起的是怒与疑——怒脏，疑赵，疑长老为何不辩。

莫长春不辩，只喝茶，茶淡，淡得像把浪当茶晾。

孙福旧党嘀咕：「老不正经达峰了……」顾小满冷眼：「达峰的是脏嘴，不是长老。」

赵玄机立堂上，面色平，平里烧，烧不出轰响——止步不入，戏缺高潮，谣言靠嚼补。

林挽月躲廊角，不敢看莫长春方向，只敢看鞋尖，尖上泥，像罪，也像将证的胆。

他低声，像钩：

「浪到峰，堂上见。」

围观弟子廊下，围观里反转尚未起，起的是怒与疑——怒脏，疑赵，疑长老为何不辩。霍镇山骂脏嘴，魏无涯观浪，低笑。脏字达峰，证人在后头——延迟的爽点，在堂上，不在廊上。

风大，脏字达峰，达峰像帖在每个人眼里——帖在堂上，证人在后头，还字在袖里。

茶淡，茶淡得像把浪当茶晾。晾完了，才好还。

魏无涯在檐角观浪，扇合，低笑：「脏字达峰，堂上见。证人已在后头，只等一张嘴肯翻。」

霍镇山骂：「脏嘴！夜袭在前，还嚼浴房！」骂声洪，骂得像鼓，鼓点催战，催得脏字也轻一寸。

林挽月躲廊角，唇白，白得像终于懂：慕错可改，证字更重，重得像恩。

苏念慈闻讯赶来，眼眶红，未哭，只道：「师兄……」「不急。」莫长春道，「浪要起，也得起完。起完了，才好还。」

柳青鸢令禁谣，铁牌光冷，却难禁人心——人心爱听脏的，脏的比除魔符传得快。林挽月躲廊角，不敢看莫长春方向，只敢看自己的鞋尖，尖上泥，像罪。

大因果还在堂上，像挽月反戈，像夜袭子时——谣言达峰只是小浪，宽恕才是大还礼的前奏。

围观弟子在廊下踮脚，踮得像等反转——反转未起，起的是怒与疑。有人低声：「长老为何不辩？」另一人答：「不辩，像把浪当茶晾。」

外门弟子踮脚问：「长老真不管？」顾小满冷眼扫来：「管。管的是你们这张嘴。」那人缩脖，余音却飘向执法堂：「达峰了，就得当众还。」

丹堂廊下，又有人压着嗓：「苏首座摔杯，摔给谁看？」旁人嗤：「摔给脏嘴看。脏嘴达峰，堂上才肯翻。」话落，廊静半息，半息里像听见铁牌将要落案——落案不在廊上，在堂上，在证人喉里。

莫长春不辩，杖点地，像点住余音；柳青鸢不软，剑在鞘，威在眸。霍镇山骂声洪，骂得像鼓，鼓点催战。东脉云低，腥风一线，像下一钩已挂在云上。
