# 第九十二章 废窑再试

雨歇。

药山废窑，窑口红湿，湿像昨夜火没尽。没尽不要紧，韩泥二回上山，筐里仍是药渣，渣上盖灰，灰像常倒。

这一回，他不急火。急火焦渣，焦渣苦，苦不是路。路在匀，匀在掌，掌在茧。

铁无言昨夜说过一句：「再试，先温炉，后投芯。」

韩泥记在心里。记，不声张。声张像求丹；不求，只试。

入窑，先以掌温炉壁。壁凉，凉退半寸，退得像坛腹应温。温够了，投青芯一撮，褐渣半把，灵泉三滴——滴少，少才试得出分寸。

火起，火不猛，不弱，像呼吸。呼吸匀，渣里白气起一线，起而复散。散前，他鼻下捕捉到培元路第二丝——丝细，细而正。

正，他不敢喜。喜了，手飘；飘了，赖扒皮来。来不要紧，窑深，声不出。

第二炉，他换投序：先褐后青。序错，香杂；序对，香单。单香在鼻下，像铁无言验渣时说的「正」。

管事弟子过路，问：「韩泥，你倒渣倒两趟？」

「渣多。」韩泥说，「筐沉，分两次。」

管事记一笔，笔丑，真在「分两次」。真，才像常活。

第三炉，火匀一寸。一寸够了，再多，窑壁裂。裂不要紧，裂是旧窑，旧窑才没人争。争穴，要末席；末席，要手稳。

炉开，无丹，只有一团凝灰。凝灰里，一点白，白像未成之珠。未成不要紧，再试；试炉本就不成丹。

韩泥收灰，灰入袖袋，袋扎紧。扎紧，沿净。沿净，坛藏得住。

下山遇白织月，袖风带香，声冷：「丑杂役，药山是你扫的？」

「倒渣。」韩泥不抬眼，「手稳，不脏内门路。」

白织月嗤：「末席也配碰药山？」

韩泥只答：「配倒渣。」

嗤声远，远不要紧。远，才试得稳。

回棚，赖福秤筐：「仍少二斤。」

「看错了。」韩泥说，「渣在筐底，底沉。」

赖福掀底，底有灰，无青。青在袖，袖在，路在。

夜，他按席角，席严。坛腹温一线，贴掌，像应「再试」二字。再试，火路熟半分；熟半分，不够成丹，够准备。

刘婆端粥过，粥稀：「又上山？」

「倒渣。」韩泥说，「手稳，饭稳。」

「饭稳，恩不断。」刘婆哼，「别偷火，偷了连坐丙九。」

韩泥点头。偷火不在他路上；路上只有试炉，试到匀，试到能匿炼。

邻床悄问：「你真只倒渣？」

「倒渣里试火。」韩泥声平，「火试过了，才藏得住。」

少年闭嘴。闭嘴，像丙九也学会忍。

丑时，廊下鼻下静半刻。粟壳香淡，香落齐，掌纹那路又温一线。温不应喜，应「稳」——稳，是再试的回响。

赖福脚步远，远里一句：「丑杂役，我盯着你。」

韩泥只答：「在。」

在，就不滚。不滚，废窑第二回就不白来。下一回，要备匿丹之料——备在暗处，备在席下，备在还恩前头。

他低声对坛：「再试过了。火匀半寸，路熟半寸。」

坛不应字，只温。温，像爹搭沿，像泥岗丑时那口白气。

更鼓尽，雨歇，药山湿，湿里藏一点窑温余韵。余韵散，散进掌，散进记——记：再试，火可匿。

卯时，他先清栏。栏腥，腥不要紧，手稳才要紧。铲下稳，稳里拾褐渣半把，把入袖，袖扎紧。扎紧，沿才净。

药渣场角，铁无言再嘱：「再试，先温炉，后投芯。芯多一分，焦；少一分，不发。」

第二炉投序照页，页浅，浅而真。真在控火半寸，半寸像席下坛腹应温。应温，不应喜；喜了，手飘。飘了，赖扒皮来。

药渣场午后，铁无言分渣，渣分三堆，堆边立牌：「青、褐、废。」韩泥只取废，取废不要紧，废里才有试路。试路在第二回，第二回要分两次投，投序写在袖袋边，边用汗渍，汗渍干，字即无。

同批少年赌他不敢再上窑：「再焦，管事记过。」韩泥只答：「倒渣。」答短，短像常活。常活里，掌纹火起半寸，半寸匀，匀像坛腹应温。应温，不应喜。

刘婆缝顶针路过，针亮，声低：「你掌又裂？」「编筐。」韩泥说，「筐勒藤，藤硬。」「硬好。」刘婆哼，「硬手，饭稳。」饭稳，恩不断——恩在粥，也在匿丹将备。

赖扒皮夜查棚，火折照席，照见席严。严不要紧，页在怀，灰在袖。袖袋扎紧，沿才净。净了，93备料才藏得住——藏三处，散像杂役藏粮。

雨歇，药山湿，湿里余温不散。不散不要紧，不散像第二回试成。韩泥独眼平，低声对坛：「再试过了。下一回，备。温一线，贴掌，像应「熟」字。熟半寸，路熟半寸，半寸够接93匿丹准备。

雨歇后第二回，韩泥分两次投——第一次青芯，第二次褐渣，中间掌按炉壁半息。半息，像控火；控火，像93匿丹之序预演。预演不要紧，预演不成丹，成路。路熟半寸，半寸够备料。

赖福秤筐，筐底沉，沉是渣，不是偷。偷，赖福眼细；眼细不要紧，细也看错了。看错了，韩泥只答「倒渣」，答短，短像常活。常活里，袖袋灰扎紧，沿才净。

刘婆端粥，仍多半勺：「又上山？」「倒渣。」「手稳，饭稳。」恩在粥，也在再试——再试过了，下一回备匿丹。备在暗处，备在席下，备在还恩前头。

邻床鼾声里，韩泥席下按坛，坛腹温一线。温不应喜，应「稳」——稳是二回回响。更鼓尽，药山湿，湿里窑温余韵不散。不散，第二回就不白来。不白来，93才接得上。

药山半腰，旧窑塌半边，塌不要紧，塌才少眼。少眼，第二回才试得出控火——控火半寸，半寸像坛腹应温。应温，不应喜；喜了，手飘，飘了正测白来。白来不要紧，坛在，根在。回棚后邻床问：「雨夜上山？」「倒渣。」韩泥声平，「筐满，手稳。」手稳，饭稳，饭稳恩不断——恩在刘婆粥里，也在93将备之料里。

雨线斜斜，斜进半塌窑口，口内韩泥第二回试火，火路分两投——先青后褐，中间掌按炉壁半息，半息像93将写之无字投序。投序不在纸，在掌；掌稳，褐才发，发而正香浮一丝，丝细，细像培元路第二丝。丝在鼻下，不在眼；眼昏，像赖扒皮盯梢，盯不要紧，窑深，雨大，声不出。
