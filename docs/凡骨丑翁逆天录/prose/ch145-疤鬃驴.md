# 第一百四十五章 疤鬃驴

药市尾巷，驴嘶一声，嘶哑，哑像疤鬃。

卖驴老汉声粗：「丑杂役，要驴不？驮药，不载人。」

韩泥验齿，验蹄，验背——背宽，宽才驮药；药在，145才实。

价砍半刻，砍像末席该砍的。砍完，钱串去一截，截去不要紧，驴在，路在。

疤鬃驴毛色灰，灰里一道旧疤，疤像刀，刀划皮，不划手——手稳，驴才跟。

老汉递绳，绳糙：「丑人骑丑驴，配。」

韩泥不辩，接绳，绳在，就不滚。

铁无言立巷口，声低：「驴在，别显摆。显摆，赖福记。」

「不显摆。」韩泥说，「驮药，不驮名。」

刘婆端饼，饼糙：「驴要喂，别空肚。」

韩泥点头。谢轻，恩重。

丑时，他牵驴回棚，驴稳，稳像145该得的——得骑，不拜暖；拜暖，路断。

他低声对坛：「疤鬃在。记着。」

兽栏旁石洞，洞浅，浅里拴一匹驴——驴瘦，鬃乱，鬃上疤一道，道像泥岗路，路硬，硬出「疤鬃」二字。

管事扬声：「丙九韩泥，宗门配畜，疤鬃驴一头。驮药，不驮人；驮人，税加倍。」

韩泥只答：「驮药。」答短，短才像145该接的——接畜，不接荣；荣在远，在七品类骑谱上那一行灵品。

驴眼钝，钝不要紧，钝才稳；稳才驮得住药包，驮得住142后坊市续摊的材。

他抚鬃上疤，疤糙，糙像丑骨——骨丑不要紧，蹄稳才要紧；蹄稳，145才像得骑，不像得宠。

铁无言验蹄，蹄铁旧，旧而齐：「驴怯人，你别怯驴。」「不怯。」韩泥说，「怯了，包散。」

刘婆端豆，豆粗，喂驴半把：「畜也要饭。饭稳，驮才稳。」

赖福瞄一眼，瞄不要紧，瞄不出粒——粒在怀，驴在栏，两不混。

沈枯芽递口信，信仍枯叶：「叶姐说，驴丑不要紧，路正。」

韩泥系药囊，囊灰，灰里白细，细像培元散余韵——韵在驮，不在名。

他低声对坛：「疤鬃在了。记着。」坛不应字，只温。

更鼓沉，他牵驴过坊市尾，尾巷人侧目，侧目不要紧，蹄声稳，稳像145该落的钉。

兽栏登记后，他牵驴回丙九，蹄声稳，稳像七品类谱上灵品那一行。

药包上肩，肩沉，沉不要紧；沉，才试驴；试过了，146验息才像实。

邻床悄问：「丑驴也配你？」「配。」韩泥说，「配不配，蹄稳不稳。」

管事记簿，记「疤鬃驴一头，驮药」，记细，是小胜。

驴拴柱后，鼻喷白气，气稳，稳像145该落的钉——钉驮药，不钉名。

他望坛，低声道：「驴在。记着。」坛腹温，贴掌。

骑不在荣，在驮。驮药，不驮人；驮人，税加倍，末席驮不起。驮得起，是疤鬃驴，灵品七品类谱上那一行：瘦，鬃乱，疤一道，道像泥岗路。路硬，硬出畜稳；稳，才配145硬锚。锚不在快，在蹄；蹄稳，药包才不散，142后坊市材才续摊，146验息才像实。实了，147名号才压得住谣；谣在舌，蹄在声，声稳，145才真。

真，不是得宠，是宗门配畜——配像例行，例行里藏路。路通向药峰，通向坊市，通向262省亲前驮药；驮药在远，145只接。接法是不怯驴，不怯包沉；沉，试手，试驴，试丑骨末席能不能把灵品骑用成常活。常活里藏线：线连146十七，连147续名，连148叶信，连149周家风——风在远，145只驴。

驴眼钝，钝不要紧，钝才稳；稳像韩泥独眼，眼丑，眼准。准，铁无言验蹄，蹄铁旧而齐；齐，刘婆喂豆半把，豆粗，畜也要饭。饭稳，驮才稳；稳了，沈枯芽枯叶信说「驴丑不要紧，路正」——路正，是145该落的字。字落，更鼓沉，牵驴过坊市尾，侧目不要紧，蹄声稳，稳像钉。

钉住145，144名才不虚，146层才不实；实虚在远，145只畜。畜在，就不滚；不滚，能还才送那套铁律才配第二部走。

蹄声稳，驮药路开；路开，146验息才像实，147名号才续得住。续得住，148叶线才接得稳。

蹄声远去，远像145该落的钉。钉住驮药，144名才不虚，146验息才像实。他望兽栏影：「疤鬃在。路正。记着。」驴在柱后，鼻喷白气，气稳，稳像七品类那一行。

白织月袖风过，韩泥不辩。辩，费舌；舌要留来守坛。

赖福廊下过，韩泥只答：「在。」在，就不滚。

柱上条角白一线，三年约在怀。怀凉，掌温。

坡下石阶湿，他数息，数帖，数债序。

同棚杂役笑，笑在耳，不在心。

废渣巷日斜，他验渣，渣正，手才静。

侧门驿亭风硬，他听风，风过，记在心里。

散队后，他过外堂阶下，分拣半刻，指稳——指稳，路就不飘。

编筐半刻，藤刺扎掌，刺浅，浅些沿才净。沿净，坛才藏得住。

丑时，练鼻下静。粟壳香淡，香一字不落。

管事记簿，记细，是小胜。胜在忍，不在辩。

他望坛，低声道：「手在，路在。更鼓沉，天边白一线。白里，他按木牌，牌凉贴胸。坊市尾巷风硬，他收摊如石，石在，就不滚。」

验息后，他理怀内册页，页黄，字丑，丑而真。

铁无言声低：「别飘。」韩泥应声：「不飘。」

刘婆端粥，仍多半勺。韩泥点头，谢轻，恩重。

药峰分拣照旧，指稳，鼻下静一息。

帖在怀，怀沉，沉的是序，不是惧。
