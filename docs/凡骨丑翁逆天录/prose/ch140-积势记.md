# 第一百四十章 积势记

积势记，记在一册薄册里。

册是管事所发，发像例行——例行不要紧，例行里藏路。

册上写：药峰、坊市、大比预选、层数登记。登记齐，齐像140收束，不像散。

韩泥一页页看，独眼平。页上无恩字，恩在记里——记刘婆已还，记叶汤在前，记142要谢老耿。

老耿名在142，142在远；远不要紧，140只积势，不越界。

沈枯芽口信到，信仍无字，只有半片枯叶：「叶姐脉稳些。」

稳不要紧，稳像95培元余温——温在脉，不在嘴。

铁无言立门外，声低：「积势够了，别躁。」

「不躁。」韩泥说，「躁，手飘；飘了，142谢不出。」

赖福最后查席，查完，哼：「丑杂役，滚远点。」

韩泥不滚。不滚，是约；约在玉牌，不在赖福舌。

白织月远廊过，袖风冷，没停。没停，像默认——默认丑骨还在走。

刘婆补扣，声低：「140了，别空肚。」

「不空。」韩泥说，「空肚，手飘。」

丑时，练鼻下静。粟壳香淡，香落齐，积势这一簇记进心里。

他望坛，低声道：「142在前。谢老耿，经叶家。记着。」

坛腹温，贴掌，像应「势」——势够了，恩才送得出；送得出，才配称能还才送。

更鼓将尽，天边白一线。白里，他按心口：「汤还是热的。」

六字落，像路标——标向142，标向175筑基丹，标向188雨夜。

大比帖在怀，怀沉，沉的是债序，不是惧。惧在，手也要稳；稳了，第二部才像第二部。

外门弟子渐散，散声远，远不过坛腹那一线温。温在，就不滚。

册页翻尽，尽像140收束——收束不越142，不越175，不越188。

管事弟子收册，声平：「积势记齐。下一簇，142培元散谢老耿。」

韩泥不辩。辩，像急；急，手飘。

药峰、坊市、栏边、坛角——四处在，手才稳；手稳，帖才压得住。

赖福最后骂一句，骂声远，远不过坛腹那一线温。

沈枯芽口信无字，只有半片枯叶——叶黄，像叶汤还在前。

铁无言声低：「142在前。别躁。」「不躁。」韩泥说。

刘婆端粥，仍多半勺：「140了，别空肚。」韩泥点头。

粟壳香淡，香落齐。

他望坛，低声道：「积势够了。142，谢老耿。记着。」

坛腹温，贴掌，像应「势」——势够，恩才送得出。白里，大比帖在怀，怀沉，沉的是路，不是惧。

册收，势足，足像140该收的——收，不越142。

管事扬声：「下一簇，142培元散谢老耿，经叶家。」

韩泥记在心里，心里排序：142、175、188——三笔在远，一笔刘婆已还。

帖在怀，怀沉，沉的是路。路通向大比，通向筑基，通向省亲。

他按心口，低声道：「汤还是热的。」

六字落，像路标——标向142，标向叶汤，标向第二部中段。

坛腹温，贴掌，像应「势」——势够，才配称能还才送。

积势记，记在一册薄册里——册是管事所发，发像例行，例行里藏路。路通向142培元散谢老耿，通向175筑基丹，通向188雨夜，通向235胜钱，通向260筑基真名；260后，262省亲。省亲远，远不要紧，140只收束首簇。收束不越界，不越142，不越175，不越188——越界，手飘；飘了，恩送不出。

送不出，就不配称能还才送；能还才送，是17第二部馈缘铁律。铁律在远，140只积势；势足，才送得出142那粒。粒在远，刘婆已还；还在叶汤，还在老耿，还在泥岗。泥岗在262，262在远；远不要紧，帖在怀，怀沉，沉的是序。序里写：药峰、坊市、栏边、坛角、层十四十五——写齐，像140该收。

收了，第二部首簇才像第二部，不像一部尾韵。尾韵在130还刘婆，新章在131帖，在132药峰，在133挂牌，在134铁兄，在135袖风，在136十五，在137续坊，在138守坛，在139风紧；风紧，140记齐。齐，像弓满；满不要紧，箭在142。更鼓将尽，天边白一线，白里，他按心口：「汤还是热的。

」六字落，像路标——标向142，标向叶汤，标向大比，标向三年约。约说站钱戾衡台前，才不滚；不滚，是丑翁活法。活法在忍，在还，在手稳。手稳，坛腹温一线，像应「势」——势够，第二部才在走。

积势记齐，142在前。在前，手稳，就不飘。

册收，势足，142在前。在前，手稳，就不飘；飘了，恩送不出。

势够，他按心口：「汤还是热的。」六字落，像路标——标向142。

温在，就不滚；不滚，142才近。近在142，不在140；140只积势，势够，才送得出。

验息后，他理怀内册页，页黄，字丑，丑而真。

铁无言声低：「别飘。」韩泥应声：「不飘。」

刘婆端粥，仍多半勺。韩泥点头，谢轻，恩重。

药峰分拣照旧，指稳，鼻下静一息。

帖在怀，怀沉，沉的是序，不是惧。

白织月袖风过，韩泥不辩。辩，费舌；舌要留来守坛。

赖福廊下过，韩泥只答：「在。」在，就不滚。

柱上条角白一线，三年约在怀。怀凉，掌温。

坡下石阶湿，他数息，数帖，数债序。

同棚杂役笑，笑在耳，不在心。

废渣巷日斜，他验渣，渣正，手才静。

侧门驿亭风硬，他听风，风过，记在心里。

散队后，他过外堂阶下，分拣半刻，指稳——指稳，路就不飘。

编筐半刻，藤刺扎掌，刺浅，浅些沿才净。沿净，坛才藏得住。

粟壳香淡，香一字不落。

管事记簿，记细，是小胜。胜在忍，不在辩。

他望坛，低声道：「手在，路在。」坛不应字，只温。

更鼓沉，天边白一线。白里，他按木牌，牌凉贴胸。

坊市尾巷风硬，他收摊如石，石在，就不滚。
