# 第一百三十六章 层十五

层十五，不是雷，是一线实。

实从十四里长出来，长像药峰分拣里多走半刻鼻下静——静到只闻辛淡，不闻辱。

韩泥独眼半阖，内视里阶影稳在十五。稳，不是喜，是活；活了，大比路才多一寸。

管事弟子验息：「十五层，登记。」

「登记。」韩泥应声。应声短，短才像常活。

同批少年嘀咕：「丑骨也能十五？」

韩泥不辩。辩，像显摆；显摆不要紧，层在，手在。

药峰活照旧，照旧才像积势——积势不在快，在稳。

编筐半刻，藤刺扎掌，刺浅，浅些沿才净。筐卖钱，米在，恩才还得上。

铁无言送栏灰，灰细：「十五层，别飘。」

「不飘。」韩泥说，「飘了，帖白接。」

刘婆端粥，仍多半勺：「层高了，饭也要稳。」

韩泥点头。谢轻，恩重。

赖福查席，火折照角：「席下啥味儿？」

「渣味。」韩泥声平，「药山活，活有味。」

赖福退，退像没查着。查不着不要紧，席严，坛不露。

丑时，练鼻下静。粟壳香淡，香一字不落。

他望坛，低声道：「十五实了。记着。」

坛腹温，贴掌，像应「层」——层在，路在，就不滚。

更鼓沉，他按心口：「汤还是热的。」

六字落，像路标——标向142谢老耿，标向省亲，标向叶汤。

外门风紧，紧像139钱戾衡影将近。近不要紧，手稳，就不飘。

内视里阶影稳在十五，稳像弓拉满——拉满不要紧，箭在142射。

管事弟子再登记，册上添一行：炼气十五。十五，不是喜，是实。

编筐棚里，末藤硬，硬像冬里规矩。规矩在，手才不飘。

铁无言送炭渣，渣细，屑亮：「层高了，别骄。骄了，手飘。」

「不骄。」韩泥说，「骄，像滚。」

白织月远廊过，没停。没停，小胜——胜在层实，不在辩。

「「」韩泥点头。142在前。记着。」坛腹温，贴掌，像应「实」——实了，势才够。验息册上，十五层那一行，字丑，真在。真，才像积势。

药峰活照旧，照旧才像140前的实——实不够，142谢不出。

编筐钱买米，米在，恩才还得上。还了，才像路。

赖福查夜，火折照角，照见厚席不见坛。坛不露，沿才净。

他低声对坛：「十五实了。记着。」坛腹温，贴掌。

邻床悄问：「层高了，怕不怕？」「怕。」韩泥说，「怕也守。」

炼气十五，不是雷，是一线实——实从十四里长，长像药峰分拣里多走半刻鼻下静。静到只闻辛淡，不闻辱；辱在耳，不在鼻；鼻准，142才谢老耿。老耿在142，142在远；远不要紧，136只实层。实层，管事验息，册上添一行，行丑，字真。真，才像积势；积势不在快，在稳。

稳了，137坊市续才听得真；真里，139钱戾衡影才接得住。接在台，不在坊；坊尾骂，骂不要紧，手稳，就不滚。滚了，130刘婆白还；还了，136才配实十五。十五像弓满——满不要紧，箭在142，不在今日喜。喜了，手飘；飘了，丹不正。正，是五年淘的，不是层赏的。

赏在远，在大比，在175筑基丹；136只积势，不越界。沿净，坛才藏得住；藏得住，137才续坊市。续坊市，听风紧，风紧不要紧，手稳，渣就正。正了，140收束才足。足在远，136只实层。更鼓沉，他望坛，低声道：「十五实了。记着。」坛腹温，贴掌，像应「实」——实了，势才够，够才配称第二部在走。

层十五实，实像弓满——满不要紧，箭在142，不在今日喜。

验息再记，十五层那一行，字丑，真在。真了，136才像实，不像虚。

更鼓沉，他望坛：「十五实了。记着。」

「铁无言送栏灰，灰细：「层高了，别飘。」「不飘。」韩泥应声。常活里藏实层，实层里藏142——142在远，136只实。实了，137坊市才续，138守坛才静，139风才紧，140势才足。记着。」

管事记簿，记细，是小胜。胜在忍，不在辩。

他望坛，低声道：「手在，路在。」坛不应字，只温。

更鼓沉，天边白一线。白里，他按木牌，牌凉贴胸。

坊市尾巷风硬，他收摊如石，石在，就不滚。

验息后，他理怀内册页，页黄，字丑，丑而真。

铁无言声低：「别飘。」韩泥应声：「不飘。」

刘婆端粥，仍多半勺。韩泥点头，谢轻，恩重。

药峰分拣照旧，指稳，鼻下静一息。

帖在怀，怀沉，沉的是序，不是惧。

白织月袖风过，韩泥不辩。辩，费舌；舌要留来守坛。

赖福廊下过，韩泥只答：「在。」在，就不滚。

柱上条角白一线，三年约在怀。怀凉，掌温。

坡下石阶湿，他数息，数帖，数债序。

同棚杂役笑，笑在耳，不在心。

废渣巷日斜，他验渣，渣正，手才静。

侧门驿亭风硬，他听风，风过，记在心里。

散队后，他过外堂阶下，分拣半刻，指稳——指稳，路就不飘。

沿净，坛才藏得住。
