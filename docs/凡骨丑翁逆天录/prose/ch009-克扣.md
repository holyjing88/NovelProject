# 第九章 克扣

馒头香未散，村头告示桩贴新纸。墨黑，韩泥独眼辨出「赖」字，像刀口向下。纸新，浆糊味冲鼻，冲得像血腥味。周嫂念给旁人听，念得高兴，高兴里带颤：

「沈家药账、芽儿罚银、帮晦气翁者连坐！」

村人围一圈，圈心空给告示，也空给晦气翁。韩泥站圈外，肩水晃，不近桩——近了就成靶。人群让缝，缝够过，没人碰他，像碰了沾墨。有人低声：「赖管事下手了。」

另一人接：「沈家这回要惨。」

韩泥不答，只把水担直。担直了，路还能走。水在桶里晃，晃出细声，细声像提醒：路还在，井还在，活还在。周嫂念完，眼往他这边一扫，扫得像刀背：「看见没？帮晦气翁的，连坐！」

韩泥独眼平：「坐我。莫坐旁人。」

周嫂一噎，没再接，像没料到他会回。回得平，像石头落地。旁人嘀咕：「晦气翁还真嘴硬。」

韩泥担水走过，只一句：「嘴硬不硬，水还打。」
沈家院墙内咳。咳得急，像娘。韩泥不进门，把昨夜留的半馒掰碎，包布，搁豁口，放下便走。墙内有人声，极轻：「……谁……」

韩泥不答。答了，连坐又多一口。墙内静一息，像有人拾了，仍没声。没声好，声会惹祸。赖扒皮的告示，是仇苗。苗不今日拔，先记胸口，刺名：赖扒皮。回屋，心口四恩在，赵辱在。告示不划——势无血，记心。「克扣，记下了。」

他低声又道：「赖扒皮，记着。芽儿，也记着。」

胸口像多了一根刺，刺不拔，先藏着。藏着不是忘，是等。等境到，等刀快，等还恩的人还清了，再拔。他低声自问：「告示能贴多久？」

自答：「贴多久，记多久。」
午后，集尾来人收摊费，多收三文，笑：「晦气翁，赖管事打过招呼，你的筐，税重。」

韩泥摸出十二文，数十文，剩两文，搁桌上：「税重，我认。筐也重，你拿不动。」

来人掂筐，沉，沉得脸变：「你……」

「十文。」韩泥说，「多一文，没有。」

来人嘟囔：「赖管事面前，你还嘴硬？」

「硬不硬，筐硬。」韩泥指节敲筐沿，敲出一声闷响，「拿不动，就别多收。」

来人收了十文，走了——欺负软柿子，碰着硬藤，少咬两口，也是小胜。韩泥不追骂。追骂像少年，他不是。两文搁桌上，像明日买盐的命。来人走远，还回头嚷一句：「赖管事不饶你！」

韩泥关门，低声：「不饶，就记着。」
傍晚，周癞子醉砸门：「韩泥！赖管事说，你晦气管够！」

门柱震，震得坛边布包挪半寸。韩泥开门一尺，递去半碗浑水：「管够，喝一口，醒醒。」

周癞子愣，骂半句：「你……老不死的……」

「醒醒。」韩泥又说一遍，「醒不醒，在你。水在这。」

水也不接，晃远。晃得鞋跟打门墩，打出一声闷响，像碰了一鼻子灰。韩泥关门，掌按心口。恩四，辱一，仇苗一。「忍。」

一字。忍不是怂，是刀还没磨快；磨刀靠境，境靠熬，熬他五十八年最熟。外头周癞子步远，步乱，骂声碎：「老不死的……晦气……」

碎了好，碎得像碰了灰，灰落在他脸上，不在韩泥脸上。夜里雨来，雨打茅草，沙沙响。坛沿微温，像吸了雨。韩泥把袄贴腹，腹微热，热得像在数恩——数到四，停，不急，急乱心记。他低声道：「赖扒皮，记着。沈家芽儿，也记着。」

前一句是仇，后一句是恩。恩仇分列，不混。混了，韩泥就不配记恩。告示桩前，风撕纸角，撕下半指宽，没人补贴——贴也贴不住风。韩泥路过，不近，独眼瞥见「连坐」二字，二字像枷。枷不在他颈，在芽儿、在叶家、在所有敢对他好的人身上。所以他更不能不记恩。记恩，才配有一天拔仇苗。雨歇时，村静，静得听见自己的呼吸。

呼吸稳，稳得像还撑得住。桌上两文铜钱，铜色暗，暗得像命还薄，薄也得攥。他望一眼门外告示方向，墨字看不见，「赖」字却在眼里。眼里有字，心里有苗。苗小，真；真苗，早晚要拔。韩泥按心口，低声道：「忍。记着。恩仇分列。」

雨又落一阵，落得更密，密得像要把告示字洗掉。洗不掉，字在桩上，苗在心里。韩泥望雨，低声：「克扣记下了。赖扒皮，记着。」

告示角被风掀起半指，半指里「连坐」二字黑白分明。韩泥不撕，撕了更祸；不近，也是护——护芽儿，护叶家，护所有敢对他好的人。

雨声里，沈家方向又咳一声。咳轻，像芽儿。韩泥不往那边走，走便是连坐。不连坐，也是护。

里正巡夜路过，敲锣一声，哑：「告示贴着，谁再帮晦气翁，记名！」

韩泥开门一尺：「记我。莫记旁人。」

里正愣，锣槌悬着：「你倒认。」

「认。」韩泥说，「恩我收了，罚我认。旁人莫连坐。」

里正哼一声，走了。四恩在，仇苗在。苗不拔，先记。记清了，才配活。坛沿微温，瓮没醒，温一线，够撑一夜。

次日，告示桩前仍围人。周嫂念给新来的听：「沈家药账、芽儿罚银、帮晦气翁者连坐！」

韩泥担筐路过，不近桩，只一句：「连坐坐我。莫坐娃，莫坐叶家。」

有人嘀咕：「晦气翁嘴硬，硬给谁看？」

韩泥独眼平：「给自己看。看得清，才活得久。」

沈家墙豁口，芽儿声碎：「韩爷爷……我娘咳……」

韩泥停半息，不近：「咳，翁记着。药账是仇，粥是恩。」

墙内静。静里像娘咳两声，咳得像还活着。韩泥走开，走开也是护——护，不是怂，是等刀快。

集尾收摊，税吏来收，伸手多要三文。韩泥把十文摆齐，摆得一字排开：「说好了十文。多一文，没有。」

税吏眼一斜：「赖管事打过招呼，晦气翁税重。」

「税重，筐也重。」韩泥指节敲筐沿，闷响实，「你拿不动，就别多咬。」

税吏掂筐，沉，脸变，嘟囔着只收十文——少收三文，也是小胜。胜在钱还在掌心，不在赖扒皮嘴里。

傍晚，周癞子醉撞门柱，灰蹭鼻，退半步，骂声碎在风里。碰灰退半步，也是小胜——胜在灰落他鼻，不落韩泥名。

里正巡夜又咳，咳过告示桩：「再帮晦气翁连坐，记名报县。」咳声沉，沉得像给全村缝一道边——边内过日，边外惹祸。

告示角冷，他读条三遍，条字像赖扒皮的笑。笑在角，不在心——心在，铺就不关，汤就能还热。他收好腰布，明日仍去集尾，活先，锤后，锤在公审。

回屋摸坛沿，沿温不退。不退，像瓮替他守着「少收三文」——守一文是一文，守到公审那日。

丑时雨歇，井台空，空得像全村嘴也歇了半寸。他低声道：「克扣记下了。赖扒皮，记着。恩四，辱一，仇苗一——分列，不混。」

桌上两文铜钱，铜色暗，暗得像命还薄，薄也得攥。攥住了，明日还去集尾卖筐——筐硬，税重，人更重。少收三文那声闷响，闷在掌心，不在舌上。仇苗在，恩也在——在，就还得活，活到拔苗那日。

赵绣春隔窗咳一声，咳得像替税吏补半句：「赖扒皮，记着。」韩泥不接，只把两文并十文，并得齐，齐得像分列——恩四，辱一，仇苗一，各归各位。分列清了，心就不乱；心不乱，公审锤落，也只一句：我记着。忍。明日仍去集尾，筐硬，税重，人更重。坛沿温一线，应钱不应谣——应着，少收三文那笔小胜才钉实。腰布再数一遍，数十文，文文落掌。稳。等锤。
