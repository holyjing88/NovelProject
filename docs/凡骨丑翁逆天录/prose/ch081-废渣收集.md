# 第八十一章 废渣收集

四层可见后，药山差事改了字。柱上条新贴：「丙九韩泥，废渣收集，日半。只收渣，不采药。私藏活药，除名。」

暗袋七包，包丑，包实，塞床角。塞住了，才不示人；示人，谣又来。谣不要紧，晦气遮瓮——遮住了，收集才稳。

坡腰更腐，腐沉底，底真。真在底，不在浮；浮的少，少也要记。记错了，废渣收集白来，废窑也不收。

铁无言指廊角废窑方向：「窑在远，渣在先。先攒够了，窑才开得。」窑才开得，82废渣路才走得通，83才识温听雨。

只收渣，韩泥熟。熟不等于轻——轻了，手就飘；飘了，诀白芽。卯时，边门开。他背两只筐，筐旧，底垫粗布，布丑，不漏。不漏，是小胜；胜在筐，不在嘴。药山三处渣场：坡下、坡腰、深堆边。三处轮转，轮转不要紧，鼻下有根才要紧。坡下灰黑，黑里辛淡。他鼻下一静，沉腐浮三字不落舌，指稳，一撮入筐。

入筐前再辨：辛沉，腐浮，假香剔净。真在底，不在浮；浮的少，少也要记——记错了，废渣收集白来。铁无言立坡角，声低：「收集别混。混了，废窑不收。」

废窑二字落耳，像钩。钩不要紧，先收——收齐了，路才宽。韩泥应声：「三堆，不混。」

午间，管事验筐，倒秤，半斤对册。对，记「丙九韩泥，收净」。记净，是小胜。赖福立秤边，眼细：「收渣也收？收出私藏，连坐丙九。」

韩泥不接茬。接茬，全丙九饿。他低声道：「只渣，无活药。」

管事冷声：「验过了。赖福，少嗓，多秤。」

赖福脸一青，青完，眼却扫韩泥袖——袖里无香，无烟，像私藏从未有过。扫完，笑更薄。午后，回棚。他把白日渣分七纸包，包丑，包实，塞床角暗袋。暗袋旧，是编筐时留的夹层——夹层不要紧，不露才要紧。不露，像坛；坛不露，晦气遮瓮，遮住了，收集才稳。邻床问：「你床角啥味儿？」

「渣味。」韩泥声平，「药山活，活有味。味在角，不在席。」

少年掩鼻，没再问。问，像沾晦气。他按席角，席严，坛不露。坛腹温一线，贴掌，像应「收」字。应完，他数纸包——七包，包轻，记重。七包不要紧，七十包才要紧。七十包在远，远像95匿丹；远不要紧，先攒。刘婆提篮过，篮里饼硬：「收渣饿了？」

「饿了。」韩泥说，「饼就行。」

刘婆塞饼，饼糙，糙得掌印深：「别谢。记着。私藏别沾，沾了婆也救不了你。」

韩泥点头。谢轻，恩重。恩在饼里，也在「别沾」二字里——二字像铁无言，像界碑，像沿净规。黄昏，药渣场角。他多留半刻，把七包样再辨一遍。辨完，并入暗袋。袋沉半分，沉不要紧，沉才像路。

铁无言折返，声低：「袋别示人。示人，赖福记过。」

韩泥应声：「不示。记着。」

铁无言又指廊角废窑方向：「窑在远，渣在先。」

窑才开得，韩泥不追问。追问像求；不求，只记。记废窑在远，记渣袋在角，记收集是路。白织月主道过，袖风带香，香冷：「杂役也收渣？收了，也炼不出丹。」

韩泥不辩。辩，费舌；舌要留来守袋，留来守坛。他侧身，让半尺，让得像避光。白织月鼻下哼，哼完，走了。少一句嘲，小胜。丑时前，他廊下练半刻闭息。息在鼻，不在嗓。嗓响，香飘；香飘，袋就藏不住。掌纹那路转一周，二层稳，远灯不熄。不熄，像催攒渣——催不要紧，脚仍稳；稳了，袋才不漏。秦霜木鱼轻敲：「藏别藏邪。邪，鼻下香飘。」

韩泥不答，答在心里——心里恩辱分列。收是活，活是路；路通向废渣路，通向温听雨初识。他独眼平，手不抖：「收了七包。袋在角，不示。废渣路在明日。」

韩泥只答：「不示。记着。」

邻床翻身，嘟囔：「啥味儿……像药……」

「渣味。」韩泥说，「睡吧。」

坛腹应温，不应字。温，像七包渣里那一点辛，辛在活里。活在，才收得长——收长了，废渣路才走得通。他低声对坛：「渣收了。袋在角。坛在，人在。人在，就不滚。」

韩泥只答：「渣味。沿净。」

在，就不滚。不滚，袋才攒得长——攒长了，才开得了废窑，才接得住丹路开。柱上条角白一线，三年约在怀。怀凉，袋凉，掌里远灯不熄——灯在，路就不绝。他低声道：「废渣收集了。舌不留，手要稳。稳了，路才宽。」

路才宽，在明日；宽不要紧，袋在角，角在，就不滚。收渣第二日，斗车辙深，深里积水冻一线。韩泥足底气沉，沉得稳，稳像踹台按实。第一斗满，过磅，记册。第二斗满，再记。记功轻，路重。管事弟子验巷尽，验足迹不扒砖，记「收净」。记净，是小胜。铁无言声低：「三斗别示人。示人，谣来。谣来，煞骨门嗅。角柜锁。」

刘婆提水过，水洒稳：「废渣臭，别空肚。」

韩泥喝一口，水凉，凉不要紧；凉也能静鼻。白织月丹房门开一线，袖风冷，没多言。没多言，小胜。赖福立巷口，眼细：「三斗也收？收出丹味。」

韩泥低声道：「只收，不炼。炼字在条上，不在掌上。」

午后回棚，四包样变六包，包实，锁柜。锁住了，才不示人。邻床问：「三斗能炼丹？」

「能。」韩泥说，「不在今日。今日只收。」

丑时前，廊下练半刻鼻下静。六包并息，并出焦回一线。路开了半步。废渣巷续，不叫。温，像焦回一线摁进影里。更鼓沉，独眼平：「收集了。废渣路尽在后，温氏初识在后。」

更鼓沉，沉里，韩泥按木牌，牌凉，掌温。

收渣第二日，坡腰斗车辙深，深里积水冻一线。记功轻，路重。

记净，是小胜。铁无言声低：「袋别示人。示人，谣来。」

「」没多言，小胜。赖福立巷口，眼细：「两斗也收？收出丹味。」午后回棚，七包变九包，包实，锁角柜。邻床问：「九包能炼丹？」「能。今日只收。」九包并息，并出焦回一线。路开了半步。

「「「邪，鼻下香飘。」废渣路在明日，温氏初识在远廊。」温，像九包渣里那一点辛，辛在活里。活在，才收得长。袋在角。坛在，人在。人在，就不滚。」廊下风硬，他把木牌系紧一线，系得像把仍末摁进掌心。同批少年远避，避像避晦气，晦气遮瓮，遮住了才稳。

编筐角藤刺仍在，他偏手，刺浅些，沿才净，坛才藏得住。

管事夜巡验棚，灯照席角，沿净，像腌菜坛，不像邪。

老耿挑水过，声哑，境别拜邪，拜了路断，他只记着。

秦霜木鱼远，一声，心别邪，邪了鼻下香飘，香飘手就滚。

柱上条角白一线，三年约在怀，愧字在怀，手仍不抖。

白织月远廊过，袖风冷，没停，胜在鼻，不在嘴。

铁无言脚步远，停半息，声低，别漏香，别叫境，只记。

刘婆粥稀，勺沉，饭稳恩不断，他喝尽，不谢，只记。

收渣第三日，斗车进巷，车辙轧霜。轧不要紧，足底二层气沉，沉得稳。稳像踹台按实，稳像影里第四阶在等。

管事验三斗，对册，记「收净」。记净，是小胜。刘婆提水过，水洒稳，嘱别空肚。他喝一口，凉也能静鼻。

午后角柜六包，包实，锁柜。示人，谣又来；谣不要紧，晦气遮瓮。

铁无言声低：「收长了，废渣路才走得通。路通，温氏才识，嗅药才小成。」韩泥不答，答在心里——心里记三斗，记焦回，记不叫。

一线轻，记重；记重了，才接得住废渣路尽那一日。

废渣十包，包实，锁角柜。柜丑，锁像规——锁住了，才不示人。示人，赖福记过；记过，恩断半寸。半寸不要紧，渣在，丹路才开；丹路开，温听雨才验手，95才匿赠。

「「记着。」老耿挑水路过，水洒稳，声哑：「袋别当宝示人。示，谣来。」示，谣来。」角柜锁丑，锁像规——备齐了，别炼；炼在远，手在近，瓮温一线应「备」。
