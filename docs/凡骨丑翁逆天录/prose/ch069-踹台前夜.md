# 第六十九章 踹台前夜

正测后第三日，柱上又贴条。条新，字利，像刀背刮纸：

踹台前夜，管事又补一条细规：末席验手，须赤掌按台，不许戴布。布厚，手抖藏不住；藏不住，滚字就真。韩泥解腕布，布旧，布边刘婆补过一线。线丑，线暖——暖在肘，不在台。台凉，掌按实，实了，明日才不抖。

「明日卯时，踹台验末。凡骨杂役韩泥，必到。不到，除名。」

韩泥读条，独眼平。除名他熟，熟不等于认——认命是滚，不认是手在。手在，就不滚。

卯时前，丙九列队。管事弟子敲柱两声，廊静。他念条，声沉：「踹台专验末席手稳。手抖，除名；除名，麸皮停，记功抹。」

末排少年脸白，脸白不要紧，韩泥只按木牌——牌在，名在，路在。赖福晃木牌，牌旧，笑薄：「踹台专踢末席。踢完，你滚，丙九脸也干净。」

韩泥不接茬。接茬，全丙九饿。他摸怀木牌，牌凉，膝热——热在坛，热在路，热在钱戾衡那一句「记住今日」。

同批少年悄问：「你真去？」

「去。」韩泥说，「不去，除名。除名，恩断。」

少年不懂「恩断」，却懂「除名」二字重。重，像麸皮停，像记功抹，像叶丫头那碗烫还不到手。

散队后，他过坡下测灵石。石凉，线灰，灰在众目下公示过——公示不要紧，手要紧。手抖，滚字就真；不抖，末排也能活。活长了，才还得了七笔恩。

韩泥只答：「手在。」

午后，兽栏仍腥。腥不要紧，手稳才要紧。他铲下稳，稳得像坡下按石那一掌——掌在，名在，路在。小兽角里咳一声，咳轻，他换干草，草干，味淡一寸。管事路过，记一笔：「丙九韩泥，踹台前仍清细。」

刘婆端粥，仍多半勺：「踹台？」

「踹台。」韩泥说，「末席，必到。」

「末席怕什么。」刘婆哼，「手稳，饭稳。饭稳，恩不断。」

韩泥点头。谢轻，恩重。恩在粥里，也在忍里——忍里藏着刃，刃对着钱戾衡，对着赖福，对着内门的眼。刃不亮，先磨；磨在丙九，亮在还恩那日。

药山远路，他担废渣过侧阶。阶上白织月立，衣白，声细：「丑杂役，踹台也配去？配去，别脏台沿。」

韩泥不辩，肩稳，渣不漏。漏了，赖扒皮记过；记过，恩也断半寸。白织月鼻下哼，哼完，袖风过——风过不要紧，手在，就不滚。

沈枯芽在药圃角弯腰，弯得像没看见。没看见不要紧，韩泥也不近——近，像求；不求，只记。记沈枯芽半块灵石那笔恩，恩在还，不在这一日。

丹房外，铁无言抱臂立侧，声低：「灵根末，手稳，渣场仍收你。别漏。」

韩泥应声：「不漏。」

铁无言又指坡：「踹台看手，不看嘴。嘴响，手飘；手飘，滚。」

「手记着七笔，不飘。」韩泥说。

秦霜远廊经过，木鱼不敲：「心别邪。邪了，鼻下香飘。」

韩泥不答，答在心里——心里七笔分列，不混。泥岗六笔沉，宗门刘婆一笔烫——烫在肘，烫在粥，烫在今日这口辱。辱在众目，手不能抖；抖了，滚字就真。

廊下有人练剑，剑光两闪，闪完，一句笑远：「丑杂役，明日踹台见。」

另一内门弟子接话，声更薄：「听说钱师兄要掷玉牌——玉牌？三年约那种？」

笑又起，起得像潮。潮不要紧，韩泥不接笑——接笑，像认晦气；认不得，只记。记仇苗，不在册，在心——心在坛，坛腹温一线，温不应辱，应「忍」。

邻床问：「踹台怕不怕？」

「怕。」韩泥说，「怕恩断。恩不断，怕也要去——去，手才不抖。」

少年不懂「恩断」，却懂「怕也记」，闭嘴睡了。睡了，棚里更静；静里，更鼓沉，沉得像踹台在前头。

管事夜巡验棚，灯照坛角，照见净沿不见宝。不见不要紧，沿要净。韩泥不辩，只擦沿——擦沿，像守瓮人。

小杂役蹭他袖，声更小：「听说玉牌是契。契接了，活不过三年。」

「活不过，也要接。」韩泥说，「不接，除名。除名，叶丫头那碗烫就还不了。」

小杂役不懂「烫」，却懂「还不了」三字重，收声走了。走了不要紧，玉牌在远，手在近——近，才验得过踹台。

外头风硬，柱上条角又白一线。白，像仇苗出土，根扎坡下，拔不出。拔不出不要紧，先记——记钱戾衡，记「记住今日」，记灰线三尺公示在众目下的那一瞬。

他按席角，席严，坛不露。坛腹温一线，贴掌，像爹的手又搭上来。爹说过：「坛别扔。」扔不得，扔了就无根；无根，辱就白挨。

丑时末，他鼻下再静半刻。粟壳香里多一丝药苦，苦像药山渣——渣不要紧，鼻下稳才要紧。稳了，明日卯时手才不抖；抖了，滚字就真。

赖福脚步在廊外停半息：「丑杂役，我盯着你。盯出邪，连坐丙九。」

韩泥只答：「在。」

在，就不滚。不滚，仇苗才扎得实——扎实了，才接得住明日踹台，接得住玉牌，接得住三年约那一踹。

更鼓将尽，天边白一线。白里，内门廊下有人影立，立得像等卯时。影瘦，袖线细，细里带金边——像钱戾衡。

韩泥不近。近，像求；不求，只记。记完了，才配等踹台——台在远，仇苗已在心；心在，就不滚。

戌时，他编筐末单，藤刺扎掌，偏手，刺浅些。浅些，茧仍裂，裂口潮，潮不是血——血要等踹台那一脚，脚在沿前，不在舌。

编筐钱买米，米在，恩才还得上。还得上，才对得起刘婆多半勺，才对得起叶丫头那碗烫还在还路前头。

管事弟子又贴细条：「踹台只验手。手抖，除名；除名，连坐丙九。」

同批少年脸更白。韩泥独眼平，只按木牌——牌在，名在，路在。连坐不要紧，手在就不连坐。

沈枯芽在药圃角收工，怯，路过坡下时停半息，停，像想问，又不敢问。不敢不要紧，韩泥也不近——近，像求；不求，只记。记沈枯芽半块灵石那笔恩，恩在还，不在踹台这一日。

沈枯芽低声：「韩……韩叔，明日……手别抖。」

韩泥只答：「记着。」

就二字，无「怕」，无「谢」。谢轻，恩重；重不在舌，在手——手抖，滚字就真；不抖，末排也能活。活长了，才还得了芽儿那半块石。

戌时末，他回棚前过粪渠，渠冰裂一线，裂不要紧，裂像仇线——仇线浅，浅在钱戾衡名，浅在「记住今日」四字。四字烫，烫在还路前头，不在这一夜。

他蹲渠边，冷水洗掌，掌茧裂处刺疼，疼不露。洗完，掌按席沿，沿净，像预演明日按台——按实，实了，脚落也不抖。

邻值少年问：「你真不躲？」

「躲，除名。」韩泥说，「除名，恩断。恩不断，就不躲。」

少年不懂「恩断」，却懂「不躲」，闭嘴了。闭嘴了，棚里更静；静里，坛腹温一线，温不应慌，应「忍」。

更鼓二响，响沉。韩泥合眼不睡实，怀三字并怀：三年约、仍末、踹台。仍末不要紧，手在；手在，就不滚。

管事又嘱半句：「踹台只验手。」韩泥只答：「手记着七笔，不飘。」

散场后，药渣场角风硬。他多留半刻，分拣一撮灰，灰里一点辛。辛正，他鼻下静，静里辨出苦与辛——苦沉，辛浮。样纸包实，塞怀，怀暖半分，暖不应符，应「忍」。

编筐末单，藤刺扎掌，偏手，刺浅些。浅些，沿才净；净了，坛才藏得住。编筐钱买米，米在，恩才还得上——还得上，才对得起刘婆多半勺，才对得起叶丫头那碗烫还在还路前头。

他低声道：「踹台前夜。玉牌在远。仇苗在近。近，才磨得亮。」

刃不亮，先记。记，是丑翁的本事。记完了，才配接玉牌那一夜——牌在远，仇苗在近。近，才磨得亮。亮在还恩那日，不在舌上。踹台在前头，玉牌在更前头。
