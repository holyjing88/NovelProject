# 第六章 稀粥

谣起第三日。

门墩多了一只粗碗。不是叶家那只——沿口缺在不同处，缺得像月牙，缺在碗东。碗里稀粥，稀得照见底，米粒几粒，浮着，像可怜人的良心：少，却有。粥面凝一层薄皮，揭起来，底下更稀，稀得像娃的眼泪。

碗底蹭着门墩石，蹭出一道湿印，印浅，像有人放下时手在抖。

韩泥开门，门外无人。泥地小脚印浅、乱，像娃的，鞋尖还沾着沈家院墙根的青苔。沈枯芽。村里喊「芽儿」。怯，瘦，爹早死，娘病。送粥像做贼——在泥岗村，对晦气翁好，就是贼。

他喝粥。淡，腥，米少，胃却暖一线。暖细，比姜汤弱，更难得——姜是叶的胆，粥是娃的胆。

他喝完，舌尖舔碗沿，舔到一粒米，含住，嚼了，嚼得慢，像嚼恩。粥腥里有灶烟味，烟味淡，淡得像沈家灶小、柴少、火怯。碗洗尽，搁回门墩。洗时水凉，指节红，红得像记。

天未亮透，他开门再看门墩。脚印边多一粒米，米粘泥，像娃慌时掉的。韩泥拾起，含进嘴里，嚼了：「芽儿，翁看见了。」

晌午，村道轰了。

马春花嗓尖，尖得像剪子：「沈家芽儿学谁不好！往晦气翁门缝塞粥，晦气沾上了，你娘药钱谁出！」

沈枯芽哭，闷在喉：「我……我没有……」

「没有？碗人赃并获！」周嫂帮腔，棒槌还攥着，「赖管事说了，沈家欠账，再牵扯，连坐全族！」

「碗在门墩，粥在我肚。」韩泥在屋里，声不高，「要罚，罚我。」

「你倒敢认！」马春花更尖，「认了就坐实，沈家跑不了！」

「跑不了，我认。」韩泥声仍平，「粥在我肚，不在芽儿手里。」

「咳？咳就是晦气！」马春花接话，「你爹早死，你娘病，你再沾晦气翁，沈家绝不绝？」

赖扒皮。沉丹宗杂役管事，手狠。芽儿一碗粥，惹了眼。哭声远，像被拖进院。韩泥没出门——出门，坐实「牵扯」；不出门，粥已入腹，恩已入手。

屋里静，静得听见漏雨。雨没下，是屋顶存的水滴，一滴，又一滴，滴在坛沿，滴得像计时。

他低声道：「记着。粥稀，情不稀。」

午后，赖扒皮的人路过泥屋，一脚踹在门墩上，墩响，木屑飞，嚷：「晦气翁，再收沈家东西，罚银加倍！」

韩泥开门，独眼平：「我收的是粥，不是沈家。粥已入腹，你要罚，罚我。」

来人愣半息，上下打量他，像打量一块啃不动的骨头：「你倒硬。」

「硬不硬，粥已硬进肚。」韩泥说，「回去告诉赖管事，要连坐，坐我。莫坐娃。」

来人靴跟又碾门墩，碾出一声闷响：「罚银呢？别装死！」

「有。」韩泥拍腰间空袋，「命在，银慢慢凑。你要现拿，拿不走。」

来人骂一句「老不死的」，走了——赖扒皮要的是怕，韩泥不怕，骂便浅。这是暗爽：仇苗种下，苗前，他先赢一口「不躲」。

他蹲下去，把门墩木屑一点点拢回缝边。拢不好看，却像把娃的脚印护住半寸——护半寸，够记。记不写在纸上，写在指腹：指腹按过缺沿，缺还温着昨夜粥气。

门墩木屑还沾着，沾着像那脚没白踹——踹不散粥恩，踹不散娃胆。

傍晚，沈家墙豁口搁小布包，里半块饼，捏实，捏得指印还在。韩泥不敲门，放下就走。墙内静一息，像有人拾起，没谢——谢字会惹祸。

「芽儿。」他低声，只对自己说，「别谢。谢了，马春花又有一嘴。」

「翁不怂。你送粥，我记粥。」

墙内咳一声，轻，像娘。韩泥脚步没停，停便是认。

夜里，心口又添一粒——叶汤、赵辱、芽粥，恩辱分列。外头童谣丫不念了，许冷，许被娘拽回。村静，静得听见沈家咳，咳两声，像活着的证明。

他摸坛沿，坛腹微温一丝，像从地底爬上来的。塞布，塞不紧，温还在。

他想起芽儿哭腔，闷在喉里，像怕。怕也要送，送一粒米，也是恩。恩小，真；真恩，韩泥认。

「娃胆大，翁不能怂。」他对自己说。怂了，对不起那碗稀粥。

门墩上粗碗还在，沿口那月牙缺，缺得像娃留的印。印不深，真。

他低声道：「罚我，莫罚娃。」

这话不说给赖扒皮听，说给心听。恩在心里，芽儿那粒米也在，像一粒胆。

次日，门墩又空。空了一夜，韩泥仍洗坛沿，仍编筐。沈家墙豁口传来咳，咳得急。韩泥不近，只把半块饼掰实，搁豁口，低声：「芽儿，别送粥了。送一次，罚一次。」

墙内静一息，娃声碎：「我娘还咳……」

「咳，翁记着。」韩泥说，「粥稀，情不稀。情记着，比再送安全。」

墙内「嗯」一声，像米粒落泥。落定了，韩泥才回屋，坛腹微温一丝，温得像应那句「情不稀」。瓮没醒，温在，够记。

稀粥入腹，芽儿远远看一眼，像一粒热姜落回汤里。韩泥不招手，招手像讨。讨轻，恩重。

门关上，风从门缝钻，钻得膝上凉。凉不怕，怕的是芽儿那声哭，哭在喉里，像怕连累娘。

晌午那阵，马春花嗓又尖了一回，尖完，反而少了看客——看客腻，腻在晦气翁不炸，戏文重复，重复便无趣。无趣，也是韩泥的活法：不炸，活；活，记；记，候还。

赖扒皮告示在村头仍贴着，贴着像枷——枷不在他颈，在敢对他好的人身上。所以他更不能怂，更不能忘芽儿那粒米。

村头告示角卷了，卷着像赖扒皮的手还搁在村里——搁着不要紧，韩泥只认门墩上那道月牙缺。缺在，娃就来过；来过，恩就真。真恩不扩声，扩声招马春花，招连坐。

门外风硬，硬里仍候下一碗「趁热」。叶家秤还在，娃娘药才有着落。有着落，芽儿胆就不白送。

他合眼，合得浅，浅梦里只有门墩月牙缺，和娃那声闷在喉里的哭。

丑时末，村犬哑叫一声，像谣也哑半寸。坛腹微温，应「情不稀」。冬前村会那盏灯，影先到——影到不要紧，要紧是芽粥记实了。

他低声道：「粥稀，恩不稀。稀粥顶一口，够活到还。」

门墩木屑未扫，稀粥温还在胃。心记只留沈家豁口那粒米——米小，真，真就不白收。赖扒皮要的是怕，韩泥不怕——不怕，才配记恩，才配候还。

娃又补一句，声更碎：「韩爷爷……你别饿。饿死了，谁还我娘？」

韩泥愣半息，独眼平：「翁不饿。你娘咳，叶家秤在。秤在，药才治。你别送粥了，送一次，罚一次。」

墙内应一声极轻，轻得像怕惊动巷口。韩泥才回屋，按坛沿，沿温一丝，应娃胆，不应罚银。

明日他还得编筐、卖药、打水——活长了，才还得了叶丫头那碗烫，才护得住芽儿那粒胆。仇苗在，恩也在；在，就不散。

编筐手不能抖，抖了，稀粥恩就还寒酸。瓮未醒，温在，够记——情记着，比再送粥安全；安全了，沈家才少挨罚，芽儿才少哭一声。

天将亮，他再开门墩看一眼。粗碗还在，月牙缺未变，变的是泥上脚印——脚印被风抹淡半寸，淡不要紧，缺还在，恩还在。还在，就够活到还。

他回屋按坛，沿温不退，退的是外头风。风硬，硬里仍候下一碗「趁热」——候着，就不怕赖扒皮那告示再贴一层。粥稀，恩不稀；恩在，路就不断。他摸门墩月牙缺，缺浅，真——真在，娃那粒胆就不白送。罚银加倍那声嚷，嚷进仇苗，不进心记。
