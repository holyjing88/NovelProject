# 第一百一十八章 还芽石

夜行外门杂院，韩泥揣着半块灵石。石旧，边角磨圆，圆得像泥岗罚银那夜——那夜芽儿送粥连坐，连坐因他。因他，就该他还。

杂院门松，松得凡役能过。他贴墙走，独眼辨窗：第三间灯亮，亮得发怯。怯不要紧，石该还。还了，第五笔徒恩落心里，不落榜；落榜，赖福又有由头。

他叩门，声低：「芽儿。」

门开一线，沈枯芽探头，眼亮又慌：「韩……韩爷爷？」

「是我。」韩泥把半块灵石托在掌心，「泥岗罚银那夜，你送粥被罚。罚在我身上。今日还你半块——旧账，清。」

芽儿退半步：「我……我不是要……」

「知道。」韩泥道，「你不要，我欠。欠着，丹不正。不正，我炼什么都味飘。」

内屋芽儿娘咳一声，声沙：「谁？」

「韩泥。」他仍站门外，「还沈枯芽半块灵石。泥岗旧账，今日清。」

娘沉默一息：「进来。」

屋窄，窄得像杂役心。韩泥不坐，立门边，把石放桌上。石轻响一下，像盖印。印盖了，心里空半寸——空得舒服；舒服不是无债，是债还了一笔。

芽儿娘瞥石，浊眼里清一线：「宗门里，你哪来灵石？」

「渣里淘，坊市捡，五年攒。」韩泥说，「攒来还恩，不攒来显摆。」

娘点头，点得像公审那日鞠半躬：「恩……我们收不得重……」

「半块，不重。」韩泥道，「重，我扛不动。半块刚好还。」

芽儿捏石，指尖红了：「韩爷爷，你……你成丹师了？」

「未成。」韩泥摇头，「成不成，恩先还。还了，才配炼；炼了，才配备战。」

芽儿抬头，怯笑浅，笑像一芽：「我记着粥。也记着狼口……」

「记着就好。」韩泥说，「记着，别学邪。邪了，鼻下香飘；香飘，路就歪。」

「我不学。」芽儿声小，却真。

娘低声：「荐帖那日，你走了。走了，芽儿仍怕连坐……」

「怕不要紧。」韩泥说，「怕，更要还。还了，你就不怕沾我。」

娘眼圈湿，没哭出声：「刘婆说，你手稳……」

「刘婆粥，恩另笔。」韩泥分得清，「芽儿石，徒恩。分开搁，别混。混了，叶丫头那碗汤就凉。」

芽儿把石攥紧，又松半寸，像怕攥碎：「那……你要喝茶吗？我娘有热水。」

「不必。」韩泥退半步，「喝了像讨。讨轻了，还就轻。我还完就走。」

出门，月白，白得像玉牌凉。凉不要紧，仇烫——烫在钱戾衡，烫在三年约。约还远，徒恩已还一笔；还了一笔，十层门槛开一线。开一线不声张；声张，赖福眼还盯着。

他回窑角盘坐。气冲关，炼气十层破。破也不叫。叫了，像破境告示；告示一出，赖福指「偷境」。管事看手不看腹——手稳，册在，就不滚。

铁无言立窑外，声低：「还了？」

「还了。」韩泥说，「还沈枯芽半块灵石，清了。」

「下一笔？」

「刘婆。复测后。」韩泥答，「序不乱。」

铁无言点头，点得像并肩过兽潮那日：「序不乱，丹才正。」

刘婆远处咳，咳轻，像听见：「还谁了？」

「芽儿。」韩泥抬声应答。

「还就好。」刘婆哼，「还了，饭更稳。饭稳，恩不断——你可别得了便宜乱响。」

「不响。」韩泥道，「静着还的。」

丑时他廊下练鼻下静。粟壳香淡，半块石那轻响才记进心里。记进心里，不进舌——舌响，白织月要说「施舍」。施舍不是。是还。还了，才配十层稳，才配复测备战。

他摸玉牌：「三年约，记着。芽儿还了，叶还在前头。」

叶在前头，省亲在前头，钱戾衡台也在前头。前头路长，徒恩落一笔，够撑十层实，够撑备战两月。两月后上台，眼风再来——来，不抖。

外头柱上新贴纸，纸边字淡：「外门复测，丹试报名。」报名不要紧，先十层实；实了再报；报了，备战才真。真备战，为夺魁，为还刘婆新衣养老钱——钱数得过三遍，不多不少。

沈枯芽窗灯灭了。灭得像睡。睡了，石多半在枕下，枕下像信——信他丑，也信他还。韩泥望一眼暗窗，低声道：「徒恩，记着。还了。」

回棚路上，同批有人截问：「夜行去哪？身上有味。」

「买藤。」韩泥随口应，「藤旧，编筐。」

「编筐还摸灵石味？」那人盯他袖。

「渣味。」韩泥摊掌给人看茧，「闻够了？够了让路。路堵着，卯时迟到，赖福又有牌晃。」

那人呸一声让开。韩泥袖里其实没有石——石已落桌上。落了，味也该淡；淡了，才像没夜行。没夜行的人，才活得过明天散队点名。

还石夜，靴底泥少。少得像没出过宗门。少泥不要紧，石已落芽儿掌心。掌心凉，恩却烫；烫在序里，等刘婆那笔接着。

他不回棚喧。喧像讨恩；讨轻了，还就轻。重，静着还，还到手才算数。算数了，心里那半寸空腾出来，正好装下一笔待还。

夜里再默念一遍泥岗徒恩。默念不落字纸——字纸只记饭，饭稀，恩不稀。稀粥顶一口饥，多活一日，多记一日。记满了就还：芽儿清了，刘婆待清，叶汤头恩还在省亲前。

邻床少年迷糊问：「你夜里又跑哪去了？」

「还债。」韩泥说，「债清半笔。睡。」

「还谁？」

「不该你问的。」韩泥道，「问多了，香飘。」

少年哼一声睡了。韩泥对坛：「还应芽儿，还应刘婆，还应叶汤。还应，路才正；路正，粒才正。」

坛腹温一线，应「还」。还应够走到十层稳，够走到复测风起。风起报名，试台在远，魁在更远——更远之前，先把十层按实，先把怀内培元散数清。

他低声道：「徒恩落笔了。不欠芽儿。手更稳——稳了，接明日报名，接钱戾衡影再来。」

影硬，手更硬。手硬，玉牌就不虚。虚了，三年约白接；白接，叶汤凉，芽儿石白还。白还不得。还了就要保得住。保得住，才配谈夺魁，才配谈把新衣递进刘婆手里。

更鼓沉。韩泥按木牌贴胸。十层气在腹里走周，周周实。实像石落桌上那一声轻响——响过了，心里空，路反而更实。实了，明日柱上那条报名的字，他才敢用丑字去押。

丑时末，铁无言又立窑外：「石递稳了？」

「稳。」韩泥说，「娘收了，芽儿捏了。捏红了指尖——红得像记得住。」

「记得住就行。」铁无言道，「你别回头讨谢。讨了，恩薄。」

「不讨。」韩泥应，「谢轻，恩重。重的在下一笔刘婆。」

「下一笔远。」铁无言瞥他，「远也别飘。十层刚破，破了就叫，叫了像求人看。」

「不叫。」韩泥摸坛沿，「沿净。净了，报名才不像花活。」

沿净。坛温。石已还。还字落地那夜，坡下风比白日软半分——软得像准他睡一刻。一刻够养手。手养实了，明日押字才不抖；不抖，复测风再起，也先让他立住十层这道坎。

他闭眼片刻，又睁开。睁开不是怕梦，是怕心里那声石响淡了。淡了像忘恩；忘不起。忘不起就再念一遍：「还沈枯芽半块灵石，泥岗旧账，今日清。」念完，指尖还留着石的凉。凉退了，恩留着。

邻棚有人梦呓骂晦气。骂声远，他不接。接了费气；气要留着养境，养到报名册前那一押。押上丑字，风再嘲也嘲不掉册上的印。

印在，路在；路在，刘婆新衣才不是空话，叶汤才不是远得看不见的热。热还在前头，石已还在今夜——今夜清了一笔，明日才配去押那条复测字。字丑不怕，怕手飘；手不飘，五年欠的首笔才算真落了地。落地了，坛才肯再温一线；温一线，明日报名那一步才踩得住。踩住了，徒恩才算真还成。

他独眼半阖，心里却醒着：下一秤不在石上，在报名册上——册上字丑，手也得稳。稳了，才接得住风声起。
