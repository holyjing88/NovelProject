# 第三十九章 体统

扑怀次日，丹堂外廊比平日更静。

静里却更闹——闹在弟子眼里，闹在窃语里，闹在「苏首座扑进莫长老怀里」这一句，比三品金丹传得还快。

苏念慈脸红仍强撑首座威严，弟子问丹样，她答得准，指节却白，白得像还攥着昨夜袖角温度。有人故意问：「首座，金丹成后，体统如何守？」

苏念慈冷声：「守体统，先守丹。丹在，堂在。」

弟子噤声，却忍不住偷瞧廊外——莫长春昨夜脸洗过了，灰袍仍素，杖仍旧，像什么都没发生，又像发生得太实在。

柳青鸢公事公办来访，青衫素，宗主令在腰，剑未佩，威在眸。她目光扫过莫长春袖角一点药渍，一言不发，像风过不留痕。

苏念慈送茶，茶热，手稳：「宗主请。」

柳青鸢抿一口，淡道：「三品金丹，青岚之幸。」

「宗主谬赞。」

「不谬。」柳青鸢起身，至门边停半息，未回头，「首座破境，可喜。东脉之事，亦可喜不起来。」

她走了。廊上静，静得能听见炉里细火。

围观弟子在远处咬耳朵：「宗主骂体统，骂的是谁？」「骂首座，也骂长老。」「骂得好轻，好狠。」

苏念慈转向莫长春，低声：「宗主……吃醋了。」

莫长春望天，云淡：「风大。」

「师兄装傻。」

「傻好。」他杖点地，「傻让人放松警惕，警惕一松，礼好送。」

苏念慈噗嗤，又赶紧收，首座模样。她望门内炉，低声：「炉还开着。」

「开着就好。」莫长春道，「开着，比多少句体统都真。」

柳青鸢在远处回廊停了一停，指节叩栏，一声，像把妒与担都叩进木里，再转身，宗主威仪又穿上，穿上得像剑回鞘。

苏念慈瞧见那背影，忽然不酸了，酸不起——东脉夜袭近了，体统二字，轻不过命。

赵玄机路过，听见「体统」，停半息，未讽，像也懂：门比体统难守，夜袭将至，谁都有戏要演，演不了的是命。

霍镇山骂：「体统个屁！夜袭在前！」

弟子围廊外，咬耳朵：「宗主骂体统，首座脸红，长老说风大——风大还能红？」

苏念慈封丹样，手稳，稳得像把羞炼进丹。莫长春望东脉，云淡，云腥，像战帖已至。

他低声，像钩，说给炉，也说给风：

「验丹文书半个时辰后到。东脉的事，才是真要命。」

风大，药香与云腥缠在一起，像战帖已至。

围观弟子在远处咬耳朵：「宗主今日来不来？」「来了怎么办？」「体统二字，够嚼半日。」另一人答：「嚼归嚼，夜袭近了，体统轻不过命。」

孙福旧党嘀咕：「老不正经……」顾小满冷眼：「闭嘴。首座守丹，长老守东脉，你守什么？」嘀咕者噤声，像体统碰上半石，又磕碎牙。

赵玄机路过丹堂，听「体统」二字，停半息，未讽。他望莫长春背影，妒里忽然空一寸——空处像还库那日挪出的一寸，挪向哪边，他自己尚说不清。

魏无涯在檐角，扇开一线，观三人气运交缠，低笑：「体统是戏，东脉是命。命近，戏轻。」

验丹文书半个时辰后到，文书到，全宗又要围一层——围金丹，围体统，围九月老头还剩几月。

林挽月罚役途中，听「体统」浪起，唇白，像终于看懂：体统是戏，证字才是路。路未说，时未到。

霍镇山骂过丹堂廊：「体统个屁！验丹文书半个时辰后到，到了全宗又要围一层——围金丹，不围嘴。」

体统还在弟子嘴里嚼，嚼不过验丹钟响，钟响不过子时。

弟子围廊外，咬耳朵：「宗主骂体统，首座脸红，长老说风大——风大还能红？」苏念慈封丹样，手稳，稳得像把羞炼进丹。莫长春望东脉，云淡，云腥，像战帖已至。验丹钟将响，响之前，体统还在嚼；响之后，全宗又要围金丹——金丹在炉，命在东脉，体统在风里。

丹堂炉火映着廊柱，柱影长，长得像体统二字也长。莫长春立廊外，不辩，只望东脉，像望一口将沸的锅——锅在丹堂，火在战前，体统在风里，风在柳青鸢眸里。

赵玄机路过，听见「体统」，停半息，未讽，像也懂：门比体统难守，夜袭将至，谁都有戏要演，演不了的是命。霍镇山骂：「体统个屁！夜袭在前！」

围观弟子在远处咬耳朵：「宗主骂体统，骂的是谁？」「骂首座，也骂长老。」「骂得好轻，好狠。」苏念慈转向莫长春，低声：「宗主……吃醋了。」莫长春望天，云淡：「风大。」
