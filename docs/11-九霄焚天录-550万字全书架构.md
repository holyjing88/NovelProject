# 《九霄焚天录》550万字 · 1700章全书架构

> **独立作品声明**：本文档为**全新原创长篇**，与《万古守灯人》无任何设定、人物、剧情关联。  
> **篇幅目标**：约 **550 万字** · **1700 章** · 均章 **3200～3500 字**（大战章 5000～8000 字）  
> **主参考**：《斗破苍穹》剧情骨架（天才跌落 → 三年之约 → 戒灵导师 → 炼药崛起 → 宗门大比 → 遗焰寻宝 → 地图升级 → 大陆争霸 → 成帝）  
> **辅参考**：《凡人修仙传》资源博弈与谨慎留后手；《斗罗大陆》战队分工与大赛结构  
> **核心主题**：莫欺少年穷；以焰焚霄，以丹立命，以诺证道

## 二、笔锋定位（斗破为主，凡人/斗罗为辅）

### 2.1 斗破式核心（占比 70%）

| 技法 | 本作落地 |
|------|----------|
| 开篇天才跌落 | 焰纹测试从「九层满溢」跌至「锻骨三层」 |
| 当众受辱 | 订婚宴上被退婚，立下 **三年焚天之约** |
| 章末留钩 | 每章断在「焰纹将熄」「古戒微热」「遗焰现世」 |
| 围观震场 | 每次破境、炼丹、大会必有旁观者哗然短段 |
| 延时复仇 | 三年之约（卷一末）→ 五年宗门仇（卷三）→ 十年中州债（卷七） |
| 戒灵导师 | 丹霄子传授丹道与焰诀，主角仍靠自己搏杀 |
| 炼药立身 | 以丹换资源，以资源养焰，弱者突破口 |
| 闯塔夺宝 | 焚天塔、镇焰塔、九霄天塔（各阶段一座） |
| 遗焰寻宝 | 太古遗焰榜九焰，逐焰升级 |
| 势力升级 | 青石城 → 大燕帝国 → 东域 → 中州 → 九霄天 |

### 2.2 凡人式辅助（占比 20%）

- **林烬三不**：不浪突破、不留真名入坊市、不欠人情不还不战
- 每一块灵石、每一味药有来路账本（隐性计数，增强真实感）
- 秘境捡漏必有代价（伤、仇、债）
- 宗门政治：站错队会死，林烬先站「弱侧」攒底牌

### 2.3 斗罗式辅助（占比 10%）

- **焚天小队**固定队：强攻/控火/丹辅/侦察/坦克
- 学院联赛式 **「东域万院战」**
- 魂环式外物突破 → 本作以 **「遗焰融合」** 为突破仪式

- 母遗物，内封 **丹霄子** 残魂与半部《焚天丹经》

- 终局：古戒化为第九遗焰「寂灭心焰」核心

#### B. 丹霄子（戒灵导师）

## 十、核心台词（原创）

1. 「三年，够了。」  
2. 「别急着笑，焰还没灭。」  
3. 「丹不成，焰不配称天。」（丹霄子）  
4. 「你说我配不上天，我用这掌告诉你——配不配，焰说了算。」  
5. 「赵家欺我林家，今日起，焰纹为证。」  
6. 「师父，这炉丹，我替你炼完了。」  
7. 「吞焰盟吞天下之焰，我便焚尽天下之吞。」  
8. 「帝不称帝无所谓，这青石城，得有人护得住。」  
9. 「苏清晏，这一战之后，你我恩怨两清，还是两缠，你选。」  
10. 「烬而不灭，是为焚天。」
