# 第五十八章 血线将至

试手日，卯时前。韩泥未睡实。他是记心翁，夜里在心里盘点七笔。分列，不混。符怀暖一丝，坛腹温一线。坡下，秦霜最后一遍：「手抖，丹歪，滚。丹歪不要紧，手不能歪。」试手凝丹，只验手，不验灵根色。色末，手抖才滚。韩泥立末排，独眼平。铁无言验鼻：「静带到手上。」丹炉前，他手不抖。手稳，丹形丑，丑而稳。稳，像辨香那日鼻下静。执事记功，不记灵。管事「嗯」一声：「麸皮不克。」记功，像窄路又宽半分。赖福脸青，像秤空一格。韩泥不喜，不喜像少年。

他只记：手稳了，路又宽半分。同批少年碰他袖：「你真过了？」「过了。」韩泥说，「过的是手，不是灵。」「灵根末，手正，正就够。」铁无言声低，只对他。韩泥「嗯」一声。嗯，像心口七笔压着。秦霜目光扫过队尾，声冷：「再嗓，记名。」队散。散时，赖福啐地，啐得虚。虚，像怕「记功」二字。韩泥不回头。回头像认谣，谣借腥，他不碰。午后，编筐补单，藤刺深扎，扎透旧茧，血珠饱满。他甩手，血甩向席盖，席吸了，血未至沿。未至，像血仍不渗沿那几日。沿在眠，血不主动碰。

他盯掌，盯席，盯坛角。角静，沿净，净，像瓮仍眠。他低声道：「血来了，沿不接。不接，不是怕，是等缘。」坛腹温一线加剧，像瓮听见血在席，不在沿。刘婆端姜汤，姜薄，热：「试手过了，别傲。」「不傲。」韩泥说，「傲，符自灭。」他喝尽，「嗯」一声。姜热，像叶汤远影。管事路过，验手：「手抖否？」「不抖。」韩泥说，「抖是恩断，恩不断，手不飘。」管事「嗯」一声，走了。

试手坡下，同批少年抖，抖像怕滚。韩泥不抖。抖是恩断，恩不断，手不飘。他盯席边，席边湿一线，湿不到沿。不到，像瓮仍眠。铁无言路过，声低：「手稳，沿净。净，像等缘的人。」韩泥「嗯」一声，不答「缘」字，只答：「等。」等，不是拜血祠，不是主动滴血。是活计扎，血找席，席挡沿，沿等韩氏守瓮血那一误。他是记心翁，恩仇都记在心里。试手记功，功在心里，不在舌。舌轻，恩重。夜，符怀暖，药涂裂口，血止。止在沿前，止得像规矩。外头风息，风息，柱上无新条。无条，有条在坛。

坛温加剧，加剧，像将醒未醒。

丑时前，沿仍净，手稳，心记着七笔。记着，才接后头两夜。他按席，席厚，厚像守了六十日的命。命跟坛，坛跟缘，缘在沿前那一止。他低声对坛：「血在席，不在沿。沿在眠，我守席，守沿，守不主动滴血。」坛不应字，只温一线，又凉。凉，像应「等」。外头更鼓二响，响沉。明日仍搬柴，仍清栏，仍末排。末排不要紧，要紧是沿净，手稳。

他低声道：「血线将至，沿不接。不接，才接得住瓮醒那一误。」邻床问：「你手咋了？」「扎藤。」韩泥说，「血在席，不在沿。」「沿是啥？」「沿是坛边。」韩泥说，「沿净，才接得住缘。」邻床不懂，哼一声，翻身。秦霜远廊经过，一句：「手稳，心别邪。邪，符灭。」韩泥怀符暖一丝，心记着七笔，不邪。他独眼平，手不抖：「试手过了，路宽半分。宽，不是灵根好，是恩没断。」坛壁水汽爬半寸，敛。敛着，像替他守着。手别飘，恩不断。恩不断，叶汤就还在前头。血到了席，没到沿。

沿在眠，血不主动碰，这是铁规。他盖席，盖严，严像藏私。私是坛，坛是命。外头风硬，风里无栖香全句。全句散了，散成席边血气那一息。管事又嘱：「试手过了，仍搬柴。搬柴手抖，记功抹。」韩泥「嗯」一声，肩稳。同批少年问：「试手过了，还搬柴？」「搬。」韩泥说，「搬柴练肩。肩稳，手才稳。」铁无言廊角，声低：「沿净，别拜邪。拜邪，恩断。」韩泥「嗯」一声：「不拜。守沿，等缘。」刘婆又端半勺粥，热：「抖的是心，不是手。心记着，手就稳。」韩泥喝尽，「嗯」一声。半勺，热在，像第七笔粥又实一分。

坡下霜白，柱上无新条。他摸木牌，与记功在并放。并放，像命与规矩挨着。挨着，才接得住瓮醒。秦霜问心时，他声哑，声平：「为还。还，先还烫。」秦霜「嗯」一声，不夸，只点头。点头，像记功在。试手后，他仍末排。编筐补刺，藤刺浅些。浅些，血线可控。控，像守瓮人。

管事验手：「试手过了，手还抖否？」铁无言立柱边，声低：「沿净，好。」刘婆端姜汤，姜薄，热：「血在席，别慌。慌，手飘。」「不慌。」韩泥说，「慌，恩断。」邻床问：「你席边咋湿了？」「席旧，吸。」韩泥说，「吸了，沿才净。」他按心口，低声道：「叶丫头汤，热着还。还，要等瓮醒后，路才长。」血是活计扎出的，不是主动滴坛的。滴坛，要等误那一甩。丑时前，坛腹温又一线，温短，像应席边血气，不应沿。他低声对坛：「你再眠一夜。眠着，我席盖严。严，不主动滴你。」坛不应字，只温，温得像「嗯」。窄路也是路。

路通向瓮醒，瓮醒通向还。还叶汤，还振东药。清晨，坡下霜白。霜贴石阶，石阶滑，滑不要紧，站要稳。韩泥仍末排。搬柴过半，肩稳，像试手练出来的。管事验沿：「沿净，记在心里。脏沿，像拜邪。」韩泥「嗯」一声：「不拜邪。」铁无言立柱边，声低：「手稳，沿净。」韩泥只答：「等。」编筐末藤，藤刺深扎，血珠饱满。甩手，血甩破席，席吸尽，沿仍净。外头风息，柱上无新条。坛温加剧，像将醒未醒。」他独眼平，手不抖：「血线将至，沿不接。」试手日午后，他仍末排。末排不要紧，要紧是沿净。

丹形丑，丑而稳。脸丑，手正，正就够。

试手日午后，肩稳，像坡下站出来的实，实不在灵，在手。血到了席，没到沿，沿不接。

试手坡下，丹形丑，丑而稳。稳，执事只记手，不记脸——脸丑，手正，正就够。同批少年抖，抖像怕滚。韩泥不抖，抖是恩断；恩不断，手就不飘。赖福脸青，青像秤空一格。一格空，韩泥路实一寸——寸小，也是胜。午后编筐，藤刺深扎，血珠饱满。饱满，他甩向破席，席吸尽，沿仍净。净，像规矩守了六十日——六十日擦沿，六十日不主动滴血，六十日席盖严。他盯席边，席边湿一线，湿不到沿。不到，像瓮仍眠，眠得深，深在等缘满。夜，符怀暖，药涂裂口，血止。止在沿前，止得像道线——线内席，线外沿。他低声道：「血来了，沿不接。不接，不是怕，是等那一误。」坛腹温加剧，加剧，像瓮听见血在席，不在沿。
