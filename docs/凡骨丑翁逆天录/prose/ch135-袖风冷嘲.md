# 第一百三十五章 袖风冷嘲

药廊风起，袖香冷。

白织月立阶上，阶高，声更高：「丑丹师挂牌三日，可辨几包渣？」

韩泥立阶下，捧空囊，囊净：「够。够辨，够挂牌，不够辩。」

「辩什么？」白织月唇角一挑，「辩你运气？」

「辩骨。」韩泥声平，「骨丑，渣正。正，是手嗅出来的，不是运气。」

「手嗅？」她袖风过，过得冷，「丑骨的手，也算手？」

「算。」韩泥说，「五年摸渣，摸出来的手。」

「预选帖在怀，真以为能上大比台？」

「上。」韩泥道，「上不上，台见。见在掌，不在舌。」

「台上见骨。」白织月声尖半分，「骨丑，台尖，别跪。」

「不跪。」韩泥说，「跪了，约白接。」

她没再尖。尖不过「上」字——上是约；约在玉牌，不在嘴。

铁无言立廊角，抱臂，没插话。没插话，像从绶前先站——站侧，站远，站得像清瘴后仍来修栏。

韩泥望他一眼，一眼像伏笔：绶在后头远日，线在并肩与修栏；线不断，恩才还得完。

赖福廊下过，啐：「丑杂役也配跟内门弟子论辩？」

有人帮腔：「配？配晦气！」又有人压嗓：「压低，管事记名。」

韩泥不辩。辩费舌；舌要留来守坛。

管事弟子记簿，记「丙九韩泥，廊下不滚」，字细，细得像小胜。

刘婆缝扣过，声低：「别跟白姑娘顶。顶，惹眼。」

「不顶。」韩泥说，「顶，像情；情在还，不在舌。」

「还归还，嘴别硬。」刘婆补半句，「硬了，粥也少。」

「粥我喝。嘴不硬。」他应。

「手别抖。」刘婆又盯他独眼，「抖了，下一层虚。」

「不抖。」

回杂役屋，他独坐坛边。袖风冷，冷不要紧；冷，才记得铁无言挡的那半步。半步近是兄弟；远是绶未纳。纳前，只站。

邻床少年悄问：「白织月是不是瞧不上你？」

「瞧不上不要紧。」韩泥说，「瞧得上，手也要稳。」

「那铁无言为何帮你？」

「帮在清瘴，不在今日辩。今日，只站。」

「站着不说话，算什么帮？」

「算线不断。」韩泥道，「线断了，日后绶也不香。」

秦霜远木鱼轻敲：「袖风嘲，心别邪。」

韩泥不答，答在心里。心里分列：嘲是嘲，恩是恩；恩在还，不在辩。

铁无言廊角声低：「袖风冷，手别飘。」

「不飘。」韩泥应声。应声短，短才像常活。

「她再尖呢？」

「再尖，仍不辩。」韩泥说，「辩像滚。」

管事扬声例行：「预选者，层数每月验一次。」

「验。」韩泥只答。

「验不过呢？」

「不过，再守。守到过。」

验什么，验手稳不稳。稳了，下一层才实；实了，大比台才踩得住。

药渣场角，他备料半刻，料细，细像积势里最后一筛。筛齐，手才静。

赖福脚步停门外，停半息，像盯。盯不要紧，席严，坛不露。

「在不在？」赖福隔门问。

「在。」韩泥道，「沿净。」

「净给谁看？」

「给自己守。」

沈枯芽远远鞠半躬，不近。不近不要紧，徒恩清了半块灵石；清了，叶汤还在前。

他低声对坛：「从绶在远。线在今。今，站侧，不叫。」

坛不应字，只温。温像应「兄弟」——兄弟线续，不混叶汤，不混刘婆。

更鼓沉，天边白一线。白里，他摸玉牌，牌凉：「三年约，记着。大比，记着。」

簿上再添半行，行丑，字真：「嘲下不辩，席下不滚。」真在不滚，不在辩赢。

韩泥捧空囊，囊净，净像没辩赢。赢不要紧，没滚就是赢。

白织月袖风又过，过得冷——冷才记得手要稳。她远廊没停。没停，小胜——胜在没滚，不在辩赢。

药峰活照旧。分拣半刻，指稳，鼻下静一息——一息里只闻辛淡，不闻辱。辱在廊下，不在指；指准，下一层才实。

他望铁无言，铁无言抱臂，抱臂像约：纳绶前，只站侧，不叫。廊下风硬，硬不要紧；硬，才记得手要稳。

刘婆端粥，仍多半勺。韩泥点头，谢轻，恩重。粥热，热像提醒：嘲过了，恩还在；恩在，路就不绝。

同棚杂役笑：「你跟白姑娘顶了？」

「没顶。」韩泥说，「顶像情。情不在舌。」

「那你怕不怕大比？」

「怕。」他说，「怕也上。不上，约白接。」

「上了输呢？」

「输了再还。」韩泥道，「还清了，输也值。」

帖沉怀，沉的是序，不是惧。惧在耳，不在坛；坛在，手才不飘。

废渣巷日斜，他验渣，渣正，手才静。渣正，像挂牌那「正」字还在掌里。

侧门驿亭风硬，他听风，风过，记在心里。风里带一点袖香余冷，冷像那眼；眼在，手更要稳。

丑时末，再练鼻下静。粟壳香淡，一字不落。落齐，袖风这一日记进心里——记嘲，记站，记不叫。

老耿路过窗下，哑声：「嘲过了？」

「过了。」韩泥说。

「过了就实层。层实了，台才接得住。」

「实。」

药廊石阶上还留着袖香余痕，痕淡，淡得像嘲也有尽。尽了不要紧，尽了才露出底下那条路：路通向验息，通向十五，通向台。

同棚少年劝他：「少跟白姑娘见面。见一面，晦气深一分。」

「见不见，她在风里。」韩泥说，「风里我在，就不躲。」

「躲不好？」

「躲像认邪。」他摇头，「邪在谣，不在我。我只守正。」

少年叹口气走开。走开后，他把「正」字在心里又按一遍——按住了，袖风再尖，也只刮衣，不刮骨。

他按心口，低声道：「汤还是热的。」

六字落，像路标——标向下一层，标向大比，标向还没纳的从绶。

更鼓沉，他望坛，独眼平：「袖风嘲过了。下一回，层十五。」

坛腹温，贴掌，像应「站」——站在侧，不叫；叫了像卖，卖了线就脏。

编筐棚角，他把藤条绕紧，绕得像把嘲也绕在外头。外头笑，里头静；静了，验息册上那行空格才填得进真数，填不进运气。

铁无言门外问：「手还飘吗？」

「不飘。」

「帖还沉吗？」

「沉。沉的是序。」

「序里有我？」

「有。」韩泥说，「有你那半步。半步在，线在。线在，我不叫。」

铁无言「嗯」一声，脚步远了。远得像站侧该有的距离——近了像情，远了像义；义能走长远，情容易脏舌。

管事夜巡又停门前：「廊下那出，记过了。别再起波。」

「不起。」韩泥说，「波在她袖，不在我舌。」

「舌省着点好。」

「省着。省给守坛。」

人走了。他才把玉牌贴回心口。牌凉，凉里有约；约不催嘴，催手。手要在分拣里稳，在鼻下静，在嘲里不滚。不滚，才有资格把空着的那一行写成十五。

秦霜木鱼最后一声落下，声短：「心正，层正。」

韩泥默应。应不出口，出口像表演。表演脏线，也脏汤。

他开册自检三行：嘲下不辩——有；席下不露坛——有；站侧不叫绶——有。有了三行，才敢把眼睛望向空格。空格不填运气，填手。手要在明天的分拣里再稳一寸，稳一寸，十五才不像喜，像实。

白织月袖香余冷在鼻下一息淡去。淡去时，他忽然想起叶青禾那碗汤，汤不在廊，在很远很远的门缝灯。灯未亮，汤仍热。热着，人就不能被嘲带跑。

他闭眼数息，一息十四，一息还空。空处留给十五。留给了，就不慌；不慌，袖风再嘲，也只是风声。风声过后，人还在坛边，帖还在怀，线还在侧——在，就够明天把空格填实，填成真层，不填半点运气。

山门风再起，帖角轻撞衣扣，叮一声。叮声像钩——钩住验息册上那行空格，也钩住他还没踩实的十五。格空着，人就不能歇。歇了，嘲会笑他；笑他不要紧，手飘了才要紧。手不飘，层才来；层来了，台前那口气才真。真了，袖风再冷，也尖不过掌里那一口「正」。
