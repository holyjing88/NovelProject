# 第一百二十五章 窑炉稳

最后七日，韩泥啥都不做了——就守窑温。

窑门柴挡压得死紧，紧得像当年席盖捂坛。席严坛不露，窑严粒才正；正了，试台才配夺魁，夺魁才配还刘婆。他把焚火残诀一一息数稳，铁片嵌在窑口，温只许走一线。一线升，像首炼那天余温还没散尽——散尽不要紧怕，怕的是跳。跳了，粒杂；粒杂，半月熬白了。

赖福没新花样了，只在廊外晃木牌：「七日后试台，看你滚。」

韩泥不还嘴。辩费气，气要留给数息、守温。铁无言送来炭渣，渣里一点灵火屑，够稳一夜：「七日，别出门。」

「不出。」韩泥说，「出门，手飘。」

刘婆在柴门外咳：「窑里闷，别闷坏。」

「闷不坏。」韩泥应，「闷着，炉才稳。」

白织月过药山道，袖风远远刮一句：「七日后，看丑骨出丑。」他仍不辩。出丑在脸，不出在手；手稳，丑也能正。

第一夜，窑温如秤。第二夜，邻床少年问：「七日不出门，饿不饿？」

「不饿。」韩泥说，「饿了手飘。」

第三夜温忽跳一跳，像有人使坏。他铁片压口，残诀一息，温又压回秤平。第四夜刘婆塞半块麦麸饼：「试台前，别空肚。」他收下，谢轻，恩重。第五夜气走圆满，十二层实到极处，像弓拉满——箭先不射，射在后头夺魁那天。第六夜再验九粒，粒粒正，正得像五年渣里淘出来的日子。第七夜封窑，封得像封坛：坛温一线不退，心才稳，步才稳，债序才稳。

丑时末练鼻下静，粟壳香混药香。香正，更要紧是静。静住了，开窑晨的天色才白得实——白得像刘婆补衣的旧线。线旧，线暖，等着一百三十日后交到她手里。

铁无言来窑外：「七日毕，手别飘。」

「不飘。」韩泥应。

刘婆又咳：「窑稳了？」

「稳了。」

「稳了，试台别抖。」

「不抖。」他只这两字。

望着坛，独眼平：「稳。记着。」七日封窑毕，九粒仍正。正，试台就在手上。

他把九粒又挨个闻了一遍。闻完不炫，只点头。点头给自己看：稳了，才配出门挨验。

第一日辰初，药山雾未散，窑烟细如线。他添半把潮柴，残诀走息，潮柴也燃稳。管事过外记一笔：「备战七日，别下山。」他答：「不下。」不下不是怂，是手要稳。稳了，第九粒才配落怀；落怀了，后头验正、听帖才接得住。

第二日借编筐棚半刻。藤刺扎掌，他偏开手，刺浅些。筐勒匀，匀像火路——勒不匀，渣散，路断。编筐钱能买米，米在，试台前恩才还得上口。同批少年掩鼻：「你七日不出窑，臭不臭？」

「臭。」韩泥声平，「臭是药山活。活有味，味在角，不在席。」

第三日黄昏，有人掷石过窑顶，石打柴挡一声响。响不要紧，温未跳。他铁片再压，一息回秤。老耿挑水路过，声哑：「备战别拜暖。拜暖，路断。」

「不拜。」韩泥说，「坛在，人在。」

第四日邻床少年发烧扰息，他仍守窑到丑时末。扰归扰，香一字不落。第五日铁无言又送炭渣，屑更细：「够一夜。」他谢轻，兄弟线仍不断。沈枯芽口信无字，只有半片枯叶——叶黄，字在记里：叶丫头怕黑，门缝留灯，还刘婆在前头。前头远，手在就不绝。

第六日白织月远廊过，袖风冷，没停。没停是小胜——胜在手稳，不在嘴辩。秦霜木鱼轻敲：「备战别邪。」他不答，答在心里：稳，是丑翁本事。本事在忍七年克扣，在还芽儿半块石，在守这七日窑。

第七日封窑前，他再验九粒。正。忍不是怂，是活；活了，才配夺魁，才配还衣还钱。散队后他数息、数粒、数债序。序在，步不飘；不飘，才接得住验，接得住帖，接得住魁与还。更鼓沉里他对坛说：「稳。记着。」

药山半腰风硬，他把剩渣摊匀，匀像秤平。摊匀了，手才静；静了，第九粒才肯安。管事夜巡灯照柴挡，见厚挡不见坛——坛不露，沿净才藏得住。同棚有人赌：「丑骨夺魁，太阳西边出。」他不接茬，只把掌茧裂处的疼按回去。疼不露，才像常活；常活里藏备战，备战里藏还路。

废渣巷尾倒半筐灰，灰里仍正。正像回响，回响在鼻下，不在舌。舌留给守窑，留给封坛。侧门风紧，紧像试台前那口气——气要匀，匀了手不飘；手不飘，九粒不杂。不杂，验才验得出正。

帖还没到，他先把木牌贴胸。牌凉，像贴着仇约：约说三年，说炼气十三，说站钱戾衡台前。台前远，窑温近。近处守住了，远处才不散。赖福脚步在廊外停半息，他只应：「在。沿净。」在，就不滚。

刘婆端粥，仍多半勺：「手稳，饭稳。饭稳，恩不断。」

韩泥点头。沈枯芽怯问：「韩叔，怕不怕？」

「怕。」他说，「怕也守。守完，才有饭。」

白织月袖风再过，没停。他低声对坛：「坛在，人在。人在，就不滚。」坛不应字，只温。温够了，刃不必先亮——先记。记完了，才配沿净，才配下一程验香。

编筐半刻藤又扎手，刺浅，沿才净。筐卖了米在，恩才还得上。入夜药渣场角他备料，指节白一息，白完坛才落稳。辰初兽栏侧雾未散，他数息，掌茧疼——疼不露，记恩在忍里。午后编筐棚，日斜，他验渣半息，鼻下静，把「守」字摁进路里。

管事夜巡过坡下石阶，灯照席角，照见厚席不见坛。坛不露，沿净才藏得住。帖在怀，怀沉，沉的是序，不是惧。惧了手软，软了九日白熬。白熬不起——刘婆粥热着，铁无言炭渣暖着，芽儿枯叶静着。静着等人站稳，他就得先把窑守成秤。

他又清了一回坛沿灰。灰细，细得像能写信。写信他不会，只会把日子记在茧里：哪夜跳温，哪夜饼热，哪夜枯叶无字。无字的更沉——沉在信上，信他还能守完这七日。守完不是赢，是够格去输赢；够格了，才上得了台。上不了台，谈什么不滚。

封窑毕那夜，他对着微温一线又钉一句：「七日稳了。明日验，别飘。」

夜里窑缝风小了些。小了好，小了温更匀。匀了，他才敢合眼半息——半息醒来，手先摸泥封。封在，心就在。

验字落在耳里，像钩。钩着坡下试台，钩着贴条那日的规矩，钩着帖角被风掀起来还能按住的手。手按住了，还刘婆才不算梦；梦醒了，钱戾衡台前才接得住「不滚」两字。两字现在不喊，喊早了像狂。狂不得。先把窑封死，把香守正，把九粒捂热——捂热了，锤子落下来，也不至于第一下就碎。

更鼓尽时，他听见远处试台方向有人拖木、钉钉。钉声疏，疏得像倒数。倒数不吓人，吓人的是倒数里手还飘。

开窑晨雾更浓，他把九粒罐抱进怀，像抱着一口不应冷的热汤。汤不能洒，罐更不能晃。晃一下，心里那根秤就斜；斜了，七日都亏。

同棚少年追出来：「真不出门硬熬完了？」

「熬完了。」韩泥说，「熬完才有资格去怕。」

「怕什么？」

「怕手飘。」他脚步不停，「怕了反而握紧。握紧，验时才不丢人。」

少年追不上，站在雾里喘气。韩泥走进坡影，影凉，他却觉得窑温还在骨上。骨上有温，帖来也不慌，锤落也不软。

他把手按在泥封上，封硬，硬像一句答应：到验香那天，香要正，息要匀，眼别乱。乱眼就会看见金线边，看见了就容易抖。抖不得。

于是他把「稳」字从窑温里挪进骨头——骨头丑，稳了也管用。管用到帖落下，管用到新衣递出，管用到仇还在玉牌上却不先咬人。先守正，再说话。话留给验，留给规矩，留给那一锤定香的日子。
