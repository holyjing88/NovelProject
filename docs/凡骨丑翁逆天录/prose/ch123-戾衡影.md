# 第一百二十三章 戾衡影

钱戾衡影，落在复测前半月。

戾衡影，廊下靴底声沉，沉里金线边亮。亮不要紧，玉牌更凉——凉烫并着，并着像三年约反面。

钱戾衡声不高，高在韩泥耳听见：「丑杂役，试台别洒。」韩泥低声道：「记着。」记着，不辩。辩，费舌；舌要留来开罐，留来捧粒上台。

影硬，手更硬。手硬，124无言挡才接得住——挡话，不挡手；手稳，簿清，盗谣虚。

内门练剑毕，廊下歇，歇里钱戾衡靴底声沉，沉里带金线边。边亮，亮得像玉牌上「三年约」三字——三字凉，仇烫。烫在心里，不进舌；舌响，香飘；飘了，手滚。

韩泥捧渣袋过廊，不近，不躲。躲像怕；怕，手抖。手不抖，影就落他身上，落得像眼风。眼风硬，手更硬；手硬，玉牌不虚，三年约不白接。

钱戾衡声不高，高在众耳都听见：「丑杂役，又见面了。」

韩泥停步，独眼平：「见。」

「见什么？」钱戾衡笑，笑薄，「见灰线三尺？见末席报名？见你想炼气十三，不配？」

韩泥不辩。辩，费舌；舌要留来守罐，留来记玉牌。玉牌在怀，贴身，不示——示了，哄笑又来；来不要紧，手稳。手稳，就不滚；不滚，才有资格站他台前。

「配不配，试台见。」韩泥声平。

钱戾衡踏前一步，靴底声沉：「试台？我让你滚，你才有资格不滚。十三层，你够吗？」

「不够，练。」韩泥说，「练够了，站你台前。」

哄笑薄起，薄也真。真在众目，真在玉牌凉——凉不要紧，仇烫。烫在心里，不进舌；舌响，香飘。飘了，滚字就真；真了，叶汤恩断，刘婆恩还迟。

钱戾衡收笑，声冷：「三年约，还剩半年。半年，你仍是丑。」

「丑不要紧。」韩泥说，「丹正，才要紧。」

「正？」钱戾衡瞥他怀，瞥得像看藏散，「丑骨炼正？正也是运气。」

韩泥捧袋，袋沉，沉的是渣，不是辩词。辩词费舌，舌要留来试台——试台见，运气不见，正见。正见，刘婆敢认恩，芽儿敢收石，叶汤还在前头烫。

铁无言立侧，声低：「少主，管事区，别越线。」

钱戾衡瞥铁无言，瞥完仍盯韩泥：「铁无言，你也护晦气？」

「护手稳。」铁无言说。

钱戾衡哼，哼得像意未尽：「半年後试台，我看你抖不抖。」

记着——仇苗又深半寸，深在影，深在玉牌，深在三年约。深不要紧，不拔，先活；活够了，站台前，才不滚。不滚，炼气十三在130破；破了，还刘婆，大比帖在怀。

钱戾衡走远，走远还留一句：「丑杂役，记住今日。」

韩泥应：「记着。今日，也记着。」

记着，像68那日眼风，像70那日踹台。踹台接了约，约接了备战；备战接了影，影接了试台。试台见，仇见，恩见——恩见刘婆新衣，仇见不滚。

散队后，刘婆端粥，仍多半勺：「内门影？」

「影。」韩泥说。

「影不要紧。」刘婆哼，「手稳，饭稳。饭稳，恩不断。」

韩泥点头。谢轻，恩重。恩在粥里，也在忍里——忍钱戾衡影，换手不抖，换试台资格。资格在，九粒在怀，怀沉，沉的是债序。

午后废窑，他盘坐冲关。气走十二周，周周实，炼气十二层。十二层，不声张。声张，钱戾衡说「偷境」；说不要紧，手稳，试台见。见了，正字自现，刘婆恩才配还。

坛腹温一线，应「影」——影在路后，路在前；前是复测，前是魁，前是还刘婆。还刘婆，芽儿还了沈枯芽半块灵石，刘婆待还；还了，叶汤还在前。还在前，省亲在后，大比帖在更后。

沈枯芽远远探头，又缩，像怕钱戾衡影沾身。沾不要紧，石已还；还了，芽在，徒恩清了半笔，半笔够撑十二层。十二层实，试台手才不飘；不飘，就配夺魁预备，就配130还刘婆。

韩泥不答，答在心里——心里玉牌凉，三年约烫，钱戾衡名在仇线顶。顶不要紧，刃磨亮；亮在试台，亮在510那日远影，不在今日舌。今日舌留来守静，守鼻下香一字不落。

丑时，练鼻下静。粟壳香淡，香落齐，戾衡影这一面记进心里。记进心里，不进舌——舌响，影就真；影真，手就抖；抖了，滚字就真。不抖，是玉牌反面答案。

邻床少年悄问：「你真不怕他？」

「怕，手抖。」韩泥说，「不抖，就不怕。」

少年闭嘴。闭嘴，韩泥望廊下——廊空，影远，玉牌凉。凉烫并着，并着像路——路向十二层稳，路向124挡话，路向130魁与还刘婆。还刘婆，新衣养老钱在怀，布包叠实，等试台後递出。

他低声道：「钱戾衡，记着。半年，试台见。」

不滚，手在，粒在，坛在。在，就接得住124铁无言挡话，接得住125窑炉稳，接得住129备战帖至。帖至，魁至，还刘婆至。

外头风硬，硬得像影还在。还在不要紧，腹里十二层实——实了，影虚半分；虚半分，够走到还刘婆那日，够走到叶汤省亲前。省亲前，大比帖在怀沉，沉的是债序，不是惧。

影去后，韩泥废窑盘坐，十二层气走周。周周实，实得像玉牌贴胸——胸稳，手稳，试台不洒。不洒，钱戾衡「上台别洒」白说。白说不要紧，末项在130；130夺魁，还刘婆，炼气十三破。破了，大比帖接得住；接得住，钱戾衡名在帖后，后在不要紧，刘婆先还。先还，坛认还，腹里微温应。应了，就够走进130，够听魁字落，够看刘婆眼浊清一寸。

钱戾衡影这面，韩泥不近廊下吵。吵，手飘；飘了，除名。除名，恩断；恩断，叶汤凉，芽儿石白还，刘婆恩还迟。

影里，钱戾衡声冷：「半年後试台，我看你抖不抖。

铁无言立侧：「少主，管事区，别越线。」钱戾衡哼，哼得像意未尽：「丑杂役，记住今日。」记着，像68眼风，像70踹台；踹台接了约，约接了备战；备战接了影，影接了试台。」「影不要紧。手稳，饭稳。恩在粥里，也在忍里——忍钱戾衡影，换手不抖。

午后废窑，气冲关，十二层破。破，不声张。声张，钱戾衡说偷境；说不要紧，手稳，试台见。见了，散正不正，影硬不硬，一并见。

沈枯芽远远探头，又缩，像怕影沾身。沾不要紧，石已还；还了，芽在，徒恩清了半笔。半笔够撑十二层，够撑复测备战。

夜，练鼻下静。香落齐，戾衡影这一面记进心里。记进心里，不进舌。舌响，影就真；影真，手抖；抖了，滚字就真。不滚，手在，粒在，坛在，玉牌在。还在不要紧，腹里十二层实——实了，影虚半分；虚半分，够走到还刘婆，够走到叶汤省亲前。省亲在筑基后，先在130还刘婆，先夺魁，先接大比帖。

十二层破，钱戾衡影硬不过手稳。手稳，玉牌不虚，三年约不白接。白接，恩断。恩断，叶汤凉，刘婆新衣还不上。还不上，先在130夺魁。魁后，炼气十三，还刘婆，帖至山门。山门帖上言伯钧名，钱戾衡影在字后——后在不要紧，刘婆这笔先还。

十二层实，影去，玉牌在怀。怀凉仇烫，烫在试台，烫在不滚。不滚，手在，路在。

影去，廊空，空得像仇暂退。暂退不要紧，记着。记着68眼风，70踹台，123这一面。这一面，十二层实，玉牌贴胸，胸稳手稳。手稳，试台不洒；不洒，末项在130；130夺魁，还刘婆，炼气十三破。破了，大比帖接得住；接得住，言伯钧名在帖上，钱戾衡监在帖后。后在不要紧，刘婆先还。先还，坛温应还，腹里微温一线。

他摸玉牌，牌凉，低声道：「三年约，记着。手稳，粒正，试台见。」见字落，坛温一线，像应。应了，路又明一寸——明向还恩，明向不滚，明向刘婆新衣养老钱。

外门弟子渐少，韩泥不近廊下笑——心在坛，坛腹温一线，温像应「备战」。
