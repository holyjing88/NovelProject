# 第十八章 厉无殇至

第七日，赤焰少谷主亲至。

青岚议事大殿，旗未入殿，威先入。厉无殇踏步上阶，玄袍金线，魔息如刃，笑挂在唇，像一把还没出鞘的刀先寒了空气。

韩铁山伴侧，甲未卸，眼四扫，像数青岚门还有几块硬骨。

柳青鸢升座，霍镇山按盾立侧，周德海捧和折子立末，像备好的退路。

莫长春拄杖立殿柱旁，灰袍素，须眉雪白，在满堂华服里，像一张旧符，却贴在最显眼处。

厉无殇目光落在他身上，停住，笑深一分：「这位，便是莫长春莫长老？化神劫后，筑基中期，寿元九月——啧啧，将死之人，占着长老位，占着青岚门的脸。」

殿内寂然。

有人吸气，有人低头，有人忍不住看莫长春——看他将死之人如何接这一刀。

莫长春抚杖，微笑，像听一句久等的话：「厉少谷主嘴利。老夫九月，确实将死。」

厉无殇扬眉：「既知将死，何不早退，把位让给有用之人？」

「退可以。」莫长春道，「死也可以。但在退与死之前，老夫还想听少谷主把东脉三成说完——说完，老夫或许还能送厉少谷主一份礼。」

全场一沸，沸在窃语：送魔头礼？

厉无殇眼底一闪，像被勾起兴：「礼？莫长老九月寿元，赠我何物？半块灵石？一包药渣？」

「礼轻。」莫长春笑，「意重。」

魏无涯立大殿梁上阴影里，扇未开，只观气运，观莫长春袖中那一缕说不清的热，像观一枚将落未落的子。

柳青鸢指节发白，未言，等师兄接招。

霍镇山盾光微亮，像随时要战。

周德海却笑，笑里喜：「莫长老都愿赠礼，和谈有望啊。」

莫长春看他一眼，像看一盆油：「和谈有没有望，不在礼。在根。」

他转向厉无殇，声仍轻：「少谷主，东脉民耕三百年，你索三成，是索命。命可以给一点，根不能拔。你若懂，便谈；不懂，老夫这九月，也够活到你后悔今天说的话——不过，还没说到那一步。」

殿内再寂。

厉无殇笑意定住，像刀终于碰见骨：「好一个『还没说到那一步』。莫长老，我记住了。」

他落座，谈判开场，话却句句带刺，像把东脉灵田切成席面，要青岚门吞刀。

莫长春不辩细节，只偶插一句，句句在「根」字上，像把柳青鸢主战之心，钉在殿心。

宴前，庭外风硬，日头毒。

厉无殇解甲披风，额汗一层，仍笑，笑里躁。韩铁山低声：「少谷主，莫长春袖中无宝光，放心。」

厉无殇嗤：「九月老头，能有什么宝？待会儿看他如何收场。」

莫长春立殿柱旁，袖中素帕压着，帕角干净，像等了很多年。他望厉无殇，笑，像望一份将送的礼。

柳青鸢在殿门内，心弦微紧，不知师兄袖里是什么。

廊下弟子窃语：「莫长老要送魔头礼？」「九月老头，疯了吧？」「手帕定身那夜，也是礼轻意重……」

簿在袖中沉，沉得像要把因果一口气记满。

他低声，仅自己听见：

「下一息，礼该出门了。」

大因果还在宴庭对峙那夜，像帖，像手帕，像全场寂然的三息。

谈判开场，话却句句带刺，像把东脉灵田切成席面，要青岚门吞刀。莫长春不辩细节，只偶插一句，句句在「根」字上，像把柳青鸢主战之心，钉在殿心。

宴前，庭外风硬，日头毒。厉无殇解甲披风，额汗一层，仍笑，笑里躁。韩铁山低声：「少谷主，莫长春袖中无宝光，放心。」厉无殇嗤：「九月老头，能有什么宝？」

廊下弟子窃语：「莫长老要送魔头礼？」「手帕定身那夜，也是礼轻意重……」簿在袖中沉，沉得像要把因果一口气记满。他低声：「下一息，礼该出门了。」

厉无殇目光落在他身上，停住，笑深一分：「这位，便是莫长春莫长老？化神劫后，筑基中期，寿元九月——啧啧，将死之人，占着长老位，占着青岚门的脸。」殿内寂然。莫长春抚杖，微笑：「厉少谷主嘴利。老夫九月，确实将死。」

「退可以。死也可以。」莫长春道，「但在退与死之前，老夫还想听少谷主把东脉三成说完——说完，老夫或许还能送厉少谷主一份礼。」全场一沸，沸在窃语：送魔头礼？厉无殇眼底一闪，像被勾起兴：「礼？莫长老九月寿元，赠我何物？」

围观弟子在殿阶下踮脚，踮得像等礼出门——礼未出，先出的是汗，汗在厉无殇额上，也在九月老头袖里。有人低声：「送魔头礼，青岚脸先碎？」另一人答：「脸碎不碎，看下一息。」
