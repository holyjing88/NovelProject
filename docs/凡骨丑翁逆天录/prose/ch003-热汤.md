# 第三章 热汤

天又冷了。泥岗村入秋，风硬，韩泥漏屋先尝寒意，风从门缝灌进来，灌得藤条更僵。他早起编筐，指尖僵，藤条绕两圈又松，骂藤没用，骂完了还得绕。张麻婆门外路过，瞥一眼：「编慢些，手别裂。」

「裂不了。」

编筐手不能抖，抖了，筐松，药散，活路断。心口那碗汤的温还没记实，像习惯跛脚一样，习惯心里空着——不是没人恩他，是恩还没记；韩泥不知该怎么记，只晓得恩不能乱记，记了就得还。晌午前，门缝才透进一点光。有人影顿在门口，没敲也没喊。韩泥抬头，见一双布鞋，洗得发白，鞋尖沾泥，站得稳。「叶丫头。」

他开口，声音哑。叶青禾应一声。村里都说她命苦，男人走得早，一个人撑药铺帮衬，清瘦耐看，不张扬。她与韩泥隔门站着，晦气翁不配让人进门，她也不进，怕闲话——闲话在泥岗村比风快，风过无痕，闲话能留三年，留三年，够毁一个人。「韩泥，开门缝就行，我不进。」

叶青禾顿一息，又道：「汤趁热，莫挡门口风。」

韩泥把门缝推开一指：「一指够。」

碗递进来，粗瓷沿口缺了一角，汤热气还在，香味淡，是姜，热气贴着门缝往外散了一线，又敛进碗里。「趁热。」叶青禾说。就两字，没有可怜，没有嫌弃，没有「你怎么活成这样」的叹息。韩泥最怕叹息，叹息比骂还轻贱；骂是明的，叹息是暗的，暗的扎心。他伸手接碗，指尖碰到碗底，一物硌手——薄片切得细，藏在汤下，姜。他没抬头：「够了。」

叶青禾在门外道：「天冷，门缝大，别冻着。」

「冻不死。」

「我知道，所以我送热的。姜别扔，晒干还能煮一回。汤里姜别挑太薄，薄了不暖肚。暖着喝，凉了别硬咽。」

韩泥捧着碗，没进门，也不让叶进门——门缝这一指宽，够递碗，够记恩，不够容闲话。「够暖。不硬咽。」

叶青禾声更轻：「记得住，比谢字要紧。」

韩泥没接。巷口周嫂咳一声，没近，只甩一句：「叶丫头，药铺还开不开？」

叶青禾应：「开。秤在，铺就在。」

韩泥在门内听见，没出声——应了，就得还；还，得等手上有东西，手上有东西，才还得不寒酸。咳声远了，谣还没起，起的是风。风硬，汤热，热抵风半寸。他捧碗站门内喝得很慢，烫，辣一点，胃暖，热从喉落到胃，落进心里那道空处——空处填了一指，不多，够记。碗沿烫手，他换了个握法，指节贴着粗瓷，烫着才记得牢。他坐下，碗搁膝上，汤已尽，指尖还留着粗瓷的温。姜片挑出窗台晾干，穷人家姜用两回。叶青禾脚步远去，布鞋踩泥，轻，稳，像把「趁热」二字留在门缝里。韩泥不会写字，也不会把恩划在纸上——懂的是欠了要还，辱了要忍，仇了要候，候在心上，比写在纸上实在。这一碗汤是头一笔恩，深，烙在心里。他低声道：「汤还是热的。」

不是对她说，是说给自己——记着，有一天还。热汤能凉，恩不能凉；凉了，人就成了嚼舌根的嘴，轻，薄，一吹就散。傍晚村童远处嚷「晦气翁」，韩泥没出门，坐到坛边。腌菜坛白日凉，入夜却温了一指，温应心口那碗汤，不应谣。赵绣春门外尖声：「叶丫头倒是心善，善到晦气翁门里去了？」

叶家药铺秤杆落一声，脆，短，赵绣春哼一声噤了，尖不过那声稳。风硬了一点，屋里更冷，可心口还暖着，暖着就够活过今夜。卖盐老汉停一息：「叶丫头心善，善在实。实，不是邪。」

韩泥不谢，谢轻，他记。老汉走了，脚步沉——公道不常来，来一次，够撑明日井台。韩泥不懂太多道理，只懂记在心里。门外暮色沉下来，泥岗村扣着全村，远处赵绣春家亮起灯，亮得张扬，他在瓮底似的屋里，不抬头，只低头看碗，看碗底那片姜辛还在不在。叶青禾为何送汤？不为名声，要名声不会沾晦气翁；不为可怜，送法不像可怜。人该送时，不图回报，图心里过得去。韩泥五十八年，少见这种过得去。还的时候，不能寒酸，不能敷衍，不能学村头那些「算了算了」。碗底缺沿不碍用，穷人家碗缺沿，汤照样烫。童谣丫远处嚷一句，嚷完被娘捂嘴，捂了，够他今夜安一息。他按坛沿，又按心口，坛壁一线水汽一闪又敛，坛不拒热，拒的是脏水。坛凉，心热，人夹在中间，五十八岁，仍活着。他轻声道：「这恩，我记了。」

心里那一笔，丑，却真。他洗尽粗瓷碗搁回门墩，夜雨敲瓦，碗底叮一声。叶家药铺灯亮到夜深，秤声又起，脆，稳，像钉住铺子没关，也钉住心口那笔恩。韩泥合眼，心里只留两字：趁热。还的时候，碗要烫手，姜要还在——还在，叶丫头才晓得晦气翁没忘。他低声道：「趁热。记着。还。」

风从门缝过，带一点姜辛，辛得短，短得够记一夜。张麻婆骂童谣丫：「少学周嫂那嘴！」

骂完塞门缝半块饼：「吃。编筐手不能抖。」

韩泥掰半留半。沈枯芽隔门问：「韩爷爷，汤烫不烫？姜辣不辣？」

「烫。辣好。趁热，才记得住。」

叶青禾在巷那头应：「记得住就好。」

韩泥没接，接轻了是讨。芽儿还想扒门缝，娘拽走，芽儿喊半句「明儿我也送姜」，娘捂嘴骂回去。张麻婆折返：「叶丫头心实，你别多想。」

「不多想。想还。」

「还什么还，先把筐编完。」

「筐编完，人还在，才还得上。」

他藤条绕膝，勒出闷响，心口那笔恩更实，实了，手就不抖，编筐手稳，才还得上那一碗汤。门外赵绣春又骂一句，骂给风听。风过，叶铺秤声又起，脆，稳，谣就矮了半寸。卖盐老汉远处补一句：「实人送实汤，别叫谣搅浑。」

韩泥不接，只把窗台姜片按平。外头风硬，门缝灌进来，他拿空碗堵住缝——碗空，还能挡半寸风。夜更深，井台桶绳吱一声，布鞋踩泥，轻，稳，没停，像叶丫头又去铺里守秤。叶铺灯还亮一线。韩泥搁碗，碗底叮一声——明儿去井台，还得听那声秤，也得见那双洗白的布鞋。碗底姜沉，沉得像恩也沉。沉不怕，怕浮——浮了，像施舍；沉了，像该记。记实了，门缝这一指宽，才够活过下一个霜晨。心里记恩，从此起——恩在，公审来，也只一句还。霜晨里，他仍去井台——井台有秤声，秤声在，叶家就在；叶家在，汤就不凉。
