# 第四十四章 嗅诀

盯秤后第三日，渣场灰又热。热罩脸，脸罩心，心仍要分堆——药渣、炭渣、废渣，堆乱则路断。韩泥拨渣，鼻尖一动，苦里带甘，甘是叶振东那包药的余味，余味淡，淡得真。同场杂役呕：「天天嗅，嗅成晦气鼻！」

韩泥「嗯」一声：「鼻晦气，手不晦气。手不晦气，渣不漏。」

管事喝：「丙九，酉时清完。漏一粒，记名。」

韩泥指不停。不停，是盯秤后练出来的——练出饭，练出命，练出还的路。铁无言再来，立桩，声低：「渣分三堆，别混。混了，丹房不收。」

韩泥点头。铁无言走近，看他拈渣：「鼻下这味，叫什么？」

韩泥停半息：「苦蒿余气，带一点甘。」

「甘从哪来？」

「叶家药铺方向。」韩泥说，「甘短，记在心里，不声张。」

铁无言看他一眼，眼里无嘲：「短，也能记？」

「短，才真。」韩泥说，「真，才不混。」

铁无言翻簿，从袖取薄册，册旧，字密：「嗅诀三句，杂役能记多少记多少。」

他低声道：「一，鼻先静，静才辨。二，苦甘分列，不混则准。三，渣不漏，路才宽。」

韩泥复述，声哑，一字不落。铁无言点头：「笨人记全，比灵人记半强。」

同场杂役笑：「丑杂役也配学嗅诀？」

铁无言扫一眼：「配不配，渣场见。漏了，你配也不配。」

笑断。杂役凑近：「三句有什么用？」

「漏了，丹房不收。」韩泥说，「不收，记名滚。」

「滚了怎么办？」

「滚了，汤还不了热。」韩泥说。杂役啧一声，学他分堆，手仍乱。乱，灰扬一脸，管事骂：「学也学像点！」

杂役揉眼，不学了。午后，丹房外，药烟浓。韩泥隔街立，闭息，试诀——鼻先静，静里苦甘分列。甘短，苦长，长里带涩。涩是废渣残药，残药不能丢。白织月过，袖风冷：「站街嗅，嗅出灵根了？」

韩泥应声：「嗅路。」

白织月唇线绷一息，走了。少一句嘲，小胜。街角赖福瞥他伫立，喝：「丑杂役，站街装仙？」

韩泥挪半步：「站街记味。记错了，渣混；混了，滚。」

赖福啐，走了。啐，像怕铁无言第二眼。验场前，铁无言折返，扬声：「丙九韩泥，拈三渣，报三味。」

韩泥停，闭息，鼻先静。第一拈，苦长，蒿气；第二拈，甘短，叶家方向；第三拈，涩重，废药残。他声哑报：「苦蒿、短甘、涩废。三堆，不混。」

铁无言点头：「笨人全对，灵人常漏。」

同场杂役脸热，低头分堆，手稳半寸。管事验场：「丙九韩泥，渣清完，香篓齐。明日仍渣场。」

回场，他把涩渣并入香篓，篓满，满而不泄。同场杂役笑：「丑杂役也配跟铁师兄说话？」

铁无言远处扫一眼，声不高：「渣不漏，话就少。话多，往往是手乱。」

笑断。断，韩泥记——记铁无言第二眼，记嗅诀三句。午后末，赖福带人过渣场，故意踩乱第三堆。韩泥不骂，蹲身重分。分完，手稳，鼻仍静。铁无言折返，只看堆：「谁踩的？」

赖福脸白。白，韩泥不指——指，费舌；舌要留来报三味。铁无言翻簿：「踩乱者，明日顶酉时渣场。」

小胜落在韩泥掌上：掌不乱，堆不乱，名还在册。同场杂役低声：「踩乱也不滚你？」

「滚看手。」韩泥说，「手稳，堆齐，铁师兄看堆，不看嘴。」

夜，油灯暗，丙九棚。韩泥把嗅诀写在破布角——不写全，写「静·列·漏」三字，字丑，丑真。邻床凑看：「什么鬼字？」

「路字。」韩泥说，「路通向渣不漏，通向还。」

「就三字？」

「三字记全，比百句记半强。」韩泥说。邻床撇嘴，睡了。刘婆门外咳：「鼻下别漏。漏了，粥更稀。」

韩泥只答：「不漏。」

他把香渣篓藏床底，藏实。实，路不张扬；张扬，赖福来克扣。铁无言夜巡过渣场外，没进来，只扬声：「丙九韩泥，明日仍别漏。」

扬声远，远得像规矩比人长——长得住，路才宽。他低声复念三遍。念完，破布角上三字在，篓底香在，鼻下甘短仍在。坛壁水汽爬半寸，应「静」字，不应灵根。灵在石上，诀在鼻下，恩在心里。管事门外贴条：「冬深编筐轮换，三日后丙九韩泥。误者，记过。」

刘婆门外咳：「三日后编筐，手别裂。裂了，麸皮又浅。」

韩泥应：「不裂。嗅诀记着，筐才接得上。」

他低声复念嗅诀末句：「渣不漏，汤才热。热着还，还着活。」念完，合眼。合眼不是睡实，是等三日后藤条勒指——勒得住，筐才编得稳。坛壁水汽一线，应「编筐」，不应慌。三日后帖在柱，手在，鼻在，路不断。嗅诀三句落破布角：静、列、漏——丹道辅线起，不越阶，只铺瓮醒前鼻下路。铁无言夜巡后再止，止在渣场外，扬声：「丙九韩泥，破布别示人。示人，像求。」

韩泥只点头：「不示。示人，嗅诀就飘。」

扬声远，远里管事又贴细条：「三日后编筐，掌裂者备布。误卯时，记过。」

同舍有人梦呓「热」，韩泥不答，只按破布角——角旧，字丑，真。真在「不漏」：不漏沿，不漏渣，不漏恩。他低声道：「三日后筐，七日后香。香净了，才近栖香谣。」

卯时前，渣场雾白。韩泥先到半刻，把三堆再理一遍。理完，鼻下静，苦甘分列，涩渣入篓。同批杂役来迟，喘：「你又早？」

「早，手稳。」韩泥说，「稳，铁师兄才翻簿。簿上有名，名在末，也在路。」

铁无言到时，只看堆，不问人：「丙九韩泥，报三味。」

韩泥闭息，一字不落。落完，铁无言点头，册上画圈——圈小，真，像嗅诀起头。同场杂役脸热，低头分堆，手稳半寸——稳半寸，也是小胜。小胜落在掌里，像破布角三字又实一分。实一分，三日后编筐才勒得住指。管事验场，点头：「明日仍渣场。三日后编筐，别误。」

韩泥应一声，肩稳。稳，像破布角三字在怀，在，路就不飘。同批少年递绳：「三日后编筐，我帮你捆。手别裂。」

韩泥低声应：「捆。裂了，嗅诀白记。」

绳结实，实像三日后的藤条——藤条勒指，勒得住，筐才编得稳，香才问得准。赖福远处骂一句，骂不重——重的是韩泥没回头。回头像认谣，谣借腥，他不碰。坛腹微温一丝，应「静」字，不应慌。

午后，渣场日头斜，斜光照苦堆，苦堆边甘短一线。韩泥蹲半刻，鼻下先静，静里才把涩渣拨进香篓——拨得准，准得像拈叶家药须。同批杂役捂鼻路过，仍嘟「晦气鼻」，却没人再嚷他漏渣——嚷不动了，嚷不过铁无言册上那一圈。圈在末，也是名；名在，路才不飘。

刘婆提粥远逝，粥香淡，淡不过鼻下甘短；甘短记着，像记着叶家方向。方向远，还字不远。他低声复念嗅诀首句：「鼻下静，苦甘列。」列清了，三日后编筐手才不慌；慌了，藤刺勒指，血蹭坛，不像丑翁。坛壁水汽应「静」，不应灵根——灵在石上，诀在鼻下，恩在心里，在心里才还得热。

同舍有人翻身，翻身带起一股药烟味——味苦，苦里一点甘，甘短，像叶家方向。短，也记；记短，才不混长苦。酉时末，他把破布角再塞深一寸，塞得像把嗅诀藏进掌纹。掌纹丑，丑得真；真掌，才拈得准渣。三日后编筐帖在柱，手在，鼻在，路不断。铁无言夜巡后再止，止在渣场外，扬声：「丙九韩泥，明日仍别漏。」韩泥只点头。

扬声远，远得像规矩比人长——长得住，路才宽。嗅诀在怀，怀在掌，掌在明日藤条下。稳。等编筐。
