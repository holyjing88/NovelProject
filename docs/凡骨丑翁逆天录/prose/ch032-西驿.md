# 第三十二章 西驿

西驿在坡下，坡下人稠。稠的是赶考的、赶药的、赶宗的。韩泥背篓入驿，篓沉，肩稳，稳得像泥岗村那两只筐——筐卖过钱，钱买过米，米养过命，命走到这，不算白走。驿门有吏查路引。泥岗村无引，有里正荐帖抄本。吏展帖，瞥「言伯钧」三字，眉一抬，抬得像看见缝：

「凡骨？」

点头。「五十八？」

吏敲桌：「排队。别插队。插队，滚。」

韩泥排队。队里少年多，骨嫩，眼亮，亮得像嫌他老。老，他不辩。辩骨，像求；不求，只等。候到午后，验骨。木牌凉，贴掌，掌糙，糙得牌滑。执事弟子瞥他麻脸，鼻下哼一声：「凡骨，下品里的下品。」

韩泥点头。弟子恼：「去杂役处登记。别挡测灵根的人。」

旁侧少年测灵，灵光一闪，青白，白得像新瓷。瓷亮，亮得众人「哟」一声——哟声落，韩泥已转身，转得像避光。光不刺眼，刺眼的是比：比不过，不比。这是辱，辱在宗门门口，味同泥岗，只是换了衣裳。韩泥不辩，只记——记在心里，不写在脸上。脸丑，丑惯了。登记处，簿上写：

韩泥，五十八岁，凡骨，泥岗村，言伯钧荐，杂役试录。写罢，掷一枚木牌：丙九。「丙九杂役房。」吏道，「明日卯时，山门侧门入。今夜宿驿西棚。棚费三文。」

韩泥付三文，三文从泥岗带来，带来得像根。小胜在：没被拒在门外。门缝窄，窄也进了。验骨前，队里有人嘀咕。「五十八也排队？占名额。」

韩泥不辩，只把木牌抄本收好。收好，像收绳——绳拴命，命拴门缝。前头少年测完，灵光青白，执事弟子赞一声「中品」。赞声落，少年胸挺，挺得像门已开。韩泥瞥一眼，不比。比是少年，他不比。轮到，执事弟子不耐烦：「贴掌。别磨蹭。」

韩泥贴掌，掌糙，牌凉，凉得像泥岗冬井。弟子瞥结果，鼻哼：「凡骨，下品里的下品。下一位！」

旁侧有人笑：「这种也进宗？晦气。」

韩泥转身，转得像避光。光不刺眼，刺眼的是比——比不过，不比。登记处前，又有人拦：「言伯钧荐的？那老儿云游去了，名额作数？」

吏翻簿，翻半息：「作数。抬棺手稳，记过。」

韩泥应声，不谢言伯钧。谢轻，记——记门缝是言伯钧名开的，名在末，缝在。棚里潮，潮里汗酸。邻铺少年嘀咕：「五十八做杂役，占棚位，晦气。」

韩泥铺席角，席角潮，他不嫌。嫌，像少年。少年不是他，他是记恩记辱那一个。他掏叶振东药包，拣一叶，嚼，嚼得苦，苦得清醒。清醒里，摸坛沿——坛在篓底，沿温一线，温得像应「丙九」二字。丙九是末，末不是绝，是缝。外头驿铃响，响得像山门在远处叩。邻铺少年又嘀咕：「听说言伯钧长老年少云游，留过名额给抬棺稳的人……这老丑，抬棺？」

另一人笑：「抬棺手稳，炼丹手未必稳。」

韩泥闭眼，不辩。辩，费舌。舌要留命用，命要活长，活长了，才还得尽泥岗六笔恩。棚外有人卖热饼，饼香，香里他想起张麻婆的馒。馒恩在，饼不买——买饼像馋，馋轻，恩重。驿吏夜巡，敲棚柱：「丙九韩泥，明日卯时集合。迟到，除名。」

韩泥只答，应得像泥岗里正敲锣——锣哑，也得听。他低声道：「明日入山。入山，不回头。」

坛沿温一丝，温得像耿恩在底，经片在底，叶汤在胸——在，就够睡。夜深，驿外马蹄。有人喝：「让路！沉丹宗内门弟子过！」

棚里少年涌出去看，韩泥不起。起，像凑热闹。凑热闹要还人情，他还不起。独坐，听马蹄远，远成钩。钩向山门，钩向杂役房，钩向瓮醒那日。他按心口，心沉，沉得像六笔恩又加一笔：宗门收了他，收不是恩，是路。路开了，还才有道。丙九木牌搁膝上，牌凉，膝热。热在心，心在，丑翁便不走绝路。天将亮，邻铺少年催：「老头，起！卯时点名！」

韩泥已醒。醒在丑时，醒得像泥岗擦坛——习惯在，根在。他背篓起身，篓底坛闷，闷得像凡物，像耐心的等。等瓮醒，等还。山门在雾里，雾里丹香一线，线细，够引凡骨钻。驿棚外，白织月影子过，过得快，像内门弟子。弟子衣鲜，他不比。比不过，不比——比是少年，他不比。邻铺少年嘀咕：「丙九，末席中的末席。」

韩泥闭眼：「末席，也是席。席在，饭在。饭在，命在。」

少年不懂，翻身睡了。睡，韩泥不睡实——实睡，像认输。丑时将近，坛沿温一线，温得像应「入册」。册在，名在末，末不是绝，是缝。卯时前，驿棚外集合。执事弟子点名，名到丙九，韩泥应「在」。在，像钉——钉在西驿，钉向山门。弟子瞥他：「凡骨，侧门入。别走正门，丢人。」

邻铺少年凑近：「你真五十八？」

「不怕兽栏臭？」

「臭过泥岗井台。」韩泥说，「臭，不算事。事是活。」

少年缩回去，像被「活」字堵住。点名毕，弟子掷单：「丙九韩泥，兽栏清粪，入门先练手稳。」

韩泥接单，单旧，字丑，丑得像里正荐帖——帖让他进门，单让他干活。活，先活下来。篓底坛闷，闷极轻，轻像应「兽栏」二字。应步，不应名。名未醒，步先稳。他随队出棚，雾厚，丹香一线，线细，够引凡骨钻。钻，不是钻光，是钻缝——缝在末席，末席也是席。弟子又喝：「凡骨，走路靠边。别挡内门弟子。」

韩泥贴边，让。让半步，够——够活，够看，够记。邻铺少年悄问：「你真不怕？」

「怕。」韩泥说，「怕恩断。恩不断，怕也要走。」

少年不吭声，像被「走」字堵住。执事弟子点名末，又瞥丙九队：「凡骨记住，侧门入，别闯正门。闯了，打断腿。」

有人悄问：「五十八，兽栏臭，你图什么？」

「图活。」韩泥说，「活长了，还恩。」

「还什么恩？」

「泥岗六笔。」韩泥独眼平，「宗门一笔，另算。」

问的人闭嘴，闭嘴，像被「另算」堵住。驿吏夜巡，敲棚：「丙九，明早卯时，别迟到。」

邻铺少年又嘀咕：「你坛真不扔？」

「不扔。」韩泥说，「坛在，根在。」

「赖福说晦气。」

「晦气在嘴。」韩泥独眼平，「嘴贱，手稳就行。」

少年翻身睡。韩泥摸坛沿——沿一线水汽，应「丙九」，应完又敛。同棚又有人问：「言伯钧荐的，真作数？」

「作数。」韩泥说。「云游去了，谁认？」

「册认。」韩泥独眼平，「册在，名在。名在末，也是名。」

丑时前，他把荐帖抄本又叠一遍，叠得像把门缝压实在怀底。压实了，言伯钧名在末，缝在。缝在，丙九才作数。作数，不等于轻——轻了，像欺恩；他不轻，只等卯时点名。册落「丙九杂役」，字丑实；韩泥应一声：入门不是仙，是活路——活路通侧门，侧门通兽栏，手稳先分活，分活，才见挡石誓。邻铺少年梦呓：「丙九……末席……」

韩泥闭着眼：「末席，也是席。」

驿吏又敲：「迟到除名！」

韩泥只点头：「不迟。」

邻铺少年问：「侧门入了，还能辞村吗？」

「能。」韩泥说，「辞村在前，还恩在后。先后不乱，路才真。」驿西棚里，更鼓将尽。更鼓沉，沉得像泥岗丑时擦坛——习惯在，根在。韩泥摸木牌，牌凉，凉不过心口六笔恩。恩背着，肩才稳；肩稳，侧门才进得。侧门窄，窄也进；进了，才有丹香引路，才有兽栏练手，才有将来还恩的秤。坛沿温一线，应「丙九」，不应名。名未醒，步先稳。稳。等卯时。
