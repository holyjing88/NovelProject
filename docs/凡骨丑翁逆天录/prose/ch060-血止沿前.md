# 第六十章 血止沿前

第六十日，雪细。雪细像粉，粉在肩，肩沉，沉得住。

韩泥清西栏，粪锹沉，掌茧裂口渗一丝潮。潮不是血，血不碰沿。

卯时前，他又去搬柴半刻。霜厚，柴湿，斧声闷，闷像更鼓。

管事验垛：「沉得住，记功在。」

记功在，麸皮不克。克粮，克不走手稳。

午后编筐，末藤刺深，血珠再饱满。他甩手，血甩破席，席吸尽，沿仍净。

净，像瓮守的最后一夜前奏。

坛腹温加剧，比廊下讲规矩那日又多一线。又多，像将醒未醒，醒在缘，不在人催。

秦霜远廊经过，一句：「手稳，心别邪。邪，符灭。」

符温贴腕，记着七笔，不往邪路偏。

铁无言立柱边，声低：「沿净，好。净，像等缘的人。」

韩泥只答：「等。」

等，不是拜血祠，不是主动滴血。是活计扎，血找席，席挡沿，沿等韩氏守瓮血那一误。

同批少年问：「坛温加剧，像醒吗？」

「像眠。」韩泥说，「眠着，不急醒。」

「急醒咋办？」

「急，像拜邪。」韩泥说，「拜邪，路断。」

少年懂「断」，闭嘴。

老耿挑水路过，水洒稳：「六十日了，还守沿？」

韩泥只点头：「守。守席，守缘。」

老耿不再言。言少，水稳，像褐袍人当年嘱托的余音。

管事查棚，灯晃坛角：「杂物收起。」

韩泥席盖不动。动像心虚。不虚，是守。

管事盯半息，走了。只见破席，不见仙缘。

邻床笑：「坛角盖那么严，藏宝？」

韩泥不辩，只把席盖又按实。按实，像编筐扎掌那日。

掌茧再裂，裂在冬，不在沿。

刘婆端姜汤，姜薄，热：「六十日了，沿还净？」

「净。」韩泥说，「净，才接得住后头缘。」

他喝尽，只点头。姜热，像灶下还着的一星火。

赖福讽：「守席守出仙缘？」

管事路过，声冷：「再嗓，克半月麸皮。」

赖福哑了。哑，韩泥按席的手却更稳。

夜，雪停，棚静。他揭席一角，看沿。沿尘，沿滑，沿上无血。

无血，盖严，盖得更严，像席盖更严预演。

他低声道：「血止沿前，瓮温加剧。加剧，不急喊醒。」

掌稳，心稳，符暖，席严。外头更鼓二响，响沉。

明日仍搬柴。搬柴，藤刺还会扎。扎了，血还会到席。席还会挡沿。挡到那一夜，缘才满。

满到误触那一滴，瓮才醒。

邻床鼾雷，雷里，坛角静。

他低声对坛：「你再眠半夜。半夜后，缘满不满，看活计，不看心急。」

坛腹温一线，又凉。凉，像应「等」。

六十日擦沿，六十日不主动滴血。滴，要等误那一甩。

同批少年又碰他袖：「你守坛六十日了？」

「守。」韩泥说，「守席，守沿，守不主动滴血。」

「滴血干啥？」

「主动滴，像借腥。」韩泥说，「腥借来，路易断。」

他按心口，低声道：「丫头那碗汤，热着还。还，要等瓮醒后，路才长。」

编筐末藤，藤刺再扎，血珠饱满。甩手，血落破席，席厚，吸得尽。吸尽，沿边仍无尘。

管事查沿，烛晃一线：「六十日，沿要更净。净，才接得住瓮温加剧。」

韩泥用旧布再擦沿一圈。擦罢，沿滑，沿尘，沿上无血。

同批少年蹭过来：「坛温加剧，你怕不怕？」

「怕。」韩泥说，「怕也记。记着，手就不飘。」

少年一愣，退半步。退，像懂「怕也记」。

管事验垛，斧痕新，柴纹直。直，像手稳留下的印。

韩泥肩勒旧印，印深，沉得住。沉得住，瓮温加剧才好接。

同批少年夜间问：「六十日，你图啥？」

「图沿净。」韩泥说，「净了，才接得住误那一甩。」

少年不懂「误」，却懂「净」，闭嘴。

他撕旧袄角，补席边破处。补罢，席更厚，厚像后头那一夜。

席厚，血好找席；席好找，沿才好接沿前那一止。

扫罢，又去搬柴一担。柴沉，肩勒旧印，印深，像丙九五年。五年磨出的肩，才扛得住瓮温加剧。

符怀暖一丝，护骨寒，掌裂不僵。僵，下一刺便偏；不偏，缘才正。

他按心口，七笔分列，不混。混了，手就飘。飘，像借腥。腥不借，路才长。路长，才还得上汤。

他摸木牌，七笔分列。叶丫头汤，振东药，陈姑袄，刘婆粥，符恩，耿恩，僧恩。笔笔在，手才不飘。

卯时前搬柴半刻，霜厚，斧声闷，落得像把心口七笔一下一下钉实。

黄昏，雪停，他扫棚前雪半刻。雪薄，帚短，扫出一路净。净不要紧，沿要净。

铁无言路过，看一眼肩：「六十日了，肩还稳？」

「稳。」韩泥说，「稳，才接得住沿前那一止。」

铁无言不再言。

他指节叩木牌，叩出七笔分列。七笔分列，不混。

他盖席，盖严，破布又加一层。层厚，厚像后头那一夜预演。

他低声对坛：「六十日了。六十日沿净，才配说你暖加剧。加剧，我仍不喊醒。」

坛腹颤极轻，轻像应「等」。止得住，才配得上瓮温加剧那一夜。

二更，他揭席一角，看席边。席边湿一线，湿痕止在沿前三指。三指外，沿净。

他对坛低声：「血止沿前，席严，符暖，手稳。稳，才配接席盖更严那一夜。」

坛颤极轻。

三更将尽，雪又细一丝。细不要紧，席要紧。他把破席又加一层角，角压住沿边湿痕，压得像给后头那一夜垫一道坎。

同批少年翻身，梦中喊「滚」。韩泥不扰，只低声：「滚在舌上，不在沿上。」

坛腹温加剧，他掌心贴坛底一瞬，贴完即离。离，像敬，也像守——守规矩，不借温拜邪。

四更，铁无言巡廊而过。灯晃坛角，晃一下便远。远，像说「沿净，手稳，够了」。

他独眼合半息，不合实。合实，明日搬柴手就沉。沉得住，才配得上瓮温加剧那一夜。

五更，天边白一线。白像席盖更严的前奏。前奏里，更鼓尽，尽处一声鸦叫，叫短，叫得像缘将满未满。

秦霜远廊一句：「瓮温加剧，别拜。拜，符灭。」

他低声：「加剧是缘，不是命。缘在，七笔就在。」

雪细停，棚静。静里更鼓远。远，像瓮将醒未醒，醒在缘，不在人催。

席盖更严那一夜，已在眼前。眼前不要紧，沿要紧。沿净，才配接。接得住，才配醒。

五更末，他按席，席厚，沿净。沿净处，坛腹温一线又凉回去——像六十日守出来的界，界内席严，界外缘候。候的是活计误那一甩，不是人喊醒。

六十日清晨，他扫棚前雪，雪下露出一块旧砖。砖冷，冷像泥岗井台——井台远，冷还在记里。记里，沿净，席严，才配接瓮温加剧那一夜。

祖父嘱过：「席厚，沿净，韩家不散。」嘱在耳，他加第四层破布。布旧，旧更能吸；吸的不是仙缘，是活计扎出的血路。

管事查棚第二趟，灯照席角，照见厚席不见坛。不见不要紧，沿要净。沿净，管事走了，像默认守瓮也是杂役活计里的一种。

雪停后，他搬柴半刻，肩勒旧印，印深，沉得住。沉得住，瓮温加剧才好接下一夜席严。沿净在前，醒在缘，不在人催。

六十日午后，他编筐末藤，藤刺偏深半分，深半分，血珠更饱满。饱满，仍甩向破席，席吸尽，沿仍净。净，像给后头那一夜预演——预演，不主动滴血；滴血，要等活计误那一甩。甩在缘里，不在舌上。

管事查棚第三趟，灯照席角，照见厚席不见坛。坛不露不要紧，沿要净。沿净，管事记簿一笔，笔丑，真在「六十日」——六十日沿净，才配说瓮温加剧。加剧，他仍不喊醒；喊醒，像拜邪。拜邪，路断。席严一线，才接得住下一误。
