# 第七十三章 兽栏杂务夜

冬夜更深。丙九排了夜值，值在兽栏，值在腥与粪之间。韩泥接牌，牌木，木旧，字浅：「丙九韩泥，戌至丑，清栏换草。」

赖福瞥牌，眼细：「夜值也给你？末席专干晦气活。」

韩泥不辩。辩，费舌；舌要留来清栏，留来守坛。他应一声：「在。」

戌时初，栏里灯豆两盏，豆光黄，黄得像丙九的饭。饭稀不要紧，活要细——细了，管事才不记名。小兽角里咳一声，咳轻。韩泥换干草，草干，味淡一寸。淡不要紧，鼻下静一息，静里闻见粪里一点药回——回得像白日渣场那撮辛。第二趟清栏，雪混粪，铲下仍稳。稳，像踹台按实那一掌——掌在，名在，路在。同值少年掩鼻：「丑杂役，夜值你也接？」

「接。」韩泥说，「接才有饭。饭在，恩才还得上。」

少年退半步，没再问。问，像沾晦气。晦气不要紧，晦气遮瓮。管事弟子巡栏，灯照角里：「湿粪换净。换不净，记名。」

韩泥应声：「换净。」

换净，是小胜。胜在活，不在嘴。管事又验水桶，桶沿净，水浊而稳。稳了，才记「丙九韩泥，夜清细」。子时，风硬，灌进栏角。灌不要紧，肩勒旧印，印深，沉得住。沉得住，掌纹那路才不飘——飘了，粪铲就偏。栏外铁无言抱臂立，声低：「夜值别漏香。漏香，像求。」

韩泥指稳：「不漏。鼻下静。」

铁无言点头，走了。走，也像认可——认可不重，重的是手不抖。刘婆提灯过，灯昏，照见韩泥肘袄针脚：「夜值也漏风？」

「漏一线。」韩泥说，「清栏挂袋，挂的。」

刘婆哼：「挂完回去补。补完，别谢。记着。」

韩泥点头。谢轻，恩重。恩在针脚里，也在夜值这碗稀粥前头。丑时前，栏角最静。静里，小兽齐睡，睡实了，铲声才显得沉。沉不要紧，韩泥趁静，席下掌贴坛沿半息——沿温一线，应夜，不应字。他不揭坛，只让鼻下息贴近沿边。沿边无香，有凡气；凡气里药香回涌，回涌像一层在活。邻值少年打盹，盹浅，头一点。

韩泥不喊，喊，全丙九记名。他多铲半刻，替少年角里清净——净了，像还刘婆一勺粥恩。更鼓二响，管事来验：「丑时末，角里仍湿？」

「干了。」韩泥说，「草换两遍。」

管事记簿，声平：「丙九韩泥，夜值细。」

记细，是小胜。赖福立栏外，眼细，像嗅：「夜值也嗅？嗅晦气？」

韩泥不接茬。接茬，全丙九饿。他低声道：「鼻下静，手在。」

兽栏夜，粪铲落栏，落得要实。实了，二层气才稳；稳了，鼻下才静。静里，粪里一点药回，回像65那日嗅渣——渣不要紧，回进掌纹，回进暗修那路。更鼓二更，小兽角里咳两声，咳重些。重不要紧，韩泥添草，草干，又添水。水少，少才显勤；勤在册，勤不在舌。舌响，赖福记「邪」。丑时前，他廊下立半刻，任腥风硬。

风硬不要紧，掌贴沿，沿温一线。温不应腥，应「忍」——忍里藏刃，刃对着三年约，对着钱戾衡，对着内门的眼。赖福盯半息，终是没掀席——掀不得，沿净，坛不露。不露，像腌菜坛，不像邪。丑时尽，天边白一线。白里，韩泥收铲，肩酸，酸不要紧，手不抖。不抖，夜值才算完。回棚路上，药渣场角风硬。

他多留半刻，分拣一撮灰，灰里一点辛。辛正，他鼻下静，静里辨出苦与辛——苦沉，辛浮。样纸包实，塞怀。怀暖半分，暖不应符，应「记」。记牢了，嗅诀才生根；根在，药山才走得深。邻床问：「夜值累不累？」

「累。」韩泥说，「累也清。清完，才有饭。」

少年闭嘴，像也学会忍。忍不是怂，是活路——活路通向暗修，通向药山，通向还恩那日。他按席角，席严，坛不露。坛腹温一线，贴掌，像应「夜清」二字。应完，他合眼不睡实，鼻下仍静一息。外头鸡未鸣，沉丹宗夜禁脚步远。钱戾衡在玉牌里，在三年约里——约还在，夜值先做完。做完，才配暗修二层那一步。他低声对坛：「兽栏夜清了。一层在掌。二层在远，手在近。」

韩泥只答：「不拜。坛在，人在。」

更鼓沉，沉里，他摸怀木牌，牌凉，膝热。热在坛，热在路，热在夜值记下的那一笔「细」。独眼平，手不抖：「夜值了。嗅到了。明日药山，渣里活。」

刃不亮，先记。记完了，才配丑时暗修，才配接得住二层微进。柱上条角白一线，冬深，像正测已过，像三年才开头。开头不要紧，手在；手在，帖在，名在册。他廊下练半刻闭息，息在鼻，不在嗓。嗓响，香飘；香飘，煞骨门就嗅得到。嗅得到不要紧，沿净，壳才像腌菜坛。白织月远廊经过，袖风冷，没停。没停，小胜。

秦霜木鱼远，一声，像提醒：心别邪。韩泥不答，答在心里——心里恩辱分列，夜值是活，活是路；路通向药山捡渣，通向废渣炼丹，通向95那盏门缝灯。坛腹应温，不应字。温，像夜栏里那一点药回，回进掌纹，回进一层境里。境微，不要紧；微了，才接得住二层。他低声道：「夜清了。坛应了。暗修在丑时，二层在掌前。」

二层在掌前，不在舌上。舌响，香飘，玉牌白接。不接笑，只接手——手稳了，夜值才不是白值。散队后，他编筐末单，藤刺扎掌，偏手，刺浅些。编筐钱买米，米在，恩才还得上。同值少年打盹醒来，愣半息：「我角里……干了？」

「干了。你睡，我铲。」

少年脸热，没谢——谢像欠。欠不要紧，韩泥不记欠，只记夜值那一笔「细」。沈枯芽远廊经过，步子轻，像也没看见。没看见不要紧，韩泥不近——近，像求；不求，只记半块灵石恩在还路前头。他摸怀木牌，凉进骨。凉不要紧，掌里一层温——温在暗，不在众目。众目看灰线，掌里多了一尺暗线将开。

戌时末，栏角水桶翻半寸，水泼，泼不要紧，他舀净，舀得像还刘婆一勺粥恩。舀净了，管事才记「角净」——角净，是小胜。小兽里有一只独眼兔，眼浊，浊不要紧，它舔他靴边，舔轻，舔像认人。认人不要紧，韩泥换草，草干，味淡一寸。味淡了，栏角才像能睡。同值少年打盹，头一点一点。

他多铲半刻，替少年角里清净——净了，像还恩，也像忍。铁无言又过，脚步停半息：「夜值别省水。省了，腥更重；重了，瘴近。」

韩泥应声：「不省。」

不省，不为显勤，为栏活。栏活，丙九才不被连坐。子时二更，风灌栏缝，缝响，响不要紧，他肩勒旧印，印深，沉得住。沉得住，粪铲才不偏——偏了，管事记「飘」。丑时前，他席下掌贴坛沿半息，沿温一线，应夜，不应字。更鼓三响，天边白一线。白里，他收铲，肩酸，酸不要紧，手不抖。坛应了。天白，韩泥回棚，肩酸，酸不要紧，手不抖。完在册，完在管事笔尖。邻床问：「兽栏夜值咋不睡？」

「睡实了，秘密散。」韩泥说，「散不得，坛在。」

少年不懂「秘密」，却懂「坛在」，闭嘴。他换干衣，衣潮，潮不要紧，沿要净。沿净，像腌菜坛，不像宝。不像宝，赖福才掀不得席。午后，药渣场风硬，他多留半刻。留，不为赏，为鼻下根——根粗了，夜值才不虚。铁无言指一堆灰：「夜值后鼻易飘。飘了，腐当辛。腐当辛，手就滚。」

韩泥分拣，指稳，分出腐与辛。分出，铁无言点头，走了。刘婆端粥，仍多半勺：「夜值饿不饿？」

「饿。」韩泥说，「饿也清。清完，才有饭。」

「饭稳，恩不断。」刘婆哼，转身走了。韩泥喝尽粥，粥稀，稀不要紧；稀也能暖腹。暖腹，鼻下才静，静才接得住二层暗修。白织月远廊过，没停。秦霜木鱼远，一声。赖福秤杆落盘，落轻，落轻不要紧，眼仍细。韩泥不辩，只按木牌。木牌旧，字深：「仍末。」仍末，他熟。熟不等于认命——命在坛，坛醒了，夜值也是路。

路通向药山，通向废渣，通向95门缝灯。灯在远，手在近；近在，就不滚。丑时前，他再清栏半刻，替同值少年角里湿处。替，不为名，为恩——恩在刘婆粥，也在丙九不连坐。管事夜巡，灯照栏角，角净，记「夜清细」。记细，是小胜。他回棚，按席角，席严，坛不露。坛腹温一线，贴掌，像应「夜」字最后一笔。低声道：「夜值了。嗅到了。明日暗修二层，仍不叫。」
