# 第十六章 跟风

公审日未定，风先定。

周癞子醉，醉到韩泥门口，砸门，砸得响：「晦气翁！赖管事说，你克全村！克到公审都要开！」

韩泥开门一尺，独眼平：「公审开，开的是谣，不是我。」

周癞子红眼：「谣？叶寡妇送你汤，睡没睡，谁知道！」

这话毒，毒到连醉鬼都信自己的嘴。

韩泥不辩睡没睡。辩，成戏。他只道：「睡没睡，天知道。汤热不热，我知道。」

周癞子愣半息，骂：「老不死的，装圣！」

扬手要打，韩泥抬臂挡，布条下伤未愈，疼一抽，他没退。

「打。」他嗓哑，「打完，去祠堂说。说你有证，我认。没证，认谣。」

周癞子手悬半空，像没料到他不躲。躲，像怕；不躲，像心记着。记着压人，压到醉鬼手落不下，落不下，就骂一句「疯子」，晃远。

这是小胜：硬扛，不伤她，不伤叶，只伤谣的胆。午后，周癞子醒半分，跟赵绣春党在井台复读：「叶家寡妇，晦气！」

韩泥打水，桶满，不溅。

周嫂递眼色，周癞子来推他肩，推不动——筐肩练出来的劲，不是醉劲。

「推我，不如推井绳。」韩泥道，「绳断，你渴。」

周癞子啐他脚边，啐不到人，像唾沫自辱。

周嫂帮腔：「韩泥，你硬，硬到全村跟着受罪！」

韩泥拎桶：「受罪的是病。病等着抓药，不等你的棒槌。」

周嫂棒槌一紧，没砸——砸，里正记名。记名，公审前，胆先缺一截。

村人见，见晦气翁不倒，倒的是周癞子的步。步乱，像跟风踩空。

马春花远处看，看两息，没帮腔——帮腔要力，力要胆，胆被公审钩住，钩住便少一截。

赵绣春却近前，嗓尖：「公审辨了，你也跑不了！」

韩泥拎桶，水不洒：「跑不了，就不跑。站给全村看。」

赵绣春唇动，没再尖——尖需要众，众散一半，尖便短。

井台边，童谣丫被娘拽住，仍探头。探头里，念半句，又卡壳：

「叶家铺，晦气铺——」

娘捂她嘴：「再念，回家挨揍！」

童谣丫眼圆，圆里怕，也像好奇：好奇晦气翁怎么还不倒。

韩泥路过，不停，只道：「念完再念，嘴累。累的是嘴，不是铺。」

童谣丫挣出半声：「翁……狼……真打？」

「真打。」韩泥道，「牙怕棍，人不怕谣。」

娘拽紧，拽远——远，风仍吹，吹不到娃心里那一寸胆。

韩泥回屋，心口四恩在，赵辱旁，他又记一辱——辱增，不写名，写风。

风无姓，风有痕。痕在心，不在口。

「周癞子，记着。」

他低声道，「跟风的人，风停，先摔。」

坛沿微凉，心口微热。热里，他想起公审未定日——未定，谣更急；急，就更要记恩，不能乱拔仇苗。

夜里，沈家咳两声，咳得轻。轻，像芽儿娘还活。

韩泥合袄，塞怀，望叶家药铺方向，灯弱，仍亮。

亮着，叶汤就不倒。

风过村道，童谣丫不念了——不是怕，是娘拽回。拽回，风仍吹，吹到告示桩「连坐」二字，字黑，像等下一阵更醉的嘴。

韩泥不追风。追风像少年。他只记辱，候公审。

辱二，比赵辱短，却更硬——硬在跟风者名虽不写，形已在。形在，将来公审上，一并算。

陈姑远处咳，咳两声，像提醒：还活着的人，别被风带倒。韩泥「嗯」一声，应给自己。

井台边，周癞子又晃来，想再推，见韩泥桶满，推不动，自骂「晦气」，晦气落在自己脚面。

韩泥打水，水浑，稳。稳，像告诉跟风者：井还在，路还在，公审之前，他还不倒。

心里辱二，干硬，像风干的仇。仇不干，将来拔苗，一手滑。

周嫂过巷，嘴还想动，见韩泥心口沉，沉得像有凭。凭不在嘴，在四恩——四恩像四张证，证公审来，翁不怕空身去。

风停一瞬，停里，叶家灯仍亮。亮，叶汤还在。还在，跟风的人，先摔一半胆。

夜里，叶铺秤又落一声。

落声远，远过周癞子骂街，远过赵绣春门内摔碗。韩泥在屋听，不近，近像牵扯。听，就够——够知叶家秤还在，够知公审前叶汤不倒。

周癞子又晃井台，晃空，啐空台，啐像啐自己。

马春花远处嚷：「公审开，看你还硬！」

韩泥开门缝一线：「硬的不是我，是汤。汤热，你嗓尖也尖不凉。」

马春花噎，嚷声收——收，像风又断一截。

坛沿微温一丝，温应秤声，不应骂声。骂声在舌，秤声在理。理在，翁便候公审。

他低声道：「辱二记着。恩四笔，一笔不能浅。浅了，公审白开。」

风再起时，村人分三拨：一拨跟赵绣春骂，一拨跟里正等，一拨只绕叶铺走。

绕铺者停门阶外，低声：「叶掌柜，明儿还开？」

叶青禾帘内：「开。病不等公审。」

问者「嗯」一声，步快，快里像松半口气——气松，风就短一寸。

周癞子醉醒又晃，晃到告示桩，读「连坐」，读得舌打结：「连……连我？」

韩泥路过，独眼平：「连不连，公审判。你先醒。」

周癞子啐自己脚面，啐像自辱——辱二记在心里，形在，名不写，写风。

赵绣春想借风再尖，尖到一半，见里正锣在巷口，咽回去。咽回去，也是钩：锤未定，人心已定一半。

沈家咳两声，咳得轻。轻，像芽儿娘还活；活着，跟风的人就先摔一半胆。

叶铺秤又落，落声压风。压得住风，压不住公审——公审来，恩辱一并算。

坛沿微温应秤，不应骂。骂在舌，秤在理，理在，翁便候。

韩泥低声道：「风停，摔的是跟风的。站住的，是记恩的。」

风尾一过，村静半息。静里，叶铺灯还亮，亮得像给公审留一盏烛。

里正锣余音又荡一回，荡得短：「霜降后第三日卯时。少嚼。」

韩泥不追风，只记辱二。记完，臂布下伤还疼——疼，像提醒：第三日，坐末席，听锤。

风再起时，童谣丫不念了——娘拽回，拽得住嘴，拽不住帖。帖在，公审在。

童谣丫念半句，张麻婆帚飞：「再跟风，摔的是你！」

韩泥路过，不高声：「站住别滚。滚的，是谣。」

祠堂门侧，里正又敲锣半声，收住：「公审帖，霜降后第三日卯时。今日起，各家少嚼。」

半声锣，像钉。钉住了跟风人的脚，也钉住了韩泥的候。

巷口，昨日跟风嚷「晦气铺」的汉子低头，放一竹篮鸡蛋在叶铺阶前，声小：「昨嘴贱……叶掌柜，还开铺不？」

汉子步快逃走——逃，像跟风先摔一截胆。

叶青禾隔帘对韩泥方向，声低：「蛋我收。铺还开。公审上，少一张邪嘴。」

韩泥路过，不停，只道：「胆摔了，汤别砸。砸秤的，帖上见。」

张麻婆帚过巷，骂：「再跟风，我先打腿！」

周癞子醒半分，听见，脸涨，涨得像昨日推不动的桶——推不动，就摔自己的胆。

韩泥不笑。笑轻。他只低声道：「霜降后第三日卯时。帖在，脚就站住。」
