## 五、扩写铁律

1. 新增人物：**姓+名**不得与还礼表、上表撞全名；避免同姓同字辈（如再出「霍×山」）。  
2. 新增教派/地名：须带**灯/焰/照/守**语义（天煞门、守夜林为已锁定例外）。  
3. 登记符箋称 **灯符册**；账本称 **灯箓账**。  
4. 洞府称 **照心斋 / 守灯静室**，不写「洞府」「仙府」。  
5. 地图命名：**×灯位图 / ×库图 / ×路线图**，与剧情功能绑定。
