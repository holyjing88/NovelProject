# 第一百四十二章 再谢老耿

药山废窑夜里落着细雨，雨脚密，窑口红得沉。

韩泥把门掩严，柴挡卡死，像把外头闲话全关在缝外。炉灰清完，投序按页：灵砂先，药渣后，沉腐浮落齐。齐了，鼻下才静；静了，火才肯听。

铁无言立在窑外十步，不进门，只丢一句：「今晚别贪粒。贪，味就杂。」

「不贪。」韩泥说，「只取正。」

「正了就送。」铁无言提醒，「别自己跑泥岗。跑了，赖福鼻子就醒。」

「经芽儿手。」韩泥应，「我不进门。」

第一炉起，香淡淡浮上。他半阖独眼，听炉里那点噼啪，像听五年前淘渣时的响。响稳了，心才稳。散成三粒，粒丑，泥纹浅，嗅之辛淡——正。

他只取一粒，其余两粒埋灰。取多像贪；贪了，恩就变成买卖。灰布囊旧，旧得像当年经片那一包，线脚歪，却结实。

窑外沈枯芽蹲在坡影里，衣裳湿半边，手却伸得很直：「韩叔？」

「在。」韩泥递囊，立在三步外，「培元散，一粒。给耿爷。经你手，我不进门。」

芽儿接囊，指尖抖：「叶姐说……门缝开一线就行。您别近。」

「不近。」韩泥声平，「近了像攀。攀，她难做人。」

「我懂。」芽儿咬唇，「我也难。可这粒得送。」

「送稳。」韩泥看着他，「绳不抖那天，他帮过我。粒不正，我宁可毁。」

「嗅着呢？」芽儿把囊凑鼻前又缩回，「我闻不出。」

「你不用闻。」韩泥说，「老耿心不聋，他懂。」

人走远后，叶青禾那边门缝始终没大开。韩泥能感觉出——铺里灯还亮着，像那年门缝留灯的温还在；那温是匿赠培元丹的旧账，跟今次送散不是一笔。一笔是叶，一笔是耿，他心里分得清。

窑里炉温退了。他蹲下，掌按坛沿，低声：「散正。耿恩，送出了。」

坛腹温一线，像应个「还」字。还得浅，还得真——浅在一粒，真在五年没忘。

雨更大些。铁无言把蓑衣往他肩上一搭：「送出去了？」

「出去了。」

「别回头看。」铁无言道，「回头像讨谢。讨谢，路就脏。」

「不回头。」韩泥站起，「回头的人，炼不出正粒。」

回丙九的路上，白织月恰好从廊角掠过，袖风冷：「半夜出窑？丑丹师炼私货？」

「炼还债。」韩泥不辩长短，「债还清，再听你笑。」

「还债？」她冷笑，「杂役也配谈债？」

「配。」韩泥说，「骨丑，债真。」

袖风一滞，没再尖。他进棚，刘婆不知从哪端来半碗姜汤：「雨夜喝点。喝完睡。」

「谢。」

「谢啥。」刘婆盯他灰手，「手黑成这样，还敢给人炼药？」

「正药不怕黑手。」韩泥擦指，「黑的是灰，不是心。」

刘婆哼一声：「心正就行。心正，老婆子帮你熬姜。」

丑时将近，沈枯芽又摸回来，眼圈红，裤脚全泥：「送到了。叶姐隔缝递进……耿爷那边接了。」

「服了？」韩泥问。

「还未。」芽儿喘，「他说天亮看。看粒正不正。」

「正就好。」韩泥点头，「不正，你把囊退我。」

「退？」芽儿一愣。

「退。」韩泥认真，「能还才送。送假的，不如不送。」

芽儿把枯叶又塞他手里：「叶姐让捎一句——别谢她。谢在脉里稳着就行。」

「记着。」韩泥收叶，「叶汤仍在前头。今日只还耿。」

邻床少年迷糊翻身：「你们半夜嘀咕啥？偷丹啊？」

「偷恩。」韩泥随口，「恩要还，才睡得着。」

少年砸吧嘴又睡了。韩泥坐坛边，把抬棺那日绳印在心里又按一遍：绳不抖，棺不歪；粒不正，恩白送。今日粒出了，路经叶家缝，缝窄，正好藏。

赖福夜巡火折远远晃过，没进棚。晃得像闻味。味在远窑里散尽了，棚里只剩粟壳香。他摸怀中木牌，牌凉贴胸：「散在途。人别滚。」

天快亮时，他仍没睡死。睡死了像忘炉。炉冷了，粒还热——热在别人掌心，热在他记恩的那本账上。

雨脚停了一线。他望瓮，独眼平：「明早听回音。回音来了，这笔才算落。」

炉前第二息，他几乎要加减一撮砂。手抬起又按住。加减是聪明，稳才是正。正字不在巧，在忍。

「今日忍住。」他对炉低语，「忍住，粒才配出窑。」

窑缝外突然有人踢柴挡：「谁半夜炼私货！」

「外门允备散。」韩泥不应门，「簿上有名。」

踢的人骂两句走了，脚步像赖福那路狠劲，却不如他熟。铁无言在暗处哼一声：「吓你的。吓你手抖。」

「不抖。」韩泥说。

散成时，他偏头再嗅。辛、淡、正——三线齐。齐了才装囊。囊口绳打两个死结，结丑，结实。

芽儿走后，他没立刻回棚，反而在窑口站了半刻，听雨打瓦。瓦响密，像提醒：送出去了还不等于收。收，要等天亮。

「等到就等到。」他抹脸上水汽，「等到假，我就毁另两粒，绝不拿不正的充数。」

回棚倒姜汤时，邻床忽然醒：「好香……你炼丹？」

「炼完了。」韩泥说，「你睡。睡你的，别跟嘴。」

「跟谁？」

「跟风。」韩泥放碗，「风一吹，恩就变戏。」

丑时过后，他短睡一小段。梦里绳在肩，棺在木，老耿哑着嗓子喊「抬稳」。他抬稳了，梦却醒了。醒时坛仍温，温得像说：粒在路上。

路上有雨，有缝，有芽儿发抖的手。手抖不要紧，接稳了就成。成了，这一夜才不算白烧炭。

囊送走后，他在灰里写了个「耿」字，又用手抹平。写是记，抹是藏。藏好了，赖福来也只能闻见窑冷。

「你写字？」铁无言问。

「写完就没了。」韩泥摊掌，「没了才安全。」

「小心过了。」

「过了才活到今。」韩泥望雨，「活到今，才送得出这一粒。」

雨夜泥滑，他回棚时踩空半步，膝一沉。沉得像提醒：人老了，步子更要稳。稳了，才配谈还。

到门时沈枯芽又折返，嗓子哑：「耿爷让人捎话——别上泥岗谢。谢在粒里，别谢在嘴上。」

「正好。」韩泥点头，「我本就不去。」

「叶姐也说了，门缝不会再开第二次。」芽儿咬牙，「开多了，闲话杀人。」

「开一次够。」韩泥按他肩，「一次把恩递出去，够了。」

肩上泥点子搓掉，芽儿才走。韩泥对着门板说了句只有自己听见的话：「老耿，粒到了就行。我还欠你远的，远的往后。」

后半夜窑冷透了。他最后看一眼灰坑里埋着的两粒余散，用土盖严：「不许被人捡。捡了，正也变邪。」

盖严后离开，步子刻意放慢。慢不是怂，是怕踩滑再摔——摔了，怀里账本会晃。账本晃不得。晃了，「还」字就容易写成「逞」。

「不逞。」他自己咬字，「只还。」

路过药峰下水沟，他舀一勺冷水洗鼻。洗完，今夜辛淡残留才散尽。散尽好——好在回去睡觉时闻见馍香，不至于把丹香梦成财香。

「财香害人。」他对流水说，「恩香才留得住。」

天将亮，他把窑门再检查一遍插销。销牢，鬼才进得去翻灰。

「怕贼？」铁无言不知何时又立在雾里。

「怕心邪的人。」韩泥说，「心邪手黑，专捡别人恩。」

「那你呢？」

「我心也黑过。」韩泥坦白，「黑在仇上。仇归仇，恩归恩。今日只许恩出窑。」

雾淡时两人分手。韩泥摸到怀帖边角，角磨圆了，圆得像年头。年头里藏着抬棺、补瓮、今次一粒——粒小，年头大。

灶房方向飘来粥香。他没有去讨早粥，只在心里说：粒先落，粥后喝。落了，才对得起抬棺那根不抖的绳。

瓮温贴掌，像点头。点头不响，钩却亮：亮向泥岗哑嗓子那半句「粒正」，也亮向他还没走完的债序。序未完，夜就不能全黑。
