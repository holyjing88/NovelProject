# 第一百二十八章 十二层圆

卯时，试台坡下鼓响第一声。

韩泥立丙九队末，怀九粒，袖刘婆针脚，胸贴玉牌。贴紧，像贴仇约——约里写滚，滚要手抖；不抖，十二层圆满才不虚。

管事弟子扬声：「外门复测，丹试开！培元散，验手验丹！」

白织月立廊角，袖风冷：「末席也上台？」韩泥不辩。上台不要紧，手稳；稳了，末席也能挂牌——牌歪，字丑，人更丑；丑不要紧，丹正。

铁无言立左侧三步，抱臂，像108那日并肩。并肩不要紧，今日独献粒；粒正，兄弟线就清半分仇谣。

钱戾衡声不高，高在韩泥耳听见：「丑杂役，上台别洒。」韩泥低声道：「记着。」

第一列验丹，有人手抖，粒洒，除名。除名不要紧，韩泥记——记手抖滚，记自己不抖。

第三列，管事喝：「韩泥！」韩泥上前，开罐，捧粒。粒丑，泥纹浅，掌稳，稳得像捧一碗不洒的热汤。

执事两指捏粒，对日，又鼻下嗅：「正。」

正字落，坡下嗡一声。嗡里嘲少，惊多——丑，能正；老，能上台。

钱戾衡眉一蹙，蹙得像不服：「运气。」韩泥不辩，只收罐，声平：「骨丑，散正。正，五年淘的，不是运气。」

刘婆立人群末，蓝布包头，手抖，抖得像怕沾晦气。怕不要紧，恩在；在，韩泥望她一眼，一眼像预告130还。

沈枯芽远笑浅，笑像芽。芽在，徒恩清了沈枯芽半块灵石；清了，叶汤还在前头。

铁无言声低：「正了。末项比试，别飘。」「不飘。」韩泥说。

午后，管事扬声末项预告：「明日卯时，培元散比试夺魁。魁者，挂牌丑丹师，递大比预选帖。」

韩泥不辩，只收罐回丙九。回不要紧，粒正，手稳，十二层圆满像弓满。

日暮，他回丙九，不近廊下笑。笑在耳，不在心——心在坛，坛腹温一线，温像应「圆」。

丑时，练鼻下静。粟壳香淡，香落齐，十二层圆这一日记进心里。

邻床少年悄问：「你真验正了？」「正了。」「正了，明日夺魁？」韩泥不答，答在坛——坛腹温一线，温像应「圆」。

他摸玉牌，牌凉，低声道：「十二层圆。十三，在明日。」

试台收鼓，鼓息里，备战帖影已至山门。山门风硬，帖角将掀，掀出「大比」二字影。

卯前一刻，丙九队末风硬，硬像试台灰线名。灰线名如刀，刀在皮上划，划不出血，划得出忍——忍不是怂，是活。活，才配上台；上台，手才不飘。

管事弟子验袋，袋灰，灰里白细，细像药。药不要紧，管事不验名——名在远，在130「丑丹师」。韩泥怀九粒，袖刘婆针脚，胸贴玉牌。贴紧，像贴仇约。

抖了，刘婆新衣还不上；不抖，丑也能「正」。

第二列，执事两指捏粒，对日，又鼻下嗅。嗅齐，声平：「正。钱戾衡立廊下，声不高，高在韩泥耳听见：「丑杂役，上台别洒。」记着，不是服，是活；活了，才配站台前，才不滚。午后散队，他编筐半刻，藤刺扎掌，偏手，刺浅些。编筐钱买米，米在，恩才还得上。还了，才像130预告——预告在针脚，不在嘴。

管事扬声末项预告：「明日卯时，培元散比试夺魁。圆，不是喜，是实；实了，箭在129射。心里不骄，只记。记正，记还，记刘婆这笔在前头。候末项时，风硬，吹得灰线名像刀。刀在皮上划，划不出血，划得出忍。忍不是怂，是活；活了，才配夺魁，才配接帖。十三，在明日。管事夜巡记簿，记「丙九韩泥，验正」，记细，是小胜。

胜在掌，不在嘴；嘴要留来守罐，留来候末项。」」

坡下石阶湿，湿里他数息，数粒，数债序。债序在，步不飘；步不飘，才接得住129帖，130魁与还刘婆。

赖福脚步在廊外停半息，停，像盯。盯不要紧，粒正，手稳，十二层圆就不白圆。

外堂阶下，管事弟子再验袋，袋扎紧，紧像席严。紧不要紧，粒在怀，怀沉，沉像债序将落。落不要紧，落要正。

药渣场角，他分拣半刻，渣灰，灰里正。正不要紧，正像试台回响——回响在掌，不在嘴。嘴要留来守罐，留来候末项。

廊下暗处，赖福脚步停半息，停，像盯。盯不要紧，粒正，手稳，就不滚。滚了，刘婆新衣还不上；不滚，丑也能挂牌。

侧门驿亭，风硬，吹得帖角掀。掀不要紧，帖在远，手在近。在近，就不飘；飘了，末项夺魁就虚。

坛边席角，他低声对坛：「圆了。下一回，帖。」坛不应字，只温。温不应喜，应「稳」——稳，是十二层圆满回响。

试台坡下，鼓余音未散，散不要紧，正字还在耳。还在耳，不在舌；舌要留来养息，留来守最后一粒。

刘婆端粥过，粥稀：「验正了？」「正了。」「正了，别飘。」韩泥只答：「不飘。」不飘，是丑翁本事；本事在忍，在还，在130前头。

铁无言送炭渣，渣细，屑亮：「末项前，手别抖。」「不抖。」韩泥应声。应声短，短才像常活；常活里藏夺魁，夺魁里藏还刘婆。

丑时末，练鼻下静半刻。粟壳香淡，香一字不落。落齐，十二层圆这一夜记进心里——心里不骄，只记还刘婆在前头。

试台木柱色浅，浅像70踹台。踹台接了约，约接到129帖；帖到，末项夺魁才像路，不像赌。

散队后，他过侧门驿亭，风硬，分拣半刻，指稳，鼻下静一息——静，是把还叶汤摁进路里。

更鼓沉，沉里，他过药山半腰，清栏，独眼平，手不抖——抖了，手稳就白挨。

午后药渣场角，风硬，他编筐，指稳，稳里带一点恩不断的回响。

邻床悄问一句，韩泥只答：「分列。」答短，短才像常活。

柱上条角白一线，三年约在怀。怀凉，掌温——温在路在，不在众目。

赖福脚步在廊外停半息，韩泥只答：「在。沿净。」在，就不滚，才守得住还叶汤。

老耿挑水路过，水洒稳，声哑：「别拜暖。拜暖，路断。」韩泥只答：「不拜。坛在，人在。」

秦霜远廊过，木鱼轻敲：「心别邪。邪了，鼻下香飘。」韩泥不答，答在心里——心里恩不断，分列，不混。

铁无言声低：「手在就行。」韩泥应声：「在。」在，就是证——证分列，不在嘴。

刘婆端粥，仍多半勺：「手稳，饭稳。饭稳，恩不断。」韩泥点头，谢轻，恩重。

沈枯芽怯，悄问：「韩叔，怕不怕？」「怕。」韩泥说，「怕也守。守完，才有饭。」

白织月远廊过，袖风冷，没停。没停，小胜——胜在手稳，不在嘴。

他低声对坛：「坛在，人在。人在，就不滚。刃不亮，先记。记完了，才配分列，才配下一程。编筐半刻，藤刺扎掌，刺浅，浅些沿才净。筐卖钱，米在，恩才还得上。入夜坡下石阶，风沉，他备料半刻，指节白一息——白完，还叶汤才落稳。辰初废渣巷，雾未散，他数息，独眼平，掌茧裂处疼——疼不露，手稳在忍里。午后外堂阶下，日斜，他验渣半刻，鼻下静一息——静，是把恩不断摁进路里。」

管事夜巡过廊下暗处，灯照席角，照见厚席不见坛——坛不露，分列才藏得住。

散队后，他过坛边席角，风硬，练诀半刻，指稳，鼻下静一息——静，是把路在摁进路里。

更鼓沉，沉里，他过侧门驿亭，分拣，独眼平，手不抖——抖了，还叶汤就白挨。

午后药山半腰，风硬，他清栏，指稳，稳里带一点手稳的回响。

邻床悄问一句，韩泥只答：「恩不断。怀凉，掌温——温在分列，不在众目。散队后，他望试台坡，坡下灰线名如刀——刀划皮，不划手；手稳，帖才近。更鼓沉，他按木牌，牌凉贴胸。验正这一日记进心里——心里不骄，只记还刘婆在前头。」
