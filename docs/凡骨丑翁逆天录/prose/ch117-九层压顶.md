# 第一百一十七章 九层压顶

除名试炼摆在坡下石坪。赖福押着木牌，牌面新字刺眼：「末席试炼，手抖除名。」

韩泥站末位，怀里玉牌凉，掌心里八层气烫。烫不是吓，是关隘在顶——顶过去，才是九层；过不去，名就从册上抹掉。

辰时，丙九列队。赖福扬牌，嗓子故意尖：「韩泥，先验！」

队里嗡一声。有人偷眼看他，像看晦气会不会当场滚下山。韩泥不接那嗡，只上前，掌按试炼石。

石凉进骨，灰线仍短——短不要紧，管事看手。手不抖，光才稳；稳里气走周，他故意把鼻下那一字摁死：静。静齐了，层才记进心里，不进舌头。

气聚掌纹，冲关隘。隘裂一线，九层开了——开像内视里阶影又亮一阶，亮在暗里，不写进册。册仍写末席，掌却见了九层。九层给他自己看，不给人看。

层破，他不露喜。喜像认晦气。他只收掌，声平：「验毕。」

赖福眼赤：「九层也末！末，仍滚！」

管事弟子冷声：「试炼过。手稳，不除名。赖管事，再闹，记过。」

旁有少年小声：「真过了？」

「嘘！」旁边人急扯袖，「想惹赖管事？过了也别提。」

赖福噎住，噎完换招：「不除名，永排末。末，永吃最后，永清最脏栏。」

「清。」韩泥只一字。

一字，手稳。小胜在活，不在热闹——胜在九层落地，胜在赖福压顶到头。到头仇苗却更深：深在玉牌，深在三年约，深在钱戾衡那一脚的反面。反面烫，正面手硬；手硬，就不滚。

白织月立廊角，袖风冷：「九层灰线，仍像桩。」

韩泥侧身让路，不辩：「桩稳，台不塌。」

「塌的是你丹道骨。」她笑。

「骨丑不要紧。」韩泥道，「手稳要紧。稳了，你袖风尖不过试台。」

她唇动了动，没再尖。铁无言抱臂走近，声低：「九层，别飘。」

「不飘。」韩泥说，「飘了，丹不正。」

「正，才配还。」铁无言道，「还谁，心里排个序。序乱，粒就杂。」

韩泥不应嘴，心下清着：芽儿半块灵石在前，刘婆新衣在后，叶汤还在省亲前——笔笔分开搁，不搅成一锅粥。

散队后赖福堵窑道，嗓哑得像压尽了：「韩泥，你赢今日，赢不了三年。三年后钱少主让你滚，你滚不滚？」

韩泥摸出玉牌，牌对着他：「三年约，记着。滚要手抖——我不抖。」

赖福退了。退时像被「三年」两字噎住。噎住也好——压顶到此为止。韩泥把七年克扣、三日公示、今日试炼一并记进心里。记一寸，刃亮一寸；亮一寸，够还芽儿，够备复测。

刘婆端粥过来，仍多半勺，勺沿像奖：「九层了？」

「九层。」韩泥应。

「层在腹，不在嘴。」刘婆哼，「嘴响，饭稀。稀了，恩也不断——你可别学会响。」

「不响。」韩泥点头，「谢粥。」

「谢什么。」刘婆又瞪他一口，「饿着破境，境虚。虚了，明日还债也抖。」

「不抖。」韩泥喝粥，「姜在碗里，暖在掌里。」

谢轻，恩重。恩在粥里，也在忍里——忍赖福压顶，换九层实，换明日还芽儿的资格。资格在怀，半块灵石碎也在怀：沉的是第五笔徒恩，终于要落。

午后他数碎石，凑够半块。石旧，边角磨圆，圆得像泥岗沈家罚银那夜——那夜芽儿送粥被罚，罚因沾他。沾得真，还就要真。真还，首笔正式报恩；报恩不张扬，张扬像施舍。施舍不是他还法，他要静静递到芽儿手里才算数。

散队后兽栏仍脏。他照样清最脏那格。铲子落，粪落，气却稳在九层掌里——稳得像试炼石没离开过手心。管事路过记一笔：「试炼过，栏仍清。」字丑，真。

赖福远处啐：「清栏的九层，也配还人灵石？」

韩泥不抬头：「配不配，债知道。债清了，才炼得正。」

「债？」赖福冷笑，「杂役也谈债？」

「谈。」韩泥声平，「粥债、汤债、石债。债不清，丹就飘。」

赖福噎半息，甩袖走了。走了，韩泥才把碎石袋系紧——系紧像把明日夜行也系住：系住了，才不半路洒。

铁无言在径外丢一句：「九层别示人。示了，偷境谣又来。」

「不示。」韩泥应，「示在还上，不示在嘴上。」

邻床少年又悄问：「你真要去还沈枯芽？」

「半块石，欠五年。」韩泥说，「欠着，丹不正。不正，我不敢上复测台。」

「怕不怕？」少年眨眼。

「怕手抖。」韩泥摸牌，「不抖就不怕。怕也得还——还了，心才空出半寸装下一笔。」

少年闭嘴。韩泥望外门方向：方向远，心近。近的是明日夜行杂院；远的是刘婆那笔新衣，得等夺魁后再还。

丑时他廊下练鼻下静。粟壳香淡，一字不落。落齐了，九层才算实。实了，外头风软半分——软得像压顶尽了。尽了，不是忘；是把「赖扒皮」三字磨进刃背。

他低声道：「赖扒皮，记着。压顶到此。刃亮一寸，不白忍七年。」

坛不应字，只温。温像推他出门——出门，还沈枯芽半块灵石；还了，十层门槛才开，复测备战才有底气。备战开，两月后上台；上台，钱戾衡眼风再来——来，他接。

玉牌在怀，半块石在怀，恩在序里。序不乱，步不飘。步不飘，才接得住明日那一还，才接得住往后那张大比帖。帖在远，还在近；近的先落，落得不张扬；不张扬，坛才温，丹才正。

夜里他再理一遍怀内碎石。旧石烫手——烫手也该还。该还就不拖到天亮后。天亮后嘴杂，夜清，石响才清楚。他按坛沿，沿温一线，应着「九」：九层是赖福最后一关；关过了，徒恩在前。

前头半块石，后头刘婆粥，旁边三年约仍烫。烫在仇线上，不在还线上。还线要静。静着还，芽儿才肯信；信了，罚银那套粥才不算白热。

外头更鼓沉。韩泥睡半息，醒半息——睡实了怕忘还，醒着又怕露喜。喜露不得。他只把掌心九层气按实，按得像把试炼石那点凉也按成路。

路通向外门杂院。杂院门松，窗灯若亮，他就叩一声「芽儿」。叩了，石递出去；递出去，心里空半寸——空不是无债，是债还了一笔，还能还下一笔。

他摸玉牌，低声道：「手稳，粒正，试台见。明日还石——还了，再谈十层稳不稳。」

见字与还字并着落，坛温一线。温像答应：答应他不飘，答应他不抖，答应他把赖福压顶的七年，换成芽儿枕下那一点凉。凉里有恩，烫在心里；心里记清了，明日夜行靴底才不慌。

刘婆夜里又添姜一片：「九层了也得吃。饿着还债，债要变馋。」

「吃。」韩泥应，「姜暖，手不抖。」

「抖什么。」刘婆哼，「抖了石递不稳。递不稳，芽儿怕，你白跑。」

「不白跑。」韩泥道，「跑这一趟，把五年欠拧干。」

拧干了，心里空半寸。空出来装刘婆，装叶汤，装复测台上那粒正。正字未落地，先把明日杂院的门敲响——敲响了，才算压顶真尽。尽了，刃才亮；亮了，钱戾衡那影再硬，也硬不过他掌心里的九层气。

他坐回席角，把试炼石那点凉又忆一遍。凉进骨时他没叫，破九层时他没笑。不叫不笑，才像丑翁活法：境藏掌里，债推前头。前头是芽儿，再前头是刘婆，最前头那碗叶汤还远着——远也得按序走。

风从窑口进，带着一点坡下石凉。凉里有钩：明日夜行，半块旧石要落桌上——落了，压顶才算真正翻过去。翻过去，刃才亮一寸；亮一寸，就不白忍赖福这些年。
