# 第六十一章 丑时坛温

丑时，棚静，鼾雷断了一息。韩泥未睡，盘坐，油灯豆，豆光照坛角。角在，席严，符怀暖一丝。暖像半闪，不燃，光太亮，管事查。坛腹温一线加剧，像心跳，像瓮里翻身半次，又眠。眠，不认主。认主，要等血缘碰沿。他不主动滴血，不揭席诱沿，不拜邪求醒。求，像赖福。等，像守瓮人。掌茧涂药，药凡，稳手。手稳，明日藤刺才正。正刺，血才找席，席才挡到沿前。邻床翻身，梦呓「温」。韩泥盖席，声低：「你看错了。睡。」他低声对坛：「你再眠半夜。缘满不满，看活计，不看心急。心急，恩断。」坛不应字，只颤极轻，轻像「嗯」，像祖父远处应。---

沿上符影浮半瞬，又敛。韩泥独眼盯沿，沿净，沿滑，沿上无血。无血，符影才像半闪，不像邪。符在怀，暖一丝，不燃。燃，烟起，管事查。不燃，暖够扛丑时寒。铁无言门外脚步远，远，像夜禁巡过。巡过不要紧，要紧是席严，沿净。他低声道：「符影半闪，不燃。燃，管事查。暖一丝，够扛丑时寒。」坛腹又温一线，温短，像应符影，不应沿。刘婆门外咳：「丑时寒，别空肚。」韩泥应：「不空。姜在记里。」

鸡未鸣，狗未吠，夜禁脚步远。三年约、钱戾衡，都在另一时辰。时辰未到，瓮先温，温而不醒。丑时将尽，温一线敛，敛，像瓮又眠回去。眠回去不要紧，要紧是席盖更严那一夜将至。将至，接席盖更严，接瓮醒，接鸿蒙九劫瓮认主。他按掌，掌贴席，席贴坛角。角温一线，温短，像廊下香字。又像凡符暖字，又像叶汤那个「热」字。热在恩里，不在舌上。舌轻，恩重。他是记心翁，恩仇都记在心里。符是物，物护骨，不护傲。傲，符自灭。邻床又梦呓，梦呓轻，像怕「温」字。韩泥盖席，声低：「睡。坛在眠，不在醒。」外头风硬，风硬了，柱上无新条。无条，有条在坛。他低声道：「丑时坛温，一线加剧。加剧，不急喊醒。喊醒，像拜邪。」坛不应字，只温，温得像答应，像等，像后半夜。掌贴席，席贴角，角温贴掌，像认亲前的试探。试探不要紧，认主要等血缘碰沿。油灯豆，豆光晃沿，沿上符影又浮半瞬，又敛。敛，像凡符不越阶，像瓮仍眠。他低声对坛：「你再眠。眠着，我席盖严。严，不主动滴你。」坛颤极轻，轻像「嗯」。外头更鼓将尽，将尽，像丑时走了一圈。走一圈，走到编筐扎掌，走到血误触席，走到席渗沿，走到瓮醒。醒，不喊，不认，等血缘碰沿那一误。他合眼，不睡实。恩仇在心里，符在怀，坛在角，席盖严。管事门外脚步停半息，又远。远，像查棚过了。过了，只见破席，不见仙缘。他低声道：「符影半闪，瓮温加剧。加剧，像将醒未醒。未醒不要紧，沿净要紧。」

沿净，手稳，符暖，席严。严，才接得住后头那一误。将至，接席盖更严，接瓮醒那一误。他摸木牌，牌边磨毛，毛得像心口七笔。笔笔磨过，磨出实。实了，才配接瓮醒。坛壁水汽爬半寸，敛。敛着，像替他守着。手别飘，恩不断。恩不断，叶汤就还在前头。外头更鼓远，远，像瓮将醒未醒。醒在缘，不在人催。秦霜远廊一句，声轻：「心别邪。邪，符灭。」韩泥「嗯」一声，不辩。辩轻，恩重。同批少年翻身，问：「你坛角咋那么暖？」「眠着。」韩泥说，「暖，不等于醒。」少年不懂，又睡。睡，棚静一寸。静一寸，丑时坛温那一线又加剧半分。他低声对坛：「闪罢，仍眠。眠着，别燃符光。」坛腹温一线，又凉。凉，像应「眠」。眠着，等活计误那一甩。老耿门外挑水，水洒稳，声哑：「丑时寒，符别燃。」韩泥「嗯」一声：「不燃。暖一丝，够用。」老耿不再言。水稳，像凡符不越阶那条界。油灯豆将尽，豆光缩一线，缩了，沿上符影却浮半瞬。半瞬，像韩氏守瓮血还未到，符先探路。探路不要紧，沿要净。他独眼盯符影，影淡，影敛，敛回坛角。角温，席严，符暖。符暖，像老耿赠符那日。那日符录起，起在怀，不起在舌。他低声道：「符录在怀，符影在沿。沿上闪半瞬，不算邪。」坛腹温一线又加剧，加剧，像丑时走第二圈。第二圈，不走沿，走席。席厚，血未来，缘仍等误那一甩。同批少年咳嗽，咳醒半息，问：「你还不睡？」「丑时守坛。」韩泥说，「守，不等醒。等误。」少年不懂「误」，又睡。睡，鼾雷起，起又断一息。断一息，像坛也翻身半次，又眠。天边白一线，白像瓮醒前夜。前夜，席盖更严。更严，他要加破布，加旧席，加压柴角。压不要紧，沿要净。秦霜远廊说过：心邪，符灭。他不邪，他只记七笔。七笔压着，手才不主动滴血。滴血若主动，像拜血祠。拜血祠，恩断。断了，叶汤凉，路也凉。他低声对坛：「丑时尽，坛温仍加剧。加剧，我仍不揭席诱你。」坛颤极轻，轻像守瓮人远处「嗯」。外头风息，风息了，符影却似在沿上再浮一息，又敛。敛，像钩，钩瓮醒，钩正测，钩还。还，先还沿净这一夜。他摸怀符，符暖一丝，贴骨。骨暖，手稳，沿净，席严。严了，才配接后头席盖更严那一夜，才配接瓮醒那一误。误在缘，不在心急。心急，血偏。偏，席挡不住。他独眼平，手不抖，低声道：「丑时坛温，符影半闪。半闪罢，仍等活计那一刺。」木牌在枕下，与席严并放。并放，像命与坛挨着。挨着，才接得住瓮醒那一误。外头更鼓尽，尽处，天边白一线。白一线，像瓮醒前夜将至。刘婆门外咳：「丑时尽，沿还净？」韩泥应：「净。符影半闪，不燃。」

丑时，油灯豆，豆光照坛角。角在，席严，符怀暖一丝，暖像半闪，不燃。眠，不认主，认主要等血缘碰沿。他不主动滴血，不揭席诱沿，不拜邪求醒——求，像赖福；等，像守瓮人。手稳，明日藤刺才正，正刺，血才找席，席才挡到沿前。邻床翻身，梦呓「温」，他盖席，声低：「你看错了。」错不错，他按掌，掌贴席，席贴坛角，角温一线，温短，像廊下香字，像凡符暖字，像「热」字。他低声对坛：「你再眠半夜。半夜后，缘满不满，看活计，不看心急。」坛不应字，只颤极轻，轻像「嗯」，像祖父远处应，像沿上符影浮半瞬。鸡未鸣，狗未吠，夜禁脚步远。远，像三年约、钱戾衡都在另一时辰。
