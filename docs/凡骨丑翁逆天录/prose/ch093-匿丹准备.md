# 第九十三章 匿丹准备

霜降后。

韩泥不再白天上山。白天清栏、分拣、编筐，活计照做，做稳了，才挪出丑时与雨夜。

匿丹准备，首在藏料。青芯、褐渣、灵泉，分三处：袖袋、筐底夹层、兽栏干草下浅坑。坑浅，浅不要紧；浅，才像藏粮。

铁无言药渣场角，声低：「你要试，别用内门药。用渣，用灰，用捡的。」

韩泥应声：「用渣。」

用渣，像晦气杂役该用的。该用的，才不惹眼。

第二备，藏炉。瓮底泥炉拆成三片，片入坛席下，席厚一层。厚，不为保暖，为挡目。管事夜巡，灯照席角，照见厚席不见炉。不见，才匿。

第三备，藏诀。墟翁丹经残页，页黄，字丑，他从废经片里抠出，抠出不示人。示人，赖扒皮来。来不要紧，页在怀，怀贴坛温。

叶青禾那边，他不敢近。近，像连累；连累，叶丫头说过「别来丑」。丑不要紧，恩要还——还在暗处，还在门缝，还在她怕黑那一线。

他托沈枯芽递一句口信，信无字，只有半块干饼：「冬夜别灭灯。」

沈枯芽怯，点头，点头像懂「怕黑」——怕黑是恩，恩要还，还不能说。

韩泥备第四件：温汤囊。囊小，陶缝，缝像叶汤碗沿。沿热，热在记里；囊空，空不要紧，成丹后装温，装门缝留灯那一路。

赖福查库，库门响：「丙九，领麸皮！」

韩泥排队，队末，独眼平。赖福眼细：「你袖鼓，鼓什么？」

「木牌。」韩泥说，「三年约，贴身。」

赖福想掀袖，管事喝：「查库快些，误时克半日！」

赖福哑半息，掷麸皮，皮轻半分。轻不要紧，韩泥不接谢——谢轻，记在心里。

夜，他铺经页于席，页上「培元」二字浅，浅而真。真，他默念，念不进舌，念进鼻。鼻下嗅，渣里辨，辨出投序。

投序写在筐底，写用柴灰，灰干，扫即无。无字，才匿；匿，才备得成。

刘婆补肘过，肘布厚：「你席盖又加一层？」

「风硬。」韩泥说，「沿净要紧。」

刘婆哼：「沿净，手稳。手稳，饭稳。」

韩泥点头。饭稳，恩不断；恩不断，匿丹才值得备。

邻床问：「你丑时还不睡？」

「编筐。」韩泥说，「筐卖钱，钱买米。」

钱买米，也买匿丹之静。静，在丑时，在雨夜，在席下坛温一线。

他低声对坛：「备齐了，不叫。叫出来了，路断。」

坛腹温，贴掌，像应「备」字。备，不是炼；炼在下一回，炼在门缝前，炼在还恩前头。

更鼓二更，雨意又来。雨里，他把温汤囊挂床柱，柱旧，挂得住。挂得住，才留得住门缝那一灯。

赖福脚步停门外，停半息，像嗅。嗅不要紧，席严，页在怀，料在三处。三处散，散得像常杂役藏粮。

韩泥独眼平，低声道：「备过了。下一回，炼。」

炼，不在众目，在暗处。暗处还恩，才配叶丫头那碗烫。

晨雾里，他分拣药渣。指稳，稳里辨青褐——辨在鼻，不在眼。眼昏，像赖扒皮盯梢。

赖扒皮查库，库门响：「丙九，领麸皮！」

韩泥排队末席，独眼平。赖福眼细，细得像秤：「你袖鼓，鼓什么？」

管事弟子冷声：「查库快些！」赖福哑半息，掷麸皮。

沈枯芽怯，递口信：「叶姐……怕黑。」

韩泥点头。点头像应「备」——备在暗处，备在门缝，备在还恩前头。

丑时前，他廊下静半刻。粟壳香淡，香落齐，掌纹温一线。

白日里，他仍编筐、清栏、分拣，一步不乱。乱，像邪；邪了，管事来。管事记簿，记「丙九韩泥，清细」，记细，是小胜。胜在活，不在嘴。嘴闭，手开，开里藏料三处，藏炉三片，藏诀一页。

秦霜远廊经过，木鱼不敲，只瞥一眼，瞥像问「鼻下香飘否」。韩泥不近，不近，像守规。守规，才匿得住。匿得住，才炼得住95那一粒——粒小，小才赠得进门缝。

样纸三包锁角柜，柜丑，锁像丙九末席。席末也要锁，锁住了才不示人。

第五备藏路，废窑火路熟半分，够备95那一粒。

愧线别来丑在记里，分列——愧是愧，恩是恩。
