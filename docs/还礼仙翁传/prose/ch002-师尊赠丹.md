# 第二章 师尊赠丹

晚晴殿里没有熏香，只有药气与旧书卷味，像一座尚未封死的墓，又像一间还肯留灯的静室。

沈晚晴背对殿门，白发以木簪挽起，簪尾垂一缕，随呼吸微动。她闭关在即，周身灵压却收得干净，不留半分吓人锋芒，只留师徒间那点温。殿外雨声如线，织在门槛上，把山门喧嚣隔成远潮。

「长春。」她唤名，不是唤长老，「劫伤脉案，我看过了。」

莫长春拄杖行礼：「让师尊费心。」

「你倒还知道费心。」沈晚晴转身，目光落他左腿，落他新杖，落他须眉上洗不净的灰，「化神九重雷劫，能活下来已是异数。修为跌落……罢了，活着比境界值钱。」

柳青鸢在侧，唇线抿紧。她想说师兄本可退劫，想说执法堂有人提前关了护阵一角——话不能在这说。她只向前半步：「师父，师兄的脉……」

「我知。」沈晚晴抬手止住，自袖中取出一匣，匣开，丹香如春暖，一粒宝丹卧在丝垫上，色泽青碧，像将一整条春河炼进了一颗珠子。

续命丹。

莫长春瞳孔微缩。此丹名他听过，丹堂谱上标为宝品，服之可延寿元，虽不能破境，却能让将尽之人多喘几十年。对寻常长老已是天恩；对他，几乎是明着的怜悯。

沈晚晴将丹推至案沿：「再活五十年，找个伴，也不枉此生。」

殿内一静。

柳青鸢看向莫长春，眼神复杂——有痛，有急，还有一点她不愿承认的慌。她怕师兄拒丹，更怕师兄收丹后仍走那条「送礼」的疯路；她尚说不清，只觉得心口像被剑穗扫了一下，不疼，却麻。

莫长春却笑起来，像听见一句旧玩笑：「师尊，弟子都七十二了，上哪找伴？找回来，人家还以为我拐骗小姑娘。」

沈晚晴也笑，笑里带叹：「嘴还硬。」

她把丹匣又推近一分：「拿着。入关之后，青岚门靠青鸢，你……靠这根杖，好好走完九月。」

「九月」二字落下，柳青鸢指尖一颤。她上前一步，欲言又止，终是低声：「师兄，丹堂在炼辅丹，外头流言……你别听。」

「听了一点，正好。」莫长春道，「不听，怎么知道礼该送谁。」

柳青鸢一愣。

沈晚晴深深看他：「你总说送礼。长春，你这一生收礼多，送礼少。入关前，为师只想问你一句——你怕死吗？」

莫长春拄杖立定，雨意从殿外扑进来，凉在颈后。

「怕。」他答得干脆，「所以不急着死。怕死的人，才肯把情分一点点送出去，换别人肯替你多撑一息。」

沈晚晴沉默良久，取出一枚玉符，置于丹旁：「入关后，神念可传讯三次。用完，便真是关中了。」

「够了。」莫长春接过玉符，触手温，像师恩还有温度，「三次，够弟子把该还的还一笔。」

她目光又落他袖中，像觉出那里有什么将醒未醒：「袖里……」

「旧伤发热。」莫长春截话，笑，「师尊别操心。弟子九月，最会发热。」

沈晚晴不再追问，只道：「青鸢，扶你师兄下山。雨大，别滑。」

---

出殿时，雨终于落下，密而凉。

柳青鸢撑伞相送，伞沿压低，遮住两人半张脸。石阶湿，莫长春杖点地，声闷，每一步都像敲在人心上。

「师兄。」柳青鸢忽然开口，「续命丹……」

「师尊的丹，」莫长春道，「得收。」

「可你只有九月。」她声音发紧，「你不服丹，外人会说你矫情。」

「外人说得对。」莫长春侧脸在雨雾里淡得像墨，「老夫是矫情。矫情的人，才舍得把好东西再送出去。」

柳青鸢停步：「送出去？」

莫长春没答。识海中塔影轮廓又颤，这一次贴着续命丹匣，像两物相识，一热一凉，在袖里轻轻碰了一下。

他抬头望关门的方向，殿内灯影已远，沈晚晴背影化作一点温黄，像将熄未熄的炉。

「青鸢。」他低声，「师兄若做一件荒唐事，你别拦。」

柳青鸢握伞柄，指节发白：「什么荒唐事？」

莫长春笑：「把救命丹，再送给救命的人。」

雨声变大，盖过弟子的远语。柳青鸢站在雨里，忽然明白师兄说的不是寻死，是另一种活法——她尚说不清，只觉得心口被什么撞了一下，像剑穗无风自动。

她再问：「送给谁？」

「送给该活得更长的人。」莫长春道，「师尊入关，命比弟子长；弟子九月，够做别的事。」

柳青鸢伞沿微抬，雨线斜打进来，她眼睫一湿，却硬声：「师父若知，会骂你傻。」

「傻徒弟，」莫长春道，「才配收傻回赠。」

下山途中，执法堂弟子迎面，见莫长春袖中丹匣轮廓，窃语：「真给了？」

柳青鸢目光一扫，弟子散。她忽然道：「师兄，若塔……真在你识海里，你别把九月玩成儿戏。」

莫长春看她，像看一块终于肯问真话的玉：「九月从来不是儿戏。儿戏的是世人以为九月只能等死。」

柳青鸢不再言，只把伞往他那边倾了一寸，像倾一点宗主不便说的护。

回到小院，他独坐案前，未点灯。

续命丹在左，玉符在右，薄册在膝上缓缓显形——册面无字，唯「缘」字将起未起，像等人先开口。雨歇后瓦檐滴水，一滴一滴，像给九月计时。

莫长春伸手，指尖落在「缘」上。

「师尊说再活五十年。」他自语，「弟子偏不先活。弟子想先问问——这塔，认不认还礼？」

塔壁金铭无风浮起，空白，却有一行淡字浮起，像墨迹从水底慢慢浮上水面：

**真心赠予，天道记之。**

字落，又起，更淡，像怕吓着他：

**赠出不计价。**

莫长春长出一口气，笑意真了些。他握玉符，符温；握丹匣，丹凉——凉与温在袖中碰面，像师恩与塔规握手。

窗外钟又响一声，慢，稳，不再像送终。

像等人还礼。

他低声：「明日，去还师尊一粒丹。换一根杖，换一条路。」

万缘塔一颤，像应。

远处丹堂一线夜灯，苏念慈在炉前守药，不知师兄已将九月的第一笔大恩，记在师尊心上。

续命丹之事，半日传遍青岚门，比脉案还快。

赵玄机在执法堂听讯，铁牌在腰，冷声：「私赠续命丹，门规第十条——」话没说完，像被「师尊所赐」四个字堵住，咽回去。

苏念慈在丹堂炉边，炉火未开，心却开，开得像听见塔鸣细响。她低声：「师兄若真把丹再赠……那丹堂算什么？」弟子不敢答，答了就像站队。

柳青鸢遣伞至静室门外，伞骨旧，柄上刻「青岚」二字，像宗主不便说的护。莫长春接伞，不撑，只道：「青鸢，伞是好物，雨来时再撑。」

顾小满在浆洗房听见「续命丹」，手里衣角一紧，紧得像又记起半块灵石——石未送，恩已起。

夜，塔壁「缘」字金铭微亮，亮得像等人先开口。莫长春抚杖，袖仍空，一念收塔，杖落掌，像把师恩先入塔中，再论还不还。

围观弟子在院外踮脚，踮得像等塔鸣——鸣没来，来的是塔壁无风自亮，亮得像天道已认第一笔恩。青岚门这一夜，都在等师兄把续命丹再送出去，送给该活得更长的人。塔影在识海里微荡，荡完又静，像钩向下一章——塔鸣初缘，首赠得杖，换一条路。明日，去还师尊一粒丹。

雨歇，塔壁「缘」字又亮一线，亮得像催他明日出门——续命丹在匣，该送的人，还在山上某处等着。
