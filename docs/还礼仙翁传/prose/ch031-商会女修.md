# 第三十一章 商会女修

缘箓二转后半月，青岚门比秋末更忙。

天试风声在，东脉战备在，丹堂炉火也在——忙得像一锅将沸的药。百宝阁管事入山这天，执法堂客厢早早收拾出来，茶未沏，先摆了账册，像摆一副算盘，等秦商言来拨珠。

弟子围在客厢外三层，不敢噪，却忍不住偷瞧——百宝阁女修入青岚，比赤焰来使还稀奇，稀奇在买卖，买卖里也许有情报。

秦商言入厢，青衣利落，笑挂在唇，像把买卖也穿成衣裳。她目光在莫长春杖与柳青鸢剑穗间游移，游得准，像数灵石也数人情。

「秦管事。」柳青鸢公事公办，「灵石周转、情报互换，按宗门例。」

「宗主言重。」秦商言拱手，「商言此行，不只谈灵石。」

她看向莫长春，目光停得久一分：「听闻莫长老将死之人，手却松——坊市送棉衣，东脉定魔少，风声都传到百宝阁了。」

廊外弟子听见「定魔少」，窃语起，像潮。

莫长春拄杖笑：「手松？老夫手紧。紧在只紧该紧的地方。」

秦商言挑眉：「何处该紧？」

「赤焰最近在哪花钱。」莫长春道，「老夫想知道这个。」

厢内一静。柳青鸢眸光微动，未拦——师兄要情报，宗主也要。霍镇山立在廊下，粗眉一扬，像听见「花钱」二字就精神。

秦商言沉吟，笑收半分，像终于认这是一场不对等的开价：「在妖与田之间。买路，买命，买你们青岚门犹豫的那七日。」

「很好。」莫长春点头，「茶还没沏，先记一笔。」

袖中簿无字，却一烫，像等人把下一礼送出去。

秦商言目光扫他袖，扫得细，像商人的眼能看灵石，也能看「记一笔」的重量：「莫长老，百宝阁做买卖，也做情报。情报有价，莫长老打算拿什么付？」

莫长春从袖中取出一盒茶饼，纸包旧，边起毛，角磨圆：「家乡茶。不值钱。请你尝。」

秦商言一愣，厢内弟子也愣——谈赤焰夜袭，付的是茶饼？

莫长春笑：「难还才好。还得起的，就不叫礼。」

秦商言打开纸包，茶香扑出，苦后回甘。她拈起一块，细嚼，细嚼出涩，也细嚼出暖，像第一次被人把「无价」放在舌上。

案上多一卷薄简，墨迹新干，像茶气里突然长出字来：

**赤焰东脉调动：三日后佯攻，七日后夜袭。**

秦商言抬眼，声低：「这笔买卖，商言还没算清。」

「茶饼算清了。」莫长春摆手，「简是回赠。你问赤焰在哪花钱，老夫答在妖与田之间——茶，是谢你肯答。」

她沉默两息，忽然问：「若商言把简卖给别人呢？」

「卖就卖。」莫长春道，「情报值钱，人情更值钱。你卖简，是商道；你记恩，是礼。各算各的账。」

秦商言起身，郑重揖手：「秦商言记下了。」

出厢时，柳青鸢在廊下路过，目光相触，各自点头，像宗门与商会之间，多了一根看不见的线，线那头拴着东脉，这头拴着茶饼。

弟子炸开一层议论：「茶饼换情报？」「莫长老又送礼！」「百宝阁管事竟真吃了！」

秦商言听在耳里，笑，笑里像亏本也乐意。

霍镇山在廊下听完全程，骂：「茶饼换夜袭？好！」周德海在远处听，笑僵，笑里像袖中有什么湿了。

秦商言下属问：「管事，这茶真比灵石贵？」秦商言答：「贵在人肯记恩。恩是另一条商路。」

莫长春出厢，弟子围问：「长老又送礼？」莫长春笑：「礼送了，简收了。接下来看宗主怎么战。」

百宝阁车马下山，山下弟子踮脚，踮得像看一场亏本却乐意的买卖。

她低声，像钩，说给下属，也说给风：

「这茶，比灵石贵。记恩账，不记入买卖账。」

大因果还在赤焰夜袭那夜，像妖骑，像杖剑，像全场寂然的斩。

秦商言听在耳里，笑，笑里像亏本也乐意。她低声，像钩，说给下属，也说给风：「这茶，比灵石贵。记恩账，不记入买卖账。」

百宝阁车马下山，山下弟子踮脚，踮得像看一场亏本却乐意的买卖。莫长春出厢，弟子围问：「长老又送礼？」莫长春笑：「礼送了，简收了。接下来看宗主怎么战。」

案上多一卷薄简，墨迹新干，像茶气里突然长出字来：**赤焰东脉调动：三日后佯攻，七日后夜袭。**秦商言抬眼，声低：「这笔买卖，商言还没算清。」「茶饼算清了。」莫长春摆手。

围观弟子在客厢外三层踮脚，踮得像等茶饼换情报——换得轻，简却重。有人低声：「茶饼也能付情报？」另一人答：「付的是恩，不是价。」

延迟的爽点不在茶香，在简上墨迹新干那一瞬——新干，像茶气里突然长出字来。
