# 第四十七章 问香

卯时，药库坡，霜白。韩泥到得早。早，像备检那日——石前不抖，坡前也不抖。

秦霜立坡顶，青衣，袖线细。她脚边三只小囊，囊口松，各泄一丝香。「韩泥。」她声不高，「帖上写了，问香，不问人。人到，鼻到。」

韩泥只答。同坡下，赖福带人围了一圈，围得像看晦气杂役出丑。秦霜冷眼：「退三步。退不干净，记名。」

众人退。退，赖福仍尖：「丑鼻辨香，辨出邪脉，正好坐实谣！」

秦霜不答他，只掀第一囊：「报。」

韩泥闭息，鼻先静。静里，苦蒿淡，带一点丹烟——丹房常香，不邪。「丹烟，苦底。」他声哑。

秦霜掀第二囊。香甜，甜得腻，腻里藏腥——腥像肉腐，又像脂粉。韩泥眉心一沉，沉得像泥岗村听谣那日。谣里说栖香邪脉，说沾不得；这香，沾不得。「甜腻，藏腥。腥在底，不在面。」他说。

赖福笑：「他瞎蒙！」

秦霜掀第三囊。香清，清里带草木，草木里有一线温——温不像丹，像人身上热气，像热汤揭盖那一瞬。韩泥指节发白，半息，才道：「温短，草长。草里有人气。人气，不是邪。」

秦霜看他一眼，眼里无嘲，有问：「你怎么知温短？」

「叶家药铺方向，甘短。」韩泥说，「短，才真。真，才不混。」

秦霜应一声。声落，坡下静了一息——静，像谣被剥开一层皮。她目光落韩泥掌上旧疤，疤丑，丑真：「编筐扎的？」

韩泥应声：「血落篓底，不蹭坛。」

秦霜停半息，像记这一句：「不蹭坛，手才紧。手紧，鼻才静。」

她收囊，声仍冷：「赖福，谣是你唱的。唱可以，别借外香吓人。」

赖福脸白：「执事，我……」

「第三囊，栖香正脉试香。第一第二，丹房常香与邪脉残香。」秦霜说，「他辨得出温短，辨得出腥藏。赖福辨得出吗？」

赖福哑了。同批少年碰韩泥袖：「你真辨出来了？」

「嗅诀三句，够活。」韩泥说，「够辨，够不漏。」

秦霜掷木牌，牌薄，给韩泥：「冬日起，丹房外廊，辨香半刻。辨得准，记功；辨不准，滚。」

韩泥接住。牌薄，真——真在「记功」二字，真在「不滚」。赖福啐：「末等也配丹房廊？」

秦霜扫他：「配不配，鼻下见。再唱谣，先查你秤。」

赖福退，退得像克字被人挖了根。同批少年低声：「你怎知第三囊不是邪？」

「温短。」韩泥说，「邪腥长，温短。短，像叶家方向甘气——甘短，才真。」

少年似懂，闭嘴。

午后，韩泥仍搬柴。柴沉，肩实。廊下新贴条，条在丙九柱旁，字利：

「冬后正测，七日后。丙九末排，先验。」

七日。韩泥看条，看七，像看七笔——笔未还，先活七日。活七日，手不能抖；抖了，廊下记功白立。白立，克字又实；实了，全丙九苦。

刘婆端粥，稀，热：「问香这日，多喝一口。一口，顶谣一句。」

韩泥喝尽，粥热在肚。她低声：「秦执事凶，凶在理。理在，你少挨一句骂。」

应声。她瞥七日条，哼一声：「七日，手别抖。抖了，老婆子粥也不够补。」

韩泥声平：「不抖。」

赖福远处骂，骂得虚——虚，像怕执事查秤。韩泥不辩。舌要留来辨明日廊香，留来数七日，留来记木牌。

酉时，丹房外廊。铁无言立桩，不摆香囊，只掷册：「记功验簿。韩泥，报卯时三囊。」

韩泥声哑复述：「丹烟苦底，甜腻藏腥，草温人气。温短，真。」

铁无言翻簿，笔尖一顿：「一字不差。册上记功，麸皮按册。」

同廊弟子凑近试嗅空囊，嗅完脸皱：「我怎嗅不出腥在底？」

铁无言扫一眼：「嗅不出，别尖。尖了，你顶漏报。」

笑断。管事验簿，敲印：「丙九韩泥，辨香准。赖福，明日你发粮，我盯。」

赖福脸青。青，比卯时更深——深在册上有印，印在，克字又虚一寸。韩泥不谢。谢轻，功在印，功在七日，七日在命。

同廊弟子愣了一息，退半步——退，像怕晦气从册上爬过来，又像怕韩泥鼻下那股静。

夜，丙九棚。邻床问：「栖香邪脉，真那么吓人？」

韩泥「嗯」一声：「腥的吓人，温的不吓人。温短，像汤。汤是热的。」

「汤谁送的？」

「叶丫头。」韩泥说，「热的。记着，以后还。」

邻床懂「热」，闭嘴。外头赖福骂远，骂「辨香辨出邪」，骂得虚。韩泥不辩。舌要留来数七日——七日正测，先验末排；验不过，滚；验过，仍末，仍活。

他低声道：「七日正测，廊香半刻，谣记着。记着，熬到能炼丹。能炼丹，先还热的。」

坛腹微温一丝，应「七日」，不应腥。腥记着，不碰；碰了，像求邪。

秦霜声从远廊来，不高，像留给风听：

「明日卯时，坡再开。开，讲栖香谣。讲清了，丙九少死一半谣。」

韩泥睁眼，独眼平。讲谣。讲清了，仇苗分得出——哪句赖福，哪句外推。外推，才钩人。七日条在柱，木牌在怀，记功在册。都在，命就在。

他低声道：「明日听。听清了，鼻才静。静了，路才宽。」

邻床悄问：「执事为何独叫你？」

「鼻下辨过。」韩泥说，「辨过，才分得清腥温。分得清，克字就虚。」

三更，廊下风歇一息。韩泥仍闭目，鼻息如线——线在，静在，静了才分得清明日坡开讲谣里哪句赖福、哪句外推。外推钩人，钩人不沾手；手沾腥，像求邪。

秦霜那三囊香还在鼻里各留一丝影：苦底像丹房常日，腥腻像栖谣皮，温短像叶家方向。影在，辨在；辨在，克字就虚一寸。虚一寸，够活七日。

他把七日条在心里又数一遍：七减一，六；六减一，五——数到正测前一日，手仍要不抖。抖了，廊香记功白立；立住了，才进得丹房门缝学还恩的丹。丹成，先还叶丫头，还，先还烫。

邻床翻身，梦中还念「栖香邪脉」。韩泥不扰，只按木牌。按实，像把辨香那三囊各留一丝影——影在，辨在；辨在，克字就虚一寸。

卯前一刻，他睁眼，鼻先静。静了，才配听坡开讲谣；讲清了，七日正测末排仍到——到，不抢，先听。

廊下验簿收进丹房，印泥尚湿。湿不要紧，印干了，克字也干半分。他摸木牌边，牌薄，真在「记功」二字——功不在嗓，在鼻。

同廊弟子退半步，像怕晦气从册上爬过来。韩泥不辩，只按木牌。按实了，明日坡开讲谣，耳才分得清哪句赖福、哪句外推。

秦霜声从远廊来，不高，像留给风听：「明日卯时，坡再开。开，讲栖香谣。」

静了，才配听坡开讲谣——讲清哪句赖福、哪句外推，正测才验得过。

木牌在怀，记功在册，七日条在柱。都在，命就在；命在，坡下辨香才站得住。

廊下风硬，硬里仍候下一声坡开。坡开讲谣，讲清了，克字就虚；虚了，全丙九少饿半口。

丑时，坛腹微温一线，应记功，不应慌——木牌在怀，七日条在柱，命在，坡下辨香才站得住。

问香帖在柱，验簿记印。印干了，克字也干半分——干半分，够活，够候坡开讲谣，够候正测末排仍到。

廊下记功册上印干了，印干，克字也干半分。半分也是活路——活路宽一寸，坡下辨香才站得住。鼻静了，耳才分得清哪句赖福、哪句外推。

明日卯时，坡再开，讲栖香谣——讲清了，克字就虚；虚了，全丙九少饿半口。手在，路在，命在，正测末排仍到，坡下辨香才站得住。问香在，帖在，路就不绝，命在，坡见。

刘婆端粥，仍多半勺。韩泥点头，谢轻，恩重。
