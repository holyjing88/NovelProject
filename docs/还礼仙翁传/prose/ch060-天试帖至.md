# 第六十章 天试帖至

战后第五日，五宗联名帖至宗主殿。

帖金边，字肃，像把天试从风声变成铁律：**三月后，五宗天试启；青岚须选派护法长老与参赛弟子。**

殿内烛火稳，稳得像要把「三月」二字，钉进每个人的寿元里。帖面还有一行小字，小字硬：武试、丹试、阵试，各宗不得缺席，缺席者降等，降等者割脉。

执事堂长老捧帖上阶，阶下弟子屏息，屏息里仍带夜战余悸——余悸未散，帖已至，至得像把青岚从守田，直接推上五宗台。

霍镇山抱拳，声洪：「镇妖司请战！」

柳青鸢升座，目光落莫长春：「师兄，护法长老位，宗门给你。」

莫长春接帖，塔影一烫，烫得像等这一刻。帖与塔意相合，淡金字浮，成句：

**缘箓三转·结缘——羁绊四十以上，两人已成。**

他辨「两人」，不点破，只笑，像笑给塔听：柳在，苏在，顾在，秦在——结缘二字，宽，宽得像路，路还长。塔壁金铭在识海里亮一线，亮得像回赠记在心里，记在心里不在袖，在缘。

寿元行跳：三月。

因果沉，约七十五，像东脉一夜的雨，终于落进心里。

殿外，弟子议论未歇。

「夜战三夜，莫长老杖剑齐出……」

「九月？谁还信九月！」

「我信。」顾小满在廊角，声轻，「长老给符那夜，我就信。」

苏念慈听见「天试」二字，握拳，又松，松得像丹师终于敢信：青岚门还有三月，还有师兄，还有炉。她未入门，只立廊下，像陪炉，也像陪战后的静。

赵玄机立执法堂侧，铁牌在腰，目光扫过莫长春背影——妒还在，却薄了，薄得像私丹还库后空出的那一格，格空，正好装「继续还」。

周德海抚须不语，像一盆浇了一半的油，等下一阵风。风若来，是盟会，是侧门，是闸纹——天试帖至，和战线也要重新量。

林挽月立殿柱侧，唇白，白里却有一线定，定得像终于选对路：恩对路，不能改；帖来了，各自还。

莫长春望天，云薄，像帖上的金边。他低声，仅柳青鸢听见：「护法长老，老夫接。打，仍是你打。」

柳青鸢点头：「你赠，我打。」

他扬声，像给全殿听：「礼还没送完呢。天试门前，够再送几程。」

殿内一静，静后是一层低低的应——不是欢呼，是信，像东脉灵田雨后，根还在土里。

秦商言的飞符几乎与联名帖同到，符上八字：「天试跟行，茶恩记厚。」柳青鸢收符，指节发白，未言，像把百宝阁也记进缘里。

夜，小院无灯。

莫长春独坐廊柱旁，不坐主位，杖横膝，竹纹游动，气音清亮：

够。

他点头：「够。」

塔壁浮半句，像卷二记在心里：

**结缘有时，天试在前。**

东脉方向云低，赤焰旗远，像厉无殇未完的恨；天试帖金边在案，像三月后的雷劫——大因果还在后头，部二却在此刻记在心里。

秦商言在山下驿站收讯，讯短，却重：「青岚夜战三夜，莫长春杖剑齐出。」

她对下属道：「茶饼那笔恩，记厚了。天试三月，百宝阁跟青岚走一路——路在礼上，不在价上。」下属应声，声里有一线敬，敬得像也懂：买卖之外，还有缘。

苏念慈回丹堂，封最后一炉培元，封得稳，像把战后余悸也封进瓷瓶。她望峰顶方向，低声：「师兄，三月……够炼几炉了。」

霍镇山在营帐外磨刀，刀声霍霍，像给天试磨刀，也像给东脉磨刀。磨完，刀锋映月，映得像帖上金边。

顾小满回杂役堂，册子添新页，页上写：「天试帖至，长老护法。」写罢，揣怀，揣得像把三月也揣成半块灵石。

魏无涯在远城酒楼听报，扇合，低笑：「缘箓三转，结缘四十。莫长春，你把路走宽了。」笑里无恨，有棋，棋未到终。

赵玄机披甲巡丹堂，甲补，刀换，巡得像把帖也巡成门。周德海在长老院独对侧门图，图冷，冷得像等盟会风来。

午后，宗主殿再议，议武试人选，议丹试人选，议护法随行。霍镇山请战，声洪；苏念慈请炉，声稳；顾小满请记，声轻——轻里硬，硬得像册子也要上天试台。

柳青鸢一一准，准到最后，目光落林挽月：「罚役未止，不得行。」林挽月唇白，点头，点得像把慕错人，再改一寸。

莫长春全程不抢，抢是宗主的戏；他只接帖，接护法，接「你赠，我打」，接得像把部二最后一笔，接在袖里，接在塔里。

殿外弟子传：「缘箓三转，结缘四十。」传的人半懂，懂的人不再问九月，问的是：三月后，青岚跟谁结缘，跟谁还礼，跟谁打。

夜更深，山门钟未响，响的是远城驿马——秦商言的车队已调头，调头像跟青岚同路。百宝阁不欠茶饼恩，恩要还，还到天试门前，还在礼上，不在价上。

苏念慈封炉后，独坐丹房，望峰顶小院无灯，无灯里她懂：师兄不坐主位，主位在柳，在帖，在三月。她在，炉在，也是结缘之一。

霍镇山磨刀至月斜，刀锋映帖金边，映像给东脉再开锋。顾小满册子写：「部二记在心里，部三开场。」写罢，揣怀，揣得像把雷劫也揣成可还的礼。

莫长春独坐，杖横膝，笑淡：「三月，够送，够还，够活到你后悔——厉无殇那句，天试门前还能再说一回。」

杖答：够。

大因果还在天试门前，像帖，像雷劫——部二记在心里，部三才刚开场。册沉，缘升，升在帖上，升在三月，升在每个人将要还的下一段路上。

晨，山门布告柱空出一格，像等下一道金纹诏。弟子不知，知的是：帖已在殿，缘已在塔，护法已在识海——袖空，塔意却沉。

柳青鸢独对帖，帖金边映面，映得像把「你赠，我打」也映成宗门下一季的规矩。规矩里，莫长春不抢武试名，不抢丹试名，只护；护不是弱，护是还礼的鞘。

厉无殇远山听帖，听缘箓三转，听结缘四十，骂声哑，哑里恨更深：「三月……够本少谷主再备一刀。」

韩铁山听帖，笑冷，笑里算：青岚夜战虽胜，胜后还要天试，试就是耗，耗就是机会。

周德海听帖，抚须，须动，动像也懂：闸纹之争，盟会要辩；天试之局，盟会也要量。量完，和战线才真分。

莫长春收帖入袖，一念出塔，帖入塔格，塔意温，温像缘箓三转记在心里。一念收塔，杖落掌，他念，仅风听：「部二尽，部三起。起在帖，起在三月，起在还礼未完的下一程。」

顾小满夜抄帖文，抄给杂役堂能识字的人看，看的人少，少却亮，亮像雨夜那道光还在。她抄完，加一句：「护法长老，莫长春。」写罢，揣怀，揣像把部二终钩，也揣成部三开门的钥。

柳青鸢阅抄本，未删，删像不信。信，是东脉雨后，根还在土里；根在，帖在，缘在，路还在礼上。

杖中竹纹再游，气音二字：够。莫长春点头，像对杖，也像对部二最后一夜：够送，够还，够把缘箓三转，落在青岚门这张将满的册上。

秦商言车队离驿，旗换，换青岚同行色。她对心腹道：「天试不是买卖，是缘。缘在，百宝阁就在。」

赵玄机夜守丹堂，帖影在云，云像压在门上。他立，像把部二最后一功，也立成部三第一门。

莫长春小院无灯，杖横膝，塔壁浮：**结缘有时，天试在前。** 部二在此记在心里，部三在帖上开门。

礼还没送完呢，天试门前，够再送几程。

缘箓三转，帖在柱，柱响像卷二盖印——盖印之前，旨还在路上。
