# 第七十四章 暗修炼气二层

丑时后第三夜。兽栏夜值毕，韩泥未睡实。睡实了，秘密才藏得住；藏得住，才配暗修。他盘坐床沿，席盖严，坛不露。露了，赖福来，管事来，煞骨门旧债也来。油灯豆，豆光晃一下，又稳。稳，像掌按测灵石——石凉，手不抖；坛温，心也不抖。他揭席半寸，掌贴沿。沿温一线，贴到掌纹那路里，路微颤，颤像一层在呼吸。

呼吸在暗，不在公示；公示写仍末，掌写一层。邻床鼾声如雷。雷不要紧，他鼻下静一息。粟壳香淡，淡不要紧；要紧是香一字不落。气从鼻入，沉腹，沉到掌纹。掌纹亮半分，亮不像光，像认——认韩氏守瓮血，认瓮穴灵穴。一层境里，气转一周，周转涩，涩不要紧，涩才真。真在凡骨壳，真在腌菜坛沿，真在五十九年擦沿的手。

他低声道：「二层在远，手在近。」

不在舌上，在掌里。舌响，香飘；香飘，手就滚。滚了，玉牌白接。气再转一周，周转快半分。快不要紧，快也要稳——稳了，才不破席，不叫邻床醒。坛底极轻一颤，像臾墟残念应一息。应完，又寂。寂里，气冲关隘，关隘薄，薄像席布。冲半寸，退半寸。退不要紧，再冲。第三周，气聚掌纹，掌烫，烫得踏实。

踏实了，关隘裂一线，裂声闷，闷在骨里，不在屋里。二层开了。开不像雷，像瓮唇吐白气——白气冷，冷里带药香。药香淡，淡不要紧；淡也是境。韩泥屏息，独眼不眨。眨了，境就散。散不得，散了，二层白开。气稳在二层，稳像粪铲落栏——落得要实，实了，才不飘。飘了，管事记「抖」。他收功，收得像盖席。

盖严，沿净，坛不露。不露，像腌菜坛，不像邪。掌纹那路比昨夜厚半分，厚不要紧，厚在暗，不在册。册仍写仍末，掌写二层——写给自己，不写给别人。邻床翻身，眯眼半息：「你……身上咋热？」

「编筐扎的。」韩泥声平，「看错了。睡。」

小杂役哼一声，又睡实。睡实了，二层才藏得住。更鼓将尽，天边白一线。白里，他廊下练半刻鼻下静，静到只闻辛淡，不闻辱。二层在掌，一层在忆里，两层分列，不混。混了，香飘；香飘，煞骨门就嗅得到。铁无言脚步远，停半息，声更低：「境升别叫。叫了，像求。」

韩泥只答：「油灯豆。豆爆，爆完即灭。」

赖福盯半息，终是没掀席。掀不得，沿净，坛不露。不露，晦气遮瓮，遮住了，二层才稳。卯时前，兽栏仍腥。腥不要紧，二层在掌，铲下更稳。稳，像踹台那一掌——掌在，境在，路在。管事弟子巡栏，记一笔：「丙九韩泥，清细。」

二层破关那夜，关隘裂声闷，闷在骨里，不在屋里。屋里邻床鼾声如雷，雷不要紧，韩泥收功像盖席——盖严，沿净，境敛。敛了，二层在掌，一层在忆，分列不混。卯时兽栏，铲下比昨日稳半分。稳半分，管事眉松一线，记「春初仍清细」。记细，是小胜；胜在活，不在嘴。赖福讽：「二层也末等。」韩泥不接，舌留来鼻。

铁无言立栏边，声低：「二层配渣，渣才说话。说话了，药山才走得深。」走得深，在76；远不要紧，脚在近。近，才暗修，才嗅诀。刘婆端粥，仍多半勺：「昨夜又守丑时？」

「守了。」韩泥说，「沿净。」

「沿净怕什么。」刘婆哼，「饭稳，恩不断。」

韩泥点头。恩在粥里，也在二层境里——境轻，记重；记重了，才接得住药山。秦霜远廊经过，木鱼不敲：「心别邪。邪了，鼻下香飘。」

韩泥不答，答在心里——心里七笔分列。愧线71，臾墟恩72，二层境74——各走各的，走长了，才还得了叶汤。他摸怀玉牌，凉进骨。凉不要紧，掌烫——烫在二层，不在众目。众目看灰线三尺，掌里多了一尺暗线。白织月远廊经过，袖风冷：「杂役也守丑时？守出邪法？」

韩泥不抬头：「守沿。沿净，不叫。」

白织月唇线绷一息，走了。少一句嘲，小胜。午后，药渣场。灰堆高，风里一点香。韩泥分拣，指稳，二层气走指梢，指梢温半分，温像辨渣更细。铁无言走近半步：「二层别漏。漏了，渣场不收。」

韩泥应声：「不漏。鼻下静。」

铁无言又指两堆灰：「这堆偏腐，这堆偏辛。辛沉，别当浮。」

韩泥分拣，分出，分出才点头。点头轻，记重——记二层境配嗅渣，配活，配暗修。他低声对坛：「二层开了。不叫。坛在，人在。」

坛不应字，只温。温，像爹的手又搭上来。爹说过：「坛别扔。」扔不得，扔了就无根；无根，二层白开。更鼓沉，沉里，他合眼不睡实，掌纹那路仍转半周。半周不要紧，转多了，邻床就醒；醒不得，醒不得。他独眼平，手不抖：「二层在掌。三层在远。药山在明日，渣里活。」

韩泥只答：「不拜。记着。」

夜禁脚步远，风里像煞骨门三字一闪——闪不要紧，沿净，壳才像腌菜坛。像腌菜坛，魔教才嗅不准。他按心口，恩辱分列。二层是境，境是路；路通向药山深行，通向废渣路，通向丹路开那一日。散队后，他过坡下测灵石。石凉，线灰，灰在众目下公示过——公示不要紧，掌里有二层，手要紧。同批少年悄问：「你身上有时热，邪不邪？」

「沿净。」韩泥说，「沿净，不叫。不叫，就不邪。」

少年不懂「沿净」，却懂「不叫」，闭嘴了。他廊下再练半刻鼻下静，粟壳香里多一丝药苦，苦像药山渣——渣不要紧，鼻下稳才要紧。稳了，明日药山手才不抖。钱戾衡名在玉牌里，在三年约里——约还在，二层先藏住。藏住了，才接得住药山捡渣，才接得住白织月再嘲。丑时前第二夜，他未敢再冲关。

冲不要紧，冲急了，邻床就醒；醒不得，醒不得。他只让气在二层转半周，半周稳，稳像粪铲落栏——落得要实，实了，才不飘。坛底极轻一颤，像臾墟残念应一息：「……二层在暗，暗在沿。沿净，煞骨门嗅不准。」

韩泥不答，答在心里——心里记二层境微进，进在丑时，进在席下，进在沿净处。管事弟子夜巡验棚，灯照席角，照见净沿不见坛。不见不要紧，沿要净。沿净，像腌菜坛，不像邪。沈枯芽远廊经过，步子轻，像也没看见。没看见不要紧，韩泥不近——近，像求；不求，只记半块灵石恩在还路前头。

散队后，他过丹阁侧廊，廊扫痕直，直得像秤。他不踩灰，肩稳，筐空——筐空，像只清栏，不像嗅药。同批少年掩鼻：「你身上咋有时热？」

「编筐扎的。睡。」

少年退半步，没再问。问，像沾晦气。应完，他不叫。不叫，才守得住暗修，守得住药山捡渣那一日。他低声对坛：「二层在掌。三层在远。二层开后第二日，坡下石阶霜化。韩泥过阶，阶滑，足底气沉，沉得稳。稳像踹台按实，稳像一层在忆、二层在掌。管事弟子列队，喝：「冬深杂役，手抖记名！」

韩泥立队末，独眼平，手不抖。不抖，是小胜；胜在肩，不在舌。赖福立队首，眼细：「丑杂役身上热，二层？邪！」

韩泥不接茬。接茬，全丙九饿。他低声道：「油灯豆。沿净。」

管事冷声：「赖福，再嗓，克麸皮！」

赖福哑了。哑，韩泥掌按木牌，牌凉，掌烫——烫在二层，不在众目。兽栏清栏，铲下稳，稳里二层气走臂，臂不僵。不僵，粪铲才不偏；偏了，管事记「抖」。小兽角里咳一声，韩泥换草，鼻下静，静里闻见药回一线。药回，像嗅诀将芽，像渣场辛沉底。铁无言立栏边，声低：「二层别漏。漏了，像破境示人。示人，谣来。境在席下。」

刘婆端粥，仍多半勺：「二层别飘。」

「不飘。」韩泥说，「飘了，恩断边缘。」

刘婆不懂「边缘」，懂「不飘」，哼一声走了。午后药山边门，仍只捡渣。捡渣手稳，二层气走指梢，辨渣更细。细了，辛腐才分得清。管事验筐，筐净，记「捡净」。记净，是小胜。黄昏编筐，藤刺扎掌，他偏手。偏手像守沿——沿净，血不来；不来，坛才像腌菜坛。三层在远。药山深行，仍不越界。

温，像把二层摁进掌纹那路里——路在，就不滚。更鼓沉，他合眼不睡实，气转半周。半周稳，稳不像飘。飘了，煞骨门就嗅得到。独眼平：「二层稳了。嗅芽将起。冬深续活，四层将显。」

管事贴条，条丑字真：「杂役夜禁。」禁在，坛在，瓮温不外泄，明日仍沿净。
