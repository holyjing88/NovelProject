# 第一百四十一章 势足

积势足，足在册，册在管事手。

管事扬声：「三日后验息，炼气十六，层实才记。」

韩泥只答：「在。」在，就不滚。

药峰分拣照旧，照旧才像141该过的日——日苦，日真；真了，142才炼得实。

坊市末摊续听风，风里有「丑丹师」三字，三字在舌，不在掌；掌要留来备散材，材细，才配142。

沈枯芽口信到，信仍无字，只有半片枯叶：「叶姐说，142那路，她接。」

韩泥不接话，接在心里。心里排序：142、耿恩、叶汤——三笔分列，不混。

铁无言声低：「备材别躁。躁，手飘。」

「不飘。」韩泥说，「飘了，散不正。」

刘婆端粥，仍多半勺。韩泥点头，谢轻，恩重。

丑时，练鼻下静。粟壳香淡，香一字不落。

他望坛，低声道：「势足了。142，在前。」

坛腹温，贴掌，像应「备」——备齐，才叫炼；炼了，才送得出。

外门晨鼓未迟，韩泥已醒。醒在丑时前一刻，一刻够数息，数帖，数债序里下一笔——142老耿，经叶家。

管事弟子传话，话平：「夺魁者，三日后验息登记。炼气十六，层实才记。」

韩泥只答：「记。」记，就不飘；飘了，142谢不出。

药峰活照旧，照旧才像141该过的日——日里分拣，日里编筐，日里备散材。材细，细像110首炼那筛；筛齐，手才静。

坊市末摊续留，续在听，不在荣。荣在远，在大比，在235；141只听风改一半。

铁无言立阶下，声低：「142在前。备材别躁。」「不躁。」韩泥说，「躁，手飘。」

刘婆端粥，仍多半勺：「登记前，别空肚。」韩泥点头。恩在勺，不在舌。

赖福查税，税半文，文旧。旧不要紧，旧像常杂役该交的；交了，才像141积势，不像显摆。

沈枯芽口信到，信仍无字，只有半片枯叶——叶黄，像叶汤还在前，像142经叶家那条路还温。

他低声对坛：「十六登记在三日后。142在前。记着。」坛不应字，只温。

更鼓沉，他摸怀帖，帖厚，厚像债序又落一笔——笔是散，笔经叶家，笔要还耿恩。

散队后，他过验息堂外，堂门闭，闭不要紧——三日后开，开才记十六。

坛边席角，他低声对坛：「备材在。142在。人不滚。」

邻床翻身，眯眼半息：「真要谢老耿？」

「要。」韩泥说，「能还才送。」

管事记簿再翻，翻见「培元散备材」四字——四字丑，真在登记前。

药渣场角，他摊材半刻，材细，细像110首炼那筛。筛齐，手才静；静了，142才像丹，不像试。

他摸怀帖，帖厚，厚像债序第三笔——笔是耿，笔经叶，笔在三日后送出。

更鼓沉，练鼻下静。落齐，141这一日记进心里。

141这一日，外门像140收束后的第一口气——气要匀，匀了，142才谢得出。谢在142，不在141；141只积势，只备材，只候三日后炼气十六登记。登记不是贺，是钉——钉住层实，钉住大比路，钉住丑骨末席还能不能踩稳。踩稳，靠手，不靠嘴；嘴要留来听管事传话，留来听坊市风改一半，留来听沈枯芽那半片枯叶。

叶黄，像叶汤还在前，像142经叶家那条缝还窄还真。真，才配送培元散；送，不是攀耿恩，是还七笔⑥——⑥起在28抬棺绳不抖，起在15屋漏补瓮，起在110首炼自用与142馈缘的分界。分界在心，不在舌；舌若显摆，赖扒皮来，白织月来，钱戾衡眼风来。

来不要紧，手稳，材细，筛齐，142才像丹A该写的章，不像110试炉。试炉在己，馈缘在人；人在老耿，老耿耳聋，心在；心在，粒才收得住。收得住，143才像余波，144才像泥纹名起，145才像疤鬃驮药。驮药在远，141只备。

备完，更鼓沉，按木牌，牌凉贴胸，坛腹温一线，像应「备」——备了，142在前，就不滚。

更鼓沉，他望验息堂影，影在门后，像142在门后。门后不要紧，材在，手在，就不滚。

外门风硬，验息堂门闭，闭不要紧；三日后开，开才记十六。开前，材要齐；齐，142才谢得出。他望坛，独眼平：「备材在。十六在三日后。142在前。」更鼓沉，按木牌，牌凉贴胸，像贴141这一日——日在备，不在喜。

铁无言声低：「别飘。」韩泥应声：「不飘。」

药峰分拣照旧，指稳，鼻下静一息。

帖在怀，怀沉，沉的是序，不是惧。

白织月袖风过，韩泥不辩。辩，费舌；舌要留来守坛。

赖福廊下过，韩泥只答：「在。」在，就不滚。

柱上条角白一线，三年约在怀。怀凉，掌温。

坡下石阶湿，他数息，数帖，数债序。

同棚杂役笑，笑在耳，不在心。

废渣巷日斜，他验渣，渣正，手才静。

侧门驿亭风硬，他听风，风过，记在心里。

散队后，他过外堂阶下，分拣半刻，指稳——指稳，路就不飘。

编筐半刻，藤刺扎掌，刺浅，浅些沿才净。沿净，坛才藏得住。

管事记簿，记细，是小胜。胜在忍，不在辩。

他望坛，低声道：「手在，路在。」

更鼓沉，天边白一线。白里，他按木牌，牌凉贴胸。

坊市尾巷风硬，他收摊如石，石在，就不滚。
