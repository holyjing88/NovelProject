# 第 21 章 拦拳

泼水后，堂上更乱。

乱不是声，是气——气里水腥，笑腥，赵绣春掌腥，腥混成一锅，锅沸在堂心，沸在陈寻湿衣上。湿衣贴骨，骨不抖，抖的是人群里那些等着看「跪」的眼。眼多，多成墙，墙缝里漏进来的天光灰，灰得像没洗干净的盆沿，盆沿还沾着方才泼出去的那一层冬水。

赵绣春要陈寻跪，声尖，尖里带胜：「洗了晦气，还不跪？跪，才像悔！」

她掌拍得响，响在棚顶绳上，绳颤，颤出一阵细尘，细尘落，落得像雪，雪不落人，落人心——人心里早有人替她叫好，好的是「四灵根也配站堂」这句老话终于落了地，落了水，落了跪。

陈寻不跪，膝直，直得像桩：「无罪，不跪。有水，无罪。」

「你辱村风！」

「辱在舌，不在行。」

八字落，落得像石，石击水，水花溅到赵绣春脸上，她脸一扭，扭得像被反泼。反泼不在水，在理；理一硬，跪就成了赵家要的台阶，台阶他不给。不给，赵绣春就更急，急里她看赵村长，赵村长眼转，转得像翻页——页上写着「除名」，写着「三分二」，写着「今日还差几票」。

萧惊鸿挤前，肩撞人群，撞出一条缝。缝里赵小满退，退半步，半步像怕拳，怕的不是拳，是拳后的案。赵小满袖仍湿，湿得像烫，烫在赵家，赵家当烫在陈寻身上；陈寻身上水顺颌滴，滴在土上，土黑，黑像墨，墨在记。萧仍追半步，半步追的是水，水泼在陈寻脸上，泼在他眼里——眼里不是泪，是火。

拳起，带风，风硬，硬得像三月前集口粮铺柱上那悬着的拳。拳未落，人群已静一瞬，一瞬里连赵绣春的掌都停了，停得像等一声响——响若出，堂便成案，案若成，测灵名便没了。

陈寻抓住腕，抓得准，准得像练过千次。低喝，声低，低里烫：「还没到时候！」

萧惊鸿眼红，红里水与怒混，混成一片：「都泼成这样了！你还要忍？」

「泼水是辱，拳是案。」陈寻说，一字一字，字像钉，「案上加拳，测灵名没了。名没了，站什么石？」

萧惊鸿喘，喘得像弓满又被人按住弦。弦在骨，骨响，响而不出。他腕上脉跳，跳得像鼓，鼓在忍，忍比打更难——三月前集口，他悬拳在赵小满肩窝上，陈寻也是这一句；一句重复，重复不是软，是把怒按进骨里，按到能站石的那一日。

拳落，砸在自家腿上，疼，疼得闷；闷里不吭，像把怒砸进骨头，砸成三月前那夜祭父母时说的：听你的。腿骨一震，震得他眼前白了一瞬，白里仍看见陈寻湿衣贴颈，颈侧水痕像一道未干的墨线。

人群静了一息，一息里赵村长眼转，转得急——急的不是理，是势；势若成案，除名便有名目，名目要拳，不要水。水已泼，泼成了礼；礼若再成拳，礼就碎了，碎的是赵家今日搭的台。

他改口，改得像翻页：「以证为据。周嫂，你男人，可是陈寻推下山？」

周嫂哑，哑得像被人掐住喉。她看赵绣春，赵绣春眼刀，刀快，快得她舌乱：「我……我听说……」

「听说，不是见。」沈振东声冷，冷得像霜落药柜，「见者，是我。醉者，是你男人。伤者，已证。推人者，无。」

沈振东从人群后出，灰袍，步稳，稳得像把一条线画在堂上——线从后山路口起，起至队尾，尾在陈寻，陈寻未近醉人，未夺刀，未推人。他指路线，指得像画在土上，土湿，指痕浅，浅里硬，硬得像证。

人群分波，像水遇石，石是证，水是舌；舌退一寸，波就散一寸。散处，有人倒吸气，有人看赵绣春，赵绣春嘴角仍翘，翘得像刀回鞘一半，鞘响，响而卡。

老耿独眼出，眼亮，亮里旧：「瓮我修，与他爹旧恩有关，与沈家无涉。陈寻封门练体，我瞧见，未夜出。」

瓮在堂边摆着，像一件静物，静物里沉着旧恩，旧恩与谣无关，无关便是一环。老耿指瓮，指得像指碑，碑不言语，言语在独眼里——眼亮，亮得像还能守夜，亮得像不信谣能淹瓮。

张麻婆骂，骂得直，直里白：「周家酒虫，关陈寻屁事！我盐白送，送在他门口，未送在沈家帘里！」

她盐字硬，硬得像她人，人从人群侧出，出得像一块盐，盐白，白得不掺沈家，不掺夜入，不掺帘内。方小沅抖，抖得像叶，叶仍立：「粥是我送，与他无夜入。门边，未进。」

方母拽女，拽得更紧，紧里像护，护的不是陈寻，是女儿的名。名若污，粥也污；粥不污，门边二字就够。够，便是一环，环小，环硬。

赵绣春要拦，拦成尖叫：「小丫头懂什么！装神！心字——」

钱管事记，笔快，快得像怕漏链：「证词在此。除名票不够。」

笔落纸响，响里像关门，门关一半，一半在票，票不够，不够便是理在。陈寻湿衣站直，像钉，钉在堂心。钉住了，赵绣春笑不出来；笑僵，僵里她仍拍掌，掌响空，空得像敲在铁上。

他侧目萧惊鸿，一眼，眼里有谢，谢不在言，在「不出拳」。不出拳，名在；名在，链才能继续。链继续，才是公审；公审未完，反噬未至。萧惊鸿腿疼，疼而闷，闷里他看陈寻，陈寻不看回，只看堂上——堂上链在齐，齐才公；公，才名在。

赵村长拍案，案响，响里催：「再听！还有谁见？」

堂前风紧，紧里陈寻任水冷，任笑热，只立。立里，他觉出仇那痕仍温，温而毒，毒在「当众」——当众的辱已记心，辱不收拳，拳只添案。他低声，只给自己：「审。继续。」

赵绣春脸扭，扭得像被反泼，反泼不在水，在萧那一拳未落。未落，势便泄；泄一寸，链紧一寸。她急，急里换声：「周嫂！你再说！」

周嫂抖，抖得像寒，寒里跪不得，跪了是给赵家台阶。台阶陈寻不给，萧也不给——不给，靠证。

猎户自人群出，声粗，粗里直：「我见刀落，落在他自家臂。陈寻在队尾，未近醉人。」

猎户臂粗，粗里直，直得像柴，柴不弯，弯的是谣。又一环，环扣伤者。伤者臂抬，抬得像证：「拉我，不是推。我若滚，是他拉。」

臂上布新，新里渗旧血，血干，干得像一条线，线证「自伤」，不证「被推」。赵村长眼沉，沉里仍笑，笑僵：「还有呢？」

还有，是老耿瓮，是张麻婆盐，是方小沅粥——粥小，盐硬，瓮静，静里都是「未夜入」。沈振东再指：「山下路口，我指，他未推人。队先下，他断后。」断后不是推，是挡，挡谣，挡醉刀，挡村眼。

陈寻任水顺颌滴，滴在土上，土黑。黑像墨，墨在记；记谁泼，记谁拦拳，记谁把拳砸在自己腿上。他按心口，人在怀，恩六笔温，温像六盏小灯；仇痕温而毒，毒已收过峰，峰在泼水，峰后靠链。

钱管事念条，念得像钉：「沈氏，汤衣，未夜入。张麻婆，盐，门边。方小沅，粥，门边。老耿，瓮，旧恩。」每念一条，赵绣春脸白一分。白，是谣在退；退不退，看最后一扣——扣在醉者嘴上，不在他拳上。

萧惊鸿立人群，拳硬，硬而不出。不出，是陈寻那一眼；一眼，够堂上，也够三月前。他看链起，眼里火退半寸，退成闷，闷在骨，骨里响，响而不爆——爆，是给赵家除名；不爆，是给陈寻留名。

赵村长再拍案，声里带虚：「今日，除名票不够……」

不够，是理；理在，审就不能私结。私结是赵家想要的，想要不成，成的是下一章：证齐之后，醉者该上了。他眼亮，亮得像抓住最后一缝，缝在「听说」，听说若成，链或能翻——翻不了，便等反噬。

风紧，棚顶绳响，响像催。陈寻湿衣贴骨，骨立，立成堂心那根桩——桩不倒，赵家的水就白泼。泼了，辱在阴；阴在，阳不乱；阳六笔仍温，温得像说：恩在，名在，审在。

他抬眼，看棚外天，天低，低得像压。压不垮桩，压垮的是心虚者——心虚者，下一息该拖上堂了。堂前有人低声，低声像沸前闷：「周家男人呢？」

赵村长耳尖，尖里喜，喜像抓住扣：「拖！」

拖字未落，陈寻不拦——拦在此时，像怕；不怕，等反。反是链的最后一扣，扣在醉嘴上，不在他拳上。

陈寻湿衣滴水，低声，只给自己：「拦住了。审。继续。」
