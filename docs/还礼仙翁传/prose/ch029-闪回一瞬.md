# 第二十九章 闪回一瞬

夜坐院中，秋虫声细，像塔鸣。

莫长春闭目，识海中塔影无风自动，「缘」字发烫，烫得像雷劫那夜碎掉的那道影，又要拼回一线。

他看见荒野，百年外，青年莫长春，衣破，杖缺，捧一瓶劣药，递向一名陌生修士。

修士衣锦，马高，眼冷，像看一块沾泥的讨好：「你要什么回报？」

青年答不出，唇动，动不出字。他只会送，不会说，说了，就像交易，交易了，就不叫赠。

药瓶被拒，跌碎，药汁渗土，像恩没送出去，反成耻。修士走远，留一句：「赠非交易。你不懂，就别送。」

青年跪土边，掌按药渍，忽然懂了一半：不是不要回报，是不能先算回报。先算，手就脏了，礼就轻了。

他又看见第二幕，同一片荒野，另一人濒死，青年仍送药，仍不问回报，那人咽气前，只吐二字：「迟了。」

青年立尸边，懂另一半：赠有时，还亦有时；还不了，不是耻，是命；怕的是先算，把命算轻。

影碎。

闪回里还有第三幕：青年莫长春捧药被拒后，遇一老翁，老翁不拒，只问：「你要回报吗？」青年摇头。老翁叹：「那这药，吃得不安。」药仍被收下，欠下一笔说不清的人情。

青年因此更懂：赠有时，收的人不安，也是因果；怕的不是还不还，是先算。

莫长春睁眼，当下，青竹杖横膝，杖轻颤，像应那一瞬。

院外桂香，山远，人近，像百年缩成九月。柳青鸢传讯至，简上六字：「师兄，别死在天试前。」

莫长春回简四字：「尽量不死。」

简下，他笑，笑像霜，凉而不苦。

弟子门外经过，窃语：「莫长春又在院中独坐，像等什么。」

「等死？」

「等礼。」

「九月老头，有什么礼好等？」

莫长春听见，不辩，只抚杖。杖中竹纹游动，气音轻，像说：还有。

他低声，像对百年外的青年，也像对当下九月：

「第一次赠礼失败，原来记在这里。记下了，才懂为何这一回，只能送一半在席上，一半在袖里。」

缘字亮一息，又敛，像伏笔埋妥，不在今夜揭完。

风大，桂香扑脸，像有人从百年外走来，又停在门口，不进门。

柳青鸢又传讯：「天试前，别送一半。」莫长春回：「不送一半。」

魏无涯远观缘字亮一息，扇合，低笑，像看伏笔埋妥，不在今夜揭完。

苏念慈在丹堂远处，炉火细，细得像也听见缘字亮一息。她低声对炉：「师兄又在等礼。等礼的人，最忌先算。」

赵玄机在执法堂抄门规，听见院中桂香与秋虫，指节叩铁牌，一声，像把「问心」二字，又记进铁里。

霍镇山在远处骂阵，骂声洪，像把战意敲进每个人心里，却敲不进百年外的荒野。

顾小满在杂役堂值夜，听见院中桂香与秋虫，无端心稳，像半块灵石在怀，温了一路。她不知百年，只知长老又在等礼。

闪回一瞬那夜，莫长春独坐，闭目，识海里旧影比现实还亮。

莫长春袖空，一念出塔，杖横膝，不追旧，只记旧。

柳青鸢隔窗望，不进门，像怕进门就惊破闪回。她低声：「师兄，旧伤别撕。」

顾小满远处守，守得像守半块石。她低声：「闪回记下了。不往外传。」

赵玄机验夜灯，灯稳，稳里仍想挑刺——刺挑不出，挑到的是静。

苏念慈闭炉，闭得像怕闪回进丹堂乱了火。她低声：「旧别传成咒。」

院灯长明，明里像等一句给全门的话——话没来，来的是桂香，香里藏着百年旧恩仇的一页。

闪回散后，莫长春睁眼，塔中旧影收，收像记完一笔。百年外的青年跪土边，当下的九月抚杖——两影叠在一息，叠完又分。

闪回里第四幕忽至：青年莫长春在雨夜赠伞，伞旧，收伞人连名都未留，只把一枚铜钱塞回他掌——铜钱凉，凉得像说「赠可以，别指望记住」。青年懂了：礼有时轻，轻到只剩一声「谢」，也仍算赠。

当下，院灯被风吹斜，斜影落在簿上，塔壁无新铭，缘字却像被旧影喂饱。莫长春以指节轻叩膝上杖，杖应，应得像百年外那青年也在叩。

林挽月罚役归途，望见院灯长明，明里像有人把旧伤摊在灯下晒。她未近，只把一枚新制符纸塞门缝——符纸薄，意厚，厚得像「别死在天试前」之外，还有一句未写的「别先算」。

赵玄机验完夜灯，灯芯无爆，爆不了，像挑刺的手无处下。他回思主道「帕从心而来」，忽然觉得妒火里掺进一丝说不清的空——空处像还库挪出的那一寸，挪向哪边，尚说不清。

莫长春再闭目不半柱香，识海里青年又试一次赠药——这一回药被收下，收的人仍不安，不安像另一种还礼。青年因此更懂：赠有时，收的人不安，也是因果；怕的不是还不还，是先算。

睁眼时，桂香已淡，淡得像旧影退去。顾小满在院外守至更鼓，守得像守半块石，石温，温成路。她不知百年，只知闪回不往外传——传了，就轻了。

柳青鸢隔窗未进，未进像怕惊破师兄最后一瞬。她低声：「旧伤别撕。」撕开了，天试前来不及缝。

院灯长明至更鼓，更鼓过，桂香散尽，散得像旧影也肯歇。莫长春睁眼，当下与百年在眸里合又分，分完，只余一句：还不了，就别先算。

风大，桂香扑脸，像有人从百年外走来，又停在门口，不进门——不进，才是还礼。

识海里，青年赵氏曾问：「师兄为何赠药？」莫长春在闪回里答：「怕你死在天试前。」答完，他自己也愣——愣像发现百年前的怕，与百年后的九月，竟是同一句。

院灯将灭时，顾小满听见师兄低语「还不了，就别先算」，低语像落进册里，册里多一行别人看不见的字。

闪回尽头，青年赵氏递药的手在识海里停住，停住像百年后的妒，也曾有过敬。莫长春睁眼，当下与百年又分，分完，只觉桂香里多了一丝铁锈味——味像赵氏铁牌，也像天试前的刀。

柳青鸢隔窗未进，未进像怕惊破师兄最后一瞬。她低声：「旧伤别撕。」撕开了，天试前来不及缝。顾小满在院外守至更鼓，守得像守半块石，石温，温成路。

赵玄机在思过崖，听院灯长明，指节叩石，像把闪回当风听——风里有敬，也有妒，妒敬缠在一起，缠成百年。

顾小满不知百年，只知闪回不往外传——传了，就轻了。她守至更鼓，守得像守半块石，石温，温成路。

莫长春再闭目不半柱香，识海里青年又试一次赠药——这一回药被收下，收的人仍不安，不安像另一种还礼。青年因此更懂：赠有时，收的人不安，也是因果。

柳青鸢隔窗未进，低声：「旧伤别撕。」撕开了，天试前来不及缝。

闪回里，少年赵氏抬头，眼里还有敬——敬若还在，百年后的妒，就还有救。敬若不在，百年后的路，也还有问。

弟子渐散，散里仍回头瞧一眼灰袍袖空——瞧得像瞧一眼九月还能走多久。

闪回里，少年赵氏抬头，眼里还有敬——敬若还在，百年后的妒，就还有救。
