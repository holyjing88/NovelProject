# 第四十一章 备检

入门两月余，雪意落了半寸。

丙九棚外贴条：凡骨杂役，测灵备检。备检不是正测，是筛——筛手抖的，筛嘴硬的，筛活不下去的。韩泥读条，独眼平。冬测备检过了，测灵备检仍排末。末，他熟。

邻床少年抖：「听说正测在冬后，备检不过，正测没名。」

「没名就滚。」另一人接，「」

韩泥不插嘴。嘴要留来按石，不留来吵。管事弟子喝：「丙九集合！测灵石前站直，抖的，滚！」

测场在侧门石坪，坪白，石冷。灵石竖中央，石青，青得钝——钝得像秤，秤灵根，秤命，秤你配不配留。第一个少年按掌，石光一闪，绿线三寸。三寸，外门边缘。「过。下一。」

韩泥看手，不看光。手按石前，气先沉，肩后塌。记法，不记色。

轮到丙九，赖福先推人：「丑杂役最后，别沾石晦气。」

管事冷声：「按序。韩泥，末。」

韩泥上前，掌茧裂处疼，疼不露。掌按石，石凉，凉进骨。光起，灰线，线短，短不过指节。众人嗤。嗤声薄，薄也真。

管事瞥一眼：「凡骨末等。备检过——手紧，石前不抖。」

赖福阴笑：「末等也过？丙九丢脸。」

韩泥收掌，不辩。舌要留来嗅渣，留来活。管事掷木牌：「韩泥，备检记名。正测冬后，仍末排。」

木牌薄，真在「记名」二字——有名，不被滚。邻床少年按石，绿线两寸，抖了半寸。管事喝：「抖！滚！」

少年哭腔。韩泥不回头——回头，手易软。

同批有人低声：「灰线也过？」

「过的是手。」韩泥说，「石前不抖，才滚不了。」

那人点头。点头像学紧，学紧，也是小胜。

白织月经坪外，袖风冷：「末等灰线，也配站石前？」

韩泥「嗯」一声：「管事看手，不看嘴。」

白织月唇线绷一息，走了。少一句嘲，小胜。

铁无言抱臂立栏边，声低：「灵根末，手紧，渣场仍收你。别漏。」

「不漏。」

管事收石，喝：「备检毕。记名者，正测冬后。未记名者，明日滚。」

丙九静一息。一息里，有人看韩泥木牌——牌薄，牌真。赖福阴笑，笑浅：「末等记名，正测仍是末。末，滚得快。」

韩泥不答。答，费舌；舌要留来记木牌。笑里藏着冬后滚字——滚字离韩泥三尺远，远一寸，多活一日。

午后兽栏仍清。雪混粪，铲下实。小兽角里咳，咳声细。韩泥多清半刻，湿粪换干草。草干，味淡一寸。管事路过：「备检过了，兽栏别抖。冬角仍你，清细，记着。」

「记着。」

赖福晃秤：「末等灵根，多吃一口也是浪费。」

韩泥不接茬。接茬，全丙九饿。

刘婆舀粥，仍多半勺。半勺，热在。她低声：「备检过了？」

「过了。末等，手不抖。」

「末等怕什么。」刘婆说，「手紧，饭在，命在。命在，丹以后才学。」

韩泥点头。谢轻，恩重。

同批少年凑近：「你怎么石前不抖？」

「搬石练肩，编筐练指，清粪练铲。」韩泥说，「练到紧，石前才紧。」

「可你是灰线……」

「灰线在石上，不在手上。」韩泥说，「手紧，饭在。饭在，才熬得到正测。」

少年闭嘴，像被「紧」字堵住。管事扫赖福一眼：「你也去按石。抖了，你滚。」

赖福脸白，闭嘴。白，小胜。

酉时渣场，韩泥分三堆，堆齐，香渣单放。鼻下苦甘分列，甘短，苦长。铁无言再瞥：「备检过了，仍别漏。」

街角赖福喝：「末等也配分堆？」

韩泥不回头。辱记着，手仍分——分堆，路才宽。管事验场：「丙九韩泥，渣清完，香篓齐。明日仍渣场。」

韩泥拈香渣，篓满。满而不泄，泄则路窄。

夜，韩泥对坛低声：「灵末，手不末。」

坛壁水汽一线，爬半寸，敛。敛着，应手，不应灵色。邻床问：「末等，你还熬什么？」

「熬还。」韩泥说，「手紧能活。活了，才还得上。」

「还得上谁？」

「叶丫头汤，刘婆粥。」韩泥说，「泥岗六笔，宗门一笔。分列，不混。」

邻床又问：「正测冬后，你仍末排，图什么？」

「图活。」韩泥说，「活一日，记一日。记满了，才还得上。」

邻床懂「活」，翻身睡了。外头雪落，落声细，细得像秤盘轻响。响在耳，记在心——备检过了，正测还远，远不要紧，手紧一日，多活一日。

管事巡夜，敲柱：「备检记名者，正测前别生事。生事，饭少。」

韩泥只答一声，声平，平得像木牌落掌。他低声道：「灵根末，命未末。命未末，就接着做粪，做渣，做筐。做到了，路才宽。」

坛沿温一丝，应手，不应灵。灵是石上的灰线，命是手上的茧。他合眼，不睡实，是记名。记名在枕下，命在手上。

丑时，油灯豆，豆光照坛角。他擦沿三遍，沿净。坛壁里浮出极淡影，像半个「瓮」字——一闪，灭。掌心微烫，烫得像血脉里有什么，与坛底沉睡了不知多少年的东西，隔了一层腌菜坛壳，对上了半息。他没喊，没拜，把席盖压严。

邻床翻身，梦呓半句：「……啥光……」

「你看错了。睡。」

他低声对坛：「你眠着。我守。守到沿净，守到缘满。」

坛不应字，只沿口一线温，温短，像应他「守」字，不像醒。

翌日卯前，雪又落薄一层。韩泥先去兽栏，铲下实，小兽咳声轻了半分。管事弟子路过，只瞥他木牌：「末等，兽栏别散。」

「不散。」韩泥说，「散兽栏，先散手；手散，石前就抖。」

弟子无话，走了。白织月袖风远过，仍不近——不近，像备检过了，嘲也省半句。韩泥不追嘲，追活：搬干草，换湿粪，铲磨掌茧，茧裂处疼，疼不露。

午后渣场，铁无言掷香篓。篓口松，香泄一丝，一丝就够认。韩泥分堆，苦甘分列，分列得像辨恩辱——恩苦短，辱腥长，长记着，不碰。铁无言「嗯」一声：「手紧，篓齐。明日仍你来。」

韩泥点头，不谢。谢轻，记在木牌边。木牌薄，真在还能站；站得住，才配廊下嗅路，才配冬后正测末排仍到。

夜归丙九，邻床梦呓问灰线。韩泥声低：「灰线在石上，手在拢柴、清粪、按石。手在，命未末。末等，手不抖。」

备检末席过，灵仍末。灵末手不末——手在，廊下嗅路才长，长向辨香关，长向还烫。木牌在枕下，与备检帖并放。并放，像命与规矩挨着——挨着，才验得过正测。

丑时前，丙九柱上又贴细条：冬测备检记名者，渣场勿漏。韩泥看条，看木牌，看坛沿——沿净，牌在，名在，漏不得。漏了，备检白过；白过，冬后正测没排。他低声道：「末席过，不是灵过。灵在石上灰线，命在掌上茧。茧裂，仍勒条；勒条，仍分堆。」邻床梦呓问「灰线」，他声低：「灰线在石上，手在铲上。手在，命未末。」

「」韩泥只答一声。丑时符影半闪后，他仍不喊醒。喊醒像拜邪；守沿，才配等缘满。备检记名在怀，怀暖，牌更真。真在还能站，站得住，才配廊下嗅路。雪又落薄一层，落声细，细得像秤盘轻响——响在耳，记在心，备检过了，正测还远，远不要紧，手紧一日，多活一日。备检帖角起霜，霜化字更真。真在「记名」，不在灵色。

他摸木牌边，边糙，糙得像心口分列——恩辱不混，混了手就飘。同批少年学他分堆，手仍乱。乱，铁无言不看人，只看堆——堆齐，名在末，也在路。铁无言掷香篓，篓口松，香泄一丝。备检木牌在怀，怀暖，牌更真。真在还能站，站得住，才配廊下嗅路，才配冬后正测末排仍到。管事巡夜敲柱：备检记名者，正测前别生事。

管事记簿，记细，是小胜。胜在忍，不在辩。
