# 第一百二十六章 刘婆补

试台前夜，刘婆又补衣。

刘婆补，肘袄针脚又补一线。线丑，线暖——暖在肘，不在册。册写仍末，肘写恩，恩在还，不在这一日。

「试台？」刘婆哼，「试台别空肚。空肚，手飘。飘了，新衣还不上。」新衣二字，落得像130实锤——实锤在怀，布包棉实。

韩泥点头。谢轻，恩重。恩在粥里，也在针脚里——针脚丑，挡风；风不进，臂才抬得起筐，才捧得住粒。

丙九灶边，灯豆，豆光照她顶针，顶针亮，亮得像36那夜。那夜补肘，这夜补袖口——袖口磨破，破在搬渣磨，磨不要紧，风不进才要紧。风进，手僵；僵了，试台粒洒；洒了，丹不正，刘婆恩还迟。

韩泥坐凳角，凳歪，歪得像末席。末不要紧，手伸，伸得像抬棺不抖：「不用补。」

「用。」刘婆咬线，线紧，「试台明日，袖口磨破，像穷酸。穷酸不要紧，风进，手僵。僵了，粒洒。」

韩泥噤。噤，任她针走。针脚糙，糙得牢，牢得像恩——恩旧，不敢认；认不要紧，韩泥记着，等130还新衣养老钱。养老钱数过三遍，新衣棉实，袖口多缝一层，耐磨，像今夜这针脚。

门外赖福路过，瞥一眼，没笑。没笑，像怕刘婆真扣盆——扣盆不要紧，刘婆眼一瞪：「再看，明日你也没粥！」

赖福走，走，韩泥记——记刘婆恩，记第七笔烫。烫在肘，烫在粥，烫在今夜袖口。袖口暖了，明日试台手才不飘；不飘，就配夺魁预备，就配还刘婆。

「老婆子补，不收钱。」刘婆低声，像36那夜，「钱要，你还不起。还恩，恩太重，老婆子扛不动。」

「扛不动，记着。」韩泥说，「记着，活长了，才还得起。」

「还就好。」刘婆哼，「还了，老婆子敢穿新衣出门，不必躲赖福眼。」

新衣二字，落得像130钩——钩在怀，钩在明日试台后。试台后夺魁，夺魁还刘婆；还了，芽儿清了沈枯芽半块灵石，叶汤还在前。还在前，省亲在后，大比帖在更后。

针收，她塞半块布条：「裹着试台，别挂栏。」

韩泥收下，记。记刘婆顶针，记叶丫头针脚，分列，不混。混了，还乱；还乱，粒不正。不正，刘婆恩还迟，叶汤还凉；凉不得，汤是头的。

铁无言立门外，声低：「明日试台，我站侧。」

「站就好。」韩泥说。

沈枯芽远远鞠半躬，躬浅，像118那夜娘——夜还沈枯芽半块灵石，日补袖；袖暖，手稳，试台不抖。不抖，钱戾衡「看你抖不抖」就虚半分。

韩泥应声。急不得，急像讨魁；魁不要紧，先正粒，先稳手，先还刘婆序不乱。序不乱，丹才正；正了，钱戾衡眼风硬不过「正」。正了，大比帖接得住，叶汤还在省亲前烫。

他回窑，盘坐。十二层圆满，圆满像弓满；弓满不要紧，箭在明日射——射向试台，射向炼气十三，射向还刘婆。还刘婆，恩偿静着还，还到该还的人手里，才算数。

坛腹温一线，应「补」——补袖，补恩，补明日还的路。路在130，路在试台后新衣养老钱；钱不多不少，数过三遍，够一个补衣婆活到老，不必再偷偷摸摸。

丑时，练鼻下静。粟壳香淡，香落齐，刘婆补这一夜记进心里。记进心里，不进舌——舌响，白织月说「可怜」；可怜不是，是还；还了，才正。正了，试台见；见了，魁名落，刘婆手抖能止。

邻床少年悄问：「刘婆为何总帮你？」

「帮活，不帮脸。」韩泥说，「脸丑，袖口不能破。破了，试台手僵。」

少年闭嘴。闭嘴，韩泥摸玉牌：「三年约，记着。明日试台，钱戾衡影再来。」

再来不要紧，袖口暖，手稳，九粒正——正，就够上台，就够夺魁预备，就够还刘婆。还刘婆后，大比帖在怀，言伯钧名在帖上，钱戾衡影在字后。

外头风软，软得像恩将还。还将还不要紧，先试台；试台过，炼气十三破，魁名落，新衣递，养老钱递，刘婆眼浊里清一寸。清一寸，像恩还了一半；另一半，是她终于敢穿新衣出门。

他低声道：「刘婆恩，记着。明日试台，后日还。」

后日还，像127债序清，像129备战帖至。帖至，大比影至，钱戾衡名在帖后——后在不要紧，刘婆这笔先还。先还，叶汤仍在省亲前，芽儿仍在徒线清。

坛不应字，只温。温像夸「补」——补了袖口，暖了手，稳了明日。明日散队听鼓，听试台，听130前最后一夜。最后一夜，还刘婆在望，大比帖在望，叶汤在省亲前。

刘婆门外又咳：「明日试台，别抖。」

韩泥只答：「不抖。」

不抖，是玉牌反面答案；答案在手，在手在坛，在坛在七年忍。忍了，还沈枯芽半块灵石了，补袖了，备战尽了——尽了，就配魁，就配还刘婆，就配听执事扬声「韩泥，魁」。

「刘婆补袖这夜，针脚糙，糙得牢，牢得像36补肘。肘暖了，才走得动还路；袖口暖了，试台手才不僵。僵了，粒洒；洒了，丹不正；不正，刘婆恩还迟。」刘婆低声，「钱要，你还不起。记着，活长了，才还得起。还了，老婆子敢穿新衣出门。」新衣二字，落得像130钩——钩在怀，钩在试台后。

试台后夺魁，夺魁还刘婆；还了，芽儿清了，叶汤还在前。还在前，省亲在筑基后，大比帖在仇路前。铁无言立门外：「明日试台，我站侧。」沈枯芽远远鞠半躬，躬浅，像118那夜——夜还石，日补袖；袖暖，手稳，试台不抖。路在130，路在试台后新衣养老钱；钱不多不少，数过三遍，够补衣婆活到老。

香落齐，刘婆补这一夜记进心里。记进心里，不进舌——舌响，白织月说「可怜」；可怜不是，是还。」后日还，像129帖至。忍了，还芽儿了，补袖了，备战尽了——尽了，就配魁，就配还刘婆。补袖毕，韩泥试抬陶罐，罐沉，沉得像捧热汤。汤是头的，粒是今日的，刘婆新衣是130的。130未到，先稳手，先不抖。

不抖，钱戾衡半年後试台，也不抖。不抖，就配127债序清，配128验正，配129备战帖，配130魁与还刘婆。针脚暖进骨，骨暖，天寒不怕。不怕，试台手才不僵。他摸玉牌，牌凉，低声道：「三年约，记着。他低声对坛：「钱戾衡，记着。赖福克扣，记着。」记着，刃磨亮。亮在还刘婆，亮在夺魁，亮在大比帖至山门。

刘婆粥仍多半勺，姜仍多一片。片暖，恩不断。恩不断，手才稳；手稳，粒才正；粒正，试台才配说话。说话了，白织月袖风尖不过「正」，钱戾衡影硬不过手。

袖口暖，手不僵。不僵，试台不洒，不洒钱戾衡白说。

补袖毕，韩泥试抬陶罐，罐沉，沉得像抬棺不抖。不抖，试台手才不僵。僵了，粒洒；洒了，丹不正。不正，刘婆恩还迟。迟了，叶汤也还凉。凉不得，先稳手，先验正，先末项，先130魁与还。还了，敢认恩；认了，眼浊清一寸。清一寸，像恩还了一半；一半不要紧，新衣养老钱在怀，在130递全。

「手稳，粒正，试台见。」见字落，坛温一线，像应。应了，路又明一寸——明向还恩，明向不滚，明向刘婆新衣养老钱。袖口暖，试台不洒，不洒仇不滚。更鼓沉，沉里，韩泥按木牌，牌凉贴胸。126章手在，名在册，坛沿净。舌不留，鼻下静。恩辱分列，不混。活长了，才还得了叶汤，才接得住门缝灯。坛腹微温，应「稳」字。

稳不是飘，是丑翁专精；专精在渣，专精在窑，专精在五年忍。忍了赖福压顶，还了沈枯芽半块灵石，备战才真；真备战，夺魁，还刘婆，大比帖在怀。刘婆门外咳：明日试台，别抖。韩泥只答：不抖。忍了，还芽儿了，补袖了，备战尽了。尽了，就配魁，就配还刘婆，就配大比帖影里钱戾衡名。名在帖后，后在不要紧，刘婆这笔先还。
