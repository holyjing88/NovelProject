# 第五十五章 杖剑出

剑起时，没有少年天才的吼，只有杖点地的一声轻响。

那响很轻，轻得像老人敲茶盖；可东脉灵田边上，所有还在喘气的人，都听见了。

灵田中央，稻茬还冒着烟，血把泥染成暗褐。赤焰金丹修士凌空立，灵光罩体，像一座移动的炉，炉盖掀开，压得四周弟子跪地，压得柳青鸢剑意一滞。

韩铁山在远处举旗，旗红映甲，笑深：「青岚门底牌，不过如此。」

弟子群情骇然，有人退，有人跪，有人仍咬牙往前，像半块灵石撑着他们不退——撑到九月老头拄杖前行，一步，两步，像把寿元走成剑距。

顾小满从侧径奔回，浑身泥血，眼里仍亮雨夜那道光。她看见长老袖中青霜剑鸣，剑未全出，剑意先出，霜落刃上，像秋晨的露。

「将死之人……」赤焰金丹修士冷笑，「也配——」

话未落，剑已落。

莫长春杖尖点地，青霜剑在手。剑是柳氏，意是莫氏；筑基灵压按理推不动金丹——可塔影在识海沉，沉得像七十五斤人情同时了结恩仇，沉得像手帕定身那夜没用完的因果，今夜一并兑。

不快，像斩乱麻；却准，像斩因果。霜色一线从灵田中央划过，划过护体灵光，划过喉前一线，像划过一张欠久的恩。没有花哨的剑诀名，只有说不清的话：**大因果，延迟兑。**

金丹修士目睁，不信，身子坠，坠进田里，泥溅，像一颗炼坏的丹。灵光碎成无数细屑，细屑落进未收的稻茬，落进血泥，像一场迟来的霜，霜过即融。

田里静了一瞬，静得连火星跳的声音都听得见。霍镇山盾举在半空，忘了骂，像一尊石化的山。丹堂方向，地火阵仍亮，仍响，苏念慈在阵眼里把最后一点灵力榨干，却笑了一线——笑像炉里细火，笑给灵田那边看。

全场寂然。

寂了三息，像手帕定身倒放——这一次，定住的是全场人的嘴。三息里，有人数心跳，有人数稻茬上的血，有人数九月老头须眉在火光里白得刺眼——白得像一张旧符，符旧，意满。

外门弟子先是呆，继而炸：「斩……斩了？！」

「莫长老一剑……」

「九月老头？！」

霍镇山忘了骂，盾举在半空，像一尊石化的山。韩铁山在远处，喉头一紧，旗杆滑手，险些坠地——他忽然想起手帕定身，想起妖骑被杖扫，想起魏无涯八字：气运异数，勿轻敌。

柳青鸢剑未回鞘，眸光剧震，震后定，定成宗主的稳：「护法长老……斩了？」

莫长春收剑，剑归袖，杖再点地，声轻，像什么都没做，又像做尽了：

「一剑，够还一段恩。恩还没还完，剑先歇。」

他须眉在火光里白得刺眼，笑淡，淡得像霜落在脸上，凉，却不苦。

赵玄机在丹堂门洞，离得远，却听得清。他妒了半辈子，此刻只剩喉头发紧，紧得像咽下一颗培元丹——苦，却补脉。他低声，仅自己：「原来……不是演。」

苏念慈在阵眼，灵力将尽，却笑了一线，笑像炉里细火：师兄在灵田斩金丹，她在丹堂守炉，各算各的戏，各还各的礼。

围观弟子炸成浪，浪里慕，也怕——慕的是一剑，怕的是九月老头还能出第二剑。

有人低声：「手帕定身，杖扫妖骑，如今一剑斩金丹……」

「三连击？」

「九月老头，还是人吗？」

另一人答：「是人。是将死之人，把命押在还礼上。」

韩铁山在远处，面色铁青，旗挥，妖骑再冲，冲得像用妖帅的怒，盖住自己刚才的骇。

厉无殇立高坡远望，面色骤变，骂声更厉，却不敢再上前半步——手帕定身那三息，妖骑被杖扫，金丹被一剑，像三连击打在他脸上，打得他恨与惧绞在一起。

魏无涯在远处梁上，扇合，悄然退，像看一枚子落盘，落满，仍未终局。

周德海在后阵棚下，令旗举起又落，落得像侧门阵闸仍不开——顾小满暗线报来的空隙，他选择不信，不信得像把青岚门一半胜算，关在闸后。

莫长春不追韩铁山，只望丹堂方向，地火阵仍亮，仍响。

「苏丫头守得住。」他低声，「赵小子……该还人情了。」

风过焦土，腥里掺一点丹香，像两条战线在这一夜，终于拧成一股。

苏念慈在阵眼，单膝微屈，灵力见底，却笑，笑很轻，像三月前整理衣襟那一下：师兄在灵田斩金丹，她在丹堂守炉，各算各的戏，各还各的礼。

弟子扶伤退，退时仍回头看莫长春背影，像看一块旧符，符旧，意满。

柳青鸢收剑，剑意未散，望向师兄，只一字：「够。」

莫长春点头：「够一夜。不够一生。礼，还没送完。」

塔壁浮字，淡金，像迟来的围观层记在心里：

**一剑有时，恩还在袖。**

大因果，却未兑尽——韩铁山还在，厉无殇还在，妖雾还在东方酝酿。

莫长春抚杖，笑，像对簿说，也像对九月说：

「下一阵，换更重的还。」

顾小满从侧径望见一剑，眼里亮，亮得像雨夜那道光还在。她低声，仅自己：「长老……不是演。」

霍镇山盾举在半空，骂声终于出口，却变了调：「莫长老在，青岚还在！」

浪声里，有人把这一夜讲成戏本，戏本里只有三息——三息定魔少，三息定全场，三息后，剑归袖，杖再点地，像什么都没做，又像做尽了。

魏无涯在远处梁上，扇合，悄然退，像看一枚子落盘，落满，仍未终局。周德海在后阵棚下，令旗举起又落，落得像侧门阵闸仍不开——不信暗线，像把一半胜算关在闸后。

柳青鸢收剑，剑意未散，望向师兄，只一字：「够。」莫长春点头：「够一夜。不够一生。礼，还没送完。」

围观弟子在田埂上踮脚，有人低声：「筑基斩金丹？」另一人答：「斩的不是境，是恩。」

妖雾在东方再起一线，像牙已露，露在双妖帅那夜之前。寿元在塔壁跳：两月。莫长春瞧见，笑淡：「两月够还，不够吵。」

大因果还在双妖帅那夜，像妖核，像周犹豫——一剑斩金丹只是小兑，青鳞妖君才是大还礼的前奏。

厉无殇立高坡远望，面色骤变，骂声更厉，却不敢再上前半步——恨与惧绞在一起。

田里静了一瞬，静得连火星跳的声音都听得见。外门弟子先是呆，继而炸——炸在延迟那一息，炸在斩完之后全场仍不敢信。有人低声：「筑基斩金丹？」另一人答：「斩的不是境，是恩。」

魏无涯在远处梁上，扇合，悄然退，像看一枚子落盘，落满，仍未终局。周德海在后阵棚下，令旗举起又落，落得像侧门阵闸仍不开。

围观弟子在田埂上踮脚，踮得像等第二剑——第二剑未出，剑已归袖，杖再点地，像什么都没做，又像做尽了。

苏念慈在阵眼，灵力将尽，却笑，笑很轻，像三月前整理衣襟那一下：师兄在灵田斩金丹，她在丹堂守炉，各算各的戏，各还各的礼。

塔壁浮字，淡金，像迟来的围观层记在心里：**一剑有时，恩还在袖。**大因果，却未兑尽——韩铁山还在，厉无殇还在，妖雾还在东方酝酿。

莫长春抚杖，笑：「下一阵，换更重的还。」
