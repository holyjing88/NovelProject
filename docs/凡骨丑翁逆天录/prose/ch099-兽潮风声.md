# 第九十九章 兽潮风声

条至。

药山界柱上，新贴兽潮预警，预警丑，字真。真在「杂役备障、夜禁加严、外门轮值」。

韩泥读条，不近，不躲。躲，像怕；怕不要紧，怕也备。

备，首在耳。耳里，铁无言说：「兽潮带瘴，瘴入栏，栏先倒。」

韩泥记：「栏先倒，手稳。」

记，次在鼻。鼻下嗅诀，嗅过瘴气描述——描述在经片，经片说「腥里带甜，甜为毒」。

甜毒，他记在心里。记，不在舌；舌要留来清栏，留来守坛。

赖福召集丙九：「兽潮来，丙九守兽栏外围，不许逃！」

韩泥立队末，应声：「在。」

在，就不逃。不逃，才并肩——并肩在108，在远；远不要紧，风声已到。

白织月立廊下，袖风带香：「丑杂役也守栏？守住了，别拖内门后腿。」

韩泥只答：「不拖。手稳。」

刘婆缝布条，条粗，塞韩泥袖：「蒙鼻，别嗅太过。」

韩泥收，收不谢。谢轻，恩在布条里。

沈枯芽怯，悄问：「韩叔，怕不怕？」

「怕。」韩泥说，「怕也守。守完，才有饭。」

饭稳，恩不断。

夜，远岭兽吼，吼低，低像潮前滚雷。滚雷不要紧，滚在远；远，一日近一日。

他席下按坛，低声道：「风声到了。下一回，前夜。」

坛腹温，像应「潮」字。潮未到，备先到。

邻床睡不实，问：「真会冲药山？」

「冲不冲，都要当冲。」韩泥说，「当冲，手才不飘。」

丑时，静半刻。香落齐，掌纹温一线。温，像培元路稳半分——稳，为清瘴备。

赖福查夜，火折照栏：「丙九别睡，睡出瘴，连坐！」

韩泥立，立像桩。桩不要紧，桩稳，栏才稳。

更鼓尽，兽吼又近。近不要紧，近，心记多一行：兽潮风声，已至耳。

柱上新贴兽潮预警，字真。真在「杂役备障、夜禁加严」。

铁无言说：「兽潮带瘴，瘴入栏，栏先倒。」

赖福召集丙九：「不许逃！」

韩泥立队末：「在。」

刘婆缝布条，条粗，塞袖：「蒙鼻，别嗅太过。」

沈枯芽怯问：「怕不怕？」

「怕。」

远岭兽吼，吼低，低像潮前滚雷。

赖福查夜，火折照栏：「丙九别睡！」

桩稳，栏才稳。

他读条三遍，三遍不要紧，三遍才记牢。记牢「甜毒」二字，记牢「栏先倒」，记牢「不许逃」。不许逃，不是勇，是活。活，才还恩。

沈枯芽送刘婆布条时，怯声：「叶姐脉……稳了。」韩泥不接话，接在心里。脉稳，是95；风紧，是99。两笔不混，混了，七笔乱。乱，路断。

条至第三日，韩泥清栏加趟。栏净，净不要紧；净，少瘴入。

铁无言分障草，草湿，湿不要紧；湿，才挡甜腥。甜腥在风里，风一过鼻，他仍嗅出一线毒——毒细，细而记。

沈枯芽送水，水凉：「叶姐说……别逞能。」

韩泥接水，不接话。话在还，不在嘴。
