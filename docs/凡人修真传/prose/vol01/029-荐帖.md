# 第 29 章 荐帖

老耿养伤，陈寻每日送水。

瓮搁他床头，泥缝干，像一座小碑。碑不言语，言语在耿叔独眼里——眼亮，亮得像还能守夜，亮得像不信自己会废半力。臂吊着，吊得像提醒第三夜；第三夜过了，命还在，臂废半力，刀还得磨，磨给下一夜，磨给还恩的路。

陈寻送水，不多说，多说就成恩；恩多了，恩价更乱；乱，三月前就站不稳。送水是工，工在沈铺，工也在邻屋；邻屋咳声远，远里像嘱：活着，站直。

老耿臂吊着，独眼仍亮，见陈寻不谢，只问：「心里，又记了？」

「记了。」陈寻说，「你活，我记。你死，我也记。」

老耿哼一声，像骂，也像认：「小崽，别把自己记穷了。」

「穷不穷，看还得完还不完。」陈寻说，「还得完，就不穷。」

陈寻不辩，担水，换布，像还一笔放长线还的工。工在沈铺，工也在路上——路上肩肿，肿是昨夜背下来的；背下来了，就不能当没发生。肩肿消得慢，消得慢，便多跑一圈，多闻一味，多念一遍恩——页里八笔温，温得像八根桩，桩住他，让他别飘，别省，别病。

萧惊鸿来骂猎户逃，骂到一半，见老耿睡，收声，只帮陈寻劈柴，柴声脆，像把北岭那夜劈碎一点。碎一点，不够，够的是名还在，帖还在，路还在。

---

第七日，破庙又见程不二，这次有酒无字，有帖。

庙香仍冷，冷得像北岭夜风没散尽。供桌侧，程不二坐，膝稳，酒壶搁膝，话也稳：「杂役是后门，荐帖是侧门。」

帖黄，比杂役帖正，角齐，印浅：**碧云宗外门 · 荐**

「测灵亮，荐帖助入外门；不亮，荐帖仍保你不被刁福来随意除。」

陈寻接帖，问：「代价？」

「活。」程不二说，「活到还完七笔……不，活到还完你心里所有阳。」

他指陈寻怀：「恩多，走得慢，也走得远。远，不是躲赵绣春，是站得住石。」

陈寻看帖背，无地图，只有一行小字，笔稳，瘦，像药铺里常见的那种：「别病。病，站不稳石。」

他心一动，不入心——未落款，不算恩，算嘱，像有人隔空递药，不递名。嘱比恩轻，轻里更紧，紧在三月，紧在石，紧在那块还未亮的灵石上。

程不二笑一下，笑里无温：「沈家那铺子，你少登门。登门，是工；不登门，是规矩。」

他又指北岭：「狼患未平，你别逞。逞，活不到三月。」

人又没，酒壶留，帖在手。没得像风，风过，香仍冷，冷里帖暖，暖得像第二条路——路在纸上，路也在骨上；骨上要过两月，月像绳，绳勒告示角，角在村头，也在心里。

陈寻双帖在怀——杂役，荐。像两条腿，一条粗，一条细，都通向三月。粗可走，细可攀；攀不攀，看石；走不走，看命。命在，两条腿就不废；废了，帖也只是纸。纸与石，都不替他活——活，还得自己喘。

回村，沈家药铺灯仍亮，亮得稳。陈寻路过，不停步，只心里记：缝臂、止血、一句「先活」——工未还，恩已压在心里，压得不言。不言，不是忘，是怕言多了，情多了，心乱了。

第二日，院门又有一包，无署名，布粗，里头采药单，单上列三十味，旁注「练体后闻，别省」。陈寻识笔迹——沈振东，稳，瘦，像他的人。单末一行小字，非沈振东笔，像沈清弦：**「别病。病，站不稳石。」**

陈寻把字记心里，不入心——未落款，不算恩，算嘱。嘱与帖背那行字同，同得像沈家隔帘递话：话不到口，意到了；意到了，就得活。

他不登门谢，谢是价；价一出，工就变恩，恩他记，不在谢里。按单练：晨跑，闻药，午识，夜默想。跑时喘，喘匀；闻时苦，苦记；翻时阳温未退，退不了，像老耿那笔命还压在心里，压得他不敢省一息。

萧惊鸿来，见单，骂：「沈家又帮？帮，你还不去谢？」

「工。」陈寻说，「我搬过货。单，是续工。谢口，欠情；欠情，恩价更乱。」

萧惊鸿不信，却见陈寻练得更狠，汗落，步稳，不再劝。劝不动，便信「名在」三个字——名在，拳才不出；不出，路才直；直，才站得上石。

老耿在沈铺养，隔日问：「荐帖，是什么？」

陈寻答：「路。」

老耿独眼一眯：「路窄，别摔。」

陈寻抚袖中石「三」，不言语——石凉，帖暖，心沉，像三条线拧一股，往三月收。拧一股，不是乱，是排期——排什么时候该登门，什么时候该封门；排什么时候该守夜，什么时候该练体；排到最后，排向石。

夜里，陈寻数心里阳温：食、盐、汤、衣、名、粥、瓮、命——八笔，尚未还一笔。八笔像八根桩，桩住他，让他别飘，别省，别病。桩在，人就在；人在，帖才不是废纸。

他摸青冥瓶，箱底，灰扑扑，无动。无动，不是无；无，是还没到时候——到时候，也许在北岭，也许在石前，也许在更远的山门。摸一下，像数心跳；心跳在，人在，恩仇在心里。

测灵，两月余。他展杂役帖、荐帖，帖背无图，只有嘱。嘱短，命长；命长，就得练。练体，闻药，数息，像数恩，数到三月，数到石上。

窗外，北岭远，狼嚎偶一，像催。方小沅在告示下贴一小纸，无人知，纸上两字：**等你。** 陈寻未看见。看见了，也不会应——应，是还不起时的事。

他摸袖中石「三」，又摸荐帖角，角齐，像路已划好，只欠脚。脚若不病，石便能站；石若站不稳，帖便是纸。纸与石，都不替他活——活，还得自己喘，还得自己闻苦，还得自己把三十味一味不漏地记进鼻，记进骨。

老耿在沈铺咳一声，像远钟。陈寻侧耳，耳静，心稳：命记了，荐到了，沈家暗助已至。谢口不说，练体不止。不止，不是狠给人看，是给石看——石前站不稳，一切恩都成笑柄；笑柄，赵绣春会唱，童谣丫会学，冷眼会比嘴更久。

他把采药单夹进杂役帖，与荐帖并列，像两条腿，一条练，一条进。练罢，闻药，苦味入鼻，他想起沈振东缝臂时那句「先活」——活，才是还恩的时辰。时辰未到，先练；练到稳，才配谈还。

窗外狼嚎又起，远，像催，也像试。陈寻不抬头，只数息，一息，两息，像数恩，数到三月，数到石上「三」。三在袖，三在帖背，三在北岭第三夜——三不是咒，是数；数得准不准，不由程不二，由步。

他想起破庙程不二：酒无字，帖有印。印浅，路却深。深路走法只有一条——别病，别死，别在还恩前倒下。倒下，八笔阳温就成恩落空；恩落空，恩价就脏；脏，心就乱。

沈家药铺灯灭时，村东仍亮一灯，像有人醒着缝纱、配药、记恩。陈寻不入那光，只把采药单再展一遍，三十味，一味不漏——漏一味，闻不准；闻不准，站不稳石。石在三月外，名已先在；名先在，人就得先站直。

他关窗，睡。睡里，帖角「荐」、石「三」、心里「命」，三条线并一股，往三月拽。拽他的，不是程不二，不是沈家，是他自己数的那八笔阳温——一笔未还，就不配站稳石。

醒时，天未亮，他先闻药，再展单。单在，帖在，命在，路就在。路在，就不怕北岭远。远，是狼；近，是三月；中间，是练，是闻，是数息。数到稳，才站得住。

他低声，只给自己：「荐到了。练，不止。」
