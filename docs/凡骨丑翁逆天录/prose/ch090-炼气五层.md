# 第九十章 炼气五层

丹路开后第三夜，丑时。韩泥未睡实。睡实了，五层才冲得开；冲得开，才接得住95那盏门缝灯。他盘坐床沿，席严，坛不露。油灯豆，豆光稳，稳像暗袋十包渣那一点齐——齐在底，不浮。掌贴沿，沿温一线。四层气转，转一周，转两周，转第三周时，丹路那一字「开」在鼻下跳一息。跳不要紧，跳才活。

丹路开后第七夜，丑时，气聚掌纹，冲关隘。隘裂一线，五层开了——开像内视里阶影又亮一阶，亮在暗，不在册。册仍写仍末，掌见五层，见给自己。

五层稳在掌，稳像加验十堆全对——全对才稳，稳才接得住91试炉，接得住95匿丹。他收功，收得像盖席。盖严，沿净，境敛。

活在渣，活在暗，活在沿净处。气沉腹，沉到掌纹。掌纹烫，烫得踏实。踏实了，关隘裂一线，裂声闷，闷在骨里，不在屋里。五层开了。开不像雷，像瓮唇吐白气——白气冷，冷里带药香，药香里带一点甘短，甘像叶家方向。甘短不要紧，甘在记里，不在舌上；舌响，香飘，煞骨门就嗅得到。韩泥屏息，独眼不眨。眨了，境就散。

散不得，散了，五层白开。气稳在五层，稳像踹台按实——掌在，境在，路在。路通向95匿丹，通向坊市挂牌，通向还叶丫头那一碗烫。盖严，沿净，坛不露。不露，像腌菜坛，不像邪。掌纹那路比昨夜厚半分，厚不要紧，厚在暗，不在册。册仍写仍末，掌写五层——写给自己，不写给别人。

邻床翻身，眯眼半息：「你……身上咋又热？」

「编筐扎的。」韩泥声平，「看错了。睡。」

小杂役哼一声，又睡实。睡实了，五层才藏得住。更鼓将尽，天边白一线。白里，他廊下练半刻鼻下静，静到只闻辛淡，不闻辱。五层在掌，四层在忆，丹路「开」在鼻——三层分列，不混。混了，香飘；香飘，手就滚。铁无言脚步远，停半息，声更低：「五层别叫。叫了，像求境。」

韩泥不答，答在心里——心里记一笔：暗修五层，境微进。进在丑时，进在席下，进在沿净处。坛底轻颤，臾墟残念弱一息：「……五层在掌，匿丹在远。远不要紧，渣在先。」

韩泥低声道：「记着。」

韩泥只答：「油灯豆。豆爆，爆完即灭。」

赖福盯半息，终是没掀席。掀不得，沿净，坛不露。不露，晦气遮瓮，遮住了，五层才稳。卯时，兽栏仍腥。腥不要紧，五层在掌，铲下更稳。稳，像踹台那一掌——掌在，境在，路在。管事弟子巡栏，记一笔：「丙九韩泥，清细。」

刘婆端粥，仍多半勺：「昨夜又热？」

「豆爆。」韩泥说，「沿净。」

「沿净怕什么。」刘婆哼，「饭稳，恩不断。」

韩泥点头。恩在粥里，也在五层境里——境轻，记重；记重了，才匿赠叶。

卯时兽栏，铲下比昨日稳一分。稳一分，管事记「仍清细」。刘婆端粥，仍多半勺：「昨夜又静？」「静了。五层在掌。不叫。」

韩泥应声：「不炼。试温，记着。」

铁无言又道：「95在远，门缝留灯。灯在，别贪名；贪名，路断。」

95在远，门缝留灯——八字落耳，像钩。钩不要紧，先记——记五层，记丹路开，记匿丹在远。白织月远廊过，袖风冷：「杂役也五层？五层也炼不出丹。」

韩泥不辩。辩，费舌；舌要留来守境，留来守坛。他只答：「境在掌，不在舌。丹在远，渣在近。」

白织月唇线绷一息，走了。少一句嘲，小胜。秦霜木鱼轻敲：「到境心别贪。贪，邪。」

韩泥不答，答在心里——心里七笔分列。五层是境，境是路；路通向95，通向还恩。他低声对坛：「五层到了。不叫。坛在，袋在。袋在，95才接得住。」

坛不应字，只温。温，像叶汤碗沿又热回半分——热在记里，不在舌上。丑时前，他再练半刻。四层在忆，五层在掌，六层在远——远在130，远不要紧，脚不盲。他独眼平，手不抖：「五层到了。95在远。门缝灯在远。」

韩泥只答：「不恋。开在渣，记着。」

更鼓沉，沉里，他摸暗袋，袋沉，渣像种。种轻，记重；记重了，才炼得出95那一粒培元丹。坛腹应温，像应「五层」最后一笔。应完，他不叫。不叫，才守得住暗修，守得住门缝灯。邻床悄问：「你身上热，真没事？」

「没事。」韩泥说，「沿净，不叫。不叫，才藏得住。」

少年不懂「藏境」，却懂「不叫」，闭嘴睡了。他低声道：「五层到了。舌不留，手要稳。稳了，95才匿得出，叶丫头那碗烫才还得了。」

还得了，在远，在95；远不要紧，境在近。近在掌，近在袋，近在丹路开那一字。赖福讽远：「末等也五层？五层也滚。」

韩泥不接。舌要留来记，留来忍。忍不是怂，是刃——刃对着三年约，对着95，对着还恩那日。柱上条角白一线，三年约在怀。怀凉，掌烫——烫在五层，不在众目。众目看灰线，掌里多了一尺暗线。风硬，风里一点甘短回半息，回像叶家方向。回不要紧，记在心里，不进舌。在，就不滚。

不滚，五层才扎得实——扎实了，才接得住95暗炼赠丹，才接得住门缝那一盏灯。五层破后第二日，药渣场春验。韩泥分拣，鼻下清，十堆全对，全对稳。管事记「春验仍准」。准，是小胜。胜在掌，不在舌。铁无言声低：「五层配槽，废窑明日寻。寻到，别炼今日。炼在95，匿在门缝。」

韩泥应声：「匿，不叫。还，不在舌。」

温听雨声清：「眼更静了。静，像能守炉的人。」

静，韩泥不谢。谢轻，记在心里——记缘苗，记各做各的，记498在远。赖福讽远：「末等五层？五层也滚不出沉丹宗。」

韩泥不接。舌要留来掌，留来守坛，守三年约这一条命。刘婆端粥，仍多半勺：「五层别飘。飘了，恩断。」

「不飘。」韩泥说，「飘了，叶丫头汤白挨。」

叶丫头三字落耳，愧线「别来丑」又烫半分。烫不要紧，分列——愧是愧，恩是恩。午后废渣巷口，他立半息，记废窑方向。方向在远，手在近；近在，就不滚。秦霜木鱼一声：「五层别骄。骄了，香飘，匿就败。」

韩泥不答，答在心里——心里不骄，只记95，记门缝留灯，记培元丹匿赠叶。黄昏，角柜锁实，五层稳，细槽明半分。明，像丹路将行；将行不是今日炼，将行是还恩前头。邻床问：「95啥日子？」

「还恩一日。」韩泥说，「远在废窑后，近在手。手稳，灯才留得住。」

丑时前，廊下练半刻。五层气转一周，稳，细槽温，影淡现门缝灯。他低声对坛：「五层在掌。废窑明日寻。95在后，叶丫头，门缝灯叔会来。更鼓沉，独眼平，手不抖：「五层稳了。91废窑试炉，93匿丹准备，95恩还暗爽。」钱戾衡名在仇线，玉牌在怀，三年约在怀——约在，就不滚。不滚，才磨得出刃，才还得了热汤。

柱上条角白一线，春风硬，硬不要紧，手在；手在，路就不绝。廊下风硬，他把木牌系紧一线，系得像把仍末摁进掌心。同批少年远避，避像避晦气，晦气遮瓮，遮住了才稳。编筐角藤刺仍在，他偏手，刺浅些，沿才净，坛才藏得住。管事夜巡验棚，灯照席角，沿净，像腌菜坛，不像邪。」

老耿挑水过，声哑，境别拜邪，拜了路断，他只记着。

秦霜木鱼远，一声，心别邪，邪了鼻下香飘，香飘手就滚。

柱上条角白一线，三年约在怀，愧字在怀，手仍不抖。

更鼓沉，沉里，韩泥按木牌，牌凉贴胸。90章手在，名在册，坛沿净。舌不留，鼻下静。恩辱分列，不混。活长了，才还得了叶汤，才接得住门缝灯。

挂牌末，有人询价，价低——低不要紧；低，才像试水，不像求名，试台还远。
