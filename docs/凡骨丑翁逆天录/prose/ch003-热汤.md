# 第三章 热汤

天又冷了。

泥岗村入秋，风硬，韩泥的漏屋先尝寒意。他早起编筐，指尖僵，藤条不听话。心口那碗汤的温，他还没记实，像习惯跛脚一样，习惯心里空着。

不是没人恩他，是恩还没记——韩泥不知该怎么记，只晓得一件事：恩不能乱记，记了，就得还。
晌午前，门缝透进一点光。

有人影顿在门口，没敲，也没喊。韩泥抬头，见一双布鞋，洗得发白，鞋尖沾泥，站得稳。

「叶丫头。」他开口，声音哑。

门外那人「嗯」了一声。

叶青禾。

村里都说她命苦，男人走得早，一个人撑药铺帮衬，清瘦，耐看，不张扬。她与韩泥隔门站着，像隔一道看不见的线——晦气翁不配让人进门，她也不进，怕惹闲话。闲话在泥岗村比风快，风过无痕，闲话能留三年。

碗递进来。

粗瓷，沿口缺了一小块，汤热气还在，香味淡，是姜。

「趁热。」叶青禾说。

就两字。没有可怜，没有嫌弃，没有「你怎么活成这样」的叹息。韩泥最怕叹息，叹息比骂还轻贱人。骂是明的，叹息是暗的，暗的扎心。

他伸手接碗。

指尖碰到碗底，一物硌手——薄片，切得细，藏在汤下。姜。

韩泥没抬头，只道：「够了。」

叶青禾在门外停一停：「天冷，门缝大，别冻着。」

韩泥指尖一顿：「冻不死。」

「我知道。」她声轻，却硬，「所以我送热的。」

门外有人咳，咳得像周嫂在巷口。周嫂没近，只甩一句：「叶丫头，药铺还开不开？」

叶青禾应：「开。秤在，铺就在。」

咳声远了。远，像谣还没起，起的是风。风硬，汤热，热抵风半寸。

他捧碗站门内。汤入喉，烫，辣一点，胃暖。姜片挑出，窗台晾干——穷人家，姜用两回。叶青禾脚步远去，布鞋踩泥，轻，稳。
他坐下，碗搁膝上，汤已尽。

韩泥不会写好看字，也不会把恩划在纸上——老农民哪懂这些？懂的是欠了要还，辱了要忍，仇了要候。候在心上，比写在纸上实在。

叶青禾这一碗汤，是头一笔恩。深，像一道旧伤，烙在心里。

他低声道：「汤还是热的。」

不是对她说。

是说给自己听——记着，有一天还。热汤能凉，恩不能凉。凉了，人就成了泥岗村那些嚼舌根的嘴，轻，薄，一吹就散。
傍晚，村童又在远处嚷「晦气翁」。

韩泥没出门。他坐到坛边，腌菜坛白日又凉，入夜却温了一指——温应心口那碗汤，不应谣。

赵绣春门内声尖，尖不过叶家药铺那边秤杆落盘的脆响。脆响稳，稳得像把谣压回半寸。

韩泥听见了，不笑，只按心口。心暖，秤稳，够他活过今夜。

卖盐老汉路过，在门外停一息：「叶丫头心善，善在实。实，不是邪。」

韩泥「嗯」一声，不谢。谢轻，他记。

老汉走了，脚步沉，沉得像公道。公道不常来，来一次，够撑明日井台。

他不懂太多道理。

叶青禾为何送汤？不为名声，要名声不会沾晦气翁。不为可怜，送法不像可怜。像人该送——人该送时，不图回报，图心里过得去。韩泥五十八年，少见这种过得去。

他只懂记在心里。

心里多了一笔恩，就得活久些，活稳些，活到能还那一碗汤。还的时候，不能寒酸，不能敷衍，不能像村头那些「算了算了」。

童谣丫远处嚷一句，嚷完被娘捂嘴。捂了，够。够他今夜安一息。

他伸手，按坛沿，又按心口。

汤入腹，心口热。坛沿本凉，竟温了一线——温得轻，像应那碗汤，像应「趁热」二字。蒙尘里水汽一闪，敛进壁里，瞧不出仙气，只瞧得出：坛不拒热，拒的是脏水。

坛凉，心热，人夹在中间，五十八岁，仍活着。

门外暮色沉下来，泥岗村像一口旧瓮，罩着全村。远处赵绣春家亮起灯，亮得张扬。韩泥在瓮底似的屋里，轻声道：

「这恩，我记了。」

心里那一笔，丑，却真。

他洗尽粗瓷碗，搁回门墩。夜雨敲瓦，碗底叮一声。

叶家药铺灯亮到夜深，秤声又起一声，起得像钉。钉住「铺子没关」，钉住他心口那笔恩。

韩泥合眼。心里只留两字：趁热。

坛沿一线温，贴掌不退。不退，像替他守着这笔恩，守到还得上的那日。还的时候，碗要烫手，姜要还在——还在，叶丫头才晓得晦气翁没忘。

门外赵绣春又骂一句，骂给风听。风过，秤声又起——起一声，骂就哑半寸。哑半寸，够活。

他低声道：「趁热。记着。还。」

三字，丑，真。真，便按实，按到心口，按到坛沿，按到明日井台。

张麻婆门外骂童谣丫：「少学周嫂那嘴！」骂完，塞门缝半块饼：「吃。编筐手不能抖。」

韩泥「嗯」一声，饼掰半，半留门墩。留，像还恩——恩在，路在，井台明日还去。

叶铺秤声又起，起得像替他把谣压回三寸。三寸够活，够还，够走到宗门风声那日。

沈枯芽隔门问：「韩爷爷，汤烫不烫？」

韩泥「嗯」一声：「烫。烫好。趁热，才记得住。」

芽儿还想问，娘拽走：「别叨晦气翁。」

韩泥不接，只把窗台姜片按平——按实，像按心。

张麻婆折返，嗓粗：「叶丫头心实，你别多想。」

韩泥：「不多想。想还。」

还的时候，不能寒酸，不能敷衍。敷衍像泥岗村那些「算了算了」——算了，恩就凉。凉不得。

张麻婆：「还什么还，先把筐编完。」

韩泥「嗯」一声，藤条又绕膝：「筐编完，人还在，才还得上。」

卖盐老汉远处补一句：「实人送实汤，别叫谣搅浑。」

韩泥不接，只按坛沿。沿温一线，像应「实」字。

沈枯芽扒门缝：「韩爷爷，明儿还喝烫汤？」

韩泥「嗯」：「喝。烫着，才记得路。」

芽儿缩回去，像一粒热姜落回汤里。

汤入喉，烫。烫过风，风就短半寸。姜片挑出，窗台晾干——穷人家，姜用两回。用两回不是抠，是活路长。叶丫头没进门，门缝递碗，像隔一道看不见的线。线外闲话快，线内汤热。热是真，闲话是虚，韩泥专记真。他摸碗沿，姜辛还在。辛像字，字没写进册，写进心——心册丑翁自己管，漏不得。恩不能乱记，记了就得还。还什么，还热。热凉了，恩就乱。秤杆在叶铺里落一声，落得稳。稳，是给全村看：铺子没关，谣没赢。韩泥隔门听见，独眼抬了一下，又落。落，像把「趁热」按进胃，按进心。坛角微凉，胃里却暖。暖不大，够撑半日编筐。他低声道：「叶丫头，这碗记下了。」不喊名，名惹祸；记恩，恩要还——还远，先记半笔。周嫂在巷口咳，咳像探。探不到门内，门内只有空碗和姜辛。辛在，谣就浅；谣浅，叶铺才开——开铺如开还路，路长，先记第一笔。韩泥是记心翁，恩仇都记在心里。心里第一格留给烫，烫字不写，按在胃上。胃暖，手稳；手稳，明日编筐卖十二文，十二文再买米——米能活，活才能还。空碗洗净，缺沿像月。月缺，恩不缺——缺的是还日，还日远，先记；记了，门缝才配再开。

碗沿姜辛还在，叶脚步已远；韩泥捧空碗：恩记半笔，还远不怕，怕的是汤凉——凉了，这门缝白开，明日秤杆还响，他还要听那声稳。
