# 第七十八章 嗅诀萌芽

嘲后第五夜，丑时。韩泥未睡实。睡实了，芽就藏不住；藏得住，才配开诀。他盘坐床沿，席严，坛不露。油灯豆，豆光稳，稳像筐底那撮深灰——沉在底，不浮。掌贴沿，沿温一线。温进掌纹，掌纹里二层气转，转一周，转两周，转第三周时，鼻下忽静到极。极静，不是无气，是气只走鼻。

走鼻，不走舌；舌响，香飘，白织月就笑得更真。他取白日样纸三包，摊开，灰里辛淡。辛淡不要紧，辛在，芽就在。第一包，鼻下一吸，辛沉。第二包，苦浮。第三包，腐底一线，底真。三包并息，息在鼻，掌纹微颤。颤不像惧，像认——认渣说话，认药性归位。忽然，颅骨里一声轻，非钟非雷，像瓮唇吐息。

药渣场角，他分拣，指稳，不需细想，鼻先落。落一次，指一次，稳一次。稳了，管事记「辨性初准」——初准，是小成前那一线。

秦霜木鱼一声：「诀萌别骄。骄了，香飘。」韩泥不答，答在心里——心里不骄，只记。记沉腐浮，记四层影仍在，记废渣路在前头。

吐息里，三字浮现，浮在鼻端，不入口：

嗅诀萌芽，颅骨里一声轻，像瓮唇吐息。吐息里三字：沉、腐、浮。三字不落舌，落鼻——落鼻，才萌芽；萌芽，才接得住81收集，83初识，85小成。

「沉腐浮。」

三字丑，丑而真。真在诀，不在册；册无诀，诀在鼻。韩泥屏息，独眼不眨。眨了，诀就散。散不得，散了，嗅诀白芽。他低声复诵，只复给自己听：「沉腐浮。沉者药，腐者毒，浮者假。」

复诵不在舌，在鼻。鼻里，辛苦腐三线并，并出一线清——清像渣场风硬那日，铁无言指的三堆灰。芽起了。起不像光，像鼻下多了一寸根。根细，细不要紧；细才藏得住，藏得住，煞骨门才嗅不准。邻床翻身，眯眼半息：「啥味儿……」

「坛味。」韩泥声平，「睡吧。」

小杂役哼一声，又睡实。睡实了，诀才藏得住。他收功，收得像盖席。盖严，沿净，样纸仍包，包实。实了，芽才不离鼻。更鼓将尽，天边白一线。白里，他廊下立半刻，任风硬，鼻下仍静。静里，沉腐浮三字不落舌，只落鼻。铁无言脚步远，停半息，声更低：「芽起了别叫。叫了，像求诀。」

韩泥只答：「守沿。沿净。」

赖福盯半息，终是没掀席。掀不得，沿净，坛不露。卯时，兽栏腥重。腥不要紧，鼻下有根，铲下更稳。稳，像踹台那一掌——掌在，诀在，路在。管事弟子巡栏，记一笔：「丙九韩泥，清细。」

刘婆端粥，仍多半勺：「昨夜又静？」

「静了。」韩泥说，「鼻下有根。」

刘婆不懂「根」，懂「静」：「静好。饭稳，恩不断。」

韩泥点头。恩在粥里，也在诀芽里——芽轻，记重；记重了，才接得住嗅药小成。

骄了，香飘。午后，药渣场。灰起，他分拣，不需细想，鼻先辨：辛左，腐右，假香上浮。浮起，指到，指稳，分堆。

分堆快半分，快不要紧，快也要稳——稳了，管事才不记「飘」。铁无言走近，瞥分堆：「对。芽起了，手才快。」

韩泥应声：「沉腐浮。记着。」

铁无言点头，走了。走，也像认可——认可不重，重的是渣场仍收他。白织月远廊过，袖风冷，没停。没停，小胜——胜在鼻，不在嘴。秦霜木鱼轻敲：「诀在鼻，心别贪。贪，邪。」

韩泥不答，答在心里——心里恩辱分列。诀是路，路通向废渣收集，通向温听雨初识，通向丹路开。他按席角，席严，坛不露。坛腹温一线，贴掌，像应「芽」字。应完，他低声对坛：「诀起了。三字记着。不叫。」

坛不应字，只温。温，像把「沉腐浮」摁进掌纹那路里——路在，就不滚。邻床问：「你分拣咋快了？」

「鼻下有根。」韩泥说，「根在，手才稳。」

少年不懂，闭嘴。闭嘴，像丙九也学会忍。丑时前，他再练半刻。三包样再辨，辨得更快，快不像炫，像活——活长了，丹路才开。掌纹那路转一周，二层稳，稳里芽根又细半分。细不要紧，细才接得住三层暗修。他独眼平，手不抖：「芽起了。嗅药在小成前。冬深续活，四层将显。」

韩泥只答：「不拜。沉腐浮，记着。」

更鼓沉，沉里，他摸样纸三包，纸糙，糙得像恩。恩轻，记重；记重了，才接得住95那盏门缝灯。坛腹应温，像应「萌芽」最后一笔。应完，他不叫。不叫，才守得住诀，守得住暗修。他低声道：「沉腐浮。芽在鼻。舌不留，手要稳。稳了，她断的丹路，叔自己来开。」

自己来开，在远，在89章；远不要紧，芽在近。近在鼻下，不在白织月笑里。赖福讽远：「末等也分拣快？快出邪。」

韩泥不接。舌要留来鼻，留来守坛。守坛，守诀，守「沉腐浮」三字不落舌。柱上条角白一线，冬深，风硬。风硬不要紧，鼻下有根——根在，就不滚。散队后，他过丹阁侧廊，廊扫痕直，直得像秤。他不踩灰，肩稳，渣在筐——筐在，就不污廊。同批少年悄问：「沉腐浮啥意思？」

「沉者药，腐者毒，浮者假。」韩泥说，「记住了，手才快。」

少年不懂「浮者假」，却懂「手才快」，闭嘴了。他按心口，七笔分列，不混。嗅诀芽在鼻，愧线71在怀，玉牌凉进骨——凉不要紧，芽暖在鼻；芽暖，路就不绝。更鼓二响，响沉。他合眼不睡实，鼻下仍静一息。静里，沉腐浮三字像根，根细，细才藏得住煞骨门那一嗅。芽起后第二日，药渣场加验五堆。

韩泥分拣，不需细想，鼻先落，五堆全对。全对，管事记「丙九韩泥，芽后仍准」。记准，是小胜。铁无言取未知灰一撮：「再闻。」

韩泥鼻下一息，指落：「沉三，腐一，浮一。」

铁无言分秤，对，点头：「芽起了，别叫诀名。」

韩泥应声：「不叫。沉腐浮，记着。」

温听雨远廊过，裙素，袖香温，她没近，只一眼。一眼，也像验手稳。赖福讽远：「末等也诀？诀出邪。」

韩泥不接。刘婆端粥，仍多半勺：「芽起了别飘。」

「不飘。」韩泥说，「飘了，手就滚。」

午后兽栏，铲下稳，稳里鼻下有根。根在，粪里药回才辨得出。管事记「清细」。记细，是小胜。黄昏，样纸三包再验，连中。连中稳，稳不像炫——炫像求，不求，只记。邻床问：「沉腐浮啥意思？」韩泥说，「三字记着，不在舌。」

丑时前，廊下练半刻。三包并息，并出一线清。清像嗅诀小成前那口息。他低声对坛：「芽起了。不叫。温，像把「沉腐浮」摁进掌纹那路里。更鼓沉，独眼平，手不抖：「嗅诀萌芽了。杂役续在后，温氏在远。」更鼓沉，沉里，韩泥按木牌，牌凉，掌温。记准，是小胜。」

「「沉腐浮，记着。」「静了。」饭稳，恩不断。」他低声对坛：「诀起了。三字记着。不叫。」廊下风硬，他把木牌系紧一线，系得像把仍末摁进掌心。同批少年远避，避像避晦气，晦气遮瓮，遮住了才稳。

编筐角藤刺仍在，他偏手，刺浅些，沿才净，坛才藏得住。

管事夜巡验棚，灯照席角，沿净，像腌菜坛，不像邪。

老耿挑水过，声哑，境别拜邪，拜了路断，他只记着。

秦霜木鱼远，一声，心别邪，邪了鼻下香飘，香飘手就滚。

柱上条角白一线，三年约在怀，愧字在怀，手仍不抖。

白织月远廊过，袖风冷，没停，胜在鼻，不在嘴。

铁无言脚步远，停半息，声低，别漏香，别叫境，只记。

散队后，他过坡下测灵石。石凉，线灰，灰在众目下公示过——公示不要紧，鼻里有根，手要紧。

更鼓沉，沉里，他摸样纸三包，纸糙，糙得像路。路轻，记重；记重了，才接得住冬深杂役续那一轮。

「不叫。」铁无言声低：「手稳就行。」韩泥应声：「稳。」稳了，废窑火路熟半分，试手还远。
