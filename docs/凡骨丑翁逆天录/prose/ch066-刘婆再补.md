# 第六十六章 刘婆再补

克扣第三日，麸皮又少半袋。

赖福秤杆落盘，落得轻，轻得像笑话。半袋少下去，丙九每人嘴边就少一口；少一口，粥更稀。灶边刘婆勺底仍沉，沉得像一辈子顶针顶布——她不认这半袋“风干”。

韩泥不辩。一辩，赖福就会扯连坐；连坐一来，沈家那套短签又会飘到丙九棚。他按了按怀中木牌，牌上旧字仍深：「仍末。」这两个字他熟，熟不等于认命——饭还能进肚，路就还在。

管事弟子巡灶，鼻下一嗅，皱眉：「麸皮按册发。少一粒，记赖福名。」

赖福脸一白，慌忙把秤杆抬半寸。抬迟了，笔已经落簿。笔迹丑，字却真：克半袋，丙九补发。

同批少年低声：「管事真管了？」

「管了。」韩泥端碗，声平，「鼻下静，才闻得出少。」

赖福眼细，细得像牙：「丑杂役，你告的？」

「我没告。」韩泥说，「你的秤会说话。」

刘婆眼一瞪：「再嗓，粥也别喝。粥稀，不是他克的？克人克到灶上，像扒皮！」

赖福唇动，终是没再尖——尖不过「灶」字。灶在，刘婆在，丙九才有一口稀粥。

有人小声接：「赖福也有怕的？」

刘婆嗓更高：「怕什么？灶是活路。活路断了，全丙九饿。」

韩泥坐下喝粥。肘袄漏风一线，风凉进骨缝。刘婆瞥见针脚：「肘又漏？」

「漏一线。」韩泥道，「清栏挂袋，挂裂的。」

「挂袋也挂，风也进。」刘婆从袖底摸出针线包，顶针发亮，「坐下。站着，风仍钻。」

他坐凳角。凳歪，歪得像末席——末席也是席。刘婆穿针，手稳：「老婆子补，不收钱。钱你们绝户拿不出。拿恩，恩太重，我扛不动。记着就行。」

「肘破，风进。」韩泥只接事实。

「进就补。」刘婆咬线，线紧，「补完别谢。谢轻。」

针脚走得糙，却牢。牢了才挡风；风不进，臂才抬得起粪铲。她不问他肩背那道旧疤哪来的，只问：「抬粪还抖不？」

「不抖。」韩泥说，「抖了，铲偏。」

「偏了，管事记。」刘婆哼，「滚字在柱上，不在肘上。肘不漏，手才还在丙九。」

针走一圈，像给肘上旧裂重新钉扣。刘婆低声：「顶针顶一辈子，顶的是活，不是脸。脸丑，肘不能漏风。」

「不漏。」韩泥说，「漏了，清不了栏。」

「知道就好。」她咬线，「别再挂粪栏。挂了，我又补。」

门外赖福路过，笑得薄：「刘婆，你也沾晦气？沾了，粥更稀！」

刘婆头也不抬：「粥稀是你克的。他要真能克你，你早倒了。」

赖福脸一青，脚步硬生生走开。走时还回一句：「补衣补到杂役身上，你也晦气！」

「晦气也比克扣干净。」刘婆咬线，「再嗓，你连稀粥都别想近灶。」

午后，麸皮补发到。粥厚半分，厚半分像窄路又宽半寸。不是灵根变好，是丙九这口饭没被半袋骗走。碗糙，糙里热意真——真到掌心都暖一分。

管事当众验袋：「丙九补发齐。赖福，半月无赏。」

场子静一息。有人拿眼看赖福，看得他晃秤晃得不甘：「补发一次，下次还克。」

韩泥不接话。话说多了，明日又饿。铁无言路过灶角，声低：「鼻下静，手才稳。稳了，渣场仍收你。」

「稳在鼻。」韩泥应。

管事又记一笔：「丙九韩泥，克扣日仍清栏细。」细字轻，像活路多钉了一钉。同批少年碰他碗：「这半袋弹回去了？」

「弹在秤上。」韩泥说，「不在嘴上。」

补完肘，他立刻清栏。新针脚挂袋角，又扯开半寸，风又钻。刘婆灶边瞧见，顶针一亮：「回来。再补半针。」

韩泥回灶，不辩。刘婆再穿针，针脚更糙，更牢：「挂袋也挂，风也进。进风臂僵，僵了坡下站不住。」

「站得住。」韩泥说，「站不住，叶家那碗汤还不了烫。」

刘婆哼一声，没再嗓。嗓要留来舀粥。粥再稀，她仍多舀半勺到他碗里：「喝。正测前，别空肚。」

「姜汤呢？」小杂役凑过来问。

「姜在记里。」刘婆瞪他一眼，「记里热，手就不飘。」

小杂役吐舌退开。韩泥喝完，把碗扣稳：「谢轻，恩我记。」

「记就记。」刘婆收针，「别当着赖福的面记。当面记，他下次更克。」

翌日卯时，韩泥仍清栏第一趟。赖福堵栏口：「补发了，就能偷懒？」

铲一落，粪均匀下去。「偷懒，管事记。」韩泥说，「记了，滚的是我，不是你克的那半袋。」

赖福噎半息，走了。管事路过，瞥一眼：「手稳。」

二字很轻，却够丙九听见。管事又当众念补发令：「麸皮齐。再克，连坐。」念完，赖福笑不出来，韩泥也不近前谢——谢轻，恩在肘上新针脚里，在粥热半勺里。

午后又过渣场半刻。灰堆里辛淡，淡得像提醒他：饭实一日，鼻也不能松。铁无言远远看一眼，没近——没近也好，近了像护短，护短容易惹赖福报复灶边。

夜，他擦坛沿。沿温一线，像应「刘婆」二字，应完又敛。邻床问：「刘婆为何补你？」

「风进肘。」韩泥说，「补风，不补脸。」

少年懂「脸」，闭嘴。刘婆那边又咳一声，咳低：「正测前，姜在记里。」

「在。」韩泥应，「在，才还得烫。」

更鼓沉，廊外赖福脚步远，远里仍扔一句：「丑杂役，我盯着你。」

「在。」韩泥只答一个字。

在，就不滚。不滚，肘上那线针脚才不算白补。他抚新针脚，针脚暖进骨——暖不是虚礼，是第二天还能抬铲。抬得起，坡下才站得住；站得住，第七笔才不凉。

坛腹微温贴掌，像顶针又顶了一下。路还长，长得够还，够挨过克扣，够等到冬后那一验。

他低声道：「克扣弹回去半袋。半袋不大，肩却轻一点。」

翌日秤杆落盘，落实了。赖福没再落轻。没落轻，也是小胜。胜在饭，不在嘴。韩泥清栏，铲下仍稳，肘上针脚不再挂袋角——风不进，臂就不僵。

老耿挑水路过，水洒稳，声哑：「肘补了，别拜邪。拜，路断。」

「不拜。」韩泥说，「恩在肘，不在舌。」

刘婆收针，顶针在灶沿敲一下，轻得像盖印：盖在布上，不盖在脸面上。她又叮一句：「三日后坡下见。见在末排也行，手别抖。」

「手不抖。」韩泥端碗，「粥热半分，就够记了。」

有人隔灶问：「刘婆，你偏他？」

「偏活路。」刘婆瞪回去，「活路在肘上，也在勺里。勺没了，你们喝风？」

那人缩脖子。韩泥不插话。插话像把恩推到明处，明处一亮，赖福夜里就可能再动手脚。

夜里棚里又静。他把木牌按在胸口，牌凉，肘暖。凉暖并着，像两本账：一本写仍末，一本写刘婆。两本都不撕。撕了，汤就凉，粥也稀。

灶边有人又递话过来：「韩泥，你肘补了，敢不敢跟管事讨赏？」

「不敢。」韩泥摇头，「讨赏像邀功。邀功，赖福夜里就给你拆针。」

「那你图啥？」

「图粥热半分。」他说，「半分够扛明日。」

那人啧一声走了。刘婆听见，只哼：「图半分的人，比图赏的人靠得住。」

韩泥应了一声，把碗洗了。水凉，手不凉——肘上针脚还留着指温。他把这点温按进心里，当第七笔盖印：盖在布上，也盖在还路前头。

三日后卯时，坡下见——见在末排，不在今日嗓子上。手在，就不滚；恩在粥里，也在肘上那道丑针脚里。针脚丑，挡风；风不进，臂才抬得起粪，才站得住冬测前那些硬日子。日子硬不要紧，肘暖半分，路就还直着。直着，才接得住冬测那口冷风。
