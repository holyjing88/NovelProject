# 第九十五章 匿赠培元丹

雨夜。

药山废窑，窑口红，红退，退到掌纹能控。控火第三回，韩泥不再试——这一回，炼。

青芯五撮，褐渣两把，灵泉七滴，投序照经页。页黄，字丑，丑不要紧；丑，才像他的路。

炉温起，起一寸，一寸稳。稳里，渣白气聚，聚而不散。散不得，散则败；败不要紧，败过两回，第三回要成。

鼻下嗅诀动，动如秤——秤香，秤色，秤那一点「正」。正来慢，慢不要紧；慢，才匿得住。

丑时中，丹形初聚，聚如豆，豆小，小不要紧。小，才塞得进门缝囊。

成丹一瞬，窑内无雷，无光，只有掌纹那路烫一线。烫，像叶汤碗沿；沿热，热进心记。

他取丹，丹温，温而不灼。灼了，像邪；邪了，路断。路不断，丹入温汤囊，囊缝，缝丑，缝暖。

不下山。下山，赖扒皮嗅。他在窑内等到雨歇，雨歇，星晦，星晦才好走。

坡下小舍，门缝仍宽一线。一线光里，叶青禾未睡，眼望缝，像在等，又不敢等。

韩泥不现形。现形，像连累。他把囊系细线，线过缝，轻轻一拽，囊入，油豆在门外爆一声，爆完即灭——灭，像无人来过。

缝内，叶青禾触囊，触到温。温，她识得，识得像当年姜汤藏碗底那一点热。

她低声：「……韩叔？」

缝外无声。无声，才是还恩。

她开囊，囊里一粒丹，丹旁一张粗纸，纸上三字：「趁热。」

就三字，无署名，无「韩」。无韩，才匿；匿，才赠得出。

她指节白，半晌，丹入口。入口温，温进脉，脉里一线培元，弱而真。真，她不敢声张——声张，连累丙九。

沈枯芽在远舍，怯，未眠。未眠不要紧，他不知丹，只知「别灭灯」已传。

韩泥回兽栏，雨泥没踝，踝冷，心热。热不应喜，应「还」——还一半，还暗处；暗处还，才配叶丫头。

赖福查夜，火折照韩泥袖：「什么味儿？」

「药渣。」韩泥声平，「栏腥，洗不净。」

赖福鼻动，动半息，没再追。没追不要紧，丹已赠，赠在缝，赠在温。

席下，他按坛，低声道：「还了。不叫。」

坛腹温，贴掌，像应「趁热」。趁热，是叶汤的回响；回响在，恩就不凉。

刘婆不知。刘婆只端粥，粥稀：「雨夜别乱跑。」

「在栏。」韩泥说，「手稳。」

邻床睡实，睡实了，秘密才藏得住。藏得住，才接得住坊市风声——风声在远，在下一程。

他独眼平，摸怀经页，页上「培元」深了一线。深不要紧，深是起；起在暗处，起在还恩。

更鼓尽，坡下门缝灯灭。灭，是她睡了；睡了，丹在脉里温着。温着，韩泥胸空半寸，空得舒服——舒服不是无债，是债还了一笔，还能还下一笔。

他低声道：「叶汤，还了一半。剩下一半，在省亲前。」

坛不应字，只温。温，像爹搭沿，像门缝那一线黄。

雨夜里，废窑深，深像藏得住秘密。

经页「培元」二字浅，浅而真。真，他默念，念不进舌，念进鼻。

成丹一瞬，掌纹烫一线。

坡下门缝，叶青禾触囊，触到温。温，她识得，识得像姜汤藏碗底那一点热。

窑内雨声如鼓，鼓在顶，不在心。心要静，静里掌按炉壁，壁凉，凉退，退得像坛腹敛温。韩泥不现形。现形，像连累。他把囊系细线，线过缝，轻轻一拽，囊入，油豆在门外爆一声，爆完即灭。

赖福巡外围，火折照栏：「丑杂役，别乱窜！」韩泥立暗处，不应。不应，像石。石在，路在。窜了，连累叶家线；不窜，恩在暗处还。

他数七笔，笔分列，不混。叶汤这一笔，还了一半。刘婆端粥，仍多半勺：「又上山？」「倒渣。」韩泥说，「手稳，饭稳。」刘婆哼，没再问——问，像沾晦气。

更鼓沉，韩泥按木牌，牌凉贴胸。不叫。下一程，坊市风声。

暗炼培元丹，雨夜为幕。韩泥入废窑，窑深，深不要紧，深才无眼。无眼，掌炉才正。正不在亮，在匀——匀一寸，不多不少，像脉里培元温着的那一线。

投序照页，页浅，浅而真。青芯四撮，褐渣一把，灵泉六滴，滴齐，火起。起时，坡下门缝仍宽一线，一线像等。等不要紧，等的是囊，不是人。

成丹一瞬，窑内无雷，无光，只有掌纹烫。烫，像叶汤碗沿，像门缝灯，像93备料之静。静里，他取丹，丹小，丹温，温而不灼。灼了，叶丫头受不住。

匿赠用细线系囊，线过缝，轻轻一拽，囊入。油豆门外爆一声，爆完即灭——灭，像93无字投序。无字，才匿；匿，才赠得出。

叶青禾触囊，触到温，温她识得。开囊，一粒丹，粗纸三字：「趁热。」就三字，无韩，无署名。无署名，她才敢服；敢服，脉才稳半分。

韩泥数七笔，叶汤还一半。省亲远，远不要紧，远，手先还。还了，不叫，坛沿净。下一程，坊市风声——风声里，仇线也在，仇线在钱戾衡名上，名在远，手在摊。

暗炼培元丹，雨夜为幕，窑深无眼。无眼，掌炉才正——正不在亮，在匀。匀一寸，成丹小，丹温，温而不灼。灼了，叶丫头受不住；受得住，脉才稳半分。

细线系囊，线过缝，一拽，囊入。油豆门外爆，爆完即灭——灭像无字投序，灭像94留灯。灯留了，丹才赠；赠不示名，名示则赖扒皮来。来不要紧，匿才不连。

叶青禾触囊，触到温，开囊，见丹，见「趁热」三字。就三字，无韩，无署名。无署名，她才敢服；敢服，95还一半。剩下一半，在省亲前——省亲远，远不要紧，手先还。

韩泥数七笔，不混。混，路断。下一程坊市风声——风声里，仇线也在，仇在玉牌，约在怀，烫在还路前头。前头有兽潮，有挂牌，有130远。远不要紧，95只还一半。

坡下门缝仍宽一线，一线像等囊，不等人人。人等，像连累；连累不要紧，匿才不连。囊过缝，叶青禾触温，温她识得——识得像姜汤藏碗底那一点热。热在记，不在炉；炉只要匀，匀一寸，成丹小，丹温。温而不灼，灼了受不住；受得住，脉稳半分，95还一半。不叫，像93无字投序；不叫，像94留灯不现人。

不现人，恩在暗处；暗处还，才配「别来丑」——丑在脸，不在恩。恩在七笔里，笔分列，不混。混，路断。下一程坊市风声，风声里仇线也在，仇在玉牌，约在怀。怀贴身，像守坛沿。手在，摊在，路在。

坊市方向，尘起，尘是96。96在远，95只还一半——一半在叶脉，一半在省亲。省亲远，远不要紧，手先还。不叫，像93无字，像94留灯。留灯，赠丹，分列七笔，不混。

暗炼培元丹，雨夜为幕，幕深，窑深，深不要紧，深才无眼。匀一寸，成丹小，丹温，温而不灼；灼了，叶丫头受不住。受得住，脉稳，95还一半；剩下一半，在省亲前。

匿赠用细线系囊，线过缝，一拽，囊入。油豆门外爆，爆完即灭——灭像94留灯，灭像93无字。叶青禾触温，开囊，见丹，见「趁热」三字。就三字，无韩，无署名；无署名，她才敢服。服，不叫；不叫，坛沿净。不叫，赖扒皮嗅不到；嗅不到，丙九不连坐。连坐不要紧，匿才不连。

下一程坊市风声——风声里，仇在玉牌，约在怀，兽潮在条上。条在远，95只还一半。一半，够了；够了，手在，路就不绝。

秦霜木鱼远敲，声低：「暗赠别骄。」韩泥不答，答在心里——心里不骄，只还。下一程坊市，风声里兽潮也在，兽潮在条，条在远，手在，就不滚。叶汤还一半，剩下一半，在省亲前。

散队后，他过外堂阶下，分拣半刻，指稳——指稳，路就不飘。
