# 第一百零七章 钩108

静半日。

兽散，散像退潮，潮退，界外仍腥。腥不要紧，腥是钩——钩在余毒，余毒在栏，栏在，角才不能散。

韩泥被管事点名：「丙九韩泥，随铁无言清余瘴。」

点名，像钩；钩108，落在清。

清，不是扫，是嗅。嗅余毒，毒在角草，草黑，黑里甜。甜，他熟，熟于99风声，熟于101备。

铁无言臂仍绷，绷粗，声低：「清完，我请你吃一袋麸皮——不，请不动。手在就行。」

韩泥应声：「在。」

在，就是请。请，不在嘴，在并肩。

清角时，他鼻下动，动出三处漏——漏在障根，根补，补不要紧；补完，仍剩一处，一处深，深像毒窝。

窝，要丸，丸要炼。炼在108，不在107；107只钩——钩出深窝，钩出经页「清瘴」二字将实。

实，他摸怀经页，页黄，字深一线。深不要紧，深是序；序齐，清瘴丸才肯落怀。

赖福来瞧，瞧像不信：「还能嗅？」

「嗅。」韩泥说，「嗅完，手稳。」

赖福没再尖，尖不要紧；尖过了，像也怕余毒。

白织月立远，声低：「余毒进内墙，谁都跑不了。」

韩泥只答：「不跑。」

不跑，是钩的另一头——头在铁无言，尾在清瘴丸。

叶青禾在坡下，门缝灯又亮。亮，他不在近；在近，像连累。连累不要紧，灯在，脉里培元仍温——温，像恩还着。

沈枯芽悄递：「叶姐说，别逞能。」

韩泥点头。逞能不在路；路在清，清在108。

夜，管事弟子贴条：「余瘴清尽前，丙九不准撤。」

不准撤，就是钩。钩住，韩泥东角又立，立像续后的终前。

终前，他按坛，低声道：「钩在了。下一程，炼清瘴丸。」

坛腹温，贴掌，像应「钩」字。钩，不是戏，是序。

丑时，静半刻。香落齐，掌纹烫——烫像五层将满，满而不破，破在下一程。

邻床问：「真要再炼？」

「炼。」韩泥说，「炼了，栏才活。」

更鼓尽，深窝仍甜，甜像等丸。等不要紧，等的是108。

他独眼平，低声道：「钩108。手在，铁无言在。」

在，就不滚。不滚，清瘴路才接得上。

兽散，界外仍腥。腥是钩——钩在余毒。

管事点名：「丙九韩泥，随铁无言清余瘴。」

清角时，鼻下动，动出三处漏。仍剩一处深窝，深像毒窝。

炼在108，不在107。

叶青禾门缝灯又亮。亮，他不在近。

管事贴条：「余瘴清尽前，丙九不准撤。」

深窝甜，甜像等丸。108一事，清瘴；清瘴一事，并肩。并肩一事，铁无言。铁无言说「手在就行」，就行，是契的芽。

条贴「不准撤」，撤不得，撤了，余毒进内墙。进内墙，白织月也怕；怕不要紧，角仍要人。人要，韩泥在；在，就不滚。不滚，108才接得上。

静半日，界外仍腥。腥不要紧，腥是钩——钩在余毒。

秦霜木鱼轻敲：「钩在，心别慌。慌，手飘。」

韩泥不答，答在心里——心里不慌，只记。记深窝，记清瘴，记并肩在前头。

序齐，清瘴丸才肯落怀。107只钩——钩出深窝，钩出清瘴二字将实。
