# 第二十五章 余波

公审过了第二天，泥岗村像退了一潮。滩上露出旧泥，泥里还嵌着昨天嚼剩的字：晦气、牵扯、绝户。字还在，人却嚼不动了——嘴酸了。

韩泥天一亮还是编筐。藤条浸软绕两圈，臂布勒得紧，狼口那道旧伤一勒就胀。半只筐编完，担到集尾，卖十二文。买主掂了掂，还是付了。他当这是夸手艺，不当骂。回村又编半只，指节闷响，心里把叶汤那笔恩按实——按得像给叶家垫了一块石。

赵绣春家门闭得紧。路人路过也不再围了。围了半辈子戏台，戏台塌了一角，塌在叶青禾那句「谁没受过一碗饭」上。饭字轻，却压得人哑。周嫂洗衣，棒槌轻了半分，像怕再砸出「汤是邪」来。水仍浑，她骂自己小声：「砸多了，手也疼。」

叶家药铺开门。秤声稳，药气苦，苦里却有活路。叶青禾在柜后不抬头，也不辩，只称药。称得准，比辩十句硬。

韩泥担空筐回村，村人让路，让得别扭——让半步，也算松一口气。马春花啐得老远，像啐给自己听：「疯子……说得倒也在理。」在理二字她不肯吞全，吞一半够她今日不堵叶铺门。

午后，沈枯芽端一碗药汤进叶铺。汤能进门，公审才算真落了地。张麻婆骂街骂到一半改口：「叶丫头硬，硬得像我这馒。馒热，心热，热不过诬人嘴。」

陈姑路过，棉袄边还新，新得像冬恩还在。她不看韩泥的脸，看筐：「筐编得紧，人便紧。紧了，谣追不上。」

韩泥不谢。谢轻了，恩显得轻。他站在巷尾望叶铺灯，白日也亮着，亮得像全村绷了十几日的绳松了一寸。绳松了人不松；人一松，恩就散。

他低声道：「气松了，恩没松。」

心口叶汤深，他不增笔，只按。按实了，像按住叶家脊梁还在。脊梁在，将来省亲那日才有路。坛角微凉，坛腹微温。温里，他听见远处赵家门内一声摔碗——摔得脆，像认输，也像不服。

里正巡巷，咳一声：「少嚼。嚼多了，连坐。」

他又补一句，声沉得像钉：「公审过了，谁再聚众辱人，先记名，再报县。县吏来，不比今日温和。」

村人噤了，像退潮。韩泥回屋编筐，指紧——紧着，给明日留活路。

傍晚里正又巡到叶铺，咳道：「公审过了，秤别停。停秤，比哑口还毒。」

叶青禾在帘内应：「不停。秤在，人在。」

里正向韩泥屋扬声，却不近：「韩泥，少生事。」

韩泥开门，独眼平：「我生事？我编筐。」

里正瞥他一眼，愣半息：「心收好。收不好，也是祸。」

「恩在心里，祸在嘴。」韩泥说，「嘴我管不了，心我管得住。」

里正无话，走了。走也是松气——松给全村看：公审不是结局，是缝；缝上了，日子还得过。

韩泥回屋，听叶铺最后一声秤落，落得像钉，钉住「叶家还在」。他低声道：「松气是松气，叶汤不能浅。浅了，日后便无根。」

赵绣春门缝探头，像不甘：「松什么气？天还旱！」

张麻婆啐她：「旱是天，汤是人。人还在，你少泼。」

赵缩回去，又哑半寸。半寸够韩泥记。叶铺帘内药杵又响一声，像秤还在量活路。

沈枯芽又送一碗粥到叶铺。粥稀，热，热得像公审后第一碗「正常」。韩泥路过不近，只望秤杆。杆稳，心便安一寸。

安一寸，够编筐，够活。他低声道：「公审过了，谣没死，嘴还在。嘴在，心别软。」

巷口有人扯周嫂袖：「真不关铺了？开着，赵家再闹怎么办？」

周嫂棒槌顿在石上，声闷：「闹了公审审过了。审过了还关，谁给娃抓药？抓药的地方在，日子才算过。」

扯袖的人缩手，仍低声：「我就怕牵扯……」

「怕牵扯就少嚼。」周嫂说，「嚼多了，连坐连到你自家门。」

人散了。韩泥担筐从旁过，没停。停了像凑热闹；热闹他不要，他只要那口秤声还在。

村北水井边，马春花洗盆，盆沿磕石，磕得脆。她见韩泥，鼻里出气：「公审赢了，你就能编一辈子筐？」

韩泥独眼平：「能编就编。编不了，才饿。」

「嘴硬。」

「嘴硬比嘴脏好。」他说完走了。走两步，听见她骂「老疯」，骂声轻了半分——轻半分，也是余波。

入夜，叶青禾隔帘问，声平得带倦：「公审过了，你筐还编？」

韩泥在门外，不近帘：「编。不编，集尾空，空像认输。」

「认输给赵绣春？」

「认输给谣。」韩泥说，「谣没死，手不能停。手停，秤也停；秤停，你爹药白送。」

帘静半息。秤杆轻碰一声，像盖章。叶青禾又问：「沈枯芽那帖药，你用了？」

「用了。臂能抬筐。」

「能抬，别逞。」她顿一息，「外头的路远，远里也有嘴。嘴比泥岗尖，尖了你别硬顶。」

韩泥应：「不顶。顶是活，不是辩。辩赢了公审，活路还在后头。」

里正巡到巷口，咳：「叶丫头，帘内少议外头。议多了，像牵扯。」

叶青禾道：「里正叔，我只说药。药不断，人不断；人不断，村不断。」

里正转向韩泥：「你，少往叶铺门缝凑。凑了，赵绣春又有话说。」

韩泥独眼平：「我凑集尾，不凑门缝。门缝是她的，筐是我的。」

里正「哼」一声走了。走，也是给全村看——公审过了，规矩还在；规矩在，谣便不能再借规矩咬一轮。

里正差人送抄本贴柱角：「沉丹宗验骨，十日后卯时，西驿见。」韩泥瞥一眼：「十日，够把该记的记清，该还的路上还。」

张麻婆改口改得短，真：「叶丫头硬，硬得像馒。」

丑时，他在灯下数藤条，数到第十根停住——像数十日。十日不远，帖却在远。远不要紧，尺在柱上，手就不飘。末席站着，也是席。

沈枯芽隔巷探头又缩，像怕牵扯。韩泥隔巷应，声哑：「公审过了，你娘还咳，药还得抓。抓药找叶家，别找别处。」芽儿点头，跑远。

叶铺秤声又起一声——声在，铺子没关，谣没赢透。盐担老汉蹲阶下，嗓粗却低：「公审那日叶丫头说得在理。理在秤，秤在，我娘那两帖药才不算白送。」

韩泥听见了，不点头。点头像拉帮。他只把叶汤按深——深了，等还。

赵绣春门内又摔碗，摔得脆里带虚——虚在药铺门仍开。门开就还有病人，还有竖着的理。韩泥不近赵门，也不近叶帘。近是害，远是护。护到柱上那十日，护到帖来那日。汤仍记烫，烫是余波里唯一要守的字。

丑时过后，巷里犬吠两声又止。韩泥起身，摸黑点灯，灯豆黄，黄里他把臂布重新勒过。勒紧，疼一下，疼一下像提醒：公审过了，人还在编筐。编筐的人，才配把恩记到日后去还。

灯下他默念叶青禾没跪下的背影，念张麻婆那句改口，念村人让出的半步。半步不大，够暖一夜。

天亮前有人敲门。门缝外是盐担老汉，递半把腌菜：「叶丫头理直，你也别软。软了，墙里头的人又起劲。」

韩泥接了，只道：「记着。恩不软。」

老汉点头走了。走得轻，像怕吵醒谣。

夜里细雨打檐。韩泥合眼，手按坛沿。沿在，根就不散。公审的锤子落过了，村里却还悬着一口闷气——气散了才松，松了才看得见往后的路。柱上「十日」两字在风里不动，动的是他心里那点热：热着，才不白过这一夜。
