# 第六十八章 公开测灵

三日后卯时。

坡下霜白，白里柱上条已干。条干，字更显——「凡骨杂役，公开测灵。末席公示，滚者除名。」

韩泥读条，独眼平。正测不是备检，是公示——公示给内门看，公示给全宗看，公示谁滚谁留。滚字在条上，不在心上；心在坛，坛醒了，滚也要手稳才滚。

丙九列队，队末是韩泥。赖福立队首，眼细，像秤——秤人，秤灰线，秤谁多喘一口气。管事弟子喝：「按序！手抖的，先滚！」

测灵石抬到坡下中央，石青，青得钝。钝，不像仙，像秤——秤灵根，秤命，秤你配不配留。内门弟子立廊下，袖风冷，冷里有人笑：「凡骨也配公开测？」

「配不配，看手。」韩泥声平，「手抖才滚。」

同批少年抖，抖半息，管事喝：「抖！先滚！」少年立刻按住腕，腕稳，才排到石前。

第二个按石，绿线一寸半，光稳，稳里手也稳。管事记：「过。下一。」第三个按石，绿线两寸半，光一闪，闪里肩沉半分——沉不要紧，沉了，光才正。韩泥看，不看光，看手——记法，不记色。

第四个按石，绿线两寸，抖了半寸。管事冷声：「抖！滚！」少年哭腔，被架走。丙九静一息，静里，有人看韩泥——看，也像怕末席传染晦气。

赖福阴笑：「丑杂役最后。最后，专看滚。」

管事冷声：「按序。韩泥，末。」

韩泥上前，掌茧裂处疼，疼不露。掌按石，石凉，凉进骨。光起，灰线，线短——短不过指节，却公示在众目下，像把「末」字钉进全坡。

有人嗤：「灰线三尺，也配留？」

嗤声薄，薄也真。白织月立廊角，袖风带香，香冷：「凡骨末等，丹路断。断了的，站石前，像笑话。」

韩泥不辩。辩，费舌；舌要留来按石，留来守坛。管事瞥他掌：「手抖没？」

「没抖。」韩泥说。

管事又瞥一眼灰线，灰线三尺，真在公示——公示末席，公示灵根，公示全宗都看见的「仍末」。他记簿，声沉：「韩泥，凡骨末等。手稳，石前不抖——仍末排，公示。」

执事念榜，念声沉：「公示末席，韩泥。灵根凡骨末等，灰线三尺。手稳者，冬后仍杂役；手抖者，除名。」念完，丙九有人松一口气，有人看韩泥——看，也像看笑话。

仍末排，公示。公示，比备检更辱——辱在众目，辱在内门的眼，辱在赖福的笑。

赖福笑出声：「公示也末等！丙九的脸，让你丢尽了！」

韩泥收掌，不辩。舌要留来活，留来忍。忍不是怂，是刃——刃不亮，先记。记仇苗，苗不深，深在将来。

廊下有人踏前一步，靴底声沉，沉里带金线边——内门弟子，衣鲜，眉锐。他瞥韩泥灰线，瞥完，唇角一挑：「钱戾衡。」

名过，风冷。冷不是冬风，是眼风——眼风落在灰线上，像踩。踩不要紧，踩实了，才记得住。

钱戾衡声不高，高在众耳都听见：「灰线三尺，也配站这坡？配站，就配滚。滚了，省得污内门眼。」

内门有人哄笑，笑薄，薄也真。真在众目，真在灰线三尺公示那一瞬——瞬短，辱长；辱长，才扎成仇苗。

韩泥指节白一息，白完，又平。钱戾衡——名在仇线里，名在今日，不在舌上。舌响，香飘；香飘，手就滚。

他独眼平，只答：「管事看手，不看嘴。」

钱戾衡笑一声，笑薄：「嘴硬。正测后，看你还硬不硬。」

笑里藏着踹字——踹在约里，不在今日。今日，他只记名，不近，不拜。拜邪，路断；不拜，各守各线。

铁无言抱臂立侧，声低：「灵根末，手稳，渣场仍收你。别漏。」

韩泥应声：「不漏。」

秦霜远廊经过，木鱼不敲：「心别邪。邪了，鼻下香飘。」

韩泥不答，答在心里——心里七笔分列，不混。泥岗六笔沉，宗门刘婆一笔烫——烫在肘，烫在粥，烫在今日这口辱。辱在众目，手不能抖；抖了，滚字就真。

管事收石，喝：「正测毕。公示末席：韩泥。滚者除名者，已滚。仍末排者，冬后仍杂役，手抖再滚。」

公示榜前，有人念末行字，念完，嗤一声。嗤不要紧，嗤在舌，不在手；手在，名在，路才不断。韩泥立末，不近，不躲——躲，像怕；怕，手就抖。

丙九静一息。一息里，有人看公示榜——榜上新添一行，行丑，真在「韩泥·仍末·手稳」。手稳，是小胜。胜在活，不在嘴。

邻床少年悄问：「你真不滚？」

「滚看手。」韩泥说，「手在，就不滚。」

少年闭嘴，像也学会忍。忍不是怂，是活路——活路通向还恩，通向省亲，通向钱戾衡那一句「记住今日」的反面。

赖福晃木牌，牌旧：「公示了，更滚得快。」

韩泥不接茬。接茬，全丙九饿。他摸怀木牌，牌凉，膝热——热在坛，热在路，热在钱戾衡那一句眼风。

钱戾衡已走远，走远还留一句：「丑杂役，记住今日。」

韩泥低声道：「记着。」

记着——仇苗又深半寸。深半寸，不拔，先活。活长了，才还得了叶丫头那碗烫，才接得住将来那一踹。

他回棚，不近廊下笑。笑在耳，不在心——心在坛，坛腹温一线，温不应辱，应「忍」。忍字丑，丑翁专精；专精了，才配记钱戾衡这一眼风。

午后，兽栏仍腥。腥不要紧，手稳才要紧。他铲下稳，稳得像坡下按石那一掌——掌在，名在，路在。管事路过，记一笔：「丙九韩泥，正测后仍清细。」

刘婆端粥，仍多半勺：「公示了？」

「公示了。」韩泥说，「仍末排。」

「末排怕什么。」刘婆哼，「手稳，饭稳。饭稳，恩不断。」

韩泥点头。谢轻，恩重。恩在粥里，也在忍里——忍里藏着刃，刃对着钱戾衡，对着赖福，对着内门的眼。刃不亮，先磨；磨在丙九，亮在还恩那日。

散队后，他过坡下石阶，阶滑，霜白。阶上有人吐唾，唾落阶边，不落他靴——落靴，像沾晦气；不沾，也像内门仍不肯认他。他不辩，只按木牌——牌在，名在，路在。

老耿挑水路过，水洒稳，声哑：「公示不要紧。末，不滚；滚，看手。」

韩泥只答：「手在。」

更鼓将起，柱上条角又白一线。白，像仇苗出土，根扎坡下，拔不出。拔不出不要紧，先记——记钱戾衡，记「记住今日」，记灰线三尺公示在众目下的那一瞬。

他按席角，席严，坛不露。坛腹温一线，贴掌，像爹的手又搭上来。爹说过：「坛别扔。」扔不得，扔了就无根；无根，辱就白挨。

廊下仍有内门弟子练剑，剑光两闪，闪完，一句笑远：「灰线三尺，也配留？」

韩泥不答。答，费舌；舌要留来记仇苗。记仇苗，不在册，在心——心在坛，坛醒了，刃才磨得亮。

他低声对坛：「公示了，仍末。仍末，也不滚。滚要手抖，我不抖。」

坛不应字，只温。赖福脚步在廊外停半息：「丑杂役，我盯着你。」

韩泥只答：「在。」

在，就不滚。不滚，仇苗才扎得实——扎实了，才接得住踹台前夜，接得住玉牌，接得住三年约那一踹。

他廊下练半刻鼻下静，粟壳香淡，淡不要紧；要紧是香一字不落。落齐了，钱戾衡那一句「记住今日」，才记进心里，不进舌。

他独眼平，手不抖：「钱戾衡，记着。记着，刃才磨得亮。」

刃不亮，先记。记，是丑翁的本事。记完了，才配等踹台前夜——夜在远，仇苗已在心；心在，就不滚。手在，帖在，名在册；册在，才接得住玉牌那一夜。玉牌在远，仇苗在近。近，才磨得亮。亮在还恩那日，不在舌上。踹台在前头。

帖在怀，怀沉，沉的是序，不是惧。
