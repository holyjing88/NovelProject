# 第一百零九章 炉温稳

雨后。

兽潮散，药山界解夜禁，禁解，韩泥回废窑——回，不为观，为稳炉。

稳炉，是109一事。一事，不在名，在温——温一寸，不多不少，像呼吸。

窑口红，红退，退到掌纹能控。控，他不再急，急过，急在104至；至后，要稳。

铁无言送渣一包，包灰，声低：「清瘴剩的料，别浪费。浪费，像逃。」

韩泥应声：「不逃。」

不逃，就炼余韵——余韵不是丹，是手。手熟，路才正。

第一炉，培元路，无丹，只有灰里一点白。白像未成，未成不要紧；未成，是余韵。

第二炉，清瘴路，丸不成，成灰香。灰香正，正不要紧；正，是109的稳。

稳，他记炉壁烫半寸，烫像坛腹应温。应温，不应喜；喜了，手飘。

赖福克扣复原半分，半分不要紧；半分，像兽潮后风变——变里，少盯梢半息。

少盯，韩泥不谢。谢轻，记在心里：盯少了，窑可多待半刻。

半刻，够稳。稳里，他鼻下嗅，嗅出投序将变——变在110，变向培元散。

培元散，是散，不是丹。丹在95赠叶；散在110，在自用，在丹道起点。

起点，不在嘴，在炉。炉温稳，稳像首丹余韵——韵在掌，不在舌。

刘婆端粥，粥稀：「兽潮过了，手还稳？」

「稳。」韩泥说，「饭稳，恩不断。」

叶青禾托沈枯芽递一句：「别再来门缝。」

韩泥接话，不接辩。辩，费舌；舌要留来稳炉。

接在心里：不来，是因清；清，是因匿。匿，恩还在脉里温。

夜，席下按坛，低声道：「炉温稳了。余韵在掌。」

坛腹温，贴掌，像应「稳」字。稳，是109的钩——钩110。

丑时，静半刻。香落齐，掌纹温，温而匀。匀，像六层实，实而不飘。

邻床问：「你真要炼散？」

「炼。」韩泥说，「炼了，才还得了下一笔。」

下一笔，在省亲前；前，先110。

更鼓尽，废窑温退，退像敛。敛不要紧，敛，为明炼。

他低声道：「稳了。下一程，培元散。」

培元散，丹道起点，起点在丑手，不在名。

雨后，夜禁解，韩泥回废窑稳炉。

铁无言送渣：「清瘴剩的料，别浪费。」

白像未成，未成是余韵。

第二炉，清瘴路，成灰香。灰香正，正是109的稳。

赖福克扣复原半分，少盯梢半息。

刘婆端粥：「兽潮过了，手还稳？」

韩泥接在心里：不来，是因清；清，是因匿。

废窑温一寸，一寸稳，稳像109的余韵。余韵不是丹，是手。手熟，110才成散。成散，不在急，在稳；稳，是95赠叶后，108清瘴后，这一路走来的回響。

赖扒皮少盯半息，半息不要紧，半息够多炼半刻。半刻，够嗅出投序将变——变向培元散，散在一品，品在渣，渣在栏，栏在手稳里。
