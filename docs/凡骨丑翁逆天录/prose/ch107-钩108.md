# 第一百零七章 钩108

静了半日。

兽散了，像退潮，潮退后界外还腥。腥不是闲闻，是钩——钩在余毒，余毒在栏，栏在，角就不能散。韩泥站在东角石根旁，独眼先扫了一圈草色：草黑的地方，甜味还黏着。黏着就不算完。

管事点名：「丙九韩泥，随铁无言清余瘴。」

点名本身就像钩，钩子落在「清」字上。

清不是拿扫把扫，是拿鼻子嗅。角草黑着，黑里发甜——这味他熟，熟于兽潮前的风声，也熟于备潮那几夜。他把经页摸了一下，页黄，字深一线。深是序，序齐了，清瘴丸才肯落怀。

铁无言臂仍绷着，声低：「清完，我请你吃一袋麸皮——不，请不动。手在就行。」

韩泥应声：「在。」

在，就是请。请不在嘴，在并肩。

清角时，他鼻下一动，动出三处漏。障根补了两处，仍剩一处深得像毒窝。窝要丸，丸要炼。炼不在这一章，这一章只钩——钩出深窝，钩出经页里「清瘴」二字将要坐实。

赖福凑来瞧，不信：「还能嗅？」

「嗅。」韩泥说，「嗅完，手稳。」

赖福嘴尖到一半又收了。收着像也怕余毒。白织月立得远，声冷：「余毒进内墙，谁都跑不了。」

韩泥只答：「不跑。」

不跑是钩的另一头——头在铁无言，尾在清瘴丸。

叶青禾在坡下，门缝灯又亮。他不靠近。靠近像连累；灯亮着，脉里培元丹的温还在，恩还记着就行。沈枯芽悄递话：「叶姐说，别逞能。」

韩泥点头。逞能不在这一步，这一步在清，清在下一段炉前。

夜里管事弟子贴条：「余瘴清尽前，丙九不准撤。」

不准撤，就是钩住。韩泥又立回东角，立得像续战终前那一息。铁无言指着草黑处：「深窝在前。跟我进，进三步。」

「三步。」韩泥应。

三步出去，像当初填障挪位——挪过就不怕。怕也进，进了才嗅得真。秦霜木鱼轻敲：「钩在，心别慌。慌，手飘。」

他不答，答在心里：不慌，只记窝、记清、记并肩。

丑时静半刻。香落齐，掌纹烫——烫像五层将满，满而不破，破要等下一程。邻床问：「真要再炼？」

「炼。」韩泥说，「炼了，栏才活。」

次日还清余瘴。漏补得更实，窝里甜退了半寸，却更黏。黏像等丸。等不要紧，等的是炉与手。管事弟子记簿，笔尖停：「清余瘴，记功。」功还在远，远在丸成之后；这一章功劳只算钩实。

白织月袖风拂过，韩泥不辩。辩费舌，舌要留来守坛、守鼻。沈枯芽又探头：「你真跟铁师兄进窝？」

「进。」韩泥说，「进了才清得尽。」

「死了呢？」

「死在角上，也好过死在逃路上。」他声平，「我不逃。」

赖福第三次来瞧，鼻动动，终于没尖：「丑杂役，别把毒嗅进肚子里！」

「进肚子的是丸。」韩泥道，「丸还没炼。」

「没炼你吹什么？」

「吹完好炼。」韩泥独眼平，「你怕，记住这条线就行。」

第三日，深窝前风止半息。铁无言：「再进两步，别硬顶。」

「不顶，嗅。」韩泥跟进。

甜味贴着鼻梁走，他嘴一张，像要把味咬碎记牢：「沉、腐、浮——浮在最里。」

铁无言皱眉：「你还能分三层？」

「能。」韩泥说，「分不出，丸就投错。」

第四日，管事又来核条。条角被风掀起半分，字仍在：「不准撤」。赖福跟在后面阴阳：「贴条吓鬼呢？鬼比他先撤。」

管事冷眼：「你撤，丙九留下。」

赖福闭嘴。韩泥不看他，只看窝。窝里甜浓，浓像睁着眼等炉火。

夜深，他把经页摊在膝上，一字一字对投序。对完低声：「料齐了没？」

铁无言丢来半袋褐渣：「清瘴剩料。明日障后起炉——挡目，别被赖扒皮看见。」

「挡。」韩泥应。

「成不成？」

「鼻说了算。」韩泥道，「鼻正，丸正。」

回棚他席下按坛，低声道：「钩在了。下一程，炼清瘴丸。」

坛腹温贴掌，像应「钩」字。钩不是戏，是序。更鼓尽，深窝仍甜，甜像睁着眼等。他独眼平，只说：「钩住了。手在，铁无言在。」

在，就不滚。不滚，清瘴路才接得上。接上了，才有并肩那一夜。

同袍夜里咳嗽，咳得像余毒钻缝。韩泥递半块饼：「先垫肚。明日清完，再睡死。」

那人苦笑：「你比管事还狠。」

「狠的是窝。」韩泥说，「我不狠，窝先吃人。」

坡下灯还亮着一隙。他背过身，不往近走。近处留给清，远处留给恩。两笔各记各的，明天只拿一笔去开战：余毒、深窝、炉。

晨光刚白，铁无言在门口砸了砸鞋底泥：「走不走？」

清余瘴的日子最磨人。不是没有仗打，是仗打完了，味还赖着不走。韩泥每天寅时起，先按坛，再摸经页，再跟铁无言对一眼——对上了，再进角。进角时不喊勇，只喊漏：「左二指甜。」「根下浮一层。」「深处沉，别踩实。」

铁无言应得慢，应得准：「听见了。你闻，我铲。」

铲开一层黑草，底下泥像炖坏了的酱。酱气扑鼻，旁边少年捂嘴干呕。韩泥按住他后颈，力道不重：「呕完接着干。干完，味才退。」

少年眼泪都出来了：「这比兽潮还恶心……」

「潮打人，毒吃人。」韩泥道，「吃人的更慢，更要清。」

赖福第四次探头，终于冒出一句像样的：「丑杂役，嗅多了会不会把自己嗅疯？」

「疯了也比你跑得慢。」韩泥不抬头，「你跑了，条还贴着。条贴着，丙九还得清。」

赖福脸一抽，溜了。溜也罢，溜了角里清净一分，鼻子就准一分。准了，他才敢对铁无言说：「窝口这层浮味，明日炉要先压浮，再克沉。顺序反了，丸白炼。」

铁无言点头：「你排，我挡眼。」

挡眼二字落地，韩泥肩上松一点。松的不是劲，是慌。慌在夜里最多：夜里灯少，甜味会骗人，骗人把漏当风。他便把石灰线拉直，拉成一条不说谎的边——边外是毒，边内是人。人出边，就要喊。

坡下沈枯芽又递话，这次话更短：「叶姐问你，手还在不？」

「在。」韩泥说，「在才还。」

「她说……别拿命还。」

韩泥停半息：「命拿来守角。还的是恩。两样分着。」

芽儿听不懂分账，却懂点头。点完头就跑，跑得像怕沾上余毒。沾不上也好——娃儿清着，往后还有石要还。

第四日傍晚，管事把「不准撤」的条又按紧一点：「风掀斜了，字还在。字在，人就别想装看不见。」

韩泥看着那条，忽然觉得钩不止钩他。钩也钩住了铁无言，钩住了丙九那一列还喘着的人。钩住了就不散；不散，明日障后那口炉才有资格起火。

他夜里对坛又说一次，话更土：「钩挂好了。明天别飘。」

坛腹一温，像回他：不飘，就炼。

第五日清晨起雾，雾白得盖住角。韩泥用石灰重新描了一圈边：「雾里最爱骗人。边画清，人就不踩错。」

铁无言跟在边内：「今日只嗅口沿，不硬进。进留给炉后。」

「行。」韩泥应，「口沿浮味若淡半寸，炉才敢起。」

同袍远远递来湿布：「捂口鼻？」

「捂外表，别捂死嗅。」韩泥接布搭在肩上，不捂满，「鼻子是秤。秤蒙了，丸就瞎。」

赖福远处阴阳一句：「装神弄鬼。」没人接。没人接，他的阴阳就落在泥里，溅不起浪。韩泥只当没听见——听见也费不着气，气要留给明日障后那一掐火候。

「走。」韩泥摸了摸坛沿，又摸了摸经页，「钩挂上了。今日障后起炉——起了，就把这钩钉死。」钉死了，坡下那盏灯才敢灭，他才敢把「还」字从清瘴里兑现一截。
