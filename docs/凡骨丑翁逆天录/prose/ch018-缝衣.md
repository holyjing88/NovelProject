# 第十八章 缝衣

叶青禾夜里穿针，针尖对着自己指腹，不对着人。

对人，就成了勾引；对己，只是补肘。肘在袄上，袄在门外泥岗村最漏那户——户里人不喊她名，只喊晦气翁。晦气翁的肘破了，风钻，钻得像针；她补，不是可怜，是人该补。

线过，针脚密。密得像药铺秤，秤药也秤恩——恩轻，秤出来不重；不重，才真。

药铺灯芯她捻紧一分。捻紧，不是怕黑，是怕谣。谣说寡妇送汤就是邪，邪字贴门，贴得比旱还干。她偏不贴。贴的是针脚，针脚不写字，只挡风。

袄是旧陈姑的，陈姑走了，袄还在。还在，就该暖。暖不必进门，进门就坐实「一对」。一对是脏名，脏名比破肘更漏风——她懂，所以隔窗送，隔门墩送，送完就走，走得不回头，像没发生过。

巷口风硬，硬里赵绣春声尖。尖，她早料；料了，仍送。送布包，不进门，不越线。
韩泥臂伤结痂，袄肘又破。破不大，风能钻，钻得像针。针不大，骨里记得疼——疼让人活，活让人还。

夜里，叶家药铺灯亮得迟。迟，像寡妇还在秤药、还在穿针。韩泥编筐，藤条勒指，独眼偶尔往巷口瞥——瞥不是望，是怕。怕有人骂叶家，骂到灯灭，骂到铺关。

门缝外，脚步轻，停。停得稳，像叶青禾一贯站法：不进门，不越线，线外也能送恩。

韩泥没开门。开，坐实牵扯；不开，恩也能落。落得像雪，雪无声，也能厚。

布包搁门墩，轻响。叶青禾声低：「袄破了。我补了肘。」

就一句。没有「你冷」，没有「别冻着」。她懂他怕什么——怕的不是冷，是名。名一脏，汤也脏，姜也脏，针脚也脏。

韩泥在门内，应一声：「够了。」

够了，不是谢。谢字轻，还字重。

叶青禾在门外停半息，没再问。问多了，像求。她不求。只把针收进布包，布包旧，角磨毛，毛得像她这些年的名声——磨毛了，仍包得住针，包得住药，包得住「人该补」三个字。

门内韩泥没开，她也不催。催，像逼。逼恩成债，债就薄了。薄了，不如不送。
巷口忽然尖起来，尖得像盆泼剩水。

赵绣春嗓先到：「半夜缝绝户翁的袄？寡妇还要脸不要？」

周嫂帮腔，棒槌没拿，嘴比棒槌狠：「药铺不开，倒有闲针！晦气沾衣，洗都洗不掉！洗不掉，全村跟着晦气！」

叶青禾在门外，没辩。辩，赵绣春更来劲；辩，周嫂更添词。词里早备着「血祠」「旱」，备着明日公审。

韩泥在门内，隔窗缝看见她侧影——青布素，肩绷着，手里还捏针，针尖对着自己指腹，不对着人。对人，就成了「勾引」；对己，只是「补肘」。

他喉动，想开口。想道：汤是热的，袄也是热的，热不是邪。开口，叶家多一口「一对」的脏名，脏名比破肘更漏风。

他咽回去。咽得像咽刺。刺不进谣，进心。心进，坛便温。

赵绣春逼前两步，绣花鞋踩门墩尘，尘起，像也要沾晦气：「说！图什么？图他绝户给你送终？图血祠旱落你铺里？」

叶青禾抬眼，一眼，平：「图肘不漏风。」

「漏风？」赵绣春冷笑，袖口香粉味硬，硬得像体面，「你漏的是德！德漏了，药也卖鬼！」

叶青禾没退：「德在秤上。秤准，药就不鬼。」

赵绣春一噎，噎半息，转尖：「秤准？你秤的是绝户翁的袄！袄晦气，秤也晦气！」

周嫂啐地：「明日公审，看你怎么辩！赖管事可等着！」

叶青禾收针，针尖仍对自己：「公审审谣，不审针脚。针脚挡风，不挡人嘴。」

就一句。不多。多了，赵绣春更来劲。她走，走得稳，稳得像秤砣落盘——盘沉，声轻，轻里不让步。

笑散去，像潮退，退得脏。脏在巷石缝里，缝浅，浅到章内消不尽。可针脚还在袄肘上，还在，风就进不去半寸。
门墩上，布包还在。还在，像没人敢碰。

韩泥开门一尺，取包，肘里细密针脚，密得像叶家药铺的秤——秤人，也秤恩。恩轻，秤出来不重；不重，才真。

他指尖抚针脚，热。热不是袄，是人手留下的。手五十八岁见过冷，仍肯深夜穿针，针脚就是话。

隔窗那刻，他一句没说。不说，不是认辱，是护她名。名比嘴贵，贵就得忍。忍不是怂，是老头能选的勇——勇不炸，勇记，勇候还。

他进屋，心里叶汤旁，又添衣纹一捺——不增新形，只让叶汤旁多一线布面。布面能挡风，挡不了全村嘴，嘴明日再说。

「肘不漏。」他低声道，「心也别漏。」

坛角微凉。坛腹微温。温里，他望门缝，像望那碗曾热的汤。汤气散了，叶汤还在，捺新了，像衣补在心里。

叶家灯熄了。熄得稳，稳得像没怕。没怕也可能是怕到极致，极致成静。

韩泥穿上补袄，暖进骨。暖是情，情不出口，出口就浅。浅了，像施舍；深了，像欠。他宁可欠。

他只把袄勒紧，勒得像把「先还」勒进肘里。勒完，又编筐。筐稳，手稳，心也得稳——稳到明日公审，稳到叶家秤还在。

巷尾，赵绣春声又起，起得远，像不甘心：「补袄？明日看里正怎么判！」

韩泥独眼不抬，藤条绕膝，绕出一道细棱。棱不尖，尖的是谣。谣尖，他就不尖——尖了，叶更苦。

小胜落在肘上：肘不漏，风不进；风不进，人还能编筐。筐卖出，米买来，命活着，恩才能还。

补袄记在心里，不必声张。声张害叶。害不到，就是今夜的情——情在窗缝，在针脚，在忍住的半句。

叶青禾走后，巷里静一息。静里，韩泥把袄肘对着灯，看针脚，像看一封不会写字的信。信说：我还在。还在，公审就还有理。理在，翁就不开口。不开口，也是还。

张麻婆路过，瞥他窗缝灯，嗓粗，像故意让巷里听见：「补了？谁补的？」

韩泥在门内：「叶家。」

张麻婆：「叶家胆肥。胆肥好，胆肥的人，药才真。」

韩泥：「嗯。」

张麻婆顿半息，又道：「明日公审，你少吭声。吭声，害她。害她，恩就白记。」

韩泥：「知道。坐着听。」

张麻婆啐地，啐向赵家方向：「针脚挡风，嘴挡不了。嘴挡不住，手还得稳。」

韩泥「嗯」一声，不谢。谢字欠着，欠到能还。还字比谢字重，重就得活，活得久，才还得上。

张麻婆走远，远里又喊半句：「肘不漏，心也别漏！」

韩泥低声应：「不漏。」

应完，合窗，灯豆。豆光里，袄肘针脚密，密得像叶家秤还在——秤在，汤就还能热；汤热，公审就还有理。

叶缝衣被骂，骂在帘外，针在帘内。针密，像路——路通将来还烫。韩泥隔窗，不语。语，成牵扯；不语，像没听见，听见了也只记。衣破有人补，补的是路。路窄，恩宽。宽不喊，喊惹祸。第一笔又深一分——深在名节，名节公审要护。护，不是辩，是等叶丫头自己把秤立住。坛温一线，像应针脚。针脚密，心就不散。散，汤就凉。他低声道：「缝的是衣，记的是恩。」恩在心里，不在舌。舌留来等锤。隔窗不语，语成牵扯。牵扯害叶，害叶不如护名——名节公审要清，清了才还热。热是后话，话先在针脚里：针脚密，路就直；路直，缝衣恩才实。骂在帘外，针在帘内。内里的稳，像叶家秤——秤杆弯，不断；不断，韩泥就不近。不近，是护；护到公审，护到「我记着」落地。牵扯害叶，害叶不如护名——名节清了，缝衣恩才还到实处。隔窗不语，护名节——名节清了，缝衣恩才还到实处，实处是还热。

针脚密，骂在帘外；韩泥隔窗不语：语成牵扯，不语成护——护叶名节，护到公审；公审清了，这缝衣恩才还到实处，实处是还热。
