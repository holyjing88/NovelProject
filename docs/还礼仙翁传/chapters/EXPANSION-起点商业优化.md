## 二、金手指差异化（对抗「系统赠礼流」同质化）

### 2.1 一句话卖点（上架简介用）

> **七十二岁，寿元九月。别人修仙靠打坐，莫长春靠送礼——送出去的是人情，还回来的是命。**

### 2.2 与同质作品的切割点

| 常见赠礼流 | 本作差异 |
|------------|----------|
| 面板+数值刷屏 | **鸿蒙万缘塔**：万物可藏，一念收放；人情账，不拖章 |
| 少年开挂 | **老年视角**：敬名分不敬战力；喜剧来自装穷与将死反差 |
| 后宫倒贴直白 | **馈缘俘获**：先恩后情，纳绶含蓄留白 |
| 双主线修炼 | **铁律**：赠礼驱动一切，打坐不能替代赠缘契机 |
| 塔内分层阁字 | **v16废止**：万物可藏，无缘阁/符阁 |

### 2.3 备选书名（点击率 A/B）

| 优先级 | 书名 | 适用 |
|--------|------|------|
| A | **还礼仙翁传** | 精品向、老读者 |
| B | **鸿蒙万缘塔：老头还能活九月** | 黄金三章、新媒体 |
| C | **天道还礼** | 短平快搜索 |

正文内**不出现**「系统」「面板」「倍率」「缘阁」；用「塔鸣一荡」「一念出塔」「善缘入账」。
