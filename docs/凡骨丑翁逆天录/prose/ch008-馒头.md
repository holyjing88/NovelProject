# 第八章 馒头

陈姑棉袄第三日，风仍硬。

韩泥早起理藤。野藤浸了一夜水，皮软了，才绕得紧。先刮老皮，再入水浸，浸软了绕两圈，勒指，留半寸尾，尾塞结里——结在，筐不散；人不散，恩也不散。

门墩上脚印浅，像昨夜有人来过又退。他编筐底，指节起皮，起皮不慌；慌了结松，底就塌。塌不得，塌了，十二文也没处落。

日头未高，他先编两只小筐备集尾。筐沿用指节敲，声闷，闷而实——实了，才配装药，配卖钱。编完一只，搁门墩晾，晾得藤条回半分韧，韧了才勒得住药草。

棉袄袖口长，长出一截，他卷了卷，卷得像把寒挡在门外。陈姑那袄第三日还暖，暖进骨，骨缝里那阵僵，松了半分。松了，手才编得实。

张麻婆骂街骂到晦气翁门口，声却落软：

「韩泥！筐编得歪！歪了卖谁！」

韩泥「嗯」一声，藤条没停。张麻婆骂是习惯，习惯里藏一点别的东西——真嫌，转身走；骂，是还肯当邻。

「张麻婆。」他开门。

她叉腰，脸皱，皱得像晒干的橘皮：「开门干啥，晦气散出来？」

「你骂到门口，不开门，不像话。」

张麻婆噎半息，从怀里掏出布包，甩他怀里：「吃！别饿死我门口，晦气！」

布包烫，实。白面馒头一个，硬，像这老太的嘴。热气扑脸，扑得像一小团白日光。

「我不是可怜你，是烦你！」她唾沫星子飞，「你饿死，抬棺还得找你，累我！」

韩泥接稳。「累你就别送。」

张麻婆眼一瞪：「送都送了，还嘴硬！吃热的，别噎死！」

吐唾沫，三步远，骂着走了：「王家鸡又拉我门口，缺德哟……」

骂声远，远里还有一句，轻，像怕人听见：「筐编结实点，别砸我名声。」

韩泥掰馒，慢慢吃。面实，噎人，噎得清醒。麦香粗，粗得像这村子的日子——不细，真。馒皮硬，内里软，软得像张麻婆骂完还肯塞过来的手。吃一半，留一半，布包好，搁坛边。穷人家，一顿分两顿。

心里又添一圈——叶汤、芽粥、陈袄、张馒。四笔齐了，像四口粮碰在一处：粮少，够活；活够了，才还得上。

门外赵绣春过，瞥见韩泥啃馒，鼻哼：「还有人喂晦气翁，谣传不尽。」

韩泥抬头，独眼平，不接话，只抬了抬手里剩的半馒：「吃馒。嚼得动，人就没死。」

赵绣春唇动，没再尖，走了——骂刃碰着闷石头，钝了。门外风过，带起一点馒香，香淡，淡得像反驳。反驳不必响，嚼得动，就是活。

午后，韩泥担两筐去集尾。筐结实，张麻婆骂出来的结实。藤条勒肩，勒出印，印浅，像手艺的证。

集尾风硬，吹得棉袄领子往脖子里钻。他缩了缩，又直腰——直腰像卖筐，也像活。

半个时辰，卖出十二文，比昨日多三文——手艺是穷翁的刀，刀不快，却稳。三文不多，够盐，够米，够多活一日。

买主掂筐沿：「晦气翁筐，倒实在。」

「结实。」韩泥不接「晦气」二字，「下回还来。」

买主又多摸一文：「多一文，不亏。」

韩泥点头，铜钱落掌心，十二文一文不少。腰布沉了一截——这是今日小胜，辱在村道，钱在掌心；钱不暖，能买米，米能活命。

另一买主凑近，低声：「赖管事要收你税，你小心。」

韩泥应声：「税重，筐也重。」

他买粗盐一小把，米半升，盐粒粗，米色黄，黄得实在。回村。

村道童谣丫念：「晦气翁，吃馒翁……」

念到一半，张麻婆帚子飞来，唬得娃跑：「再念，撕你嘴！」

韩泥路过，不笑，心里松一寸——有人骂着护，也是邻。张麻婆扫他一眼，哼：「吃馒别噎着，噎着还得我抬棺！」

「麻婆。」他停半息，「筐我编结实了。」

张麻婆一愣，哼更大：「结实就卖！别饿死我门口！」

娃躲在墙后探头，小声：「麻婆，他真吃馒？」

张麻婆帚子一举：「吃！活人！再念晦气，撕嘴！」

娃一缩，缩得像信了——晦气翁嚼得动，就不是鬼。

井台边，韩泥打水。棉袄领子竖着，竖得像一层薄盾。村人见棉袄，让了半步。半步够走，够打，够让陈姑那袄在村道上有用。

有人低声：「晦气翁也配吃馒？」

另一人哼：「你配嚼？」

声息散了，散得像风咬不动这半步。韩泥拎绳，放桶，水起，浑，稳稳落地：「吃馒，也打水。井是村的。」

周嫂棒槌悬着，终是没砸响。旁边妇人低声：「张麻婆送的吧？」

周嫂脸一沉：「少嚼。」

韩泥打水，桶满，拎回。水晃，晃在桶沿，不洒。洒了，更像晦气。他不辩，不炫，只穿，只嚼。穿和嚼本身就是答：晦气翁没冻死，有人把他当人。

夜里，米缸有米，灶生火，粥香薄，香得实在。粥里米粒翻，翻得像四口粮在锅里碰。碰不着，心碰着了。韩泥喝粥，按了按心口：四笔恩齐了，心齐一截。活够了，才配说还。

丑时前，他摸坛沿，沿温一线，贴掌不退。瓮没醒，温在，像替他守着张麻婆这一个馒。

沈枯芽隔巷探头，声小：「韩爷爷，馒……香吗？」

「香。」韩泥说，「面实，人实。你娘还咳？」

「咳。」娃点头，「我娘说，别送了，会罚。」

「别送。」韩泥蹲低，「情记着，比再送安全。」

娃「嗯」一声，跑远。跑远里回头一眼，一眼像徒线又实一分。

里正巡巷路过，咳一声：「井是村的，袄是人送的。谁再嚼，记名。」

赵绣春门内一滞，声哑半寸。哑半寸，够他胃里的暖再实一分——实了，第四笔恩就不白收。

路过沈家墙根，芽儿远远看一眼，像一粒热姜落回汤里。韩泥不招手，招手像讨。讨轻，恩重。

张麻婆巷口又骂一句，骂尾却落「别冻死」——冻死了，谁还她的馒恩。韩泥不应骂，只应活。

路过赵绣春墙根，有人窃语：「张麻婆老糊涂，绝户也送馒？」

赵绣春嗓尖，尖不过桶底碰石台的闷响：「送馒？送晦气！」

韩泥不辩，只把桶放稳，水不洒。馒在胃，水在桶，路还在。

坛沿温不退，应馒不应谣。四形齐了——汤、粥、袄、馒，各有一形，各有一暖。暖在，手还要编筐；筐卖一文是一文，一文也是还路的肉。

丑时末，村犬哑叫一声，像谣也哑半寸。他低声道：「汤还是热的。馒也是实的。」

漏雨滴在灶沿，滴成细线，线断，断在坛边。坛沿微凉，凉得手指一触，沉。他低声道：「冬来了，人没冻死。」

外头张麻婆家咳两声，像确认她还活着，他还活着。活着，恩就能还；能还，就不白收。

丑时前，叶家药铺灯弱着，弱也是亮；亮着，叶汤就不倒。坛沿一线温，贴掌不退，像替他守着张麻婆这一个馒，守到还得上的那日。

他摸腰布，十二文还在，布沉，踏实。村嘲在远，坛在角，钱在腰。沿在，根不散；人不问，坛也不答。

丑时坛壁微凉，午时又温，他当天候作怪，从不多想。今日擦坛，指尖顿了顿，沿仍跟凡物无二；他按了按心口，四笔恩在，像四口粮齐了。

明日还要去集尾卖筐，还要去井台打水。井是村的，谣烫不了胃里的实馒。手还得稳，稳了，四恩才齐；齐了，才配说还。馒在胃，筐在手，恩在心——三处齐，路就不绝。四形齐了，还路不断，明日还要去编筐，还要去把藤条理齐，还要去井台打水。走着。
