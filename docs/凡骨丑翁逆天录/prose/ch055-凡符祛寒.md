# 第五十五章 凡符祛寒

冬深，霜厚，宗门外驿亭终于有人起身。

韩泥搬柴过侧门，老耿在旁挑水，水洒半勺，声哑：「去。不白拿。拿了，记耿恩，记僧恩，不拜血祠。」

褐袍僧不诵长经，袖里取出黄符，符边毛，毛得像凡物。凡物，才入凡人手。风从驿亭柱边刮过，刮得符角轻颤——颤的不是仙光，是冬气。

「寒侵骨，骨寒侵手。」僧声低，「符不入丹房，不越阶。阶在远，寒在近。近，先祛。」

韩泥不接，先问：「为何是我？」

僧指他掌茧裂口：「你鼻下静，手下正，坛边席严。严的人，符不辱主。」

老耿哼一声：「僧是我引的。引，不是邪，是冬里少死人。」

「少死人？」韩泥又问。

「少死，谣就少。」老耿道，「谣少了，你那窄路才宽半分。」

韩泥这才接符。符薄，贴掌，掌暖一丝，暖得像从前经片贴怀——贴完，掌纹里留一线浅烫。烫浅，像凡物认手，不认脸。

「凡符祛寒。」僧道，「燃则一日暖，不燃则贴怀。贴怀，护骨寒，不护心邪。心邪了，符自灭。」

韩泥低声：「心记着七笔，不邪。」

僧又嘱：「符是凡品，祛寒，防。勿示人——示人，谣又来。」

僧合十，不拜坛，只拜冬：「瓮在眠，人别睡死。睡死，是拜邪。眠，是等。」

僧离亭，脚步远。远了，像凡符使命落了地。驿亭空，空得像赠毕——毕了，怀才暖。韩泥把符贴进衣里，贴着心口偏左：左近心，心要不邪，符才肯暖。

回棚，符藏怀，贴身，不燃。燃了烟起，管事查。赖福嗅他袖，嗅像找由头：「藏啥？」

「寒。」韩泥说，「寒在骨，不在袖。」

管事路过，声冷：「宗门外的事，少议论。议论，像借商线腥。」

赖福哑了。管事又道：「符凡物，别傲。傲了，邪。」

刘婆端姜汤，姜薄，热：「怀里有符，别傲。」

韩泥喝尽。姜热像远村灶火——远火在，还路在。邻床问：「啥符？」

「凡符。」韩泥说，「祛寒，防。不燃，贴怀。」

「贴怀干啥？」

「贴怀，手不僵。」韩泥说，「手僵了，藤刺偏。刺偏了，血找错路。」

「血找错路又咋？」邻床还不放手。

「找错了，沿就不净。」韩泥道，「沿不净，后头缘就接不住。」

午后西栏清粪，风硬，符暖护骨，手不僵。粪锹沉，沉得像搬柴。搬柴手正，清粪手也要正——正了，试手才不滚。小兽咳，咳得细。他多铺一层干草，草旧，旧能暖。管事盯梢：「清粪加半刻。半刻偷懒，克扣麸皮。」

他不偷懒。克的是粮，克不走还路。铁无言巡栏边，声低：「符在怀，别示。示了，谣来。」

「不示。」韩泥只答。

夜里他对坛低声：「凡符起了，仍凡。凡，够活命。活命，才还恩。」

坛腹温一线，不与符抢。符护人，瓮眠等缘——各守一线，不拜邪。秦霜远廊经过，一句：「心别邪。邪了，符灭。」

韩泥怀符暖一丝，心记着七笔，不邪。丑时他按席，席严，沿净。净了，才接后头血近沿，接瓮醒那一误。外头更鼓沉，柱上五日条还在。五日，试手凝丹——凝丹前，符不燃，沿不血，席盖严。

同批少年碰他袖：「你真得了符？」

「得了。」韩泥说，「凡符，祛寒。不示，不傲。」

「示了咋办？」

「示了，谣来。」韩泥说，「谣来了，路易断。」

少年懂「断」，闭嘴。搬柴经侧门，风更硬。韩泥不凑亭，只瞥一眼——褐袍亭已空，空像赠毕。铁无言又巡过来：「符在怀，手别僵。僵了，藤刺偏。」

「不僵。」韩泥应，「僵了，血找错路。」

四更邻床翻身，袖风扫过他怀。怀暖一丝，符不灭。灭，心邪；心不邪，符自护。卯时搬柴，风更硬，硬像西栏粪锹。符暖护骨，骨不寒，手不僵。管事验垛，声沉：「冬里沉得住，记功在。」记功在，麸皮不克。午后铁无言廊下见他：「符凡品，别傲。傲了，邪。」

「不傲。」韩泥答，「傲了，符灭。」

黄昏坛腹温一线，不与符抢。缘在沿前，不在符烟。他不燃不示——示了谣来，谣来路易断，粥就凉。他独眼平：「符护骨，沿在净。净了，才接后头缘。」

丑时他揭怀角透气，符边不燃，只暖一线。暖线贴骨，骨不僵，坛腹却凉回去——像符护的是人，瓮等的仍是缘，各不相欠。他对坛低声：「凡符在怀，瓮仍眠。眠，不争。争了，像拜邪。」

侧门风硬如刀。刀贴骨，符暖护着。护骨不寒，手才不僵，缘才正。刘婆又端半勺姜汤，热：「符在怀，别空肚。空肚，鼻下香飘。」

「不空。」韩泥喝尽，「姜在记里。」

「记里有姜，怀里有符。」刘婆低声，「两样都别拿去跟人吵。吵了，一样也不长。」

「不吵。」韩泥说。

老耿夜过侧门，水洒实，声哑：「僧是我引的。引，不是邪，是还经片那一恩。」

韩泥不答，只摸怀符。符边毛，毛得像凡物——凡物，才配丙九末排。管事夜巡路过，鼻下嗅袖：「无烟，好。」他不辩，只按席。按实了，像把符恩与耿恩、僧恩并记第七笔里。秦霜远廊又一句：「符凡品，心别邪。」

韩泥不答，答在心里——心里七笔不混，符才护骨。护骨不寒，手才不僵；不僵，五日后试手才站得住。西栏第二趟，粪锹沉，符仍暖。暖，不抢瓮温。小兽栏角眼湿，咳得细。他再铺干草，暖兽，像冬里少死一人。邻床问符味——味淡，淡得像凡物。他不燃符：燃了管事查，像借腥；借腥，恩易断。

五更前柱上五日条被风吹得猎猎。猎猎不是催滚，是催手正。手正，符护骨，瓮眠等缘——各守一线，不拜邪。铁无言第二趟巡栏，不问怀里有啥，只验鼻：「鼻下静，符才暖。静乱了，符自灭。」

「静在记里。」韩泥应。

刘婆第四趟端姜汤，热：「符在怀，别傲。」

「不傲。」韩泥喝尽，「傲了，符灭。」

搬柴最后一趟，肩勒旧印，沉得住。侧门风过，驿亭柱影斜，斜处已无褐袍——无袍不要紧，黄符已在怀。

回棚路上，他遇着老耿。老耿桶沿滴水，滴得慢：「符贴稳了？」

「贴稳了。」韩泥说，「不燃，不示。」

「稳就好。」老耿点头，「我引僧，不是引你去拜邪。拜邪，我那点水也白挑了。」

「不拜。」韩泥道，「记恩，记耿恩，记僧恩。记下了，才敢用暖。」

老耿「嗯」一声，挑着水走了。走远了，韩泥才把怀口掖紧。掖紧不是怕冷，是怕风把符边掀给旁人看。旁人一看，谣就长脚；长脚的谣能从驿亭追进丙九棚，追进麸皮账，追进试手名单。

丑时末，他揭席验沿，沿仍滑，滑里无血。无血，符才肯继续暖；暖着，他才敢把「僧恩」填进第七笔那一格空位。填完，他低声对坛：「符护我，你仍眠。眠着等缘，缘不在烟里。」

邻床在梦里哼一声，他听见了也不解释。解释像示符；示符像邀谣。他只把怀口再掖深一分，深了，暖才稳——稳了，明早搬柴肩才不打飘。飘不得：肩一飘，藤刺就偏；刺一偏，沿前那点规矩就险。险了，暖也白贴。白贴不得——贴了就要护到手正那一日。手正了，符恩才算接实了。接实了，才敢往下走。

符录起了，仍凡，不越阶。凡，够活命；活命，才还恩。他按席，沿仍净：净了，符才护骨；护骨，手才不僵。僵不了，五日条上那一验，才站得住坡下那口气。气在，窄路在；路在，汤就不凉。凉不得——凉了，冬深这一赠就只剩下纸边那点毛，毛里存不住还。还在前头，暖先贴骨；骨暖了，手才扛得住后头那些试。
