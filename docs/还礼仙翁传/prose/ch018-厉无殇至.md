# 第十八章 厉无殇至

第七日，赤焰少谷主亲至。

青岚议事大殿，旗未入殿，威先入。厉无殇踏步上阶，玄袍金线，魔息如刃，笑挂在唇，像一把还没出鞘的刀先寒了空气。

韩铁山伴侧，甲未卸，眼四扫，像数青岚门还有几块硬骨。

柳青鸢升座，霍镇山按盾立侧，周德海捧和折子立末，像备好的退路。

莫长春袖空，杖在塔中，立殿柱旁，灰袍素，须眉雪白，在满堂华服里，像一张旧符，却贴在最显眼处。

厉无殇目光落在他身上，停住，笑深一分：「这位，便是莫长春莫长老？大乘劫败，筑基中期，寿元九月——啧啧，将死之人，占着长老位，占着青岚门的脸。」

殿内寂然。

有人吸气，有人低头，有人忍不住看莫长春——看他将死之人如何接这一刀。

莫长春抚杖，微笑，像听一句久等的话：「厉少谷主嘴利。老夫九月，确实将死。」

厉无殇扬眉：「既知将死，何不早退，把位让给有用之人？」

「退可以。」莫长春道，「死也可以。但在退与死之前，老夫还想听少谷主把东脉三成说完——说完，老夫或许还能送厉少谷主一份礼。」

全场一沸，沸在窃语：送魔头礼？

厉无殇眼底一闪，像被勾起兴：「礼？莫长老九月寿元，赠我何物？半块灵石？一包药渣？」

「礼轻。」莫长春笑，「意重。」

魏无涯立大殿梁上阴影里，扇未开，只观气运，观莫长春袖中那一缕说不清的热，像观一枚将落未落的子。

柳青鸢指节发白，未言，等师兄接招。

霍镇山盾光微亮，像随时要战。

周德海却笑，笑里喜：「莫长老都愿赠礼，和谈有望啊。」

莫长春看他一眼，像看一盆油：「和谈有没有望，不在礼。在根。」

他转向厉无殇，声仍轻：「少谷主，东脉民耕三百年，你索三成，是索命。命可以给一点，根不能拔。你若懂，便谈；不懂，老夫这九月，也够活到你后悔今天说的话——不过，还没说到那一步。」

殿内再寂。

厉无殇笑意定住，像刀终于碰见骨：「好一个『还没说到那一步』。莫长老，我记住了。」

他落座，谈判开场，话却句句带刺，像把东脉灵田切成席面，要青岚门吞刀。

莫长春不辩细节，只偶插一句，句句在「根」字上，像把柳青鸢主战之心，钉在殿心。

谈判至午，灵田亩数、划界碑位、民户迁徙，条条带刺。

厉无殇指地图东脉三成，指尖如刀：「此处，此处，此处。七日谈，谈的是线，不是脸。」

柳青鸢声稳：「线可以谈，命不能卖。」

霍镇山按盾，盾光微亮，像随时要把地图撕了。

周德海赔笑插话：「少谷主，民户可迁，迁了仍耕，耕仍归青岚——」

厉无殇冷笑：「迁了，根就断了。断根的人，还耕什么？」

莫长春偶插一句，句句不落席面，只落「根」字：「根在民，民在田，田在青岚。索三成，索的是三代人的脚。」

厉无殇目光一凛，像被「脚」字扎到，扎到仍笑：「莫长老，脚能砍，头也能砍。」

「头能砍，」莫长春淡道，「砍完，你拿什么吃稻？」

殿内再寂，寂里，魏无涯在梁上扇开一线，观厉无殇额汗，观莫长春袖空，低笑：「脚与稻，比三成有趣。」

宴前，日头毒，风硬。

厉无殇解甲披风，额汗一层，仍笑，笑里躁。韩铁山低声：「少谷主，莫长春袖中无宝光，放心饮。」

厉无殇嗤：「宝光在袖外，不在袖里。九月老头，专让人看不透。」

廊下弟子三层围庭，围得像等戏开场——开场不在酒，在帕。

林挽月挤在后排，踮脚，竟也怕错过。她想起赵师兄白日里冷，夜里却去叩莫长老门，忽然觉得青岚门这两日，比执法堂还难站边。

苏念慈在丹堂远廊，丹香与酒气隔一条风，她指尖掐木栏，掐得像怕师兄又赠出一份「只有别人收得下」的东西。

赵玄机立人群前，铁牌在腰，冷眼观莫，心里却翻：赠魔头礼？找死？还是找活？

他想起还库那日，心挪的一寸，一寸像半块灵石，硌得睡不着。

莫长春立殿柱旁，袖空，一念出塔，素帕在塔中压着，帕角干净，像等了很多年。他望厉无殇，笑，像望一份将送的礼。

柳青鸢在殿门内，心弦微紧，不知师兄袖里是什么。

塔影在识海沉，沉得像要把因果一口气记满。

他低声，仅自己听见：「下一息，礼该出门了。出门之前，先让全场看清：席是席，界是界。」

宴设庭前，灵桌玉盏，酒未温。韩铁山验席，席设得足，足里藏刀——刀不在刃，在汗。厉无殇额汗一层，仍笑，笑里躁，躁得像刀还没出鞘，鞘先热了。

廊下三层弟子围庭，围得像等戏开场——开场不在酒，在帕。有人低声：「送魔头礼，青岚脸先碎？」另一人答：「脸碎不碎，看下一息。」

林挽月挤后排，踮脚，怕错过。她想起赵师兄白日冷，夜里却去叩莫长老门，忽然觉得青岚门这两日，比执法堂还难站边。

苏念慈远廊掐木栏，像怕师兄又赠出「只有别人收得下」的东西。她不问帕，问也没用——有用的是收得下的人。

赵玄机立人群前，冷眼观莫，心里翻：赠魔头礼？找死？还是找活？他想起还库那日，心挪的一寸，一寸像半块灵石，硌得睡不着。

魏无涯梁上观，观厉无殇额汗，观韩铁山按刀，低笑：「汗比刀先出，好戏在后。」

周德海陪笑斟茶，茶洒手，笑也洒，洒得像和已成——成不成，不在茶，在下一息。

霍镇山盾立廊下，不入席，像随时掀桌。柳青鸢礼数足，眼神更足，足里像手帕那夜未到的因果，今夜一并等。

莫长春立柱旁，袖空，一念出塔，素帕在塔中压着，帕角干净，像等了很多年。塔影在识海沉，沉得像要把因果一口气记满。

他望厉无殇，笑，像望一份将送的礼：「少谷主，东脉三成谈不完，席上的汗，倒先谈完了。」

厉无殇眼底一闪，像被「汗」字烫到，烫到仍傲：「莫长老，别装神。九月之人，能有什么礼？」

「礼轻。」莫长春道，「意重。重不重，你接便知。」

谈判再开，条条带刺，刺在灵田亩数、划界碑位、民户迁徙。莫长春不辩亩数，只偶插「根」字，像把柳青鸢主战之心，钉在殿心。

柳青鸢眸光微动，不拦，不追，像懂：师兄不喊战，战字却在根上。廊下窃语未歇，歇处是九月老头袖空那一瞬——袖空，塔在，帕在格子里等。

莫长春立柱旁，袖空，一念出塔，素帕在塔中压着。他低声，仅自己听见：「下一息，礼该出门了。」

赵玄机验魔帖，帖黑边，低声：「魔少至，门里该紧。」苏念慈闭窗，低声：「魔少在，炉更要稳。」

柳青鸢礼数足，眼神更足，足里像手帕那夜未到的因果，今夜一并等。廊下三层弟子围庭，围得像等戏开场——开场不在酒，在帕。

莫长春念：「少来了，席要来。席比刀热，热里藏帕。」

宴庭灯亮，少谷主笑浅，笑里刀已出鞘半寸——半寸够全场寂然。

弟子渐散，散里仍回头瞧一眼灰袍袖空——瞧得像瞧一眼九月还能走多久。
