> **效力**：自 **2026-07-14** 起，**后续新写 / 返工改笔** 的 `prose/` 正文，**一律采用本规范**。  
> **样板章**：`prose/ch014-村议辱叶.md`（通俗白话落地样）  
> **硬闸不变**：字数 2000～2500 · dup&lt;2% · 单段≤180 字 · 破壁/禁词仍按 `30`/`07`。  
> **历史档**：此前「仙凡笔锋 v2」电报短句体记入 `30` §六附，**仅作旧稿参考，不再作为新稿标准**。

---

## 一、一句话定义

用**正常人说话、正常小说叙述**写韩泥的故事：句子读得懂、气息顺、像说书也像过日子，**不要**文言腔、不要电报体、不要公式口号凑字。

---

## 二、要什么（正例）

| 项 | 做法 |
|----|------|
| **白话完整句** | 有主语、有着落：「他天一亮还是编筐」「祠堂前挤满了人」 |
| **叙述可朗读** | 一口气读下去不拗口；少用「语，成…；不语，成…」式对偶暗号 |
| **对话口语** | 村里人怎么骂就怎么骂；老人怎么应就怎么应（「嗯」「记着」「够了」可留，但别章章套模板） |
| **一场一事** | 仍守 `07` 一章一事；场景里写清动作与选择 |
| **章末戏内钩** | 用人事悬念收（公审、帖、咳、柜），别写「怒值满」「读者若…」 |
| **老头味** | 忍、记、还——用动作和短对白，不少年吼、不文青长抒情 |

### 对比例

| 旧（仙凡 v2 / 电报·文言味） | 新（通俗白话） |
|------------------------------|----------------|
| 天阴，泥岗村像扣着一只瓮。 | 天阴着，泥岗村闷得像扣了口大锅。 |
| 语，成「一对」；不语，成「默认」——他选第三条：听，记。 | 开口像坐实「一对」；不说又像默认。他选第三条：听着，记着。 |
| 声浪顶到顶。顶得指节响，响在袖里，响不出声——满，满到里正不得不抬手。 | 声浪顶到顶。韩泥袖里指节响，响不出声——怒满了，里正不得不抬手。 |
| 村议散，怒值满；舌留公审，心留恩辱—— | 村议散了，怒气却堆满了。舌头上留着公审，心里留着恩和辱。 |
| 柱上条角白一线，三年约在怀。（后部垫字窜章） | **禁**：后部模板句窜入前部；章内情节要对本段 `05`/`16` |

## 三、不要什么（禁）

1. **文言腔**：半文半白、硬对偶、字典词堆砌（「俟」「遂」「焉」「之谓」等无必要不用）。  
2. **电报体**：为押「短」而碎裂信息，读起来像提纲或评分备注。  
3. **公式口号**：「分列不混」「怒值」「小爽落点」「舌要留来」**连续复读**；必要交代一句即可。  
4. **提纲入文**：章号自指（「135只嘲」）、策划话术入戏。  
5. **机械垫字**：`更鼓沉，他按木牌——N息…`、跨章 PART2 通用块、同句复贴。  
6. **错时代号窜章**：如泥岗村章出现「三年约玉牌/大比帖」未到剧情却硬贴。
