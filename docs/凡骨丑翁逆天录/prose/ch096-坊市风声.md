# 第九十六章 坊市风声

坊市人稠，风也杂。

外门坊在药山南脚，棚挤着棚，价乱得像没人管。乱里藏真货，也藏嘴仗。嘴仗起了第三日，有人嚼舌头：「听说丙九有个丑杂役，夜里嗅渣，嗅得像炼丹。」

韩泥不往棚心凑。凑近了像显摆；显摆一出，赖扒皮就跟着来。他立在末摊，破筐反扣着，筐上搁三包药渣，渣灰扑扑的，标价压到最低。

标最低，才像末席该做的事。做自己该做的，才少招眼。

铁无言从棚边过，脚步停了半息：「你要挂牌，先听风声。」

「听。」韩泥说。

「风声有两条。」铁无言声音压低，「一条说你能嗅；一条说你是晦气，碰谁谁倒霉。」

韩泥独眼平看着他：「晦气我熟。」

铁无言没笑：「熟不要紧，别接骂。接骂了，坊市不收你。」

韩泥点头。收不收，不在这一日；这一日，他只听。

听着听着，白织月袖风飘过，声冷：「丑杂役也摆摊？摊什么，卖腥？」

韩泥只答：「卖渣。渣净，栏也清。」

白织月嗤了一声，那嗤声很快远了。远不要紧，远了才听得全。

全了才知道，棚尾有人在议兽潮——议得远，像说下个月的事：「南荒妖兽聚着，不定哪天冲药山界。」

韩泥耳尖一动，不动声色，把那几个字按进心里。兽潮不在舌上练，舌要留来守摊，也留来守坛。

刘婆买线经过，线粗得扎手，声音却低：「少摆摊，多吃饭。」

「饭稳。」韩泥说，「摊是听风。」

刘婆哼一声：「风硬，别吹歪路。」

「不歪。」韩泥应。

午后，管事弟子巡坊，声硬：「杂役摊，税半文。」

韩泥交半文。文旧，旧得像常杂役的口袋。巡完，他收摊，收得像从没来过。从没来过不要紧，风已经进了耳——进耳两条：嗅渣像丹，还有兽潮将至。

入夜，席下按坛。坛腹温一线，贴掌，像应了个「听」字。听完不算完；完是下一回挂牌试水。

赖福堵门：「坊市去了？去干什么？」

「听风。」韩泥声平，「风硬，手稳。」

赖福眼细：「风里有你的晦气？」

「有。」韩泥说，「我熟。」

赖福啐一口走了。走不要紧，风已经听过。

邻床悄问：「兽潮真会来？」

「来不来，都要备。」韩泥说，「备手，备渣，备着别飘。」

「备什么？」

「备耳朵。」韩泥道，「耳先听全，手才不慌。」

丑时，廊下静半刻。粟壳香淡，香里混一点坊市尘。尘淡，才像他真去过、又没显摆过。

他低声对坛：「风声过了。下一回，挂牌。」

挂牌不在图名，在试手。试坊市认不认这双丑手，认不认这末席的鼻子。

同批少年睡不实，又问：「你真敢挂牌？」

「敢试。」韩泥说，「试不过就撤。撤了，栏还得清。」

「不丢人？」

「丢人不如死人。」韩泥道，「死人不还清恩。睡。」

少年闭嘴。韩泥再按坛沿，沿净，心才稳。

次日他又去末摊。仍是破筐，仍是三包渣，价仍低。药摊婆远看他一眼，没买，也没骂。没骂也是风——风里少半分毒，他就多半分活路。

铁无言立在十步外，声很低：「风硬，别辩。辩费舌；舌留着守坛。」

「听风，不叫。」韩泥应。

管事弟子又在柱上贴条，字丑，却真：「南荒兽聚，或冲药山界。」韩泥读条，不近，也不躲。躲像怕；怕也可以，怕了更要备。备，先在耳，再在鼻，末在手。

赖福晃木牌堵坊口：「丑杂役，风里就是你的晦气！」

韩泥不辩：「听。」

白织月袖风又过，冷一句：「还卖腥？」

「卖渣。」韩泥说，「腥在栏里，不在包里。」

她唇一抿，走了。走时同门低笑：「晦气摊。」韩泥当耳旁风。风进耳要记，笑进耳只过。

刘婆傍晚又拦他一步：「听够了没有？」

「够听，不够挂。」韩泥说，「挂要再等一日。」

「等就等。」刘婆道，「别为半文把自己卖进去。」

「卖不进去。」韩泥应，「半文只买耳。」

夜里赖福又堵过一次门缝：「三日看你摊。摊瘟了，连坐丙九。」

韩泥应声：「三日。」

三日在前，像一根绳勒着。勒着也好，勒着才知道挂牌不是耍。

他摸怀里玉牌，牌凉，仇名远着。远名烫不得；烫了，手飘，摊就歪。歪了，兽潮条也不用听——听了也白搭。

散队后他过药渣场角，风硬，鼻下却静。静里分得出谣与真：谣在舌尖，真在耳根。兽潮二字贴在界柱上，贴实了，才催他挂牌试水那一步。

邻床又问：「你听风听出什么？」

「听出人多嘴杂。」韩泥说，「也听出潮在后头。后头不怕，怕今日手飘。」

「那你明日挂不挂？」

「挂。」韩泥道，「牌歪字丑都行，鼻子要正。」

坛腹又温一线。温像应「听」字。听过了，才配试。试过了，才知道坊市到底认不认他这双手。认不认别喊，记着就够——记着，明日再去挂那块歪牌。

更鼓将尽，棚外风还硬。他按木牌贴胸，牌凉，心里却热半分。热不是喜，是刃磨过耳——耳听全了，刃才不虚砍。

午后棚尾更吵。有人赌兽潮来，赌杂役先逃。赌声刺耳，韩泥当没听见。接赌像认晦气；认了，鼻子再准也白搭。他只把破筐挪半寸，挪回末席该蹲的位置。

药摊婆远远递话：「末摊那位，渣还卖不？」

「卖。」韩泥说，「价低。」

「低就对了。」婆子哼，「高了，像要成精。」

韩泥没笑。笑一响，棚里嘴便多。他低头理渣，理得包口齐，齐得像清栏那几下——手上活做干净，风里的话才伤不到骨。

同批少年蹲过来：「你就不怕他们嚼？」

「怕。」韩泥说，「怕也听。听全了，才知道哪句是刀，哪句是屁。」

「哪句是刀？」

「兽潮那句。」韩泥道，「刀在后头，先别拿舌头去接。」

少年吐舌头走开。韩泥望向界柱，柱上字被风吹得猎猎。猎猎不是热闹，是提醒：听完风，还得把日子过到挂牌那一步。一步跨早，像往赖福秤上跳；跨晚，潮来时鼻子也用不上。

铁无言黄昏又现，扔过来半块硬饼：「吃。听风也得有力气。」

「谢。」韩泥接。

「少谢。」铁无言道，「谢多了像欠。欠多了，手软。」

「不软。」韩泥说，「饼进肚，耳还在。」

「耳在就行。」铁无言看一眼摊，「明日要挂，今夜别炼出味。味一飘，巡夜鼻子比狗尖。」

「不炼。」韩泥应，「只守坛。」

夜里他果真没碰窑火。席下坛温一线，温得老实。老实才像听风人该有的动静——动了嘴，不动规矩。

赖福半夜踹门框：「丑货，摊上赚了多少晦气钱？」

「半文税交了。」韩泥说，「赚的是耳。」

「耳能喂狗吗？」

「能喂活。」韩泥道，「活着清栏，比喂你嘴强。」

赖福骂骂咧咧走了。走后廊下静下来，静里还能听见远岭偶尔一声兽鸣。鸣远，心却近。近的是备，不是慌。

他把三包渣重新码好，码进筐底。码齐了，像把「听」字码实。听实了，明日挂牌才不像做梦。做梦挂牌，一碰就碎；醒着挂牌，碎了还能撤，撤了还能清栏。

刘婆丑时前又塞他一缕线：「补袖用。袖破了，像丐。」

「补。」韩泥说。

「补完听风？」

「听完了。」韩泥道，「下一回伸手。」

「伸手也别伸太长。」刘婆叮嘱，「长了，被人拽进去。」

「只伸半文那么长。」韩泥应。

她这才走。走后他把线塞进袖，袖里凉，心里热。热不是喜，是刃又磨了一道——听风磨耳，挂牌磨手，两道磨完，才配在兽潮前站住。

天边白一线时，他仍坐着。坐着把两样摁实：嗅渣的谣，兽潮的条。谣要躲开赖福的嘴；条要记进清栏的手。两样都不混。混了，路断；不混，明日才能把牌挂出去，把丑手伸到尘里试一试。
