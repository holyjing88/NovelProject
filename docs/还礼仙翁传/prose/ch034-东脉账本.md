# 第三十四章 东脉账本

长老会再议东脉，烛比人长。

周德海捧册上前，册厚，厚得像七年和平能称斤卖：「割让三成，七年不战，灵石收益、伤亡减免，皆在此表。」

他指节敲表，敲得自信，像数字能当盾：「第七年，宗门弟子守田，轮换休整，成本可控。」

霍镇山拍案，震得茶杯跳：「割地换安，安在谁枕上？第七年赤焰再要五成，你拿什么填表？」

周德海笑：「霍长老，打仗也要算账。」

「账可以算，」霍镇山吼，「根不能拔！」

殿内低议如潮，年轻长老看表，年老长老看周，柳青鸢不言，目光落莫长春——师兄拄杖立窗下，像一根不表态的秤砣。

周德海趁势推：「莫长老，九月将尽，战起耗命，和谈省命，长老以为呢？」

全场目光齐落，像等九月老头也给「和」字添一笔。

莫长春忽然开口，声轻，却清，像秤砣落盘：

「周长老，第七年，谁守田？」

周德海一噎，像被问住的不是表，是魂：「……宗门弟子。」

「弟子会老，田不会。」莫长春道，「根在民，不在表。表上的灵石，买不回迁走的民，买不回被妖踏过的稻。」

全场一寂。寂里，有人倒吸一口气，像第一次听见「守田」二字还能这么问。

周德海笑容僵，揖手退半步：「莫长老……言重了。」

「不重。」莫长春道，「重的是表。表厚，心薄。」

霍镇山抱拳，声洪：「莫长老这一句，比盾还硬！」

柳青鸢指节叩案，一声，两声，第三声前，周德海已识趣闭嘴，册子仍捧在手里，像捧一块烫手山芋。

廊外弟子听见「谁守田」，窃语炸开：「周长老答不上来？」「莫长老又一句话定场？」「九月老头，专问要命处。」

会后，莫长春出厅，霍镇山抱拳：「莫长老，东脉这一仗，霍某跟你。」

莫长春笑：「硬不过东脉泥。泥在，人才在。」

周德海从侧廊过，目光与莫长春一触，一触即分，像谁都没看谁，又像谁都看见了谁袖中的简。

魏无涯在檐角，扇合，低笑一声，像看一枚子将落。

周德海退席后，袖中和折子仍湿，湿得像七年和平泡过汗。孙福远闻「谁守田」，嘀咕：「守田？守的是克扣吧。」

霍镇山再骂：「表厚，盾更厚！莫长老一句，比表硬！」

弟子围殿外，围得像听一场定场，定场不在拍桌，在问魂。

魏无涯扇合，低笑：「第七年，谁守田？好问。」

柳青鸢眸光微动，未言，像把「谁守田」三字记进宗主令里。周德海揖手退，退得像捧烫手山芋，山芋烫的是和，不是表。

顾小满在杂役堂听见消息，冷眼压回孙福旧党嘀咕。

莫长春望东脉方向，像望一场将到的夜。低声，像钩：

「表可以捧，根不能拔。下一声拍桌，该宗主了。」

风腥，云低，像秦氏简上的「夜袭」二字，已贴到东脉田埂。

殿外弟子围成一层，围得像等下一声拍桌——拍桌不在周，在柳；不让字一落，表就成废纸。

大因果还在赤焰夜袭那夜，像妖骑，像杖剑，像全场寂然的斩。

孙福在角隅听「谁守田」，面色白，像听见自己的克扣将被翻。霍镇山再骂：「表厚，盾更厚！莫长老一句，比表硬！」弟子围殿外，围得像听一场定场，定场不在拍桌，在问魂。

风腥，云低，像秦氏简上的「夜袭」二字，已贴到东脉田埂。表可以捧，根不能拔——下一声拍桌，该宗主了。

周德海笑容僵，揖手退半步：「莫长老……言重了。」「不重。」莫长春道，「重的是表。表厚，心薄。」霍镇山抱拳，声洪：「莫长老这一句，比盾还硬！」

莫长春望东脉方向，像望一场将到的夜。风腥，云低，像秦氏简贴在天上。

围观弟子在殿外踮脚，踮得像等周德海再答——答不上来，表就成废纸。有人低声：「第七年，谁守田？」另一人答：「弟子会老，田不会。」

延迟的爽点不在拍桌，在问魂那一问——问完，和字薄一寸，战意厚一寸。顾小满在杂役堂听见消息，冷眼压回孙福旧党嘀咕。魏无涯扇合，低笑：「第七年，谁守田？好问。」

秦商言在客厢外听见，笑收半分，像终于认这是一场不对等的开价：「青岚问魂，赤焰才急。」

莫长春望东脉，风腥，云低，像秦氏简贴在天上。大因果还在赤焰夜袭那夜——表可以捧，根不能拔。
