# 第十六章 周癞跟风

村会日子定了，村里的风先紧起来。

韩泥早起编筐，胳膊上的布还没拆。藤条刮皮、泡软、绕膝，绕得结实。他不慌——慌了结松，筐一塌，十二文就没了。晌午前编完两只，用指节敲敲底，声闷而实，才配去集尾换钱。

担着筐过集尾，买主是个挑盐的老熟人。他打量筐沿，捏了捏底：「编得紧。不松。」

「紧才装得住药。」

买主从腰里摸出铜钱，数了两遍才递过来：「十二文两只。晦气翁的手艺，倒实在。」

韩泥收下，腰布沉了一截。钱不暖身，能买米；米在锅里，人就还能活。他不回嘴，担起空筐，把十二文也编进脚程里。

回村道上，有人指指戳戳：「晦气翁还卖筐？」另有人压低嗓子：「明日辨诬，他要是倒了，筐也没人要。」韩泥听见了，步子不乱。乱了像认谣；不乱，十二文才算真正落到腰里。

午后过井台，周癞子醉了，红着眼晃过来推他肩膀。推不动——那是担筐练出来的劲，不是酒劲。

「晦气翁！」周癞子嗓子哑，「赖管事说，你克全村！克到村会都要开！」

韩泥拎着桶，水没洒：「村会开，开的是谣，不是我。」

周癞子愣了半息，骂：「叶寡妇送你汤，睡没睡，谁知道！」

这话脏，脏到连醉鬼都信自己的嘴。韩泥不辩睡没睡。辩，成戏。他只道：「睡没睡，天知道。汤热不热，我知道。」

周癞子的手悬在半空，像没料到他不躲。不躲像心里有底。有底就压人，压到醉鬼的手落不下来，只好骂句「疯子」，晃远了。

井台边周嫂递眼色，周癞子又来推。还是推不动。「推我，不如推井绳。」韩泥道，「绳断，你渴。」

周癞子啐他脚边，啐不到人，像唾沫先羞了自己。周嫂帮腔：「韩泥，你硬，硬到全村跟着受罪！」

韩泥拎桶：「受罪的是病。病等着抓药，不等你的棒槌。」

周嫂棒槌握紧了，到底没砸——砸了，里正记名；村会前胆先缺一截。村人看着，见晦气翁没倒，倒的是周癞子的步子。步子乱，像跟风踩空。

马春花远处看了两息，没帮腔——帮腔要胆子，胆子被村会钩住了，就少一截。赵绣春却近前，嗓子尖：「村会辨了，你也跑不了！」

韩泥拎桶，水仍不洒：「跑不了，就不跑。站给全村看。」

赵绣春嘴唇动了动，没再尖——尖要人多，人散一半，尖就短。井台边童谣丫被娘拽住，还探头，探着探着念半句又卡壳：

「叶家铺，晦气铺——」

娘捂住她嘴：「再念，回家挨揍！」

童谣丫眼睛圆着，圆里有怕，也有好奇：好奇晦气翁怎么还不倒。韩泥路过，不停脚，只道：「念完再念，嘴累。累的是嘴，不是铺。」

童谣丫挣出半声：「翁……狼……真打？」

「真打。」韩泥道，「牙怕棍，人不怕谣。」

娘拽紧她远去。韩泥回屋，心口那几笔恩仍在，赵辱旁他又记下一辱——辱不写人名，写「风」。风无姓，风有痕；痕在心里，不在口上。

「周癞子，记着。」他低声道，「跟风的人，风停了，先摔。」

他把水担回屋，倒进缸里一半，剩一半备着煮粥。水花一响，倒比井台骂声干净。骂声像风，吹过就散；水留下来，才够活命。他擦了擦掌心的井绳印，印红，手却稳——稳了，明日坐末席才不抖。

夜里沈家咳两声，咳得轻，像芽儿娘还活着。他合上袄，望叶家药铺方向——灯弱，仍亮。亮着，叶汤就不倒。风过村道，童谣丫不念了，不是怕翁，是被娘拽回。风仍吹到告示桩上「连坐」二字，字黑，像等下一张更醉的嘴。

韩泥不追风。追风像少年。他只记辱，等村会。这第二笔辱比赵辱短，却更硬——硬在跟风的人姓名不写，影子已经在。影子在，村会上一并算。

陈姑远处咳了两声，像提醒：还活着的人，别被风带倒。韩泥「嗯」一声，应给自己。井台边周癞子又晃来想推，见桶满了推不动，自骂「晦气」，晦气落在自己脚面上。

风停一瞬，叶家灯仍亮。夜里叶铺秤又落一声，远过周癞子骂街，也远过赵绣春门内摔碗。韩泥在屋里听，不近——近像牵扯；听着就够，够知秤还在，够知村会前叶汤不倒。

马春花远处嚷：「村会开，看你还硬！」

韩泥开门缝一线：「硬的不是我，是汤。汤热，你嗓尖也尖不凉。」

马春花噎住，嚷声收回去，像风又断一截。韩泥低声道：「辱二记着。恩几笔，一笔不能浅。浅了，村会白开。」

风再起时，村人分成三拨：一拨跟着赵绣春骂，一拨跟着里正等，一拨只绕叶铺走。绕铺的人停在门阶外，低声问：「叶掌柜，明儿还开？」

叶青禾在帘内答：「开。病不等村会。」

问的人「嗯」一声，步子快了，像松了半口气——气一松，风就短一寸。周癞子醉醒又晃到告示桩，读「连坐」，舌头打结：「连……连我？」

韩泥路过，独眼平：「连不连，村会判。你先醒。」

周癞子啐自己脚面，像自辱。赵绣春想借风再尖，尖到一半，见里正锣在巷口，又咽回去。锤还没落，人心已定一半。

叶铺秤再落，落声压住一阵风。压得住风，压不住村会——村会来了，恩和辱一并算。韩泥低声道：「风停，摔的是跟风的。站住的，是记恩的。」

巷口，昨日还嚷「晦气铺」的汉子低头，把一只竹篮鸡蛋放在叶铺阶前，声很小：「昨嘴贱……叶掌柜，还开铺不？」放下就逃，逃得像跟风先摔一截胆。

叶青禾隔帘朝韩泥方向低声说：「蛋我收。铺还开。村会上，少一张邪嘴。」

韩泥路过不停，只道：「胆摔了，汤别砸。砸秤的，帖上见。」

叶青禾把竹篮收进帘内，秤砣又落一声，落得短。短里像说：蛋收下了，铺不开邪——开的仍是药。韩泥听见了，脚步更实一分。实不是硬，是风里终于有人先摔了自己的嘴。

巷口还有个女人拉住汉子袖子嘀咕：「你昨儿不也跟着喊？如今赔蛋，赵家听见了咋办？」汉子脸红：「听见就听见。娃还要抓药。」女人不说话了，只把袖口攥紧。攥紧像怕风再把人卷回去。

张麻婆扫帚扫过巷，骂：「再跟风，我先打腿！」

周癞子醒了半分，脸涨得像昨日推不动的那桶水——推不动，就摔自己的胆。韩泥不笑。笑轻。他只道：「明日巳时。帖在，脚就站住。」

丑时末，沈家灯芯跳了一下，像芽儿娘还醒着。韩泥听着叶铺秤声，仍不近。赔蛋的汉子胆先摔了，他的脚还站着——站着，等锤。

盐担老汉从阶下经过，粗声压低：「周癞子再推你，公审上我作证——推不动的，是谣，不是人。」韩泥没点头。点头像拉帮。他只道：「你记你的。我记汤。」

老汉「嗯」一声远了。远里风还吹，吹不到他袖里那十二文——钱会花掉，记不会花掉。

夜里门外脚步近了又远。不像砸秤的胆，像巡夜人的锣柄擦过墙角。韩泥编完最后一道藤棱，收筐靠墙：「明日巳时，祠堂见。谣开成风，风停了，谁跟谁摔，公审也会记一笔。」

他合上眼，手按坛沿。瓮没醒，他不喊。五十八岁，绝户，晦气翁，还住在村西最漏的屋里。村狗叫一声又哑了，像全村都在等那个日子。手实了，筐实了，汤才能立。帖贴在那儿，明日巳时，祠堂村会，叶汤得立给全村看。
