# 第一百零二章 兽潮近

黄昏。

兽吼不再远，远变成近，近变成栏外草动。草动，像手，像爪，像潮头探界。

韩泥东角立，火把照脸，脸丑，丑不要紧；丑，才不被派去墙心。墙心，是内门；内门不要紧，角才实。

实里，甜腥扑面，扑面他蒙条，条湿，袖囊灰香在指间。指间不要紧，指稳，才撒得匀。

「撒！」铁无言喊。

韩泥撒灰香，撒东角一线，一线白，白像雪，雪里苦。苦不要紧，苦清甜毒。

毒退半寸，半寸够了；够了，小兽嘶声低一线。低，像活。

赖福退半步，半步不要紧；退，是怕，怕不要紧，怕也记——记谁稳，谁飘。

墙外人影动，动像兽。兽影 primera 贴界，贴不要紧，贴的是瘴，瘴厚，厚要清。

韩泥再撒第二囊，囊尽，尽不要紧；尽，还有第三囊。第三囊，留至中。

铁无言侧声：「近处了。别省。」

「不省。」韩泥说。

不省，不为勇，为栏。栏在，丙九才不被连坐。

白织月在内墙，声远：「外门挡，杂役别乱！」

韩泥不答。答，费舌；舌要留来喊角漏。

角未漏。未漏，像手稳——手稳，是丑翁的本事。

夜，兽影退又进，进像试界。试不要紧，试实了，明日就至。

他回棚半刻，席下按坛，低声道：「近了。撒过了。」

坛腹温，像应「近」字。近，不退；退不在路。

邻床抖：「我听见爪刮石。」

「听见不要紧。」韩泥说，「手稳，栏在。」

丑时，静半刻。香落齐，掌纹又温一线——温，像境微动，动不要紧；动，仍五层，五层够守。

赖福查夜，火折照他袖：「又上山了？」

「守角。」韩泥声平，「角在，人在。」

赖福啐，啐完走。走不要紧，近，已至眼。

更鼓尽，远岭火起，火像信号。信号真，真在「兽潮近」。

他独眼平，低声道：「近了。下一更，压。」

压，是气；气在胸，不在嘴。嘴闭，手开。

黄昏，兽吼不再远，变成栏外草动。

韩泥东角立，火把照脸。脸丑，丑才守角，不守墙心。

韩泥撒灰香，撒东角一线。毒退半寸，小兽嘶声低一线。

赖福退半步，退不要紧；退，是怕，怕也记谁稳。

白织月内墙，声远：「杂役别乱！」

角未漏。未漏，像手稳。

兽影退又进，进像试界。

第三囊留至中，留不要紧，留的是续。续在105，续在106，续在107钩108。钩108，落在清瘴丸；丸在108成，成在并肩，成在正。

他回棚半刻，编筐半刻，藤刺扎掌，刺浅。浅些，沿才净；净了，坛才藏得住。藏得住，丑时仍静，静里掌纹温一线，温像五层实，实而不飘。

近处第二更，墙外人影动，动像兽。兽影贴界，贴不要紧，贴的是瘴，瘴厚，厚要清。

第一兽扑障，扑轻，轻不要紧，矛尖稳，稳刺，刺退。退兽，是小胜；胜在肩，不在嘴。

同批少年脸白，脸白不要紧，韩泥只按木牌——牌在，名在，恩不断。

黄昏第二更，甜腥扑面，扑面他蒙条，条湿，袖囊灰香在指间。指间剩第三囊——第三囊，留至103。留不要紧，留，才压得住。

「「撒！」铁无言喊。」答，费舌；舌要留来喊角漏。角漏，比内墙荣耀小，比活命大。大不要紧，大在栏里小兽还喘着。赖福过，火折照脸：「丑脸别挡道！」韩泥让半步，让完仍立角。立角，不在脸，在手。手稳，撒才匀；匀，毒才退。同袍问：「你还有几囊？」「够。」韩泥说，「够到压。」够，不在多，在准。

准在鼻下，鼻下一动，知毒浓淡——浓，多撒半指；淡，省着用。更鼓沉，韩泥按木牌，牌凉贴胸。下一更，压。兽潮近，近在草动，近在兽吼不再远。远变近，近变栏外一线甜腥——甜腥扑面，扑面他蒙条，条湿，袖囊灰香在指间。指间第三囊，留103；留不要紧，留，才压得住。「撒！」铁无言喊。

韩泥撒东角一线，一线白，白像雪，雪里苦。苦清甜毒，毒退半寸，半寸够小兽嘶声低一线。低一线，像活；活，是小胜。白织月内墙，声远：「外门挡，杂役别乱！」乱不乱，在手。手稳，撒才匀；匀，角才稳。角稳，103才压得住——压，是气，气在胸，不在舌。舌要留来喊角漏，漏比荣耀大。

赖福过，火折照丑脸：「别挡道！立角，不在脸，在桩。桩稳，栏稳；栏稳，内墙才稳。稳，不在白织月袖风，在灰香尽未尽。」「够。」够到压，压到103。103压，104至——至在远，今夜近实了。近实，不要紧，近实，手才不飘。飘了，障破，栏倒。试界实了，明日就至。下一更，压。」压，角还在，铁无言还在，经页还在。

还在，就不滚。兽潮近，草动像爪，爪探界。探界不要紧，探实了，103才压。压前，102撒——撒东角一线，灰香苦，苦清甜毒。毒退半寸，小兽嘶声低一线，低像活。白织月内墙：「别乱！角稳，内墙才稳——稳不在袖风，在桩。桩是韩泥，韩泥不让道，让半步仍立角。

第三囊留103，留不要紧，留才压得住。压，是气，气在胸。胸要稳，稳像编筐勒到第四匝——匝匀，筐不散；角匀，栏不倒。倒，99白备，100白守。试界实了，明日至。更鼓沉，按木牌，近了，下一更压。压，铁无言在，经页在，坛温一线在怀。在，就不绝。

黄昏第三更，兽吼不再远，远变草动，草动像爪。爪探界，探不要紧，探实了，103才压。压前，102撒——撒东角一线，白像雪，雪里苦，苦清甜毒。毒退，小兽缓，缓像活。活，胜在活，不在嘴。白织月内墙「别乱」，乱不乱，在手；手稳，角稳，内墙才稳。

赖福过，火折照丑脸，照完走，走也记名——名在簿，簿上「近了」二字，二字真，真在角还在。还在，第三囊留103，留不要紧，留才压得住。压，是气，气在胸，不在舌。压，铁无言在，经页在，坛温在怀。在，就不绝；不绝，103在前，104在远。远不要紧，手在，桩在，就不滚。

兽吼贴界，贴不要紧，贴实了，103压。压前，102撒尽第二囊，囊尽，第三囊留。留不要紧，留才压得住。压，铁无言喊，韩泥挪，挪不移角——角在，桩在，就不滚。滚在钱戾衡脚，脚在远，守在角。

毒退半寸，小兽嘶声低一线；低像活，活是小胜，胜在活，不在嘴。

白织月内墙：「外门挡，杂役别乱！手稳，角稳，内墙才稳。赖福过，火折照脸，照完走——走也记「近了」二字，二字真，真在角还在。还在，第三囊留103；留不要紧，留才压得住。」「够到压。」够，不在多，在准；准在鼻下，鼻下一动，知浓淡。浓，多撒半指；淡，省着用。省着，103才接得住。

接得住，104才至，105才潮中，106才续，107才钩，108才清瘴。清瘴在远，102只撒。撒尽第二囊，更鼓沉，按木牌，近了，下一更压。在，就不绝。

第四更，兽吼贴界，贴不要紧，贴实了，103才压。压前，撒尽第二囊——尽不要紧，尽还有第三囊。第三囊，留103；留，才压得住。压，气在胸，手在角，铁无言在侧。侧在，就不散。散，赖福喊；喊，韩泥不应，应的是撒。撒匀，毒退，小兽活。

兽潮近实，近实不要紧，近实，103才压。压前，102撒尽第二囊——尽不要紧，尽还有第三囊。第三囊留103，留才压得住。压，铁无言喊，韩泥挪，挪不移角。角在，桩在，就不滚。滚在远，守在角，清瘴路在。路在，经页在怀，怀贴坛，坛温一线，像应「近」字。近字落齐，更鼓沉，按木牌。

按牌，牌凉贴胸，胸要稳，稳像编筐勒到第四匝。匝匀，筐不散；角匀，栏不倒。不倒，103才压，104才至，105才潮中。潮中，更尽，手还在。还在，就不绝。

手在，就不绝。

验息后，他理怀内册页，页黄，字丑，丑而真。
