## 一、体系总览

《凡人修仙传》修炼体系 = **三界飞升** × **境界阶梯** × **灵根定资质** × **丹器阵符火补足**。

| 维度 | 凡人流特征 |
|------|------------|
| 节奏 | 慢、稳、资源战；闭关 alone 难以破境 |
| 资质 | 灵根决定速度；伪灵根可成道，靠机缘与算计 |
| 战力 | 境界为本，符阵丹器可**有限越级**，硬扛有代价 |
| 飞升 | 人界封顶化神 → 灵界续修 → 渡劫升仙界 |

```
人界：炼气 → 筑基 → 结丹 → 元婴 → 化神
                    ↓ 破界飞升
灵界：化神(稳固) → 炼虚 → 合体 → 大乘 → 渡劫
                    ↓ 渡劫成功
仙界：真仙 → 金仙 → 太乙 → 大罗 → 道祖
```

---

## 二、三界结构

| 界域 | 修士常规上限 | 灵气/资源 | 叙事功能 |
|------|--------------|-----------|----------|
| **人界** | 化神 | 稀薄；灵石、坊市、宗门秘境 | 入门、宗门政治、乱星海式争宝 |
| **灵界** | 大乘 → 渡劫 | 浓郁；万族、界面战争、跨界材料 | 换天、种族博弈、大乘前总账 |
| **仙界** | 道祖 | 仙灵之气；仙门、仙域、法则之争 | 证道、余烬、全书顶点 |

**飞升铁则**：

- 人界**化神后期/巅峰**可尝试破开界面，进入灵界（常伴随界面风暴、灵压反噬）。
- 灵界飞升者落地时多为**化神**，需先稳固再破炼虚。
- 灵界**大乘巅峰**面临**渡劫**；成功则升**仙界**，起点为**真仙**。

## 三、境界阶梯（全书）

### 3.1 小境通则

除炼气按**层数**外，各大境常见小境为：

**初期 → 中期 → 后期 → 巅峰（大圆满）**

部分境界在「后期」与「巅峰」之间还有**假丹/假婴**等不稳态（结丹、元婴期常见）。

### 3.2 人界（炼气 → 化神）

| 大境 | 小境 | 寿元约 | 地位概要 |
|------|------|--------|----------|
| **炼气** | 1～13 层 | 百余岁 | 引气入体；杂役/外门主体 |
| **筑基** | 初/中/后/巅峰 | ~200 年 | 凝液成基；内门门槛 |
| **结丹** | 初/中/后/巅峰 | ~500 年 | 凝丹成道；宗门真传/长老候选 |
| **元婴** | 初/中/后/巅峰 | ~1000 年 | 元婴出窍；一峰之主、乱星海霸主 |
| **化神** | 初/中/后/巅峰 | ~2000 年 | **人界天花板**；太上长老级 |

> 人界极少出现炼虚修士；化神修士已是顶流，多数宗门镇派老祖不过元婴。

### 3.3 灵界（化神稳固 → 渡劫）

| 大境 | 小境 | 寿元约 | 地位概要 |
|------|------|--------|----------|
| **化神** | 初/中/后/巅峰 | 延续人界基数 | 飞升落地；重新积累 |
| **炼虚** | 初/中/后/巅峰 | 更长 | 炼化虚空；灵界中坚 |
| **合体** | 初/中/后/巅峰 | 更长 | 神魂肉身合一；一方诸侯 |
| **大乘** | 初/中/后/巅峰 | 更长 | 灵界顶尖；临近飞升 |
| **渡劫** | 一至九重 | — | 扛天劫；成败在此 |

### 3.4 仙界（真仙后）

| 大境 | 说明 |
|------|------|
| **真仙** | 入仙界起点；仙籍、仙门、仙域法则 |
| **金仙** | 仙界中坚战力 |
| **太乙** | 高阶仙修（如玉仙等变体） |
| **大罗** | 顶级战力；掌法则雏形 |
| **道祖** | 法则极致；全书战力顶点之一 |

> 仙界篇篇幅远短于人界+灵界，但每一境跨度极大，常以**仙劫、法则感悟、道心**为主要矛盾。

## 四、灵根体系

| 类型 | 属性数 | 修炼速度 | 说明 |
|------|--------|----------|------|
| **天灵根** | 单属性 | 最快 | 各大宗门争抢 |
| **异灵根** | 变异单属性 | 极快且强 | 风、雷、冰等 |
| **真灵根** | 2～3 属性 | 中等 | 宗门外门/common |
| **伪灵根** | 4～5 属性 | 极慢 | 韩立式凡人流起点 |

**本作映射**：韩泥为**末席伪灵根 + 丑骨凡躯**，不改根骨，靠丹道、符阵、鸿蒙九劫瓮与算计补足。

## 五、突破机制

### 5.1 五要素（凡人流核心）

| 要素 | 说明 |
|------|------|
| **修为满** | 丹药、灵脉、时间积累 |
| **破境丹** | 筑基丹、结丹丹、育婴丹、化神丹等（品阶对应境界） |
| **护关** | 阵法、洞府、护法、队友拖时间 |
| **天时** | 灵气潮、秘境、界面法则窗口 |
| **天劫** | 筑基后渐严；渡劫为最重 |

### 5.2 天劫概要

| 阶段 | 天劫类型 | 强度 |
|------|----------|------|
| 筑基 | 心魔/小雷 | 低 |
| 结丹 | 丹劫 | 中 |
| 元婴 | 婴劫 | 高 |
| 化神 | 神劫 | 很高 |
| 炼虚～大乘 | 大天劫 | 极高 |
| 渡劫 | 九九重劫 | 全书人间/灵界最高 |

**凡人流铁则**：宁可晚破、不可浪破；破境前备足遁符、护关阵、备用丹。

### 5.3 本作破境硬条件（执行层）

见 `02` §一、`11` 因果系统：

```
修为满 + 丹药/真火/护关人 + 天时 + 阳因≥阈值
```

## 七、道具品阶

### 7.1 原著常见叫法（人界→灵界）

| 品阶 | 适用境界 | 特征 |
|------|----------|------|
| 法器 | 炼气～筑基 | 低阶，可交易 |
| 法宝 | 筑基～结丹 | 认主，威力随祭炼提升 |
| 古宝 | 结丹～元婴 | 上古遗留，威能诡异 |
| 灵宝 | 元婴～化神 | 通灵，神识认主 |
| 玄天之宝 | 化神～炼虚 | 界面级至宝，难得 |
| 通天灵宝 | 合体～大乘 | 灵界顶流 |

### 7.2 本作九阶（原创命名 · 与境界绑定）

> 详表 `02` §九、`12` §2.0。自低到高：

**凡品 → 灵品 → 宝品 → 玄品 → 地品 → 天品 → 圣品 → 仙品 → 道品**

口诀：**凡灵宝玄，地天圣仙道**。凡人原著 **法器→法宝→古宝→灵宝→玄天之宝→通天灵宝** 与本作九阶数值对齐、命名独立。

## 九、越级战斗规则（凡人流）

| 越级类型 | 条件允许 | 典型手段 |
|----------|----------|----------|
| 炼气 → 筑基 | 符箓+阵盘+地利 | 不可硬扛 |
| 筑基 → 结丹 | 真火+二品丹+队友拖时间 | 逃杀为主 |
| 结丹 → 元婴 | 地利+舟+敌方内乱 | 乱星海式 |
| 元婴以上 | 最多越一小阶 | 需古宝/灵宝+代价 |

**本作公式**（`02` §二）：

```
有效战力 = 境界基础 × 道具阶 × 符阵加成 × 丹药状态 × 遁逃准备 × 道侣同心(≤1.15)
```
