# 第七十一章 别来丑

三年约立过第七天，侧门驿亭来了驿卒。肩湿着，风从泥岗那头刮来，带着旧灶烟。

驿卒一嗓子喊开：「丙九韩泥！泥岗来字！」

韩泥正在兽栏清粪，铲子顿了一顿。不是字烫，是字后头那个人烫——叶青禾。管事弟子验封，蜡印浅却真：一角「叶」字，还有里正那枚旧印。「念。」韩泥说。

驿卒念得急：「韩叔，别来丑。青禾。」

四字落进耳朵，像碗沿热气退了一寸。热退不要紧，字还在。同批少年挤眉弄眼：「叶家丫头嫌你丑？」

韩泥不辩。辩费嘴，嘴要留着记愧。他把字条塞进贴身衣袋，不给人瞧——瞧见了，谣言又会生腿。赖福晃着秤杆，眼细成缝：「泥岗来字？来字也晦气。」

韩泥只当没听见。接茬就会惹事，丙九一整排都跟着饿。他摸了摸衣袋，纸粗糙，糙得像当年叶丫头送汤那碗边。「别来丑。」他低声念给自己听——不来，是怕连累；丑，是别把她的名蹭脏。名干净，他更不能去叫门。

午后栏里仍腥。腥不怕，手稳才怕掉。刘婆端粥来，仍多舀半勺：「泥岗来字了？」

「来了。」韩泥说，「四个字。」

「四字怕什么。」刘婆哼一声，「手稳，饭稳。饭稳，恩就不断。」

韩泥点头。谢说轻，恩说重——粥里有，字条里也有。字条没套客套，却有礼：礼是愧，愧是叶丫头替他挡过的谣、挨过的辱。

铁无言抱臂立在栏边，嗓子压低：「字别贴脸上。贴了，像求人。」

「不贴。」韩泥应，「记着。」

秦霜从远廊过，木鱼没敲，只丢一句：「心别邪。邪了，鼻下的香就飘。」

韩泥不答。心里七笔账各搁各位：泥岗那六笔沉着，宗门刘婆这一笔烫着；如今又开第七笔——愧。愧不重礼重名。名是叶青禾，名也是「别来丑」四个字。丑可以长在脸上，不能长进她名里。心在「还」，还就得先活长。

回棚后，他掀席半寸，不碰坛，只把字条摊在膝上歇一息。纸黄，字稚，稚得真——真在「青禾」，也真在「别来」两个字像割手。「丫头，叔记着。」他低声道，「丑着活。活长了才还得上。你别来，叔去。」

邻床悄声问：「你真不恼？」

「恼在愧里。」韩泥说，「不恼字。字是护我，不是辱我。」

叶青禾三个字只进愧线，不抬到舌头上。舌头一碰，名就脏；名一脏，那碗烫汤就算白喝。他编筐时藤条绕指，像把「别来丑」绕进掌纹——绕进掌，不绕进廊下笑里。

赖福堵在棚口，晃木牌：「丑杂役，叶家丫头跟你啥关系？说清楚，记过。」

「恩。」韩泥只答两个字，「恩在还，不在讲。」

刘婆提水路过，水溅半滴落在筐沿，湿了一线。那一线让他想起叶汤碗沿的热。众目问丑，他只记三样：记别来，记别污名，记手稳。玉牌在怀，三年约在怀，字条也在怀。仇与愧都凉，只有恩烫。

傍晚他托沈枯芽带一句口信回去。没字，只有半块干饼，饼边刻了浅痕：「好。」

沈枯芽怯怯点头：「叶姐……收下了。她说，叔丑着活，别来。」

「不来。」韩泥说。

不来才叫还；一去就叫连累。午后从坡下望，叶家小舍远着，门缝没亮灯——不亮，正因为「别来」。白织月从远廊过，袖风冷，没停。没停也算赢一寸——赢在活着，不在嘴上。

赖福远远讽一句：「叶家也嫌你晦气。」

韩泥不接。忍不是怂，是把刃磨暗：对着三年约，对着「别来丑」，对着以后还恩那一天。编筐末张单时藤刺扎掌，他偏了偏手，刺得浅些。浅些沿才净，沿净坛才藏得住。筐钱买米，米在，恩才还得上——还得上，才对得起叶丫头这一刀字。

更鼓二响，沉着。他合眼不睡实，怀里三样并着：三年约、别来丑、仍末。仍末不怕，手还在。手在，就不滚。他低声道：「别来丑。叔丑着活。活长了，门缝灯才照得到你。」

夜巡管事照席角，见净沿不见宝。不见也好，净才要紧。「守沿？」管事随口问。

「不拜。」韩泥说，「记着。」

散场后他在药渣场角分拣半刻，指稳，鼻下先静一息。静不是忘愧，是把愧按进恩里。字条没署名，角上却沾着一点泥岗灶灰——淡，像叶青禾在灶前写字时袖口蹭上的。韩泥拂去，像拂字外辱，留字里恩。

「丑」字是护：护名，护路，护她不必因叔再沾谣。沉丹宗有谣，泥岗也有；丑叔不去，她才能站正。他摸衣袋里三样凉物，坛底却温着一线。温应着恩，也应着将来——还热汤、还省亲、还叶家一席不丢人的面。

赖福脚步在廊外又停半息：「丑杂役，字条也晦气？」

「在。」韩泥说。

在，就不滚。愧扎实了，才配听坛开口，才配挨「煞骨门」那一问旧债。他对着坛低声：「叶丫头护名，叔护坛。坛在，名在，路就不绝。」

过丹房外廊时，有个少年拽他袖：「韩叔，你真不回泥岗看看？字都来了。」

「看了，名就脏。」韩泥摇头，「脏了，那碗汤就算白喝。」

少年挠头：「就四字，你记这么深？」

「深。」韩泥说，「四字能护一个人名。护名的字，比金贵。」

少年不懂「贵」，却懂「护」，闭嘴走了。韩泥立廊角半息，风从侧门灌进来，仍带着泥岗灶烟味。味旧，旧得像村头石碾。碾过的不是粮，是他脸上那层丑壳。丑壳可以留，叶丫头的名不能沾。

管事弟子来收筐，翻看沿口：「编得倒稳。稳归稳，别因字分神。分神，麸皮扣一半。」

「不分神。」韩泥应，「手在筐上。」

「手在筐上最好。」管事哼一声，「泥岗字再烫，烫不过丙九这条命。」

韩泥不辩烫与不烫。辩了像把愧摊开给人看。他把字条往怀更深处按了按，按得纸角发软。软不要紧，字还硬——硬在「别来」，硬在「丑」。

铁无言从灰堆后头出声：「字收妥了？」

「妥了。」韩泥说。

「妥就别摸。」铁无言道，「摸多了，脸会软。脸软，手就飘。手飘，渣场不收你。」

「不摸。」韩泥应，「只记。」

铁无言看他一眼，像验一句真话，点头走了。走后韩泥才把灰堆里一点辛拨出来，辛正，入纸包。包进怀，跟字条隔着一层布——隔着也好，恩一条线，愧一条线，别搅。

日头偏西，他过侧门驿亭再站一息。亭风硬，硬里仍有泥岗方向一线。驿卒扛空袋走过，随口问：「还回不回信？」

「不用。」韩泥说，「饼刻了『好』，她懂。」

「懂就行。」驿卒笑笑，「泥岗丫头字少，情重。你这种丑脸，得着了。」

韩泥不接「得着」。得着的是护，不是脸。脸丑着，护住了，才叫还得起。他摸怀，玉牌凉，字条温——温不是纸热，是人心热。热得他独眼发涩，涩完又平。平了，才回棚编最后一捆藤。

棚外更鼓又响一记，沉。他指腹摩过字条边角，像摩过碗沿那圈旧热。热还在，人就不能贱卖那四个字。

坛不应字，只温。温像在说反面：别来丑，那就来还；迟可以，不算不算。天边将白，他把字条塞回怀更深处，像把愧摁进心记第七笔。第七笔此刻不亮，亮在往后。外头风硬，风里仍有泥岗那头的味道——旧，却真。真得让他知道，下一夜丑时，该守沿了。守沿守到那声哑，坛底才会开口。
