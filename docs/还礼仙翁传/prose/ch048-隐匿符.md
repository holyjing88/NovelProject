# 第四十八章 隐匿符

雨后，顾小满登门，未跪，立。

双手捧符，黄纸朱纹，比除魔符更深一重，像把夜色裁进符里——她裁了七夜，指节割破，血染纸边，染得像半块灵石的延续，染得像杂役堂里那些年的苦，终于肯换一种还法。

「长老，小满只能做到这些。」

莫长春接符，符温，温得像活命。他不看符纹先看她：「眼睛还亮？」

「亮了。」顾小满道，「雨夜那道金光之后，小满看得更清——赤焰探子换岗，田埂火把，侧径空隙，都进眼里，像地图展开。」

「地图展开，」莫长春点头，「比杀一人有用。」

「可小满……」她摇头，「比起半石、除魔符，这符太轻。」

「轻？」他笑，「宝品隐匿符，裁纸如裁命。你以血裁，还叫轻？」

顾小满怔。她不懂宝品不宝品，只懂长老递半石时，她跪过；递除魔符时，她抖过；今日捧符，她竟敢不跪，像被那道金光洗直了腰。

莫长春将符收入袖中隔层，与簿相贴，簿一烫，烫得像因果涨了一截，涨得沉，却不吓人。

「够了。」他道，「礼不在重，在心真不真。心真，簿才记。」

顾小满抬眼，忽然道：「小满能走暗线了。赤焰探子，小满能跟；周长老侧门不开，小满也能报。」

「跟可以。」莫长春递她一枚薄护身符，非宝品，贴肉，「别跟丢命。暗线走侧面，别走正面。正面是霍老头和宗主的戏，侧面才是杂役的戏。」

她接符，泪落，笑也落：「小满记恩。」

门外，赵玄机路过，停半息，听「宝品」二字，指节叩铁牌，一声。他没进门，只拂袖走，走前目光扫过顾小满背影——妒吗？少一点了；怕吗？多一点了：怕九月老头连杂役都能养成暗线，怕自己的心还在私囊里。

林挽月远立廊角，唇动，像想说对不起，终未近。

莫长春关门，簿上浮字：

**隐匿有时，情报成形。**

他念，像延迟层落账：「小因果够了。大因果——夜袭子时，看这张符能不能送命回来。」

风停雨住，东脉方向云低，像夜还未来，牙已露。

廊下，弟子窃语：「顾小满献宝品符……杂役也能回赠？」

赵玄机路过，停半息，指节叩铁牌，一声，像把妒意敲实，又敲松一寸。他没进门，只拂袖走，走前目光扫过顾小满背影——怕吗？多一点了：怕九月老头连杂役都能养成暗线。

林挽月远立廊角，唇动，像想说对不起，终未近。

顾小满回杂役堂，把符贴身藏好，藏得像藏半块灵石——恩在掌心，还不了嘴，还得了命。

大因果还在夜袭子时，像妖骑，像杖剑——隐匿符只是小回赠，挽月反戈才是大还礼的前奏。

顾小满怔。她不懂宝品不宝品，只懂长老递半石时，她跪过；递除魔符时，她抖过；今日捧符，她竟敢不跪，像被那道金光洗直了腰。莫长春将符收入袖中隔层，与簿相贴，簿一烫。

莫长春接符，符温，温得像活命。他不看符纹先看她：「眼睛还亮？」「亮了。」顾小满道，「雨夜那道金光之后，小满看得更清——赤焰探子换岗，田埂火把，侧径空隙，都进眼里，像地图展开。」

「够了。」他道，「礼不在重，在心真不真。心真，簿才记。」顾小满抬眼，忽然道：「小满能走暗线了。赤焰探子，小满能跟；周长老侧门不开，小满也能报。」她接符，泪落，笑也落：「小满记恩。」莫长春关门，簿上浮字：**隐匿有时，情报成形。**

围观弟子在廊下踮脚，踮得像等长老再收一张符——收的是暗线，不是宝光。有人低声：「杂役也能回赠？」另一人答：「回赠的是路，不是符。」

廊下一人道：「七夜血染纸边，也叫轻？」另一人答：「轻的是杂役的身分，不是符的纹。」赵玄机拂袖走远，走远两步，却停，停得像终于听懂——懂的是暗线在侧面，妒在正面，正面留给他自己咽。

莫长春独坐廊凳上，不坐主位，望东脉方向云低，轻声：「小满以血裁符，裁的是命，不是纸。裁得了，才走得进暗线。」

风停雨住，东脉方向云低，像夜还未来，牙已露。隐匿符入了袖，情报成形——成形了，挽月反戈才肯在堂上把脏字翻过来。大因果还在夜袭子时，像妖骑，像杖剑。
