# 第五十六章 编筐季末

试手前二日，丙九编筐季末。韩泥接末单，藤条硬，硬得像冬里的规矩。

管事盯梢：「末单交不出，试手名单划掉。」

韩泥不辩，只编。编筐是本行，深秋末筐也是本行。筐成，手稳，稳比美重要。

藤刺扎掌，扎旧茧，茧裂，裂口潮，潮不是血。血不碰沿，沿才净。

赖福讽：「丑手编筐，筐也晦气。」

刘婆路过，端粥，稀，热：「手稳，筐就圆。圆，能装柴，能装恩。」

韩泥喝粥，「嗯」一声。热在，像第七笔又实。筐沿齐，齐像秤杆落盘。

管事验筐，指筐底：「底不塌，记功。」

记功，像窄路又宽半分。

午后，符怀暖一丝，护骨寒，手不僵。僵，藤刺偏。不偏，缘才正。

搬柴过半，肩稳，像筐成后的余韵。余韵不在夸，在手还在。

铁无言路过，声低：「季末了，手还在。在，才接后头几日。」

韩泥只答：「在。」

「试手验啥？」

「验手。」韩泥说，「手抖，丹歪，滚。丹歪不要紧，手不能歪。」

铁无言「嗯」一声：「静带到手上。手上静，丹才正。」

同批少年碰他筐：「末单也编得齐？」

「齐不齐，管事验。」韩泥说，「我验手。手稳，筐就圆。」

少年碰筐沿，沿不塌：「晦气筐，倒结实。」

韩泥不辩。结实，像小胜。胜在实，不在嘴。

黄昏，坡下风硬，柱上条在：「后日卯时，试手凝丹。」

后日，比五日条又近一步。近不要紧，要紧是手还在。

管事验末捆，声沉：「季末了，藤刺还会扎。扎了，别硬扛。扛裂，手抖。」

韩泥「嗯」一声：「不硬扛。扛，路断。」

赖福偷懒，管事记名。韩泥不偷懒。克粮，克不走还路。

末藤刺深时，韩泥停半息，停，像听藤条说话。落盘稳，谣压半寸。

夜，坛腹温一线，像应筐成。成，不急醒。醒在缘，不在筐。

他低声道：「季末了，手还在。在，才接血近沿那几日。近，不碰沿。」

坛不应字，只温，温得像「嗯」。外头更鼓沉，沉里，试手条在柱，柱上「后日卯时」。

后日，凝丹试手。试手不是醒瓮。醒瓮在更后，在误触那一滴。

邻床问：「你怕不怕滚？」

「怕。」韩泥说，「怕也记。记着，手就不飘。」

「飘了咋办？」

「飘了，路断。」韩泥说，「路断，粥就凉。」

邻床懂「凉」，闭嘴。

筐是活计，活计扎掌，掌裂在冬，不在沿。

丑时前，裂口渗潮，潮不是血。无血，沿便安。他擦沿，沿滑，沿净。净，像祖父手还在。

刘婆门外咳：「后日试手，别空肚。」

韩泥应：「不空。粥在，姜在记里。」

符怀暖一丝，护手不僵。僵，后日藤刺便偏。

他低声对坛：「筐成了，手还在。还在，才接后头试手，接血近沿，接瓮醒。」

外头风息，柱上「后日卯时」更真。真在试手，真在末排，真在窄路。

季末筐成，底不塌。底不塌，像秤杆回正——回正，手才配后日试手。

后日卯时，试手凝丹。凝丹验手，不验灵色。灵色末，手不能末。

卯时编筐，藤条硬，硬得像冬规矩。规矩在管事眼，也在筐底。底塌，试手名单划。

第一筐成，筐沿齐，齐像秤杆落盘。落盘稳，手才稳。手稳，后日丹才不歪。

第二筐，藤刺扎掌，扎旧茧，茧裂，裂口潮，潮不是血。无血，沿便安。安，才接后头试手。

同批少年偷懒，管事记名。韩泥不偷懒。偷懒，像心飘。心飘，手就抖。

午后搬柴，肩稳，稳像筐成后的余韵。

铁无言验鼻，声低：「鼻下静，带到手上。」韩泥「嗯」一声，只答「在」。

黄昏，柱上条在：「后日卯时，试手凝丹。」条薄，真。真在「仍末」，真在「后日」。

成，不急醒。筐是活计，活计扎掌，掌裂在冬。

偏刺，血找错路。错路，像拜邪。

第二日末单仍编，编，手在，缘才正。管事验筐沿：「沿齐，底不塌，季末记功。」

季末第一日，管事验第三筐：「沿齐，底不塌。」塌了，试手名单划。

编到第四筐，藤刺扎旧茧，茧裂，裂口潮，潮不是血。无血，沿便安。

同批少年碰筐沿：「晦气筐，倒结实。」韩泥不辩。结实，像小胜。

午后第二趟搬柴，肩稳。稳，像筐成练出来的。练出来，才配后日试手。

铁无言第二趟路过：「季末了，手还在。」韩泥只答：「在。」

黄昏第二日，柱上条更真：「后日卯时，试手凝丹。」真在试手，真在末排。

夜第二日，坛腹温一线，像应筐成。成，不急醒。

邻床第二日问：「你怕滚吗？」「怕。」

丑时第二日，他擦沿，沿滑，沿净。净，才接后头血近沿。

刘婆第二日咳：「后日试手，别空肚。姜在记里。」

季末最后一日，筐底不塌。不塌，像秤杆回正。回正，手才配试手。

第五筐成，管事验沿：「齐。」齐，像秤杆落盘。落盘，手才稳。

第六筐，藤刺扎掌，茧裂无血。无血，沿便安。安，才接试手。

搬柴第四趟，肩稳。稳，像编筐练出来的。

铁无言第四趟：「后日试手，带上静气。」韩泥「嗯」一声。

夜第三日，柱上条更真。真在「后日卯时」。

他对坛低声：「筐季末了。末了，手还在。还在，才接血近沿。」

邻床第三日闭嘴。闭嘴，像懂「凉」。

丑时第三日，符暖护手。暖，不燃。燃了，管事查。

季末第七筐，筐沿齐，底不塌。管事验过，记功。宽不要紧，要紧是手还在。

后日前夜，他数柱上条。条薄，真。真在「仍末」，真在「卯时」。卯时，试手凝丹，验手，不验灵色。

编筐末刺深时，他停半息，听藤条响。响闷，闷像更鼓。更鼓里，掌茧裂无血。无血，沿便安。

刘婆季末最后一碗粥，热：「后日试手，鼻下别飘。」韩泥喝尽：「不飘。粥在，静在记里。」

他对坛低声：「筐季末了，掌茧裂无血。无血，沿净。净了，才接后日坡下，接血近沿。」

铁无言路过，声低：「后日试手，静带到手上。」韩泥「嗯」一声：「带上。带上，丹才正。」

第八筐成，季末记功在。在，像窄路又宽半分。宽不要紧，要紧是后日卯时条真。

后日前夜丑时，他数更鼓。更鼓沉，掌茧裂无血。无血，沿净，手才配坡下。

邻床问：「编筐也试手？」「编筐练手。」韩泥说，「手稳，筐圆，丹才不歪。」

铁无言季末最后一嘱：「静气带上坡。

后日条贴柱，韩泥摸条，条薄，真。

筐底不塌那日，管事声沉：「季末记功在。」韩泥不辩，只把手擦干。
