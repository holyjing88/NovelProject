# 第一百一十一章 六层愈压

春末，药山废窑里还剩一点首炼的暖。暖浅，却够他盘坐窑角，掌按坛腹——气在丹田里涨了半寸。

半寸不响，像把培元散那口「正」又咬实一分。六层到了。他睁独眼，脸上不动。动了，赖福闻得出；闻得出，灵石折麸又要加重。

铁无言蹲窑口外，胳膊架膝，声压得很低：「六层？别漏。漏了，白织月来嗅。」

「手稳。」韩泥只两字。

「稳就行。」铁无言顿了顿，「赖福今早改册，三折变两折。别以为他松，他记仇。」

韩泥点头。册上折数，他早摸清——丑杂役末席，灵石兑麸皮，克到骨头里。克不要紧，先活；活长了，才还得了芽儿那半块石。

窑角有渣灰味，混一点青芯余辛。他鼻下轻轻走一遍嗅诀，辛正，不邪。正了，才敢把气往里压；压实了，六层才不像偷来的。

铁无言又扔来半截枯藤：「编筐用。筐实，管事少问；少问，你夜才有空。」

「谢。」韩泥接过。

「谢轻。」铁无言说，「轻点活着，比什么都重。」

卯时散队，丙九列队领饭。赖福立队首，木牌一晃，嗓子尖得像秤针：「韩泥！灵石折麸，两折。两折，省宗门粮。」

队里嗡一声。有人偷眼看他，像看晦气会不会蹭过来。韩泥不接那嗡，只端碗。碗糙，粥稀，稀里却多半勺——是刘婆舀的。他喝得慢，慢得像个只会干活的凡人。

赖福眼细，针尖对着他腕：「暗修？」

「管事看手。」韩泥声平，「手抖才滚。」

「手不抖，灵根也末。」赖福笑得薄，「末等也想偷气？偷气，除名。」

丙九一个少年忍不住嘀咕：「两折也太狠……」

旁边人急忙扯他袖子：「小声！想连坐？」

「连坐不怕，饿怕。」少年嗓子更低，「韩泥哥手不抖，我看他……」

「看什么看。」有人嘘，「被赖福听见，你也两折。」

韩泥听得见，仍不接。辩费舌；舌要留着守坛、按窑温。他把玉牌摸进怀里——牌凉，凉里是钱戾衡那一脚，也是三年约。脚在远，克扣在近；近的先忍。

管事弟子路过，瞥一眼册：「两折？执事批了？」

赖福脸一青，嘴却顺势软半寸：「批了。舌响，香飘；香飘，手就滚。」

「记过了。」管事哼一声，「折归折，手抖照样除名。韩泥，午后渣场，别偷懒。」

「听着。」韩泥应。

午后药渣场，太阳晒得灰发白。韩泥分拣渣灰，灰里一点辛，正，像培元散余韵。他鼻下静着，一字不落，把六层气往里压。旁人骂天骂地，他当没听见。

铁无言抱臂立侧：「六层稳不稳？」

「稳。」

「别飘。」铁无言说，「飘了，折麸册还要洒。」

「不飘。」韩泥铲下不停，「折麸册在，手也在。」

「手在就好。」铁无言低声，「昨夜账我核过，外门丹房那几袋，单干净。赖福想找茬，暂时找不到。」

韩泥「嗯」一声。账干净不是恩，是活路。活路宽半分，气才配往里涨。

渣场尽头有人喊渴，管事弟子喝：「喊什么喊！末席也配喊？」韩泥不抬头，只把一筐渣压实，沿齐了，推到边上。推得稳，稳像把六层又按进一寸。

傍晚，赖福靴底沉沉地踩进渣场：「韩泥，明日搬粪加倍。加倍，赎你偷气。」

「没偷。」韩泥声仍平，「气在坛，不在偷。」

「坛？」赖福眼一眯，「什么坛？」

「腌菜坛。」韩泥独眼抬了抬，「凡人坛。管事看粪清不清，不看坛。」

赖福凑近半步，鼻头乱动：「我嗅得到邪。」

「管事也嗅。」韩泥说，「嗅着了，再罚。」

赖福哼了一声，哼得像没逮住：「凡人坛也晦气。晦气，多搬一日粪。」

多搬就多搬。韩泥手仍稳——稳是小胜，胜在活，不在嘴。他把最后一筐渣灰压实，筐沿齐，管事弟子远远看一眼，没说话。不说话，就是过。

散队后，刘婆端来一碗姜汤，姜薄，热：「两折？」

「两折。」

「折什么。」刘婆哼，「手稳，饭稳。饭稳，恩不断。」

「嗯。」韩泥喝尽，谢得轻。

刘婆盯他腕上茧：「茧厚了。厚了别臭美，赖福专撕厚茧的人。」

「撕不掉。」韩泥说，「手还在。」

「贫。」刘婆却又笑了半寸，「粥我留了口底，你回棚再喝两口。姜多一片，昨晚那片不够。」

「够。」韩泥顿了顿，「够记。」

恩在姜汤里，也在这多半勺粥里——他心里七笔分列，这一笔记「刘婆」，不跟仇搅。仇对着赖福，对着玉牌上那三字，对着钱戾衡；恩对着叶汤、芽儿石、刘婆粥。分列，丹才正。

回漏舍，他把席角压石按实，坛藏严。坛腹温一线，贴掌，像爹当年那句：「坛别扔。」扔不得；扔了无根，六层白涨，韩家也散。

邻床少年悄问：「你真六层了？」

「层在腹，不在嘴。」韩泥说，「嘴响，赖福来。」

「我不说。」少年咽了咽，「两折……你恨不恨？」

「恨。」韩泥独眼平，「恨着先干活。」

「干活能还吗？」少年问，「还灵石，还脸？」

「能还的先还。」韩泥说，「还不起的，记着。」

少年闭嘴。韩泥又摸玉牌——牌凉，仇烫。烫不在这一日，在三年约前头。这一日只手稳、坛在、六层实。

外头风硬，柱上条角白一线，像仇苗又深半寸。他不拔，先活。活长了，才接得住外门复测那一帖；帖在远，克扣在近。

他坐棚口编筐，藤条浸软，绕两圈勒紧，指节敲底，声闷而实。筐紧，管事弟子路过无话；无话，赖福脸青半分。青半分，也是小胜。编筐时肘新针脚不挂袋角——针脚是叶丫头那路细，细了挡风，风不进，臂才能抬粪、抬筐、守窑温。

廊外，赖福脚步停半息：「丑杂役，明日两折，别怨。」

「晓得。」韩泥应。

「晓得什么晓得。」赖福冷笑，「在席下躲着修？我迟早掀开。」

韩泥不答。答了像认。他只听见靴底远了，才把席角再压紧一分。

在，就不滚。不滚，六层才配往后熬——熬查席，熬诬告，熬公示，熬到坊市尾摊那一寸路。

夜半，秦霜远廊下过，木鱼不敲，声钝：「进境急，心别邪。邪了，鼻下香飘。」

韩泥立起身，微微颔首：「心不邪。」

「不邪就好。」秦霜远看他一眼，「丑骨也能走人路。路歪了，散也邪。」

「散正。」韩泥说。

「正了，别急着亮。」秦霜远道，「亮早了，人先把你掐灭。」

「不亮。」韩泥说，「先活。」

秦霜远不再言，木鱼声远了。韩泥回坐，答在心里——心里刃对着赖福，对着玉牌，对着钱戾衡。刃不亮，先磨；磨在活里，磨在每一勺稀粥里。

丑时，他廊下练半刻鼻下静。粟壳香淡，一字不落。落齐了，才低声对坛：「六层，记着。赖福两折，记着。钱戾衡三年约，记着。」

坛不应字，只温。温像听懂了「进」——进一寸，忍一寸；忍一寸，活一日。活一日，刘婆粥多顶一口，叶汤那笔头恩又近一步。近一步，还恩序不乱。

他再摸一次玉牌，低声：「手稳，粒正，试台见。」见字落，坛温一线，像应。应了，路又明一寸——明向还恩，明向不滚，明向刘婆新衣养老钱。

更鼓沉。他按席角，席严，坛不露。外头药山风又硬了一分，硬得像有人半夜要踹门查棚。

韩泥独眼半阖，心里却醒着：下一秤若不在麸皮上，就在席角下。席角下有坛——坛得严，严了，命才还在。

风从门缝钻进来，带着一点粪栏腥。他不躲，只把掌按实：「查也好。查，就让他查空。」

空了席，空不了腹。腹里六层实着，像钉子钉进木里——钉住了，明天的靴声再碎，他也接得住。

远处更鼓又一声，像提醒：丑时将尽，靴声不远。他闭眼，却把挪坛的步骤在心里过了一遍——先石，后席，再缝。过完，嘴角不动，心里已备好下一夜。
