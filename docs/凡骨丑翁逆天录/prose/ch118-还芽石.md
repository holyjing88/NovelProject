# 第一百一十八章 还芽石

夜行外门杂院，韩泥揣半块灵石。

还芽石，沈枯芽柱角，手捧小布包。包轻，里半块灵石已不在——不在，在石已还那夜；夜清了，日来送行，行不送，送的是记。

「韩爷爷，石还了，芽儿记着。」芽儿声小。韩泥点头：「记着。抖了，还你石那夜就白还。」

徒恩清了，叶汤还在前——在前，省亲路，在大比帖后，在钱戾衡台前。台前不要紧，先还芽，先稳119。

石旧，边角磨圆，圆得像泥岗沈家罚银那夜——那夜芽儿送粥被罚，罚因沾他；沾不要紧，恩真。恩真，还也要真；真还，首笔正式报恩，报恩不张扬。张扬，像施舍；施舍不是，是还。还了，才配炼，才配复测，才配130夺魁还刘婆。

杂院门松，松得像凡役可过。韩泥贴墙，独眼辨窗——窗亮第三间，亮得像芽儿仍怯。怯不要紧，石该还；还了，徒恩落笔，落心里，不落榜。落榜，赖福来；赖福来，除名借口又来。

他叩门，声低：「芽儿。」

门开一线，沈枯芽探头，眼亮又慌：「韩……韩爷爷？」

「爷爷。」韩泥把半块灵石托掌心，「泥岗罚银那夜，你送粥连坐。连坐不要紧，恩记着。还沈枯芽半块灵石，泥岗旧账，今日清。」

芽儿退半步：「我……我不是要……」

「知道。」韩泥说，「你不要，我欠。欠着，丹不正。」

芽儿娘咳内屋，声沙：「谁？」

「韩泥。」韩泥声平，「还沈枯芽半块灵石。泥岗旧账，今日清。」

娘沉默一息，声低：「进来。」

屋内窄，窄得像杂役心。韩泥不坐，立门边，像怕沾晦气——沾不要紧，石要放桌上。放桌上，石轻响，响得像盖印。盖印了，第五笔恩落了；落了，心里空半寸，空得舒服。

芽儿娘瞥石，眼浊里清一线：「宗门里，你哪来灵石？」

「渣里淘，坊市捡，五年攒。」韩泥说，「攒来还恩，不攒来显摆。」

娘点头，点得像公审那日鞠半躬：「恩……我们收不得重……」

「半块，不重。」韩泥说，「重，我扛不动。半块，刚好还。」

芽儿捏石，石凉，凉得她指尖红：「韩爷爷，你……你成丹师了？」

「未成。」韩泥说，「成不成，恩先还。还了，才配炼。」

芽儿抬头，怯笑浅，笑像芽：「我……我记着粥。也记着狼口……」

「记着就好。」韩泥说，「记着，别学邪。邪了，鼻下香飘。」

芽儿应一声，声小，真。真，韩泥记——记第五笔恩落了，落得不张扬；不张扬，坛才温。温了，路才正；路正，粒才正。

他退半步，不讨谢。谢轻，恩重；重了，半块刚好抵。抵了，心里空半寸，空得舒服——舒服不是无债，是债还了一笔，还能还下一笔。下一笔，刘婆在复测后；复测后，新衣养老钱，在130递出。

娘低声：「荐帖那日，你走了。走了，芽儿仍怕连坐……」

「怕不要紧。」韩泥说，「怕，还更要还。还了，不怕沾我。」

娘眼圈湿，没哭出声：「刘婆说，你手稳……」

「刘婆粥，恩另笔。」韩泥说，「芽儿石，徒恩。分列，不混。」

混了，还乱；还乱，叶汤就凉。凉不得，汤是头的；头是叶丫头那碗烫，还在省亲前，还在大比帖后，还在钱戾衡台前。

出门，月白，白得像玉牌凉。凉不要紧，仇烫——烫在钱戾衡，烫在三年约。约还远，徒恩已还一笔；还了，炼气十层门槛开一线。开一线，不声张；声张，赖福虽压顶尽，眼仍盯。

他回窑角盘坐，气冲关，炼气十层。十层，不声张。声张，赖福指「偷境」；指不要紧，手稳，管事看手，不看腹。腹里气走周，周周实，像坛腹应「还」——还应芽儿，还应刘婆，还应叶汤。

铁无言立窑外，声低：「还了？」

「还了。」韩泥说，「还沈枯芽半块灵石，清了。」

「下一笔？」

「刘婆。在复测后。」韩泥说，「序不乱。」

铁无言点头，点得像108并肩：「序不乱，丹才正。」

刘婆远处咳，咳轻，像听见：「还谁了？」

「芽儿。」韩泥说。

「还就好。」刘婆哼，「还了，饭更稳。饭稳，恩不断。」

韩泥点头。谢轻，恩重。恩在粥里，也在还里——还里静，静得像一碗热汤递到该递的人手里。手递石，不讨谢；不讨谢，恩才真；恩真，坛才温。

丑时，练鼻下静。粟壳香淡，香落齐，半块石那响才记进心里。记进心里，不进舌——舌响，白织月说「施舍」；施舍不是，是还。还了，才配十层稳，才配复测备战。

他摸玉牌，牌凉：「三年约，记着。芽儿还了，叶还在前头。」

叶在前头，省亲在前头，钱戾衡在前头。前头路长，徒恩落一笔，够撑十层稳，够撑复测备战。备战开，两月后上台；上台，钱戾衡眼风再来——来，不抖。

外头风软，软得像恩还了一半。一半不要紧，刘婆新衣在远，远得像130那日。那日未到，先备战；备战，先炼气十层实，先粒正，先刘婆粥多半勺。

坛不应字，只温。温像夸「还」——还到了芽儿手里，瓮便温一线。温一线，够走到119，够走到复测风声起。风声起，报名帖至，试台在远，魁在130，还刘婆在130后。

沈枯芽窗灯灭，灭得像睡。睡了，石在枕下，枕下像信——信韩泥丑，信韩泥还，信恩不邪。韩泥望窗，独眼平，低声道：「徒恩，记着。还沈枯芽半块灵石，还了。」

还石夜风软，韩泥靴底泥少，少得像没出过宗门。少泥不要紧，半块石已落芽儿掌心。掌心凉，恩却烫——烫在序里，排在刘婆前头第二位待还。

回窑后，韩泥把泥岗第五笔徒恩又默念一遍。默念，不落字纸；字纸只记饭，饭稀，恩不稀。稀粥顶一口饥，多活一日，多记一日。记满了，就还——还了芽儿，还刘婆，还叶汤。还路长，先十层实，先复测风，先夺魁，先大比帖。帖在怀沉，沉的是债序，不是惧。惧要手抖，他不抖。

不抖，就配119十层稳，配120风声起，配130还刘婆新衣养老钱。新衣袖口多缝一层，耐磨，像刘婆顶针顶到半夜。半夜冷，她手抖，抖也没停——停不得，肘暖了，人才走得动还路。还路，芽儿这笔已清，清出一寸胸空，空出来装下一笔刘婆待还。

还芽儿后，韩泥不回棚炫耀。炫耀像讨恩；讨轻，还重。重，静着还，还到芽儿手里，才算数。算数了，心里空半寸，空得舒服——舒服不是无债，是债还了一笔。

回窑角，气走周，十层破。破，不声张。声张，赖福虽压顶尽，眼仍盯；盯不要紧，手稳，册在，就不滚。

铁无言立窑外：「还了？试台后。」韩泥答在心里排序——芽儿清了，刘婆待清，叶汤头恩在省亲前。序不乱，粒才正；粒正，魁才配。

刘婆远处咳，咳轻：「还谁了？」「芽儿半块石。」「还就好。还了，粥更稳。」韩泥喝粥，稀，热。热顶饥，饥顶备战。备战从十层始，始向复测，始向130夺魁还刘婆。

邪了，鼻下香飘；不飘，粒才正。

夜，他再数债序：叶汤①，芽儿⑤已还，刘婆⑦待还，钱戾衡仇在玉牌。混了，还乱；还乱，丹不正。

坛腹温一线，应「还」——还应芽儿，还应刘婆，还应叶汤。还应，路才正。路正，粒才正；粒正，试台才配说话。

他低声道：「徒恩，记着。还了。」还了，就不欠芽儿；不欠，手更稳。手稳，接119十层稳，接120复测风，接钱戾衡影再来——来，不抖。

外头柱上条贴新纸，纸边字淡：「外门复测，丹试报名。」报名不要紧，先十层实；实了，报；报了，备战才真。真备战，两月后上台；上台，玉牌才不凉透。

还芽儿夜，韩泥不回棚喧。喧像讨恩，讨轻，还重。重，坛温应「还」。还应刘婆在130，还应叶汤在省亲前，还应钱戾衡仇在玉牌。玉牌凉，三年约烫。烫在试台，烫在大比，烫在不滚。不滚，手在，粒在，坛在。在，就配119十层稳，配120复测风。
