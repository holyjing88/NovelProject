# 第八十八章 暗修四层

三层落地后第三夜，丑时又到。远灯近了半寸。

近不是到，是灯芯更清——清得像废窑余温里那一点亮，亮在渣里，不在舌头上。韩泥盘坐床沿，席严，坛不露。掌贴沿，沿温一线。他先把呼吸调匀，匀得像编筐绕藤：紧了断，松了散。

三层气转一周，两周；第三周时，远灯忽然定住，定得像碑。碑钝，钝得像测灵石——石秤过命，灯秤境。气再转一周，掌纹烫得踏实。

关隘裂一线，裂声闷在骨里。四层开了。开不像雷，像药山深雾散开半寸，散出界碑影：影外一步没脚印，影内是境。韩泥屏息，独眼不眨。眨了，境散；散了，白开。他把气稳住，稳得像筐底那撮辛——沉在底，不浮。

邻床鼾声如雷。雷不要紧，他鼻下静一息。沉腐浮三字不落舌，落鼻。收功像盖席——盖严，沿净，坛仍不露。沿净了，夜里才像腌菜坛，不像邪修。

掌纹那路又厚半分，厚在暗，不进册。册仍末席，掌写四层。写给自己。他把那厚度按进记忆，不按进义气——义气会嚷，记忆只闷。

温听雨清晨洒扫，巾湿，水净。她瞥韩泥指头，声清半分：「指比洒扫那日更稳。」停一停，又补半句：「稳，像能炼丹的人。」

「像」字落地，韩泥不谢。谢轻了像讨。他记——记缘苗浅，记界还在，记真炼不在今朝，在渣路往后走实。

她像听见他心里那道界，巾一甩：「各做各的。我说稳，不是叫你攀门。」

「不攀。」韩泥说，「听过就够。」

「够就好。」她看他一眼，目光不黏，「黏了，丹也不香。」

角柜十包样再锁。锁住了才不示人。示人谣来；谣来，晦气倒可能遮瓮——遮住也好，守四层，守三年约那条命。他把钥匙绳勒紧，勒出一道浅印。

铁无言在径外停脚，声压低：「四层别叫。叫了，煞骨门嗅得到。」

韩泥心里记一笔：暗修四层，见境已到。到在暗，不在众目。坛底轻颤，臾墟残念弱一息：「四层在掌，丹路将开。开在渣，别叫丹。」

「记着。」韩泥低声应。

「记着还不够。」铁无言道，「明日废窑只能试温，不能占。占了，名册上除你。」

「不占。」韩泥说，「试温。」

赖福盯棚缝半息，终没掀席。掀不着净沿，掀了像自取其辱。卯时他背筐上药山边门，四层气沉足底，足底稳一分——稳像界碑前那一步：不过界，只捡渣。深雾起，鼻下小成，手快而稳。

管事验筐：「深捡净。」记簿。赖福远讽：「四层也捡？捡出邪丹！」

韩泥不接茬：「只捡渣，不叫境。」

「你倒会装！」赖福凑近半步，「装到什么时候？」

「装到你饿。」韩泥说，「你挨饿那天，我还在捡。」

赖福噎住，骂骂咧咧退开。刘婆端粥，仍多半勺：「昨夜又亮？」

「豆爆。」韩泥说，「沿净。」

「沿净怕什么。」刘婆哼，「饭稳，恩不断。你飘了，粥也救不了你。」

「不飘。」他把粥喝尽，「飘了，废窑不收。」

午后他到废窑门槛外，再试温，不占窑——占窑除名，除名恩断。凡火匀一寸，掌温应上去，渣灰里亮多一丝，像路要开。铁无言立径外：「四层到了，别贪炉。贪炉，除名。」

「不贪。」韩泥道，「试温。」

「试温可以。」铁无言说，「炼，不许。炼要日子，日子没到，手就先滚。」

「日子我记着。」韩泥应，「手不滚。」

白织月主道过，袖风冷：「杂役也四层？四层也炼不出丹。」

韩泥不辩：「炼渣，不炼名。名在册末，渣在袋里。」

「渣能炼出体面？」她鼻下一哼。

「体面不急。」韩泥说，「活命要紧。」

她走了。少一句嘲，又是小胜。秦霜木鱼轻敲：「到境心别骄。骄，邪。」

韩泥心里把恩和辱分开搁着：四层是境，境通向丹路，通向五层，不是通向嘴硬。丑时前他再练半刻——三层在忆，四层在掌，五层在远。远也不盲。

邻床悄问：「你身上热，真不怕？」

「怕。」韩泥说，「怕也藏。不叫，沿净，才藏得住。」

「藏啥？」

「藏活命的东西。」韩泥道，「懂不懂都睡。」

少年闭嘴睡了。韩泥摸暗袋，袋沉，渣像种。种轻，记重；记重了，明日丹路才敢开。他对坛低声：「四层到了。不叫。坛在，袋在。袋在，路才开。」

坛不应字，只温。温像把远灯那一点清摁进掌纹。

药渣场次日验指，他分拣更快，快也要稳。十堆仍全对，管事记「春验仍准」。铁无言声低：「四层别示人。示人，像破外门境。」

「不示。」韩泥说，「境在席下。」

「席下好。」铁无言道，「席上热闹的，多半短命。」

温听雨又远远一句：「指更稳了。稳，像能守炉。」

守炉比炼丹更贴他——他还没炉，只有废窑门槛外那一点温。赖福讽：「末等四层？四层也滚！」

他仍不接。舌要留给鼻子，留给守坛。刘婆嘱：「四层别飘。」

「不飘。」韩泥说，「飘了，废窑也不收我。」

黄昏他立在废渣巷口，不进窑。进是收，今日只记四层配槽。邻床又问：「四层能打赢钱戾衡吗？」

「赢不在四层。」韩泥道，「赢在活得长。活长了，刃才亮。」

「那你刃亮不亮？」

「还早。」韩泥说，「亮早了，人先死。睡。」

丑时前内视，第四阶亮着，阶上白线像炉门缝——缝远，像门缝留灯，还远着。他按木牌贴胸：「四层稳了。丹路开在后头。手不抖。」

老耿挑水过：「境别拜邪。」

「不拜，记着。」韩泥应。

管事夜巡验棚，灯光照席角，沿净，记「手稳」。手稳是饭票，不是贴金。他把木牌系紧一线，系得像把「仍末」摁进掌心。同批少年远避，避像避晦气——晦气遮瓮，遮住了才稳。

更鼓沉时，他独眼平平：「四层在掌。明日废窑门槛再试温。温到渣说话，丹路才算开。」

风硬，廊下一冷。他不打颤。颤了像求；不求，只记温听雨那句「像能炼丹」——像，就够催一催；真炼，还要渣答一句「开」。

他把暗袋按在肋下，袋温微微。温不在炉，在记。记够了，门槛外那寸凡火才肯听话。

午后编筐角，藤刺又扎了一下。他偏手，刺浅些，血珠抹掉。血不能沾坛沿——沾了，沿就不净；不净，赖福就有借口。借口一来，四层再稳也白搭。

刘婆路过递半块硬馍：「吃。别空肚试温。」

「谢。」韩泥接过。

「谢什么。」刘婆瞪，「谢完把馍吃了。吃了，手才不抖。」

他咬一口，硬得咯牙，却踏实。踏实了，才敢想废窑那一寸火。火匀一寸够，两寸就裂——裂了没人争，正好藏；可争的人一旦闻到丹香，藏也藏不住。

同批少年远远问：「你真到四层了？」

「你听见我叫了？」韩泥反问。

「没有。」

「没有就别问。」韩泥说，「问多了，香就飘。」

少年闭嘴。韩泥把馍渣拍净，拍得像把「末席」也拍进尘里——尘里站着也能活，活着才等得到渣开口。

丑时将到，他掌贴坛沿，低声道：「四层稳了。瓮在，袋在。明日试温，专等那个『开』字。」

棚外更鼓又一声，像催，又不像催。他不跟着慌。慌了手抖，手抖火不匀；火不匀，渣就不肯开口。他把暗袋再按实一分：「开」字可以等，手不能先乱。

秦霜木鱼远远再敲半下，像提醒心口：境到了也别邪。他心里应完，独眼平着等明日——明日废窑门槛外那寸凡火，才轮到他问话。问到了，丹路才算真开，手也才算没白练。
