# 第一百四十四章 泥纹扬名

坊市一开，尘土里就滚出两个字：泥纹。

不是榜上的雅号，是尾摊传出来的土叫法。有人递渣，韩泥鼻下静一息，辨完辛淡，对方退时留下半句：「泥纹丑，效不丑。」

半句比锣响。韩泥没接喜，只把破筐反扣稳些，歪牌照旧：「丑丹师·嗅渣辨性·续留。」

第一日来了个老散修，鬓毛花白，递上一捧灰渣：「闻闻。真假？」

韩泥捻渣，指腹一道：「沉在下，腐气浮。可炼，不宜急火。」

「价？」

「半文看渣。」韩泥说，「看完你拿走主意。」

老散修付了钱，临走又瞄牌：「泥纹……是他们叫的？」

「他们爱叫就叫。」韩泥收摊布，「我认渣，不认嘴。」

白织月偏偏这时候过来，袖风冷：「名号也丑。泥纹，像沾泥乞儿。」

「骨丑，渣正。」韩泥抬头，「正不正，不看袖香。」

「你得意了？」她笑薄。

「没得意。」韩泥说，「得意手就飘。飘了，名是假的。」

她滞半息，没再尖，转身走了。婢子跟在后头小声：「姑娘，他怎么总顶您？」

「顶得住，才讨厌。」白织月淡淡丢给风。

管事弟子午后贴条：「丑丹师续留，税照旧。泥纹之名，坊间自传，不作官授。」

韩泥看完条，交半文税：「不作官授更好。官授像债。」

「你怕债？」弟子奇望。

「不怕还的债。」韩泥道，「怕空名。空名压手。」

铁无言立在三步外，不进摊，声低：「名起了，别飘。」

「不飘。」韩泥说，「名在牌上，活在手里。」

「手里稳，名才不臊。」铁无言点头，「臊了，赖福有词。」

刘婆买盐路过，盐袋一晃：「少听嘲。嘲多了，耳根软。」

「听。」韩泥应，「听是为了避。避了，才活得长。」

「活长比名响强。」刘婆塞他一块糙饼，「吃。辨渣饿着，鼻子不准。」

「谢。」

「少谢。」她瞪，「谢多了像生分。」

第二日，递渣的人多了半圈。有人真求辨，有人存心搅：「泥纹？闻闻我裤脚行不行？」

韩泥不气，只推开：「裤脚不是渣。不是渣，我不收钱。」

「假清高。」那人啐。

「清不高。」韩泥说，「规矩在。规矩破了，摊就脏。」

赖福远远盯着，终究凑近：「成名了？税加倍如何？」

「条上写照旧。」韩泥指贴条，「照旧就半文。」

「半文也是钱。」赖福阴声，「钱从哪来？谢恩剩的？」

「从编筐来。」韩泥声平，「筐丑，钱真。」

赖福没搜出岔子，只甩袖：「名臭了别怪我。」

「名臭我认。」韩泥道，「效臭，我不认。」

第三日收摊前，老散修又来，这回递的渣正，正得像给个证：「昨日听你的，火没毁。今天加一文。」

「半文规矩。」韩泥只收该收的，「多的你收回。」

「真倔。」老散修笑出声，「泥纹丑，效不丑——这句我帮你传。」

「传效力，别传人。」韩泥提醒，「传人，像显摆。」

「懂。」老散修收渣，「传效就行。」

回丙九路上，沈枯芽从巷角蹦出来：「叶姐听着坊市叫泥纹了。她说……丑点没关系，路正就行。」

「告诉她，路正着。」韩泥说，「名是旁人喊的。我喊瓮。」

「喊瓮？」芽儿懵。

「瓮在，心才在。」韩泥拍怀，「心在，名才不虚。」

夜里他按坛。坛腹温一线，像应「名」字——名起在渣里，不起在舌头上。舌头要留着收税、答嘲、跟刘婆道谢，留着听下一桩活。

邻床少年兴奋得睡不着：「韩叔，你火了！泥纹泥纹！」

「火是炉里的。」韩泥说，「名是风里的。风一转就灭，炉得自己守。」

「那我也能有名？」

「先把鼻子练准。」韩泥道，「准了，名自己来；不准，喊破天也假。」

更鼓沉，他收好摊上最后一点灰。灰里还留着辛淡。辛淡在，就说明今日没骗人。

外头尾巷有人打着哈欠还在念：「泥纹丑，效不丑……」

名起之后，摊前多了偷瞧的人。有人甩下一包染色假渣，想砸他招牌。

韩泥只闻一口：「染过。染过的我不辨。」

「你怎知道？」

「鼻子知道。」韩泥推回去，「假渣进摊，我就毁自己牌子。」

「吓唬谁？」

「吓唬我自己。」韩泥认真，「吓自己，手才稳。」

管事弟子听说此事，过来补记一笔：「拒假渣，可嘉。」

「别嘉。」韩泥摇头，「嘉了像表功。表功，就有人盯着摘。」

铁无言远处举手示意：摘吧。他懂——名刚起，最怕被人抬轿又撤轿。轿上的人摔得响。

午后，一个女修路过，瞧见牌冷笑：「泥纹？像村名。」

「村名也认路。」韩泥说，「认路比好听要紧。」

「你嘴倒俐。」

「俐在渣上。」韩泥低头继续摊灰，「渣外的嘴，我不练。」

第三日傍晚，他把半文税外多出来的两文另纸包起，塞回本金盒：「名带来的钱，先不花。花了，像吃名。」

「那你吃啥？」芽儿问。

「吃筐钱，吃粥。」韩泥道，「吃得素，名才不腻。」

夜里雨停，牌上湿痕干了。他摸牌沿，沿仍歪。歪着好——歪着提醒他：这不是华服，是饭碗。饭碗稳，泥纹俩字才有根。

「根在鼻。」他对瓮说，「鼻准，名就准；鼻飘，名就先死。」

名传开的那天，有人出钱买他「泥纹秘诀」。银两摆到摊上叮响。

「不卖。」韩泥当场推回，「诀在鼻子里，鼻子不批。」

「开价？」

「没价。」韩泥说，「卖诀的人，渣先假。」

那人悻悻走了。铁无言远处勾唇：「吓跑一个财神。」

「财神不认路。」韩泥收拾，「认路的是老实鼻子。」

傍晚收摊，牌被顽童画了两笔黑。他没骂，用水擦净：「牌丑点没事。字在，就成。」

「你太惯人。」少年不平。

「惯的是娃娃。」韩泥说，「成年人使坏，我记账。账满了，用手回。」

「用手？」

「用手炼更正的粒。」韩泥抬头，「正压邪，比骂管用。」

回棚路上听见有人唱「泥纹丑」，唱得油。他只当听歌。歌散了，摊子还在。在，就够明朝支起来。

瓮边他多问一句：「名起了，你会不会嫌吵？」

坛温一线，像说：吵在外，净在内。内净，外名随便叫。

收摊账上今日进项不多，他偏抄得工整。工整给自己看：名不是利，名是验鼻子的考题。考过了，才配继续挂。

「抄账这么认真？」芽儿蹲旁边。

「认真，免得以后赖。」韩泥吹干墨，「赖账的人，先赖自己糊涂。」

「叶姐让问：泥纹二字，要不要辟谣？」

「辟什么。」韩泥合账，「丑就丑。效正，谣自己散。」

芽儿走后，他把「效不丑」三字写在摊底板反面，写完又铲掉。留在心里就行，写出来像标语。标语一响，手就容易飘。

夜里他梦见末摊被抬进内门廊，牌金光刺眼。刺醒后一身冷汗：「金的不要。歪的才像我。」

坐起来喝水，水入喉，名才又变回土腥味。土腥味亲近——亲近才踏实。

临出门他特地把破筐沿修齐一圈。沿齐，人不齐也不要紧——人丑，器皿干净，来辨渣的才肯伸手。

「修筐也算丹道？」有人打趣。

「算活。」韩泥说，「活不干净，丹再正也被嫌。」

修好筐，他在心里把门坎放低：名起了，门槛不能高。高了，像内门做派。做派一起，鼻子就傲；傲了，假渣都闻不出来。

「不傲。」他提起筐，「末席到死都是末席。末席闻得更清。」

筐修好，他特意在摊角压一块净石。石沉，牌才不摇。牌不摇，来辨渣的人才觉得这丑摊可信。可信比好听要紧，要紧到他愿拿整日尘土去换。

念声远了，像钩——钩住明朝还要续挂的牌，也钩住他心里隐隐响的蹄声。蹄声还没落地，影却先贴到兽栏边。畜一来，药包才有脚；有脚，这名才不像吹出来的泡。
