# 第五十一章 夜袭预兆

秦商言的第二道飞符，比第一道更急。

符上十二字，墨迹未干：**子时东脉，妖帅随行，先锋佯攻丹堂。**

柳青鸢立宗主殿，符在指间，指节白。霍镇山甲未卸，声洪：「镇妖司前移！东脉灵田，丹堂，侧径，全线布防！」

莫长春拄杖入殿，灰袍素，袖中却沉——青霜剑在，剑未佩，剑意像霜贴在袖里，贴了一路。

「师兄。」柳青鸢看他，「你能赠，不能打。」

「能打一点。」莫长春道，「打多了，像演。打一点，像还礼。」

柳青鸢一愣，随即懂：师兄不是变，是簿上因果满了，满到不得不兑。

她递令旗：「护法长老位，师兄自己占。」

「占。」他接旗，杖点地，「占而不骄。骄了，又像九月将死前的回光。」

殿外，弟子奔走，火把明灭，像把子时提前点在东脉边上。苏念慈遣人来报：丹堂地火阵已温，炉不灭。赵玄机戴罪出思过崖，甲胄带血痕，眼却亮，亮得像终于等到还人情的夜。

弟子围观殿前调度，窃语如潮：「妖帅随行？」「佯攻丹堂？」「莫长老袖里……有剑意？」

顾小满在杂役堂分班，路线图在怀，像第二张半石，眼里仍亮雨夜那道光。

周德海立后阵棚，令旗在指间，迟迟不落，落得像犹豫，犹豫得像奸。

莫长春望东脉，笑：

「礼还没送完。他们先动手，正好省老夫敲门。」

寿元在簿上，三月。因果沉，像七十五斤，今夜要称一称。

魏无涯观调度，低笑，像看因果满到不得不兑。

他低声，像钩：

「子时两时辰后。三线布防，缺不得一环。」

风腥，火把明灭，像弓已张满。韩铁山在远山冷笑，笑里刀，刀向田，也向丹堂。

杖中竹纹急游，气音二字，像应：

备好。

「九月老头要打？」「打一点，像还礼。」窃语里怕，也燃——怕妖帅，燃的是长老袖里有霜。

苏念慈在丹堂温地火阵，炉不灭，阵纹淡红，像另一颗心也在跳。

赵玄机戴罪出思过崖，甲胄带血痕，眼却亮，亮得像终于等到还人情的夜。路过莫长春，抱拳，礼深：「长老，赵玄机今夜守门。」莫长春道：「守。门比剑难守。守住了，恩就还了一半。」

厉无殇在帐内听报，声哑：「莫长春，青霜？」他砸杯，杯碎，像砸不碎袖中记忆：手帕三息，杖扫妖骑，如今霜入袖——礼还没送完，恨还没记够。

子时两时辰，弓张满，霜在袖——缺一环，田就失；失一环，丹堂先危。

杖中竹纹急游，气音二字，像应：备好。风腥，火把明灭，像弓已张满。殿外弟子奔走，火把明灭，像把子时提前点在东脉边上——点上的是霜，是杖，是簿上因果满到不得不兑那一夜。

柳青鸢一愣，随即懂：师兄不是变，是簿上因果满了，满到不得不兑。她递令旗：「护法长老位，师兄自己占。」「占。」他接旗，杖点地，「占而不骄。骄了，又像九月将死前的回光。」弟子围观殿前调度，窃语如潮：「妖帅随行？」「佯攻丹堂？」「莫长老袖里……有剑意？」

子时两时辰，弓张满，霜在袖——缺一环，田就失；失一环，丹堂先危。韩铁山在远山冷笑，笑里刀，刀向田，也向丹堂。

「能打一点。」莫长春道，「打多了，像演。打一点，像还礼。」寿元在簿上，三月。因果沉，像七十五斤，今夜要称一称。

魏无涯观调度，低笑，像看因果满到不得不兑。厉无殇在帐内听报，声哑：「莫长春，青霜？」他砸杯，杯碎，像砸不碎袖中记忆：手帕三息，杖扫妖骑，如今霜入袖。

赵玄机路过莫长春，抱拳，礼深：「长老，赵玄机今夜守门。」莫长春道：「守。门比剑难守。守住了，恩就还了一半。」

围观弟子在殿前踮脚，踮得像等青霜出袖——袖未开，剑意先至。有人低声：「能打一点，像还礼。」另一人答：「打多了，像演。」

殿前人群里，有人伸颈看袖，看不成，只得问：「剑意能不能出袖？」旁人冷笑：「出不出袖，两时辰后说了算。」霍镇山甲甲相碰，碰出闷响，闷响把「打一点」三字撞进众人耳里，耳里先怕，后燃——燃的是霜未见，腥先到。

苏念慈遣人再报：「丹堂炉不灭，地火阵纹已红一线。」报声入殿，殿外围观里有人低声：「红一线像帖将至，帖在云上，云在东脉。」
