# 第二十六章 坊市闲逛

备战间隙，莫长春下山，青岚坊市。

未乘轿，未带弟子，灰袍素，杖点青石板，像寻常老翁赶集。摊贩认得他，却不敢大声招呼——「莫长老」三字在坊间比灵石还硬，硬到砍价都像冒犯。

可今日偏偏有人敢围——围得不近，围在巷口、茶楼下、药铺檐下，像看一场不像长老的戏。

莫长春在棉衣摊前停，摸布料，厚，糙，暖，问价，摊主伸五指，他伸三指，摊主苦脸，他笑：「三指，送杂役堂十件，记青岚门账。」

摊主一愣：「长老……真记门账？」

「记。」莫长春道，「杂役堂人冻，手抖，搬药洒；洒了药，丹堂骂；丹堂骂，你少卖一瓶伤药。一环扣一环，暖的是手，赚的是命。」

摊主终点头，像被说服，也像被九月老头一种说不清的气压服。隔壁摊主见状，凑过来：「长老，俺这伤药也记门账？」

「记。」莫长春道，「但先砍价。门账不是冤账。」

围观人群笑，笑轻，像松一口气——定身魔少的老神仙，下山却像抠门老翁，反差比戏还好看。

他又至低阶符纸摊，同样砍价，同样嘱送杂役堂，灵石从袖中出，出得慢，像真穷，数得细，像真抠。一张黄纸扯半炷香，扯得认真，掌柜额头见汗，最后仍让了半指。

路人瞧见，窃语：「莫长老定身魔少，下山却像抠门老翁？」

另一人答：「抠门才真，灵石撒给杂役，不撒给面子。」

第三人低声：「听说他九月将尽，还这么省……省给谁看？」

第四人答：「给咱们看。杂役手暖了，咱们伤药好卖。」

有人嚷：「莫长老，也给俺记一件棉衣！」

莫长春笑：「排队。门账一笔笔记，乱不得。」

茶楼二层，白漱玉凭窗，远观莫长春在符纸摊前与掌柜扯半炷香，扯得认真，像不是化神劫后的长老，是凡间会为一张黄纸计较的人。

她心口微动，慕意初起，像茶烟，轻，却缠。

同伴笑：「漱玉，你看什么？」

白漱玉收目，淡道：「看一个人，把钱花在别人看不见的地方。」

同伴撇嘴：「看不见的地方，怎么赚名声？」

白漱玉指楼下：「楼下都在看他砍价。名声不在他花多少，在他花给谁。」

同伴一愣，不语，却忍不住又瞧一眼楼下——九月老头数灵石，数得像数命，数得围观者都静了半息。

日暮，莫长春归山，袖中少灵石，心中多烟火。

途经杂役堂，把棉衣单掷给管事：「十件，明日穿。穿了手稳，别洒药。」

管事接单，手抖，抖得像接旨。顾小满在廊下瞧见，眼里亮，亮得像又多记了一笔。

簿无字，他却觉善缘又涨一分，涨在凡人摊位的三指砍价里，涨在杂役堂将到的棉衣里。

坊市弟子尾随半里，尾随里议论：「长老真记门账？」「司库认吗？」「认不认，杂役手暖了就行。」

符纸掌柜私下对伙计说：「莫长老砍价像砍价，记门账像记命。」伙计问：「咱赚吗？」掌柜答：「赚在青岚门还在。」

白漱玉下楼，欲递名帖，又止，止得像怕买卖玷了慕。她只买一包粗茶，茶苦，苦后回甘，像莫长春那日递茶饼。

归山路上，孩童追问：「老爷爷，你真定住魔少？」莫长春笑：「定住的是戏。你们别学戏，学记门账。」

弟子尾随半里，议论门账，议论里敬，也疑——疑长老真穷，疑长老真抠，疑得越久，越像看懂：抠的是面子，松的是手。

他念，像钩，钩向远山：

「坊市烟火，是根的另一面。妖讯再来时，这双手，还得暖着握杖。」

杖答，气音轻，像等。

莫长春笑：「不等。」

大因果还在远处，像妖潮，像赤焰——坊市烟火只是小暖，第二次妖讯才是大还礼的前奏。

坊市弟子尾随半里，尾随里议论：「长老真记门账？」「司库认吗？」「认不认，杂役手暖了就行。」

莫长春归山，袖中少灵石，心中多烟火。簿无字，他却觉善缘又涨一分，涨在凡人摊位的三指砍价里，涨在杂役堂将到的棉衣里。

他念，像钩，钩向远山：「坊市烟火，是根的另一面。妖讯再来时，这双手，还得暖着握杖。」杖答，气音轻，像等。

围观人群在巷口踮脚，踮得像等长老再砍价——再砍的不是价，是面子。有人低声：「定身魔少的老头，为三指灵石扯半炷香？」另一人答：「扯的是门账，不是价。」
