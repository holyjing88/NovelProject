# 第十五章 宗主裁断

执法堂正殿，柳青鸢升座，未佩剑，威在眉睫。

孙福跪地，顾小满立侧廊——莫长春嘱她勿进，她便在柱后听，指节把袖角拧成绳。

周德海立殿左，抚须不语，像一盆浇了一半的油，等风。

莫长春拄杖立殿心，不跪不辩，灰袍素，像一块不会被嚼舌嚼碎的旧玉。

柳青鸢开口，声冷而清：「孙福，你说莫长老勾引杂役，证据何在？」

孙福叩头：「外门皆言……」

「外门皆言，」柳青鸢截话，「便是证据？」

孙福一窒。

柳青鸢目光扫过廊下弟子：「谣言止于宗主。信谁，我来担。」

殿内一静，静得能听见铁牌光落在地砖上的冷。

她续道：「莫师兄不是那种人。半石之事，是师兄赠恩，顾小满受恩，杂役堂孙福克扣在前，顾小满举证在后——恩在堂，不在床。」

孙福脸白，膝软：「宗主……孙福一时糊涂……」

「糊涂？」柳青鸢道，「糊涂到敢拿宗主师兄名节换你三月罚役的轻？」

她拍案，声不大，意重：「孙福，罚役三年，逐出杂役堂，发往灵田苦役。再敢诬长老，逐出山门。」

孙福瘫软，被执法弟子拖出，拖出一串求饶，求到廊外，像风扯碎纸。

周德海揖手：「宗主英明。只是外门流言……」

「流言我压。」柳青鸢看他，「周长老，东脉妖潮将起，赤焰使者已在路上。你若还有闲心拱火，不如去长老会备折子。」

周德海笑容终于收尽，退席。

莫长春抬眼，与柳青鸢对视一瞬，像二十年同门，一句话不必说全。

柳青鸢低声，仅他听见：「师兄，我不解释你，只信你。」

莫长春点头：「够了。」

柱后，苏念慈听见「莫师兄」三字，与心里那声「师兄」叠在一起，妒与忠绞成一股，勒得她喉紧。

她看见柳青鸢维护时，眼底没有妒，只有宗主的担——那一担，比她丹堂首座的火更烫。

苏念慈退半步，未现形，像怕自己一露面，就把私心里那点不该有的酸，溅到堂上。

廊外，弟子群情反转：「宗主都说了，还传什么！」

赵玄机立远处，抄门规的手停住，墨滴在纸，晕开一字「心」，像莫长春那句定场词又浮上来。

他忽然觉得，妒一个人，若妒到连宗主都信他，自己便显得很小。

午后，莫长春回院，顾小满仍在阶下，未跪，立。

「长老清白了。」她低声。

「清白本来在。」莫长春道，「只是世人爱听脏的。」

他袖空，一念出塔，薄符落掌，递给她：「收好。不是除魔符，是护身。孙福虽逐，周德海还在。」

顾小满接符，泪落，却笑：「小满记恩。」

塔影在识海一烫，字成：

**宗主一担，忠诚记在心里。**

莫长春望青岚峰顶云，念：「柳氏信我，苏氏妒我，赵氏疑我——好，人多，礼才好送。」

杖中竹纹游动，气音仍是一字：

等。

他笑：「不等了。下一阵，赤焰旗要来。」

裁断书贴出执法堂外，朱印未干。八字横排：「回赠有据，私赠不究。」

有人故意念错：「私赠有据——」话没念完，被同伴打手，像怕念错也传染。

柳青鸢佩旧剑巡行，剑穗无风，声稳：「裁清，门清。再传诬长老者，与孙福同罪。」

赵玄机验裁书，书真，真里仍想挑刺——刺挑到「宗主担」，挑不实。他低声：「裁断了，才辩得清。」

苏念慈开炉，炉旺。她对弟子道：「裁了，妒别烧丹。丹堂只认药，不认舌。」

霍镇山在营帐外磨刀，刀声霍霍：「宗主硬，东脉才硬。」

周德海回府，折子角皱，遣弟子探赤焰来使行程，探得像怕妖潮与赤焰真同至。

顾小满回杂役堂，新管事姓李，见她眼神，客气里多一分怕。她声平：「规矩在册上。册在，规矩就在。」

丹堂里，苏念慈遣汤被长老推回。她笑苦：「他还是这样，把甜推给该喝的人。」

柳青鸢夜归宗主殿，独对东脉图，指节发白。她嘱执事：「执法堂一贴，东脉驿一贴——贴给敢诬的人看，也贴给敢担的人看。」

莫长春院中，顾小满隔门守了一夜，天明问：「长老，还查孙福旧党吗？」

「查在册上。」他笑，「册在谁手，谁查。老夫袖空，只记恩。」

塔壁浮第二行，淡金：**裁清门风，东脉弦绷。**

他望东脉云：「孙福事小，赤焰事大。大因果在旗，旗在七日。」

夜风过，桂香淡。青岚门像被同一根弦拽紧——弦在东脉，弦在七日，弦在赤焰旗将来的那一声哗响。

廊下，年轻弟子念裁断书，念到「恩在堂，不在床」停住，像舌被洗。师兄拍他肩：「念对了，就闭嘴。闭嘴，也是还礼。」

赵玄机夜归执法堂，报告写到一半，笔尖悬——「私赠」二字写不下去，「回赠有据」挡在前头。他掷笔，笑苦：「裁书贴的是门风，贴不住的是塔。」

莫长春院中，杖中竹纹游，他问：「下一阵，旗红还是人心红？」气音不答，塔壁却沉，沉得像七日后那条席，已在识海里摆好。

顾小满回屋，把护身符贴身，贴得像第二张半石。她念：「宗主担了，我记着。记着，就不必跪。」

周德海密室里，心腹问：「裁断了，还拱吗？」周冷笑：「裁断裁的是孙福，没裁东脉。东脉，才是咱们的田。」

苏念慈夜未眠，炉边搁一只空碗，碗像等汤回来。她忽然明白：柳青鸢担的是名，她妒的是名，名不同，火却同烫——烫得她更要把丹炼稳，稳给东脉，稳给师兄九月。

霍镇山营中，刀磨至锋，锋映烛，像映七日后的血。他对副将道：「宗主硬，东脉才硬。硬不在嘴，在田。」

柳青鸢独对东脉图至三更，指节发白，白里不是惧，是算——算妖潮、算赤焰、算师兄九月。她低声：「师兄，这一回，我替你担名，你替我送礼。」

莫长春望云，云缝里有星，星淡，淡得像九月里多出来的一息。他抚簿，念：「裁清了，才好送下一礼。礼轻，意重——重的是东脉那根弦，不是孙福那张嘴。」

执法堂外，裁断书朱印映日，映得像给孙福三年苦役钉一枚钉。弟子念「恩在堂，不在床」，念得像洗舌，洗得脏字退潮，潮退处，东脉弦却绷得更紧。

赵玄机立远处，像不服，又像服一半，半里仍空一线——空的是塔，塔不在裁书上。他忽然想把培元还库的念头又翻出来，翻得像还库之前，心还在两边。

莫长春院中，顾小满隔门守了一夜，天明问：「长老，还查孙福旧党吗？」「查在册上。册在谁手，谁查。」

柳青鸢夜嘱执事：「执法堂一贴，东脉驿一贴。」莫长春望东脉云：「孙福事小，赤焰事大。大因果在旗，旗在七日。」

杖中竹纹游，气音仍是一字：等。他笑：「不等了。下一阵，赤焰旗要来。」

丹堂里，苏念慈把药勺放回架，勺响，响得像妒火退潮一寸。她未去执法堂，却明白：柳青鸢担的是名，她妒的是名——名不同，火却同烫。

赵玄机夜抄门规，抄到「诬长老」一条，停住，像第一次发现门规里也有他踩不实的缝——缝在宗主担，不在私赠。

莫长春抚簿，念：「裁清了，才好送下一礼。下一阵，赤焰旗要来。」

裁断落定，孙福膝软，柳青鸢眸光却亮——亮得像把东脉妖讯，也一并裁进了三月之约。
