# 第一百章 兽潮前夜

前夜。

兽栏外，火把密，密像星落人间。人间不要紧，要紧是栏里小兽不安，不安则腥重，腥重则瘴近。

韩泥蒙刘婆布条，条粗，粗不要紧；粗，挡甜腥。甜腥在风里，风一过鼻，他仍嗅出一线毒——毒细，细而记。

记，不在舌。舌响，手飘；飘了，栏倒。

铁无言分障，障是湿草叠石灰，叠法粗，粗不要紧；粗，能挡一时。一时，够了。

「你守东角。」铁无言说，「角漏，喊我。」

韩泥应声：「喊。」

喊，不为勇，为活。活，才还恩。

赖福巡外围，声尖：「丑杂役，别缩！缩了记过！」

韩泥立东角，不缩。不缩，像桩；桩稳，赖福瞥一眼，瞥完走。

走不要紧，瞥是试——试他会不会逃。不逃，才配并肩。

夜深，远岭兽吼连，连像潮拍岸。岸是药山界，界内，外门弟子列阵——阵字不写，写人；人列，列成墙。

墙不要紧，韩泥不在墙心，在栏角。角小，小才藏得住丑翁炼丹之备——备在袖，备在经页，备在清瘴路。

叶青禾在坡下，灯未灭。未灭，他不在近处；在近，像连累。连累不要紧，灯在，脉里培元温着，温着，她才敢立前夜。

他低声对坛，坛在怀贴温：「前夜了。别叫。」

坛不应字，只温。温，像应「守」。

邻床同守，抖：「我腿软。」

「软不要紧。」韩泥说，「扶栏，手稳。」

丑时，静半刻。香落齐，掌纹烫一线——烫，像清瘴诀在经页里醒半分。

醒半分，不够炼；够备。备足，等潮至。

更鼓尽，兽吼又近。近里，一线甜腥扑面，扑面他蒙条，条湿，湿不要紧；湿，才挡毒。

他独眼平，低声道：「前夜过了。下一更，备足。」

备足，不在嘴，在手。手在，栏在，坛在。

兽栏外，火把密。小兽不安，不安则腥重。

铁无言分障，湿草叠石灰。韩泥守东角，角漏，喊他。

赖福巡外围，声尖：「别缩！」

韩泥不缩。

未灭，他不在近；在近，像连累。

远岭兽吼连，连像潮拍岸。界内，外门弟子列成墙。

他低声对坛：「前夜了。别叫。」

同袍运石灰，石灰呛，呛不要紧，呛才挡毒。挡毒靠障，靠灰香，靠手稳。手稳，他扶栏，栏扶同袍，同袍扶他——扶，不是情，是活。

丑时前，他数囊三只，数经页一页，数清瘴路一条。一条，不够战；够备，备足，等104至。至，不在嘴，在角；角在，就不滚。

前夜第二更，甜腥扑面，扑面他蒙条，条湿，湿不要紧；湿，才挡毒。

更鼓沉，沉里，他摸怀玉牌，凉进骨。凉不要紧，掌温——温在诀，不在众目。众目看兽潮，他看手——手不抖，障才稳。

秦霜木鱼轻敲：「前夜别慌。慌，手飘。」

韩泥不答，答在心里——心里不慌，只记。记东角，记备足，记潮至在前头。

前夜第二更，韩泥换湿草两遍。两遍不要紧；两遍，才挡甜腥。铁无言送石灰一包：「撒角，别撒眼。」韩泥撒，撒匀，匀像编筐勒藤——勒匀，才稳。缩了记过！走不要紧，眼在，眼会记过。

同袍问：「你真不怕？」「怕。」韩泥说，「怕也守。守完，才有饭。」饭稳，恩不断。饭在刘婆粥里，也在栏里小兽喘气的声里——声轻，轻不要紧；轻，说明毒还没入深。沈枯芽悄递水：「叶姐说，别逞能。」韩泥接水，不接话。

丑时，廊下静半刻。香落齐，掌纹温一线。温，像培元路稳半分——稳，为清瘴备。管事弟子记：「东角韩泥，前夜在。」在，就不逃。

更鼓沉，韩泥按木牌，牌凉贴胸。下一更，备足。兽吼又近一线，近不要紧，角还在。

兽潮前夜，火在栏外，不在心。心要稳，稳像桩——桩是东角，角在，102才撒线。线白，白像雪，雪在远，远不要紧，远，今夜只守。

韩泥蒙刘婆布条，条粗，粗不要紧，粗挡甜腥。甜腥一过鼻，他仍嗅出毒细——细而记，记99，用在103压。压在远，今夜只喊角漏。角漏，比内墙荣耀小，比活命大。

铁无言分障，障湿草叠石灰，叠法粗，粗能挡一时。「你守东角。活，才并肩108；并肩在远，远不要紧，眼先在。

赖福巡外围，火折照脸：「别缩！」韩泥不缩。不缩，像桩；桩稳，赖福瞥完走。走不要紧，走也记名——名在管事簿，簿上「前夜在」，在就不逃。

叶青禾坡下灯未灭。沈枯芽递水，水凉，怯声：「别逞能。」韩泥接在心里。

丑时，廊下静半刻，香落齐，掌纹温一线。备在101废窑，备在102囊，备在103诀。诀在经页，页在怀，怀贴坛。前夜过了，下一更，备足。备足，手在，栏在，坛在。

前夜第二更，湿草换两遍，两遍挡甜腥。甜腥一过鼻，毒细而记——记99，用在103压。压在远，今夜只守东角。守，不为勇，为活；活，才108并肩。

铁无言分障：「你守东角。」「喊。」喊角漏，漏比荣耀大。荣耀在内墙，内墙要柱；柱要桩，桩要韩泥不缩。不缩，赖福瞥完走，走也记「前夜在」。

叶青禾坡下灯未灭，未灭，他不在近。近像连累，连累不要紧，灯在，脉里培元温着。温着，她才敢立前夜——立，是恩；恩不混，混则七笔乱。

丑时廊下静半刻，香落齐，掌纹温一线。温像培元路稳半分，稳为清瘴备。备在101窑，备在102囊，备在103诀。更鼓沉，按木牌，前夜过了，下一更备足。

前夜第三更，兽吼近一线，近不要紧，角还在。还在，赖福记「东角在」；在，就不逃。不逃，102才撒线，103才压，104才至。至在远，100只守——守不为勇，为活；活，才108并肩。并肩在远，远不要紧，眼先在。眼先在，甜腥扑面，蒙条，条湿，袖囊灰香在指间。指间稳，撒才匀；匀，毒退半寸。

退半寸，小兽嘶声低一线，低像活。活，是小胜；胜在活，不在嘴。嘴要留来喊角漏，漏比内墙荣耀大。叶青禾坡下灯未灭，未灭，他不在近；近像连累，连累不要紧，灯在，脉里培元温着。坛温一线，像应「守」字。

前夜第四更，火把照席，照不见坛——坛在怀，怀贴温一线。备在101窑，102囊，103诀。诀在经页，页在怀，怀在，就不滚。滚在远，守在角，角在，栏在。

心要稳，稳像桩——桩是东角，角在，102才撒线，103才压，104才至。至在远，100只守。守不为勇，为活；活，才108并肩。

韩泥蒙刘婆布条，条粗，粗挡甜腥。甜腥扑面，扑面仍嗅出毒细——毒细而记，记99，用在103压。角漏，比内墙荣耀大；荣耀在白织月袖风，风冷，冷不要紧，冷才真。真在「别乱」，乱不乱，在手；手稳，撒才匀，角才稳。

赖福巡外围，火折照丑脸：「别缩！」不缩，像桩；桩稳，赖福瞥完走。走也记「前夜在」——在，就不逃。

同袍换更，换不要紧，角不换。不换，韩泥仍桩——桩稳，栏稳，内墙才稳。稳，不在白织月袖风，在蒙条，在灰香，在经页。经页在怀，怀贴坛，坛温一线，像应「守」字。守字落齐，101备齐，102撒线，103压，104至。至在远，100只前夜。前夜过了，手在，就不滚。

前夜第五更，火把照栏，照见小兽仍喘——喘轻，轻不要紧，轻说明毒未入深。未入深，因蒙条，因湿草，因韩泥不缩。不缩，赖福记「东角在」；在，就不逃。眼先在，甜腥扑面，扑面仍嗅出毒细。毒细而记，记99，用在103。用在远，今夜只喊角漏。角漏，比内墙荣耀大。更鼓沉，按木牌，前夜过了，备足了。备足，手在，栏在，坛在，就不滚。

备足了，手在，就不滚。

前夜将尽，火把照栏，照见小兽仍喘——喘轻，轻不要紧，轻说明毒未入深。不缩，102才撒线，103才压，104才至。守完了，备足了，手在，角在，坛在，就不滚。
