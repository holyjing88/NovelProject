# 第五十四章 东脉夜战

子时，青岚峰的钟只响一声，像被人掐住喉咙——随后，东脉灵田那边杀声炸了。

火光冲天，赤焰先锋踏田而入，灵稻成灰，灰里混着血。韩铁山立旗冷笑，旗红映甲，像把七日之期提前点着：「青岚宗主，今夜之后，东脉改姓赤！」

柳青鸢剑出，青霜色洗夜，洗出一道冷路：「田在，姓在。人散，姓才改。」

霍镇山盾阵横移，声洪得像雷滚进田里：「镇妖司在！东脉根在！」盾面扛住第一波冲击，弟子惨呼，血溅田埂，血被脚碾进泥——泥仍算东脉的根，根在，战就有意义。

同一时刻，丹堂方向地火阵鸣，淡红阵纹撑起半座堂。

苏念慈立阵眼，金丹灵力灌入地砖，炉不熄。赵玄机率执法堂二十人杀到侧翼，刀卷血，嗓子哑了仍清：「苏首座，赵玄机今夜守门！」

「守。」苏念慈头也不回，袖风送出三颗疗伤丹，丹丸穿门洞，落盾后，落倒地处，落得不浪费一颗——抛丹作战，金丹境的基本功，今夜头一回用在活人身上。

莫长春坐丹堂门外石阶，杖点节拍，不进阵。进阵，阵眼易乱；不乱，也抢苏念慈的戏。他像给这场夜战敲鼓，鼓点不快，却稳。

后阵指挥棚下，周德海令旗举起，又落。

副将急声：「周长老，侧门阵闸已开，只等你令——包抄赤焰侧翼，丹堂可解！」

「不开。」周德海目光阴沉，「防中伏。后阵不稳，妄动者军法从事。」

顾小满从凡阵侧径冲至后阵边缘，浑身泥血，嗓子哑：「周长老！赤焰绕田埂，再不开闸，丹堂阵眼必破——」

周德海看她一眼，淡道：「杂役弟子，回你的路。」

顾小满咬牙，转身再奔，奔回灵田，像要把「侧门不开」四个字奔成一把刀，递给宗主，递给每一个肯听的人。

灵田中央，灵压暴起。

赤焰金丹修士破阵一角，护体灵光如炉盖掀开，压得弟子跪地，压得柳青鸢剑意一滞——

韩铁山远观，笑深：「青岚门底牌，不过如此。」

弟子群情骇然，有人退，有人跪，有人仍咬牙往前，像半块灵石撑着他们不退。

莫长春站起身。

不快，像老人散步；却准，杖点地，袖中青霜剑鸣，剑未全出，剑意先出，霜落刃上，像秋晨的露。

灰袍素，须眉白，在火光里像一张旧符，贴在最要命处。

「将死之人……」韩铁山瞳孔一缩，「也配出剑？」

全场一寂。

寂里，柳青鸢剑意复起，像师兄替她挣回一息。霍镇山盾阵再挪半步，骂声又洪：「莫长老在，青岚还在！」

赵玄机在丹堂门洞听见，刀锋更硬，硬得像终于把「心在谁那边」刻进铁里。

莫长春拄杖前行，一步，两步，像把九月走成剑距。

风腥，田燃，杀声未绝——

下一息，剑要落。

簿在袖中沉，沉得像七十五斤人情，同时结账。

韩铁山远观，笑深，笑里却有一线骇——九月老头杖点节拍，像给这场夜战敲鼓，鼓点不快，却稳，稳得让人不信他只有九月。

弟子群情骇然，有人退，有人跪，有人仍咬牙往前，像半块灵石撑着他们不退。

莫长春望丹堂方向，地火阵仍亮，仍响，像苏念慈在阵眼里把金丹灵力榨到最后一滴。

他低声，仅自己：「苏丫头守得住。赵小子……该还人情了。」

大因果还在下一息，像杖剑出，像全场寂然——夜战只是小开幕，一剑斩金丹才是大还礼的前奏。

赵玄机率执法堂二十人杀到侧翼，刀卷血，嗓子哑了仍清：「苏首座，赵玄机今夜守门！」苏念慈袖风送出三颗疗伤丹，丹丸穿门洞，落盾后，落倒地处，落得不浪费一颗。

莫长春坐丹堂门外石阶，杖点节拍，不进阵。后阵指挥棚下，周德海令旗举起，又落。顾小满从凡阵侧径冲至后阵边缘，浑身泥血，嗓子哑：「周长老！赤焰绕田埂，再不开闸，丹堂阵眼必破——」周德海淡道：「杂役弟子，回你的路。」

灵田中央，灵压暴起。韩铁山远观，笑深：「青岚门底牌，不过如此。」莫长春站起身，不快，像老人散步；却准，杖点地，袖中青霜剑鸣。全场一寂。下一息，剑要落。

柳青鸢剑出，青霜色洗夜，洗出一道冷路：「田在，姓在。人散，姓才改。」霍镇山盾阵横移，声洪得像雷滚进田里：「镇妖司在！东脉根在！」同一时刻，丹堂方向地火阵鸣，淡红阵纹撑起半座堂。苏念慈立阵眼，金丹灵力灌入地砖，炉不熄。

莫长春望丹堂方向，地火阵仍亮，仍响，像苏念慈在阵眼里把金丹灵力榨到最后一滴。他低声，仅自己：「苏丫头守得住。赵小子……该还人情了。」簿在袖中沉，沉得像七十五斤人情，同时结账。大因果还在下一息，像杖剑出，像全场寂然。

围观弟子在田埂上踮脚，踮得像等剑落——剑未落，剑意先至，全场一寂。有人低声：「将死之人，也配出剑？」另一人答：「配不配，下一息见。」
