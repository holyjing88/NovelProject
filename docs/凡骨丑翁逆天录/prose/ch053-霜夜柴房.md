# 第五十三章 霜夜柴房

霜夜，柴房风硬。风从板缝钻进来，贴骨，像克扣的麸皮贴肩。克粮，克不走手稳。

韩泥搬柴至三更。柴湿，沉，肩勒旧印，印深，像五年杂役磨出来的。丙九末排，柴房不能摆坛——坛在棚角，心却近坛。近，像守瓮人日夜都在。

铁无言提灯巡夜，灯远，声近：「柴湿，少劈，别闪腰。腰闪，试手滚。」

韩泥应声：「腰记着，不闪。」

铁无言停半息，灯照他掌：「驿亭褐袍人，又候三日。冬深才赠，赠不白受。」

韩泥独眼平：「不借。」

「袖腥，洗得掉。」铁无言说，「心邪，洗不掉。鼻下静，别凑亭。」

铁无言不再言，灯远。远，像门缝又开一线。线窄，够走。

赖福路过，讽一句：「丑杂役守柴，守出仙缘？」

韩泥不答。答轻，恩重。

管事贴条于柱：「五日后试手凝丹。误者，滚。」

五日，比辨香又近一步。韩泥摸条，条薄，真。真在「仍末」，真在窄路。

同批少年碰他袖：「凝丹你也敢想？」

「不想。」韩泥说，「手想稳。稳了，才不滚。」

「滚了咋办？」

「滚了，恩断。」韩泥说，「恩断，汤就凉。」

少年懂「凉」，闭嘴。

三更后，柴房只剩他一人。斧钝，他劈浅些。浅些，腰不闪。闪腰，试手滚；不滚，路在。

柴湿，劈浅，斧声闷，闷像更鼓。更鼓里，他歇半刻，掌茧裂口渗汗，汗咸，不是血。血不碰沿，沿才净。

他数更鼓，更鼓沉。沉里，柱上五日条像活了半分。活了，试手在望；望不要紧，手要紧。

四更，最后一担柴入库。库门落锁，锁声沉，沉得像更鼓落底。

铁无言灯照柴垛，垛纹直，直像手稳留下的印。他低声：「沉得住，记功在。」

韩泥只点头。记功在，麸皮不克；克粮，克不走手稳。

他回棚，席角潮，沿仍净。邻床鼾雷，雷里，他按席，席贴坛角。角温一线，温短，像应「等」字。等冬深，等缘满，等活计扎出那一刺。

五更将尽，天边白一线。白，像柱上五日条活了。活了，试手在望，末排仍末，路仍窄。

他低声道：「柴沉，手稳。稳了，才验得过凝丹。凝丹，先还热的。」

坛不应字，只温一线，又凉。凉，像瓮仍眠。

卯时前，他又去柴房半刻。霜厚，板缝响，响像牙咬。他摸怀，尚无符，骨寒贴掌。寒不要紧，心记着七笔，便不易飘。

刘婆端粥来，稀，热：「霜夜别空肚。空肚，手飘。」

「不空。」韩泥喝尽。热在，像第七笔又实。

管事路过，验柴垛：「湿柴沉，沉得住，记功。」

记功，像窄路又宽半分。宽不要紧，要紧是五日条在柱。

赖福远处骂：「晦气也配记功？」

管事声冷：「再嗓，试手名单划你名。」

赖福哑了。哑，韩泥肩上的柴却捆得更实。

午后，搬柴过半，肩稳。稳，像坡下站了一上午辨香，站出了实。

编筐午后，藤运到丙九，他领末捆。藤硬，刺扎掌，掌茧裂口渗潮。潮不是血，血不碰沿。

管事验韧：「韧，编得出筐。筐圆，手才不抖。」

韩泥不偷懒。偷懒，麸皮克；克的是粮，克不走手稳。

铁无言廊下见他：「五日条在柱，别贪快。贪快，丹歪。」

韩泥应声：「不贪。贪，恩断。」

「沿净？」

「净。」韩泥说，「血不碰沿，沿才接得住后头缘。」

铁无言退回廊角。

第二趟搬柴，侧门窄，窄得只容半眼。半眼，够看驿亭。亭里褐袍人仍坐，坐得不急。急便邪，不急，像等冬寒够深。

韩泥不凑。凑，像贪。贪，恩易断。

褐袍人袖里黄纸角露一线，露了即收。收，像试他凑不凑。他不凑，只绕远三步。三步，像界——界内末排，界外赠缘。

黄昏，雪粒起，粒细，细像粉。粉落肩，肩沉，沉得住。

他回棚，擦沿。沿滑，滑像祖父韩老栓的手。手不在，滑还在。

沿上无尘，沿上无血。无血，才守不主动滴血的规矩。

坛腹午温，夜凉，像瓮呼吸。吸，不应人；吐，不应灵。

邻床笑：「拜坛拜出邪脉。」

管事路过，声冷：「再嗓，滚去西栏清粪。」

邻床闭嘴。韩泥不辩，只把辱记在心里，与恩分列。列清，心不乱。

夜，他对坛低声：「霜夜柴沉，手要稳。稳了，才接后头五日。」

坛腹温一线，又凉。凉，像应「等」。

外头风息，柱上五日条更真。真在试手，真在末排，真在窄路。路通向凝丹，凝丹通向还。还，先还烫。

丑时，坛腹又温一线，温短，应「柴」字，不应慌。

他摸木牌，牌边磨毛，毛得像心口七笔。笔笔磨过，磨出实。实了，五日后才站得住。

同批少年翻身，梦中喊「滚」。韩泥不扰，只低声：「滚在舌上，不在沿上。」

刘婆门外咳：「试手前夜，别空肚。空肚，鼻下香飘。」

韩泥应：「不空。粥在，姜在记里。」

他低声道：「五日，试手。试得过，关才开第二道。关开了，才学凝丹。学了，先还烫。」

木牌在枕下，与五日帖并放。并放，像命与规矩挨着。挨着，才验得过。

辨香已过，坡下记功在。在，像窄路又亮半分。亮不要紧，要紧是五日条真。

秦霜远廊经过，停半息：「霜夜别空肚。」

她不再言，木鱼在指间，不敲，只托着。托着，像规矩还在耳。

老耿门外挑水，水洒稳：「褐袍人候冬。冬深，才赠。」

韩泥应声，不凑亭。

刘婆又端半勺姜汤，姜薄，热：「霜夜腰寒，姜压一压。压住了，手才不僵。」

韩泥喝尽，只点头。姜热，像灶下还着的一星火。

四更前，他又劈半担。劈浅，斧钝，钝不要紧，腰记着就行。腰记着，试手才不滚。

铁无言灯远前，又嘱：「沿净，别拜邪。拜邪，恩断。」

韩泥只答：「不拜。」

铁无言停半息，灯照他肩：「柴沉，肩还稳？」

「稳。」韩泥说，「稳，才验得过凝丹。」

灯远，像门缝又开一线。

卯时，坡下又起一层霜。霜贴石阶，石阶滑，滑不要紧，站要稳。

外头风硬，风里无栖香全句。全句散了，散成柱上五日条那一息。一息在，试手在，末排在。

霜夜将尽，柴房斧声歇。歇了，手还稳。稳，才配接后头凝丹那一关。

柱上五日条在晨光里更真——真在试手，真在末排，真在窄路。路窄，也要走；走了，才还得上那碗热汤。霜白覆条角，字边起一层细冰，冰化即见「五日后试手」——像倒计时从柱上落到掌上。

他按心口，七笔分列，不混。混了，手就飘；不混，手就稳。坛腹午温夜凉，起伏如瓮呼吸——吸不应人，吐不应灵；起伏在，五日后试手才接得住那一关。关开了，先还烫；烫在叶汤，不在舌上。五日条在晨光里，字边细冰化尽，化尽像试手从柱上落到掌上，掌还稳，坡下才站得住。
