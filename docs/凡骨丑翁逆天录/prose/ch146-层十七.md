# 第一百四十六章 层十七

验息堂前排队时，韩泥袖口还沾着药峰灰。

灰不丢人。丢人的是层虚、手飘。他站队末，独眼半阖，听前面少年们互相问：「今天验谁？听说有人冲十七。」

冲字落地，像要把人抬起来。他不想被抬。抬起来摔得狠。

执事翻册，声平：「韩泥！炼气十六，夺魁挂过牌。今日验十七。」

「到。」韩泥上前一步，开罐捧息。息从丹田起，稳，像筛齐了的材——筛齐才敢称实，实了才敢让人写进册。

执事两指对着他鼻息停半刻，又对日，像怕看走眼：「稳。」

旁头少年倒抽气：「丑骨也十七？」

「稳就行。」执事不抬眼，笔尖落下，「册上添：炼气十七层。」

字丑不要紧，字真。真了，十七才像一根钉，钉在泥里，不钉在嘴上。

管事弟子把木牌递回：「登记完了。别在廊上嚷。嚷了，赖管事爱记账。」

「不嚷。」韩泥接牌，「层在，手在。」

铁无言立在阶下，抱臂，声低：「层高了，别骄。」

「不骄。」韩泥说，「骄了，火不稳。」

「火在后头。」铁无言点一下下巴，「今日只当常活。驴还得驮，渣还得分。」

「驮。」韩泥应得短。

出堂口，赖福正堵着，火折在手里晃：「十六才昨天，十七就来？偷境？」

「验息堂验过了。」韩泥不躲，「偷不得的东西，册上写得明白。」

赖福噎半息，冷笑：「丑杂役也学会顶嘴了？」

「不是顶。」韩泥道，「是交税。十七层也交半文，跟十六层一样。」

「……哼。」赖福把火折一甩，走廊灰起又落。他查席习惯还在，夜里会再来照角——照见厚席就行，坛从不给人见。

药峰活照旧。疤鬃驴立在石桩旁，鬃上旧疤一道，鼻喷白气。韩泥把药渣包往驴背上一扣，包沉，驴蹄稳。稳了，十七层才像实，不像喜。

刘婆端来粥，仍多半勺：「层高了，饭也要稳。空肚逞能，气会散。」

「稳。」韩泥接粥，「谢。」

「别谢我。」刘婆瞪他一眼，「你把话记牢：别显摆。显摆了，外头闲人就来。」

「不显摆。」韩泥说。

同批少年跟上来，探头探脑：「韩叔，你真十七了？就……就这么平？」

「平才活。」韩泥喝一口粥，「雷在心里响没用。响完手还得编筐。」

少年面面相觑，有人小声：「这老头真怪。」

怪就怪吧。怪人不爱抢舌头。

午后他编筐半刻，藤刺扎进掌心，浅，疼得清醒。清醒好——疼提醒他：层在册上，不等于路走完。白织月远廊过了一影，袖风冷，没停。没停就是便宜。便宜他收下，不做声。

沈枯芽在远处探头，不敢近，只比了个「叶」的手势。韩泥略一点头。点头不说话。说话容易把线搅乱。

老耿挑水路过，嗓子粗：「层高了别拜暖。」

「不拜。」韩泥道，「拜暖，路断。」

「那就驮你的。」老耿走远。水桶晃，晃出一点点人情，人情不入册，却入心。

丑时回丙九，他关严门，练鼻下静。粟壳香淡，一字不落。坛腹贴掌，温一线，像听得懂「实」字。

「十七实了。」他低声道，「记着。」

坛不应字，只温。温够了。够了就不往外张扬。

邻床翻身问：「韩叔，十七层吃不吃补？」

「吃稳。」韩泥说，「补在手，不在嘴。」

「那坊市还去不去？」

「去。」韩泥道，「名号是人传的，手是自己稳的。手不稳，名再响也是风。」

邻床又问：「赖福夜里来不来？」

「来就来。」韩泥把席角压实，「席厚，角净。他照得到的，只是席。」

更鼓沉下去，他把木牌贴在胸口。牌凉，凉得像把十七层按进骨头里——按住了，才配去续坊市那半句泥纹。

夜里外头有人路过，故意提高嗓门：「丑骨登十七？太阳西边出！」

韩泥不开门。开门像辩论，辩论费力。力要留给明天驴背上的包，留给坛，留给还没还完的账。

铁无言在门外咳一声，没进屋，只丢半句：「睡。明早驮药。」

「睡。」韩泥应。

应完，他又望一眼坛沿：「手在，路在。喜留远。」

远的是大比，是废窑火，是日后还得起的恩。近的只有这一层实。实了，明天的日子才接得住。

药峰散队时，管事又过来核一遍：「韩泥，十七层，登记无误？」

「无误。」韩泥说。

「无误就好。无误别拿层当鞭子抽人。」管事压低声，「抽了，回头抽到自己脸上。」

「不抽。」韩泥道，「层给自个儿踩路，不给人听。」

管事点头走了。走后，废渣巷日斜，他蹲下再验一把渣。渣正，鼻下静一息。静里他忽然明白：十七不是雷声，是从前十六里一点点长出来的皮——皮厚，才扛得住后头的火。

天边白一线时，他按心口，觉得那碗从前的叶汤又热了一点。热不热，别人看不见。看得见的，是他掀门帘出去时，背脊仍旧直。

直着过日子。直着，十七才不算白验。验过了，下一脚便是坊市尾那面歪牌——牌歪不要紧，效要正。正了，名才续得住；续得住，叶线才接得稳。

药峰坡下有人抽烟点火折，火折一亮，照见他独眼那半边疤：「十七层？你怕不是梦里验的。」

「梦里验不算。」韩泥把绳结系紧，「册上写了才算。」

「册会造假。」

「造假验息堂不认。」韩泥道，「不认，我就还是十六。十六也活，十七也好活。活着比名好听。」

那人噎住，吐一口烟走了。烟散在坡下，散得像闲话。韩泥不追烟。追烟像追名声。名声留给大比台，台还远。

午后他过外堂阶下，分拣半刻。指稳，鼻下静一息。旁边杂役笑：「层高了，手指头还这么笨？」

「笨得点准药。」韩泥说，「巧的人爱耍花样。花样进不了我的罐。」

「你罐里什么？」

「常活。」韩泥道，「常活里藏还。还清了，才谈别的。」

验息登记那一行字，晚上他掏出来看了三遍。三遍不是喜，是怕手飘。手飘了，明天渣会闻偏；闻偏了，泥纹二字就塌。塌不得。塌了，叶青禾那碗汤在心里也烫不稳。

夜里赖福真来了。火折照角，照见厚席，鼻孔里哼：「席下有没有妖坛？」

「有席。」韩泥坐起，「席厚，腿凉。你照完走人。」

「口气不小。」

「口气小过赖管事。」韩泥道，「税交了，层验了，驴拴了。你还能记哪一笔？」

赖福脸色难看，终于退。退后把门带响。响不要紧。坛温在，席还厚，十七还在册上。

铁无言半夜又敲门，只塞半截炭：「明日驮两包。包沉，你别逞。」

「不逞。」韩泥接炭，「沉才像末席该扛的。」

「末席也会登。」铁无言说，「登了别把自己登丢。」

「不丢。」韩泥应。

黎明前他再练一遍鼻下静。香落齐，像把「实」字从骨头里再捺一遍。捺完，他忽然明白：十七层实了，不是给外人看的花，是给自己踩路的砖。砖一块块铺，才能走到废窑那口火。

走到火以前，别踩空。踩空了，还不起的债全会跟着摔。

药峰午歇，管事叫住他：「十七层登记了。活量暂时不加。加了你容易飘。」

「不加正好。」韩泥说，「加活像奖。奖会晃眼。」

「你倒懂。」管事看他一眼，「懂就去驮药。驮完别在廊下站着让人围观。」

「廊下不停留。」韩泥牵驴走。

疤鬃驴蹄声稳，稳得像替他把话落地：层实了，日子仍旧是日子。日子里编筐、分拣、交税、守坛——一样不缺。缺一样，十七就成了空壳。

夜里邻床又问：「韩叔，你拿层换过什么没有？」

「换稳饭。」韩泥说，「稳饭比换笑脸实在。」

「没人请你喝酒庆祝？」

「请我也不去。」韩泥道，「酒散，手飘。飘了，明日验渣会闻歪。」

邻床嗤一声，倒头睡了。韩泥仍坐坛边半刻。半刻里他把「实」字再按一遍——按到指节发白，才躺下。

躺下前低声：「十七按实了。明日不拿层换热闹。热闹散，手会歪。」

他拴好驴绳，低声对空气也像对坛：「明日续摊。今日先把实字按死。」

按死就好。按死了，锤子才轮得到后头落。
