# 馈缘逐章赠礼表（按剧情权重拆分 · v2）

> **原则**：不是每章都送礼，只列 **有赠礼/收恩/靠拢/纳绶/心记** 的章；其余章写修炼、打斗、日常。  
> **链路**：赠礼 → 羁绊/亲密/忠诚/馈缘↑ → 靠拢 → 报恩 → 纳绶 → 眷升（`12`）
