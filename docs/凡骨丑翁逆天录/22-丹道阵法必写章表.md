# 丹道·阵法必写章表（全书 1560 章 · 分章互斥版）

> **用途**：正文落笔前查本表——**本章只写炼丹，或只写布阵，或都不写**；二者 **不同章混顶**。  
> **铁律**：炼丹与布阵 **分开写、交错穿插**；「丹阵双绝」用 **跨章联动**（上章炼丹余韵 → 下章丹气入阵），不在同一章混写高潮。  
> **关联**：`20` 丹道细则 · `21` 阵法细则 · `05`/`16` 剧情 · `17` 馈缘

---

## 〇、分章铁规（必读）

| 规则 | 说明 |
|------|------|
| **互斥** | 一章若有 **S/A 炼丹**，则本章 **不写 S/A 布阵**；反之亦然 |
| **交错** | 同部内尽量 **丹章→阵章→丹章** 交替，避免连写五章全是炼丹 |
| **丹阵联动** | 允许 **上一章末** 留丹气余韵，**下一章开篇** 丹气入阵；各算各章高潮 |
| **符/战/情** | 与丹、阵分立；378 可符辅阵，但 **阵为主、符为辅** |
| **B 级** | 宜写项可并入邻章，但仍遵守互斥 |

**查表口诀**：先看 **§一 互斥总表**「本章标 丹 还是 阵」→ 再查 §二 或 §三 细则。

## 七、单章落笔流程

```
① 查 §一：本章标「丹」「阵」还是「—」
→ ② 标「丹」：只查 §二，本章不写布阵 S/A
→ ③ 标「阵」：只查 §三，本章不写炼丹 S/A
→ ④ 若跨章联动：查 §五，用上/下章衔接句
→ ⑤ 写后勾选：是否混写了另一系统高潮？（忌）
```
