# 第四十三章 盯秤

备检后第五日，麸皮又浅。浅在秤盘，盘在赖福手里。

他晃得欢：「晦气杂役克宗门粮，粮都臭了！」

丙九有人顶：「臭的是你手！」

赖福举鞭：「再顶，明日没麸皮！」

韩泥低头清粪，不接茬。接茬，全丙九饿——饿比辱快，快就先活。

管事路过，秤盘一落：「册上三两，你发几两？」

赖福脸僵：「风干了……」

「风没干你的良心。」管事冷声，「今日起，发粮我盯。少一粒，记名。」

丙九静了一息。一息里，有人看韩泥——看他仍低头，看他手紧。韩泥不抬头。抬头，像邀功；功在饭，不在脸。

发粮场，队排丙九。管事亲执秤，秤杆实，实得像公审里里正的锣——锣定规矩，规矩定饭。韩泥接粮，三两，实。实粮入手，沉，沉得真。赖福旁侧，唇动，没尖。尖了，管事在。

韩泥眼角扫过秤底，底缝沾麸，沾得薄——薄也记。记在心里，不声张；声张，赖福反咬，全丙九饿。

同批少年低声：「忍出来的？」

「忍半月，等这一盯。」韩泥说，「盯了，秤才实。」

「能实多久？」

「实到管事腻，或者实到我滚。」韩泥说，「滚前，先记仇。」

少年闭嘴。闭嘴，也是小胜。有人数粒，数完，脸松一寸。松一寸，丙九都沾光。

韩泥把粮袋系紧，系法与泥岗编筐同——底密，口收，一粒不漏。管事喝：「数完干活。杵着，粮也臭。」

众人散。散时，有人碰韩泥肩：「谢你忍那一回。」

韩泥只答：「谢轻。饭实，是真。」

铁无言路过栏外，声低：「手紧的人，秤上才站得住。」

韩泥不抬头。夸轻，记重。

午后，发粮场空，秤盘仍温。韩泥收粪路过，见赖福蹲秤底，指刮盘缝，刮得轻，轻得像风干了粮。他没喊，立三息，等管事折返。管事来时，韩泥只道：「盘缝有麸。」

管事掀盘，麸落掌心，脸色沉：「赖福，明日你发，我盯。再刮，记名。」

管事记名一笔，笔沉，沉得像公审锤落半寸——半寸不响，响在秤上。赖福退时，眼里阴，阴韩泥记住，不顶。顶，全丙九饿；饿急，汤就凉。

赖福脸白，骂不敢骂，退半步。同批少年碰韩泥肩：「你早看见？」

「看见，不等管事，没用。」韩泥说，「有用的是饭实。实了，才还得上。」

刘婆端粥来，粥仍稀，勺却立得住半息——半息，比昨日强。她瞪赖福：「再克粥，盆扣你头上。」

赖福退半步，骂「疯婆」，脚真退。退，胜在灶边。韩泥喝粥，不道谢。谢轻，恩重——宗门第七笔，灶边另记。

午后，赖福报复，掷单加倍：「兽栏冬角，韩泥再加半刻。」

韩泥只答：「加。」

管事折返，声沉：「加倍？册上没这一条。」

赖福脸白：「冬角脏……」

「脏，全丙九轮。」管事说，「单上加字，先问我。」

赖福收单，收得哑。哑，小胜。韩泥铲下仍实。实，让管事肯盯第二回。

同批有人低声：「韩泥，要是管事不盯了呢？」

「不盯，仍忍。」韩泥说，「忍到他又盯，或者忍到滚。滚前，记全仇。」

那人闭嘴，干活去了。酉时，韩泥仍去渣场半刻。渣分三堆，香篓齐——粮实一日，手不能松；手松，明日秤又浅。

同批少年跟来，捏鼻：「你还做？粮都实了。」

「粮实一日，手不能松。」韩泥说，「手松，秤又浅。」

渣场灰热，灰里苦甘分列。甘短，苦长——分列，才不混；不混，嗅渣才准。准了，正测辨香才站得住末排。少年不学，走了。

刘婆端晚粥，稀，热：「盯秤这日，多喝一口。一口，顶三口克扣。」

韩泥喝尽，不谢。第七笔恩，又落半勺——勺少，热在。赖福门外骂：「疯婆子偏晦气！」

刘婆回骂：「偏的是人，不是秤！」

韩泥不掺。舌要留来记三两实粮落袋那声沉。发粮场散后，秤盘空着，空得像赖福的嘴——嘴还硬，手已虚。

夜，赖福踹棚柱，骂：「盯秤盯到老子头上！」

韩泥不辩。舌要留来记秤——记今日三两实，记明日还盯不盯。邻床问：「你真不闹？」

「闹，饭少。」韩泥说，「饭少，汤还不了热。」

「还什么汤？」

「叶丫头那碗。」韩泥说，「热的。凉了，还乱。」

邻床又问：「七笔是什么？」

「泥岗六笔，刘婆一笔。」韩泥说，「分列，不混。」

邻床懂「热」，闭嘴睡了。韩泥摸坛沿，沿温一线，温得像灶边半勺粥。管事巡夜，敲柱：「明日仍我盯秤。赖福，少玩秤。」

赖福应声虚。虚，韩泥记——记第二回盯秤，记三两落袋那声沉。同批有人梦呓：「饭……」

韩泥睁眼，替盖袄：「排着。秤实了。」

梦呓止。止，今晚丙九睡得着——睡得着，也是小胜。外头风硬，吹得秤盘空响一声。响在耳，记在心——响的是赖福的虚，不是韩泥的滚。

同批有人低声：「盯秤这日，你掌心不抖？」

「抖。」韩泥说，「抖在袖里。秤盘实了，袖里才紧。」

「紧了呢？」

「紧了，明日仍盯。」韩泥说，「仍盯，赖福才虚。虚了，丙九才多一口实粮。」

他低声道：「秤实不是闹出来的，是半月忍出来的。忍到管事肯盯，饭才实。」

坛沿温一线，温应「实」字，不应骂。骂记着，熬到瓮醒，熬到还。

翌日卯前，发粮场仍排丙九。管事再来执秤，秤杆实，实得像昨日那声锣没落。韩泥接粮，仍是三两，实。实粮入手，他眼角又扫秤底——底缝净，净得像告诉全棚：盯过一日，虚字才短半寸。

丑时，刘婆门外咳：「盯秤这日，明儿仍多半勺。」韩泥点头：「心不少。」心不少，秤才实第二日；实第二日，恩才不断。

酉时渣场，他把三堆再理一遍——粮实一日，手不能松；手松，明日秤又浅。理齐了，渣味分列，苦甘各归各堆；各归各堆，鼻下才静。静了，正测辨香才站得住末排。还，先还烫——烫在叶丫头，不在舌上。

同批少年梦呓又问粮，他替盖袄：「排着。秤实了，梦就紧，手才不抖。」

外头赖福骂远，骂远，今晚丙九睡得着——睡得着，也是秤实的小胜。刘婆门外又咳半声，没说话——没说话，像说心不少，勺才能多。

第二日发粮场，底缝净了。净了，像告诉全棚：虚字短半寸，短半寸，够活。赖福那侧秤仍浅半分——浅在他，不在全棚。全棚有实粮一口，便是小胜叠小胜。

备检木牌在怀，他摸牌边，边磨毛，毛得像心口那笔烫——烫记着，不泄；泄了，秤实也白实。同批有人凑近半尺，问：「你为什么不谢管事？」韩泥只答：「谢在粮里。粮实了，丙九都少饿半口。」

管事盯秤，三两落袋。掌心不抖——抖在袖里，秤盘实了，袖里才紧。

赖福旁侧，唇动，没尖——尖了，管事在。韩泥不抬头，只记三两落袋那声沉。记沉了，才配正测末排仍到，才还得了叶丫头那碗烫。

丑时，坛沿温一线，温应「实」字，不应骂。木牌在怀，帖在柱——帖在，辨香在，盯秤这日就不白忍。

发粮场第三更，风硬，秤杆仍实。实了，仇记着，恩也记着——记着，才配正测末排仍到。丑时末，他把三两落袋那声沉，又在心里记一遍；记一遍，路就不绝。

渣场风硬，他把三堆又理一遍——理齐了，鼻下才静。静了，辨香才站得住末排；站得住，盯秤这日就不白忍。

赖福虚了，虚不是终——终在正测辨香，在还叶丫头那碗烫。虚字丑，丑翁专精在忍；忍到管事肯盯，饭才实。备检记名在怀，手在，路在，辨香在。
