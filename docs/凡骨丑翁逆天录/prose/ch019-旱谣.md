# 第十九章 旱谣

十月尽，井浅了。

韩泥早起上山采药，篓绳旧了，系紧——系法仍是爹教的：绕两圈，勒紧，留半寸尾。后山旱，苦蒿枯了大半，蛇藤也蔫。胳膊伤还没全好，坡滑，起根慢，一株一株拔，泥沾在指上，红得像记着账。

井浅得桶底蹭泥，泥腥混着水腥，喝进喉，喉里也像旱。他打水，绳放长，水起，浑，少半指。少，也得打。不打，米煮不成；不煮，臂伤也好得慢。

井台辘轳吱一声，吱得像骨节在叫。绳勒掌心，勒出红印——印浅，手还实。慢不要紧，要紧的是村还要活。村道龟裂，娃不跑；一跑就起尘，尘像灰雪。

马春花骂天，骂到一半咽了——天骂不动，骂韩泥更动不得，憋着就生谣，谣比旱还快。井台边老汉蹲着，桶排成队，低语像旱虫爬：「再不下，秋种全废。」

「废就废，废也比祭血祠强。」

「强不强，嘴说了不算，里正说了才算。」

韩泥路过，脚步没停。停，成靶。他拎桶，桶沉，沉得像榜末那个名——名在末，手还实。

午后告示桩前聚人。不贴赖扒皮的黑纸，贴的是口述。口述最毒，毒在不用墨，用嘴就能杀人。周嫂嗓门大，像替天传话：「后山土地干裂，里正求雨，求不动！龙王爷不管泥岗！」

「有个过路货郎说，西边旱了三年，有人祭血祠，雨就落！落得像瓢泼！」

「血祠要什么？要晦气！要绝户！要寡妇送汤！汤一送，旱就缠！」

人群「嘶」一声，像牙酸。酸里带怕，怕里带贪——贪雨，贪替罪羊，贪一个能骂能打的出口。

赵绣春站荫下扇风，扇得闲，像看戏不花钱：「祭什么血祠？叶家药铺不就是小血祠？抓药抓晦气，送汤送灾星！」

众人笑，笑短，笑里带怕。怕旱，怕死，怕比死更慢的渴。渴让人信邪，邪让人找近的人下手。

韩泥停步，独眼平，筐不晃：「旱是天。祠是人编的。人编来吓人，不是救雨。」

赵绣春眉一挑：「你懂祠？绝户翁成半仙了？」

「我懂渴。」韩泥拎筐转身，「渴了，井还要打。打了，村才活。活不靠祭，靠手。」

就三句。不辩邪，只辩活。活字落，像石进浑水——水仍浑，石在。石在，就有人噎。

井台有个妇人抱着空碗，把碗口对着天空，像能接下一滴似的。接不住，她才把手缩回来，低声道：「手……手还能打。」说完像怕自己太大胆，赶紧低头看地。韩泥从她身边过，没停，也没笑。笑轻。能把话说出口的人，已经比血祠近了一尺。周嫂噎半息，脸青，啐他背影：「晦气翁装神仙！神仙怎在榜末？」

有人低声接：「榜末……也是人打的井。」

周嫂瞪眼：「你替他说话？」

那人缩回队列。队列仍浑，浑里韩泥背影已远。远像井绳——绳沉，仍拎得动桶。

村头老槐下，货郎早走了，走前留下一句像咒：「血祠不开，雨不开；开了血祠，先死晦气。」

咒落在旱土里，土干，咒不干。韩泥低声道：「记着。封的不是雨，是谣的根。根在怕，怕在旱，旱在天——天不许人赖，人偏要赖。」

沈枯芽在巷口，手里空篮望天，望得像等雨。韩泥停半息：「娃，别信血祠。」

芽儿声小：「我不信……可娘咳，赵婶说叶丫头邪……」

「叶丫头不邪。邪在嘴。嘴信谣，药就抓不着。」

芽儿点头跑了，跑得像一粒米落进沈家锅——锅小，米真，真米能活，活就不靠祭。

傍晚叶家药铺门开一线，搁一只陶罐，罐口湿，湿的是凉茶。无人声，像叶青禾也知道：旱天送汤不如送水；水也能救命，命在，铺就在。韩泥不取罐。取了，坐实血祠谣——谣会说「寡妇送水也牵扯」。他只隔门道：「心领了。」

门内静一息，叶青禾声低，像从秤后传来：「旱天，汤烫不得。水凉些，别渴着。」

「嗯。你也别熬太晚。」

「秤还要摆。摆实了，明日才好说话。」

「说话的是你。我坐着听。」

门内秤砣落一声，沉。沉后再无声，像应完了。应完，铺还在，理还在。

韩泥退后两步才走。旱天里这点凉茶香，淡得像针脚——挡不了全村嘴，却能提醒他：谣说血祠，叶家给的仍是水。水比血干净，干净就值得记。

夜来风干，像磨刀。磨刀不砍人，砍人心。村童不敢念谣，怕。怕里有人低声学货郎那句：「血祠开，雨下来……」

学的人自己先卡住，像咬了舌——舌疼，比血祠轻。韩泥独坐灶前，听一句记一句。记不进心的，记进眼。眼记着，候还，候破。破不是杀人，是破谣，破到明日公审。

旱加重，谣也加重，加重到今夜像一根绳，往明日公审上勒，勒紧了，钩就出来。他摸坛沿，坛腹回一丝温，像吸了「血祠」二字又吐不出。吐不出不要紧——瓮没醒，醒不在今夜。今夜只一件事：井还要打，心还要按，叶家灯还不能灭。

他舀半桶浑水，浇门口一小畦枯姜苗。苗是叶家曾送姜的根，根活着，「趁热」就不死。苗颤了颤，像在旱里点头。点头，就是一口气松下来。

井台边有人窃语血祠，说完又闭嘴——闭嘴不是信，是怕。怕正好给公审留缝。井壁裂一线，裂得像干渴的唇；唇裂了，还得打水。打，才活。

马春花路过，瞥他门口那畦姜苗，嗓尖又收：「还浇？浇了也长不活。」

「根在，就活。」

「根在，旱更大！」

韩泥不辩，只舀水，水浑，仍浇。浇完苗颤，像点头。马春花愣半息，啐地：「晦气翁跟苗说话！」

「苗不骂人。」

她语塞走了。走远了仍骂天——天骂不动，人也骂不出雨。雨没来，苗还在，就是那口气。

里正巡夜，敲锣一声，哑着：「血祠再拜，抓人。抓人，连坐。」

韩泥担水路过，脚步没停：「井是村的。拜邪，关不了井。」

里正灯影晃过他肩，像照见榜末名——名在末，肩还直。直，就是旱夜里的小胜。

天将亮，姜苗叶尖凝一粒夜露，露少，真——真得像「趁热」在旱里也没死绝。露滚落进土，土仍裂；裂不怕，根还在。

他又多打半桶，放给邻舍寡老婆门阶，不叩门——叩门成牵扯；放水成活路。寡老婆开门缝见桶，没出声，只还一只小空陶罐，像说「我接住了」。韩泥收罐不收谢——谢轻，他记。记的是旱天不拜邪，也给人一口活路。

旱风硬，血祠词出了街。他不碰邪——碰像求，求轻。他多担一担水，水给活，不给谣。血祠鼓声远，远得像催命；要紧的是井台还开着，叶铺秤还在。

短签在桩上，风撕角，撕不掉名。名在仇苗里，苗还在，公审前汤还得立住。旱谣在街，姜苗在田。田旱，心不能旱——心热，秤还在，坛还在，公审前汤就不凉。井台排队仍长，像全村还信手，不信嘴。他担水回屋，水浑，仍煮稀粥。粥稀，像旱天的命——命稀，还在。还在，血祠谣就短半寸。

锣声远，远里姜苗仍颤，像点头。天将亮，云层压极低，低得像一口扣在村上的锅。锅不开，雨不落；不落，手还得打。

他低声道：「旱还在，手还实。手实，公审前汤就不凉。血祠谣在街，姜苗在田——田旱，心不能旱。」坛沿一线温贴着虎口。瓮没醒，他不喊醒。醒在后头，不在这一更。桶放下，他仍准备天一亮去井台——验的不是雨，是村子还肯不肯靠手活着。
