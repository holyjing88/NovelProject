# 第七十六章 药山深行

第二日，药山边门再开。管事弟子验牌，声沉：「仍只捡。深一步，不许过界碑。过了，除名。」

药山深行，深雾起一寸，雾里界碑影钝。碑影钝，钝像测灵石——石秤命，碑秤规矩。规矩在，脚停；停，不是怂，是不贪活药。

深堆边，他捡一撮灰，灰里辛淡。辛淡不要紧，鼻下有根，根细，细才辨得出沉腐浮。辨得出，78嗅诀才萌芽，81收集才有着落。

白织月主道过，袖风冷：「丑杂役也深行？深行，别越界。」韩泥侧身让半尺：「不越。捡渣，沿净。」越界不要紧，手在就不滚。

韩泥应声：「不过界。」

界碑在雾后，碑石青，青得钝。钝，像测灵石——石秤命，碑秤规矩；规矩在，手才在。他背筐上坡，坡滑，霜白。霜不要紧，二层气沉足底，足底稳半分，稳像踹台按实。深堆在坡腰，堆边有弃炉基，基破，破里灰热半分——热不是火，是余药性。余药性淡，淡不要紧；淡也要记。韩泥鼻下静，静里闻见苦辛并一线。

并一线，像叶家药远路飘来半息。飘半息不要紧，恩在胸，胸有恩，手才不飘。他拣一撮，指稳，入筐。入筐前，再辨：辛沉，苦浮，腐在最底。底真，浮假；假剔净，筐才净。雾起一寸，一寸不要紧，一寸也要辨。辨错了，深行白来。白来不要紧，手在，名在，路在。坡角有杂役弃牌，牌裂，字浅：「过界除名」。

韩泥瞥一眼，不近——近，像贪。贪，路断。铁无言立雾外，声低：「深一步，鼻下更静。静了，才听得见渣说话。」

韩泥应声：「静在鼻。」

铁无言又道：「听见别叫。叫了，像求。只求手稳，不求丹名。」

韩泥分拣，又半刻，筐底多一撮黑灰。黑灰里一点亮，亮像未烬的引火——引火轻，记重；记重了，废渣路才走得通。午间，管事来验，验筐，验足迹。足迹不过界碑，界碑外一步无印——无印，是小胜。「丙九韩泥，深捡净。」管事记簿，笔丑，真。赖福立雾外，眼细：「深也捡？捡出邪药，连坐丙九。」

韩泥不接茬。接茬，全丙九饿。他低声道：「只捡渣，不碰活药。」

白织月主道过，笑声远：「杂役进深雾，像老鼠进丹房。」

韩泥不辩。辩，费舌；舌要留来记，留来忍。忍里藏着刃，刃对着三年约，对着白织月，对着内门的眼。午后，雾散半寸，散出界碑影。影斜，斜像在提醒：再一步，除名。除名，恩断；恩不断，他就停。停，不是怂，是规。规在，坛在；坛在，根在。根在，才还得了叶汤。他拣最后一撮，撮小，小不要紧，小里辛正。

辛正，鼻下静，静里掌纹那路微颤——颤像嗅诀将芽。将芽不要紧，芽在鼻，不在舌。舌响，香飘；香飘，煞骨门就嗅得到。回坡下，筐交管事，管事称半斤，半斤对册，对，记「丙九韩泥，深捡净」。刘婆提篮过，篮里饼硬：「深行饿了？」

「饿了。」韩泥说，「饼就行。」

深行，别越界。

捡渣，沿净。刘婆塞饼，饼糙，糙得掌印深：「别谢。记着。深行别贪，贪了婆也救不了你。」

韩泥点头。谢轻，恩重。恩在饼里，也在「别贪」二字里——二字像铁无言，像秦霜木鱼，像坛沿净规。回棚，他把深捡样分三纸包，包丑，包实。实了，才记得住；记住了，嗅诀才萌芽。邻床问：「深雾怕不怕？」

「怕。」韩泥说，「怕也捡。捡了，才有路。」

少年闭嘴，像也学会忍。忍不是怂，是活路——活路通向冬深杂役，通向四层可见，通向丹路开。丑时前，他廊下练半刻鼻下静。粟壳香淡，掺一点药山深灰的辛——辛淡，淡不要紧；淡也是芽。芽在鼻下，不在册。册写仍末，鼻写嗅——写给自己，不写给别人。他按席角，席严，坛不露。坛腹温一线，贴掌，像应「深」字。应完，他低声对坛：「深行了。界没越。嗅芽在鼻，不叫。」

韩泥只答：「只捡渣。沿净。」

赖福哼一声，脚步远。远不要紧，筐已交，名已记——记了，深行才算完。秦霜远廊经过，木鱼轻敲一声：「界在，心在。心越界，鼻下香飘。」

韩泥不答，答在心里——心里七笔分列，深行是活，活是路；路通向白织月再嘲，通向冬深杂役续。他独眼平，手不抖：「深捡了。界没越。嗅诀将芽，冬深续活。」

韩泥只答：「不恋。记着。」

夜禁脚步远，风里一点辛回，回进鼻下，回进二层境里。境微，不要紧；微了，才接得住将芽那一日。散队后，药渣场角风硬。他多留半刻，把深捡样并入白日样。样多，多不要紧；多，才记得住药性深浅。同值杂役嗤：「深雾也捡？捡出命来。」

韩泥不辩。辩，费舌；舌要留来辨渣，留来守坛。他只答：「命在手。手在，就不滚。」

管事夜巡验棚，灯照席角，照见净沿不见袋。不见不要紧，沿要净。沿净，坛才藏得住。他低声道：「深行了。坛应了。明日白织月嘲，舌仍不留。」

舌仍不留，手仍要稳。稳了，嘲才白挨；白挨不要紧，记仇苗，也记嗅芽。深行第二趟，雾起半寸，界碑影斜。韩泥驻足，足离碑三步，三步是规。规在，坛在。他拣炉基边余烬，余烬少，少里焦回正。焦回正，鼻下静，静里掌纹微颤——颤像四层影在等。管事验足迹，足迹不过碑，记「深捡净」。记净，是小胜。

「温听雨主道远，裙素，没近。没近，不要紧；要紧是手不越界，筐不藏活药。捡出邪，连坐。」韩泥低声道：「只捡渣。界没越。」铁无言声低：「深一步，心别贪。贪炉砖，像偷火。偷火，路断。」韩泥应声：「不偷。只走路。」午后散雾，坡下筐过磅，半斤对册。对，记功。记功轻，恩重。刘婆端粥，仍多半勺：「深行腰还直？」「直。」韩泥说，「腰直，手才稳。」

粥稀，韩泥喝尽。喝尽，才配回棚守丑时，才配嗅诀将芽。黄昏，深捡样分三纸包，包实，塞角柜。柜锁丑，锁像规——锁住了，才不示人。邻床问：「炉基扒不扒？」

「不扒。」韩泥说，「扒了，记名。记名，恩断。」

丑时前，廊下练半刻鼻下静。三包并息，并出焦回一线——线轻，记重。他低声对坛：「深行了。界没越。嘲在明日，舌仍不留。温，像药山深灰那一点引火。更鼓沉，独眼平，手不抖：「深捡了。嗅芽将起。冬深续活，四层将显。」更鼓沉，沉里，韩泥按木牌，牌凉，掌温。深行归途，坡下雾散半寸。

散出界碑影，影钝，钝像提醒——再一步，除名。沈枯芽药圃角弯腰，弯得像没看见。没看见不要紧，韩泥也不近——近，像求；不求，只记半块灵石恩在还路前头。散队后，兽栏仍腥。腥不要紧，二层在掌，铲下更稳。稳，像踹台那一掌——掌在，渣在，路在。记着。」

柱上条角白一线，三年约在怀。怀凉，掌温——温在诀将芽，不在白织月笑里。

「坛应了。嗅芽在鼻，冬深续活在远。」坛腹应温，像药山深灰那一点引火。引火轻，记重；记重了，废渣路才走得长。廊下风硬，他把木牌系紧一线，系得像把仍末摁进掌心。同批少年远避，避像避晦气，晦气遮瓮，遮住了才稳。编筐角藤刺仍在，他偏手，刺浅些，沿才净，坛才藏得住。管事夜巡验棚，灯照席角，沿净，像腌菜坛，不像邪。

更鼓沉，沉里，韩泥按木牌，牌凉贴胸。76章手在，名在册，坛沿净。舌不留，鼻下静。恩辱分列，不混。活长了，才还得了叶汤，才接得住门缝灯。

「「界没越。」刃不亮，先记。记完了，才配再嘲那一句，才配四层将显那一境。刃不亮，先记。界没越。更鼓二更，他廊下练半刻鼻下静——静齐，才配接匿丹备料，才配守坛。」
