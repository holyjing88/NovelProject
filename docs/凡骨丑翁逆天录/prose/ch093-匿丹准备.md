# 第九十三章 匿丹准备

霜降后。

韩泥不再白天上山。白天清栏、分拣、编筐，活计照做，做稳了，才挪出丑时与雨夜。

匿丹准备，首在藏料。青芯、褐渣、灵泉，分三处：袖袋、筐底夹层、兽栏干草下浅坑。坑浅，浅不要紧；浅，才像藏粮。

铁无言药渣场角，声低：「你要试，别用内门药。用渣，用灰，用捡的。」

韩泥应声：「用渣。」

用渣，像晦气杂役该用的。该用的，才不惹眼。

第二备，藏炉。瓮底泥炉拆成三片，片入坛席下，席厚一层。厚，不为保暖，为挡目。管事夜巡，灯照席角，照见厚席不见炉。不见，才匿。

第三备，藏诀。墟翁丹经残页，页黄，字丑，他从废经片里抠出，抠出不示人。示人，赖扒皮来。来不要紧，页在怀，怀贴坛温。

叶青禾那边，他不敢近。近，像连累；连累，叶丫头说过「别来丑」。丑不要紧，恩要还——还在暗处，还在门缝，还在她怕黑那一线。

他托沈枯芽递一句口信，信无字，只有半块干饼：「冬夜别灭灯。」

沈枯芽怯，点头，点头像懂「怕黑」——怕黑是恩，恩要还，还不能说。

韩泥备第四件：温汤囊。囊小，陶缝，缝像叶汤碗沿。沿热，热在记里；囊空，空不要紧，成丹后装温，装门缝留灯那一路。

赖福查库，库门响：「丙九，领麸皮！」

韩泥排队，队末，独眼平。赖福眼细：「你袖鼓，鼓什么？」

「木牌。」韩泥说，「三年约，贴身。」

赖福想掀袖，管事喝：「查库快些，误时克半日！」

赖福哑半息，掷麸皮，皮轻半分。轻不要紧，韩泥不接谢——谢轻，记在心里。

夜，他铺经页于席，页上「培元」二字浅，浅而真。真，他默念，念不进舌，念进鼻。鼻下嗅，渣里辨，辨出投序。

投序写在筐底，写用柴灰，灰干，扫即无。无字，才匿；匿，才备得成。

刘婆补肘过，肘布厚：「你席盖又加一层？」

「风硬。」韩泥说，「沿净要紧。」

刘婆哼：「沿净，手稳。手稳，饭稳。」

韩泥点头。饭稳，恩不断；恩不断，匿丹才值得备。

邻床问：「你丑时还不睡？」

「编筐。」韩泥说，「筐卖钱，钱买米。」

钱买米，也买匿丹之静。静，在丑时，在雨夜，在席下坛温一线。

他低声对坛：「备齐了，不叫。叫出来了，路断。」

坛腹温，贴掌，像应「备」字。备，不是炼；炼在下一回，炼在门缝前，炼在还恩前头。

更鼓二更，雨意又来。雨里，他把温汤囊挂床柱，柱旧，挂得住。挂得住，才留得住门缝那一灯。

赖福脚步停门外，停半息，像嗅。嗅不要紧，席严，页在怀，料在三处。三处散，散得像常杂役藏粮。

韩泥独眼平，低声道：「备过了。下一回，炼。」

炼，不在众目，在暗处。暗处还恩，才配叶丫头那碗烫。

晨雾里，他分拣药渣。指稳，稳里辨青褐——辨在鼻，不在眼。眼昏，像赖扒皮盯梢。

赖扒皮查库，库门响：「丙九，领麸皮！」

韩泥排队末席，独眼平。赖福眼细，细得像秤：「你袖鼓，鼓什么？」

管事弟子冷声：「查库快些！」赖福哑半息，掷麸皮。

沈枯芽怯，递口信：「叶姐……怕黑。点头像应「备」——备在暗处，备在门缝，备在还恩前头。」「编筐。霜降后第三夜，雨意沉。愧线「别来丑」在记里，烫半分。分列——愧是愧，恩是恩。叶青禾在坡下，脉弱，沈枯芽悄递：「叶姐说，别来丑。」韩泥不接话，接在心里。话在还，不在嘴；还在缝，还在灯。

灯未留，料先备——备齐，才叫炼。老杂役问：「你编筐编到丑时？」「末席编。」韩泥声平，「编完，才有饭。」饭稳，恩不断。编筐末藤勒指，指疼，疼不要紧；疼，手才记牢投序。经页边添小划，划完即抹，抹得像无字。无字，才匿。匿，才备得成下一回暗炼。更鼓沉，韩泥按木牌，牌凉贴胸。匿丹准备，首在分藏。

韩泥把青芯分三处：一处席下坛边，一处筐底灰里，一处兽栏干草下。下处最险，险不要紧，险才像常杂役乱藏。乱藏，赖扒皮才不细查。」

经页页边，他用柴灰写投序，写毕即抹，抹净，像无字。无字，才匿；匿，才炼时不叫。叫出来了，路断；路断，叶汤那碗烫就还不了。

叶青禾在坡下咳，咳轻，沈枯芽送刘婆姜汤，汤热，热像95将赠之温。韩泥不近，不近不要紧，近像连累。连累在赖扒皮眼，眼细，细像秤。

铁无言过渣场，脚停：「你备什么？」「备栏。」韩泥说，「栏清，瘴少。」答非所问，问才止。止不要紧，真答在怀——怀里有页，有料，有投序。

赖福秤麸皮，麸皮少半勺，少不要紧，少在赖福手，不在韩泥筐。筐满，满像常活。常活里，韩泥丑时编筐，筐卖钱，钱买匿丹之静。静，在雨夜，在席下坛温一线。他按坛，低声道：「备齐了。下一回，炼。」炼在暗处，暗处还恩，才配门缝那盏将留之灯。灯未留，料先齐——齐，不叫，坛沿净，席严。

匿丹准备，料分三处，序写筐底。筐底柴灰写字，字干，扫即无——无字，才匿；匿，才暗炼时不叫。叫出来了，赖扒皮记过，丙九连坐，叶汤那碗烫还不了。

叶青禾在坡下，脉弱，沈枯芽递话：「别来丑。」丑在脸，不在恩；恩在还，不在嘴。嘴要留来编筐，留来清栏，留来守坛。坛在席下，席严，赖福嗅不到页边药香。

铁无言过，脚停半息：「备什么？」「备栏。止不要紧，真答在怀——怀里有页，有料，有投序，有95将赠之温。温在远，备在近。近齐，才叫炼。丑时编筐，筐卖钱，钱买米，也买匿丹之静。静，在雨夜，在霜降后第三夜。雨意沉，愧线「别来丑」烫半分。恩不断，94灯才留，95丹才赠。

霜降后第三夜，雨意沉，沉不要紧，沉才藏备料。备料不在一处，分三处散，散像杂役藏粮——粮乱，赖扒皮才不细查。细查不要紧，席严，页在怀，坛不露。不露，沿才净；净了，94灯才留，95丹才赠。赠在远，备在近；近齐，才叫炼。炼不在众目，在暗处；暗处还恩，才配叶丫头那碗烫。

烫在记，不在舌；舌要留来编筐，留来清栏，留来守坛。编筐丑时，藤硬勒指，指裂不要紧，裂像坛沿朱红——朱红在记里，不在席。席下按坛，低声道：「备齐了。」不叫，坛沿净，路不断。路不断，下一回暗炼；暗炼，雨夜为幕，幕在95，不在93。93只备，备齐，手在，就不滚。药田边，沈枯芽探头：「韩叔，叶姐说别来。

」别来不要紧，别在丑，丑在脸，不在备。备齐，94留灯，95赠丹。丹在远，料在近——近齐，手在，就不滚。滚在钱戾衡脚，脚在远，约在怀。怀贴身，像守坛沿。匿丹之备，备在雨夜第三更。更鼓沉，韩泥编筐，筐勒硬藤，藤刺扎掌，刺浅不要紧——刺浅，沿才净。净了，坛沿才藏得住三处料：席下一点，筐底一点，干草下一点。

点散，散像杂役藏粮；粮乱，赖扒皮才不细查。细查不要紧，页在怀，投序写筐底，柴灰写字，扫即无。」韩泥不接话，接在分列——愧是愧，恩是恩。

恩在还，不在嘴；还在94门缝，还在95将赠之温。铁无言过渣场，脚停半息，声低：「手别飘。」「不飘。」韩泥应，应完继续分拣，分拣像清栏——清栏他熟，熟才稳。稳了，下一回暗炼；暗炼，雨幕为帘，帘在95，不在93。93只备齐，备齐，不叫，坛沿净，路不断。

废渣巷日斜，他验渣，渣正，手才静。
