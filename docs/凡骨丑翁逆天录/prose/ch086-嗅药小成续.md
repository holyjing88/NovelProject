# 第八十六章 嗅药小成续

小成后第七日，药渣场加验。

管事贴条：「丙九韩泥，辨性加验。十堆混灰，错两堆，记名。」

韩泥读条，独眼平。错两堆，恩断边缘；边缘不要紧，鼻清才要紧——清在，手才准。

十堆灰，灰混，混里各有沉腐浮。他分拣，指稳，不需细想，鼻先落。落一次，指一次，稳一次。

第七堆，腐底藏辛，辛浅，浅不要紧，浅也要剔——剔出，管事眉松一线。

第十堆，假香浮厚，浮厚也要剔。剔净，十堆全对——全对，管事记簿，声平：「丙九韩泥，加验准。」

加验准，是小胜。胜在鼻，不在嘴。温听雨立风口，裙素，袖香温，她没近，只远看一眼——看一眼，也像验；验完，袖风一转，走了。

走了，留半句在风里，风不清，韩泥不追——追，像求。不求，只记：记她远验，记她傲里少一分冷。

铁无言抱臂立侧，声低：「小成续了，别飘。飘了，加验白来。」

韩泥应声：「不飘。沉腐浮，各归位。」

铁无言又取一撮未知灰，灰少，少不要紧，少更要准：「这撮，你闻。」

韩泥鼻下一息，指落：「沉七，腐二，浮一。」

铁无言分秤，分完，点头：「对。续成了，才接得住暗修三层。」

三层在丑时，不在今日。今日只续鼻——续长了，根才粗；根粗了，境才破。

午后，外堂洒扫。阶尘少，帚仍稳。温听雨擦栏，巾湿，二人仍各做各的。

她忽然停半息，声清：「加验十堆全对？」

「全对。」韩泥应，不多字。多字，像攀。

温听雨唇线停半息：「全对不容易。不容易，像手稳的人。」

第三回「手稳」，韩泥仍不谢。谢轻，记在心里——心里缘苗温一线，心里嗅药小成续，心里四层影在等。

她巾一收：「别攀温氏门。门在丹，不在缘。」

「不攀。」韩泥说，「各做各的。」

各做各的，是界。界在，恩苗才不邪。邪了，秦霜木鱼就敲第二声。

刘婆端粥，仍多半勺：「加验过了？」

「过了。」韩泥说，「十堆全对。」

「全对怕什么。」刘婆哼，「饭稳，恩不断。」

韩泥点头。恩在粥里，也在全对里——对轻，记重。

黄昏，角柜样增至十二包，包实，锁柜。锁住了，才不示人；示人，赖福来，管事来，煞骨门旧债也来。

邻床问：「十堆全对，你成丹师了？」

「不成。」韩泥说，「成鼻。鼻成，手才稳。手稳，丹才来。」

少年不懂「鼻成」，却懂「不成丹师」，闭嘴睡了。

丑时前，他廊下练半刻，十二包并息，并出一线清。清里，掌纹影颤——颤像四层阶影在催三层破。

他按席角，席严，坛不露。坛腹温一线，贴掌，像应「续」字。应完，他低声对坛：「小成续了。三层在丑时。不叫。」

坛不应字，只温。温，像把十堆全对，摁进掌纹那路里。

韩泥只答：「鼻清。沿净。」

赖福盯半息，终是没掀席。掀不得，沿净，坛不露。

秦霜木鱼一声：「续成别骄。骄了，香飘。」

韩泥不答，答在心里——心里不骄，只记。记续成，记三层将破，记丹路将开。

他独眼平，手不抖：「续成了。暗修三层在丑时。四层影在等。」

刃不亮，先记。记完了，才配暗修三层，才配暗修四层。

更鼓沉，沉里，他摸怀玉牌，凉进骨。凉不要紧，鼻暖——暖在诀，暖在十二包样，暖在缘苗记里。

韩泥只答：「不恋。记着。」

坛腹应温，像应「小成续」最后一笔。应完，他不叫。不叫，才守得住鼻清，守得住95那盏门缝灯。

他低声道：「续成了。坛应了。丑时破三层，仍不叫真名。」

不叫真名，臾墟子嘱过。嘱过，煞骨门才嗅不准；嗅不准，晦气才遮瓮。

柱上条角白一线，春风硬，硬不要紧，手在；手在，就不滚。

管事夜巡验棚，灯照席角，沿净，记「手稳」。同批少年悄问：「丑时你又静？」

「静。」韩泥说，「静破境，不叫。叫了，恩断边缘。」

少年不懂「恩断边缘」，却懂「不叫」，闭嘴。

他合眼不睡实，鼻下仍清一息。清里，沉腐浮各归位，位位稳，稳才接得住三层破那一夜。

白织月远廊过，没停。没嘲，小胜——胜在鼻，不在嘴；嘴稳，她断的丹路，叔自己来开。

续成后第三日，兽栏铲下稳，稳里鼻下清。清，像十堆全对那口息，息不落舌，落鼻。

管事记「清细」。记细，是小胜。铁无言声低：「续成了，丑时破三层。破时别叫，境在席下。」

韩泥应声：「不叫。」温听雨远廊一眼，验指稳，袖风一转，走了。

刘婆端粥，仍多半勺，嘱十堆全对手别飘。韩泥点头，喝尽，不谢，只记。

午后药渣场，十二堆混灰再验，全对。全对稳，稳才接得住87三层破，接得住88四层破。

黄昏角柜锁实，十二包样在柜，柜丑，锁像规。规在，坛在；坛在，根在。

他低声对坛：「续成了。三层在丑时。不叫真名，不叫境名。」

温，像把续成最后一笔，摁进掌纹影里。

更鼓沉，独眼平：「小成续了。87丑时破三层，丹路将开，95灯在远。」

续成后第三日，加验再加两堆，十二堆混灰。韩泥分拣，指稳，十二堆全对——全对，管事记「续验准」。

铁无言分秤末撮，点头：「沉七腐二浮一，对。续成了，丑时破三层，破时别叫境。」

温听雨远廊一眼，验指稳，没停。没停，小胜——胜在鼻，不在嘴。

丑时前，十二包并息，并出一线清。清里掌纹影颤，颤像四层阶影催三层破。

更鼓沉，沉里，韩泥按木牌，牌凉，掌温。掌温在暗，不在公示；公示写仍末，掌写续成。

续成后第四日，坡下春风硬，硬里药香回一线。回不要紧，鼻下清，清才接得住三层破。

兽栏午后，铲下稳，稳里十二包样在忆里，忆里沉腐浮各归位，位位稳，稳不像飘。

管事弟子验渣场，十堆再验，全对，记「加验仍准」。准，是小胜，胜在鼻，不在舌。

温听雨外堂洒扫，瞥指稳，没说话，袖风温一线。温一线，记在心里，不在舌上。

刘婆端粥，仍多半勺，嘱加验别空肚。韩泥喝尽，粥稀，稀不要紧，稀也能暖腹。

黄昏角柜锁实，十二包在柜，柜丑，锁像丙九末席。席末，也要锁，锁住了才不示人。

丑时前，廊下练半刻，十二包并息，并出一线清。清里，掌纹影颤，颤像四层在等三层破。

他低声对坛：续成了，三层在丑时，不叫。坛不应字，只温，温像十堆全对摁进影里。

赖福脚步在门外停半息，鼻清，沿净，终是没掀席。掀不得，晦气遮瓮，遮住了境才稳。

更鼓沉，独眼平，手不抖：小成续了，87丑时破三层，88四层，89丹路开，95灯在远。

铁无言声低：续成了，丑时破三层，破时别叫，境在席下。韩泥应声：不叫。

秦霜木鱼一声：续成别贪名。贪名，路断。他不贪，只记十堆全对这一笔。

老耿挑水过，声哑：续成别恋诀名。恋，路断。韩泥只答：不恋，记着。

柱上条角白一线，三年约在怀，愧字在怀，加验全对在掌，掌不示人。

白织月远廊过，没嘲，小胜。胜在鼻，不在嘴；嘴稳，丹路将来自己开。

清里，位位稳，稳才接得住三层破那一夜。

十堆全对那一夜，他合眼不睡实，鼻下仍清一息。清里，沉腐浮各归位，位位稳，稳才接得住87三层破。刃不亮，先记——记续成，记温听雨远验，记角柜十二包锁实。

十堆全对，加验准。准在鼻，不在嘴。嘴稳，温听雨远验也近不了沿。沿净，坛不露；不露，三层丑时才破得稳。

更鼓尽，雨歇，药山湿——湿里藏一点窑温余韵；余韵散，记：再试，手稳。
