# 第三十六章 刘婆

入门第七日，丙九灶粥更稀。稀，像赖福克过的麸皮——克半袋，粥薄一层，薄得勺子立不住。立不住，刘婆仍舀，舀满，满得像多给半勺。「丑脸那个，来。」

韩泥端碗，碗糙，糙得掌印深。深，像编筐的印，像抬棺的印，像五十八年活出来的印。刘婆瞥他肘袄，瞥见针脚——针脚密，密得像叶家，又不是叶家：歪一线，真一线，真像补衣婆的手。「肘谁补的？」

「叶丫头。」

刘婆鼻哼，哼得像不信寡妇好心：「寡妇心善，善到晦气？」

韩泥独眼平：「恩。不是晦气。」

就三句。刘婆愣半息，愣完，从袖底摸针线包，包旧，顶针亮：「坐下。肘我看看。」

灶边少年窃语：「刘婆也沾晦气？」

刘婆眼一瞪：「晦气在你们嘴，不在他肘。肘漏风，风进肺，肺坏了谁清粪？你们清？」

少年噤。噤，韩泥听见了，听见像泥岗公审前里正那声「记名」——记名，嘴就短半寸。韩泥不坐。坐，像讨。讨轻，恩重。刘婆把针线搁桌，声沙：「不坐也行。站着，风仍钻。钻久了，臂僵，僵了抬不了粪，抬不了筐，赖福正好克你饭。」

韩泥「嗯」一声，坐凳角。凳歪，歪得像丙九末席——席末，也是席。刘婆剪线，穿针，针老，眼浊，手却稳。稳得像顶针顶了一辈子布，顶得针脚比叶家还糙，糙得牢。「老婆子补，不收钱。」她低声道，「钱要，你们绝户拿不出。拿恩，恩太重，老婆子扛不动。」

韩泥不接「扛不动」，接事实：「肘破，风进。」

「进就补。」刘婆咬线，线紧，「补完，别谢。谢轻。记着就行。」

记着——韩泥最会。会所以指节白了一息，白完，又平。门外赖福路过，瞥一眼，笑：「刘婆，你也沾晦气？沾了，粥更稀！」

刘婆头也不抬：「粥稀是你克的，不是他克的。他克你，你早倒了。」

赖福脸一青，青完，走。走，也是小胜——胜在灶边，胜在粥里，胜在有人肯给丑杂役顶针顶布。韩泥低声道：「记着。」

刘婆哼：「记着记着，老婆子又不少块肉。」

肉不少，恩却多了一笔——多在心里，不落字纸。字纸只记饭，饭稀，恩不稀。补肘时，灶边少年噤声。韩泥肘伸，伸得像抬棺——抬棺不抖，补肘不躲。刘婆针走，针脚糙，糙得牢：「老婆子顶针顶一辈子，顶的是活，不是脸。脸丑，肘不能漏风。」

「不漏。」韩泥说，「漏了，清不了粪。」

「知道就好。」刘婆咬线，线紧，「别挂粪栏。挂了，我又补。」

韩泥「嗯」一声，记——记刘婆恩，记叶丫头恩，分列，不混。夜，丙九房。韩泥抚肘新针脚，脚糙，糙得暖。暖进骨，骨里想起叶丫头那针，又想起刘婆这针——两针不同，同一意：不漏风，不漏名，不漏还的路。坛腹微温，温得像应「刘婆」二字。二字丑，丑得真，真便按实，按到坛沿一线水汽。他低声道：「恩又多一笔。笔多，命要活长。活长了，才还得起。」

邻床少年梦呓，呓得像嫌粪臭。臭，韩泥不怕，怕的是恩断——恩断，瓮醒也无用，醒了就还。外头更鼓，鼓沉，沉得像顶针顶在心上。顶一下，又一下，顶出将来还刘婆那一笔，顶出将来还刘婆那一身新衣。刘婆收针，针入包，包入袖：「明日起，肘别挂粪栏。挂了，我又得补。」

韩泥「嗯」一声，起身，粥碗已空，空得像泥岗米缸见底——见底过，活过来了，宗门也会活过来。他回丙九，路过赖福，赖福瞥肘新针脚，瞥完，没笑。没笑，像怕刘婆真扣盆。灶边少年悄问：「刘婆为何帮你？」

「帮肘，不帮脸。」韩泥说，「脸丑，肘不能漏风。漏风，清不了粪，编不了筐，活不了。」

少年似懂，闭嘴。闭嘴，韩泥抚针脚，针脚糙，暖。暖，像恩又多一笔——笔在心里，不落字纸。他低声道：「刘婆恩，叶丫头恩，分列，不混。混了，还乱。」

坛腹微温，应「补肘」，应「记着」。记着，够走到将来还刘婆那一身新衣。外头丹香一线，线细，细得像钩——钩向药渣，钩向嗅诀，钩向下一章每一日苦熬。丑时将近，邻床少年醒，瞥他抚肘：「刘婆真不收钱？」

「不收。」韩泥说，「钱贵，恩重。重，先记着。」

「你怎么还？」

「活长了，才还得起。」韩泥独眼平，「活一日，记在心里一日。记满了，就还。」

少年翻身睡。睡，韩泥摸坛沿——沿温一线，温像应「刘婆」二字，应完又敛。外头赖福骂远，骂像风。风过，坛温不退——退的是怕，不退的是命。命在，恩在，仇苗在，瓮眠在。眠着，步先稳；稳着，够走到130还那一身新衣。刘婆门外咳：「肘别挂粪。挂了，明早又来。」

韩泥「嗯」一声：「记着。」

刘婆哼一声，声沙：「记就记，老婆子又不少块肉。」

韩泥不谢。谢轻，记——记刘婆顶针，记叶丫头针脚，记赖福秤，分列，不混。坛腹微温一线，应「记着」，应完又敛。敛，像睡；睡，像等瓮醒那日还粥。赖福门外骂：「刘婆，你也沾晦气？」

刘婆门外应：「粥稀是你克的，不是他克的。再骂，明早你也没粥！」

赖福哑了，哑，像秤空一格。韩泥低声道：「刘婆粥，叶丫头针，记两笔恩，心里分列。」

他心记叶丫头针脚——针脚在，人在泥岗；人在，恩还得。第七笔落在心，不落字纸；宗门恩另册，不混泥岗六笔。混则还错人，错人，汤就凉，凉不得。

刘婆肘布丑，粥热半勺。韩泥喝尽，只「嗯」一声，像盖第七笔的印。印盖得糙，糙得牢——牢，才走得动将来还路。

丑时将近，他抚肘新针脚，针脚暖进骨。骨暖，天寒不怕；天寒，灶边热粥还稀着，稀也顶一口饥。顶一口，多活一日，多记一日。

刘婆翌日早起，仍多半勺粥。勺在，韩泥端碗，碗糙，烫手不虚。烫，像第七笔又温半分。温半分，够熬到编筐课、清粪栏、兽栏冬角。

编筐课上，管事弟子验筐松紧，韩泥筐紧，紧得弟子无话。无话，赖福脸又青半分——青半分，也是小胜。小胜累加，够活，够记，够走到还刘婆那日。

午后他去搬麸皮，肘新针脚不挂袋角。不挂，风不进；风不进，臂能抬。能抬，才抬得了将来省亲路上的篓，才端得住将来还恩的丹。

夜里他摸坛沿，沿温应「刘婆」，应完又敛。敛着，瓮眠；眠着，步先稳。稳着，第七笔与泥岗六笔分列——列清了，还才不乱。

刘婆补肘，肘暖了，才走得动还路。赖福哑口，哑也像小胜。
