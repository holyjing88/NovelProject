# 第二十九章 荐帖

集尾贴榜。泥岗村采药榜末，韩泥名在末，末得稳定，像旱里一滴水——滴少，也是水。水在，活路在。榜旁又贴一张，墨新，沉丹宗字样，像刀光：

「外门杂役招募：凡骨可试，年五十上限六十，有荐者优录。」

村人嗡，嗡向韩泥：「晦气翁也能去？」

嗡声里，有羡，有嘲，有怕。怕晦气沾身，羡宗门路远。马春花尖：「去了也是杂役，丢人！」

丢人二字，韩泥不入耳。入耳的，是「上限六十」——五十八岁，两年窗。窗小，够钻。钻不是偷，是活路。活路通向还恩，通向三年约后报仇，通向瓮醒那日。日远，路要先迈。叶青禾在药铺门内，声平：「能去，就去。别困死在泥岗谣里。」

韩泥「嗯」一声。里正咳，咳出一句：「言伯钧长老昔年云游，留过一纸荐帖名额，给抬棺稳、手不脏的人。韩泥……在册上。」

在册上。三字，像绳，绳拴沉丹宗，拴仇线，拴省亲那日路。路长，先上车。沈枯芽娘拽娃，远远朝榜鞠半躬，躬得浅，真，真像公审那日。公审过了，荐帖来，来得像缝——缝进宗门，谣才追不上半步。赖扒皮的人路过，瞥榜，瞥韩泥，冷声：「去了宗门，也别以为洗得净晦气。」

韩泥独眼平：「洗不净，我带着。带着，才记得泥岗。」

来人一愣，骂「老疯」，走了。走，也是让半步——让得丑，够。午后，韩泥站榜前，站如一截老木。木丑，根深。张麻婆塞他两个馒，馒热：「路上吃。别饿死，饿死谁还我馒恩。」

韩泥不收全，收一个，另一个推回：「留着你。我记了，不欠馒，欠情。」

情欠着，将来还。还，是叶汤深的那条路。沈枯芽躲柱后看，看怯，怯里亮。亮，像不怕了。不怕，韩泥记，记在芽儿那粒米旁，不再加笔，笔够。心口微热，热得像应帖。应帖，不是天命，是人活。人活，先离谣窝。他低声道：「去。」

就一字，像钉，钉进五十八年。钉深，拔时带泥，泥是恩，恩不丢。坛在屋，经片在底，荐帖在眼，恩在心。心厚，厚得过榜末名。夜里，韩泥展荐帖抄本，抄本丑，字是里正让人念给他听的。听清，记牢：

凡骨可试，杂役优录，言伯钧名在末。名在末，像韩泥在榜末——末，不是绝，是缝。缝进宗门，才有翻身的秤。叶青禾关门前来，送来一只小药囊，囊轻：「路上若泻，别硬扛。硬扛，死在半路，谁还恩？」

韩泥接囊，不接话多。话多，像恋。恋，误路。他只道：「记着。」

叶青禾退半步，半步像公审后第一次近：「我爹药，我汤，我姜，你一并记着。别漏。」

「不漏。」韩泥说，「漏了，坛会烫我。」

叶青禾走了，布鞋稳，稳得像把叶家秤交给他带走——带走的是记，不是欠。欠，将来还。韩泥摸心口，心烫，烫得像应「去」字。去，不是逃，是活。活长了，才还得全。外头旱风仍硬，硬不过一字。字钉进五十八年，钉出第六部前的所有忍。忍，是丑翁的刃。刃不亮，先磨。磨在泥岗，亮在宗门。周嫂洗衣，棒槌轻敲，敲得像也想问：「真去？」

韩泥「嗯」一声：「真去。不去，困死谣里。」

周嫂闭嘴，闭嘴，像怕再砸出祸。祸在嘴，不在宗门路。里正补一句：「荐帖名额就一名。名给了韩泥，别人别抢。抢，连坐。」

众人散，散得快，像怕连坐。快，韩泥心更定——定，不是喜，是路开一线，线够钻。陈姑在人群后，没说话，只把棉袄领子拢紧，拢得像公审那日。公审过了，帖来了，来得像门缝又宽了一指——宽的是活路，不是牵扯。耿恩经片在底，荐帖在怀，叶汤在胸，六笔恩可答。答得齐，辞村才不算逃。韩泥低声道：「去。不是逃，是活。」

叶青禾帘内秤响一声，响得像送行——行不送，送的是记。记着，就不怕半路死。坛腹微温，应「去」字。去字钉进五十八年，钉出宗门篇，钉出瓮醒，钉出省亲那日。天未亮，韩泥再至榜前。榜墨新，新里旧：采药榜末，韩泥名在末。末，不是绝，是缝。缝旁沉丹宗三字，像刀光，也像活路。沈枯芽娘拽娃来，远远鞠半躬，没近。近晦气翁，像牵扯；鞠，是恩。芽儿怯问：「韩爷爷，你真去宗门？」

韩泥蹲低：「去。」

「宗门远吗？」

「远。」他说，「远，才追不上泥岗嘴。嘴追不上，你叶姑姑秤才能稳。」

芽儿点头，点得像米粒。米粒旁，韩泥心里又稳半分——稳，不是喜，是路开一线。里正来，咳：「帖在，路在。路开了，别生事。生事，名额收回，连坐全村。」

韩泥独眼平：「我生事？我编筐，我记恩，我上路。」

里正瞥他怀，瞥见荐帖抄本角，愣半息：「言伯钧长老的帖，别糟蹋。糟蹋了，抬棺那笔恩也白记。」

「不糟蹋。」韩泥说，「糟蹋恩，坛会烫我。烫，我受；糟蹋，不受。」

叶青禾开门一瞬，门开又关，关前声平：「路上若病，别硬扛。硬扛，死在半路，六笔恩谁还？」

韩泥「嗯」一声：「记着。不漏。」

门关了，关得像秤交给他带走——带走的是记，不是欠。欠，将来还；还，得活着到宗门。一字，钉进五十八年。钉深，拔时带泥；泥是恩，恩不丢。」

张麻婆在人群后喊：「路上吃馒！别饿死，饿死谁还我馒恩！」

韩泥「嗯」一声，不回头。回头，像恋；恋，误路。去字起，宗门篇开；篇开，省亲那日才有路。里正又咳：「一路。」

韩泥「嗯」声：「走。」

他低声道：「荐帖在怀，像绳。绳在，沉丹宗门就进得。」

荐帖风声过耳，快不得——先辞村，先验骨，先丙九活稳；活稳，才接帖，帖到，路才开。荐帖风声过耳，心动一瞬；韩泥按心：动不是贪，是想还快——快不得，先辞村，先验骨，先丙九活稳，活稳，才接帖，帖到，路才开。荐帖风声过耳，心动一瞬。动不是贪，是想还快——快不得，先辞村，先验骨，先丙九活稳。活稳，才接帖，帖到，路才开；路开，坛应一步，一步还恩不空。荐帖风声过耳，快不得——先辞村，先验骨，先丙九活稳。**状态**：大境·凡人 · 资质·丑骨末席（伪灵根） · 鸿蒙九劫瓮·眠 · 洞府·漏舍凡舍
