# 第四十一章 三十年

限期第二夜，丹堂后院，炉声细，像有人在远处哼歌。

苏念慈绕炉走了一圈，终是止步莫长春面前，月白衫子被火光映得透明，像把心事也映透明。后院无弟子，只有风与药香，像特意把热闹留在前堂，把真话留在后炉。

「师兄。」她声稳，眼不稳，「我等了三十年。」

莫长春看她，须眉白，笑温：「念慈啊，老夫剩四月了。你想清楚。」

她竟笑，笑里倔，倔里酸：「那正好，不用等太久。」

莫长春一愣，随即也笑，像被一句反话撞中软处：「你这丫头，倒会算账。」

「丹堂算账，」她道，「我比你熟。四年也好，四十年也好，回赠不是还债——是你教的。」

风过，药香扑怀，像昨夜重现，却不再有扑的动作，只并排立，像两盏灯。远处前堂仍有弟子窃语「体统」，窃语被风切断，断在后院这一瞬。

莫长春低声：「想清楚了就好。清楚的人，才收得下苦，炼得出甜。」

苏念慈点头，望炉：「炉还开着。」

「开着。」他应，「别误了明日验丹。」

她转身回炉，背影挺直，像把三十年折进一炉火里，烧给青岚门看。

廊角，柳青鸢立过一停，未入后院，像刻意把空间留给别人，又像怕自己一入，就要说体统。她指节叩栏，一声，转身，宗主令在腰，威在回。

苏念慈瞧见那影，眸光微动，不妒了——妒轻不过命，命在东脉，在子时，在夜袭。

霍镇山在远处骂阵，骂声洪，像把战意敲进每个人心里。赵玄机思过崖方向，风硬，像妒火还在冷，冷到要上场守门。

弟子在前堂窃语「三十年」，窃语轻，轻不过后院这一瞬真话。有人猜苏首座表白，有人猜莫长老拒，猜成七种版本，版本里都没有后院炉边并排立的两盏灯。

魏无涯在檐角，扇开一线，观气运，低笑：「三十年，四月，好账。」

莫长春望东脉，低声，像钩，说给炉火：

「验丹明日。今夜之后，丹堂只有战，没有等。」

风大，炉声细，像一句未说完的话，留给子时。

顾小满在杂役堂值夜，听后院炉声细，无端心稳，像半块灵石在怀，温了一路。她不知三十年，只知首座守炉，守得像战。

赵玄机在思过崖，听丹堂炉声远，妒火又冷一寸。他忽然想：若守门能还恩，妒人还妒什么？想完，他自己也愣，像没料到思过崖风能吹出这一句。

柳青鸢在回廊停一停，未入后院，像刻意把空间留给别人。她指节叩栏，一声，像把妒与担都叩进木里——妒轻不过命，命在东脉，在子时，在夜袭。

三十年落在炉里，四月落在簿上——等字收束，战字开门。苏念慈不回看，莫长春不追问，像多年同门，终于不必问「你还要等什么」。
