# 第六十一章 丑时坛温

丑时，棚静，鼾雷断了一息。韩泥未睡，盘坐，油灯豆，豆光照坛角。

角在，席严，符怀暖一丝。暖像半闪，不燃——光太亮，管事查。

坛腹温一线加剧，像心跳，像瓮里翻身半次，又眠。眠，不认主。认主，要等血缘碰沿。

他不主动滴血，不揭席诱沿，不拜邪求醒。求，像赖福。等，像守瓮人。

掌茧涂药，药凡，稳手。手稳，明日藤刺才正。正刺，血才找席，席才挡到沿前。

白日编筐半刻，末藤硬，他偏手，刺浅些。浅些，血仍来。血落席，沿不接——不接，像沿前即止那日。止在席边，不在沿上。

邻床翻身，梦呓「温」。韩泥盖席，声低：「你看错了。睡。」

他低声对坛：「你再眠半夜。缘满不满，看活计，不看心急。心急，路断。」

坛不应字，只颤极轻，轻像远处应，像祖父韩老栓远处应。

沿上符影浮半瞬，又敛。韩泥独眼盯沿，沿净，沿滑，沿上无血。

无血，符影才像半闪，不像邪。符在怀，暖一丝，不燃。燃，烟起，管事查。不燃，暖够扛丑时寒。

铁无言门外脚步远，远，像夜禁巡过。巡过不要紧，要紧是席严，沿净。

脚步又停半息，声低，只对他：「符影半闪，不燃。燃，管事查。查，像借腥。」

韩泥应：「不燃。沿净，才像探路，不像邪。」

脚步再远。远，像门缝又开一线——线窄，够走。

刘婆门外咳：「丑时寒，别空肚。」

韩泥应：「不空。姜在记里。」

鸡未鸣，狗未吠，夜禁脚步远。三年约、钱戾衡，都在另一时辰。

时辰未到，瓮先温，温而不醒。

丑时将尽，温一线敛，敛，像瓮又眠回去。眠回去不要紧，要紧是席盖更严那一夜将至。

他按掌，掌贴席，席贴坛角。角温一线，温短，像廊下香字，又像凡符暖字，又像灶下那个「热」字。热在恩里，不在舌上。

外头风硬，柱上无新条。无条，有条在坛。

他低声道：「丑时坛温，一线加剧。加剧，不急喊醒。喊醒，像拜邪。」

坛不应字，只温，温得像答应，像等，像后半夜。

六十日沿净，六十日不主动滴血。滴，要等活计误那一甩——甩在缘里，不在舌上。舌急，血偏；偏，席挡不住。

秦霜远廊一句，声轻：「心别邪。邪，符灭。」

符在怀底，温一线，记着七笔不误。

油灯豆将尽，豆光缩一线，缩了，沿上符影却浮半瞬。半瞬，像韩氏守瓮血还未到，符先探路。探路不要紧，沿要净。

同批少年咳嗽，咳醒半息，问：「你还不睡？」

「丑时守坛。」韩泥说，「守，不等醒。等误。」

少年不懂「误」，又睡。

天边白一线，白像瓮醒前夜。前夜，席盖更严。更严，他要加破布，加旧席，加压柴角。

他低声对坛：「丑时尽，坛温仍加剧。加剧，我仍不揭席诱你。」

坛颤极轻，轻像守瓮人远处应。

外头风息，符影却似在沿上再浮一息，又敛。敛，像钩，钩瓮醒，钩正测，钩还。

他摸怀符，符暖一丝，贴骨。骨暖，手稳，沿净，席严。严了，才配接后头席盖更严那一夜，才配接瓮醒那一误。

符影第二瞬浮，又敛。敛，沿上无血，无血才像半闪——半闪，像凡符祛寒那日赠的暖，像探路，不像拜邪。拜邪，恩断；恩断，汤凉。

误在缘，不在心急。心急，血偏。偏，席挡不住。

他独眼平，手不抖，低声道：「丑时坛温，符影半闪。半闪罢，仍等活计那一刺。」

刘婆门外咳：「丑时尽，沿还净？」

韩泥应：「净。暖一丝，不燃。」

木牌在枕下，与席严并放。并放，像命与坛挨着。挨着，才接得住瓮醒那一误。

外头更鼓尽，尽处，天边白一线。白一线，像瓮醒前夜将至。将至，接席盖更严，接瓮醒，接鸿蒙九劫瓮认主。

他按席，席贴坛，坛腹温仍加剧——加剧，不认主；认主，等误那一甩。误在活计，不在舌；舌急，血偏。

符影沿上再浮一息，又敛。敛，像凡符探路完毕，像韩氏血还差半寸。半寸，靠等，不靠催。

老耿门外挑水，水洒稳，声哑：「前夜别拜。拜，路断。」

韩泥应：「不拜。席严，沿净。」

耿不再言，脚步远。远，像冬深那一赠还在等——等不要紧，要紧是沿净。

丑时尽，风息，油灯豆灭。灭不要紧，沿净，席严，符暖，手稳——稳了，才配接瓮醒前夜那一夜。

他低声对坛：「符影半闪，仍等误。误了，根在。」

坛不应字，只温一线，温短，像应「等」。等，像守瓮人。守，不等醒，等误。

温而不醒，醒在缘，不在舌。还，先还烫——烫在瓮醒后，不在丑时这一更。

他加一层破布，布旧，旧更能吸。吸，挡沿。沿净，才配明日席盖更严，才配后头瓮醒那一误。

管事夜巡路过棚外，靴声沉，沉得像钉：「丑时守坛，别生事。生事，连坐丙九。」韩泥应声：「不生事。生事在舌，不在沿。」

管事靴声远，远里像也给正测留一寸缝——缝在沿，不在舌。舌急，手就飘；手飘，符就灭。铁无言远处咳一声，咳得像验沿：「沿还净，符才暖。」

棚外雪粒打瓦，打瓦声细，细像有人在远候。候不要紧，要紧是沿上无血——无血，才像探路，不像拜邪。邻床第二声梦呓「温」，他盖席，声低：「温在符，不在坛。」说完，符怀暖一丝，坛腹仍凉——各守各线，不混。

同批少年又梦呓「红」。韩泥盖席，声低：「红在符，不在坛。睡。」

他按心口，低声道：「丫头那碗汤，热着还。还，要等瓮醒后，路才长。路长，先过席盖更严那一夜。」

坛腹温仍加剧，加剧，像沿边已等半寸。半寸，靠活计误那一甩，不靠舌上催。耿恩在底，叶汤在胸，分列，不混——混了，沿就污；不混，沿才净。

席严，沿净，符影在沿上又浮了半瞬，随即敛去。敛去，像钩——钩正测，钩还，钩瓮醒。瓮醒在下一夜，不在这一更。

他把木牌翻过来，背面糙，糙得像心口七笔。笔笔按实，按出稳。稳了，才配接后头席盖更严那一夜。木牌与坛沿并放，并放，像命与缘挨着——挨着，才接得住瓮醒那一误。

刘婆第三声咳：「丑时尽，沿还净？符还暖？」

韩泥应：「净。暖一丝，不燃。」

天边白一线更深，白像瓮醒前夜就在下一更。铁无言提灯远过棚角，灯照坛席，照出一圈净沿，嘱半句：「沿还净，才接得住瓮醒。」

韩泥应声，不追问。问急，像借腥；等，像守瓮人。油灯豆将尽，他独眼盯沿半息——沿尘，沿滑，沿上无血。无血，才像探路，不像邪。

沈枯芽娘在远处咳两声，咳轻，像怕扰丑时守坛。咳不要紧，芽在，叶家秤在，汤就不凉。刘婆第四声咳，声更低：「沿还净，别主动滴血。滴血，像借腥。」韩泥应：「不滴。等缘，不在舌。」

油灯豆爆一粒，粒灭，沿上符影随灭即浮。浮半瞬，像凡符那日在掌纹里留下的浅烫——烫探路，探的是沿净不净，不是叫他拜。坛腹温仍加剧，加剧，像沿边已等半寸；半寸，靠活计误那一甩，不靠舌上催。

这一更，只守席，只守沿，只守不主动滴血。风息半更，席盖与沿贴合一线——贴实，坛温才不外泄。下一夜席盖更严；严了，才接得住瓮醒那一误。沿还净，符还暖，手还稳。丑时坛温，一线加剧；沿仍净。坛腹温不退，不退，才等得起下一误。稳。等。
