# 第七十六章 药山深行

第二日，药山边门又开了一线。管事弟子验牌，声沉：「仍只捡。深一步可以，过界碑不许。过了，除名。」

韩泥应声：「不过界。」

他昨夜几乎没睡实。柜角那三撮浅堆样还在怀里发热——热不是火，是记性。记性催他上坡：浅堆闻过了，深堆才配摸。摸不是贪，是把门缝里那点路往粗了拓。拓粗了，往后攒废渣才不瞎子摸象。

牌一收，他背筐上坡。坡滑，霜白，脚底下像抹了油。他把气往足底沉，一步一步走稳——稳不是逞能，是怕一滑就滚进界里。雾深了一寸，界碑影在雾后发钝，钝得像堵秤：过了，命就断在规矩上。

深堆在坡腰。弃炉基破着，灰里还能摸到一点余热。热不是火，是药性没散尽。韩泥鼻下静着，拣一撮灰放掌心，再闻：辛沉底，苦浮上，腐在最下。浮的先剔，沉的入筐。筐底垫粗布，布丑，却不漏。

坡角有杂役弃牌，牌裂，字浅：「过界除名」。他瞥一眼，不近——近像贪；贪了，路断。远处主道香冷，渣道苦真。苦真才配捡；捡假香，等于给赖福送把柄。

白织月从主道过，袖风冷，一眼扫他：「丑杂役也敢深行？深行就行，别越界。越了，脏我丹房风。」

韩泥侧身让半尺，肩稳，渣不漏：「不越。捡渣，沿净。」

她笑一声，笑声远了，像钉子丢进井里：听见了，水面还晃。韩泥不接。接了费舌头；舌头得留着记路、记仇、记恩。同批少年蹲在坡下不敢上，只远远骂：「疯了吧，深雾也能进？」

「派了就进。」韩泥扛筐不回头，「进了也不过碑。」

铁无言立在雾外，声低：「深一步，鼻下更静。静了，才听得见渣说话。听见别叫。叫了，像求名。」

「静在鼻。」韩泥道，「不叫。」

铁无言又补一句：「别贪炉砖。贪炉砖，像偷火。偷火，路断。」

「不偷。只走路。」

他再拣半刻，筐底多一撮黑灰。黑灰里一点未烬的亮，轻得像引火，他却记重——记重了，往后才走得通废渣路。雾又起半寸，他故意停在界碑外三步。三步是规矩：规矩在，坛才藏得住；坛藏得住，叶家那碗汤才还得上。

他蹲下，指甲抠炉基边一条焦纹。焦纹里辛正，正像昨天浅堆里没有的。他把那撮单独包一层纸，纸丑，包紧。包紧不是藏私，是怕混堆把辛搅没。搅没了，鼻子练不出芽；练不出芽，白织月那句「炼不出丹」才真要成真。

雾腰里忽然有人脚步乱。乱的是别棚杂役，贪炉砖被当场喝住。管事鞭影一甩：「再近界碑半寸，除名！」那人跪下去哭。韩泥眼皮都不抬——抬了像同情，同情惹事。他只把自己足尖又往后挪半分，挪进规矩里。

午间管事来验。验筐，验足迹。足迹不过碑，碑外一步无印。管事眉松半分，记簿，笔丑，字真：「丙九韩泥，深捡净。」

赖福立在坡下，眼细：「深也捡？捡出邪药，连坐丙九。」

韩泥低声道：「只捡渣，不碰活药。」

赖福哼一声，脚步远了。远不要紧，筐已交，名已记——记了，深行才算完。刘婆提篮路过，篮里饼硬得硌牙，塞进他掌：「深行饿了？」

「饿了。」韩泥说，「饼就行。」

「别谢。」刘婆掌印在饼上还在，「记着。深行别贪，贪了婆也救不了你。」

韩泥点头。谢轻，恩重。恩在饼里，也在「别贪」两字里——两字像铁无言，像秦霜木鱼，像坛沿那条净规。他咬一口饼，饼糙得刮嗓子，却暖胃。暖进胃里的东西，不能拿去换炉砖。

回棚前，他把深捡样分成三纸包，包丑，包实，塞进角柜。柜锁破，却像规矩：锁住了，才不示人。邻床少年探头：「深雾怕不怕？」

「怕。」韩泥说，「怕也捡。捡了，才有路。」

「捡出命来咋办？」

「命在手。」他声音平，「手在，就不滚。」

少年闭嘴，像也听懂了半句。秦霜远廊经过，木鱼轻敲一声：「界在，心在。心越界，鼻下香飘。」

韩泥不答。答在心里——心里账清：深行是活，活是路。路通白织月再嘲，通嗅诀将芽，通冬深那些脏活。黄昏他又去兽栏清半刻，铲下稳，稳得像踹台那一掌。粪里混着一点药回，他鼻下一静，静里分出浮假，指却不停——不停才像只清栏，不像嗅丹。

管事夜巡验棚，灯照席角，沿净，像腌菜坛，不像邪。赖福跟在后面瞧，瞧见沿净，嘴角抽一下，没掀席。掀不得，沿净就是甲。沈枯芽在药圃角弯腰，弯得像没看见。没看见也好，韩泥也不近——近像求；不求，只记半块灵石恩还在前头。

回棚路上，老耿挑水擦肩而过，声哑：「深了也别贪香。」

「不贪。」韩泥道，「香飘，人就短。」

老耿扁担挂霜，点一下头，走了。走得像把一句规矩钉进他后背。他顺着那规矩把筐绳又勒紧一分。

丑时前，他廊下立半刻，任风硬，鼻下仍静。静里，掌纹那路微颤——颤不像惧，像芽要顶土。他按席角，席严，坛不露，低声对坛：「深行了。界没越。芽在鼻，不叫。」

坛腹温一线，贴掌，像应「深」字。应完他不笑。笑像漏香；漏香，路就薄。他把木牌系紧一线，牌凉贴胸，独眼平：「名在册了。明日嘲来，舌仍不留。」

夜里风硬，药山那边雾气还往棚缝钻，带一点焦回的辛。他睡不实——睡实了像忘路。邻床又闷问：「你明天还去深堆不？」

「管事派就去。」韩泥说，「派了也不过碑。碑在，人在。」

「不过碑，能捡到啥？」

「能捡到记。」他掌心按在柜角，「记牢了，鼻子才开。鼻子开了，才配谈别的。」

「记能当饭？」

「记能保命。」韩泥说，「命在，饭才轮得到你。」

少年哼一声翻过去。棚柱吱呀一声，像跟他说：规矩还钉着。他摸摸怀里三年约玉牌，牌凉进骨。凉不要紧，手还热——热在掌茧，热在筐绳印。印在，名就在；名在，滚字就还只写在柱上，没写进骨头。

散队后他还在药渣场角多留半刻，把深捡样跟白日浅堆样并在一块闻。并息不是炫鼻子，是怕记混。混了，明日再深行就瞎。同值杂役嗤：「深雾也捡？捡出命来。」

「命拴在指头上。」韩泥声音更哑，「指头稳，命才稳。」

管事灯笼晃过来，他立刻停鼻，只装理筐。理筐手笨，笨好看——笨才像杂役，不像偷学嗅诀的人。灯笼远了，他把最后一撮焦灰掐进纸包，包口勒紧，勒进柜角。柜角冷，掌却热。热里那一点颤又起，像芽顶霜土，顶一下，又缩回去。

他没急着追。追芽像掐苗。苗得自己拱。拱出来的夜，才配叫萌芽。萌芽前，先把界守死：碑外三步，一步都不多。

更鼓沉下去。坡下界碑影还像钉在他眼里：再一步是除名，退一步是活路。活路窄，窄也要走——走稳了，嗅诀那一芽才舍得冒头。冒了也不叫；不叫，白织月的嘲才挨得住，挨住了，手才更稳。

明日一到，嘲来也好，验筐也好，他只守一条：界没越，鼻下静。静住了，深行这一日才不算白冻。白冻的人会越界；他越不起，叶汤还等着还。还字一沉，脚就稳。稳着，界才钉得住。钉住就活。
