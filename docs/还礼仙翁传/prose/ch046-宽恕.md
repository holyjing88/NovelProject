# 第四十六章 宽恕

谣言达峰次日，赵玄机自请思过崖。

崖口风硬，石冷，像把全宗的脏字都吹到这里来晾。赵玄机跪，膝磕石，声哑：「弟子……陷害长老，罪该万死。」

莫长春拄杖立他面前，不扶，不骂，只一句，像记在心里：「去吧。恩是我留的，你得还人情——以后。」

赵玄机抬头，愣，愣得像听不懂「以后」二字：「长老……不罚？」

「罚了，」莫长春道，「你就只记着恨。记着还，才记得住守门。丹在谁手里不要紧，心在谁那边要紧。」

赵玄机眼眶红，重重磕头，额血沾石，像把妒火磕裂一缝。风过崖口，血痕未干，干得像妒火也干了一寸——干一寸，才知门往哪守。

远处，林挽月瞧见，心摇，摇得像廊边竹。她想近，又止，止得像怕自己一近，就要在堂上把话说尽。证字还压在喉里，时未到，可莫长老这一句「以后」，像把还字提前亮了一角。

执法堂弟子在崖下窃语：「赵执事思过，莫长老不罚？」「恩字……什么意思？」「听说夜袭将至，罚轻了？」

窃语里，慕与怕交织，像全宗都在等赵玄机是忠是奸的最后一笔。柳青鸢立崖下远处，未近，眸光冷，冷里却有一线松——松的不是奸，是师兄又把一条路留出来了。

霍镇山骂：「赵小子，思过别白思！夜袭要守门！」

赵玄机跪姿更直，直得像终于把「还库」之后的那一寸，跪进了石里。他低声，像对自己说：「思过崖风硬，正好把妒火吹冷。冷透了，才知道门往哪守。」

莫长春转身下山，杖点每一步，塔影微烫，字淡：

**宽恕有时，忠诚记在心里。**

他念：「赵小子，别浪费半月思过。半月后，东脉要人守门。」

弟子让路，让路里仍有人低声：「不罚像纵奸。」霍镇山自后喝：「纵的是路，不是奸！路开了，夜袭才有人守。」喝声洪，洪得像把「以后」二字也砸进每个人耳里。

苏念慈在丹堂听讯，炉火旺，旺得像替思过崖也担。她低声：「宽恕不是纵，是留路。」又遣人送汤至崖下，汤温，温得像把恕炼成甘。

魏无涯观崖口，低笑，像看妒火线收束一截，收束里夜袭已在门外。他扇合，自语：「恕比罚难。难在赵小子得懂——懂的是还，不是逃。」

孙福旧党在角隅嘀咕：「宽恕？宽恕谁？」顾小满冷眼：「宽恕守门的人。你守什么？」

顾小满把思过崖时辰记进册子，记得像记一条暗线——暗线不在正面，在侧面，在杂役堂敢不敢记。

赵玄机跪至日暮，额血干，眼却亮，亮得像等还恩夜。他望丹堂方向，望得像望一条将守的门。莫长春不回头，只袖空下山，袖里万物可藏，藏得住恕，也藏得住将来的战。

崖下角落，两弟子压着嗓：「思过还能改十日？莫长老年事糊涂了吧。」另一人却摇头：「你糊涂。门一开，恩不开。」话没落，赵玄机膝再磕石，声哑，那一瞬静了——静得像妒火被风按住，按住片刻，才又窃起。

赵玄机跪至申时，风更硬，硬得像要把私字从袖里吹出来。他反复想莫长老那句「以后」——以后是还恩，还是还妒？还恩，得守门；还妒，得毁门。两条路都在膝下，他选哪条，堂上才肯翻。

林挽月午后再度途经崖下，未近，只远立。她看见赵玄机额血，看见莫长春下山背影，看见霍镇山骂声洪。她忽然觉得，宽恕不是纵，是留路；留路的人，要在子时还恩——还恩之前，她得先还证。

苏念慈送汤至崖下，汤温，温得像把恕炼成甘。她对执法堂弟子道：「思过崖风硬，汤别凉。凉了，恕也凉。」弟子接汤，接得像接一门难懂的规。

周德海在长老院听闻「莫长春不罚赵」，面色平，平里冷。他低声：「不罚，是留赵守门。留了赵，东脉侧门谁开？」心腹不敢答，答了也像奸。

霍镇山骂罢思过崖，又骂东脉防务：「赵小子，半月思过别白思！夜袭要守门，门比妒人难！」骂声洪，洪得像把「以后」二字也砸进每个人耳里。

魏无涯传讯至崖下，只八字：「恕落崖，戈将起。」莫长春袖中一念出塔，符落掌，阅后焚，焚得像把下一程也点着。

顾小满傍晚把思过崖时辰、在场人物、霍镇山骂声，一条条记进册子。她不懂宽恕，只懂记——记清了，暗线才走得稳。

赵玄机夜跪，月白，石冷，血痕干。他低声，像对自己说：「思过崖风硬，正好把妒火吹冷。冷透了，才知道门往哪守。」

莫长春回院，不坐主位，独坐廊柱旁，望思过崖方向，杖横膝，塔壁无新铭，袖却沉，沉得像把赵玄机那一缝妒火，暂存待后兑。

柳青鸢批宗主令：「赵玄机思过半月，不加刑，加巡东脉侧翼。」令冷，冷里像把「以后」二字也写进去了。

苏念慈闭炉，闭得像怕思过崖风进丹堂。她低声：「宽恕宽恕，别传成弱。弱的是妒，不是恩。」

赵玄机思过第一日，执法堂弟子送饭，饭冷，冷得像规。弟子问：「赵执事，长老真不罚？」赵玄机声哑：「罚了，我只记恨。记着还，才记得住守门。」弟子退，退得像终于懂：恩字比铁牌难懂。

林挽月第二日再途经崖下，仍未近。她看见赵玄机跪姿更直，看见额血干，看见风硬。她忽然觉得，莫长老留的不是赵的命，是赵的路——路开了，证才肯开口；证开口，堂才肯翻。

苏念慈第三日遣药至崖下，药温，温得像把恕炼成甘。她对执法堂弟子道：「思过崖风硬，药别凉。凉了，恕也凉。」弟子接药，接得像接一门难懂的恩。

周德海听闻「莫长春不罚赵」，面色平，平里冷。他低声对心腹：「不罚，是留赵守门。留了赵，东脉侧门谁开？开谁，闭谁，得看夜袭。」心腹不敢答，答了也像奸。

霍镇山第四日骂罢思过崖，又骂东脉防务：「赵小子，半月思过别白思！夜袭要守门，门比妒人难！」骂声洪，洪得像把「以后」二字也砸进每个人耳里。

顾小满把思过崖五日时辰、在场人物、霍镇山骂声，一条条记进册子。册厚一寸，厚得像把「留路」二字压实。

孙福旧党在角隅嘀咕：「宽恕？宽恕谁？」顾小满冷眼：「宽恕守门的人。你守什么？」嘀咕者噤声，噤得像谣言碰上半石，又磕碎牙。

莫长春第五日独坐廊柱旁，望思过崖方向，杖横膝，塔壁无新铭，袖却沉。他低声，像记在心里：「年轻人，急什么。急出来的，是妒；慢出来的，是还。」

全宗尚不知：宽恕不是纵，是留路；留路的人，要在子时还恩。大因果还在夜袭子时，像妖骑，像杖剑；宽恕只是小留路，赵死守才是大还礼的前奏。

识海塔影一荡，一念收塔，杖落掌，他念：「恕完，醒要来。小满觉醒，响里换命。」

风过崖口，血痕干，妒火冷一寸——冷透了，才是守门。

顾小满把思过崖七日时辰、霍镇山骂声、赵玄机跪姿，又记一行：「恕非纵，是留路。」行细，细得像半块石硌深。

莫长春第五日独坐廊柱旁，低声：「年轻人，急什么。急出来的，是妒；慢出来的，是还。」一念收塔，杖落掌，他念：「恕完，醒要来。」

思过崖风硬，妒火冷一寸——冷透了，小满那双眼，该亮了。
