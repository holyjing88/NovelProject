# 第五十七章 周犹豫

战后清点，硝烟未散，东脉侧门仍闭。

顾小满立侧门阵闸外，浑身泥血，指侧门，指节白：「周长老，战时三回求开，闸纹未动。」

周德海抚须，面色平，平里冷：「侧门一开，中伏谁担？老夫守后阵根本，不错。」

「丹堂差点破。」顾小满声哑，「赵师兄在门洞流血，苏首座在阵眼榨干灵力——」

「若丹堂都守不住，开侧门不过是送头。」周德海截话，声不高，理太冷，冷得像早就备好的稿。

围观弟子渐聚，渐聚成浪，浪里怒，怒周长老见死不救，怒侧门不开，怒杂役一句话敢顶长老。

柳青鸢立不远处，青衫沾灰，未言，只目光扫过周德海指间令旗，令旗干净，干净得像从未举起过。

她淡淡道：「周长老之言，本宗记下了。盟会之前，不辩；盟会之后，再辩。」

周德海揖手，笑，笑里一丝松，松得像逃过一劫：「宗主英明。」

霍镇山骂声远，骂得洪，却骂不到后阵——后阵稳，稳得像奸线也稳。

莫长春拄杖过，不看周，只看顾小满：「话送到了？」

「送到了。」顾小满点头，「也记下了。」

「记下就好。」莫长春道，「记比杀难，比杀有用。」

他望侧门，低声，像落账：「门不开，有时防伏，有时防忠。忠奸，不在嘴，在闸。」

袖中簿无字，却沉，像一笔大因果，暂存，待后兑。

弟子窃语反转，反转成惧：惧周德海，惧侧门，惧盟会之后那一辩。有人低声：「秦管事说过周长老见信使……」

「别乱传。」

「可侧门真没开。」

浪声里，周德海仍笑，笑里冷，冷得像不怕日后辩。

魏无涯在檐角，扇开一线，观气运，观周德海袖，低笑一声，像看线深了。

他低声，像钩：

「闸在，账在。盟会之前不辩，盟会之后，一笔不漏。」

风过焦土，硝烟像洗不净的灰，灰里藏着下一卷的剪线。

有人低声：「秦管事说过周长老见信使……」「别乱传。」「可侧门真没开。」浪声里，怒与惧交织，像全宗都在等宗主一句话。

孙福旧党面色白，像听见「侧门」二字，也听见自己的旧账将被翻。韩铁山在远山听报，笑冷：「侧门不开，青岚自断一臂。」厉无殇在帐内，声哑：「周德海，好棋。」

赵玄机在丹堂门洞包扎，听见「侧门不开」，停半息，未讽，像也懂：门不开，有时防伏，有时防忠。忠奸，不在嘴，在闸。

柳青鸢立不远处，青衫沾灰，未言，只目光扫过周德海指间令旗，令旗干净，干净得像从未举起过——干净，有时比脏更难辩。

侧门仍闭，闸纹未动——动与不动，盟会之后，一笔不漏。

魏无涯在檐角，扇开一线，观气运，观周德海袖，低笑一声，像看线深了。风过焦土，硝烟像洗不净的灰，灰里藏着下一卷的剪线——剪线之前，闸在，账在，盟会之后，一笔不漏。

顾小满立侧门阵闸外，浑身泥血，指侧门，指节白：「周长老，战时三回求开，闸纹未动。」周德海抚须，面色平，平里冷：「侧门一开，中伏谁担？老夫守后阵根本，不错。」围观弟子渐聚，渐聚成浪，浪里怒，怒周长老见死不救，怒侧门不开，怒杂役一句话敢顶长老。柳青鸢淡淡道：「周长老之言，本宗记下了。盟会之前，不辩；盟会之后，再辩。」

「丹堂差点破。」顾小满声哑，「赵师兄在门洞流血，苏首座在阵眼榨干灵力——」「若丹堂都守不住，开侧门不过是送头。」周德海截话，声不高，理太冷，冷得像早就备好的稿。

莫长春拄杖过，不看周，只看顾小满：「话送到了？」「送到了。」「也记下了。」「记下就好。记比杀难，比杀有用。」

围观弟子在侧门阵闸外踮脚，踮得像等闸开——闸未开，记字先至。有人低声：「杂役敢顶长老？」另一人答：「顶的是闸，不是嘴。」

围观弟子中有人替周辩解：「后阵不开，未必奸。」立刻遭冷眼：「奸不奸，闸纹比嘴真。」周德海揖手笑意不散，笑意散在风里，散得像把「盟会之后」四字先寄在众人心上，寄得每个人脚下都像踩在延迟的痛处。

顾小满指节仍白，白得像要把三回求开刻进闸纹。她低声，像说给围观：「闸不开，记下了。记下比骂难，比骂有用。」
