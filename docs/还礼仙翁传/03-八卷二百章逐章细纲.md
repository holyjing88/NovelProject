# 《还礼仙翁传》八卷二百章逐章细纲

> 正文目标 **500 万字**（十二部约 1250 章，见 [`06-五百万字全书架构.md`](./06-五百万字全书架构.md)）。  
> 阶段性目标：核心 200 章（50 万）→ 加厚 335 章（100 万，见 `05-百万字扩充方案.md`）→ 全书 1250 章（500 万，见 `06-五百万字全书架构.md`）。  
> 核心骨架均章 2500 字；500 万阶段均章 **3800～4200**，高潮章 **5000～8000**。  
> 规划总纲见 `02-原创剧情.md` **第十七章 + 第二十一章**。  
> **逐章详细纲要**（承上/启下/状态衔接）见 [`chapters/`](./chapters/README.md) 目录；**插入章**见 [`chapters/EXPANSION-百万字插入章规划.md`](./chapters/EXPANSION-百万字插入章规划.md)。  
> **500 万分部细纲**：第一部 1～100、第三部 221～360、忠奸恩仇专章见 `chapters/EXPANSION-五百万字*.md`。  
> **衔接/系统校验**：第五版见 [`chapters/AUDIT-衔接校验.md`](./chapters/AUDIT-衔接校验.md)、[`chapters/AUDIT-系统一致性校验.md`](./chapters/AUDIT-系统一致性校验.md)。  
> 缘箓九转解锁章：4 / 30 / 60 / 90 / 120 / 145 / 165 / 185 / 200。  
> **赠礼密度铁律（500万扩写）**：每 **5 章** ≥1 次 ①赠礼+②四维（目标占比 22%，见 `07-剧情分类`）；200 章骨架 A 类仅 8%，**壳章内必嵌赠礼**。各部「赠礼节拍表」见 `EXPANSION-五百万字第X部细纲.md`。
