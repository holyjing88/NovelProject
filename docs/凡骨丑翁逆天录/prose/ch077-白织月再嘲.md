# 第七十七章 白织月再嘲

深行后第三日，内门小比余波未散。廊下仍有人练剑，剑光两闪，闪完，笑也两声。笑声薄，薄也真——真在凡骨杂役仍末席，真在灰线三尺仍公示。韩泥过廊，独眼平，不近。近，像凑热闹；凑热闹要还人情，他还不起。白织月立丹房阶前，裙素，袖香冷。她瞥韩泥筐沿，筐空，空里仍沾一点渣灰：「丙九也走丹房阶？阶上沾灰，像老鼠尾巴。」

白织月再嘲，声细，细里带刃：「丑骨末席，深行也配？配行，别脏主道。」韩泥肩稳，渣不漏，不接刃——接刃，全丙九饿。

她袖风过，带过一点冷香。冷香不要紧，韩泥鼻下仍静。静里，他记仇苗又深半寸——深在嘲，深在众目，深在「末等」二字。二字重，像麸皮停。

韩泥不辩。辩，费舌；舌要留来清栏，留来守坛。他侧身，让半尺，让得像避光。白织月声不高，高在众耳都听见：「凡骨末等，丹路断。断了还嗅药山，嗅出邪味，污我丹房风。」

内门有人哄笑，笑薄。薄不要紧，辱在众目，手不能抖；抖了，滚字就真。韩泥指节白一息，白完，又平。平，像踹台按实——掌在，名在，路在。他只答：「走阶，不进门。灰在筐，不在阶。」

白织月眉一挑：「嘴硬。三年后钱师兄让你滚，你滚不滚？」

三年约在怀，凉进骨。凉不要紧，仇烫——烫在还路前头，不在这一日。韩泥低声道：「滚看手。手在，就不滚。」

白织月笑一声，笑冷：「手稳有屁用。灵根末，稳也炼不出丹。」

笑里藏着断字——断丹路，断名，断凡骨幻想。断不要紧，韩泥不接断——接断，像认晦气；认不得，只认鼻下静。秦霜远廊经过，木鱼敲一声：「嘲归嘲，心别邪。」

白织月收笑，袖风一转，走了。走，也像给韩泥留一口辱——辱长，才扎仇苗；仇苗浅，深在钱戾衡，也在内门的眼。韩泥不追。追，像求；不求，只记。记白织月，记「丹路断」，记众目下那一瞬。午后，兽栏仍腥。腥不要紧，铲下稳，稳得像廊下那一句「手在」——在，就不滚。管事路过，记一笔：「丙九韩泥，正测后仍清细。」

赖福晃秤：「白织月也嘲你？嘲对了，末等炼什么丹。」

韩泥不接茬。接茬，全丙九饿。他低声道：「嘲在舌，不在掌。」

掌里有二层，掌里有嗅芽将开——开在鼻，不在舌。舌响，香飘；香飘，手就滚。刘婆端粥，仍多半勺：「丹房阶前挨嘲了？」

「挨了。」韩泥说，「不辩。」

「不辩怕什么。」刘婆哼，「手稳，饭稳。饭稳，恩不断。」

韩泥点头。恩在粥里，也在不辩里——不辩不是怂，是刃；刃不亮，先记。铁无言抱臂立栏边，声低：「嘲别贴脸。贴了，鼻下飘。」

韩泥应声：「不贴。静在鼻。」

铁无言又道：「白织月话毒，毒里七分真——真在灵根末。末不要紧，渣里活，活长了，丹路自开。」

丹路自开，韩泥不声张。声张像求丹；不求，只记。记铁无言这一句，记白织月那一句，分列，不混。黄昏，药渣场。灰起一线，他分拣，鼻下静，静里辨出假香浮——浮不要紧，浮也要剔。剔净，小胜。胜在指，不在嘴。胜了，才配回棚守丑时，才配嗅诀萌芽。邻床悄问：「白织月漂亮，你恼不恼？」

「恼在辱。」韩泥说，「恼不在貌。貌在皮，辱在众目。」

少年不懂「众目」，却懂「不辩」，闭嘴睡了。丑时前，他廊下练半刻闭息。息在鼻，不在嗓。嗓响，香飘；香飘，白织月就笑得更真。他按席角，席严，坛不露。坛腹温一线，贴掌，温不应嘲，应「记」。记仇苗，也记丹路将来开——开在渣，不开在舌。他低声对坛：「她嘲丹路断。叔记着。断不断，鼻下见。」

坛不应字，只温。温，像把「嘲」字摁进仇线，不摁进恩线。玉牌在怀，愧字在怀，白织月那一句在耳——耳响，心不响；心响，手就飘。他独眼平，手不抖：「嘲了。记着。嗅诀将芽，冬深续活。」

韩泥只答：「不跪。记着。」

秦霜木鱼远，一声，像应「心别邪」。邪了，鼻下香飘；香飘，煞骨门就嗅得到。白织月嘲是辱，刘婆粥是恩，铁无言话是路——三路分列，不混。混了，心就乱；心乱，手就抖。坛腹应温，不应字。温，像应「再嘲」最后一笔——笔丑，真；真在忍，真在手稳。他低声道：「丹路她断，叔不断。断在舌，不断在渣。渣里活，活长了，让她愣。」

韩泥只答：「鼻下静，手在。」

在，就不滚。不滚，嘲才扎得实——扎实了，才磨得出刃，才还得了叶汤。柱上条角白一线，三年约在怀。怀凉，掌温——温在二层，不在众目。众目看灰线，掌里看嗅芽。散队后，他编筐半刻，藤刺扎掌，偏手，刺浅些。编筐钱买米，米在，恩才还得上——还得上，才对得起白织月这一口嘲。

钱戾衡名在玉牌里，在三年约里——约还在，嘲先忍完。忍完，才配嗅诀萌芽，才配冬深杂役续那一轮。外头内门练剑，剑光两闪，闪完，笑也薄。薄不要紧，韩泥不接笑——接笑，像认晦气；认不得，只记。他摸怀木牌，牌凉，膝热。热在坛，热在路，热在「丹路断」三字还没真断的那一日。嘲后第二日，药渣场十堆混灰。

管事指堆，声沉：「错两堆，记名。」

韩泥分拣，指稳，鼻下静，十堆连对。连对，管事眉松，记「辨性准」。准，是小胜。胜在鼻，不在嘴。白织月若知，大概仍说丹路断——断在舌，不断在渣。赖福立远处，眼细：「辨性准也末等。」

韩泥不接茬。舌要留来记，留来忍。忍里藏刃，刃对着三年约，对着白织月，对着钱戾衡眼风。铁无言走近：「嘲后别飘。飘了，准也白准。」

韩泥应声：「不飘。沉腐浮，将芽。」

刘婆端粥，仍多半勺：「挨嘲了，粥仍要喝。」

「喝。」韩泥说，「喝尽，手才稳。」

粥稀，稀不要紧；稀也能暖腹。暖腹，恩不断。午后外堂阶前，尘少，他扫阶，帚稳。稳像踹台按实——掌在，帚在，路在。管事验阶，阶净，记「洒扫净」。记净，是小胜。黄昏，样纸再包，包实。包实，才记得住；记住了，嗅诀才萌芽。邻床问：「白织月咋总嘲你？」

「她断丹路。」韩泥说，「叔不断。」

丑时前，廊下练半刻闭息。嗓响，香飘；香飘，手就滚。坛腹温一线，贴掌，像应「嘲」字。低声道：「嘲了。记着。」

同批少年远避，避像避晦气，晦气遮瓮，遮住了才稳。

编筐角藤刺仍在，他偏手，刺浅些，沿才净，坛才藏得住。

管事夜巡验棚，灯照席角，沿净，像腌菜坛，不像邪。

老耿挑水过，声哑，境别拜邪，拜了路断，他只记着。

秦霜木鱼远，一声，心别邪，邪了鼻下香飘，香飘手就滚。

柱上条角白一线，三年约在怀，愧字在怀，手仍不抖。

白织月远廊过，袖风冷，没停，胜在鼻，不在嘴。

铁无言脚步远，停半息，声低，别漏香，别叫境，只记。

刘婆粥稀，勺沉，饭稳恩不断，他喝尽，不谢，只记。

散队后，他过丹阁侧廊，廊扫痕直，直得像秤。他不踩灰，肩稳，筐空——筐空，像只清栏，不像嗅药。

沈枯芽怯，悄问：「韩叔，你恼不恼？」
