# 第九十四章 门缝留灯

冬夜。沉丹宗外门侧，坡下那间小舍门旧，缝宽一线——宽得够透光，不够窥人。

韩泥不去敲门。敲门像连累；她说过「别来丑」。丑可以不见，恩却要还。还在缝，还在灯，还在她怕黑那些年。

沈枯芽先过去，怯声：「叶姐，韩……韩叔让带话：冬夜别灭灯。」

叶青禾愣半息，眼低：「他……他还记着？」

「记着。」芽儿说，「说怕黑。」

「我……我不怕。」她指节白一息，又松，「别让人看见。」

「无字。」芽儿递半饼，「只有饼。他说加字像情。」

「情……」叶青禾苦笑一声，「就叫记住吧。记住也行。」

韩泥立在远廊，独眼不往门里探。探像邪；邪了，秦执事来问。他只备了灯油半壶——壶小，嘴封泥，泥上无字。油是凡油，豆油耗得慢，慢才留得住长夜。

壶搁门侧石下，石下垫湿草，湿得像雨夜常有人放物。常有人，才不惹眼。

第二夜他仍不近。赖福夜里爱盯梢，他换法：门楣悬一根编筐剩的细藤，藤尾系油豆一粒。豆小，遇热会爆一声就灭——灭前够她伸手取壶。

第三夜，叶青禾推开缝。石下油壶旁多了个小囊，囊空，缝线丑，像刘婆顶针路。她取壶，手抖。抖像终于敢把恩接到手里。

她对缝外低声：「……还不起。别再来。」

缝外无人应。无人应，才是韩泥。他已在兽栏席下按坛，低声道：「灯在，就不黑。」

不黑是一半；另一半在丹。丹未成，灯先留——留缝，留温，留她冬夜敢合眼。

赖福查夜，火折照坡：「谁放油？」

韩泥立队中：「不知。丙九在栏。」

「石下有痕迹。」赖福眼细。

「雨痕。」韩泥说，「雨夜石头都湿。」

管事弟子扫一眼：「外门失窃？无。散。」

散队后，芽儿白天悄问：「叶姐收下了。」

「好。」韩泥只一词。

「她还说……还不起。」

「还不还由她。」韩泥道，「我还是我的事。两回事别缠。」

「那你还去吗？」

「灯油还会去。」韩泥说，「人仍不见。见了，她又要说别来丑。」

刘婆缝布条路过，声低：「门缝那丫头，脉还弱？」

「弱。」韩泥说，「弱才留灯。」

「留灯别留祸。」刘婆哼，「祸连丙九，粥也连。」

「不留祸。」韩泥应，「只留光。光是凡光，不像灵火。」

「凡光也好。」刘婆点头，「凡光不招旁人眼。你记住，招眼的事，一件都别做。」

「一件不做。」韩泥说。

丑时廊下静。粟壳香里多一线豆油气，气淡——淡才像凡灯。他摸怀经页，「培元」二字仍浅。浅，下一回要炼实；炼实了，才塞得进门缝那只空囊。

更鼓尽，坡下门缝一线灯黄。黄是凡光，凡光守得住秘密。韩泥独眼平：「灯留了。下一回，丹。」

丹要在暗处炼，暗处还，才配「别来丑」后头那碗烫。

邻床翻身问：「你夜归迟，干什么？」

「清栏。」韩泥说，「栏腥，多铲两遍。」

「闻起来像豆油。」

「灯芯炸了。」韩泥道，「看错了。睡。」

少年睡了。他才把细藤收回袖里。藤细如虫丝，丝在暗处，暗处看不见人。看不见人，她才敢取壶——取壶时，怕黑会退半寸。退半寸就够。

第四夜他又补半壶油。壶旁空囊仍在，像等。等的不是人，是温。温要等丹；丹要等雨夜；雨夜要等火路再匀半寸。半寸他心里有数。

叶青禾怕黑那句，沈枯芽又转过一次：「她说灯亮着，能睡实一点。」

「好。」韩泥把「好」字咬短。

「韩叔，你到底还她什么？」

「先还她合眼的夜。」韩泥说，「大的往后。往后有日子。」

芽儿不懂「大的」，却懂「合眼」。合眼对怕黑的人是恩，恩不示名。名一示，赖福记过，丙九连坐，叶家线也会断。线不断，囊才过得了缝。

韩泥按坛：「伏笔立了。伏笔不是戏，是还。还一半在灯，一半在将成的那粒。」

坛温一线，像应「缝」字。缝在，灯在，根在。根在，门缝就不白留。

坡下风硬，硬里那一线黄不肯灭。不肯灭，就像叶汤碗沿那点热不肯凉。他睡得浅，浅着听更鼓——更鼓尽了，明日还要清栏；清栏完，再瞅雨线。雨线一斜进废窑，炉壁最听话。听话那天，把丹炼进囊里，把「趁热」两字送到缝那头。

送到了，他也不现形。现形会把恩做成祸。祸不配汤。汤要趁热，灯也要趁黑——黑里亮着，她才敢把眼睛合上。

白织月袖风远过，声冷：「坡下谁夜灯？杂役家也敢亮到丑时？」

「外门各有火。」韩泥不抬头，「火凡，不脏规矩。」

她唇线绷一息，走了。走了最好——走了，灯才只亮给一个人。一个人够了。

铁无言渣场角忽然道：「你手上豆油气。」

「编筐灯炸了。」韩泥说。

「炸给该炸的人看。」铁无言声更低，「别炸给赖扒皮。」

「不炸给他。」韩泥应。

夜里他又摸一次门侧石。石湿，壶在，囊空。空囊像一张张开的手，等着雨夜那粒温。温还没来，灯先到——灯到了，怕黑的人就敢把夜熬过去。

韩泥回棚按坛，低声：「灯留住了。下一夜若下雨，我就炼。炼成了，塞缝里，人仍不见。」

翌日坡下路过的杂役咬耳朵：「那寡妇屋夜夜亮，是不是养着什么？」

「养灯。」韩泥从旁走过，声不高，「黑着才邪。亮着是活人。」

两人噎住，散开。散开好。散开了，叶青禾的灯才不像邪火。

韩泥回棚编筐，筐边缠细藤。细藤他留专用——门楣那根同批。同批藤，手熟，手熟才爆豆准。准了，她取壶的时间刚好；不准，豆早灭，壶冷在石下。

刘婆过来看藤：「你编虫丝做什么？」

「练手。」韩泥说，「细活稳，粗活才不飘。」

「稳就好。」刘婆哼，「稳着别往外门小舍钻。钻了，闲话比狼多。」

「不钻。」韩泥应，「远着留。远着，她安全。」

午后秦执事廊下问话：「丙九有人近坡下女舍？」众人摇头。韩泥摇得最慢，慢得像个听不清的聋。聋有时是护身。

问完散。沈枯芽躲在药田垄后，脸色白：「刚才吓死。」

「怕就对了。」韩泥说，「怕了嘴才严。严了，灯才长。」

夜里他又去一次，不碰门，只听缝内呼吸平不平。呼吸平，说明她睡实一点。睡实一点，他袖里油壶就能少补一晚。少补，痕迹少；痕迹少，赖福少由头。

回棚路上撞见铁无言。铁无言看他一眼：「手上豆油味。」

「灯芯。」韩泥说。

「灯芯也好。」铁无言极低，「灯芯别变成火把。火把一亮，全宗都看见。」

「不变。」韩泥道，「只留一线。」

一线在坡下亮着。亮着，就是伏笔。伏笔立了，丹才不是凭空砸下来的东西——丹要沿着灯走，走缝，走温，走到「趁热」两个字里。字未写，灯先在。在了，明日雨若肯斜，窑里就能成。

他按坛入睡前，把「别来丑」三字又温一遍。丑可以远，恩必须近。近在缝，不在面。面不见，魂才安。魂安了，还才算人做的事，不是妖做的戏。

邻床忽然问：「你今晚又晚归，咋不歇？」

「歇了。」韩泥说，「歇完还得听雨。雨斜，我才有活。」

「什么活？」

「倒渣。」韩泥道，「你懂不懂都睡。」

少年哼一声翻身。韩泥闭眼前看了一眼棚缝外的黑——黑里没有坡下那一线，可他心里有。心里有灯，脚才稳得住明日。

坡下那一线黄灯终夜不灭。他合眼前只记一句：灯留了，明日若雨斜进废窑，就把培元那粒炼出来——炼出来，才能把汤热真正还一半。
