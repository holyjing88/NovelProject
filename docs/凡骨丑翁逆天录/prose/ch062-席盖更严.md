# 第六十二章 席盖更严

瓮醒前夜，风硬。硬像最后一道冬关。关不过，缘不满。缘满，才误触那一滴。

韩泥清栏归来，掌茧新裂，裂口渗血，血珠饱满。他甩手，血甩破席，席吸尽，沿仍净。

他不主动滴血，不揭布诱沿。只用活计扎出的血喂席，席挡沿，沿等误触那一瞬。

邻床笑：「坛角盖那么严，藏宝？」

韩泥不辩，只把席盖又加一层。层厚，厚像秘密。秘密是坛，坛是命，命跟了五十八岁。

管事查棚，灯晃坛角：「杂物收起。」

韩泥只答一声，席盖不动。动像心虚。不虚，是守。

管事盯半息，走了。只见破席，不见仙缘。

夜，符怀暖一丝，护骨寒，手不僵。僵，明日藤刺偏。不偏，缘才正。

他低声对坛：「明日或后日，藤刺还会扎。扎了，血还会来。来席，不来沿。沿上，等你认的血。」

坛腹温一线，贴掌，贴到心口也暖半分。暖，不像火，像认亲前的试探。

更鼓三响，响沉。丑时将至，像丑时坛温那一线又要走一圈。

走一圈，走到编筐扎掌，走到血误触席，走到席渗沿，走到沿颤，走到瓮醒。

他合眼，不睡实。符在怀，坛在角，席盖严。

他低声道：「席盖严了。严了，才接得住那一误。误了，根在。根在，路长。」

坛不应字，只温，温得像答应，像等。

刘婆端姜汤，姜薄，热：「前夜别空肚。空肚，手飘。」

「不空。」韩泥喝尽，只点头。

赖福讽：「盖席盖出仙缘？」

管事声冷：「再嗓，克半月麸皮。」

赖福哑了。哑，韩泥加席层的手却更稳。

秦霜远廊经过，停半息：「前夜别空肚。」她不再言，木鱼在指间，不托，不敲——像规矩还在耳，像正测三关还在柱上。

铁无言门外脚步停半息，声低：「沿净？」

「净。」韩泥说，「血在席，不在沿。」

「不主动滴？」

「不滴。」韩泥说，「滴，要等活计误那一甩。」

铁无言点头，脚步远。

韩泥又加一层破布，布旧，旧更能吸。吸血，挡沿，沿等韩氏守瓮血那一误。

邻床问：「盖那么严，热不热？」

「热不要紧。」韩泥说，「沿净要紧。净了，才接得住瓮醒。」

席盖严，严像守了五十八年的命。命跟坛，坛跟缘。

他按心口，低声道：「丫头那碗汤，热着还。还，要等瓮醒后，路才长。」

坛腹温一线加剧，加剧，像明日就在沿边等那一滴守瓮血。

他低声对坛：「席盖严了。守得严，韩家不散。」

坛不应字，只温，温得像答应。祖父韩老栓嘱过：「坛在，韩家不散。」

误触那一滴，缘才满。满，不在心急，在活计，在席，在沿。

他独眼平，手不抖：「守住了，才配醒。醒在缘，不在舌。」

编筐第三刻，末藤硬，刺偏，他仍偏手，刺浅些。浅些，血仍来。血落席，席边湿，沿不接。不接，像等缘。

管事第三趟查棚，灯晃坛角，只见破席，不见仙缘。不见不要紧，沿要净。

老耿挑水路过，声哑：「前夜别拜。拜，路断。」

韩泥应声：「不拜。席盖严，不主动滴血。」

他低声对坛：「钩瓮醒，钩在沿。沿净，钩才牢。牢了，才误那一滴。」

坛颤极轻，轻像应钩。

管事验筐：「季末记功在。手稳，沿还净？」

「席边湿，沿仍净。」管事记簿，走了。

午后，搬柴一担，肩勒旧印，印深，像丙九五年。肩沉不要紧，手稳才要紧——稳，明日藤刺才不偏。

清栏第二趟归来，掌茧渗血，血珠饱满。甩手，血甩破席，席吸尽，沿仍净。他加第三层破布，布更旧，旧更能吸。层厚，厚像钩——钩瓮醒，钩在沿，钩在误那一甩。

刘婆第二趟姜汤，热。韩泥喝尽，不空，手才不飘。

他独眼平，手不抖，低声道：「最后一夜，席盖严。不主动滴血，活计喂席，席挡沿。」

同批少年又碰他袖：「你守坛六十日了？」

「守。」韩泥说，「守席，守沿，守不主动滴血。」

「滴血干啥？」

「主动滴，像借腥。」韩泥说，「腥借来，路易断。」

外头更鼓又三响，响沉。沉里，瓮醒将至。将至，沿要更净，手要更稳，席要更严。

邻床问：「误不了咋办？」

「误不了，就等。」韩泥说，「等，像守瓮人。」

他不拜邪，不揭席诱沿。诱沿，像借腥。腥，路易断。

席盖更严，钩瓮醒。钩在缘，钩在误那一甩。

最后一夜，风硬，符暖，手稳。稳，才配明日藤刺再扎那一误——扎了，血找席，席挡沿，沿等认主那一滴。

他摸木牌，牌边磨毛，毛得像心口七笔。七笔分列，不混。混了，手飘；不混，手稳。

祖父韩老栓嘱过：「坛在，韩家不散。」嘱在耳，不在舌。舌急，血偏；偏，席挡不住。

天边白一线，白像瓮醒就在下一更。下一更，或明日编筐，或丑时坛温——温加剧，沿颤，瓮醒。

他低声对坛：「席严，沿净，符暖。稳了，才配你醒。」

坛不应字，只温一线加剧，加剧，像沿边已等那一滴守瓮血。等，不在舌，在活计；活计误那一甩，缘才满。

铁无言最后一嘱：「沿净，不主动滴。滴，等藤刺误那一甩。」韩泥只答，手不抖。

外头风硬，更鼓沉。沉里，席严，沿净，符暖，手稳——稳了，才配丑时坛温再走一圈；走一圈，走到沿颤，走到瓮醒。

他按席，席贴坛，坛腹温加剧，像心跳。跳，不认主；认主，等误那一滴。滴在缘里，不在心急。

最后一夜，他不睡实。不睡实，像辨香前夜——鼻下静，手上才静。静了，才接得住瓮醒那一误。

他擦沿半刻，沿滑，沿净，沿上无血。无血，才像席盖更严守了六十日——守席，守沿，守不主动滴血。

坛角破席层厚，厚像秘密。秘密是坛，坛是命。命跟了五十八岁，跟到瓮醒前夜这一更——更尽，或明日藤刺，或丑时沿颤，瓮醒。

秦霜远廊又一句：「手稳，心别邪。邪，符灭。」韩泥怀符暖一丝，不邪。不邪，才配沿边那一误；误了，根在，路长。

还，先还烫——烫在瓮醒后，不在这一更。这一更，只守席，只守沿，只等误那一甩。

席盖更严，钩瓮醒。钩在缘，钩在沿，钩在活计误那一滴——滴在下一更，不在舌上。

第七层破布压上，压得像给坛角戴了冬帽。帽丑，厚才真。真在能吸活计血，不在藏宝。

管事第三趟路过，靴底带泥。泥不进沿，沿仍净。净了，他加第八层角，角旧，旧更能挡——挡到沿前，沿不接，缘才满。

祖父嘱在枕下与木牌并放。嘱短，字丑，丑在还能读：「误了，根在。」根在，路才长；路长，瓮醒后才还得了那碗热汤。

天边白一线更深，白里他独眼盯沿——沿尘，沿滑，沿上无血。无血，像给六十三翁预备的那一滴，还在缘里等着。更尽，沿颤，瓮醒。
