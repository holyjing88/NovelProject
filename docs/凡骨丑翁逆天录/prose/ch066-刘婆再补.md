# 第六十六章 刘婆再补

克扣第三日。

赖福秤杆落盘，落轻半袋麸皮。半袋，丙九每人少一口，少一口，粥更稀。稀，刘婆勺仍沉，沉得像顶针顶了一辈子的布。

韩泥不接辩。辩，连坐；连坐，沈家那套短签又来。他按木牌，木牌旧，字深：「仍末。」

仍末，他熟。熟不等于认命——命在坛，坛醒了，克扣克不走根。根在，饭路就在。

管事弟子巡灶，鼻下嗅：「麸皮按册发。少一粒，记赖福名。」

赖福脸一白，慌忙把秤杆抬半寸——抬迟了，管事已记簿一笔。笔丑，真；真在「克半袋」，真在「丙九补发」。

韩泥不谢。谢轻，记在心里：克扣反弹，弹在秤上，不在舌上。这是小胜——胜在饭落肚，胜在赖福哑半息。

赖福眼细，细得像牙：「丑杂役，你告的？」

「我没告。」韩泥说，「管事有鼻。鼻下静，才验得出麸皮少。」

赖福想再尖，刘婆眼一瞪：「再嗓，粥也别喝。粥稀，不是你克的？克人克到灶上，像扒皮。」

同批少年窃语：「赖福也有怕的？」

刘婆嗓更高：「怕什么？灶是活路。活路断了，全丙九饿。」

赖福唇动，终是没再尖——尖不过「灶」字。灶在，刘婆在，丙九才有一口稀粥喝。

韩泥端碗，碗糙，糙得掌印深。刘婆瞥他肘袄，瞥见针脚——针脚密，密里又松一线，松在风进处。「肘又漏风？」

「漏一线。」韩泥说，「清栏挂袋，挂的。」

刘婆哼：「挂袋也挂，风也进。进风，臂僵，僵了抬不了粪。」

她从袖底摸针线包，包旧，顶针亮：「坐下。站着，风仍钻。」

韩泥坐凳角。凳歪，歪得像丙九末席——席末，也是席。刘婆穿针，针老，手稳：「老婆子补，不收钱。钱要，你们绝户拿不出。拿恩，恩太重，老婆子扛不动。」

韩泥不接「扛不动」，接事实：「肘破，风进。」

「进就补。」刘婆咬线，线紧，「补完，别谢。谢轻。记着就行。」

针走，针脚糙，糙得牢——牢，才挡风。风不进，臂才抬粪；臂抬得动，正测才站得住。韩泥肘伸，伸得像抬棺——抬棺不抖，补肘不躲。

顶针顶在布上，顶得布发紧，紧里又松一线——松在旧裂处，裂处像泥岗冬里那道疤。刘婆不问他疤从哪来，只问：「抬粪还抖不抖？」

「不抖。」韩泥说，「抖，粪铲偏。」

「偏了，管事记。」刘婆咬线，「记了，仍滚。滚字在柱，不在肘；肘不漏，手才在。」

针脚走一圈，走得像给第七笔盖印——印不在纸，在肘上；在肘上，粥才热，恩才不断。

刘婆低声：「老婆子顶针顶一辈子，顶的是活，不是脸。脸丑，肘不能漏风。」

「不漏。」韩泥说，「漏了，清不了粪。」

「知道就好。」刘婆咬线，线紧，「别挂粪栏。挂了，我又补。」

记着——韩泥最会。会所以指节白了一息，白完，又平。门外赖福路过，瞥一眼，笑薄：「刘婆，你也沾晦气？沾了，粥更稀！」

刘婆头也不抬：「粥稀是你克的，不是他克的。他克你，你早倒了。」

赖福脸一青，走了。走，也是小胜——胜在灶边，胜在粥里，胜在有人肯给丑杂役顶针顶布。

韩泥低声道：「记着。」

刘婆哼：「记着记着，老婆子又不少块肉。」

肉不少，恩却多了一笔——多在心里，不落字纸。字纸只记饭，饭稀，恩不稀。第七笔与泥岗六笔分列，不混；混则还错人，汤就凉。

午后，麸皮补发到。发到，粥厚半分。厚半分，像窄路又宽半寸——宽，不是灵根好，是七笔没断。韩泥喝粥，热在，热得像盖第七笔的印。

管事当众验袋：「丙九补发齐。赖福，半月无赏。」

丙九静一息，静里，有人看赖福——看，也像小胜。赖福晃秤，晃得像不甘：「补发一次，下次还克。」

韩泥不辩。辩，费舌；舌要留来活。活长了，克扣迟早还——还不在秤，在将来，在正测，在能还恩的那一日。

铁无言路过，声低：「鼻下静，手才稳。稳了，渣场仍收你。」

韩泥应声：「稳在鼻。」

管事记簿，记丙九韩泥「克扣日仍清栏细」。细，是小胜。胜在活，不在嘴。

补完肘，他立刻清栏第一趟。新针脚挂袋角，扯开半寸，风又钻。刘婆在灶边瞧见，顶针一亮：「回来。再补半针。」

韩泥回灶，不辩。辩，费舌；舌要留来抬粪。刘婆再穿针，针脚更糙，糙得牢：「挂袋也挂，风也进。进风，臂僵；僵了，坡下站不住。」

「站得住。」韩泥说，「站不住，叶汤还不了烫。」

刘婆哼一声，没再嗓。嗓要留来舀粥，粥稀，恩不稀。

翌日卯时，韩泥仍清栏第一趟。赖福堵栏口：「补发了，就能偷懒？」

韩泥铲下稳：「偷懒，管事记。记了，仍滚。」

赖福噎半息，走了。走，也是小胜——胜在舌留住了，胜在粪仍清细。管事路过，瞥一眼：「手稳。」

二字，轻，真。真在活，不在灵色。管事又当众念补发令：「丙九麸皮齐，赖福半月无赏。再克，连坐。」

念完，丙九静一息，静里有人看赖福——看，也像小胜。韩泥不近，不谢；谢轻，恩在粥里，在肘上，在第七笔里。

夜，他擦坛沿，沿温一线，应「刘婆」，应完又敛。

敛着，瓮醒；醒着，步先稳。恩在——恩在粥，在肘，在第七笔里。他低声对坛：「克扣反弹，弹在秤上。秤稳，手才稳。」

坛不应字，只温。邻床问：「刘婆为何补你？」

「风进肘。」韩泥说，「补风，不补脸。」

少年懂「脸」，闭嘴。闭嘴，像也学会忍。忍不是怂，是活路——路通向正测，通向还，通向瓮醒后的每一日。

刘婆第四声咳，声更低：「正测前，姜在记里。」

韩泥应：「在。在，才还烫。」

更鼓沉，赖福脚步远，远里仍有一句：「丑杂役，我盯着你。」

韩泥只答：「在。」

在，就不滚。不滚，恩才不断——不断，汤就不凉。刘婆粥热半勺，热在，像第七笔又实。他抚肘新针脚，针脚暖进骨，骨暖，臂才抬得起粪，才站得住坡下末排。

坛腹温一线，贴掌，像刘婆顶针顶的那一下，也顶进了心里。顶进了，路还长——长得够还，够活，够等冬后那一验。

他低声道：「第七笔在，克扣弹了，粥厚半分。厚半分，肩才扛得住坡下那日。」

恩在，不在册。册废了，心还在——还在，汤就不凉。

翌日，麸皮仍按册发，秤杆落盘，落实——落实，像克扣那笔弹回去了。赖福眼细，没再落轻。没落轻，也是小胜。胜在饭，不在嘴。

韩泥清栏，铲下稳，稳里肘上新针脚不挂袋角——不挂，风不进；风不进，臂才抬。柱上条角白一线，白不要紧——白在远，热在粥；粥热，手就不飘。

老耿挑水路过，水洒稳，声哑：「肘补了，别拜邪。拜，路断。」

韩泥只答：「不拜。恩在，不在舌。」

刘婆收针，顶针在灶沿敲一下，敲声轻，轻像盖印：盖在肘，不在舌。

韩泥端碗，粥仍稀，稀里热半勺。热半勺，像第七笔又实一寸——寸小，肩才扛得住克扣弹回去那半袋，扛得住正测前这三日。他抚肘新针脚，针脚暖进骨，骨暖，臂才抬粪。抬粪不抖，正测才站得住——站得住，恩才不断；不断，汤就不凉。

坛腹微温，应「刘婆」二字，应完又敛。敛着，才接得住正测前三日——三日里，手在，恩在，第七笔在。恩在粥里，也在肘上新针脚里——针脚丑，挡风；风不进，臂才抬得起粪，才站得住坡下末排。

三日后卯时，坡下见——见在末排，不在舌。手在，就不滚。
