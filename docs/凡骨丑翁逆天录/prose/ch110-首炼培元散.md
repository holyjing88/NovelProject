# 第一百一十章 首炼培元散

丑时。

药山废窑，窑口红，红稳，稳像109余韵落地。落地，才配首炼——炼培元散，散在一品，品在路，路在渣。

韩泥分拣，指稳。青芯五撮，褐渣三把，灵泉七滴，投序照经页——页深，深不要紧；深，是散，不是丹。

丹在95，赠叶；散在110，自用，自用是丹道起点。

炉温起，起一寸，一寸稳。稳里，白气聚，聚而不散，散则败——败不要紧，败过，这一回要成。

成，在鼻下。鼻下嗅诀动，动如秤，秤香，秤色，秤「正」。正来慢，慢不要紧；慢，才正。

正来一瞬，窑内无雷，无光，只有掌纹那路烫。烫，像叶汤碗沿，像门缝灯，像清瘴并肩——并肩，手稳，手稳，才成散。

散成，成灰白，白细，细像雪，雪像药。药入粗纸包，包丑，丑不要紧；丑，才像杂役该得的。

得，不在名，在正。正，他嗅半息，半息够——够自用，够备130远，够还下一笔恩。

不自服。服，像破境；破境不在110，在260远。远不要紧，110只起点——起点在散，散在废窑，窑在灵脉浅穴。

浅穴，他占住了。占，不声张。声张，赖扒皮来。来不要紧，窑深，声不出。

铁无言立窑外，不进来，声低：「成了？」

「成了。」韩泥说，「散，不是丹。」

「正？」

「正。」

正，一字，铁无言点头，点像证。证，不在嘴，在鼻。

下山，遇管事弟子，弟子验包，包灰，灰里白细，细像药。药不要紧，管事不验名——名在远，在130「丑丹师」。

赖福堵路：「又上山？上山成什么？」

「倒渣。」韩泥声平，「手稳。」

赖福鼻动，动半息，没再追。没追不要紧，散在袖，袖扎紧。

回棚，席下按坛，低声道：「首炼过了。散在，路在。」

坛腹温，贴掌，像应「起点」二字。起点，不是终；终在1555远。

刘婆端粥，粥稀：「兽潮后，你还炼？」

「炼。」韩泥说，「炼了，才还得了恩。」

叶青禾在坡下，门缝灯未亮。未亮，像避；避不要紧，脉里培元仍温——温，是95；散，是110，两笔不混。

混，七笔分列；列在心，不在舌。

丑时后，他廊下静半刻。香落齐，掌纹温，温而匀。匀，像六层实，实而不飘。

邻床问：「你真成丹师了？」

「不成。」韩泥说，「成路。路在散，散在渣。」

更鼓尽，废窑温退，退像敛。敛不要紧，敛，为下一程——下一程，赖扒皮压，炼气六层往上，坊市再挂牌。

他低声道：「培元散，正。丹道，起点。」

起点在丑手，不在名；名在远，路在掌。掌在，就不滚。

铁无言点头，点像证。鼻下静，手不飘。飘了，散不正；不正，路断。路不断，111起，赖扒皮压，炼气六层往上——往上，不在嘴，在掌。掌在，就不滚。

首炼丑时，星晦。星晦不要紧，晦才好走。

秦霜木鱼轻敲：「首炼别骄。骄了，手飘。」

韩泥不答，答在心里——心里不骄，只记。记正，记起点，记六层愈压在前头。

散成灰白，细像雪，嗅之正。正了，才配首丹——首丹在暗，不在众目。

骄了，手飘。愈压不要紧，压，手才稳。走下山，遇管事弟子，弟子验包，包灰，灰里白细，细像药。管事不验名——名在远，在130「丑丹师」。不验不要紧，正才验。
