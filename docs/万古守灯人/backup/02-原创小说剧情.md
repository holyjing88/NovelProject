## 世界观

### 时代与地理

**大玄历·承平三十七年**，承平盛世表象之下，田赋沉重、豪强兼并。

四方格局：
- **玄京**：王朝都城，皇权、宦官、世家三角博弈
- **青萝镇**：主角故乡，江南水乡，药铺茶肆烟火气
- **云岚宗**：西南修仙大宗，外门杂役、内门真传等级森严
- **幽灯集**：每月朔日子时开启的地下集市，以灯为契、以魂为价
- **枯骨岭**：南疆禁地，上古战场遗址，时间流速异常

### 力量体系：「灯道」十二阶

不同于传统灵气修炼，灯道以「心念」为火、以「记忆」为油：

| 阶位 | 名称 | 能力 | 代价 |
|------|------|------|------|
| 一阶 | 微光 | 感知十步内生灵情绪 | 消耗当日一份温暖记忆 |
| 二阶 | 烛火 | 短暂照亮隐匿之物 | 失明一炷香 |
| 三阶 | 灯芯 | 凝练灯油（可储存时光） | 须发变白一缕 |
| 四阶 | 灯盏 | 以灯油加速或减缓自身时间 | 折寿三年 |
| 五阶 | 灯影 | 投射过去某一时刻的幻影 | 遗失一段真实记忆 |
| 六阶 | 灯骨 | 以骨为灯架，硬抗致命伤 | 永久失去一种感官 |
| 七阶 | 灯魂 | 窥见他人命灯明暗 | 承受对方一炷香内的痛苦 |
| 八阶 | 灯河 | 操控小范围时间流速 | 沉睡七日 |
| 九阶 | 灯域 | 展开灯域结界 | 灯油耗尽则神魂俱灭 |
| 十阶 | 灯天 | 照亮一城气运走向 | 与一城因果绑定 |
| 十一阶 | 灯道 | 言出法随，但仅限"照亮真相" | 不能说谎七日 |
| 十二阶 | 万古灯 | 为人间留一盏永不熄灭的守灯 | 永世不得飞升 |

**核心规则**：灯道修行者越强大，越无法远离人间。十二阶「万古灯」的终极境界，是化作青萝镇街口那盏风雨无阻的长明灯——照见他人前路，自己再无归途。

## 人物设定

### 主角：顾迟年

- **年龄**：六十岁，青萝镇退休书吏
- **身世**：年轻时曾赴玄京赶考，名落孙山；回乡后替人代写书信、誊抄账册三十年
- **性格**：表面温吞啰嗦，爱念叨陈年旧事；骨子里执拗，认定的事十年不改
- **口头禅**：「急什么，灯还亮着呢。」
- **弧光**：从「只想安稳过完余生」→「为身边人守灯」→「为一座城守灯」→「万古长明」

### 重要配角

**沈青禾**（女主之一）
- 二十七岁，青萝镇药铺掌柜
- 丈夫三年前死于豪强逼债，独自撑起药铺
- 顾迟年少年时曾暗恋其母，对青禾如父女又如故交
- 后服食灯油开窍，走上灯道，与主角并肩

**铁柱**
- 四十五岁铁匠，顾迟年发小
- 性格憨直，力气惊人但灵根驳杂
- 云岚宗杂役弟子，是主角进入修仙界的引路人
- 代表「凡人尊严」的镜像角色

**云照长老**
- 云岚宗藏经阁守阁人，外表七十岁老妪
- 实为两百年前灯道七阶高手，因耗尽灯油自封修为
- 传授顾迟年《守灯经》的真正含义
- 临终遗言：「灯不是为自己亮的。」

**陆承安**
- 玄京世家子弟，云岚宗内门首席
- 并非脸谱反派：他真心想以修仙拯救乱世，但方法是用秩序碾压一切
- 与主角冲突核心：「照亮真相」vs「塑造真相」

**裴无妄**
- 幽灯集神秘摊主，卖「后悔药」「忘情烛」「续命油」
- 真实身份是上古灯道十一阶残魂
- 每次交易都要收走买家最珍贵的一盏「命灯记忆」

## 主线剧情（五卷结构）

### 第一卷：青萝镇的最后一盏灯（第1-40章）

**开篇**
承平三十七年秋，顾迟年六十大寿。整理阁楼时发现一本无字书册，午夜自燃，浮现《守灯经》三字。书册燃尽，留下一盏青铜小灯——**守岁灯**。

守岁灯能力：每积攒一日「人间烟火」（帮邻人做一件实事），可凝一滴灯油。灯油可延缓衰老（一年灯油延缓一岁衰老）。

**关键情节**
1. 沈青禾药铺遭豪强逼债，顾迟年以微光阶感知识破对方伪造借据
2. 铁柱推荐他参加云岚宗「杂灵根选拔」——六十岁上限，今年最后一次机会
3. 选拔试炼：在守夜林中独守一夜灯火，顾迟年以「给迷路猎户指路」攒够灯油，成为史上最老杂役弟子
4. 云照长老暗中观察，留下一句：「你这盏灯，不是为自己点的。」

**卷末高潮**
豪强夜袭青萝镇，烧毁镇口长明灯。顾迟年以三阶灯芯之力，燃尽三十年记忆，重现长明灯幻影，吓退豪强。全镇人以为神迹，顾迟年却忘了自己少年时为何落第。

**卷末感悟**：「原来照亮别人路的时候，自己的过去也会变暗。」

---

### 第二卷：云岚宗的杂役老头（第41-90章）

**核心冲突**：宗门资源分配不公，杂役弟子被当作消耗品。

**关键情节**
1. 杂役堂日常：挑水、炼丹渣、清理盏兽厩。顾迟年用「烛火」照出丹渣中的残余灵材，被克扣的杂役们暗中感激
2. 坊市倒卖：与铁柱合作，以灯道感知鉴别灵药真假，小赚灵石
3. 幽灯集初探：朔日子时，以灯油为入场券，购得「疏脉丹方」
4. 内门倾轧：陆承安推行「优胜劣汰」，杂役弟子被派往枯骨岭外围采药，伤亡惨重
5. 顾迟年献计：以「灯影」投射上次平安归来的路线，救下十二名杂役

**感情线**
沈青禾来宗门送药，被陆承安看中灵药天赋，强邀入内门。顾迟年沉默三日，最终以五滴灯油为代价，在幽灯集换来「拒婚符」，让青禾自主抉择。青禾选择留下药铺，但承诺「等你筑基，我再来」。

**卷末高潮**
天煞门突袭云岚宗，陆承安设局让杂役殿后。顾迟年燃灯骨阶（暂失嗅觉），以身为灯架挡住致命一击，救下铁柱。云照长老出手击退天煞门，公开宣布顾迟年为其记名弟子。

---

### 第三卷：幽灯集与枯骨岭（第91-140章）

**核心冲突**：时间流速异常区「枯骨岭」中，上古灯道遗迹现世。

**关键情节**
1. 裴无妄揭示守岁灯真相：此灯是上古灯道十二阶「万古灯」的碎片，集齐三相可重燃人间长明
2. 枯骨岭外围：时间紊乱，一日如一年。顾迟年以四阶灯盏减缓自身时间，苦修三年（外界三日）
3. 遇见「时间残影」：看见自己若少年修仙的平行人生——飞升瑶台，但青萝镇无人守灯，乱世中全镇覆灭
4. 选择：平行人生的自己伸出手邀他交换，顾迟年拒绝：「瑶台太冷，我这盏灯得在镇上亮。」
5. 陆承安亦在枯骨岭，走捷径吞噬他人命灯，突破六阶灯骨，但开始遗忘自己姓名

**卷末高潮**
枯骨岭核心「万灯冢」开启，需以真心记忆为祭。陆承安献祭他人记忆强行闯入，神魂错乱。顾迟年献祭自己「初恋记忆」，获得第三块守岁灯碎片，晋升七阶灯魂，窥见陆承安命灯将灭。

顾迟年伸手拉住他：「你的灯，还没灭呢。」

---

### 第四卷：玄京风云（第141-190章）

**核心冲突**：王朝气运衰竭，修仙界与皇权联手「封灯」——禁止民间灯道修行，垄断光明。

**关键情节**
1. 玄京诏令：民间私修灯道者，以「乱心妖术」论处
2. 顾迟年潜入玄京，以五阶灯影在朝堂重现豪强逼债真相，震动朝野
3. 女宰相**谢长缨**（非女帝设定）推行「开灯令」：允许灯道用于刑狱照明、赈灾指引
4. 云岚宗分裂：陆承安率亲信投靠皇室「镇灯司」，成为「封灯」执行者
5. 青萝镇遭镇灯司围剿，铁柱以凡人之躯挡在镇口长明灯前，重伤濒死

**感情线**
沈青禾以灯油开窍，晋升四阶灯盏，率药铺学徒组成「青萝灯会」，为灾民照明引路。她与顾迟年在烽火中重逢，只说：「灯还亮着，我就来了。」

**卷末高潮**
玄京城下，陆承安率镇灯司围攻谢长缨。顾迟年九阶灯域展开，照亮全城气运——百姓命灯皆暗，唯镇口长明。陆承安看见自己命灯中最后一缕光，是少年时母亲为他点的那盏油灯。他放下剑，以六阶灯骨承接镇灯司致命一击，身死前说：「原来……我也想有人为我守灯。」

---

### 第五卷：万古长明（第191-220章 · 终卷）

**核心冲突**：守岁灯三相归一，顾迟年面临终极选择——飞升或留人间。

**关键情节**
1. 裴无妄揭示终极真相：万古灯不是境界，是一种「选择」——化作人间一盏永不熄灭的长明灯
2. 天煞门余孽联合域外天魔，欲熄灭人间最后一盏命灯，让世界陷入永恒黑暗
3. 云照长老燃尽残魂，以两百年底蕴为顾迟年铸灯架
4. 铁柱以凡人之躯率青萝镇百姓，以柴火、烛光、目光汇聚成「万家灯火」
5. 沈青禾将毕生灯油注入守岁灯，青禾变白发，顾迟年泪落

**终章：青萝镇的雨夜**

天魔退散，雨过天晴。顾迟年站在镇口，手中守岁灯与长明灯融为一体。

裴无妄问：「飞升之门已开，你走不走？」

顾迟年摇头，将灯挂在镇口槐树上：「我走了，谁给赶夜路的人照路？」

他的身体渐渐透明，化作灯火融入长明灯。从此，青萝镇的长明灯风雨不灭。

**尾声（百年后）**
沈青禾七十大寿，青萝镇已成灯道圣地。铁柱的孙子在镇口朗读：

「顾先生说过，急什么，灯还亮着呢。」

远处，长明灯微微闪烁，像是有人在笑。

## 支线剧情

### 铁柱线：凡人的骄傲
铁柱始终无法突破灯道一阶，但以凡人之躯守护长明灯，最终获得「万家灯火」加持，成为唯一一个「无阶却能点灯」的守灯人。

### 裴无妄线：后悔药的代价
裴无妄曾是上古灯道天才，因用灯影篡改过去导致全族命灯熄灭。他在幽灯集万年，只为等一个愿意「不修改过去、只照亮未来」的守灯人。

### 谢长缨线：制度之灯
女宰相不以武力，以「开灯令」让灯道走进刑狱、赈灾、科举，证明人间变好不需要仙人降临。
