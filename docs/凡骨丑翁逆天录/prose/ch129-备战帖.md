# 第一百二十九章 备战帖

试台末项前夜，备战帖至。

山门执事传话，话像鼓余音：「明日末项，培元散比试夺魁。魁者，挂牌丑丹师，递大比预选帖。」

话落丙九，丙九嗡一声。嗡里，赖福脸白，白得像嚼不出晦气：「丑杂役夺魁？魁，太阳西边出！」

韩泥不辩。辩，费舌；舌要留来养息，留来守最后一粒——粒在怀，怀沉，沉得像129末这夜。

玉牌贴胸，凉，凉里烫着三年约——约说炼气十三，站钱戾衡台前，才不滚。

刘婆端粥，仍多半勺，半勺像送行：「明日末项？」「末项。」韩泥说。

「末项别空肚。」刘婆哼，「空肚，手飘。飘了，新衣还不上。」韩泥点头。谢轻，恩重。

铁无言立门外，声低：「明日我站台侧。魁名落，别辩。辩像显摆恩。」

「不辩。」韩泥说，「恩偿，静着还，还到刘婆手里，才算数。」

钱戾衡影在山门帖背面，背面字淡，淡得像眼风：「言伯钧预选，内门少主监。」

白织月袖风过，过得冷：「明日魁，仍是运气。」韩泥捧陶罐，罐稳：「运气试台见。见正，见还，见刘婆。」

沈枯芽远远鞠半躬，不近。不近不要紧，徒恩清了半块灵石；清了，韩泥心里排序再落一笔。

秦霜远木鱼敲一声：「帖至，心别邪。邪了，鼻下香飘。」韩泥不答，答在心里。

丑时，练鼻下静。粟壳香淡，香落齐，备战帖这一夜记进心里。

邻床少年悄问：「你真敢夺魁？」「敢。」韩泥说，「不夺，三年约白接。白接，恩断。」

他望坛，低声道：「明日魁，后日还刘婆。还了，帖在怀，大比在路，钱戾衡在台前。」

外头山门风硬，帖角掀，掀出「大比」二字影。影在字后，像仇在路后；路后不要紧，刘婆这笔先还。

更鼓将尽，天边白一线。白里，试台木色浅，浅像70踹台——踹台接了约，约接到明日魁。

他按心口，低声道：「汤还是热的。」

六字落，像路标——标向130还刘婆，标向大比帖，标向钱戾衡台前不滚。

丑时末，他再理布包——新衣叠实，钱串齐，陶罐封泥，九粒培元散在怀。怀沉，沉得像债序最后一笔待落。

外门弟子渐少，韩泥不近廊下笑。笑在耳，不在心——心在坛，坛腹温一线，温像应「帖」。

赖福廊外最后骂一句：「明日夺魁？晦气！」韩泥不辩。舌要留来养息，留来守最后一粒。

他再按玉牌，牌凉，低声念：「三年约，记着。钱戾衡，记着。明日末项，记着。」
