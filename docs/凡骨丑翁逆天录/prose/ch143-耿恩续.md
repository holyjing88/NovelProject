# 第一百四十三章 耿恩续

送出的次日，风硬。

沈枯芽回，眼红，红像叶姐脉稳半分——稳了，散才服；服了，耿爷才肯收。

「耿爷服了。」芽儿声低，「他说，粒正，他懂。绳不抖，粒也不抖。」

韩泥不答懂，答在心里。心里记28抬棺，记15修瓮，记142这一粒。

坊市议：「丑杂役也送丹？」议在舌，韩泥不辩。辩，费舌；舌要留来守坛。

管事记簿，记「丙九韩泥，培元散谢恩」，记细，是小胜。

刘婆哼：「耿老头聋，心可不聋。」

韩泥点头。心聋，粒白送；心不聋，恩才落簿。

铁无言验渣，渣灰，灰里正：「送出了？」

「送出了。」韩泥说，「经叶家，不进门。」

赖福路过，鼻动，没嗅出粒——粒在远，在耿爷怀。

丑时，练鼻下静。香落齐，143这一日记进心里。

他望坛，低声道：「耿恩收了。记着。」

坛腹温，贴掌，像应「⑥」——⑥还了，144才配传名。

泥岗风硬，硬里传来半句哑声——声是老耿，老耿耳聋，听不全话，听全了心。

韩泥未近岗，近岗像攀恩；恩在，记便够。够了，143只收余波，不收戏。

沈枯芽回，回时眼红，红像叶姐脉里那一线稳——稳了，散才收得住；收得住，老耿才肯服。

「耿爷服了。」芽儿声低，「他说，绳不抖，粒也不抖。粒正，他懂。心里记28抬棺，记15修瓮补漏，记142这一粒——粒小，恩大；大，不在声，在稳。坊市有人议：「丑杂役也送丹？」议在舌，真在耳。耳听全了，143才像余波，不像扬名。」「送出了。」

「「」韩泥点头。胜在还，不在辩。赖福路过，鼻动，没嗅出粒——粒在远，在耿爷怀，在七笔⑥那页。记着。」坛腹温，贴掌，像应「⑥」——⑥还了一半，一半在262。更鼓沉，天边白一线。白里，143过了，144在前——在前，丑丹师名号才配传。

回丙九，不近廊下议。议在舌，不在心——心在坛，坛腹温一线。

沈枯芽走后，廊里静，静像142那粒还在耿爷怀——还在，143才像收，不像送。

铁无言点一下头，点得像28经片那记——记耿恩，不拜邪。

坊市有人议，议声远，远不过掌里那半分稳。稳在，143才像余波，不像扬名。

韩泥按心口，低声道：「⑥落半笔。262在远。记着。」

白里，144泥纹名该起了——起在渣，不在舌。

143不写炉，写收——收老耿服散那半句哑声，收沈枯芽眼红那一线，收坊市舌上「丑杂役也送丹」的议。议在舌，不在掌；掌里已无粒，粒在耿爷怀，在七笔⑥那页。页在远，262才还尽；143只落半笔，半笔也要记。记28抬棺，记15补瓮，记142经叶家——家缝窄，缝能递，递了才像报恩②，不像显摆。

显摆，赖福来嗅；嗅不到，粒在远，143才像余波，不像扬名。扬名在144泥纹，在147续；143只温耿恩，温像经片压坛底那丝温——温浅，够记。记了，铁无言验渣，渣正，正像142后手未飘；未飘，146十七才实，147名号才续。续在远，143只收。

收完，韩泥不近泥岗，近岗像攀；攀，费舌；舌要留来守坛，留来候144。候144，不是急，是序；序里写：142送出，143收波，144名起——名起在渣，不在嘴。嘴漏，用心堵；堵法是不辩，是不滚，是坛在人在。人在，耿恩就在；在，写哑声，写绳不抖，写粒也不抖。抖了，恩假；不抖，143才真。

真，更鼓沉，天边白一线，白里，144在前。

耿恩收了，风改一半；一半改，144泥纹名才配起。名起在渣，不在舌；舌要留来守坛，留来候144。

廊静，议远，远不过坛腹那一线温。温在，耿恩就在；在，不必近泥岗。他望帖：「⑥落半笔。144名在前。记着。」更鼓沉，天边白一线，白里，泥纹二字影将起——起在渣，不在舌。

药峰分拣照旧，指稳，鼻下静一息。

帖在怀，怀沉，沉的是序，不是惧。

白织月袖风过，韩泥不辩。

赖福廊下过，韩泥只答：「在。」在，就不滚。

柱上条角白一线，三年约在怀。怀凉，掌温。

坡下石阶湿，他数息，数帖，数债序。

同棚杂役笑，笑在耳，不在心。

废渣巷日斜，他验渣，渣正，手才静。

侧门驿亭风硬，他听风，风过，记在心里。

散队后，他过外堂阶下，分拣半刻，指稳——指稳，路就不飘。

编筐半刻，藤刺扎掌，刺浅，浅些沿才净。沿净，坛才藏得住。

粟壳香淡，香一字不落。

管事记簿，记细，是小胜。胜在忍，不在辩。

他望坛，低声道：「手在，路在。」坛不应字，只温。

白里，他按木牌，牌凉贴胸。

坊市尾巷风硬，他收摊如石，石在，就不滚。

验息后，他理怀内册页，页黄，字丑，丑而真。

铁无言声低：「别飘。」韩泥应声：「不飘。」

刘婆端粥，仍多半勺。韩泥点头，谢轻，恩重。
