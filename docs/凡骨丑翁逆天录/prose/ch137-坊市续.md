# 第一百三十七章 坊市续

坊市续，续在听，不在荣。

韩泥仍立末摊，牌仍歪，字仍丑——丑不要紧，听真。

真里，有人议大比：「言伯钧预选，内门少主监。」

议在远，在舌，在耳。耳要留来辨谣与真——谣在舌，真在渣。

一老散修递渣，渣里藏石屑——藏像使坏。使坏不要紧，鼻下一息，辛里出涩；涩是石，石要剔。

老散修脸白：「你……你真辨？」

「辨。」韩泥声平，「辨了，才续摊。续摊，不在名，在手。」

人走，走时风言：「丑丹师，有点真。」

真不要紧，真像96风声那日——那日听，这日续。

管事弟子贴条：「坊市丑丹师，续留三日。」

韩泥收条，条丑，字真。真，才像外门积势。

白织月远廊过，没停。没停，小胜——胜在渣正，不在辩。

刘婆买盐过，盐粗：「少听仇话。」

「听。」韩泥说，「听，才避；避，才活。」

铁无言声低：「坊尾有人盯你。」

「盯不要紧。」韩泥说，「盯在皮，不在手。」

回丙九，他按坛。坛腹温一线，像应「续」字。

丑时，练鼻下静。粟壳香淡，香落齐，坊市续这一日记进心里。

邻床悄问：「你真打算上大比台？」

「上。」韩泥说，「不上，约白接。」

更鼓沉，他望坛：「听风紧。下一回，守坛夜。」

坊市续，尘起，尘里混一点大比二字——二字在远，在帖，在139。

管事贴条续留，条丑，字真。

侧门驿亭风紧，紧像试台前那口气。那口气要匀，匀了，手才不飘。

编筐半刻，藤刺扎掌，偏手，刺浅些。编筐钱买米，米在，恩才还得上。

赖福查税，税半文，文旧。旧不要紧，旧，像常杂役该交的。

铁无言声低：「坊尾盯梢，别近。近，像挑衅。」

「不近。」韩泥说，「近，费舌。」

粟壳香淡，香一字不落。

他低声对坛：「坊市续了。听风紧。记着。」

老散修再递渣，渣正，正像续摊的回响——回响在掌，不在嘴。

管事弟子记一笔，笔丑，真在「续留」。真，才像137收风。

坊尾盯梢，盯不要紧，盯在皮，不在手。

韩泥收摊，收得像石。石在，风过，风过不留痕。

坛腹温一线，像应「续」。

更鼓沉，他望帖：「听风紧。139过了。140在前。」

坊市续，续在听，不在荣——荣在远，在大比，在235胜钱；137只听。听谣，听真，听坊尾盯梢；盯梢不要紧，盯在皮，不在手。手稳，老散修递渣那石屑才剔得出；剔得出，续留三日才像证。证不在嘴，在鼻；鼻下静一息，辛淡正，正像96那日。那日听风，这日续摊；续摊，牌仍歪，字仍丑；丑不要紧，渣正。

正了，138守坛夜才静；静了，139风声紧才接得住。接在玉牌，接在三年约，接在「站钱戾衡台前，才不滚」。不滚，是约；约在，手就稳。稳了，140积势才收束；收束不越142，142谢老耿，经叶家，叶汤还在前。前在远，137只续。续了，风改一半；一半改，像140该记的册页。

册页在管事手，手丑，字真；真，才像外门积势。势在，就不绝。更鼓沉，按木牌，牌凉贴胸，他低声对坛：「坊市续了。听风紧。记着。」坛不应字，只温。温像应「续」——续在耳，不在舌；舌要留来守坛，留来候138守夜。

坊市续，听风紧，紧不要紧，手稳，渣就正。

续摊第三日，条丑，字真。真了，137才像听风，不像摆摊。

风紧，不要紧；紧，才试忍不忍。更鼓沉，他收摊，收得像石。

「」「听。」活，才像137续坊市，不像一部尾韵。尾韵在130，新章在听，在稳，在忍。更鼓沉，他收摊：「听风紧。记着。」

他望坛，低声道：「手在，路在。更鼓沉，天边白一线。白里，他按木牌，牌凉贴胸。坊市尾巷风硬，他收摊如石，石在，就不滚。」

验息后，他理怀内册页，页黄，字丑，丑而真。

铁无言声低：「别飘。」韩泥应声：「不飘。」

刘婆端粥，仍多半勺。韩泥点头，谢轻，恩重。

药峰分拣照旧，指稳，鼻下静一息。

帖在怀，怀沉，沉的是序，不是惧。

白织月袖风过，韩泥不辩。辩，费舌；舌要留来守坛。

赖福廊下过，韩泥只答：「在。」在，就不滚。

柱上条角白一线，三年约在怀。怀凉，掌温。

坡下石阶湿，他数息，数帖，数债序。

同棚杂役笑，笑在耳，不在心。

废渣巷日斜，他验渣，渣正，手才静。

侧门驿亭风硬，他听风，风过，记在心里。

散队后，他过外堂阶下，分拣半刻，指稳——指稳，路就不飘。

编筐半刻，藤刺扎掌，刺浅，浅些沿才净。沿净，坛才藏得住。

管事记簿，记细，是小胜。胜在忍，不在辩。
