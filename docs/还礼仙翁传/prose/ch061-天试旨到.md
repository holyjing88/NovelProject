# 第六十一章 天试旨到

战后第七日，山门布告柱新贴金纹诏。

不是青岚门自贴，是五宗联名，纸边烫金，像把「天试」二字从卷二末尾，直接钉进每个弟子的眼里。诏文不长，却硬：

**灵息复苏三十七年，五宗天试重启。三月后，东南试炼场开。青岚须选派武试弟子、丹试弟子各若干，并立护法长老一位。**

风一过，金纹哗响，响得像帖在人心上。

演武场方向，弟子已聚了半圈，半圈里窃语如潮：

「三十年一届，咱们这辈子赶上一回？」

「听说赤焰、玄木、合欢、真言、圣辉都派人……」

「青岚门夜战刚毕，还派得出人？」

「派不出人，也得派脸。」

赵玄机立人群前，铁牌在腰，衣尚带战后补缀的线头，线头不显眼，显眼的是他背挺得直。顾小满立他侧后半步，灰袍洗得发白，怀里揣册子抄本，像揣半块灵石，揣一路，揣到布告柱前。

执事堂长老唱名布告，声平：「武试预选榜，赵玄机第一，顾小满第二，燕不归第三——」

「燕不归」三字落，外门那边一名抱剑青年抬下巴，笑浅，像剑快人也快。

赵玄机眸光微动，动得像妒又起，却起得克制——战后他还库、守门、思过，妒火薄了，薄不等于灭。

顾小满却低头，只问：「第三是谁？」

「燕不归，」赵玄机淡道，「剑快。你第二，够了。」

「不够。」顾小满声轻，「第二，是记出来的，不是藏出来的。」

赵玄机看她，看她册子边雨痕旧印，忽然不语，像想起挽月反戈那日堂上冷，冷里也有人把路记清了。

---

布告柱另一侧，人更少，却更静。

林挽月立静处，袖无林苑青纹，只剩罚役灰边，像三月之罚才走了一半。她不敢挤进人群，只远远读诏，读到「武试」二字，指尖微颤。

有人认她，窃语起：「林挽月还敢来？」

「罚役三月，来读榜也不行？」

「行，可她证过赵师兄……」

话未落，林挽月转身欲走，身后有人唤：「林师妹。」

顾小满追来，气未喘匀，只递一张薄简：「杂役堂抄的预选细则。你若还想走武试路，得先消罚役记——我不帮你求情，只帮你把字抄清。」

林挽月接过，简温，像半块灵石温：「你为何……」

「因为证字。」顾小满道，「证过了，路要走得直。直路抄本在，比慕值钱。」

林挽月眼眶一热，热得急，急急低头，像怕热被人当成悔。

围观弟子瞧见这一幕，窃语又变一层：「顾小满递简给林挽月？」「杂役堂什么时候管外门罚役？」「莫长老说的记路……真记到册上了？」

宗主殿内，柳青鸢已议过三轮。

霍镇山拍膝：「镇妖司子弟优先！夜战流过血的，别让他们坐冷板凳！」

周德海抚须，笑浅：「霍长老，天试是五宗脸面，不是镇妖司校场。护法长老位，更须稳重。」

柳青鸢目光落莫长春，停半息：「师兄，布告已贴。弟子里，赵玄机、顾小满风头最劲。护法长老……仍是你。」

莫长春袖空，一念出塔，杖落掌，笑：「老夫拄杖看热闹，热闹里若有人摔了，扶一把。」

「扶人可以，」霍镇山吼，「别让人家看轻青岚！」

「看轻才好。」莫长春道，「看轻了，礼好送。送出去，才有人还。」

周德海眼神一闪，一闪像想起侧门闸纹，像想起盟会之后那一辩，终是揖手：「莫长老说得轻巧，天试门前，轻巧不得。」

「轻巧不得，」莫长春看他，「所以老夫去。去，不是打，是镇。镇场二字，比斩字省命。」

殿外弟子听见「镇场」二字，传成几种版本，版本里都有九月老头要抛头露面，版本里也都有一丝不信——不信里夹着怕，怕里夹着盼，像东脉雨后灵田，根还在，苗还颤。

山下驿站，秦商言收讯比山门晚半日，却到得准。

讯是金边复印件，她指节敲纸边，敲得像敲价，敲完却笑：「茶饼那笔恩，又涨价了。」

下属低声：「管事，百宝阁跟不跟队？」

「跟。」秦商言利落，「天试三月，跟青岚走一路。路在礼上，不在价上——莫长老原话，商言记在心里。」

她另取飞符，符上六字：**「天试需茶，茶需人。」**

符去，像把百宝阁也绑进青岚门三月之约。

坊市茶楼，白漱玉凭窗，远观山门方向金纹诏影映云，映得像一场远戏开场。

侍女低声：「小姐，还去递名帖吗？」

白漱玉摇头：「不递。天试人多，名帖薄，薄不过茶饼。」

她指楼下符纸摊：「你看，九月老头若在，必在人群里。人群里找得到他，找不到价。」

侍女不懂，只觉小姐眼里光又起，光里疑少敬多，像慕从价里抽出来，剩一点纯。

白漱玉低声，像说给自己：「三月……够看清一个人，够不够，看他还礼还不还。」

夜，布告柱金纹在灯下仍亮，亮得像三月倒计时。

莫长春立廊柱旁，不坐主位，袖空，一念出塔，杖横膝，望东南，像望试炼场方向。

识海中塔影微烫，塔壁无全铭，只浮半句，淡金：

**天试在前，结缘未竟。**

他辨「未竟」，不点破，只笑，像笑给风听：柳在，苏在，顾在，秦在，白在远处，林在罚役——结缘二字，仍宽，宽得像三月的路，路还长。

寿元行跳：三月。与诏合，合得像帖与命终于对齐。

围观弟子渐散，散里仍窃语：「莫长老又要出山门？」「护法长老，拄杖看热闹？」「九月够去天试吗？」

有人答：「够。不够的人，连布告柱都不敢站近。」

又有人悄道：「顾小满第二，赵师兄第一——战后，榜还能这么排？」

「榜这么排，」另一人道，「像青岚门告诉五宗：门还在，人还在，礼也还在。」

莫长春听见尾音，不辩，杖中竹纹游动，气音轻，像一字：

等。

他改口，声轻却清：「不等了。三月，够送，够还，够让五宗记住——青岚门的老头，袖里没东西，塔里有恩。」

风大，布告金纹哗响，像天试在门前先敲了一声鼓。

燕不归在演武场边抱剑，剑未出鞘，鞘响像替「第三」二字敲了一下。他笑浅，笑里无妒，只有快——快得像告诉全门：榜上有名，名要凭剑挣。

苏念慈不进门，只遣人送汤至廊外，汤温，温得像怕旨进丹堂乱了炉。她低声：「天试是门的事，丹堂是火的事。火不能因旨乱，门也不能因丹怠。」

魏无涯在远檐收扇，像收一局新棋，低语：「五宗天试，是盘大棋。九月老头不抢子，只守格——格在，礼才走得远。」

赵玄机独立柱前，读「燕不归」三字，读成剑影，影里妒火薄了，薄得像战后还库留下的规矩。顾小满抄细则，抄到禁药一条，停住，像想起丹堂炉声。林挽月抄「证字」，笔稳，不知三月后有没有她的路，只知今日这一步没歪。

风大，布告金纹哗响，像天试在门前先敲了一声鼓。

秦商言另取飞符，符上六字：「天试需茶，茶需人。」符去，像把百宝阁也绑进青岚门三月之约。

燕不归在演武场边抱剑，剑未出鞘，鞘响像替「第三」二字敲了一下。他笑浅，笑里无妒，只有快。

莫长春听见尾音，不辩，杖中竹纹游动，气音轻，像一字：等。他改口，声轻却清：「不等了。三月，够送，够还。」
