# 第一百四十六章 层十七

炼气十七，不是雷，是一线实。

管事验息，册上添一行：十七层，登记。

韩泥应声，声平。层在，手在，就不飘。

药峰活照旧，驴驮药渣，疤鬃稳——稳了，146才像实，不像喜。

编筐半刻，藤刺扎掌，刺浅，浅些沿才净。

赖福查席，火折照角，照见厚席不见坛。坛不露，沿才净。

铁无言声低：「层高了，别骄。」「不骄。」韩泥说。

丑时，练鼻下静。香一字不落。

他望坛，低声道：「十七实了。记着。」

验息堂前，队如常，常里藏一针——针是登记，针是炼气十七，针是146该实的那一层。

执事验牌，牌歪，字丑：「韩泥，炼气十六，夺魁。今日验十七。」

韩泥上前，开罐，捧息。息稳，稳像141备材那筛——筛齐，层才实；实了，147坊市才续得住。

册上添一行，行丑，字真：炼气十七。十七，不是喜，是实；实了，大比路又踩实半寸。

同批少年窃语：「丑骨也十七？」语在舌，韩泥不接。接，费舌；舌要留来守坛，留来驮药。

铁无言立阶下，声低：「层高了，别飘。」「不飘。」韩泥应声。应声短，短才像常活。

药峰活照旧，疤鬃驴驮包，包沉，沉不要紧，沉才像146——像实层，不像虚涨。

刘婆端粥，仍多半勺：「层高了，饭也要稳。」韩泥点头。

赖福查税，税半文，文旧。旧不要紧，十七层也交半文，才像末席，不像显摆。

他低声对坛：「十七实了。记着。」坛腹温一线，像应「实」——实了，势才够，够才配150收束。

更鼓沉，他按木牌，牌凉贴胸。胸贴紧，像贴债序——序里写十七，写驴，写泥纹名。

验息后，他理怀内散样，样少，少不要紧——少，才像实层，不像囤粒显摆。

药峰活照旧，疤鬃驮包，包沉，沉像末席该扛的分量。沿净，坛才藏得住。

赖福查夜，火折照角，照见厚席不见坛。

同批少年问层，问在舌，韩泥只答：「实。」答短，短才像常活。

他望坛，低声道：「记着。」坛腹温，贴掌。更鼓沉，按木牌，牌凉贴胸。胸贴紧，像贴146这一层实。炼气十七，不是雷，是一线从十六里长。长像141备材，像142送粒，像145驴稳那几步累加；累加不够，十七就虚；虚，165火不稳，175丹不正。

正，靠验息堂前开罐捧息那半刻——半刻稳，册上添一行，行丑，字真。真，才像146该写的实，不像层赏的喜。喜了，手飘；飘了，147名号就虚，149周家风就接不住。接不住不要紧，146只实；实层，药峰活照旧，疤鬃驮包，包沉，沉像末席该扛的分量。

分量在肩，不在嘴；嘴若辩「丑骨也十七」，辩，费舌；舌要留来守坛，留来练鼻下静。静到只闻辛淡，不闻辱；辱在耳，不在鼻；鼻准，146才配十七。十七像弓满——满不要紧，箭在165青木药火，不在今日喜。喜在远，在大比，在260筑基真名；146只积势，不越界。界在150收束，收束在远，146只钉一层。

钉完，铁无言说别飘，刘婆说饭要稳，赖福查税半文照旧——照旧，才像末席，不像显摆。显摆，赖扒皮来；不来，146才静。静，更鼓沉，按木牌，坛腹温一线，像应「实」——实了，147坊市才续，148叶线才稳，149只听不架。架在151，146不架；不架，是忍，是活，是丑翁活法。

层十七实，实像弓满——满不要紧，箭在165，不在今日喜。喜了，手飘；飘了，火不稳。

验息册上，十七那一行，字丑，真在。真了，147才续，165才钩。更鼓沉，他望坛：「十七实了。记着。」层实，帖才压得住；压得住，大比路才又实半寸。实，不在喜，在手。

赖福廊下过，韩泥只答：「在。」在，就不滚。

柱上条角白一线，三年约在怀。怀凉，掌温。

坡下石阶湿，他数息，数帖，数债序。

同棚杂役笑，笑在耳，不在心。

废渣巷日斜，他验渣，渣正，手才静。

侧门驿亭风硬，他听风，风过，记在心里。

散队后，他过外堂阶下，分拣半刻，指稳——指稳，路就不飘。

粟壳香淡，香一字不落。

管事记簿，记细，是小胜。胜在忍，不在辩。

他望坛，低声道：「手在，路在。」坛不应字，只温。

更鼓沉，天边白一线。白里，他按木牌，牌凉贴胸。

坊市尾巷风硬，他收摊如石，石在，就不滚。

验息后，他理怀内册页，页黄，字丑，丑而真。

铁无言声低：「别飘。」韩泥应声：「不飘。」

刘婆端粥，仍多半勺。韩泥点头，谢轻，恩重。

药峰分拣照旧，指稳，鼻下静一息。

帖在怀，怀沉，沉的是序，不是惧。

白织月袖风过，韩泥不辩。辩，费舌；舌要留来守坛。
