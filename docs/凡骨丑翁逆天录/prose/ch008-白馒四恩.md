# 第八章 白馒四恩

棉袄刚暖上身，手心又来了烫。

韩泥在村口编完一只筐，抬眼，张麻婆扫帚一扫，骂声先到：「喂！晦气翁！手伸着干啥？等风吃吗！等风吃，风也不肯喂你！」

他一愣。张麻婆已经把一个白馒头塞进他手心，面皮还热，烫得指腹一缩。馒皮白得刺眼——村里少见白面。

「拿着！」她骂得更凶，「看什么看？施舍！听见没有？施舍！别给脸不要脸！」

村里人爱听骂，不爱听好话。她把「好」裹进骂里，骂得像刀，刀刃上却沾着面香。童谣丫在旁边捂嘴，眼睛亮得像偷到蜜。

「张婶……」韩泥嗓子哑。

「叫麻婆！」她扫帚往地一顿，「婶字留给活人听。你——吃！吃完闭嘴编筐。再饿晕倒门口，我还得给你收尸！收尸我不收钱，收晦气！」

童谣丫忍不住笑出声。张麻婆扫帚一扫：「笑什么笑！少学舌头！学舌头迟早学成马春花！」

韩泥把馒头掰开，热气扑脸。白面少见，村里多是杂粮。他咬一口，面香冲到眼眶。眼眶热了，他不敢让热掉下来——掉下来，像讨。嚼得慢，慢得像怕吞掉这份骂里的热。

旁人起哄：「麻婆肯给绝户白馒？图什么？」

张麻婆眼一翻：「图你们闭嘴！他编筐手不能抖，手抖了我屋上雨漏谁补？你们补？补个屁！」

有人还想刺，赵绣春脚步过来，嘴开一线：「张麻婆，你也不怕沾晦气？」

「怕。」张麻婆痛快承认，「怕你们嘴比晦气脏。馒头我塞了，你们吞回去。吞不下去，拿回家泡茶。」

赵绣春噎住。周嫂在远处看了一眼，没帮腔——井台那日让道的半步，还搁在她眼里。她把棒槌换了下手，像换掉一句闲话。

韩泥把馒头留半边，往袖里塞。张麻婆看得清楚：「藏啥？吃完！藏起来发霉，霉了怪我？」

「留着。」韩泥说，「有人比我更饿。」

「沈家那芽儿？」张麻婆哼，「你倒会分。分完自己饿，饿了我再骂你一顿。」

「分得动，就分。」韩泥点头，「骂归骂，馒归馒。」

他路过沈家墙根，半馒从墙缝推进去。里头立刻有吸鼻子的轻响。芽儿压着声：「韩爷爷……又……又给人骂了吧？」

「别嚷。」韩泥说，「嚷了，又给人撕。馒吃了，声留肚里。」

墙内一静。随后传来娘亲咳嗽似的呜咽。呜咽里夹半句「谢」。他没停。停像招祸。招祸就连张麻婆也拖脏。

集上，筐又卖了——十二文外头，买方多拍了两文：「今儿这筐更紧。你手气好？」

「手热。」韩泥拍拍棉袄领，又拍拍空了的袖，「热了就紧。紧了就不漏。」

买方不解，但仍把钱付清。钱落袋，袋底响一声，像日子也响一声。旁边卖葱的汉子瞅他：「你嘴边有面渣。」

韩泥擦了擦：「白的。少见。」

回村过药铺侧巷，叶青禾正在晾姜片。姜辣味淡淡飘出来，跟面香不对味，却都能暖人。她看他一眼，没多话，只道：「吃了？」

「吃了。」韩泥应。

「那就好。」叶青禾把秤砣轻轻一落，「人有热，路才直。热冷了，路就歪。」

就这一句。他记下。热可以是汤，可以是粥，可以是袄，也可以是馒——热不同，热过胃的都算。算在心里，不贴门口做榜。

巷口忽然有短影一闪，铁无言路过，檐下站了半息，嗓子极低：「别飘。」

韩泥独眼一抬。铁无言已经走了，像从未来过。那三字却钉在耳里——白馒香，棉袄暖，人容易觉自己站高了。站高了，下一脚就崴。崴了，恩人的热全摔碎。

「记下。」韩泥对空影说，「不飘。」

傍晚，马春花又敲盆：「谁给绝户白馒？谁想给全村招邪？白面都喂晦气，你们还吃不吃饭！」

张麻婆扫帚一扬：「我给的！你有本事罚我！罚不了，就把嘴借我扫帚擦擦！擦干净再骂，省得脏风。」

人群哄笑。马春花脸涨红，还要骂，里正远远咳嗽：「少闹。闹到夜里，记名。记名的人不挑姓。」

笑散一半。韩泥不掺进笑里。笑里容易飘，铁无言那句还烫着。他掂掂袖，袖空了，心不空。

夜里他坐漏屋，把今日摸了一遍：叶汤暖过，芽儿粥稀过，陈姑袄压过，麻婆馒烫过。摸着摸着，不叫账单——账单冷，像税吏的脸。这是四道热，热在不同的手，热过同一个肚子。

四道热摆开，不是清单，是场面——井台让道、墙缝塞饼、骂里塞馒、秤砣轻落，一样样在眼前过。过完，心里就实。

他低声：「四样，都在。在就不灭。」

坛沿微温，像应了一下。他没喊瓮醒，只把馒头屑捏碎，埋进泥地角落，挨着前日那粒米：「恩人给的，不当风吹。风吹走面香，吹不走记性。」

门外有犬吠一声，随即寂。寂里他更听得清自己的呼吸。呼吸稳，就不飘。

第二日清晨，童谣丫偷偷跟到村口，小声问：「韩爷爷，麻婆骂你，你真不气？」

「骂里有馒。」韩泥说，「气就傻。傻了手抖，筐编不圆。」

童谣丫眨眨眼，被张麻婆帚子隔空一点：「少问！回去洗衣！问多了学坏！」

「我没学坏！」童谣丫跑了，跑两步又回头吐舌头。

韩泥看着那帚子护着娃似的姿势，心里又实一分。实了，才懂「别飘」——飘的人记仇不记恩，或者记恩记成自己本事。本事是假的，馍是真的。

集尾又遇上赵绣春。她嘴钝了半日，仍不甘：「你以为馒头能洗晦气？」

「洗不了。」韩泥说，「馒头只填肚子。肚子满了，人不跪谣。谣不顶饥。」

「你嘴倒利。」

「馒利的。」韩泥道，「我跟着利半寸。」

赵绣春想尖，尖不出来。周嫂在旁边拽她袖：「走吧。里正眼尖。眼尖就不分谁先闹。」

走了。韩泥掂掂袋里十四文，不多，却够过两日。够，就不去想罚银吓人。吓人的话像空雷，雷响过，田还是那块田。

路过陈姑门前，陈姑正在晒袜。她看他一眼：「吃白的了？」

「嗯。」

「那就成。」陈姑说，「袄别敞。馒别夸。夸了，明日就没。」

「不夸。」韩泥应。

夜里风大，棉袄扣严。他想四道热叠在身上，叠成一条还能走的路。路不宽，够一个人把背挺直。挺直不是撑场面，是别跪着把恩踩进泥里。

「四恩在，」他对黑屋说，「还路在。」

说完，听门外有人轻轻跺脚——像张麻婆路过，又像陈姑查夜。他不开门。开门像炫耀。炫耀就飘。飘了，铁无言那句就白钉。

夜里他又摸了摸泥地角落里那点屑。屑埋住了，香还在鼻底回。香回一下，铁无言那句「别飘」也回一下。两下叠着，人才压得住。

天亮前，他把筐绳勒紧，准备明日再出村。出村的人，只要背上还压着旧棉袄，手里还记得白馒烫，就不会把自己走丢。

村口风一刮，面香似有若无。有人远远说：「绝户也能吃白的？」他不接。接了就像炫耀，炫耀就飘。

他把筐绳再勒一勒，勒紧像勒住心口那点浮气。浮气压住了，四道热才贴得实。贴实了，明天出村也不飘。不飘，恩才像恩。像了，才配说还。还在后头。先把路留住。

丢不起。还字还在后头，路得先留着。路留着，四道热才不至于凉成一句闲谈。闲谈里的恩，最容易被人当笑话笑没。笑没了，还路就断。
