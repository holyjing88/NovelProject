# 第十章 绝户记恩

马春花这回敲的不是铜盆，是棺材板。

板薄，敲得咚咚响，像给活人试寿材：「绝户翁！听清楚！你无儿无女，死了谁抬？村里给你预备一口——省得脏土！脏土也嫌你！」

人群围着，像看戏。有人笑，有人咂嘴，有人把娃的眼睛捂住，捂完自己还偷看。赵绣春站台阶上补刀：「绝户就该绝！你还收汤收粥收袄收馒，收够了？收够了给全村下葬！」

「四样，」马春花掰着手指数，敲板敲得更响，「叶家汤、沈家粥、陈姑袄、麻婆馒——件件沾你，件件晦气！绝户还收恩？绝户就该空手埋！空手埋，干净！」

韩泥站在圈外，听完，嗓子哑却清楚：「欠着。」

场子一顿。板声也顿住半息。

「欠着？」赵绣春尖笑，「你拿什么还？拿墓穴？拿这块板？」

「拿命还。」韩泥独眼平，看着棺材板，也看着人，「欠四样，还四样。还不起银，还得起命。命在一天，欠字就在一天。」

静了一息。笑声卡在喉里。有人低头。低头的人里，也包括昨儿跟着嚷连坐的。有人嘴硬还想接，被旁人拽了拽袖。

马春花愣半息，又尖起来：「嘴硬！绝户嘴硬也是绝户！命值几个钱！」

「绝户不绝恩。」韩泥说，「恩在，户就不空。空的是你家嘴。」

这句话像石头丢进污水里——水仍脏，石已沉底。陈姑在人群后咬牙，把怀里干饼往前一递，递到韩泥手上：「拿着。别空腹挨骂。空腹挨骂，骂更尖。」

「陈姑……」

「少话。」陈姑瞪他，「绝户听着难受，活着的更难受。吃！吃完再还你的命。」

他接过干饼，饼糙，糙得硌牙，却香。香里有灶火，火里有盐。张麻婆在更远的地方扫帚杵地，没过来，只骂童谣丫：「看什么看！回家！看戏看出毛病！」骂完，嗓子却软了一线，软得像心里也堵。

叶青禾没进圈。她在巷口站住，声不响，只对韩泥唇形一样：「往心里记。」

韩泥「嗯」了半息。记字比还字先落地。先落地，才不乱。

围散后，有人还戳着棺材板笑。笑够了，板还竖着，竖着像等着把他吓跪。韩泥回屋，干饼搁坛沿，手按着沿，眼前忽然一闪——爹临终那夜，也按着这坛。灯油将尽，火苗跳，爹手枯，枯得像柴，柴还抖：「泥啊……户绝人不绝。人绝，恩别绝。绝了，爸白养你。白养你，我在地下也堵得慌。」

「爹。」他低声，「恩没绝。汤也在，粥也在，袄也在，馒也在。」

坛腹微温。温得像应，又像催。催他把「还」字立住。立住不是喊给人听，是先按进自己骨头。骨头硬了，板再响也钻不进髓。

他掰了半角干饼细嚼，嚼出盐腥。盐腥里忽然想起集上人看热闹的眼——眼多，心少。心少不怕，他心多一笔就够。

午后他上集卖筐。买方摸藤条，又摸筐沿：「今儿更紧。十五文，行不行？紧得像你生了气。」

「行。」韩泥点头，「气在，手更稳。」

十五文落袋，袋比前几日沉。沉一点，命就多一点底。底多了，「拿命还」才不空喊。空喊吓不住人，沉袋能撑日子。

回村过井台，周嫂想说「晦气」，看见他棉袄领口扣得死紧，干饼渣还在唇边，话又咽了。赵绣春瞪她：「你哑巴了？」

「水浑。」周嫂装忙，「先打水。」

夜里马春花还把棺材板竖在村西路口，像立碑：「让绝户天天看见！看见才老实！老实了少收恩！」

韩泥从路口过，停半步，对着薄板道：「板我看见了。恩我也看见了。板吓不住还。还字比板硬。」

沈枯芽躲在暗处，小声：「韩爷爷，他们……好凶。板……吓人。」

「凶是嘴。」韩泥说，「还是手。手还没动，先记。记清了，手才不乱。」

「记什么？」

「记谁递饼，记谁塞馒，记谁送粥，记谁给汤。」韩泥顿了顿，「也记谁敲棺。敲棺的人，响一声，名下一笔。」

芽儿使劲点头。点头像发狠。发狠也好，狠比怕有用。

第二日，里正到村西，看见棺材板，眉皱：「春花，寿材不是戏。再立路口，报官。官来了，你板子也蒙尘。」

马春花不服：「绝户吓人！吓人就该看棺！」

「吓人的是谣。」里正冷声，「冬前村会，别把板子带进祠堂。带进了，吵翻天，也定不了理。」

「村会？」有人接嘴，「真开？」

「嗯。」里正说，「日子近了。有话会上说。会前少拿死人吓活人。活人吓慌了，集也散。集散了，税也没处收——赖管事爱听不听。」

影到了——村会的影。不亮，却压人。压得赵嘴也软半寸。软半寸也好，半寸能让人喘气。喘气时，韩泥听见有人低声念「绝户」，念完又把眼躲开——躲开，就是心里虚了。

他没追着虚看。追了像逞。逞不是还。还要沉。

韩泥听在耳里，回屋把十五文钱分成小堆：一堆过冬，一堆凑罚，一堆先空着——空着留给「还」。还不一定是银，也可以是命里那一口硬气。硬气不是顶撞，是欠着不忘。

他把空着的那一堆按了按，按出一个浅坑：「坑留给恩。恩填进来，才叫还。」

陈姑又来，看了眼干饼渣：「吃了？」

「吃了。」

「那就成。」陈姑道，「绝户两字让他们喊。你别自己喊进心里。喊进心里，真绝。真绝了，袄也白塞。」

「没喊进。」韩泥说，「我只喊欠着。欠着，就活着。活着，就还能还。」

陈姑眼眶红一下，又骂：「死倔。」转身走了，走时袖口一动，像还想塞点什么，终究没塞——塞多了，又给人口实。口实多了，村会未开，谣先开。

叶青禾夜里送一包姜末到门缝，不多话，只一句：「往心里记。记清了，还才不乱。乱了，热也凉。」

「记下。」韩泥应，「汤我没忘。饼也没忘。四样我数得清。」

「嗯。」叶青禾在门外站半息，背影仍直，「绝户两字，别替他们咬自己。咬自己，就中了他们的板。」

门外风过，姜辣一丝钻进鼻。他合眼，把手按在坛沿上。爹的话、四恩的热、棺材板的响，全按进同一处。按实了，人才不散。不散，村会来了才站得住。

路口那块棺材板夜里还竖着，风一过，薄板发颤，颤出细响。细响像催命，催不走他眼里的四道热。热在，板就只是板。板竖着吓唬人，吓唬不住记恩的人。记恩的人，见板如见账。账在，就不慌。慌的是忘。忘了，板才赢。赢不得这绝户。

路口板影斜斜投进窗纸，像一道灰杠。他盯了一息，低声：「杠吓不住命。命用来还。」

丑时，他起身对空屋说：「还字立了。村会的影，也到了。」

影到了，就不怕板子再敲。板敲的是绝户名，敲不碎心里的恩。恩在，来日还得起；还得起，绝户也不绝。绝的是谣，不是他心里那本账。

他把干饼渣扫进掌心，埋进泥角，挨着米和馒屑：「四样热，一处埋。埋住了，就不怕人当街数落。」

沈家墙根忽然极轻地传来一声：「韩爷爷……你别怕棺。」

「不怕。」韩泥隔墙应，「怕的是忘恩。恩不忘，棺也抬不走我。抬不走，我就还能走到村里开会那日。」

明日锣响之前，他先把筐编圆。圆了，人才站得住——站住了，等村会议事，他把「欠着」二字当众按实。按实了，棺材板再响，也只是响，不是判。判不了恩，就判不了他这口还字。
