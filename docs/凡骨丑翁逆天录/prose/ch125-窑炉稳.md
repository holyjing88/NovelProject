# 第一百二十五章 窑炉稳

备战最后七日，韩泥只守窑温。

窑门柴挡，挡严，严得像62席盖。席严坛不露，窑严粒才正；正了，试台才配夺魁，夺魁才配还刘婆。

焚火残诀走息，息稳，铁片嵌口温一线。一线升，升得像110首炼余温未散——散不要紧，稳才要紧。稳了，第九粒成，成最后一粒备战粒；粒丑，泥纹浅，嗅之正。

赖福已无新招，只廊外晃木牌：「七日后试台，看你滚。」韩泥不辩。辩，费舌；舌要留来数息，留来守温。

铁无言送炭渣，渣里一点灵火屑，屑细，够稳一夜窑温：「七日，别出门。」「不出。」韩泥说，「出门，手飘。」

刘婆柴门外咳：「窑里闷，别闷坏。」「闷不坏。」韩泥说，「闷，炉才稳。」

白织月过药山道，袖风远：「七日后，看丑骨出丑。」韩泥不辩。出丑在脸，不出在手；手稳，丑也能「正」。

第一夜，窑温如秤，秤平，粒不杂。第二夜，邻床少年悄问：「七日不出门，饿不饿？」「不饿。」韩泥说，「饿，手飘。」

第三夜，窑温忽跳，跳像赖福余党使坏——使坏不要紧，铁片稳口，残诀压温，压回一线。

第四夜，刘婆送半块饼，饼糙，糙得像泥岗麦麸：「试台前，别空肚。」韩泥收下，谢轻，恩重。

第五夜，气走圆满，十二层实到极致，极致像弓拉满——拉满不要紧，箭在130射。

第六夜，他再验九粒，粒粒正，正得像五年淘的。淘在渣，淘在克扣，淘在忍。

第七夜，他封窑，封得像封坛。封坛，坛温一线不退；不退，心稳，步稳，债序稳。

丑时末，练鼻下静。粟壳香混药香，药香正，正不要紧；要紧是静一字不落。

开窑晨，天色白，白得像刘婆补衣线。线旧，线暖，线记着，等130还。

铁无言来窑外，声低：「七日毕，手别飘。」「不飘。」韩泥应声。

刘婆柴门外咳：「窑稳了？」「稳了。」「稳了，试台别抖。」韩泥只答：「不抖。」

他望坛，独眼平：「稳。记着。」七日封窑毕，九粒仍正。正，试台在手。

第一日辰初，药山雾未散，雾里窑烟细如线。线直，直像投序。韩泥添柴半把，柴潮，潮不要紧，残诀走息，潮柴也燃。燃稳，温才上秤。

管事弟子过窑外，记一笔：「备战七日，别下山。」韩泥只答：「不下。」不下，不是怂，是手要稳。稳了，第九粒才配成；成了，128才配验，129才配帖。

第二日午后，编筐棚借半刻。藤刺扎掌，他偏手，刺浅些。筐勒匀，匀像火路——勒不匀，渣散，路断。编筐钱买米，米在，试台前恩才还得上。

同批少年掩鼻：「你七日不出窑，臭不臭？」「臭。」韩泥声平，「臭，药山活。活有味，味在角，不在席。」少年退半步，没再问——问，像沾晦气。

第三日黄昏，赖福余党掷石过窑顶，石落柴挡，挡响一声。响不要紧，温未跳。跳才要紧——跳了，粒杂。韩泥铁片压口，残诀一息，温回秤平。

老耿挑水路过，水洒稳，声哑：「备战别拜暖。拜暖，路断。」韩泥只答：「不拜。坛在，人在。」人在，就不滚；不滚，才守得住七日。

第四日夜里，邻床少年发烧，烧像扰息。扰不要紧，韩泥仍守窑，守到丑时末。丑时末练鼻下静，粟壳香混药香，香一字不落。落齐，温才匀。

第五日晨，铁无言送炭渣第二回，渣里灵火屑细，屑亮，亮像108并肩余温。余温在，兄弟线就不断。韩泥收渣，谢轻，声平：「够一夜。」

沈枯芽口信无字，只有半片枯叶——叶黄，叶上无字，字在记里。记叶丫头怕黑，记门缝留灯，记130还刘婆在前头。前头远，远不要紧，手在，路就不绝。

第六日午后，白织月远廊过，袖风冷，没停。没停，小胜——胜在手稳，不在嘴辩。

秦霜木鱼轻敲：「备战别邪。邪了，鼻下香飘。」韩泥不答，答在心里——心里只稳。稳，是丑翁本事；本事在忍七年赖福，在还沈枯芽半块灵石，在守七日窑温。

第七日封窑前，他再验九粒，粒粒正，正像五年淘的。忍不是怂，是活；活了，才配夺魁，才配还刘婆。

散队后，他数息，数粒，数债序。债序在，步不飘；步不飘，才接得住128验正，129帖至，130魁与还刘婆。更鼓沉，沉里，他望坛，独眼平：「稳。记着。」

药山半腰，风硬，他把剩渣摊匀，匀像秤平——秤平，窑温才平。摊匀了，手才静；静了，第九粒才肯落怀。

管事夜巡过窑外，灯照柴挡，照见厚挡不见坛。坛不露不要紧，沿要净。沿净，席严，坛才藏得住。

同棚杂役赌明日试台：「丑骨夺魁，太阳西边出。」韩泥不接茬，接在掌茧里——茧裂处疼，疼不露。不露，才像常活；常活里藏备战，备战里藏还路。

废渣巷尾，他倒渣半筐，渣灰，灰里正。正不要紧，正像备战回响——回响在鼻下，不在舌。舌要留来守窑，留来封坛。

侧门驿亭风紧，紧像试台前那口气。那口气要匀，匀了，手才不飘；飘了，九粒就杂。杂了，128验不出正。

坛边席角，他按木牌，牌凉贴胸。胸贴紧，像贴仇约——约说三年，说炼气十三，说站钱戾衡台前。台前远，远不要紧，窑温在近。

编筐棚借半刻，末藤硬，硬像冬里规矩。规矩在，手才不飘。飘了，正测白来；白来不要紧，坛在，根在。

兽栏侧清栏，栏臭，臭不要紧。臭，药山活；活有味，味在角，不在席。席严坛不露，坛不露，备战才匿得住。

更鼓沉，韩泥按木牌，牌凉贴胸。七日封窑毕，九粒仍正。正，试台在手；手在，路在，就不滚。

窑门柴挡再压半指，压严，严像62席盖。盖严了，外风不进；不进，温才匀到试台前那一息。

散队后，他过外堂阶下，风硬，分拣半刻，指稳，鼻下静一息——静，是把坛在摁进路里。

更鼓沉，沉里，他过廊下暗处，清栏，独眼平，手不抖——抖了，记恩就白挨。

午后坛边席角，风硬，他编筐，指稳，稳里带一点守角的回响。

邻床悄问一句，韩泥只答：「沿净。」答短，短才像常活。

柱上条角白一线，三年约在怀。怀凉，掌温——温在不叫，不在众目。

赖福脚步在廊外停半息，韩泥只答：「在。沿净。」在，就不滚，才守得住坛在。

秦霜远廊过，木鱼轻敲：「心别邪。」韩泥不答，答在心里——心里守角，分列，不混。

铁无言声低：「手在就行。」韩泥应声：「在。」在，就是证——证沿净，不在嘴。

刘婆端粥，仍多半勺：「手稳，饭稳。饭稳，恩不断。」韩泥点头，谢轻，恩重。

沈枯芽怯，悄问：「韩叔，怕不怕？」「怕。」韩泥说，「怕也守。守完，才有饭。」

白织月远廊过，袖风冷，没停。没停，小胜——胜在记恩，不在嘴。

他低声对坛：「坛在，人在。人在，就不滚。」坛不应字，只温。

刃不亮，先记。记完了，才配沿净，才配下一程。

编筐半刻，藤刺扎掌，刺浅，浅些沿才净。筐卖钱，米在，恩才还得上。

入夜药渣场角，风沉，他备料半刻，指节白一息——白完，坛在才落稳。

辰初兽栏侧，雾未散，他数息，独眼平，掌茧裂处疼——疼不露，记恩在忍里。

午后编筐棚，日斜，他验渣半刻，鼻下静一息——静，是把守角摁进路里。

管事夜巡过坡下石阶，灯照席角，照见厚席不见坛——坛不露，沿净才藏得住。

帖在怀，怀沉，沉的是序，不是惧。

白织月袖风过，韩泥不辩。辩，费舌；舌要留来守坛。
