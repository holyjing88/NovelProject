# 第三十九章 克扣月半

半月下来，麸皮短了三成。短在秤上，秤在赖福手里。他举秤盘晃：「晦气杂役吃多了，兽栏粪都臭！」

丙九有人嘀咕：「再克，咱们也滚。」

韩泥仍清粪，铲下稳，稳才少挨骂。心里臭着，记着，不辩。辩，赖福正等他辩——辩了就扣饭，扣了祸及全丙九。管事路过，瞥秤：「麸皮按册发，少一粒记名。」

赖福脸一僵，秤手松了半寸。松半寸，也是小胜。有人低声：「韩泥不闹，咱们跟着沾光。」

他听见了，不答。功在饭，不在舌。舌赢一时，秤输一月。月初麸皮还足，月中秤悄悄浅了，月末短三成。短成仇苗，苗不拔，先活。赖福每日晃秤，晃得丙九眼酸：「晦气沾谁谁穷！」

有人顶嘴：「穷是你克出来的！」

赖福举鞭：「再顶，明日没麸皮！」

韩泥低头清粪，不接茬。接茬，全丙九跟着饿。管事冷声：「吵完了？吵完韩泥仍清粪，你仍发麸皮。少一粒，你记名。」

赖福收鞭，脸青。兽栏深处小兽咳，咳声细，扎耳。他多清半刻，把湿粪换干草。稳了，管事少骂一句，饭就多一口余地。管事路过，瞥他铲下：「韩泥，兽栏这角，明日仍你。」

「清。」

「清得细，记着。记着，麸皮不能短你的。」

赖福远处听见，脸一抽，抽完装没事。刘婆端粥来，粥更稀，勺几乎立不住。她眼里一瞪赖福，赖福竟退半步。

「再看粥，我去管事那告。」刘婆嗓沙，盆沿捏得响，「告不成，我把盆扣你头上。」

赖福骂「疯婆」，嘴硬脚却退。韩泥端碗，碗温——稀薄也是热。刘婆低声：「稀也喝，别硬扛。」

「扛惯。」

「扛惯，也会倒。」刘婆说，「倒了就滚。滚了，谁还叶丫头那碗汤？」

韩泥独眼抬了一下，又垂：「记着。」

「粥稀，心不能稀。」她又补一句，「心稀了，手就抖。手抖漏渣，漏渣就滚。」

「不滚。滚了，谁记赖福的秤？」

刘婆愣半息，哼：「记着记着，丑脸倒会说话。」

午后他编筐补单，指茧裂，裂里渗血。血擦布上，不往坛沿蹭——坛不该因奋而醒，醒要等该醒的日子。邻床少年问：「你手裂了还编？」

「编了，有饭。饭在，命在。命在，恩才还在。」

「命都没几口粥了，还恩？」

「少三口，记着。记着，以后还。」

少年叹气：「咱们都被克，你就不能闹一回？」

「闹一回，爽一口。」韩泥说，「爽完，滚半年。滚半年，恩断。」

少年不吭了。管事查单，瞥他掌：「裂了换活。」

「不换。单没完，饭没稳。」

管事哼一声，走前却道：「明日麸皮按册，我亲自盯秤。」

赖福远处啐：「装硬。」

韩泥不回头。辱记着，不在今日还。还，要等手稳。同批有人低声：「韩泥，咱们要不要一起去管事那？」

「去，赖福记你们名。不去，他记我名。名都记，饭先稳。」

「你就忍？」

「忍半月，换明日盯秤。盯秤，胜。」

众人面面相觑，仍各自干活。夜，赖福踹门：「韩泥，明日兽栏加倍。做不完，滚。」

「加倍，也做。」

门内，仇苗又深半寸。深半寸，不拔，先活。坛腹微温，应「忍」。他低声道：「克扣记着。刘婆粥记着。嗅渣香记着。必还记着。」

半月少三口饭，三口记着——先还自己的命。命还清了，才还得起别人的恩。刘婆门外咳：「还喝不喝？」

碗递进来，稀，热。热粥稀，也胜冷秤。他喝尽。赖福门外骂：「疯婆子多事！」

刘婆回骂：「多事你也滚远点！秤在你手，良心在谁肚？」

韩泥摸坛，坛温应粥温。温一线，不烫人，烫仇。邻床少年梦呓：「饭……」

他睁眼，替少年盖破袄：「排着。管事盯秤了。」

少年哼唧睡了。管事巡夜，敲棚柱，声沉：「赖福，麸皮按册。再克，记名。」

赖福应了一声，声虚。虚，也是小胜——胜在明日还能喝一口稀粥。同批有人低声：「克扣半月，你胃还暖？」

「暖在记里。记着，才还得上。」

「还得上什么？」

「还得上叶丫头的汤，还得上刘婆的粥，还得上泥岗那几笔恩，还得上宗门这一笔。」

那人闭嘴翻身。韩泥合眼，三口饭少在心里刻着。外头风息了，稀粥的热还在肚里。热在，人活；人活，明日盯秤才有戏。

管事靴声再沉，沉得像钉：「冬测备检，两月后。备检前，手别抖。」

两月——比瓮醒远，比克扣近。近不要紧，要紧是钉在耳里。钉住了，赖福的秤才虚一日；虚一日，丙九就多喘一口气。一口气，够熬到备检帖上柱那日。

邻床少年又梦呓「饭」，他替盖袄，低声：「排着。秤实了，粥就实。」粥实不实，看明日管事盯不盯——盯了，半月三口才有回。少年迷糊问：「你真能还那么多？」

「还不了，也记。」韩泥说，「记全了，才活得久。活得久，才还得上。」

门外又敲柱：「明日丙九转渣场。韩泥，酉时到。误者，记过。」

渣场——鼻下那条路，明日要走实。坛壁一线水汽，应粥温，不应骂。骂记着，不泄；泄了，害全丙九。丑时他展掌看茧裂，裂里新皮嫩，疼真。疼真，记着，等对的日子扎得准。

午后补单时藤条又勒指，茧再裂一线。他不喊，把血擦破布上。筐底密，壁直，管事路过数一只：「丙九里，你筐最不塌。」最不塌，赖福想扣麸皮，也找不着由头——由头在秤，秤虚在嘴，实在手。

手实，半月三口才有回；有回，刘婆粥才不白护。同批少年咬着唇跟过来：「盯秤真有用？我们是不是该闹一回？」

「有用。」韩泥把藤条绕紧，「秤旁多一只眼，赖福就少伸一寸手。少一寸，丙九就多半口气。先干活。闹留给该闹的日子——日子没到，闹只送饭。」

酉时渣场，他把三堆再理一遍。灰热扑面，鼻下静一息，静出苦甘——甘短，苦长，分开才不乱。乱了香就飘；飘了，两月后帖上柱，手就抖。抖不得。

铁无言远远立着，没过来，只扫堆一眼。一眼够：堆齐，他便转身走。走得干脆，像说「行了」。韩泥心里却不飘——行了只是今日，明日秤还在赖福手里。手里有秤，仇苗就还活着。

渣场风硬，风里带药苦，苦里一线甘，短得像刘婆粥里那半勺热。热记着，忍半月才不算白少。回房时路过灶间，刘婆把空碗一顿：「喝了？还敢空肚熬夜？」

「喝了。」

「喝了就睡。冬测备检不是靠熬夜熬出来的，是靠手不抖熬出来的。」

「知道。」韩泥顿了顿，「两月后，帖上柱。上柱前，我手不能飘。」

「飘了，」刘婆哼，「老婆子补的肘也白补。」

他应一声回房。夜里柱影斜着，像把备检日子远远吊在上头。吊着，他就按实掌茧：「手在，秤在。秤落了盘，粥才能落肚。」坛温应了一线，应忍，不应骂。骂留给簿子，忍留给命——命在，两月后那张帖才有人站得住。

邻床少年睡熟了，嘴里还咕哝「麸皮」。韩泥替他把破袄角压实，低声道：「排着。管事明日盯秤，盯一寸，你们肚里就多一寸。」

他自己却睁着眼，把半月短下的三口饭一笔笔记完。记完，仇苗更深，更深也不拔——拔早了流血，流血误冬测。冬测帖未上柱前，他先把鼻子里的甘、肘上的暖、秤上的亏，全都按进心里，等那两月走到头。走到头，帖落下，他才认那是锤。锤未落前，先把肚里那口稀粥喝热了。
