# 第五十三章 杂役阵

子时将至，东脉侧径比主道更黑，黑得像一口吞人的井，井口小，只容得下报信的人。

顾小满率杂役分班潜伏，路线是秦商言简上标的三条，交错，像网眼。第一班出发，第二班接隙，第三班备换。她走在最前，隐匿符贴胸，符温，像长老的手按在心上，按得她呼吸都轻了半分。

田埂方向火把骤起——赤焰先锋贴地行，果然佯攻侧径，果然要断丹堂耳。腥风过坎，顾小满低声：「退，不战。消息比命贵。」

一队人缩进土坎，呼吸压到极轻。先锋过，过出一阵腥风，风后，她抬手，三短一长，信号哨，极细，像虫鸣。丹堂方向，有人接应，接应成一线——线细，却活着。

第二路险在断崖，有人跌，顾小满扶，扶完仍跑，跑得像半块灵石在掌里跳，跳成一颗心。有人吓软，她冷眼：「跪着送不出消息。站起来。站起来，才像还恩。」

第三路更黑，黑里赤焰先锋回头搜，搜不到人，搜得到记号——止血散撒路口，像给后来者留字：活着送到。顾小满在第三处拐点停下，取出莫长春备的止血散小包，撒路口，撒得像撒一粒不会熄的灯。

子时钟声远传，一声，像弓弦满。她抬头，东脉灵田方向杀声已起，杀声里丹堂一线地火阵鸣，淡红，像另一颗心也在跳。杂役阵动，动得无声，动得比剑有用——有用在耳还在，耳在，门就不盲。

后阵边缘，她冲至周德海指挥棚，浑身泥血，嗓子哑：「周长老，侧门阵闸，开一次！赤焰绕田埂，再不开闸，丹堂阵眼必破——」

周德海淡道：「杂役弟子，回你的路。」目光一沉，像看一粒尘，不像看一条命。

顾小满咬牙，转身再奔，奔回灵田方向，像要把「侧门不开」四个字奔成一把刀，递给宗主，递给每一个肯听的人。她不知谁肯听，只知不听，田就失；田失，丹堂先危。

阵后弟子不知暗线，只知侧径静得反常，反常得像一根耳还在。有人低声：「侧径怎么没动静？」另一人答：「没动静，才是动静。」答完，仍静，静得扎人后颈。

赵玄机在丹堂门洞听见信号哨，极细，像虫鸣，却懂：侧径耳还在，耳在杂役，不在长老。他刀锋更稳，稳得像终于把「心在谁那边」刻进铁里——刻给门看，也刻给自己看。

苏念慈在阵眼石，地火阵纹淡红，炉中火舌细稳，像替暗线也担。她低声：「消息比命贵，命在门。」霍镇山在前阵骂：「赤焰先锋贴地行？老子骂醒你们！」骂声滚进田里，滚得像替杂役也撑胆。

莫长春仍坐丹堂门外石阶，杖点节拍，不进阵。进阵，阵眼易乱；不乱，也抢苏念慈的戏。他袖空，望侧径方向，望得像望一根细线还拴在门上。线还在，礼就还没送完。

顾小满第二班接隙，第三班备换，换得像把网眼补密。补密了，赤焰搜三次，搜不到人，只搜到止血散味，味淡，淡得像长老留的字：活着送到。

她冲至灵田高台侧，嗓子哑，报柳青鸢：「侧门仍闭，赤焰绕后。宗主，阵闸再不开，腹背受敌！」柳青鸢眸光一冷，令旗微抬，却未直落周德海棚——落令像称忠，称忠要等战后，战前要救门。

顾小满退入暗线，不再求周德海。求一次，够记；求两次，像私。她低声，说给夜风：「消息送到了。接下来，看谁还装瞎。」

第二班有人崴脚，她背起，背得像背半块灵石——石不重，路重。第三班在拐点接应，接应得像把网眼再补一线。补上了，赤焰先锋回头再搜，仍搜不到人，只搜到虫鸣般的哨音，音细，细得像针扎在赤焰耳里。

第一路折返，喘而稳，报：「火把两处，先锋贴地，搜三次未果。」顾小满点头，点头像把「耳还在」四字也点实。她把秦商言简上第三条路线又核一遍，核得像核半块灵石——核清了，才走得直。

她奔回丹堂侧径，报赵玄机：「侧门仍闭，绕后已确。」赵玄机颔首，不多言，言多像私。他刀锋更稳，稳得像终于把「守门」二字刻进铁里。

秦商言在侧径接应，衣沾泥，声低：「消息已入宗主耳。接下来，看阵闸。」顾小满摇头：「阵闸在周长老指间。指间不落，落得像等战后称忠。」秦商言点头，点头像把「称」字也记下。

风腥，杀声起，像东脉夜战开幕，缺不得这一根耳。网眼小，容不下英雄，容得下报信——报信送到了，侧门仍闭，闭的人，尚要战后称一称。

霍镇山在前阵听见虫鸣哨音，骂声一顿，顿得像终于听懂：侧径没动静，才是动静。他声洪：「杂役耳还在！东脉根还在！」

苏念慈抛丹不停，丹丸落处，倒者能爬——爬不是为了名，是为了门还在。她低声训弟子：「听哨音，别慌。慌一声，阵眼就乱。」

顾小满再奔灵田，再报柳青鸢：「侧门仍闭，赤焰绕后，腹背受敌！」柳青鸢眸光更冷，令旗微抬，仍未直落周棚——救门要紧，称忠要等战后。

莫长春仍坐石阶，杖点节拍，不进阵。他袖空，望侧径，望得像望一根细线还拴在门上。线还在，礼就还没送完。识海塔影一荡，一念出塔，杖点石阶，声轻：「阵亮，战要来。东脉夜战，响里见杖剑。」

一念收塔，杖落掌。顾小满没入最后一道侧径，隐匿符贴胸，符温，像长老的手还在心上。她低声，说给夜风：「消息送到了。接下来，看谁还装瞎。」

下一息，杀声要炸。炸之前，杂役阵已亮，亮在无声，亮在记号，亮在还活着回来的足音里——足音细，细得像半块灵石在掌里跳，跳成一颗心，跳成青岚门的耳。

林挽月罚役途中听见杀声起，唇白，步稳，稳得像把「证路」二字也带进林苑。她不敢回头，回头像慕；不回头，才像改。

莫长春望丹堂门洞，望灵田高台，望侧径黑线，三线像三根弦，弦满，才响。他袖空，低声：「耳还在，门就不盲。盲了，夜袭就不是试炼，是乱局。」

秦商言在侧径接应最后一班，声低：「三条路都活着。消息已入宗主耳，入赵执事耳，入苏首座耳。」顾小满点头，点头像把「活着送到」四字也点实。

周德海在后阵棚令旗仍不落，不落得像刀背，不向前，只向后。副将不敢再问，问多了像私。霍镇山在前阵骂，骂声滚进田里，滚得像替杂役也撑胆。

苏念慈在阵眼石，炉中火舌细稳，地火阵纹淡红，像替暗线也担。赵玄机在门洞听见最后一记虫鸣哨音，刀锋更稳，稳得像终于把「守门」二字刻进铁里。

顾小满撒最后一包止血散，撒在拐点，撒得像给后来者留一盏不会熄的灯。灯小，路直，直了，消息就能活着送到。她没入最后一道侧径，像把耳再往前送半寸——半寸，有时比一剑更有用。风腥扑面，杀声已起，起得像东脉夜战开幕，缺不得这一根耳。杂役阵亮，亮在无声，亮在记号，亮在还活着回来的足音里。

杂役阵成，石在脚，脚稳，暗线才走得直——直路尽头，是东脉夜战。
