# 第三十四章 杂役

入门第三日，赖扒皮来。赖扒皮名赖福，皮笑肉不笑，笑里带秤——秤克扣，克半袋麸皮，克一把柴刀，克一句「晦气杂役」。「韩泥，丙九末席，兽栏你包。」

「嗯。」

「嗯？」赖福眼一眯，「不会说话？」

「会。说嗯。」

赖福恼，恼半息，又笑：「老丑，嘴硬。硬没用，宗门看手。」

韩泥举筐——筐是宗门发的破篓，篓底补过，补得像他的脸：丑，牢。赖福瞥筐底，瞥见席角一截陶色，色暗，像腌菜坛沿。沿在，他眼停半息，停完，啐：「还带村里晦气坛？扔了！」

韩泥独眼平：「扔不得。祖辈传的。」

「传？传给你克宗门？」

「克不克，天知道。坛在，人在。」

赖福想再尖，管事路过，咳：「赖福，麸皮按册发，少一粒，我记你名。」

赖福脸一收，收得像刀入鞘。鞘在，刀还在——刀是克扣，克扣是仇苗，仇苗韩泥记在心里，不拔，先活。韩泥低声道：「坛在，人在。人活，克扣迟早还。」

这是小胜：坛没扔，麸皮也没少发——少发是明日的事，今日赖福退了半步。赖福出门，又折返，笑里带秤：「丙九，柴刀钝了，自己磨。磨钝，没饭。」

韩泥「嗯」一声：「钝，手稳。」

「稳？」赖福眼眯，「稳有屁用，凡骨炼不了丹！」

韩泥独眼平：「丹以后再说。先说粪。粪清完，才有饭。」

赖福唇动，没再尖——尖不过「饭」字。饭在，路在；路在，克扣迟早还。晨，杂役灶。刘婆端粥桶，粥稀，勺沉。沉，刘婆手也沉，沉得像顶针顶了一辈子的布。她瞥韩泥丙九牌，瞥完，瞥他肘袄——袄补过，针脚密，密得像叶家手艺，又不是叶家：针脚歪一点，真一点，真像补衣婆自己的手。「新来的，丑脸那个？」她嗓沙。韩泥「嗯」一声。刘婆舀粥，舀满，满得像多给半勺——半勺，恩轻，轻也记。她低声道：「肘破了，找我。别找外门裁缝，裁缝嘴贱，嘴贱收缝钱，钱贵。」

韩泥不接话多。话多，像欠。他只道：「记着。」

刘婆愣半息，愣完，哼：「记着就记着。老婆子不图你还，图你活。」

活，韩泥懂。懂所以「嗯」一声，算收了一句暖——暖在粥里，暖在仇线外，暖得像将来还刘婆那一笔，已有根。午后，药渣场。韩泥分拣渣，渣灰，灰里偶有点香。香淡，淡得像将来炼丹的命。命在渣里淘，淘是丑杂役的活，活脏，脏能活。白织月过，袖风带香，瞥他，瞥得像看灰：「丙九也分拣？浪费香。」

韩泥不辩，只低头，指稳，稳得像抬棺。白织月走远，走远还留一句：「凡骨，炼什么丹。」

话冷，冷不过叶汤。叶汤热过，热在胸，胸热，手便稳。药渣场角，铁无言路过，抱臂，没看他，只瞥分拣堆：「渣里别漏香。」

韩泥「嗯」一声，指稳，稳得像抬棺。铁无言走了，走了也像认可——认可不重，重的是手不抖。管事记簿，记他分拣量足。足，像泥岗卖筐十二文——钱不多，够活。沈枯芽若知他入门，会怯笑一下。笑浅，像芽。芽在泥岗，他在宗门——门隔着，恩不隔。夜，杂役房。韩泥盘坐床角，擦坛沿。沿厚，厚得像爹的手，像祖父的手，像祖辈不知几代人的手。赖福在门外骂邻床，骂完，踢门：「韩泥，明日兽栏加一倍。做不完，没饭。」

门内，他不怒。怒像少年。少年不是他，他是近六十岁，记恩记辱记仇，记到能还那日。坛腹微温，温得像应赖福那脚——脚狠，坛不应狠，应「忍」。忍不是怂，是活路。他低声道：「仇苗在。恩也在。忍得，才醒得了瓮，还得了叶。」

邻床少年悄问：「你跟谁有仇？」

韩泥独眼闭一息：「跟克扣的人。跟嚼谣的人。跟欠恩未还的人——欠恩未还，先还自己命。」

少年不懂，不懂便睡。睡，韩泥不睡实。实睡，像认输。丑时将近，坛沿一线温光，一闪，灭。灭得像公审前夜那闪——闪的是血渗沿，闪的是爹那句「坛在，根在」。他不追问。追问像少年。他只按掌，按实。赖福在门外又骂一句，骂远，骂像风。风过，坛温不退——退的是怕，不退的是命。刘婆门外咳：「肘别挂粪，挂了，我又补。」

韩泥「嗯」一声，算又收一句暖——暖在仇线外，暖在粥里，暖得像将来还刘婆那一笔，根已下。白织月白日那句「凡骨炼什么丹」，他记着，不辩。辩，费舌。舌要留来清粪，留来活，留来还。他低声道：「仇苗在，恩也在。」

丑时将近，符影半闪，闪完即敛。敛，像公审前夜，像瓮醒在远等。邻床少年又醒，悄问：「赖福为何总盯你？」

「盯末席。」韩泥说，「末席好克。克了，他秤满。」

「你不告管事？」

「告了，饭少。」韩泥独眼平，「饭少，恩断。恩不断，先忍。」

少年似懂，闭嘴。闭嘴，韩泥按坛沿——沿温不退，退的是怕，不退的是命。外头赖福骂邻床，骂完，踢门：「韩泥，明日兽栏再加半倍！」

韩泥应：「加。手稳，粪不洒。」

管事门外贴条，条压门柱：「三日后，凡骨测骨。丙九末排，韩泥到。误者，滚。」

三日测骨——比瓮醒近，比还叶汤远。远不要紧，要紧是帖在柱上。刘婆门外又咳：「明早肘别挂粪。挂了，老婆子又得顶针。」

韩泥「嗯」一声：「记着。」

白织月廊下冷声：「末席别碍眼。」

韩泥不接：「眼碍不住。碍的是屎，不是你的贵。」

他低声道：「三日测骨。帖在柱，手别抖。」

外头赖福骂远，远，今晚丙九仍要活——活，才醒得了瓮，还得了叶。叶在远，恩在近。近，够活一夜。一夜活，一日记。赖福秤虚，坛腹应忍；韩泥咽稀粥：忍不是怂，是存锤——锤在叶挡石，石未落，心先稳，稳到能立誓必还，还，先还烫。赖福秤虚，坛腹应忍。咽稀粥——忍不是怂，是存锤，锤在叶挡石，石未落，心先稳。稳到能立誓必还；还，先还烫，烫在叶汤，在刘婆粥，在七笔里分列。咽稀粥，忍不是怂，是存锤——锤在叶挡石，石未落，心先稳。**状态**：大境·凡人 · 资质·丑骨末席（伪灵根） · 鸿蒙九劫瓮·眠（瓮温加剧） · 宗门·丙九杂役 · 漏舍凡舍

白织月袖风冷，冷不过刘婆半勺粥。半勺热，够撑一日杂役。
