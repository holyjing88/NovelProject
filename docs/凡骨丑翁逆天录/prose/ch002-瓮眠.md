# 第二章 瓮眠

雨停在后半夜。

天将明，东边云缝漏出一线灰白，泥岗村仍黑着。韩泥天不亮出门——米缸见底，筐没编完，得上后山采药。苦蒿、蛇藤，能换铜板的都要。篓绳旧了，勒肩勒出印。他习惯疼，疼久了就不哼。

出门前他摸了一下坛沿：凉，沉，照旧。坛留屋里，他不声张。声张像求怪，求怪惹祸；祸来了，筐编不成，药卖不出，活路就断。

山路滑，露水重，深一脚浅一脚。独眼是幼时染病留下的，看远无碍，辨药足够。村里说独眼晦气，他当听不见。麻脸被草叶划了两道也不擦——擦了耽误时辰，篓就装不满。

坡上有霜，踩上去咯吱一声。霜化在草叶上，化出一点腥甜，腥甜里混着苦蒿气。气对了，根就在附近。韩泥用竹铲尖试探土松处：土松，根才长；根长，买主才肯摸第二遍钱。

半山坡，猎户路过，见他铲药，停一停：「晦气翁，这坡蛇多。」

韩泥嗯一声：「看见绕。」

猎户走了，没多话。多话在泥岗村也是晦气。他蹲下身，用竹铲起苦蒿。根须带泥，气味冲鼻。两株换四文，够买一把粗盐。铲到第三株，独眼眯了眯，辨出叶脉偏老——老叶价稳，价稳则碗不空。

石缝里潮，潮气贴着苦蒿，苦得醒神。他又入石缝起一株老蛇藤，根粗须长，味比苦蒿更冲。药篓沉了，篓绳勒出旧印。他在石上啃干饼，饼硬噎喉，咽下去没咳。远处村烟直，把他拴回泥岗村——拴得住，就跑不了；跑不了，就得活。

手心忽然一热，热得莫名。他搓了搓手，热气散了，只剩茧。后山风紧，草伏倒一片，野兔窜过，他不追。追了也不够一锅，还费腿。

回村已是午后。泥屋比早晨更潮。韩泥先看坛：封口破布还在，坛里空，坛壁蒙尘，瞧不出半点古怪——本该如此，腌菜坛能有什么仙气？他伸手一摸，微凉沉，跟死陶无二，缩回手，没声张。村里若知屋里有古怪，明日唾沫就能淹死他。他不怕死，怕麻烦。

夜里就着油灯坐坛边编筐。油灯豆大，光昏，昏里坛角像一块扣着的墨。藤条在膝上绕，绕得指节红。空篓等着明日再装药。坛在角上，人在泥屋里，各安各位，谁也不嚷。

他不会写字。老农民哪懂滴血认主？懂的是欠了要还，辱了要忍，仇了要候。候在心上，比写在纸上实在。他低声道：「人活着，账得记着。记在心里，不乱。」

编一半他停，独眼瞥坛角。坛仍沉，仍蒙尘：「爹，坛在，根在不在，我说不清。」

坛不应。不应好，应了才吓人。爹常说，瓮醒之前，人得醒够。醒够，就是别指望；别指望，掌心里才装得住文钱。

丑时，村狗不叫，鸡未鸣，泥岗村静得像一口扣着的大锅。韩泥没睡实，掌心又热，摸向坛沿——坛壁比白日温了一线，温得极轻，像有人翻身又睡回去。

他睁眼。屋黑，油灯灭。坛沿一线温光一闪即没，蒙尘里凝出水汽，沿水汽爬半寸，爬完即敛。瓮未醒。若醒了，村里不会还叫他晦气翁。可坛比昨多了一丝活气，藏在蒙尘里，不肯露脸。他坐起按心口——心口空了几十年，今夜却填了一指，填不多，够醒着。

天边泛白，他担药篓往集尾去。路上卖盐老汉担担避让，连招呼都省了。韩泥不当回事，药贱也卖，卖一文是一文。

集尾，老蛇藤先出手四文，苦蒿六株十二文，合计十六文，比往日多一倍。韩泥把铜钱一枚一枚拢进掌心，数到第十六枚，停一息，才收进腰布。停这一息，活路才坠实。

买药的问：「晦气翁，今日走运？」

韩泥摇头：「手稳，药老。」

买主笑，笑短却真：「实诚。」

另一买主掂蛇藤根，鼻嗅：「味冲。你这独眼，辨药倒准。」

「眼一只好，够用。药贱，人不贱。」

买主一愣，多摸两文：「拿去买盐。别饿着手。」

韩泥推回去：「说好了十六，不少赚，不多欠。」

「你这老头，倒硬。」

「硬在手，不在嘴。」

买主终收回，却道：「下回还来。」

「还来，药还老。」

他担空篓出村，担得平，把十六文也担进了步子。村口周嫂在井台洗衣，棒槌悬着：「……今日多？」

「药老。」

「少沾晦气。」

韩泥不辩，绕远半步。张麻婆门外骂：「老东西，数钱数到天亮？」

「数三遍，三遍一样，心里才踏实。」

「踏实了就睡。明儿还得卖药，别数没了手。」

回屋他把十六文数三遍，三遍一样，心里才踏实。布结贴腹，钱沉坠坠的。他把药篓绳又检查一遍——绳毛了，毛了结就松，松了药就洒。洒不得，洒了，十六文买来的稳就白稳。

他低声道：「十六文，活路宽一指。」

丑时那点水汽爬坛沿，爬完即敛——瓮还在睡，却像听见了十六文落袋的脆响。脆响在掌心，不在舌上。沿口温罢又敛，敛了瓮仍眠，人仍醒。

门外风紧，茅草屋顶沙沙响。打棺人听过太多瓦上脚步，十回有九回是风。他往腰布摸了一把，昨夜擦沿的布还在兜里，潮了，他没扔。布潮不要紧，沿净才要紧。

瓮眠里，人不急。急像少年，他是记心翁——翁慢，账不凉。沿在，根就不散。醒在缘，不在这一更。

次日天光未亮，他还是先望一眼坛角。角仍沉，仍蒙尘，沉得好。至宝不露相，露相惹祸；惹了祸，十六文买来的宽一指也白宽。出门时露水又起，亮如细盐。盐粒踩碎，碎声轻，轻得像告诉后山：今日还来。

回村路上，他数了数篓里剩的苦蒿，数清楚了，心就不乱。乱则手飘，手飘，药就贱。贱不要紧，贱也卖——卖一文是一文，文文落掌，掌里有数。

猎户又在坡下遇见他，远远喊：「蛇藤别砍光，明儿我们还走这道！」

韩泥抬手应了一下，不多说。多说像邀功，邀功惹嘴。他宁可药篓沉，也不愿舌头滑。舌头滑了，村里能把一株好药说成晦气。

夜里再摸坛沿，沿仍凉多温少。温来得短，像借一炷香的火，香尽就灭。他不求它醒，只求它别出事。出事比饿更难对付——饿能扛，出事扛不住。

他把空篓叠平搁墙角，叠平了，辱也被叠远半步。钱布系成死结，结死了才睡得着。睡浅，掌心又热一线，热得像坛在远处轻轻应了一下。应一文是一文，不应多。

丑时那热又来过一次，短得像误点火星。他把手贴坛壁，贴住了才信：不是梦，也不是手心自己发烧。温来得轻，轻得不敢嚷；不敢嚷，才配守这口眠着的瓮。醒早了惹祸，醒迟了也得先活——活到缘满那一天。

卖盐老汉白天躲过他，夜里路过漏屋时却干咳一声，像认了个熟人。韩泥不应。应了像攀交情，攀交情在泥岗村容易变成「一对」。他宁可各走各的，钱归钱，骂归骂。

张麻婆送来的话还在耳边：明儿还得卖药。话糙，却实。实在比好听有用。他检查药铲刃口，刃有缺口，缺口不影响起根。有缺口的家伙也能干活——跟他这副麻脸独眼跛驼一样。

他把十六文按进腰布最里层，布结再勒一道。勒紧了，心才放得下。放得下，梦才不散；梦不散，丑时坛壁那点温才记得住。

明日上山，瓮未醒。山不认晦气，只认药老不老，手稳不稳。手稳，瓮眠也好，人醒也好，都得先活。
