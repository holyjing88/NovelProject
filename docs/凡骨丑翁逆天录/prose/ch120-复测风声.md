# 第一百二十章 复测风声

外门复测风声，半月内传遍丙九。

柱上条日日换新，新字像秤：「丹试，培元散；手抖除名，末席可报。」韩泥读条，独眼平，报名册上那字还在——字丑，人稳，字也稳。稳了，赖福找不到滚字；滚字在管事册，不在赖福嘴里。

赖福晃木牌，笑薄：「丑杂役也报名？报名，省我除名工夫。」

「除名看手。」韩泥声平。

「手稳，灵根末。」赖福说，「末，丹道断。断了，上台像笑话。」

「笑话上台也得带粒。」韩泥道，「粒正，笑话变哑。」

棚口有人低声接：「他真要上？钱少主那约……」

「小声！」旁人急扯，「约是他家的事，你掺什么！」

韩泥不跟赖福多掰。辩费舌头；舌头留给炼粒、守坛。坛在角，角稳，风声再嘲也不飘——飘了，滚字就真；真了，叶汤断，刘婆迟，芽儿那半块石白还。

报名第二日，内门廊下议论嗡成一片。嗡里提起钱戾衡，提起像影：「三年约那丑杂役，也报丹试？」

「报，让他滚得明白。」有人笑。

笑薄，薄也真。韩泥路过，不近，不躲——躲像怕；怕，手抖。手不抖，笑在耳，不在心。心在坛，坛醒了，备战才真；真备战，两月后上台，眼风再来。

铁无言立渣场，声低：「风声紧，别漏。」

「不漏。」韩泥说，「漏了，粒不正。」

「正，才配上台。」铁无言道，「上台，我站侧。不替你辩，替你挡粪话。」

「谢。」韩泥点头。兄弟线记一笔——记并肩过潮，记今日挡话。挡话不要紧，手稳才要紧。手稳，粒在怀，怀沉的是债序；债序在，步不飘。

刘婆端粥，仍多半勺：「复测风？」

「起了。」韩泥说。

「起风别空肚。」刘婆哼，「空肚手飘。飘了，恩断——断了，你别怪我不舀。」

「不空。」韩泥吃尽，「热顶饥，饥顶备战。备战两月，一日一日熬。」

「熬就熬。」刘婆又道，「熬出正粒，才配谈我那衣。谈早了，衣飞。」

「不飞。」韩泥应，「衣我记三遍。」

午后白织月带婢至药山道，袖风冷：「丑杂役报名，像桩也上台？」

韩泥捧渣袋：「桩稳，台不塌。」

「塌的是丹道骨。」她笑，「骨丑，散必杂。」

「杂不杂，试台见。」韩泥道，「见了，你袖风尖不过正。」

袖风过，过像不服。不服不要紧，试台在远；远，先炼，先稳十层，先藏好培元散。三粒不够，往后闭门再炼到七粒九粒；粒正，手稳，风声再嘲也尖不过「正」。

废窑里他又开一炉。炉温稳，稳过报名那日。多炼两粒，粒在怀，怀沉像债序：芽儿清了，刘婆待还，叶汤在省亲前。序不乱，粒才正。

秦霜远木鱼远敲：「风大，心别邪。」

韩泥心里应：不邪。玉牌凉，三年约烫，钱戾衡影在条后。条后不要紧，手在前；手稳，影就虚半分。虚半分，够走到闭门再炼，够走到廊下影再来，够走到夺魁还刘婆。

邻床少年悄问：「钱戾衡会来吗？」

「来。」韩泥说，「来，记着。记着，不抖。」

「怕不怕？」

「怕手飘。」韩泥摸牌，「不飘就不怕。怕也得上台——上台才还得上清的。」

少年闭嘴。韩泥望坡下试台方向：方向像魁，魁像还刘婆。还刘婆前，先复测；复测过，才往十三走，才夺魁名。魁名在牌，牌歪字丑，人更丑；丑不要紧，丹正才要紧。

风声这一半月，柱上条日日新。脸丑不要紧，手稳；稳了，风声是鞭子——鞭子抽耳，不抽手。手在，粒在，坛在，玉牌在。

赖福廊外骂：「报名也白报！白报，滚得更快！」

韩泥只答：「在。」

在，就不滚。不滚，风声再嘲也尖不过「报」字——报了名，备战才真；真备战，两月后验手；验手，为魁，为还刘婆新衣养老钱。

丑时练鼻下静。粟壳香淡，香落齐，风声这一半月记进心里。记进心里，不进舌——舌响，赖福记「报名晦气」。晦气不要紧，手稳夺魁；夺魁，还刘婆，大比帖在怀。

执事又贴条：「丹试限培元散，私带除名。」韩泥捧陶罐，罐封泥，泥封严。严了，私带谣虚；虚了，舌净，手稳，试台见。见了，正字落，嘲哑半分。

第二周，他闭门半日炼粒，半日清粪，日夜不乱。不乱，赖福找不到由头；由头在腹，腹有气，气不露。不露，白织月袖风再冷，也冷不过坛腹一线温。温应备战，不应嘲。

沈枯芽远站廊下，怯笑浅。笑浅像芽——芽在，徒恩还了；还了，她更怕沾魁的晦气。沾不要紧，韩泥不招手；招手像讨。讨轻，恩重。他只低声对风：「还了就不欠。欠的在下一笔。」

夜里再炼，多两粒，手里凑到五粒。五粒够试台预备——预备在夺魁，不在今日。今日风声紧，紧不要紧，粒正，手稳。

管事弟子路过废窑，随口问：「复测风里，你别闲。」

「不闲。」韩泥铲渣，「炉在转，栏在清。」

「闲了手就飘。」管事道，「飘了，报名字白押。」

「不白。」韩泥应，「字在，粒在。」

他摸玉牌，低声道：「三年约，记着。钱戾衡，记着。赖福克扣，记着。复测风，起——起不要紧，手稳。」

坛腹微温，应「报」。报了名，风就真；风真，备战才真。真备战，接闭门再炼，接织月再嘲，接戾衡影，接帖至那天。帖在山门，影在字后——影在路后，刘婆这笔先还。先还，坛认「还」字。

日暮他回丙九，不近廊下笑。笑在耳，备战在心。心在坛，坛温一线，温像应「圆」——圆不到全债，先圆手稳这一圈。圈圆了，鞭子抽耳也抽不散掌心里的散。

廊下两名杂役撞肩而过，一个咬耳：「听说钱少主要来看丹试？」

「来又怎样。」另一个啐半口，「看完不还是笑他末？」

韩泥脚不停。停了像听进心里；听进心里，刃弯。刃弯了，玉牌就虚。他把那两句骂声当风——风抽耳，不抽掌。

回到废窑，他先把罐泥摸一遍。泥裂一丝，他便重封；重封像把「私带」两字闷死。闷死了，执事贴条才钉不进他罐里。罐里五粒，粒丑，却正——正是他半月扛嘲的底气。

铁无言傍晚又过窑口，扔半袋边角渣：「炼不够再掺。掺多了杂，掺少了稀。你自己拿捏。」

「拿捏。」韩泥应。

「拿捏不住，别上台丢脸。」铁无言顿了顿，「丢脸不要紧，丢刘婆那勺粥才要紧。」

「丢不起。」韩泥说，「勺在，衣还在后头。后头要有正粒垫着。」

外头更鼓沉，沉得像试台在等。等在远，备战在近；近，炼粒，稳十层，喝刘婆粥。粥稀，热，热得像第七笔恩又温半分。半分够撑到魁，够撑到新衣递进刘婆手里——递之前，先把明日窑温守住，把罐泥封严，把「不抖」两字按进掌纹。明日风再紧，也扯不过鼻下那一字不落。

邻床少年翻身问：「你夜里灯又不灭？」

「灭一半。」韩泥道，「留一线给窑。窑冷了，粒就杂。」

「杂了上台？」

「不上。」韩泥截话，「不上白报。白报，赖福更欢。」

少年憋住了嘴。韩泥独眼半阖，掌按罐泥。泥严，粒静，风从窑口进——进得像钩：风起了，下一秤便在再炼、再嘲、再影里等着。等着，也不慌；慌了手飘，飘了就滚。

不滚。先把这半月风声按进刃背，再把明日炉火点起来。炉火一亮，闭门再炼便不再是空话——空话挡不住影，正粒才能挡。


刘婆夜来，看他眼圈青：「风吹的？」

「吹耳。」韩泥说，「吹不进手。」

「吹进手，碗我收回。」她把姜汤放下，「喝。明早还清栏。栏清，名才不像滚。」

「清。」韩泥应，「名在，报在，粒在。在，就不让这阵风白刮。」

风白刮不得。刮过半月，条贴旧了又新，新字仍像秤——秤手，不秤脸。他脸丑认了，手稳守了；守到验手那天，钱戾衡影若来，影硬，他掌更硬。

硬过了，才配谈还刘婆，才配谈帖进山门。帖未到，风先到；风到了，他把备战按实——按实，像把「明日」两字钉进窑缝，钉住就不散。

罐泥干了，他指甲轻抠边，抠出一圈严缝。严缝像规矩：规矩在，私带谣咬不穿；咬不穿，舌净，手稳。稳到手按测台那日，灰线长短随人说，效正却瞒不住鼻子。

他最后对坛说一句：「复测风起了。起了就炼，炼了就验，验了才还。」坛腹微温，像应「验」。

牌凉仇烫。烫着催他走：走出这半月风声，走出闭门再炼，走出钱戾衡影下那一步不抖。不抖，明日窑温仍要守；守住了，条上「手抖除名」四字才咬不到腕子。腕子稳，粒就稳；粒稳，这阵复测风才算被他按住了半寸。半寸够用——够用到明日炉火再起。
