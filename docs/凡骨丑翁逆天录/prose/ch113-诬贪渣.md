# 第一百一十三章 诬贪渣

赖福诬告，落在药渣场。

辰时，丙九刚散队，管事弟子便召韩泥：「渣场少三袋灵材渣，赖管事指你私藏。怎么说？」

韩泥独眼平：「没藏。分拣有账，账在铁无言处。」

赖福立侧，笑薄：「丑杂役手快，快就能藏。藏了，除名。」

铁无言抱臂，声低：「昨夜账我核过。三袋渣，是运去外门丹房，单在这。」他掷簿，簿上朱印新，新得像赖福诬告刚落笔就撞上墙。墙硬，账清，诬就虚；虚了，韩泥记——记在心里，对着赖福，对着三年约，对着玉牌。

管事弟子翻簿，脸色沉：「赖管事，指证不实。」

赖福脸青，青完改口：「不实，也是他晦气招贼。贼偷，算他看管不力。」

「不力，罚。」管事冷声，「韩泥，手稳，账清，不除名。赖管事，诬告罚一月灵石折换权。」

赖福眼一眯，眯得像记仇——仇不落在管事，落在韩泥。落在韩泥，就要变着法压：「不力也要担。担三日全栏粪，粪臭，熏他晦气。」

「搬。」韩泥只一字。

一字，手稳。手稳，管事点头：「三日後，仍渣场。赖管事，再诬，逐出管事位。」

赖福退，退时靴底声沉，沉得像把下一压踩实。踩实不要紧，韩泥记——记在心里，不进舌。记了，刃磨亮半分；亮半分，够撑过公示，够撑过坊市一行。

三日搬粪，粪腥灌鼻。灌鼻不要紧，鼻下静——静，才不飘；不飘，气才凝。凝着，炼气七层门槛又近半寸。半寸，不露；露了，白织月又来嘲。

第一日，坡下有人指：「诬告不成，罚搬粪，晦气活该！」

韩泥不辩。铲下稳，稳得像按灵石——石凉，手不抖；粪沉，手也不抖。抖了，滚字就真。滚了，叶汤恩断，芽儿半块灵石也还不上。

刘婆端姜汤至栏外，姜薄，热：「诬你？」

「诬。」韩泥说，「账清，手稳。」

「稳就好。」刘婆哼，「饭少一口，姜多一片。姜暖，手别僵。」

韩泥喝尽。谢轻，恩重。恩在姜里，也在忍里——忍三日粪臭，换一日不除名；不除名，恩才不断。恩不断，还序才不乱；序不乱，丹才正。

第二日，赖福立栏外，阴笑：「搬粪也搬不出灵根。灵根末，炼气七层？七层也假。」

韩泥铲下不停：「管事看粪清不清，不看层。」

「层在腹，腹在席下坛。」赖福眼亮，「坛呢？」

「灶边腌菜坛。」韩泥声平，「要看，去看。」

诬贪渣，赖福掀筐，掀见灰，灰里无青——青在袖袋，袖袋扎紧。扎紧，沿才净；净了，坛才藏得住。

管事冷声：「验过了。赖福，少嗓，多秤。」赖福脸一青，青完，眼却扫韩泥袖——袖里无香，无烟，像私藏从未有过。

韩泥不辩。辩，费舌；舌要留来守袋，留来守坛。守坛，守114公示辱，守115尾摊潜。

赖福没去。没去，像怕再撞铁无言的账。铁无言傍晚来栏边，掷半袋麸皮：「赏的。别给赖福看见。」

韩泥收下，记——记铁无言兄弟线，记渣场账清这一胜。胜小，胜在活。活在还恩前头，活在复测前头，活在三年约反面。

第三日，全栏粪清。清完，管事验栏，声沉：「韩泥，三日粪清，手稳。记功半笔。」

半笔，比灵石重。重在心里，不落炫耀——炫耀，赖福来；赖福来，诬告又来。又来不要紧，手稳，册在，就不滚。

韩泥回窑角盘坐，气冲关，关开一线，炼气七层。七层，仍不声张。声张，赖福指「偷气」；指不要紧，手稳，账清，诬告已败一次。败一次，刃亮一分；亮一分，够接公示辱。

他摸玉牌，牌凉：「钱戾衡，记着。赖福，也记着。」

坛腹温一线，应「清」——粪清，账清，心清。心清，鼻下香才落齐；香落齐，七层才实。实了，才配去坊市尾摊，才配捡漏那一页残诀。

邻床少年悄问：「赖福还会诬？」

「会。」韩泥说，「诬不要紧，手稳。手稳，册在，就不滚。」

少年似懂，闭嘴。闭嘴，韩泥望药山——山有坊市道，道在远，远得像钩。钩不要紧，先熬过赖福下一压；下一压，在坡下公示。公示三日，一日一日熬——熬出头，是坊市；熬出头，是还芽儿。

秦霜远木鱼又敲一声：「心别邪。」

韩泥不答，答在心里。心里排序：叶汤头恩，芽儿半块灵石，刘婆粥，分列，不混。混了，还乱；还乱，丹不正。不正，刘婆恩还迟，叶汤还凉。

丑时，他练鼻下静。粟壳香淡，香落齐，诬告这三日才记进心里。记进心里，刃又磨亮半分——亮在还芽儿那日，亮在复测备战，不在粪栏。粪栏臭，忍三日；忍三日，换不除名，换七层实。

外头赖福骂远：「三日过了，还有三年。三年，丑杂役必滚。」

韩泥低声道：「三年约，记着。」

记着，就不滚。不滚，七层才实；实了，才配去坊市尾摊，才配捡漏那一页残诀。残诀在远，公示在近——近，忍三日；三日过，路又宽半寸。宽半寸，够走到114坡下公示，够走到还沈枯芽半块灵石前头。诬告败后，赖福消停半日，半日像暴风雨前闷。韩泥不趁闷偷懒，仍渣场分拣，仍兽栏清粪，仍编筐交差。交差，手稳，册上「韩泥」二字才不像滚字。

管事弟子当众念罚：「赖管事诬告，折换权收一月。」丙九有人窃笑，笑薄。韩泥不笑——笑像认晦气；认不得，只记。记诬告这一笔，进仇线，不进恩线。

午后，七层气走稳，他废窑盘坐。气周而不泄，泄了，白织月说「偷境」。白织月果然过药山道，袖风冷：「搬粪也搬出七层？七层，仍末。」韩泥不辩：「末，看手。」袖风过，过不要紧，腹里七层实。

沈枯芽远远站柱角，手捧空陶罐，罐像曾装粥。粥恩在泥岗，石恩在宗门——宗门半块灵石，他已在怀里攒。攒不要紧，先七层稳，先诬告尽，先赖福下一招。

铁无言掷一小袋麸皮：「赏的。诬告败了，别飘。」韩泥收下：「不飘。」不飘，才配接114公示。公示辱在坡下，辱在众目；众目下，手更不能抖。

夜，他数怀内灵石碎，碎少，少得像芽儿半块石还差一截。差一截不要紧，渣里淘，坊市捡，两月可凑。凑齐，118还；还了，徒恩首笔落，落得不张扬。

坛腹温一线，应「清」——粪清，账清，诬告清。清了，七层才实；实了，赖福脸青，青得像秤空一格。一格空，韩泥路宽半寸；半寸，够走到公示三日，够走到尾摊潜行。

秦霜远木鱼远敲：「仇深，忍住。忍住，鼻下才静。」韩泥不答，答在心里——心里玉牌凉，三年约烫，钱戾衡名在仇顶。顶不要紧，刃磨亮；亮在还芽儿，亮在复测，不在粪栏舌上。

他低声道：「诬告，记着。七层，记着。」记着，就不白搬三日粪。白搬，辱白挨；不白挨，仇苗深，深得将来试台手不抖。

诬告败后第三日，韩泥仍立坡下公示榜侧，不躲。躲像怕，怕手抖。手不抖，辱在舌，不在心。心在坛，坛温应「清」。清了，七层实；实了，尾摊可潜。潜了，捡焚火残页，稳窑温，炼粒备复测。复测在远，赖福压顶尽在即。尽在即，118还芽儿，119十层，130还刘婆夺魁。

他摸玉牌，牌凉，低声道：「三年约，记着。手稳，粒正，试台见。」见字落，坛温一线，像应。应了，路又明一寸——明向还恩，明向不滚，明向刘婆新衣养老钱。

更鼓沉，沉里，韩泥按木牌，牌凉贴胸。113章手在，名在册，坛沿净。舌不留，鼻下静。恩辱分列，不混。活长了，才还得了叶汤，才接得住门缝灯。

刘婆粥仍多半勺，姜仍多一片。片暖，恩不断。恩不断，手才稳；手稳，粒才正；粒正，试台才配说话。说话了，白织月袖风尖不过「正」，钱戾衡影硬不过手。

试台坡下鼓响，鼓不要紧——鼓在耳，手在掌；掌稳，粒才正，帖还远。
