# 第一百三十二章 药峰行

药峰晨雾未散，雾里药香硬。

硬才像峰。峰高，活苦；苦，才像外门积势，不像一部爽完就散。韩泥立东垄口，破筐靠膝，等管事分活。怀帖仍沉，沉得像提醒：夺魁了，也得摸渣。

管事弟子喊：「丙九韩泥，东垄分拣，午前清栏，午后编筐。」

「在。」韩泥只答。在，就不滚。

东垄渣灰，灰里带着昨天没闻净的味。他鼻下静一息，静里辨出辛淡——辛淡正，正像嗅诀回响。回响在鼻，不在舌。

同批少年掩鼻：「你夺魁了还分拣？」

「分拣。」韩泥声平，「魁在牌，活在手。手不停，牌才不虚。」

「牌虚了呢？」

「虚了，帖也凉。」韩泥道，「凉了，大比白接。」

少年退半步，没再问。问像沾晦气；晦气他不怕，怕的是手飘。

刘婆送线过垄，线粗，声低：「药峰活重，别空肚。」

「不空。」韩泥说，「空肚，手飘。」

「筐勒匀，别勒破手。」刘婆又补半句，「破了手，挂牌闻不准。」

「匀。」他应。

刘婆哼一声。话在还衣那日余温——温在，线才送得心安。她把线塞他袖口：「省着用。线断了，筐散。」

铁无言挑水路过，水洒稳：「药峰别拜邪。拜邪，路断。」

「不拜。坛在，人在。」韩泥说。

「人在就行。」铁无言望东垄，「挂牌近了。近了别急。」

「不急。急了，渣不真。」

两人目光一碰，碰得像兽潮并肩那半步还在。半步不显，显了赖扒皮嗅得出；嗅得出，丙九连坐。

午前清栏，栏臭，臭不要紧。臭是药山活；活有味，味在角，不在席。他倒渣半筐，摊匀，匀像秤平——秤平，手才静。

赖福余党掷石过垄，石落筐边，筐未翻。有人嚷：「翻啊！翻了好看！」

韩泥不抬眼。未翻，手稳，活像常杂役。常了，才藏得住嗅。

午后编筐半刻，藤刺扎掌。他偏手，刺浅些，筐勒匀——匀像火路；勒不匀，渣散，路断。

赖福秤筐，秤杆偏：「仍少二斤。」

「看错了。」韩泥说，「渣在筐底，底沉。」

「底沉？」赖福冷笑，「你会装秤？」

「装不了。」韩泥道，「底沉，秤自平。」

赖福眉一蹙，再称，杆回正。正像吃瘪。他啐一口：「丑丹师也会算。」

「不算。」韩泥说，「只看底。」

管事弟子验袋，袋灰，灰里白细：「清栏了？」

「清了。」韩泥说。

「明日照旧。」

「照旧。」

照旧才像积势。积势不在快，在忍。忍五年赖福，忍兽潮并肩，忍帖在怀仍上山摸渣——摸渣，才像第二部开笔。

日暮收工，他回丙九，不近廊下笑。笑在耳，不在心——心在坛，坛腹温一线。

邻床悄问：「你天天这么活，不累？」

「累。」韩泥说，「累，才像活。活，才配帖在怀。」

「那魁名不虚？」

「不虚。」他答，「魁在牌，活在手。」

「手不停，不怕人笑你？」

「笑在耳。」韩泥道，「耳可听，心不随。」

沈枯芽再递口信，信仍无字，只有一片枯叶——叶黄，字在记里。韩泥把叶夹进册页，低声道：「叶在。汤在前。」

白织月袖风没上山，山里只有药香和渣灰。渣灰好——灰里没有嘲，只有真。真了，挂牌才像试水，不像赌名。

丑时，练鼻下静。粟壳香混药香，药香正，正里只求静一字不落。落齐，这一日记进心里。

管事夜巡记簿，记「丙九韩泥，分拣清栏」，字细，细得像小胜。胜在忍，不在辩。

他望药山半腰，雾散一线白。白像还刘婆那晨，晨远，路在；在，就不绝。

半腰有人往下扔空袋，空袋打在垄边，扬起灰。灰进鼻，他不咳，只把气沉下去：「灰进鼻，辨还在。辨在，就不滚。」

少年在下风处喊：「韩泥！你鼻不要了？」

「要。」韩泥说，「要用来闻正。正了，才挂牌。」

「挂牌给你棺材铺用？」

「给活人用。」他弯腰继续分拣，「活人要栏清。栏清了，兽才少一寸邪。」

扔袋的人骂了两句走开。走开也好，垄上又只剩药香硬和手稳。

铁无言又过棚口，声低：「别飘。」

「不飘。」韩泥应。

「明日坊市？」

「近了。」韩泥说，「近了，先听风。」

铁无言点一下头，像认，又像约下一回站侧。

刘婆柴门外咳：「山上别沾晦气。」

「不沾。栏清，手净。」韩泥说。

她递半块饼：「吃。手抖了，榜虚。」

「不抖。」他接饼，谢轻。饼粗，温实，像恩还没散尽。

老耿路过，哑声：「药峰过了？」

「过了一日。」韩泥说。

「过了就记。记着，挂牌才有底。」

「记着。」

更鼓沉，他按木牌，牌凉贴胸。胸贴紧，像贴债序——序里写药峰，写坊市，写大比。

他低声对坛：「药峰过了一日。下一回，挂牌。」

坛不应字，只温。温像应「分拣」——分拣净，嗅才准；准了，丑丹才配试。

坡下石阶湿，他数息，数债序。债序在，步不飘；步不飘，才接得住坊市那三日。

同批少年掩鼻过，又问：「你咋天天上山？」

「倒渣。」韩泥只答。

「倒渣里闻什么？」

「闻正不正。」他说，「正了，牌才挂。」

倒渣里藏分拣，分拣里藏嗅——嗅在鼻，不在舌。舌要留来守坛，留来答管事，留来对赖福说「底沉」。

散队后再编筐半刻，藤刺浅，沿净。沿净，坛才藏得住；藏得住，帖才像活。

验息后他理册页，页黄，字丑：炼气十四。十四下边空一行，像等下一层——等，不要紧；等要用手等，不能用嘴等。

赖福廊下过，啐一声：「活着？」

「活着。」韩泥道，「活着才分拣。」

「分拣到何时？」

「分拣到挂牌。挂上了，再听你啐。」

啐的人噎住。噎住也好，噎住就不占他鼻。

天边白一线，他望药峰雾散处，独眼平：「药峰过了。记着。」

记着，是丑翁本事。本事在指稳，在鼻准，在累也还上山。上山过了，挂牌才有底；有底，风才改一半。

他按心口，帖角硌肋，硌得像提醒：明日不是歇，是挂牌。挂上了，才听得全坊尾的风。

山风再硬，硬得像催。催也催得正经——正经催的是坊市木棚、歪牌、三日试水。水试过了，帖才不悬；帖不悬，大比台才踩得实。

丙九廊下，有人揉鼻子：「你身上药味重，像背着一座山。」

「山要背。」韩泥说，「背过了，鼻才准。」

「准了卖给谁？」

「不卖。挂牌听风。风里有真，有嘲，嘲听全了，真才不丢。」

人耸肩走开。走开也好，走开他才把木牌按实。

侧门风过，带一点药香余冷。冷里，他望坛：「手在，路在。挂牌在前。」

坛温一线，像点头。点头不响，路却明。明向南脚坊市，明向那块丑牌，明向还没改完的风。风未改，人不能歇。歇了，嗅会钝；钝了，试水就虚。虚了，帖便成摆设——摆设挡不住钱戾衡那眼风，也挡不住大比台前那口气。

夜更深，他再摸怀帖，帖角已磨毛。毛像路：走过药峰，还要走进棚乱价乱的坊。坊乱不要紧，牌歪不要紧；要紧是鼻下那一息正。正住了，才配把「丑丹师」三字钉在破筐上。

刘婆隔篱又塞进一截布条：「勒手用。藤刺深了，闻味会飘。」

「谢。」韩泥接过。

「别谢。谢多了像还不清。」刘婆哼，「你清了衣，别再欠鼻子。」

「不欠。」他说，「鼻子留着用。」

篱外脚步远了。他看布条，布旧，针脚密，密得像恩还没散尽。恩在，苦也接得住；苦接住了，挂牌才不像赌气。

更鼓将尽，药峰影黑成一块。黑里有路。路上有渣。渣里有正。正到手，风才会改。改之前，他得先把明日坊市那块歪牌在心里挂一遍——挂稳了，手才不慌。

邻床翻身问：「明日真去挂？」

「去。」韩泥说。

「不怕空摊？」

「空摊也听风。风听全了，就不空。明日挂牌一开，听的才算入账。」
