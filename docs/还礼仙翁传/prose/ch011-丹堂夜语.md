# 第十一章 丹堂夜语

回春丹落案那夜，丹堂炉火又亮。

苏念慈本已回房，却在廊下停步——她听见自己方才说过的话在脑子里反复响：「回赠……不是还债。」师兄点头时，像认了一个她藏了十年的字。她怕那字烫，又怕那字凉，最终折返，药篓换成空盏，只带一袖丹香。

丹堂外廊风硬，灯影从门缝漏出，金红一线，像炉心吐的舌。

莫长春已在廊柱旁。

不坐石阶，不坐主位，只让半边灰袍被丹光洗过，像守门，也像陪炉。杖点廊石，竹纹暗动，与白日不同，夜里竹纹像醒着。

苏念慈止步三步：「师兄还没睡？」

「等一个人问明白话。」莫长春看她，「你折返，便是那人。」

苏念慈脸热，硬撑：「我来问药渣。外头说师兄赠剑赠符，赠到杂役头上，唯独赠丹堂一包渣——是不是羞辱？」

「羞辱你？」莫长春摇头，「羞辱丹堂首座，青岚门先羞的是自己。」

「那为何不给好丹？」她声音发紧，「回春丹是回赠，药渣是赠。你先赠渣，再等我熬红了眼，才给甜——师兄，你当我什么？」

莫长春沉默片刻，道：「好丹给你，是应当。你是首座，是苏氏丹道的门面，宗门该给你的，老夫没资格替宗门赖账。」

苏念慈一怔。

「药渣给你，」他续道，「是只有你收得下。」

廊外弟子远散，只剩风与炉息。苏念慈指尖掐进掌心，疼得清醒：「我从十二岁入丹堂，收得下苦，收得下夜，收得下赵玄机冷脸、周德海阴阳——唯独收不下师兄把心意当渣扔过来。」

「渣里有什么？」莫长春问。

「炭灰，药筋，还有我熬夜的气。」她脱口而出，像把篓底翻给他看，「你以为我不知道？你每回赠渣，渣里都有我当日炼废的方子——你记得我哪一夜失手，哪一夜守炉。」

莫长春须眉在灯下白得刺眼，笑却温：「记得，才赠渣。丹堂首座若只记得甜，炼不出三品。」

苏念慈眼眶一烫，别脸：「……我缺的不是丹。」

「缺什么？」

她唇动，字到齿边，终于落下，轻得像怕惊动炉盖：「缺一个人，在我熬到三更时，不说『首座当表率』，只说……念慈，歇一歇。」

廊上静极。

丹堂内火噼啪一声，像替人答话。莫长春不躲她目光，却也不接那句烫人的字，只抬杖指门内：「夜深，火还亮着。明日要炼培元批次，别误了丹。」

苏念慈站在原地，像被定住一半。

她想说：我等了十年，你就回我一句别误炼丹？

话未出口，却见师兄侧脸在灯影里极淡，淡得像九月寿元——不是拒绝，是把她往门内推，推回她该站的位置，不伤她名节，不借夜风坐实外头谣言。

苏念慈忽然懂了：他装聋，不是不知；是知得太清楚，才不能把「心意」点破在廊上。

她低声：「……好。我进门。」

转身时，袖中无簿，心口却一跳，像有人在她脉门上记了一笔，不记名，只记温。

莫长春等她入门，火色吞没她背影，才轻声，像说给风听：「心意不能点破在廊上。点破了，就变成债。债要还，还急了，就俗了。」

杖中竹纹游动，气音仍是一字：

等。

后半夜，培元丹入库的消息贴到丹堂库房，墨迹新干。

苏念慈在炉前看见榜单，莫长春名字在末，份额一栏写着「长老例份」。她指尖抚过那三字，没哭，把火调稳，像把一句话也调稳在心里：

他说不出口的那句，她替自己收好。

廊外，莫长春已回小院。

簿上浮字，淡得几乎看不见：

**丹堂有人，门内是戏，门外是礼。**

他合簿，望丹堂方向一线夜灯，念：「下一份，是培元被扣，还是妒火进门？」

杖答：等。

莫长春笑：「等。」

九月还剩八月，够把一句说不出口的心意，熬成将来还得了的回赠。

大因果还在远处，像妖潮，像赤焰——丹堂夜语只是小回赠，大还礼还在后头。

后半夜，培元丹入库的消息贴到丹堂库房，墨迹新干。苏念慈在炉前看见榜单，莫长春名字在末，份额一栏写着「长老例份」。她指尖抚过那三字，没哭，把火调稳。

廊外，莫长春已回小院，簿上浮字，淡得几乎看不见：**丹堂有人，门内是戏，门外是礼。**他合簿，望丹堂方向一线夜灯，念：「下一份，是培元被扣。」

弟子远散，远散里仍有人嚼舌，嚼成「首座与长老廊上夜语」——嚼得轻，轻不过炉盖下的火，火还在，话就不必点破。

围观弟子在丹堂阶下踮脚，踮得像等一句点破——点破没来，来的是炉盖下的火，火还在，话就不必点破。有人低声：「首座红过脸，长老坐门外，老不正经？」另一人答：「不正经的是嚼舌，不是陪炉。」
