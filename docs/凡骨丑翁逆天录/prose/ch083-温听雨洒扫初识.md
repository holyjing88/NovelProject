# 第八十三章 温听雨洒扫初识

废渣路后第五日，丹阁侧廊新扫。水痕干一线，干得像界——杂役走廊，不进门；进门，就是另日的事。韩泥担渣过侧廊，肩稳，渣不漏。漏一粒，记过；记过，粥勺就浅。

廊角帚丝匀，匀在步，匀在腕，匀在鼻下那一点冷香。冷香不像白织月袖风，也不像刘婆粥香，像丹阁药洗常年浸过的味。淡，淡也要记。记了，才知道丹阁风跟兽栏腥不是一路东西。

同批少年已经窃语温氏名。他不插话。插话像凑热闹，凑热闹要还人情，他还不起。还不起人情，就别先欠眼色。

铁无言立栏边，声低：「温氏眼毒。毒里三分真——真在末席。你渣里活长了，她眼才肯改。改在远，别盼。盼了，手先飘。」

「不盼。」韩泥道，「她扫她的，我担我的。」

廊侧那人抬眼，眼清，清里带傲。傲在皮，不在帚——帚仍匀。她瞥韩泥筐沿，筐丑，人更丑。眉蹙一息：「丙九也走丹阁廊？廊刚扫，别踩灰。」

韩泥不辩。辩费舌。他停半息，让出整尺，让得干净，干净得像把路让给规矩。

「让就好。」她声不高，落耳仍尖，「末席担渣，像污廊。污了，我还得重扫。」

「渣在筐，不在廊。」韩泥只这一句。

答完就走。走稳，肩不晃，筐不洒。洒了，她更嘲；不洒，小胜。胜在肩，不在嘴——嘴赢不了丹阁风，肩赢过记过。记过一次，名上就沾灰；名上沾灰，废渣路白走。

她未再嗓，帚丝又匀。匀像尺。尺在，他不近；不近像不求。不求，只记：丹阁廊，帚丝匀，冷香淡，眉蹙那一息。名他还不知道，名不要紧；人在帚上，帚在，像钩。钩不急着咬，先挂着。

午后少年们说开了：「丹阁廊扫的是温听雨，温氏的外门，灵根不差。」

「温氏？」有人咂舌，「那咱别近，近了就晦气。晦气粘温氏，门都嫌。」

「嫌就嫌。」另一个低声，「丑杂役敢还嘴？还嘴更脏。」

韩泥仍不插。心里只按着：恩线不添，仇线不添，中间留一线空——空才装得下日后的事。日后远，近的是筐沿别晃。

兽栏仍腥，他铲得稳，稳里那一点冷香又回半息，回像药山深灰的辛。辛淡，淡也提醒：廊上那人不是粥，不是仇，是一眼尺。尺量肩，不量脸。

管事路过，记一笔：「丙九韩泥，过廊不洒渣。」

赖福讽远：「温氏也嫌你？嫌对了，末等污廊。污廊就该扫出去。」

韩泥把那句再钉一遍，低得像对自己说：「渣在筐，不在廊。在廊，才叫污。」

刘婆端粥，仍多半勺：「丹阁廊挨眼了？」

「挨了。」韩泥说，「不辩。辩赢嘴，输肩。」

「不辩怕什么。」刘婆哼，「手稳，饭稳。饭稳，恩不断。断了恩，丹阁风再香也顶不了饿。」

韩泥点头。恩在粥里，不在丹阁那点傲里。傲冷，粥烫；烫的才能记长久。

黄昏渣场分拣，鼻下有根，手稳。冷香回进沉腐浮——浮是假香，剔净，真香才留得住。真香淡，淡也是路。路上多一眼，不等于多一份攀；攀了，路就斜。

丑时前廊下闭息，息在鼻，不在嗓。嗓响，香飘；香飘，温氏就真嫌。嫌了记过，记过就损丙九。他按席角，席严，坛不露。坛腹温一线，贴掌，温不应傲，应「记」。记初识，不记攀；攀字一落，缘线先脏。

邻床问：「温听雨漂亮，你怕不怕？」

「怕在污廊。」韩泥说，「怕不在貌。貌在皮，污在筐。筐稳，她嘲也嘲不进肩。」

「那你喜欢？」

「不近。」韩泥道，「近了还不起。还不起，就别伸手。」

少年懂「不辩」，睡了。秦霜木鱼轻敲：「识人别识皮。识皮，心邪。邪了，帚丝也成绳。」

白织月远廊过，袖风冷：「杂役也过丹阁？过出温氏眼？眼脏了，廊都要重刷。」

韩泥不抬头：「过廊，不进门。渣在筐。筐不洒，廊不脏。」

白织月唇线绷一息，走了。少一句嘲，小胜。小胜攒着，不声张。

同批又悄问：「温氏会不会记你晦气？」

「记不要紧。」韩泥说，「手稳，筐不洒，廊就不污。廊不污，记也白记。白记伤不了册。」

第三日外堂阶尘更少。更少不要紧，帚仍稳，稳像渣场分堆。温听雨擦栏，巾湿，水净；韩泥扫阶，帚偏，灰不扬到她脚边。二人仍各干各的。她瞥帚痕，眉松一线，没再多说。没多说，也好——多说像欠，欠了难还。

管事验阶栏，皆净，各记一笔。赖福立堂外，眼细，讽晦气：「晦气扫阶，阶都晦。」

韩泥不接茬。铁无言声低：「温氏傲，敬才，不敬貌。你别求敬，手稳就行。稳了，她眼毒也毒不到你袋里。」

「手稳。」韩泥应，「袋也稳。」

午后他分拣，指稳。温听雨远廊过，袖风温一线——温进记里，不进舌上。他对坛低声：「初识了。再遇在后。不叫，不近。近一步，像把尺往自己脖子上搁。」

洒扫毕，阶尘少。二人不多废一句闲话——闲话像软绳。软绳在，缘苗才邪不了。邪了，秦霜木鱼敲第二声，敲完仍救不了心。

老耿挑水过，声哑：「境别拜邪。拜了，路断。断路人，扫帚也嫌。」

「不拜。记着。记帚，记香，记让尺。」

廊下风硬，木牌系紧。同批远避仍避。藤刺偏手，沿净。夜巡灯照席角，仍像腌菜坛。刘婆粥稀，勺沉，他喝尽，不谢，只记。记粥，也记廊上那一眼蹙——蹙不像情，像规矩落地。

更鼓沉下来。他按木牌，牌凉贴胸。名在册，坛沿净，鼻下静。活长了，叶汤才还得上；还得上，门缝那盏灯才接得住。灯在远，廊在近；近的先走稳。

丹阁侧廊水痕又干一线。界在，他不越——不越像不求；不求，只记帚丝匀、冷香淡。明日若不碰，后日也会碰。碰的时候，肩还要稳，筐还要不洒。洒一粒，初识就脏；脏了，再遇也站不直。站不直，十包渣、废窑试温，全像笑话。

他合眼，心里把「渣在筐」又按一遍。按实了，丹阁廊上那一点冷香才只是味，不是绳。味能记，绳不能绑——绑了，手就飘。手飘，废渣路白攒，叶家那碗烫也还不上。

丑时过半，棚柱吱一声。有人起夜，踩过他床尾，低骂「晦气挡路」。他侧身让，仍不吭。吭了像认晦，不吭像让尺——跟丹阁廊同一套活。让完，再摸坛沿。沿仍温，温得老实。

铁无言说过眼毒。毒不怕，怕自己先把筐沿晃脏。脏了，再遇那天她不必嘲，管事先记过。记过一回，三年约玉牌捂在胸口也烫手。

他半坐起来，摸过床角暗袋——袋沉，十包还在。在就好。渣在，肩才有底；肩有底，丹阁廊让尺才不像讨好。讨好脏，让尺干净。干净了，冷香只是冷香，不变成绳。

天将亮时他又闭一息。息里扫过帚丝匀、筐不洒、粥多半勺。三样记齐，初识才算入账，不是路过看了一眼热闹。

远处丹阁方向隐约有帚响。响不进棚，却进记。记里她仍傲，傲不要紧——傲量得了廊，量不了他暗袋里的十包。十包在，肩就硬；肩硬，明日再碰面，筐仍不洒。

坛腹温一线回来，像应他：初识落了，线空着，空着才接得住再遇那一眼验。验肩，验筐，验他还敢不敢让尺。敢让，就还有明日；有明日，空线才不白留。留着，才接得住后头那一眼。
