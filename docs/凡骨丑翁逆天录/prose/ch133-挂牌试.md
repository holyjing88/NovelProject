# 第一百三十三章 挂牌试

坊市开市，尘起。

外门坊在药山南脚，棚杂，价乱——乱里藏真，也藏嘲。嘲在舌，真在耳；耳要留来挂牌，留来试水。韩泥立末摊，破筐反扣，扣上木牌。牌歪，字丑，八个字，无夸：「丑丹师·嗅渣辨性·试水。」

管事弟子验牌：「试水三日，价自定，税照旧。」

「照旧。」韩泥交半文。文旧，旧像末席该交的。

「价怎么定？」管事随口问。

「半包渣，半文听。」韩泥说，「听得准，再加成。」

管事「嗯」一声，像不信，也不拦。

第一日，无人近摊。近像沾晦气。晦气他不怕，怕的是耳不聪——聪了，才听得全坊尾风。

白织月袖过，声冷：「丑丹师？试什么，试晦气？」

「试渣。」韩泥只答，「渣净，栏清。」

「清给谁看？」

「给手看。手准了，牌才不虚。」

「手准？」她唇角一挑，「丑骨的手，也配准？」

「配不配，渣见。」韩泥道。

她袖香擦过棚角就走了。走了也好，空摊才像试水；试水不在客，在耳。

棚旁有人小声：「真挂了？」又有人嘘：「嘘，别沾。沾了，赖管事记名。」

韩泥不抬眼。抬眼像讨，讨了，正字就脏。

第二日，有人递渣，渣灰，灰里杂。韩泥鼻下静一息，静里辨出辛淡——辛淡正。

递渣人愣：「你真辨？」

「辨。」韩泥声平，「辨了，才挂牌。挂牌不在名，在手。」

「辨错了呢？」

「错了，牌摘。摘了，我认。」韩泥说。

人退，退时多留半包渣——半包像试。试出「正」，风就改一半。

又有人远远扔一撮灰：「闻啊！闻晦气！」

韩泥捡起，闻一息，放下：「灰假。真渣不这么燥。」

扔的人噎住。噎住也好。

第三日，管事贴条：「丑丹师试水，暂留。留，不在荣，在听。」

韩泥收摊，收得像没来过。没来过不要紧，牌已挂——挂丑，挂稳，挂得像从前那三日。

铁无言过，脚停半息：「牌钉住了？」

「钉住了。」韩泥说，「钉住了，才接风声。」

「风声紧。」铁无言声低，「钱戾衡的人，在坊尾。」

「不近。」韩泥道，「近了像挑衅。手稳，就不滚。」

「三日别辩。辩像显摆。」

「不辩。显摆，像滚。」

铁无言点一下头：「我站坊口。不近摊。」

「不近也好。」韩泥说，「近了像认。认了，赖扒皮嗅得到。」

刘婆买米过，米糙，声低：「少摆摊，多吃饭。」

「饭稳。」韩泥说，「摊是试水，水试过了，才谢得老耿。」

刘婆愣半息，没听懂后半句，听懂「饭稳」：「稳，别飘。」

「不飘。」

她端粥仍多半勺：「试水别空肚。空肚，鼻钝。」

韩泥点头，谢轻，恩重：「鼻钝了，牌虚。」

邻床少年晚间悄问：「你真挂牌了？」

「挂了。」

「挂了，不怕人笑？」

「笑在耳。」韩泥说，「耳听嘲，手辨渣。渣正，笑就远。」

「客有几个？」

「有几个是几个。客少，耳清。」

赖福余党围棚外起哄：「丑丹师卖晦气啦！」

韩泥不接茬，只把半包渣再验一遍。渣正，嘲就远；远了，听才全。

管事弟子再验牌：「续留三日，税照旧。」

「照旧。」他又交半文。声平，平像末席该交的。

「续留，别惹事。」

「不惹。」韩泥说，「只听风。」

回丙九，他按坛。坛腹温一线，像应「牌」字。

丑时，练鼻下静。粟壳香淡，落齐，挂牌这三日记进心里。

更鼓沉，他望坛，独眼平：「试水过了。下一回，铁兄续。」

坛不应字，只温。温像应「听」——听风紧，手更稳。

坊市尾巷，谣与真并起——谣在舌，真在耳。钱戾衡影在坊尾，影淡，淡像将紧。紧不要紧，手稳，就不滚。

他再摸木牌，牌歪，字丑：「丑不要紧。渣正，才要紧。」

摊前忽然来了个买参须的老太，眼神浊，却把一撮灰递过来：「老头子说你闻得准。闻闻，这灰能入药不？」

韩泥闻一息，摇头：「燥。假灰。入药伤脾。」

老太愣住，又像信了：「我儿说你丑，丑骗人。今日我信鼻，不信嘴。」

「信鼻就好。」韩泥把灰退回，「假的别买。真的贵，贵得值。」

老太走时留下半块粗糖：「给你润嗓。闻灰的人，嗓子苦。」

「谢。」韩泥收下，不说谢重——谢轻，恩才不像债。

白织月远廊再过，袖风冷，冷也尖不过「正」字。正字在掌，不在牌。

沈枯芽远远鞠半躬，不近摊。不近不要紧，徒恩清了半块灵石；清了，叶汤还在前。

秦霜木鱼轻敲：「牌挂了，心别邪。」

韩泥不答，答在心里。心里分列：嘲是嘲，试是试；试过了，才续兄弟线。

废渣巷日斜，他验渣，渣正，手才静。静了，收摊如石——石在，牌在，风过不留痕。

散队后编筐半刻，藤刺浅，沿净。沿净，坛才藏得住；藏得住，牌才不虚。

管事夜巡记簿，记「丑丹师试水」，字细，细得像小胜。胜在渣正，不在嘴。

同棚杂役笑：「挂牌三日，赚了几文？」

「赚了半包真渣。」韩泥说，「渣真，够了。」

「真渣能当饭？」

「能当耳。」他说，「耳清了，坊尾才听得住。」

笑的人噎住。噎住也好，噎住就不占他鼻。

他低声对坛：「挂牌试水过了。记着。」

坛腹温，贴掌，像应「试」。试过了，风改一半；一半改，铁兄那半步才接得住。

天边白一线，坊市木棚色浅。浅里，他按心口，帖与牌并贴——帖沉，牌歪，人更丑。丑不要紧，正字落了，路就明。

明向东角栏，明向兽潮后那半步兄弟。栏若不稳，挂牌白挂；挂牌稳了，下一回该修栏，该把并肩续成站。

老耿夜里路过棚外，哑声问：「牌子还在？」

「在。」韩泥说。

「在就守。守到风改。」

「守。」

药廊远处有人仍嚼「晦气摊」，嚼声远，远不过他掌里那半包真渣。渣真，嘲就浅；浅了，耳才空得出地方听坊尾。

坊尾灯笼晃，晃出人影一截。影淡，淡像钱戾衡的眼风还没落全。落不全不要紧，他不追影。追影像挑衅；挑衅坏手，坏手就坏牌。

他回屋把木牌擦干净，歪字仍丑：「丑就丑。丑里正，才算数。」

铁无言窗外站一息，不进门：「东角栏缺灰。缺了，我等你。」

「等。」韩泥应，「试水过了，就补灰。」

「别迟。」

「不迟。迟了，线断。」

白织月再没近摊，可坊里仍有人学她的冷腔：「丑丹师，卖晦气啦——」

韩泥抬头一次，只道：「渣假的，我不收。渣真的，半文听。」

学腔的人噎住，转去别处起哄。起哄远了，他鼻下更静。静里听得见坊尾靴声两三次停顿——停顿像盯梢。盯就盯，他不追。追了像送货上门给钱戾衡。

管事收市锣响，锣硬：「散！明日照开。」

「照开。」韩泥收摊。木牌收进怀，贴着预选帖，一歪一正，硌得肋骨发麻。麻得好——麻着才记得：试水过了不算完，栏角灰还缺，兄弟线还悬。

回丙九路上，沈枯芽塞来半片新叶，仍无字。韩泥看叶脉，脉细，像提醒叶汤尚烫：「叶在。我不催。催了像讨。」

芽儿鞠半躬就跑。跑得好，跑了才不像绑在丑摊上。

刘婆灶边喊他：「吃饭！闻了一天，肚子空了鼻子也飘。」

「来。」他应。粥仍多半勺，勺里姜多。姜辣一口，辣出明日修栏的影——影里有铁无言，有灰，有「只站不叫」。

他摸怀中木牌，牌角磨毛，毛像三日风。风改了一半，另一半还在东角栏上挂着，等他去续。

山门风起，棚帘轻响。响声像钩——钩住铁无言臂上那道旧伤，也钩住他还没说出口的「续」。续不在嘴，在灰，在栏，在站。站侧去了，袖风才接得住。风再紧一寸，东角栏木就会响；响了，并肩那半步才算真续上，不叫一句空话。
