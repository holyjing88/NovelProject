# 第五十一章 先验商队

三日后，卯时，廊下霜白。

丙九列队，末排先验。韩泥立最前——前不是荣，是险；险在「滚」字，滚了，麸皮停，记功抹，七笔恩断。

管事敲锣两声，廊静。秦霜立廊角，木鱼不敲，只托：「辨香半刻，报香一字不落。漏一字，记过；记过两次，滚。」

香囊三只。韩泥闭息，鼻先静。粟壳淡，丹烟苦底，第三囊甜腻藏腥——腥在底，不在面，像栖香谣散了，散成规矩。「粟壳淡，丹烟苦底，甜腻藏腥。」他声哑，声平。

秦霜「嗯」一声，不夸，只记簿。铁无言立柱边，验他鼻下静气，声低：「静，别贪快。贪快，香飘。」

半刻尽，韩泥一字未漏。末排先验过——过了，不是灵根好，是鼻下稳。秦霜目光扫到队尾赖福，声冷：「再嗓，记名。」赖福哑了。

管事贴新条：「辨香过，候凝丹试手。试手在三日后。这三日，仍搬柴。」韩泥接条，条薄，真——真在「仍末」，真在路未断。

午后，韩泥担柴过宗门外侧门。侧门窄，窄得只容半眼，半眼够看驿道。

尘起。西驿商队过，驼铃远，车辙深。五脉旗号杂：佛门灰、密宗赭、西教白、密院黑、沉丹青——青是主场，杂是客。齐不要紧，要紧是界；界在门内门外，门内末排，门外商线。

驼队里有人咳，咳得寒，像北风贴骨。商队却不停——停不得，停，货腐，命也腐。

佛门灰旗前，有人施粥，粥稀，热。热在客路，不在丙九棚。密宗赭旗后，褐袍人坐驿亭，坐得不急。急便邪，不急，像等冬寒够深。西教白旗风里猎猎，猎猎不是谣，是界。密院黑旗压车辕，车沉，沉得像克扣的麸皮。

老耿挑水路过，水洒半勺，洒得实：「五脉齐，冬里货紧。紧，别凑秤。」

「不凑。」韩泥说，「凑秤，手飘。」

老耿又指驿亭：「那人等冬。冬深，才赠。赠，不白受。」

韩泥瞥褐袍人袖里黄纸角露一线，露，像凡物藏锋。密宗线不在村庙了，线在宗门外。他不凑亭。凑，像贪；贪，恩易断。

第二趟搬柴，侧门霜白。褐袍人抬眼，眼淡，淡得像凡物看人——不像看仙缘，像看骨寒。韩泥独眼平，平过去。平，是守。

商队领队验秤，秤杆落盘，落得像叶家那杆。韩泥旁站，不凑，只记：商线在，心不乱。领队扫他一眼，扫过，像看尘，不看人。

黄昏，丹房外，铁无言翻簿，见他一眼：「鼻下稳，三日后再验手。手抖，仍滚。」

韩泥只答：「手记着七笔，不抖。」

秦霜声从远廊来：「五脉商队在，宗门戒躁。躁者，借商线生谣——谣借腥，你别沾袖。」

同批少年碰他袖：「你真过了鼻子？」

「过了。」韩泥说，「过鼻子，不过命。命在坛。」

赖福远处讽：「看商队能看出仙缘？」管事声冷：「再嗓，克半月麸皮。」赖福哑了。

夜，韩泥对坛，声低：「第一关过了。试手前，冬来了。」

坛腹温一线，应「香」字，不应慌。心里七笔分列，不混。混了，手就飘；不混，手就正。

外头驼铃余音散尽，褐袍人影还在驿亭——在，像密宗线从村庙挪到宗门，挪得不急。柱上旧条作废，新条写「三日后试手」。路通向凝丹，通向还。还，先还烫。

刘婆门外咳：「试手前，别空肚。空肚，手抖。」

韩泥应：「不空。粥在，姜在记里。」

丑时，坛腹又温一线，温短，像瓮听辨香过耳。过耳，不应灵，应规矩。外头风硬，柱上试手条却更真——真在末排，真在验手，真在「滚」字后那一条窄路。

他低声道：「三日，试手。试得过，关才开第二道。」

木牌在枕下，与试手帖并放。并放，像命与规矩挨着。挨着，才验得过。

管事嘱：「商线腥，袖洗三遍。洗不净，别近坛。」他近坛，不近亭。袖洗三遍，三遍净了，胸才贴坛；坛应一线温，不应商线腥。

夜雪粒细，落肩即化。化不要紧，肩沉，沉得住，冬深才赠。商队铃远去，五脉线在界外；韩泥不借腥，心在界内。界内，手别飘——飘了，试手滚；不滚，路还在。

天边白一线，白像试手前三日也白了半分。驿亭褐袍人未走，黄纸角仍在袖里——在，像冬深那一赠还在候。候不要紧，要紧是三日后手正。手正，才验得过凝丹；凝丹，先还烫。

同批少年窃语：「末排也配先验？」管事冷眼：「再嗓，滚。」窃语散了。韩泥木牌在怀，帖在袖——袖里旧条作废，作废，像昨日数完。数完，今日只验鼻。

辨香前，秦霜又嘱：「温短贴胸，腥长沾袖。沾袖的，洗三遍。」韩泥应声：「洗三遍。」三遍净，胸才贴坛；贴坛，不应商线腥。

铁无言验他袖，袖净，鼻下静。静，像辨香那日廊角——廊角过，路还在。过路，不在舌；舌响，手就飘。

午后第一趟，侧门风硬，他肩柴停半息。停，不是懒，是听——听驼铃，听界。界在，心不乱；心不乱，手就不飘。

密宗赭旗后，褐袍人袖里黄纸角又露一线。露，像凡物将出，不出急，急像邪。他不凑亭，只记：冬深才赠，赠不白受。

辨香毕，同批少年围上来，问香囊里第三囊是啥味。韩泥不细说，只答：「甜腻藏腥。藏，不在面，在底。」少年不懂「底」，却懂「藏」，闭嘴散了。

管事又嘱：「商队过，袖腥三遍。三遍净，才回棚。」韩泥记在心里，回棚先洗袖。水冷，指缝腥洗不尽，洗的是面，不是底——底记着七笔，底才正。

第三趟搬柴，侧门风硬，他肩柴停半息，听驼铃余音。铃远，界还在；界在，袖洗三遍，三遍净了，才近坛。坛应一线温，不应商线腥。

佛门施粥那锅，粥稀，热在客路。热在客路，不在丙九棚——棚里粥也稀，稀在刘婆第七笔里。第七笔在，心不乱；心不乱，手就不飘。

铁无言第二趟巡廊，不问他商队看了啥，只验鼻：「鼻下静，商线才不乱。乱，手就飘。」韩泥应声：「静在记里。」记里，七笔不混；不混，手就不飘。

邻床夜间问：「五脉商队，跟你啥关系？」

「没关系。」韩泥说，「关系在界。界内末排，界外商线。线过，袖洗；袖净，坛近。」

少年不懂「界」，却懂「袖洗」，闭嘴睡了。

丑时第二遍，他近坛不拜，只验席严。席严，坛才肯应温；应温，不应商线腥。腥在袖，不在心。心在，符才接得住——符未至，怀仍空。空不要紧，沿净，席严，才接得住后头缘。

同批少年夜间又问：「商队跟你啥关系？」韩泥不答。答轻，恩重。他低声道：「关系在界。界内守坛，界外商线。」

管事夜巡验袖，鼻下嗅三遍：「袖净，好。」韩泥不辩，只按席。按实，像把「不借腥」按进肩里。

铁无言第三趟巡廊，声低：「商线过，袖洗三遍。三遍净，才近坛。」

同批少年夜间碰他木牌：「三日后验啥？」韩泥只答：「验手。手正，丹才不歪。」少年不懂，却闭嘴——闭嘴，廊下更静；静里，试手条才真。

柱上试手条在风里贴实，贴实，像倒计时。倒计时在柱，七笔在心，木牌在怀——怀在，路在；路在，恩不断。

他独眼平，手不抖，低声道：「商队验讫，铃远去。线在界外，心在界内。三日后，试手。」

夜雪又细，落肩即化。化不要紧，肩沉，沉得住，三日后手才正。风息了，符未至，怀仍空——空不要紧，沿净，席严，才接得住后头缘。

赖福廊下过，韩泥只答：「在。」在，就不滚。

柱上条角白一线，三年约在怀。怀凉，掌温。
