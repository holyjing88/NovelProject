# 第六十二章 席盖更严

瓮醒前夜，风硬。硬像最后一道冬关。关不过，缘不满。缘满，才误触那一滴。

韩泥清栏归来，掌茧新裂，裂口渗血，血珠饱满。他甩手，血甩破席，席吸尽，沿仍净。

他不主动滴血，不揭布诱沿。只用活计扎出的血喂席，席挡沿，沿等误触那一瞬。

邻床笑：「坛角盖那么严，藏宝？」

韩泥不辩，只把席盖又加一层。层厚，厚像秘密。秘密是坛，坛是命，命跟了五十八岁。

管事查棚，灯晃坛角：「杂物收起。」

韩泥只答一声，席盖不动。动像心虚。不虚，是守。

管事盯半息，走了。只见破席，不见仙缘。不见不要紧，沿要净——净了，管事才肯记「季末记功在」；功在，饭才正，恩才不断。

清栏第二趟归来，掌茧又渗血，血珠饱满。他甩手，血甩破席，席吸尽，沿仍净。他加第三层破布，布更旧，旧更能吸。层厚，厚像钩——钩瓮醒，钩在沿，钩在误那一甩。他不主动滴，不揭席诱沿；诱沿像借腥，腥借来，路易断。

夜，符怀暖一丝，护骨寒，手不僵。僵，明日藤刺偏。不偏，缘才正。

他低声对坛：「明日或后日，藤刺还会扎。扎了，血还会来。来席，不来沿。沿上，等你认的血。」

坛腹温一线，贴掌，贴到心口也暖半分。暖，不像火，像认亲前的试探。

更鼓三响，响沉。丑时将至，像丑时坛温那一线又要走一圈。

走一圈，走到编筐扎掌，走到血误触席，走到席渗沿，走到沿颤，走到瓮醒。

他合眼，不睡实。符在怀，坛在角，席盖严。

他低声道：「席盖严了。严了，才接得住那一误。误了，根在。根在，路长。」

坛不应字，只温，温得像答应，像候。

刘婆端姜汤，姜薄，热：「前夜别空肚。空肚，手飘。」

「不空。」韩泥喝尽，只点头。姜辣醒指，指醒，茧才不白裂。他低声对坛：「席盖严了。守得严，韩家不散。」坛不应字，只温，温得像答应。祖父韩老栓嘱过：「坛在，韩家不散。」嘱在耳，不在舌；舌急，血偏，偏了席挡不住。

赖福讽：「盖席盖出仙缘？」

管事声冷：「再嗓，克半月麸皮。」

赖福哑了。哑，韩泥加席层的手却更正。

同批少年碰他袖：「盖那么严，坛里真有宝贝？」

「有。」韩泥说，「有命。命跟坛，坛跟缘。」少年缩手，像被「命」字烫到。他不辩宝贝，只把第七层破布角压实——压实了，沿前多挡半寸风。风硬，符在怀，符暖护骨寒；骨不寒，明日藤刺才不偏。

秦霜远廊经过，停半息：「前夜别空肚。」她不再言，木鱼在指间，不托，不敲——像规矩还在耳，像正测三关还在柱上。

铁无言门外脚步停半息，声低：「沿净？」

「净。」韩泥说，「血在席，不在沿。」

「不主动滴？」

「不滴。」韩泥说，「滴，要等活计误那一甩。」

铁无言点头，脚步远。

韩泥又加一层破布，布旧，旧更能吸。吸血，挡沿，沿等韩氏守瓮血那一误。

邻床问：「盖那么严，热不热？」

「热不要紧。」韩泥说，「沿净要紧。净了，才接得住瓮醒。」

他按心口，低声道：「丫头那碗汤，热着还。还，要等瓮醒后，路才长。」

坛腹温一线加剧，加剧，像明日就在沿边等那一滴守瓮血。

误触那一滴，缘才满。满，不在心急，在活计，在席，在沿。

他独眼平，手不抖：「守住了，才配醒。醒在缘，不在舌。」

编筐第三刻，末藤硬，刺偏，他仍偏手，刺浅些。浅些，血仍来。血落席，席边湿，沿不接。不接，像候缘。

老耿挑水路过，声哑：「前夜别拜。拜，路断。」韩泥应声：「不拜。席盖严，不主动滴血。」老耿点头，水晃正，正得像告诉他：沿净，路就不绝。沿净，比拜邪实——实路走六十日，走到瓮醒前夜这一更。

邻床问：「误不了咋办？」

「误不了，就等。」韩泥说，「等，像守瓮人。」诱沿像借腥，腥借来，路易断；他不诱，只守席，守沿，守不主动滴血。

管事验筐：「季末记功在。手正，沿还净？」

「席边湿，沿仍净。」管事记簿，走了。管事走后，他把破席角又掖进坛底，掖得像给沿多挡半寸风。风硬，符在怀，符暖护骨寒——骨不寒，明日藤刺才不偏。偏了，血落错地方，缘就薄。

管事第三趟查棚，灯晃坛角，只见破席，不见仙缘。不见不要紧，沿要净。净了，他加第八层角，角旧，旧更能挡——挡到沿前，沿不接，缘才满。

午后，搬柴一担，肩勒旧印，印深，像丙九五年。肩沉不要紧，手正才要紧——正，明日藤刺才不偏。柴担落地，他掌心又裂一线，血珠饱满，甩手，血甩破席，席吸尽，沿仍净。净，是六十日守出来的；守出来，才配瓮醒前夜这一更。

外头更鼓又三响，响沉。沉里，瓮醒将至。将至，沿要更净，手要更正，席要更严。最后一夜，风硬，符暖，手正。正，才配明日藤刺再扎那一误——扎了，血找席，席挡沿，沿等认主那一滴。误在缘里，不在舌上。

天边白一线，白像瓮醒就在下一更。他低声对坛：「席严，沿净，符暖。正了，才配你醒。」坛不应字，只温一线加剧，像沿边已等那一滴守瓮血。候，不在舌，在活计；活计误那一甩，缘才满。

刘婆第二趟姜汤搁棚柱，姜薄，热得钝；他喝尽，不空，手才不飘。第七层破布压上，压得像给坛角戴了冬帽。帽丑，厚才真。真在能吸活计血，不在藏宝。

同棚少年梦呓问「醒了吗」，他不答。答，像求；不求，只守。他把席边湿处又按干一遍，按得像把血关在席里，关在缘里。席湿，沿仍净——净是规，规比命长。

他独眼盯沿半刻，沿尘，沿滑，沿上无血。无血，才配韩氏守瓮那一滴落对地方。秦霜远廊下过，停半息：「手正，心别邪。」他不拜，亦不揭席。第八层角压实，角旧，旧更能挡——挡到沿前，沿不接，缘才满。

丑时前，更鼓又沉。他不睡实，独眼盯沿——沿净，沿上无血。铁无言最后一嘱：「沿净，不主动滴。滴，等藤刺误那一甩。」韩泥只答，手不抖。木牌在怀，七笔分列。还，先还烫——烫在叶丫头，不在这一更。祖父嘱在枕下，嘱短，字丑，丑在还能读：「误了，根在。」根在，路才长。坛温一线加剧，像沿边已等那一滴守瓮血。

席严，沿净，符暖，手不飘。更尽，沿颤，瓮醒——醒在缘，不在舌。缘满不在心急，在活计误那一甩。手不飘，缘才满。

丑时前第二刻，邻棚有人烧纸，纸灰飘过一缕，飘到沿边即止。止，像沿也认界——界内席，界外尘；尘不落沿，沿才不脏。脏沿，像借腥；腥借来，路易断。他不辩，只把第八层角又压实一分。压实，像给明日藤刺留一条正路。

同批少年夜间又问：「瓮醒了你怕不怕？」「怕。」韩泥说，「怕恩断。恩不断，怕也要守——守席，守沿，守不主动滴血。」少年不懂「恩断」，却懂「怕也记」，闭嘴睡了。睡了，棚里更静；静里，更鼓沉，沉得像瓮将醒未醒，醒在缘，不在人催。

丑时前第三刻，他摸祖父嘱，嘱短，字丑，丑在还能读：「误了，根在。」根在，路才长。路长，才还得上叶丫头那碗烫。烫在还路前头，不在这一更。

他独眼平，手不抖，低声道：「席盖严了，沿还净。下一更，藤刺还会扎——扎了，血找席，席挡沿，沿等认主那一滴。」
