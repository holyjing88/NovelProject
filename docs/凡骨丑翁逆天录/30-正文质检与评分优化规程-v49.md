## 二、硬闸 P0（不过则 ALL_OK False）

### 2.1 字数闸（v48）

| 项 | 值 | 出处 |
|----|-----|------|
| 下限 | **2000** 汉字 | `prose_utils.TARGET_LO` |
| 上限 | **2500** 汉字 | `prose_utils.TARGET_HI` |
| 理想参考 | **2250** | `TARGET_IDEAL`（非硬闸） |

- **不足**：补**戏内独段**（对话 / 小爽 / 情感锚 / 钩），禁模板叠句凑字。  
- **超标**：先删复读与末段叠收墙，再删弱过渡。

### 2.2 句级重复 dup

- **算法**：按 `。！？` 分句（≥8 字），完全相同句出现两次即计 dup。  
- **硬闸**：单章 **dup < 2%**（`dup >= 0.02` 即 HIGH_DUP）。  
- **目标**：全书 dup 均值尽量压至 **< 1%**，抬节奏/文笔十维。

### 2.3 结构与其他

| 检查项 | 规则 |
|--------|------|
| WALL | 单段（段内无空行）**≤ 180 字**；超标扣 0.3 分并标 WALL |
| 破壁 | 禁「下一章」「这章」「读者若问」等元叙事 |
| 禁词 | 禁 **恩册 / 恩簿 / 记账翁**（改用「记恩」「心记」「分列」等） |
| 章末 | 必须有 `**状态**` 行（在 `---` 章末脚注区，勿进正文） |
| 乱码 | 无 `\ufffd`、无控制字符 |

## 三、分章评分规则（`_score_full.py`）

**基准分 9.9**，扣分项：

| 条件 | 扣分 |
|------|------|
| dup ≥ 10% | -1.2 |
| dup ≥ 5% | -0.8 |
| dup ≥ 2% | -0.4 |
| META_NARR 破壁 | -0.5 |
| WALL 长段 | -0.3 |
| 字数 <2000 或 >2500 | -0.5 |
| ch1-3 且 dup>0 | 额外 -0.3 |

**封顶加分（v50 · 通俗笔锋高峰）→ 10.0**：同时满足——

1. 扣完后仍 ≥9.9，且 **dup=0**、无 WALL、无 flags、字闸达标  
2. 章末脚注含 **通俗笔锋**  
3. 正文对白「」≥12 处  
4. 末段命中 ≥1 个 `HOOK_KEY`（公审/帖/还…）

**目标**：普通合规章 **9.90**；白话高峰样板章（如 `ch014`）**10.00**。

## 六、修稿铁律（本方案核心）

> **仙凡笔锋 v2**（2026-07-13 · 历史统一稿）：见 **§六附**（仅旧稿参考）。  
> **通俗白话笔锋**（2026-07-14 · **现行新稿标准**）：见 **`31-通俗白话小说笔锋`** · **§六附B**；样板 `prose/ch014-村议辱叶.md`。  
> 新写 / 点名改笔时，v49 叠收末段律与 v2 电报体 **均让位于 `31`**；P0 硬闸（字闸/dup/WALL/破壁）不变。

### 6.1 顺序：先删后补

```
① 跑终检定位问题章
② 删末段叠收墙 / 跨段复读句（模板句优先）
③ 合并重复场景（同人物同功能只留一处）
④ 若字数 <2000：补独段戏（对话/小爽/情感锚）
⑤ 若 WALL：按句拆段（>180 字无空行）
⑥ 复跑终检至 ALL_OK
```

**禁止**：未去重先凑字；末段模板三连（「稳。等。」「帖在/沿净」「四恩齐/六笔齐」叠收）。

### 6.2 补字有效手段（优先级从高到低）

1. **独段对话**（叶青禾/芽儿/陈姑/里正/买主——每人一句不可替代）  
2. **小爽实锤**（卖筐多文、税吏少收、赵退半步、赖爪牙骂走）  
3. **情感锚**（趁热、先还汤、分列恩辱仇、公审前自检）  
4. **单线末钩**（冬前村会 / 八日后上路 / 荐帖风声 / 等锤）  

**禁止**：同义短句排比凑字数；「真就不白收」「还定了」「四恩记实了」末段复读墙。

### 6.3 硬锚与口传链（勿乱改）

| 类型 | 内容 |
|------|------|
| 口传链定稿 | `这是你爷爷传我的，我也这么传你` + `坛在，韩家不散` |
| 里程碑锚 | ch024 公审 · ch030 辞村 · ch035 必还 · ch062/ch063 瓮醒钩 |
| 自检锚 | ch010 四恩齐 · ch030 六笔恩可答 |
| 策划对照 | 章末脚注 `（对照 05 §…）` 勿删改锚点语义 |

### 6.4 文笔辨识度

- 保留「短句—像—」韩泥笔法，但**同章内喻体不重复**。  
- 对话占比高的章（公审/村议）优先保对白，少内心排比。  
- 拆 WALL 时在**句号后空行**，勿 mid-sentence 断段。

### 6附.1 章结构（四段式）

| 段 | 要求 | 示例 |
|----|------|------|
| **景起** | 首段 1～3 句：时令/天气/村景，再入人事 | 「冬前。」「天又冷了。」「谣起第三日。」 |

### 6附.2 笔法

- **短句—像—**：保留韩泥体感喻（风像针、桶底声闷），**同章喻体不重复**。  
- **对话优先**：暖/还恩用对白（「趁热」「记着」「别谢」），少内心排比凑字。  
- **压释**：辱后 500 字内须有暖/硬扛/小爽（`07` §压释 · 与 v49 同）。  
- **段落**：单段 ≤180 字（WALL 硬闸）；景起段可略长，对话段宜短。
