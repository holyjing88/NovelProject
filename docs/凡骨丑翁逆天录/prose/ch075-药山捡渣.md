# 第七十五章 药山捡渣

卯时前，药山边门开一线。边门窄，窄给杂役，不给内门。

内门弟子走主道，主道香冷；杂役走渣道，渣道苦，苦才真。管事弟子验牌：「丙九韩泥，捡渣半日。只捡，不采。采了，记名。」

「只捡。」韩泥应。

牌在，名在，路在。赖福立门外，眼细：「药山也给你？末席专捡晦气渣。」

韩泥不辩。辩费嘴，嘴要留着分拣，也留着守坛。药山坡下废渣堆三层，灰黑里偶有一点亮——亮不是宝，是未烬的药性。风过，灰起一线。韩泥鼻下静，静到只闻辛淡，不闻辱。辛淡正，他指稳，拣一撮入筐。筐底垫粗布，布丑，不漏。

第二层渣堆更腐，腐沉底才真。真在底，不在浮；浮的少，少也要记——记错了，鼻下就飘。铁无言立坡角，嗓子压低：「捡渣别贪。贪，像求。不求，只记。」

韩泥分拣，分出三堆：辛、腐、假香。假香浮，浮也要剔——剔净了，手才稳。铁无言又取半片枯叶，叶上沾灰：「苦浅，像叶家药。药在远，恩在胸。」

韩泥鼻下一静，静里辨出苦与辛。苦沉，辛浮。浮少，记牢；记牢了，嗅诀才生根。坡滑，霜白。霜下碎石响半声。脚轻才要紧——脚重了，界碑那一步就踏错；踏错贪活药，除名。

午后日斜，药山深雾起一线。雾不怕，他仍只捡边堆——边堆许可，深堆不许；不许，就不进。筐满半寸，半寸也是路。路通向废渣炼丹，通向往后还叶丫头那一碗烫。白织月主道过，袖风带香，香冷：「杂役也碰药山？碰了，也炼不出丹。」

韩泥不抬头：「碰渣，不碰丹。丹在远，渣在近。」

白织月唇线绷一息，走了。少一句嘲，小胜。管事验筐，筐底无活草，无整药，只有渣。渣对，记一笔：「丙九韩泥，捡净。」

捡净是小胜：胜在筐，不在嘴。赖福讽远：「捡渣也捡，末等也末等，三年仍滚。」

韩泥不接茬。接茬饿一整排。「滚字在柱上，」他低声道，「不在筐上。」

筐在，渣在，路在。二层气走指梢，指梢温半分，辨渣更细——细了，辛腐才分得清。回棚前他在药渣场多留半刻，把白日样再辨一遍。样少不要紧，要紧是记。记辛，记腐，记假香浮。样纸包实，塞怀。怀暖半分，暖不应符，应「记」。记牢了，往后深行才走得稳。

邻床问：「药山累不累？」

「累。」韩泥说，「累也捡。捡了，才有渣练。」

少年不懂「渣练」，懂「累也捡」，闭嘴睡了。他按席角，席严，坛不露。坛腹温一线，贴掌，像应「捡」字。丑时前他廊下练半刻闭息。息在鼻，不在嗓。嗓响，香飘；香飘，手就滚。

掌纹那路转半周，二层境稳，稳像筐底那撮辛——沉在底，不浮。浮了像求；不求，只记。他对坛低声：「渣捡了。辛记了。深行在明日，别贪。」

坛不应字，只温。温像把「渣」字摁进掌纹那路——路在，就不滚。刘婆端粥，仍多半勺：「药山去了？」

「去了。」韩泥说，「只捡渣。」

「只捡怕什么。」刘婆哼，「手稳，饭稳。饭稳，恩不断。」

韩泥点头。恩在粥里，也在筐底那撮灰里——灰轻，记重；记重了，才接得住深行。秦霜远廊过，木鱼不敲：「心别邪。邪了，鼻下香飘。」

「不贪。」韩泥说，「只捡。」

更鼓沉。他摸怀木牌，牌凉，筐渣凉，坛温一线。温应活，不应慌——慌像少年，他是记心翁。独眼平：「渣捡了。深行在明日。嗅诀在鼻下，不在舌上。」

刃不急着亮，先记。记完了，才配药山再深一步，才配白织月再嘲那一日。三年约在怀，愧字在怀，渣筐搁角里——藏着，不示人。示人，谣又来。坛腹又温一息，像药山边堆那一点辛进了活里。活在，才嗅得到；嗅得到，丹路才开。

「仍只捡，不采。」他对自己说。「采了，记名；记名，恩断。」

恩不断，才还得了叶汤，才接得住门缝灯。沈枯芽在药圃角弯腰，弯得像没看见。韩泥也不近——近像求；不求，只记半块灵石那笔恩在还，不在这一日。散队后兽栏仍腥。腥不怕，二层在掌，铲下更稳。

管事另记一栏「药山」，笔尖停半息：「捡净——写上。」写上，像把晦气也记成一笔活；活记在册，滚字才不好随便抹。同批少年掩鼻：「药山渣也臭。」

「臭不要紧。」韩泥说，「臭也捡。捡了，鼻下才有根。」

「渣味。」他又补一句，「睡吧。」

在，就不滚。不滚，渣才捡得长——捡长了，废渣路才走得通。药山第二趟，坡角风硬，硬里苦辛。韩泥筐沿压布，布厚，厚了灰才不漏。漏了像求药性；不求，只捡。管事弟子立界碑旁，声沉：「再深半步，除名。」

「不越。」韩泥应。

不越，是规。规在，手在。手在，就不滚。第二筐满，他歇半息，歇不像懒，像鼻下静。静里辛沉底，假香浮，浮少也要剔。剔净，铁无言点头：「捡渣如辨性。性在底，不在嘴。」

韩泥心里记下这一句，记下边堆，记下二层境配指稳。白织月主道再过，裙素袖香冷，瞥筐一眼，瞥完走了。没嘲，小胜。赖福立门外又眼细：「捡渣也捡，末等也末等。」

韩泥不作声。舌头留来辨鼻，留来守坛。午后回坡交筐，管事记「丙九韩泥，药山捡净」。刘婆提篮过，篮里饼硬：「药山饿不饿？」

「饿。」韩泥说，「饼就行。」

刘婆塞饼，饼糙：「别谢。记着。药山别贪活药。」

他应一声，收下。恩在饼里，也在「别贪」二字里。回棚样纸包实塞怀。邻床问：「药山香不香？」

「香在灰里。」韩泥说，「灰在活里。活在，手才稳。」

丑时前廊下再练闭息。低声道：「渣捡了。深行在明日。界碑不越，恩不断。」

坡下归途过药渣场角，风硬，他多留半刻，把样纸并入白日样。样轻，记重；记重了，深行才走得稳。同批少年掩鼻：「你身上咋有药味？」

「渣味。」韩泥说，「味在灰里，不在身上。」

秦霜远廊过，木鱼轻敲：「捡渣心别贪。贪，鼻下香飘。」

韩泥不答，心里只留四个字：不恋，只捡。坛腹应温，像应「捡」最后一笔。应完，他不叫。不叫，才守得住深行那一日。柱上条角白着一线，三年约在怀。怀凉，筐渣凉，掌里二层温——温在暗，不在众目。

第二趟出山前，他在界碑旁蹲一息，用指尖蘸点边堆湿灰，在碑脚划了一道浅痕。「痕浅，」他对自己说，「过痕一步，就是贪。」

同批少年扛空筐过来：「韩叔，你划啥？」

「记界。」韩泥说，「记了，才不迷。」

「迷啥？」

「迷香。」韩泥站起，「香一迷，手就采。采了，名就没。」

少年咋舌：「捡渣也讲究？」

「讲究。」韩泥拍拍筐沿，「渣里有饭。饭里有恩。恩断了，谁还叶家那碗汤？」

少年不接「汤」字，却把筐绳勒紧了些。勒紧也好，像把贪勒死在绳结里。韩泥领他走边道，不碰主道，也不碰深雾。雾里有人影一闪，像白织月袖角——闪过就没了。没嘲笑，也是一天的运气。

交筐时管事翻底：「无活茎，好。再有一回掺活药，丙九连坐。」

「无。」韩泥说，「只有渣。」

「只有渣才像你。」管事难得不阴，「滚字还在柱上，你先靠手稳住。」

韩泥看着柱上那「滚」字，点一下头。点头不是服气，是记账——账本里又多一笔：渣捡净了，深行才能写。写之前，先把鼻下那根再养粗一分。粗一分，明日雾深一尺，他也不慌。

他把木牌系紧一线，系得像把「仍末」摁进掌心。更鼓沉，牌凉贴胸。辛记了，界碑不越。明日再往深一步，仍只捡，不采。采了断恩；不采，路才长。路长了，叶汤才还得热，门缝灯才照得到。
