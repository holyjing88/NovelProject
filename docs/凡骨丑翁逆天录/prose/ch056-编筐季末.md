# 第五十六章 编筐季末

试手前二日，丙九编筐季末。韩泥接末单，藤条硬，硬得像冬里的规矩。

管事盯梢：「末单交不出，试手名单划掉。」

韩泥不辩，只编。编筐是本行，深秋末筐也是本行。筐成，手稳，稳比美重要。

藤刺扎掌，扎旧茧，茧裂，裂口潮，潮不是血。血不碰沿，沿才净。

赖福讽：「丑手编筐，筐也晦气。」

刘婆路过，端粥，稀，热：「手稳，筐就圆。圆，能装柴，能装恩。」

韩泥喝粥，只点头。热在，像第七笔又实。筐沿齐，齐像秤杆落盘。

管事验筐，指筐底：「底不塌，记功。」

记功，像窄路又宽半分。

午后，符怀暖一丝，护骨寒，手不僵。僵，藤刺偏。不偏，缘才正。

搬柴过半，肩稳，像筐成后的余韵。余韵不在夸，在手还在。

铁无言路过，声低：「季末了，手还在。在，才接后头几日。」

韩泥只答：「在。」

「试手验啥？」

「验手。」韩泥说，「手抖，丹歪，滚。丹歪不要紧，手不能歪。」

铁无言应声：「静带到手上。手上静，丹才正。」

同批少年碰他筐：「末单也编得齐？」

「齐不齐，管事验。」韩泥说，「我验手。手稳，筐就圆。」

少年碰筐沿，沿不塌：「晦气筐，倒结实。」

韩泥不辩。结实，像小胜。胜在实，不在嘴。

黄昏，坡下风硬，柱上条在：「后日卯时，试手凝丹。」

后日，比五日条又近一步。近不要紧，要紧是手还在。

管事验末捆，声沉：「季末了，藤刺还会扎。扎了，别硬扛。扛裂，手抖。」

韩泥只答：「不硬扛。扛，路断。」

赖福偷懒，管事记名。韩泥不偷懒。克粮，克不走还路。

编到第三筐，末藤刺深时，韩泥停半息，停，像听藤条说话。落盘稳，谣压半寸。刺深，裂口渗潮，潮不是血——无血，沿便安；安，才接后日坡下。

第四筐，筐沿齐，底不塌。管事验过，笔尖停他名上：「季末记功在。」

韩泥不谢。谢轻，记在木牌边。

第五筐成时，同批少年偷懒被记名。韩泥仍编，藤刺再扎，茧裂无血。无血，像给后日试手又钉实一钉。

夜，坛腹温一线，像应筐成。成，不急醒。醒要等缘，不在筐成。

他低声道：「季末了，手还在。在，才接血近沿那几日。近，不碰沿。」

坛腹温一线，又凉。外头更鼓沉，沉里，试手条在柱，柱上「后日卯时」。

后日，凝丹试手。试手不是醒瓮。醒瓮在更后，在误触那一滴。

邻床问：「你怕不怕滚？」

「怕。」韩泥说，「怕也记。记着，手就不飘。」

「飘了咋办？」

「飘了，路断。」韩泥说，「路断，粥就凉。」

邻床懂「凉」，闭嘴。

筐是活计，活计扎掌，掌茧裂在冬，不在沿。

丑时前，裂口渗潮，潮不是血。无血，沿便安。他擦沿，沿滑，沿净。净，像祖父手还在。

刘婆门外咳：「后日试手，别空肚。」

韩泥应：「不空。粥在，姜在记里。」

符怀暖一丝，护手不僵。僵，后日藤刺便偏。

他低声对坛：「筐成了，手还在。还在，才接后头试手，接血近沿，接瓮醒。」

外头风息，柱上「后日卯时」更真。真在试手，真在末排，真在窄路。

季末筐成，底不塌。底不塌，像秤杆回正——回正，手才配后日试手。

后日卯时，试手凝丹。凝丹验手，不验灵色。灵色末，手不能末。

第六筐成时，藤刺偏深半分，他停半息，听藤条响。响闷，闷像更鼓。裂口渗潮，潮不是血——无血，沿便安；安，才接后日坡下那一验。

第七筐，筐沿齐，底不塌。管事验过，记功。记功在，麸皮不克。

编筐末刺深时，他偏手，刺浅些。浅些，血仍不来——不来不要紧，手要在。手在，后日丹才不歪。

刘婆季末最后一碗粥，热：「后日试手，鼻下别飘。」韩泥喝尽：「不飘。粥在，静在记里。」

铁无言路过，声低：「后日试手，静带到手上。」韩泥只答：「带上。带上，丹才正。」

邻床问：「编筐也试手？」

「编筐练手。」韩泥说，「手稳，筐圆，丹才不歪。」

后日条贴柱，韩泥摸条，条薄，真。真在「仍末」，真在「卯时」。

筐底不塌那日，管事声沉：「季末记功在。」韩泥不辩，只把手擦干。

回正的手，明日才配站上坡下末排——帖在柱，名在帖，卯时在。掌茧裂无血，无血，沿才净。净了，血线才找得到席。席盖严，沿在前。手在，缘才正。

丑时前，他涂药最后一遍，药凡，凉后稳。稳，像把「仍末」两个字摁实了——摁实，不惧验脸，只惧手抖。

铁无言前夜最后一嘱：「沿净，别涂药上沿。」

韩泥只答：「沿净。药只涂裂，不涂沿。」

坛腹温一线，像瓮听掌茧稳了。仍眠，不等醒。醒要等误，不在筐成。

五更将尽，坡下霜白，石阶滑。滑不要紧，站要稳。稳，才站得住坡下末排——末排不要紧，手正才要紧。

季末记功在，窄路又宽半分。宽，不是灵根好，是七笔没断。断了，粥就凉；不断，后日卯时条才真。

后日前夜丑时，他数更鼓。更鼓沉，掌茧裂无血。无血，沿净，手才配坡下。坡下空，空得像白纸上等明日那一笔——笔在执事手里，手在他掌上；掌上凉后稳，稳，才验得过凝丹。

第八筐成，筐沿齐，底不塌。管事验沿：「齐。」齐，像秤杆落盘。落盘，手才稳。季末最后一刺深时，他停半息，听藤条响——响闷，闷像更鼓；更鼓里，茧裂无血，无血，沿便安。

入库前，管事用尺量筐口。尺短，口圆，圆得像秤杆落盘。韩泥立末排旁观，旁观看自己的掌——掌茧裂无血，血不碰沿，沿才净。净了，后日卯时，坡下才站得住。

同批少年碰第八只筐：「晦气筐，倒能装柴。」韩泥不辩：「能装柴，也能装恩。」装恩，不在筐，在手；手在，帖在，卯时在。

五更前，他摸柱上后日条，条角起霜。霜化，字更显——「仍末」二字，像被编筐季末又摁实一摁。摁实，不惧脸丑，只惧手抖。

坡下石阶将晚又起霜。霜白，像给后日试手铺一层白纸。纸上写字，字是「验手」——验手，不验灵色；灵色末，手不能末。

他低声对坛：「筐季末了，掌茧裂无血。无血，沿净。净了，才接后日坡下，接血近沿，接瓮醒。」

搬柴季末最后一担，柴湿，肩勒旧印。印深，像编筐练出来的稳。稳，才配后日坡下——坡下验手，不验灵色。

管事验末捆入库，声沉：「八筐齐，季末记功在。」韩泥不谢，只把手擦干。擦干，像把「仍末」摁回掌上。

后日前夜丑时，他数更鼓。更鼓沉，掌贴膝，膝稳，掌才稳。稳里，柱上「后日卯时」更像倒计时——计在柱，不在舌；舌急，手就飘。

刘婆季末最后一咳：「后日试手，姜在记里。」韩泥应：「在。在，才还烫。」还烫，先还叶丫头汤；汤在缘里，等瓮醒后路才长。

铁无言季末最后一嘱：「静气带上坡。」韩泥只答：「带上。带上，丹才正。」带上，像把编筐季末又摁实一摁——摁实，不惧脸丑，只惧手抖。
