# 第三十三章 公平买卖

秦商言连夜下山，驿站在山脚，马未卸，先吩咐下属。

「莫长老送的茶，」她道，「比灵石贵。记入百宝阁恩账，不记入买卖账。」

下属愣：「管事，无价的礼……」

「所以贵。」秦商言截话，「公平买卖，卖的是货；恩情，卖不了。」

她翻那卷东脉调动简，指节轻叩纸面：「三日后佯攻，七日后夜袭——青岚门若只守不攻，田还在，命没了。」

下属问：「若情报卖了别人……」

秦商言望青岚方向，笑：「卖就卖。商道是商道，恩是恩。各算各的账。」

下属又问：「若赤焰夜袭成真，商言赚不赚？」

秦商言笑深：「赚。赚在青岚若赢，百宝阁有路走；若输，商言也有恩可还——恩，是另一条商路。」

马嘶，夜风腥，像东脉的云已低头。驿外弟子偷听，偷听里惊，惊茶饼竟换夜袭简，惊百宝阁管事竟说「亏本乐意」。

---

同一夜，莫长春院中。

他展简，烛火映字，像映一张将到的刀。佯攻、夜袭、妖与田之间——秦商言没卖假情报，这礼还得干净。

柳青鸢遣人来问：「师兄，可是秦氏简？」

「是。」莫长春道，「宗主主战，简只是让战打得明白。」

来人退。廊下弟子窃语：「长老手松，连百宝阁都松？」

另一人答：「松的是茶，紧的是命。你看简上写什么。」

「写什么？」

「夜袭。」

窃语骤静，静里，怕意起，像三日之期提前贴到每个人心里。

霍镇山闻讯，骂声洪：「好！打得明白，死也明白！」

周德海在长老院听闻，笑僵，笑里像有什么在袖中湿了又干。

莫长春望简，笑而不辩坊间那句「长老手松」。手松给杂役棉衣，手紧给赤焰命门——世人只见松，不见紧，正好。

簿无新字，袖却暖，像秦氏茶香还在。

他低声：「公平二字，商言说得对。老夫偏不爱公平，爱还礼。」

杖中竹纹一游，气音轻。

弟子围简抄本，围得像看刀，刀上写：佯攻、夜袭。有人怕，有人燃，燃的是终于明白赤焰花钱在哪。

苏念慈在丹堂听讯，炉火旺，旺得像战意也旺。她低声：「七夜丹成了，东脉也要成。」

秦商言驿中写信，写恩账，不写买卖账，写得像亏本，亏得像赚。

魏无涯在檐角观简传讯，扇合，低笑：「茶比灵石贵，商言说得对。」

莫长春念，像钩，钩向东脉账本：

「东脉账本，该翻开了。周长老的表，该有人问问——第七年，谁守田。」

风大，烛焰直，像战意也直。

驿外弟子偷听，偷听里惊，惊茶饼竟换夜袭简，惊百宝阁管事竟说「亏本乐意」。有人低声：「秦管事亏？亏得像赚。」

另一人答：「你不懂。恩账比买卖账难还，难还才贵。」

孙福在角隅听简讯，面色白，像听见「第七年」三字，也听见自己的旧账将被翻。他嘀咕：「周长老守田……」旁人瞪：「你看简，别看嘴。」

外门弟子把简抄成七种版本，版本里都有「夜袭」二字，像把怕字写进每个人眼里。弟子问苏念慈：「首座，夜袭丹堂危不危？」苏念慈冷声：「危。所以炉不能熄。」

秦商言驿中写信，信末加一句：「商言欠莫长老半块茶，欠青岚一条暗线。此债，商言认。」魏无涯传讯厉无殇，只四字：「简已至。」厉无殇阅讯，焚，笑冷：「准不准，莫长春都要接。接，才是戏。」

顾小满在杂役堂浆洗，听驿站消息，指节白：「茶换简？简换命？」杂役师兄答：「你不懂账。秦管事记恩，不记卖。」

赵玄机在思过崖听风，风里有「夜袭」二字。他妒火冷一寸，忽然明白：莫长春手松给茶，手紧给东脉田埂——紧的地方，才是该守的门。

秦商言马队远去，远得像把恩账留在青岚，把买卖留在百宝阁——各算各的，才算干净。

风大，烛焰直，像战意也直。简在案上，茶香在袖里，恩在账上——公平买卖，卖的是货；还礼仙翁，还的是命。

弟子围简抄本，围得像看刀，刀上写：佯攻、夜袭。有人怕，有人燃，燃的是终于明白赤焰花钱在哪——钱花在田，花在命，花在子时那一刀。

茶饼入驿那夜，有人把故事讲成「秦管事亏了一笔买卖」，亏得像赚。莫长春在院中展简，简上「佯攻」二字像鼓，「夜袭」二字像刀——刀还没落，鼓先响，响得全宗睡不着。顾小满在杂役堂叠衣，听「夜袭」二字，指节白，像半块灵石在怀，温了一路，温成警。
