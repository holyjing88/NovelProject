# 第五十四章 坛温起伏

连雨一日，棚漏风。风进，坛席潮。

韩泥擦沿，沿滑，滑像祖父韩老栓的手。手不在，滑还在。坛腹午温，夜凉，晨又温一线。起伏，像瓮呼吸。吸，不应人；吐，不应灵。

邻床笑：「晦气翁拜坛，拜出邪脉。」

管事路过，声冷：「再嗓，滚去西栏清粪。」

邻床闭嘴。韩泥不辩，辱记心里，与恩分列。列清，心不乱。

雨歇，驿道泥浅。小商队过，两驼，铃轻。褐袍人起身，像等雨停后冬寒才够深。

韩泥搬柴，不凑亭。凑，像贪符。

老耿挑水，水洒实：「那人等冬。冬深，才赠。赠，不白受。」

韩泥应声：「白不受。受，记恩。」

午后，藤运到丙九，藤硬，硬像冬规矩。韩泥领末捆，管事验韧：「韧，编得出筐。筐圆，手才不抖。」

同批少年偷懒，管事记名。韩泥不偷懒。

刘婆端姜汤，姜薄，热：「雨潮，别空肚。」

韩泥喝尽。赖福讽：「丑手编筐，筐也晦气。」

刘婆哼一声：「手正，筐就圆。圆，能装柴，能装恩。」

赖福脸青，退半步。藤刺扎掌，扎旧茧，茧裂，裂口潮，潮不是血。

黄昏，铁无言廊下见他：「坛温起伏，别当醒。醒，要等缘，不等人催。」

韩泥独眼平：「不催。催，路断。」

「沿净？」

「净。」韩泥说，「血不碰沿，沿才接得住后头缘。」

同批少年碰他袖：「坛温起伏，你怕不怕醒？」

「不怕。」韩泥说，「怕也守。守着，沿才净。」

夜，雨又细。细敲棚顶，顶响像更鼓。

他低声对坛：「温凉起伏，像你眠。眠着，不急醒。醒，要等血缘碰沿。碰，不是今日。」

他摸掌茧，茧裂，裂口无血。无血，沿便安。

坛腹温一线，又凉。凉，像应「等」。

外头更鼓三响，响沉。沉里，褐袍人影仍在驿亭。在，像密宗线候冬，不急，急便邪。

丑时前，邻床梦呓：「……温……」

韩泥盖席，声低：「你看错了。睡。」

他按席，席厚。雨里小商队铃轻，像界。界内末排，界外五脉。

他低声道：「起伏不要紧，沿净要紧。净了，才接后头赠。」

符未至，怀尚空。空不要紧，褐袍人袖里黄纸边露一线，露，像凡物将出。

韩泥绕远，绕，像避腥沾袖。袖腥，洗得掉。心邪，洗不掉。

雨歇后，泥路仍滑。搬柴经侧门，他瞥驿亭一眼——亭里人仍坐，袖角黄纸毛边在风里微动。动，像凡符将出，不出急，急像邪。

管事盯梢：「坛边别拜邪。拜邪，滚。」

韩泥不拜。守沿，缘在。

同批少年问：「那人还不走？」

「不走。」韩泥说，「走，冬未深。深，才赠。」

管事验柴：「沉得住，记功在。」

刘婆端粥，稀，热：「雨停了，别空肚。」

韩泥喝尽，只点头。热在，像第七笔又实。

霜白那日，侧门窄。驼铃远，尘落。褐袍人仍坐，坐得像等冬寒再深半分。半分，像凡符出袖前的最后一试。

韩泥搬柴，肩勒旧印，印深，像丙九五年磨出来的实。他不凑亭，只瞥一眼。一眼，黄纸边毛，毛得像凡物。

铁无言廊角，声低：「坛温起伏，别当醒。醒在缘，不在舌。」

韩泥只点头。点头，像把「等」字按进掌上。

夜，坛腹午温，夜凉，起伏像瓮呼吸。呼吸不应人，不应灵。

他低声对坛：「你再眠。眠着，我守席，守沿，守不主动滴血。」

凉，像答应「等」。

外头更鼓沉，柱上五日条在。五日，试手在望。望不要紧，要紧是沿净，手正。

他独眼平，手不抖：「赠来仍凡，不越阶。越阶，像拜邪。」

雨又起那夜，棚顶响密，密像算盘落子。韩泥披旧袄，袄薄，薄能挡半分潮。潮进席角，他不挪席，挪了沿要再验。

邻床问：「坛温像醒吗？」

「像眠。」韩泥说，「眠着，不急醒。醒，要等缘。」

五更将尽，天边白一线。白里，驿亭灯灭，褐袍人影仍在。仍在，像冬寒还差半分才够赠。半分，他等得起。

卯时，管事验棚：「坛角杂物，收干净。」韩泥只擦沿。沿滑像祖父韩老栓的手——手不在，滑还在，还在，像血脉日常。

秦霜远廊经过，木鱼不敲，只托着。她停半息：「坛温起伏，别拜。拜，像借腥。」韩泥只答：「不拜。」

午后西栏风硬，粪锹沉，沉得像湿柴。湿柴沉，肩也沉。沉不要紧，沿要净。管事验沿，烛照一线：「脏一指，像拜邪。」韩泥用旧布再擦，擦得沿尘，沿上无血。

夜雨第三夜，邻床翻身，碰他席角。席角潮，沿仍净。净，像六十日擦出来的规矩。

老耿夜过侧门，桶沿轻碰栏边，声闷：「褐袍人候冬，冬深才动。动，是赠，不是邪。」韩泥不凑亭——凑，像贪符；贪符，像拜邪。

丑时前，木牌贴胸，胸口吃住七笔。七笔分列，不混。混了，手就飘。

驿道干了半寸，驼铃又轻。铃轻，亭里人仍坐，坐得像等霜再厚一层。厚了，黄纸才肯出袖。

赖福路过侧门，嗓尖：「褐袍人还不走，专候丑杂役？」管事冷声：「再嗓，克半月麸皮。」赖福退，退得像吞了苍蝇。韩泥不接话——接话，像邀功；功在沿净，不在舌。

管事验垛：「冬里沉得住，记功。」记功，麸皮不克。韩泥不谢，只把柴摞齐——齐，像给后头试手留一条正路。

同批少年编筐偷懒，管事记名。韩泥不偷懒，藤刺扎掌，茧裂无血。无血，沿便安。

编筐末藤硬，他仍偏手，刺浅些。浅些，茧仍裂，裂口潮，潮不是血——血要等活计误那一甩，误在沿前，不在舌。

坛腹起伏，呼吸不应人。不应人，像瓮仍眠——眠着，坛温才不外泄；外泄，像借腥。

柱上五日条与坛温并放着。并放，像命与规矩挨着——挨着，才接得住试手，接得住凡物将出袖那一日。

管事夜巡验棚，灯照坛角，照见净沿不见宝。不见不要紧，沿要净。韩泥不辩，只擦沿——擦沿，像祖父还在。

雨歇第七日，侧门驼铃又轻一声。袖里黄纸角又翻半分，翻完贴回——贴回，像有人在试这冬够不够冷。冷够了，凡符才肯落怀。

秦霜远廊又过，木鱼不敲，只一句：「商线腥，袖洗三遍。洗不净，别近坛。」韩泥袖已洗三遍，三遍净了，胸才贴坛。坛应一线温，不应商线腥。

夜巡最后一趟，风息半更，席盖与沿贴合一线。贴实，坛温才不外泄。他不催，像守瓮人——守，不等醒，等缘。

丑时，他把席角又压严一层，压得像给坛沿戴冬帽。沿净一寸，袖里黄纸才肯露角——露角，凡符将出；出，仍凡，不越阶。

铁无言夜巡嘱半句：「沿净，才接得住赠。」韩泥应声：「净。净，像守瓮人。」

同批少年蹭袖角，声小：「五日条上写试手，你怕不怕滚？」

「滚看手。」韩泥说，「手在，就不滚。不滚，才接得住后头凡符。」

少年似懂，收声走了。走了不要紧，五日条在柱上，试手在望——望不要紧，沿净才要紧。

轮换第十日，柱上五日条角起霜，霜化字更真——真在沿净，真在冬深将够。够深，褐袍僧才肯离亭。

雨歇了，轮换稳了，亭里人还在候冬。风硬一寸，硬像凡符出袖前的最后一试。

刘婆第二趟端姜汤来，姜薄，热得钝：「雨潮，手别飘。飘了，试手滚。」韩泥喝尽，不空，手才不飘。热在，像第七笔在冬里又实一分。

韩泥独眼平，手不抖：「沿净，才接得住后头凡符。符来，仍凡，不越阶。越阶，像拜邪。」

轮换第十一晨，柱上五日条角白一线，侧门驼铃又轻一声。铃轻不要紧，凡符将出袖前须沿净——沿净，瓮温不外泄，试手才站得住坡下。
