# 第七十章 三年玉约

卯时，踹台。

坡下霜白，台木色浅，浅得像专给末席准备的辱。辱在众目，辱在内门眼，辱在赖福那点薄笑。丙九列队，韩泥立台前——前不是荣，是险。险在「滚」字：滚了，麸皮停，记功抹，七笔恩断。

管事敲锣：「踹台验末，手抖除名。」

内门弟子列廊下。钱戾衡立台侧，靴底沉，金线边亮，瞥一眼，唇角一挑：「丑杂役，上来。」

韩泥上前。掌茧裂处疼，疼不露。掌按台沿，沿凉进骨。台不高，高在众目——众目下，末席像笑话。

钱戾衡声不高，众耳都听见：「灰线三尺，也配站这坡？配站，就配滚。滚了，省得污内门眼。」

哄笑薄，也真。韩泥指节白一息，又平。他不辩。管事冷声：「踹台验末，手抖除名。」

钱戾衡抬脚，脚落台沿，像踩。踩实了，才记得住。韩泥独眼平：「管事看手，不看嘴。」

脚落半寸，台木吱呀一声。吱呀不要紧，手要紧——他掌按实，实得像坡下按石那一掌，一丝不抖。

有人嗤：「末席也硬？」

钱戾衡收脚，收得像意未尽。辱已够扎仇苗。他从袖里取出玉牌，牌小，牌青，青里刻三字：「三年约。」

牌掷台心，掷得像契。契在，仇在，路在。

钱戾衡声冷：「三年。三年后外门大比，你若还是末席，滚出沉丹宗，永不再入。若你炼气十三，站我台前——我让你滚，你才有资格不滚。」

哄笑又起，起得像潮。韩泥不接笑。接笑像认晦气；认不得，只认台心那行字：三年约。

有人喊：「接着！不接就滚！」

管事抬手压场：「契自愿接。接了，记册；不接，另议。」

韩泥看着台心玉牌，声平：「接。」

一字落地，棚下有人倒抽气。他把牌拾起，掌纹贴着「三年约」三字，凉意像钉子。

管事记簿，声沉：「韩泥，踹台手稳，不除名。三年约，记册。」

簿上又添小字：「契在怀，示人当记过。」示人不要紧，怀要紧——贴身，像守瓮人守沿。

廊角有人影立得远，衣暗带一点煞。韩泥不近，只记：影在，像远观，远观不要紧，手要紧。

铁无言抱臂：「灵根末，手稳，渣场仍收你。别漏。」

「不漏。」韩泥应。

秦霜远廊经过，木鱼不敲：「心别邪。邪了，鼻下香飘。」

他心里仇苗又深半寸——深在玉牌，深在三年，深在钱戾衡这一脚。玉牌凉进掌纹。凉不要紧，仇烫，烫在还路前头。

钱戾衡走远，还留一句：「丑杂役，记住今日。」

「记着。」韩泥低声。

仇因又实一分。实了，才配活三年。活三年，才还得了叶丫头那碗烫，才接得住往后该赢该诛的那些日子——日子未到，手先要稳。

赖福晃木牌：「三年？丑杂役活不了三年。」

韩泥不接茬。接茬，全丙九饿。他把玉牌藏怀，贴身，不示人。示人，谣又来。

台面还震着半息余韵。他掌曾按实，脚曾落台沿——骨记仇，不记滚。滚在远，约在怀。

同批少年围上来：「玉牌凉不凉？」

「凉进掌。」韩泥说，「烫在还路前头。」

「三年够长吗？」

「够记。」韩泥道，「记清了，才活得过。」

前头有三年，有十三层，有让他滚的那一日。日未到，手先要稳。

铁无言夜过渣场，瞥他怀：「契贴身，别示。示了，赖福记过；记过，恩也断半寸。」

「贴身。」韩泥应。

刘婆端粥，仍多半勺：「踹了？」

「踹了。」韩泥说，「手稳，不除名。」

「不除名怕什么。」刘婆哼，「手稳，饭稳。饭稳，恩不断。牌呢？」

「在怀。」韩泥拍拍胸口，「不示人。」

「不示对。」刘婆说，「示了，嘴杂，粥也稀。」

他点头。谢轻，恩重。恩在粥里，也在忍里——忍里藏刃，刃对着钱戾衡，对着三年约，对着玉牌上那三字。

回棚后邻床悄问：「你真敢接三年约？」

「不接，除名。」韩泥说，「除名，恩断。」

少年懂「除名」二字重，闭嘴。管事夜巡验棚，灯照净沿，记一笔：「丙九韩泥，踹台后仍手稳。」

韩泥把玉牌系肚兜。兜布粗，粗像泥岗旧袄——粗才贴肉，贴肉才像守沿。同批窃语：「玉牌是契，契接了，三年后必滚。」

他不接语。接语像认晦气；认不得，只认手在。手在，三年就不白接。

午后编筐换米，米少半升，也煮。煮时他把玉牌按在米缸沿，沿凉，牌凉，凉进骨——骨记仇，皮记约，两笔分开走，别搅成一锅糊涂。

钱戾衡弟子过廊，靴底带笑：「丑杂役，三年后记得滚。」

韩泥铲粪，铲下稳，稳像按台：「滚看手。」

三字声平，是小胜——胜在忍，不在嘴。白织月远廊经过，袖风冷，没停。没停，也好。赖福讽远：「三年？活不过。」

他不接。舌要留来记，留来忍。忍不是怂，是刃——刃对着钱戾衡，对着玉牌，对着还恩那日。

沈枯芽在药圃角弯腰，弯得像没看见。韩泥也不近——近像求恩；不求，只记半块灵石那笔，恩在还，不在这一日。

廊角煞影仍远，像看客。看客不要紧，手要紧。手稳了，三年才站得住，才接得住往后开口的人，才接得住药山捡渣。

他廊下再练半刻鼻下静，粟壳香淡，一字不落。落齐了，玉牌才记进心，不进舌。

踹台后几日，柱上条角仍白。韩泥仍末排，仍不滚。玉牌贴怀，凉进骨；仇烫在还路前头。管事簿上「三年约，手稳」一行在，名在册就不除名。除名，恩断；恩不断，忍里才藏得住刃。

更鼓沉。他按木牌，牌凉贴胸。手在，名在，坛沿净。鼻下静，恩与辱分开记——记清了，活才长；活长了，才还得了叶汤，才接得住门缝那点灯。

邻床又问：「三年，真接？」

「接了。」韩泥说，「接才有名。名在，就不滚。」

「玉牌给人看过吗？」

「不给人看。」韩泥拍肚兜，「给人看，就给人拿捏。」

他独眼平，手不抖，低声道：「三年约，记着。记着，刃才磨得亮。」

夜里他又把玉牌掏出来看一眼。青玉小，字浅，浅得像故意让人看轻。看轻不要紧，轻的是玉，重的是约。他把牌重新系紧，系结打得跟编筐底一样密——密了，掉不了，也拿不出给别人随便掂。

同批有人隔帘问：「韩泥，你接约是为出口气？」

「为留名。」韩泥说，「名在，才有气可出。名没了，气也没处出。」

「三年很长。」

「很长才够还。」韩泥合眼，「还叶汤，还仇，还今日这一脚。」

玉牌在怀，仇苗在心。心在，就不滚。手在，路就不绝。路不绝，三年后台前那一脚，才轮得到他去接——接的不是求饶，是把今日这口辱连本带利还回去。还之前，先活着；活着，才配谈滚与不滚。滚字在远，约字在近；近的先捂热，远的慢慢磨。

更鼓尽时，他把肚兜系带再勒紧一寸。勒紧了，玉牌贴肉，像第二个心脏——跳得慢，跳得冷，却跳个不停。邻床梦呓「三年……滚……」，他听着不恼。恼什么？约是自己接的。接了，就得拿命去填三年的空。空填满了，才轮到把钱戾衡今日那一脚还回去。还之前，粥还要喝，栏还要清，坛还要守——守住了，约才不是笑话。

刘婆第二日见他袖里鼓一块，盯一眼：「牌？」

「牌。」韩泥不瞒她。

「捂好。」她把姜塞他手心，「捂三年。捂不住，老婆子这半勺也白给你。」

「捂得住。」韩泥说，「捂不住，汤也还不了。」

姜热进掌，掌里玉牌更凉。一热一凉，像两条绳拴住同一只手——手被拴住了，反而不乱。不乱，三年才走得完。
