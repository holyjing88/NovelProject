# 第五十二章 入冬轮换

入冬第三日，丙九轮换兽栏。锣声一响，韩泥分到西栏——栏风硬，粪腥厚，厚得像冬一口气压下来。

赖福啐地：「晦气专分西栏，克全栏运。」

管事冷眼：「再嗓，西栏你一人清到年尾。」

赖福哑了。韩泥肩上的筐却实，实得像七笔压着——压着也好，手就不敢飘。西栏角落霜更厚，厚得像故意为难末排。为难就为难，粪还得清，兽还得活。

清粪时，西栏角风灌进骨缝。粪锹沉，沉得像搬柴。他不偷懒：偷懒麸皮克，克走的是粮，克不走手正。栏里最冷处，一只幼兽蜷在干草下，鼻息浅，浅得像要断。它咳了一声，细细的，细得像替整个西栏诉苦。

韩泥蹲下，掌心贴它额。额凉，凉才要暖。他多铺两层草，草里塞半块省下的麸皮边角——边角少，少也暖。幼兽舔他指背，舔得像谢。谢轻，他记在心里，不出口。出口像邀功；邀功像借腥。

管事走近，烛照幼兽鼻息，鼻息匀了半分。他低声：「多铺草，记功在。」

韩泥只点头，不谢。谢轻了，就记进第七笔里——刘婆那勺粥旁，又添了一粒粟。同批少年偷懒，管事记名。韩泥多留半刻，不邀功，只记。记着，心里那点实才不散。

清粪毕，他洗手。水冷，指缝腥洗不尽。洗的是袖，不是心——心记着恩，袖腥得避；胸热着，记。他把旧布拧干，挂回栏角，再看一眼幼兽：鼻息匀了，草窝鼓着，像冬里少死一条命。少死一条，恩便多记一笔。

午后搬柴过宗门外。驿亭褐袍人仍在，像等冬寒够深。商队余尘已尽，只剩五脉旗号在风里猎猎——猎猎不是谣，是界。韩泥不凑亭，只瞥一眼：袖里似有一张黄纸，纸边毛，毛得像凡物。凡物，才入凡人手。侧门窄，窄得只容半眼。半眼够看商线还在不在。还在，心就不乱；乱了，手就飘。

老耿挑水路过，水洒得实，声哑：「那人等冬。冬深了才赠。赠，不白受。」

「白不受。」韩泥应，「受了，记恩。」

「记就够。」老耿又道，「别凑过去问。问急了，像讨。」

「不讨。」韩泥说。

第二趟搬柴，风更硬，吹得侧门霜白。霜白，像辨香那日廊下——廊下过了，路还在。褐袍人未起身，袖里黄纸像多了一丝轮廓。轮廓丑，真。韩泥绕远三步，三步就是界：界内末排，界外赠缘。管事验垛，声硬：「冬里柴沉，沉得住，记功在。」

韩泥只点头。记功在，窄路又实一寸。他擦了擦掌心茧，茧裂口潮着，不是血。血不碰沿，沿才净。

回棚，坛腹时温时凉。温时像瓮在听柴声，凉时像瓮还睡着。睡着不要紧，要紧是沿上无血。他揭席一角，看沿：沿尘、沿滑，沿上无血。无血，才守住「不主动滴血」那条老规矩。邻床抱怨：「坛占角，晦气。」

韩泥不辩，只按席角。按实了，像编筐扎掌那日——掌茧再裂，裂在冬里，不裂到坛沿上。他想起祖父韩老栓的手：滑，稳，不许人拿坛闹着玩。手不在，滑还在。

刘婆端粥来，稀，热：「西栏腥，别空肚。」

「不空。」韩泥喝尽。热在胃里，第七笔又实一分。

「兽咳得凶不凶？」刘婆问。

「浅。」韩泥说，「铺草了，匀了半分。」

刘婆点头：「匀了就好。冬里死兽，谣就多。谣多，赖福那嘴就更尖。」

赖福远处骂：「晦气清粪，兽也晦气。」

管事路过，声冷：「再嗓，克半月麸皮。」

黄昏，铁无言巡到栏边，灯远，声近：「兽栏腥，洗尽再回棚。」

「洗尽。」韩泥应，「袖腥避着，胸热记着。」

铁无言停半息，又道：「试手前两日，手别僵。僵了，藤条不听。」

韩泥独眼平：「手记着，不僵。」

夜里他对坛，声低：「冬第三日，栏换了，商线还在。在，就不急。急了，像借腥。」

坛腹微温一线，又凉回去。凉回去像呼吸——吸温，吐凉，不急醒。外头更鼓二响，沉沉的。沉里，试手又近一日。「席盖严，沿不碰。」他低声道，「碰，不是今日。今日只清栏，只搬柴，只记恩。」

同批少年碰他袖：「西栏腥，你咋不躲？」

「躲，手飘。」韩泥说，「手飘了，试手就滚。」

「滚了呢？」

「滚了，粥就凉。」韩泥道，「凉了，还不起。」

少年懂「凉」，闭嘴。

丑时坛腹又温一线，温短，像瓮听西栏那声咳。咳细，不应人，应冬。他拇指压在木牌刻痕上，刻痕浅，浅得像心口七笔。笔笔按实，按出正——正了，试手才站得住。秦霜远廊一句，声轻：「心别邪。邪了，符灭。」

三更，西栏幼兽又咳一声。他想起白日那舌舔指背——轻，却真。真，像第七笔里又添一粒粟。他不奔栏：奔了像邀功。功在记里，不在舌头上。四更，铁无言提灯路过棚角，灯照坛席，照出一圈净沿。他低声：「沿净，好。好了，才接得住冬深那一赠。」

韩泥应声，不追问。问急了像借腥；守着，才像守瓮人。他又摸木牌，七笔分着记：叶丫头汤，振东药，陈姑袄，刘婆粥，符恩还没落到手，耿恩先记着，僧恩也先空着一格。笔在，手就不飘。

五更将尽，天边白一线。白像试手前两日也白了半分——白不要紧，手要紧。他按席，席贴坛，坛腹时温时凉：温时像听，凉时像眠。眠着，不急醒。醒要等缘，不等人催。西栏第二趟清粪，幼兽鼻息又浅一息。他多铺半层草，草旧，旧能暖。暖兽，像冬里少死一条命。管事验栏，烛照兽眼，眼湿，湿像怕冬。怕不要紧，手正了，兽才少死。

同批少年夜里问：「你图啥守坛？」

「图沿净。」韩泥说，「净了，才接得住后头那一甩。」

少年不懂「甩」，却懂「净」，闭嘴。管事夜巡名册，笔尖停他名上半息：「西栏清栏正，记功在。」他不谢，只记到木牌边。木牌薄，真在还能站；站得住，才配试手前那一坡。

「试手验啥？」少年又问。

「验手。」韩泥说，「手抖，丹歪，滚。丹歪不要紧，手不能歪。」

邻床哼一声，翻身睡了。韩泥侧耳听西栏风，风里幼兽鼻息匀了——匀了，西栏那角草还留着他的麸皮边。边角少，少也暖；暖着，柱上试手帖才贴得住。

入冬轮换第四趟清栏，粪锹磕着栏板，磕出闷响。闷响里他看见幼兽睁眼，眼湿，却不躲人。不躲，像认了草窝那点暖。管事远远喊：「西栏别歇！歇够了滚东栏搬水！」

「不歇。」韩泥应，锹又起。起锹不是逞能，是怕记功一抹——抹了，窄路就松。松了，手就敢偷懒；偷懒一次，试手那日腕子就软。

傍晚他又洗了两遍袖。水更冷，冷得指节发木。木不要紧，腥要下去。下去了，回棚才敢近坛。近坛时他先闻自己袖口，袖口无粪腥，才把席角压严。压严了，坛腹那点温凉起伏才像自家呼吸，不像外头借来的邪气。

天将亮，侧门霜白一线。白里，驿亭褐袍人仍在，像冬深那一赠还候在门外。候不要紧，沿净、手正，缘才接得住。灯远了，门缝像开了一线窄缝——缝窄，够走；够走，恩就不急断。断不得：断了，西栏那点暖、棚里那勺粥，都要跟着凉。他再听西栏一声细咳，咳过又匀，匀了才收心回棚——收心，才把轮换这一日过实。过实了，入冬这页才算翻过去。
