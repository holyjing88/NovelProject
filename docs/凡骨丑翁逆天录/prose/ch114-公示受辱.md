# 第一百一十四章 公示受辱

坡下新榜，贴的是「杂役末席公示」。

条新，字利：「丙九韩泥，贪渣诬，公示三日。」诬已败，辱还来——赖福换招换到了众目之下。韩泥按怀中木牌，牌在，名在册，手就不能抖。

有人早早围着看：「贪渣？不是清白了吗？」

「清白管屁用。」旁人撇嘴，「赖管事要辱，就辱。辱给你看，也给全丙九看。」

辰时，全丙九列队。赖福立榜侧，嗓尖得像刀刮木：「韩泥，凡骨末等，炼气七层亦末。末，公示三日，以警众人——沾晦气者，连坐克扣。」

队里嗡一声，少年退半步，像怕传染。韩泥独眼平，立末席，不躲。躲像怕；怕，手就抖。

「真连坐？」有人发毛。

「嘘！」旁边人扯袖，「你退远点就不是。」

管事弟子念条：「公示末席，手稳者仍留；手抖者，除名。韩泥，上石阶。」

石阶九级，级级像秤。他一步步上，掌茧疼，疼不露。末级立定，坡下百双眼睛——嘲的、惊的、白织月那双冷的。冷不要紧，手稳；稳了，辱就白挨一半。

白织月袖风带香：「七层也末？末，丹路断。断了，站阶上，像桩。」

韩泥不辩。辩费舌；舌要留着记仇苗。钱戾衡今日不在，玉牌在怀，三年约在怀。约在，就不滚。

赖福扬声：「跪半刻，谢宗门不除名之恩！」

「管事看手，不看膝。」韩泥声平。

场子里静了半息。有人等他跪，有人等他骂。他都不给——给了，辱才落得实。

管事弟子蹙眉：「赖管事，公示无跪令。」

赖福脸青，阴笑接上：「不跪，站半刻。站，让全宗看灰线三尺配七层。」

风硬，吹得灰线名像刀。刀在皮上划，划不出血，划得出忍。忍不是怂，是活。

铁无言立人群侧，声低，只够他听见：「站直。站直，像清瘴那日并肩。」

韩泥应声，脊不弯。半刻尽，管事挥手：「毕。韩泥，下阶，仍渣场。」

下阶每一步稳，稳得像首炼培元散时按炉。有人嘀咕：「他还真不跪。」有人回：「不跪又能怎样？吃饭还不是最后。」

赖福凑近，声只有两人听见：「公示三日，三日你吃饭最后。最后，像狗。」

「狗也清粪。」韩泥说，「粪清，宗门净。」

赖福噎住，甩袖走。韩泥把这一辱记进心里——刃又亮一分。

散队后，刘婆端粥，勺小，仍舀满：「公示了？」

「公示了。」

「公示怕什么。」刘婆哼，「手稳，饭稳。最后吃，老婆子等你。」

韩泥点头。谢轻，恩重。忍三日公示，换一日不除名；不除名，还序才不乱。

「你真不跪？」有少年追到棚口问。

「膝硬，手更硬。」韩泥说，「手硬，才配还恩。」

「还谁？」

「该还的人。」韩泥不多讲，「一日一日记。」

「我也想手硬。」少年嘟囔，「硬了就不怕榜。」

「硬从干活起。」韩泥说，「榜吓人，手不吓人。」

午后渣场，他分拣更细。细得像把辱按进灰里——灰脏，手不脏。铁无言又记一笔账清：「别近廊下吵。吵，手飘。」

「不飘。」韩泥说。

「不飘就好。」铁无言顿了顿，「公示这日子，别往热闹处凑。凑着，人指你。」

「不凑。」韩泥说，「干活。」

「干活是盾。」铁无言低声，「盾在，矛才扎不穿。」

傍晚，内门廊下有人笑：「灰线三尺，七层也末！」

笑薄，他也听进心里。玉牌凉，三年约烫——烫在还恩前头，不在这一声笑话上。

白织月再过，袖风更冷：「站半刻，像桩。桩也配公示？」

韩泥不答。桩稳，台不塌；塌不塌，看手，不看嘴。

婢子忍不住：「他怎么一声不吭？」

「吭了像讨饶。」白织月淡淡道，「不吭，像石头。石头也讨厌。」

第一日，他吃饭最后。碗糙，粥薄，刘婆仍舀满。满像恩顶着克扣。坡下有人念他名：「韩泥·仍末·手稳。」手稳，是小胜，胜在活，不在榜。

「手稳写上了？」有人嘀咕。

「写了。」另一个人说，「写了也末。末就末吧——他粪清着。」

第二日仍末，仍最后吃。刘婆在远处等着他空碗——等像补衣婆等恩还那日。那日在远，先熬这三日。韩泥把碗递回时，声低：「谢。」

「谢什么。」刘婆哼，「碗空了，人别空。人空了，恩也空。」

「不空。」韩泥说。

夜，窑角盘坐，七层气走周，坛腹温应「站」——站直半刻，不跪。不跪像挡石那年：石砸肩，膝不弯；弯了，恩像讨。

赖福廊外停半息：「明日仍末。末，吃饭最后。」

「在。」韩泥应。

「在什么在。」赖福冷笑，「站阶上风光？风光完了，还是狗。」

韩泥没接话。答了像跟狗辩。他只听见靴底远了，才把玉牌按实一分。

第三日，榜揭。名仍在末行。末行丑，丑得像脸；脸丑不要紧，手稳。赖福压顶压到头，头之外却露出一条缝——缝向坊市尾摊，向捡漏，向九层。

有人揭榜时还啐：「最后一日，晦气散了吧？」

「散不散看他。」旁人说，「他手还不抖。」

夜，他摸玉牌，低声念「三年约」三字。三字凉，仇烫。烫在心里，不进舌。舌响，香飘；香飘，滚字就真。

坛腹微温，应「站」。他低声对坛：「公示三日，一日一日记着。记着，刃亮。」

亮在试台，亮在还刘婆，亮在钱戾衡台前不滚。不滚，先偷空去坊市——公示第二日夜，尾摊人少，漏才好拣。

更鼓沉。韩泥按木牌，牌凉贴胸。坡下条角白一线，像钩，钩向明天那一寸缝。

「潜。」他对自己说，「拣漏，别贪名。」

风从门缝过，带一点药山冷。冷里有路——路窄，却真。真了，才配把「末」字熬成手里那点正。

他闭眼，却把侧门那条小路在心里走热——走热了，夜行方不慌。慌了，漏捡不成，公示还真把他钉死在末席上。钉死以前，他得先把铁和页摸到手。

公示第一夜，棚里话密：「他真站半刻？」「真站。脊都不弯。」

韩泥躺着听，听完不接。接像表演。表演要观众，他只要活。

玉牌贴胸，凉进肉。凉提醒他：三年约还远，远不怕，怕忘。忘了，站阶白站。

第二日他分拣时把筐沿磕齐。齐在灰里，不齐在脸上。脸上不动，笑话就短。

刘婆又塞半块糗：「站着饿得快。吃。吃了膝才硬。」

「嗯。」韩泥接过，「记着。」

柱上条角卷起，他用石子压住。压住的不是字，是明日要踩实的夜路。

第三日揭榜，名仍在末行。有人啐他靴边，痰沫未干。韩泥不擦给旁人看，只弯腰用草叶抹掉——抹掉的是脏，不是脊梁。

铁无言只在人群外抬了抬下巴。下巴轻，意思重——站过了，夜路就属于你。

韩泥把那一抬收下。收下不是恩宴，是路引。路引向侧门，向尾摊，向豆灯下那堆「垃圾」。垃圾里有温，温够把末席熬成进境。

夜里他数息到换班那一刻。一刻到，靴底才准响。响早了，公示的辱就白挨。

同棚少年夜里又问：「明儿还末席吗？」

「名末。」韩泥说，「手不末。手不末，才配出门拣漏。」

少年点头，不敢再问。不问也好——问多了像敲锣，锣一响，赖福耳朵就竖。

韩泥把糗咽尽，热意从头顶落到脚底板。落稳了，才配把「潜」字按进靴底。靴底一热，侧门那半刻换班便不再像梦，像一条真路。

石子沉，心也沉。沉着去潜，潜着去拣——拣回来的温，才配把三日公示从辱变成台阶。台阶不在旁人嘴里，在他靴底那条窄路上。路窄，容不下跪，只容一双稳手。手稳，漏才归得了废窑。
