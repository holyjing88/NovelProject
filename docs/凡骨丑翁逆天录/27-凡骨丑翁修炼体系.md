## 一、体系总览

```
凡人流（凡人修仙传） × 迟暮丑翁 × 丹道主业 × 瓮记恩仇 × 馈缘纳绶
```

| 维度 | 本作特征 |
|------|----------|
| 节奏 | 慢、稳、资源战；**58 岁开局**，破境更惜命 |
| 资质 | **末席灵根 + 丑骨凡躯**；不改根骨，靠丹、符、瓮、算计 |
| 战力 | 境界为本；符遁＞硬扛；丹名扬威补脸 |
| 驱动 | **有恩必还、有仇必报**；赠礼对口涨馈缘（`12`） |
| 金手指 | **鸿蒙九劫瓮**（腌菜坛凡相）；恩仇铭瓮、一念感瓮 |
| 飞升 | 人界化神 → 灵界炼虚～渡劫 → 仙界真仙 → **1555 证道眷升** |

**境界总线**（与凡人原著一致）：

```
人界：炼气 → 筑基 → 结丹 → 元婴 → 化神
                    ↓ 破界飞升（约 781）
灵界：化神(稳固) → 炼虚 → 合体 → 大乘 → 渡劫
                    ↓ 渡劫成功（约 1300）
仙界：真仙 → 金仙 → 太乙 → 大罗 → 道祖（远景）
```

## 四、资质体系（灵根 · 丑骨 · 定性）

> **资质**决定修炼速度与破境难度；**不改韩泥末席根骨**为全书铁规。与丹药品阶、道具品阶 **独立计量**。

### 4.1 灵根五级

| 档位 | 属性数 | 修炼速度 | 破境难度 | 本作角色例 |
|------|--------|----------|----------|------------|
| **天灵根** | 单属性 | ×3.0 | 易 | 钱戾衡（金） |
| **异灵根** | 变异单属性 | ×2.8 | 易且强 | 萧断雁（剑/风变） |
| **真灵根** | 2～3 属性 | ×1.5 | 中 | 沉丹宗内门多数 |
| **伪灵根** | 4～5 属性 | ×1.0 | 难 | **韩泥（末席四～五灵根）** |
| **废灵根** | 混杂/感灵极弱 | ×0.5 | 极难 | 杂役预筛淘汰线 |

**测灵表现**：石秤三色（青/黄/灰）；韩泥 **踹台末席、灰线克丙九**（ch041 备检）。

### 4.2 丑骨凡躯（韩泥独有资质）

| 项 | 定性 | 叙事功能 |
|----|------|----------|
| **丑骨** | 颈瘤、疤、迟暮相；**1555 不改骨** | 低谷羞辱→丹道扬名反差 |
| **凡躯** | 58 岁开局，寿元紧 | 逼「不浪突破」 |
| **末席灵根** | 四～五属性伪灵根 | 靠丹、符、瓮、算计补足 |
| **证道换相** | 终局意象，非换皮设定 | 丑仙证道，骨相仍在 |

### 4.3 资质 × 修炼（硬规）

```
有效修炼效率 = 灵根速度系数 × 洞府聚灵 × 丹药状态 × 天时 × 阳因系数
```

| 规则 | 说明 |
|------|------|
| 伪灵根可至化神 | 凡人逻辑；韩泥 **130～780** 人界全程 |
| 资质不替代道具品阶 | 末席亦可用天品符应急，但 **不可跳两阶** |
| 丑骨拒延寿丹 | 叶线 **延命丸拒用**；馈缘重情不重寿 |
| 丹师天赋≠灵根 | 韩泥丹道靠嗅控凝封验，不靠天灵根 |
