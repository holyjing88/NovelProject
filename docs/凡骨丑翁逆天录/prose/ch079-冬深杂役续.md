# 第七十九章 冬深杂役续

嗅诀萌芽后，冬更深一寸。柱上条换新，字丑，真：「冬深轮换，丙九韩泥，兽栏、药山、编筐，三日一轮。」

冬深轮换第三日，编筐末单，藤刺扎掌，偏手，刺浅些。编筐钱买米，米在，恩才还得上——还得上，才对得起刘婆多半勺，才对得起冬深这三日活。

药山远路，他担废渣过侧阶，阶霜白，肩不晃。晃了，赖扒皮记过；记过，恩也断半寸。白织月鼻下哼，哼完，袖风过——风过不要紧，手在。

丑时前，他闭息半刻，二层稳，远灯不熄。不熄，像四层将显——显在80，远不要紧，手在近。近，才内视，才看得见远境。

轮换不要紧，手在才要紧。手在，名在册，册在管事手，不在赖福秤上。卯时，丙九列队。霜白，呵气成线。赖福晃木牌，眼细：「冬深也轮换？轮换专给末席晦气活。」

韩泥不辩。辩，费舌；舌要留来清栏，留来守坛。他应一声：「在。」

第一日，兽栏。腥重，粪硬，铲下仍稳。稳里鼻下有根，根细，细不要紧；细才辨得出粪里一点药回——回像兽栏夜那日，回进掌纹，回进二层境里。小兽角里咳一声，咳轻。他换干草，草干，味淡一寸。管事路过，记一笔：「丙九韩泥，冬深仍清细。」

同批少年掩鼻：「冬深还兽栏，臭死你。」

「臭不要紧。」韩泥说，「臭也清。清完，才有饭。」

少年退半步，没再问。问，像沾晦气。晦气不要紧，晦气遮瓮。第二日，药山。边门开一线，他背筐上坡，坡滑，霜白。二层气沉足底，足底稳半分，稳像界碑前那一步——不过界，不贪活药，只捡渣。深雾起一寸，鼻下静，沉腐浮三字不落舌。沉者入左堆，腐者入右堆，假香上浮，指到即剔。铁无言立雾外，声低：「冬深别怠。怠了，渣混；混了，诀白芽。」

韩泥应声：「不怠。沉腐浮，记着。」

筐满，交管事，记「深捡净」。赖福讽远：「冬深也深捡？捡出邪，连坐丙九。」

韩泥不接茬。接茬，全丙九饿。他低声道：「只捡渣，沿净。」

第三日，编筐。藤刺扎掌，掌茧旧裂处又渗潮。潮不是血，血不碰沿。他偏手，刺浅些——浅些，沿才净；净了，坛才藏得住。编筐末单，钱买米，米在，恩才还得上。邻床少年瞧：「冬深还编？」

「编。」韩泥说，「筐卖钱，钱买米。米在，愧字才还得上。」

刘婆端粥，仍多半勺，肘袄针脚又补一线：「冬深轮换累不累？」

「累。」韩泥说，「累也转。转了，名才在册。」

「名在怕什么。」刘婆哼，「手稳，饭稳。饭稳，恩不断。」

韩泥点头。谢轻，恩重。恩在粥里，也在针脚里——针脚丑，挡风；风不进，臂才抬得起筐。午后，药渣场角风硬。他分拣，鼻下有根，手快半分，快不要紧，快也要稳。稳了，管事才不记「飘」。

铁无言走近，瞥分堆：「冬深仍对。对，才配废渣收集。」

废渣收集四字落耳，像钩。钩不要紧，先记——记冬深轮换，记三日一轮，记手不怠。秦霜远廊经过，木鱼轻敲：「冬深心易惰。惰了，鼻下香飘。」

韩泥不答，答在心里——心里恩辱分列。冬深是活，活是路；路通向四层将显，通向废渣路。黄昏，回棚。席严，坛不露。他按席角，坛腹温一线，贴掌，像应「续」字。应完，他合眼不睡实。丑时前，他廊下练半刻闭息。息在鼻，不在嗓。嗓响，香飘；香飘，赖福就记「邪」。掌纹那路转一周，二层稳，稳里芽根又细半分。细不要紧，细才接得住内视那一境。邻床问：「冬深还守丑时？」

「守。」韩泥说，「沿净，不叫。」

韩泥只答：「不拜。坛在，人在。」

更鼓沉，沉里，他摸怀玉牌，又摸愧字条，凉进骨。凉不要紧，鼻暖——暖在诀，不在众目。他独眼平，手不抖：「冬深续了。轮换三轮，手不怠。四层将显，在丑时。」

刃不亮，先记。记完了，才配内视四层，才配废渣收集那一袋。坛腹应温，不应字。温，像冬深霜白里那一点掌烫——烫在暗，不在册。册写仍末，掌写二层，鼻写沉腐浮。他低声对坛：「冬深杂役续了。坛在，根在。根在，就不滚。」

韩泥只答：「守沿。鼻下静。」

在，就不滚。不滚，冬深才熬得长——熬长了，才接得住95那盏门缝灯。柱上条角白一线，三年约在怀。怀凉，掌温——温在诀，不在廊下笑里。他低声道：「冬深续活。舌不留，手要稳。稳了，四层才看得见。」

看得见，在明日丑时；远不要紧，手在近。近，才内视，才显境。轮换第二日，编筐与兽栏对调。韩泥编筐半刻，藤刺扎掌，偏手，刺浅些。浅些，沿才净。筐成，管事验沿，沿稳，记「手稳」。手稳，是小胜。兽栏午后，雪混粪，铲下仍稳。稳里鼻下有根，根在，药回才辨得出。管事弟子验角，角净，记「冬深仍清细」。记细，是小胜。赖福克扣未反弹，眼却细：「冬深也清？清出晦气。」

韩泥不辩。辩，费舌；舌要留来续活，留来守坛。刘婆顶风端粥，粥稀，勺沉：「冬深别僵。」

「不僵。」韩泥说，「肘针脚挡风。」

铁无言立风里，声低：「冬深寒咬骨，鼻下更静。静了，四层影才不散。」

韩泥应声：「静在鼻。影在掌。」

白织月远廊过，没停。秦霜木鱼一声：「冬深心易缩。缩了，恩就凉。」

韩泥不答，答在心里——心里恩不凉，愧不混，仇不拜。冬深是压，压后仍要活。黄昏，药渣场灰冷，他分拣，沉腐浮并一息，分堆快，快也要稳。管事记「辨性准」。准，是小胜。邻床问：「冬深啥时候完？」

「完在春。」韩泥说，「春未至，手要在。」

丑时前，内视半息，第四阶影淡，淡不要紧；淡也是可见。可见是钩，钩三层四层将破。他低声对坛：「冬深续了。四层影见了。不叫。温，像把「续」字摁进掌纹里。更鼓沉，独眼平：「杂役续了。废渣将收，温氏将识。」

更鼓沉，沉里，韩泥按木牌，牌凉，掌温。掌温在79，不在舌上。舌响，香飘；香飘，手就滚。滚了，恩断。他不滚，手在，名在，路在。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。

滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。

滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。

廊下风硬，他把木牌系紧一线，系得像把仍末摁进掌心。

同批少年远避，避像避晦气，晦气遮瓮，遮住了才稳。

编筐角藤刺仍在，他偏手，刺浅些，沿才净，坛才藏得住。

管事夜巡验棚，灯照席角，沿净，像腌菜坛，不像邪。

老耿挑水过，声哑，境别拜邪，拜了路断，他只记着。

北栏轮换第十日，雪深一寸，铲下仍稳。稳像70按台——掌在，境在，路在。四层影在掌前，五层在远；远不要紧，冬深先过完。冬深过完，废渣才齐，温听雨才验手，95门缝灯才照得到。

更鼓沉，沉里，韩泥按木牌，牌凉贴胸。79章手在，名在册，坛沿净。舌不留，鼻下静。恩辱分列，不混。活长了，才还得了叶汤，才接得住门缝灯。

经页黄边再翻一线，章短，短不要紧——短也要记投序，记火路，记门缝留灯。
