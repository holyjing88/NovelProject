# 第一百四十九章 风势渐紧

坊市尾巷起风时，先起舌头。

有人凑堆议：「周家要嫁女，嫁给丑杂役？拿债压人？」

韩泥收摊不抬头。抬头像接架。接架，手就飘；飘了，收束不住，后头的火也不稳。

刘婆端菜路过，声压得更低：「周家那风声，别入耳？」

「入。」韩泥说，「入，才避。避不在辩，在记。」

「记什么？」

「记它还远。」韩泥道，「远着只听。听全了，才知道哪条缝不能踩。」

铁无言抱臂堵在巷口：「周家线，别沾。」

「不沾。」韩泥说，「沾了像滚。」

「滚给谁看？」铁无言冷笑半息，「给赖福看热闹？给你自己埋坑？」

「不埋。」韩泥牵驴，「蹄稳就行。」

沈枯芽又来一趟，枯叶上多了一字，字小：「周家别逼叶姐。」

字小，真。真像把叶家信续到今天。韩泥把叶片按进怀：「收到。今日不接。」

「不接我怕……」芽儿咬唇。

「怕也别喊。」韩泥道，「喊了，叶姐更紧。」

管事弟子传话，话硬：「外门少议家事。议了，记簿。」

「不议。」韩泥只答两个字。

赖福幸灾乐祸，嗓子尖：「丑骨也怕周家？怕就滚回泥岗！」

韩泥不辩。辩费舌。舌要留来数息，留来守坛。他贴着胸口摸玉牌——牌凉，烫着三年约：约说站稳钱戾衡台前，才谈得上不滚。滚在周家闲话里，连大比影都接不住。

邻床夜里问：「韩叔，周家真逼嫁？」

「风在。」韩泥说，「风在，后头见。今日不见。」

「你就不气？」

「气留给活命。」韩泥道，「活命才还得起恩。」

刘婆再送一碗菜汤：「别接架。接了，粥也凉。」

「不接。」韩泥喝汤，「接了手飘。飘了，散不正。」

疤鬃驴在柱后喷鼻，短而稳，像学他忍。忍不是怂。怂是瞎跑；忍是把劲存着——存到该解的雨夜，存到该还的省亲，存到不该把叶青禾拖进泥潭的时候。

坊尾又有人喊：「听说债压着叶家？那丑翁脱得了？」

「脱不脱，看路。」韩泥收绳，「路不歪，债就变不成娶。」

「你嘴倒硬。」

「硬的是规矩。」韩泥说，「规矩没到，风先到。风先到，我只擦耳朵。」

白织月袖风冷冷掠过，留下半句：「末席也配听周家笑话？」

「听笑话不花钱。」韩泥声平，「花钱的是上手接。我不接。」

袖风顿了顿，没再尖。尖不过「不接」两个字。

夜里他回丙九，按坛。坛腹温一线，像应「听」。听清了：逼嫁是风，解围在后头，架不能今天接。今天接，青木药火都会躁。

「周家风起了。」他低声道，「后头在前。记着。不滚。」

铁无言推门，扔进半截炭：「睡。明日照样分拣。分拣比吵架贵。」

「贵在活。」韩泥应。

「你真不打算吭一声？」

「吭给谁听？」韩泥问，「听的人要看戏。我不演。」

更鼓沉。他望帖，帖角微颤，像大比影在远。远不要紧。近的是忍——忍赖福五年，忍送出那半笔散，忍叶线不混，忍今日只听不架。

药峰早课，有个杂役贴耳问：「你要是被周家点名，咋办？」

「到时再说。」韩泥道，「现在点名的是风，不是人。」

「风也能伤人。」

「伤耳不伤手。」韩泥说，「手在，坛在，路就在。」

管事巡到药峰廊下，目光扫过他：「外门少议家事。你懂？」

「懂。」韩泥应。

「懂就好。」管事压册，「懂了才活得久。」

活得久，才配谈还。还叶汤，还泥岗，还那些说过热话的人。热话留给将来；今天只把风声记进心里，记下日期，记下「别逼叶姐」四个字，不多添一笔火。

天将亮，巷口又飘来一句「周家」二字。二字像钩子，钩浅，却越钩越紧。

清晨门外霜贴石阶。韩泥扫霜出门，听见巷口已经有人把「逼嫁」两字说得跟真契一样。

他把驴牵出来，对路过闲人只丢一句：「契没贴山门，先别当真。」

「你开脱？」

「我擦耳朵。」韩泥说，「耳朵脏了，手会跟着脏。」

铁无言送来一块压帖的石镇：「帖角别让风掀。掀乱了，你心里也乱。」

「领了。」韩泥把石镇压住大比帖。

「周家线你真一声不吭？」铁无言问。

「吭给戏看。」韩泥道，「我不卖票。」

午间赖福又来呛：「听说叶家寡妇要让周家娶进门？你当陪嫁还是当搅屎？」

韩泥抬眼，独眼平：「我当听的人。听完记。记完干活。」

「干活躲事？」

「干活活命。」韩泥道，「活命才有明日可走。」

赖福噎住片刻，啐一口走开。唾沫星子落在尘里，像风的碎渣。

沈枯芽下午哭着来：「他们说叶姐……叶姐脸都白了。」

「白也先忍。」韩泥把她拽到墙根，「你回去告诉她：别跪。别认嫁。风紧时跪一次，后头更难直。」

「你怎么不亲自说？」

「我一近，谣就成真。」韩泥说，「真了，架就来。架来早，火探不成。」

芽儿抹脸：「那你到底管不管？」

「管在后头。」韩泥道，「后头有日子。日子没到，我先把耳朵磨利。」

夜里刘婆坐门槛：「我活了这么大，见过逼嫁。逼嫁最爱找没势的。你别逞。」

「不逞。」韩泥说，「逞死没用。有用的是忍到能解的那一夜。」

「你算得出夜？」

「算不出。」韩泥老实，「算得出的只有今日不接。不接，叶姐还能喘气；喘气，才等得到还。」

他望坛温：「风紧了。紧着也好。紧了才显谁还能站住。」

站住的方法：税照交，摊照摆，驴照驮，周家二字只进耳不出手。出手留给验渣，留给数息，留给那块压帖的石镇。

石镇沉，帖才不飞。帖不飞，三年约还在胸前烫着。烫着提醒他——台前的仇还远，台下的风先别把自己吹歪。

风紧的第二天，外门廊下贴了半张提醒：少议家事，违者记名。韩泥看一眼就走。走得像把提醒按进胸口——提醒是护栏，不是枷。

药峰上，有人故意放大嗓门讲周家嫁娶笑话。韩泥把药包往驴背一扣，驴喷鼻，他不接话。接话像入戏。入戏的人最后都得掏钱买掌声。

铁无言把水递来：「喝。别嗓子干了忍不住。」

「忍不住也忍。」韩泥喝水，「忍到验完今天的渣。」

「渣验完，耳还脏怎么办？」

「脏了就再擦。」韩泥道，「擦到能睡着。睡着了，明日才有力气。」

沈枯芽傍晚来，不敢哭出声：「叶姐让我问——你还记不记得她？」

「记得。」韩泥说，「记得不等于现在冲去。冲去害她。」

「那我该怎么回？」

「回她：别跪，别认，别把风当契。」韩泥道，「契没盖印，风再响也是风。」

芽儿走后，韩泥按帖。石镇沉，帖角不动。不动好。不动说明他还没被风掀起来。

夜里刘婆塞来一颗枣：「吃甜的。苦听多了，人会僵。」

「谢。」韩泥吃枣，「僵了手会飘。飘了，规矩也扶不住。」

他低声对坛：「风紧。只听。明日仍不架。」

天色将白时，他把石镇又按实一分。帖不飞，约还在，风再响也掀不开他胸口那层皮。

铁无言从廊下过，丢半句：「今天坊尾嘴脏，你少停。」

「不停。」韩泥说，「停了像接。接了像滚。」

「滚给你仇家看？」

「不看。」韩泥道，「看戏的人多。演戏的人死得快。」

他把驴牵出棚，蹄声稳。稳声里他只记四个字：只听不架。不架，叶姐才能喘气；喘气，才等得到该还的日子。日子没到，锤子先别自己落。

他披衣出门，对坛也对人：「只听，不架。后头见。」

后头见。风紧了，锤子还没落——没落就好，落在它该落的日子。日子没到，人先别自己把心敲碎。
