# 第八十二章 废渣路

暗袋七包之后，铁无言指了一条路。那路不写在册上，只落在嘴里：

「药山废窑，半塌。杂役可过，不可占。过，捡渣试温；占，除名。」

韩泥把字啃进骨头里。过，不占——跟坛沿擦净一样，是规矩，不是商量。商量了就松，松了就贪，贪了名就没。

卯时前，他绕药山侧径。径窄，霜白，霜下碎石响半声。他脚轻，脚重了，赖福就嗅「偷占」。废窑在坡背，窑门塌半扇，门内灰热一线——热不是火，是余炉温。温淡，淡也要记。淡里有路：路从药山起，经暗袋，指废窑，再往远处指开炉的日子。日子远，他不急；急的人进深窑，深窑吞名。

他不入窑深，只立门槛外。外头风硬，硬里带苦辛。鼻下有根，沉腐浮三字不滚舌面，指捻一撮暗袋渣，渣灰里一点亮，像未烬引火。引火轻，记却重。记重了，才敢把掌贴上去试。

「试温，不占。」他自己哑声钉一句。

掌贴门槛石，石凉进骨。凉不要紧，炉温应掌，应得像坛腹那一线温——温不应字，应「路」。二层气走指梢，指梢温半分，温进渣灰。灰里亮多一丝，多一丝也是路。路上没有丹字，只有焦回——焦回进鼻，进记，不进嘴。进嘴，就叫邪炉。

铁无言立径外：「温到了别贪。贪占窑，路断。断了，十年杂役也白熬。」

「不占。试温。」

试半刻，他收指，收得像盖席。渣回暗袋，袋沉，沉才像攒。回坡下，管事验足迹：足迹不过窑深，深内一步无印。无印，是小胜。

「丙九韩泥，过窑试温，未占。」管事记簿，笔丑，真。

赖福讽得远：「废窑也过？过出邪炉，连坐丙九。」

有人跟着起哄：「丑杂役摸窑门，门都脏了！」

韩泥不接茬。接茬，全丙九饿。他只把肩沉半寸，沉够了，才走回渣场。

午后他把七包并成十包——新收三包，路才长。同批少年嗤鼻：「丑杂役也过废窑？过出丹了？」

他不辩，只答：「试温。丹字不在今日。」

少年不懂「试温」，却懂「不辩」，闭嘴了。旁边又有人压嗓：「他袋沉，沉里藏啥？」

「藏渣。」韩泥头也不回，「你爱掀，掀给管事看。」

那人噎住。掀不开，谣言就短半寸。

刘婆端粥，仍多半勺：「废窑去了？」

「去了。」韩泥说，「试温，不占。」

「不占怕什么。」刘婆哼，「手稳，饭稳。饭稳，恩不断。断了恩，窑温再好也喂不饱肚子。」

韩泥点头。恩在粥里，也在「不占」二字里。二字比粥烫，烫着提醒：过门可以，进屋不行。

黄昏他编筐半刻，藤刺扎掌，偏手，刺浅些。编筐钱买米，米在，恩才还得上——还得上，才对得起这一步废渣路。筐编完，廊下立着，任风硬，鼻下仍静。静里，路像一条线：坡下捡，坡腰捡，废窑试温，暗袋攒，将来开炉。线远不要紧，线头在近，近才走得长。走长了，才谈得上把叶汤那笔恩一点点还清。

丑时前盘坐床沿，席严，坛不露。掌贴沿，沿温一线，远灯不熄。气转一周，二层稳，十包渣像十粒种。种轻，记重。记重了，丹路才有根；无根叫丹，是嘴炮。

坛底轻颤，臾墟残念弱一息：「……渣里活，路在渣。路在渣，别叫丹。」

韩泥心里应：路在渣，不在舌。舌一响，赖福就来闻邪。

邻床问：「你床角袋越来越沉？」

「渣沉。」韩泥说，「沉不要紧，沉才像路。路空了，人先滚。」

少年不懂「路沉」，睡了。秦霜木鱼远敲一声：「路在心，心别邪。邪了，炉温也变成祸。」

他不答。路是活，活还要往前——往前才会撞上丹阁廊上那把帚。帚匀的人眼毒，毒不怕，怕自己肩晃、筐洒。

第二日巷尽炉基前，他驻足。驻足不为贪砖，为记焦回——回在鼻，不在舌。斗第三趟满，过磅，记「废渣路净」。铁无言验斗沿，沿无活草，无整药：「路尽了，别回头贪。贪砖的，名先没。」

「不回头。界在。」

风口远处，一个素裙人立半息，袖香温，她没近，只远看一眼斗沿。看一眼，也像验净。赖福讽远：「走路也走，末等也末等。末等看温氏一眼，也算攀？」

韩泥舌留给鼻子。攀字脏，脏了路就斜。刘婆端粥又多半勺：「路尽了腰还直？」

「直。」韩泥说，「腰直，槽才稳。槽稳，斗不翻。」

午后角柜十包锁实。邻床问：「炉基为啥不扒？」

「扒了，记名。」韩泥说，「记名，恩断。恩断了，粥也喝不稳。」

「那你图啥？」

「图把路走完。」韩泥道，「走完了，才有下一步。」

丑时十包并息，并出一线清。清里掌纹影颤，颤像远境在催。他低声对坛：「路尽了。焦回记着。丹阁廊那帚，隔几日就碰上——碰上了，别近。近了像求，求了像脏。」

第三日药山侧径仍霜白。径角有杂役弃牌，牌裂，字浅：「占窑除名」。他瞥一眼，不近——近像贪，贪则路断。管事再验足迹，深内仍无印。无印，名才干净。干净了，才配把「未占」两字继续背着。

巷口风硬，硬里焦回。界碑影斜在雾里，像提醒：再一步，除名。他收最后一斗，过磅，记「路尽净」。素裙人远廊一眼，一眼也像验沿。验得轻，他不迎——迎像把缘线扯早，扯早了就断。

刘婆端粥嘱：「路尽了别贪砖。」

韩泥点头，谢轻，恩重。砖里没有米，米在粥勺里。

废窑试温那半刻，温稳一线，像嗅诀里「沉」字落地。落地不要紧，记火路，记投序，记门里门外两套规矩——门外可过，门里禁占。界在，后头的人缘才邪不了。邪了，秦霜木鱼会敲第二声，敲完仍救不了名。

老耿挑水过，声哑：「境别拜邪。拜了，路断。断路的人，挑水也没人让。」

「不拜。」韩泥说，「只记。记温，记界，记袋。」

白织月远廊袖风冷，没停：「废窑路也走？走过也炼不出丹。」

铁无言脚步远，停半息：「别漏香，别叫境。只记。记够了，炉门自开。」

他把粥喝尽，木牌系紧。同批远避仍避，他由他们避。藤刺偏手，沿净，坛藏得住。灯照席角，像腌菜坛，不像邪。不像邪，夜巡才懒得掀席。

更鼓沉，沉里他摸暗袋，袋凉，坛温一线。温应路，不应慌。嗅渣场风硬不要紧，要紧鼻清、手稳——稳了，才藏得住以后再试炉温。炉温再试，也是在规矩里试，不在贪里试。

夜里他睡不实。睡实了像忘路。路在十包渣里，也在门槛那一点炉回里。炉回淡，淡也够钉住人：丹阁廊要新扫，担渣过廊，肩稳就行，别踩灰，别沾事。沾事了，「过窑未占」白过。白过一次，十年起不来。

坛沿贴掌，温一线回来。温得像告诉他——废渣路走到了，下一段不是窑，是人。人比窑难，难也得过。过的时候肩要稳，明日廊上那一眼验，才接得住。

他把十包渣在心里再数一遍：坡下三，坡腰四，窑门试温后并进去的三。数完，掌心潮潮的，不是怕，是记住了火路那点回。回在鼻里过夜，醒来还在，才配说「路在渣」。路在渣，丹字就别急着叫——叫早了，除名条比丹香先到。

外头更鼓又沉一记。他翻了个身，木牌硌着肋骨，硌得清醒。清醒好——清醒才记得门槛外那一线炉温。炉温淡，淡也够当尺子：尺子量脚，不量贪。量住了，明日廊上那一眼验，肩才撑得直。
