# 第一百四十三章 耿恩续

送出的第二天，风比昨夜更硬。

韩泥在渣场分拣到一半，沈枯芽就跌跌撞撞冲进来，眼红肿，嗓子却亮：「耿爷服了！」

「细说。」韩泥放下渣筐。

「他说粒正，他懂。」芽儿比划着，「还说——绳不抖，粒也不抖。老辈子话，我差点儿听哭。」

韩泥没笑。笑轻了像得意。他只点头：「收了就好。」

「他让我问你。」芽儿喘匀，「问你还记不记得抬棺。」

「记。」韩泥声哑，「记得比丹方清楚。」

「他还说屋漏那年补瓮……」芽儿犹豫，「补瓮也算？」

「算。」韩泥道，「恩不分大小。大的记，小的也记。」

坊市那边风已经先吹开了。有人压低嗓子议：「丑杂役也送丹？送谁啊？」

「泥岗一聋老头。」另一人撇嘴，「听说抬过棺。奇了怪了。」

「送丹还走叶家寡妇缝？」第三者挤眼，「啧，够弯。」

韩泥路过尾巷，一句都不接。接了像解释；解释多了，像求人认你的恩。恩不用求认——收了，就是收了。

管事弟子找上门，簿子一摊：「丙九韩泥，培元散谢恩一事，如实记。记细了，省得后头闲话变讼。」

「记。」韩泥说。

「经叶家？」

「经。我不进门。」

管事瞥他一眼：「聪明。进门就吵。」

「吵费舌。」韩泥应，「舌留着守坛。」

刘婆端粥来，听完哼出声：「耿老头聋，心可不聋。心聋，你这粒白送。」

「没白送。」韩泥喝粥，「他服了。」

「服了就成。」刘婆眯眼，「往后少听坊市嚼舌头。嚼得你手飘，才亏。」

「不飘。」

铁无言在棚外验完一筐渣，过来时只问一句：「送出了？」

「送出了。」韩泥说，「经叶家，人不进门。」

「好。」铁无言点一下头，「点头像记账。账清半分，下半分别急。」

「不急。」韩泥摸怀帖，「还有一半在后头。后头路远。」

「远也别停。」铁无言道，「停了，像装完人。」

午后验息堂开门。管事扬声：「夺魁者验息！炼气十六，层实才记！」

韩泥上阶，开息。息稳，稳得像昨夜炉边那口气没散。执事捺印：「十六层，登记。」

同批少年有人酸：「谢完恩就涨层？运气叠运气。」

「层是守出来的。」韩泥下阶，「运气闻不出正香。」

「你嘴硬。」那人不服。

「嘴硬不如手硬。」韩泥说，「手硬，册才肯写。」

赖福在阶下扫他一眼，鼻子动了动，没嗅出那粒散——散早在耿爷怀里。嗅空了，他就甩话：「谢恩？谢到我账上没有。」

「你账是税。」韩泥掏半文，「税照交。恩不入你簿。」

赖福脸青半息，终于骂着走了。韩泥把「十六」两个字按进心里——不是贺，是钉。钉住层实，钉住还能继续还的力气。

沈枯芽傍晚又来，递半片新枯叶：「叶姐说，别再让人把叶家跟丹绑一块嚷。」

「嚷的是他们。」韩泥收叶，「我闭嘴。」

「她还说……」芽儿耳尖红，「脉里稳着就行。稳了，她喘得匀。」

「好。」韩泥应，「匀比谢声管用。」

白织月廊上路过，冷冷丢一句：「杂役也玩报恩戏？」

「不是戏。」韩泥望她，「戏假，粒真。」

「真也丑。」她袖风一掠，「丑名传远了，看你怎么收。」

「远近不重要。」韩泥说，「收在该收的人手里，才重要。」

入夜，他坐坛边，把七笔里那道耿恩又描深半分。抬棺、补瓮、今次一粒——三截接到一处，处还不圆满，远省亲那笔还在后头。后头有日再走；今日只把余波收稳。

邻床少年突然问：「韩叔，你高兴吗？」

「高兴。」韩泥老实，「高兴也不嚷。嚷了，像炫耀债。」

「债还能炫耀？」

「有人炫耀骨。」韩泥淡道，「我只记还。」

更鼓沉时，坛腹又温一线。他低声：「耿恩收了。记着。」

温像应字。应完，外头坊市尾巷隐隐起了新词——词短、词土，像拿泥纹说事。词还没贴到他牌上，影却先到了。

他按木牌贴胸，望向天边那道白：「名要来，先让它闻着正。不正，宁可无名。」

登记后他没回棚庆祝，照旧去编筐。藤条勒膝，勒出那点实在疼——疼提醒他：层高了，活还是末席的活。

「韩叔，十六了还不歇？」少年跟来。

「歇了手生。」韩泥说，「生了，明日辨渣就不准。」

「你就没有高兴的时候？」

「有。」韩泥抬头，「老耿收下那一刻，高兴。高兴放心里，不放嘴上。」

坊市有个熟人硬拉他喝酒：「谢完恩，喝一个！」

「不喝。」韩泥扳开袖，「喝了，鼻钝；鼻钝，摊上砸锅。」

「真古板。」

「古板活得久。」韩泥走了。

刘婆后来听说喝酒那事，骂得痛快：「谁劝你喝你踹谁。喝坏鼻子，泥纹名也毁。」

「没喝。」韩泥应。

「没喝就好。」她又软了，「软饭多吃。吃饱了，闲话才顶得住。」

傍晚铁无言把一张旧纸塞他手里：「耿恩这笔，我帮你在我账上也画了一笔。画完，你别天天悬着脸。」

「脸悬着，是怕忘。」韩泥收纸，「忘了，丹还会邪。」

「怕忘就对。」铁无言说，「怕忘的人，走得远。」

夜里瓮边，他把「十六」和「收散」并排放。并排放不等于糊成一锅——层是层，恩是恩。糊了，后头泥纹名一起也脏。

「名要来就来。」他望黑暗，「来之前，把手洗净。」

有个外门文书故意把谢恩一事抄成笑话贴廊下。韩泥看见了，没撕。

「不撕？」少年急，「丢人！」

「撕了像心虚。」韩泥说，「留着。留着让人看：我还能再炼第二炉正的。」

「你真横。」

「不横。」韩泥摇头，「横的人爱辩。我把下一炉炼正，笑话自己会挂。」

管事后来叫他去，问要不要擦掉：「影响体面。」

「外门体面不在纸条。」韩泥答，「在手里有没有正香。」

「你这老头……」管事叹，「去吧。别惹事。」

「不惹。」

出屋时白织月拦路：「笑话贴你脸上，你还能笑？」

「没笑。」韩泥绕开，「哭更没用。没用的表情，我不做。」

她一顿。他已经走远。走远时心里其实烫——烫的是怒，怒按住，手才不废。废了手，老耿那粒也白送。

入夜他把笑话纸条的内容在日记里反写一行：「让他们笑。我还。」写完撕了日记页，烧进炭盆。灰白，心定。

更鼓后再敲时，他去兽栏外看了一眼空桩。桩上空着，像等人，也像等畜。畜还没来，名却先来了。

「看桩？」巡夜问。

「看空位。」韩泥说，「空着也好。空着提醒我：名起了，脚还没多生蹄。蹄以后再说。」

巡夜骂他神经病。他不争，回棚喝凉水。水凉，怒烫，两下一中和，人就能睡。

睡前他又把管事簿上那行「培元散谢恩」默背一遍。背的不是光荣，是压力：压着他后头每一炉都得正。不正，笑话纸条就会变刀。

「变刀就接。」他合眼，「接住了，还能还。」

晨雾未散，他又把笑话纸条看了一眼。纸条被露水泡软，字更糊。

「糊了才好。」他低声道，「糊了，人还在，粒还在。」

路过刘婆灶边，灶里烟一喷。刘婆塞他半颗蒜：「吃。去火。」

「火在哪？」

「火在你眼底。」刘婆道，「怒压着也是火。压久了，鼻子先伤。」

韩泥把蒜嚼了。辣得醒，醒得刚好听见远处有人又提起「泥纹」——提起就提起，先让余波里的耿恩沉淀一夜，再去接名。

蒜味未散，他编完最后一道藤。藤紧，像把余波也勒紧——勒紧了，明朝的名才冲不散今日的还。勒紧时指节咔一声，声小，却够他把「收」字钉住。

白里风一紧，紧得像钩——钩住明朝末摊上那半句闲话，也钩住他还没放下的「还」字。字在，夜就不空。
