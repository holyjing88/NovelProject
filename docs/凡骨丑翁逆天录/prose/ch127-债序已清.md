# 第一百二十七章 债序已清

试台清晨前，韩泥理怀里九粒培元散。

粒粒泥纹浅，浅像他那张丑脸——脸丑不要紧，嗅之正。正了，试台才配说话；说了，才配把钱戾衡那点眼风顶回去。

他坐在席角，把账一条条理清，生怕乱成一锅粥。

「一，叶汤，头恩，省亲前还。」

「五，芽儿，半块灵石，已还。」

「七，刘婆粥与补袖，试台后还新衣养老钱。」

「仇，钱戾衡三年约，玉牌在怀。」

铁无言敲门，声很轻：「卯时列队，试台坡下。我站你左侧三步。」

「三步就好。」韩泥说，「挡粪话，不挡手。」

「手是你的。」铁无言道，「手稳，我掷簿；簿清，盗谣虚。」

韩泥点头。兄弟线记着——挡话的是铁无言，献粒的仍是他自己。替证可以，替炼不行。

九粒封进陶罐，泥封严。严不是藏奸，是蔽眼：蔽赖福眼，也蔽白织月那股冷香。

罐沉，袖口刘婆针脚暖，玉牌凉烫并着——并着，像路：先上坡，再夺魁，再还人。

刘婆灶边盛粥，仍多半勺：「债序清了？」

「清了。」韩泥说。

「清就别乱。」她哼，「乱还，恩成债。债重，手飘。」

他喝尽。谢轻，恩重。恩在粥里，也在序里——序清，粒才正；粒正，魁才配；魁配了，还刘婆才不丢人。

沈枯芽柱角站着，小布包空了半边。石已还过，他却仍来送行，行也不送什么，只递一句小声：「韩爷爷，试台别抖。」

「不抖。」韩泥说，「抖了，还你石那夜就白还。」

芽儿点头，点得像米粒。米粒旁，徒恩清了，叶汤还在前头——前头远，也不抢。抢像花账，花账脏。

赖福廊外最后晃一下木牌：「试台滚，省我工夫！」

韩泥只答：「在。」

在，就不滚。不滚，手才稳；手稳，九粒才肯听招呼。

他摸玉牌，贴胸：「三年约，记着。今日试台，半年后大比，都不抖。」

坛腹温一线，像应「序」字。温一线，又像爹说：坛在，韩家不散。不散，就不滚。

不滚，钱戾衡那句「让你滚，你才有资格不滚」才接得住。

邻床少年悄问：「你真能夺魁？」

「魁在试台，不在嘴。」韩泥说，「嘴问，手答。」

少年闭嘴。韩泥望坡下——坡下试台新木，木色浅，浅得像当年踹台那块板。

踹台接了约，约接到今日；今日先验手，验完才谈魁。

外头鼓声远，远得像要开又没开。他不急。急像讨魁；魁不要紧，先正粒。

正了，执事才肯扬声；扬声了，刘婆眼浊里才能清一寸。

白织月袖风从远处扫过，声冷：「末席也列队？」

韩泥不辩。上台不要紧，手稳；稳了，末席也能挂牌。牌歪字丑人更丑——丑不要紧，丹正。

管事弟子喊：「丙九，出！」

队伍动了。韩泥走末席，怀九粒，袖针脚，胸贴牌。贴紧，仇约也在胸。约在，手在，步就不飘。

铁无言落在一条线外三步：「记住，别洒。」

「不洒。」韩泥说。

老耿在侧门挑水，见他一眼，嗓子哑：「别拜暖。」

「不拜。」韩泥应。

秦霜木鱼又敲一声：「心别邪。」

他还是不答。答在掌茧里——茧裂处疼，疼不露。不露，才像常活；常活里藏试台，试台里藏还路。

坡下灰线已经划开。有人说笑，有人抹汗。韩泥只闻自己陶罐：粟壳香淡，正。正，五年淘的，不是今日运气。

「序清。」他低声道，「试台，见。」

见字一落，坛又温半分。温像推他出门——出门上台，上台见正；见正之后，才递新衣养老钱。

布包与钱串又在怀里压了压。不多不少，够一个补衣婆活到老，不必再躲赖扒皮眼。

同列少年嘀咕：「丑骨也带钱上台？」

「不是上台花的。」韩泥说，「是还人的。」

「还谁？」

「该还的人。」他收住嘴。说破像显摆，显摆像把恩拿去卖。

钱戾衡影在廊下，声不高，高在韩泥耳朵里：「丑杂役，别洒。」

「记着。」韩泥答。

记着不是服，是活。活了才配站台前。台前远半步，鼓却近了。

卯时鼓再响一记。队末的他抬脚上台阶，台阶凉，袖口暖。凉暖并着，像把债序踩实。

踩实了，步就不飘；不飘，粒才落得住掌心。

他最后看一眼人群末。蓝布包头的人还没来，或者来了躲着。躲不要紧。还，不在当众认亲，在当众递物。

递到了，她才敢信：杂役五年，恩没忘。

「叶汤在前，芽儿已清，刘婆在后。」他在心里再说一遍，「后日那笔，今日先不混。」

列队间隙，管事弟子抽验行囊。抽到韩泥，翻开夹层，见布包与钱串，眉一挑：「试台带钱？」

「带还人的。」韩泥说，「不是赌资。」

「还谁？」管事又问。

「补衣的。」他答得短，「验完还我。」

管事盯他一眼，最终合上夹层，只在簿边点一点。点一点比骂轻，轻里却是活路：钱没被扣，序就没乱。

白织月远远听了，冷笑：「丑骨还学起报恩了？」

韩泥不抬头：「学不起。只会记。」

「记有用？」

「有用。」他把罐抱紧，「忘了，丹不正。」

铁无言在侧轻咳一声，像替他挡后话。挡得好。后话费气，气要留给上台。

坡下有人练嗓子嚷：「末席也想正？正了太阳西出！」

韩泥听着，袖里指节响一下，又平下去。响给自己听就行。听完，手更稳——稳了，嚷的人只会更噎。

沈枯芽在更远处踮脚，朝他摆摆手，手很轻。韩泥没摆回去。摆回去像拉帮手。他只把「芽儿已清」再按进心里一寸。

鼓第三声起，坡下齐静。静里，他把手按在罐盖上：盖严，手稳，序清。清了，才见。

见完正字，才接末项；末项夺魁，魁名落地，新衣才递得出。递得出，一部债序才算圆半寸——圆半寸也好，总比空嘴强。

风从坡下上来，吹得灰线名像刀。刀划皮，不划手。手稳，帖才近；帖近了，还刘婆的日子才真。

队伍前移半寸，他怀里钱串轻轻一磕。磕声极小，他仍伸手压住——压住了，像把「还」字也按实。

按实了，才不会在上台前抖成笑话。

赖福远远喊：「丑脸，别摔了钱！摔了你连晦气都买不起！」

韩泥头也不回：「买不起晦气，买得起还。」

「还？」赖福愣，「你还啥？」

「该还的。」他声音平，「你听不懂也罢。」

听不懂最好。听懂了，便要拦，拦了便乱序。序一乱，粒便洒；粒一洒，刘婆那包新衣又要推迟。

铁无言侧目：「少跟他费话。」

「费不起。」韩泥说，「话留给执事。」

执事在台口已经翻簿。簿页翻得脆，脆得像立刻要点名。点到他时，他只要罐开得稳，掌托得平——平了，「正」字才肯出声。

他在心口把叶、芽、刘、仇四字最后过一遍。过完，肩才松半分。松不是懈，是腾出手来捧粒。

捧粒比捧仇重要——仇在三年后的台上，粒在眼前这一尺木板。

木板近了。近到能看见木刺。木刺不扎人，扎的是心虚。他心不虚。不虚，是因为粥刚喝过，针脚还暖，钱还在，衣还在。

「上台。」管事喝。

他抬腿。腿不飘。不飘，债序就跟进了灰线里。

他把「清」字按进掌茧，茧疼一分，账就更实一分。实了，才踏稳那段凉木台阶。

台阶尽头，是验丹；验丹尽头，是还。还还没落地，脚步先落地。落地了，人就不虚。

他把罐盖按得更严一分，严得分明：序在心，粒在掌，人在灰线里。

灰线外头嘲还响，响不过袖口那点暖。暖在，就不飘；不飘，债才清得成。
