# 第一百三十五章 袖风嘲

药廊风起，袖香冷。

白织月立阶上，阶高，声更高：「丑丹师挂牌三日，可辨几包渣？」

韩泥立阶下，捧空囊，囊净：「够。够辨，够挂牌，不够辩。」

「辩什么？」白织月唇角一挑，「辩你运气？」

「辩骨。」韩泥声平，「骨丑，渣正。正，是手嗅出来的，不是运气。」

白织月袖风过，过得冷：「预选帖在怀，真以为能上大比台？」

「上。」韩泥说，「上不上，台见。见在掌，不在舌。」

她没再尖。尖不过「上」字——上，是约；约在玉牌，不在嘴。

铁无言立廊角，抱臂，没插话。没插话，像从绶前先站——站侧，站远，站得像108后仍来134修栏。

韩泥望他一眼，一眼像伏笔：绶在580，线在108与134；线不断，恩才还得完。

赖福廊下过，啐：「丑杂役也配跟内门弟子论辩？」

韩泥不辩。辩，费舌；舌要留来守坛。

管事弟子记簿，记「丙九韩泥，廊下不滚」，记细，是小胜。

刘婆缝扣过，声低：「别跟白姑娘顶。顶，惹眼。」

「不顶。」韩泥说，「顶，像情；情在还，不在舌。」

回杂役屋，他独坐坛边。袖风冷，冷不要紧；冷，才记得108并肩时铁无言挡的那半步。

半步远，半步近——近，是兄弟；远，是绶未纳。纳在580，纳前，只站。

丑时，练鼻下静。粟壳香混一点袖香余冷，冷落齐，心别邪。

他低声对坛：「从绶在远。线在今。今，站侧，不叫。」

坛不应字，只温。温像应「兄弟」——兄弟线续，不混叶汤，不混刘婆。

更鼓沉，天边白一线。白里，他摸玉牌，牌凉：「三年约，记着。大比，记着。」

邻床少年悄问：「白织月是不是瞧不上你？」

「瞧不上不要紧。」韩泥说，「瞧得上，手也要稳。」

他望坛，独眼平：「袖风嘲过了。下一回，层十五。」

药峰活照旧。分拣半刻，指稳，鼻下静一息——一息里只闻辛淡，不闻辱。辱在廊下，不在指；指准，136才实层。

秦霜远木鱼轻敲一声：「袖风嘲，心别邪。」

韩泥不答，答在心里。心里分列：嘲是嘲，恩是恩；恩在还，不在辩。

铁无言廊角声低：「袖风冷，手别飘。」

「不飘。」韩泥应声。应声短，短才像常活。

预选帖在怀，帖边磨毛，毛像133挂牌那牌角——角歪，字丑；丑不要紧，帖在，路在。

外堂阶下，风硬，吹得帖角掀半分。掀不要紧，帖在怀，手在。

管事扬声例行，声平：「预选者，层数每月验一次。」韩泥只答：「验。」

药渣场角，他备料半刻，料细，细像积势里最后一筛。筛齐，手才静。

赖福脚步停门外，停半息，像盯。盯不要紧，席严，坛不露。

刘婆缝布条过，声低：「白姑娘嘴冷，心别冷。」韩泥不接茬，接在坛——坛腹温一线。

沈枯芽远远鞠半躬，不近。不近不要紧，徒恩清了半块灵石；清了，叶汤还在前。

他低声对坛：「从绶在580。580前，只站。站侧，不叫。」不应字，坛腹温一线。

更鼓将尽，天边白一线。白里，他练鼻下静半刻，静到只闻辛淡。廊下暗处，赖福脚步又停半息，停，像盯；盯不要紧，手稳，就不滚。

韩泥捧空囊，囊净，净像没辩赢，赢不要紧，没滚就是赢。

白织月袖风又过，过得冷——冷不要紧，冷才记得手要稳。

簿上再添半行，行丑，字真：「嘲下不辩，席下不滚。」真在不滚，不在辩赢。

他望铁无言，铁无言抱臂，抱臂像从绶前先站——站，不叫。

更鼓沉，他按坛：「袖风嘲过了。580在远。线在今。」

白织月袖风冷，冷像内门阶上那眼——眼在丑骨，眼在帖，眼在大比台见不见真章。真章在台，不在廊；廊下韩泥不辩，辩，费舌；舌要留来守坛，留来候136层十五。

层在远，只嘲——嘲不要紧，嘲才试手稳不稳。稳，铁无言抱臂那半步就是答；答不在嘴，在站。

站侧，像从绶前先站；从绶在580，580断臂再生丹，丹在远，线在108与134。线不断，伏笔不是纳，是站；站，不叫，才像兄弟。

兄弟不是情，是义；义在108清瘴，在134修栏，在廊下不滚。不滚，管事记簿记细，记细是小胜。

胜在忍，不在辩；辩，像显摆恩；恩偿，静着还，还到该还的人手里。刘婆说「别顶白姑娘」，韩泥不顶；顶，像情；情在还，不在舌。

舌要留来数息，留来守坛。坛腹温一线，像应「兄弟」——兄弟线续，不混叶汤，不混刘婆。混，七笔分列；列在心，不在舌。

更鼓沉，天边白一线，白里，他摸玉牌：「三年约，记着。」记着，是丑翁本事；本事在忍，在还，在580前只站。

他望铁无言，一眼像约——580纳从绶，580前，只站侧，不叫。廊下风硬，硬不要紧；硬，才记得手要稳。稳了，从绶在580，线在今。

风过，他回杂役屋，不近廊下笑。笑在耳，不在心——心在坛，坛腹温一线。

舌要留来候136层十五。层在远，只嘲；嘲过了，手才配实。

内视里阶影稳在十四，稳像弓未满——未满不要紧，136才实；实了，142才近。

白织月袖风余冷，冷在鼻下一息里淡去。淡去不要紧，淡才静；静了，帖才沉怀。

更鼓沉，他按心口：「汤还是热的。」六字落，像路标——标向136，标向层十五。

编筐半刻，藤刺扎掌，刺浅，浅些沿才净。沿净，坛才藏得住。

散队后，他过外堂阶下，分拣半刻，指稳——指稳，路就不飘。

外门预选册页黄，黄像常活里藏路。路通向大比，通向580，通向142；142在远，今只站。

沈枯芽叶信无字，字在叶脉，脉像叶汤——汤在前，绶在580；580前，只站不叫。

废渣巷日斜，他验渣，渣正，手才静。渣正，像133挂牌那「正」字续——续在忍，不在辩。

侧门驿亭风硬，他听风，风过，记在心里。风里带一点袖香余冷，冷像白织月那眼；眼在，手更要稳。

坡下石阶湿，他数息，数帖，数债序。序里108在前，134在中，580在后；后不要紧，今只站。

药廊阶上尘起，尘落袖风里。落不要紧，尘落齐，心才静。

白织月远廊再过，没停。没停，小胜——胜在没滚，不在辩赢。

赖福火折照角，照见厚席下无坛。坛不露，沿才净；沿净，丙九才不连坐。

邻床少年又悄问：「铁无言为何帮你？」

「帮。」韩泥声平，「帮在108，不在今日辩。今日，只站。」

柱上条角白一线，三年约在怀。怀凉，掌温。

刘婆端粥，仍多半勺。韩泥点头，谢轻，恩重。

帖沉怀，沉的是序，不是惧。惧在耳，不在坛；坛在，手才不飘。

侧门风紧，紧像139将近。近不要紧，只嘲；嘲试手，不试舌。

同棚杂役笑，笑在耳，心不随。心在坛，不在廊。

管事弟子夜巡过，记「廊下不乱」，记细。细不要紧，细才是席严。

药渣场角，他倒渣半筐，渣灰，灰里正。正不要紧，正像133那「正」字——字在掌，不在舌。

铁无言送栏灰，灰细，屑亮：「袖风过，别辩。辩像滚。」

「不辩。」韩泥说，「滚了，刘婆新衣白还。」

铁无言点一下头，点得像134修栏那日——栏在，就在；只站，不叫。

编筐钱买米，米糙，糙不要紧；米在，恩才还得上。

丑时末，他再练鼻下静。粟壳香淡，香一字不落。落齐，135这一日记进心里——记袖风，记580，记只站不叫。

药廊尘落，落齐，像133挂牌那风改一半——一半改，134修栏才接，袖风才试，136层十五才实。

赖福廊外骂，骂声远，远不过韩泥掌里那一线稳。稳在，就不滚。

秦霜远木鱼再敲一声：「站侧，心别邪。」韩泥只答：「不邪。」答在心里，心里稳。

袖风嘲尽，嘲在耳，不在手；手稳，136才近。近，在实，不在喜。喜了，手飘。

更鼓沉，他望坛，独眼平：「136在前。记着。」坛腹温，贴掌，像应「站」——站在侧，不叫，580在远。
