## 一、因果系统（统领恩仇与突破）

### 1.1 设计定位

丑翁账 → 筑基后升为 **「泥瓮簿」**（漏底聚元壶凝元液浸染，与瓮同源）。  
**凡人修仙**式：因果不清，突破有劫；**斗破**式：恩怨到了该还该报的时候，读者爽。

### 1.2 四类因果

| 类型 | 记号 | 来源 | 后果 |
|------|------|------|------|
| **恩因** | ＋青 | 施恩、救人、赠丹 | 不清则心魔；还清可减劫 |
| **仇因** | ＋赤 | 受辱、迫害、背叛 | 不清则道心裂；报清可凝煞助战 |
| **善因** | ＋白 | 救凡人、赈灾、开渠 | 天劫减一分、鸡犬名额＋ |
| **恶因** | ＋黑 | 滥杀、夺恩、背诺 | 天劫加一分、心魔加重 |

### 1.3 泥瓮簿规则

1. **低谷入账**：恩因必记；仇因只记不立刻报（韩泥报仇三则）。  
2. **突破清算**：每次大境突破前，簿子自发示警「尚有×笔未清」。  
3. **天劫挂钩**：渡劫 **1300** 时，善因抵恶因；未清仇因化心魔劫。  
4. **与丑账关系**：七笔丑账 = 恩因核心；报仇账 = 仇因核心。

### 1.4 关键章锚

| 章 | 事件 |
|----|------|
| 2 | 得空白册（后变泥瓮簿） |
| 35 | 蘸血立「必还」→ 恩因固化 |
| 130 | 还刘婆，恩因－1 |
| 260 | 筑基，簿子变泥瓮簿（瓮元液浸染） |
| 262～270 | 省亲连还，恩因大减 |
| 510 | 诛戾衡，仇因－1 |

| 1300 | 渡劫因果清算 |
| 1555 | 终证道，泥瓮簿化入天道 |

## 三、一人得道鸡犬升天系统

### 3.1 设定名：**「眷升名录」**

天道规则：**终证道**（1555）时，可携 **恩因未尽者** 一缕魂入仙界边陲（非全复活，是「眷升」）。

### 3.2 名额规则

```
眷升总名额 = 境界基础名额 + 善因加成 − 恶因扣减
```

| 境界 | 基础名额 | 可携对象 |
|------|----------|----------|
| 元婴 | 0 | — |
| 化神 | 1 | 灵宠/器灵 |
| 合体 | 2 | ＋弟子 |
| 大乘 | 3 | ＋道侣残念 |
| 真仙 | **7** | ＋道侣魂（正遗念+副+剑）+恩人 |

### 3.3 韩泥眷升名单（1555 · 七名额 · 见 `19` §六）

| 序 | 对象 | 类型 |
|----|------|------|
| 1 | 臾墟子残念 | 器灵 |
| 2 | 沈枯芽 | 弟子 |
| 3 | 叶青禾执念因 | 遗念正绶 |
| 4 | 温听雨 | 副绶魂 |

### 3.4 叙事功能

- 前期施恩 = 后期眷升伏笔（读者二刷爽）。

## 七、写法硬规

1. 因果变化 **显性写**（簿上多一笔/少一笔）。  
2. 道侣后宫：主次分明，纳则不负；家族入戏（`19`）。  
3. 道具升级跟境界，不跳阶。  
4. 眷升名单前期只埋不剧透，**1555** 揭晓。  
5. 馈缘链见 `12`：赠礼对口、倒贴有因、纳绶不滥。
6. **禁自刎**：角色死亡一律战死拖垫背，不写自尽、吞毒、自毁金丹（寿尽坐化除外）。
