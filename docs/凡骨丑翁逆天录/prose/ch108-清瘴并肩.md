# 第一百零八章 清瘴并肩

深窝前。

角草黑，黑里甜，甜像毒眼。眼盯着，韩泥掌炉起，起在障后，障挡目，目挡，才炼得正。

青芯三撮，褐渣一把，石灰半指，灵泉五滴——滴齐，投序照经页。页深，深不要紧；深，是清瘴。

炉温一寸，一寸稳。稳里，白气聚，聚成丸，丸青灰，灰像药。

药成三粒，粒正，正字在鼻下，不在眼。

第一粒，掷深窝，窝嘶一声，甜退。退半丈，半丈够同袍进。

第二粒，铁无言吞，吞而臂肿退，退像活。活，是并肩。

「跟我进。」铁无言说。

韩泥进，进三步，三步撒第三粒灰雾，雾里清路。清路时，爪风再至，至，铁无言挡，韩泥不躲——不躲，像并肩。

并肩，不是喊，是手。手稳，他铲障，障改，改出通道。通道里，同袍拖倒兽，兽毒，毒不要紧；毒，清瘴丸正克。

克尽，深窝平，平像清。清，是108一事。

管事弟子记：「清瘴有功，丙九韩泥，铁无言。」

记，铁无言拍他肩：「兄弟，手在。」

韩泥应声：「在。」

在，就是契——契在580远，在远不要紧；在，是108的果。

赖福瞧簿，脸青，没嗓。没嗓不要紧，簿上「有功」二字，像把晦气按远半寸。

白织月递水，水凉，声低：「……正。」

韩泥接，接不谢。谢轻，记在心里：并肩，清瘴，正。

叶青禾在坡下，门缝灯灭。灭，像睡；睡，脉里培元温着，温着，像恩还着。

韩泥不在近。在近，像连累。连累不要紧，清瘴是还另一笔——还栏活，还同袍活。

回棚，掌裂，裂渗潮。潮不要紧，裂是活计。

按坛，低声道：「清过了。并肩过了。」

坛腹温，像应「正」字。正，是清瘴丸，正，是并肩。

丑时，静半刻。香落齐，掌纹烫——烫像境动，动一线，五层圆满，圆满后微开——开六层，开不要紧；开，像兽潮里实出来的。

邻床问：「你真成丹师了？」

「不成。」韩泥说，「成清。清完，仍倒渣。」

更鼓尽，界外腥退，退像终。终不要紧，终后有余韵——余韵在109，在炉温。

他低声道：「清瘴丸，正。铁无言，在。」

在，就不滚。不滚，才接得住首丹余韵。

深窝前，角草黑，黑里甜。

韩泥掌炉起，障挡目，目挡，才炼得正。

丸成三粒，粒正。第一粒掷深窝，甜退半丈。

第二粒，铁无言吞，臂肿退。

韩泥进，撒灰雾，清路。爪风再至，铁无言挡，韩泥不躲。

管事记：「清瘴有功。」

铁无言拍肩：「兄弟，手在。」

白织月递水，声低：「……正。」

坡下门缝灯灭。灭，像睡；脉里培元仍温。

深窝平，平像清。清，管事记功，记功，赖扒皮脸青。脸青不要紧，功在簿，不在嘴。嘴闭，手开，开里铁无言拍肩：「兄弟，手在。」

境开六层，开不要紧，开像兽潮里实出来的。实出来，109稳炉，110炼散——散，是丹道起点；起点，不在名，在正。正，一字，白织月也递水，水凉，声低，低像认。

深窝前，风止半息。止不要紧，止才炼得正。

第三粒，灰雾清路。

爪风再至，铁无言挡，韩泥不躲——不躲，像并肩。

管事记：「清瘴有功，丙九韩泥，铁无言。」

清瘴并肩，铁无言挡，韩泥不躲——不躲，像并肩。粒正，一字，够证。

「「深窝前，角草黑，黑里甜，甜像毒眼。韩泥掌炉起，起在障后，障挡目，目挡，才炼得正。「跟我进。」铁无言说。并肩，手稳，手稳，才成丸。成丸，不在名，在正。正，他嗅半息，半息够。」掌纹烫，烫像境动，五层圆满后微开——开六层。六层开，不滚，才接得住109。更鼓沉，韩泥按木牌，牌凉贴胸。铁无言，在。

」不滚，才接得住首丹余韵。清过了，六层开了。下一程，炉温稳。清瘴并肩，肩在障后，后在掌炉。掌炉起，障挡目，目挡，才炼得正。正不在亮，在匀——匀一寸，聚成丸，丸青灰，灰像药。药成三粒，粒正，正字在鼻下。第一粒掷深窝，甜退半丈，半丈够进。第二粒铁无言吞，臂肿退，退像活。「跟我进。

」韩泥进三步，三步撒第三粒灰雾，雾里清路。清路时，爪风再至，铁无言挡，韩泥不躲——不躲，像并肩。并肩，不在嘴，在肩。肩拍一下，声低：「兄弟，手在。」手在，才六层开——开微，微不要紧，微实而不飘。飘了，109不稳；稳了，110才散。管事记功，记「丙九韩泥，铁无言」。

记，像证；证，赖福改簿，改「晦气」为「有功」——改不要紧，改在簿，不在脸。脸仍丑，丑才像末席该得的。

刘婆远送粥，没近窑：「活着，别骄。」骄了，手飘；手飘，散不正。不正，路断。路不断，109稳炉，110首炼。首炼在远，远不要紧，108只并肩——并肩，清瘴丸正，六层开。铁无言，在。」在，就不滚。不滚，才接得住109余韵，110起点。起点在散，散在远，远，手先在。清瘴并肩，障后掌炉，目挡才正。

正，匀一寸，丸成三粒，粒青灰，嗅之正。正，掷深窝，甜退半丈；半丈，铁无言吞第二粒，臂肿退，退像活。「跟我进。」进三步，撒第三粒灰雾，雾里清路。并肩，肩拍一下：「兄弟，手在。」手在，六层微开，微实而不飘。管事记功，赖福改簿，改「晦气」为「有功」——改在簿，不在脸。脸仍丑，丑才像末席该得。

得，不在名，在正；正，接109余韵，110起点。更鼓沉，按木牌，清瘴丸正，铁无言在。在，就不滚。不滚，109稳炉，110首炼培元散。散是丹道起点，起点在丑手，不在130远名。深窝前，障后掌炉，目挡才正。正，掷深窝，甜退半丈；半丈，铁无言吞第二粒，臂肿退。「跟我进。并肩，肩拍：「兄弟，手在。

管事记功，赖福改簿——改「晦气」为「有功」，改在簿，不在脸。在，就不滚。名在远，路在掌。掌在，就不绝。」

清瘴第二粒，铁无言臂肿退，退像活。「跟我进。」进三步，撒雾，雾里清路。清路时，爪风再至，挡，不躲——不躲，像并肩。并肩，六层微开，微实而不飘。散是起点，起点在丑手，不在名。

「跟我进。不正，路断。在，就不滚。不滚，六层开了，109在前，110在远。远不要紧，108只并肩——并肩，手在，路就不绝。清瘴第三粒，雾浓，清路半丈——半丈，够同袍进，够深窝甜退。退，铁无言臂肿退，退像活。活，六层微开，微实而不飘。散在远，108只并肩——并肩，肩拍，手在，就不滚。

滚在远，名在130，路在掌。掌在，就不绝。清瘴第四粒，雾浓，清路半丈——半丈，够同袍进，够深窝甜退。散在远，108只并肩——并肩，肩拍：「兄弟，手在。」手在，管事记功，赖福改簿。改「晦气」为「有功」，改在簿，不在脸。得更鼓沉，按木牌，清瘴丸正，铁无言在。在，就不滚。不滚，109稳炉，110首炼。

首炼在丑时，星晦，晦才好走。走，掌在，路在，就不绝。清瘴丸正，六层开了。开了，109稳炉，110首炼培元散。掌在，铁无言在，就不滚。滚在远，并肩在108，并肩过了，就不白并。清瘴将尽，雾浓，清路半丈——半丈，够同袍进，够深窝甜退。铁无言臂肿退，退像活。「兄弟，手在。

管事记功，赖福改簿——改「晦气」为「有功」。更鼓沉，按木牌，清瘴丸正，109稳炉，110首炼。并肩过了，掌在，路在，就不滚。清瘴丸正，六层开了，109稳炉，110首炼。并肩过了，铁无言在，掌在，就不滚。」」

更鼓沉，按木牌，清瘴丸正。109稳炉，110首炼。并肩过了，掌在，铁无言在，就不滚。

六层开了，109稳炉，110首炼培元散。

并肩过了，109稳炉，110首炼。掌在，就不滚。

「」韩泥点头，谢轻——骄了，手飘；手飘，109不稳。秦霜木鱼远敲，声低：「清瘴别骄。」韩泥不答，答在心里——心里不骄，只记并肩。
