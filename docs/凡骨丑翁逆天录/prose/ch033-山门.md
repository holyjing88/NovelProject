# 第三十三章 山门

卯时，雾厚。沉丹宗山门在雾上，石阶千级，级级青苔，青苔滑，滑不过人心滑。人心滑，韩泥见过——泥岗村见过，井台见过，祠堂见过。他走侧门。侧门窄，窄给杂役，窄也给凡骨。守门管事瞥木牌：「丙九？韩泥？」

点头。「抬棺的？」

「抬过。」

管事鼻哼，哼得像不信，又懒得查：「进去。兽栏、药渣、劈柴，哪叫去哪。别闯内门，闯了，打断腿。」

韩泥「嗯」一声，入。山门内，丹香扑面。香浓，浓得像把泥岗霉气洗掉半寸。洗不全，够活。活在这香里，像活进另一条命。道旁弟子衣鲜，衣鲜者瞥他，瞥完绕远。绕远，韩泥习惯。习惯里，他望见远处丹塔影，影高，高得像还不起的恩——恩要还，塔要爬，爬不急，急便摔。侧门石阶旁，废筐堆着，筐破，破得像泥岗村西头。

西头他走过五十八年，走过才识得：破筐能补，破命能活，破骨……且活着看。有人喝：「杂役！避让！」

韩泥贴墙，让。让半步，够。够活，够看，够记。这是小胜：山门进了，腿没断，谣没跟进来——跟进来的是他自己，和篓底那只坛。管事领至丙九杂役房。房小，四床，床乱，乱里霉。韩泥选床角，角暗，暗好藏坛。坛从篓取出，用破席盖，盖严，像泥岗旧习。「晦气东西别摆明面。」邻床少年嫌，「克修炼。」

韩泥不辩克不克，只道：「我盖。」

盖了，像盖住泥岗，盖住五十八年，盖住「晦气翁」三字。三字还在舌上，舌在门外，门内换名：丑杂役。名换，人不换。人还是韩泥，还是记恩、记辱、记仇那一个。入门册上还有一行小字：凡骨试录，三年为期。期为绳，绳拴杂役，拴将来测灵，拴大比，拴还恩的路——路长，先活三年。午后分活，管事掷单：

兽栏清粪，药渣分拣，劈柴二十捆。韩泥接兽栏。粪臭，臭得像泥岗井台脏水，臭不过命。命在这臭里，先活下来。清粪时，独眼眯，眯见栏角一滩血——血旧，旧得像有人伤过灵兽。血不干，像有人故意留着吓人。韩泥不碰血，不碰像碰邪。他清净，净得地面见石，石凉，凉得清醒。灵兽栏深处，小兽抬眼，眼怯，怯得像沈枯芽。

韩泥投半把草，草落，小兽啃，啃得像活。活，他懂——懂所以清粪，不图谢，图手稳。管事路过，瞥一眼：「老丑，手稳。」

三字，像庙祝那句「手稳」。韩泥应声，不谢。谢轻，记重。清粪毕，同组杂役歇栏外。有人嫌：「丙九老丑，分活总多。」

韩泥只答：「多，手稳。稳，才有饭。」

另一人问：「你抬棺真不抖？」

「抖一次，掉一次。」韩泥说，「掉一次，记一次。记多了，滚。」

两人噤声，噤，像被「滚」字堵住。管事再经，扔一把草镞：「栏角别留血。血留，灵兽躁。」

韩泥只点头，清净，净得地面见石。夜，兽栏远吠。韩泥回房，擦手，手糙，糙里带腥。腥洗不尽，尽的是心——心要净，净才能修，修才能炼丹，炼丹才能还叶。他低声道：「山门进了。下一步，活。活到达得到还的那日。」

午后，劈柴二十捆。斧沉，肩驼，斧落仍匀——匀得像泥岗编筐勒条，一下一下，不飘。同批少年劈到第十捆，手抖，柴飞半尺。管事喝：「抖！滚！」

韩泥不笑——笑费气。他劈完二十捆，捆齐，齐得管事不用数。管事瞥他：「老丑，斧也稳。」

谢轻，记重。记「手稳」二字，记侧门，记丙九末席。坛腹微温一丝，应丹香，应「手稳」，应丙九末席。二更后，邻床少年梦呓醒，醒来看他盘坐：「你不睡？」

「睡不实。」韩泥说，「实睡，像认输。」

「怕什么？」

「怕恩断。」韩泥独眼平，「恩不断，命才肯活。活，才有还。」

少年不懂，翻身又睡。睡，韩泥摸丙九木牌，牌凉，凉得像榜末名——末不是绝，是缝。外头更鼓三响，响沉，沉得像瓮在睡，睡里听见的脚步，一步步，走近瓮醒。他低声对坛：「山门进了，兽栏清了。」

坛不应字，只沿口一线水汽，应完又敛。敛，像睡；睡，像等。等瓮醒，等还——还叶，还六笔泥岗恩，还省亲那日。三更，管事巡房，敲柱：「丙九，明日药渣分拣。手别抖。」

韩泥应一声：「不抖。」

管事瞥席下陶色：「坛？」

「祖辈传的。」

「别摆明面。」管事鼻哼，「晦气杂役，少惹眼。」

韩泥低声应，不辩晦气。辩，费舌；舌要留来分拣，留来活，留来还。管事走了，邻床少年悄问：「你真带坛？」

「带。」韩泥说，「坛在，根在。根在，人不在绝路。」

「赖福说扔。」

「扔不得。」韩泥独眼平，「扔坛像扔命。命扔不得。」

少年不懂，翻身睡。睡，韩泥按坛沿——沿一线水汽，应「侧门」，应「手稳」，应完又敛。敛，像睡瓮听步；步稳，名未醒，路先开。管事又经，扔单：「丙九韩泥，明日药渣分拣。」

「不抖。」

「抖了，赖福扣饭。」

韩泥不答：「扣饭，记名。记名，迟早还。」

管事一愣，没再尖，走了。走，像记下一笔「手稳」。邻床少年梦呓醒：「你真五十八？」

「五十八。」韩泥说，「兽栏臭，你受得住？」

「臭过泥岗井台。」韩泥说，「受不住，滚。」

少年不吭，又睡。他低声道：「渣场有香，香是路。路开了，活。」

更鼓又响，韩泥合眼。合眼不是睡实，是记名——记侧门，记手稳，记药渣路。管事门外贴条：「入门第七日，凡骨测骨登记。丙九末排，韩泥到。误者，滚。」

七日测骨——比瓮醒近，比还汤远。远不要紧，帖在，手就别抖。丹香淡，兽栏活臭；韩泥手不抖：侧门进，正路未开——未开也要站丙九末席。帖在柱上，手就别抖；抖了，恩断，汤就凉，必还誓就白立。他低声道：「七日后测骨。帖在柱，手别抖。」

丑时将近，坛沿一线水汽，一闪，灭。他不追问，只按掌。按实了，才配七日后那验。稳像泥岗抬棺，稳像公审末席。末席也是席，席在，饭在；饭在，帖才验得过。山门丹香淡一寸，淡不过兽栏活臭。活臭里，铲下仍稳——稳，手才不抖；抖了，恩断，汤就凉。

入门册上「三年为期」四字，他默念一遍——三年是绳，绳拴杂役，拴将来测灵，拴还恩的路。路长，先活三年。三年里，手稳一日，多活一日；多活一日，多记一日恩。恩记满了，才还得上热的。刘婆若来查房，多半又咳：「坛别摆明面。」

「盖了。」韩泥说，「盖严，像泥岗旧习。」

管事门外又贴细条：「药渣分拣，酉时到。鼻笨没事，手别抖。」

酉时渣场——鼻下那条路，明日要走实。走实了，丹香才引得深；引得深，还汤才近一步。同批少年梦呓：「灵根……」

韩泥声低：「灵根在石上，手在铲上。手在，命未末。」

侧门风硬，兽栏活臭扑面。臭不要紧，要紧是铲下稳——稳，管事才夸手稳；手稳，七日后测骨帖才验得过。他低声道：「侧门进，正路未开。未开也要站丙九末席。席在，饭在；饭在，恩不断。」丑时末，他把木牌又按实一遍，按得像把丙九名钉进掌纹。名在末，步先稳。

明日酉时渣场，鼻下要走实——走实了，丹香才引得深，还汤才近一步。同批少年又梦呓「灵根」，韩泥声低：「灵根在石上，诀在鼻下。鼻下静了，铲下才稳。」侧门风硬，兽栏活臭扑面，臭不要紧，要紧是手别抖。坛沿一线水汽，应「入门」，不应慌。

柱上条角白一线，三年约在怀。怀凉，掌温。
