# 第一百二十二章 织月嘲

榜一贴出来，白织月那句嘲就跟过来了。

坡下人挤得满当当，像粥锅翻了沫。榜纸被风掀角，末行字却清楚：「韩泥，丑骨，报培元散。」丑骨两字贴在纸上，像贴在他脸上。他不抬头抢看，只站远一点，把怀里陶罐按稳。罐一稳，心才稳；心一稳，嘲才刮不进骨缝。

白织月袖风带着冷香过来，声音细，细里带刃：「丑杂役也再炼？炼了也末等。」

韩泥仍低着眼：「再炼，不叫。沿净，不叫。」

「丑名也上榜？」她笑，「上榜像污榜。」

铁无言恰立在侧，声沉：「榜看试台，不看脸。」

白织月瞥他一眼，尖得不收：「铁师兄也护晦气？护了，丹道跌价。」

「跌价试台见。」铁无言说。

韩泥独眼平，只回三个字：「试台见。」

三个字砸下去，场子嗡了半息。嗡完，赖福挤进人堆，嗓子尖得像锥子：「丑杂役上榜，丙九晦气满坡！」

管事弟子喝：「聒噪！再聒，逐出坡！」

赖福噤了。噤归噤，韩泥把这一声记进心里——记织月嘲，记赖福余毒，记末席风声。记不是为骂回去，是为把舌头留着护罐。罐里七粒培元散，护不住，汤也好、粥也好，全都凉。

刘婆站在人群末尾，蓝布包头，手里端着空碗，像来占个位子。碗空着，人却热。韩泥望她一眼，那一眼像把还恩的日子往前挪了半寸——挪半寸，也得先过试台；不过试台，新衣养老钱只是布包里的梦。她冲他轻轻摇头，像怕沾魁晦气。

旁边有人嘀咕：「末行也算报名？」又有人接：「算。管事册上有名就得上。」嘀咕远了，像潮退；潮退了，嘲还挂在耳边。怕不要紧，粥仍多半勺，姜仍多一片。一片姜暖手，手不僵，粒才不洒。

散队后，药山道上白织月又拦了一拦，袖风更冷：「你那七粒散，怕是杂的。」

「杂不杂，鼻下知。」韩泥声平。

「鼻丑也配嗅丹？」

「鼻丑，嗅得正。」韩泥顿一下，「正，是五年渣场淘出来的。淘出来了，嘴尖也尖不过一粒正。」

她唇线绷了绷，没再跟。没跟，是小胜——胜在掌，不在嘴。韩泥也不逞，回窑盘坐，气走十一层，走得沉，沉得像罐在怀里压着。十二层门槛裂开一线，他不露。一露，白织月就会咬「偷境」；偷境不是，是忍。忍到试台见了，再说。

秦霜的木鱼在远处轻敲：「嘲声大，心别邪。」

韩泥不答，答在心里。心里排着序：芽儿还了，刘婆待还，叶汤是头恩，钱戾衡是仇。仇恩分两边摆，就不糊涂；糊涂了，散先不正。不正，刘婆这笔就得再拖；拖了，叶汤还更凉。

邻床少年悄问：「白织月咋总盯你？」

「末席丑，好嘲。」韩泥说，「嘲来了，手稳就行。」

「手稳能赢她？」

「不赢她。」韩泥摸了摸玉牌，「赢试台。钱戾衡，也记着。记着，试台见。」

牌凉贴胸，像又提醒一遍三年约。约不在今天结算，今天只把嘲挨过去。挨过去了，榜末行也不丢人——丢人的是手抖。手抖，除名；除名，芽儿那半块石就像白还，刘婆粥也喝不安生。

外头榜风又掀，掀起一角白，白得像大比还远着的影。影在远，嘲在近。近的可以忍，远的得靠粒正手稳去接。接不住，帖来了也白。帖白了，还恩就断在半道，断不得。

赖福在廊外吐了口唾，唾落阶边，不落他靴。像内门仍不肯认他，他也懒得讨认。认不认，试台认正就够。正了，刘婆才敢穿新衣，芽儿才敢收石，叶青禾那碗汤才还烫在前头。

夜里他练鼻下静，粟壳香淡，香落齐。织月这一日的嘲，他一寸寸按进心里，按完刃亮半分——亮给试台，亮给还刘婆，不亮给袖风。袖风冷，忍过去；过去了，见字才真。

「试台见。」他又低声说了一遍，像钉钉子。

他把罐口泥封又抹了一道，抹匀了才睡下。睡下不等于撒手；撒了手，榜上那行丑骨就真成笑话。

坛腹微温一线，像应。应了，备战又实一分。实一分，就够往十二层挪，够把后面铁无言挡话接住，够把刘婆那包衣钱焐热到能递出去的日子。日子未到，先护罐。罐在怀，怀沉，沉得像债；债清了，步才不飘。

第三日韩泥仍吃得最末。末席吃饭，刘婆却等着，等得像补衣婆等一口热气。他端碗时她低声：「榜末也是榜。末不怕，怕手虚。」

「手不虚。」韩泥说，「虚了，不敢喝你粥。」

刘婆哼笑：「油嘴。喝吧。喝完炼你的。」

他喝完，掌心热了半寸。热半寸，嘲就退半寸。退半寸也好——退够了，才听得见明日廊下那道靴底声。靴底声来了，金线边就亮；亮了，玉牌更凉。凉烫并着，才叫三年约还活着。活着，就别躲。躲像怕，怕了手就抖；抖了，钱戾衡那句「让你滚」才算应验。

散榜那夜，丙九棚里有人瞎起哄：「末行丑骨也想夺魁？太阳打西边出来！」

韩泥铺席不理。不理像怂，也像稳。稳的人少开口；一开口，嚼舌的人就有料可加。他只检查泥封裂纹——裂纹一丝，他就补；补完了，才肯躺下。躺下不等于睡实，睡实了怕梦里洒手。手在梦里也得端着罐，端着才像活人还有路。

铁无言深夜来过一回，站在廊柱影里：「嘲听完了？」

「听完了。」韩泥说。

「听完就炼气，别炼嘴。」铁无言道，「嘴赢她一次，她能嘲你十次。手赢一次，她得哑半天。」

「我晓得。」韩泥摸罐，「哑不哑，试台见。」

铁无言走了。棚外虫鸣细，细得像针。针扎耳，他把仇和恩仍分开搁：嘲归嘲，粥归粥，汤归汤。搁混了，散先不正；不正，榜末行就真成污。污不得——污了刘婆那空碗也跟着羞。空碗来占位，是信他还能还；信丢了，粥再热也烫不进心。

他把罐又封一遍泥，封完对坛轻声：「正。见。」

清晨他又去坡下看了一眼榜。榜角被露水打湿，末行字更深了些。深不是荣耀，是钉。钉在纸上，也钉在人眼里。眼里人多，嘴里料就多；料多了他不躲，只把步子放慢——慢，才不像偷。

有个外门弟子指他脊背：「就是这丑的。」

「是我。」韩泥头也不回，「试台见。」

指的人噎住。噎住好。噎住了，他才能把罐子护得更紧。紧不是怕抢，是怕自己一个恍惚把泥封磕裂。裂了，正也散。散不得，正是他这张丑脸唯一能拿得出手的东西。

刘婆在坡脚远远望他，没喊。不喊像懂：喊会惹事，惹事会抖。她只把空碗换成装着热水的罐，等他路过再塞。韩泥接过，低声：「谢。」

「谢什么。」刘婆道，「喝完炼。炼正了，我才好等。」

见字挂在夜风里，像钩。钩着十二层，钩着复测，钩着帖。帖若落到坡下，大比影也跟着近；近了，还刘婆那一日才不算空话。空话他不说。他说不过白织月，也不想说——说赢不如炼正。炼正了，明日锤落试台，嘲声自己哑。

他把热水喝尽，掌心回温半分。温半分，榜末行也不那么刺眼——刺眼的是后面要见的那双手，不是纸上两个「丑骨」。

远处更鼓又沉一记，沉得像把「见」字钉进木里。木里有纹，纹里有路。路通向验香，通向夺魁，通向把新衣袖口多缝那一层的日子。日子未到，榜角白影还在风里翻。

翻就翻吧——翻到他眼前，他也只按罐，不按人。按人费劲，按罐保命。保命不是怕死，是怕恩断。恩一断，叶汤、刘婆粥、芽儿石，全变成笑话。笑话他不当。不当，就熬到锤子真正落下那天。
