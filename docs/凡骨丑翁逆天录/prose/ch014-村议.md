# 第十四章 村议

韩泥臂伤未愈，村议先起。

天阴，泥岗村像扣着一只瓮。他早起编筐，藤条绕膝，臂布勒紧，勒得疼——疼着，才记得狼口那夜不是梦。筐编了半只，里正锣已响，响得全村往祠堂聚。

马春花敲铜盆，盆声刺耳：「叶家寡妇勾引晦气，药铺藏邪！今日说清，不说清，全族连坐！」

村人聚祠堂前，男多女多，嘴多心少。赵绣春站阶上，像唱戏：

「叶青禾，你送汤送姜，图什么？图药铺名声？还是图绝户翁报恩？」

叶青禾站出，青布素，不跪。「图人该送。」她声平，「天冷，门缝大。送姜，送热。」

「热？」赵绣春笑，「热到铺子没人抓药？热到沈家被罚？热到全村晦气？」

周嫂帮腔：「寡妇不知羞！」

众人「嗯」成一片，像潮，潮往叶家淹。

韩泥坐末席，臂缠布，不语。语，成「一对」；不语，成「默认」——他选第三条：听，记。

赵绣春逼前：「你说，韩泥给你什么好处？药？钱？还是……」

叶青禾抬眼，一眼，冷：「他给我什么，我不知道。我给他一碗热汤，我知道。」

祠堂梁上灰落，落在叶青禾青布肩头，叶青禾没拂。拂，像认罪；不拂，像认人。

韩泥末席看着，指节白——白不是因为怕，是因为恩在眼前被人唾。

陈姑在人群后拢紧棉袄领，低声问邻人：「祠堂议什么？」

邻人答：「议叶家牵扯。少嚼，连坐。」

陈姑没再嚼。嚼多了，连坐。韩泥与她无冤也无恩，只晓得那人嘴利——嘴利不要紧，记不要紧，恩辱分列，不混。

「热汤？」赵绣春嗓尖，「热汤能当饭？能当药钱？」

「能。」叶青禾道，「能当人心。人心若都凉了，你抓药抓谁？」

祠堂静一息。马春花盆又敲：「嘴硬！赖管事说了，叶家若不认牵扯，迁铺！」

叶青禾唇白，仍不跪：「牵扯的是谣，不是我。」

「谣？」赵绣春指韩泥，「他绝户，克邻克亲，你靠近，不是牵扯是什么？」

众人附和，声大，大得像要把叶青禾压进地。

韩泥指节发白，仍不开口。开口，害她更深。他只把叶汤那笔恩按心，按出形，像按叶家脊梁。

里正再敲铜盆：「够了！再辱，记名！」

记名二字，像绳，勒住赵绣春半寸。她嗓仍尖，尖里缺后力：「记！叶家欠全村一个交代！」

周嫂忽然道：「要我说，叶家关铺三日，谢罪！」

「对！关铺！」

「绝户翁搬出村西！」

周嫂棒槌往地一墩，墩响，没砸人，嗓却硬：「关铺三日，娃咳谁治？汤……汤不算邪。我洗衣水早浑了，也没见邪！」

静里，有人「嗯」了一声，像潮退一寸。赵绣春瞪她，周嫂脸白，不退：「绣春，你饿不着，你别替全村饿。」

声浪顶到顶。顶得指节响，响在袖里，响不出声——满，满到里正不得不抬手。

韩泥这时起身，慢，跛，却直。「关铺。」他嗓哑，「病谁治？搬我，谣谁担？担我，不如担真话。」

祠堂阶下，有人抄「公审」二字，抄歪，歪得像心也歪着悬。韩泥末席看，不看抄帖人，看叶青禾背影——背影直，直得像秤杆。

赵绣春一愣：「你认邪？」

「我认恩。」韩泥独眼平，「汤是热的。我记着。谣烫不了这恩。」

他不辩奸，只辩恩。恩字落，像石进潮，潮未退，石已在。

里正抬手压：「今日议到这。再闹，报官。」

赵绣春脸青，想再尖，被里正一眼压住。尖收半寸，后力没了——没了，公审的缝又开一线。韩泥最后一个出祠堂，独眼扫过空阶，阶上还留着叶青禾站过的印，浅，真。

散场，散得不甘。不甘，就是钩：叶家未安，谣未止，怒未泄。

祠堂侧，有人低声问里正：「公审，开不开？」

里正板脸：「开。日子未定，理要定。定不了，村先乱。」

叶青禾在侧巷，没近，只站。站一息，转身回铺，背影直，像竖。

她停半步，没回头，声低：「韩翁，别认邪。认恩，够了。」

「嗯。」韩泥道，「恩我认。邪，等公审判。」

就两句。她步稳，回铺，帘动，秤砣落一声——落给全村听，铺还在。

回屋，心口叶汤那笔恩按实，按得像给叶家垫一块石：石不亮，承得住。

外头赵绣春党散，散时仍骂，骂声远，远里夹着一句「公审」，像锤未落，声先至。

祠堂檐下铜铃被风碰响，响一声，像替叶家秤砣应和。韩泥出祠堂，臂疼，疼里冷。冷不是怕，是忍——忍到公审，忍到还。

夜里只做一件事：把叶汤按深，把赵辱记着，把叶家背影记住——记住，将来还汤时，不让她孤。

叶铺灯终夜弱亮，弱亮像叶汤的影，影在，韩泥就不睡实——睡实像忘恩，他忘不起。

怒在村里蓄，蓄不爆，爆要等公审——公审来，叶汤才能当众立，立给全村看：汤是热的，不是邪。

韩泥独眼闭半息，再开，像把怒压进心——压进，不泄，泄害叶；候公审，才是还叶汤的地方。

散场后，村道仍沸。沸里，有人拽周嫂袖：「真关铺？关了三日，我娃咳谁治？」

周嫂脸僵，僵半息，嘟：「关的是晦气，不是药……」

嘟声虚，虚得像潮退一寸——退一寸，叶铺灯还亮，亮就是钩。

赵绣春党有人嚷：「绝户翁认恩？认恩就是认邪！」

韩泥担筐过，不停，只道：「恩是汤。邪在谣。谣你熟，汤你不熟。」

嚷者噎，嚷声裂——裂成窃语，语里「公审」二字又浮上来。

坛角微凉，腹里一线温，像瓮睡了，却听见「辱叶」二字——听见，不退，只记。记恩，不记逃。

里正路过，咳：「今晚水席，谁再辱叶，记名。」

韩泥「嗯」一声：「记。候公审。」

盐担老汉蹲阶下，嗓粗却低：「周嫂今日这话，公审上我帮一句。帮一句，不帮邪。」

韩泥听见了，不点头。点头，像拉帮。他只把叶汤那笔恩按深——深，等锤落。

赵绣春门内啐一声，啐碎在台阶上，碎得像人心裂半寸。祠堂阶下，有人抄帖，抄得慢，慢里却实——实，像全村第一次把「公审」写进心里。

他低声道：「汤是热的，得热着还。」

散场后棚下雨细，叶青禾撑伞走，伞骨旧，仍直。韩泥檐下收筐，雨声里听见秤砣又落一声，落像给祠堂铜盆应和。雨丝打在他独眼上，他不揉——揉，像心虚；心不虚，雨就打湿一点皮。

村议散，怒值满；舌留公审，心留恩辱——周嫂当众说汤不算邪，赵党裂半寸，锤还在前头候着。

韩泥回屋，汤字不在口，在心——叶汤深，像叶青禾没跪下的脊。脊在，公审就有骨。

他低声道：「辱叶这笔，不今夜算。候公审，当众还汤。」坛腹温一丝，应「立」字，不应「辱」字。辱记着，不泄；泄，害叶。

丑时，叶铺秤声又起，脆，短，钉住铺子没关，也钉住心口那笔恩。张麻婆骂童谣丫：「少学周嫂那嘴！」骂完塞门缝半块饼：「吃。编筐手不能抖。」

沈枯芽隔门问：「韩爷爷，汤烫不烫？」

「烫。辣好。趁热，才记得住。」

叶青禾在巷那头应：「记得住就好。」

韩泥没接，接轻了是讨。他藤条绕膝，勒出闷响，心口那笔恩更实——实了，手就不抖，还的时候才不寒酸。恩在，明日打水的手就不会抖；手不抖，还的时候才不寒酸。风从门缝过，带一点姜辛，辛得短，短得够记一夜。合眼，手仍按坛沿。沿在，根不散；人不问，坛也不答。瓮未醒，他不喊醒；醒在缘，不在雨夜这一更。恩在，手就不抖。

柱上条角白一线，三年约在怀。怀凉，掌温。
