# 第十七章 送药

村会前一日，风紧。

韩泥臂伤结痂，早起编筐，门墩多了一小包，布旧，结打得怯——像娃的手。里三层，外两层，最里是几株干药：止咳的，清热的，量不多，够熬三帖。纸角二字，歪，小：

别病。

不是叶家笔，是芽儿笔。芽儿送药，像送粥，像做贼——贼在泥岗村，是对晦气翁好。韩泥收药，不入腹，先闻。闻得苦，苦里一点甘，像叶家铺子的气。

「娃胆大。」他低声道。

药包布角磨毛，毛得像娃手心出的汗。韩泥把药叶摊在膝上，一叶一叶看，看叶脉，像看芽儿有没有退。藤条在膝上绕半圈，勒紧，再绕半圈——绕两圈，勒紧，留半寸尾，尾塞结里，跑不了。

编筐至晌午，筐底圆了，沿齐了。用指节敲了敲底，声闷，闷而实——实了，才配装药，配卖钱。他搁下藤条，往沈家方向望一眼，望得像确认豁口还在。

晌午，马春花嗓尖，找沈家麻烦：「芽儿又送！送药送晦气，你娘还要不要活！」

沈枯芽哭，闷在喉：「我……我没有……」

「没有？包上泥印，全村都看见！」

韩泥在巷角，没出。出，坐实。他只把药包分出一半，搁沈家豁口，放下便走——走快，像从没来过。墙内静一息，像有人拾起，没谢。谢字惹祸，不谢，恩也在。

马春花骂两声，见无人应，啐远，像一拳打空——打空，也是小胜：沈家还接得住，芽儿还肯送。沈枯芽娘在墙内咳，咳轻，像知道药回来了。

韩泥不叩墙，只把剩药包搁豁口——还一半，留一半，留给自己臂上未愈的肿，他不配全收。他自熬半碗，苦得皱眉，皱眉像把村会前的夜也熬进去——进去，臂肿退一线，心记深一线。

墙内芽儿声细，像对娘：「药……回来了……」

娘咳里夹半句：「别……再送……」

芽儿哽：「不送……娘咳……」

静。静里，韩泥已远——远，像没听见，听见了也只记：娃还肯送，点就不灭。

巷角有人贴条，条旧，写着「晦气勿近」。贴条人手抖，抖得像自己也怕条。韩泥瞥一眼，不撕——撕，成闹；不撕，当风。

夜里，叶青禾隔巷低声问：「韩翁，村会上……我怎么说？」

「说汤。」韩泥道，「热，送，该送。别辩奸，奸在谣嘴里。」

「你呢？」

「我认恩。」韩泥独眼平，「恩我担。担了，正好连。」

叶青禾「嗯」一声，步远，远里药铺灯仍亮——亮弱，像村会前的烛，烛不救嘴，救抓药的人。

午后，赖扒皮的人路过，棍尖点门扇：「晦气翁，村会前别生事。沈家那包药，别藏你灶角！」

韩泥开门，独眼平：「我生事？我编筐。」

来人瞄他腰，又瞄坛角：「腰里别藏东西。灶角也别藏。」

韩泥摊手：「藏心。心里恩辱分列。你要看，看脸。」

来人愣半息，骂「老疯」，走了——怕心，不如怕赖扒皮的令；令在，苗在。韩泥合袄，这是暗爽：仇在，他也不怕示心——示的是恩，不是邪。

傍晚，韩泥熬一帖药，不自己喝，装粗碗，搁叶家门阶。他不叩门，只留字，丑，二字：

趁热。

叶家与沈家，同一词。词不同人，意同一脉——热，是情，不是奸。叶青禾开门，见碗，见字，站一息，端碗回帘内。帘动，像叹，叹轻，像谢，谢不出声。韩泥远观，不近。近，害她。

他回屋，心口芽儿那粒米旁，又添药叶一笔——浅，真。「粥稀，药苦。」他低声道，「情不稀，不苦。」

坛腹微温，温得像恩又记在心里。记在心里，不是多欠，是多证：证芽儿没退，证沈家还活，证村会来之前，叶汤还立。坛角微温一丝，像吸了药气。韩泥按坛，按心。「记着。」

外头风硬，吹不凉那二字：趁热。药气淡，飘不进他的漏屋，飘进的是雨意——雨未落，意先至，像村会前的潮。

他自留那半包药，熬一碗，苦，入喉，臂肿似退半分——退半分够，够他明日还去集尾，还去卖筐，还去等村会锤落。芽儿的药叶在心，浅，真；真，就不白收。白收是施舍，他不做施舍翁，只做记心翁——恩仇都记在心里，灵根记性好，漏不得。

「趁热」二字，叶家收过，沈家送过，如今又从翁手回叶阶——回，不是还情，是还礼的形，形先立，情后存。村会风声紧，紧里这一点暖，像姜，辣，小，够他再撑几日。

巷口，张麻婆骂马春花，骂得响，却不去沈家闹：「闹娃，你算啥好汉！」

马春花尖：「送药送晦气，不该管？」

张麻婆啐：「管嘴不如管咳。咳死了，你赔？」

马春花噎，尖收——收，像今日送药的小胜：沈家还接得住，芽儿还肯送，嘴再尖，也尖不进豁口。

韩泥门缝听，不答。答，成牵扯。他只低声道：「药苦，恩不苦。记着了。」

集尾卖筐，有人见袖里药香，退半步，又上前：「筐还卖？」

韩泥道：「卖。药香在袖，不在筐。」

买主掂筐，仍付十二文——十二文落掌，像村会前还能活一日。回村，童谣丫远远看他袖，不敢近，近，像沾药香也沾晦气。韩泥路过，不停，只道：「药香不传染。传染的是嘴。」

公审前夜，风更紧。紧里，芽儿又摸豁口，摸空，摸得怯。韩泥早把药送尽，门缝只留一句，哑：「别病。病在娘，不在包。」

芽儿声细：「翁……药苦……」

「苦才治病。」韩泥道，「怕苦，别怕送。送是点，点在，你娘就活。」

芽儿「嗯」一声，步远，远像做贼——贼在泥岗村，是对晦气翁好，好得真，就得记在心里。

沈家墙内，娘咳轻，轻半分，够撑一日。一日一日攒，攒到村会，攒到还。马春花远处还尖，尖到叶铺阶下，见碗里「趁热」二字，尖收——收，像风又断一截。

丑时，他把药叶夹进旧纸，纸角再写二字，丑：别病。写给自己，也写给娃——娃若再来，看见字，就知道翁记着，不必再做贼。药气散尽，屋里只剩坛腹一线温。温像第四笔记心落妥——落妥，不示人；示人，成邪。

天将亮，风更紧，紧里沈家咳一声，咳得像给村会前又添一证：证娃还肯送，证娘还肯活，证药叶记心，不虚。韩泥合袄，独眼闭半息，再开——开，明日坐末席，备一句「我记着」，够了。

沈枯芽门缝塞一片姜，姜薄，薄得像娃也懂「村会」二字重。韩泥拾起，不尝——尝，像收谢；不收，只记。记在心里，娃的胆，也算一证。坛壁蒙尘里，极淡一道影游过，像半个符，像半个字，一闪，灭。灭得他以为眼花——眼花也罢，坛应了药气，应得他心定半分。

药叶记心，芽儿怯送。怯是真，送是恩——恩在，村会前心就不空；心不空，叶丫头证白才立得住。韩泥摸坛，腹温不退，像瓮睡了，理醒了半分。沈家豁口咳一声，咳轻，像证：日子定了，芽儿娘还能喘，村会就能辨。

他低声道：「药叶记心。恩因④，落妥了。」

韩泥擦坛沿三遍，沿净，心定。药泥味冲，冲得像恩不必甜。甜在叶汤里，冲在芽儿手里——手里有药，村会就不空。

叶铺帘动，似有人端碗，碗热，气苦里带一点姜辣——辣小，真，像「趁热」又立住一回。韩泥隔巷不闻，闻了，像牵扯；不牵扯，也是护。护到村会，护到叶丫头还能开秤。

明日巳时，他坐末席，备一句「我记着」，够了。沿口一线温，贴住虎口，脆，短，够撑明日村会。瓮未醒，他不喊醒；醒在缘，不在这一更。
