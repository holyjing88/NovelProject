# 第六十三章 瓮醒

沉丹宗杂役房，夜深。韩泥清完兽栏，洗手，水冷，指缝腥。五十九岁，入门半年，须发更白，背更驼，独眼仍利。

村里叫他「晦气翁」，丙九里叫他「丑杂役」，叫法不同，辱味一样。赖福今日又克扣，克去半袋麸皮。丙九里仍有人喊他赖扒皮——名不同，手一样黑。

韩泥不辩，只记在心里，也记在坛边。半袋少了，明日清栏还得干，干完手还得编筐。

外头赖福骂远：「丙九别嚷，嚷了加倍克。」

韩泥不答。答，费舌。同屋小杂役笑：「丑老头，认命吧。」

韩泥声平：「命在坛边。坛在，不认。」

小杂役一愣，没再笑，又探头：「半袋麸皮，赖扒皮也不怕天打？」

韩泥只答：「天打不紧。坛在，命在。」

小杂役没再嘴，嘴闭了，鼾声又起。小杂役含含糊糊问：「坛能当饭吃？」

韩泥：「不能。能当根。」

他回屋。门闩扣好，外头的辱进不来。屋小漏风，比泥岗泥屋强在不漏雨，弱在隔壁鼾如雷。

他摸坛沿，坛厚，承得住三代人擦过的手。床角摆着泥岗带来的腌菜坛。宗门不许私摆，他用破席盖着。入门时管事骂过晦气，他应一声，席盖更严。

他又压平最上一层席角，掌根着力，只怕压出响。席下陶腹凉，凉得凡，凡才藏得住。六十日里席一层层加厚，厚不为保暖，为挡管事的目，挡宗门的查，挡自己手去揭坛的冲动。

恩辱仇分列心里，不混。坛腹这几月微温过几回，温完又敛，候人来碰这一碰。

今夜丑时。韩泥没睡，盘坐床沿，油灯豆，豆光照坛。他吹熄半粒豆，豆灭一息，又点亮——点亮，要把丑时掐在掌里，不准邻床梦呓把这一更掐散。

邻床有人梦呓：「丑时了……还不睡……」

韩泥只答：「在。」

半只补单筐搁梁下，白日管事点过末席他的名字，名在末，活不能末。藤条干硬，韩泥绕膝两圈，第三圈勒紧，勒得疤边发白发痒。茧磨藤，藤找疤，疤在食指第二道旧裂上——冬测备检后就没合透，今夜藤刺一扎，血冒出来。

他甩手，血甩到破席上，席旧，吸了血，血渗到坛沿。不是有意滴，是活计扎的。血自己找路，找的是沿，不是席，也不是他喊出来的。

破席布纹竖起，吃住了血，血沿纹往坛脚走。席挡出巴掌大的暗印，挡不住细线——细线找沿，找沿不认理。

风从窗缝进来，灯豆晃一下，又稳了。他揭席，扯开坛口破布。

坛里空，坛壁蒙尘，内息尽敛，瞧不出半点仙气。本该如此，腌菜坛能有什么仙气？泥岗村那夜丑时，坛温一线；入门后时凉时温，总在丑时前后醒半分。坛沿蒙尘被豆光照出一圈细边，边旧，沿上还留着祖父擦过的旧迹。

今夜血从席渗到坛沿，一线朱红。血一落沿，坛壁颤——极轻，却是对 **他的** 血的颤，不是随便谁的血都能碰。

外头巡夜执事敲柱：「丙九，丑时了，熄灯。」

韩泥低声：「灭了。」

执事脚步远去。韩泥独眼一紧，心口突跳。

爹临终那夜浮上来：铁山手搭这坛沿，嗓哑：「坛别扔。韩家就剩你，坛在，根在。」

又翻出祖父老栓擦沿的冬夜；再翻出祖辈口传的「守瓮」二字——字不传外，传血，传擦沿的手。老耿经片里提过，祖上守过一只瓮，他那时只当故事听。三十八年了，他以为根是穷人家安慰绝户的话。

坛沿蒙尘里浮出极淡一道影，半个「韩」，一闪即灭。掌心跟着烫，血脉里有什么，与坛底沉睡多年的东西对上了。

不是随便谁的血。是韩氏一代代擦过的沿，是爹搭过手的沿，是他五十九年每日再擦的沿——守瓮人的血，碰守瓮人的瓮。

掌纹里细线亮起，灶下灰里一点红，沿脉走到腕，走到心口。心口不燃，只温，深过叶汤那一点。心记里「先还」二字，坛应了一声。「嗡——」

一声闷响，非雷非钟，在颅骨里。韩泥独眼一花，见坛底水光聚成一线，爬壁，爬出符形——丑，歪，孩童乱画一般，却亮。

符不是贴上去的，是从坛壁里 **长** 出来的；半个「韩」影被血喂饱，喂出第二笔、第三笔，钝得真，真得丑翁。

符形沿壁游走半圈，圈口如瓮唇，一线白光吞吐。吐的不是仙气，是 **认** ——认他掌按处，认他五十九年擦沿的手，认韩家最后一脉守瓮血。

韩泥屏息，独眼不眨，怕眨一下，这认就散了。亮一闪，灭。灭前符形投在窗纸上，影丑影真，像有人把「瓮」字拆开给他看——不全不要紧，认主要紧。

坛口吐出一口白气，冷里带药香、泥腥、腌菜坛旧日的酸，还有一丝说不清的古气。很远的仙界有人争过的那只瓮，在凡间腌菜坛的壳里，睁了半只眼。

韩泥换指再按。未破皮的指腹贴沿，符不亮，光不应——应了，仍是那道血线。换指不应，他才知道这认，认的是沿，不是喊出来的声势。

血线是钥，钥配了，瓮才肯醒。血脉认主，不是喊名，是沿上擦出来的认。手背起皮，火舔冰咬交替。他没躲。

瓮醒了——不是梦，不是丑时幻觉。坛壁符形隐去，余温灼掌，掌纹里多了一道路，通向还恩，通向省亲，通向泥岗那碗还没还尽的叶汤。

韩泥独眼盯住窗纸，符影已灭，灭处仍留一点暖，暖得像叶青禾送汤那日碗沿的热。「醒了？」他低声问。韩泥又补半句：「沿在，手在。」

无人答。坛沿一线温光，像爹的手又搭上来。韩泥按实，按到掌烫，烫得踏实。

里屋兽栏有夜鸟叫一声，哑。臾墟子是否在坛底、是否在将来开口，此刻不知——此刻他只要坛醒，命牵到了掌上。

外头鸡未鸣，沉丹宗夜禁脚步远。钱戾衡在玉牌里，在三年约里——约还在，瓮先醒，醒在血缘碰沿那一瞬，不急在这一夜。

韩泥把席盖回，盖严，像盖一个刚出生的秘密。秘密不能叫，叫了，赖福来，管事来，煞骨门旧债也来。

外头管事又喝一声：「丙九，还不静？」

韩泥只答：「在。」

他擦掌血，血入布，布黑，布上印痕丑，丑而真。「记着。」他对坛说，「瓮醒了。爹说的根，原来在这儿。路还长。汤还要还。」

他又低声补一句：「不认名，认沿。」

邻屋又有人嘟囔：「丑老头，又不睡……」

韩泥只答：「行。」

窗上符影余亮未散，隔壁小杂役眯眼半醒：「你那边……啥在亮……」

「油灯豆。」

「不像油灯……你手还滴血……」

韩泥盖席，席严，声更低：「编筐扎的。看错了。睡。」

小杂役哼一声，又嘟囔：「晦气……啥味儿……」

韩泥：「坛味。睡吧。」

韩泥不辩。辩轻，命重。窗上余亮散尽，屋里又只剩鼾声，一切照旧。

至宝仍肯叫他「腌菜坛」，不吓他——真名什么，仙界谁争，他不懂，也不追问。少年才追问。

鼾雷里，他按掌不松。克扣是压，瓮醒是释——释不在喊，释在认。

他低声对坛：「爹，根在这儿。我活长些，还恩。」

坛底极轻一颤，像应。掌纹那路还在，在就不怕明日麸皮再克——克的是粮，克不走瓮。

更鼓将尽，天边白一线。丑时尽了，坛腹温一线不退——不退，像瓮从眠里抬了半寸眼。路才长，长得够还。
