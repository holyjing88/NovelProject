# 第一百三十一章 荐帖在怀

复测散后第三日，山门晨鼓迟了半刻。

鼓迟不要紧，帖要在——帖在怀里，这一路才像第二部，不像一部尾韵。韩泥立丙九队末，衣襟里那张预选帖沉甸甸，厚得像刘婆新衣那包棉。棉旧，恩新；帖新，债序又落一笔。

执事验牌，牌歪，字丑，念得却真：「韩泥，炼气十三，夺魁。帖，收好。」

「收好。」韩泥应声。应声短，短才像常活。

验息册又添一行：炼气十四，登记。十四不是喜信，是钉——钉住大比路，钉住三年约，钉住钱戾衡那眼风。风在帖背，字淡，淡得像偷看；他不抬眼，眼要留来守坛。

赖福廊下过，脚步乱，乱像嚼不出晦气：「丑杂役也接帖？帖，太阳西边出！」

有人跟着起哄：「丑骨上台，台要塌！」又有人压嗓子：「压低点，管事记名。」

韩泥不辩。辩费舌；舌要留来听药峰派活，留来守「丑丹师」那块歪牌子。

铁无言立阶下三步，声低：「帖在怀，手别飘。」

「不飘。」韩泥说，「飘了，刘婆那笔白还。」

「还了的别丢。」铁无言又补半句，「丢了，我挡那半步也白挡。」

「不丢。」韩泥应。

铁无言没再多话。话在兽潮那日并肩，并肩续到今日——续不在嘴，在站。他站侧，像还半分仇谣。

白织月远廊过，袖风冷：「帖是预选，预选仍是运气。」

韩泥捧帖，帖稳：「运气大比见。见在台，不在舌。」

「台上见骨。」她唇角一挑，「骨丑，台尖，别摔。」

「摔不摔，掌见。」韩泥说。

她唇动，尖不过「见」字，袖香擦过檐角就散了。散不要紧，帖还在怀里硌着。

沈枯芽口信无字，只有半片枯叶——叶黄，像提醒：叶汤还在前，省亲还在远。远不要紧，刘婆已还；还了一笔，帖才配贴心口。

韩泥把枯叶夹进册页，低声道：「叶在。汤在。帖也在。」

秦霜木鱼轻敲：「帖至，心别邪。邪了，鼻下香飘。」

韩泥不答，答在心里。心里四笔分列：帖、药峰、坊市、叶汤——不混成一锅。

管事弟子扬声：「夺魁者，明日药峰派活。丑丹师，别躲。」

「不躲。」韩泥说。躲像邪；邪了，管事记簿「丙九韩泥，躲活」，路就断。

旁有杂役嘀咕：「夺魁了还上山摸渣？丢人。」老耿挑水路过，水洒稳，声哑：「丢人？丢人的是拜暖。帖在，别拜暖。拜暖，路断。」

「不拜。坛在，人在。」韩泥说。

「人在就好。」老耿点头，「在了，药峰才像活，不像罚。」

回杂役屋，他按坛。坛腹温一线，贴掌，像应「帖」字。应，不应喜；喜了，手就飘。

邻床少年悄问：「你真接帖了？」

「接了。」

「接了，大比不怕？」

「怕。」韩泥说，「怕也接。不接，三年约白挂。」

「约说什么？」

「说站钱戾衡台前，才不滚。」韩泥摸玉牌——牌凉，烫着那约。

少年噎住，半晌又问：「那钱戾衡是谁？」

「台前那口气。」韩泥答，「气在，手更要稳。」

同棚杂役赌大比：「丑骨上台，太阳西边出。」有人笑他，有人等他摔。他不接茬，把藤条绕过膝盖，编筐半刻——编筐钱买米，米在，恩才还得上。

藤刺扎掌，他偏手，刺浅些。浅些沿才净；沿净，坛才藏得住。

刘婆端粥，仍多半勺：「接帖了？接了别飘。」

「不飘。」韩泥点头，谢轻，恩重。

她又补一句：「药峰活重，明日别空肚。」

「不空。」他应。空肚，手飘；飘了，帖像纸，纸挡不住台前那口气。

「粥里姜辣。」刘婆盯他独眼，「辣一口，记得还。还过的，别当玩。」

「记得。」韩泥把半勺喝尽。姜辣进心口，像提醒：帖不是奖，是债序第二笔。

丑时，练鼻下静。粟壳香淡，一字不落。落齐，十四层这一日记进心里。

更鼓沉，他望坛，独眼平：「帖在怀。记着。」

记着，是丑翁本事；本事在忍，在还，在叶汤那碗还没还尽的烫。

外门风硬，帖角掀半分，掀出「言伯钧」三字影。影在字后，像仇在路后。路后不要紧，药峰在前。

赖福廊下又啐：「在不在？」

「在。」韩泥只答。

「在什么？在装？」

「在守。」韩泥道，「守坛，守帖，守还没还的汤。」

啐的人噎住。噎住也好。

他低声对坛：「明日药峰。手在，路在。」

坛不应字，只温。温像应「十四」——层在，帖在，就不滚。

管事夜巡过，灯火照簿，记「丙九韩泥，帖在怀」，字细，细得像小胜。胜在忍，不在辩。

散队后他过坡下石阶，阶湿，湿里数息——数帖，数层，数债序。序在，步就不飘。

验息后，他理怀内册页，页黄，字丑，丑而真：炼气十四，登记。登记不是贺，是钉。钉住了，大比才像活，不像运气。

钱戾衡名在帖背，背字淡。韩泥拇指按住淡处，按得实：「眼风在。手更要稳。」

稳了，明日分拣才像积势，不像罚。罚不要紧，活才像第二部开笔。

白织月再从远廊过，没停，没讥。没讥也是风——风过，他只把帖往怀里按深半寸。

铁无言离队前又问：「怀沉不沉？」

「沉。」韩泥说，「沉的是序，不是惧。」

「惧也行。」铁无言道，「惧着守，比喜着飘强。」

「懂。」韩泥应。

天边白一线时，他把木牌贴胸。牌凉，帖温——温在序，不在惧。惧在，也要守；守完，才有药峰那一日。

巷口卖炊饼的汉子招呼：「魁爷，赏个脸？」

「不是魁爷。」韩泥摇头，「杂役。饼钱照给。」

「照给就好。」汉子笑，「照给，比赊账的少年强。」

他把两文放下，接过半个饼，掰开热气。热气里没有嘲，只有面香。面香进肚，帖更沉一分——沉得像日子还得过，过完才谈台上争。

柱上条角白一线，三年约的影还在。影不吓人，吓人的是手飘。他握拳又松开：不飘，就接得住明日药峰的臭与苦。

叶青禾门缝灯未亮。未亮像避，脉里培元仍温。他不近灯，近了像惹她。他只在心里把汤按实：汤热，帖沉，下一程是药峰。

侧门驿亭风硬，他听风，风过，记在心里。风里没有嘲，只有山门帖角轻响。

废渣巷日斜，他验渣，渣正，手才静。静了，帖才压得住。

药渣场角，他把剩渣摊匀，匀得像秤。秤平了，手才静；静了，帖才压得住心口那一下硌。

同棚有人夜里还赌：「大比台上，丑骨第几？」

「第几都行。」韩泥说，「第几也得手在。」

「手在就能赢？」

「手在才不滚。不滚，才谈赢。」

赌的人散了。散了也好，散了屋里才静得下坛温。

他按心口，低声道：「汤还是热的。」

六字落，像路标——标向药峰，标向挂牌，标向大比台前不滚。

编筐棚借半刻，藤条绕膝，膝上旧疤发紧。紧得像提醒：帖新，骨旧；旧骨接新帖，接不住就碎。碎不得。碎了，刘婆新衣白穿，叶汤白烫，铁无言那半步也白挡。

管事弟子远处又喊一遍：「明日药峰，卯时点名！」

「到。」韩泥应。应声落进雾里，雾冷，帖仍热。

验息册合上时，执事多看他一眼：「十四了。别漂。」

「不漂。」韩泥应。

「帖丢了，重领难。」

「不丢。贴胸口。」

执事挥手让他离开。离开台阶，夜露湿鞋。湿不要紧，湿提醒他脚下还有泥——泥岗的泥远着，叶汤的烫近着，药峰的灰明早就扑脸。扑脸也得去。去了，才像接第二部，不像坐在魁字上晒太阳。

沈枯芽夜里又远远站一站，站一会就走，像怕沾光。韩泥不招手。招手像显摆；显摆坏芽，也坏汤。

山门风再起，帖角轻撞衣扣，叮一声。叮声像钩——钩住药峰派活，也钩住他还没走完的债序。序未完，人就不能停。停了，帖会凉；凉了，大比台上那口气就散。
