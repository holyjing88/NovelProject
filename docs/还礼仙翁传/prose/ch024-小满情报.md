# 第二十四章 小满情报

雨后，青岚石道洗得发亮，像刚擦过的剑。

顾小满登门，未跪，立，怀里捧册，册厚，厚得像她这些年在杂役堂攒下的胆。册边磨毛，角起卷，像被掌心的汗浸过许多夜——记比告状难，她记了三年，才敢捧到长老门前。

莫长春在书房接册，翻页，一页一页。孙福历年克扣杂役灵石、口粮、例份，条目清楚，旁注日期，有的旁注还画小小半块灵石，像画给自己看，提醒恩从哪来。

「长老不查，小满也会记。」顾小满低声。

莫长春合册，抬眼：「记住比告状难。你做到了。」

顾小满脸红：「小满……怕长老嫌琐事。」

「琐事堆成山，才是宗门根。」莫长春把册收入袖中隔层，「册子我留抄本，原本你还拿着。哪日堂上需要，你拿得出来，不必跪。」

顾小满点头，像接第二张除魔符。她退到门边，停住，忽然道：「长老，手帕定身那日……小满在人群里。小满不怕魔少，小满怕长老九月不够。」

莫长春笑：「九月不够，够送；不够还，簿会记。」

顾小满不懂簿，却懂「记」字，像记半石，记除魔符，记今夜册子。

她走后，簿在袖中微烫，字成：

**记住难，告状易；忠在此册。**

莫长春望雨停后的云，云缝里露出一线青岚峰影，像根还在天上挂着。

杂役堂，孙福苦役未归，堂里换管事，新管事姓李，说话客气，客气里仍怕——怕莫长老，怕那本册子，怕半块灵石的故事再被翻出来。

弟子私下仍传「莫长老勾引杂役」，传一句，便被顾小满冷眼压回去：「宗主裁过，再传，去执法堂。」

传话者噤声，像谣言终于碰上一块半石，磕碎牙。有人不服，嘟囔：「一个杂役，凭什么压人？」

顾小满把册子往案上一放，册子声不重，却响：「凭这个。你也能记，记完了，再来压我。」

堂里静了。静里，有人第一次觉得杂役堂不只有浆洗与苦役，还有一本能上台面的账。

孙福旧党暗中嘀咕，嘀咕被顾小满听见，她只一句：「孙福在苦役，册子在堂上。谁克扣，谁入账。」

嘀咕者噤声，像克扣也怕「记」字。

赵玄机路过，听见，未讽，只拂袖走，袖中已无培元私囊，走得轻了一分。他走过顾小满身侧，停半息，低声：「记得好。」

顾小满一愣，赵已走远，背影硬，像不愿人看见他软了这一句。

林挽月远立门外，瞧见，心口复杂，复杂得像一根线松了，又不敢全松。

夜，东脉方向钟响比往日急。

莫长春立院中，杖点地，像给远方敲节拍。簿无新字，袖中册抄本却沉，沉得像把孙福的克扣、杂役的胆、青岚的根，一并收进袖里。

柳青鸢传讯来，简上二字：「备战。」

莫长春回简二字：「已备。」

简下，他望东脉，稻浪声远，像半块灵石拼回的人间，正在等一场更大的阵。

司库听闻册子，面色变，变得像怕半块灵石的故事再演。赵玄机路过司库，低声：「堂上见。」

夜雨又至，顾小满在杂役堂抄册，抄得像抄自己的胆，抄到指节白，白得像半石又长出一块。

霍镇山闻讯，骂：「孙福那厮，早该入账！」弟子围杂役堂，围得像等一场小堂辩，辩的是克扣，不是谣言。

魏无涯檐角观气运，观册子入袖，低笑：「记字，比杀字难。」

他低声，像钩：

「册子入账了。下一声钟响，该把人都赶到田上去。」

风大，钟声急，像催阵，也像催命——催的不是九月，是根。

大因果还在远处，像妖潮，像赤焰——册子只是小账，东脉备战才是大还礼的前奏。

孙福在苦役地听讯，面色白，像听见册子二字，也听见自己的克扣将被翻。他嘀咕：「杂役也敢记？」旁人瞪：「你看册，别看嘴。」

莫长春立院中，杖点地，像给远方敲节拍。簿无新字，袖中册抄本却沉，沉得像把孙福的克扣、杂役的胆、青岚的根，一并收进袖里。

风大，钟声急，像催阵，也像催命——催的不是九月，是根。册子入账了，下一声钟响，该把人都赶到田上去。

顾小满在杂役堂抄册，抄到指节白，白得像半石又长出一块。新管事姓李，说话客气，客气里仍怕——怕莫长老，怕那本册子，怕半块灵石的故事再被翻出来。

围观弟子在杂役堂外踮脚，踮得像等册子上堂——册子未上堂，先上了胆。有人低声：「杂役也敢记？」另一人答：「记字，比杀字难。」
