# 第二十五章 卷末集结

青岚门备战东脉，钟响比往日急，却不再像催命，像催阵。

东脉灵田，镇妖司列阵，霍镇山盾立前方，声洪传田：「民不散，根不拔！迁屋不迁心！」

民户拖家带口，却不乱跑，像被这句话钉在田埂上。老农问：「霍长老，妖来了往哪躲？」

霍镇山吼回去：「往阵后躲！阵在，你们在；阵破，田也没了！」

柳青鸢巡视阵眼，青衫沾泥，不避，像宗主也下地。她指一处水渠，令弟子填埋又掘开，像给阵脚留活路。弟子问：「宗主，为何不令民先撤？」

柳青鸢淡道：「民撤了，谁守田？田无人，战为何战？」

围观弟子窃语，窃语里敬，也怕——怕三日之期，敬的是宗主肯沾泥。

莫长春袖空，杖在塔中，立田埂，望杖，望民，望远处赤焰旗残影，笑：「礼还没送完呢。」

霍镇山回头：「莫长老，这一仗你要上？」

「老夫上不上，杖得上。」莫长春道，「杖扫过妖骑，再扫，扫得动。」

苏念慈率丹堂弟子送伤药、培元、回春，车队过长，像一条丹香铺成的河。她经过莫长春，停半息，未唤师兄，只递一瓶培元：「八月了，别省。」

莫长春接瓶，笑：「好丹给你，是应当；这瓶给老夫，是你收得下九月。」

苏念慈耳根热，转身走，像把丹堂夜语那夜的话，在田埂上还了一半。柳青鸢远处看见，眸光微动，不妒，只担——担师兄，担苏首座，担东脉。

顾小满在杂役队里搬阵材，怀里仍揣册子抄本，像揣第二张胆。她抬头看莫长春，莫长春也看她，目光一触，像说：记着，比跪有用。

午后，赤焰旗又在远山露头，露头便隐，像试探，像催。

弟子惊呼，阵脚微乱。霍镇山盾亮，骂声更洪：「来啊！霍老子等着！」

莫长春却不急，杖点田埂，声轻，却清晰，像说给全场听：「他们急，我们不必急。急的人，先露破绽。」

全场静一息，静里，慌意退了一寸。赵玄机在执法堂队里听见，指节叩铁牌，一声，像把这句话记进铁里。

魏无涯在远处檐角，扇开一线，观气运，观田上民户，观莫长春杖尖，低笑一声，像看一枚子将落。

夜，灵阵盘预热，光在田上铺一层薄金，像给根盖被。

莫长春立盘边，塔壁寿元行跳：七月。

他瞧见，不慌，像瞧见季节换牌。柳青鸢来传话：「师兄，宗主令，你守盘边，不必入阵心。」

「好。」他应，「阵心给你们。老夫在门口坐着，像守炉。」

他不在阵心，只坐盘边石阶——不坐主位，像给阵守门口。

杖中竹纹游动，气音轻：

够。

莫长春笑：「够就好。」

风过灵田，稻似浪，浪里有人家灯火，像半块灵石拼回人间。远处赤焰方向，云低，腥风一线，像第二卷的人情债，已催到门前。

弟子围盘边窃语：「寿元七月了……」「莫长老还笑得出来？」「笑的人，杖才稳。」

魏无涯在檐角观阵，观民，观莫长春杖，低笑：「根还在。」

民老问莫长春：「长老，妖来了，我们往哪？」莫长春指阵后：「往人心齐处。齐，比符管用。」

赵玄机率执法堂巡阵，与莫长春目光一触，一触即分，像妒还在，嚼舌却止。

霍镇山骂：「赤焰旗又露头！来啊！」莫长春笑：「让他们探头。头探多了，颈就露。」

司库验门里实价，册上多一笔棉衣、一笔符纸，笔笔有莫长春三指砍价的备注。执事低声：「长老真记门里实价？」司库答：「宗主令，认。认恩，才认根。」

他低声，像钩：

「七月了。够送，够还，够把该来的人，统统赶到灯下。」

苏念慈送药车队过长，过长像丹香铺成的河，河过田埂，田埂上宗主沾泥，泥里像根还在。

顾小满搬阵材，册子抄本揣怀，揣得像第二张胆，胆在，阵材才稳。

霍镇山盾立前方，吼：「民不散，根不拔！」

莫长春坐盘边石阶，像守炉，也像守门口，低声：「集完，坊市要逛。逛比集轻，轻里藏线。」

集结后，各脉忙乱，忙乱里仍有人围观莫长春巡行，巡行像看九月老头还能送几礼。他不坐，只记，记脚步像记在心里——识海深处，万缘塔影一荡。

周德海赔笑，笑洒，洒得像怕集结也洒到自己。他遣人探坊市、探百宝阁，探得像怕老头又买实价藏线。

霍镇山按刀，刀稳，稳里战意仍热。营中有人问：「莫长老不上阵心，只守盘边？」霍镇山吼：「守盘边，也是守根！」

苏念慈开炉，炉旺，旺得像把集结炼成丹香。她递培元时耳根热，热得像把丹堂夜语在田埂上还了一半。

顾小满守册，册厚，厚像半块石又硌深。她搬阵材，目光与莫长春一触，像说：记着，比跪有用。

魏无涯檐角低笑：「根还在。」民老又问：「长老，妖来了，我们往哪？」莫长春指阵后：「往人心齐处。齐，比符管用。」

午后赤焰旗远山露头又隐，弟子惊呼，阵脚微乱。莫长春不急，杖点田埂：「他们急，我们不必急。急的人，先露破绽。」

夜，灵阵盘光薄金，寿元七月，莫长春不慌，像季节换牌。杖中竹纹游，气音轻：够。他笑：「够就好。」

他低声，像钩：「七月了。够送，够还，够把该来的人，统统赶到灯下。」

风过灵田，稻似浪，浪里灯火，像半块灵石拼回人间。远处腥风一线，像第二卷人情债，已催到门前。

赵玄机率执法堂巡阵，与莫长春目光一触，一触即分，像妒还在，嚼舌却止。弟子围盘边窃语：「七月了……笑的人，杖才稳。」

柳青鸢巡视阵眼，青衫沾泥，不避，像宗主也下地。霍镇山骂：「赤焰旗又露头！来啊！」莫长春笑：「让他们探头。头探多了，颈就露。」

司库验门里实价，认莫长春三指砍价备注。执事答：「认恩，才认根。」

集结后，各脉忙乱，忙乱里仍有人围观莫长春巡行。他不坐，只记，记脚步像记在心里——识海深处，万缘塔影一荡。

周德海赔笑，笑洒，洒得像怕集结也洒到自己。霍镇山按刀，刀稳，稳里战意仍热。

苏念慈开炉，炉旺，旺得像把集结炼成丹香。顾小满守册，册厚，厚像半块石又硌深。

午后赤焰旗远山露头又隐，莫长春不急，杖点田埂：「他们急，我们不必急。急的人，先露破绽。」

夜，灵阵盘光薄金，寿元七月，莫长春坐盘边石阶，像守炉，也像守门口。杖中竹纹游，气音轻：够。他笑：「够就好。集完，坊市要逛。逛比集轻，轻里藏线。」

民老又问：「长老，妖来了，我们往哪？」莫长春指阵后：「往人心齐处。齐，比符管用。」

魏无涯檐角低笑：「根还在。」赵玄机巡阵，与莫长春目光一触，一触即分，像妒还在，嚼舌却止。

苏念慈递培元，耳根热，像把丹堂夜语在田埂上还了一半。顾小满搬阵材，册子抄本揣怀，揣得像第二张胆。

莫长春立田埂，笑：「礼还没送完呢。」霍镇山回头：「莫长老，这一仗你要上？」「老夫上不上，杖得上。」

弟子围盘边窃语：「寿元七月了……」「笑的人，杖才稳。」柳青鸢巡视阵眼，青衫沾泥，不避，像宗主也下地。

莫长春低声：「七月了。够送，够还，够把该来的人，统统赶到灯下。」

风过灵田，稻似浪，浪里有人家灯火，像半块灵石拼回人间。远处腥风一线，像第二卷人情债，已催到门前。

集结令下，东脉风硬，硬里像把卷一最后一笔恩，推到每个人面前。
