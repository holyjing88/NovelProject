# 第三十八章 金丹成

第七夜末，雷小，像天公肯给丹堂留一点面子——随后，丹成。

药香先于人声到了山门，温甜里掺雷息，像七月暑气被滤过一遍，又掺极淡焦气，像炉盖留的三道细纹，此刻被地火余温一烘，化成全宗都能闻见的讯号。

外门弟子在阶下踮脚，踮得酸，仍不肯退：「苏首座……成了？」

「听说是三品……」

「莫长老又在门口吧？」

「廊柱旁，不进门——跟七夜前一样。」

议论被风扯散。丹堂门扉半掩，里头静得出奇，静得像炉心还在跳，跳得只有金丹境才听得见。

苏念慈捧托盘出来的时候，袖口沾炭灰，鬓发散了几缕，眼底血丝，可那双眼睛亮得吓人——不是狂喜，是熬到极致之后，终于敢相信自己没辜负那一炉火的笃定。

金丹卧白绸，拇指大小，淡金薄膜流转，像晨露，一碰要碎，却实实在在托在掌心。

莫长春拄杖立在**丹堂门外廊柱旁**——不坐石阶，不坐主位，只让门内丹光能照到他半边灰袍，像守门，也像陪炉。

他昨夜也在。小丹劫起时没进门，只把清心符贴门框，像给炉盖加一道俗世能懂的锁。此刻他看着她，没说话。

苏念慈走到廊下，举托盘，让金丹反光落在他须眉上。

「师兄。」她嗓子哑，却笑，「成了。」

莫长春指尖在托盘边停一停，没碰那颗丹，只问：「几颗？」

「三颗。一颗验品，一颗封存，这一颗——」她顿顿，像把许多年没说出口的话在舌头上理了一遍，「回赠。不是还债。」

他接过，掂了掂，轻得像玩笑：「二品炉，三品丹？」

「你给的残页，你送的渣，你门口七夜又七夜——」她下巴微扬，露出一点少年时的倔，「我总不能还你二品货色。」

莫长春哼一声，像笑，又像叹：「口气倒大。」

识海中塔影翻过，丹道一栏淡金：**三品**。字迹与掌中金丹同息。苏念慈没看塔壁，只看他。

廊外弟子越聚越多，却不敢噪。

有人小声：「三品……苏首座要扑怀了吧？」

苏念慈耳根一热，没扑向议论，扑向他——额头抵肩窝，灰袍与丹袍相贴，药香与松香缠在一起。托盘搁在石栏上，双手攥他袖角，指节发白。

「我等了三十年……」

她声音闷在他肩头，颤，却清楚。

「丹炉还开着呢。」

莫长春僵住。杖滑半寸，磕廊石。手落她发顶，极轻按一下，像按一炉刚成的火，怕重手惊碎。

「……嗯。开着。」

门外脚步停住，不重，很稳，像有人刻意放轻鞋底，却仍瞒不过金丹境的耳，更瞒不过门外那人自己的骄傲。

柳青鸢未入门，声冷如秋晨的露：「丹堂首座，注意体统。」

苏念慈肩一颤，像被针扎，却未立刻退。她抬头，目光越过他肩头，正对门缝里的影。柳青鸢青衣，宗主令在腰，眼底一瞬极快的光——像妒，又像痛，更像咽回去的担。

苏念慈笑，笑里带愧，却不躲：「宗主说得是。体统。」

她退半步，整理衣襟，首座威仪又穿上，脸红到颈根，眼神却不再躲师兄的眼。

莫长春杖点廊石，转向门缝，声平：「首座破境，可喜。验丹文书半个时辰后可至。」

门外静一息。脚步声远去，没有停。

风把门缝吹开一线，炉内细火仍燃，像一句未说完的话。

苏念慈低声：「她走了。」

「骂了。」莫长春淡淡道，「体统二字，比骂狠。」

她噗嗤，又收住，首座模样：「我去封丹衣。师兄……脸也洗。」

她笑，推门回炉。门合上前回头一眼，像杂役院外送药的小师妹，又像三品丹师苏首座。

丹堂阶下，验品弟子捧验丹文书上山，路长，像把「成」字也走长。有人跪，有人拜，传「私授」者被令罚，罚不及敬快。

林挽月罚役途中，听「三品」与「扑怀」并传，唇白，像终于看懂：体统是戏，证字才是路。

霍镇山按刀，刀稳，声洪：「金丹既成，东脉也有胆！」

廊外弟子炸开一层欢呼，欢呼里慕，妒，怕——怕青岚门真还有底，怕九月老头还能陪炉七夜。

赵玄机立远，铁牌在腰，像等败，又像等成。败没来，来的是更浓丹香，香压过妒火半寸。他最终只说一句：「首座，辛苦了。」

魏无涯檐下观成丹一瞬，扇合，低笑：「重的推给她，轻的才送得出。送得出，才还得起。」

成丹之后，苏念慈没有扑进任何人怀里——她只把托盘举稳，像举一门规矩：丹成，先谢炉，再谢人。莫长春终于迈半步，半步又收，收得像怕进门就坏了礼：「收好。金丹不是谢礼，是下一段恩的开头。」

外门弟子比划定身三息，内门弟子摇头：「扑怀那一下，比三品丹还响。」「像戏。」「戏？你上去接宗主一句体统试试。」

莫长春不辩，只抚袖。塔壁浮半句：**寿元四月**；又起一行：**丹成有时，回赠有时。**

苏念慈捧丹立阶，未扑怀，却把托盘举稳，举得像举一门规矩——规矩比扑怀难，难在担，不在妒。

赵玄机立远，铁牌在腰，妒火被丹光压矮半寸，矮里仍像藏下一招。他最终只道：「首座，辛苦了。」

柳青鸢脚步在远处停，停得像体统也将至。她未近，只遣执事传话：「围金丹，不围嘴。」

验丹文书半个时辰后至。执事捧匣上山，匣开，三品丹样卧白绸，像把全门的疑也卧进绸里。周德海抚须笑：「样既出，该入公库，体统才正。」苏念慈不接话，只把样丹举高一寸，举给廊下万人看：「样在此，人在此。库是门的，丹是炉的。」

莫长春杖点阶，声轻：「围样，不围嘴。嘴嚼体统，嚼不过三品光。」

全场一寂，继而炸开——炸在阶下，炸在每个等着看「扑怀」的人眼里。扑怀来了，体统也来了，体统比扑怀狠——狠的是担，不是妒。

丹香未散，体统二字已在廊外等着——等着把成丹，走成门规。

苏念慈看他，看他眼里妒火还在，却矮了半寸：「赵执事，执法堂清场，清得不错。」一句话，像把太阳打西边出来又按回东边。

莫长春念：「开着就好。炉开着，门就开着。」金丹成那声，比钟还响，响得全门一静，静里一沸。

验品弟子捧匣上山，匣开，三品丹样卧白绸，像把全门的疑也卧进绸里。有人跪，有人拜，传「私授」者被令罚，罚不及敬快——敬快得像丹香压过流言。

周德海抚须笑：「样既出，该入公库，体统才正。」苏念慈不接话，只把样丹举高一寸，举给廊下万人看：「样在此，人在此。」

莫长春杖点阶，声轻：「围样，不围嘴。嘴嚼体统，嚼不过三品光。」全场一寂，继而炸开——炸在阶下，炸在每个等着看「扑怀」的人眼里。

成丹之后，苏念慈没有扑进任何人怀里——她只把托盘举稳，像举一门规矩：丹成，先谢炉，再谢人。莫长春终于迈半步，半步又收，收得像怕进门就坏了礼。

柳青鸢脚步在远处停，停得像体统也将至。她未近，只遣执事传话：「围金丹，不围嘴。」验丹文书半个时辰后至，执事捧匣上山，匣开，三品丹样卧白绸。

他念：「开着就好。炉开着，门就开着。」

丹香未散，体统二字已在廊外等着——等着把成丹，走成门规。
