# 第五十八章 血线将至

试手日，卯时前。韩泥未睡实。夜里在心里盘点七笔。分列，不混。

符怀暖一丝，坛腹温一线。

坡下，秦霜最后一遍：「手抖，丹歪，滚。丹歪不要紧，手不能歪。」

试手凝丹，只验手，不验灵根色。色末，手抖才滚。

韩泥立末排，独眼平。铁无言验鼻：「静带到手上。」

丹炉前，他手不抖。手稳，丹形丑，丑而稳。稳，像辨香那日鼻下静。

执事记功，不记灵。管事「嗯」一声：「麸皮不克。」

赖福脸青，退半步。韩泥不喜，不喜像少年。他只记：手稳了，路又宽半分。

同批少年碰他袖：「你真过了？」

「过了。」韩泥说，「过的是手，不是灵。」

「灵根末，手正，正就够。」铁无言声低，只对他。

秦霜目光扫过队尾，声冷：「再嗓，记名。」

队散。赖福啐地，啐得虚。韩泥不回头。回头像认谣，谣借腥，他不碰。

午后，编筐补单，藤刺深扎，扎透旧茧，血珠饱满。

他甩手，血甩向席盖，席吸了，血未至沿。未至，像血仍不渗沿那几日。

他盯掌，盯席，盯坛角。角静，沿净，净，像瓮仍眠。

他低声道：「血来了，沿不接。不接，不是怕，是等缘。」

刘婆端姜汤，姜薄，热：「试手过了，别傲。」

「不傲。」韩泥说，「傲，符自灭。」

管事路过，验手：「手抖否？」

「不抖。」韩泥说，「抖是心飘，心不飘，手不飘。」

他盯席边，席边湿一线，湿不到沿。不到，像瓮仍眠。

铁无言路过，声低：「手稳，沿净。净，像等缘的人。」

韩泥「嗯」一声，只答：「等。」

等，不是拜血祠，不是主动滴血。是活计扎，血找席，席挡沿，沿等韩氏守瓮血那一误。

夜，符怀暖，药涂裂口，血止。止在沿前，止得像规矩。

坛温加剧，加剧，像将醒未醒。

丑时前，沿仍净，手稳，心记着七笔。他按席，席厚，厚像守了六十日的席。

他低声对坛：「血在席，不在沿。沿在眠，我守席，守沿，守不主动滴血。」

坛不应字，只温一线，又凉。凉，像应「等」。

他低声道：「血线将至，沿不接。不接，才接得住瓮醒那一误。」

邻床问：「你手咋了？」

「扎藤。」韩泥说，「血在席，不在沿。」

「沿是啥？」

「沿是坛边。」韩泥说，「沿净，才接得住缘。」

他盖席，盖严，严像藏私。私是坛，坛是命。

坡下霜白，柱上无新条。他摸木牌，与记功在并放。并放，像命与规矩挨着。

秦霜问心时，他声哑，声平：「为还。还，先还烫。」

秦霜「嗯」一声，不夸，只点头。

试手后，他仍末排。编筐补刺，藤刺浅些。浅些，血线可控。

管事验手：「试手过了，手还抖否？」

「不抖。」

他独眼平，手不抖：「血线将至，沿不接。」

丹形丑，丑而稳。脸丑，手正，正就够。试手过了，路宽半分。宽，不是灵根好，是七笔没断。

血到了席，没到沿。沿在眠，血不主动碰，这是铁规。

卯时前，坡下霜白。霜贴石阶，石阶滑，滑不要紧，站要稳。稳，才验得过试手。

静，才配末排。

」克粮，克不走还路。

试手坡下，同批少年抖，抖像怕滚。韩泥不抖。抖是心飘，心不飘，手不飘。

午后编筐，藤刺深扎，血珠饱满。饱满，甩向破席，席吸尽，沿仍净。净，像瓮仍眠。

止在沿前，止得像道线。线内席，线外沿。

醒在缘，不在人催。催，像拜邪。

丑时前，他按席，席厚，沿净。净，才接得住瓮醒那一误。误在缘，不在心急。

清晨，编筐末藤，藤刺深扎，血珠饱满。甩手，血甩破席，席吸尽，沿仍净。

试手坡下第二遍，他手仍不抖。不抖，丹形丑，丑而稳。稳，执事只记手。

队散后，赖福啐得虚。虚，像谣借腥。腥，他不碰。不碰，路才净。

午后第二单编筐，藤刺深扎，血珠饱满。饱满，甩向破席，席吸尽。

他盯席边第二遍，席边湿一线，湿不到沿。

管事第二趟验手：「手抖否？」「不抖。」韩泥说，「抖是心飘。」

刘婆第二趟端姜汤：「试手过了，别傲。」

铁无言第二趟声低：「手稳，沿净。」

夜第二遍，符怀暖，药涂裂口，血止。

坛温加剧第二夜，像将醒未醒。催，像拜邪。

他按席第二遍，席厚，沿净。

清晨第二趟编筐，血甩破席，沿仍净。净，像瓮仍眠。

试手后第三趟搬柴，肩稳。稳，像坡下练出来的。

编筐第三单，藤刺扎，血珠饱满。甩向破席，沿仍净。

管事第三趟：「手还抖否？」「不抖。」

刘婆第三趟姜汤，热。韩泥喝尽，不傲。傲，符灭。

铁无言第三趟：「沿净，等缘。」韩泥只答「等」。

夜第三遍按席，席厚，沿净。净，才接瓮醒那一误。

他低声第三遍：「血线将至，沿不接。」

坛温加剧第三夜，像将醒未醒。醒在缘，不在催。

试手日午后，执事记功，不记灵。韩泥不喜，只记手稳了。稳了，路又宽半分。宽，是七笔没断，不是灵根好。

编筐补单第二回，血珠饱满，甩向破席。席吸尽，血未至沿。未至，像血线将至，沿仍不接。

夜试手后，他对坛低声：「手过了，沿还净。」

邻床问沿，他答：「沿是坛边。沿净，才接得住缘。」答短，像心口七笔压着。

符怀暖，药涂裂口，血止沿前。止，像道线。沿在眠，等韩氏守瓮血。

窄路也是路。路通向瓮醒，瓮醒通向还。还，先还烫。试手坡下，丹形丑，丑而稳。稳，执事只记手，不记脸。午后编筐，藤刺深扎，血珠饱满，甩向破席，席吸尽，沿仍净。

试手日清晨，坡下霜白。霜贴石阶，站要稳。稳，才验得过手。

午后第三单，血线将至。将至，沿不接。

夜试手后第二遍，坛温加剧。加剧，像将醒未醒。

他摸记功帖，帖薄，真。真在「手稳」，不在「灵色」。

丹炉前第二遍，他手仍不抖。丑而稳，稳而真，真在末排站住了。

试手过了，他仍末排。末排不要紧，要紧是沿净，手稳。

血线将至那日，他独眼平，手不抖。抖，缘便偏。偏，席挡不住。

窄路也是路。

沿在眠，血不主动碰。

还，先还烫。

沿不接，缘在等。

他低声：「等。」

血线将至，手仍稳。

血线将至，沿仍不接。
