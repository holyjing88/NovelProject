# 多绶道侣与家族线（后宫收集 · 主次分明）

> **定位**：道侣系统 v4 · 取代旧「仅一主绶」设定。  
> **原则**：**有情有义才纳绶**；倒贴→报恩→纳绶→家族并入主线；**正不妒，侧不越**。  
> **最后修订**：2026-07-11（AUDIT v3.9）

---

## 一、总纲

| 项 | 内容 |
|----|------|
| 道侣绶上限 | **4 位**（正绶 1 + 侧/缘绶 3） |
| 绑缘簿总名额 | **7/7**（4 道侣 + 3 伴飞：萧/铁/柯） |
| 收集节奏 | 赠礼→倒贴→报恩→**纳绶礼**→家族线→鸡犬升天 |
| 主次 | **沈清弦正绶**永占主位；侧绶依录入顺位；720 后正位魂占位，侧绶同修 |
| 禁止 | 无恩强收、无家族戏、同章纳两人、侧绶夺正礼 |

## 八、写作硬规

1. **每位道侣纳绶前** ≥2 章家族戏 + ≥1 章倒贴/报恩  
2. 纳绶礼 **沈正绶必须在场**（720 前本人，720 后灵位/名分）  
3. 侧绶/缘绶 **不抢 420 契缘礼** 戏量  
4. 男欢女爱：正绶可写帐暖(719)；侧缘写吻/抱/共战，**272 前仅沈意缘**  
5. 家族灭门/辱女戏 **禁自刎**；仇用 510/655/1440 报  
6. 收集满 4/4 道侣绶 ≈ **655 章**；之后写后院日常与灵界续缘  
7. **丹道/阵法**：破境章写控火成丹+护关阵；合阵章沈魂占主位；见 `02` §十六/§十七
