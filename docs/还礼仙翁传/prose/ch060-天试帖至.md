# 第六十章 天试帖至

战后第五日，五宗联名帖至宗主殿。

帖金边，字肃，像把天试从风声变成铁律：**三月后，五宗天试启；青岚须选派护法长老与参赛弟子。**

霍镇山抱拳，声洪：「镇妖司请战！」

柳青鸢升座，目光落莫长春：「师兄，护法长老位，宗门给你。」

莫长春接帖，袖中簿一烫，烫得像等这一刻。帖与簿意相合，淡金字浮，成句：

**缘箓三转·结缘——羁绊四十以上，两人已成。**

他辨「两人」，不点破，只笑，像笑给簿听：柳在，苏在，顾在，秦在——结缘二字，宽，宽得像路，路还长。

寿元行跳：三月。

因果沉，约七十五，像东脉一夜的雨，终于落进账里。

殿外，弟子议论未歇。

「夜战三夜，莫长老杖剑齐出……」

「九月？谁还信九月！」

「我信。」顾小满在廊角，声轻，「长老给符那夜，我就信。」

苏念慈听见「天试」二字，握拳，又松，松得像丹师终于敢信：青岚门还有三月，还有师兄，还有炉。

她未入门，只立廊下，像陪炉，也像陪战后的静。

赵玄机立执法堂侧，铁牌在腰，目光扫过莫长春背影——妒还在，却薄了，薄得像私丹还库后空出的那一格。

周德海抚须不语，像一盆浇了一半的油，等下一阵风。

莫长春望天，云薄，像帖上的金边。

他低声，仅柳青鸢听见：「护法长老，老夫接。打，仍是你打。」

柳青鸢点头：「你赠，我打。」

他扬声，像给全殿听：「礼还没送完呢。天试门前，够再送几程。」

殿内一静，静后是一层低低的应——不是欢呼，是信，像东脉灵田雨后，根还在土里。

夜，小院无灯。

莫长春独坐廊柱旁，不坐主位，杖横膝，竹纹游动，气音清亮：

够。

他点头：「够。」

簿上浮半句，像章末钩：

**结缘有时，天试在前。**

风大，正好送礼。

东脉方向云低，赤焰旗远，像厉无殇未完的恨；天试帖金边在案，像三月后的雷劫——大因果还在后头，部二却在此刻落了账。

秦商言在山下驿站收讯，讯短，却重：「青岚夜战三夜，莫长春杖剑齐出。」

她对下属道：「茶饼那笔恩，记厚了。天试三月，百宝阁跟青岚走一路——路在礼上，不在价上。」

苏念慈回丹堂，封最后一炉培元，封得稳，像把战后余悸也封进瓷瓶。她望峰顶方向，低声：「师兄，三月……够炼几炉了。」

霍镇山在营帐外磨刀，刀声霍霍，像给天试磨刀，也像给东脉磨刀。

莫长春独坐，杖横膝，笑淡：「三月，够送，够还，够活到你后悔——厉无殇那句，天试门前还能再说一回。」

杖答：够。

大因果还在天试门前，像帖，像雷劫——部二落了账，部三才刚开场。

殿外，弟子议论未歇：「夜战三夜，莫长老杖剑齐出……」「九月？谁还信九月！」顾小满在廊角，声轻：「长老给符那夜，我就信。」苏念慈握拳，又松，松得像丹师终于敢信：青岚门还有三月，还有师兄，还有炉。

赵玄机立执法堂侧，铁牌在腰，目光扫过莫长春背影——妒还在，却薄了。周德海抚须不语，像一盆浇了一半的油，等下一阵风。莫长春望天，云薄，像帖上的金边：「护法长老，老夫接。打，仍是你打。」

秦商言在山下驿站收讯，对下属道：「茶饼那笔恩，记厚了。天试三月，百宝阁跟青岚走一路。」苏念慈回丹堂，封最后一炉培元。霍镇山在营帐外磨刀，刀声霍霍。杖答：够。大因果还在天试门前。

帖金边，字肃，像把天试从风声变成铁律：**三月后，五宗天试启；青岚须选派护法长老与参赛弟子。**莫长春接帖，袖中簿一烫。帖与簿意相合，淡金字浮，成句：**缘箓三转·结缘——羁绊四十以上，两人已成。**寿元行跳：三月。因果沉，约七十五。

殿内一静，静后是一层低低的应——不是欢呼，是信，像东脉灵田雨后，根还在土里。夜，小院无灯。莫长春独坐廊柱旁，不坐主位，杖横膝，竹纹游动，气音清亮：够。簿上浮半句：**结缘有时，天试在前。**风大，正好送礼。

围观弟子在殿外踮脚，踮得像等缘箓三转落字——字没落进眼里，落进袖中簿，簿沉，像七十五斤人情。有人低声：「九月？谁还信九月！」另一人答：「我信。长老给符那夜，我就信。」

大因果还在天试门前，像帖，像雷劫——部二落了账，部三才刚开场。礼还没送完呢，天试门前，够再送几程。
