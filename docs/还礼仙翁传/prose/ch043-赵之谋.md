# 第四十三章 赵之谋

赵玄机妒火至峰，峰不在莫长春，在林挽月看他的眼神——那眼神里多了犹豫，少了慕，像一根线松了。

外门林苑，夜，竹影碎，月白得像一张纸，铺在赵玄机脚边，纸上该写局，不该写恩。

「挽月，」赵玄机声低，「莫长老浴房外徘徊，你瞧见了吧？」

林挽月一颤：「赵师兄……那是谣言……」

「谣言要有人传。」赵玄机截话，「长老辱你、辱宗门，你忍得，执法堂忍不得。」

「可……」

「可什么？」他目光冷，「你慕我，还是慕他装的神仙样？」

林挽月脸白，唇抖，终未答。竹影晃，像有人也在听，听在廊角，听在将来要翻的堂上。

赵玄机递她一枚传讯符：「明日戌时，浴房外径。你只需『偶然』路过，看见『偶然』徘徊的长老。余下，执法堂会办。」

林挽月捏符，像捏一块烫手铁：「若错了呢？」

「错了，」赵玄机淡道，「也是长老九月将死，急出来的丑。与你何干？」

他走了，袖风冷。林挽月立月下，立成一根将断的竹。

远处，莫长春院中，簿无字，却沉，像知奸谋将起，不急，只念：「年轻人，急什么。」

顾小满在杂役堂浆洗，抬头望林苑方向，无端心口一紧，像半块灵石在掌里跳，跳成警。她不知局将起，只知赵玄机白日路过执法堂时，眼神太稳，稳得像已写好明日嚼舌的稿。

孙福旧党在角隅嘀咕：「莫长老老不正经，终于有人敢办。」

顾小满冷眼：「办错了，堂上见。」

嘀咕者噤声，像谣言碰上半石，又磕碎牙。

夜更深，林挽月捏符欲碎，又止，止得像怕碎的不是符，是自己最后一点慕。她想起赵师兄还库，想起莫长老止步不入的传闻尚未起，忽然觉得两条路都在脚下，一条慕，一条证，证字更重，重得像恩。

赵玄机在执法堂偏厅，指节叩铁牌，一声比一声轻，轻得像妒火终于找到出口，出口却是脏的。他低声，像对自己说：「明日，全宗都要看。」

执法堂弟子问：「赵执事，明日若长老不入浴房……」

赵玄机淡道：「不入，也是徘徊。徘徊，够嚼。」

弟子笑，笑里脏，像已看见明日浪。

魏无涯在檐角，扇开一线，观气运，低笑：「局将脏，脏里也有人要证。」

林挽月欲碎符又止，止得像慕将断。风过竹，沙沙，像答，又像不答。她低声，说给月：「若真是局……明日戌时，该有人止步。」

苏念慈在丹堂值夜，炉火旺，旺得像替林苑也担。她不知局将起，只知浴房二字若成浪，丹堂也要被溅脏。

柳青鸢在宗主殿批文书，笔尖一顿，像觉出竹影里的局，却不点破，只落笔：「戌时，执法堂巡。」——巡是真的，局也是真的，真假要等莫长春止步才算。

月白，符烫，像两条路都在脚下——一条脏，一条证。证字还无人说，脏字已在赵玄机袖里写好。

翌日未至，林苑已静得反常。林挽月未眠，符贴枕下，烫得像一块未落的铁牌。她反复想赵师兄那句「与你何干」——若错了，脏的是长老；若对了，脏的是她。

赵玄机回偏厅，又召两名心腹，低声交代引路灵息该怎么放：「莫长老若往丹堂，必过岔口。灵息只引，不逼——逼急了，像陷害；引到了，像巧合。」心腹问：「若他真进浴房呢？」赵玄机冷笑：「进了，更好。不进，也够。」

顾小满夜里未睡，把赵玄机白日去过的地方一条条记进册子：执法堂、林苑外径、孙福旧党聚处。她不懂大谋，只懂记——记比猜有用，猜比传有用。

莫长春院中，茶仍煎，煎得淡。他不点破，不预辩，只把明日戌时当成一笔将落的恩：「谋在明，才防得。藏在暗，才毒。」袖空，杖在塔中，像等误闯自己响。

霍镇山在营里磨刀，听外门风言，骂一句：「浴房？放屁！」又骂：「赵小子若敢借体统咬人，老子先剁他铁牌。」骂完仍磨刀，刀声霍霍，像给夜袭磨刀，也像给明日嚼舌磨刀。

孙福旧党聚在角隅，低声：「赵执事这回要成。」孙福本人未至，余党却像闻腥的猫。顾小满自后经过，册子一翻，只记时辰与地点，不记脏字——脏字日后堂上见，见真了才罚。

秦商言飞符至杂役堂，符上八字：「浴房局小，夜袭局大。」像商言也懂：针若来，先记针；针未到，先记路。

戌时之前，执法堂偏厅又亮了一盏灯。赵玄机把明日巡者名单再过一遍：谁「恰好」路过，谁「恰好」记录，谁「恰好」把止步三息写成稿。心腹问：「若林师妹临阵反水……」赵玄机目光冷：「反水，也是戏。戏要真，才像谣言；谣言要真，才像门风。」

顾小满天将亮时把册子抄完，抄得像抄自己的胆。册上无脏字，只有时辰、地点、谁去过林苑外径——胆还压着，时未到；时到了，堂上才肯翻。

林挽月林苑独坐，传讯符贴枕下，烫得像罪。她反复咀嚼「偶然路过」四字——偶然若成，她成刀；偶然若败，她成证。慕字还挂在颈上，证字还压在喉里，两条路都在脚下，她选哪条，堂上才肯翻。

柳青鸢声低：「师兄，赵氏谋起，门要稳。」莫长春道：「谋在明，才防得。藏在暗，才毒。」

赵玄机铁牌在腰，笑浅，笑里像藏下一招。顾小满声平：「谋记下了。针若来，先记针。」

苏念慈不进门，送汤至廊外，汤温，温得像怕谋进丹堂。她低声：「谋谋别传成盟。」

周德海在长老院听闻林苑夜会风声，抚须不语，像一盆浇了一半的油——油未沸，风先到。他不对赵玄机表态，只令心腹：「明日戌时，后阵巡一遍，别让人说长老院见死不救。」话是公，意是私。

魏无涯夜来传讯，只六字：「局将脏，证在后。」莫长春阅讯，点头，像谢提醒。他自语：「证字不在老夫嘴里，在肯翻的人喉里。」

翌日卯时，外门尚未起浪，林苑已静得反常。赵玄机铁牌在腰，笑浅，笑里像藏下一招——一招不在刀，在「恰好」；恰好若成，全宗都要看；看够了，才好还。

赵之谋起那夜，比百宝阁价还阴，阴里像针。

风过林苑，竹影又碎一层。林挽月终未碎符，符仍在枕下，烫得像将证的胆——胆还压着，时未到；时到了，该有人止步。谋起那夜，阴里像针；针若来，先记针；针未到，先记路。

苏念慈夜回丹堂，炉仍开，像把妒与担都炼进火里。她低声：「谋谋别传成盟。盟若成，丹堂先脏。」风过，炉火旺。

霍镇山在营帐外磨刀，刀声霍霍，像给明日嚼舌也磨一刀——刀不斩人，先斩谣。

戌时将至，林苑竹影愈静。赵玄机心腹在浴房外径布巡，布得像布一张网——网眼是「恰好」，网绳是谣言。莫长春不进门，只袖空立远，立得像告诉全宗：戏在门外，不在门内。

顾小满册上又记一行：谋起，戌时，浴房外径，止步候。行细，细得像半块石硌深。

戌时将至，林苑竹静，静里像针——针未落，先疼在证字上。

弟子渐散，散里仍回头瞧一眼灰袍袖空——瞧得像瞧一眼九月还能走多久。

赵玄机指节叩铁牌，一声，像把谋与妒又绞在一起——绞成更硬的墨，墨里仍留一寸空。
