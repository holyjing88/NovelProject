# 第四十七章 小满觉醒

夜雨，杂役堂后山，泥滑，路窄。

顾小满抄近路报讯，怀里揣着孙福私账抄本，像揣半块灵石，揣一路，揣到簿意一跳——她无簿，却觉天地静了一瞬。

金光冲霄，一瞬即敛，像有人在她头顶记了一笔，又迅速藏起。

她跌坐泥中，喘，却看得更清：左径三人，右径火把，赤焰探子换岗的间隙，全进眼里，像地图展开。雨线变成线，线变成路，路变成田埂，田埂变成丹堂侧径——她以前走夜路只懂怕，如今懂「看」。

莫长春立在雨里，不知何时至，杖点地，声轻：「小满，看见了？」

「看见了……」她颤，「小满……以前瞎，现在不瞎了。」

「不瞎就好。」他道，「恩不用嘴还。你记路，就是还。」

顾小满跪又想立，终是立，雨打脸，像洗一双新开的眼睛：「长老，赤焰探子在田埂三处，左径三人，右径火把，换岗间隙半炷香……」

「慢慢说。」莫长春道，「说清了，比跪清了有用。」

她一条条说，莫长春听，听进袖中，像听进一张将成的网。雨里远处，赵玄机思过崖方向，风硬，像另一根妒火还在冷。

顾小满说完，忽然问：「长老，小满这眼睛……能走暗线吗？」

「能。」莫长春递她一枚薄护身符，非宝品，贴肉，「别跟丢命。暗线走侧面，别走正面。正面是霍老头和宗主的戏，侧面才是杂役的戏。」

她接符，泪落，笑也落：「小满记恩。」

雨歇，云破，一星露，像气运刚抬头，抬头便藏，藏得像丁级，又像将成网的眼。

杂役堂弟子日后窃语：「顾小满雨夜见金光？」「胡说。」「可她说探子换岗，准得像亲眼见。」

霍镇山闻讯，骂声洪：「好！暗线要活，丹堂才稳！」

周德海在长老院听闻，面色平，平里冷，像早就备好的稿。

魏无涯观金光一瞬，扇合，低笑，像看网将成，成在子时。

他低声，像钩：

「眼开了。下一程，看这张网能不能在子时送命回来。」

风停，东脉云低，腥一线，像夜还未来，牙已露。

杂役堂弟子日后窃语：「顾小满雨夜见金光？」「胡说。」「可她说探子换岗，准得像亲眼见。」「丁级杂役，哪来的眼？」「半块灵石给的。」窃语轻，轻不过雨夜那一瞬金光。

孙福私账抄本仍在她怀，像第二张半石。她忽然懂：金光不是显圣，是开眼——开眼的人，走暗线，不走正面。

秦商言飞符至杂役堂，符上六字：「暗线需耳，耳在丁级。」像商言也懂：网眼小，容不下英雄，容得下报信。

顾小满摸护身符，符温，像长老的手还在。眼开了，路记了，网将成——成在子时，成在侧径，成在杂役堂敢不敢走夜路的那一步。

魏无涯观金光一瞬，扇合，低笑，像看网将成，成在子时。风停，东脉云低，腥一线，像夜还未来，牙已露——露在丁级杂役眼里，露在暗线将走的那一步。

她一条条说，莫长春听，听进袖中，像听进一张将成的网。雨里远处，赵玄机思过崖方向，风硬，像另一根妒火还在冷。雨歇，云破，一星露，像气运刚抬头，抬头便藏，藏得像丁级，又像将成网的眼。

莫长春立在雨里，不知何时至，杖点地，声轻：「小满，看见了？」「看见了……」她颤，「小满……以前瞎，现在不瞎了。」「不瞎就好。」他道，「恩不用嘴还。你记路，就是还。」

围观弟子日后在杂役堂外踮脚，踮得像等金光再亮——再亮没来，来的是探子换岗准得像亲眼见。有人低声：「丁级杂役，哪来的眼？」另一人答：「半块灵石给的，记恩记的。」

杂役堂檐下，有人嗤：「金光？丁级也配？」另一人悄道：「配不配，探子换岗准不准说了算。」话出，众人噤声半息，半息里像听见田埂上的火把噼啪——噼啪先远，后近，像夜还未来，眼已至。

周德海在长老院听闻，面色平，平里冷，像早就备好的稿。他低声对心腹：「杂役开眼？眼开了，也得看谁肯闭闸。」

雨歇后，顾小满把路线一条条画进册子抄本，画得像画自己的胆——胆画了，路才走得稳。莫长春收讯，不夸，只杖点地，声轻：「路记清了，比金光更显圣。」

风停，东脉云低，腥一线，像夜还未来，牙已露。眼开了，下一程，看这张网能不能在子时送命回来——送回来了，半块灵石的故事，才算又长了一块。
