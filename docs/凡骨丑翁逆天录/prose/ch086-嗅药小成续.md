# 第八十六章 嗅药小成续

嗅药小成后第七日，药渣场又贴了条。

管事墨未干，字却硬：「丙九韩泥，辨性加验。十堆混灰，错两堆，记名。」

韩泥蹲下把条读完，独眼平平的。错两堆就要记名，记名就靠边站——可他不怕靠边，只怕鼻子糊。鼻子不清，往后指头再稳也没用。他把条角折齐，塞进袖口，像把规矩先收进口袋。

十堆灰横排开，灰里混着沉香、腐底、浮腥，像故意把人鼻子搅晕。旁有人吭嗤：「晦气翁也加验？验出灰饭来也不稀奇。」

韩泥不抬头，只道：「验灰，不验嘴。」

那人还想刻薄，铁无言抱臂往栏杆一靠，声不高，够听见：「嘴闲着，滚去铲栏。他验，你们别搅。」

场子静了半息。韩泥伸手入堆，指尖不先碰，鼻子先落——沉的往左，腐的往中，浮的拨开。第七堆腐底里藏一点浅辛，浅也剔；第十堆假香浮得厚，厚更要剔干净。剔完举手，管事对照样册，笔一顿：

「丙九韩泥，加验准。十堆全对。」

全对落进簿里，像钉子敲进木板。同场杂役有人倒抽口气：「真全对？」

「真。」管事没抬头，「真就去干活。杵着，灰也臭。」

有人凑近看簿，被管事一瞪缩回。韩泥袖里指节松开——紧过了，也不敢松成得意。得意像香，香一飘，赖福那鼻子比狗还灵。

温听雨站在风口远处，裙素，袖香温。她没走进，只远远看一眼指头落得稳不稳——看完，袖风一转，走了。走了也留下半句在风里，韩泥不追。追像求，求就脏。他只把那一眼记在心口，记浅一点：浅了，才不像攀。

铁无言走过场边，另取一撮未知灰：「这撮，你闻。」

韩泥鼻下一息，指落：「沉七，腐二，浮一。」

铁无言分秤验完，点头：「对。小成续上了。别飘。飘了，加验白做。」

「不飘。」韩泥说，「沉腐浮，各归位。」

铁无言又低半声：「续成了，夜里才接得住破境。丑时别叫。叫了，全棚听。」

韩泥「嗯」一声：「不叫。」

「不叫比会叫值钱。」铁无言道，「末席能活，靠的是闷。」

午后外堂洒扫，阶尘少，扫帚仍一下下稳。温听雨擦栏擦到半道，忽然停，声清：「加验十堆，真全对？」

「全对。」韩泥应，不多字。多了像攀。

她嘴唇动了动：「全对不容易。不容易，像手稳的人。」

「手稳」第三回落耳，韩泥仍不谢。谢轻了像讨巧，他只记——记她远验，记她傲里少了一分冷，记各做各的界还在。

她把巾一收：「别攀温氏门。门在丹，不在缘。」

「不攀。」韩泥说，「各做各的。」

「各做各的好。」她补半句，像怕自己话说软，「你手稳，我也只当看见扫帚。」

韩泥应一声，扫帚继续走。走完外堂，他心里那条缘苗温一线，温得克制——克制住了，才不邪。邪了，秦霜木鱼会敲第二声。

刘婆端粥过来，勺口又多半勺，热气扑脸：「加验过了？」

「过了。」韩泥接碗，「十堆全对。」

「全对怕什么。」刘婆哼一声，「怕你空肚逞能。喝。饭稳了，才挨得住下一茬。」

「挨得住。」他喝尽，稀粥也能暖腹。暖腹比暖脸重要——脸暖了容易飘，腹暖了才扛得住丑时。

邻床少年凑过来：「十堆全对，你成丹师了？」

「不成。」韩泥道，「成鼻子。鼻子成了，手才稳；手稳了，丹才轮得到。」

少年愣住：「鼻子还能成？」

「能。」韩泥说，「能闻对的，比嘴夸强。」

「那你夸一句自己？」

「不夸。」韩泥摇头，「夸了，鼻子就糊。」

少年闭嘴睡了。韩泥却把角柜里样包数到十二——包实，锁实。锁住了才不示人；示人，赖福来，管事来，旧债也会来。他把钥匙绳缠腕上，缠得像把命拴紧。

赖福傍晚路过棚外，脚停半息，鼻尖抽了抽：「沿这么净？净出邪味？给我掀！」

韩泥坐在席角，声平：「掀得着，你就掀。掀不着，别赖全棚。」

赖福伸手摸席缝，缝严，坛不露，像腌菜坛盖死了。他哼一声：「暂记你。丑时再静，我听见。」

「听见你也听见瞌睡。」韩泥道，「困了就睡。」

「嘴硬！」赖福脸色铁青，「等你破境破到告示上，我再看你硬！」

韩泥不接。接了要辨，辨了要露。秦霜木鱼远远敲一声，像隔墙递话：「续成别骄。骄了，香就飘。」

韩泥心里应：不骄，只记。记十堆全对，记十二包锁柜，记丑时破境——境要藏在席下，不贴告示。

丑时前他廊下练半刻，十二包样息并一线清。清里掌纹影颤了颤，像更远的阶在催。他掌贴坛沿，低声道：「小成续了。三层在丑时。真名不叫。」

坛不应字，只温。温像把十堆全对摁进掌纹那路里。臾墟子当年嘱过：不叫真名，煞骨门才嗅不准；嗅不准，晦气才遮住瓮。他把玉牌贴胸，牌凉，凉提醒三年约还摁着命。

同批少年从廊下一过，悄问：「丑时你又静？」

「静。」韩泥说，「静着破，不叫。叫了，恩也断边。」

「啥叫断边？」

「像碗摔了边。」韩泥道，「还能盛，盛不稳。」

少年似懂非懂，走开了。白织月远廊经过，没嘲，也没停——不停就是放过这一回。韩泥不稀罕她放过，稀罕自己手还不抖。

更鼓沉下去。坡下春风硬，药香回了一线。他合眼，鼻下仍清一息：沉、腐、浮各归位。位稳了，才接得住那一夜破关。

铁无言夜巡到渣场，瞥一眼角柜：「样别示人。沿净。」

「净着。」韩泥应。

「明日管事还要抽验两堆。」铁无言说，「验得过，日子才好过。验不过——」

「验得过。」韩泥打断他，不傲，只实，「鼻子在，就不会糊。」

铁无言盯他一息，终是点头走了。韩泥躺回席上，手按木牌，牌凉，掌却暖。暖在诀里，也在十二包样里。他低声道：「续成了。坛应了。丑时破三层，仍守口。」

守口，不是怂，是活命。活命，才等得及破境，才等得及丹路往后一点点开。他把「全对」那一笔按实——按实了，明日抽验才不怕手抖。

天边白一线时，他仍没睡实。鼻尖那口清气像钉，把他钉在「别飘」两个字上。飘了，加验白来；不飘，丑时那一破，才可能压得住。

兽栏那边隐约有铲声，有人骂牲口。骂声远，规矩近。韩泥把席角再压紧一分，对坛道：「续成了。三层在后。后头再开，也不叫。」

坛腹温一线，像听懂了「后」字。他还不喊醒更大的——明天村里不是他的村，药山才是。药山上，鼻子要准，嘴要闷，手要稳。

管事弟子夜里提灯路过棚外，灯火一晃，照见席角净。净得他愣半息，低声对同伴：「晦气翁这铺，比有些外门还齐。」

同伴嗤：「齐能当灵根？仍末席。」

韩泥听见了，不气。末席也要齐——齐了，抽验才站得住。他摸角柜锁，锁凉，包实。包实了，才不像炫耀；不像炫耀，才活。

丑时将尽，他把十二包样息又并一次，并出那一线清。清着清着，掌纹影又颤。颤不是吓，是催——催他别把鼻子用飘了。他低声道：「催可以。飘不行。」

廊外风硬，风里有人问：「明日抽验，你慌不慌？」

「不慌。」韩泥答，「慌的是鼻子糊。鼻子在，验什么都一样。」

那人愣住走了。韩泥合上独眼，心口按着「全对」与「不叫」。按牢了，明日抽验才不怕手抖，丑时破三层才不至于开成热闹一场。
