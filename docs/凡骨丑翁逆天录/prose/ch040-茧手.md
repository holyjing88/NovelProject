# 第四十章 茧手

入门两月，须发更白，背更驼，独眼仍利。利，是渣里香、编筐、克扣里熬出来的——熬成茧，茧厚，将来握丹才握得住。同批杂役走了两人。韩泥不笑，只记——末席也要站直，站直了，测手才过。管事唤丙九集合：「凡骨杂役，冬测备检。冬测备检不是测灵，是测手——手稳的，留；手抖的，滚。」

韩泥站末排。末席，他熟，熟所以稳。测前，管事喝：「凡骨别妄想测灵。手抖的，明日滚。」

众人噤声。韩泥呼出一口长气，气稳。赖福阴笑：「丑杂役，别抖。抖了，丙九丢脸。」

韩泥不答。手稳，才是回辱。辱在记，不在顶嘴。同批少年抖着手搓衣角：「我搬铁定抖。」

「抖，先吐气。」韩泥低声道，「气稳，肩才吃重。」

少年照做，肩松一点。松一点，也是小胜。管事扫队：「冬测备检只测手，不测嘴。嘴稳的，去搬第三块石。」

众人噤。韩泥抬眼，独眼平——平，是两个月粪渣筐熬出来的。管事扫队时，目光如秤——秤在手，不在脸。脸丑不要紧，手抖才滚。韩泥站末排，肩平，指茧贴裤缝，缝着，手才不飘。赖福低声：「过了冬测备检，测灵仍是末。末，滚得快。」

末席他熟，熟所以不争前，争前手易抖。测手场，青石地，线白。搬石过线，石沉，沉得掌茧疼。疼，是编筐、清粪、嗅渣攒下的底。第一石，抬得稳；第二石，指茧裂处迸血，血不疼窗，疼不露；第三石，换肩，肩驼，驼得稳。管事又掷一块小石，石轻，轻测细稳：「再一块。这块抖了，仍滚。」

韩泥吸气，气沉肩，石过线，线白，白得干净。管事「嗯」一声：「韩泥，过。」

旁侧一少年手抖，石落半寸。管事喝：「滚！」

少年哭：「我家里还有人……」

管事冷声：「家里有人，手就不能抖。抖了，宗门不要。」韩泥不笑，只记——记手稳的价值，记滚字离自己多远。同批未过的少年被架走，哭腔远，远得像泥岗公审前夜的风——风硬，他仍站末排，不回头。回头手易软，软了，冬测白过。刘婆在灶边，瞧他过，舀粥多半勺。半勺，恩轻，轻也真。韩泥不道谢，只点头。

谢轻，恩重，重就记着。管事掷木牌，牌上刻「冬测备检过」三字。三字薄，薄也真。赖福阴笑：「过了又如何？凡骨仍是凡骨。」

韩泥举掌，掌茧裂，裂里新皮，皮嫩，嫩所以还疼。「骨凡，手不凡。」他说，声哑，不高，「手不抖，粪不洒，渣不漏，筐不塌。」

赖福唇动，没尖。尖了，明日加倍，加倍也挡不住记名。管事补一句：「丙九韩泥，手稳，冬测备检记名。」

同批有人嘀咕：「凡骨也能过？」

管事喝：「手稳就过，嘴稳不算。嘴稳的，去丹房门口试试，看渣漏不漏。」

少年凑近：「你怎么做到的？」

「做。」韩泥说，「粪做，渣做，筐做。做多了，手就不抖。」

「做两年？」

「做一天，也要做稳。」

少年点头，眼里还是怕。同批又一人未过，跪求：「再试一次！」

管事冷声：「冬测备检不是人情。手抖，明日滚。」

韩泥不劝，不笑，只记——记滚字离自己三尺远，远一寸，多活一日。赖福啐：「滚一个少一个晦气。」

管事喝：「你也去搬石。抖了，你滚。」

赖福脸白，闭嘴。白，也是小胜——胜在测手场，胜在韩泥仍站着。夜，编筐末单，补冬测空隙。藤条脆，炸刺，刺进指肚，血珠饱满。他甩手，血甩到破席，渗向坛沿方向——渗到沿前，收手，席盖严。邻床少年醒了一瞬：「你又流血了？」

「编筐末单。」韩泥说，「单完，手才稳。手稳，明日测灵备检才不抖。」

「测灵备检，凡骨也测？」

「测。测不过，滚。滚了，恩断。」

少年又睡了。睡了，韩泥才低声补：「别学我忍。学我稳。稳了，才少挨滚。」

血不该故意醒瓮。醒要活计扎，扎要时机到，到瓮醒。坛沿仍温，温不应血，应手。他低声道：「两月了。泥岗恩，未竟。宗门债，新起。再熬，熬到瓮醒，熬到还。」

外头雪意，风硬，雪未落，意先冷。冷是钩，钩着冬测名单，钩着坛温，钩着掌血停在沿前那一寸。他低声自问：「手稳了，灵仍末。末，怕不怕？」

不怕。怕手抖，不怕末席。末席站直，站直了，测灵备检才有名；有名，才不被滚出丙九。刘婆收碗，低声：「手裂了，明儿我给你缝块布。」

韩泥点头：「记着。」

他念泥岗六笔恩，念宗门刘婆粥另记第七笔，分列，不混。赖福门外啐：「过了也是凡骨！」

韩泥不答——舌要留来辨香，留来还，不浪费在赖福嗓上。管事远处喊：「丙九韩泥，明日测灵备检名单，你在末。别抖。」

韩泥应声：「不抖。」

邻床少年问：「测灵备检，凡骨有希望？」

「希望不在灵，在手。」韩泥说，「手稳，名单就有名。有名，不被滚。」

韩泥覆席，席盖坛，盖得严。严，血不渗，瓮不醒；不醒，时机在瓮醒——瓮醒那日，掌茧再裂，裂得准，才推得动恩债。冬测备检过了，手记下了。记下，冬测不滚。不滚，才熬得到瓮醒。瓮醒前，血不渗沿。掌按石，灰线短，短不过指节。指节短，手不短——手稳，备检才过。过了，木牌在怀，怀暖，牌更真。

坛沿温一线，应手，不应血。血停在沿前，停在瓮醒之前那一寸。一寸之间，全是活计。邻床少年问：「你掌流血，还盖坛席？」

「盖，是不主动醒。」韩泥说，「醒要等活计扎，扎要等瓮醒那日。」

少年不懂瓮，懂「盖」，翻身睡了。外头雪意又近一分，雪未落，意先冷——冷是钩，钩着冬测名单，钩着坛温，钩着掌血停在沿前那一寸。管事巡夜，敲柱：「丙九韩泥，冬测备检过了。测灵备检名单在末，别抖。」

刘婆夜里端姜水，姜辣，热：「茧手别空攥。攥出血，备检白过。」

韩泥喝一口，姜辣醒指，指醒，茧才不白裂。管事门外贴条：「冬后测灵备检，十日后。丙九记名者到。误者，滚。」

十日备检——冬测过了，测灵在远。远不要紧，手记下了，手就不抖。备检前夜，灰线再过石，石过，手稳。木牌在怀，怀暖，牌更真。雪意又近一分，钩在雪，钩在坛，钩在瓮醒前那一寸。血停在沿前，停在活计里——活计在，冬测这关才开瓮门缝；缝开，路才向正测。

掌按石，灰线短，短不过指节；指节短，手不短——手稳，备检才过，瓮眠着等那一刺扎对。他低声对坛：「备检记名在怀，测灵在远。远不要紧，手记下了，手就不抖；抖了，恩断，汤就凉。」

天将亮，掌茧裂处结痂，痂硬，按石不疼。不疼，才敢想十日备检——备检在远，今宵只睡半息。半息里，他仍覆席盖坛，盖严，像告诉邻床：血不渗沿，瓮不醒；不醒，对。刘婆护粥，粥稀，热在；热在，第七笔又温半分。温半分，够熬到备检帖、清粪栏、兽栏冬角。

外头雪意又近一分，雪未落，意先冷——冷是钩，钩着十日备检，钩着坛温，钩着掌血停在沿前那一寸。血停在沿前，停在活计里——活计在，冬测这关才开瓮门缝；缝开，路才向正测。掌按石，灰线短，短不过指节；指节短，手不短——手稳，备检才过。木牌在怀，怀暖，牌更真。稳。等测灵备检。十日帖在柱，手在，路不断。

丑时末，他把破席角又掖实一遍，掖得像把「不抖」钉进掌纹。
