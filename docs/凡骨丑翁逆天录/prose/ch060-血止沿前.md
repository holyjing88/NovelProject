# 第六十章 血止沿前

第六十日，雪细。雪细像粉，粉在肩，肩沉，沉得住。韩泥清西栏，粪锹沉，掌茧裂口渗一丝潮。潮不是血，血不碰沿。卯时前，他又去搬柴半刻。霜厚，柴湿，斧声闷，闷像更鼓。管事验垛：「沉得住，记功在。」记功在，麸皮不克。克粮，克不走手稳。---

午后编筐，末藤刺深，血珠再饱满。他甩手，血甩破席，席吸尽，沿仍净。净，像瓮守的最后一夜前奏。坛腹温加剧，比廊下讲规矩那日又多一线。又多，像将醒未醒，醒在缘，不在人催。秦霜远廊经过，一句：「手稳，心别邪。邪，符灭。」韩泥怀符暖一丝，心记着七笔，不邪。铁无言立柱边，声低：「沿净，好。净，像等缘的人。」韩泥只答：「等。」等，不是拜血祠，不是主动滴血。是活计扎，血找席，席挡沿，沿等韩氏守瓮血那一误。同批少年碰他袖：「坛温起伏，像醒吗？」「像眠。」韩泥说，「眠着，不急醒。」「急醒咋办？」「急，像拜邪。」韩泥说，「拜邪，恩断。」少年懂「断」，闭嘴。老耿挑水路过，水洒稳：「六十日了，还守沿？」韩泥「嗯」一声：「守。守席，守缘。」老耿不再言。言少，水稳，像褐袍人当年嘱托的余音。管事查棚，灯晃坛角：「杂物收起。」韩泥「嗯」一声，席盖不动。动像心虚。不虚，是守。管事盯半息，走了。只见破席，不见仙缘。邻床笑：「坛角盖那么严，藏宝？」韩泥不辩，只把席盖又按实。按实，像编筐扎掌那日。掌茧再裂，裂在冬，不在沿。刘婆端姜汤，姜薄，热：「六十日了，沿还净？」「净。」韩泥说，「净，才接得住后头缘。」他喝尽，「嗯」一声。姜热，像叶汤远影。赖福讽：「守席守出仙缘？」管事路过，声冷：「再嗓，克半月麸皮。」赖福哑了。哑，韩泥按席的手却更稳。夜，雪停，棚静。他揭席一角，看沿。沿尘，沿滑，沿上无血。无血，盖严，盖得更严，像席盖更严预演。他低声道：「血止沿前，瓮温加剧。加剧，不急喊醒。」坛温一线，贴掌，贴到丑时前。掌稳，心稳，符暖，席严。外头更鼓二响，响沉。明日仍搬柴。搬柴，藤刺还会扎。扎了，血还会到席。席还会挡沿。挡到那一夜，缘才满。满到误触那一滴，瓮才醒。邻床鼾雷，雷里，坛角静。他低声对坛：「你再眠半夜。半夜后，缘满不满，看活计，不看心急。」坛不应字，只温一线，又凉。凉，像应「等」。他是记心翁，恩仇都记在心里。六十日擦沿，六十日不主动滴血。滴，要等误那一甩。同批少年又碰他袖：「你守坛六十日了？」「守。」韩泥说，「守席，守沿，守不主动滴血。」「滴血干啥？」「主动滴，像借腥。」韩泥说，「腥借来，恩易断。」清栏时，粪锹沉，肩勒旧印。印深，像五年杂役磨出来的。符怀暖一丝，护骨寒，手不僵。僵，藤刺偏。不偏，缘才正。他按心口，低声道：「叶丫头汤，热着还。还，要等瓮醒后，路才长。」

黄昏，雪停，他扫棚前雪半刻。雪薄，帚短，扫出一路净。净不要紧，沿要净。扫罢，又去搬柴一担。柴沉，肩勒旧印，印深，像丙九五年。铁无言路过，看一眼肩：「六十日了，肩还稳？」「稳。」韩泥说，「稳，才接得住沿前那一止。」铁无言「嗯」一声，不再言。言少，眼却验他鼻下静气。静气在，坛温起伏，就不像邪。他摸木牌，牌边磨毛，毛得像心口七笔。七笔分列，不混。叶汤热，振东药，陈姑袄，刘婆粥，符恩，耿恩，僧恩。笔笔在，手才不飘。飘了，血偏。偏，席挡不住。挡不住，沿就脏。沿脏，瓮不醒真。他是记心翁，恩不挂舌，仇也不挂舌。舌轻，心重。秦霜远廊木鱼不敲，只托着。托着，像规矩还在风里。她一句，声轻：「瓮温加剧，别拜。拜，符灭。」韩泥「嗯」一声：「不拜。守席，守沿。」

二更，他又揭席一角，看席边。席边湿一线，湿痕止在沿前三指。三指外，沿净。净，像血也懂规矩。席湿，他不换。换，沿要再验。验，手就忙。忙，心就飘。他盖席，盖严，破布又加一层。层厚，厚像后头那一夜预演。坛腹温加剧，贴掌，贴到腕骨也暖。暖，像将醒未醒。未醒不要紧，沿净要紧。要紧是血止沿前，不主动碰沿。邻床梦呓「红」。韩泥盖席，声低：「你看错了。睡。」他低声对坛：「六十日了。六十日沿净，才配说你暖加剧。加剧，我仍不喊醒。」坛不应字，只颤极轻，轻像「嗯」。外头更鼓三响，响沉，沉里坛温又一线。一线短，短像席边湿痕止于沿前。止得住，才配得上瓮温加剧那一夜。夜尽，席盖更严将至。将至，沿要更净，手要更稳，符要更暖。暖，不燃，燃了管事查。他按心口，低声道：「血止沿前，瓮温加剧。加剧是缘，不是命。」缘在，七笔就在。在，叶汤就还烫得动。坛腹温一线加剧，加剧，像明日就在沿边等那一滴守瓮血。窄路也是路。路通向瓮醒，瓮醒通向还。还叶汤，还振东药。还，先还烫。铁无言立柱边，声低：「六十日沿净，像守瓮人。守，不是拜，是等。」外头风息，柱上无新条。无条，有条在坛。坛温加剧，像将醒未醒。醒，不喊，不认，等血缘碰沿那一误。他独眼平，手不抖：「血止沿前，沿不接。不接，才配接瓮醒。」坛壁水汽爬半寸，敛。敛着，像替他守着。手别飘，恩不断。恩不断，叶汤就还在前头。血到了席，没到沿。止在沿前，止得像规矩。他盖席，盖严，严像后头那一夜预演。丑时前，沿仍净，手稳，符暖，席严。外头风硬，风里无栖香全句。全句散了，散成坛温加剧那一息。一息在，缘在，叶汤还烫在前头。雪细停，棚静，静里更鼓远。远，像瓮将醒未醒，醒在缘，不在人催。清栏时，粪锹沉，掌茧渗潮。潮不是血，血不碰沿，沿净像守瓮日常。午后编筐，末藤刺深，血珠再饱满。饱满，甩向破席，席吸尽，沿仍净。净，像最后一夜前奏。前奏不急，急像赖福，急像借腥。不邪，凡符才不辱主。」等，不是拜血祠，不是主动滴血，是活计扎，血找席，席挡沿。夜，他揭席一角，看沿——沿尘，沿滑，沿上无血。无血，盖严，盖得更严。坛温加剧，贴掌，贴到丑时前。前，掌稳，心稳，符暖，席严。」
