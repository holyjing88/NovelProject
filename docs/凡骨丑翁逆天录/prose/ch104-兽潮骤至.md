# 第一百零四章 兽潮骤至

至。

卯时前，界忽然裂开一线。线细，细却像刀口——毒先涌，涌得像潮；潮头是瘴，瘴后头是爪。爪刮障外，障内韩泥仍钉在东角。钉子歪了，角就散。

「至了！」铁无言吼。

韩泥灰香早空，空就炼。炼不回废窑，窑远，远了一息都像送命。袖底还剩废渣和青芯，芯少，少够一炉小丸。掌纹里火起半寸，投料顺序乱不得。乱了败，败过一次他记得——这一回只能成。

成在急里，急却不能飘。飘了栏倒，栏倒丙九连坐。丸成三粒，粒小，小也值。一粒塞进自己嘴里，含半息，半息把体内那股甜毒往外顶；另两粒掷给铁无言。

铁无言接住，吞下，嗓子哑：「正。」

正。一字够了。够当并肩的信物——信不在嘴甜，在药真。

兽影贴障，爪破障角。破口一开，风里全是腥。铁无言挡在前头，挡时臂上旧伤崩开。韩泥抄起铲粪叉改障，改得粗，粗能硬扛一爪也成。一爪砸下来，他臂上也开花，开花人还在角上。

「别散！」赖福躲在内墙尖声喊，「丙九别散！散了老子记你们全家！」

韩泥不应。应的是把障再堵实，堵完又撒最后一撮灰渣。渣尽，指间火又起——再起不是逞，是同袍倒了两只，倒了得清，不清活不成。

内墙援手终于到。兽影退一线。退不是终，终得看潮中那几更还咬不咬人。管事弟子手抖着记：

「丙九韩泥，角稳；铁无言，挡爪。」

记在簿上是小胜。胜不拿来喊，拿来换口气。白织月远远望着，声音意外地低：

「丑杂役……没逃。」

韩泥仍不答。答费舌，舌要留给第二更。第二更兽又近，近时丸已尽，尽了还有经页。诀出，灰雾再起，角里又清一线。清一线，像多活一寸。

夜将尽，他掌心裂口渗潮，潮不是血糊坛。坛沿仍净，怀中仍温。回棚半刻，他按住坛沿：

「至了。守过了。」

坛腹温着，像应「至」字。至不是清。清还在后头，后头还有更深一窝的瘴等着练人手。

邻床捂着脸哭：「我差点跑……跑出去就活了吧？」

「跑也活不长。」韩泥说，「怕也守。守了，才真活。」

「你臂流血了。」

「流着。」韩泥扯布勒紧，「流完还能铲。铲得动，就还没完。」

更鼓尽，界外仍吼，吼像没完。没完也好，至少说明今晚「至」字已经验过眼——验过了，人还在角上，角还在名下。

第二更一开始，又一头兽扑障。扑得重，韩泥侧身让半寸，矛再刺。刺退，肩背发木。木里他听见刘婆在远处，篮里饼硬得磕篮底，却没递上来——递不上火线，恩就藏在「手稳」两个字里。他懂。

「韩叔！」沈枯芽不知何时被撵到更后头，嗓子破，「叶姐叫……叫你别死！」

「死不了。」韩泥回半句，「死了还谁扫栏？」

铁无言哼一声，像笑又像喘：「臭嘴。再来一茬，你左我右。」

「左。」韩泥应。

左不是抢功，是角的方位他熟。熟了才不跟铁无言抢同一块空。

白织月又瞥一眼，像验货。验就验，角稳了，内墙才稳；稳不在她袖风香不香，在韩泥这根桩裂不裂。桩裂了一线，裂是活计，不是滚。

赖福躲着喊：「火再起！再起给同袍清！清不干净，拿你们填坑！」

韩泥火再起半寸，起给路，不起给嗓门。同袍接过他掷去的半截热灰，胡乱洒，洒完竟也能喘。喘就是链。链这一环先接上，后头才谈得上并肩更远的事。

第三更毒又涌一线。一线不长，清路才要紧。路清了，人能换位；换位了，伤的才能往后挪。韩泥看见铁无言臂布透红，仍不叫停：

「还能挡？」

「能。」铁无言咬牙，「你炉呢？」

「还在。」韩泥抬掌心，「不用尽。尽了潮中没后手。」

「留着。」铁无言点头，「留着比嘴硬强。」

管事弟子二次记名时，笔锋重了些：「角仍在。」三字落下，赖福嘴张了张，骂不出来。骂不出来也好，簿子比嘴硬。

韩泥抽空回棚喝了半口凉水。水入喉，喉里那股甜退一点。退一点，他才敢再出去。出去路过刘婆粥桶，刘婆盯他臂伤，喉咙滚动一下，终究只说：

「还活着就盛粥。死了粥冷。」

「嗯。」韩泥点头，「饭稳，恩不断。」

这句话说完，他自己也一愣——愣完便收下。收下来当钉子，钉在心里。

界外吼声忽然密成一片。密得像把「至」字咬碎了再吐回来。韩泥立回东角，独眼扫过铁无言、扫过障、扫过那道细得吓人的裂口。裂口还在，人就还得顶。他低声道：至了。下一程，潮中。潮中没有中间好看，只有更尽——更尽时，手还在不在，才算数。

风从裂口灌进来，灌得火把猛偏。偏影里他摸到怀中木牌，牌凉，条上「备战」的墨味似乎还沾在鼻尖。墨味提醒他：规矩写在柱上，命写在角上；角上站到更鼓再沉一截，才配把「还」字往下接。接不住潮中，备了压了至了，都算白。

至后第四更，兽影稀一阵，毒却更黏。黏得人指甲缝发麻。韩泥炉火再起半寸，不起丸，只烤干角背湿气——湿气重，下一更炼丸容易败。铁无言看见了，骂得短：

「还挺会省。」

「省给潮中。」韩泥道，「中段比至更磨人。磨人处，没炉等于没命。」

「你倒会算。」

「算活。」韩泥抬眼，「不算脸。」

同袍被人抬下去一个，抬时还抓着韩泥裤脚：「别……别让他们撤我名……」

「名在。」韩泥按他手，「人先保。名后写。」

管事弟子过来勾簿，笔尖顿：「丙九抬下一名，东角仍在。」仍在俩字落下，赖福在后头想找茬都难。难也好，难说明钉还钉着。

白织月忽然派人丢下一壶清水：「别跟我说谢。脏手别沾内墙石。」

韩泥接壶，先浇铁无言臂上止血布，再自己漱一口。漱完吐掉，甜味淡了些。淡了些，他才把壶还回去：「水到了。」

派人的外门弟子皱眉：「你还真会用。」

「会用才不浪费。」韩泥转回角，「浪费在这夜里，比偷药难看。」

卯时鼓将起未起时，裂口又鼓胀一线。一线里爪尖闪过。铁无言矛杆横拦，韩泥叉尖跟上，两人像把门闩死。闩死一息，援手再至一息。一息接一息，至字才算咬过而不把人吞尽。

「还能顶到潮中吗？」同袍喘着问。

「能。」韩泥说，「顶不住也顶。顶着，才有续。」

他这句话说完，自己掌心又裂开些。裂了用布勒，勒完继续铲障泥。泥腥、血腥、石灰腥搅在一块，腥得像把药山外的兽潮煮进鼻子。鼻子受苦，心更得醒——醒着才记得：下一程不是散场，是更长的「中」。

夜风把「全员备战」的条边掀起来一角。掀角像眨眼。韩泥看着那角，忽然觉得条也在催人：催人把命写整齐。写整齐不是毛笔字，是角还在、人还在、丸还能再炼一粒。

一粒够不够都得进潮中去验。验之前，他把坛沿再按一下，像跟「还」字约好——先过中，再谈还。还得有命；命在角上钉到更尽，潮中那几更才接得住。接不住，这条备战的字，便只剩纸上规矩，照不出一条活路。活路还在角上——角在，才能进潮中去挨更。
