# 第一章 村嘲晦气

天阴着，泥岗村闷得像扣了口大锅。

五十八岁的韩泥蹲在村西最漏的泥屋门口编筐。藤条绕着膝盖，绕一圈勒紧，再绕一圈——这手艺爹教过，他编了半辈子。村里叫他晦气翁：麻脸一块块，独眼那只好的常年眯着，另一只蒙着白翳，腿跛，背也驼。模样摆在那儿，骂声就自动跟上。

村童隔着老远喊：「晦气翁来了！」

另一个娃扔土块，没扔准，被娘一把揪住耳朵：「别挨他！晦气沾上了，洗三天都洗不掉！」

娃哇一声哭了。娘拽着跑远，还回头啐一口。韩泥眼皮都没抬。户籍册上写着绝户，一人过，无儿无女，死了连摔盆的人都没有。抬棺、扫庙、编筐、采药，五十八年就这么过来。

土岗西坡土薄，雨多风也多。茅草屋顶一响，人心就发空。别人家漏一点补一点，他补不起，旧了就让它旧。藤条是前日后山采的野藤，刮皮、浸水、编紧——结松了，筐就塌；筐塌了，钱就没处落。

「韩泥！」

马春花的声音从井台砸过来，尖得扎汤。棒槌跟着在盆沿上敲一下：「编你的筐！少往村道中间杵，挡了运！挡了运，你赔得起？」

韩泥嗯了一声，藤条没停：「赔不起。所以挪了。」

马春花哼一声，没再近。近也近不出钱来，骂也骂不出钱来。她骂她的，他编他的。他心里另有一本账——去年抬棺路过她家，她没拦。恩轻，记着；谢字欠着，等有还的时候。

日头偏西，筐底圆了，沿齐了。他用指节敲敲底，声闷而实。实了，才配装药，配卖钱。起身时腿麻，拄了一下门框。门框朽，跟指节一样起皮裂口。

泥屋里黑潮，霉气贴脸。灶上无米，瓮里无水，墙角堆着苦蒿和半捆野藤。能换铜板的留着，换不来的晒干当柴。穷归穷，哪块砖松、哪根椽朽，他身子比眼记得牢。

他把两只筐叠好，草绳捆上，绳结仍是爹教的。爹咽气那夜手搭坛沿，嗓子哑得很：「坛别扔。韩家就剩你，坛在，根在。」

沿凉，摸上去跟冬夜一样。祖父老栓从不卖、从不扔，只留这句话。韩泥不知真名，只唤腌菜坛。每日用旧布擦沿，布洗得发白，擦得惯了，习惯也成了根。

屋角那只坛蒙尘无光，酸菜早吃完了，空着当摆设。摆设也是家当——坛在，屋在，人在。村里人笑：「绝户翁连坛都蒙尘，命也灰。」他不答。册上绝户二字擦不掉，坛沿还能擦亮一层。

午后，他担两只筐赴集尾。道窄人杂，货担铃铛响两声又远了。卖盐老汉冲他点点头，很快把脸别开，怕晦气从跨过的影子里跟着走。韩泥不近不躲，担筐走稳——走稳了，辱也追不上步。

买主是个挑盐的，老熟人，打量筐沿：「编得紧。不松。」

「紧才装得住药。」

买主捏了捏底，底不塌，从腰里摸出铜钱，数了两遍才递过来：「十二文两只。晦气翁的手艺，倒实在。给你。」

「下回还来。还编这种底。」

「还来，底还紧。」

铜钱落掌心，十二文一文不少。韩泥收进腰布，布沉了一截——这是今日小胜。辱在村道，钱在掌心；钱不暖，能买米，米能活命。他不回嘴，担起空筐走稳，把十二文也编进步子。

买主在身后嚷：「底别塌！」

韩泥没回头：「塌不了。」

巷口沈枯芽娘拽娃让开。芽儿仰脸问：「韩爷爷，筐紧不紧？」

娘捂嘴拖到暗处：「紧不紧，别问晦气。」

韩泥脚步没停：「紧。装药不紧，人倒霉。」

话落，自己也觉晦。晦归晦，手不停。他绕远半步出集，风把货摊上的草屑卷起来又落下，像村道上的话——起得快，散得也快。

回屋时雨细绵，专找漏瓦钻。韩泥拿缺口陶碗接屋顶漏水，碗小，接得慢，水漏进碗又漏进泥地，转眼没了。他忽然想，这辈子是不是也这样接雨——接一碗漏一碗，到头来还是干的。门口半筐野藤往里拢了拢，免得雨泡烂。明日天不亮还得上山，苦蒿蛇藤都得备。

雨大了，漏打湿半边衣襟。雨丝扎在麻脸上，针扎，也洗，他没躲。村道那头，赵绣春跟几个妇人笑，笑里带刺。赵家做皮货，走路带风，利起来比刀还快。韩泥与她无冤也无恩，只晓得那人嘴利——嘴利不要紧，心里自有一本账。

他把碗搁下，坐到坛边，低声道：「不漏人就行。」

话音落下，坛壁轻轻一颤，极轻，像呼吸。雨珠落沿，蒙尘里浮出一线极淡水汽，像热汤刚揭盖那一瞬，一闪又敛。韩泥以为手抖，没理会。丑时坛壁微凉，午时又温，他当天候作怪，从不多想。

今日擦坛，指尖顿了顿，沿仍跟凡物无二。他按了按心口——空了几十年，近来梦里常梦见欠着什么，醒来手心发热。急什么，五十八年都过来了。

泥岗村入夜早，远处喊娃吃饭，喊声拖长，断在雨里。灶冷，他不喊饿，饿惯了。村童又嚷：「晦气翁，别近！」张麻婆咳一声：「少嚷。嚷多了嘴烂。」童散了，张麻婆也没近这间漏屋——近了，也沾晦气。

韩泥嘴角动了一下，不知是笑还是咳。摸腰布，十二文硌着，踏实。村嘲在远，坛在角，钱在腰。沿在，根就不散；人不问，坛也不答。他合上眼，手仍按坛沿。五十八岁，绝户，晦气翁，还在村西这间最漏的屋里。雨声里沿口一线温，贴住虎口，脆而短，够撑一日，脆如十二文落袋。沿温罢又凉——凉也罢，瓮未醒，他不喊醒。

隔壁张麻婆骂街骂得凶，尾声却落一句：「韩泥那筐，编得倒是结实。」他听见了，没接话。接话像讨夸，夸轻，筐重。庙祝那边，昨儿扫庙尘土还沾着裤腿，他没拍——拍着也嫌邀功。

雨夜长，他盘算明日的活：藤条再浸一盆，苦蒿还得上山看一趟。盘算完，心就不乱。乱则手飘，手飘，筐就松。松不得，松了，十二文买来的那半寸活路就白稳了。

风从门缝灌进来，灌得藤条更僵。他抬手塞了块破布堵缝，堵不住雨，堵得住半寸寒。屋里坛角静着，静得像一口扣着的小瓮。静也罢，静着就别惹人注意；惹人注意了，唾沫比雨还密。

他低声又道：「坛别扔——记住了。」

坛不应。不应好，应了才吓人。爹的话还在耳边，比铜钱更沉。铜钱会花完，话不会。花完还能挣，话忘了就散了根。

夜里远处棒槌又砸了一下盆沿，砸得短而脆。像给明日留半寸活路。活路在，手就还得稳。他摸了摸腰布死结，结死了才睡得着——睡浅一点也好，浅着才听见雨，听见漏，听见屋角那点动静。

他想起爹临终那夜屋里也漏。爹不看天，只看坛沿，手指青白，却把「坛别扔」咬得很死。韩泥那时不懂，只懂点头。如今五十八年过去，点过的头还在，坛还在，人才没散成路边一把土。

集尾卖过的那两只筐，底紧沿齐，是他唯一拿得出手的体面。体面不大，大不过骂；骂再大，也大不过十二文落袋的脆响。脆响一过，肚子就知道该往哪边挪——挪向米，挪向明天的藤条，挪向还能编下去的手。

邻舍有人夜里路过，隔墙嘀咕：「绝户翁还点灯？晦气也耗油。」他又不当回事。油灯豆小，照得见坛角，照不见闲话。闲话照不见，就不配进这间屋。

他翻了一次身，肩骨响，响完又静。静里只有雨。雨大了，缺口碗接着，听得见滴答；滴答也算声响，至少证明漏屋还在接，人还在听，日子还没断。

坛在角，明日还编。编了筐，换钱，买米，活。这日子丑，却真；真就够他再撑一日。
