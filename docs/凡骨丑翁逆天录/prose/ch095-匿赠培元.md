# 第九十五章 匿赠培元

雨夜。药山废窑窑口红，红退到掌纹能控的程度。

第三回，韩泥不再试。这一回，炼。青芯五撮、褐渣两把、灵泉七滴，投序照经页。页黄，字丑——丑路才像他的路。

炉温起一寸，稳。渣白气聚而不散。散则败；他败过两回，第三回要把气拢住。鼻下嗅诀如秤：秤香，秤色，秤那一点「正」。正来得慢，慢才匿得住。

丑时中，丹形初聚，小如豆。小正好——小才塞得进门缝囊。成丹一瞬，窑内无雷无光，只有掌纹烫一线。烫像叶汤碗沿，沿热进心记。

他取丹。丹温，温而不灼。灼了像邪，邪了路断。路不断，丹入温汤囊，囊缝丑，缝却暖。

他不立刻下山。下山赖福会嗅。他在窑内等到雨歇、星晦，晦了才走。

坡下小舍门缝仍宽一线。一线光里，叶青禾未睡，眼望缝，像等，又不敢等。

韩泥不现形。现形像连累。他把囊系细藤，线过缝，轻轻一拽——囊入。门外油豆爆一声，爆完即灭，灭像无人来过。

缝内，叶青禾触囊，触到温。温她识得，识得像当年姜汤藏碗底那一点热。

「……韩叔？」她低声。

缝外无声。无声才是还恩。

她开囊。囊里一粒培元丹，旁一张粗纸，纸上三字：「趁热。」

就三字。无署名，无「韩」。无韩才匿；匿才赠得出。

她指节白，半晌，丹入口。入口温，温进脉，脉里一线培元，弱而真。真，她不敢声张——声张连累丙九。

「趁热……」她对着缝又说一遍，嗓子哑，「我……记下了。」

沈枯芽在远舍未眠，却不知丹，只知「别灭灯」已传过去。韩泥回兽栏，雨泥没踝，踝冷，心热。热不应喜，应「还」——还一半，还在暗处；暗处还，才配叶丫头。

赖福查夜，火折照袖：「什么味儿？」

「药渣。」韩泥声平，「栏腥，洗不净。」

「夜里上山了？」

「倒过夜渣。」韩泥说，「雨大，路滑，回来晚。」

赖福鼻动半息，没再追。没追也好——丹已赠，赠在缝，赠在温。

席下他按坛：「还了。不叫。」

坛腹温，贴掌像应「趁热」。趁热是叶汤的回响；回响在，恩就不凉。

刘婆端粥，粥稀：「雨夜别乱跑。」

「在栏。」韩泥说，「手稳。」

「手稳好。」刘婆哼，「乱跑的人，粥也烫不热肚。」

邻床睡实。睡实了，秘密才藏得住。他摸怀经页，「培元」二字深了一线。深是起；起在暗处，起在还恩。

更鼓尽，坡下门缝灯灭。灭是她睡了；睡了，丹在脉里温着。韩泥胸口空半寸，空得舒服——舒服不是无债，是债还了一笔，还能还下一笔。

「叶汤，还了一半。」他对坛低声，「剩下一半，在省亲前。」

坛不应字，只温。温像爹搭沿，像门缝那一线黄。

清晨芽儿跑来，眼红：「叶姐让我说……灯灭了，可身子暖了。她问……是不是你。」

「别答是。」韩泥道，「答了，她会怕。怕了，灯白留。」

「那我答什么？」

「答不知道。」韩泥说，「不知道最稳。稳了，她才能继续睡着。」

「韩叔，你真狠。」芽儿嘟囔。

「狠在嘴上。」韩泥应，「心不狠。心还在记着热。」

铁无言午后在渣场看他一眼：「手上有焦。」

「试温。」韩泥说。

「试温别炫。」铁无言声低，「炫了，像求名。求名，路断。」

「不求。」韩泥应，「只还。」

白织月袖风远扫：「雨夜窑红，谁烧的？」

「渣场水汽。」韩泥不抬头，「看错了。」

她唇线绷一息，走了。少一句嘲，够了。秦霜木鱼轻敲：「暗赠别骄。」

韩泥心里应：不骄，只还。还一半就够这一夜——够她脉稳半分，够他把七笔旧恩里那一笔叶汤按实。

夜再静时，他数了数：叶汤这一笔，暗还一半；剩一半，留给省亲那日当面。当面远，手先还。还了不叫，坛沿净，赖福嗅不到，丙九不连坐。

同棚有人哼坊市小调，调子轻浮：「听说外门要开挂牌，小散也能换钱……」

韩泥听进耳，不接嘴。换钱是后头的事。后头有坊市风声，有挂牌，有更远的兽潮条。条在远，这一夜只还一半——一半进了叶青禾脉里，温着，像汤，像灯，像门缝那声油豆爆。

他编筐半刻，末藤勒实：「下一程，听坊市。听清楚了，再决定手往哪伸。手伸偏了，刚还的恩也会脏。」

坛温一线，像应「还」字。应完仍不叫丹名——丹名留在缝里那三字「趁热」上就够。够了，人才能睡；睡前把木牌贴胸，牌凉掌烫。凉的是约，烫的是还。

雨停后第二日，叶青禾在坡下晒药叶，药叶薄，薄里手稳一分。稳，旁人看不出培元丹，只看出她脸色不像前几天那般灰。

沈枯芽小声：「姐，你夜里暖了？」

「暖了。」她说，「别问谁。问了，福气跑。」

「我不问。」芽儿拍胸，「韩叔也不让我问。」

她眼睫颤一下，没再说话。说话容易把「趁热」三字说漏。漏了，丙九那些人就要倒霉。她宁可欠着，也不准恩变成祸。

韩泥远处过药田，不抬头，只听得见叶家方向风轻一点。轻一点就够。够了把胸口那半寸空再按实——空是还过的位置，位置在，下一笔才写得下。

管事弟子巡到废窑远看一眼：「昨夜雨大，窑塌没有？」

「没塌。」韩泥说，「旧窑经雨。」

「杂役少靠近。」

「倒渣才靠近。」韩泥应，「渣倒完就走。」

人走了，他才在心里把成丹那一瞬再过一遍：匀一寸、聚而不散、温而不灼。灼不得，灼了叶丫头脉受不住；受得住，才叫还，不叫害。

夜里他又摸温汤囊，囊空了。空得好——空说明东西到了该到的地方。地方在缝里，缝外无我；无我，恩才干净。

刘婆端粥多半勺：「你今早眼神稳。」

「粥稳。」韩泥说。

「粥稳就行。」刘婆哼，「眼神稳的人少惹事。惹事的，坊市风再响也进不了肚。」

「不惹。」韩泥应，「听风可以，伸手要慢。」

第二日坊市方向烟尘起了一截。有人喊：「外门登记挂牌预告贴出来了！」贴远韩泥也没挤去看。看了容易手痒；手痒则骄；骄则把夜里那粒丹的静弄脏。

他守着清栏。栏清完，鼻下仍稳。稳里，叶汤那一笔已经从「欠」挪到「还一半」。一半不声张，声张会把另一半也逼到悬崖上。

沈枯芽中午悄悄来：「姐脉稳了。她让我谢谢……谢谢送灯的人。她没说名字。」

「别谢。」韩泥说，「谢了像交易。交易脏。」

「那我怎么说？」

「说知道了。」韩泥道，「知道就够。够了，她继续吃药叶，你继续别问。」

芽儿应着跑开。韩泥望药田一口井，井水映半张丑脸。丑脸可以送丹，也可以不露面——不露面，正是这一章的骨。骨硬，恩才不软成纠缠。

午后白织月又讽：「听说有人雨夜炼了粒破丹？破丹也配称丹？」

「灰多。」韩泥说，「灰里偶有糊。糊不当丹。」

「你嘴倒滑。」

「倒渣滑。」韩泥应，「不倒滑，灰糊嘴。」

她走了。铁无言远远丢一句：「糊说得好。真丹更要装糊。装糊，活得长。」

「记着。」韩泥应。

夜里棚外风硬，硬里有人念挂牌价。价高低与他此刻无关。关的是坛温、席严、袖净——袖净说明温汤囊空了，空了说明培元丹已经到了叶青禾脉里。到了，就不许再碰她门缝，除非省亲那日能堂堂正正站在亮处。

亮处还远。远着也好，远着逼他继续熬。熬到能把剩下那一半也还清。还清以前，坊市的尘、兽潮的条、三年约的凉，都只能一件一件接。一件一件，才不乱。

他最后摸一次经页。「培元」深了一线。深是起，起在暗处，起在「趁热」。趁热二字，比署名更重——署了名，恩变债据；不署名，恩仍是人温。人温够了，这一夜就算落地。

雨沟滴尽。他贴木牌入眠前，只把坊市方向那一点尘听进耳里：尘起之日，钩在后头；这一夜的还，已经落在叶脉里，温着，像灯，像汤，像必还的未完那一半。
