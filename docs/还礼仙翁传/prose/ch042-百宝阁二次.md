# 第四十二章 百宝阁二次

秦商言二次入山，驻点设在坊市驿，人未到，货先到——隐匿符材料一小匣，附字条：「夜袭需暗线。茶饼之恩，商言还一半。」

莫长春至驻点，未带弟子，灰袍素，像老客串门。驿外弟子偷瞧，偷瞧里好奇——百宝阁女修二次来，送的不是灵石，是符材，像又一出「茶比灵石贵」的戏。

秦商言斟茶，茶比上次好，仍非灵品，像刻意让他记得「家乡」与「买卖」的差别：「赤焰夜袭时，东脉侧径最险。暗线若断，丹堂先危。」

「所以你来送材料。」莫长春道。

「来换线索。」秦商言笑，「公平。」

他取茶饼余半块，推过去：「上次你说无价最难还。老夫再给你半块，把难还再推一推。」

秦商言怔，继而笑出声，笑里真了几分：「莫长老，商言亏本乐意。」

「亏本好。」莫长春道，「亏本的人，记得恩。」

她低声：「公平二字，我常说。遇见你，才知公平不是等价，是肯不肯还。」

莫长春不接情话，只问：「材料谁手？」

「百宝阁最好的裁纸人。」秦商言道，「裁符纸如裁命，贵，值。」

他收匣，袖中簿一烫。秦商言望他袖，像觉出那里有什么，终未问，只道：「商言还有一句。周德海近月与赤焰信使频见，驿馆后巷，三次。商言不卖此条，记恩账。」

莫长春点头，像落账：「记下了。恩账比买卖账难还，你还得起。」

秦商言起身，郑重揖手。出驿时，弟子窃语：「长老又送半块茶？」「百宝阁管事又笑了。」「这次笑得像真亏。」

霍镇山闻讯，骂：「好！暗线要活，丹堂才稳！」

周德海在长老院听闻，面色平，平里冷，像早就备好的稿。

弟子围驿外，围「半块茶饼再送」，围得像看亏本乐意。

顾小满在杂役堂分班，眼里亮，亮得像雨夜金光还在。

莫长春不辩，杖点青石板，像钩，钩向东脉侧径：

「材料到了。暗线能不能活，看杂役堂敢不敢走夜路。」

魏无涯在檐角观，扇合，低笑：「周德海三次见信使，线深了。」

风腥，云低，子时像已贴在靴底。

驿外弟子偷瞧，偷瞧里好奇——百宝阁女修二次来，送的不是灵石，是符材，像又一出「茶比灵石贵」的戏。「上次半块茶，这次一匣符。」「秦管事又亏？」「亏得像赚。」

顾小满在杂役堂分班，眼里亮，亮得像雨夜金光还在。她问：「隐匿符……小满能学吗？」杂役师兄答：「学不学，看长老肯不肯给路。」

秦商言出驿，望青岚山门，低声对下属：「莫长老的难还，商言认。认账的人，不走绝路。」下属问：「若周德海真是奸……」秦商言截话：「奸不奸，商言不卖。商言只记：茶比灵石贵，暗线比货贵。」

赵玄机在思过崖，听坊市驿消息，指节白，像妒里又掺进一丝说不清的东西——百宝阁女修二次来，不为卖，为还。

茶饼又少半块，符材多一匣——还礼仙翁的账，从来不算公平，只算肯不肯还。

风腥，云低，子时像已贴在靴底。弟子围驿外，围「半块茶饼再送」，围得像看亏本乐意——乐意的人，记得恩；记得恩的人，不走绝路。

秦商言望他袖，像觉出那里有什么，终未问，只道：「商言还有一句。周德海近月与赤焰信使频见，驿馆后巷，三次。商言不卖此条，记恩账。」莫长春点头，像落账：「记下了。恩账比买卖账难还，你还得起。」隐匿符材在匣，匣在袖，袖里有网，网在子时。

莫长春不辩，杖点青石板，像钩，钩向东脉侧径：「材料到了。暗线能不能活，看杂役堂敢不敢走夜路。」驿外弟子仍围，围得像看亏本乐意——乐意的人，走夜路。

秦商言起身，郑重揖手。出驿时，弟子窃语：「长老又送半块茶？」「百宝阁管事又笑了。」「这次笑得像真亏。」茶饼又少半块，符材多一匣——还礼仙翁的账，从来不算公平，只算肯不肯还。

「亏本好。」莫长春道，「亏本的人，记得恩。」她低声：「公平二字，我常说。遇见你，才知公平不是等价，是肯不肯还。」

魏无涯檐角观，扇合，低笑：「周德海三次见信使，线深了。」材料到了，暗线能不能活，看杂役堂敢不敢走夜路。

秦商言出驿，望青岚山门，低声：「莫长老的难还，商言认。认账的人，不走绝路。」风腥，云低，子时像已贴在靴底。
