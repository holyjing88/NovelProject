# 第五十九章 沿前即止

冬更深，丙九静。静里，坛温却醒半分。醒半分，不是醒全，像瓮翻身，翻完又眠。韩泥编筐末藤，藤刺再扎，扎得深，血珠再饱满。他仍甩向破席。席旧，吸血，血渗至席边，沿前止。止，像一道看不见的线。线内是席，线外是沿。沿上无尘，沿上无血。邻床梦呓：「……红……」韩泥盖席，声低：「你看错了。睡。」血到了席，没到沿。沿在眠，血不主动碰，这是铁规。符怀暖一丝，护手不僵。僵，下一刺便偏。不偏，缘才正。老耿门外咳：「沿净，好。净，才接得住赠。」韩泥「嗯」一声，不追问。问急，像邪。等，像守瓮人。耿脚步远。褐袍僧影已散，像凡符使命已了。余下瓮与血缘，余下席与沿。---

午后，清西栏，粪锹沉，掌茧裂口渗潮。潮不是血，血不碰沿。管事盯梢：「沿净，记在心里。脏沿，像拜邪。」韩泥「嗯」一声：「不拜邪。拜邪，恩断。」赖福讽：「守坛守出仙缘？」管事声冷：「再嗓，西栏你一人清到年尾。」赖福哑了。哑，韩泥肩上的粪锹却捆得更实。刘婆端粥来，稀，热：「冬深，别空肚。」「不空。」韩泥喝尽，「嗯」一声。热在，像第七笔又实。铁无言巡栏边，声低：「沿净？」「净。」韩泥说，「血在席，不在沿。」铁无言「嗯」一声，灯远。丑时前，坛腹温又一线，温短，像应席边血气，不应沿。他低声对坛：「你再眠一夜。眠着，我席盖严。严，不主动滴你。滴，要等扎那一刺，误那一甩。」坛不应字，只温，温得像「嗯」。外头风息，柱上无新条。明日仍搬柴，仍清栏，仍末排。末排不要紧，要紧是沿净，手稳，心记着七笔。记着，才接血止沿前，接丑时坛温，接席盖更严。他按席，席厚，厚像守了六十日的命。命跟坛，坛跟缘，缘在沿前那一止。丙九早课远，远像另一个时辰。时辰内，只有编筐声，柴房声，坛温起伏。他盖席，盖严，严像藏私。私是坛，坛是命。他是记心翁，恩仇都记在心里。血是活计扎出的，不是主动滴坛的。滴坛，要等误那一甩。秦霜远廊一句：「手稳，心别邪。邪，符灭。」韩泥怀符暖一丝，心记着七笔，不邪。同批少年碰他袖：「你席盖那么严，藏啥？」「坛。」韩泥说，「坛在眠，不在醒。」「醒了呢？」「醒了，缘满。」韩泥说，「缘满，不在急。」少年不懂，哼一声，走了。他独眼平：「沿在眠，血不主动碰。碰，要等活计误那一甩。」坛腹温一线，贴掌，像听「止」字，不急应。外头更鼓三响，响沉。沉里，瓮温醒半分，又眠回去。眠回去不要紧，要紧是沿净。他低声道：「沿前即止，不接沿。不接，才接得住瓮醒。」沿滑，像祖父韩老栓的手。手不在，滑还在。滑不要紧，沿上无血，无血才净。刘婆门外咳：「沿净，好。净，才少病。」韩泥应：「净。净，像守瓮人。」他擦沿，沿净，沿滑。净，像六十日擦出来的规矩。规矩在沿，不在舌。坛壁水汽爬半寸，敛。敛着，像替他守着。手别飘，恩不断。外头风息，风息了，席却更严。严，像后头那一夜预演。血渗至席边，沿前止。止，像看不见的线。线内席，线外沿。他低声对坛：「你再眠。眠着，我守沿，守席，守不主动滴血。」窄路也是路。路通向瓮醒，瓮醒通向还。还叶汤，还振东药。还，先还烫。第二日，仍编筐，仍清栏。清栏腥，符暖护骨，手不僵。藤刺再扎，血珠再饱满，仍甩向破席。席旧，吸血，沿前止。管事验沿：「沿净，好。」韩泥「嗯」一声：「净。」铁无言巡栏边，声低：「血在席，不在沿。沿净，像等缘的人。」韩泥只答：「等。」邻床又问：「你席盖那么严，藏啥？」哑，韩泥按席的手却更稳。刘婆端粥，稀，热：「冬深，别空肚。」第三日，丙九静。醒半分，不是醒全。编筐末藤，藤刺再扎，扎得深，血珠再饱满。仍甩向破席。管事验沿：「沿净，记在心里。」刘婆端粥，稀，热：「沿净，好。」他盖席，盖严，严像后头那一夜预演。邻床梦呓「红」，他盖席，声低：「你看错了。」管事又嘱：「沿净，仍搬柴。搬柴手抖，沿易脏。」韩泥「嗯」一声，肩稳。同批少年问：「沿净干啥？」「沿净，才接得住缘。」韩泥说，「缘，不在急。」铁无言廊角，声低：「沿净，别拜邪。」韩泥「嗯」一声：「不拜。守沿，等缘。」刘婆又端半勺粥，热：「抖的是心，不是手。心记着，手就稳。半勺，热在，像第七笔粥又实一分。坡下霜白，柱上无新条。无条，有条在坛。他摸木牌，与席严并放。并放，像命与规矩挨着。挨着，才接得住瓮醒。清晨，丙九静。韩泥清西栏，粪锹沉，掌茧渗潮。编筐末藤，藤刺再扎，血珠再饱满。」缘在沿前那一止。止了，才接得住瓮醒。」清晨，丙九早课远。远像另一个时辰。」天未亮，丙九棚里静。静里，编筐声细，细像针。针扎掌，掌茧裂，血珠饱满。他甩手，血甩向破席。席旧，吸尽，血渗至席边，沿前止。」韩泥「嗯」一声：「记在心里，不在舌。」赖福远处骂：「晦气翁守坛。」哑，韩泥擦沿的手却更稳。」睡。」刘婆端粥，稀，热：「冬深，沿净，好。」眠着，我守沿，守席。」清晨霜白，丙九静一寸。静一寸，像沿前那一止。编筐末藤，藤刺再扎，血珠饱满。甩手，血甩向破席。席边湿一线，湿不到沿。不到，像瓮仍眠。他独眼平：「沿前即止，不接沿。沿前即止那日，席边湿一线，湿不到沿，沿不接，缘在等。赖福远处骂，骂轻，韩泥擦沿的手却更稳。清晨霜白，丙九静一寸，静一寸，像沿前那一止。他独眼平，手不抖：「沿前即止，不接沿。」坛腹温一线，贴掌，像听「止」字。静里，丙九早课远，远像另一个时辰。藤刺再扎，血珠再饱满。饱满，仍甩向破席——席旧，吸血，血渗至席边，沿前止。线内席，线外沿；沿无尘，沿无血，沿滑像祖父手。私是坛，坛是命，命跟了五十八年。老耿门外咳，咳像提醒：「沿净，好。问急，像邪；等，像守瓮人。褐袍僧影已散，散，像凡符使命已了。

沿前即止，席吸尽，沿仍净。净，才接丑时坛温，接席盖更严那一夜。
