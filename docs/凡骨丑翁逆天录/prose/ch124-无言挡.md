# 第一百二十四章 无言挡

试台风声紧那几天，赖福一伙最爱嚼一句：「盗渣炼散。」

这话在丙九灶边散开，臭得像馊锅。韩泥端着糙碗喝粥，粥稀，刘婆仍舀满。满碗像挡半天饿，挡不住嚼舌。挡嚼舌的，是铁无言。

铁无言立在灶侧，声沉：「盗不盗，渣场账在我这。韩泥账清，手稳。再嚼，我去管事处。」

角落里还是有人嘟囔：「铁师兄护晦气，晦气传染！」

铁无言不抬杠，直接掷出账簿，朱印拍得脆：「昨夜三袋渣，运丹房，有单。谁盗，谁指印。」

谁也不敢按指印。赖福脸青一阵，骂骂咧咧走了。韩泥只谢一字：「谢。」

「谢轻。」铁无言说，「试台那天，我站坡侧。不替你炼，替你挡粪话。」

「挡就好。」韩泥道，「舌头要留给按罐，留给守坛。」

白织月廊下过，袖风冷：「铁师兄护久了，也沾晦气。」

铁无言抱臂：「晦气在嘴，不在手。」

韩泥不插嘴。嘴一开，嚼舌的人就又有料。他宁可把这一日挡话按进心里：挡的是粪话，不是替他动手。动手的仍是他自己——手稳了，十二层才稳；十二层稳了，试台上才不飘。

刘婆端来姜汤，姜薄，热：「铁哑巴护你？」

「护。」韩泥说。

「护就好。」刘婆哼，「有人护舌，你护手。手稳，饭稳。饭稳，恩不断。」

他把姜汤喝尽。谢轻，恩重。

灶边又有人咬耳朵：「铁师兄真护他？」对面压声：「账在他手里。你敢咬，你先对账。」耳朵话散了，韩泥当没听见。听见了也只护手，不护脸。恩在姜里，也在那一掷簿里——簿清，盗谣就虚；谣虚，试台前口才净半分。净半分，够把夺魁预备实一点，够把刘婆布包焐得更热。

午后废窑，他开最后一炉备战。炉温稳得像秤。秤平了，再炼两粒，九粒齐。九粒够上台，够把末项拴住——拴住了，后头还衣还钱才不飘。芽儿半块石已清，刘婆待还，叶汤在省亲前：序清了，粒才正。

赖福闯窑，被铁无言拦在门外：「管事区，验账再来。」

赖福悻悻退。韩泥把门关严，只记两字：余毒尽。尽了，才顾得上试台近。近不要紧慌，粒正手稳，钱戾衡影再硬也硬不过「正」。正了，刘婆敢认恩，芽儿敢收石，叶汤还烫在前头。

沈枯芽送来一小束干药草，草凡，凡得像泥岗路边：「叶姑姑说，试台手别抖。」

韩泥收下，像接一点徒线余温。温浅，够记。记着芽儿恩还了，叶汤还在更前头——前头那碗烫，还到了手，才谈得上一轮圆满。圆满不在魁名，在债序清。

邻床少年悄问：「铁无言为啥帮你？」

「并肩走过一回。」韩泥说，「并肩，不替炼，替挡话。」

「挡完你就赢？」

「挡完口净。」韩泥摸玉牌，「赢，在试台见。」

牌凉贴胸。外头风声又紧一分，他只多稳一分手。一分一分熬，熬到十二层圆满，熬到十三层该破再破。破早了像叫，破晚了像怂；不早不晚，要看帖和台。

丑时练鼻下静，粟壳香淡。挡话这一日进心里，刃亮半分——亮给试台，亮给还恩，不给嚼声。嚼声远了，远不过九粒「正」。坛腹微温，像应着「挡」字。挡了口舌，挡不了仇。仇在玉牌，在三年约，在钱戾衡那句「记住今日」。记着，就不抖；不抖，才配上台，才配把养老钱交到补衣婆手里。

第二日炉再稳一天，第九粒泥纹仍浅，嗅之却正。正了才封罐，封泥，封严。示人一天，赖福就多一天由头；由头多了，除名近。除名近了，三年约白接，叶汤凉，刘婆更拖后——拖不得，所以先把口堵住，再把火守住。

铁无言坡侧路过，又丢一句：「末项前，别辩。辩多了，手散。」

「不辩。」韩泥应。

「不辩，就还。」铁无言顿了顿，「还到该还的人手里，才算数。」

韩泥点头。算数的事不喊，喊了像讨彩。他只把九粒理进怀，理得像理账。账清了，步才不飘；步不飘，试台才不洒；不洒，钱戾衡那句「上台别洒」才落空。落空了，白织月说「运气」也尖不过一鼻正香。

夜里更鼓沉，他按木牌贴胸。牌凉，心热。热的是粥，是姜，是并肩那一掷簿。簿在，盗谣虚；虚了，他端碗仍稳，窑温仍稳，九粒仍正。正了，明日验香、后日听帖，才一个接一个挨得住。

「试台，见。」他对坛低声。

窑角药草束轻轻晃了一下。晃得小，却够他把叶青禾那句「手别抖」又按实一遍。按实了，明日再听嚼舌也不慌。

见字落，路又明一寸——明向封窑七日，明向刘婆补袖，明向把布包递出去那一刻。递出去那天若还远，今天就把嚼声挡在门外。门外挡干净了，门内火才匀；火匀了，帖来了也不乱。乱不得。一乱，锤子还没落，手先软。

午后他特意把账簿字号在心里过一遍：哪三袋、哪日、哪押印。过完不是背书，是防赖福再咬。再咬时他能指得出路，路清楚，铁无言那一掷才不落空。落空了，兄弟线也丢脸；丢脸事小，试台口不净事大。口不净，验香前先腥一半。

同棚有人凑近：「你偷没偷？」说完自己先缩脖子。

「账在铁师兄那儿。」韩泥不抬头，「要看，去看朱印。」

那人噎住，骂了句晦气走开。走开好。走开了，他才能安心开炉。炉火映得脸丑更丑，丑不要紧，香正才要紧。香正了，九粒才算粒；算粒了，挡话才算帮到点儿上。帮到点儿上，他谢得起；谢不起的人情，他不揽。

赖福夜里又在廊外晃木牌，牌撞击声刺耳：「试台滚，省我工夫！」

韩泥隔门只答：「在。」

挡话后的夜里，他没急着睡，把九粒一粒粒对鼻下过。过完不像炫耀，像过关。关口是香，香正才算没浪费铁无言那一掷。掷簿挡的是嘴，挡不住他自己手飘——手飘只能自己按。

沈枯芽白天塞来的干药草有点潮，他挪到暖处烘干。烘干时低声：「告诉叶姑姑，手我稳着。」

话没人听见，听见也没关系。关系大的是明日封窑前，这九粒还能否闻出同一路正。同一路了，盗谣再起也咬不穿。咬不穿，试台前那口气才匀。

铁无言再度路过，敲敲窑门：「口净了？」

「净了半边。」韩泥说，「另半边看验。」

「半边够活。」铁无言道，「活到验，再活到帖。帖来了，你别先跪。」

「不跪。」韩泥应，「跪了，恩先断。」

在，就不滚。不滚，才守得住这九粒，守得住铁无言替他挡下的那半张嘴。嘴挡了，手还得自己硬。硬到试台开，硬到验字落，硬到坡下帖角被风掀起还能按住——按住了，还恩的路才不断。

窑角草香渐渐干透，干透后更像泥岗路边的味。味凡，却勾着他记叶汤。汤还在前，簿挡在侧，九粒在怀——三样都在，步才迈得开。迈开了，验字才接得住。

夜风把柴挡吹得轻响，他听响辨缝——缝大了就塞，塞严了炉才匀。炉匀，九粒才共一路香。一路香了，才配把「盗」字彻底晾干。

他把干药草束挂在窑角，草香淡得几乎没有，却够提醒：叶青禾还在前头望。望的不是魁名，是他手还稳不稳。稳，汤才还得动；不稳，汤凉在谣里。谣他见得多了，从村议到宗门，变的是地方，不变的是有人爱泼脏水。

脏水今朝被簿挡住，明朝还可能换个法子来。来了再挡，挡完再炼——炼到锤子落下那天，香自己会给答案。答案若正，嚼舌的人只能咽回去。
