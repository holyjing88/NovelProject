# 第八章 青狼袭村

东脉李家村的妖讯，是在顾小满喝完热粥的翌日清晨到的。

报讯弟子满身泥，喘着撞进山门：「青狼妖围村，伤三人，请长老援！」

霍镇山已在石道拦人，盾未亮，嗓门先亮：「莫长老若去，镇妖司记恩；若不去，老夫自己去送死！」

莫长春拄杖出院：「霍老头，别喊。去。」

村外炊烟断，犬吠绝，只剩妖风腥。

青狼妖伏在塌了一半的院墙上，眼绿，爪利，像把夜撕剩的布挂在牙上。村民缩在灶下，孩子哭，被母亲死死捂住。

顾小满扶老妇转移，脚下一滑，险些栽进妖口——她本不该来，却跟着杂役队送药，像半块灵石还在掌心里推她往前。

妖扑。

莫长春杖至。

不快，像老人散步；却准，杖尖点地，竹影暴起，如索如鞭，缠妖腰，一提一摔，青狼妖砸在晒谷场，尘起，骨断。

妖未绝，第二只从墙后扑出，爪直取顾小满——她退半步，除魔符未亮，莫长春已至，袖空，一念出塔，杖落，竹索再缠，再摔，第二只青狼砸在第一只旁，腥风止。

村民呆住，继而跪谢，谢得乱，像一场迟来的雨。

霍镇山抱拳，声洪：「莫长老，镇妖司记在心里！」

老村正颤巍巍上前，捧出一袋干栗，栗小，意重：「长老，村里穷，只有这个……」

莫长春推回：「栗留给孩子。老夫不取栗，取一句记恩。」

老村正红眼，点头，点头像把「九月老头救村」四个字，刻进自己心口。

莫长春不居功，袖空，一念出塔，除魔符落掌，黄纸朱纹，灵品，递顾小满：「下次别一个人挡。」

顾小满指尖抖，接符，像接第二块半石：「长老……我……」

「记着就行。」莫长春道，「恩不用嘴还，用命还的时候，别还错了人。」

---

回山路上，弟子群情振奋，外门终于不再只说「九月将死」。

林挽月望莫长春背影，忽然觉得「老」字不一定等于「弱」。她没敢说出口，怕赵师兄笑她动摇，却把除魔符的事记进心里——记得像半石，记得像恩。

苏念慈在丹堂廊下听讯，炉火正旺，她低声对弟子：「除魔符给了杂役……师兄总把重的给别人。」

弟子不敢接话。苏念慈望东脉方向，妒里掺一点敬，敬得像药渣里的甘。

霍镇山声洪，像要把救村功劳喊实：「莫长老除魔，镇妖司记恩！」

赵玄机在石道旁冷听，指节叩铁牌，一声一声，像把妒意敲实。

孙福在杂役堂阴脸，对弟子道：「半块石算什么，救村是长老本分！」

弟子不敢大声应，却有人小声：「孙管事，你克扣月例时，莫长老怎么算？」

孙福脸色一变，拂袖走，走得像怕下一阵，恩仇算到他头上。

夜，识海塔壁金铭浮起一行，淡金：

**除魔一念，善缘记在心里。**

莫长春合眼，望东脉方向，村灯重亮，像有人把掰开的灵石又拼回一半。

他念：「小满，符收好。下次别挡前面，挡侧面也行。」

风过，无人答，像话进了土里，等着长。

他望杖，笑：「霍老头记在心里，赵执事妒在心里。好，人多，恩才厚。」

杖中竹纹一游，像应。

村民扶老弱回屋，屋灯重亮。顾小满抱伤童回，童睡，睡得像半块石还在怀里。她低声：「长老，村保住了。」

莫长春点头：「村保住，门才保住。」

青狼退后，村外泥血未干，外门已起新话。

莫长春袖空，杖在塔中，望狼退方向，不追。弟子围上来，围得像盼他再出一手，一手没出，出的是一句：「狼退了，恩没退。」

赵玄机验阵痕，痕乱，乱得像想挑刺，刺挑不出——阵是杂役半块石撑的，名分上挑不出刺。

苏念慈送药至村口干，药苦，苦里带甘。她看莫长春，欲言又止，止得像妒火压进炉里。

村口干上，弟子把救村讲成「九月老头施法」，施法越传越邪，邪到莫长春自己听了也笑。

他袖空，杖在塔中，对传谣者道：「老夫若会施法，先施你们去睡觉。」弟子哄笑，笑里敬惧参半。

顾小满分药，药苦，苦里带甘，甘像半块石温。她声平：「狼退，恩未退。记下了。」

赵玄机验爪痕，痕深，深里挑不出刺——刺若挑，先挑迟到的自己。

周德海抚须，笑和气缓：「莫长老救村有功，外头嚼施法，嚼得没趣。赵执事，您也验了，验不出毛病，便验验自己的嘴。」

赵玄机脸一沉，未答。

霍镇山补一句，声洪：「再嚼救村是施法，镇妖司记名！」

弟子散，散得比妖风还快。

柳青鸢遣人送帖，六字：「师兄，别听风。」莫长春回帖四字：「听风也好。」

回山路上，顾小满把除魔符贴身藏好，藏得像藏第二块半石。霍镇山声洪，一路喊到山门：「莫长老除魔，镇妖司记恩！」喊声里，外门弟子第一次把「九月将死」四个字，说得小声。

赵玄机在石道旁冷听，指节叩铁牌，一声比一声轻——他妒，却开始疑：疑自己迟一步，迟得像故意；疑九月老头不居功，居功在理。

苏念慈送药至村口干，药苦，苦里带甘。她看莫长春，欲言又止，终只道：「师兄，符给了杂役，丹堂……也能给。」

莫长春笑：「你能给的，比符重。别急，渣在明日。」

顾小满在通铺角，把除魔符与半块灵石并藏，藏得像并藏两条还不清的恩。她低声：「狼退，恩未退。记下了。」

赵玄机验爪痕，痕深，深里挑不出刺——刺若挑，先挑迟到的自己。他自语：「将死之人……我何必要怕。」却把培元丹藏进暗格，藏得比罪还深。

村口干上，老村正把「记恩」二字教给孩童，孩童不懂修仙，只懂栗留给谁。莫长春在远处瞧见，不近，只点头，点头像收了一句将来要还的礼。

回山路上，弟子群情振奋，外门终于不再只说「九月将死」。林挽月望莫长春背影，忽然觉得「老」字不一定等于「弱」——弱的是谣，不是杖。

赵玄机验阵痕，痕乱，乱得像想挑刺，刺挑不出——阵是杂役半块石撑的，名分上挑不出刺。周德海抚须，笑和气缓：「莫长老救村有功，外头嚼施法，嚼得没趣。」

回山路上，霍镇山声洪，一路喊到山门：「莫长老除魔，镇妖司记恩！」喊声里，外门弟子第一次把「九月将死」四个字，说得小声。赵玄机在石道旁冷听，指节叩铁牌，一声比一声轻——他妒，却开始疑：疑自己迟一步，迟得像故意。

村口干上，弟子把救村讲成「九月老头施法」，施法越传越邪，邪到莫长春自己听了也笑。他袖空，杖在塔中，对传谣者道：「老夫若会施法，先施你们去睡觉。」弟子哄笑，笑里敬惧参半。

杖中竹纹一游，像应。风止。

村口烟未散，狼嚎远，远里像妖潮在试青岚门的胆——胆在，杖在，礼还在路上。

弟子渐散，散里仍回头瞧一眼灰袍袖空——瞧得像瞧一眼九月还能走多久。

青狼退后，村外泥血未干，外门已把故事讲成三出戏。

顾小满抱伤童回，童睡，睡得像半块石还在怀里。她低声：「长老，村保住了。」莫长春点头：「村保住，门才保住。」

风过，泥腥散，像把救村送进谣言里——谣来之前，先换一句笑。

他袖空，杖在塔中，对传谣者道：「老夫若会施法，先施你们去睡觉。」弟子哄笑，笑里敬惧参半，惧里掺盼。
