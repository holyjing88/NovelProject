# 第一百三十九章 风声紧

风声紧，紧在坊尾，紧在帖背。

钱戾衡的人不近身——近身像挑衅。挑衅他不接，只记影：影在条角，条丑，字真。

条上写得损：「预选丑骨，台上方出丑。」

有人念完就笑：「出丑都写明白了，他还挂牌？」

「挂牌是他的事。」旁人扯袖，「你笑大声，仇家听见，当你跟班。」

韩泥不辩。辩费舌；舌要留来守坛，留来候谢老耿那一日。

玉牌贴胸，又凉又烫——烫着三年约。约说：站到钱戾衡台前，才不算滚。

滚了，恩断；不滚，刘婆那笔才不算白还。

铁无言立三步外，声低：「风声紧，别接骂。」

「不接。」韩泥说，「接骂，像滚。」

「骂飞过来呢？」

「飞过来当风。」韩泥道，「风进耳，不进手。手稳，帖才压得住。」

坊市收摊，他立末，像石。石在，风过不留痕。

白织月袖风过，声冷：「钱戾衡的人，你接得住？」

「接。」韩泥声平，「接在台，不在坊。」

「台上看见丑骨，可不只嘲一句。」她唇角微挑，「你那三年约，是玩笑吗？」

「不是玩笑。」韩泥说，「是命。命说站，就站。」

她袖过，没停。没停，小胜。胜在不接坊骂，把火留到台上烧。

刘婆端姜汤来，汤烫，仍多半勺：「风硬，别吹歪路。」

「不歪。」韩泥说，「歪了，约白接。」

「白接啥。」刘婆骂得轻，「接不住仇，先接不住自己肚。喝。」

「喝。」他接碗，「记着。」

管事弟子传话，话硬：「风声紧，别接骂。接骂出事，簿上记你。」

「不接。」韩泥只答两字。

药山半腰风硬，他听半刻——风里有仇，仇在远，手在近。近的是筐、是渣、是席下坛，远的才是钱戾衡那只眼。

编筐棚借半刻，藤刺扎掌，刺浅。刺浅些，沿才净；沿净了，坛才藏得住。

铁无言抱臂，像清瘴那日站位：「台见，别飘。」

「不飘。」韩泥应。

「飘了呢？约废？」

「约不废。」韩泥独眼平，「废的是手。手废了，人才废。」

回杂役屋，他按坛。坛腹温一线，像应「风」字。

丑时练鼻下静。粟壳香淡，香一字不落。

他低声对坛：「钱戾衡，记着。台见。」

坛不应字，只温。温像应「约」——约在，手就稳。

邻床悄问：「你怕不怕？」

「怕。」韩泥说，「怕也守。守完，才有路。」

「怕还上预选？」少年舌头打结，「我怕就躲。」

「躲得过今日，躲不过约。」韩泥道，「约到了，躲也是滚。」

坊尾条角字丑，真在「预选」。真不要紧，真在台，不在坊。人在坊尾骂，他当听风——风紧，才试忍不忍。

忍过赖福五年，忍过并肩那半步，忍过还刘婆那一夜。还了，才配接这仇风。

仇风接，不接骂。骂像滚；滚了，约白接。约说炼气十三，他已走到十五；约还说站台前——站在掌上，不在嘴上。

赖福廊下过，啐：「钱家盯上你，你还挺胸？挺胸给谁看？」

「给自己看。」韩泥说，「胸挺着，手才不抖。」

「哼，抖不抖台上看。」赖福走了，靴底故意响。

韩泥不追声。追声像接骂。他把玉牌按实，凉意进肉，肉里烫约。

沈枯芽远远探头，又缩回去，只敢丢一句：「叶姐说……别跟人硬撞。」

「不撞。」韩泥回，「撞在台，不在巷。」

芽儿走了。叶汤还在前，前不动怒，怒了汤也凉。

更鼓沉，他望帖，帖角掀半分：「风声紧。下一回，积势收。」

药峰上照旧分拣，指稳，鼻下静一息。息里只辛淡，没有辱。辱在条上，条不进鼻子。

白织月远影再过一次，弟子跟屁虫似的问：「白师姐，那丑骨真敢接钱戾衡？」

「敢不敢，他自己写在脸上了。」白织月道，「我们看台，不看脸。」

韩泥把这话当针，扎进心里：看台，不看脸。脸丑，台正就成。

帖在怀，怀沉。沉的是序，不是惧。惧也有，惧着也要把风声听真——听真了，积势册才写得齐。

午后渣场，有人故意把那张损条拓了两份，往药廊柱上贴。贴完还拍掌：「让丑骨自己揭！」

韩泥路过，看了一眼，不揭。揭像恼；恼像接骂。

管事弟子沉着脸撕条：「谁贴的？贴私条，罚扫坊尾！」

人群一散而空。韩泥把扫把接过来：「我扫。扫了干净，省得落土进鼻。」

「你……你自己扫诬你的条？」管事愕然。

「条是纸。」韩泥说，「纸脏，土扫掉就行。骨还在，台还在。」

铁无言在廊角差点骂出声，最后只挤出一句：「你真能忍。」

「忍不是怂。」韩泥边扫边道，「怂是滚。滚了，三年约变笑话。」

「笑话可不是我要听的。」铁无言抱紧臂，「台前你要还像今日这样站，我才认你这兄弟。」

「认不认随意。」韩泥把土收进簸箕，「我认约。约在，人就在。」

傍晚刘婆又来塞半块糗饼：「扫街饿得快。吃。吃完别跟人瞪眼，瞪眼惹祸。」

「不瞪。」韩泥接饼，「眼只一只，留给验渣。」

「油嘴！」刘婆骂完却笑，「笑完了去歇。风这么紧，夜里更该守肚子。」

夜里棚里低语成片：「钱家的人今儿在驿亭喝茶。」「喝茶？打听谁？」「还能有谁，那挂歪牌的。」

韩泥听着，像听雨。雨落瓦，瓦不裂；裂的是沉不住气的人。

他按坛沿，沿净，低问：「怕吗？」

坛腹温一线，像回「在」。在就够了——在席下，在息里，在三年约的凉意里。

秦霜木鱼忽远忽近：「风紧时，心松一分。松了，邪不好落脚。」

韩泥对暗处拱手：「松得住。松不住的是仇，仇我留台上。」

夜巡小役端着冷茶路过棚门，见他醒着，纳罕：「韩叔还不睡？风这么紧，睡了耳根清静。」

「清静留给台上。」韩泥接茶润喉，「台上不清静，耳根清静也没用。」

「您……真打算跟钱家硬碰？」

「硬碰不在巷口。」韩泥把茶还他，「巷口碰是吵。台上碰是约。约到了，躲也丢脸。」

小役咂嘴走了。棚里有人竖耳朵假装打鼾，鼾声假得刺耳。韩泥不揭穿——揭穿像多生一条仇。

他盘膝再静一刻。一刻里只闻粟壳，不闻条上辱。辱是纸，纸怕火；火他留着炼散，不留着烧巷。

铁无言临天亮又在窗外站定：「条撕了，土扫了，人呢？」

「人在。」韩泥应，「在就不滚。」

「钱家茶座散了没有？」

「散没散不关我开门。」韩泥摸玉牌，「关我手稳不稳。手稳，茶凉了也烫不到指头。」

「行。」铁无言丢下两个字就走，「积势去。」

药峰派活鼓响起时，韩泥已经蹲在分拣筐边。有人故意把损条碎片踢到他靴尖：「哟，还敢来上工？」

「工是工。」韩泥拨开碎片，「仇是仇。分列，不混。」

「分列？」那人讥笑，「钱家听见你文绉绉，怕要笑掉牙。」

「笑让他笑。」韩泥拈起一片潮叶，「笑进炉里，炉也不认。炉认药性，不认牙。」

管事路过喝止：「少贫！验叶！」

场子一静。韩泥鼻下一息，报：「此包可用，偏干，宜配润。」

报完，他把靴尖碎片扫进垃圾筐。扫的是纸，不是脊梁。

午歇片刻，他靠廊柱啃糗饼。有人路过甩话：「丑骨还吃得下？我要是被条点名，三日吃不下饭。」

「饭是饭。」韩泥嚼完才道，「条是条。饭养手，条养仇。两样都扔了，人才真垮。」

甩话的人骂一声远了。廊风过去，糗饼渣掉在他膝上，他弹掉，起身回活。

他望坛，最后一句落得哑：「影在钱戾衡。衡在台。台在前。守完，才有路。」

天边白一线。白里像有人已经在抄下一张更损的条。条还没贴到门上，鼓点却先到了胸口——下一册要合上了，合上才知道，手里这点势到底够不够送到老耿跟前。
