# 第一百零七章 钩108

静半日。

兽散，散像退潮，潮退，界外仍腥。腥不要紧，腥是钩——钩在余毒，余毒在栏，栏在，角才不能散。

韩泥被管事点名：「丙九韩泥，随铁无言清余瘴。」

点名，像钩；钩108，落在清。

清，不是扫，是嗅。嗅余毒，毒在角草，草黑，黑里甜。甜，他熟，熟于99风声，熟于101备。

铁无言臂仍绷，绷粗，声低：「清完，我请你吃一袋麸皮——不，请不动。手在就行。」

韩泥应声：「在。」

在，就是请。请，不在嘴，在并肩。

清角时，他鼻下动，动出三处漏——漏在障根，根补，补不要紧；补完，仍剩一处，一处深，深像毒窝。

窝，要丸，丸要炼。炼在108，不在107；107只钩——钩出深窝，钩出经页「清瘴」二字将实。

实，他摸怀经页，页黄，字深一线。深不要紧，深是序；序齐，清瘴丸才肯落怀。

赖福来瞧，瞧像不信：「还能嗅？」

「嗅。」韩泥说，「嗅完，手稳。」

赖福没再尖，尖不要紧；尖过了，像也怕余毒。

白织月立远，声低：「余毒进内墙，谁都跑不了。」

韩泥只答：「不跑。」

不跑，是钩的另一头——头在铁无言，尾在清瘴丸。

叶青禾在坡下，门缝灯又亮。亮，他不在近；在近，像连累。连累不要紧，灯在，脉里培元仍温——温，像恩还着。

沈枯芽悄递：「叶姐说，别逞能。」

韩泥点头。逞能不在路；路在清，清在108。

夜，管事弟子贴条：「余瘴清尽前，丙九不准撤。」

不准撤，就是钩。钩住，韩泥东角又立，立像续后的终前。

终前，他按坛，低声道：「钩在了。下一程，炼清瘴丸。」

坛腹温，贴掌，像应「钩」字。钩，不是戏，是序。

丑时，静半刻。香落齐，掌纹烫——烫像五层将满，满而不破，破在下一程。

邻床问：「真要再炼？」

「炼。」韩泥说，「炼了，栏才活。」

更鼓尽，深窝仍甜，甜像等丸。等不要紧，等的是108。

他独眼平，低声道：「钩108。手在，铁无言在。」

在，就不滚。不滚，清瘴路才接得上。

兽散，界外仍腥。腥是钩——钩在余毒。

管事点名：「丙九韩泥，随铁无言清余瘴。」

清角时，鼻下动，动出三处漏。仍剩一处深窝，深像毒窝。

炼在108，不在107。

叶青禾门缝灯又亮。亮，他不在近。

管事贴条：「余瘴清尽前，丙九不准撤。」

深窝甜，甜像等丸。108一事，清瘴；清瘴一事，并肩。并肩一事，铁无言。铁无言说「手在就行」，就行，是契的芽。

条贴「不准撤」，撤不得，撤了，余毒进内墙。进内墙，白织月也怕；怕不要紧，角仍要人。人要，韩泥在；在，就不滚。不滚，108才接得上。

静半日，界外仍腥。腥不要紧，腥是钩——钩在余毒。

秦霜木鱼轻敲：「钩在，心别慌。慌，手飘。」

韩泥不答，答在心里——心里不慌，只记。记深窝，记清瘴，记并肩在前头。

序齐，清瘴丸才肯落怀。107只钩——钩出深窝，钩出清瘴二字将实。

连累不要紧，灯在，脉里培元仍温。温，是95；钩，是107。两笔不混，混了，七笔乱。」韩泥不接话，接在记里。逞能不在钩，在清——清完，108才炼。炼，要铁无言在，要障挡目，要手稳。

铁无言声低：「深窝在前。跟我进，进三步。」韩泥应声：「三步。」三步，像挪103填障——挪过，就不怕。怕不要紧，怕也进。进，才清瘴。」清，不是扫，是嗅。甜，是99记的；记牢了，108才正。

更鼓沉，韩泥按木牌，牌凉贴胸。终前，他按坛，低声道：「钩108。」不滚，清瘴路才接得上。坛腹温，像应「钩」字。

钩108，钩在余瘴，瘴在角草，草黑，黑里甜。甜是99记的，101备的，102撒的——撒尽，103压过，104至，105潮中，106续，107才钩深窝。

韩泥随铁无言清余瘴，清不是扫，是嗅。嗅三处漏，漏补两处，剩一处深，深像毒窝。窝，108才炼丸；107只钩，钩出经页「清瘴」二字将实。将实，不在嘴，在掌。

叶青禾坡下，门缝灯又亮。连累不要紧，灯在，脉里培元仍温——温95，钩107，两笔不混。混，七笔乱；乱，路断。」逞能不在钩，在清——清完，108并肩才真。并肩，要铁无言在，要障挡目，要手稳。手稳，六层才开；六层开，109才稳炉，110才散。进三步。」三步，像103填障——挪过，就不怕。怕也进，进才清。

清，管事记「清余瘴有功」——功在远，功在108全成。钩在了，108在前。

钩108，钩在余瘴深窝。窝在角草，草黑，黑里甜——甜是99记的，102撒的，103压的。压过，104至，105潮中，106续，107才钩实。

清角嗅三处漏，补两处，剩一处深。深窝，108才炼丸；107只钩，钩出经页「清瘴」将实。将实，要铁无言在，要障挡目，要手稳。手稳，六层才开。

叶青禾门缝灯又亮，亮，他不在近。近像连累，连累不要紧，灯在，脉里培元仍温——温95，钩107，不混。

更鼓沉，按木牌，钩108。手在，铁无言在，坛腹温像应「钩」字。钩在了，108在前——在前，清瘴并肩，六层开，109稳炉，110培元散。散在远，远不要紧，108只清瘴。

兽散，界外仍腥，腥是钩——钩余毒，余毒在栏，栏在，角才不能散。散不要紧，107只钩——钩深窝，钩经页「清瘴」将实。将实在108，不在107；107只清，清不是扫，是嗅。甜是99记的，记牢了，108才正。正，要铁无言在，要障挡目，要手稳。散在远，107只钩。

钩了，叶青禾门缝灯又亮，亮，他不在近；近像连累，连累不要紧，灯在，脉里培元仍温——温95，钩107，不混。在前，清瘴并肩，六层开，就不滚。

清余瘴第二日，铁无言：「深窝在前。」韩泥应声「三步」——三步，像103填障。填障稳，108才炼丸。炼丸要障挡目，要铁无言在，要手稳。手稳，六层开；六层开，109稳炉，110散。钩了，108在前。

叶青禾门缝灯又亮，亮，他不在近；近像连累，连累不要紧，灯在，脉里培元仍温——温95，钩107，不混。沈枯芽悄递：「别逞能。进三步。管事点名清余瘴，清不是扫，是嗅。嗅牢了，108才正。钩在了，108在前——在前，清瘴并肩，六层开，就不滚。

清余瘴第三日，深窝在前，前像108将炼之炉。炉在障后，后在掌；掌稳，丸才正。正，不在名，在鼻——鼻下嗅诀，沉腐浮落齐。落齐，钩才实；实了，108在前。在前，铁无言在，障挡目，手在，就不滚。滚在远，钩在107，钩住了，就不白钩。
