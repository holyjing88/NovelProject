# 第二十三章 审前

先还叶次日，里正敲锣。锣声哑，哑得像旱。哑也得听——听清了，是村议：「明日巳时，祠堂公审叶家牵扯，全村见证，缺席连坐。」

「连坐」二字又出，出得像枷。枷不在韩泥颈，在敢对他好的人的颈上。韩泥在门内，指停藤上。停一息，又绕。绕得像绕命。命要过公审这一关。关不过，叶家关铺；关过了，谣也还在。还在，也得走。走路不辩嘴，辩心。祠堂前，赵绣春已摆阵势，妇人围半圈，像看戏台。戏台不用搭，泥岗村从来不缺戏，缺的是真话。「明日，叶青禾自己说清！」她嗓尖，尖得稳，「送汤送姜补袄，是不是勾引绝户？是不是引血祠旱？是不是害全村榜末？」

周嫂帮腔，棒槌在怀，像随时能砸：「里正说了，不说清，迁铺！迁了铺，谁治病？治不了病，就等死——死也别赖我们！」

众人附和，声浪一层压一层。压里，叶青禾未出，药铺门闭，闭得像一块青布，布后秤声稳。稳，是最硬的辩。韩泥担筐路过，不近祠堂。近，成靶。他绕远半步，半步就够——够看，够听，够记。记不是怂，是候还。祠堂侧，里正与赵家低语，语里「坐实」二字清楚。清楚不要紧，要紧的是叶家秤还在——还在，公审就有理。里正抬眼，瞥韩泥背影，没叫。叫，成戏。戏明日才演，今日只敲锣。马春花瞥他，啐：「晦气翁，明日你也去！去听听寡妇怎么辩！辩不清，连你一起滚村西！」

韩泥「嗯」一声：「去。听真话。」

马春花一愣，想再尖，赵绣春拽她袖，低声却亮：「让他去！去了，坐实一对！坐实了，公审好收尾！」

韩泥不辩。辩，成嘴。嘴不如心。心里叶汤已深，深得够扛明日。午后，沈枯芽在巷口徘徊，手里攥小纸包，纸包抖，抖得像叶。叶干，仍要送药，送药像还粥。韩泥停步，筐放脚边，不挡路。「韩翁……」芽儿声怯，怯得像那日狼口，「我娘让送药叶……不敢进门……赖管事说，签字连坐……」

「敢。」韩泥接包，轻，「药我收。名你别签。签字会连坐。连坐是扒皮的牙，别让它咬你。」

芽儿抬眼，眼红，红得像旱天的沙：「明日……叶丫头要公审……我娘说，要是关铺，我们没处抓药……」

「公审审谣。」韩泥独眼平，「不审人心。人心若凉了，抓药抓谁？」

就三句。芽儿点头，跑。跑得像一粒米，落进沈家锅，锅小，米真。真米能熬粥，熬粥也是活。这是暖，暖在压前，暖得像给明日留一口气。气在，鼓才敲得响。沈枯芽跑远，韩泥望一眼，望得像记——记芽儿还肯送药，记沈家还肯捎话，记公审前不是全村的嘴都毒。毒有，暖也有，暖就是释。傍晚，叶家灯亮。亮得比往日久。久，像在缝，像在称重，像在把明日要说的话，一句句过秤。秤准，话就稳；话稳，铺就稳。韩泥站巷尾，望灯，不往前。往前，赵绣春要笑；不往前，叶青禾也要站明日台上。台高，高不过一句「趁热」。他回屋，心口叶汤深，深得像血里泡过。他不增笔，只按，按实，像按叶家脊梁。脊梁不弯，谣就压不垮。「明日公审。」他低声道，「我坐着，不害她。她说话，我记着。记着，以后还。」

坛角微凉。坛腹微温。温里，他候巳时，候赵绣春哑口的那刻——那刻未到，但绳已绷紧。绷紧到今夜，像鼓，鼓面是全村嘴，鼓槌是旱与谣。明日，鼓要响。响完，要么叶家秤还在，要么全村嘴更毒。韩泥合袄，塞怀，独眼闭一息。闭不是睡，是蓄。蓄到明日，蓄一句：汤是热的，我记着。里正又敲一遍锣，锣尾拖长，像绳结：「明日巳时，祠堂见。赵家作证，周家作证，叶家自辩。韩泥——」

人群「嗡」一声，嗡向他。里正顿半息，像也怕「晦气」二字：「韩泥也到。到，不许闹。闹，连坐。」

韩泥在屋里应不了，应也在心里：到。不闹。不害她。害她，恩就白记了。应完，他摸心口叶汤，汤深，深得像能扛明日一整场潮。潮是压，深是释——释不在辩，释在坐稳，释在记着以后还。赵绣春笑声穿过墙，笑里带胜：「明日，看绝户怎么坐！」

周嫂接：「看寡妇怎么辩！」

声远去，远成钩。钩尖是「明日公审」，钩尾是叶家灯——灯还亮，亮得像没认输。韩泥摸袄肘针脚，针脚密，密得像叶青禾明日要说的话。话不必多，多不如稳。稳，他信她。信，也是还的一种预备。预备不是怕，是候——候叶开口，候赵哑口，候叶汤透光。候是压里的定。定了，明日才不慌——不慌，才坐得住末席，才不害她名。里正锣声远去，远成旱风里一线哑。哑声里，叶家灯仍亮。韩泥望灯，望一息，像望明日台上那一句话——不必多，一句「趁热」就够全村哑半息。够半息，就够还的起头。里正锣声又近，近得像催。韩泥合袄，不睡，只编筐。筐稳，手稳，明日才坐得住。张麻婆路过，瞥他窗灯，敲两下门：「还不睡？」

韩泥：「编筐。手稳，心稳。」

张麻婆：「心稳就好。叶家那秤，你也稳着。」

韩泥：「不害她。」

张麻婆：「害她，你那些还字全白写。」

韩泥：「不写白。记着。」

张麻婆走远，远里嘟囔：「老头倔，倔好。倔才能还。」

韩泥应声，藤条又绕膝。绕出一道细棱，棱不尖，尖的是明日——明日巳时，公审见真，真在秤，也在叶汤。他摸袄肘针脚，针脚密，密得像叶青禾明日要说的理——理在针脚里，也在汤里，也在「不害名」三个字里。理在，就不怕末席。末席坐着，也能听真。

明日公审，村静如绷紧绳。他不备词，备心——心实，叶丫头一句证白就立得住；立得住，汤就还热，锤落，才是明爽。爽是后话，话先在「我记着」里。丑时前，他仍编筐，指稳，筐紧——紧，像把还字编进藤里，编进命里，编进明日末席那一坐。

赵绣春门内又摔碗，摔得脆，脆里带虚——虚在药铺门仍开，开，就还有病人，还有竖立的理。韩泥不近祠堂，不近叶阶——近是害，远是护。护到明日巳时，护到叶丫头把秤摆稳。沈枯芽隔巷探头，探头又缩，像怕连坐。韩泥隔巷应，声哑：「别送药了。公审前，你娘要活，叶姑姑要稳——稳了，药才抓得着。」芽儿点头，跑。跑远，像把暖又送一遍；送一遍，够撑到锤落。

坛角微温，像瓮替他守着「我记着」——守着，就不怕谣一夜吞村。里正锣声又远，远成钩，钩尖是「明日公审」，钩尾是叶家灯——灯还亮，亮得像没认输。
