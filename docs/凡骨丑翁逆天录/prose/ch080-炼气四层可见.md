# 第八十章 炼气四层可见

冬深续后第七夜，丑时。

韩泥未睡实。睡实了，内视藏不住；藏得住，才看得见远境。轮换条在柱上挂了好多日，手茧又厚了一层，厚得像催他往里看一眼：看清四层在不在前头，看清了，废渣袋才知道往哪背。

他盘坐床沿，席严，坛不露。油灯豆，豆光晃一下，又稳。稳，像踹台按实——掌在，名在，路在。

掌贴沿，沿温一线。温进掌纹，二层气转一周、两周；转到第三周，鼻下忽静到极。极静里，眼前不是黑，是灰——灰像药山深雾，雾里有一点碑影。碑影钝，钝像测灵石：石秤命，碑秤规矩。

气再转一周，灰散开半寸。散开处，四层境像远灯，灯不亮，可见。

可见，不是到，是见。见到，手才知往哪走；知往哪走，暗修才不盲。韩泥屏息，独眼不眨。眨了，灯就灭。灭不得，灭了，四层白见。他低声道：「四层可见。未到，先见。」

话不落舌上，落在掌里。舌响，香飘；香飘，赖福来掀席。坛底极轻一颤，像臾墟残念应一息：「……见境，不叫境。叫境，煞骨门嗅得到。」

韩泥不答。答在心里：记一笔——四层可见，见在暗，不在公示。公示写仍末，掌见四层；见给自己，不见给别人。气稳在二层，稳里那盏远灯不熄。不熄像指路；指路不要紧，脚仍停——停不是怂，是规。规在，坛在；坛在，根在。

邻床翻身，眯眼半息：「你……身上咋亮？」

「油灯豆。」韩泥声平，「看错了。睡。」

小杂役哼一声，又睡实。睡实了，内视才藏得住。他收功，收得像盖席。盖严，沿净，坛不露。不露，像腌菜坛，不像邪。更鼓将尽，天边白一线。他廊下立半刻，鼻下仍静。静里，四层那盏远灯仍在——仍在，像钩，钩三层暗修，钩四层将到。

他把「见」字按进胸口。按完不像高兴，像多了笔债：见到了，就得往前凑；凑，还得藏好。藏不好，三年约还没到期，人先被人翻出来。

廊下风穿过木格，格缝里的霜白像细盐。他伸手搓一下，盐融了，掌心更醒。醒着好。醒着才记得：见境不是登天，是看见坡还长。坡长，就得一步一步拿废渣垫脚。垫脚前，先别把灯喊亮给全棚听。

铁无言脚步远，停半息，声更低：「见境别叫。叫了，像求。」

韩泥心里记着：见是路，路通向废渣收集，通向丹路开。卯时兽栏腥重。腥不要紧，远灯在掌，铲下更稳。管事弟子巡栏，记一笔：「丙九韩泥，冬深仍清细。」刘婆端粥，仍多半勺：「昨夜又亮？」

「豆爆。」韩泥说，「沿净。」

「沿净怕什么。」刘婆哼，「饭稳，恩不断。」

韩泥点头。恩在粥里，也在远灯里——灯不亮，可见；可见，记重。午后药渣场，灰起，他分拣，鼻下有根，手快半分。快也要稳——稳了，四层那盏远灯才不晃。

铁无言走近，瞥分堆，又瞥他掌：「见境了？」

「见了。未到。」韩泥应声。

「见好。到，在远。远不要紧，渣里活，活长了，才到得了。」

到得了，他不声张。声张像求境；不求，只记：记四层可见，记三层在远，记二层在掌。白织月远廊过，袖风冷：「杂役也守丑时？守出四层？」

韩泥不抬头：「守沿。沿净，不叫。」

白织月唇线绷一息，走了。少一句嘲，小胜。秦霜木鱼轻敲：「见境心别贪。贪，邪。」

他没接话。心里账本清晰：见境是路，路通向废渣路，通向温听雨初识。他按席角，低声对坛：「四层见了。未到，先记。不叫。」

坛不应字，只温。温像把远灯那一点灰，摁进掌纹那路里。邻床悄问：「你身上有时热，怕不怕？」

「怕。」韩泥说，「怕也不叫。不叫，沿净，才藏得住。」

少年不懂「藏境」，却懂「不叫」，闭嘴睡了。丑时前，他再练半刻。二层气转，转时远灯仍在——仍在，像四层在等他。等不要紧，脚仍停；停稳了，三层才开得了。掌纹那路比昨夜厚半分，厚在暗，不在册。册仍写仍末，掌见四层。

他独眼平，手不抖：「四层可见。三层在远。废渣收集在明日。」

刃不亮，先记。记完了，才配攒渣，才配废渣路那一袋。玉牌在怀，凉进骨。凉不要紧，掌里远灯不熄——不熄在暗，不在众目。众目看灰线，掌里见四层。

内视后第三日，药渣场风硬。他分拣，指稳，忽觉掌纹影里第四阶边多一丝辛——辛像深灰焦回，也像废渣将收。辛不要紧，记牢；记牢了，明日那袋才有着落。管事验五堆，五堆全对，记「春验仍准」。铁无言声低：「影见了，别求破。稳二层，破三层四层在后。」

「不求。影记着。」韩泥应声。

铁无言走后，他在灰堆边蹲着绑筐绳。绳结打了又松，松了又打——不是手笨，是逼自己别飘。飘的人见了灯就想扑。扑了，灯灭，人也裸。他宁可绳结丑，把「到」字往后挪。挪到渣够沉、手够稳、嘲够记的时候，再谈破。

刘婆又端粥：「见着了别贪。贪了，舌就响；舌响，香飘。」

「不贪。」韩泥点头，「粥先喝尽，手才稳。」

温听雨外堂过，裙素，没停。没停不要紧；要紧是手稳，境不露。露了像求；不求，只记四层可见这一钩。老耿挑水过，声哑：「见境别拜灯。拜，路断。」

「不跪灯。」韩泥只答，「灯在掌里，不在嘴上。」

黄昏回棚，按席角，席严，坛不露。坛腹温一线，像应「见」字余韵。废渣将收，温氏将识——他都不叫。丑时前内视再半息，远灯仍在，仍在就不慌。慌像少年；他是记心翁，翁慢，恩不凉。

同批少年悄问：「你身上有时热，邪不邪？」

「沿净，不邪。」韩泥说，「邪在舌，不在坛。」

赖福廊外停半息，冷声甩过来：「第几天了，还守丑时？守出鬼了不成？」

「守沿。」韩泥应一声，「沿净。」

脚步远了。他才松半寸肩。松完又紧——紧着，才接得住明日废渣袋。袋还不在手里，影先在掌纹里。影在，路就不盲。

夜禁前，他去废渣巷口立了半息，不进巷。进巷是收，今日只记影——影在掌，不在众目。众目还盯着灰线三尺，他犯不着把远灯递给人看。巷风硬，硬得像提醒：见了就得攒渣；攒渣，才能把「可见」变成「能走」。

棚里少年又嘀咕：「你老站巷口干啥？邪不邪？」

「验风向。」韩泥随口扯一句，「风正，明天背袋不呛。」

少年信了，蒙头睡。他才回席角，再内视半息。远灯仍在，比昨夜近半分——近不是破境，是眼熟了。眼熟了，脚才不慌。慌的人爱叫；叫的人短命。他不叫。

枕边木牌硌着胸。硌一下，醒一下。醒一下，就把「明日废渣」四字在心里重钉一遍。钉牢了，丑时灯火才不算白费。

更鼓沉，独眼平，手不抖。柱上条角白一线，冬深将尽，春风声远。远不要紧，掌里灯不熄——灯在，就不滚。活长了，才还得了叶汤，才接得住门缝灯。

灯不叫人看见；看不见，才配把明日废渣袋一袋一袋攒实。攒实了，路才宽；路宽了，四层那盏远灯才不只是影。影要变成路，先从明天那一袋废渣起。袋沉了，灯才肯近；灯近了，也不许叫出声。不叫，才熬得到还叶汤的那一日。那一日还远，袋先实。实了再谈近。
