# 第八十七章 暗修三层

小成续后第三夜，丑时。韩泥未睡实。睡实了，三层才冲得开；冲得开，才藏得住。他盘坐床沿，席严，坛不露。油灯豆，豆光稳，稳像渣场五堆那一点齐——齐在暗，不在众目。掌贴沿，沿温一线。温进掌纹，二层气转，转一周，转两周，转第三周时，鼻下芽根一紧。紧不要紧，紧才真。真在冲关，真在凡骨壳，真在腌菜坛沿。

三层破关，关隘裂一线，裂声闷，闷在骨里。开不像雷，像瓮唇吐白气。白气冷，冷里带药香，药香里带焦回——焦回像废渣路，像各归位。

温听雨外堂过，远远一瞥，瞥韩泥肩稳，瞥完，袖风一转，走了。走了，也像验；验肩稳，像验手稳。韩泥不追，不谢，只记。

十堆再验，十堆全对。全对，管事记「辨性准」。铁无言走近：「三层配鼻，鼻更要清。清了，才接得住四层。」四层在88，丑时才破。

三层破关，瓮唇吐白气。白气冷，冷里带药香。香在暗，不在公示；公示写仍末，掌写三层。

气沉腹，沉到掌纹。掌纹亮半分，亮不像光，像认——认韩氏守瓮血，认瓮穴灵穴，认四层那盏远灯。远灯不熄，像指路。指不要紧，脚仍停稳；稳了，关隘才裂。关隘薄，薄像席布。气冲半寸，退半寸。退不要紧，再冲。第四周，气聚掌纹，掌烫，烫得踏实。踏实了，关隘裂一线，裂声闷，闷在骨里，不在屋里。三层开了。

开不像雷，像瓮唇吐白气——白气冷，冷里带药香。药香淡，淡不要紧；淡也是境。韩泥屏息，独眼不眨。眨了，境就散。散不得，散了三层白开。气稳在三层，稳像粪铲落栏——落得要实，实了，才不飘。飘了，管事记「抖」。他收功，收得像盖席。盖严，沿净，坛不露。不露，像腌菜坛，不像邪。

掌纹那路比昨夜厚半分，厚不要紧，厚在暗，不在册。册仍写仍末，掌写三层——写给自己，不写给别人。邻床翻身，眯眼半息：「你……身上咋更热？」

「编筐扎的。」韩泥声平，「看错了。睡。」

小杂役哼一声，又睡实。睡实了，三层才藏得住。更鼓将尽，天边白一线。白里，他廊下练半刻鼻下静，静到只闻辛淡，不闻辱。三层在掌，二层在忆里，远灯仍见——见四层，未到，不要紧；见着了，脚就不盲。铁无言脚步远，停半息，声更低：「三层别叫。叫了，像求境。」

韩泥只答：「油灯豆。豆爆，爆完即灭。」

赖福盯半息，终是没掀席。掀不得，沿净，坛不露。不露，晦气遮瓮，遮住了，三层才稳。卯时，兽栏仍腥。腥不要紧，三层在掌，铲下更稳。稳，像踹台那一掌——掌在，境在，路在。管事弟子巡栏，记一笔：「丙九韩泥，清细。」

刘婆端粥，仍多半勺：「昨夜又热？」

「豆爆。」韩泥说，「沿净。」

「沿净怕什么。」刘婆哼，「饭稳，恩不断。」

韩泥点头。恩在粥里，也在三层境里——境轻，记重；记重了，才接得住四层。午后，药渣场。灰起，他分拣，鼻下小成，手快而稳。稳里三层气走指梢，指梢温半分，温像辨渣更细。铁无言走近半步：「三层别漏。漏了，渣场不收。」

韩泥应声：「不漏。沉腐浮，记着。」

铁无言又指暗袋方向：「袋别示。示了，赖福记私藏。」

韩泥点头轻，记重——记三层境配嗅诀小成，配废渣路，配丹路将开。白织月远廊过，袖风冷：「杂役也境升？升出邪法？」

韩泥不抬头：「守沿。沿净，不叫。」

白织月唇线绷一息，走了。少一句嘲，小胜。秦霜木鱼轻敲：「境升心别贪。贪，邪。」

韩泥不答，答在心里——心里七笔分列。三层是境，境是路；路通向四层，通向丹路开。他低声对坛：「三层开了。不叫。坛在，人在。」

坛不应字，只温。温，像爹的手又搭上来。爹说过：「坛别扔。」扔不得，扔了就无根；无根，三层白开。更鼓沉，沉里，他合眼不睡实，掌纹那路仍转半周。半周不要紧，转多了，邻床就醒；醒不得，醒不得。他独眼平，手不抖：「三层在掌。四层在远灯。丹路将开。」

韩泥只答：「不拜。记着。」

夜禁脚步远，风里像煞骨门三字一闪——闪不要紧，沿净，壳才像腌菜坛。像腌菜坛，魔教才嗅不准。他低声道：「三层开了。远灯仍见。四层在明日丑时。」

四层在明日丑时；远不要紧，灯在掌前。掌前，才冲得开，才开丹路。赖福讽远：「末等也热？热出四层？」

韩泥不接。舌要留来守境，留来守坛。柱上条角白一线，冬深，风硬。风硬不要紧，三层在掌——掌在，就不滚。三层破后第二日，药渣场春验十堆。韩泥分拣，鼻下清，十堆仍全对。管事记「春验仍准」。准，是小胜。铁无言声低：「三层配鼻，四层在丑时。破四层，仍不叫。」

韩泥应声：「不叫。境在暗。」

温听雨外堂过，瞥肩稳，瞥完，声清半分：「肩比洒扫那日更稳。」

更稳，韩泥不谢。谢轻，记在心里。赖福讽远：「末等三层？三层也滚。」

韩泥不接。舌要留来鼻，留来守坛。刘婆端粥，仍多半勺：「三层别飘。」

「不飘。」韩泥说，「飘了，手就滚。」

午后废渣巷口，他立半息，记焦回。记，不为炼，为槽将开。秦霜木鱼一声：「境升别骄。骄了，香飘。」

韩泥不答，答在心里——心里不骄，只记三层，记四层影更近。黄昏，角柜锁实，掌纹三阶亮在忆里，亮在暗，不在册。邻床问：「三层厉害？」

「厉害在掌。」韩泥说，「掌不示人，示人，谣来。」

丑时前，内视半息，第四阶影近半寸。近，是钩；钩四层将破。他低声对坛：「三层在掌。四层在丑时。不叫。温，像三阶亮摁进影里。更鼓沉，独眼平：「三层稳了。丹路将开，五层在远。」

更鼓沉，沉里，韩泥按木牌，牌凉，掌温。掌温在87，不在舌上。舌响，香飘；香飘，手就滚。滚了，恩断。他不滚，手在，名在，路在。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。

滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。

滚了，恩断。

廊下风硬，他把木牌系紧一线，系得像把仍末摁进掌心。

同批少年远避，避像避晦气，晦气遮瓮，遮住了才稳。

编筐角藤刺仍在，他偏手，刺浅些，沿才净，坛才藏得住。

管事夜巡验棚，灯照席角，沿净，像腌菜坛，不像邪。

老耿挑水过，声哑，境别拜邪，拜了路断，他只记着。

秦霜木鱼远，一声，心别邪，邪了鼻下香飘，香飘手就滚。

柱上条角白一线，三年约在怀，愧字在怀，手仍不抖。

白织月远廊过，袖风冷，没停，胜在鼻，不在嘴。

铁无言脚步远，停半息，声低，别漏香，别叫境，只记。

三层破，破在丑时，破在席下。席严，坛不露；不露，赖福掀不得。掀不得，四层影才等得实，丹路才开得正。

更鼓沉，沉里，韩泥按木牌，牌凉贴胸。87章手在，名在册，坛沿净。舌不留，鼻下静。恩辱分列，不混。活长了，才还得了叶汤，才接得住门缝灯。

管事夜巡验棚，灯照席角，沿净，记「手稳」——手稳，是小胜；胜在活，不在嘴。
