# 第六十二章 席盖更严

瓮醒前夜，风硬。硬像最后一道冬关。关不过，缘不满。缘满，才误触那一滴。韩泥清栏归来，掌茧新裂，裂口渗血，血珠饱满。他甩手，血甩破席，席吸尽，沿仍净。他不主动滴血，不揭布诱沿。只用活计扎出的血喂席，席挡沿，沿等误触那一瞬。邻床笑：「坛角盖那么严，藏宝？」韩泥不辩，只把席盖又加一层。层厚，厚像秘密。秘密是坛，坛是命，命跟了五十八岁。管事查棚，灯晃坛角：「杂物收起。」韩泥「嗯」一声，席盖不动。动像心虚。不虚，是守。管事盯半息，走了。只见破席，不见仙缘。---

夜，符怀暖一丝，护骨寒，手不僵。僵，明日藤刺偏。不偏，缘才正。他低声对坛：「明日或后日，藤刺还会扎。扎了，血还会来。来席，不来沿。沿上，等你认的血。」坛腹温一线，贴掌，贴到心口也暖半分。暖，不像火，像认亲前的试探。更鼓三响，响沉。丑时将至，像丑时坛温那一线又要走一圈。走一圈，走到编筐扎掌，走到血误触席，走到席渗沿，走到沿颤，走到瓮醒。他合眼，不睡实。他是记心翁，恩仇在心里，符在怀，坛在角，席盖严。他低声道：「席盖严了。严了，才接得住那一误。误了，根在。根在，路长。」坛不应字，只温，温得像答应，像等。明日，或丑时，瓮醒。刘婆端姜汤，姜薄，热：「前夜别空肚。空肚，手飘。」「不空。」韩泥喝尽，「嗯」一声。姜热，像叶汤远影。赖福讽：「盖席盖出仙缘？」管事声冷：「再嗓，克半月麸皮。」赖福哑了。哑，韩泥加席层的手却更稳。铁无言门外脚步停半息，声低：「沿净？」「净。」韩泥说，「血在席，不在沿。」「不主动滴？」「不滴。」韩泥说，「滴，要等活计误那一甩。」铁无言「嗯」一声，脚步远。远，像门缝又开一线。韩泥又加一层破布，布旧，旧能吸。吸血，挡沿，沿等韩氏守瓮血那一误。邻床问：「盖那么严，热不热？」「热不要紧。」韩泥说，「沿净要紧。净了，才接得住瓮醒。」邻床不懂，翻身。翻身，鼾声起。他是记心翁，恩仇都记在心里。席盖严，严像守了五十八年的命。命跟坛，坛跟缘。外头风息，风息了，席却更严。严，是最后一夜守。守住了，才配接瓮醒那一误。他按心口，低声道：「叶丫头汤，热着还。还，要等瓮醒后，路才长。」坛腹温一线加剧，加剧，像明日就在沿边等那一滴守瓮血。他低声对坛：「席盖严了。守得严，韩家不散。」坛不应字，只温，温得像答应。祖父韩老栓嘱过：「坛在，韩家不散。」他当家训听，不懂为啥。为啥不要紧，重要是祖辈传下来的手，一代接一代，最后接到他这只糙手上。误触那一滴，缘才满。满，不在心急，在活计，在席，在沿。他独眼平，手不抖：「守住了，才配醒。醒在缘，不在舌。」外头更鼓远，远，像瓮将醒未醒。醒在缘，不在人催。白日，他又编筐半刻。末藤硬，藤刺偏，他偏手，刺浅些。浅些，血仍来。血落席，沿不接。沿等误那一甩。管事验筐：「季末记功在。手稳，沿还净？」韩泥说，「席边湿，沿仍净。」管事「嗯」一声，走了。走，像只看见编筐，看不见坛。窄路也是路。路通向瓮醒，瓮醒通向还。还叶汤，还振东药，还陈姑袄，还刘婆粥。还，先还烫。他低声道：「最后一夜，席盖严。严了，才误触那一滴。误那一滴，根才在。根在，汤还要还。」

他加第三层破布，布更旧，旧更能吸。吸尽席边血气，血气止在沿前三指。三指外，沿滑，沿尘，沿净。净了，才像钩住瓮醒那一缘。坛壁水汽爬半寸，敛。敛着，像钩。钩瓮醒，钩正测，钩还。还，先活过这一夜。夜过，缘才满。满，不在舌，在沿。符怀暖一丝，不燃。不燃，暖够扛前夜寒。他按席，席厚，沿净。净，像六十日擦出来的规矩。规矩在沿，不在舌。秦霜远廊一句：「心别邪。邪，恩断。」韩泥「嗯」一声，席盖又加一层。层厚，厚像守。同批少年碰他袖：「你真不滴血？」韩泥说，「滴，要等藤刺误那一甩。」「误不了咋办？」「误不了，就等。」韩泥说，「等，像守瓮人。」少年哼一声，睡了。睡，棚静。静里，坛腹又温一线。温一线，像瓮翻身半次，又眠。眠，不认主。认主，在沿。老耿挑水路过，声哑：「前夜别拜。拜，恩断。」韩泥「嗯」一声：「不拜。席盖严，不主动滴血。」老耿水洒稳，走了。走，像凡符那条界还在。他摸木牌，牌边磨毛，毛得像心口七笔。笔笔磨过，磨出实。实了，才配误触沿边那一滴。滴了，鸿蒙九劫瓮认主。认主，不在今宵他喊，在缘自己碰沿。他低声对坛：「钩瓮醒，钩在沿。沿净，钩才牢。牢了，才误那一滴。」坛颤极轻，轻像应钩。外头更鼓四响，响沉，沉里风又硬一线。风硬，席更严。严，像五十八年命跟坛，坛跟席，席跟沿。沿净，席厚，符暖，手稳。稳，才配明日藤刺再扎那一误。他独眼平，手不抖，低声道：「最后一夜，席盖严。不主动滴血，活计喂席，席挡沿。」坛腹温加剧，加剧，像瓮将醒未醒。未醒不要紧，钩已攒动。攒动在缘，不在他舌。舌喊醒，像拜邪。拜邪，恩断。他按心口，七笔压着，低声道：「守住了，才配钩成瓮醒。钩成，路才长。长，先还烫。」铁无言门外脚步又停半息，声低：「席盖严了？」「严了。」韩泥说，「严了，才配等沿边那一误。」铁无言「嗯」一声，灯远。远，像夜禁又过了一巡。坡下霜白，柱上无新条。无条，有条在坛。坛在，缘在，还路在。瓮醒前夜，风硬，硬像最后一道冬关。关不过，缘不满；缘满，才误触那一滴。清栏归来，掌茧新裂，血珠饱满。饱满，甩手，血甩破席，席吸尽，沿仍净。他不主动滴血，不揭布诱沿，只用活计扎出的血喂席，席挡沿，沿等误触那一瞬。邻床笑藏宝，他不辩，只加席层。层厚，厚像秘密——秘密是坛，坛是命。管事查棚，灯晃坛角，只见破席，不见仙缘。不见，才好守。夜，符怀暖一丝，护骨寒，手不僵。僵，明日藤刺偏；不偏，缘才正。来席，不来沿——沿上，等你认的血。沉里，丑时将至，将至，像丑时坛温那一线又要走一圈。他合眼，不睡实，实睡像认输。」坛不应字，只温，温得像答应，像等，像明日。
