# 第 25 章 冷眼

公审后第三日，赵绣春闭声，周嫂闭户，村议暂平。

平得像一层薄霜，日头一出，霜还在——还在赵绣春眼里，还在井边闲话里，还在孩童嘴里。陈寻回家换衣，晒袄，肘上针脚干，像一枚静印。院中瓮立墙角，缝处泥新，老耿的手还在。他练体，汗落，不辩，不登门谢沈家——谢是还，还须有工；工未做，谢便是价，价一出，恩就脏。

晨跑过巷，石板凉，凉得像公审那日棚下的风。风已过峰，峰后暂平，平不是无仇——赵绣春赔过礼，仇那道泼水痕却还在，浅，像提醒：仇在心里，不在嘴里。

巷口，童谣丫蹲玩泥，嘴小，眼大，唱：

「四灵根，假神仙，沈家女，丢脸面——」

调子熟，熟得像赵绣春嗓，熟得像从井边晒被处飘来的风。风不带笑，带冷眼——冷眼不是赵绣春一人，是村风剩下来的那半截谣，半截谣卡在孩嘴里，卡得比成人舌更直，更久。

陈寻停步。

丫抬头，不躲，冷眼，像小兽：「我听赵婶唱。你赢了吗？」

「审了。」

「审了，他们还唱。」丫说，「我将来，不唱。」

陈寻看她，半晌，从怀摸糖——方小沅曾放坛边那块，他未吃，今日给丫。糖硬，纸皱，皱得像一块化不开的冷。

丫不接：「不要。赵婶说，你的糖，有毒。」

陈寻收糖，走。糖回怀，怀里有恩，恩八笔温，温得像八盏小灯，灯照不亮童谣，照得清：名在，谣还在唱；唱在孩，比在婶更难断。

冷眼，也是村风——比泼水软，比泼水久。软，不伤皮；久，磨骨。他走过沈家帘边，帘静，药香稳，稳里像沈清弦那四字证白还在日光里——沈，清，弦。三字齐，谣才短；短了，孩嘴里仍会漏，漏成半截。

午后，陈寻去井边担水。

井绳吱呀，水晃，晃出天光。赵小满在旁玩水，见他就缩，像盆水是烫的。缩处，水溅他脚背，他不敢骂，骂要赵婶撑腰，赵婶暂不能嚷，只能给眼。眼从远处晒被处来，赵绣春在晒被，被角扬，扬得像旗，旗不举声，举冷。

她眼扫过陈寻，没嚷，只嘴角一撇，像记一笔未还的嘴辱。嘴辱不入心，入眼；眼冷，冷得像针，针不扎肉，扎名——名在告示下，在钱管事心里，也在童谣丫泥里那个歪字上。

钱管事路过，传话：「测灵名册，陈寻名在，不可私除。」

陈寻接话，只谢规矩，不谢人——规矩是规矩，恩是恩，不混。钱管事眼细，细里像还想提「邪术」，提了一半，咽回去，咽得像怕恩八笔的温烫到自己手里。

担水归，萧惊鸿翻墙：「平了，你还不笑？」

「暂平。」陈寻说，「赵绣春，还在。心里八笔，还在。」

萧惊鸿懂，不再逼笑，只帮他把水缸灌满，像门神，像兄弟。水缸满，满得像把公审那盆冬水隔在院外——隔不住，寒还在记忆里，记忆里泼水的痕在仇，仇淡，淡而毒，毒提醒：辱发生过，名却晒回来了。

张麻婆打门前过，塞一把盐，声低：「公审后，赵家少嚷，你少出门。出门，他们找别的由头。」

陈寻接盐，不入怀，只点头：「知道。」

张麻婆叹：「娃子，冷眼比嘴狠。嘴嚷三天，眼冷三年。」

陈寻不辩，把盐搁灶边——盐已记过，再收是情，情不记恩，记心里。心里他数：八笔恩，一笔未还；还不起，先记，先练，先站。

午后又过，童谣丫仍在巷口，这次不唱，只用小棍戳泥，戳出一个字，像「仙」，又像「假」。陈寻路过，丫抬眼，仍是冷眼：「赵婶说，你三月要出丑。出丑，我更不唱。」

陈寻停半息，说：「出不出，三月见。你唱不唱，随你。」

丫撇嘴，把泥抹平，像抹平一句没出口的谣。抹平了，眼还冷——冷里有一点东西，像不信赵婶全对，又像还不信陈寻全对。不信，是孩；孩的眼，比婶的嘴难缠。

赵绣春在远处晒被，见陈寻与丫说话，眼一冷，拉赵小满走，走前丢下一句，声不大，刚好让井边人听见：「少沾晦气，将来好测灵。」

陈寻不接，只担水，水晃，晃出天光，天光里，他自己的影，稳。稳，是给石看的，不是给赵婶看的。

周嫂门开一线，眼红，不骂陈寻，只骂赵家「借谣害人」——骂声小，像不敢让赵绣春听见，也像让全村听见：公审后，不是只有冷眼，还有怨。怨在周家，怨在赵家，怨成村议暂平后的另一种静。

夜里，陈寻默想，恩八笔，一笔未还。仇泼水痕淡，未消。八笔温，温得紧，紧不是怕，是愧；愧不急还，急的是站直。他吹灯，对空屋说：「暂平，不是终。」

窗外，北岭远，风过林，像一声低嚎——不是狼，是狼前的静。静里，里老贴告示的梆子响了一下，响得远，远得像预告。

第二日，里老贴告示：北岭羊被咬，蹄缺，血溅草，猎户明夜守林。村人围看，有人怕，有人骂，有人指陈寻：「你惹了谣，别连累羊。」

陈寻看告示，看瓮，按心，只应一句：「守。」

羊伤图在告示角，血点粗，像一枚未干的印。印红，红得像北岭草上真血，血引慌，慌引骂，骂又找陈寻——找陈寻，是眼冷的一种，眼冷找由头，由头找祸。

赵绣春在人群外，冷眼，不嚷，像把嗓门的刀收回鞘里，鞘还在。她身边赵铁柱想帮腔，被她一眼压住——公审输了嘴，赵家暂不能再用嘴，只能用眼。眼冷，冷得像说：祸来，看你守不守得住。

老耿从人缝过，独眼扫告示，扫陈寻，声低：「明夜我值。你要来，别冲前。」

陈寻应：「不冲。」

不冲，是还瓮的工，也是还命的路——路未起，工先起。起在守夜，起在北岭脚，起在火边。

萧惊鸿夜里来，带柴刀：「北岭我去。」

「我去。」陈寻说，「老耿修瓮，我守他一夜，不亏。」

萧惊鸿咬牙，终是点头：「第三夜最险。第三夜，我陪你。」

陈寻望北岭，林黑，像一张合不上的嘴。里老又补一句：狼昼近，孩勿出村。话落，巷里孩哭，娘拽，拽得紧，像怕林里那张嘴伸到门槛。童谣丫也在娘拽处，不唱，只冷眼望北，望得像懂：有些祸，嘴赢不了。

陈寻回院，磨刀，磨的是柴刀，不是猎刀——测灵在即，刀不能乱。磨罢，抚瓮，瓮凉，泥缝干，像老耿的手还在。心里落一字：狼患。字沉，像第八笔未记的债，不是恩，是祸，祸来，也得接。

夜更深，他默想，不记新字，只数旧字。数罢，对瓮说：「你修过，我守过。守得过守不过，北岭见。」瓮不应，风应，风从北来，带草腥，像预告。

萧惊鸿走后，院复静。陈寻把糖仍搁怀，糖硬，像一块化不开的冷。他想起童谣丫的眼——冷眼，不接糖，比赵婶的嘴还直。直，不伤他；伤他的，是眼背后的谣，谣还在唱，唱在井边，唱在晒被处，唱在北岭风里。

他合衣，未睡，听村犬偶吠。犬吠止，北岭又静，静得像一张合不上的嘴，在等第三夜。等，也是守。守，不必嚷。

天将明，鸡鸣，村议暂平，北岭未平。陈寻肩未肿，肿在将来；将来第三夜，火要够，步要稳，背要空——背若满了恩与辱，就背不动命。命还未记，记须在狼口之后，须在沈铺针下，须在恩顶格那一笔。

他望北岭，林黑，嘴合不上。合不上，村就不宁；宁不了，冷眼也会变成喊——喊在羊死时，喊在血溅时，喊在第三夜。第三夜，他要去，不为逞，为还瓮，为还一点，为让赵绣春那眼冷看一看：桩，也会守夜。
