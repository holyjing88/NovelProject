# 第四十二章 兽栏

冬深，霜咬棚顶，咬得丙九人缩肩。兽栏更深，深在角。角里小兽卧干草，卧不稳，喘细，喘得像漏风的箱。韩泥清粪，粪铲下稳。稳，是备检后第三日练出来的——练在雪里，练在腥里，练在赖福的秤旁。管事掷单：「丙九韩泥，冬角仍你。清细，别漏。」

「嗯。」韩泥说。赖福阴笑：「晦气杂役，兽栏专收晦气兽。」

韩泥不答。舌要留来哄兽——兽不懂灵根，懂手温。午后，小兽咳重，重一声，轻一声。韩泥换干草，草旧，潮。潮草撤，新草铺，铺得平，平才卧稳。同批杂役掩鼻：「病兽，别沾身！」

「沾了，洗手。」韩泥说，「手净，粪才净。」

那人退半步，仍嘟：「晦气。」

小兽眼半阖，半阖里还有一点活气。活气薄，薄也真。韩泥蹲，掌轻按兽背，背烫，烫得浅。浅烫，比冷强。「活一日，清一日。」他低声道，「清一日，少挨一句骂。」

兽不懂，咳轻半声。轻半声，也是小胜。同批杂役又凑近半步，仍掩鼻：「你按它，它传染你怎么办？」

「传染不了饿。」韩泥说，「饿传染全丙九。你躲，栏臭；栏臭，管事骂你。」

那人噎住，退远，仍嘟：「晦气。」

韩泥从破袄内摸出半块麸皮，掰指甲大，搁干草边。兽鼻动，舔不到，舔，也是活气。管事路过，瞥一眼：「还管病兽？」

「管。」韩泥说，「角清不净，全栏臭。臭了，你骂丙九。」

管事「哼」一声，走前却道：「清细，记着。」

赖福远处啐：「装善。」

韩泥不回头。善不善，兽不管；兽只管草干不湿。黄昏，雪又落。落进栏，落进角，落进小兽鼻下。韩泥加半刻，把湿粪清尽，尽到角外三尺。三尺净，味淡一寸。小兽喘更细，细得像线将断。他低声道：「断不得。你断，我少一笔记功；你活，我多一口饭。」

兽眼动一下，动，像应。夜前，管事来查，靴底踩雪，声沉：「角清了？」

「清了。管事蹲，看兽：「这畜，活不过今夜。」

「知。」韩泥说，「活不过，仍清。」

管事看他一眼，眼里没有笑，也没有嘲：「丙九里，你话少，手稳。」

「话多，手抖。管事起身，掷半块麸皮：「赏你的。别给赖福看见。」

韩泥接住，麸皮糙，糙也香。香在冬里，冬里一口，够撑丑时。同批杂役嘀咕：「病兽死了，你也捞赏？」

「兽死栏净。」韩泥说，「净了，少骂丙九。少骂，也是饭。」

嘀咕断了。刘婆提粥来，粥稀，热：「守角别抖手。抖了，备检白过。」

「不抖。她哼一声，多舀半勺。半勺，热在。赖福远处骂：「疯婆子又偏晦气！」

刘婆回骂：「晦气在你秤上！秤歪，粥才稀！」

韩泥不掺。掺，费舌；舌要留来清兽栏。同批杂役躲远：「死兽角，晦气！」

韩泥「嗯」一声：「晦气在嘴，净在铲。」

管事路过，鼻下动一息：「味淡了。淡一寸，丙九少骂一句。赖福阴笑：「味淡有屁用，凡骨仍是凡骨。凡骨在石上，不在栏上。栏净，饭在；饭在，命在。他把角外三尺又清一遍。三遍净，味淡两寸——两寸不大，够丙九喘一口气。丑时，小兽咳止。止，不是好，是尽。韩泥立半息，不拜，不嚎。嚎像少年，他是记心翁，恩仇都记在心里。他清最后一铲，铲下稳，稳得像替兽收尽——命在栏，恩在心里；心在活，不在哭。移兽时，同批少年躲远。躲，韩泥独力抬兽，抬得稳，稳得像抬棺老把式。赖福啐：「抬晦气畜，晦气上加晦气！答，费舌；舌要留来记管事那句「明日仍你」。管事复查角，靴底踩净地：「死兽移走，角消毒。韩泥，明日仍你。邻床少年悄问：「你真不怕晦气？」

「怕晦气，怕不过饿。」韩泥说，「饿狠了，手抖；手抖，备检白过。白过，滚。」

少年闭嘴，睡了。韩泥把半块麸皮收进破袄内袋。赏不重，重在管事看见了手稳——看见了，明日少踹一脚。他摸坛沿，沿温应「净」字，不应哭。哭费气，气要留来清明日粪。外头风息，栏味淡一寸。淡一寸，丙九喘一口气——一口气，够活，够熬。他低声道：「冬深兽栏，死一畜，净一角。角净，人少骂一句。」

同批少年问：「兽死了，你难过？」

「难过像少年。」韩泥说，「恩仇都记在心里。兽命尽，心里记着清了。」

少年不懂，懂「栏净」，点头睡了。韩泥把麸皮掰半，半留己，半塞邻床梦呓者手里。塞了，不声张。声张，赖福来抢。邻床少年梦呓：「冷……」

韩泥替盖袄，袄破，破也暖。暖一线，够活。管事巡夜，靴声沉：「冬角清了，明日仍你。别抖。丑时，栏味又淡半寸。淡半寸，够他清明日粪，够他备检后第四日仍站冬角。冬角在，命在。命在，瓮才醒得正。刘婆门外咳一声，不高：「栏净了，睡。睡实，手才不抖。」

韩泥「嗯」一声，不回头。回头像认恩，恩在粥里，不在栏哭。淡一寸，丙九喘一口气——一口气，够活，够熬到备检后仍站冬角。他低声道：「栏净一日，少骂一句。少骂一句，丙九多喘一口气。」

赖福远处骂：「晦气扫栏，扫出妖！回头像认谣，谣在秤上，不在栏净。管事靴声沉，沉得像钉：「冬角清了，备检前仍你。坛壁一线水汽，应「净」字。净在栏，不在骂。骂在赖福秤上，秤歪，栏仍要清。淡半寸，够他清明日仍站冬角，够他备检后手仍稳。<!-- v38-thicken -->冬深兽栏，粪臭，病兽喘。喘，像命薄——薄，也要清。小兽尽，尽，多清半刻。半刻，像多一口气——气给同批，也给己。赖福骂「晦气扫把星」，骂，兽仍尽。尽，不是咒赢，是手稳。韩泥不辩，只多干半刻。半刻，小暖——暖不大，够丙九有人信「丑翁手稳」。坛应忍，温一线。温像说：活过兽栏，才配盯秤那日。那日，饭要实，恩要热。小兽尽，多清半刻——半刻，像多一口气，气给同批，也给己。赖福骂扫把星，骂，兽仍尽。尽，不是咒赢，是手稳——稳，才配四十三管事盯秤，才配三两实粮落袋，才配正测不倒。兽尽多清半刻，半刻是小暖。暖给丙九——丙九信手稳，才配盯秤，才配三两实。兽栏病尽，多清半刻——半刻暖丙九，暖到盯秤那日，饭才实三两。<!-- v38-topup -->

病兽尽，多清半刻。半刻是小暖，暖给丙九——丙九信手稳，才配盯秤那日。饭才实三两，三两落袋，才真；真，赖福才虚，虚不是终，终在正测。半刻暖丙九——暖了，手才稳，才配盯秤那日三两落袋。半刻暖丙九，暖了，手才稳，才配盯秤。他低声道：「手稳，恩不断；恩不断，汤就不凉。」

坛沿他仍擦。擦时指节白，白像证：辱在册，恩在胸，分列不混。还远不怕，怕的是汤凉——汤在叶铺，在刘婆灶，在心里。泥岗根在，坛沿在，还路在——还路在，手就在；手在，恩就不断。独眼平，像看秤——秤在，铺就不关；铺不关，汤就能还热。<!-- v38-end -->

病兽尽，多清半刻；韩泥收扫把：半刻是小暖，暖给丙九——丙九信手稳，才配盯秤那日，饭才实三两，三两落袋，才真。**状态**：大境·凡人 · 资质·丑骨末席（伪灵根） · 鸿蒙九劫瓮·眠 · 宗门·丙九杂役 · 漏舍凡舍

兽栏夜，他多清半刻粪。半刻暖丙九，暖了，手才稳，才配盯秤那日三两落袋。

多清半刻，小兽鼻息匀了。匀了，像丙九又暖一寸——寸暖，手才稳。手稳，恩不断。稳了。
