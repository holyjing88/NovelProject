# 第 20 章 泼水

> 纲：青牛入宗衔接专纲 · 011～035 块 B 
> 连载名：《凡人修真传》

```
【第 20 章】泼水
已记：张麻婆食/盐 · 沈氏女汤 · 陈姑衣 · 方小沅粥 · 老耿瓮 · 沈氏女衣
```

公审续，棚外风紧，像要把棚顶掀了。

赵村长一声「礼」，声软，软里藏硬，像把刀包在棉里递出来。陈寻昨夜便觉不对——礼不对，是坑；坑不在审，在审前，在众目，在把「公审」先变成「公辱」。他昨夜换的干衣，此刻又要在众目下湿回去，湿回去的不是衣，是「四灵根也配站堂」这句话的实物。

赵小满自侧出，袖湿，盆在袖下，盆沿冰，冰映天灰。他不敢看陈寻眼，只看陈寻脚，脚稳，稳得像钉，钉得他心虚，心虚里又狠——狠是赵绣春教的，教成一种本能：先泼，后审，泼净晦气，审才「公」。教的时候，赵绣春说，水洗不净，人心才净；净的是看笑话的人，脏的是站堂的人。

「洗晦气！」赵绣春拍掌，掌响脆，脆里带笑，「四灵根，也配站堂？站堂之前，先洗！洗过了，才配听审！」

人群哄，像潮，潮从棚下涌起，涌到陈寻面前，不是水，是眼，是舌，是等着看笑话的声。有人往前挤，挤成半圈，半圈里水还没泼，笑先泼了——笑泼在陈寻名上，泼在测灵帖上，泼在沈家帘方向，像要把谣先洗成「众口一词」。

赵村长不拦，只笑，笑僵在脸，像水已泼到自己额上，却还要装「礼法」：「洗过，再审。审过，再论。青牛村讲理，也讲礼。」

陈寻没退，也没躲。他看赵小满，看赵绣春，看赵村长——三人成网，网是辱，不是刀；刀在明处，辱在礼里，礼一出口，他若躲，躲的是审；他若怒，怒的是案。他站的位置，恰在堂心，心不是堂，是众目钉出来的点，点上站着，才不算逃。

盆起，水泼。

水泼陈寻面颈，透，响，像当众剥一层皮。冬水寒，寒进骨，额发贴，眼不闭。水顺颌滴，滴在干土上，土黑，像墨；墨不是污，是记——记谁泼，记谁笑，记谁把「公审」写成「公洗」。水里有冰碴，碴划过颈侧，划出一条细线，线疼，疼里他反而不想动——动一下，像躲；躲一下，像服。

人群哄笑更大，有人学赵绣春嗓，学「晦气」，学「四灵根」，学「沈家」，笑混成一锅，锅沸，沸在堂前，沸不在理。赵铁柱在笑，笑闷；钱管事在记，笔快，快得像怕漏了这场「礼」。沈振东立人群后，眉沉，沉里没出声——出声在此时，是给赵家添柴。

陈寻不擦。

擦，像认；认，像服。他不服，也不辩——辩在此时，是给口水加盐。他任水冷，任笑热，只立，立得像一根没倒的桩，桩上挂着湿衣，湿衣下是干骨，干骨里是心，心在心里。

恩仇在怀，**记在心里**；**仇，痕**。

极淡，像指甲划，又像水渗进冷纸，三息，在，不散——比集上斗殴那痕深一线，深里带湿，湿里带寒，像仇第一次真正「吃」了辱，吃了，却不吐字，只留印。印起时，他胸口一窒，窒像有人把堂上哄笑折成薄纸，塞进仇，塞进去，纸不鼓，只沉——辱已在心，痕只是证。

陈寻指节一紧，没动。他低眼，见阴痕沿纸纹扩半分，扩像潮退后沙上留的线，线浅，线毒，毒不在形，在「当众」二字。当众的辱，比暗巷的刀更毒——刀只伤皮，辱伤名；名若伤了，测灵帖上那两个字，就脏了。

恩六笔仍温，温在，像六盏小灯照他湿脸——照不乾水，照得清：恩在阳，辱在心里；仇可留淡痕，不收恩；留痕，不是认，是留证。六笔温与阴痕冷，一怀两温，温冷不相混，像程不二说过：恩仇分立，心才不乱。

萧惊鸿在人群，骨节响，像弓满。他眼里水与火绞，火要出，水要压，压不住，弓便满。他往前挤了半步，半步像要破网——网是赵小满，是赵绣春，是堂上这盆冬水。

陈寻侧目，一眼，只一眼——

那一眼里有「三月前」，有「测灵名」，有「名在，站才有机会」，有「案上加拳，名没了」；还有集口粮铺柱上那悬着的拳，拳落下去，是爽，爽完，名没了。

萧惊鸿喉结一动，像把火咽回肚里。萧的弓松半寸。半寸，像刀回鞘半寸，鞘里仍响，响而不出。他拳硬，硬在衣下，指节白，白得像要把自己的皮也勒破——破不得，破就是案，案就是除名。

「审。」陈寻说，声不高，却穿笑，「继续。」

赵村长笑僵，像水泼自己脸上，抹不掉：「审，审……周嫂，你说。」

周嫂本哭，此刻哭更大，话更乱，像被水滑了舌，滑向「听说」「赵婶说」「沈家女夜入」「陈寻封门是假」——越说越虚，虚里带抖，抖是怕，怕陈寻不辩，怕沈振东在，怕伤者已证，怕老耿、张麻婆、方小沅的舌已在链上。她每说一句，赵绣春就拍一下掌，掌像给她壮胆，也像给她封口，封不住，就显虚。

陈寻任水冷，任笑热，只立。

羞辱到顶，像浪峰——峰顶，水寒，笑烫，赵绣春还在拍掌，掌每一下，像往峰上再添一勺；赵小满退半步，盆空，袖仍湿，湿得像烫，烫在赵家脸上，赵家却当烫在陈寻身上。峰顶最毒处，不是冷，是静——陈寻不擦、不跪、不躲，静得像把赵家所有准备好的台阶都拆了，拆得赵绣春笑不出来，笑僵，僵成一张干皮。

陈寻看棚顶灰天，天低，低得像要塌。塌不塌，是天的；站不站，是人的。他站。站里，他觉出面皮在收，收得像布，布会干，干后皱，皱是痕，痕在脸上，也在众目里——众目记住的，有时比唾沫还难抹。

峰后，或沉，或反——沉，是赵家要的；反，不是拳，是证，是名，是心里还没落完的笔。仇那道水痕还在，像等公审把唾沫收成字。

算，不是现在。现在，是把峰扛过去，扛到证词齐，扛到该来的人出来——名在，人在，辱才不能私除测灵名。

水顺颌再滴，滴在土上，土黑。他指节又紧，仇那痕，三息，仍不散，像心替他说了一句仇的话：我看见了，**记在心里**，我不还嘴。不还嘴，不是没嘴，是把嘴留给该说的时辰——时辰在证，不在水。

萧惊鸿在人群里，拳仍硬，硬而不出——不出，是陈寻那一眼；一眼，够三月前，也够堂上。他退回去半步，半步像把自己钉回人群，钉住了，怒还在，怒在骨，骨里响，响而不爆。

公审未毕，泼水是峰，峰后还有链，还有反噬，还有帘后的人——人未出，名先在谣里脏着；脏着，就要有人把它摘出来，晒在日光下。

陈寻不擦脸，像不给赵家一个「他服了」的台阶。台阶在审，不在水；水会干，寒会留，留里，他反而更清醒——清醒像刀背，刀背不杀人，只把怒按平，按到心能收的时辰。清醒里，他竟能听见自己牙关没咬——没咬，也是礼；礼是忍，忍不是怂，是把峰扛到链齐。

赵村长再拍案，案响，响里催周嫂。周嫂哭，哭里漏「赵婶让我说」，漏半句，像链扣一环，环在赵绣春，赵绣春脸一白，白里仍嘴硬：「胡说！公审公审，以证为据！」

「以证为据。」陈寻重复，重复像钉，钉回正题，「继续。」
