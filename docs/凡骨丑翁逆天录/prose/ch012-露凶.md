# 第十二章 露凶

叶家灯弱第二日，张麻婆骂到韩泥门口，骂得反常——没骂筐，骂赖扒皮。

「孙福那扒皮的！沈家欠几文，利滚利滚到卖娃！缺德！」

布包甩门墩，烫，实，两个馒，硬过昨日。

韩泥开门，独眼抬一下。

张麻婆叉腰：「吃！别饿死！你死了，谁看赖扒皮吃人！」

「我不是为你。」她唾沫飞，「我是气！气不过！」

三步远，又骂：「晦气翁，你命硬，硬到把全村晦气都吸了？」

骂是护。护里带刺，刺向外，不向他。

韩泥掰馒，慢慢吃。面实，噎人。噎得他想起告示桩「连坐」二字——字黑，像赖扒皮的牙。

张麻婆骂完，又折返，嗓低半寸：「十文别硬扛。扛不住，跟我说。我不是帮你，是烦扒皮踩我门口。」

韩泥「嗯」一声，不接谢。谢字会害张麻婆也被写进短签。村头柜新漆，漆黑，像张嘴。村人绕柜走，走快，像怕沾上沈家的债。

韩泥路过，摸出五文，搁柜边，不声响。声响成「牵扯」。赖扒皮的人数文，数完，眼细，没追他——追了，少一文，沈家仍滚。

柜旁有人低声：「晦气翁也放钱？放钱，是不是认债？」

韩泥独眼平：「认活。活要十文，十文先攒。」

那人退半步，没再问——问，像沾晦气；退，像沾一点胆。午后，赖扒皮亲至村头。

孙福，沉丹宗杂役管事，脸笑，笑薄，像刀背。周嫂迎上去，腰弯：「赖管事，晦气翁又收东西！」

赖扒皮不看韩泥屋，看沈家：「沈家药账，利加三成。芽儿罚银，十日内不齐，逐出村籍。」

沈家院门开一线，沈枯芽娘咳，咳不出声，只跪：「管事……宽几日……」

赖扒皮脚尖一抬，像要踢，又收：「宽？谁宽我？晦气翁收你们东西，宗门问罪，谁担？」

韩泥站院外三步，没进。进，成「牵扯」证据。

他开口，嗓哑，不高：「我收的是粥，不是沈家。罚我，别罚娃。」

赖扒皮转头，眼细：「老绝户，你担？」

「担。」

赖扒皮笑，笑冷：「好。十文放村头柜，少一文，沈家滚。」

笑完，袖里摸出一短签，掷地：「这是记。记你韩泥，记沈家，记叶家。谁再送，连坐写全名。」

短签黑，像仇苗出土，根扎地，拔不出。

张麻婆远处啐：「扒皮的！迟早雷劈！」

赖扒皮不恼，只道：「骂够，交十文。」

周嫂凑前，嗓甜得假：「管事英明。晦气翁不除，全村不安。」

赖扒皮瞥她一眼，笑更薄：「安不安，看柜。柜满，宗门满意。」

转身，步稳，像踩全村喉。韩泥回屋，心口四恩在，赵辱旁，仇苗加深：赖扒皮，露凶了。

他低声道：「记着。」

他摸出三文，又摸出集尾剩的两文，凑五文，包布，塞床板缝——穷，也要攒。攒够十文，先救沈家滚不了。

门外风硬，吹动门墩空布。张麻婆的馒已入腹，暖还在胃——暖是恩，叉是仇。恩仇分列，不混。

沈家院门再开一线，芽儿探头，又缩。缩里，似有人递声，怯：「韩翁……娘说……谢谢……」

韩泥不答谢。答，成牵扯。他只道：「别谢。活。十文，翁攒。」

芽儿「嗯」一声，门合，合得像把怕关在里面——关着，人还在。

夜里，沈家院静。静里，似有人摸铜板，摸得轻，像数命。

韩泥合袄，低声道：「恩在。仇也在。」

他又摸床板缝，五文还在，文文碰，碰出脆响，像十文的头。头小，真；真，就要攒到沈家不被滚的那日。

赖扒皮的短签贴桩，风撕角，撕不撕名——名在心里，不在风上。风过，仇苗又深半分，深得像牙根。

坛底微温一丝，像吸了赖扒皮的笑。韩泥不指望瓮醒，指望活：活到达得到还、达得到拔苗的那日。

张麻婆远处仍骂，骂声像隔墙的暖，暖不进门，进门的是馒的实——实的恩，他记着，仇也记着，分列，不混。

沈家院门再开一线，芽儿探头，探头又缩，像怕短签。怕，仍活着——活着，点就不灭。

韩泥低声道：「十文，我攒。攒够，先止滚。」

仇苗在心，恩在胃。胃暖，苗冷，冷暖分列，像冬里两盏灯，各照一路。

赖扒皮步出村，步稳，像把短签的影拖长——影长，村人更绕柜走。走快，沈家更静；静，韩泥更须攒够十文。

短签黑字「连坐」，他不在签上划，只在心里加深——加深，等将来有一日，叉能变刃。刃不急，急乱心。

五文在床缝，文文碰，像沈家命在数。数到十，先止滚；止滚，再谈还。还，是后话；先活，是前话。前话在，后话才有着落。里正路过，停桩前，读短签，脸沉。

「连坐写全名……」他低声，像念给全村，「这字，太重。」

韩泥开门缝一线：「重，才露凶。露了，才记。」

里正看他，独眼平，臂无伤：「你攒你的十文。公审前，别撞他刀口。」

「不撞。」韩泥道，「刀口在桩，我在缝。缝里攒，桩上记。」

里正「嗯」一声，步远，远里锣未敲，心已紧——紧，像公审的影又近半寸。

韩泥摸坛，坛底温一丝不退，像瓮睡了，牙印却留在腹里——留在，仇苗就不浅。

夜里，沈家院又静。静里，铜板轻碰，像数到第三文。

韩泥不睡实，合袄塞怀，低声道：「露凶了，更要攒。攒到十文，先止滚。」

床缝五文，文文碰，碰得像沈家命在跳——跳得慢，跳得真，真就不白忍赖扒皮的笑。

门外风过桩，短签角又撕一分，撕不撕名。名在仇苗里，苗在，公审前就须立住「不滚」二字。

张麻婆远处骂声歇了，歇了，馒的暖还在胃——胃暖，手还要编筐，筐卖一文是一文，一文也是十文的肉。

张麻婆远处哼：「睡不实就编筐。」

韩泥「嗯」声：「十文，攒。」

馒烫，赖凶露。恩在，仇现——现，不混，混则汤凉。今宵只记，明日仍编筐，仍擦沿，仍候锤；锤来，证白才爽，爽是后话，话先在掌里。

馒烫，赖凶露。恩在，仇现——现，不混，混则汤凉；今宵只记，明日仍编筐。

他低声道：「手稳，恩不断；恩不断，汤就不凉。」

坛沿他仍擦。擦时指节白，白像证：辱在册，恩在胸，分列不混。

馒烫，赖凶露；韩泥分列：恩在，仇现——现，不混，混则汤凉；今宵只记，明日仍编筐，仍擦沿，仍候锤，锤来，证白才爽。
