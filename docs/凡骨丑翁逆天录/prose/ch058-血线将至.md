# 第五十八章 血线将至

试手日，卯时前。韩泥夜里没睡实，心里把七笔一笔笔盘过——恩是恩，辱是辱，别搅成一锅浑汤。

符怀暖一丝，坛腹温一线。温着仍眠，像瓮也在等规矩开锣。

坡下霜白贴石阶。秦霜最后一遍叮：「手抖，丹歪，滚。丹丑不要紧，手不能歪。」

试手凝丹，只验手，不验灵根色。色可以末，手一抖就滚。

韩泥立末排，独眼平。铁无言验鼻：「静带到手上。」

同批少年肩抖得像怕「滚」字落地。韩泥不抖。抖是心先飘；心不飘，手就不飘。丹炉前他掌稳，丹形丑，却正——正得像辨香那日鼻下那口静气。

执事记功，不记灵。管事只点头：「麸皮不克。」

赖福脸青，退半步。韩泥不喜。少年才喜；他只记一件事：手正了，窄路又宽半分。宽不是灵根忽然变好，是七笔没断。

同批少年碰他袖：「你真过了？」

「过了。」韩泥说，「过的是手，不是灵。」

「灵根末，手正，正就够。」铁无言声低，只对他。

秦霜目光扫过队尾，声冷：「再嗓，记名。」

队散。赖福啐地，啐得虚。韩泥不回头。回头像认谣，谣借腥，他不碰。坡下柱上无新条，他把木牌与记功帖并放——命挨着规矩，挨着才算过关。

秦霜问心时，他声哑却平：「为还。还，先还烫。」

秦霜不夸，只点头。点头够了。脸丑，手正，正就够。

午后清西栏半刻，粪锹沉，掌裂不僵——符暖护着骨。编筐补单时，藤刺深扎，扎透旧茧，血珠这回饱满了起来。

他甩手。血甩向席盖，席吸住，血未到沿。未到，像规矩还在掌心里掐着。

他盯掌，盯席。角静。他低声道：「血来了，往席上甩。甩干净，像干活，不像邪。」

刘婆端姜汤来，姜薄，热：「试手过了，别傲。」

「不傲。」韩泥说，「傲，符自灭。灭了，手就僵。」

管事路过验手：「手抖否？」

「不抖。」韩泥说，「抖是心飘。心不飘，手就不飘。」

席边湿了一线，湿不到沿。铁无言路过，声低：「手正。正了，饭才稳。」

「等。」韩泥只答一个字。

他不为拜血祠。掌茧裂出血珠，甩向破席，席吸尽——像杂役常活，不叫旁人当邪脉看。席上湿一线，他只当干活脏了，赶紧盖严。

管事贴条于柱：「试手记功在。记功，麸皮不克。」条薄，真在「手正」，不在「灵色」。他仍末排。编筐第二单，藤刺浅些扎，血线可控。血珠又起时，他甩向破席，席厚，吸得尽，沿边仍无尘。

管事再验：「试手过了，手还抖否？」

「不抖。」

老耿挑水路过，水洒实：「试手过了，手还正？」

「净。」韩泥说，「席盖着。」

管事验棚，烛照席角：「席掀一角，像晦气。」

韩泥用旧布再擦，擦得沿滑、有尘、无血。赖福远处骂：「试手过了也晦气。」

管事声冷：「再嗓，克半月麸皮。」

赖福哑了。邻床终于忍不住问：「你手咋了？」

「扎藤。」韩泥说，「血先找席，席挡着才不到沿。」

「沿是啥？」

「沿是罐子边上。」韩泥说，「擦干净就行。擦急了，像拜邪。」

他盖席，盖严，严像藏私——私是坛，坛是命。秦霜远廊一句：「手正，心别邪。邪，符灭。」他心里七笔都在，不邪。

夜里符怀暖，药涂裂口，血止住，止在沿前，止得像规矩本身。坛温加剧，像将醒未醒——醒在缘，不在人催。催像拜邪。

丑时前，他按席，席厚，厚像守了六十日。对坛低声：「席盖严，你安眠。安眠，像爹嘱过的腌菜坛。」

坛腹温一线，又凉。凉像应「等」。同批少年蹭袖角，声小：「你那罐血了咋办？」

「席挡着。」韩泥说，「挡着就不像邪。」

少年不懂，却走了。他把旧袄角撕下来，补席边破处——补罢席更厚，厚像后头那一夜。席厚，血落席上，脏不到旁人眼里。

小兽栏角小兽咳得细，他多铺干草。暖兽像冬里少死一口，少死一口，恩便多记一笔。摸记功帖，帖边磨毛，毛得像心口七笔。笔笔在，手才不飘。

五更前他数更鼓，掌贴席边湿痕——湿痕止于沿前三指。三指外，沿尘、沿滑、沿上无血。无血，才不像晦气。晦气在赖福嘴上，不在他手上。

丑时末，他掀席角再看湿痕。湿痕止在沿前三指，三指外沿尘仍干——像血已至席边，却在沿前认了规矩，不肯再进半步。邻床梦中喊「滚」，他不扰，只低声：「滚在舌上，不在沿上。」

执事收帖，末排名在尾。尾不要紧，帖上有「手正」二字才真。他补席边第三处破角，挡到沿前，沿不接；不接，误才来。坡下炉灰冷，冷像试手毕了——毕了，活计才开始：藤刺还会扎，血还会找席。席在，缘在；缘在，七笔就在。

刘婆又端半勺姜汤：「试手过了，别空肚。」

「不空。」韩泥喝尽，「姜在记里。记里暖了，刺才不偏。」

铁无言夜巡，灯停棚角半息，只低喝：「别拜邪。拜邪，恩断。」

「痕在席边，止沿前三指。」韩泥答。

管事夜巡验棚，烛再照：「席边湿，席仍严。」

「严着。」韩泥说，「痕止席上。脏不到旁人眼里。」

他独眼平，手不抖。抖了手飘，飘了席挡不住脏。席严着，血落席上，旁人少说话。风里柱上记功条还晃，晃着也真。真在手正，真在席边那一线湿痕——湿在席，像干活留下的，不像邪。

试手后第一夜，他把干草又铺一层在席下。草旧，旧能吸潮；吸潮了，裂口才不乱渗。乱渗了，明日藤刺偏；偏了，编筐就歪。歪不要紧，手正要紧——正是坡下那炉丹形丑而正教出来的。

管事收试手帖进册，册边多一行「手正」。行丑，真在印。印在，麸皮不克；克粮，仍克不走还路。还路先还烫，烫在叶汤，不在舌尖争一句「我过了」。

西栏角落小兽又咳，他多垫一把草。垫草不算功，功簿上不写，心里却添一笔暖。暖兽，像冬里少死人；少死人，恩账就不断。

赖福夜里堵在棚口：「试手过了又怎样？灵根末还是末。」

「末就末。」韩泥说，「末排也出过手正。手正了，血有席接；席接了，沿就不脏。」

赖福噎住。噎住正好——嗓子一闲，沿边那一线湿痕更安静。安静里他再量沿前三指：三指外仍干。干着，才叫血线将至而不接。今晚只把席补厚、手擦干、七笔按实。按实了，天一亮藤刺再扎，血仍往席上落，不往旁人眼里送。

丑时后他再用旧布擦沿一周。擦完指尖闻不到腥，只闻得到干草与姜末。姜末是刘婆留下的，碎，却够提神。提神不是为夸「我过了试手」，是为下一藤刺来时甩得准——甩准了，血落席；落席了，管事少说话。

邻床迷迷糊糊问：「你还熬？」

「熬沿。」韩泥说，「沿脏一夜，六十日白守。」

邻床翻过去打呼噜。呼噜里坛腹又温一线，温完仍眠。眠着，他把记功帖角按平，帖上「手正」二字摸得起毛。毛了也好，毛像提醒：血线将至，手仍正，席仍严。
