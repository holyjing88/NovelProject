# 第九十七章 挂牌试水

晨雾里，韩泥把木牌挂上了。

牌歪，字丑，三个字：「丑丹师」。师字写得偏大。大不是认自己是师，是试——试坊市打不打他，打到什么份上。

牌下仍卖渣，三包灰扑扑。包旁多一小签：「可嗅，不包治。」

不包治，才像杂役该写的。该写的，管事少问。

第一个问价的，是外门瘦弟子：「嗅渣多少钱？」

「半文。」韩泥说，「嗅一次，不加价。」

瘦弟子递渣过来，渣灰。韩泥鼻下一动，动半息：「青芯过多，火路会焦。减一撮。」

瘦弟子愣住：「你真嗅？」

「嗅。」韩泥说，「嗅完，手稳。」

瘦弟子付钱，付得像试人。试了也不嚷，谣便远着。远不要紧，近处半文已落到筐里。

第二个是药摊婆，眼睛浑浊：「帮我嗅这包。嗅不出，不赖你。」

韩泥嗅过，鼻下分得清褐多青少。他指节在包上点三下：「加青，减褐，先温后投。」

药摊婆喜了半分，嘴上却不谢。谢轻，他记在心里就够。

白织月路过，瞥牌一眼，声冷：「丑丹师？也配？」

韩泥不抬眼：「配试。试不过，撤牌。」

白织月袖风一掠而过。那是风，不是判。判在丹，丹还没到他这摊上。

铁无言立得远，抱着臂，不近摊。不近像不认；眼却在。眼在，就是证。

午后管事弟子来了：「谁许你挂师字？」

韩泥声平：「试。试不过，撤。」

管事记簿，笔停半息：「试三日。三日无投诉，再议。有投诉，连坐丙九。」

「三日。」韩泥应。

三日够了；够了才知道坊市认不认这双丑手。

赖福闻讯晃秤杆过来：「丑丹师？晦气师！」

韩泥只答：「牌歪，人丑，渣真。」

真不在嘴上，在鼻子里。鼻下静，手就不飘。

收摊他数半文，文少。少是试水，不是养家。夜里按坛，低声道：「试过了。坊市没赶。」

坛腹温，像应「试」字。试不是成；成还远。

邻床问：「你真敢挂师字？」

「敢偷闲试。」韩泥说，「试不过就撤。撤了，仍清栏。」

「不怕连坐？」

「怕。」韩泥道，「怕也试。不试，鼻子白养。」

丑时静半刻，香落齐，掌纹温一线。温像培元路又熟半分——熟着，才撑得住三日那根绳。

第二日牌仍歪。歪像他人，才少惹眼。有人远骂「卖腥」，韩泥不接。他接渣，嗅，半文，不加价。加价像营生；营生一响，赖扒皮秤就跟来。

药摊婆又来一趟，递包时低声：「昨日那法，炉里稳了。」

「稳就好。」韩泥说，「半文。」

「不添？」

「不添。」韩泥道，「添了像招摇。」

白织月第二回路过，只丢半句：「卖腥的还活着？」

「活人卖渣。」韩泥应，「死人卖不了。」

她哼一声走了。铁无言远远点了点下颌，像说：手别抖。韩泥手没抖。

赖福堵路：「三日没人告你？」

「没有。」韩泥说。

「走运。」赖福眯眼，「走完了，看看连坐。」

韩泥不辩。辩费舌；舌留着给第三日。

第三日午后，瘦弟子又来，像认手不认名。韩泥仍半文，仍不加价。管事弟子收簿，墨新，声硬：「试三日，再议。」

再议不是功名，是暂留的缝。缝窄，够他喘一口气。

韩泥收摊时把「师」字用布一蒙——蒙不是撤尽，是明天改小。改小，才藏得住。藏得住，才接得住往后续摊那几日。

刘婆夜里端粥：「挂了？挂了嘴硬没？」

「硬过。」韩泥说，「硬完还活着。」

「活着就行。」刘婆道，「兽潮风紧，别拿半文买命。」

「半文买耳。」韩泥应，「命留给清栏。」

沈枯芽怯怯递过：「叶姐说……别逞能。」

韩泥接话在心里，嘴上只「嗯」。恩在那边热，试在这边冷，两头不能糊成一锅。

更鼓沉，他把木牌按到胸口。牌凉，三日之约却烫着。烫着催下一程：续摊，听条，备兽潮。

邻床又问：「三日过了算过？」

「算试过。」韩泥说，「过没过，看簿上字。」

「簿上写啥？」

「再议。」韩泥道，「再议就是还没赶。没赶，就能再挂一日。」

他望向界柱方向。柱上兽潮二字早已贴着，猎猎的，像催备。催备不怕；怕的是牌一响，手就飘。手不飘，渣才正；渣正，三日才算踩实。

第三日黄昏，簿墨未干，棚里却已起碎语：「丑丹师真敢写师？」碎语他不接。接了像抬杠；抬杠，管事笔尖一转就是投诉。投诉一落，三日白试。

韩泥把半文稳进袖底，袖底凉，钱不多，规矩倒沉。沉得好——沉着提醒他：试水不是开铺，开铺惹眼，试水要藏。

药摊婆临收摊过来，塞他一撮干青芯：「赠的。不为丹，为鼻子。」

「不收礼。」韩泥摇头。

「不是礼。」婆子瞪，「是谢。谢你没让我炉子焦。」

韩泥停半息，收下青芯，仍道：「半文够。这撮记人情。」

「记就记。」婆子走了。人情轻，轻到不压坛；不压坛，七笔才排得开。

铁无言立远，终于走近一步：「师字太大。」

「大才试得出骂声。」韩泥说。

「骂声够了就改小。」铁无言道，「小了，赖福盯不紧；盯不紧，耳才闲得听条。」

「改。」韩泥应，「今夜改。」

夜里他果真用炭笔把「师」涂淡，旁注小字「嗅」。字丑，丑得像他脸。丑不要紧，丑才像末席该挂的。

邻床挤过来看：「这不就不叫师了？」

「叫嗅。」韩泥说，「嗅能活命，师能招打。」

「你怕打？」

「怕连坐。」韩泥道，「连坐了，清栏的人就少一个。少一个，小兽先遭殃。」

少年不说话了。韩泥把木牌用布包好，包进筐侧。包着不是藏奸，是藏锋——锋露早了，刃口先崩。

沈枯芽又递来半句：「坡下灯……还亮。」

「亮着好。」韩泥说，「亮归亮，别传。传了，她麻烦。」

「我不传。」芽儿点头。

韩泥看他一眼：「你也别逞能守潮。潮来，跟在我后头，别跑前头。」

「嗯。」

「嗯不够。」韩泥道，「要记：角漏喊人，人不瞎冲。」

芽儿应了两声才走。韩泥这才按坛。坛温稳，稳得像三日没翻车。没翻车不是运气，是半文不加价、不加戏、不接石。不接的东西多了，活路才宽。

赖福丑时踹过一脚席角：「三日再议？再议也滚不出正经丹师！」

「不滚。」韩泥说，「滚的是渣。渣净，栏清。」

「嘴硬。」

「嘴短。」韩泥道，「短才少祸。」

赖福被噎住，骂了一串走了。走后廊下只剩风。风里仍有兽鸣远响。响一声，他心口那根绳就紧一分——紧着催备，不催他逞能。

他把青芯用布一裹，塞进暗袋。袋里已有渣种，种旁多一撮人情。人情不炼，只记。记着，往后还得起；还不起，就别先把「师」字吼出去丢人。

廊下更鼓又敲一记。他蹲着补袖线，线脚丑，补得结实。结实了，明日挂「嗅」字才不像要饭棍晃荡。晃荡招笑；招笑不要紧，招投诉才致命。投诉一落，三日试水就成笑话。

老杂役路过叹：「丑丹师也敢写？」

「写过，改了。」韩泥说，「改成嗅。嗅能填半文。」

「半文够买命？」

「够买耳和鼻。」韩泥道，「命还靠栏上那双手。」

他把改小后的牌竖在膝上，指腹摩过「嗅」字木刺。刺扎一下，醒一下：试水过了，不是成了；成不成，要看到兽潮后还能不能把鼻子摆正。

天边白一线时，他仍坐着。坐着把两样分开：师字是试，嗅字是活。活要留下，试可收起。收起不是怂，是藏——藏到坊市肯续，藏到兽潮条落到眼前那天。那天还没到，今夜先把鼻子养稳。鼻子稳了，明日改牌，后天续摊，潮来也不至于先倒自己。
