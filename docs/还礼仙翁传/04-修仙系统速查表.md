## 修炼公式

```
修为 ≈ 凡修(15%) + 缘修(70%) + 战修(15%)
缘箓系数 = 1 + 转数 × 0.08
有效战力 = 境界基础 × (1+器灵×0.1) × (1+缘箓×0.08) × 因果系数
```
