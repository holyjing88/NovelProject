# 第七十四章 暗修炼气二层

丑时后第三夜。兽栏夜值刚做完，韩泥没睡实。

睡实了，秘密藏不住；藏不住，就谈不上暗修。他盘坐床沿，席盖严，坛不露。露了赖福来，管事来，煞骨门旧债也来。油灯豆爆一下，又稳了。稳像掌按测灵石——石凉，手不抖；坛温，心也不抖。他揭席半寸，掌贴沿。沿温一线贴进掌纹那路，路微颤，颤像一层在呼吸。

呼吸在暗处，不在公示。公示仍写「仍末」，掌上写「一层」。邻床鼾声如雷。雷不怕，他鼻下静一息。粟壳香淡，淡不要紧；要紧是香一个字都不落。气从鼻入，沉腹，沉到掌纹。掌纹亮半分，亮不像光，像认——认韩氏守瓮血，认瓮穴灵穴。

一层境里，气转一周，周转涩。涩才真。真在凡骨壳，真在腌菜坛沿，真在五十九年擦沿的手。他低声道：「二层在远，手在近。」

不在舌头上，在掌里。舌一响，香就飘；香一飘，手就滚。滚了，玉牌白接。气再转一周，周转快了半分。快也要稳——稳了才不破席，不叫邻床醒。坛底极轻一颤，像臾墟残念应一息。应完又寂。寂里，气冲关隘。关隘薄，薄像席布。冲半寸，退半寸。退了再冲。

第三周，气聚掌纹，掌烫，烫得踏实。踏实了，关隘裂一线。裂声闷在骨里，不在屋里。二层开了。开不像雷，像瓮唇吐了一口白气——气冷，冷里带药香。药香淡，淡也是境。韩泥屏息，独眼不眨。眨了，境就散。散不得，散了二层白开。气稳在二层，稳像粪铲落栏——落得实，才不飘。

他收功，收得像盖席。盖严，沿净，坛不露。不露，像腌菜坛，不像邪。掌纹那路比昨夜厚半分，厚在暗里，不在册上。册仍写仍末，掌写二层——写给自己，不写给别人。

邻床翻身，眯眼半息：「你……身上咋热？」

「编筐扎的。」韩泥声平，「看错了。睡。」

小杂役哼一声，又睡实。睡实了，二层才藏得住。更鼓将尽，天边白一线。他廊下练半刻鼻下静，静到只闻辛淡，不闻辱。二层在掌，一层在忆里，两层不能搅。搅了香飘；香飘，煞骨门就嗅得到。

铁无言脚步远着，停半息，声更低：「境升别叫。叫了，像求。」

「不叫。」韩泥应，「油灯豆。豆爆完就灭。」

赖福盯半息，终没掀席。掀不得：沿净，坛不露。晦气遮瓮，遮住了，二层才稳。卯时兽栏仍腥。腥不怕，二层在掌，铲下更稳。稳像踹台那一掌——掌在，境在，路在。管事弟子巡栏，记一笔：「丙九韩泥，清细。」

赖福讽远：「二层也末等。」

韩泥不接。舌头留来辨鼻。铁无言立栏边，声低：「二层配渣，渣才说话。说话了，药山才走得深。」

走得深还在后头，脚先在近处。近，才暗修，才养嗅。刘婆端粥，仍多半勺：「昨夜又守丑时？」

「守了。」韩泥说，「沿净。」

「沿净怕什么。」刘婆哼，「饭稳，恩不断。」

韩泥点头。恩在粥里，也在二层境里——境轻，记重；记重了，才接得住药山。秦霜远廊过，木鱼不敲：「心别邪。邪了，鼻下香飘。」

韩泥不答。心里七笔各走各位：愧线、臾墟恩、二层境，走长了才还得了叶汤。他摸怀玉牌，凉进骨；掌烫在二层，不在众目。众目看灰线三尺，掌里多一尺暗线。

白织月远廊过，袖风冷：「杂役也守丑时？守出邪法？」

韩泥不抬头：「守沿。沿净，不叫。」

白织月唇线绷一息，走了。少一句嘲，小胜。午后药渣场灰堆高，风里一点香。韩泥分拣，指稳，二层气走指梢，指梢温半分，辨渣更细。铁无言走近半步：「二层别漏。漏了，渣场不收。」

「不漏。」韩泥应，「鼻下静。」

铁无言又指两堆灰：「这堆偏腐，这堆偏辛。辛沉，别当浮。」

韩泥分拣分出，才点头。点头轻，记重——记二层配嗅渣，配活，配暗修。他对坛低声：「二层开了。不叫。坛在，人在。」

坛不应字，只温。温像爹的手又搭上来。爹说过：「坛别扔。」扔不得，扔了就无根；无根，二层白开。更鼓沉，他合眼不睡实，掌纹那路仍转半周。转多了邻床就醒；醒不得。独眼平：「二层在掌。三层在远。药山在明日，渣里活。」

夜禁脚步远，风里「煞骨门」三字闪一下。闪一下就够——沿净，壳才像腌菜坛；像腌菜坛，魔教才嗅不准。二层是境，境是路。路通向药山，通向废渣，通向丹路开那一日。过坡下测灵石时，石凉线灰，公示过的灰线仍旧。公示不要紧，掌里有二层，手要紧。

同批少年悄问：「你身上有时热，邪不邪？」

「壳还净。」韩泥说，「净就不叫。不叫，就不邪。」

少年懂「不叫」两字，闭嘴了。他廊下再练鼻下静，粟壳香里多一丝药苦，苦像药山渣。渣不怕，鼻下稳才要紧。稳了，明日药山手才不抖。钱戾衡名在玉牌里，在三年约里——约还在，二层先藏住。藏住了，才接得住捡渣，才接得住再嘲。

丑时前第二夜，他未敢再冲关。冲急了邻床就醒。他只让气在二层转半周，半周稳像粪铲落栏。坛底极轻一颤，臾墟残念应一息：「……二层在暗，暗在沿。沿净，煞骨门嗅不准。」

韩泥心里记下：进在丑时，进在席下，进在沿净处。管事夜巡验棚，灯照席角，见净沿不见坛。不见也好，净才要紧。沈枯芽远廊过，步子轻，像没看见。韩泥也不近——近像求；不求，只记半块灵石恩在还路前头。

散队后过丹阁侧廊，廊扫痕直得像秤。他不踩灰，肩稳，筐空——筐空像只清栏，不像嗅药。同批少年掩鼻：「你身上咋有时热？」

「筐刺扎的。」韩泥说，「睡吧。」

少年退半步。应完他不叫。不叫，才守得住暗修，守得住捡渣那一日。管事贴条，条丑字真：「杂役夜禁。」禁在，坛在，瓮温不外泄。

二层开后第二日，坡下石阶霜化。韩泥过阶，足底气沉，沉得稳——稳像踹台按实，稳像一层在忆、二层在掌。管事列队喝：「冬深杂役，手抖记名！」韩泥立队末，独眼平，手不抖。

不抖是小胜：胜在肩，不在舌。赖福立队首，眼细：「丑杂役身上热，二层？邪！」

韩泥不接茬。接茬饿一整排。他低声道：「油灯豆。沿净。」

管事冷声：「赖福，再嗓，克麸皮！」

赖福哑了。韩泥掌按木牌，牌凉掌烫——烫在二层，不在众目。兽栏清栏铲下稳，二层气走臂，臂不僵。臂不僵，粪铲才不偏。小兽角里咳一声，他换草，鼻下静，静里闻见药回一线。药回像嗅诀将芽，像渣场辛沉底。

铁无言立栏边：「二层别漏。漏了像破境示人。示人，谣来。境在席下。」

刘婆端粥多半勺：「二层别飘。」

「不飘。」韩泥说，「飘了，恩就断边。」

刘婆不懂「断边」，懂「不飘」，哼一声走了。午后药山边门仍只捡渣。捡渣手稳，二层气走指梢，辛腐分得更清。管事验筐，记「捡净」。黄昏编筐，藤刺扎掌，他偏手。偏手像守沿——沿净，血不来；不来，坛才像腌菜坛。

二层稳后第一个上午，他过测灵石再看一眼。石凉，灰线仍三尺。三尺在众目里，二层在袖里。袖里烫，袖外平。平才像「仍末」。仍末遮得住，钱戾衡那枚玉牌才暂时只凉不烫。

同棚少年嚼着冷馒：「你昨儿发热，不会真要破境吧？」

「发汗。」韩泥说，「编筐编的。」

「编筐也能发热？」

「能。」韩泥把馒皮撕开，「手重了就热。热完，人还是人。」

少年笑一声，当笑话听了。笑话正好——笑话比谣轻。谣一重，掀席的人就来。韩泥把笑当雨布，遮住二层那口新气。

午后他在药山边门外站半息，只看，不进。界碑字浅：「杂役只捡边堆。」边堆灰冷，深堆雾起。雾起处有活药香，香诱人。他鼻下一静，把诱摁回去：「不进。」

管事弟子从碑旁过：「看什么？」

「看界。」韩泥说，「界在，手才在。」

「识相。」管事点头，「明日正式捡，别越线。」

「不越。」韩泥应。

不越，是给二层留命。命留住了，嗅芽才长；芽长了，深行那一步才不是送死。他摸怀木牌，「仍末」两字硌得掌心疼。疼得好——疼提醒他：境在暗，名在末，末才能活到药山开筐那一日。

温，像把二层摁进掌纹那路。路在，就不滚。更鼓沉，气转半周，半周不像飘。飘了，煞骨门就嗅得到。独眼平：「二层稳了。嗅芽将起。明日药山，仍只捡。」
