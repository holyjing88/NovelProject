# 参考小说剧情调研

> 本文档仅作创作参考与题材分析，不直接引用原文内容。最终游戏使用原创剧情《万古守灯人》。

---

## 一、《我得长生后》（驿城长豇豆 · 起点中文网）

### 核心设定

| 维度 | 内容 |
|------|------|
| 主角 | 陈可观，十八岁贫家子，为活命净身入宫 |
| 金手指 | 《造化长春功》——补缺长生，寿元无限（仍可被杀） |
| 力量体系 | 精神力九品至神品；兼修武道、儒道、佛门、剑修 |
| 世界观 | 架空大乾王朝，东莱-大乾-北魏-南疆四极并立 |
| 主题 | 人间烟火气，最抚凡人心；长生不是逃离人间 |

### 剧情脉络（两卷结构）

**第一卷：皇宫里的老太监（1-105章）**
- 开元元年入宫，净身濒死，旧书楼获功法
- 旧书楼擦灰读书，跟李公公学生存之道
- 结识小太监高欢，为其"撑伞"成为初心
- 鬼市搏命，初窥九品精神力感知
- 与太平公主市井同游，卷入宫廷权谋
- 栖凤殿前截停公主出嫁，情感高潮

**第二卷：军队里的定心丸（106-211章）**
- 出宫从军，成为军中精神支柱
- 赵凌月女帝改革：减田赋、开商路、设监察使
- 北伐布局十三年，对抗定国郡王霍无敌
- 南疆煞气洞苦修（主观三千万年，客观三年）
- 擎天峰突破超品造物主境
- 结局：携妻归故里，敲响童年家门

### 可借鉴的叙事元素（非照搬）

1. **时间变量**：长生者用三十年练一招，百姓一日三餐不可断
2. **底层视角**：太监、青楼清倌、伙夫、酒楼打工剑仙
3. **力量代价**：精神力反噬需付出代价（如"用吾须发，换尔性命"）
4. **群像塑造**：高欢、齐浩然、霍千机各有独立弧光
5. **喜剧外壳+严肃内核**：荒诞对话解构权谋，笑着流泪

### 重要配角原型

- 李公公：旧书楼管事，以懒散为铠甲的清醒守护者
- 高欢：底层小太监，务实热望，主角人性温度参照
- 赵凌月：女帝，制度革新者，与主角理念共鸣
- 霍无敌：功高震主的悲剧武将，非脸谱恶人
- 兰雅/景梦：青楼女子自我成长，拒绝猎奇叙事

## 二、《老人修仙记》（李大开版 · 凡人流）

### 核心设定

| 维度 | 内容 |
|------|------|
| 主角 | 李大开，五十岁开始修仙 |
| 风格 | 凡人流、无系统、杀伐果断、慢热、搞笑 |
| 特点 | 体内阳气澎湃，需提防仙子觊觎 |
| 力量体系 | 传统修仙：炼气→筑基→金丹→元婴→化神 |

### 剧情脉络（卷结构）

**第一卷：杂役弟子**
- 开局成为"老爷爷"（被误认为老者）
- 青楼舌战、讹银子进城
- 七星宗选拔，成为杂役弟子
- 黑风谷采药，获修士尸体与储物袋
- 坊市倒卖丹药，宗门内斗

**中期：宗门风云**
- 七香院修行，师姐秦茹、师兄乾青柳
- 天尸门威胁，五行炼尸、七星宗大战
- 冲击筑基，千机塔探险
- 上古炼气士遗迹，获千机塔和炼火诀

**后期：跨界冒险**
- 化神中期，穿越法则封锁
- 千机塔九九八十一难
- 界面漩涡，多界穿梭
- 金色飞剑、仙元石、芥子空间

### 可借鉴的叙事元素（非照搬）

1. **高龄起步**：打破"少年天才"惯例，老年主角的逆袭
2. **资源博弈**：坊市交易、丹药倒卖、储物袋争夺
3. **宗门生态**：杂役弟子、内门斗争、派系倾轧
4. **杀伐果断**：吃里扒外者杀、借刀杀人、金蝉脱壳
5. **幽默调剂**："老李你要老婆不要"、左眼皮跳财

## 三、其他同名作品辨析

| 作品 | 作者/平台 | 核心差异 |
|------|-----------|----------|
| 《老头要修仙》 | 起点中文网 | 六十八岁采药人张进，万灵图录，多界穿越 |
| 《老头修仙记》 | 番茄小说 | 徐大爷渡劫失败，万倍返还系统 |
| 《老头修仙记》 | 江怀瑾毛利兰 | 名柯同人，二次穿越 |

**结论**：用户所指的"老头修仙记"最可能为李大开版《老人修仙记》，因其在凡人流老年修仙题材中影响力最大、章节最完整（1200+章）。

---

## 四、两部作品的共性主题

1. **逆龄/长生起点**：一个从残缺中获得长生，一个从老年开始修仙
2. **凡俗扎根**：力量体系与社会肌理深度绑定，修仙要银子
3. **时间哲学**：漫长生命中的选择、孤独与坚守
4. **底层逆袭**：从社会最边缘处向上攀爬
5. **人间温情**：在杀伐与权谋中保留对普通人的关怀

## 五、原创化方向（版权安全）

### 必须替换的元素

- 所有角色姓名、地名、功法名称
- 具体情节桥段（如净身入宫、栖凤殿截婚）
- 力量体系数值与命名
- 经典台词与梗

### 可保留的题材方向

- 老年/长生主角视角
- 凡人流慢热成长
- 人间烟火主题
- 力量有代价的修行观
- 市井+宗门+禁地三段式地图
- 选择驱动的叙事结构

### 原创作品命名

**《万古守灯人》**（Legend of the Elder Cultivator）
