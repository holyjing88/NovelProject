## 六、扩写状态行（可选 · 丹/阵章）

破境章、天试章、合阵章可在八要素后追加：

```
丹道【X品·炉温稳/险】 · 阵道【X阶·阵名】
```

**示例（第 500 章）**：
```
丹道【四品·结丹丹成】 · 阵道【四阶·聚灵护关】
```

**示例（第 404 章）**：
```
丹道【二品·丹阵定魁】 · 阵道【四阶·天试丹阵】
```
