# 第四十章 掌茧裂血

入门两月，韩泥须发更白，背更驼，独眼却仍利。

利不是天生的，是粪里、渣里、筐里熬出来的。同批杂役走了两人，他不笑，只把日子一寸寸过。末席站惯了，站惯就不晃，不晃才配挨测。

天未全亮，丙九棚里已踩出碎步。有人搓手，有人咬唇。韩泥坐在末铺，把指茧在裤缝上蹭了蹭——茧厚，才像还能握住活。

管事唤集合，嗓子硬：「凡骨杂役，冬测备检！不是测灵根，是测手——手稳的留，手抖的滚！」

韩泥站末排，呼出一口长气，气贴着肩沉下去。赖福阴笑着贴过来：「丑杂役，别抖。你抖，丙九都跟着丢脸。」

韩泥不接嘴。嘴接得赢一时，手抖了就全输。旁边少年搓着衣角，小声抖：「我搬铁石，手肯定抖。」

「抖就先吐气。」韩泥低声道，「气稳了，肩才吃得住沉。」

少年照做，肩松了半寸。管事扫过队伍：「冬测备检只看手，不看嘴。嘴硬的，自己去搬第三块！」

有人想出列辩一句，被同伴扯住袖子。扯住也好——辩赢了嘴，搬不赢石，仍是滚。

棚外风硬，吹得人眼干。韩泥眨了眨独眼，把视线钉在脚下。脚稳，手才跟得上；脚飘了，石就先笑你。

同批里有人骂：「凡骨还测手？测完不还是滚？」

「滚不滚，看手。」另一人顶回去，嗓子却虚，「你抖了，骂也没用。」

韩泥听见，不掺。掺了像帮腔，帮腔容易被赖福记名。他把肩再沉半寸，沉够了，才跟着队伍进场。

测手场是青石地，地上划着白线。搬石过线就过，石沉得掌茧直疼。韩泥心里清楚：疼是本钱，不是丢脸。第一石抬得稳；第二石指缝迸出血丝，他也不哼；第三石换肩，驼背反而更吃力，仍稳。

场边有人小声数：「过了……又过了……」数到韩泥，声音忽然矮下去，像怕晦气沾身。他不理，只顾把气沉进腰。

管事又掷来一块小石，石轻得反常：「再一块。这块抖了，照样滚。」

石轻更难——轻了，手心里找不到分量，肩就容易飘。韩泥先把脚踩实，脚实了，才抬掌。掌茧贴石，贴实，不抓狠，也不松。

韩泥吸气，气沉到腰，石从白线那边过去，干净利落。管事「嗯」了一声：「韩泥，过。」

旁侧少年手一抖，石落半寸。管事喝：「滚！」

少年哭：「我家里还有人……」

「家里有人，手也不能抖。」管事冷着脸，「抖了，宗门不要。」韩泥不笑，只把「滚」字离自己多远记进心里。少年被架走，哭声远了，他仍站末排，不回头——回头手易软，软了就白过。

刘婆在灶边瞧见他过了，舀粥多半勺。半勺不重，却热得真。她低声道：「手裂了就裂了，别空攥。攥出血，备检白过。」

「记着。」韩泥点了下头，不多话。

管事掷木牌过来，牌上刻着「冬测备检过」——字薄，他也收下。赖福又阴笑：「过了又怎样？凡骨还是凡骨。」

韩泥举起掌，掌茧裂口露出新皮，新皮还嫩：「骨凡，手不凡。手不抖，粪不洒，渣不漏，筐不塌。」

同批里有人倒吸一口气。赖福嘴唇动了动，没敢再尖。管事补一句：「丙九韩泥，手稳，记名。」

有人嘀咕：「凡骨也能过？」

管事喝：「手稳就过！嘴稳不算。嘴稳的，去丹房门口试，看渣漏不漏！」

少年凑近问：「你怎么做到的？」

「做。」韩泥说，「粪做，渣做，筐做。做多了，手就不抖。」

「做两年？」

「做一天，也要做稳。」

又一人没过，跪下求：「再试一次！」

管事一挥手：「冬测备检不是人情。手抖，明日滚。」

赖福啐一口：「滚一个少一个晦气。」

管事盯他：「你也去搬石。抖了，你滚。」

赖福脸刷白，闭嘴了。场子里静了一息，喘息声乱。韩泥把木牌塞进破袄，牌角硌着心口，硌着也好——硌着才记得自己还站得住。

散场时，有人故意撞他肩：「晦气也过？」他只侧身让开，不开口。开了口，像对骂；对骂赢不了测，还耽误傍晚的粪。

管事收牌匣，冷冷补一句：「过了别飘。飘了，十日测灵备检照样滚。」

「不飘。」韩泥应得短。短了，才把力气留给手。

夜里他编筐补末单，藤条炸刺进指肚，血珠圆圆的。他甩手，血甩到破席上——席盖严，脏不到旁人眼里。

邻床少年醒一瞬：「你又流血了？」

「编筐末单。」韩泥道，「单完，手才稳。手稳，往后备检才不抖。」

「测灵备检，凡骨也测？」

「测。测不过，滚。」

少年又睡了。韩泥才低声补：「别学我忍。学我稳。稳了，才少挨滚。」

他摸坛沿，沿还温着，像应着他的茧，不应着血。外头雪意压来，雪未落，风先冷。棚柱一响，像有人夜里路过又不敢停。他不做声，只把席角掖实。

邻床又迷糊问：「你掌流血，还盖坛席？」

「盖，是防人掀。」韩泥说，「掀见了，像晦气。晦气了，麸皮账盯着。」

少年不懂坛，懂「盖」，翻身睡了。韩泥把木牌贴在心口，牌凉着，心里却暖——过了冬测，不是灵过了，是手记住了。

赖福门外又啐：「过了也是凡骨！」

韩泥不应。舌头留给以后辨香，不浪费给赖福嗓子。管事远处喊：「丙九韩泥，测灵备检名单，你在末。别抖。」

「不抖。」他应得平。

邻床睡眼迷糊：「测灵备检，凡骨有希望？」

「希望不在灵，在手。」韩泥说，「手稳，名单就有名。有名，就不被滚出丙九。」

丑时末，他把破席角又掖实一遍，像把「不抖」钉进掌纹。天将亮，掌茧裂处结了痂，痂硬，按石不疼。不疼，才敢想后面的日子——备检在远处，今宵只睡半息。

半息里他仍覆席盖坛。盖严了，血不往沿上爬。沿净着，后头凡活才接得住。刘婆护粥，粥稀却热，他心里把这半勺另记一笔，跟泥岗的恩挨着，不搅成一锅浑汤。

卯前，管事门外贴条，字硬：「冬后测灵备检，十日后。丙九记名者到。误者，滚。」

十日帖在柱上，像一根钉子。同批有人伸脖子看，看完又缩回去。韩泥看一眼，记下日子，转身去兽栏拎铲——日子记在心里，铲落在手里，手才不飘。

赖福远远骂一句，没人接。骂声像雪屑，落不到茧上。韩泥铲下第一下，实。实了，才配挨那十日。

午后他又去渣场半刻，把香撂齐。齐了，管事路过只瞥一眼，没骂。没骂，便是多喘半口气。傍晚回棚，他把掌茧对着灯看——裂口结痂，痂硬，硬得像提醒：疼过了，别松。

邻床小声问：「十日很远吗？」

「远。」韩泥说，「远也按天熬。熬一天，手就稳一天。」

「稳了就能测灵过？」

「灵末不要紧。」韩泥道，「手末，才真滚。」

夜里他又按石练三次。石不大，练的是气沉肩。邻床翻身问：「你还练？」

「练。」韩泥说，「练到手不记抖，抖才离得远。」

「管事不在，你图什么？」

「图十日帖。」他道，「帖在，名在；名在，才不被滚出丙九。」

刘婆夜里又塞半块干饼过来：「吃。编筐手不能空着。」

「记着。」韩泥接饼，饼硬，硬也顶得住饿。饿顶住了，茧才长得住人用。

门外雪意又近一分。帖角起了一层白霜，霜化开，字更清楚。韩泥独眼平平，手按在膝上，低声道：「十日到了，仍不抖。不抖，才熬得到下一关。」
