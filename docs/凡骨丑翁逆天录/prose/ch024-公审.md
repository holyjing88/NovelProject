# 第二十四章 公审

公审日，霜气未散。

韩泥早起编半只筐，藤条浸软绕两圈，指节糙。浸藤条的水凉，凉得像公审前的夜——夜凉不要紧，要紧是手不能抖。筐未成，里正锣已响三遍，响得全村往祠堂聚。他洗手上路，独眼低垂，坐末席——今日不辩，只坐。坐，是给叶丫头留台；台在，话才立得住。

供桌挪开，空地中央，里正站着，赵绣春站着。沈枯芽娘咳在人群后，芽儿拽她袖。张麻婆骂：「挤什么挤！」骂完，又瞥韩泥一眼，像挡在娃前头。

赵绣春穿新袄，声脆：「今日公审，审晦气，审牵扯！」

周嫂拍腿：「叶家寡妇往晦气翁门缝送汤送姜，药铺生意都淡了！」

马春花嗓尖：「沈家芽儿也送粥，谁沾晦气翁，谁连坐！」

里正抬手：「一个一个说。韩泥，你可起身？」

韩泥不起。「赵家娘子，你先讲。」

陈姑在人群后拢紧棉袄领，低声问邻人：「公审辨什么？」

邻人答：「辨诬，不辨生意。」

赵绣春下巴抬：「韩泥五十八岁绝户，麻脸跛足独眼。叶青禾寡妇半夜送汤送姜，写『趁热』，不是邪是什么？不是勾引是什么？」

周嫂帮腔：「寡妇守节，守到门缝送汤？沈家芽儿学她，学坏了！」

马春花啐：「晦气翁编筐，叶丫头送汤，喝了的倒霉！」

里正敲桌：「谁再嚷，记名。」

赵绣春更来劲：「韩泥，你说话！你敢说你没勾她？」

韩泥嗓哑：「我勾不起。」

四字，平，沉得深。赵绣春又尖：「你勾不起，她上赶着！」

「是恩。」

声从侧席来，不高，却清。叶青禾站着，没跪，没哭。蓝布袄补过，袖口线密。她眼不红，红的是腮，冷风吹的。

赵绣春眉一扬：「你送晦气翁，全村跟着晦气！药铺客人绕路走，你赔得起？」

叶青禾看她，不躲：「赔不起，我也不悔。天冷，门缝大，他冻不死也会病。病死了，谁抬棺？你抬？周嫂抬？」

周嫂脸一白：「关我什么事！」

「晦气事你们抢着做。」叶青禾声仍不高，「韩泥抬棺多少年，抬到你们家，你们没拦。拦的是汤，不拦的是命。」

里正敲桌：「赵家娘子，讲证据，不许骂街。」

赵绣春撇嘴：「证据？全村都看见她门缝送汤！」

叶青禾转向里正：「里正，送汤是白天门缝递，没进门，没过夜。赵家娘子若说有邪，请她拿出邪来。」

卖盐老汉嗓粗：「去年冬，我娘病，叶丫头送过两帖药，没收钱。」

赵绣春眼一瞪：「你帮他说话？」

老汉把脸别开：「我说实话。」

里正敲桌：「再嚷，记名。」

叶青禾转向村人：「诸位，你们谁家没饿过？饿时谁递过一口？韩泥五十八岁绝户，没人给他递过，我递了。递汤，不是邪，是人该做的。」

她目光扫过檐下：「赵绣春，你饿不着。周嫂，你饿不着。马春花，你饿不着。可泥岗村还有饿着的——嘴能杀人，汤能活人。我选活人。」

马春花啐：「寡妇该做？你守不住名节——」

「名节在心上，不在嘴上。」叶青禾截住，「赵绣春泼脏水，名节干净？」

沈枯芽忽然抬头，声小：「汤……不邪。叶姑姑送药引，不是晦气。」

马春花瞪她：「娃懂什么，闭嘴！」

里正敲桌：「让娃说。」

沈枯芽娘拽她袖，芽儿却站直了：「韩爷爷狼口救我，臂流血，没要谢。邪人不救人。」

张麻婆哼一声：「娃都懂，大人装不懂？」

赵绣春冷笑：「你跟绝户翁牵扯，是不是图他将来成仙回报你？」

叶青禾摇头：「图他还我一碗热汤。汤我送了，他还不起，我也不要他还。可有人不许他活，就是不许我做人。」

她目光扫过韩泥，不黏，不媚：「汤是热的。」

韩泥独眼抬了一下，又垂。嗓哑，慢，只一句：「我记着，以后还。」

里正问：「韩泥，你可认叶氏所言？」

韩泥应声：「认。汤我喝了，恩我记着。不辩，是恩记着，不用嘴吵。」

里正问：「可曾求她？可曾许诺回报？」

韩泥摇头：「不求。许诺在心里。还，不在这台上。」

里正又问：「韩泥，你可曾逼她送？」

韩泥只答：「不曾。门缝大，她递，我接。接的是汤，不是牵扯。」

赵绣春跺脚：「他认！他认就是牵扯！」

叶青禾看她：「牵扯是你们扯出来的。我只扯一碗汤，扯够了，该断了。」

周嫂忍不住，声小：「绣春……汤若真是邪，我洗衣水早浑了。」

赵绣春回头一瞪，周嫂缩回去。赵绣春嗓拔高：「他还？他拿什么还？一个编筐的晦气翁——」

「拿命还。」韩泥说。就三字。不是威胁，是恩记着。

里正沉脸：「公审辨诬，不辨生意。赵家娘子，你可还有证？」

赵绣春唇动，扫一眼村人——村人眼神腻了，腻她泼脏水，腻她把公审当私怨。她卡住了。马春花退半步，嘟囔：「我……我也是听说的。」

周嫂低声：「汤……好像也不算邪。」

张麻婆哼一声：「一碗汤，算什么邪。我送馒，送得，她送汤，送不得？」

卖盐老汉又开口：「我娘去年冬喝过叶丫头的药，人善，药苦。苦药救命，脏水害人——这话，公审该记。」

里正沉声：「公审毕。叶氏送汤，恩在，非邪。赵氏泼辱，已过。今后不许再聚众辱人。」

里正敲桌：「记了。再嚼，连坐。」

赵绣春脸青，唇动：「里正，你——」

里正截住：「赵家娘子，再闹，记名。」

赵绣春甩袖，走快。村人散，窃语：「叶丫头说得在理。」「汤是热的，这话硬。」「赵绣春这次，过了。」

马春花退到井台边，没再嚷。嚷累了，或怕了再哑口。韩泥末席坐到最后。人走尽，祠堂空。叶青禾收供桌旁粗碗，碗沿缺，像送汤那只。她经过韩泥，低声：「公审过了，你少惹事。惹事，连累芽儿。」

韩泥只点头：「不惹。事惹我，我记。」

叶青禾停半步：「汤我送了，不图你还。可你若还，还热就行。」

韩泥独眼抬了一下：「热。记着。」

她点头，走。肩松了半寸。里正收锣前，经过末席，低声：「韩泥，心收好。恩在心里，祸在嘴。」

韩泥应一声：「祸我管不了。心我管得住。」

他按心口，叶汤那笔恩加深。坛沿一线温光，指腹微烫。贴胸「趁热」印痕浮。他低声道：「这恩，还定了。」

风过，谣在远处巷。今日，汤在心里，在人心上一寸——够活，够还，够走到省亲那日。

叶家药铺那边，秤杆又落一声，落得脆——铺子没关，谣没赢。沈枯芽隔人群喊，声小：「叶姑姑，汤是热的，我记住了！」

叶青禾回头：「记住就好。」

韩泥听见了，不答。答，费舌；舌要留来还。里正收锣，对人群道：「公审过了。谁再聚众辱人，先记名，再报县。」

陈姑在人群后拢紧棉袄领，低声对邻人：「公审辨诬，不辨生意。辨对了，秤才准。」邻人「嗯」一声，像也记了。

赵哑口，锣余韵在村；韩泥只「我记着」——定了，一半在名节，一半在省亲那日；那日远，烫先不能凉，凉了，白写。

韩泥起身，腿瘸，走回村道。担空筐过叶铺，不近帘，只望秤杆——杆落，心便安一寸。公审明爽，爽在叶青禾一句「汤是热的」，爽在赵绣春哑口。

回屋，他仍编那半只筐，藤条绕完，绕实了，像把公审那日也编进筐里。坛腹微温一丝，应「热」字，不应「辱」字。丑时，叶铺秤声又起，脆，短，钉住心口那笔恩。

谣在远处巷，今日烫在心里——心里烫，路就不绝；路不绝，才还得上省亲那日。
