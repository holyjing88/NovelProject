# 第三十三章 公平买卖

秦商言连夜下山，驿站在山脚，马未卸，先吩咐下属。

「莫长老送的茶，」她道，「比灵石贵。记入百宝阁恩心，不记入买卖价。」

下属愣：「管事，无价的礼……」

「所以贵。」秦商言截话，「公平买卖，卖的是货；恩情，卖不了。」

她翻那卷东脉调动简，指节轻叩纸面：「三日后佯攻，七日后夜袭——青岚门若只守不攻，田还在，命没了。」

下属问：「若情报卖了别人……」

秦商言望青岚方向，笑：「卖就卖。商道是商道，恩是恩。各算各的恩。」

下属又问：「若赤焰夜袭成真，商言赚不赚？」

秦商言笑深：「赚。赚在青岚若赢，百宝阁有路走；若输，商言也有恩可还——恩，是另一条商路。」

马嘶，夜风腥，像东脉的云已低头。驿外弟子偷听，偷听里惊，惊茶饼竟换夜袭简，惊百宝阁管事竟说「亏本乐意」。

驿卒悄问：「管事，这简卖了赤焰，青岚知道了……」秦商言摆手：「知道了，才显恩。恩显了，路才宽。」她翻身上马，马蹄溅泥，溅得像把公平与还礼先踩成两条道。

---

同一夜，莫长春院中。

他展简，烛火映字，像映一张将到的刀。佯攻、夜袭、妖与田之间——秦商言没卖假情报，这礼还得干净。

柳青鸢遣人来问：「师兄，可是秦氏简？」

「是。」莫长春道，「宗主主战，简只是让战打得明白。」

来人退。廊下弟子窃语：「长老手松，连百宝阁都松？」

另一人答：「松的是茶，紧的是命。你看简上写什么。」

「写什么？」

「夜袭。」

窃语骤静，静里，怕意起，像三日之期提前贴到每个人心里。

霍镇山闻讯，骂声洪：「好！打得明白，死也明白！」

周德海在长老院听闻，笑僵，笑里像有什么在袖中湿了又干。

莫长春望简，笑而不辩坊间那句「长老手松」。手松给杂役棉衣，手紧给赤焰命门——世人只见松，不见紧，正好。

塔壁无新铭，袖却暖，像秦氏茶香还在。

他低声：「公平二字，商言说得对。老夫偏不爱公平，爱还礼。」

杖中竹纹一游，气音轻。

弟子围简抄本，围得像看刀，刀上写：佯攻、夜袭。有人怕，有人燃，燃的是终于明白赤焰花钱在哪。

弟子问苏念慈：「首座，夜袭丹堂危不危？」苏念慈冷声：「危。所以炉不能熄。」她把药渣拨进火，火舌舔炉壁，舔得像三日佯攻已烧到丹堂檐下。

苏念慈在丹堂听讯，炉火旺，旺得像战意也旺。她低声：「七夜丹成了，东脉也要成。」

秦商言驿中写信，写恩心，不写买卖价，写得像亏本，亏得像赚。信末加一句：「商言欠莫长老半块茶，欠青岚一条暗线。此债，商言认。」

魏无涯在檐角观简传讯，扇合，低笑：「茶比灵石贵，商言说得对。」

孙福在角隅听简讯，面色白，像听见「第七年」三字，也听见自己的旧恩仇将被翻。外门弟子把简抄成七种版本，版本里都有「夜袭」二字，像把怕字写进每个人眼里。

魏无涯传讯厉无殇，只四字：「简已至。」厉无殇阅讯，焚，笑冷：「准不准，莫长春都要接。接，才是戏。」

顾小满在杂役堂浆洗，听驿站消息，指节白：「茶换简？简换命？」杂役师兄答：「你不懂价。秦管事记恩，不记卖。」她洗了第三盆浆，盆底映出天际一线红——红像日头，也像鼓面。她把名册握紧，握紧像握紧即将摊在殿上的一页纸。

赵玄机在思过崖听风，风里有「夜袭」二字。他妒火冷一寸，忽然明白：莫长春手松给茶，手紧给东脉田埂——紧的地方，才是该守的门。

秦商言马队远去，远得像把恩心留在青岚，把买卖留在百宝阁——各算各的，才算干净。

简在案上，茶香在袖里，恩在心里——公平买卖，卖的是货；还礼仙翁，还的是命。

他将简折角，折得像给东脉事本先留个缺——缺不填，表就厚；厚了，心就薄。

莫长春念：「东脉事本，该翻开了。周长老的表，该有人问问——第七年，谁守田。」

简在案上展开第二遍，他以指节轻叩「佯攻」二字，声轻：「三日鼓，七日刀。鼓响，全门别睡；刀落，霍老头先叫。」霍镇山在远处听见，骂声起：「叫就叫！镇妖司不怕夜！」

同一夜，长老会急议，柳青鸢指节叩案：「简既至，主战不必再辩。」周德海笑僵，令旗在指间转，转得像信使已在后巷候过三次。孙福在角隅面色白，白像听见自己的克扣将被翻。

茶饼入驿那夜，简上「佯攻」二字像鼓，「夜袭」二字像刀——刀还没落，鼓先响，响得全宗睡不着。

莫长春袖空，一念收塔，杖落掌。塔壁金铭不显，他心里却记下一行：商言记恩，青岚记田——两条线，终将缠到殿上。

数日后坊市开市，百宝阁一棚先搭，棚上悬匾：实价不赊。

公平买卖那日，客堂里算盘响，响得像给全门听价。

柳青鸢声稳：「买卖公平，门才稳。」女修笑：「实价成交，下回还来。」她目光掠过赵玄机腰牌，掠过像掠过一条未落的价。

赵玄机铁牌在腰，笑浅，笑里空一线——价若硬，私情就软；软了，就踩不实。他插价半寸，半寸落空，脸白里仍像藏下一谋。

苏念慈不进门，送汤至廊外，汤温，温得像怕算盘进丹堂。她低声：「公平公平，别传成辱。价稳了，丹堂七夜也稳。」

顾小满声平：「价记下了。不赊，不欠。」册上添一行：坊市·实价·不赊。她低声：「东脉事该摊了——摊在殿上，不在嘴上。」

围观弟子客堂外屏息，比划定身三息——比试不赌价，赌「实」字谁先走样。有人赌赵玄机再插价，有人赌灰袍袖中会走出一件「实」字样的旧货。算盘每响一声，围观里便静一分，静得像等东脉事本提前翻页。

成交后，算盘响第三遍，响像给全门听：实价不赊。秦商言收摊前，将一枚空茶饼盒搁在案角，盒轻，像提醒青岚：价可以实，线不能断。她笑：「茶饼走了，恩要来。」飞符告知：「价平，恩在。」

莫长春回符四字：「田埂先问。」袖空离坊，杖点地，点得像给公平盖印——印不重，重的是下一本表将摊在殿上。

坊市夜雨初歇，石板映灯，映得像把「公平」二字也映进水里——水里晃，晃得像东脉事本将起的浪。赵玄机立坊市口半柱香，像等下一局——局不在价，在统。

秦商言收摊时，将茶饼盒底的暗线符折进青岚名册——名册无字，名册有路。她朝塔顶方向略一颔首，颔首像还莫长春半块茶，半块茶后面还跟着半条田埂。

霍镇山闻讯，声洪：「买卖公，战也要公——公在守田，不在让表！」周德海在长老院听闻，令旗转了一圈，仍不落，落得像等殿上再问魂。

魏无涯在檐角瞧这一幕，扇骨轻敲掌心：「价平了，恩才厚。」扇合，笑浅，像看见争点已从茶移到了表。

弟子渐散，散里仍回头瞧一眼灰袍袖空——瞧得像瞧一眼九月还能走多久。有人低声：「恩要来了，先听鼓。」

买卖响在坊市，响得像给东脉事本，先敲了一声开门鼓——鼓后，便是殿上问魂。
