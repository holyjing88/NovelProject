# 第一百二十一章 再炼散

复测前一月，韩泥闭门再炼培元散。

再炼散，废窑火匀，渣细，序对。散未成粒，成在记里——记火路，记投序，记122织月嘲在前。

铁无言再嘱：「再炼，别贪。贪，手飘。」韩泥应声：「不贪。沿净。」沿净，守坛，守122织月嘲。

邻床问：「你真能成丹？」「能。」韩泥说，「不在今日。今日再炼，成长了，粒才来。」

窑门柴挡，挡得像凡人烤薯——烤薯不要紧，炉在腹，炉在窑。窑温按焚火残诀走，走七七四十九息，息稳，香起。香起时，他鼻下静，静得像首炼110那日——那日是起点，这日是备战。备战，为复测；复测，为三年约反面；反面烫，正面手稳。

渣灰里一点辛，辛正，像108清瘴余韵。余韵不要紧，效正；正了，粒成，成五粒，粒粒泥纹浅。浅不要紧，嗅之正，正了藏陶罐，罐封泥，泥封得像腌菜坛——坛相凡，凡相蔽眼。蔽眼，赖福查不出；查不出，手稳，就不滚。

赖福踹柴，踹不开：「丑杂役闭门，偷炼！」

管事弟子来，嗅缝香，香淡而正：「培元散试手，允。赖管事，再闹，记过。」

赖福脸青，青完走。走，韩泥记——记风声里最后一压，压不过「正」字。正字在粒，粒在罐，罐在窑缝。缝窄，窄得像席严那夜；那夜守坛，这日守粒。

铁无言送外门边角渣，渣里灵材丝，丝细，够再开两炉。两炉不要紧，手稳；稳了，七粒在手，手才不飘。飘了，粒洒；洒了，丹不正；不正，刘婆恩还迟。

白织月立窑外，袖风冷：「闭门炼，像做贼。」

韩泥不辩。贼偷，他还；还恩炼，丹才正。正了，试台见；见了，钱戾衡眼风再来——来，不抖。不抖，玉牌才不虚，三年约才不白接。

「做贼也做不出骨。」白织月笑。

「骨丑，散正。」韩泥声平，「正，试台见。」

白织月袖过，过得快，快得像怕「正」字。怕不要紧，韩泥继续炼——炼第二炉，第三炉，炉炉稳，稳得像十层在腹扎根。扎根了，气冲关，炼气十一层。十一层，不声张；声张，赖福指「偷境」。

他摸玉牌，牌凉：「三年约，记着。十一层，为十三铺路。」

铺路不要紧，一步一脚；一脚稳，才配刘婆还，才配叶汤省亲前还。还了沈枯芽半块灵石，刘婆待还，叶汤头恩还在前——还在前，省亲在后，钱戾衡更在后。

刘婆送粥至柴门外，粥稀，热：「闭门炼？」

「炼。」韩泥说，「炼备复测。」

「备就别空肚。」刘婆哼，「空肚，手飘。飘了，粒洒。」

韩泥开门缝，取粥，喝尽。谢轻，恩重。恩在粥里，也在粒里——粒正，恩才正；恩正，还刘婆才配。还刘婆在130，新衣养老钱数过三遍，够补衣婆活到老。

午后，沈枯芽远远站柱角，怯，不近。不近不要紧，韩泥点头，点得像118还石那夜——夜还了沈枯芽半块灵石，日备复测；复测成，芽儿才不怕沾魁晦气。晦气不要紧，手稳夺魁；夺魁，还刘婆。

芽儿退，退得像信。信韩泥还，信韩泥炼，信恩不邪。邪了，鼻下香飘；不飘，粒才正。正了，试台见；见了，白织月说「运气」也尖不过「正」。

韩泥应声。贪多不要紧，七粒够；够了，藏怀，不示。示了，风声说「盗宗门灵材」——盗不得，渣里淘，淘了五年。五年淘出来的正，不是运气；运气试台见，正见。

第三炉成，气走圆满，十一层实。实了，再开两炉，粒成九粒，九粒够试台，够夺魁预备——预备在130，预备在还刘婆。还刘婆，恩偿静着还，还到该还的人手里，才算数。

丑时，练鼻下静。粟壳香混药香，药香正，正不要紧；要紧是静一字不落。落齐，十一层记进心里，不进舌。记进心里，才实；实了，才配122织月嘲，才配123戾衡影。

坛腹微温，应「炼」字。炼，是丑翁路；路在渣，路在窑，路在忍。忍了赖福压顶，还了沈枯芽半块灵石，再炼七粒——粒在怀，怀沉，沉得像债序。债序在，步不飘；步不飘，才配上台，才配魁，才配还刘婆。

外头风声又起，起得像试台近了一寸。近一寸不要紧，粒正一寸，手稳一寸。一寸一寸熬，熬到130，熬到还刘婆新衣养老钱。养老钱不多不少，新衣袖口多缝一层，耐磨，像36那夜补肘。

赖福骂远：「闭门也闭不出灵根！」

韩泥低声道：「灵根末，散正。正，就够上台。」

够上台，就够接钱戾衡影。影在玉牌后，牌凉，手热——手热在坛，坛应「炼」，炼到十二层，炼到复测帖至。帖至，大比影至，还刘婆在130落地。

闭门第六日，管事弟子嗅缝香，声平：「正，别示。」韩泥应声。示了，赖福说盗；盗不得，渣里淘，淘五年。五年淘出正，淘出还恩资格。资格在，十一层破；破了，不露喜。喜像认晦气；认不得，只封罐，封泥，封严。严了，七粒不散；散了，手飘；飘了，试台除名。除名，玉牌白凉，恩断。恩断，叶汤凉，刘婆新衣还不上。还不上，先在130夺魁。魁后，炼气十三，帖至山门，钱戾衡台前不滚。

闭门再炼第一日，柴挡窑门，挡严，严像62席盖。席严坛不露，窑严粒才正。正了，试台才配夺魁，夺魁才配还刘婆新衣养老钱。

赖福踹柴，踹不开。管事嗅缝香，香淡而正：「培元散试手，允。走，韩泥记——记风声里最后一压，压不过「正」。

第二炉，第三炉，炉炉稳。稳得像119十层在腹扎根。扎根不要紧，十一层破，破，不声张。声张，赖福指偷境；指不要紧，管事看手，手稳，册在。两炉成，七粒在手，手才不飘。飘了，试台洒；洒了，除名。」韩泥不辩：「骨丑，散正。正，试台见。」袖风过，过得快，快得像怕「正」字。

刘婆送粥至柴门外：「闭门别空肚。」「空肚，手飘。恩在粥里，也在粒里——粒正，恩才正。

沈枯芽远远站柱角，怯，不近。不近不要紧，韩泥点头——点得像118还石那夜。夜还了石，日备复测；复测成，芽儿才不怕沾魁晦气。

夜，十一层气走稳，稳像坛腹应「炼」——炼七粒，炼手，炼忍。忍了赖福七年，还了芽儿，再炼备战——备战尽，试台开。

他低声道：「十一层。七粒。试台，见。」见字落，接122织月嘲，接123戾衡影，接124铁无言挡话。挡了，舌净，手稳，粒正。一寸一寸熬，熬到十二层，熬到129帖至，熬到130魁与还刘婆。

闭门七日，七粒培元散成，十一层破。破，不声张。声张，赖福指偷境。指不要紧，管事看手。手稳，册在，就不滚。不滚，122织月嘲，123戾衡影，124挡话，125窑稳，126刘婆补袖，127债序清，128验正，129帖至，130魁与还刘婆。还刘婆，芽儿清了，序不乱，步不飘。

他摸玉牌，牌凉，低声道：「三年约，记着。钱戾衡，记着。赖福克扣，记着。」记着，刃磨亮。亮在还刘婆，亮在夺魁，亮在大比帖至山门。

七日毕，开窑，七粒仍正。正，试台在手，手不飘，飘了除名。除名，玉牌白凉，恩断不得。

第六日黄昏，窑门柴挡，挡严，严像席盖。正了，七粒封罐，封泥，封严，不示人。示人，赖福来；赖福来，除名借口来。不来不要紧，手稳，册在，就不滚。不滚，试台见，见还，见刘婆新衣养老钱在怀，在130递出。递出，芽儿石已清，序不乱，步不飘。飘了，叶汤凉；凉不得，汤是头的。头的先记着，手先稳着，粒先正着。

手稳，粒正，试台见。」见字落，坛温一线，像应。应了，路又明一寸——明向还恩，明向不滚，明向刘婆新衣养老钱。

韩泥不辩。辩，费舌；舌要留来守坛，留来按罐。手稳，粒正，就不滚；不滚，才配还刘婆，才配接大比帖，才配叶汤省亲前还尽。
