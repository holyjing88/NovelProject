# 第三十七章 丹堂大炼

七夜，丹堂火不灭。

苏念慈按残页配药，控火，凝丹，每一息都像在刀尖上走。外门弟子在阶下轮班，踮脚，不敢噪，像等一声「成」，好回去讲戏本——这一回戏本不敢乱添，添多了，丹堂首座能把他们炼进药渣里。

第一夜，她只开小火，像试路。莫长春在廊柱旁立半柱香，不进门，袖空，一念出塔，杖横膝，声轻：「试火别试心。心若急，炉先炸。」

苏念慈隔门答：「我不急，急的是缺三味。」

「缺味，」他道，「用情补。补得，页才真。」

第二夜，缺味之困显了，丹色发乌，弟子在外窃语「要败」。莫长春命顾小满送粥，粥里无药，只有热，热得像让人先稳住手。

第三夜炸炉险象，炉盖跳起，药烟苦辣，弟子惊退，有人喊：「首座！」

莫长春未进门，袖空，一念出塔，杖立门外廊柱旁，把一包药渣递给守门弟子：「送进去。渣是心意，不是废料。」

苏念慈接过渣，入炉，火竟稳了，稳得像有人把乱麻理成线。她满头汗，却笑，笑里倔：「师兄……你又使诈。」

「不使诈，」门外声音平，「你肯收渣，炉才肯成。」

第四夜，地火阵脚微裂，苏念慈以灵力补纹，补得指尖见血。莫长春隔门问：「撑得住？」

「撑不住也得撑，」她答，「页是你送的，炉是我守的。守不住，页就真成灰。」

门外静半息，静里像塔也在听。莫长春道：「撑住了，我请你喝茶。茶不必贵，贵的是成丹那口信。」

第五夜，柳青鸢遣执事送阵材，材轻，意重，重得像体统也在等丹。霍镇山派人送盾粉，粉细，像把镇妖司的硬气磨成护炉的一层皮。

赵玄机执法堂路过，停半息，未讽，只拂袖走。走前对守门弟子低声：「执法堂轮值，闲人退三十步。」

弟子窃语：「赵执事替丹堂清场？」「太阳打西边出来了？」又有人道：「清场不清心，心还在妒里。」话没敢大声，大声了就像站队。

第六夜，顾小满送粥时多留半句：「杂役堂听见谣言，说莫长老要进丹堂夺功。」苏念慈哼：「让他来夺，看炉答不答。」

莫长春在廊下笑，笑给风听：「功不在夺，在成。成了，谁夺都像夺香。」

第六夜末，赤焰方向腥风一线，像夜袭在提前嗅。苏念慈眸光一紧，莫长春袖空，一念出塔，门外杖点地，声轻：「你守炉，老夫守门口。妖来了，也是霍老头先叫。」

阶下围观弟子听见，窃语里怕，也敬——怕夜袭，敬的是长老七夜不进门，像把成丹的戏，全留给丹堂。

第七夜，地火阵鸣变，像炉心有人敲钟。

苏念慈立阵眼，灵力尽灌，丹田里有什么东西将破未破，像一颗种子顶破土。她唇白，眼亮，亮得像要把三十年压进这一夜。

莫长春袖空，一念出塔，杖点廊石，一声，两声，像给七夜敲结案鼓。

「成不成，」他低声，「都在这一炉。」

炉内，丹香起，温甜里掺雷息，像七月暑气被滤过一遍。阶下弟子齐呼，呼又被苏念慈一喝压住：「未成，退！」

呼声退，心不退，像全宗都在等下一息。

雷小，像天公肯给丹堂留一点面子——随后，丹成。

药香先于人声到了山门。苏念慈捧托盘出，袖口沾炭灰，眼底血丝，可眼睛亮得吓人。

金丹卧白绸，拇指大小，淡金薄膜流转。

莫长春仍立门外廊柱旁，不进门，像七夜前一样。他看着她，没说话。

苏念慈走到廊下，举托盘，让金丹反光落在他须眉上：「师兄。成了。」

全场一寂，继而炸开——炸在阶下，炸在廊外，炸在每个等着看「扑怀」的人眼里。

魏无涯在檐角观成丹一瞬，扇合，低笑：「重的推给她，轻的才送得出。送得出，才还得起。」

柳青鸢脚步在远处停，停得像体统也将至。她未近，只遣执事传话：「师兄，成与不成，门都在。」

赵玄机立远，铁牌在腰，像等败，又像等成。败没来，来的是更浓丹香，香压过妒火半寸。

顾小满守阶，声平：「炼记下了。」

成丹之后，苏念慈没有扑进任何人怀里。

她只把托盘举稳，像举一门规矩：丹成，先谢炉，再谢人。

莫长春终于迈半步，半步又收，收得像怕进门就坏了礼。他道：「收好。金丹不是谢礼，是下一段恩的开头。」

苏念慈眸光一闪：「下一笔，是什么？」

「赵罚，夜袭，天试，」他笑，「一笔一笔来。来的都不急，急的是人心。」

丹堂弟子跪了一片，跪里有人真心，有人作态，有人还在等「扑怀」戏码落空。戏码没落空，空的是流言——流言在丹香里薄了，薄得像七月暑气退潮。

赵玄机在远处站了很久，久到铁牌都被掌汗温了。他最终只说一句：「首座，辛苦了。」

苏念慈看他，看他眼里妒火还在，却矮了半寸：「赵执事，执法堂清场，清得不错。」

一句话，像把「太阳打西边出来」又按回东边，东边有规，规里有门。

七夜将尽，丹香浓得压过暑气，浓得像把东脉也炼进去一缕。弟子终于肯散，散时仍忍不住回头——回头瞧灰袍袖空，瞧九月还能走多久。

顾小满守阶至天明，守得像把心守牢。她问：「长老，七夜不进门，怕什么？」

「怕抢了戏，」莫长春道，「戏是苏丫头的。抢了，她还什么？还不了，塔里恩就歪。」

柳青鸢遣人传话，话短：「成与不成，门都在。」苏念慈回话更短：「成了。门在，丹也在。」

魏无涯在檐角收扇，收得像收一局棋。他自语：「重的推给她，轻的才送得出——送得出，才还得起。」

东方既白，丹堂火熄，熄得像七夜终于肯歇。可山门外的窃语未歇：有人讲药渣稳火，有人讲袖空立阶，有人讲金丹淡金——讲到最后，都讲成一句：青岚门这老头，不进门，却成全了丹堂。

苏念慈捧丹入匣，匣关，关得像关七夜风雨。她隔门问：「这粒丹，算谁的功？」

「算炉的，算页的，算你守的，」莫长春道，「不算我的。我若算进去，你就还不清了。」

阶下还有人踮脚，踮得像不甘心戏本没写成扑怀。顾小满笑浅：「戏本可以改，丹不能改。改了丹，全门都苦。」

霍镇山从镇妖司赶来，只问一句：「丹成了，东脉能不能稳？」莫长春答：「稳不稳，看你们守不守。老夫守了七夜门，你们守一夜脉。」

霍镇山咧嘴，像终于肯信：「行。守脉的，不缺你这一粒丹，缺你这一句话。」

赵玄机在远处听见，听见像听见另一种「清场」——清场清的是人，清不了塔里恩。他摸铁牌，牌凉，凉得像提醒：思过半月，只是开始。

苏念慈在丹堂隔窗望见东方既白，低语：「成了。」又补一字：「该还了。」

药香涌出丹堂门，涌得像替全门洗眼——洗眼之前，先让全门闻够这口香。

柳青鸢佩剑，声低：「师兄，成与不成，门都在。」苏念慈回话更短：「成了。门在，丹也在。」

莫长春杖点地，点得像给七夜盖印——印落，丹堂门开一线，七夜大炼，至此才算收功。

弟子渐散，散里仍回头瞧一眼灰袍袖空——瞧得像瞧一眼九月还能走多久。
