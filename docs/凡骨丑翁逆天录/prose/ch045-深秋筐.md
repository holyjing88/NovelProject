# 第四十五章 深秋筐

入门三月，霜厚，厚得藤条脆。

脆条勒指，指痛，痛里旧疤未合——合过两回，两回记着，记着等瓮醒那日扎得准。

管事掷单：编筐三十只，深秋补库。

韩泥接单，单旧，字丑。丑字他熟，熟得像泥岗筐价——价低，饭在，饭在，命在。

库役嗤：「丙九也领三十？编不完，别糟蹋条。」

「编不完，我赔饭。」韩泥说，「饭赔不起，手赔。」

库役一愣，扔捆。捆沉，压肩，压实。

赖福倚门：「末等灵根，编筐给谁看？」

韩泥不答。舌要留来勒条，勒紧，底才不松。
午后，院角日斜。

他编，慢而稳。底密，壁直，口收圆。圆，筐不塌；塌了，记名滚。

编到第十只，管事路过，瞥一眼：「手熟。」

赖福啐：「手熟有屁用，凡骨炼不了丹！」

韩泥抬眼，独眼平：「筐熟，饭熟。饭熟，人活。」

赖福唇动，没再尖。尖了也没丹炼，尖了只讨骂。

编到第十五只，藤刺炸，刺进掌心，血珠饱满。

他吮掉，血落篓底——不落坛，不蹭席，不故意滴。

管事数筐：「十五只，底都不松。松一只，扣饭。」

韩泥「嗯」一声，指不停。

编到第十八只，条脆，勒指，指茧起血线。血线细，细也记——记第三回，仍落篓底。

同批少年看：「你又流血！」

「编筐流血，正常。」韩泥说，「正常，不喊。」

编到第二十只，管事路过，数筐：「二十只，底密。韩泥，别松。」

「不松。」韩泥说，指不停。

编到第二十一只，条更脆，藤刺再炸，刺旧疤，血又落篓底。

他看血一眼，不看坛——看坛，手易往席蹭；蹭了，像主动醒。

「不蹭。」他低声道，「蹭了，不像活计。像求。」

编到第二十五只，指茧裂处又渗血，血仍落篓底——规矩，不蹭坛。

同批少年问：「你不怕疼？」

「怕。」韩泥说，「怕也编。编了，有饭。」

管事过，瞥筐底密：「丙九韩泥，手熟。熟，不是灵根，是命。」

韩泥「嗯」一声，指不停。

库役傍晚来收筐，数到三十，哼：「丙九也能编齐？」

「齐。」韩泥说，「齐，饭才齐。」

库役嘟囔，仍入库。入库，记功一笔——功在册，功也在手茧里。

管事折返，掷新单：「深秋补库急，再加五只。编不齐，扣半月麸皮。」

赖福笑：「末等也配三十五？」

管事冷眼：「配不配，筐底见。」

韩泥「嗯」一声，接新捆。捆沉，压肩，他肩不塌——塌了，像求；求不像丑翁。

编到第三十二只，藤条脆，勒指，血落篓底。编到第三十五只，末只收口，底密，壁直，口收圆。

库役再来，数到三十五，哼声轻了：「丙九也能编齐？」

「齐。」韩泥说，「齐，饭才齐。」

赖福倚门，唇动，没尖。尖了，管事在。管事在，筐在，饭在。

黄昏，末只收口，血甩到破席，席旧，吸了血，血渗向坛沿——渗到沿前，收手，席盖严。

渗不到沿上，瓮不醒。不醒，对。

坛腹温比往日重一线，重，仍不应血。血停在沿前，停在一寸，一寸里全是活计。

刘婆收碗，低声：「手又裂了，明儿缝布。布丑，能挡风。」

「记着。」韩泥说。

她哼：「记着记着，老婆子又不少块肉。」

韩泥「嗯」一声。第七笔，记在心里——泥岗六笔另列，宗门刘婆单记。

同批少年看筐：「你编得慢，饭不加。」

「饭不加，筐不塌。」韩泥说，「塌了，记名滚。」

少年闭嘴，仍学他勒条，勒不紧，筐歪。歪，管事骂，不学了。
夜，丑时将近。

坛沿温光一闪，灭。灭得像公审前夜那闪——闪的是沿，不是灵。

邻床问：「你对坛嘀咕什么？」

「记血停沿前。」韩泥说，「血不该醒瓮。醒瓮，要等活计扎。」

「疼不疼？」

「疼。」韩泥说，「疼记着，不喊。喊像少年。」

「记什么？」

「记血不渗沿。」韩泥说，「渗了，像求醒。求醒不像丑翁。」

邻床翻身睡了。

韩泥覆席，盖严。严，是不主动滴血。

管事巡夜，敲柱：「深秋筐三十五，入库。记功一笔，麸皮别短他的。」

赖福应声虚。虚，小胜。

韩泥「嗯」一声，不谢。谢轻，功在饭，饭在命，命在还。

同批少年嘀咕：「记功也不加灵根。」

「记功加饭。」韩泥说，「饭加命。命加还。」

赖福门外骂：「还什么还，末等滚得快！」

韩泥不答。滚在正测后，不在今夜。

外头风硬，空筐响一声。响像钩，钩向瓮醒——醒前，血不渗沿；不渗，对。

刘婆门外咳：「手裂了，明儿缝布。布丑，能挡风。」

韩泥「嗯」一声：「记着。」

廊下忽有人声，尖，像赖福在教谣：

「编筐三十五，漏粮气——」

声未唱全，管事喝断：「再唱，记名！」

韩泥合眼。合眼不是睡实，是记两条线：一条血止沿前，一条谣在起。谣起不要紧，要紧是冬后正测——测在远，手不能抖；抖了，备检白过。

管事巡夜再敲柱，身后贴条：「冬后正测，丙九末排先验。备检记名者，勿误。」

条旧，字丑。丑字他熟——熟像木牌，像记名，像多活一日。

他低声道：「正测冬后。冬前，手稳一日，多活一日。」

同批少年梦呓：「筐……」

韩泥睁眼，不答。答，费舌；舌要留来记三十五只筐入库——入库，记功，记功加饭。

他展掌，掌茧裂，裂里新皮嫩。嫩，疼真；疼真，记着，等瓮醒那日扎得准。

木牌在怀，条在柱，筐在库，血在沿前。都在，命就在。

坛壁一线水汽，应「记功」，不应谣。谣在起，不要紧；要紧是手稳，手稳才验得过。

他低声道：「三十五只，血三回，沿前止。止住了，瓮才醒得正。正醒，才还得了热的。热的，先还汤。记着。」

深秋编筐末单，藤刺旧疤，疤裂，血渗。渗，沿前即止——止，像瓮在数。数，不催，只温。温比白日多一线，多，像关将近。七笔心记，六沉宗外，一热刘婆粥。分列，不混。混，坡下站不住。筐交，钱少，够盐，够药，够活。活，才配问香、问谣、问正测。他低声道：「血不渗沿，命还要。」还要，才还。还，先还烫。烫，像末单交差那声实——实，手稳，瓮温。数，不催，只温；温比白日多一线，多，像关将近。分列，不混——混，坡下站不住。站不住，四日讲规矩就听不进，听不进，末排先验就滚。末单交差，瓮温加剧。加剧像关近——近，先数七笔，分列，不混。深秋末单，血不渗沿——沿前即止，止像关将近，将近先数七笔。编筐末单，血不渗沿——沿前即止，关将近。

末单交差，血至沿前止；韩泥按掌：瓮温加剧，关将近——将近，先数七笔，先分列，先不混，不混，坡下才站得住。

深秋编筐末单，藤刺旧疤，疤裂，血渗。渗，沿前即止——止，像瓮在数。七笔心记，六沉宗外，一热刘婆粥；分列，不混，混，坡下站不住。筐交，钱少，够盐，够药，够活；活，才配问香、问谣、问正测。

三十五筐入库，藤刺再入旧疤，血渗，沿前即止。止，像瓮在数关。

末单交差，钱少，够盐，够药，够活；活，才配问香、问谣、问正测。

瓮温加剧，关将近——将近，先数七笔，先分列，先不混。

编筐末单交差，血至沿前止。止，像瓮在数；数满了，坡下才站得住。

他低声道：「手稳，恩不断；恩不断，汤就不凉。」

坛沿他仍擦。擦时指节白，白像证：辱在册，恩在胸，分列不混。

还远不怕，怕的是汤凉——汤在叶铺，在刘婆灶，在心里。

泥岗根在，坛沿在，还路在——还路在，手就在；手在，恩就不断。

独眼平，像看秤——秤在，铺就不关；铺不关，汤就能还热。
