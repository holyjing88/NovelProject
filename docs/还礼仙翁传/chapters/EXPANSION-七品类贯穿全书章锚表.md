## 一、正文贯穿铁律

### 1.1 每章必检（涉及道具/修炼时）

| 项 | 要求 |
|----|------|
| **名称** | 首次出场写全名；回赠/再用可简称 |
| **品阶** | 凡/灵/宝/仙/神（丹药另标丹师可炼品） |
| **境界** | 适用境界或 HOLDER 境界 |
| **攻/防** | 灵器/法宝/符/阵/宠/骑须标子类（丹药标功能类） |
| **塔铭** | 赠礼/用符/布阵须写塔壁反应（品阶+对象） |

### 1.2 状态行模板（破境/大战/赠礼章可选）

```
修为【筑基中期】· 大境【筑基】· 小境【中期】· 资质【枯荣灵根·跌境残根】
修为【炼气十二层】· 大境【练气】· 小境【第十二层】
修为【金丹圆满】· 大境【金丹】· 小境【巅峰】
修为【渡劫第三重】· 大境【渡劫】· 小境【第三重】
丹【D05宝品·结金·破境·筑巅峰→金】· 器【W01灵品·攻】· 宝【暖手炉凡品·防】
符【T03灵品·防】· 阵【—】· 宠【P04·防】· 骑【杂役驴·防】
洞府【D2宝舍·丹堂侧室】· 闭关【宝阵+8%】· 丹房【Z03·三品】
```

> 小境通则：炼气 **1～13 层**；筑基～大乘/真仙 **初/中/后/巅峰**；渡劫 **1～9 重**。见 `08` §2.0.4。

### 1.3 穿插密度 × 品类

| 密度 | 品类要求 |
|------|----------|
| 每 **5 章** ≥1 次赠礼 | 须落 **具体品类+品阶**（丹/器/宝/符/宠/骑/阵材） |
| 每 **25 章** | 至少 **2 类** 品阶升级或新 ID 登场 |
| 每 **100 章**（部末） | 本部 KPI 七品类 **各≥1 锚点** |
| 低谷章（部一～二） | 符/阵/骑 **防侧≥60%** |
