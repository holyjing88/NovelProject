# 第四十四章 嗅诀三句

盯秤后第三日，渣场灰又热。热罩脸，脸罩心，心仍要分堆——药渣、炭渣、废渣，堆乱了路就断。

韩泥拨渣，鼻尖一动，苦里带甘。甘是叶振东那包药的余味，淡，却真。同场杂役呕：「天天嗅，嗅成晦气鼻！」

韩泥「嗯」一声：「鼻晦气，手不晦气。手不晦气，渣不漏。」

管事喝：「丙九，酉时清完。漏一粒，记名。」

韩泥指不停。不停，是盯秤后练出来的——练出饭，练出命，也练出还的路。灰热贴脸，贴出一层细汗，汗里混着苦蒿气。苦得醒，他反倒喜——醒了才分得清。

同场有人捂鼻退：「你天天闻这玩意儿，不怕鼻子废？」

「废了才怕。」韩泥说，「鼻子废了，手就不准；手不准，丹房不收。」

铁无言再来，立桩，声低：「渣分三堆，别混。混了，丹房不收。」

他应了一声。铁无言走近，看他拈渣：「鼻下这味，叫什么？」

韩泥停半息：「苦蒿余气，带一点甘。」

「甘从哪来？」

「叶家药铺方向。」韩泥说，「甘短，记在心里，不声张。」

铁无言看他一眼，眼里无嘲：「短，也能记？」

「短，才真。」韩泥说，「真，才不混。」

铁无言翻簿，从袖里取出薄册，册旧字密：「嗅诀三句，杂役能记多少记多少。」

他低声道：「一，鼻先静，静才辨。二，苦甘分开，分开才准。三，渣不漏，路才宽。」

韩泥复述，声哑，一字不落。铁无言点头：「笨人记全，比灵人记半强。」

同场杂役笑：「丑杂役也配学嗅诀？」

铁无言扫一眼：「配不配，渣场见。漏了，你配也不配。」

笑断。杂役凑近：「三句有什么用？」

「漏了，丹房不收。」韩泥说，「不收，记名滚。」

「滚了怎么办？」

「滚了，汤还不了热。」韩泥说。杂役啧一声，学他分堆，手仍乱。灰扬一脸，管事骂：「学也学像点！」

杂役揉眼，不学了。

午后丹房外药烟浓。韩泥隔街立，闭息试诀——鼻先静，静里苦甘分开。甘短，苦长，长里带涩。涩是废渣残药，残药不能乱扔。白织月过，袖风冷：「站街嗅，嗅出灵根了？」

韩泥应声：「嗅路。」

白织月唇线绷一息，走了。街角赖福瞥他伫立，喝：「丑杂役，站街装仙？」

韩泥挪半步：「站街记味。记错了，渣混；混了，滚。」

赖福啐，走了。啐完像怕铁无言第二眼。

验场前，铁无言折返，扬声：「丙九韩泥，拈三渣，报三味。」

场边立刻围上几个人。围着看热闹的，比围着学本事的多。韩泥不管围不围，只把脚站稳。

韩泥停，闭息，鼻先静。静不是闭眼装神，是把风声、骂声、脚步声都推远半尺。第一拈，苦长，蒿气；第二拈，甘短，叶家方向；第三拈，涩重，废药残。他声哑报：「苦蒿、短甘、涩废。三堆，不混。」

铁无言点头：「笨人全对，灵人常漏。」

同场杂役脸热，低头分堆，手紧半寸。管事验场：「丙九韩泥，渣清完，香篓齐。明日仍渣场。」

回场他把涩渣并入香篓，篓满而不泄。同场又笑：「丑杂役也配跟铁师兄说话？」

铁无言远处扫一眼，声不高：「渣不漏，话就少。话多，往往是手乱。」

笑又断。韩泥记——记这第二眼，记嗅诀三句。

午后末，赖福带人过渣场，故意踩乱第三堆。韩泥不骂，蹲身重分。分完，手紧，鼻仍静。铁无言折返，只看堆：「谁踩的？」

赖福脸白。韩泥不指——指费舌；舌要留来报三味。铁无言翻簿：「踩乱者，明日顶酉时渣场。」

小胜落在掌上：掌不乱，堆不乱，名还在册。同场杂役低声：「踩乱也不滚你？」

「滚看手。」韩泥说，「手紧，堆齐，铁师兄看堆，不看嘴。」

管事远远瞥一眼，没笑，只道：「丙九韩泥，堆齐。明日别误。」

「不误。」他应得短。短了，把气留给鼻下。

夜，油灯暗，丙九棚。韩泥把嗅诀写在破布角——不写全，写「静·列·漏」三字，字丑，丑真。邻床凑看：「什么鬼字？」

「路字。」韩泥说，「路通向渣不漏，通向还。」

「就三字？」

「三字记全，比百句记半强。」韩泥说。邻床撇嘴，睡了。

刘婆门外咳：「鼻下别漏。漏了，粥更稀。」

「不漏。」

他把香渣篓藏床底，藏实。实了不张扬；张扬，赖福来克。铁无言夜巡过渣场外，没进来，只扬声：「丙九韩泥，明日仍别漏。」

他应了半声。远处管事又贴细条：「三日后编筐，掌裂者备布。误卯时，记过。」

同舍有人梦呓「热」，他不答，只按破布角——角旧，字丑，真。真在「不漏」：不漏沿，不漏渣，不漏恩。他低声道：「三日后筐，筐接得上，香才问得准。」

铁无言再止在渣场外，扬声：「丙九韩泥，破布别示人。示人，像求。」

扬声远，远得像规矩比人长。他低声复念三遍。念完，破布角三字在，篓底香在，鼻下甘短仍在。

坛壁水汽爬半寸，应「静」字，不应灵根。灵在石上，诀在鼻下，恩在心里。刘婆又咳：「三日后编筐，手别裂。裂了，麸皮又浅。」

「不裂。」韩泥应，「嗅诀记着，筐才接得上。」

他复念末句：「渣不漏，汤才热。热着还，还着活。」念完合眼。合眼不是睡死，是等三日后藤条勒指——勒得住，筐才编得实。

卯时前雾白，他先到半刻，把三堆再理一遍。理完鼻下静。同批来迟，喘：「你又早？」

「早，手紧。」韩泥说，「紧，铁师兄才翻簿。簿上有名，名在末，也在路。」

铁无言到，只看堆：「丙九韩泥，报三味。」

他闭息，一字不落。落完，铁无言册上画圈——圈小，真，像嗅诀起头。同场杂役脸热，低头分堆，手又紧半寸。管事验场：「明日仍渣场。三日后编筐，别误。」

同批少年递绳：「三日后编筐，我帮你捆。手别裂。」

「捆。」韩泥低声应，「裂了，嗅诀白记。」

绳结实，实像三日后的藤条。赖福远处骂一句，骂不重——重的是韩泥没回头。回头像认谣，谣借腥，他不碰。

午后日头斜，斜光照苦堆，苦堆边甘短一线。韩泥蹲半刻，把涩渣拨进香篓，拨得准，像拈叶家药须。同批捂鼻路过，仍嘟「晦气鼻」，却没人再嚷他漏渣——嚷不过铁无言册上那一圈。

刘婆提粥远去，粥香淡，淡不过鼻下甘短。方向远，还字不远。他把破布角再塞深一寸，塞得像把嗅诀藏进掌纹。掌纹丑，丑得真；真掌，才拈得准渣。

回棚路上，风里粟壳香淡一寸。淡不要紧，要紧是鼻下静——静了，才分得清谣与香；分清，冬后正测才验得过。他按木牌，牌薄，真在记名。同舍有人笑他抱破布睡觉。笑浅，他不辩——辩费舌，舌留来念诀。

破布塞怀，三字：静、列、漏。诀紧着，三日后编筐帖也就钉着了。帖在，手在，鼻在，路就不断。

丑时末，他把指肚旧茧按了按。茧硬，硬才勒得住藤条。藤条勒住了，筐才实；筐实了，香才问得准。准了，谣就灌不进鼻子。

邻床迷糊又问：「铁师兄那三句，你背全了？」

「全了。」韩泥说，「全了才敢睡。睡前漏一句，场上就漏一堆。」

「那破布呢？」

「破布是门栓。」他道，「栓住心，诀才不飘。」

他低声道：「三日后编筐，手别裂。手裂了，嗅诀白记；白记了，正测辨香就站不稳。」

棚外细雪又落。落在柱上帖角，帖上「三日后」三字更扎眼。扎眼也好——扎进眼里，手就不敢飘。飘了，漏渣；漏了，滚。滚了，诀再灵也不顶饿。
