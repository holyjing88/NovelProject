# 第五十九章 沿前即止

冬更深，丙九棚静得能听见呼吸。静里坛温却醒了半分——不是醒全，像瓮翻了个身，翻完又眠回去。

韩泥编筐末藤，藤刺扎得深，血珠再饱满。他仍甩向破席。席旧，吸血，血渗到席边，到沿前就止。止住了，像有道看不见的线：线内席，线外沿。沿上无尘，也无血。

邻床梦呓：「……红……」

韩泥盖席，声低：「你看错了。睡。」

血到了席，没到沿。沿在眠，血不主动碰——这是铁规，守了六十日也不改。怀中符暖一线，护手不僵。僵了，下一刺便偏；偏了，缘就不正。

老耿门外咳：「沿净，好。净，才接得住赠。」

「好。」韩泥只答这一个字，不追问。问急了像邪；守住了，才像守瓮的人。耿脚步远了。

褐袍僧影早散，凡符使命已落。余下的是瓮与血缘，是席与沿。

午后清西栏，粪锹沉，掌茧裂口渗潮。潮不是血，血仍不碰沿。管事盯梢半息：「沿净，记心里。脏沿，像拜邪。」

「不拜邪。」韩泥说，「拜邪，路断。路断，粥就凉。」

赖福讽：「守坛守出仙缘？」

管事声冷：「再嗓，西栏你一人清到年尾。」

赖福哑了。哑了，韩泥肩上粪锹却捆得更实。刘婆端粥来，稀，热：「冬深，别空肚。」

「不空。」韩泥喝尽，「空肚手飘。手飘，沿先脏。」

铁无言巡栏边，声低：「沿净？」

「净。」韩泥说，「血在席，不在沿。」

铁无言灯远。他对坛低声：「你再眠一夜。眠着，我席盖严。严了，不主动滴你。滴，要等扎那一刺，误那一甩。」

坛腹起伏一线，又沉。外头柱上无新条——条无不要紧，席边那一线湿痕要紧。

明日仍搬柴，仍清栏，仍末排。末排不要紧，要紧是沿净、手正、心里七笔还在。他按席，席厚，厚像六十日擦出来的规矩。命跟坛，坛跟缘，缘在沿前那一止。

血是活计扎出来的，不是主动滴坛的。滴坛，要等误那一甩。秦霜远廊一句：「手正，心别邪。邪，符灭。」他怀符暖着，七笔都在，不邪。

同批少年碰他袖：「你席盖那么严，藏啥？」

「坛。」韩泥说，「坛在眠，不在醒。」

「醒了呢？」

「醒了，缘满。」韩泥说，「缘满不靠急。急像拜邪。」

少年不懂，哼一声走了。韩泥独眼平：「沿在眠，血不主动碰。碰，要等活计误那一甩。」

更鼓三响，沉。沉里瓮温醒半分，又眠回去。眠回去不要紧，要紧是沿净。他低声道：「沿前即止，不接沿。不接，才接得住瓮醒。」

霜贴丙九檐角，白得像给沿描了一道边。他拿旧席角蘸水，水凉，沿滑，滑里仍干。干了，才接得住后头那一甩。

编筐午后，藤刺浅半分再扎。浅半分，血珠仍饱满。饱满了仍甩向破席，席厚，吸得尽，沿仍净——净像瓮眠里翻身那半分，翻完又沉。

清栏时赖福故意撞他肩，撞像试沿净不净。韩泥让半步，不让席。席在，缘在。

管事瞧见，声冷：「撞也没用。沿净的人，撞不动。」

赖福气青，走了。走了，沿更净。铁无言巡廊，灯远前一句：「沿前即止，是止给瓮看，不是止给牙看。」

韩泥只点头。点头像懂「牙」字指谁。刘婆门外咳：「沿净，好。净，才少病。」

「净。」韩泥应，「净，像守瓮人。」

他擦沿，沿净，沿滑。净得像六十日磨出来的契。赖福路过，故意踢起沿尘。尘落，韩泥不辩，只拿旧席角按住尘，再擦一圈。管事瞧见，声冷：「赖福，再搅，你清栏加倍。」

赖福退得不甘，却不敢再嗓。管事验沿：「沿净，记心里。心里记着，舌上不辩。」

韩泥不辩。辩了像赖福。铁无言夜里提灯来，灯照席边湿痕，一线止于前三指。他低声：「止得好。止，不是怕，是等。」

「等。」韩泥应一声，像把字按进掌上。

同批少年蹭袖角，声小：「沿前三指，啥意思？」

「三指外是沿。」韩泥说，「沿不接血，缘才等得久。」

少年不懂，却走了。黄昏他掀席看湿痕，湿痕像一线墨，墨止于沿前三指，不洇沿。

老耿挑水又过，桶沿碰栏边，闷响一声。韩泥抬眼：「栏边尘，我擦。」

耿不答，却多留半息，目光落在湿痕上，湿痕止在绳量处，止得像契。契在，赠才接得住。他对坛低声：「血不渗沿。渗沿，是主动滴。主动滴，像借腥。」

坛腹起伏，仍眠。秦霜远廊木鱼不敲，只一句：「沿前止，舌也止。舌止，符才暖。」符温在肋下一线，像第七笔在冬里添柴。

他用草绳量沿前三指，绳短，指尖到沿刚好——界内席，界外沿；沿不接，缘才等得久。夜第四更，邻床翻身碰席角，闷响。韩泥不恼，只把席角掖实一寸，像给沿前那一止加了一道坎。坎在席，不在舌；舌若辩，沿先脏。

赖福被罚清栏加倍那日，沿反而更净——净像赖福的尘替界描了一遍。他在沿边再打第二道草绳结，结丑，丑在能记：记的是界，不是舌头。

管事夜巡验沿，烛照：「席边湿，沿仍净。」

「净。」韩泥说，「痕在席边，止于绳量处。」

管事记心里走了。丑时将近，檐角霜白，白里映出沿边那道界。他独眼平，低声：「沿前即止。止住了，才接得住瓮醒那一误。误在活计，不在舌。」

刘婆第二趟姜汤热得钝：「沿净，好。冬深，手别飘。」

「不飘。」韩泥喝尽，「热在，手才在。」

铁无言巡栏最后一趟：「沿前即止，记住了？」

「记住了。」韩泥说，「止在席，不在舌。」

灯远。丑时他练鼻下静，粟壳香淡，香一字不落。管事记簿记得细，细是小胜——胜在忍，不在辩。席边湿痕仍止于绳量处：止住了，才配接血止沿前那一夜，也才配接瓮将醒未醒那一息。急像拜邪，拜邪路断；他不急，只把界守严。

卯时搬柴回来，他把草绳量过的界又检查一遍。绳结仍在，湿痕仍止绳结处，不洇半寸。不洇，像瓮翻身那半分也被规矩按回去了。

同批少年拿树枝比划三指宽：「就这么窄？窄了血往哪跑？」

「往席里跑。」韩泥说，「席是吸的，沿是守的。守住了，才叫沿前即止。」

「止不住呢？」

「止不住就像拜邪。」韩泥说，「拜邪，符灭，路也断。」

少年嘴一瘪，树枝扔了。扔得好——界不能拿树枝乱画，画乱了，心先乱。心乱了，甩血就会甩过线。

午后他清栏第三趟，粪锹沉得肩骨吱呀一声。管事盯梢半息，没再骂脏沿，只记簿。记簿细，是小胜；胜在忍，不在辩。赖福加倍清栏时故意把脏水甩近坛角，韩泥侧身挡席，脏水落地面，不落沿。

「碰着沿你陪葬？」赖福骂。

「碰着沿我断路。」韩泥说，「断路的事，不做。」

管事听见，记名加赖福一笔。韩泥不谢管事。谢了像拉帮。他只把旧布再压席边，压实那道坎。坎在，丑时前瓮再翻半分，血也过不去。过不去，才接得住血止沿前那一夜，也才接得住后头误触一滴。

夜尽前他练鼻下静半息，粟壳香淡得几乎抓不住。抓不住也好，不贪香，才不邪。符暖在怀，暖得克制，像提醒：沿前即止，舌也要止。舌止了，赖福的脏水才甩不进心。

他把草绳结再勒紧半分，勒出的不是疼，是界。界在，血珠再饱也过不去。过不去这一夜，后头那条血止沿前的路才站得住——眼下他只守眼前：席严、沿净、瓮眠、人不催。
