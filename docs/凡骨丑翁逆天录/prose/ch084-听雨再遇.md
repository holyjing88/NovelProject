# 第八十四章 听雨再遇

初识后第九日，药渣场验堆。管事弟子敲柱：

「酉时验渣，丙九韩泥主分拣。分错，记名；分对，记功。」

风硬。灰起一线，韩泥指稳，鼻下有根。沉腐浮不落舌，辛左，腐右，假香上浮即剔——剔净，堆才净。验场围了几人，内门也有。他不抬眼。抬眼像求认可；不求，只分拣。分拣慢半拍也行，错一堆不行。

人群里，那一点冷香又在。像丹阁廊那日，像帚丝匀，像眉蹙那一息。温听雨立验场外，裙素，袖净。她本不必来，来了像规矩落场——规矩在，她眼也毒。瞥分堆，瞥他指：「丙九也主分拣？末席手，别分错。错了，全场笑话。」

韩泥应一声：「在。手在，就不错。」

她递帕。他不接——接帕像受礼，受礼像攀。只道：「我擦掌。粗布就行。」擦在粗布上，布丑，布真。真了，才不像借温氏袖风洗脸。

铁无言立桩，声低：「拈三渣，报三味。报错，记名；报对，过。」

韩泥停，鼻先静。第一拈，苦长，蒿气；第二拈，辛沉，药山深灰；第三拈，涩重，废药残。他声哑报：「苦蒿、辛沉、涩废。三堆，不混。」

铁无言点头：「对。」

对，是小胜。胜在指，不在嘴。温听雨眉蹙一息，蹙完却多停半息——停像意外。他不迎。迎像求；不求，只继续分拣。分拣声沙沙，沙沙里冷香淡一半。

她低声问铁无言，声仍傲，傲里多一丝疑：「他……全对？末席也能全对？」

铁无言扫她一眼：「笨人全对，灵人常漏。漏了，丹房不收。收不收，看鼻，不看脸。」

温听雨唇线绷一息，没再嗓。验场散，她走过韩泥身侧，停半尺，停得像秤。他仍不抬眼。肩稳，筐不洒。洒一粒，全对也脏。

她声不高：「你鼻下……静得怪。怪得不像杂役。」

「静在鼻，不在舌。舌响，静就跑。」

答完就走。走稳，像初识那日。那日让廊，今日让疑——疑在皮，不在掌；掌对，小胜。她未追，袖风一转，走了。走也像留一口问。问长了才有后头，问短也不要紧，缘线浅着扎实。扎实比热闹强。

赖福晃秤，眼细：「温氏也问你？问出邪鼻？邪鼻该镊了。」

「鼻下静，手在。」韩泥不接茬，「镊不了规矩。」

刘婆端粥，仍多半勺：「药渣场遇温氏了？」

「遇了。」韩泥说，「不辩。辩了像攀。」

「不辩怕什么。」刘婆哼，「手稳，饭稳。饭稳，恩不断。恩断了，全对也饿。」

韩泥点头。粥里有恩，疑里有皮，分得开才不乱。午后兽栏仍腥，铲下稳，稳里冷香回半息。管事记簿：「丙九韩泥，验渣全对。」笔落，像把「在」字钉进册。

同批少年脸热：「丑杂役也全对？全对了，温氏都愣？」

「愣不愣，不关我。」韩泥道，「关堆净不净。」

他不辩长。丑时前廊下闭息，息在鼻。嗓响，香飘；香飘，温氏疑就成真。席严，坛不露。坛腹温一线，贴掌，像应「再遇」。应完，他不叫——叫了像把缘苗拔出来晒。

邻床悄问：「温听雨问你，你咋不抬头？」

「抬头像求。」韩泥说，「不求。手对，就够了。够了，比漂亮话实。」

「她要是肯帮呢？」

「帮不帮，看手。」韩泥道，「手飘，谁帮都滚。」

秦霜木鱼轻敲：「缘来别迎。迎，心贪。贪了，鼻先花。」

铁无言脚步远，停半息：「温氏眼会改。改在远，别盼。盼了，今日全对也白对。」

他心里记：再遇了，缘苗浅。浅才藏得住。藏得住，才接得住后头辨性那一验。

白织月远廊过，袖风冷：「温氏也看你？看也炼不出丹。炼不出，别当自己成了气候。」

韩泥不抬头：「看渣，不看丹。丹在远，渣在近。近的先做完。」

再遇后第二日，外堂洒扫，阶净，栏净。温听雨巾湿，水净；韩泥帚偏半寸，灰不掠她裙。两人仍各干各的。她忽然问：「叶渣另堆，谁教的？」

「鼻下根。」韩泥说，「根在，苦辛并一线——像叶家药远路飘来半息。半息也记，记了手才不混。」

温听雨愣半息，愣不像嘲，像真愣。愣完收巾：「闻渣，不闻人。好。好字轻，你别当赏。」

「不当赏。」韩泥应，「当记。」

好字轻，记重。记她第二回停步，记她验手，记傲里松的一线。管事验洒扫，皆净，记功。赖福远处眼细：「温氏也跟你说话？说话就奸！」

韩泥不接。舌留给鼻子。刘婆端粥嘱：「再遇别攀。攀了，粥勺都嫌。」

「不攀。」韩泥说，「她擦她的，我扫我的。」

铁无言声低：「她敬才，不敬貌。才在渣，不在嘴。嘴稳，才在。」

「嘴稳。」

午后十堆全对。温听雨远廊一眼，一眼也像验。验完袖风一转，没落下第二句。秦霜木鱼一声：「缘苗浅，心别贪。贪了，十堆也乱。」

邻床夜里又问：「温氏会不会帮咱丙九？」

「帮不帮，还看手。」韩泥说，「手飘，帮来也滚。手稳，不帮也活。」

丑时廊下鼻下静。掌纹影颤，颤像三层将破。他对坛低声：「再遇了。嗅药小成在后。不叫。叫了像求诀名。」

第三日渣场叶渣另堆。苦辛并一线，并一线像叶家远路气味。飘半息，恩在胸，手才不飘。温听雨风口立半息，验分堆，眉松：「叶渣另堆，分得清。清了，全丙九少记一笔。」

他不谢。谢轻，记在心里。管事记「辨性准」。赖福眼细，没再尖——尖不过「准」字。准在鼻，不在舌尖。

刘婆又嘱别攀。韩泥点头：「不攀。规矩在场上，不在裙边上。」

老耿挑水过，声哑：「成小别拜。拜，路断。断了，叶渣另堆也救不了你。」

「不拜。记着。记准，记浅，记还。」

廊下风硬，木牌系紧。同批远避。藤刺偏手，沿净。夜巡灯照席角，像腌菜坛。他摸暗袋，袋沉，玉牌凉，愧字凉。凉不要紧，掌里远灯不熄。灯不熄，叶汤那笔还字就不灭。

药山半腰，雨歇，窑温余韵散进掌——散不要紧，记着：再试，火可匿，路向备料。叶渣另堆别混。混了，全丙九记名；记名，恩断边缘。边缘不要紧，鼻下有根，根细，细才辨得出。辨得出，明日嗅药小成才接得住。接得住，才不算白让那半尺廊。

更鼓沉。他按木牌，牌凉，掌温。再遇落了，缘苗浅着。浅不要紧——浅着，才等得起那一回「各归位」。归位之前，先把今日「全对」按实；按实了，验场上那口问才不白留。

夜里他再把三渣三味过一遍：苦蒿、辛沉、涩废。过完，独眼缝里只剩灰味。灰味干净，人就踏实。踏实了，才敢想明日丑时并息——并息不是显摆，是把「静得怪」那句疑，变成鼻下真有根。真有根，她眼神改不改都无所谓；无所谓处，线才浅得住，也才接得住后头那一验。

角柜锁轻轻碰了一下床板。他伸手按住，生怕夜巡灯晃过来起疑。疑起来，全对也变「邪准」。邪准两字一落册，叶渣另堆那段心血就白费。白费不起——废不起的东西，才叫再遇。

他又把温听雨那句「闻渣，不闻人」在心里按实。按实了，攀字就离得远；离得远，明日辨性小成才站得正，站得正，才配把「还」字继续往前背。

棚梁上积灰落下一点，落在袖口。他弹掉，动作轻，像弹假香。假香弹净，真灰才留得住——跟验场同一理。理通了，再遇不算热闹，算把鼻下那点静，从廊上接到渣堆上。接住了，钩才钩得动后头那一验。
