```
韩泥 · 【大境+小境】 · 灵根【伪/末席】 · 丹师档【X品】 · 符录【符阶·名】 · 阵盘【阵阶·名】
```

### 11.3 自检清单

- [ ] 出场/章末标 **大境+小境**（炼气须写层数，渡劫须写重数）
- [ ] 道具品阶 ≤ 当前大境对应阶（可低不可高两阶）
- [ ] 丹药品级 ≤ 当前可炼/可服上限
- [ ] 六类道具攻/防比例符合 §2.0.1（低谷防≥60%）
- [ ] 洞府位阶 ≤ 当前大境；升府须同步六区辅区提升；择缓须写辅区锁定（`12` §4.5）
