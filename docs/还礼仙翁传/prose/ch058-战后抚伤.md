# 第五十八章 战后抚伤

黎明，东脉焦土，露冷，硝烟像一层洗不净的灰。

柳青鸢寻到莫长春时，他袖角有血，血不是他的，像妖与金丹留下的尘。她递绷带，手稳，眸不稳。

「师兄。」她低声，「青岚门可以没宗主，不能没你。」

话落，她为他缠绷带，缠得紧，指尖触腕脉，忽一停，像听见什么，又似不愿听见。她额抵他肩，一瞬，极短，像把宗主威仪借夜风放软一分。

莫长春不动，任那一瞬落，落完，才笑，声轻：「风大。」

柳青鸢退半步，脸热，热得像霜洗过的剑，仍冷在鞘，鞘外却起了一层薄红。她硬声：「……师兄别胡说。」

「不胡说。」他道，「风大，脸才红。红了，也像人，不像只有剑。」

她瞪他，瞪不出狠，只转出宗主令，令在腰，威在回：「天试之前，别死。」

「尽量不死。」莫长春答，像多年同门的旧约，像闪回夜里回过的那四字简。

她转身，袍角扫过焦土，扫出一道青痕，像剑意收鞘。

远处丹堂，苏念慈望这边，望一眼，收眼，炉仍开，像把妒与担都炼进火里。弟子在焦土边窃语，窃语里慕，也怕——慕宗主竟肯放软一瞬，怕九月老头袖角有血。

霍镇山骂着走过，骂到一半，见柳青鸢背影，噤声，像懂有些事不该洪声。

赵玄机在丹堂门洞包扎，听见「风大」，停半息，未讽，像也懂：守门之后，还有恩要还。

莫长春抚绷带，簿微烫，字淡：

**战后有人缠，恩仍在袖。**

他望天试方向，云低，像帖将至。

围观弟子渐散，散里仍窃语「宗主脸红」，窃语轻，轻不过焦土上的血。

魏无涯远观，扇合，低笑，像看恩与担缠在一起，缠得像剑收鞘，鞘外起薄红。

他低声，像钩：

「抚伤是恩，不是体统。体统二字，留给他们嚼。」

风大，露冷，像天试帖已在云上写名。

有人低声：「宗主脸红……」「风大。」「风大还能红？」「体统二字，留给他们嚼。」窃语轻，轻不过焦土上的血，轻不过天试帖将至。

苏念慈在丹堂望这边，望一眼，收眼，炉仍开，像把妒与担都炼进火里。她低声：「风大，脸才红。红了，也像人。」

顾小满在焦土边，眼里亮，亮得像雨夜那道光还在。她低声：「宗主放软一瞬，像剑回鞘。」

焦土尚温，绷带尚紧——紧的不是伤，是恩；恩在袖，帖在云，天试之前，别死。

魏无涯远观，扇合，低笑，像看恩与担缠在一起，缠得像剑收鞘，鞘外起薄红。风大，露冷，像天试帖已在云上写名——名在帖，帖在试，试在之前，别死。

柳青鸢寻到莫长春时，他袖角有血，血不是他的，像妖与金丹留下的尘。她递绷带，手稳，眸不稳。「师兄。」她低声，「青岚门可以没宗主，不能没你。」话落，她为他缠绷带，缠得紧，指尖触腕脉，忽一停，像听见什么，又似不愿听见。远处丹堂，苏念慈望这边，望一眼，收眼，炉仍开，像把妒与担都炼进火里。

焦土尚温，绷带尚紧——紧的不是伤，是恩；恩在袖，帖在云，天试之前，别死。围观弟子渐散，散里仍窃语「宗主脸红」。

赵玄机在丹堂门洞包扎，听见「风大」，停半息，未讽，像也懂：守门之后，还有恩要还。魏无涯远观，扇合，低笑，像看恩与担缠在一起。

她瞪他，瞪不出狠，只转出宗主令，令在腰，威在回：「天试之前，别死。」「尽量不死。」莫长春答，像多年同门的旧约，像闪回夜里回过的那四字简。她转身，袍角扫过焦土，扫出一道青痕，像剑意收鞘。

围观弟子在焦土边踮脚，踮得像等体统二字——体统没来，来的是风大。有人低声：「宗主脸红，风大？」另一人答：「风大，脸才红。红了，也像人。」

焦土边两弟子半声半气：「宗主竟肯缠绷带……」「缠的是师兄，不是体统。」话未绝，柳青鸢背影已远，远得像把脸红也收成鞘；莫长春抚绷带，笑轻，围观里有人忽懂——懂抚伤是恩，脸红是风，风大的地方，帖也快到了。

霍镇山骂到一半，见柳青鸢背影，噤声，噤得像也懂：体统留给旁人嚼，恩留给同门记。赵玄机在门洞听见，未讽，只把绷带勒紧一寸，像把「继续还」也勒进甲里。
