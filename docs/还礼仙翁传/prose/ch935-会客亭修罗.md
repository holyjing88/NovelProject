# 第 935 章 · 会客亭修罗

> **承上**｜ch933～934 柳剑穗夜话；族老名帖已至  
> **地点**｜**D5纳绶别院 · 会客亭**  
> **人物**｜莫长春、柳青鸢、苏念慈、顾小满、秦商言；柳氏族老、苏叔、顾母、秦老太君；木青萝（误闯）

---

会客亭不大，亭外碑温，侧碑新刻柳氏剑谱名分，像给敬茶备好的席。

柳氏族老执剑谱侧碑，苏氏旁支叔遥认丹堂匾，顾母俗世揖礼，秦老太君送商路帖——**家族观礼**，部九铁律，今日齐。

四女敬茶，座次却无人肯让。

柳青鸢捧剑穗茶，先敬，目光稳：「师兄未说出口的话，我替天下说。」

苏念慈丹香茶第二盏，袖口药渍未净：「丹炉还开着，茶也还热着。」

秦商言第三盏，笑：「亏本敬茶，商言乐意。」

顾小满第四盏，手不抖，声音抖：「半块灵石……那年的恩，今日还一盏。」

木青萝误闯亭外，捧花愣住，被弟子拉走，花留阶下，像喜剧楔子。

族老们不坐主位，只观礼。主位空着——莫长春在门外石阶坐定，杖点地，不进门。

弟子窃语：「莫长老又坐门口。」

柳青鸢眼角微跳，仍敬茶不乱。苏念慈与她对视一瞬，妒意有，不黑化，像火在炉里，不炸堂。

秦商言低声：「公平到不肯排座，才是莫长老。」

顾小满退半步，让亭内光影落在莫长春膝前。

莫长春不接茶，只道：「敬了就好。茶烫，慢喝。」

四女同时顿住，又像同时松口气，跟坐四椅——椅在门外，跟坐，跟到石阶并排，像落成礼那日，像部五坐门口，像千年老梗新讲，仍好笑，仍好暖。

柳氏族老抚须：「柳氏名分，侧碑已立。」

苏叔拱手：「苏氏丹道，认纳绶，不认债。」

顾母抹泪：「小满这孩子，跟对人。」

秦老太君笑：「商路帖是贺礼，不议价。」

莫长春望亭内，望碑温，低声：

「亭外碑温，侧碑柳氏名分已定。别的……慢慢来。」

敬茶眼神战两刻，胜在无人摔盏。章末，族老退，四女收盏，会客亭灯未熄——纳绶夜在帖上，还隔着几日，今夜只敬茶，不纳绶。
