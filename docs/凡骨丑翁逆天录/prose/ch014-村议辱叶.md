# 第十四章 村议辱叶

韩泥胳膊上的伤还没好，村里的会却先开了。

天阴着，泥岗村闷得像扣了口大锅。他天一亮还是编筐，藤条绕着膝盖，臂布勒得紧，一勒就疼——疼着也好，至少提醒他，后山那夜狼口不是做梦。筐才编到一半，里正的锣响了，一响，全村往祠堂聚。锣声硬，硬得像把人往台阶上推。推就推吧——伤还在，恩也在，他得去听清今天要糟蹋的是谁。听清了，才知道往后该怎么一个个还。

马春花敲着铜盆，声音刺耳：「叶家寡妇勾引晦气，药铺里藏邪！今天说清楚，说不清楚，全族连坐！」

祠堂前挤满了人，嘴多心少。赵绣春站在台阶上，活脱像唱戏：

「叶青禾，你送汤送姜，图什么？图药铺名声？还是图那绝户翁以后报恩？」

叶青禾站出来。青布衣裳，不跪。「图人该送。」她声音平，「天冷，门缝大。送姜，送热。」

「热？」赵绣春笑，「热到铺子没人抓药？热到沈家被罚？热到全村都晦气？」

周嫂帮腔：「寡妇不知羞！」

一片「嗯」声涌过来，像潮水往叶家淹。

韩泥坐在末席，胳膊缠着布，不说话。开口像坐实「一对」；不说又像默认。他选第三条：听着，记着。

赵绣春逼近一步：「你说，韩泥给你什么好处？药？钱？还是……」

叶青禾抬眼，一眼很冷：「他给我什么，我不知道。我给他一碗热汤，我知道。」

人群里有人小声接：「也就一碗汤罢了……」又有人嘘：「嘘！被赵家听见，连你也扯进去。」

祠堂梁上掉下灰，落在她青布肩头。她没拂。拂了像认错，不拂像认人。

韩泥在末席看着，指节发白。不是怕，是眼见那碗汤被人吐口水。那碗汤他喝过——烫，辣，暖进胃里。暖进胃里的东西，不能叫人当众往脏水里泼。

陈姑在人群后拢紧棉袄领口，低声问邻人：「祠堂议什么？」

邻人答：「议叶家牵扯。少嚼舌头，怕连坐。」

陈姑闭了嘴。韩泥跟她无冤也无恩，只晓得这人嘴利——嘴利不要紧，他心里自有一本账。

「热汤？」赵绣春嗓子尖，「热汤能当饭？能当药钱？」

「能。」叶青禾说，「能当人心。人心都凉了，你抓药抓谁？」

祠堂静了一息。马春花盆又敲：「嘴硬！赖管事说了，叶家若不认牵扯，迁铺！」

叶青禾嘴唇发白，仍不跪：「牵扯的是谣，不是我。」

「谣？」赵绣春手指韩泥，「他绝户，克邻克亲，你靠近他，不是牵扯是什么？」

众人附和，声音大得像要把她压进地里。

韩泥指节更白，还是不开口。开口，只会害她更深。他只在心里把那碗叶汤按实——按住了，叶家的脊梁才不倒。

里正再敲铜盆：「够了！再辱，记名！」

「记名」两个字，勒住赵绣春半寸。她嗓子仍尖，尖里却缺了后力：「记！叶家欠全村一个交代！」

周嫂忽然道：「要我说，叶家关铺三日，谢罪！」

「对！关铺！」

「绝户翁搬出村西！」

周嫂棒槌往地一墩，没砸人，嗓子却硬：「关铺三日，娃咳谁治？汤……汤不算邪。我洗衣水早浑了，也没见邪！」

静里有人「嗯」了一声，潮退一寸。赵绣春瞪她，周嫂脸白，不退：「绣春，你饿不着，你别替全村饿。」

马春花还想敲盆，里正抬眼一扫，盆声噎在半空。场子里只剩粗喘气。

气顶到顶。韩泥袖里指节响，响不出声——气满了，里正不得不抬手。

他这时才起身。慢，跛，却直。祠堂里一瞬间静了半息。

「关铺。」嗓子哑，却够全场听见，「病谁治？搬我，谣谁担？担我，不如担真话。」

赵绣春一愣：「你认邪？」

「我认恩。」韩泥独眼平，看着她，也像看着全村，「汤是热的。我记着。谣烫不了这恩。」

有人倒抽一口气。有人低头看地。低头看地的人里，也有昨天还跟着嚷「晦气」的。

恩字落地，像石块丢进潮里——潮还没退，石头已经沉了底。他不辩奸，只辩恩。辩完，阶下那抄帖的手也顿了一顿。

里正抬手压场：「今日议到这。再闹，报官。」

赵绣春脸青，想再尖，被里正一眼压住。尖收半寸，后力没了。韩泥最后一个出祠堂，独眼扫过空阶，阶上还留着叶青禾站过的印，浅，却真。

祠堂阶下，有人把「公审」两字抄进帖里。抄得歪，心却悬着——悬着，就是明天的日子。

散场散得不甘。叶家未安，谣没停，气还憋着。

祠堂侧，有人低声问里正：「公审，开不开？」

里正板着脸：「开。日子未定，理要定。定不了，村先乱。」

叶青禾在侧巷，没近过来，只站了一息，转身回铺，背影仍直。

她停半步，没回头，声低：「韩翁，别认邪。认恩，够了。」

「嗯。」韩泥道，「恩我认。邪，等公审判。」

就两句。她步子稳，回铺，门帘一动，秤砣落一声——落给全村听：铺还在。

外头赵绣春一伙人散了，散时仍骂。骂声远着，里头夹一句「公审」，像锤子还没落下，声音先到了。

韩泥回屋，把那碗叶汤在心口按实。臂还疼，疼里发冷。冷不是怕，是忍——忍到公审，把「汤是热的」当众说清。

夜里他只记三样：叶汤要深，赵辱要清，叶青禾没跪下的背影要在。在，将来还的时候，她才不一个人扛。

叶铺灯终夜弱亮。灯在，像汤的影还在。韩泥睡不实——睡实了像忘恩，他忘不起。

散场后村道还沸着。有人拽周嫂袖子：「真关铺？关了三日，我娃咳谁治？」

周嫂脸僵半息，嘟囔：「关的是晦气，不是药……」

嘟得发虚。叶铺灯还亮着，亮着就是底气。

赵绣春一伙有人嚷：「绝户翁认恩？认恩就是认邪！」

韩泥担筐走过，不停，只道：「恩是汤。邪在谣。谣你熟，汤你不熟。」

嚷的人噎住。窃语里，「公审」两字又浮上来。

里正路过，咳了一声：「今晚水席，谁再辱叶，记名。」

韩泥「嗯」一声：「记。等公审。」

盐担老汉蹲在阶下，嗓子粗却低：「周嫂今日这话，公审上我帮一句。帮一句，不帮邪。」

韩泥听见了，不点头。点头像拉帮。他只把恩按得更深——深了，等锤子落。

赵绣春门内啐一声，啐碎在台阶上。祠堂阶下，有人抄帖，抄得慢，却实。像全村第一次把「公审」写进心里。

棚下起了细雨。叶青禾撑伞走，伞骨旧，仍直。韩泥檐下收筐，雨声里听见秤砣又落一声。雨丝打在他独眼上，他不揉——心不虚，雨就只湿一点皮。

丑时，叶铺秤声又起，脆而短，钉住铺子没关。张麻婆骂童谣丫：「少学周嫂那嘴！」骂完往门缝塞半块饼：「吃。编筐手不能抖。」

沈枯芽隔门问：「韩爷爷，汤烫不烫？」

「烫。辣好。趁热，才记得住。」

叶青禾在巷那头应了一句：「记得住就好。」

韩泥没接。接轻了像讨。他藤条绕膝，心里那笔恩更实。实了，手就不抖；手不抖，将来还的时候才不寒酸。

风从门缝过，带一点姜的辛味。他合上眼，手按在坛沿上。沿在，根就不散。瓮还没醒，他不喊醒——明天村里还要议，公审的锤子，只等日子定。

坛腹微温一线，像听懂了「还」字。他还不起大的，先把小的记牢：汤热，人直，杖未落。杖落那日，他要把恩字当众立住。
