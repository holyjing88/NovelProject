# 第一百一十四章 公示辱

坡下新榜，贴的是「杂役末席公示」。

公示辱，柱上又贴条，条新，字利：「丙九韩泥，贪渣诬，公示三日。」公示不要紧，手在才要紧。手在，名在册，辱在众目，手不能抖。

同批少年脸白，脸白不要紧，韩泥只按木牌——牌在，名在，路在。路通向115尾摊，116捡漏，117九层。

刘婆端粥，仍多半勺：「公示？」「公示。」韩泥说，「手稳，饭稳。饭稳，恩不断。」

柱下风硬，刘婆立柱下侧，声低：「公示三日，别飘。飘了，柱条就洒。」韩泥应声：「不飘。柱条在，手在。」手在，就不滚；不滚，尾摊潜才接得住。

辰时，全丙九列队。赖福立榜侧，嗓尖，尖得像刀刮木：「韩泥，凡骨末等，炼气七层亦末。末，公示三日，以警众人——沾晦气者，连坐克扣。」

丙九嗡一声。嗡里，少年退半步，退得像怕传染。韩泥独眼平，立末，不躲——躲，像怕；怕，手就抖。手抖，滚字就真；真了，叶汤恩断，芽儿半块灵石也还不上。

管事弟子念条：「公示末席，手稳者仍留；手抖者，除名。韩泥，上石阶。」

石阶九级，级级像秤。韩泥一步步上，掌茧疼，疼不露。末级立定，坡下百双眼睛——眼里有嘲，有惊，有白织月的冷。冷不要紧，手稳；稳了，辱就白挨一半。

白织月袖风带香：「七层也末？末，丹路断。断了，站阶上，像桩。」

韩泥不辩。辩，费舌；舌要留来守坛，留来记仇苗。仇苗对着公示，对着赖福，对着钱戾衡——钱戾衡今日不在，不在不要紧，玉牌在怀，三年约在怀。约在，就不滚。

赖福扬声：「跪半刻，谢宗门不除名之恩！」

「管事看手，不看膝。」韩泥声平。

管事弟子蹙眉：「赖管事，公示无跪令。」

赖福脸青，青完阴笑：「不跪，站半刻。站，让全宗看灰线三尺配七层。」

站半刻，风硬，吹得灰线名像刀。刀在皮上划，划不出血，划得出忍。忍不是怂，是活——活长了，才还芽儿石，才接复测魁。魁在远，辱在近；近，站直，就不白挨。

铁无言立人群侧，声低，只韩泥听见：「站直。站直，像108那日并肩。」

韩泥应声，脊不弯。弯了，滚字就真；真了，叶汤恩断，芽儿半块灵石也还不上。不弯，是小胜——胜在活，胜在七层，胜在赖福压顶又深一分。

半刻尽，管事挥手：「毕。韩泥，下阶，仍渣场。」

下阶，每一步稳，稳得像首炼培元散时按炉——炉热，手不抖；辱热，手也不抖。抖了，除名试炼就白过；白过，九层开不了。

赖福凑近，声只有两人听见：「公示三日，三日你吃饭最后。最后，像狗。」

「狗也清粪。」韩泥说，「粪清，宗门净。」

赖福噎，噎完走。走，韩泥记——记公示这一辱，记赖福压顶，记白织月袖风。记在心里，刃磨亮一分。亮一分，够接坊市尾摊，够接还芽儿半块灵石。

散队后，刘婆端粥——粥薄，勺小，仍给他舀满：「公示了？」

「公示了。」韩泥说。

「公示怕什么。」刘婆哼，「手稳，饭稳。最后吃，老婆子等你。」

韩泥点头。谢轻，恩重。恩在粥里，也在忍里——忍三日公示，换一日不除名。不除名，恩才不断；不断，还序才不乱。

午后渣场，他分拣更细。细，像把「末」字筛成渣里一点香——香正，手正。手正，铁无言又记一笔账清。账清，诬告虚；虚了，试台净半分——试台在远，公示在近。

傍晚，内门廊下有人笑：「灰线三尺，七层也末！」

笑薄，薄也真。韩泥不答，答在心里——心里玉牌凉，三年约烫。烫在还恩前头，烫在钱戾衡那一脚反面。反面烫，正面手稳；手稳，就不滚。

他回窑，盘坐。气走周，周里炼气七层稳，稳得像坛腹应「站」——站直半刻，坛温一线，像夸不跪。不跪，是丑翁本事；本事在忍，忍在活，活在炼气八层前头。

丑时，练鼻下静。香落齐，公示这一日记进心里。记进心里，不进舌——舌响，赖福又来。又来不要紧，手稳，就不滚。

邻床少年悄问：「你真不跪？」

「膝硬，手更硬。」韩泥说，「手硬，才配还恩。」

少年闭嘴。闭嘴，韩泥摸玉牌：「钱戾衡，记着。公示三日，一日一日记。」

记着，就不白站。白站，辱就白挨；不白挨，仇苗才扎实——扎实了，才接得住坊市尾摊，才接得住赖福最后一压。最后一压，在除名试炼；试炼过，九层开，还芽儿石在前。

外头柱上条角白一线，白得像钩——钩向坊市，钩向捡漏，钩向还沈枯芽半块灵石。石在远，辱在近；近，忍三日；三日过，路又宽一寸。宽一寸，够走到115尾摊潜行，够走到徒恩首还。

坛腹微温，应「站」字。站直，是丑翁的本事；本事在忍，忍在活，活在炼气八层前头。八层在腹，不露；露了，公示就成了真滚。

赖福脚步廊外停半息：「明日仍末。末，吃饭最后。」

韩泥只答：「在。」

在，就不滚。不滚，公示三日，一日一日熬——熬出头，是坊市；熬出头，是还芽儿。还芽儿，首笔正式报恩；报恩，静着还，还到该还的人手里，才算数。公示第一日，韩泥吃饭最后。碗糙，粥薄，薄里刘婆仍舀满。满，像恩顶着赖福克扣——克扣不要紧，手稳，饭仍顶一口饥。饥顶命，命顶还路。

坡下有人指他名，念「韩泥·仍末·手稳」。手稳，是小胜；胜在活，不在榜。榜末，真末；真末不要紧，腹里七层实，玉牌在怀凉烫并着。

午后渣场，铁无言低语：「公示三日，别近廊下吵。吵，手飘。」韩泥应声，分拣更细。细，像把「末」筛成渣里一点香——香正，手正，手正，钱戾衡眼风硬不过「正」。

白织月第二趟过，袖风更冷：「站半刻，像桩。桩也配公示？桩稳，台不塌；塌的是丹道骨，骨丑，散可正——正在远，先熬公示。第二日，仍末，仍最后吃。最后不要紧，刘婆等——等像补衣婆等恩还那日。那日在130，日在远，先忍公示，先忍赖福「像狗」。第三日，公示榜揭。揭不要紧，韩泥名仍在末行。

末行丑，丑得像脸；脸丑不要紧，手稳。手稳，公示毕，赖福压顶到头——到头，是115尾摊，是116捡漏，是117九层试炼。夜，他摸玉牌，低声念「三年约」三字。三字凉，仇烫。烫在心里，不进舌。舌响，香飘；飘了，滚字就真。不滚，才配偷空去坊市，才配捡焚火残页。坛腹微温，应「站」——站直半刻，不跪。

不跪，像35挡石——石挡飞，膝不弯；膝弯，恩像讨。讨轻，恩重；重，站着还，跪着不像还。外头风硬，柱上条角又白一线。白，像仇苗出土。出土不要紧，先活；活，才坊市潜行；潜行，才捡漏；捡漏，才九层压顶尽，才118还芽儿半块灵石。」

他低声对坛：「公示三日，一日一日记着。记着，刃亮。」亮在试台，亮在还刘婆，亮在钱戾衡台前不滚。不滚，手在，粒在，坛在，玉牌在。

公示第三日毕，韩泥下阶，每一步稳。稳像70按台——台在众目，膝不弯，手不抖。不抖，钱戾衡眼风硬不过手。手硬，玉牌不虚。虚了，三年约白接。白接，恩断，叶汤凉，芽儿石白还，刘婆新衣还不上。还不上，魁名空。魁名不空，先在尾摊捡漏，先在117九层试炼。

他摸玉牌，牌凉，低声道：「三年约，记着。手稳，粒正，试台见。」见字落，坛温一线，像应。应了，路又明一寸——明向还恩，明向不滚，明向刘婆新衣养老钱。

刘婆粥仍多半勺，姜仍多一片。片暖，恩不断。恩不断，手才稳；手稳，粒才正；粒正，试台才配说话。说话了，白织月袖风尖不过「正」，钱戾衡影硬不过手。

钱戾衡声不高，高在韩泥耳听见：「丑杂役，上台别洒。」韩泥低声道：「记着。」
