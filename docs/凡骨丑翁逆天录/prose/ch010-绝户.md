# 第十章 绝户

十月初，霜薄。

韩泥出门编筐，藤条在膝，膝上袄角硌着。霜贴泥地，贴得鞋底凉。马春花从井台来，声先至：

「哟，绝户翁还编筐？编给自己打棺材？」

几个妇人笑，笑短，像刀背刮铁。

韩泥「嗯」一声，绕半步。马春花跟两步：「叶家寡妇敢近，沈家芽儿敢送粥，陈姑敢送袄，张麻婆敢送馒——晦气翁成香饽饽了？」

韩泥停，独眼抬一下，又垂。藤条在膝上绕半圈，绕得像把话也绕住，不让它尖。

「香不香，我不知道。」嗓哑，慢，「我欠着。欠汤，欠粥，欠袄，欠馒。」

妇人笑声收了一截，像刀背没刮响。

马春花一愣，愣得像没备好下一句：「欠？你欠谁？」

「欠敢对我好的人。」韩泥说，「欠一碗汤，欠一碗粥，欠一件袄，欠一个馒。」

「你疯了？」

「没疯。」他声更平，「汤是热的，我记着。」

马春花鼻哼：「热汤？叶寡妇那碗？你配喝？」

「配不配，汤已入喉。」韩泥说，「入喉，就得还。」

「还？你拿什么还？」

「拿命还。」韩泥独眼平，「命在，还字就在。」

妇人笑不出来。有人低声：「绝户翁还真敢说。」

马春花啐远：「疯子！」转身，步乱，像踩着自己话绊了一下。井台边，周嫂棒槌砸盆沿，溅水：「马春花，少嚼。嚼多了，里正记名。」

马春花脸一青，没再接。

韩泥担水路过，不近，只一句：「井是村的。水我打，不偷不抢。」

周嫂一噎，棒槌悬着，终是没砸。村道静一息。

韩泥回屋，心口四笔恩在：叶汤、芽粥、陈袄、张馒。赵辱一笔，胸口一根仇苗。

叶丫头汤——最深。芽儿粥——细。陈姑袄——短。张麻婆馒——实。

四笔。为何要先还叶？

因五十八年无人送汤进门缝；因谣起她仍送姜，仍写「趁热」；因恩不是可怜，是人该送；因还字比谢字重，重了，就得活到达得到还的那日。

芽儿为何记细？因娃怯，怯还肯送；因粥稀，稀里有一粒胆；因罚银在沈家头上，翁不能怂。

陈姑为何记短？因袄旧，旧里有一捧暖；因孤寡送绝户，送的是「还有人把你当人」。

张麻婆为何记实？因骂是邻，邻还肯塞馒；因馒硬，硬得像这村子还肯留他一口粮。

四笔齐，心齐。齐，才配说还。

他低声自问：「先还谁？」

自答：「先还汤。汤是头的。」

他低声道：「要还。」

心口一紧，紧得像丑时未到，坛先应半分。韩泥不声张，只按心，按实了，像按命。心口热，热得细，细得像恩在呼吸。

他低声道：「四笔恩，一笔辱，一根苗。要还。」

爹临终那夜，也在这坛边。铁山手搭坛沿，嗓哑：「坛别扔。韩家就剩你。」又补一句，像复述祖父的话：「这是你爷爷传我的，我爹也这么传。」韩泥当时二十岁，不懂根是什么，只点头。三十八年过去，坛仍在角，擦沿的习惯仍在，根在不在，他仍说不清——说不清，却不敢扔。

坛角蒙尘，坛腹微温一丝，坛壁凝一线水汽，像应他这句「要还」。

还字出口，像钉进心里。字丑，丑得真。真字压绝户二字，压得马春花啐完仍绊脚。

他低声道：「绝户不绝恩。恩在，就得还。」
门外有人敲，敲得轻。陈姑嗓沙：「韩泥，开门。」

韩泥开门一尺。陈姑眼浊，却亮：「马春花那嘴，别往心里去。」

「不往舌上去。」韩泥说，「往心里记。」

陈姑哼一声：「心？」

「恩笔。」韩泥说，「袄我记着。汤粥馒，也记着。要还。」

陈姑愣半息，摆手：「还什么还，穿活了再说。」

「穿活了，才还得上。」韩泥说。

陈姑转身，风一刮，晃一晃，像一片旧布在村道飘。日头偏西，他担筐再赴集尾。

这一回，十五文。比上回多三文，三文像活路宽了一指。买主问：「又涨价？」

韩泥摇头：「价不变。人稳了，筐好卖。」

买主笑：「晦气翁，倒实在。」

韩泥不接「晦气」，只道：「结实。」

「你筐价不高，人倒硬。」买主又多给半文，「拿去买盐。」

韩泥点头：「盐要。恩也要还。」

买米，买盐，剩三文，搁门墩下石缝——芽儿常路过，路过或拾得，不拾也无妨，他不做施舍状，只做还不起前的分毫。

回村，门墩空，尘落。

他望一眼，像望还会不会有一张纸写「趁热」。

不来，他也等。等不是空等，是活。

村道又有人笑，笑短：「绝户翁还等谁？」

韩泥不回头：「等能还的那日。」

藤条再绕膝，紧。紧得像把「要还」编进筐里——筐卖钱，钱买米，米养命，命养恩，恩养还。

米缸有粮，盐罐有盐，心有四恩，胸口有仇苗。

五十八岁绝户翁，丑，跛，独眼，仍活着，仍记着，仍卖得动筐。

这就够。

村道有人嘀咕：「绝户还攒钱，攒给谁？」

韩泥听见了，不答。答了，像认绝户。认不得，认还字就行。

他进屋，独眼瞥坛角，坛仍蒙尘如凡物，沿口却温着不退——温应四恩，不应绝户二字。

汤是头的。

他低声道：「要还。先还汤。绝户不绝恩。」
夜里，韩泥坐坛边，四恩自检一过。

叶汤——热，姜，秤声压谣。芽粥——稀，胆。陈袄——旧，暖。张馒——硬，实。四笔，笔笔真。真，才配立誓还。

坛壁水汽一线，应「要还」，不应绝户二字。瓮没醒；醒前，先把还字立住。

他低声道：「绝户不绝恩。恩在，先还汤。霜落，公审不远。公审前，手不能抖，腰不能软。」

立住了，绝户二字就压不住他。

村道远处，叶铺秤声又起一声，起得像钉——钉住「铺子还开着」，钉住他心里那笔头恩。恩在，就要活久些，活稳些。

卖筐十五文，腰布沉了一截。沉实，像今日唯一赢过村嘲的东西。

马春花嘲远，四恩在胸；韩泥按心：要还定了——定了就不急，急像少年；他是记心翁，先记热，再记辱，等公审那锤落，锤落，汤才还热。

马春花嘲远，四恩在胸。绝户册上两字，压不死擦沿的手——手在，根在，根在才配说「要还」。要还定了，就不急；急像少年，他是记心翁，先记热，再记辱，等公审锤落。
