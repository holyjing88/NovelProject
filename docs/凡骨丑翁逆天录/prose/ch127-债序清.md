# 第一百二十七章 债序清

试台清晨前，韩泥理怀内九粒培元散。

债序清，他低声分列——分列，不混：叶汤头恩，芽儿已还，刘婆试台后还，仇在钱戾衡三年约。序清，步不飘。飘了，粒洒；洒了，丹不正。

铁无言敲门，声低：「卯时列队，试台坡下。我站你左侧三步。」三步就好——三步，挡粪话，不挡手。手稳，簿清。

九粒在怀，袖口刘婆针脚暖，玉牌凉烫并着。并着，像路——路向128试台列名，路向129鼓响，路向130魁与还刘婆。

粒粒泥纹浅，浅得像丑脸——脸丑不要紧，嗅之正；正了，试台才配说话，说话才配钱戾衡影硬不过「正」。正了，刘婆敢认恩，芽儿敢收石，叶汤还在前头烫；烫着，省亲路才明，大比帖後才接得住。

他陶罐封泥，泥封严，严得像腌菜坛蒙尘。蒙尘不要紧，蔽眼；蔽了赖福眼，蔽了白织月眼，试台再开，正字自现。自现了，魁名落，还刘婆落地，大比帖在怀沉。

债序在心，他低声分列——分列，不混：

「一，叶汤，头恩，省亲前还。」

「五，芽儿，还沈枯芽半块灵石，已还。」

「七，刘婆粥补袖，试台后还新衣养老钱。」

「仇，钱戾衡三年约，玉牌在怀。」

序清，步不飘。飘了，粒洒；洒了，丹不正；不正，刘婆恩还迟，叶汤还凉。凉不得，汤是头的；头是叶丫头那碗烫，还在省亲前，还在大比帖后。」

「三步就好。」韩泥说，「三步，挡粪话，不挡手。」

「手是你的。」铁无言说，「手稳，我掷簿；簿清，盗谣虚。」

韩泥点头。兄弟线，记——记124挡话，记108并肩。并肩不替炼，替证手稳；证了，试台净。净了，舌留来守静，守鼻下香一字不落。

刘婆灶边盛粥，仍多半勺，半勺像印：「债序清了？」

「清了。」韩泥说。

「清就别乱。」刘婆哼，「乱还，恩成债。债重，手飘。」

韩泥喝尽。谢轻，恩重。恩在粥里，也在序里——序清，粒才正；粒正，魁才配，还刘婆才配。还刘婆，新衣棉实，养老钱数过三遍，布包在怀等130递出。

沈枯芽柱角站，手捧小布包，包轻，里半块灵石已不在——不在，在石已还那夜；夜还了沈枯芽半块灵石，日来送行，行不送，送的是记。记着，徒恩清了，叶汤还在前——在前，省亲路，在大比帖后，在钱戾衡台前。

「韩爷爷，试台别抖。」芽儿声小。

「不抖。」韩泥说，「抖了，还你石那夜就白还。」

芽儿点头，点得像米粒。米粒旁，徒恩清了，叶汤还在前——在前，省亲路，在大比帖后，在钱戾衡台前。台前，炼气十三，你才有资格不滚。

韩泥应声。傲像赢；赢不要紧，试台赢。赢了就还刘婆，赢了炼气十三在130破，赢了大比帖至山门。帖至，言伯钧名在帖上，钱戾衡影在字后——影在路后，路在前；前是魁，前是还刘婆，前是叶汤省亲。

他摸玉牌，牌凉，贴胸，贴得像第三列仇印：「三年约，记着。今日试台，半年后大比，都不抖。」

坛腹温一线，应「序」——序清，瓮温；温一线，像爹说「坛在，韩家不散」。不散，韩泥就不滚；不滚，钱戾衡「让你滚，你才有资格不滚」才接得住。接得住，就不抖；不抖，试台手稳，粒正。

丑时末，练鼻下静。粟壳香淡，香落齐，债序这一分列记进心里。记进心里，不进舌——舌响，仇苗浅；不浅，刃亮。亮在试台，亮在还刘婆，不在钱戾衡舌。

邻床少年悄问：「你真能夺魁？」

「魁在试台，不在嘴。」韩泥说，「嘴问，手答。」

少年闭嘴。闭嘴，韩泥望坡下——坡下试台新木，木色浅，浅像70踹台。踹台接了约，约接了今日；今日试台，明日还恩序落地。落地，还刘婆，大比帖在怀，叶汤在省亲前。

外头鼓声远，远得像试台将开。将开不要紧，九粒在怀，袖口刘婆针脚暖，玉牌凉烫并着。并着，像路——路向128十二层圆，路向129鼓响，路向130魁与还刘婆。还刘婆，恩偿静着还，还到该还的人手里，才算数。

赖福廊外最后晃木牌：「试台滚，省我工夫！」

韩泥只答：「在。」

在，就不滚。不滚，债序清，手稳，粒正——正，是五年淘出来的，不是运气。运气白织月会说，会说不要紧，试台见正；见了，执事扬声「韩泥，魁」，刘婆眼浊里清一寸。

他低声道：「序清。试台，见。」

见字落，坛温一线，像推他出门——出门上台，上台夺魁预备，上台后还刘婆新衣养老钱。养老钱数过三遍，新衣袖口多缝一层，耐磨，像36那夜补肘，像126今夜补袖。

债序清这晨，韩泥理怀内九粒培元散。粒粒泥纹浅，嗅之正；正了，试台才配说话。说话才配钱戾衡影硬不过「正」，白织月袖风尖不过「正」。

他低声分列，不混：「一，叶汤，头恩，省亲前还。」「五，芽儿半块灵石，已还。凉不得，汤是头的。

铁无言敲门：「卯时列队，试台坡下。三步，挡粪话，不挡手。手稳，我掷簿；簿清，盗谣虚。」

刘婆灶边盛粥，仍多半勺：「债序清了？」「清就别乱。乱还，恩成债。恩在粥里，也在序里——序清，粒才正。

沈枯芽柱角站，声小：「韩爷爷，试台别抖。」「不抖。抖了，还你石那夜就白还。米粒旁，徒恩清了，叶汤还在前。

他摸玉牌，牌凉，贴胸：「三年约，记着。不散，就不滚。香落齐，债序这一分列记进心里。」见字落，接128验丹正，接129鼓响帖至，接130魁与还刘婆。并着，像路——路向夺魁，路向还刘婆，路向大比帖影里钱戾衡名。名在帖后，后在不要紧，手在前；手稳，帖才接得住。

债序清，卯时列队。队末韩泥，怀九粒，袖刘婆针脚，胸贴玉牌。贴紧，仇约也在胸。约在，手在，就不滚。不滚，128验丹正，129鼓响帖至，130夺魁还刘婆。还刘婆，新衣递，养老钱递，眼浊清一寸，终于敢认恩。序清，粒正，手稳，钱戾衡影虚半分。

他摸玉牌，牌凉，低声道：「三年约，记着。钱戾衡，记着。赖福克扣，记着。」记着，刃磨亮。亮在还刘婆，亮在夺魁，亮在大比帖至山门。

刘婆粥仍多半勺，姜仍多一片。片暖，恩不断。恩不断，手才稳；手稳，粒才正；粒正，试台才配说话。说话了，白织月袖风尖不过「正」，钱戾衡影硬不过手。

序清后，新衣布包与养老钱串又数三遍，不多不少，够补衣婆活到老。

活到老不必躲赖福眼，还了敢穿新衣出门，出门像恩新，恩新命也新。

序清，粒正，手稳。稳，验正，夺魁，还刘婆，帖至。

序清，卯时列队。敢认，就够称一部圆满；圆满不在魁名，在债序。债序在，步不飘；步不飘，省亲那日才踩得实泥岗路。手稳，粒正，试台见。」见字落，坛温一线，像应。应了，路又明一寸——明向还恩，明向不滚，明向刘婆新衣养老钱。

序清，验正，夺魁，还刘婆，帖至。

更鼓沉，沉里，韩泥按木牌，牌凉贴胸。127章手在，名在册，坛沿净。舌不留，鼻下静。恩辱分列，不混。活长了，才还得了叶汤，才接得住门缝灯。

坛腹微温，应「稳」字。稳不是飘，是丑翁专精；专精在渣，专精在窑，专精在五年忍。忍了赖福压顶，还了沈枯芽半块灵石，备战才真；真备战，夺魁，还刘婆，大比帖在怀。

外头更鼓沉，沉得像试台在等。等在远，备战在近；近，九粒在怀，玉牌在怀，三年约在怀。在，就不滚；不滚，钱戾衡「让你滚，你才有资格不滚」才接得住。

铁无言验簿，簿清，声低：序清了，手别飘。韩泥点头。点头，像盖序印。印盖得糙，糙得牢。牢，才走得动130还路。还路，先验正，先夺魁，先还刘婆，先帖至，先大比，先钱戾衡台前不滚。不滚，记着68眼风，70踹台，123廊下影。影硬，手更硬；手硬，玉牌不虚。
