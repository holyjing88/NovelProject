# 第五十七章 周犹豫

战后清点，硝烟未散，东脉侧门仍闭。

闸纹在晨光里冷，冷得像从未被人碰过。顾小满立侧门阵闸外，浑身泥血已半干，泥色与血色糊在一起，糊成东脉一夜的证。她指侧门，指节白：「周长老，战时三回求开，闸纹未动。赤焰残骑绕田埂，若侧门开，本可截断——」

她报时，像把昨夜再演一遍：子时初，赤焰绕田，她第一次求开，周德海说防伏；丑时中，丹堂阵眼告急，她第二次求开，周德海说后阵不稳；寅时末，莫长老斩金丹，她第三次求开，周德海仍令旗不落。三回，闸纹未动，动的是顾小满册子上的字，字越记越硬。

周德海抚须，面色平，平里冷：「侧门一开，中伏谁担？老夫守后阵根本，不错。」

「丹堂差点破。」顾小满声哑，「赵师兄在门洞流血，苏首座在阵眼榨干灵力。霍长老盾阵裂角，莫长老两连斩——后阵稳不稳，侧门开不开，不是一回事。」

「若丹堂都守不住，开侧门不过是送头。」周德海截话，声不高，理太冷，冷得像早就备好的稿。他令旗在指间，旗面干净，干净得像从未举起过。

围观弟子渐聚，渐聚成浪，浪里怒，怒周长老见死不救，怒侧门不开，怒杂役一句话敢顶长老。也有弟子低语替周辩解：「后阵不开，未必奸。」立刻遭冷眼：「奸不奸，闸纹比嘴真。」

柳青鸢立不远处，青衫沾灰，未言，只目光扫过周德海指间令旗，令旗干净，干净得像从未举起过。她淡淡道：「周长老之言，本宗记下了。盟会之前，不辩；盟会之后，再辩。」

周德海揖手，笑，笑里一丝松，松得像逃过一劫：「宗主英明。」

霍镇山骂声远，骂得洪，却骂不到后阵——后阵稳，稳得像奸线也稳。他盾阵收拢，收拢里仍骂：「侧门不开，截什么敌？截青岚自己的臂？」骂声被风带走，带不走闸纹。

莫长春拄杖过，不看周，只看顾小满：「话送到了？」

「送到了。」顾小满点头，「也记下了。册子上，三回求开，三回不开。」

「记下就好。」莫长春道，「记比杀难，比杀有用。」

他望侧门，低声，像记在心里：「门不开，有时防伏，有时防忠。忠奸，不在嘴，在闸。」

塔壁无新铭，塔影却沉，像一笔大因果，暂存，待后兑。闸纹冷，冷得像也在等盟会之后那一辩。

弟子窃语反转，反转成惧：惧周德海，惧侧门，惧盟会之后那一辩。有人提秦管事见信使的旧话，立刻被同伴喝止：「别乱传。传了，也传不过闸纹。」

浪声里，周德海仍笑，笑里冷，冷得像不怕日后辩。他转身回棚，棚下副将欲言又止，止得像也懂：长老令，就是闸。副将姓马，马副将低声：「周长老，赤焰已退，残敌……」周德海只一瞥，瞥得像把后半句也关进闸里。

魏无涯在檐角，扇开一线，观气运，观周德海袖，低笑一声，像看线深了。他低声，像钩：「闸在，恩在。盟会之前不辩，盟会之后，一笔不漏。」

风过焦土，硝烟像洗不净的灰，灰里藏着下一卷的剪线。

孙福旧党面色白，像听见「侧门」二字，也听见自己的旧恩仇将被翻。有人退半步，退得像怕闸纹也记自己。

韩铁山在远山听报，笑冷：「侧门不开，青岚自断一臂。」报信人低语：「莫长春又斩妖帅……」韩铁山笑意一滞，滞得像臂断的不止青岚。

厉无殇在帐内，声哑：「周德海，好棋。」棋好，好在他不必再派第二刀，好在他恨意有人代挡。他摸颈侧旧伤，伤是手帕定身留的，留得像也记得九月老头的三息。

赵玄机在丹堂门洞包扎，听见「侧门不开」，停半息，未讽，像也懂：门不开，有时防伏，有时防忠。忠奸，不在嘴，在闸。他绷带勒紧，勒紧里想：若侧门开，残敌截断，他守门的血，是否白流？想完，仍立，立得像闸纹也刻进甲里。

苏念慈自阵眼退下，炉仍温，温得像战后第一口能喘的气。她望侧门方向，未言，只把一枚培元丹捏在指间，捏得像把「不辩」二字也炼进火里。

柳青鸢回宗主殿途中，令在腰，威在回。她未再提周，像把「盟会之后」四字，先寄在每个人心上，寄得每个人脚下都像踩在延迟的痛处。入殿，她独对东脉图，指节发白，白里不是惧，是担——担师兄九月，担东脉民，担盟会上的闸。

顾小满回杂役堂，册子入怀，入怀像多了一页比战报更硬的记。她低声，仅自己：「闸不开，记下了。记下比骂难，比骂有用。」

莫长春夜归小院，无灯，杖横膝，不辩周，只念闸。闸纹与塔纹，像两条线，线未交，交点在盟会。

午后，宗主殿侧厅，柳青鸢召霍镇山、苏念慈、顾小满，不召周德海。霍镇山第一句便骂：「侧门不开，老子盾阵白裂一角？」柳青鸢抬手，止骂，止得像止剑：「盟会之前，不辩；今日只记。」

顾小满呈册，册上三回求开，字迹泥血交错。苏念慈补一句：「丹堂若破，侧门开亦无用；丹堂未破，侧门不开，残敌走脱——两害，取轻。」

柳青鸢点头：「记。」

霍镇山哼：「记字比闸快，快就好。」

莫长春在廊外听，未入，入像抢宗主戏。他望侧门方向，闸仍闭，闭得像周德海的笑，笑在，闸在，恩也在。

秦商言飞符至，符短：「周见赤焰信使，第三回。茶饼恩，不欠。」符落柳案，案上像多一枚钉，钉在盟会之前，钉也在盟会之后。

魏无涯离山十里，立亭，观气运翻涌，涌在东脉，涌在侧门，涌在青岚宗主殿。他扇开一线，低笑：「和战线，周占一半。莫长春占另一半。占满，才好玩。」

赵玄机夜巡丹堂，巡至侧门远观处，望闸纹，未言，像把「忠奸在闸」也巡进铁牌。孙福旧党有人夜访周德海，访得急，急得周德海仍笑：「盟会之后，再辩。」

侧门仍闭，闸纹未动——动与不动，盟会之后，一笔不漏。东脉焦土上，弟子散去，散里仍有人回头望闸，望得像望一块迟来的判词。判词未下，痛已至，痛在每个人脚下，像延迟的爽，爽不在闸开，在闸被记。

翌日，杂役堂贴出伤号榜，榜末添一行小字：侧门闸纹，战时未动。字小，却硬，硬得像顾小满册子上了堂。周德海听闻，面色仍平，平里冷，冷得像不怕纸也记。

霍镇山率镇妖司弟子巡后阵，巡至侧门，以盾轻叩闸纹，叩声闷，闷得像问：这闸，为谁闭？无人答，答在盟会。

莫长春至侧门，不叩，只望，望半息，像把闸纹也收进塔壁。他低声，仅顾小满：「闸记下了，人也要记下。记你三回，记你泥血，记你比长老敢言。」

顾小满点头，眼里亮，亮得像雨夜那道光还在。

柳青鸢颁令：盟会之前，东脉侧门加封二道，非宗主令与护法令同至，不得开。令出，周德海揖手，笑仍冷，冷像也懂：加封，是防伏，也是防他。

韩铁山听侧门仍闭，听加封，笑更冷：「青岚自缚，省本将一兵。」报信人不敢应，应像也懂：缚的是闸，未必是门。

莫长春回小院，簿沉，沉像七十五斤人情又添一笔「闸」。一念出塔，杖点地，声轻：「周犹豫，小里见和战；和战不在嘴，在盟会上一拍。」一念收塔，杖落掌，他念：「豫分，伤要来。战后抚伤，重里见人心。」

侧门闸纹微动，动得像和战二字，先在周德海须上停了一停。
