# 第一百四十八章 叶家信

叶家口信到，信仍无字，只有半片枯叶。

沈枯芽悄说：「叶姐说，142那粒，耿爷服了。她……她谢。」

韩泥不接谢，接在心里。谢在还，不在舌；还在叶汤，还在省亲前。

门缝灯未亮，亮不要紧；脉稳，像142经叶家那路还温。

铁无言声低：「叶家线，别混情。混情，手飘。」

「不混。」韩泥说，「情在还，不在近。」

刘婆端粥，仍多半勺。

丑时，练鼻下静。香落齐。

他望坛，低声道：「叶家信到了。记着。」

叶家线，线在缝，缝在口信——口信无字，字在沈枯芽眼里，眼里红一线，线像脉稳，稳像142散收好。

韩泥不近叶府，近府像还情；情在还，不在攀。攀，路歪；歪了，七笔分列就混。

沈枯芽立三步外，声低：「叶姐说，谢。谢不在嘴，在脉。脉稳了，散才正。」

韩泥只答：「正就好。」好，就不辩；辩，费舌；舌要留来守坛，留来候149风。

芽儿又递半片枯叶，叶黄，叶背有细字，字丑：「别再来门缝。」

字丑不要紧，字真；真，才像148该记的——记叶汤在前，记142经叶家，记188雨夜在远。

铁无言立栏边，声低：「叶线，别缠。」「不缠。」韩泥说，「缠，像情；情在还，不在舌。」

刘婆端姜汤，汤烫：「叶家的事，别沾晦气。」「不沾。」韩泥说，「沾，手飘。」

坊市续摊，疤鬃驴在柱后，柱稳，稳像148——稳线，不稳名；名在远，线在今。

管事记簿，记「丙九韩泥，叶信收，不进门」，记细，是小胜。

他望坛，低声道：「叶线记了。记着。」坛腹温，贴掌，像应「分」——叶汤不混耿恩，不混刘婆。

更鼓沉，天边白一线。白里，149风将起，起不要紧，手在，就不滚。

叶信收，他不进门，进门像攀。

沈枯芽走后，他理摊材，材细，细像142备材那筛——筛齐，手才静。

白织月远影过，没停。没停，小胜——胜在不分，胜在不缠。

叶家线记在心，不记在舌——舌要留来守坛，留来候149风。

坊市续摊，疤鬃在柱后，蹄声稳，稳像148该过的日。

他低声对坛：「叶线记了。149风在前。记着。」

白里，周家二字影将至。

线在缝，缝在沈枯芽口信，信无谢字，谢在脉。脉稳，142经叶家才像送，不像攀；攀，路歪，七笔分列就混。混了，叶汤不混耿恩，刘婆不混周家——分列在心，148只记。记「别再来门缝」，字丑，字真；真，才像叶青禾仍避，仍温，仍把培元散收在脉里那一线。一线不混95匿丹，不混142馈缘；混，手飘。

飘了，149周家逼嫁风就接成架；架接了，150收不住。收不住不要紧，148只稳线。稳法是不进门，不攀情，不辩谢——谢在脉，不在嘴；嘴若谢，像还情；还情不是还恩，还恩在粒，粒已送出。送出，148只收枯叶，收细字，收芽儿眼红那半息。

半息，像142后叶姐脉稳；稳，沈枯芽才敢递「周家别逼叶姐」那一句——一句在149，148只埋。埋叶线，不埋架；架在151，188才解。解在远，148只记。记完，坊市续摊，疤鬃在柱后，柱稳，稳像线。线不断，262省亲还在远；远不要紧，148只分：叶汤在前，耿恩已还半，刘婆已还——还不混，混则坛凉。

凉，片会凉；不混，坛腹温一线，像应「分」。分，更鼓沉，天边白一线，白里149风将起——起不要紧，手在，就不滚。

叶信无谢字，谢在脉；脉稳，142才像经叶家，不像攀情。攀情，路歪；歪了，七笔就混。

叶信收，他不进门。不进门，线才正；线正，149风才只听不架。更鼓沉，他低声对坛：「叶线记了。记着。」七笔分列在心，不混叶汤，不混耿恩，不混刘婆。

坡下石阶湿，他数息，数帖，数债序。

同棚杂役笑，笑在耳，不在心。

废渣巷日斜，他验渣，渣正，手才静。

侧门驿亭风硬，他听风，风过，记在心里。

散队后，他过外堂阶下，分拣半刻，指稳——指稳，路就不飘。

编筐半刻，藤刺扎掌，刺浅，浅些沿才净。沿净，坛才藏得住。

粟壳香淡，香一字不落。

管事记簿，记细，是小胜。胜在忍，不在辩。

他望坛，低声道：「手在，路在。」坛不应字，只温。

白里，他按木牌，牌凉贴胸。

坊市尾巷风硬，他收摊如石，石在，就不滚。

验息后，他理怀内册页，页黄，字丑，丑而真。

铁无言声低：「别飘。」韩泥应声：「不飘。」

韩泥点头，谢轻，恩重。

药峰分拣照旧，指稳，鼻下静一息。

帖在怀，怀沉，沉的是序，不是惧。

白织月袖风过，韩泥不辩。辩，费舌；舌要留来守坛。

赖福廊下过，韩泥只答：「在。」在，就不滚。

柱上条角白一线，三年约在怀。怀凉，掌温。
