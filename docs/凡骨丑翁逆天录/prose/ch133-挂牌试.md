# 第一百三十三章 挂牌试

坊市开市，尘起。

外门坊在药山南脚，棚杂，价乱——乱里藏真，也藏嘲。嘲在舌，真在耳；耳要留来挂牌，留来试水。

韩泥立末摊，摊是破筐反扣，扣上木牌，牌歪，字丑：「丑丹师·嗅渣辨性·试水。」

就八字，无夸。无夸，才像常杂役挂牌，不像夺魁显摆。

管事弟子验牌：「试水三日，价自定，税照旧。」

「照旧。」韩泥交半文，文旧，旧不要紧。旧，像末席该交的。

第一日，无人近摊。近，像沾晦气；晦气不要紧，风在听。

听里，白织月袖过，声冷：「丑丹师？试什么，试晦气？」

韩泥只答：「试渣。渣净，栏清。」

第二日，有人递渣，渣灰，灰里杂。杂不要紧，韩泥鼻下静一息，静里辨出辛淡——辛淡正，正像96那日听风。

递渣人愣：「你真辨？」

「辨。」韩泥声平，「辨了，才挂牌。挂牌，不在名，在手。」

人退，退时多留半包渣——半包，像试。试不要紧，试出「正」，风就改一半。

第三日，管事贴条：「丑丹师试水，暂留。留，不在荣，在听。」

韩泥收摊，收得像没来过。没来过不要紧，牌已挂——挂丑，挂稳，挂得像97那三日。

铁无言过，脚停半息：「牌挂上了。」

「挂上了。」韩泥说，「挂上了，才接风声。」

「风声紧。」铁无言声低，「钱戾衡的人，在坊尾。」

韩泥不近坊尾。近，像挑衅；挑衅不要紧，手稳，就不滚。

刘婆买米过，米糙，声低：「少摆摊，多吃饭。」

「饭稳。」韩泥说，「摊是试水，水试过了，142才谢得老耿。」

刘婆愣半息，没听懂142，听懂「饭稳」：「稳，别飘。」

回丙九，他按坛。坛腹温一线，像应「牌」字。

丑时，练鼻下静。粟壳香淡，香落齐，挂牌这三日记进心里。

邻床少年悄问：「你真挂牌了？」「挂了。」「挂了，不怕人笑？」

韩泥不答，答在牌——牌歪，字丑，人更丑；丑不要紧，渣正。

更鼓沉，他望坛，独眼平：「试水过了。下一回，铁兄续。」

坛不应字，只温。温像应「听」——听风紧，手更稳。

坊市尾巷，谣与真并起——谣在舌，真在耳。耳要留来辨，辨了，才续摊。

管事夜巡记簿，记「丑丹师试水」，记细，是小胜。胜在渣正，不在嘴。

钱戾衡影在坊尾，影淡，淡像139将紧。紧不要紧，手稳，就不滚。

编筐半刻，藤刺扎掌，刺浅，浅些沿才净。沿净，坛才藏得住；藏得住，牌才不虚。

铁无言立坊口，声低：「试水三日，别辩。辩像显摆。」

「不辩。」韩泥说，「显摆，像滚。」

刘婆端粥，仍多半勺：「试水别空肚。」韩泥点头。谢轻，恩重。

丑时末，他再摸木牌，牌歪，字丑——丑不要紧，渣正才要紧。

他低声对坛：「挂牌试水过了。记着。」坛腹温，贴掌，像应「试」。

第三日收摊，摊空，囊净——净像没来过，来过不要紧，牌在。

管事弟子验牌，牌歪，字丑：「续留三日，税照旧。」

韩泥交半文，声平。声平，才像末席该交的。

回丙九，不近廊下笑。笑在耳，不在心——心在坛，坛腹温一线。

他低声道：「试水过了。牌在。记着。」

更鼓沉，天边白一线。白里，坊市木棚色浅，浅像96听风那日。

挂牌试水三日，像97那三日，又像96那听风——风在，手在，就不滚。第一日空摊，空不要紧，空才像试水；试水不在客，在耳。耳听嘲，听「丑丹师卖晦气」，听不要紧，听全了，133才续。第二日递渣，第三日记簿「暂留」——暂留，够了；够了不在荣，在证。证渣正，证鼻准，证沉腐浮落齐。

落齐，白织月袖风再冷，冷也尖不过「正」字。正字在掌，不在牌；牌歪，字丑，人更丑；丑不要紧，渣正才要紧。铁无言立坊口，不近摊——不近，像不认；不认不要紧，眼在，眼是134兄弟线续的前奏。前奏在108，在134，在580从绶远钩；钩在，不叫。不叫，赖扒皮嗅不到；嗅不到，丙九不连坐。

连坐不要紧，133只挂牌试水。试过，134修栏，135袖风嘲，136层十五——层层积，才像140收束。收束在远，133只试。试过了，风改一半；一半改，142才近。更鼓沉，按木牌，牌凉贴胸，他低声对坛：「试水过了。记着。温像应「牌」——牌在，手在，路就不绝。」

丑时末，他再摸木牌，牌稳，稳像97挂牌那三日——三日过，风改一半。

坊市收摊，他立末，像石。石在，牌在，风过，风过不留痕。

第三日记簿「暂留」，暂留，够了。够了不在荣，在证——证渣正，证手稳。更鼓沉，他低声对坛：「试水过了。记着。」

坛腹温一线，像应「牌」——牌歪，字丑，渣正；正了，134修栏才像续，135袖风才接得住。接在忍，不在辩；辩，费舌。

散队后，他过外堂阶下，分拣半刻，指稳——指稳，路就不飘。

沿净，坛才藏得住。

粟壳香淡，香一字不落。

管事记簿，记细，是小胜。胜在忍，不在辩。

他望坛，低声道：「手在，路在。」坛不应字，只温。

白里，他按木牌，牌凉贴胸。

坊市尾巷风硬，他收摊如石，石在，就不滚。

验息后，他理怀内册页，页黄，字丑，丑而真。

铁无言声低：「别飘。」韩泥应声：「不飘。」

刘婆端粥，仍多半勺。韩泥点头，谢轻，恩重。

药峰分拣照旧，指稳，鼻下静一息。

帖在怀，怀沉，沉的是序，不是惧。

白织月袖风过，韩泥不辩。辩，费舌；舌要留来守坛。

赖福廊下过，韩泥只答：「在。」在，就不滚。

柱上条角白一线，三年约在怀。怀凉，掌温。

坡下石阶湿，他数息，数帖，数债序。

同棚杂役笑，笑在耳，不在心。

废渣巷日斜，他验渣，渣正，手才静。

侧门驿亭风硬，他听风，风过，记在心里。
