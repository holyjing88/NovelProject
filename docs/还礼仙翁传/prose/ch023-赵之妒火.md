# 第二十三章 赵之妒火

当夜，赵玄机叩门。

三下，不重，像执法堂铁牌敲在人心上。

莫长春开门，未请入，只让出廊下凳——不坐石阶，不坐主位，借门外夜风说话，像老习惯：重话不进屋，进屋就成了私恩，私恩难还。

赵玄机立而不坐，甲未卸，铁牌在腰，月光下亮得冷。他拱手，礼足，声却平，平里烫：「莫长老，手帕定身，全场皆见。弟子有一问。」

「问。」

「为何偏帮杂役、赠剑赠符、又当众辱魔少？」赵玄机目光不移，「弟子妒你，弟子认。可弟子更不懂——你若真靠女人上位，为何又救我于微末？若不靠，为何把礼散到每个人头上，唯独不散到执法堂？」

廊下远处，有脚步停了一停，又远去——林挽月在廊角，没近，像怕惊动一场不该围观的私话。

莫长春看他，像看一块被妒火炼过的铁，铁还硬，硬里已有裂：「你今夜来，不是问礼散不散，是问你自己心在哪边。」

赵玄机一窒。

莫长春续道：「老夫赠人，不为收。你若也赠，便懂——赠出去那一刻，最怕的不是对方不还，是自己先算计还什么。」

赵玄机指节白：「弟子……私扣培元丹，是算计。」

「知是算计，便还有救。」莫长春道，「丹还库了？」

赵玄机一愣：「长老怎知？」

「心在谁那边，堂上见过。」莫长春笑，「还库，是心往那边挪了一寸。一寸够了，别贪多，贪多又像私囊。」

赵玄机沉默良久，忽然问：「长老……手帕从何来？」

莫长春看他，像看一块终于肯开口的铁：「你问帕，不如问心。心还了库，帕从心来。」

赵玄机喉紧：「弟子仍妒。妒你当众定魔少，妒霍镇山记你的恩，妒顾小满也敢记你的册……」

「妒着吧。」莫长春道，「妒可以，嚼舌不行。嚼舌伤的是你的铁牌，不是老夫的九月。」

赵玄机忽然拱手，礼比往日深，深得像把脊梁弯出一寸：「弟子……不嚼舌了。」

莫长春点头：「不嚼舌，是还库之后第二寸。两寸够了，别贪多。」

赵玄机退，退到月影里，像把妒火收进袖，收得紧，却未灭。他走过廊角时，与林挽月目光一触，一触即分，像谁都没看见谁。

林挽月在远处廊角，心口复杂——她慕赵师兄，却也看见赵师兄在莫长老面前矮了一寸，矮得像终于承认：有些人不是演，是活得真。

她想起赵师兄白日里对弟子仍冷，夜里却来叩门，忽然觉得慕字轻了，轻得像一张纸，风一吹就破。她转身，未近，像怕惊动一场不该围观的私话，也像怕自己一近，就要在两个人之间选边。

廊下又有杂役路过，窃语：「赵执事从莫长老院出来，脸不凶了？」

「还库之后，人都软一寸。」

「软的是嘴，不一定软心。」

窃语远去，像围观一层，又退一层，留给院里一夜静。

韩铁山在赤焰营地听闻赵夜访，冷笑对厉无殇：「青岚门内斗，正好。」厉无殇不笑，眼里血丝：「内斗？莫长春送的是恩，不是斗。你不懂。」

赵玄机回执法堂，弟子问：「赵执事，长老说什么？」赵玄机只一字：「问心。」弟子面面相觑，面觑里，赵执事铁牌仍亮，亮得像终于肯照自己的影。

孙福在苦役地听说赵夜访，嘀咕：「赵家也软了？」顾小满冷眼：「软的是嘴，不是册。」

林挽月回房，把慕字写在纸上，又撕，撕得像撕一根将断的线。她忽然明白：赵师兄的礼也要还，还得比慕更难。

莫长春独坐，簿无字，却觉赵氏那一寸挪动，像善缘微涨，不涨在名上，涨在心上。

杖中竹纹一游，气音轻。

他望月，月白，像半块灵石洗过的光。簿边无新字，袖却暖，暖得像赵家妒火还没灭，却肯进门说话。

他低声，像钩：

「妒火未灭，够暖。小满那册子，该上台面了。」

魏无涯在远处檐角，扇开一线，观赵退，观林挽月立廊角，低笑一声，像看一枚子将落：「问心二字，比问帕难。」

院外秋雨意起，起得像东脉的云又低了，低得像妖讯将至，将至之前，先记在心里。

苏念慈隔窗听见院外脚步声，未出门，只把炉火拨旺——旺火，像替谁守一个不能说的秘密，也像替赵家妒火温一壶真话。

执法堂墙上，「问心」二字被弟子偷偷用炭描了一遍，描得浅，像怕被人看见，又怕看不见。赵玄机次日晨课见了，未擦，只令：「描浅可以，描歪不行。心字歪了，铁牌就歪。」

苏念慈隔窗未叩门，把一帖温肺散送院外石凳，帖下压细字：「妒火别烧肺，烧了谁夜里敲门？」

顾小满听闻赵夜访，停抄册，在册边添一行旁注：「某夜，赵执事叩三门，声平，未嚼舌。」旁注小，小得像给半块石再添一枚钉。

他低声：「问心之后，册子上台。上台之后，东脉集结。」

翌晨，执法堂开早课。赵玄机立于铁牌前，未抄门规，只令弟子各读「还」字一章——还库、还恩、还心。读得慢的，他也不多斥，只补一句：「读心，不读脸。」

弟子私下嘀咕：「赵执事变了？」有人答：「昨夜去问心了。问心之后，铁牌照影更亮。」

周德海夜得讯，讯说赵执事夜访莫长春，脸不凶了。他抚须，笑冷：「软一寸，不等于站队。软了，才好拱。」

柳青鸢闻讯，隔窗对练剑，剑穗扫过石栏，声低：「赵执事，门规面前，妒也要排队。」话落风里，赵若在，也会一窒。

苏念慈闭炉半炷香，怕妒烟进丹堂。她低声对炉：「妒火别烧丹。烧了丹堂先苦，苦了谁活？」炉中火稳，稳得像把真话炼成药性。

霍镇山营中听闻，骂：「赵家软了？」副将答：「还库了，不算软，算还。」霍镇山哼：「还了库，嘴还硬，才算真还。」

顾小满在杂役院，册页翻至孙福克扣末行，末行旁画半块灵石，石小，小里亮。她声平：「赵执事说问心，小满说问册。册真，心才真。」

林挽月窗前，慕字已撕尽。秋雨细，细得像一根将断的线终于断了。她忽然明白：真的人礼也要还，还得比慕更难——难在当众，难在还库之后仍妒，却肯进门叩三下。

赵玄机午后立铁牌前，看自己的影。影里仍有妒，妒里多了一寸踏实——踏实像半块灵石，硌得睡不着，却硌得对。他低声：「问心。两寸够了，别贪多。」

魏无涯檐角收扇，收前低笑：「问心之后册子上台。上台之后东脉集结。集结之后赵氏脸到——脸到，才是大还礼前奏。」

莫长春院中，杖中竹纹一游，气音轻，像问：「够暖了吗？」他笑：「够。妒火未灭，正好温真话。」袖空，一念出塔，簿边无新字，袖却暖，念：「小满那册子，该上台面了。」

东脉方向云低，风里带稻腥，像妖讯在远处磨爪。他抚袖，低声：「问心够了，下一本表上台。上台之前，雨再洗一遍石道。」

杂役堂新管事李姓，听闻赵问心，连夜把例份册摊开对照，对照到手指发白，嘴里念：「谁克扣，谁记在心里——长老的话，比孙福的鞭子先到。」

孙福旧党想借赵氏反弹，反弹半句，被顾小满册角一磕，磕得像半块石又亮一寸：「反弹克扣，也记在心里。」

妒火在暗，情报在册——册厚一寸，厚得像把赵氏的路，也记进去了。
