# 《还礼仙翁传》500万字 · 全书架构

> **篇幅目标**：约 **500 万字**（汉字）· **约 1250 章** · 均章 **3800～4200 字**（高潮章 5000～8000 字）  
> **笔锋参照**：《斗破苍穹》章回留钩、大会打脸、拍卖夺宝；《凡人修仙传》资源博弈、秘境层进、宗门政治；《斗罗大陆》战队磨合、大赛周、测定觉醒。  
> **核心不变**：**赠缘簿** 驱动一切成长；老头喜剧 + **馈缘俘获** + **多道侣纳绶** + **鸡犬升天 E14**。  
> **六步穿插**：①赠礼→②四维→③倒贴→④感恩→⑤纳绶→⑥鸡犬升天 · [`chapters/EXPANSION-穿插主线模型.md`](./chapters/EXPANSION-穿插主线模型.md)  
> **三系统+符录+阵法丹道**：[`08-道具灵宠洞府系统.md`](./08-道具灵宠洞府系统.md) · [`10-符录系统.md`](./10-符录系统.md) · [`11-阵法丹道系统.md`](./11-阵法丹道系统.md)  
> **笔锋**：**简洁、爽快**——短句震场、章末留钩；参照斗破围观反应、凡人谨慎博弈、斗罗团战节奏。  
> **分部细纲**：[`chapters/EXPANSION-五百万字大纲总索引.md`](./chapters/EXPANSION-五百万字大纲总索引.md)（十二部完整导航）  
> **与现有文档**：200 章为**不可删锚点**；50 万→100 万→500 万为逐级加厚；详见 [`05`](./05-百万字扩充方案.md)（百万）与本文（五百万）。

## 一、作品总览

| 项目 | 内容 |
|------|------|
| 中文名 | 还礼仙翁传 |
| 英文名 | The Returning Gift Sage |
| 类型 | 东方仙侠 · 老年逆袭 · 喜剧爽文 · **多线深情** |
| 主角 | 莫长春（七十二岁貌，修龄二百八十年） |
| 核心命题 | 赠人一礼，天道还你十倍；**有恩报恩，有仇报仇**；老了，反而看得更清 |
| 叙事节奏 | 赠礼驱动（核心）+ 三年一爆（斗破）+ 大赛周（斗罗）+ 秘境换图（凡人） |
| 读者承诺 | 每 **25 章**小高潮，每 **100 章**卷终，每 **300 章**地图跃迁 |

### 篇幅公式

```
500 万 ÷ 1250 章 ≈ 4000 字/章
锚点 200 章 × 5 倍加厚 = 1000 章人间主线
+ 150 章上界/情感深描
+ 100 章番外/单元
≈ 1250 章 → 终稿删冗余压至 500 万 ±5%
```

## 五、三部经典技法融合（只借结构）

### 5.1 斗破式

- **迟暮之约**（卷一/五）：厉讽→莫立约→玉佩震场  
- **围观震场**：每次缘箓转、越级战、拍卖压轴，必有短段「全场寂然后沸」  
- **章末留钩**：断在「簿字发烫」「帖将至」「寿元一月」  
- **炼药/炼器立身**：苏丹道六卷长线  
- **拍卖三日 + 暗拍**

### 5.2 凡人式

- **装穷三策**：藏阶、假赠、因果不爆  
- **坊市捡漏**：青岚坊市、天机城、不二斋  
- **秘境七层 + 禁地**：赠缘古冢、悔过廊、无赠渊  
- **宗门政治**：周德海、长老会、九府座次  
- **闪回慢揭**：第七部双线，百年人生

### 5.3 斗罗式

- **气运测定**：测气碑甲级  
- **气运七队**：指挥/锋/眼/盾/后/商/变数  
- **赠缘塔九层**：天试前内试炼  
- **大赛三周**：初赛/复赛/决赛各扩写  
- **缘契灵宠**：百年→千年→化形
