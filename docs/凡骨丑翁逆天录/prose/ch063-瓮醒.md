# 第六十三章 瓮醒

沉丹宗杂役房，夜深。

韩泥清完兽栏，洗手，水冷，指缝腥。五十九岁了——入门半年，须发更白，背更驼，独眼仍利。村里「晦气翁」换了名，叫「丑杂役」，骂法不同，味一样。

赖福今日又克扣，克半袋麸皮。丙九里仍有人喊他赖扒皮——叫法不同，味一样。韩泥不辩，只记。记在心里，记在坛边——坛在，仇在，恩也在。

克扣时，同屋小杂役笑：「丑老头，认命吧。」

韩泥「嗯」一声：「命在坛边。坛在，不认。」

小杂役一愣，没再嘴。嘴闭了，鼾声仍雷——雷不要紧，要紧的是丑时还没到，还没到，瓮还在睡。

他回屋。屋小，漏风，比泥岗泥屋强在不漏雨，弱在隔壁鼾声如雷。鼾声里，他摸坛沿，坛厚，厚得像祖辈擦过的那些年，像祖父老栓再擦过的那些年，像爹咽气前搭过手的那些年。

坛在。

泥岗村带来的腌菜坛，宗门不许摆，他摆床角，用破席盖，像藏私。私不在坛，在命——命跟了坛半生，丢不得。入门时，管事骂过，骂他带晦气，他「嗯」一声，席盖更严，骂不过席。

恩辱仇，全记在心里。恩密，辱浅，仇深。坛腹微温过几回，温得他知：坛在，命在，路在。

今夜丑时。

韩泥没睡。他盘坐，油灯豆，豆光照坛，坛影歪，歪得像丑翁。

编筐时藤条扎破指，藤刺旧疤未合，血珠饱满，他甩手，血甩到盖坛的破席上——席旧，吸了血，血渗到坛沿。

不是有意滴，是活计扎的，是血自己找路。找路，像泥岗村那夜梦里朱红又现——现不现，他都不躲。躲了，像怕自己的命。

他揭席，坛口破布扯开。

坛里空，坛壁蒙尘，内息尽敛，瞧不出半点仙气——本该如此，腌菜坛能有什么仙气？泥岗村那夜丑时，坛温一线；入门后，坛时凉时温，像等人来碰这一碰。

血从席渗到坛沿，一线，像泥岗村梦里的朱红。

血一落沿，坛壁颤。

极轻，像心跳，像有人在里面翻身——不是对血的颤，是对 **他的** 血的颤。

韩泥独眼一紧，心口突跳。跳里翻出爹临终那夜：铁山手搭这坛沿，嗓哑：「坛别扔。韩家就剩你，坛在，根在。」

又翻出祖父老栓擦沿的冬夜，擦得沿滑，滑得像路；再翻出更早祖辈口传的「守瓮」二字——字不传外，传血，传擦沿的手，传一代代不肯扔坛的倔。

三十八年了。他以为根是空话，是穷人家安慰绝户的话。

坛沿蒙尘里，浮出极淡一道影，像篆，像字，像半个「韩」——一闪，灭。灭得他以为眼花，可掌心跟着烫，烫得像血脉里有什么，与坛底沉睡了不知多少年的东西，对上了。

对上了，才叫缘分。不是随便谁的血都能碰，是韩氏祖辈一代代擦过的沿，是韩老栓再擦了一辈子的沿，是韩铁山搭过手的沿，是他五十八年每日再擦的沿——守瓮人的血，碰守瓮人的瓮。

掌纹里像有细线亮起，亮得不艳，像灶下灰里一点红，红沿脉走，走到腕，走到心口——心口不燃，只温，温得像叶汤那一点深，又像心记里「先还」二字被坛应了一声。

「嗡——」

一声闷，非雷非钟，在颅骨里响。韩泥独眼一花，见坛底水光聚，聚成一线，线爬壁，爬出符形——符形丑，歪，像孩童乱画，却亮。

亮时，他竟看得分明：符不是外来贴上去的，是从坛壁里 **长** 出来的，像沿上那半个「韩」影被血喂饱，喂出第二笔、第三笔，笔笔短，笔笔钝，钝得真，真得像丑翁的骨。

符形沿壁游走半圈，圈口如瓮唇，唇开一线，一线白光吞吐，吐的不是仙气，是 **认** ——认他掌按处，认他五十九年擦沿的手，认韩家最后一脉守瓮血。

亮一闪，灭。

灭前，符形投在窗纸上，影丑，影真，真得像有人把「瓮」字拆开了给他看——看了，仍不全，全了反吓人。不全不要紧，认主要紧。

坛口吐出一口白气，白气冷，冷里带药香，带泥腥，带腌菜坛旧日的酸，还带一丝极淡的、说不清的古气——像很远的仙界，有人争过的那只瓮，在凡间腌菜坛的壳里，睁了半只眼。

韩泥独眼盯住坛壁，壁里水光一线，线爬，爬成符形，符形丑，歪，像孩童乱画——画的是沿，是沿上极淡的「韩」影，影一闪，又沉入蒙尘。

他试着换指按，未破皮的指腹贴沿，符不亮，光不应——应了，仍是那道血线。血线像钥，钥不配，瓮不开；钥配了，瓮才肯醒。

血脉认主，不是喊名，是沿上五十八年擦出来的认。

他手背起皮，像被火舔，又似冰咬。他没躲，躲了，像怕。

瓮醒了。

不是梦，不是丑时幻觉。坛壁符形隐去，余温灼掌，掌纹里像多了一道路——路无字，路在，路通向还恩，通向省亲，通向泥岗那碗还没还尽的叶汤。

韩泥掌心按坛，低声道：「醒了？」

无人答。答他的是坛——坛沿一线温光，温得像爹的手又搭上来，像祖父在远处「嗯」了一声。韩泥不追问，只按实，按到掌烫，烫得踏实。

里屋，兽栏那边有夜鸟叫，叫一声，哑。

臾墟子是否在坛底，是否在将来开口，此刻不知。此刻只有坛，坛醒，气吐，命牵——牵的不止是他一人，是韩家祖辈口传擦沿不知几代人，牵到今夜，牵到他这最后一脉守瓮血。

外头鸡未鸣，狗未吠。沉丹宗夜禁脚步远，远得像另一个世界。钱戾衡的名字在远处，在玉牌里，在三年约里——约还在，瓮先醒，醒得不早不晚，醒在血缘碰沿那一瞬。

韩泥把席盖回，盖严，像盖一个刚出生的秘密。秘密不能叫，叫了，赖扒皮来，管事来，煞骨门旧债也来。

他擦掌血，血入布，布黑，像符。符丑，真。

独眼望窗，窗纸白，白将亮。

「记着。」他对坛说，对自己说，「瓮醒了。爹说的根，原来在这儿。路还长。汤还要还。」

坛底余温，贴掌，贴到丑时将尽。尽前，他听见极轻一声，像叹息，像回应，像瓮里有人说了半句听不清的话——话里无字，只有暖，暖得像血脉认主之后，至宝仍肯叫他「腌菜坛」，不吓他。

韩泥不追问。追问像少年。他只按掌，按实，按到掌烫，烫得踏实。

真名什么，仙界谁争，他不懂。懂的是：这坛是韩家传的，这血是韩家的，醒了，便该活长些，好还恩，好走宗门路，好等到哪一日，坛自己肯把名字说全。

隔壁小杂役翻身，梦呓半句：「……啥光……」

韩泥盖席，席严，声更低：「你看错了。睡。」

小杂役哼一声，又鼾。鼾雷里，韩泥按掌不松，掌烫，烫得像释——克扣是压，瓮醒是释；释不在喊，释在认，认主了，路才长。

他低声对坛：「爹，根在这儿。我活长些，还恩。」

坛底极轻一颤，像「嗯」，像祖父远处应，像沿上符影又浮半瞬，半瞬里无字，只有暖。

韩泥不追问。追问像少年。他只按实，按到丑时将尽，尽前，掌纹那路还在，在就不怕明日麸皮再克——克的是粮，克不走瓮。
