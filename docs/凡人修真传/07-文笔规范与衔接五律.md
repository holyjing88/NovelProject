# 文笔规范与衔接五律

## 简洁爽快

| 规则 | 说明 |
|------|------|
| 短句 | 8～15 字为主；战斗更短 |
| 一章一事 | 不跨两个大场景 |
| 对话 | 占 30～45%；带刀，少解释 |
| 爽点 | 章末 500 字内必钩 |
| 情感 | 写动作不写空话（「他别过脸」胜过「他心中复杂」） |

## 简洁爽快 · 情感写法（v3）

| 类型 | 写法 | 禁止 |
|------|------|------|
| 男欢女爱 | 吻、抱、抵额、传药、帐暖灯灭 | 低俗直写、猎奇 |
| 低谷 | 动作羞辱、读者替主角急 | 空话励志 |
| 担当 | 「我娶」「别来连累你」「去修仙别守我」 | 长篇心理 |
| 忠诚 | 挡刀、断臂、回归 | 无铺垫洗白 |
| 背叛 | 440裘叛、445萧暂叛 | 脸谱化 |
| 恩施 | 翻册、蘸血、划账 | 同章杀人 |
| 报仇 | 510诛、258逐 | 同章温存；**自刎/自尽** |
| 生死 | 逃、诈、拖、拉垫背 | 无意义赴死、羞惭自裁 |

### 示例

❌ 陈寻心中充满了复杂的情绪，他不知道自己该如何面对沈清弦。  
✅ 陈寻别过脸。「别来。」他说，「连累你。」

❌ 沈清弦美丽动人地倒贴上来。  
✅ 门缝下塞进来一盏灯。没人说话。

## 衔接五律

1. **伏笔 80 章内回收**（特殊长线如青冥瓶除外，须标注）  
2. **状态携带**：境界/伤势/恩施/因果/道侣/绑缘  
3. **背叛报恩前三章铺垫**  
4. **换地图前明示未了人事**（部间衔接栓）  
5. **每 20 章核心情感互动**

## 上下文章节衔接检查表

写每章后自查：

- [ ] 上章末钩本章首句是否承接  
- [ ] 境界/伤势是否与上章一致  
- [ ] 八要素状态行是否正确（含馈缘、符录【品·页】）  
- [ ] 恩施笔数是否正确  
- [ ] 因果阳/阴是否合理  
- [ ] 272 前无绑缘/无合击；420 前无契缘合击
- [ ] 420 前无契符；528 和合符需 420 契缘后  
- [ ] 时间线（日/月/年）是否跳跃合理  
- [ ] 新人物是否已铺垫  
- [ ] 章末是否有钩  
- [ ] **无自刎/自尽**；绝境是否先谋活或拉垫背

## 活着优先（全书硬规）

> **活着最重要。** 禁止以自刎、自尽、自裁、服毒畏罪等方式结束性命（含反派）。

| 情境 | 写法 | 禁止 |
|------|------|------|
| 败局 | 逃、降（诈降）、拖、求饶再反咬 | 羞惭自刎 |
| 必死 | **拉仇人垫背**、同归于尽（先确保仇死） | 独自赴死 |
| 陈寻 | 不算计赴死；用符/阵/队友换命 | 圣母殉道 |
| 沈720 | **寿尽坐化**（拒延寿，非自裁） | 与440混淆 |

**440 标准写法**：裘横秋叛投赤渊杀回，**欲拉岳含章垫背**；岳含章力战，**反拖裘同陨**（裘先死）；陈寻未赶到，阴因 +20。

❌ 裘横秋自知罪孽自刎谢罪。  
✅ 裘横秋断臂仍要遁，反手一掌拍向岳含章天灵——要拉垫背。
