# 第三十七章 丹堂大炼

七夜，丹堂火不灭。

苏念慈按残页配药，控火，凝丹，每一息都像在刀尖上走。外门弟子在阶下轮班，踮脚，不敢噪，像等一声「成」，好回去讲戏本——这一回戏本不敢乱添，添多了，丹堂首座能把他们炼进药渣里。

第三夜炸炉险象，炉盖跳起，药烟苦辣，弟子惊退，有人喊：「首座！」

莫长春未进门，只在门外廊柱旁拄杖，把一包药渣递给守门弟子：「送进去。渣是心意，不是废料。」

苏念慈接过渣，入炉，火竟稳了，稳得像有人把乱麻理成线。她满头汗，却笑，笑里倔：「师兄……你又使诈。」

「不使诈，」门外声音平，「你肯收渣，炉才肯成。」

第四夜，第五夜，第六夜。顾小满送粥，柳青鸢遣人送阵材，霍镇山派人送盾粉，像全宗都知道丹堂七夜，不是苏一人之事。

赵玄机执法堂路过，停半息，未讽，只拂袖走。走前对守门弟子低声：「执法堂轮值，闲人退三十步。」

弟子窃语：「赵执事替丹堂清场？」「太阳打西边出来了？」

第六夜末，赤焰方向腥风一线，像夜袭在提前嗅。苏念慈眸光一紧，莫长春门外杖点地，声轻：「你守炉，老夫守门口。妖来了，也是霍老头先叫。」

阶下围观弟子听见，窃语里怕，也敬——怕夜袭，敬的是长老七夜不进门，像把成丹的戏，全留给丹堂。

第七夜，地火阵鸣变，像炉心有人敲钟。

苏念慈立阵眼，灵力尽灌，丹田里有什么东西将破未破，像一颗种子顶破土。她唇白，眼亮，亮得像要把三十年压进这一夜。

莫长春杖点廊石，一声，两声，像给七夜敲结案鼓。

「成不成，」他低声，「都在这一炉。」

炉内，丹香起，温甜里掺雷息，像七月暑气被滤过一遍。阶下弟子齐呼，呼又被苏念慈一喝压住：「未成，退！」

呼声退，心不退，像全宗都在等下一息。

雷小，像天公肯给丹堂留一点面子——随后，丹成。

药香先于人声到了山门。苏念慈捧托盘出，袖口沾炭灰，眼底血丝，可眼睛亮得吓人。

金丹卧白绸，拇指大小，淡金薄膜流转。

莫长春仍立门外廊柱旁，不进门，像七夜前一样。他看着她，没说话。

苏念慈走到廊下，举托盘，让金丹反光落在他须眉上：「师兄。成了。」

全场一寂，继而炸开——炸在阶下，炸在廊外，炸在每个等着看「扑怀」的人眼里。

第六夜，赤焰腥风一线，弟子怕，怕里仍不肯退阶。苏念慈喝：「怕就守炉！守不住，才怕！」

顾小满送粥，粥温，温得像杂役也押七夜。赵玄机清场，清得像执法堂终于肯护丹。

第七夜雷小，雷前全场寂，寂里弟子屏息，屏息像围观，围观等延迟爽点——成丹那一声。

苏念慈捧丹出，全场炸，炸在「成了」二字，炸在莫长春仍立门外不进门。金丹卧白绸，淡金流转，像晨露，一碰要碎，却实实在在。

柳青鸢脚步在远处停，停得像体统也将至。

魏无涯在檐角观成丹一瞬，扇合，低笑：「重的推给她，轻的才送得出。送得出，才还得起。」

阶下弟子炸开，炸在延迟那一息——成丹先香后人声，人声先「成了」后扑怀，扑怀之前，全宗都在等。

莫长春仍立门外，不进门，像七夜前一样。他看着她，没说话，眼里却亮一寸，亮得像九月里又多了一盏灯。

下一息，她要还礼了。柳青鸢的脚步，也在远处停了一停。

大因果还在远处，像天试，像赤焰——七夜大炼只是小成，金丹成才是大还礼的前奏。

下一息，她要还礼了。柳青鸢的脚步，也在远处停了一停——停得像体统也将至，停得像全宗都在等延迟的那一声「成」。

阶下弟子齐呼，呼又被苏念慈一喝压住：「未成，退！」呼声退，心不退，像全宗都在等下一息。雷小，像天公肯给丹堂留一点面子——随后，丹成。

苏念慈捧托盘出，袖口沾炭灰，眼底血丝，可眼睛亮得吓人。金丹卧白绸，拇指大小，淡金薄膜流转。莫长春仍立门外廊柱旁，不进门，像七夜前一样。他看着她，没说话。

「成不成，」他低声，「都在这一炉。」炉内，丹香起，温甜里掺雷息，像七月暑气被滤过一遍。

第六夜末，赤焰方向腥风一线，像夜袭在提前嗅。苏念慈眸光一紧，莫长春门外杖点地，声轻：「你守炉，老夫守门口。妖来了，也是霍老头先叫。」阶下围观弟子听见，窃语里怕，也敬。

第七夜，地火阵鸣变，像炉心有人敲钟。苏念慈立阵眼，灵力尽灌，唇白，眼亮，亮得像要把三十年压进这一夜。雷小，像天公肯给丹堂留一点面子——随后，丹成。

围观弟子在阶下踮脚，踮得像等「成了」二字——二字未出，先出的是药香，香先到山门。有人低声：「莫长老七夜不进门？」另一人答：「不进门，才成得了戏。」
