# 第二十六章 血祠

旱到第三月，井更深，水更浑。浑水照脸，脸瘦得像全村。全村瘦，瘦不过一张榜——泥岗村采药榜，韩泥名在末。末得稳，像旱里一滴水，滴少，也是水。

韩泥早起担空筐去集尾，卖出十二文，价落掌心，像告诉自己：旱里还能活一日。回村路过村东废祠，祠无人祭，偏有人拜。拜的是泥坛，坛口红得像干血。干成壳，壳上有人新刻字，刻得歪：「天罚晦气，血祠开，雨自来。」

韩泥过祠，不拜，只看。独眼眯起，见坛底湿——旱成这样，坛底仍潮，潮得怪。怪，他不说，记。

午后他扫庙换面。帚过碑肩，尘起又落，落得像替前人收一句。面实，比邪实。实了，人才走得稳。

童谣丫念新谣，念得小声，被娘捂嘴：「血祠翁，晦气翁，送汤送姜天不容……」

谣是新的，骨还是旧的：晦气、牵扯、绝户。骨上贴了新肉：血祠、天罚。贴得像真，真里带邪。

周嫂传得像真：「叶家公审赢了，天仍不雨，是血祠示警！」

赵绣春门开半寸，声尖：「看吧！我就说牵扯晦气，连老天都罚！」

尖声顶不过旱风。旱风刮祠，泥粉落下，像灰，像祭。韩泥没辩。辩了，像坐实他懂邪。他懂坛，不懂拜。拜是别人的心，心是坑，坑要绕。

里正带人过祠，踢倒新刻字，骂：「再刻，抓人。」拜的人散得快，像没拜过。快归快，心还在——心在，邪苗便还在。

韩泥在暗处看，不看脸，看脚。脚虚，心虚；心虚，容易拜。容易拜，便容易被人推去血祠当枪。

有个扛锄的汉子散时回头看他一眼，像要问又咽回去。韩泥先开口：「旱是天。拜坛，雨不会因为你跪下就来。」

汉子喉结滚动：「可村里说……公审赢了还不雨，是老天不肯。」

「老天肯不肯，你问井。」韩泥说，「井浑了，人更得把嘴擦干。嘴脏了，坛红了，雨还是不来。」

汉子「嗯」了一声，肩上锄沉了沉，走了。走得不甘，却没再回头拜。

夜里他低声道：「邪在祠，恩在心。心不乱，人不乱。」

沈枯芽娘咳得夜长。芽儿点灯，灯豆小，像怕黑。黑里，韩泥送半帖药到豁口，不放声。声惹祸，药不惹。

叶家药铺灯也亮一瞬，亮得像秤还在。秤在，谣便不能全吞村。叶青禾隔帘咳一声，咳短，像告诉全村：药铺开门，便是不拜血祠。不拜，秤才准。

韩泥坐坛前，独眼闭一息——不是睡，是记。记邪苗，记旱，记公审后仍不散的嘴。坛腹微温，像叶汤应了半圈。圈浅，真；真了，将来拔苗才有据。

风过废祠方向，像有人叹气。叹的是旱，不是他。他回屋对坛低声：「你应你的，我记我的。记清了，将来不冤。」

坛腹微温一丝，像答。答无声，够。

天亮童谣又起，起得低。低，比高声更毒——毒在孩口，孩口不懂，只学。

韩泥路过，停，对丫说：「谣会旧，人会老。老的不是你，是嚼谣的嘴。」

丫眨眼，半懂。半懂，够。赵绣春门内又摔碗，摔得脆，像不服公审，也不服血祠。不服，嘴还在；嘴在，邪苗还在。他走回村道，把邪苗圈在心里——圈住，不急拔。急了手乱，乱了中招。

陈姑夜送一块干饼，饼硬，硬得像冬恩：「别信血祠。信坛，信心，别信嘴。」

韩泥收饼，掰半，半给沈家豁口，半自留。留，不是饿，是分清。分清了，心不乱；心不乱，邪苗立不起来。

叶青禾隔帘咳，咳短：「别拜血祠。拜邪，秤就停。秤停，全村沉。」

韩泥「嗯」一声，不近帘。不近，是护。护她刚松的气，护公审后那一寸安。

第三夜，废祠方向有人低拜，拜声碎，碎像旱里求雨。里正带人灭灯，骂：「再拜，抓！」

拜的人散时嘟囔：「公审赢了，天仍旱，不拜拜谁？」

韩泥在暗处听，不辩。辩，像同流。他只回屋，对坛低声：「旱在天，拜解决不了。」

沈枯芽娘咳得窗纸颤。芽儿点灯，灯豆光里韩泥送半帖药到豁口。芽儿怯问：「韩爷爷，血祠真能让雨来？」

韩泥蹲低，声平：「雨在天，不在坛。坛是别人的心，心是坑。你别跳坑，跳了，叶姑姑药白配。」

「那他们为什么拜？」

「嘴怕旱，怕便找坛。」韩泥说，「你怕黑，点灯；他们怕旱，点血。血点错了，村就沉。」

芽儿点头，点得像米粒。米粒旁，韩泥心里又定半分——定，不是信血祠，是不让孩口学邪。

叶青禾隔帘又咳一声：「别往祠去。去了，里正连坐，连坐连到你编筐的手。」

韩泥在门外：「不去。我看脚，不拜坛。」

「看脚？」

「脚虚的人，易被人推去当枪。」他说，「枪不是我。我是编筐的。」

帘内药杵轻响，像秤还在量活路。量给全村，也量给怕血祠的人——怕归怕，别拜邪。

韩泥低声道：「邪苗圈住了。圈住，不急拔。」

坛腹微温一丝，应旱，不应拜。不应拜，路不歪。

里正又贴柱角告示，告示末行写：「八日后上路，误者不许」。韩泥路过，独眼辨字，字丑，却真——真在尺，尺量还路。

丑时，他把坛沿又擦一遍，擦得像把血祠谣从手上洗掉。洗不掉谣，洗得掉拜意——拜意不进心，心就不乱。心不乱，八日后上路，肩才不沉邪。

沈枯芽隔巷探头又缩。韩泥隔巷应，声哑：「别拜血祠。拜邪，叶姑姑药白配。」芽儿点头，跑远。

夜里井绳又响，响得短。有人打水，有人低骂天旱。骂完，仍有人望废祠方向多看一眼——看一眼，邪苗就还活。韩泥听着，不拦嘴，只把明日要带的干粮又塞紧半分。干粮紧了，心也紧半分：紧着，才不被谣牵走。

村狗哑叫一声，像血祠谣也哑了半寸。里正柱角又添一行小字：「误期者，除名。」除名像滚，他不滚，只记尺。

旱在，邪在，恩也在。人不拜邪，人就能活——活长了，才还得起叶汤，还得上泥岗这几笔恩。

张麻婆路过废祠，啐一口：「拜邪不如拜秤。秤在，娃咳才治。」韩泥听见了，不点头。点头像拉帮。他只把邪苗圈在心里——圈住，不急拔；拔急了，手乱。

井台旁又有婆婆念叨：「血祠一开，雨就来……」念得碎，像把渴念成咒。里正恰好路过，铜锣轻磕门框：「再念，记名。记了名，县衙问起来，别怨我。」

婆婆噤声。噤声不代表信断了；信还在喉咙里打转，只是暂时不敢出嘴。

韩泥立在一旁，把这噤声也记了。记不是记仇，是记：旱里的人心，比旱更深。

夜里他回屋，把采药袋擦净，装半截干粮，干粮旁塞一截布条。布条不长，够路上勒臂、勒坛。勒实了，才像个要走的人。

陈姑夜过巷，棉袄领拢紧，停门缝外，没进。「韩翁。」她嗓沙，「别信血祠。」

韩泥没开门：「不拜。坛在角，角稳，人不虚。」

「虚了呢？」

「虚了就容易跪下。」他说，「跪下拜邪，站起来还债就难。」

陈姑步远，远里像叹——叹轻，像也给全村留一口气。气在，秤才准得响；响完，要么叶家秤还在，要么全村嘴更毒。那一口毒，明日还要试人。柱上「八日」在风里不动，他按着心口，只记一句：人不拜邪，八日后上路，肩才不沉。沉的是旱，不是命。命还在编筐的手里，手里有筐，心就不跪邪。
