# 第二十七章 第二次妖讯

坊市归途，镇妖司急帖至。

霍镇山亲来传帖，甲未卸，喘粗，马汗溅土：「东脉边村，小妖群再袭，苦战半日，请长老援！」

莫长春瞧帖，笑：「霍老头，你又喊。」

「不喊不行！」霍镇山道，「这一回妖小，却多，像赤焰试探阵脚。弟子伤七人，民屋焚三间，再拖，根就摇了！」

二人至边村，炊烟乱，犬吠绝，小妖群伏墙头，眼绿，爪利，比青狼弱，却像蝗，多。墙头、树梢、草垛后，绿眼连成线，线像赤焰投过来的一根针，专扎青岚门刚稳的阵脚。

村民缩在井边，孩子哭，哭被捂在嘴里。霍镇山盾亮，苦战，吼声震墙，墙震，妖不散，反扑更急。

弟子惊呼：「霍长老，妖太多了！」

霍镇山骂：「多就杀！镇妖司怕多，还是怕赤焰笑？」

围观村民呆住，像又看见那日边村得救，却不敢信——九月老头何时到？

莫长春杖至，不急，杖点地，竹影起，扫、缠、摔，像青狼袭村一幕复刻，却更快，像老人赶鸡赶多了，手熟。小妖被竹影卷起，摔进田里，摔进沟里，绿眼碎成一片慌。

村民跪谢，谢又被霍镇山吼回去：「谢镇妖司！功归司！」

莫长春摇头：「妖给你们，功给你们，人情记簿上。」

霍镇山愣，继而大笑：「就爱这句话！」

案结，执法堂补发曾被扣的培元丹例份，司库亲送莫长春院，瓶温，像迟来的正。

赵玄机立司库侧，面色平，低声：「长老例份，堂上已批。私扣之罪，弟子还库，不敢再犯。」

莫长春看他，笑：「还库好。库在，心在。」

赵玄机拱手退，退得硬，像把一句软话藏进袖里。廊下弟子瞧见，窃语：「赵执事给长老送丹？」「是堂上正份。」「还库之后，果然不一样。」

莫长春服丹，闭关三日。

第一日，脉沉；第二日，脉跳；第三日，脉开，像一扇门在筑基中期里推开一线——**筑基后期**。

出关时，须眉仍白，眼神却亮了一寸，亮得像九月里多了一盏灯。廊下弟子瞧见，窃语炸开：「破境了？」「靠培元？」「九月老头还能破境？」

柳青鸢亲来，递茶：「师兄破境？」

「丹是堂里正份，境是妖是霍老头逼的。」莫长春笑，「别高看老夫，高看了，又像演。」

柳青鸢松一口气，像东脉阵前又多一根能点的杖。

苏念慈在丹堂听见，炉火旺，旺得像替师兄喜，又酸，酸得像当年远处捏盏——她喜破境，酸破境因堂里丹，不因她炉边夜语。她咬牙，把酸炼进火里，火更稳。

夜，簿上浮字：

**除魔再一念，境开一线。**

莫长春合簿，望东脉，风腥。

边村老者跪谢，谢被霍镇山扶起：「谢镇妖司！」莫长春补一句：「谢自己没跑。跑了，根就断。」

执法堂贴告：补发培元例份，因除妖有功。弟子围告，围得像看迟来的正，正里仍有疑：九月老头靠丹破境，算不算本事？

苏念慈亲检丹瓶，瓶温，温得像堂里终于肯给长老一份不酸的甜。她低声对炉：「破境好。别死在丹前。」

赵玄机在告前立一停，未讽，像把「还库」之后的路，又走了一寸。

魏无涯观莫长春闭关，低笑：「境开一线，好让赵当众来问。」

出关时，廊下弟子炸开窃语，窃语里慕，也怕——怕九月老头还能破境，怕青岚门底又厚一寸。

他低声，像钩：

「境开一线，正好让某些人当众来问——问得越响，答得越巧。」

杖中竹纹一游，气音二字，像笑：

好。

大因果还在远处，像妖潮，像赤焰——破境只是小境，赵之初见才是大还礼的前奏。

莫长春合簿，望东脉，风腥。他低声，像钩：「境开一线，正好让某些人当众来问——问得越响，答得越巧。」

杖中竹纹一游，气音二字，像笑：好。廊下弟子瞧见破境，窃语炸开，炸里仍有疑——疑九月，疑丹，疑长老还能送几礼。

苏念慈在丹堂，把酸炼进火里，火更稳。她低声：「破境好。别死在丹前，别死在战前。」

霍镇山盾亮，苦战，吼声震墙，墙震，妖不散，反扑更急。莫长春杖至，不急，杖点地，竹影起，扫、缠、摔，像青狼袭村一幕复刻，却更快，像老人赶鸡赶多了，手熟。

围观村民在井边踮脚，踮得像等九月老头何时到——到之前，绿眼连成线，线像针。有人低声：「还能破境？」另一人答：「破境在闭关，不在嘴上。」

延迟的爽点不在杖扫那一瞬，在出关时廊下窃语炸开——炸开里仍有疑，疑得越久，赵之初见越响。
