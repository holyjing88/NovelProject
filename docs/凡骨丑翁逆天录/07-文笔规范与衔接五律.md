# 文笔规范与衔接五律（v5）

> **正文写作以策划文档为准**；执行入口见 **`18-正文写作主准`**。本章号、锚点、人名、馈缘链不得与 `05`/`16`/`17`/`15` 冲突。

## 简洁爽快

- 短句 8～15 字；一章一事；对话 30～45%  
- 爽点章末 500 字内落地（突破、报仇、夺魁、我娶）  
- 情感写 **动作与选择**，不写空话、不堆形容词  
- 恩仇章标题可直给：「还刘婆」「诛戾衡」「我娶」

## 起点留存·三章法则（评分优化 v15）

| 章 | 必达（读者留存） |
|----|------------------|
| **第 1 章** | 五十八岁丑翁+泥岗村+漏雨泥屋+「晦气」村嘲——**3 秒内记住人设** |
| **第 2 章** | 生计艰难+**腌菜坛**（凡相·鸿蒙九劫瓮）——金手指 **物理在场**（瓮眠：蒙尘无光/内息尽敛/如凡物） |
| **第 3 章** | 叶青禾送汤藏姜——**全书情感与眷升总伏笔**，章末钩：韩泥**心里记恩** |
| **第 10 章** | 读者能复述「七笔恩」中至少 3 笔 |
| **第 30 章** | 小爽落地（24 证白/30 赠药） |
| **第 63 章** | 瓮醒——第一部中段金手指兑现 |

**压抑节制**：连续压抑不超过 **2 章**；第 3 章起每 **5～10 章** 必有暖/爽/钩（见 `05` §起点留存表）。

## 开篇加速·第1章三段式（v16 · 破「开篇慢」）

| 段 | 字数约 | 必写内容 | 目的 |
|----|--------|----------|------|
| **首段** | 300 | 五十八岁+麻脸跛足+村嘲「晦气翁」 | 3 秒记住人设 |
| **中段** | 800 | 编筐/采药+漏雨泥屋+**腌菜坛**（蒙尘如凡物） | 穷与瓮同在 |
| **末段** | 400 | 村童绕远或赵绣春一句辱+韩泥**默记恩辱** | 钩：这仇/这恩要记 |

**第 1 章禁止**：长篇世界观说明、修炼等级科普、作者旁白说教。  
**第 2～3 章禁止**：推迟送汤；第 3 章末 **必须** 有**心里记恩**（无册、无蘸血）。

## 文笔节奏爽点强化（v18 · 冲10分）

> **目标**：文笔执行、节奏、爽点三维 ≥9.8。慢热部也要 **章章有小回馈**。

### 前30章五类小爽（每章至少落1类）

| 类型 | 写法 | 示例章 |
|------|------|--------|
| **打脸爽** | 辱人者哑火、离去狼狈 | 4童谣卡壳、5周嫂噎、8赵绣春嘴钝、10马春花绊脚 |
| **暗记爽** | 默记恩辱、心口一紧 | 3记汤恩、10立誓要还 |
| **暖恩爽** | 送汤/姜/袄/馒落地 | 3汤、5姜、7袄、8馒 |
| **手艺爽** | 筐卖出价、稳、重 | 8十二文、10十五文 |
| **硬扛爽** | 老头一句压回去，不少年吼 | 5井台回嘴、6赖爪牙走、9递水醒醉鬼 |

### 节奏硬规

1. **压释同章**：辱/谣/告示后 **500字内** 给小爽落点，不许把爽全推到下章。  
2. **章长**：**硬性 2000～3000 字/章**（起点编辑审稿习惯）；**禁止 4000+**（铺垫拖沓、节奏松散）。见 `07` §单章字数硬闸。  
3. **对话占比**：恩辱章 **≥40%**；忌连三段心理独白。  
4. **章末禁同质**：连续两章不得同以「坛凉心热」收束；换钩：价、声、物、温、步。

### 文笔执行硬规

1. **删破壁**：禁「读者若问」等元叙事。  
2. **动词优先**：用「划/摁/拎/啐/噎」替「觉得/仿佛/像」。  
3. **对话三句制**：关键场景每人不超过三句，句句带刀。  
4. **感官一样一处**：每章只强化一处感官（烫/臭/潮/硌），不堆。

## 文笔样板（宜仿句式 · 勿照抄剧情）

**开篇**：「五十八岁的韩泥蹲在漏雨的泥屋门口编筐，麻脸冲着日头，村童绕远走。」

**送汤**：「叶青禾把碗搁在门墩，没进门：『趁热。』韩泥没抬头，指尖却顿了一下。」

**心里立誓**：「韩泥指节发白，半晌才道：『这恩，我记死了。』」

**小爽还恩**：「培元散入掌，叶青禾愣住。韩泥只道：『汤还是热的，我记着。』转身便走。」

**章末钩**：「韩泥望宗门方向，心口又沉一分——三年，够还第一笔了。」

## 衔接五律

1. 伏笔 80 章内回收  
2. 状态携带：境界/伤势/**心记恩仇**/鸿蒙九劫瓮阶段/情感位 / 馈缘四维  
3. 背叛报恩 **前三章铺垫**  
4. 换地图前明示未了人事（部间衔接栓见 `04`）  
5. 每 20 章至少 1 次核心情感互动  

## 世界观硬规（全书）

本世界 **不许自刎、自尽、吞毒、自毁金丹** 等「自行了断」收场（反派、配角、正派皆然）。

| 规则 | 写法 |
|------|------|
| **宁战死** | 绝境也战到最后一口气 |
| **必拖垫背** | 临死前至少带走一名敌手（或重创敌首），写明谁与谁同归于尽 |
| **韩泥敬战死者** | 即便仇敌，若战死拖敌，韩泥不语嘲「死得窝囊」 |
| **例外** | **寿尽坐化**（如叶青禾拒延寿）、**出家**（如白织月）≠ 自刎，不属此禁 |

**440 范式**：言伯钧引魔护宗、战死并拖魔教长老垫背；裘默川叛战至死、拖魔教护法王同陨——**禁止**写二人自刎或吞毒。

## 恩仇分立

- **报恩章**：默记、送礼、安置、娶；不杀人、不虐仇  
- **报仇章**：清算、对决、逐、诛；不省亲宴、不温存原谅  
- 同章不恩仇双高潮
