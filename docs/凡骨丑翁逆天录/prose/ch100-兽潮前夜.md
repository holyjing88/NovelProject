# 第一百章 兽潮前夜

前夜到了。

兽栏外火把密，密得像星落了一地。人忙着，栏里小兽更不安。不安则腥重，腥重则瘴近。

韩泥蒙着刘婆缝的布条。布条粗，挡得住甜腥。甜腥从风里扑过来，他仍嗅出一线毒——毒细，细也记牢。

记不在舌上。舌一响，手就飘；手一飘，栏就倒。

铁无言分障，障是湿草叠石灰，叠法粗，却挡得一时。一时够了。

「你守东角。」铁无言说，「角漏，喊我。」

「喊。」韩泥应。

喊不为勇，为活。活着，才还得起恩。

赖福巡外围，声尖：「丑杂役，别缩！缩了记过！」

韩泥立在东角，不缩。不缩像桩；桩稳，赖福瞥一眼便走。

瞥是试——试他会不会逃。不逃，才配往后并肩。

夜深，远岭兽吼连成一片，像潮拍岸。岸是药山界。界内，外门弟子列成墙。墙心轮不到他，他在栏角。角小，小才藏得住丑翁清瘴的备——备在袖，备在经页，备在路里。

坡下叶青禾那盏灯未灭。未灭，他也不近。近了像连累。灯在就够；灯在，脉里培元那点温还在，她才敢立前夜。

他低声对怀中坛：「前夜了。别叫。」

坛不应字，只温。温像应「守」。

邻床同守，腿软：「我站不住。」

「软不要紧。」韩泥说，「扶栏，手稳。」

「稳得住吗？」

「稳一分算一分。」韩泥道，「稳了，有饭。」

丑时静半刻，香落齐，掌纹烫一线——烫像清瘴诀在经页里醒了半分。醒半分不够炼，够备。备足了，等潮至。

更鼓过，兽吼又近。近里一线甜腥扑面，他收紧布条。条湿，湿才挡毒。

同袍运石灰，石灰呛人：「你真不怕？」

「怕。」韩泥说，「怕也守。守完，才有饭。」

饭稳，恩不断。饭在刘婆粥里，也在栏里小兽那口轻轻的喘——喘轻，说明毒还没入深。

沈枯芽悄递水：「叶姐说，别逞能。」

韩泥接水：「嗯。」不接话。话在还，不在嘴。

管事弟子记：「东角韩泥，前夜在。」在，就不逃。

铁无言又送一包石灰：「撒角，别撒眼。」

韩泥撒匀，匀得像编筐勒藤。勒匀，障才稳。

白织月远廊立着，袖风冷，冷一句：「别乱。乱了拖内门。」

「不乱。」韩泥应，「角漏才喊。」

她没再接，火把光里只剩她唇线绷着。绷着也好，绷着说明怕的不只是杂役。

前夜第二更，韩泥换湿草两遍。两遍才挡得住甜腥。甜腥一过鼻，毒细仍在——细而记，记到往后压瘴那一步。那一步还远，今夜只守东角，只喊角漏。

秦霜木鱼轻敲：「前夜别慌。慌，手飘。」

韩泥心里应：不慌，只记。记东角，记备足，记潮至在前头。

同袍换更时抖着问：「潮真会今晚到？」

「到不到，都要当到。」韩泥说，「当到，手才不飘。」

赖福火折又照到他脸上：「还在？」

「在。」韩泥说。

「在就别睡。」赖福骂，「睡死了，连坐！」

韩泥不睡。桩似的立着，立到第三更。兽吼近一线，近不要紧，角还在。角在，灰香在指间；指间稳，撒才匀；匀了，毒退半丈。

小兽嘶声低了一点。低像活。活是小胜；胜在活，不在嘴。嘴要留来喊角漏——漏比内墙那点荣耀大。

坡下灯仍弱亮。灯在，他心口那笔恩也在。恩不跟今夜的潮搅成一锅：潮要挡，恩要还，各走各的。

他把经页边角摸了一下，页在怀，怀贴坛。坛温一线，像应「守」字。守字落齐，才接得住往后备足、撒线、压瘴那些日子。日子未到，今夜只做一件：东角不倒。

更鼓又沉，他按木牌贴胸，牌凉，掌却烫。烫在诀里，不给众目看。众目看兽潮，他看手——手不抖，障才稳；障稳，栏才喘得过这一夜。

邻床又抖：「明日呢？」

「明日潮若到，就撒，就压。」韩泥说，「潮未到，就守。守完，再谈还。」

「还什么？」

「该还的。」韩泥道，「还之前，先活着。」

第四更，火把油将尽，有人换把，火星溅到湿草上，滋地灭了。韩泥弯腰把草皮压实，压得像门缝不容风。风一进，甜毒就进；毒一进，小兽喘就重。喘重了，栏心先乱；乱了，外围再稳也白搭。

同袍嗓子发紧：「东角风大……」

「大就叠厚。」韩泥说，「厚一指，命多一息。」

「你鼻子还闻吗？」

「闻。」韩泥道，「闻是为了早喊，不是为了逞能去吸。」

他说话时布条已经湿透，湿里仍隔着一层涩。涩着好，涩说明灰香还在指缝。指缝有灰，撒时才连成线；线断了，角就漏。

铁无言巡到东角，蹲下看石灰边：「匀。继续。」

「嗯。」韩泥应。

「叶家灯你看见没有？」

「看见。」韩泥说，「不近。」

「对。」铁无言站起，「近了两边都麻烦。麻烦来了，障白叠。」

他走后，韩泥抬眼只扫坡下一瞬。灯豆大，豆大也够。够把「趁热」那笔按住，按在潮后去还，不按在今晚去疯。

赖福第五回照脸：「还杵着？杵成桩啦？」

「桩稳。」韩泥说。

「稳给你娘看？」

「稳给栏看。」韩泥道，「栏倒，你也跑不掉。」

赖福噎住，甩袖走了。走时有人偷笑，笑很快灭——灭得好，前夜不宜闹。闹一声，兽吼像应和；应和了，心更跳。

韩泥把跳按下去，按进脚板。脚板扣泥，泥凉，凉得清醒。清醒了才看见：潮还没砸到界上，砸到的是人的慌。慌不能传；一传，外围先垮。

沈枯芽远远又探头，被他眼风一扫，立刻缩回。缩对了。娃儿留在后头，才有人清明日的栏；明日栏清了，恩和饭才接得上。

他把囊里灰香又掂了掂，三只囊，一只页，一条清瘴路，路浅，浅也是根。根今晚不能拔，只能养——养在东角，养在不缩，养在「角漏就喊」。

邻床忽然低哭一声，又硬生生忍住。韩泥没劝。劝轻了像哄，哄不走兽潮。他只伸手把那人的手腕按到栏杆上：「握住。握着就不软。」

「……嗯。」

「嗯完换气。」韩泥说，「气匀，手才匀。」

火把再密一圈时，远岭吼声掀起一浪，又落下。落像试探。试探够了，真潮才到。真潮未到，前夜的考题却已写在每个人腿上：软不软，逃不逃，喊不喊。

韩泥独眼平着前方黑影，黑影里没有脸，只有风。风甜一分，他布条紧一分；风涩一分，他心就稳一分。稳着把更熬完——熬完不是功成，是把「在」字写进管事簿，也写进自己骨头里。

灰囊绳又松一扣，他重新系死。系死时指节发白，白得像把慌绞出去。慌绞干净了，剩下的才是活：守角，换草，听吼，不逃。

铁无言远处举手示意一下，示意像说：还撑得住。韩泥不挥手，只把姿势沉低半寸。沉低了，风头从肩上掠过，甜毒就少粘一分。

有人在西角喊渴，喊声发颤。韩泥没接喊——接了像离岗。离岗一瞬，东角就空；空了，漏进来的就不只是风。

他在心里把明日的次序又排了一遍：清障，补灰，听令，备足。备足之前，先把这一夜的「在」熬成实字。实了，才配听见潮真正拍岸的那一声。那一声还没到，东角的脚先不能软。

天边白一线时，火把仍密，兽吼仍近。近得像锤子悬在半空。锤子还没落下，东角却已经站实了。站实了，就不滚；不滚，明日备足那一步才有根。根在，栏在，坛在——在，就把前夜扛过去，把潮来那天的活路留在手里。
