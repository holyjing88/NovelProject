# 第十四章 孙福告状

孙福克扣半石事败，半月光景，杂役堂对他避如瘟神。

他恨顾小满，更恨莫长春——半石是长老随手赠的，赠出一个杂役的胆，也赠出他孙福在堂里站不稳的脚。

这日，他捧礼单进执法堂，礼单上是「孝敬周长老」的薄礼，话却是献给全堂的：

「周长老，孙福有冤。」

周德海抚须，佯惊：「何冤？」

「莫长春莫长老，」孙福跪地，声大得故意让廊下弟子听见，「勾引杂役顾小满，败坏门风！半石之事，非长老慈心，是……是私情！」

殿内空气一凝。

周德海沉脸，像怒，像喜，喜在终于逮到缝：「此言可大。可有实证？」

「实证在人心！」孙福叩头，「外门皆言莫长老老不正经，赠符赠石，独赠年轻女杂役。顾小满日夜出入小院——」

「够了。」

莫长春拄杖入殿，不急不慢，像来赴一场早约好的茶。

孙福一僵，仍硬撑跪姿：「长老敢做，不敢当？」

莫长春看他，像看一粒跳得高的芝麻：「你克扣杂役，顾小满举证，罚役三月，还没罚够？」

孙福脸涨红：「那是堂务！与门风何干！」

周德海叹声，拱火恰到好处：「莫长老，孙福话糙，可外头谣言确实汹涌。宗主清誉，执法堂清誉，都系于此。不如……请长老自辩？」

莫长春笑：「老夫不辩。」

周德海一愣：「不辩？」

「辩了，」莫长春杖点地，「就像承认谣言值得辩。不值得。」

他转向周德海，目光平：「请宗主裁断。柳青鸢若在，便让她来。老夫信她，不信廊下嚼舌。」

周德海眼底一闪，笑收半分：「宗主日理万机……」

「日理万机，也理门风。」莫长春截话，「你若急，先急东脉妖讯，别急着拿老夫九月寿元炒热闹。」

孙福膝行两步，想抓莫长春袍角，被杖尖轻点，点在他腕脉，疼得缩手。

「再叩一次，」莫长春低声，「执法堂地砖硬，膝盖软的人，别磕坏了。」

孙福骇然，竟真不敢再叩。

---

消息比孙福膝盖软得更快。

半日，「莫长老勾引杂役」传遍外门，比「老不正经」更难听。苏念慈在丹堂听见，药勺又敲碗沿，这一次她骂了：「谁再传，去炉边抄百遍《净火诀》。」

弟子噤声，她却心口发堵——她妒吗？妒顾小满能跪在小院阶下？还是妒谣言把师兄名节弄脏，而她只能在丹堂调火？

柳青鸢在宗主殿收到传讯，青霜剑在架，未出鞘，剑穗不动，人动。

她只问传讯弟子一句：「顾小满昨夜在哪？」

「浆洗房。」

「莫长老呢？」

「院中独坐。」

柳青鸢起身：「备轿，去执法堂。」

莫长春小院，门闭。

他独坐廊凳上，不坐主位，袖仍空，却烫，像有人把谣言烙在布上。顾小满在门外守了一夜，未敲门，像守半块灵石那样守一句清白。

天明，执法堂钟响。

莫长春拄杖出门，对顾小满道：「今日你别进堂。心已明，不必再溅血。」

顾小满红眼：「长老……」

「记着就行。」他笑，像那日赠符，「恩不用嘴还。」

他又看她一眼，声平：「你守一夜，守的是清白，不是老夫。清白在你，不在我。」

顾小满点头，点头像把半块灵石又握紧一寸。

他往执法堂走，杖点石阶，一步一步，像把九月走成证据。

识海塔壁金铭浮起半句：

**谣言起时，裁断将至。**

孙福告状那日，执法堂外比堂内还闹。

莫长春袖空，杖在塔中，不坐，只立。弟子围看，围看像盼他出一手，一手没出，出的是一句：「状告谁，先告自己。」

顾小满立侧，声平：「石是我接的，管事告长老，先告我。」孙福一窒，像被声平烫哑。

赵玄机铁牌在腰，笑浅，笑里空了一线——状若真，私赠坐实；状若假，赵氏也脏。

苏念慈送汤至廊外，不进门，像怕进门就坐实私。她低声：「告状告到丹堂，丹堂也脏。」

霍镇山声洪：「孙福无实证，镇妖司也看不顺眼。状告长老，先告自己。」

周德海抚须，笑和气缓：「孙管事，实证在人心，人心爱听脏的——脏的，堂上见。」

围观弟子屏息，屏息等宗主出殿——殿门开，开得像把全门命数，推到一个「裁」字上。

孙福告状后，杂役院分裂成两派：一派躲管事，一派护顾小满。

顾小满不分派，只声平：「石在我，状告我。别连长老。」声平里硬，硬像半块石。

赵玄机验状，状粗，粗里漏洞多，多里仍想咬，咬到塔意，塔意咬不动，咬到脸更白。

柳青鸢令：「状先存，真先查。」孙福跪，膝更抖，抖得像半块石太重。

周德海抚须，笑和气缓，却对孙福：「实证在人心，人心不当证。滚下去，等宗主裁。」

孙福一窒，像终于明白：拱火的人，也会撤火。

苏念慈在丹堂听见告状，药勺又敲碗沿，这一次她骂了：「谁再传私情，去炉边抄百遍《净火诀》。」弟子噤声，她却心口发堵——堵像妒，也像敬，敬顾小满能守一夜，妒自己只能在丹堂调火。

柳青鸢轿至执法堂，未入，先问传讯弟子：「顾小满昨夜在哪？」「浆洗房。」「莫长老呢？」「院中独坐。」她只颔首，颔首像把裁断二字，先在心里立好。

孙福跪，膝更抖，抖得像半块石太重——石是顾小满接的，状却告莫长春，告错了人，便告错了理。

围观弟子在执法堂廊下踮脚，踮得像等宗主轿至——轿至之前，谣言已脏，脏得比除魔符传得还快。有人低声：「孙福有实证吗？」另一人答：「实证在人心——人心爱听脏的。」

杂役院分裂成两派：一派躲管事，一派护顾小满。顾小满不分派，只声平：「石在我，状告我。别连长老。」声平里硬，硬像半块石。

赵玄机验状，状粗，粗里漏洞多，多里仍想咬，咬到塔意，塔意咬不动，咬到脸更白。柳青鸢令：「状先存，真先查。」

苏念慈送汤至廊外，不进门，像怕进门就坐实私。她低声：「告状告到丹堂，丹堂也脏。」霍镇山声洪：「孙福无实证，镇妖司也看不顺眼。状告长老，先告自己。」

周德海抚须，笑和气缓，却对孙福：「实证在人心，人心不当证。滚下去，等宗主裁。」孙福一窒，像终于明白：拱火的人，也会撤火。

莫长春拄杖出门，对顾小满道：「今日你别进堂。心已明，不必再溅血。」顾小满红眼，点头，点头像把半块灵石又握紧一寸。

顾小满立侧，声平：「石是我接的，管事告长老，先告我。」孙福一窒，像被声平烫哑。赵玄机铁牌在腰，笑浅，笑里空了一线——状若真，私赠坐实；状若假，赵氏也脏。

识海塔壁金铭浮起半句：**谣言起时，裁断将至。**

宗主轿影已在山门，轿未到，谣言先沸——沸到堂上，才肯裁。

山门道上，弟子三五成群，群声像潮——潮里有人盼裁，有人怕裁，有人只想瞧九月老头还能走多久。柳青鸢剑穗无风，风像从宗主令上借来，借得像裁断将至。

莫长春拄杖立山门道，声轻：「谣言沸，沸到堂上，才肯裁。裁之前，心要明。」

弟子渐散，散里仍回头瞧一眼灰袍袖空——瞧得像瞧一眼九月还能走多久。

宗主轿影已在山门，轿未到，谣言先沸——沸到堂上，才肯裁。裁的是名节，也是东脉三月之约。
