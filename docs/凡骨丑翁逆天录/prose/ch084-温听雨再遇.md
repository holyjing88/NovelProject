# 第八十四章 温听雨再遇

初识后第九日，药渣场验堆。管事弟子敲柱：「酉时验渣，丙九韩泥主分拣。分错，记名；分对，记功。」

药渣场风硬，温听雨立风边，裙素，袖净。她递帕，韩泥不接——接帕，像受礼；受礼像攀。他只道：「我擦掌。」擦在粗布上，布丑，布真。

记功不要紧，手在才要紧。手在，名在册，册在管事手，不在赖福秤上。灰起一线，韩泥指稳，鼻下有根。沉腐浮三字不落舌，辛左，腐右，假香上浮即剔——剔净，堆才净。验场围了几人，内门也有。内门不要紧，韩泥不抬眼——抬眼，像求认可；不求，只分拣。人群里，那一点冷香又在。

她忽然压低声问叶丫头，韩泥不接「嗅」——接嗅，像坊市谣。他只道：「鼻下有根。根在，手才快。」快，像85小成前那一验。

又在，像丹阁廊那日，像帚丝匀，像眉蹙那一息。温听雨立验场外，裙素，袖净。她本不必来，来了，像验场规矩——规矩在，她眼也毒。她瞥分堆，瞥韩泥指：「丙九也主分拣？末席手，别分错。」

韩泥不辩。辩，费舌；舌要留来报味，留来守坛。他应一声：「在。」

铁无言立桩，声低：「拈三渣，报三味。」

韩泥停，鼻先静。第一拈，苦长，蒿气；第二拈，辛沉，药山深灰；第三拈，涩重，废药残。他声哑报：「苦蒿、辛沉、涩废。三堆，不混。」

铁无言点头：「对。」

对，是小胜。胜在指，不在嘴。温听雨眉蹙一息，蹙完，却多停半息——停，像意外。意外不要紧，韩泥不迎——迎，像求；不求，只继续分拣。她低声问铁无言，声仍傲，傲里多一丝疑：「他……全对？」

铁无言扫她一眼：「笨人全对，灵人常漏。漏了，丹房不收。」

温听雨唇线绷一息，没再嗓。没嗓，小胜——胜在渣场，不在丹阁廊。验场散，她帚未拿，袖却净。净不要紧，她走过韩泥身侧，停半尺，停得像秤。秤不要紧，韩泥仍不抬眼。不抬眼，肩稳，筐不洒。她声不高：「你鼻下……静得怪。」

韩泥只答：「静在鼻，不在舌。」

答完，走。走稳，像初识那日。那日让廊，今日让疑——疑在皮，不在掌；掌对，小胜。她未追，袖风一转，走了。走，也像留一口问——问长，才接得住200愣；问短，不要紧，缘线才扎实。赖福晃秤，眼细：「温氏也问你？问出邪鼻？」

韩泥不接茬。接茬，全丙九饿。他低声道：「鼻下静，手在。」

刘婆端粥，仍多半勺：「药渣场遇温氏了？」

「遇了。」韩泥说，「不辩。」

「不辩怕什么。」刘婆哼，「手稳，饭稳。饭稳，恩不断。」

叶青禾名出，韩泥独眼一紧，又平。平，像踹台按实——掌在，名在，路在。他只答：「恩。」就一字，无「情」，无「叔」。无叔，才护叶名。

根在，手才快。韩泥点头。恩在粥里，不在温氏疑里——疑在皮，恩在粥，分列，不混。午后，兽栏仍腥。腥不要紧，铲下稳，稳里冷香回半息，回进沉腐浮——浮剔净，真才留得住。管事记簿：「丙九韩泥，验渣全对。」

记功，是小胜。同批少年脸热：「丑杂役也全对？」

韩泥不辩。辩，费舌；舌要留来清栏，留来守坛。丑时前，他廊下练半刻闭息。息在鼻，不在嗓。嗓响，香飘；香飘，温氏疑就成真。他按席角，席严，坛不露。坛腹温一线，贴掌，像应「再遇」二字。应完，他低声对坛：「再遇了。她疑鼻静。疑不要紧，记着。」

坛不应字，只温。温，像把冷香那一点淡，摁进空缘线。邻床悄问：「温听雨问你，你咋不抬头？」

「抬头像求。」韩泥说，「不求。手对，就够了。」

少年不懂「不求」，却懂「够了」，闭嘴睡了。秦霜木鱼轻敲：「缘来别迎。迎，心贪。」

韩泥不答，答在心里——心里恩辱分列。再遇是缘苗，苗浅，深在200；浅不要紧，苗在，就不滚。铁无言脚步远，停半息，声更低：「温氏眼会改。改在远，别盼。」

韩泥不答，答在心里——心里记一笔：再遇84，缘苗浅。浅，才藏得住。他独眼平，手不抖：「再遇了。全对。嗅药小成在明日。」

刃不亮，先记。记完了，才配辨性小成，才配三层暗修那一境。更鼓沉，沉里，他摸暗袋，袋沉，玉牌凉，愧字凉。凉不要紧，掌里远灯不熄——灯在，路就不绝。白织月远廊过，袖风冷：「温氏也看你？看也炼不出丹。」

韩泥不抬头：「看渣，不看丹。丹在远，渣在近。」

韩泥只答：「不求。记着。」

坛腹应温，像验场那一句「全对」。对轻，记重；记重了，才接得住嗅药小成。他低声道：「温听雨再遇。舌不留，手要稳。稳了，她眼将来改。」

韩泥只答：「验渣。管事记了。」

在，就不滚。不滚，缘苗才扎得实——扎实了，才接得住505副绶，那是更远的路。柱上条角白一线，冬深，风硬。风硬不要紧，鼻下有根——根在，就不滚。再遇后第二日，外堂洒扫，阶净，栏净。温听雨巾湿，水净，二人仍各做各的。她忽然问：「叶渣另堆，谁教的？」

「鼻下根。」韩泥说，「根在，苦辛并一线，并一线像叶家药远路飘来半息。」

温听雨愣半息，愣不像嘲，像真愣。愣完，她收巾：「闻渣，不闻人。好。」

好字轻，记重。记她第二回停步，记她验手，记她傲里松的一线。管事验洒扫，皆净，记功。记功轻，路重。赖福立远处，眼细：「温氏也跟你说话？」

韩泥不接茬。舌要留来鼻，留来守坛。刘婆端粥，仍多半勺：「再遇别攀。」

「不攀。」韩泥说，「各做各的。」

铁无言声低：「她敬才，不敬貌。才在渣，不在嘴。」

韩泥应声：「嘴稳。」

午后药渣场十堆，全对。全对，温听雨远廊一眼，一眼也像验。秦霜木鱼一声：「缘苗浅，心别贪。」

韩泥不答，答在心里——心里恩愧仇缘四分列，不混。黄昏，角柜锁实，样十包。锁住了，才不示人。邻床问：「温氏会不会帮咱？」

「帮不帮，看手。」韩泥说，「手飘，谁帮都滚。」

丑时前，廊下练半刻鼻下静。掌纹影颤，颤像三层将破。他低声对坛：「再遇了。嗅药小成在后，不叫。温，像第二回「手稳」摁进恩苗。更鼓沉，独眼平：「再遇了。小成在后，三层丑时破。」更鼓沉，沉里，韩泥按木牌，牌凉，掌温。廊下风硬，他把木牌系紧一线，系得像把仍末摁进掌心。

同批少年远避，避像避晦气，晦气遮瓮，遮住了才稳。编筐角藤刺仍在，他偏手，刺浅些，沿才净，坛才藏得住。管事夜巡验棚，灯照席角，沿净，像腌菜坛，不像邪。」

老耿挑水过，声哑，境别拜邪，拜了路断，他只记着。

秦霜木鱼远，一声，心别邪，邪了鼻下香飘，香飘手就滚。

柱上条角白一线，三年约在怀，愧字在怀，手仍不抖。

白织月远廊过，袖风冷，没停，胜在鼻，不在嘴。

铁无言脚步远，停半息，声低，别漏香，别叫境，只记。

刘婆粥稀，勺沉，饭稳恩不断，他喝尽，不谢，只记。

再遇后第三日，药渣场叶渣另堆。另堆不要紧，苦辛并一线，并一线像叶家药远路飘来半息。飘半息，恩在胸，手才不飘。

温听雨风口立半息，验分堆，眉松：「叶渣另堆，分得清。」分得清，韩泥不谢，谢轻，记在心里。

管事记「辨性准」。准，是小胜。赖福眼细，没再尖——尖不过「准」字。准在鼻，不在舌。

刘婆端粥，仍多半勺，嘱再遇别攀。韩泥点头：「不攀。各做各的。」

他低声道：「再遇了。」

叶渣另堆，堆别混。混了，全丙九记名；记名，恩断边缘。边缘不要紧，鼻下有根，根细，细才辨得出。辨得出，85小成才接得住。

药山半腰，雨歇，窑温余韵散进掌——散不要紧，记：再试，火可匿，路向备料。
