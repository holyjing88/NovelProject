# 第三十九章 体统

扑怀次日，丹堂外廊比平日更静。

静里却更闹——闹在弟子眼里，闹在窃语里，闹在「苏首座扑进莫长老怀里」这一句，比三品金丹传得还快。

苏念慈脸红仍强撑首座威严，弟子问丹样，她答得准，指节却白，白得像还攥着昨夜袖角温度。有人故意问：「首座，金丹成后，体统如何守？」

苏念慈冷声：「守体统，先守丹。丹在，堂在。」

弟子噤声，却忍不住偷瞧廊外——莫长春昨夜脸洗过了，灰袍仍素，杖仍旧，像什么都没发生，又像发生得太实在。

柳青鸢公事公办来访，青衫素，宗主令在腰，剑未佩，威在眸。她目光扫过莫长春袖角一点药渍，一言不发，像风过不留痕。

苏念慈送茶，茶热，手稳：「宗主请。」

柳青鸢抿一口，淡道：「三品金丹，青岚之幸。」

「宗主谬赞。」

「不谬。」柳青鸢起身，至门边停半息，未回头，「首座破境，可喜。东脉之事，亦可喜不起来。」

她走了。廊上静，静得能听见炉里细火。

苏念慈转向莫长春，低声：「宗主……吃醋了。」

莫长春望天，云淡：「风大。」

「师兄装傻。」

「傻好。」他杖点地，「傻让人放松警惕，警惕一松，礼好送。」

苏念慈噗嗤，又赶紧收，首座模样。她望门内炉，低声：「炉还开着。」

「开着就好。」莫长春道，「开着，比多少句体统都真。」

柳青鸢在远处回廊停了一停，指节叩栏，一声，像把妒与担都叩进木里，再转身，宗主威仪又穿上，穿上得像剑回鞘。

苏念慈瞧见那背影，忽然不酸了，酸不起——东脉夜袭近了，体统二字，轻不过命。

当日午后，验丹厅外日头白，白得像把体统也晒得无所遁形。两名内门弟子比谁的碎语更快，比到一半，被顾小满册角一敲，敲得像敲在「围样不围嘴」五字上。苏念慈隔窗听见，不恼，只把封蜡又按实一分——按实了，像把羞也按进体统里。

赵玄机路过，听见「体统」，停半息，未讽，像也懂：门比体统难守，夜袭将至，谁都有戏要演，演不了的是命。

霍镇山骂：「体统个屁！夜袭在前！」

苏念慈封丹样，手稳，稳得像把羞炼进丹。她低声对身边弟子：「守体统，先守丹。丹在，堂在。堂在，门才像门。」弟子问：「那扑怀……」她冷声：「扑怀是私，成丹是公。公在前，私退后。」

莫长春望东脉，云淡，云腥，像战帖已至。他杖点地，声轻：「风大。」苏念慈噗嗤：「师兄又装傻。」「傻好，」他笑，「傻让人放松警惕，礼好送。」

丹堂炉火映着廊柱，柱影长，长得像体统二字也长。苏念慈封丹衣，指尖抚过金丹封蜡，蜡温，温得像把羞炼成能守的体统。

验丹钟将响。钟未响，嘴先嚼。嚼不过文书上的三品二字，二字硬，硬得像把私嚼也压成公。

莫长春立庭心，声轻：「统在礼，不在抢。刀近了，礼更烫；烫的是东脉旗，不是丹堂门。」

验丹文书半个时辰后到，文书到，全宗又要围一层——围金丹，围体统，围九月老头还剩几月。

林挽月罚役途中，听「体统」浪起，唇白，像终于看懂：体统是戏，证字才是路。路未说，时未到。

孙福旧党嘀咕：「老不正经……」顾小满冷眼：「闭嘴。首座守丹，长老守东脉，你守什么？」嘀咕者噤声，像体统碰上半石，又磕碎牙。

魏无涯在檐角，扇开一线，观三人气运交缠，低笑：「体统是戏，东脉是命。命近，戏轻。」

验丹厅建的初衷，本是定品，不意定住了心。苏念慈捧样站在厅心，像把三十年的守炉，也站在厅心。莫长春立廊外，不进厅，像怕进门就把体统走成私。

有年轻长老问：「首座扑怀之事，可记过？」柳青鸢声稳：「记破境，不记扑怀。围金丹，不围妒。」霍镇山骂过檐下：「体统个屁！夜袭在前！」骂声像把验丹钟也骂响了——钟响，三品光落，落得像给全门看：门在，丹在，命还在东脉等。

体统二字贴满内门那日，比金丹光还刺眼。

莫长春袖空，杖立庭心，不坐，只立。柳青鸢声稳：「体统为门，不为妒。」赵玄机冷声：「丹成该入公库——」苏念慈截话：「丹是我炼的，库是门里的。各算各的。」

内门贴条比外门还快，快里像要把丹从苏念慈手里抢成「公」。抢的人手缩半寸，缩像被塔意烫到——烫的不是手，是私心。

验丹文书到，文书厚，厚得像把体统也写进纸里。苏念慈验印，印稳，稳得像首座威仪也稳。

赵玄机立验丹厅侧，铁牌在腰，冷声：「样丹该封存公库——」霍镇山骂：「体统个屁！夜袭在前！」骂完，仍令东脉巡哨加倍。

验丹厅外，弟子围成半圈，围金丹样，不围嘴。周德海立后席，笑和气缓，笑里却像想把样丹接成「公」。莫长春杖点地：「抢样的人，先问问东脉旗答不答。」周德海手缩半寸。

苏念慈举样而立，未躲，未辩，只让三品光落在脸上——光落处，体统不再是私，是门。年轻长老问：「扑怀可记过？」柳青鸢答：「记破境，不记扑怀。」

顾小满记在心里：验丹厅·围样·不围嘴。验丹钟终于响了，响得像把三品二字敲进每个人眼里，也敲进东脉风里——风腥，旗动，像提醒：明天审丹，今夜审田。

外门有人贴条避讳苏念慈名，条被柳青鸢令撕。东脉方向云低，旗影映进验丹厅窗——光在，门在，田还在等谁守。

年轻弟子围验丹厅外，比试不赌丹，赌「体统」二字谁先走样——走样的是私，不走样的是门。比到第三息，私先退，退得像被三品光烫了一下。

苏念慈隔窗瞧见柳青鸢背影，忽然不酸了——东脉夜袭近了，体统轻不过命，命在刀前，刀在旗前，旗在田前。她低声，仅自己：「炉还开着。开着，比多少句体统都真。」

塔壁浮半句：**寿元四月**。莫长春瞧见，笑给风听：「四月够验丹，够守田，不够吵。」

体统刀未出鞘，鞘响已够苏念慈隔窗一紧——紧完，赤焰又近一日，一日腥，腥在旗前，旗在田前，田在根前。

魏无涯收扇，低笑：「体统是戏，东脉是命。戏演完了，该看旗了。」赵玄机立远，铁牌在腰，妒里空一寸，空处像还库那日挪出的，挪向哪边，尚说不清——说不清也好，说不清才肯把下一招，用在刀前而不是嘴上。

验丹文书落款那刻，全宗像被同一阵风洗过。洗过的，不是羞，是界——界在宗主令上，也在苏念慈举样的那一寸光里。莫长春立廊外，不进厅，声轻：「统在礼，不在抢。刀近了，礼更烫。」

霍镇山抱丹匣退半步，匣盖未开，匣里丹香已透窗缝，与东脉旗腥气在檐下碰了一碰。碰完，谁也没说谁赢——赢不赢，要等田埂上那面旗先落。苏念慈收样入袖，袖里还温，温得像命还在手里攥着。周德海令旗一转，转的不是风，是下一程——下一程，在田，不在厅。厅外旗影又浓半分，浓得像血还没干。风过檐角，无人应声。应声的，该是田埂上的旗。旗在等夜。夜里审田，审根。
