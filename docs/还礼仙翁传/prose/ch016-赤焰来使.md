# 第十六章 赤焰来使

孙福事毕第三日，山门外赤焰旗至。

旗红如血，绣黑焰纹，风一过，像把东脉的灵田都舔了一口。弟子列阵，剑未出，气先紧。

韩铁山跨马至，甲硬声硬，抱拳礼数足，眼神不足：「青岚宗主何在？赤焰谷韩铁山，奉少谷主令，来谈东脉灵田。」

柳青鸢出殿，青衫素，威在眸：「东脉灵田，青岚门民共耕，非一谷可索。」

韩铁山笑，笑里刀：「宗主言重。少谷主只说，七日之内，划界立碑，东脉三成归赤焰，换七年不动兵戈。若不然——」

他顿顿，像故意让风把后半句送进每个人耳朵：「妖潮与赤焰，同至。」

殿前一寂。

周德海即出，捧折子，像早备好的主和稿：「宗主，赤焰势大，东脉民散则全宗失血。割三成，换七年，可议。」

霍镇山怒目：「割地换安，安在谁枕上？民散根拔，七年和平，是葬歌！」

柳青鸢沉脸，未拍板。

莫长春拄杖立殿阶侧，不语，只看杖上竹纹——竹纹比往日游得快，像因果将涨。

韩铁山目光扫过他，像扫一块将熄炭：「这位，便是传说中的莫长老？听说……九月寿元？」

莫长春笑：「九月够听你把话说完。说。」

韩铁山一愣，未料将死之人接话接得稳，便敛笑：「少谷主敬长老曾是化神，故留七日。七日后，厉少谷主亲至，谈不成，便不必谈。」

他甩袖，赤焰旗转，马蹄去，像把血旗插在东脉边上，等七日后的雨。

长老会当夜急开。

周德海主和：「东脉三成，可保核心灵脉。」

霍镇山主战：「妖潮频仍，让地则民散，民散则门散。」

柳青鸢指节发白，终看向莫长春：「师兄，你怎么看？」

莫长春杖点地：「地可以给，根不能拔。」

不表态站队，却令柳知他心向战——根在民，在田，在青岚门东脉那条命。

周德海冷笑退席半步：「莫长老说得轻巧，九月之人，不必守田。」

莫长春看他：「正因为我九月，才懂田比命短。命短的人，最懂根。」

殿外，弟子议论未歇，议论里怕，也燃——怕七日之期，燃的是青岚门终于不再只守东脉。

苏念慈在丹堂听见「厉无殇」三字，炉火一跳，跳得像妒里掺敬。她低声对弟子：「赤焰不赤焰，别传。传多了，传成惧。」

赵玄机在执法堂抄门规，抄到「外使索地」一条，停住，墨滴成字，像莫长春那句「根不能拔」又浮上来。

霍镇山在营帐外磨刀，刀声霍霍，像给东脉磨刀，也像给七日后的宴磨刀。

夜，簿上浮字，淡金：

**外患已至，七日为期。**

莫长春合簿，望东脉方向，像望一场将来的宴——宴上有血旗，有少谷主，有全场寂然。

他念：「厉无殇……名字倒凶。希望人别比名更凶。」

杖答：等。

莫长春笑：「不等了。七日，够备一份礼。」

袖中，一方素帕不知何时压着，帕角干净，像从未擦过汗，却像等了很多年。

大因果还在厉无殇亲至那日，像帖，像手帕，像全场寂然的三息。

长老会当夜急开。周德海主和，霍镇山主战，柳青鸢指节发白，终看向莫长春：「师兄，你怎么看？」莫长春杖点地：「地可以给，根不能拔。」

殿外，弟子议论未歇，议论里怕，也燃。苏念慈在丹堂听见「厉无殇」三字，炉火一跳。赵玄机抄门规，抄到「外使索地」一条，停住，墨滴成字。霍镇山在营帐外磨刀，刀声霍霍。

夜，簿上浮字，淡金：**外患已至，七日为期。**袖中，一方素帕不知何时压着，帕角干净，像等了很多年。

韩铁山跨马至，甲硬声硬，抱拳礼数足，眼神不足。柳青鸢出殿，青衫素，威在眸：「东脉灵田，青岚门民共耕，非一谷可索。」韩铁山笑，笑里刀：「宗主言重。少谷主只说，七日之内，划界立碑……若不然，妖潮与赤焰，同至。」

莫长春拄杖立殿阶侧，不语，只看杖上竹纹——竹纹比往日游得快，像因果将涨。韩铁山目光扫过他：「这位，便是传说中的莫长老？听说……九月寿元？」莫长春笑：「九月够听你把话说完。说。」

围观弟子在山门列阵外踮脚，踮得像等血旗转走——旗转走，七日之期才真正开始。有人低声：「妖潮与赤焰，同至？」另一人答：「同至之前，先备礼。」
