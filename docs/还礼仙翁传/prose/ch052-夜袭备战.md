# 第五十二章 夜袭备战

子时前两时辰，东脉像一张未张满的弓，弓弦颤，颤在田埂，颤在丹堂，颤在每个弟子的腿骨里。

柳青鸢立灵田高台，令旗起落，镇妖司列阵，外门内门各守一线。霍镇山盾阵顶前，骂声比鼓响：「赤焰要来，老子先骂醒你们！腿抖可以，退半步不行！」

弟子列阵，腿抖，抖里仍咬牙，像半块灵石撑着。民户缩在阵后，孩子哭被捂住，哭里有人低声念青岚门，念莫长老，念三品金丹——念成胆，胆不大，却够站直。

周德海立后阵指挥棚，斗篷遮面，令旗在指间，迟迟不落：「后阵稳固，侧门慎开，防中伏。」副将急声：「赤焰若绕田埂，侧门不开，丹堂腹背受敌——」周德海目光一沉：「妄动者，军法从事。」

霍镇山回头吼：「周老头，侧门不开，丹堂腹背受敌，你防谁的伏？」周德海笑，笑里冷：「防你们的莽。」柳青鸢不辩，只记，像塔壁又记下一缕待还之纹。顾小满在侧径望后阵，望周德海令旗，令旗干净，干净得像从未举起过。

丹堂，苏念慈立阵眼石，地火阵纹淡红，炉中火舌细稳。弟子搬阵材，搬药匣，搬得急，却不敢噪，噪一声，阵眼就乱；乱一寸，炉就危。

莫长春未进门，立门外石阶，杖点地，一声，两声，像给夜战敲节拍：「阵眼你守，老夫门外坐着。炉开着，别熄。熄了，门就白守。」

苏念慈扬声：「守。」她不多言，言多像怯；守字一出，地火阵纹又红一线，像替暗线也担。她低声训弟子：「佯攻丹堂，实断东脉。断脉先断耳，耳在杂役。你们守炉，杂役守线。」

执法堂，赵玄机率二十人赴丹堂侧翼，路过莫长春，抱拳，礼深：「长老，赵玄机今夜守门。」

「守。」莫长春道，「门比剑难守。守住了，恩就还了一半。另一半，看你能不能把手从私囊里抽出来。」

赵玄机颔首，刀光出鞘半寸，像把妒火也出鞘，换成守门的硬。他低声训队：「离门三步，退半步按逃罪。刀向外，眼向外，心也向外。」

林挽月罚役途中听见，唇白，白得像终于看懂：恩对路，不能改。她不敢停步，停步像私；步稳，才像记。路过杂役堂，见顾小满率队没入侧径，她只点头，点得像把「证路」二字也送出半寸。

杂役堂，顾小满分班，路线图在怀，像第二张半石。莫长春递她三张隐匿符，符温，像长老的手按在心上：「活着把消息送到。送到，比杀一人有用。别跟丢命，别走正面，正面是霍老头和宗主的戏。」

顾小满接符，指节白：「小满记恩。」她率队没入侧径，夜吞人，像吞一根细线，线那头拴着青岚门的耳。

莫长春望线消失处，望东脉，望后阵，袖空，杖在塔中。风腥扑面，田稻似浪，浪里杀意已至门前——门前是丹堂，丹堂是炉，炉边坐着九月老头，像七夜前一样，坐着，不抢戏。

韩铁山在远山布阵，笑里刀，刀向田，也向丹堂。厉无殇在帐内听报，声哑：「莫长春，青霜入袖？」他砸杯，杯碎，像砸不碎袖中记忆。魏无涯观调度，低笑，传讯秦商言，只四字：「子时动。」

弓张满，三线布，侧门闭——闭的是伏，还是忠？全宗尚不知，尚要子时称一称。称的不只是阵，还有令旗：谁的令旗真举，谁的令旗假落。

霍镇山在前阵骂罢，又骂后阵松，松得像等谁来补。苏念慈遣第三次报：「炉不灭，阵眼稳，只等子时。」报声入夜，夜更沉，沉得像把「佯攻丹堂」四字也沉进每个人心里——沉了，才防得住。

柳青鸢在高台令旗微抬，声低：「外门顶田，内门护堂，杂役走耳。耳断了，阵就盲。」霍镇山应：「镇妖司在，东脉根在！」声洪，洪得像替田埂也撑胆。

赵玄机立丹堂门洞，甲新，眼亮。他看莫长春坐石阶，坐得像七夜前守炉，守的是人，也是门。他低声，仅自己：「门比剑难守。难，才像还。」

苏念慈第三次遣人来报：「阵材药匣齐备，弟子已分班。佯攻丹堂，实断东脉——断脉先断耳，耳在杂役，你们守炉。」报声入夜，夜更沉，沉得像把「子时」也沉进地砖。

林挽月罚役至林苑，林苑远，远得像把「慕」字隔在门外。她回望丹堂方向火光未起，火未起，夜已紧，紧得像弓将张满。她低声：「恩对路，不能改。」

民户中有老农跪地，跪得像跪田，不跪人。霍镇山骂声扫过，扫得像把他们也扫进青岚门的根里：「田在，人就在。退半步，田就失！」

莫长春杖点地，第三声，像弓弦终于拉满。他低声，像钩：「子时。弓弦满了。缺一环，田就失。」

识海塔影一荡，一念出塔，杖点石阶，声轻：「备齐，阵要来。杂役阵，亮里见石。」塔影荡完又静，静里像听见侧径里极细的足音——足音是耳，耳是还礼。

一念收塔，杖落掌，望丹堂门洞。赵玄机甲新，眼亮，像终于把「守门」二字刻进铁里。刻进了，夜袭才像试炼；刻不进，罚再轻，也是私。

风腥更重，火把明灭，东脉灵田一线，弟子屏息。屏息里，全宗都在等子时——等弓响，等霜出，等侧门闭到何时，等杂役耳还能不能活着回来。

秦商言自侧径来报，衣沾泥，声稳：「赤焰先锋已贴地，火把三处。韩铁山立旗远山，妖帅随行，不是虚张，是实断。」柳青鸢令旗再落：「霍镇山，前移！苏念慈，炉不灭！赵玄机，门三步！」

霍镇山骂声滚进田里，滚得像替民户也撑胆。周德海在后阵棚令旗微动，动而不举，举而不落，落得像在等子时称一称谁的忠。顾小满率队没入侧径，没入前回头，见莫长春坐石阶，坐得像七夜前守炉。她低声：「小满记恩。」随即夜吞人，像吞一根细线。

魏无涯在檐角传讯，只四字：「子时动。」又笑浅：「三线布，侧门闭——闭的是伏，还是忠？尚要子时称一称。」

莫长春望东脉，望丹堂，望后阵，三线像三根弦，弦满，才响。袖空，杖在塔中，低声：「礼还没送完。他们先动手，正好省老夫敲门。」

韩铁山在远山冷笑，冷笑里刀：「莫长春，九月将尽，也配谈战？」风答他，答得像从青岚山门来：「配不配，子时见。」厉无殇在帐内砸杯，杯碎，像砸不碎袖中记忆。

苏念慈在丹堂阵眼，地火阵纹又红一线。她训弟子：「搬材搬药，不敢噪。佯攻丹堂，实断东脉，你们守炉，杂役守线。」赵玄机在门洞训队：「刀向外，眼向外，心也向外。心向内，门就守不住。」

莫长春笑浅：「他们急，我们不必急。急的是符，不是胆。」子时将至，东脉弓弦颤，颤的不是怕，是等——等杂役耳，等赵守门，等霜出袖，见不见血。全宗屏息。

秦商言在侧径复核火把方位，复核得像复核半块灵石——核清了，简才直，简直了，耳才活着。霍镇山在前阵骂罢，又令盾阵前移半寸，半寸像把东脉的根也往前推。柳青鸢在高台望三线，声低：「耳断了，阵就盲。盲了，就不是试炼，是乱局。」子时将至，全宗屏息。

三线布防，阵眼将明，明里藏疑——疑在周氏侧门，不在霍氏刀。
