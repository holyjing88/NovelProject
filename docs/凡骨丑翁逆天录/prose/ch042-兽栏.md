# 第四十二章 兽栏

冬深了，霜咬棚顶，丙九人都缩着肩。

兽栏比棚更深，深在角落。角里小兽卧在干草上，喘得细，像漏风的旧箱。韩泥清粪，铲一下实一下。实是备检后那几日练出来的——练在雪里，练在腥里，也练在赖福的秤旁。管事掷单：「丙九韩泥，冬角仍你。清细，别漏。」

「清。」

赖福阴笑：「晦气杂役，兽栏专收晦气兽。」

韩泥只应一声。舌头要留给哄兽——兽不懂灵根，懂手温不温。他把破袄袖子挽高一寸，冷风刮腕也不停。

同批杂役在棚口抽烟，烟味冲，冲不过栏里腥。有人喊：「韩泥，角里畜咳得吓人，你还进去？」

「进去。」他说，「不进去，畜死了罪仍落到丙九头上。」

「你逞能？」

「我活命。」他应得平，拎铲进了角。

角里潮气裹人，人一进去，靴子先沉。沉不怕，怕的是铲下发虚——虚了，粪贴草，草贴兽，兽更喘。他把铲刃磨了又磨，磨出一线亮，亮了才下第一铲。

午后小兽咳重了，重一声轻一声。韩泥撤潮草，铺新草，铺平了兽才卧得实。同批杂役掩鼻：「病兽，别沾身！」

「沾了，洗手。」韩泥说，「手净，粪才净。」

那人退半步，仍嘟囔：「晦气。」

小兽眼半阖，半阖里还有一点活气。活气薄，也真。韩泥蹲下，掌轻按兽背，背烫得浅。「活一日，清一日。」他低声道，「清一日，少挨一句骂。」

兽不懂话，咳却轻了半声。同批杂役又凑近：「你按它，传染你怎么办？」

「传染不了饿。」韩泥说，「饿能传染全丙九。你躲，栏臭；栏臭，管事骂你。」

那人噎住，退远了仍嘟囔。韩泥从破袄里摸出半块麸皮，掰指甲大，搁干草边。兽鼻动了动，舔不到也算活气。

管事路过，瞥一眼：「还管病兽？」

「管。」韩泥说，「角清不净，全栏臭。臭了，你骂丙九。」

管事「哼」一声，走前却道：「清细，记着。」

赖福远处啐：「装善。」

韩泥低头换草。善不善，兽不管；兽只管草干不湿。他把角外三尺扫干净，雪混着粪皮，扫到篮里，篮沉，心倒轻一点。

黄昏雪又落，落进栏，落进角，落进小兽鼻下。他加半刻，把湿粪清尽。三尺净，味淡一寸。小兽喘更细，细得像线要断。他低声道：「断不得。你断，我少一笔记功；你活，我多一口饭。」

兽眼动一下，像应。

夜前管事来查，靴底踩雪声沉：「角清了？」

「清了。」

管事蹲看兽：「这畜，活不过今夜。」

「知。」韩泥说，「活不过，仍清。」

管事看他一眼，没有笑，也没有嘲：「丙九里，你话少，手紧。」

「话多，手抖。」

管事起身，掷半块麸皮：「赏你的。别给赖福看见。」

韩泥接住，麸皮糙，糙也香。同批嘀咕：「病兽死了，你也捞赏？」

「兽死栏净。」韩泥说，「净了，少骂丙九。少骂，也是饭。」

嘀咕断了。刘婆提粥来，粥稀却热：「守角别抖手。抖了，备检白过。」

「不抖。」

她哼一声，多舀半勺。赖福远处骂：「疯婆子又偏晦气！」

刘婆回骂：「晦气在你秤上！秤歪，粥才稀！」

韩泥不掺。掺费舌，舌留给清栏。卯时将至，同批路过仍掩鼻，掩不住味淡的事实。他打水淋角三遍，水凉，手仍紧。手紧，管事才肯在簿上添「冬角细」二字。字小，也真。

他抬眼望发粮场方向，棚矮，矮里藏着秤。秤虚一日，丙九颤一日；颤久了，人手就飘。飘不得，他把目光收回栏角——角净，才配盯秤那天听三两实沉。

同批少年路过掩鼻。韩泥不劝，只把铲柄磨光滑些——磨光滑，明日第一铲才落得快。快不是躁，是活。

雪停时，栏外脚印乱。乱脚印里有管事的、赖福的、他自己的。他自己的最实，实得像在雪里刻「净」。净了，病兽的罪不至丙九；不至丙九，就少挨一脚。

同批又躲远：「死兽角，晦气！」

「晦气在嘴，净在铲。」韩泥说。

管事过，鼻下动一息：「味淡了。淡一寸，丙九少骂一句。」

赖福阴笑：「味淡有屁用，凡骨仍是凡骨。」

韩泥铲下不停。凡骨在石上，不在栏上。栏净，饭在；饭在，命在。

丑时，小兽咳止。止不是好，是尽。韩泥立半息，不拜不嚎。他清最后一铲，实得像替兽收尽。移兽时少年们躲远，他独力抬走。赖福啐：「抬晦气畜！」

韩泥不辩。管事复查角：「死兽移走，角消毒。韩泥，明日仍你。」

他把角里的干草再翻一遍，翻出底层潮点，潮点全剔掉。剔干净了，鼻下才敢说淡。淡半寸，全棚就少半寸骂。

邻床少年悄问：「你真不怕晦气？」

「怕晦气，怕不过饿。」韩泥说，「饿狠了，手抖；手抖，备检白过。」

他把半块麸皮收进破袄。赏不重，重在管事看见了手紧——看见了，明日少踹一脚。刘婆门外咳：「栏净了，睡。睡实，手才不抖。」

韩泥应，不回头。恩在粥里，不在栏哭。他把湿草垛挪开，露出净地。净地白，像验单——手抖，验单就白。他不抖。

同批少年问：「兽死了，你难过？」

「难过像少年。」韩泥说，「恩仇都记心里。兽命尽了，心里记着清了就行。」

少年不懂，懂「栏净」，点头睡了。邻床又梦呓：「冷……」韩泥替盖袄，袄破也暖。管事巡夜靴声沉：「冬角清了，明日仍你。别抖。」

「不抖。」

赖福远处骂：「晦气扫栏，扫出妖！」

韩泥收铲入棚，铲柄凉，凉不过坛沿那线温。温应净，不应哭。哭费气，气要留来明日仍站这角。

他把麸皮掰半，半留己，半塞邻床手里。塞了不声张——声张，赖福来抢。邻床迷糊握紧：「……啥？」

「明天咬一口。」韩泥声哑，「别叫人看见。」

雪又落细，落进栏角，落不进他眼里。眼里只有净字，净到底，备检才验得过。兽栏夜风硬，硬里味淡一寸。淡一寸，够丙九喘一口气；一口气，够挨到盯秤那日。

他望坛，低声道：「手在，路在。角净了，才听得见秤上三两落袋那一声。」

坛不应字，只温。温着就够。明日发粮场若再浅，他便忍着——忍到管事肯盯，饭才实。饭实了，叶丫头那碗烫才还得上。

丑时末，他把铲柄又擦一遍。擦到木纹发亮，亮了才像自己还握得住事。栏外风停一瞬，停里能听见远处灶下锅响。锅响是粥，粥稀也胜过空秤。

同批少年睡前又问：「栏净了，管事还会加你的活吗？」

「会。」韩泥说，「加就加。加完还能站，就说明手还在。」

「你就没个消停？」

「消停在饭里。」他道，「饭实了，才消停得起。」

他又摸一次破袄里的麸皮。麸皮糙，糙得提醒他：赏不重，重在看见。看见了，明日发粮场上才少一分被克的借口。少一分，全棚就多一分气。

栏外有人路过掩鼻，掩完仍忍不住问：「你真把死兽抬走了？」

「抬走了。」韩泥说，「抬走了，角才净；角净了，丙九才少挨骂。」

「你不嫌脏？」

「脏怕不过饿。」他道，「饿狠了，脏也得干。」

他把角外最后一点草屑扫进篮。扫完，鼻下才真淡。淡了，管事簿上那「冬角细」才站得住；站得住，盯秤那天说话才有底气。底气在，三两落袋才听得真。听得真，仇才有根，粥才热得住。热得住，才配把门缝里那点气攒到明日。

他合眼，不睡死，是记着净——净记着，盯秤那日才站得直。
