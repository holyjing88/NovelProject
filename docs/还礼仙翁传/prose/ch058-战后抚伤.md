# 第五十八章 战后抚伤

黎明，东脉焦土，露冷，硝烟像一层洗不净的灰。

灰里混着稻茬的焦味与血腥，踩上去，软，软得像一夜还没走尽。弟子三三两两收拾残刃、断旗、倒地的灵稻，收拾里仍窃语，窃语轻，轻不过焦土上的血。

柳青鸢寻到莫长春时，他立在一截断旗旁，杖点地，袖角有血，血不是他的，像妖与金丹留下的尘。须眉在晨光里白，白得像一张旧符，符旧，意满。左腕袖裂，裂口处皮未破，破的是袍，像一夜杖剑留下的证。

「师兄。」她递绷带，手稳，眸不稳。

话落，她为他缠左腕，缠得紧，指尖触腕脉，忽一停，像听见什么，又似不愿听见。脉仍乱，乱里却稳，稳得像将死之人，把命押在还礼上，押到第三夜还没散。她缠时指节发白，白里不是力，是怕——怕这一缠，像缠住九月最后的线。

她额抵他肩，一瞬，极短，像把宗主威仪借夜风放软一分。肩上有灰，灰里有血味，血味不浓，浓的是同门许多年。许多年里，她练剑，他拄杖；她升座，他还礼；她不能放软，放软像丢剑；可焦土上，剑也要回鞘一息。

莫长春不动，任那一瞬落，落完，才笑，声轻：「风大。」

柳青鸢退半步，脸热，热得像霜洗过的剑，仍冷在鞘，鞘外却起了一层薄红。她硬声：「……师兄别胡说。」

「不胡说。」他道，「风大，脸才红。红了，也像人，不像只有剑。」

她瞪他，瞪不出狠，只转出宗主令，令在腰，威在回：「天试之前，别死。」

「尽量不死。」莫长春答，像多年同门的旧约，像闪回夜里回过的那四字简。

她还想补一句，补到唇边，又咽回，咽回像把「别死」二字也缠进绷带。她转身，袍角扫过焦土，扫出一道青痕，像剑意收鞘。走出三步，又停，停半息，像还想说什么，最终只留背影，背影里剑穗无风，风像从塔意借来。

远处丹堂，苏念慈望这边，望一眼，收眼，炉仍开，像把妒与担都炼进火里。她未入门，只立廊下，指间仍捏着未送出的培元丹，捏得像把「风大」二字也炼成灰。妒不在脸，在炉——炉里火稳，火稳里她懂：师兄抚伤，是恩，不是体统；体统留给旁人嚼，恩留给同门记。

弟子在焦土边窃语，窃语里慕，也怕——慕宗主竟肯放软一瞬，怕九月老头袖角有血。窃语不敢洪，洪的是霍镇山骂声远来，骂到一半，见柳青鸢背影，噤声，像懂有些事不该洪声。

赵玄机在丹堂门洞包扎，听见「风大」，停半息，未讽，像也懂：守门之后，还有恩要还。他望莫长春方向，目光一触，一触即分，分得像把「继续还」三字，也刻进绷带里。

顾小满在焦土边收残旗，眼里亮，亮得像雨夜那道光还在。她望柳宗主背影，又望莫长老，低声，仅自己：「宗主放软一瞬，像剑回鞘。鞘回，刃还在。」

莫长春抚绷带，塔影微烫，字淡，浮起半句：

**战后有人缠，恩仍在袖。**

他望天试方向，云低，像帖将至。云边有一线金，金浅，浅得像五宗联名帖已在路上写名。

魏无涯远观，扇合，低笑，像看恩与担缠在一起，缠得像剑收鞘，鞘外起薄红。他不近，像懂：抚伤是私，围观是公，公的不必凑。

霍镇山扛盾走过，盾面裂口已补，补得像东脉还能再扛一夜。他抱拳，声洪却收了一度：「莫长老，丹堂守住了。赵小子……像个人。」

「像不够。」莫长春笑淡，「要像到底，得守到帖来。」

林挽月自人群侧过，唇白，望柳青鸢远影，又望莫长春，像终于看懂：慕错人，还能改；恩对路，不能改。她未近，近像打扰，打扰这一瞬的鞘。

焦土尚温，绷带尚紧——紧的不是伤，是恩；恩在袖，帖在云，天试之前，别死。

弟子渐散，散里仍有人忍不住再瞧一眼灰袍袖空——袖空，塔意却沉，沉得像妖核与金丹余息，都已记在心里。有人懂，有人不懂，懂的人不再问九月，不懂的人还在等下一夜。

风大，露冷，像天试帖已在云上写名。

莫长春杖点地，往丹堂方向慢走，走得不快，像把抚伤那一瞬，也收进袖里，收进塔里，收进「尽量不死」的旧约。走至丹堂阶下，苏念慈终于送出那枚培元丹，丹落他袖，落得像迟来的伴炉。

「风大。」她重复，重复像炼丹时的口诀，口诀里无妒，有担。

「风大。」他答，答完入堂，不入阵眼，只望炉，望一眼，像确认火还在，人还在，三月还在。

赵玄机自门洞出，甲补一半，血洗一半，抱拳：「长老。」

「还疼？」

「疼。」赵玄机答得直，直得像铁牌终于不弯。

「疼，就记着。」莫长春道，「记着，比忘难。忘的人，守不住第二夜。」

赵玄机嗯一声，重，像又一把刀入鞘。

柳青鸢已远，远至宗主殿，独对东脉图，指节发白。她未再提「风大」，提的是「天试之前，别死」——别死二字，像比脸红更烫，烫在宗主令上，令在，威在，软一瞬也在，软在焦土，不在殿。

顾小满收完残旗，至杂役堂登记伤号，登记里添一行：「宗主缠绷带，莫长老袖空。」写罢，划掉后半，划得像懂：袖空不是无，是塔有。

霍镇山至丹堂，声洪却低：「赵小子，下一夜还守？」

「守。」赵玄机答，答得像不需要莫长春再留路。

莫长春笑淡，笑给九月将尽的晨看：「守到帖来，够。」

焦土上露渐晞，晞得像一夜终要过去；过去里，恩缠了，伤抚了，脸红也风大了——风大的地方，帖也快到了。

午后，宗主令传各堂：战后三日，东脉除巡防，不办庆功，只办抚恤。令到，弟子喜声压半寸，压得像懂：庆功是面，抚恤是根；根在，门在。

苏念慈开炉炼伤药，药香出丹堂，盖过焦土腥半寸。她炼时想：师兄说风大，风大脸才红——红了，也像人。人不是只有剑，也不是只有炉；炉在，人也在，三月也在。

柳青鸢再至焦土，已换净衫，衫青，青得像剑回鞘后的静。她见莫长春绷带，未再碰，只远看一眼，一眼里「天试之前，别死」仍在。

莫长春笑：「宗主放心，尽量不死。」

「别尽量。」她硬声，硬后软半寸，软在转身，转在袍角，袍角不再扫焦土，扫的是回宗主殿的路。

魏无涯远观，扇合，低笑：「抚伤是恩，体统是刀。刀给别人，恩给自己人。」笑完，离山，离得像部二这一章，也要收鞘。

赵玄机巡至焦土边，见绷带弃布，弃布染尘，尘像一夜证。他拾布，未言，像把「继续还」也拾进铁牌。霍镇山见，哼：「赵小子，连布也捡？」

「门要净。」赵玄机答，答得像守门也守心。

莫长春望一眼，笑淡，笑像给风听：风大，脸才红；门净，恩才在。

弟子渐散尽，焦土上只剩断旗与残稻，残稻像东脉的疤，疤在，根也在。柳青鸢令：三日后重开灵田，田要种，人要活，活要还礼——还礼二字，像比庆功更重。

识海塔影一荡，一念收塔，杖落掌，他念：「伤合，守要来。赵死守，硬里见规。」

抚伤声轻，轻不过布告柱上金纹将贴的那一声哗响。
