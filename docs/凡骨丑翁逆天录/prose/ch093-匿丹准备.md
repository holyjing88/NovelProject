# 第九十三章 匿丹准备

霜降后，韩泥白天不再往山上跑。

白天清栏、分拣、编筐，活计照做，做稳了，才挪出丑时和雨夜。匿丹这事，首在藏料。青芯、褐渣、灵泉，分三处：袖袋一处，筐底夹层一处，兽栏干草下浅坑一处。坑浅得像藏粮——粮乱，赖福才懒得细查。

铁无言在药渣场角压嗓：「你要试，别动内门药。用渣，用灰，用捡的。」

「用渣。」韩泥应。

用渣像晦气杂役该用的东西。该用的，才不惹眼。

第二项，藏炉。瓮底泥炉拆成三片，片片塞进坛席下，席再厚一层。厚不为保暖，为挡灯。管事夜巡扫到席角，只见席厚，不见炉影。

第三项，藏诀。墟翁丹经残页，页黄字丑，他从废经片里抠出来，抠完不给人看。页贴胸，胸贴坛温，温着才像还活着。

叶青禾那边，他不敢近。近了像连累；她说过「别来丑」。丑可以离远一点，恩却不能晾着——还得在暗处，在门缝，在她怕黑的那一线。

他托沈枯芽递口信。信无字，只有半块干饼，饼边掐缺口：「冬夜别灭灯。」

「……就这话？」芽儿怯。

「就这话。」韩泥说，「别加字。加字像情，情容易被人写成邪。」

「叶姐……她怕黑吗？」

「怕。」韩泥声平，「怕黑是她的事。我的事是别让黑压着她。」

芽儿点头跑了。韩泥备第四件：温汤囊。囊小，陶缝丑，缝纹像叶汤碗沿。囊空着也好——成丹后再装温，温着过门缝。

赖福查库门响：「丙九，领麸皮！」

韩泥排队末席。赖福眼细：「袖鼓什么？」

「木牌。」韩泥说，「贴身的，三年约。」

赖福想掀，管事喝：「查库快些！误时克半日！」

赖福哑半息，麸皮仍少掷半分。韩泥不谢——谢轻了像讨好；不谢，记在心里。

夜里他把经页摊在席上。「培元」二字浅，浅却真。他默念不进舌，念进鼻。鼻下辨渣，辨出投序。投序用柴灰写在筐底，写完扫净，像从没写过。无字才匿；匿才备得成。

刘婆补肘，肘布厚：「席又加一层？」

「风硬。」韩泥说，「沿净要紧。」

「沿净，手稳。手稳，饭稳。」刘婆哼。

「饭稳。」韩泥点头。

邻床问：「丑时还不睡？」

「编筐。」韩泥说，「筐卖钱，钱买米。」

「你卖筐卖那么多，米够吃吗？」

「够吃，也够买静。」韩泥道，「静值钱。睡吧。」

钱买米，也买匿丹需要的静。静在丑时，在雨夜，在席下那一线坛温。

他低声对坛：「备齐了，不叫。叫出来，路断。」

坛腹温，贴掌应着「备」字。备不是炼；炼在下一回，在门缝前，在还人那一步的前头。

更鼓二更，雨意又来。温汤囊挂到床柱上，柱旧，却挂得住。挂得住，才留得住门缝那盏灯。赖福脚步停门外半息，像嗅——嗅不到。席严，页在怀，料散三处，散得像杂役藏粮。

晨雾里分拣。指稳，鼻下分青褐。铁无言过，脚停：「你备什么？」

「备栏。」韩泥说，「栏清，瘴少。」

「答非所问。」铁无言盯他半息。

「问该问的，就听见了。」韩泥声平，「不该问的，我也不会答。」

铁无言极短一笑，像认可，又像警告：「手别飘。」

「不飘。」韩泥应，继续分拣。

坡下方向，沈枯芽又跑来，气喘：「叶姐说……别来丑。她还说……饼收到了。」

「好。」韩泥只一词。

「她咳，咳轻，可还是弱。」芽儿咬唇，「我送了刘婆姜汤。」

「你做得对。」韩泥说，「汤你送。人我暂时不见。」

「不见……那还怎么还？」

「还不一定要见面。」韩泥道，「见面有时会害人。害了她，我那碗汤白记。」

芽儿懵着走了。韩泥把愧和恩分开搁：愧是「别来丑」，恩是「还」。愧不挡恩，恩也不拿嘴说。嘴留给编筐、清栏、守坛。

夜雨第三更，他把三处料再核一遍。席下一点、筐底一点、干草下一点——点散，却都活着。活着，下一回才能炼；炼时不叫，叶汤那碗烫才还得动。

他按木牌，牌凉贴胸：「备过了。下一回留灯。灯留了，才好把丹塞进缝里。」

坛温一线，像听懂了。听懂了也不出声——出声是给赖福听。韩泥只把藤条勒实最后一匝，硬藤勒指，指疼，疼提醒投序。疼过了，手更稳。

霜色压在兽栏顶上，顶冰薄薄一层。他拍开霜，霜粉落袖，像倒渣灰。灰里有没有青芯，别人看不见；他看得见。看得见，就不慌。不慌的人，才能把匿丹的活计做成平常日子——平常到查库、领麸、端粥，一样都不少。

少一样，就会露。露了，路断。路不断，门缝那边才有灯可等。

白织月远廊又过：「丑杂役，夜里编筐？手还抖不抖？」

「不抖。」韩泥不抬头，「抖了筐散，散了米贵。」

她嗤远。嗤远了，备才更静。秦霜木鱼轻敲半下：「备匿别骄。骄了，香先泄。」

韩泥心里应：不骄。骄是求名的人干的；他还债的人，债没清就没有名。

他把温汤囊取下摇了摇，囊空，空得干净。空着正好——空着才等得住雨夜那粒真温。真温进囊，囊过门缝，灯油那夜就不是白留。

同棚少年睡着打呼。呼声盖住藤条响。韩泥趁势把干草坑口又埋平一分，埋得像从没动过。动过就等于给赖福指路；指路的事，他一件都不做。

天快亮时，他低声道：「料齐了，炉齐了，诀齐了。下一步只留灯——灯亮了，丹才有缝可进。」

晨起清栏，栏木结霜。霜白，白得像纸。他用袖擦开看木纹，木纹涩，涩像经页边。经页在怀，怀不离人——人离了，页就成祸。

温听雨远远洒扫，路过不停留，只留一句清声：「你眼神静了。静归静，别进丹阁后门。」

「不进。」韩泥应，「各做各的。」

「各做各的好。」她走远。缘苗不认名，名一认，风就往偏处吹。韩泥把这句话也按进备料账里：备的是还叶，不是攀阁。

午间沈枯芽又送半块饼：「叶姐让……让我吃，说我瘦。她还说，灯的事，她懂。」

「懂就好。」韩泥道，「懂了别说。说了，灯会灭。」

「我不说。」芽儿狠狠点头，「韩叔，料……你藏的那个，要不要我看堆草？」

「不要。」韩泥声硬半分，「草你离远。远了，你干净。干净，将来还敢做人。」

芽儿委屈：「我想帮。」

「帮就帮嘴严。」韩泥说，「嘴严是大帮手。」

傍晚管事喊集合，宣读冬夜巡棚规矩：「席厚不许过两层。过了，查私藏。」众人「嗯」一声。韩泥心里一沉——席下有炉片。沉完立刻想辙：丑时把炉片挪进筐底夹层，夹层更深，更深像破筐补丁。补丁不惹眼。

夜里他真把三片炉迁走。迁时掌抖一下，抖完更稳。稳着把席恢复成一层半，刚好卡在规矩边。卡边上，才活。活着，匿丹备才不算白忙。

赖福半夜踹门：「查私藏！」火折照进来，照席、照筐、照草堆角。草堆角他只扫一眼，未蹲。未蹲，浅坑就活。

「没有。」韩泥声平。

「没有才怪。」赖福啐一口走了。走了，邻床才喘：「我差点尿出来。」

「尿也别尿到坛边。」韩泥说，「尿了，沿脏。」

少年愣：「你还开玩笑？」

「不开。」韩泥躺下，「睡。醒了清栏。」

更鼓尽。他闭眼，眼前却是门缝一线。一线等灯。灯等他明日去留。留完，丹才有路。路备齐了，手就不慌。不慌的人，才能把「还」字做得不像偷。

坛温贴掌，像把「留灯」两字焐热。焐热了，他才合眼片刻。片刻够不垮，垮不得——垮了，坡下那人冬夜更黑。他撑着，只为下一夜把油壶搁进石下，搁得像雨夜常有人放物。
