# 第九十七章 挂牌试水

晨雾。

坊市南脚，韩泥挂木牌，牌歪，字丑，三字：「丑丹师」。师字大，大不要紧；大，是试，不是认。

牌下仍卖渣，渣三包，包旁多一小签：「可嗅，不包治。」

不包治，才像杂役该写的。该写的，管事少问。

第一个问价的，是外门瘦弟子：「嗅渣多少钱？」

「半文。」韩泥说，「嗅一次，不加价。」

瘦弟子递渣，渣灰。韩泥鼻下一动，动半息：「青芯过多，火路会焦。减一撮。」

瘦弟子愣：「你真嗅？」

「嗅。」韩泥说，「嗅完，手稳。」

瘦弟子付钱，付钱像试——试了，不传谣；谣在远，远不要紧。

第二个是药摊婆，婆眼浊：「帮嗅这包，嗅不出，不赖你。」

韩泥嗅，嗅出褐多青少，少不要紧；少，他指节在包上点三下：「加青，减褐，先温后投。」

药摊婆喜半分，喜不要紧；喜，不谢——谢轻，记在心里。

白织月路过，瞥牌，瞥完，声冷：「丑丹师？也配？」

韩泥不抬眼：「配试。试不过，撤牌。」

白织月袖风过，过不要紧；过，是风，不是判。判在丹，丹还未炼散。

铁无言立远，抱臂，不近摊。不近，像不认；不认不要紧，眼在，眼是证。

午后，管事弟子来：「谁许你挂师字？」

韩泥声平：「试。试不过，撤。」

管事记簿，笔停半息：「试三日。三日无投诉，再议。有投诉，连坐丙九。」

韩泥应声：「三日。」

三日，够了；够了，才知坊市认不认丑手。

赖福闻讯，晃秤杆：「丑丹师？晦气师！」

韩泥只答：「牌歪，人丑，渣真。」

真，不在嘴，在鼻。鼻下静，手不飘。

收摊，他数半文，文少，少不要紧；少，是试水，不是养家。

夜，按坛，低声道：「试过了。坊市没赶。」

坛腹温，像应「试」字。试，不是成；成在110，在远。

邻床问：「你真敢挂师字？」

「敢试。」韩泥说，「试不过，撤。撤了，仍清栏。」

丑时，静半刻。香落齐，掌纹温一线。温，像培元路又熟半分。

他低声道：「三日。三日后，再看风声。」

风声里，兽潮二字又近。近不要紧，近，要备。

晨雾里，木牌挂出，牌歪，字丑：「丑丹师」。

瘦弟子递渣，韩泥鼻下一动：「青芯过多，火路会焦。减一撮。」

管事弟子记簿：「试三日。」

赖福晃秤杆：「丑丹师？晦气师！」

「牌歪，人丑，渣真。」韩泥只答。

不近，像不认；眼在，眼是证。

收摊，半文少，少不要紧；少，是试水，不是养家。

第三日午后，有个外门弟子再来，再来，像认手。认手，不要紧；认名，才惹眼。韩泥仍半文，仍不加价。加价，像营生；营生，赖扒皮秤。

试毕，他撤「师」字，改「嗅」字，字小，小才藏得住。藏得住，才接得住98续摊——续摊，听兽潮风声更近。更近，柱上贴条，条丑，字真，真在「夜禁」。

第二日，牌仍歪。歪不要紧；歪，像他人。人丑，牌丑，手稳，坊市才肯续。

老客穿灰袍，不语，只递渣五包，包上系红绳——绳旧，像试人。韩泥一包包嗅，嗅到第三包，鼻下停半息：「这包有霉，霉在角，别投炉心。」

老客眼一细：「你怎么知角霉？」

「嗅。」韩泥说，「嗅完，不取钱。」

老客收包，收时塞一块下品灵石碎片——碎片小，小不要紧。小，不能收；收，像受贿。

韩泥推回：「半文就够。够，是试，不是营生。」

铁无言立远，抱臂，不近摊。赖福过，鼻动：「嗅渣也收钱？晦气。」韩泥只答：「半文。」半文，才像末席该收的。

更鼓沉，韩泥按木牌，牌凉贴胸。试过了，三日之约在前。

挂牌试水，牌小，字丑，字写「丑丹嗅」。嗅，不是师；师字大，大惹眼。惹眼，赖扒皮来。来不要紧，字小，小才试三日。

第一日，半文嗅一次，不加价。加价，像招摇；招摇，白织月袖风至。至不要紧，至也只冷一句：「卖腥。」卖腥不对，他卖渣，渣净，栏清——清，是常活回话。

瘦弟子问价，问完递渣，韩泥鼻下一动：「青多，减一撮。」减一撮，像试炉控火——火路匀，渣才正。正，不在嘴，在鼻。鼻准，半文才值。

药摊婆三指点头，点像记功。记功轻，轻不要紧，轻在坊市，不在册。册在管事弟子簿——簿上「试三日」，三日无投诉，再议。再议，就是98续。证，不在捧，在看你手抖不抖。抖，撤摊；不抖，续。

赖福过，记过簿悬而未落——未落不要紧，未落因无投诉。无投诉，因嗅准。嗅准，不在名，在沉腐浮三字落齐。落齐，更鼓沉，韩泥按木牌，牌凉贴胸。三日之约在前，风声里兽潮又近。

挂牌试水，「丑丹嗅」三字，字小，小才不惹眼。惹眼，白织月袖风至，至也只一句「卖腥」。卖腥不对，他卖渣——渣净，栏清，清是常活回话。回话短，短像末席。

半文嗅一次，不加价。加价像招摇，招摇招赖扒皮。赖扒皮不要紧，簿上「试三日」——三日无投诉，再议。再议，98续；续，99闻兽潮条至。条至在远，远不要紧，97只试水。

瘦弟子递渣，韩泥鼻下一动：「青多，减一撮。」减一撮像控火——火路匀，渣才正。记功轻，轻在坊市，不在册。册在管事弟子手——手稳，簿才稳。

铁无言立远，眼在，眼是证。证不在捧，在看抖不抖。三日之约在前，兽潮在条上。条在远，手在摊。摊在，路就不绝。

试三日，第一日半文嗅一次，第二日药摊婆三指点头，第三日管事记簿「试三日，再议」。再议，就是98续；续，改师为嗅，字小，小才藏得住。藏得住，99才闻兽潮条至。条至在远，97只试水——试水，不在名，在鼻。鼻准，瘦弟子减一撮青，药摊婆加青减褐，老客在98才现。现不要紧，97只证手抖不抖。

铁无言立远，眼在，眼是证；赖福记过悬而未落，未落因无投诉。无投诉，因嗅准；嗅准，因沉腐浮落齐。落齐，更鼓沉，按木牌，牌凉贴胸。条猎猎，像催备；催备，99在前。手在，摊在，就不滚。

第三日黄昏，簿上「暂留」二字，字小，字真。真不要紧，真在鼻，鼻准才暂留。暂留，98续摊；续摊，听兽潮近。近，备在99，守在100，撒在102。撒在远，97只试水——试水过了，手在，就不滚。

挂牌试水三日，第一日半文，第二日药摊婆三指点头，第三日管事记簿「试三日，再议」。条至在柱，柱在界，界在命；命在丙九，丙九守栏，守在远，守在100东角。

铁无言立远，抱臂，不近摊——不近，像不认；不认不要紧，眼在，眼是证。赖福记过悬而未落，未落因无投诉；无投诉，因嗅准。嗅准，因沉腐浮落齐——落齐，手才不飘。飘了，白织月一句「卖腥」就成真；不成真，因卖渣，渣净，栏清。清是常活回话，回话短，短像末席。更鼓沉，按木牌，三日之约在前，兽潮在条上。

散摊后，韩泥编筐半刻，筐卖钱，钱买米——米买静，静在鼻下。鼻下嗅诀，沉腐浮落齐，落齐，三日试水才算完。完，不在荣，在听兽潮条猎猎。猎猎，像催备；催备，99在前。手在，就不滚。

管事弟子第三日收簿，簿干，墨新，新不要紧，新才像试水有果。果不在名，在鼻——鼻准，沉腐浮落齐，落齐，手才不飘。飘了，撤摊；不飘，98续。续了，99条至，条至，100前夜，101备，102撒，103压，104至——至在远，97只证三日。三日证毕，丑手才认。认不认，不在这一日，在130远。远不要紧，手在，摊在，路在。路在，就不滚。
