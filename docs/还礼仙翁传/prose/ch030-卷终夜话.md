# 第三十章 卷终夜话

秋末，灵息复苏风声起。

非正式帖，是口耳相传，像风先于人到了青岚峰：五宗天试，或将重启；各宗需提前物色护法长老，年轻一辈试剑试丹，老一辈押阵护道。

柳青鸢约莫长春峰顶夜话。

月白，云薄，峰顶风硬，吹得青衫猎猎。她递茶，茶热，像把战后的松，重新系回宗主腕上。

「师兄，」她低声，「天试若开，我需护法。你能赠，不能打。」

莫长春接茶，抿一口：「我能赠，不能打。」

柳青鸢笑，笑里疲，疲里坚：「正好。你赠，我打。青岚门不缺一柄剑，缺的是还礼的路。」

她递**风声简**，简上无多字，只列五宗名、试期约、护法名额。

莫长春接简，袖中簿一烫，像等这一刻。

简与簿意相合，淡金字浮，缓缓成句：

**缘箓二转·善赠——师尊、柳氏、苏氏/顾氏，首赠已满三人。**

因果沉，再涨五分，累计约四十五，像东脉一季的雨，终于落进账里。

寿元行跳：六月。

莫长春瞧见，不慌：「六月，第二卷够开场。」

柳青鸢望他：「师兄，礼还没送完。」

「没送完。」他抚杖，竹纹游动，气音温柔：

够。

莫长春点头：「够送，够还，够到天试门前。」

峰顶远瞧，东脉灯火像坊市烟火升上山；丹堂一线夜灯，像苏氏丹香未歇；杂役堂窗下人影，像顾小满仍在记册。

柳青鸢忽然道：「苏师姐今日炼培元，炉火到天亮。她没问天试，只问……师兄九月够不够。」

莫长春笑：「够。不够也会装够。」

「赵执事还了三颗私丹。」柳青鸢续道，「他说还库，不是还你。心却松了一寸。」

「松就好。」莫长春望执法堂方向，「妒松了，才接得住下一礼。」

柳青鸢沉默片刻，低声：「厉无殇三日之期，已过两日。赤焰旗还在东脉外。」

「旗在，礼就在。」莫长春道，「手帕定身是部一的名场面，部二该有人接场了。」

月下，风更大。

柳青鸢收简，起身：「师兄，回吧。峰顶风硬，九月经不得久吹。」

莫长春拄杖起立，灰袍在月下像一张旧符，符纹淡，意却满。

他念，像卷终一句，也像开卷一句：

「礼还没送完呢。」

风大，正好送礼。

簿在袖中沉，沉得像四十五斤人情——大因果还在天试门前，像帖，像雷劫，像师尊入关前那句「够活到你后悔」。

杖答：够。

他笑：「够。」

峰顶下，外门弟子尚议论天试风声，议论里怕，也燃——怕五宗试剑，燃的是青岚门终于不再只守东脉。

苏念慈在丹堂封炉，封得稳，像把卷一余悸也封进瓷瓶。她望峰顶方向，低声：「师兄，六月……够炼几炉了。」

赵玄机在执法堂抄门规，抄到「护法长老」一条，停住，墨滴成字，像莫长春那句「我能赠，不能打」又浮上来。

顾小满在杂役堂记册，册上多一行：东脉妖讯，赤焰旗，七日之期——她不懂天试，却懂「记」字，记得像还恩。

莫长春下山，杖点每一步，袖中簿沉，沉得像四十五斤人情。

风大，正好送礼。

大因果还在天试门前，像帖，像雷劫——卷一落了账，部二才刚开场。

峰顶远瞧，东脉灯火像坊市烟火升上山；丹堂一线夜灯，像苏氏丹香未歇；杂役堂窗下人影，像顾小满仍在记册。柳青鸢忽然道：「苏师姐今日炼培元，炉火到天亮。她没问天试，只问……师兄九月够不够。」

「赵执事还了三颗私丹。」柳青鸢续道，「他说还库，不是还你。心却松了一寸。」「松就好。」莫长春望执法堂方向，「妒松了，才接得住下一礼。」

峰顶下，外门弟子尚议论天试风声，议论里怕，也燃。苏念慈在丹堂封炉，封得稳。赵玄机抄门规，抄到「护法长老」一条，停住，墨滴成字。顾小满在杂役堂记册，记得像还恩。

柳青鸢约莫长春峰顶夜话。月白，云薄，峰顶风硬，吹得青衫猎猎。她递茶，茶热，像把战后的松，重新系回宗主腕上。「师兄，」她低声，「天试若开，我需护法。你能赠，不能打。」「我能赠，不能打。」莫长春接茶，抿一口。

简与簿意相合，淡金字浮，缓缓成句：**缘箓二转·善赠——师尊、柳氏、苏氏/顾氏，首赠已满三人。**寿元行跳：六月。莫长春瞧见，不慌：「六月，第二卷够开场。」

围观弟子在峰下踮脚，踮得像等缘箓二转落字——字没落进他们眼里，落进袖中簿，簿沉，像四十五斤人情。有人低声：「天试风声，是真的？」另一人答：「真不真，六月见。」
