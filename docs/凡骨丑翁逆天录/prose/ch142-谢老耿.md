# 第一百四十二章 谢老耿

废窑夜，雨意沉。

韩泥占窑，窑深，声不出——不出，才像142该炼的夜；夜炼，不在众目，在掌。

投序照页，页浅，浅而真。灵砂先，药渣后，沉腐浮落齐。

炉温如秤，秤平，粒不杂。杂了，耿恩送不出。

老耿名在远——远在泥岗，远在抬棺绳不抖。绳不抖，棺不歪；粒不正，恩白送。

散成三粒，取一粒，粒丑，泥纹浅，嗅之正。

韩泥装灰布囊，囊旧，旧像28经片那路——路经僧，142经芽，都经叶家。

沈枯芽候坡下，候像门缝那线——线窄，能递，递了才不现形。

韩泥立三步外，声平：「培元散，给耿爷。经你手，我不进门。」

芽儿接囊，手抖，接稳了，恩才落笔。

叶青禾门缝未开，脉里培元仍温——温是95，散是142，两笔不混。

韩泥不辩，辩像显摆。显摆不要紧，正字在掌；掌稳，就不滚。

他低声对坛：「散正。耿恩，送出了。。」

坛腹温一线，像应「⑥」——抬棺恩，屋漏恩，今还一粒。

药山废窑，窑口红，红稳，稳像110首炼那夜——夜不同，路同；同在一品培元散，同在一粒还恩。

韩泥立炉前，独眼半阖，鼻下静一息。息里辨投序：灵砂先，药渣后，沉腐浮落齐。齐，手才不飘；飘了，粒不正。

老耿名在远，远在泥岗，远在抬棺那绳不抖——绳不抖，棺不歪；粒不正，恩送不出。

沈枯芽候窑外，候得像叶家那条缝——缝窄，缝能递，递了才经叶家，不经坊市眼。

散成，成三粒，粒丑，泥纹浅。浅不要紧，嗅之正；正，才配称能还才送。

韩泥不示人，示了，赖福嗅，白织月嘲。嘲不要紧，正字在掌，掌稳，就不滚。

他取一粒装灰布囊，囊旧，旧像28经片那包——包经僧，囊经芽，都经叶家。

叶青禾门缝未开，开不要紧，脉里培元仍温——温是95；散是142，两笔不混。

韩泥立门外三步，声平：「培元散，给老耿。」

沈枯芽接囊，手抖，抖不要紧，接稳了，耿恩才落笔。落笔在七笔⑥，⑥是抬棺，⑥是屋漏补瓮。他低声对坛：「耿恩，送出了。」坛腹温一线，像应「还」——还了，142才像第二部丹道，不像110自用。窑温退，退像110那夜——夜退，粒在。粒在囊，囊在芽手，芽手在叶家缝。韩泥不进门，进门像还情；情在粒，不在舌。舌要留来守坛，留来候143余波。

同窑外风硬，硬不要紧；硬，才记得粒正才要紧；正，嗅之辛淡，辛淡像95匿丹那线。

他望泥岗方向，方向远，远不要紧——远，粒才送得到；得到，七笔⑥才落半笔。

铁无言立窑外十步，不近，声低：「送出了，别回头。」「不回头。」韩泥说，「回头，像攀。」

更鼓沉，按木牌，牌凉贴胸：「散正。送出。记着。」坛腹温一线，像应「还」。

钉在丹，钉在恩，钉在经叶家那条不进门的路。路窄，窄才真；真，才像韩泥活法：坛在，人在，能还才送。送培元散，散在一品，品在110首炼已证，品在142才馈缘。馈缘经叶家，不经坊市眼——眼在赖福鼻，眼在白织月袖，眼在钱戾衡影；眼在，手更要稳。稳到废窑里只闻辛淡，不闻辱；辱在耳，不在鼻；鼻准，粒才成。

成三粒，粒丑，泥纹浅，浅不要紧，嗅之正；正，才配给老耿。老耿聋，抬棺恩人，绳不抖棺不歪——歪了，恩送不出；正了，七笔⑥落半笔，半笔在262还锻炉，142只送粒。粒经沈枯芽，芽手抖，抖不要紧，接稳像叶姐脉稳；脉稳，叶汤不混耿恩，七笔分列。

分列在心：①叶汤在前，⑥耿恩今还，⑦刘婆已还——还不混，混则手飘。飘了，丹不正；正，是五年淘的，不是层赏的。赏在远，在大比，在175筑基丹；142只还，不越界。不进门，不攀情，不拜暖——拜暖，路断；路断，143无波，144无名，145无驴。更鼓沉，窑冷，他低声对坛：「散正。送出。」坛腹温，贴掌，像应「还」——还了，第二部丹道才像走，不像停。

窑冷，粒在叶家缝，缝在耿爷怀——怀在，七笔⑥才落半笔。落半笔，262在远，142在今。

窑冷，粒在途，途在叶家缝，缝在耿爷怀。怀在，七笔⑥才落半笔。更鼓沉，他低声对坛：「散正。送出。能还才送。」坛腹温一线，像应「⑥」——⑥还了半，262在远。

刘婆端粥，仍多半勺。韩泥点头，谢轻，恩重。

药峰分拣照旧，指稳，鼻下静一息。

帖在怀，怀沉，沉的是序，不是惧。

白织月袖风过，韩泥不辩。辩，费舌；舌要留来守坛。

赖福廊下过，韩泥只答：「在。」在，就不滚。

柱上条角白一线，三年约在怀。怀凉，掌温。

坡下石阶湿，他数息，数帖，数债序。

同棚杂役笑，笑在耳，不在心。

废渣巷日斜，他验渣，渣正，手才静。

侧门驿亭风硬，他听风，风过，记在心里。

散队后，他过外堂阶下，分拣半刻，指稳——指稳，路就不飘。

编筐半刻，藤刺扎掌，刺浅，浅些沿才净。沿净，坛才藏得住。

丑时，练鼻下静。粟壳香淡，香一字不落。

管事记簿，记细，是小胜。胜在忍，不在辩。

他望坛，低声道：「手在，路在。」坛不应字，只温。

更鼓沉，天边白一线。白里，他按木牌，牌凉贴胸。

坊市尾巷风硬，他收摊如石，石在，就不滚。

验息后，他理怀内册页，页黄，字丑，丑而真。

铁无言声低：「别飘。」韩泥应声：「不飘。」
