# 第四章 脏水

韩泥去井台打水，天刚亮。

桶旧，绳磨毛，勒手心。他独眼眯着，避开井沿青苔，放绳，水面咕一声，桶沉了。泥岗村井浅，水浑，打上来要澄半日，澄不清也喝，喝了不死。

身后脚步脆，像碎瓷踩泥。

「让开。」

韩泥侧身。赵绣春穿半新棉袄，袖口绣花，在泥岗村就算体面。她身后两妇人，一个捂嘴笑，一个拿眼斜他——斜得像看地上的脏，脏的是水，不是人。

赵绣春不看他脸，看他桶：「晦气翁也配打水？莫把井打浊了。」

韩泥没应，把桶提出半寸，水晃，不洒。洒了，更像晦气。

「装聋？」赵绣春嗓拔高。井台边早起的人停了，停就看戏。泥岗村不缺戏，缺角，赵绣春最爱当角。

她扬手。

半盆脏水泼过来。

韩泥没躲全，半边麻脸和肩湿透。水臭，皂角混泥，像刷盆剩水。村人「哟」一声——同情在眼里，热闹在嘴上。

赵绣春拍手，笑：「冲冲晦气，也算帮你。绝户翁，别怨。」

韩泥低头，水滴淌进衣领。他独眼不抬，像认罚，也像把辱记在心里。

他不辩。

辩了，赵绣春更来劲；辩了，村人记住的是「晦气翁顶嘴」，不是脏水。五十八年，他懂：赢嘴，输路。

妇人里有人低声：「过分了……」

赵绣春回头一瞪，那声就灭。韩泥不指望谁出头——出头要还人情，他还不起。

里正巡巷路过，瞥井台水渍，瞥韩泥湿衣，没停，只扬声：「井是村的。谁再泼，记名。」

赵绣春脸一白，甩袖走快。走快，像怕记名——记名，戏就不好看了。

韩泥等人群散尽，才把桶打满。

水浑，照不出脸。他也不需要照。他低声道：「水脏了，人没脏。」
回村路上，童谣丫蹲墙根，见他湿半边，拍掌念新词：「晦气翁，脏水翁，靠近要倒霉——」

韩泥停步，看她一眼。

就一眼。独眼平，不凶，像看一粒石子。童谣丫声音卡壳，卡半句，抱着石子跑了。

韩泥脚步没停。吓娃不是本意，本意是：你念你的，我走我的，念不死我，就走不死路。

泥屋门口，坛在角。

湿衣搭绳上，绳弯。他坐灶前，灶冷，等干。心里叶丫头的汤还在，是头一笔，最深——赵绣春这一泼，是头一笔辱。

韩泥低声道：「记着。」

不写「赵绣春」。记名像恨，恨也记在心里，但辱与恩不能混。辱这笔，候着，不急着报。报要时机，时机靠活，活靠忍。

外头马春花嗓又响，骂鸡，啐一口「晦气」，不知骂鸡还是骂谁。韩泥不应，拾藤条，绕膝，勒指。指疼，疼让人清醒。

明日还打水。水浑，也得打。赵绣春泼得再凶，泼不进心里——恩仇记在心里，不记臭。臭会散，恩仇不散。

编到第三圈，他按了按心口，硌得踏实——叶汤在，赵辱在，分列，不混。

湿衣干了，他擦坛沿。擦过一遍，沿口比白日温一线——温应恩，不应辱。辱泼得再凶，泼不进坛；坛在，根在。

门外风过，湿衣摆一下。韩泥不看，只看藤条——藤条结实，筐就结实。

赵绣春门内骂：「关什么铺！晦气！」

骂声尖，尖不过远处叶家秤杆落盘的脆。脆响起，赵声哑半寸——哑半寸，也是小胜。张麻婆傍晚送半个馒，馒热，塞门缝：「吃。编筐手不能抖。」

韩泥掰半，半留门墩，半自食。食完，嗓稳：「记着。」

卖盐老汉路过，瞥他湿衣，没躲，只点头：「赵家那盆，过。」

沈枯芽娘在巷口咳，咳得像问：「人……没事吧？」

韩泥「嗯」一声：「没事。水脏，人没脏。」

芽娘点头，拽娃走远。走远，像收了一句公道——公道轻，够撑明日井台。

过字轻，轻过脏水。韩泥「嗯」一声，算收了一句公道——公道不暖，够撑明日再去井台。

他低声道：「水脏了，人没脏。坛还在，根还在。」

坛角蒙尘，沿口温一丝不退。不退，像替他守着分列的心记：恩归恩，辱归辱。

夜里，他编筐至三更，指稳，稳得像告诉全村：水泼得脏，手不脏。手不脏，筐就结实；筐结实，路就还在。

赵绣春门内再骂，骂不过叶家秤声。秤声稳，稳得像钉——钉住叶家还在，钉住他心口恩还在。

他低声道：「辱记着，不急报。报要时机，时机靠活。」

陈姑路过，棉袄领拢紧，低声：「水脏，人心别脏。心脏，才真晦气。」

韩泥「嗯」一声，不谢。谢轻，记重。

明日井台还去。还去，像告诉赵绣春：泼得脏，泼不散命。命在，坛在，恩在，辱分列，不混。

里正晨巡又咳：「再泼，记名报县。」咳声沉，沉得像给全村缝一道边——边内过日，边外惹祸。

翌日井台，赵绣春远，不敢近。

里正咳：「昨日那盆，记下了。」

韩泥提桶，水仍浑，却稳：「记他的名。泼我的水。水浑，路不浑。」

里正点头，走两步又停：「叶家那边，别牵扯谣。」

韩泥：「牵扯的是盆，不是叶。」

卖盐老汉蹲沿，没躲他湿肩：「赵家过。过，辱记在心里。」

韩泥「嗯」一声，桶提直。

童谣丫远处偷看，没念词，只把石子排成一串，串像路。

韩泥瞥一眼，不骂娃：「排好就排好。路长，慢慢走。」

回屋，湿衣已干，他擦坛沿，沿口温一丝——温应叶汤，不应赵盆。蒙尘里水汽爬半寸，即敛，像瓮在睡，却分得清恩辱。

沈枯芽娘隔门问：「韩叔，水还打？」

韩泥「嗯」声：「打。井是村的。」

芽娘点头，拽娃走远。走远，像替全村记一句：水浑，人不必浑。

张麻婆塞门缝半块饼：「吃。手别抖。」

韩泥「嗯」一声，饼掰半，半留门墩——留，像还恩，也像告诉赵家：泼不散手。

赵绣春门内啐：「晦气扫不净，净的是盆。」

韩泥不接，只道：「盆浑一时，手不必浑。」

衣拧干，他仍去集尾。集尾只看筐，不看脸——脸丑，筐紧，紧就能活。活才能候村会那锤；锤落，脏水才洗得清，清的是名，不是嘴。

脏水干在衣上，像旧辱一层；他不辩，只记：水脏人没脏——记实了，才撑得到公审那锤，叶丫头证白落地，赵哑口，名节才立，汤才能还热。
