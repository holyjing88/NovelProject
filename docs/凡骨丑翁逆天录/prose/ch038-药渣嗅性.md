# 第三十八章 药渣嗅性

药渣场灰热，热气罩脸，罩得人喘。韩泥分拣，慢里鼻尖一动——苦蒿、蛇藤、叶振东那包药的余味，都在鼻下排着。渣里一点香，香淡，淡得真，真就不乱。

同场杂役掩鼻：「臭！」

他不掩。掩了，闻不出路。路在香里，香在渣里，渣在手里，手不能抖。另一人呕了一声：「这灰吃进去，肺都要黑！」

「黑肺仍要清。清完，才有饭。」韩泥说。那人骂骂咧咧，仍拨渣，拨乱了两堆。管事一眼扫来，他赶紧学着分堆，又喝：「丙九的，今日渣场清到酉时。清不完，没饭。」

韩泥指不停，把灰渣拨成三堆：药渣、炭渣、废渣。拈渣时，灰里香不散。白织月再经，袖风冷：「闻什么？闻你也闻不出灵根。」

「闻药性。药性在渣，不在根。」

白织月一滞，唇线绷了一息，走了。少一句嘲，也是小胜。同场有人笑：「白师姐都懒得骂你了。」

韩泥不答。舌要留来辨香，留来记。铁无言抱臂立桩，声低：「渣别乱丢，丹房要回收。一粒不漏，回收才收你。」

他把香渣单放小篓。篓小，香聚，聚才不泄。铁无言瞥他：「鼻灵？」

「鼻笨。」韩泥说，「笨久，笨出记性。记药性，记恩一样——记错了，还错了。」

铁无言盯着他分拣的手，手稳：「笨人守规，比灵人漏渣强。」

「不漏。」

铁无言应一声，走了。同场嘀咕：「丑杂役也配跟铁师兄说话？」

铁无言扫一眼，声不高：「渣不漏，话就少。话多，往往是手乱。」

嘀咕断了。韩泥记这一眼——记的是丹道门口规矩，不是面子。酉时近，渣场渐空。香还在篓里，也在他鼻下。管事来验：「丙九韩泥，渣清完了？」

「完了。香渣单放，没乱丢。」

管事瞥篓，又瞥三堆：「分堆也对。谁教你的？」

「没谁教。鼻笨，笨出记性。」

管事哼一声，却记了一笔：「明日仍渣场。」

同场杂役凑近：「你分三堆，多费工夫，饭又不加。」

「加不加，管事看。」韩泥说，「堆乱了，丹房不收。不收，记名。记名，滚。」

那人啧一声，学他分堆，手乱，灰扬一脸。管事骂：「学也学像点！灰进眼，明日你顶他的酉时。」

杂役揉眼，不学了。韩泥指仍不停，香渣篓又满一层。午后换场，丹房外药烟浓，浓得人咳。他隔街站着，闭息，辨出一丝甘——叶家药铺、叶振东那包药、「必还」未竟的余味。会，不声张。声张，赖福又来克。

管事路过，喝：「别堵道。」

他挪半步，鼻仍辨。第二味苦里带燥，燥是蛇藤渣余气。丹房弟子过，捏鼻：「杂役也闻？」

「闻路。」

「路在哪？」

「在丹房门里。」韩泥说，「门里我进不去，门外我记。」

弟子摇摇头走了。他记下一味苦，一味甘——甘苦分开摆，混了，还时必错。街角赖福瞥见他伫立，喝：「丑杂役，站岗？站岗也闻不出灵根！」

他挪回渣场方向。赖福啐：「闻也没用，丙九一辈子闻粪！」

韩泥不回头。辱记着，鼻仍辨——第三味涩，涩是废渣里残药。残药不能丢，丢则路断。回场，他把涩渣并入香篓，篓满而不泄。

夜，丙九房油灯暗。他对坛低声：「渣里有香，香是路。路通向炼丹，炼丹通向还。」

坛壁水汽一线，爬半寸，敛。敛着不是拒，是等。他翻刘婆补过的肘，肘暖。邻床问：「你天天对坛嘀咕什么？」

「记。记香，记恩，记饭。」

「记那么多，睡得着？」

「睡不着，就别睡。醒着记，记错了少一口。」

邻床翻身睡了。另一杂役隔床问：「白师姐都说你闻不出灵根，你还嗅什么？」

「嗅饭。饭在，才嗅得出药性。药性在，才还得上恩。」

那人愣了愣，也睡了。刘婆门外咳：「明儿渣场，别漏。」

「不漏。」

他把香渣篓藏床底。路不能张扬，张扬来克扣，克扣来滚。管事夜巡路过，敲棚柱：「丙九韩泥，明日仍渣场。漏一粒，记名。」

外头风硬，吹得棚顶灰落。灰落肩，他不拍——拍了手上有香，香散了，路就窄。他摸篓沿，余香未散，像丹道门缝里的第一道光：光不进丹房，光在心里。心里记着铁无言那句「渣别漏」。

「铁无言要渣不漏，管事要渣场清。」他低声道，「都要，我就都做到。做到，路才宽。」

刘婆又咳：「别漏。漏了，我扣你盆。」

他闭眼，鼻下仍有一丝甘。甘短，够走；走，够熬到该醒的日子。丑时，渣场味淡半寸，淡得像白日铁无言那一瞥——瞥见手稳，明日少踹一脚。坛壁一线水汽，应「不漏」，不应灵。灵在后头；醒前，这一缕甘就够撑步子。

铁无言远处又咳一声，像确认还在：「丙九韩泥，明日仍渣场。」

韩泥应声。他不报灵根，只报稳——稳才配嗅渣，才配将来辨香。辨香过，还恩才不飘；飘了汤就凉。他把香篓推深半寸，像把路藏好：克扣克不到，管事却看得见。

天将亮，渣场雾起。雾薄，像丹房廊下那半刻辨香的预演。预演在鼻，不在嘴；嘴稳不算，鼻下静才算。静了，甘短苦长才分得清。他覆席盖坛，沿净，不漏，路在。

丑时近半，同场杂役又蹭过来，压低嗓子：「铁师兄真看上你手了？看上手，能换个好班吗？」

「换不换，不在嘴上求。」韩泥说，「手稳，簿上才有字。有字，路自己宽。」

「你就这么认？」

「认。」他摸床底香篓沿，「认渣，认饭，认自己鼻子笨。笨才肯慢，慢才不漏。」

那人啧啧两声走开。走开也好——话多的人堆往往乱。韩泥闭眼，复盘日间三味：苦蒿长，甘短一线，涩贴底。三味像三笔账，笔笔分开，才不至于还错叶家那包药的味。

贴席卧，息匀才睡。睡浅，鼻却醒——醒着记腥甘分界。分界清，明日不漏；漏一粒，管事记名，名压路。路在末席，末席靠忍。舌要留着，先把鼻下这一缕甘记热。

更鼓又响，赖福远处骂人踢桶。桶响刺耳，刺不进他鼻下那点静。静着，他忽然想起泥岗药铺帘后的辛味——辛得像姜，又像嘱咐：趁热。趁热记着，还时才不会拿错碗。

天光一线时，他披麻衣起身，篓不撒手——撒手气散，辨不准；准一次，名往后挪半寸。半寸虽小，末席活大。侧门外霜白，他把香篓抱紧些，对路过的管事点头：「酉时前，我在场。」

「在就好。」管事没多看他，「漏一粒，别怪簿子冷。」

「不漏。」韩泥应完，步子踩实。实着去渣场，像去赴一条约：约里没有灵根二字，只有「干净」和「记得」。记得了，丹房门缝里那缕光才肯在心里多停一刻。

霜上脚印歪，歪也一步一步。他自语似的补一句：「铁无言要的是规矩，我要的是路。规矩守住，路才不叫灰埋了。」埋不得——埋了，叶家那包药的余味就找不回来；找不回来，必还二字就悬在空里。空里站不住人，他得把脚踩在实灰上。

侧门风硬，他拢了拢肘上针脚，针脚暖。暖提醒他：恩在肘，路在鼻，辱在赖福嘴里。嘴里的东西不必还，鼻下的甘必须留。留住了，明日簿子上那笔「仍渣场」才不是重复受苦，而是又近丹房半寸。半寸够他撑过灰热一天。一天一天叠上去，门缝里的光才会肯亮长些。亮长些，他才敢把炼丹二字往心里搁。
