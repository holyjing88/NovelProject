# 第六十三章 瓮醒

沉丹宗杂役房，夜深。

韩泥清完兽栏，洗手，水冷，指缝腥。五十九岁了——入门半年，须发更白，背更驼，独眼仍利。村里「晦气翁」换了名，叫「丑杂役」，骂法不同，味一样。

赖扒皮今日又克扣，克半袋麸皮。韩泥不辩，只记。记在心里，记在册上，记在坛边——坛在，仇在，恩也在。

他回屋。屋小，漏风，比泥岗泥屋强在不漏雨，弱在隔壁鼾声如雷。鼾声里，他摸腰间册，册厚，厚得像五年泥岗村。

坛在。

泥岗村带来的漏底腌菜坛，宗门不许摆，他摆床角，用破席盖，像藏私。私不在坛，在命——命跟了坛半生，丢不得。入门时，管事骂过，骂他带晦气，他「嗯」一声，席盖更严，骂不过席。

丑翁账在腰。恩笔密，辱横浅，仇叉深。册页潮过几回，潮得他知：坛与册同源，醒一头，另一头也醒。

今夜丑时。

韩泥没睡。他盘坐，针在指，粗针，缝筐那枚。油灯豆，豆光照坛，坛影歪，歪得像丑翁。

针尖对准指肚，扎。

血珠挤出，饱满，像一颗丑红的豆。

他揭席，坛口破布扯开。

坛里空，坛壁潮，潮得像有东西在底呼吸。泥岗村那夜丑时，坛温一线；入门后，坛时凉时温，像等人来滴这一滴。

韩泥俯身，滴血。

一滴。

落坛底细缝，渗，渗进泥，渗进釉裂，像泥岗村梦里的朱红。

二滴。

坛壁颤。极轻，像心跳，像有人在里面翻身。

三滴。

「嗡——」

一声闷，非雷非钟，在颅骨里响。韩泥独眼一花，见坛底水光聚，聚成一线，线爬壁，爬出符形——符形丑，歪，像孩童乱画，却亮。

亮一闪，灭。

坛口吐出一口白气，白气冷，冷里带药香，带泥腥，带腌菜坛旧日的酸。韩泥手背起皮，像被火舔，又似冰咬。他没躲，躲了，像怕。

瓮醒了。

不是梦，不是丑时幻觉。坛底那道缝，合了半分；坛壁符形隐去，余温灼掌，掌纹里像多了一道路。

韩泥掌心按坛，低声道：「醒了？」

无人答。答他的是册——腰间册页骤烫，烫得像有人翻页。他摸册，册脊「丑翁账」三字深了一线，像墨被血喂过。纸面空白处，隐隐浮出极淡的影，影像瓮，像册，像一个人驼背的侧影。

里屋，兽栏那边有夜鸟叫，叫一声，哑。

臾墟子是否在坛底，是否在将来开口，此刻不知。此刻只有坛，坛醒，气吐，命牵。

外头鸡未鸣，狗未吠。沉丹宗夜禁脚步远，远得像另一个世界。钱戾衡的名字在远处，在玉牌里，在三年约里——约还在，瓮先醒，醒得不早不晚。

韩泥把席盖回，盖严，像盖一个刚出生的秘密。秘密不能叫，叫了，赖扒皮来，管事来，煞骨门旧债也来。

他擦掌血，血入布，布黑，像符。符丑，真。

独眼望窗，窗纸白，白将亮。

「记着。」他对坛说，对册说，对自己说，「瓮醒了。路还长。汤还要还。」

坛底余温，贴掌，贴到丑时将尽。尽前，他听见极轻一声，像叹息，像回应，像瓮里有人说了半句听不清的话。

韩泥不追问。追问像少年。他只按掌，按实，按到掌烫，烫得踏实。
