# 第一百二十九章 备战帖

试台末项前夜，备战帖至。

山门执事传话，话像鼓余音：「明日末项，培元散比试夺魁。魁者，挂牌丑丹师，递大比预选帖。」

话落丙九，丙九嗡一声。嗡里，赖福脸白，白得像嚼不出晦气：「丑杂役夺魁？魁，太阳西边出！」

韩泥不辩。辩，费舌；舌要留来养息，留来守最后一粒。粒在怀，怀沉，沉得像这一夜要把债序压实。

玉牌贴胸，凉，凉里烫着三年约——约说炼气十三，站钱戾衡台前，才不滚。

刘婆端粥，仍多半勺，半勺像送行：「明日末项？」

「末项。」韩泥说。

「末项别空肚。」刘婆哼，「空肚，手飘。飘了，新衣还不上。」

他点头。谢轻，恩重。

铁无言立门外，声低：「明日我站台侧。魁名落，别辩。辩像显摆恩。」

「不辩。」韩泥说，「恩偿，静着还，还到刘婆手里，才算数。」

钱戾衡影在山门帖背面，背面字淡，淡得像眼风：「言伯钧预选，内门少主监。」

白织月袖风过，过得冷：「明日魁，仍是运气。」

韩泥捧陶罐，罐稳：「运气试台见。见正，见还，见刘婆。」

沈枯芽远远鞠半躬，不近。不近不要紧，徒恩清了半块灵石；清了，韩泥心里再落一笔：帖、魁、还、大比——四笔分开摆，不搅成一锅粥。

秦霜远木鱼敲一声：「帖至，心别邪。」

韩泥不答，答在心里。

邻床少年悄问：「你真敢夺魁？」

「敢。」韩泥说，「不夺，三年约白接。白接，恩断。」

他望坛，低声道：「明日魁，后日还刘婆。还了，帖在怀，大比在路，钱戾衡在台前。」

外头山门风硬，帖角掀，掀出「大比」二字影。影在字后，像仇在路后；路后不要紧，刘婆这笔先还。

更鼓将尽，天边白一线。白里，试台木色浅，浅像当年踹台——踹台接了约，约接到明日魁。

他按心口，低声道：「汤还是热的。」

六字落，像路标——标向还刘婆，标向大比帖，标向钱戾衡台前不滚。

丑时，他理布包：新衣叠实，钱串齐，陶罐封泥，九粒培元散在怀。怀沉，沉像最后一笔待落。落要正；正了，才配夺魁。

老杂役问：「你真敢上台？」

「敢。」韩泥声平，「不夺，约白接。」

同棚杂役赌明日魁名：「丑骨夺魁，太阳西边出。」

他不接茬，接在陶罐上——罐稳，粒正，就不像赌。

老耿挑水路过，水洒稳，声哑：「末项别拜暖。拜暖，路断。」

「不拜。坛在，人在。」韩泥说。

赖福脚步在廊外停半息：「在不在？」

「在。沿净。」韩泥答。

在，就不滚。不滚，末项才像还恩，不像赌名。

管事弟子传话第二回，话硬：「末项卯时，别迟到。」

「不迟。」韩泥只答。

迟了，手飘；飘了，刘婆新衣还不上。

铁无言又补一句：「手在就行。」

「在。」韩泥应。

刘婆灶边再看他一眼：「手稳，饭稳。饭稳，恩不断。」

他点头，把半勺粥喝尽。姜辣一口，辣进心口，像提醒：明日不是显摆，是还。

沈枯芽怯，悄问：「韩叔，怕不怕？」

「怕。」韩泥说，「怕也守。守完，才有饭。」

白织月远廊过，袖风冷，没停。没停，小胜——胜在沿净，不在嘴。

他低声对坛：「明日魁，后日还。还了，帖在怀。」

坛腹温，贴掌，像应「热」字。热在汤，汤在还刘婆前头。

丙九廊下灯灭两盏，灭不要紧，坛腹温一线还在。还在，就不滚。

坡下石阶湿，他数息，数帖，数债序。债序四笔分开：帖近了，魁在卯时；魁后还，还后大比——一步一步，脚才能踩实。

药渣场角，他备料半刻，料细，像末项前最后一筛。筛齐，手才静；静了，夺魁才像活。

编筐棚里，藤刺扎掌，刺浅。浅些，沿才净；沿净，坛才藏得住。藏得住，这一夜才不算空。

废渣巷风硬，硬像山门帖角。角掀半寸，掀不要紧；掀在远，手在近。

外堂阶下，管事夜巡记簿，记「丙九韩泥，末项前夜」。记细，又是小胜。胜在忍，不在辩。

他再按陶罐，罐底温一线，温像应「末项」——项在卯时，还在刘婆针脚前头。

「别空肚。」刘婆最后又叮一句。

「不空。」韩泥说，「空了，手飘。飘了，递不出。」

递不出，五年针脚就白暖。白暖他受不起。

窗外帖角又掀半分。掀里「言伯钧」三字影淡淡的。淡不要紧，手稳了就能接；接住了，钱戾衡影也在字后——字后远，刘婆近。近的先还。

他闭眼半息，半息里只留六个字：汤还是热的。热着，明日鼓响才不慌；不慌，魁名才落得住；落住了，新衣才递得进她抖着手里。

中夜雨丝斜。雨打山门帖角，角更潮湿，字影却更沉。沉得像催：明日别迟到，迟到手飘；手飘了，还路断。

韩泥披袄到廊下看帖一眼。帖上「大比预选」四字歪，歪得真。真下是言伯钧名，名后是钱戾衡影。

影远，他不追。追像把自己提前送上台前——台前要等炼气十三站稳。

铁无言撑伞过来，伞旧：「看它干吗？」

「认路。」韩泥说。

「路在明日掌上。」铁无言道，「帖是影子。影子吓不死人。」

「吓不死。」韩泥应，「只提醒我不滚。」

两人站一息。雨小了。小雨里，丙九灯更昏。昏灯下他看见刘婆灶间火苗还亮——亮着，像说：空肚不准上台。

回屋后他再封一次陶罐泥口。封严了，才把新衣布包压到最贴身的一层：「明日魁。魁落，就递。」

邻床突然翻身：「你嘀咕啥？」

「还账。」韩泥说。

「还谁？」

「补过我袖的人。」

少年哼一声，又睡。睡里不知明天鼓有多硬。鼓硬不要紧，罐更硬；罐硬，粒正，就不滚。

五更前他又醒一次，把九粒倒在掌心再嗅。嗅完仍正。正了，才敢把粒收回罐里——收回时手不颤。

不颤，是今夜粥还烫，针脚还在，帖也已至。

「备战齐了。」他低声说。

齐不是热闹。是罐、衣、钱、牌、粥，五样各归其位。位正了，明日上台才不像捡漏。

捡漏是运气；他要的是五年淘出来的正。

白织月若在场，定又冷一句「仍是运气」。冷句他早准备好了回音：运气试台见。见完正，见完魁，见完还——还完，嘴尖也短。

他披衣到灶边盛半勺冷粥垫底。冷粥也能顶饿。顶住了，手才不飘。

刘婆迷迷糊糊问：「咋又起？」

「怕空肚。」韩泥说。

「怕就对。」她翻个身，「空肚的人捧不住魁。」

「我捧得住。」他把碗洗了，「也递得动。」

递得动三个字，把她说清醒半分。她睁眼看他一眼，浊眼里那点清又冒出来：「那就去。去了别丢针脚脸。」

「不丢。」韩泥带上门。

门外雨已停净。山门帖湿痕未干，字却更醒目。醒目像一条绳，一头拴明日鼓，一头拴他还没交出的布包。

绳紧了，人就不能装睡。

他走回棚，在门槛上顿了顿脚底泥：「明日，见。」

见字落，远处似有更鼓收尾。收尾后，就是末项。末项一开，备战帖便从墙上走进手里——手里沉了，还刘婆的路便真正铺开。

鼓未响，帖已至。帖至，路就短。短到明天，短到还。

还字悬在夜里，像一枚未落的锤——锤落前一夜，他把袖口针脚再摸一遍，摸暖和了，才敢睡。

枕边陶罐凉而沉。沉的是明日魁，也是后日还。还字未落，人先睡稳半息——半息够了，鼓就不会把他吓飘。

帖在山门上，像一只未落的锤。锤影压夜，夜不散，直到卯时。
