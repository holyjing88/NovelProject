# 第四十九章 挽月反戈

执法堂，雨后天青，堂上却冷。

林挽月跪心，声颤却清：「谣言是赵师兄与弟子所设，浴房局陷害莫长老。顾小满无辜，弟子愿证。」

堂内一寂。赵玄机面色骤白，霍然看她，像看背叛，又像看镜子。

柳青鸢坐主位，眸光冷：「林挽月，你可知诬告反坐？」

「知。」林挽月磕头，「弟子愿坐。」

顾小满立侧廊，腕上还有罚跪雨痕，红，肿，却挺直，像半块灵石撑着她。

赵玄机咬牙，声哑：「挽月……」

「赵师兄，」林挽月不看他，「慕错了人，还能改。恩错了人，改不了。」

堂下弟子屏息，屏息里炸开一层浪——浪不是欢呼，是惊，惊林挽月敢反戈，惊赵执事脸色白，惊莫长老立堂侧不言，只点头，像把「以后还人情」又说了一遍。

苏念慈在丹堂廊下听见消息，炉火旺，旺得像替谁把脏字炼净。霍镇山在镇妖司骂：「执法堂终于肯洗脸了！」

魏无涯在檐角，扇合，低笑一声，像看一枚子落对了格，又像看妒火线终于收束一截。

柳青鸢拍案：「赵玄机，思过崖半月，加重。林挽月，罚役三月，逐出外门林苑。」

赵玄机跪，膝磕地砖，响得实。他抬头，看莫长春——莫长春立堂侧，不言，只一点头。

林挽月退堂，与顾小满擦肩，低声：「对不起。」

顾小满摇头：「记着就行。」

围观弟子炸开反转，反转成浪：

「林挽月反戈了！」

「赵执事陷害长老？」

「莫长老昨夜止步不入，原来早知局！」

浪声里，脏字退潮，退出一寸清。赵玄机跪姿更直，直得像终于把「还库」之后的路，跪进了地砖。

莫长春出堂，杖点阶石，低声，像钩：

「证完了。夜袭的符，该递出去了。」

风腥，云低，子时像已贴在门上。

有人低声：「慕错人，还能改。」另一人答：「恩对路，不能改。」孙福旧党面色白，像听见「陷害」二字，也听见自己的旧账将被翻。

赵玄机跪姿更直，直得像终于把「还库」之后的路，跪进了地砖。他望莫长春，莫长春立堂侧，不言，只一点头——点头像把「以后还人情」又说了一遍。

苏念慈遣人送药至侧廊，药温，像替顾小满腕上雨痕也担。她低声：「脏字炼净，炉不能熄。」霍镇山在镇妖司骂罢，又骂执法堂：「洗脸洗晚了，洗得还算干净！」

堂上冷，堂外浪——浪退一寸，符将递出。全宗尚不知：挽月反戈，不是毁赵，是留路；留路之后，夜袭已在门外。

魏无涯在檐角，扇合，低笑一声，像看一枚子落对了格，又像看妒火线终于收束一截。风腥，云低，子时像已贴在门上——门上贴符，符贴命，命贴东脉。

堂下弟子屏息，屏息里炸开一层浪——浪不是欢呼，是惊，惊林挽月敢反戈，惊赵执事脸色白，惊莫长老立堂侧不言，只点头，像把「以后还人情」又说了一遍。苏念慈在丹堂廊下听见消息，炉火旺，旺得像替谁把脏字炼净。霍镇山在镇妖司骂：「执法堂终于肯洗脸了！」

林挽月退堂，与顾小满擦肩，低声：「对不起。」顾小满摇头：「记着就行。」赵玄机跪，膝磕地砖，响得实。

「赵师兄，」林挽月不看他，「慕错了人，还能改。恩错了人，改不了。」霍镇山在镇妖司骂：「执法堂终于肯洗脸了！」魏无涯在檐角，扇合，低笑一声，像看一枚子落对了格。

浪声里，脏字退潮，退出一寸清。莫长春出堂，杖点阶石，低声，像钩：「证完了。夜袭的符，该递出去了。」风腥，云低，子时像已贴在门上。

围观弟子在堂外踮脚，踮得像等反转落账——落账不在欢呼，在脏字退潮一寸。有人低声：「莫长老昨夜止步不入，原来早知局！」另一人答：「知局不辩，像把浪当茶晾。」

堂外忽有人嚷：「挽月疯了！」立刻被旁人拽袖：「疯的是造谣那张嘴，不是证字。」林挽月膝仍跪，跪得像把脏字一寸寸坐实；赵玄机咬牙，牙咬得响，响不过堂上那半息——半息拉长，拉长得像把「以后」二字也审了一遍，审完才落锤。

堂外围观渐成两拨，一拨惊，一拨疑。惊者道：「赵执事脸色白，白得像被镜子照见。」疑者道：「照见也好，照见了，夜袭才有人守门。」
