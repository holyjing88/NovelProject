# 第五十五章 凡符祛寒

冬深，霜厚，宗门外驿亭终于有人起身。

韩泥搬柴过侧门，老耿在旁挑水，水洒半勺，声哑：「去。不白拿。拿了，记耿恩，记僧恩，不拜血祠。」

褐袍僧不诵长经，袖里取出黄符，符边毛，毛得像凡物。凡物，才入凡人手。

「寒侵骨，骨寒侵手。」僧声低，「符不入丹房，不越阶。阶在远，寒在近。近，先祛。」

韩泥不接，先问：「为何是我？」

僧指他掌茧裂口：「你鼻下静，手下正，坛边席严。严者，符不辱主。」

老耿哼一声：「僧是我引的。引，不是邪，是冬里少死人。」

韩泥这才接符。符薄，贴掌，掌暖一丝，暖得像经片贴怀——贴完，掌纹里留一线浅烫，烫浅，像凡物认手，不认脸。

「凡符祛寒。」僧道，「燃则一日暖，不燃则贴怀。贴怀，护骨寒，不护心邪。心邪，符自灭。」

韩泥低声：「心记着七笔，不邪。」

僧又嘱：「符凡品，祛寒，防。勿示人，示人，谣又来。」

僧合十，不拜坛，只拜冬：「瓮在眠，人别睡死。睡死，是拜邪。眠，是等。」

僧离亭，脚步远。远，像凡符使命落了地。驿亭空，空得像赠毕——毕了，怀才暖。

回棚，符藏怀，贴身，不燃。燃，烟起，管事查。

赖福嗅他袖，嗅像找由头：「藏啥？」

「寒。」韩泥说，「寒在骨，不在袖。」

管事路过，声冷：「宗门外的事，少议论。议论，像借商线腥。」

赖福哑了。管事又道：「符凡物，别傲。傲，邪。」

刘婆端姜汤，姜薄，热：「怀里有符，别傲。」

韩泥喝尽。姜热，像远村灶火——远火在，还路在。

邻床问：「啥符？」

「凡符。」韩泥说，「祛寒，防。不燃，贴怀。」

「贴怀干啥？」

「贴怀，手不僵。」韩泥说，「手僵，藤刺偏。偏刺，血找错路。」

午后，西栏清粪，风硬，符暖护骨，手不僵。粪锹沉，沉得像搬柴。搬柴手正，清粪手也要正。正，试手才不滚。

铁无言巡栏边，声低：「符在怀，别示。示，谣来。」

韩泥只答：「不示。」

小兽咳，咳得细。他多铺一层干草，草旧，旧能暖。

管事盯梢：「清粪加半刻。半刻偷懒，克扣麸皮。」

韩泥不偷懒。克的是粮，克不走还路。

夜，他对坛低声：「凡符起，仍凡。凡，够活命。活命，才还恩。」

坛腹温一线，不与符抢。符护人，瓮眠等缘——各守一线，不拜邪。

秦霜远廊经过，一句：「心别邪。邪，符灭。」

韩泥怀符暖一丝，心记着七笔，不邪。

丑时，他按席，席严，沿净。净，才接后头血近沿，接瓮醒那一误。

外头更鼓沉，柱上五日条在。五日，试手凝丹。凝丹前，符不燃，沿不血，席盖严。

同批少年碰他袖：「你真得了符？」

「得了。」韩泥说，「凡符，祛寒。不示，不傲。」

「示了咋办？」

「示了，谣来。」韩泥说，「谣来，路易断。」

少年懂「断」，闭嘴。

搬柴经侧门，风更硬。韩泥不凑亭，只瞥一眼。褐袍亭已空，空，像赠毕。

铁无言巡栏边，声低：「符在怀，手别僵。僵，藤刺偏。」

韩泥应声：「不僵。僵，血找错路。」

四更，邻床翻身，袖风扫过他怀。怀暖一丝，符不灭。灭，心邪。心不邪，符自护。

卯时搬柴，风更硬，硬像西栏粪锹。符暖护骨，骨不寒，手不僵。管事验垛，声沉：「冬里沉得住，记功在。」记功在，麸皮不克。

午后，铁无言廊下见他：「符凡品，别傲。傲，邪。」韩泥只答：「不傲。傲，符灭。」

黄昏，坛腹温一线，不与符抢。缘在沿前，不在符烟。

他不燃不示。示了，谣来；谣来，路易断，粥就凉。

他独眼平，手不抖：「符护骨，沿在净。净了，才接后头缘。」

符录起了，仍凡，不越阶。凡，够活命；活命，才还恩。

丑时，他揭怀角透气，符边不燃，只暖一线。暖线贴骨，骨不僵，坛腹却凉回去——像符护的是人，瓮等的仍是缘，各不相欠。

他低声对坛：「凡符在怀，瓮仍眠。眠，不争。争，像拜邪。」

侧门风硬如刀。刀贴骨，符暖护之。护骨不寒，手才不僵，缘才正。符与瓮各守各线，沿在净，席盖严，才接后头试手。

管事嘱：「符在怀，少声张。」韩泥不声张。声张，谣来；谣来，路易断。

西栏第二趟，粪锹沉，符仍暖。暖，不抢瓮温。瓮眠，符护人，各守各的。小兽栏角，小兽眼湿，咳得细。他多铺干草，草旧，旧能暖。暖兽，像冬里少死一人。

刘婆又端半勺姜汤，热：「符在怀，别空肚。空肚，鼻下香飘。」韩泥喝尽：「不空。姜在记里。」

秦霜远廊一句：「心别邪。」韩泥心记着七笔，不答。不答，像把符恩按进第七笔里。

他对坛低声：「凡符录起，仍凡。凡，够活命。命在，才还恩。沿在净，席盖严，符在怀——三样各守一线，才接血近沿，接瓮醒那一误。」

搬柴第三趟，肩勒旧印，印深，沉得住。沉得住，符暖护骨，骨不寒，掌裂不僵。不僵，藤刺才正，缘才正。侧门风过，驿亭柱影斜，斜处已无褐袍——无袍不要紧，黄符已在怀，怀暖一线，像经片贴过又换了一种守。

邻床问符味。味淡，淡得像凡物。凡物，才入凡人手；人手，才贴怀。他不燃符——燃，管事查，像借腥；腥，恩易断。断，粥就凉。

五更前，柱上五日条被风吹得猎猎。猎猎不是催滚，是催手正。手正，符护骨，瓮眠等缘——各守一线，不拜邪。

铁无言第二趟巡栏，不问他怀里有啥，只验鼻：「鼻下静，符才暖。静乱，符自灭。」韩泥应声：「静在记里。」

邻床翻身，袖风扫过他怀。符暖一线，不灭。灭，心邪；心不邪，符自护。

秦霜远廊角又过，只落一句：「心记着，符才护。」韩泥只应一声，声轻，像把符恩摁进第七笔里。

老耿夜过侧门，水洒实，声哑：「僧是我引的。引，不是邪，是还经片那一恩。」韩泥不答，只摸怀符。符边毛，毛得像凡物——凡物，才配丙九末排。

管事夜巡路过，鼻下嗅他袖：「无烟，好。」韩泥不辩，只按席。按实，像把符恩与耿恩、僧恩并记第七笔里。

秦霜远廊最后一句：「符凡品，心别邪。」韩泥不答，答在心里——心里七笔不混，符才护骨。护骨不寒，手才不僵，五日后试手才站得住。

侧门最后一趟，风更硬，硬像冬深那一赠前的最后一试。符在怀，怀暖一线，像经片贴过又换了一种守。守，不燃，不示，不傲。

刘婆第四趟端姜汤，热：「符在怀，别傲。」韩泥喝尽：「不傲。傲，符灭。」

搬柴第四趟，肩勒旧印，印深，沉得住。沉得住，符暖护骨，骨不寒，掌裂不僵。侧门风过，驿亭柱影斜，斜处已无褐袍——无袍不要紧，黄符已在怀。

管事夜巡第二趟，验垛，声沉：「冬里沉得住，记功在。」记功在，麸皮不克。克粮，克不走还路。

坛腹温又一线，像瓮旁听符边毛响。旁听，不应，仍眠——眠里等缘，缘在沿前，不在符烟。

老耿侧门最后一趟，水洒实，声哑：「僧恩记第七笔。记了，符才护骨；护骨，手才不僵。」韩泥只点头，点头像把凡符与经片并放一处。

邻床鼾声起，起不要紧，符暖一线不灭。灭，心邪；心不邪，符自护——护骨不寒，五日后试手才站得住。

侧门最后一趟风硬，硬又像经片贴过那一夜。风止，他按席，沿仍净——净了，符才护骨；护骨，手才不僵。

他独眼平，低声道：「符护人，瓮等人。人等缘，缘在沿前，不在符烟。」他收息，符仍暖，不抖。
