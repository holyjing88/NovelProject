# 第三十九章 克扣

半月，麸皮短三成。

短在秤上，秤在赖福手里。他举秤盘晃：「晦气杂役吃多了，兽栏粪都臭！」

丙九有人嘀咕：「再克，咱们也滚。」

韩泥「嗯」一声，仍清粪，粪铲下稳，稳才少挨骂。心臭，记着，不辩。辩，赖福正等他辩，辩了就扣饭，扣了祸及全丙九。

管事路过，瞥秤：「麸皮按册发，少一粒记名。」

赖福脸一僵，秤手松了半寸。松半寸，也是小胜。

丙九有人低声：「韩泥不闹，咱们跟着沾光。」

韩泥听见了，不答。功在饭，不在舌。舌赢一时，秤输一月。

月初，麸皮还足。月中，秤悄悄浅了。月末，短三成。短成仇苗，苗不拔，先活。

赖福每日晃秤，晃得丙九眼酸：「晦气沾谁谁穷！」

有人顶嘴：「穷是你克出来的！」

赖福举鞭：「再顶，明日没麸皮！」

韩泥低头清粪，不接茬。接茬，全丙九跟着饿。

管事冷声：「吵完了？吵完韩泥仍清粪，你仍发麸皮。少一粒，你记名。」

赖福收鞭，脸青。青，小胜——胜在秤上暂时不敢再浅。兽栏深处，小兽咳，咳声细，扎耳。

他多清半刻，把湿粪换干草。稳了，管事少骂一句，骂少一句，饭就多一口余地。

管事路过，瞥他铲下：「韩泥，兽栏这角，明日仍你。」

「嗯。」韩泥说。

管事又道：「清得细，记着。记着，麸皮不能短你的。」

赖福远处听见，脸一抽，抽完装没事。没事，也是小胜——胜在册上，胜在明日秤旁多一只眼。刘婆端粥来，粥更稀，稀得勺立不住。她眼里一瞪赖福，赖福竟退半步。

「再看粥，我去管事那告。」刘婆嗓沙，盆沿捏得响，「告不成，我把盆扣你头上。」

赖福骂「疯婆」，嘴硬脚却退。退，胜在粥边，胜在丙九尚有一口稀的。

韩泥端碗，碗温，温得稀薄也是热。

刘婆低声：「稀也喝，别硬扛。」

「扛惯。」韩泥说。

「扛惯，也会倒。」刘婆说，「倒了就滚。滚了，谁还叶丫头那碗汤？」

韩泥独眼抬了一下，又垂：「记着。」

刘婆又补一句：「粥稀，心不能稀。心稀了，手就抖。手抖，漏渣，漏渣，滚。」

「不滚。」韩泥说，「滚了，谁记赖福的秤？」

刘婆愣半息，哼：「记着记着，丑脸倒会说话。」
午后，韩泥编筐补单，指茧裂，裂里渗血。

血擦布上。血仍规矩，不主动滴坛。坛不该因奋而醒，醒要等瓮醒。

邻床少年问：「你手裂了还编？」

「编了，有饭。」韩泥说，「饭在，命在。命在，恩才还在。」

「命都没几口粥了，还恩？」

「少三口，记着。记着，以后还。」

邻床少年叹气：「咱们都被克，你就不能闹一回？」

「闹一回，爽一口。」韩泥说，「爽完，滚半年。滚半年，恩断。」

少年不吭了。

管事查单，瞥他掌：「裂了换活。」

「不换。」韩泥说，「单没完，饭没稳。」

管事「哼」一声，走前却道：「明日麸皮按册，我亲自盯秤。」

赖福远处啐：「装硬。」

韩泥不回头。辱记着，不在今日还。还，要等手稳，要等瓮醒。

同批有人低声：「韩泥，咱们要不要一起去管事那？」

「去，赖福记你们名。」韩泥说，「不去，他记我名。名都记，饭先稳。」

「你就忍？」

「忍半月，换明日盯秤。盯秤，胜。」

众人面面相觑，仍各自干活。干，也是小胜。夜，赖福踹门：「韩泥，明日兽栏加倍。做不完，滚。」

韩泥「嗯」一声：「加倍，也做。」

门内，仇苗又深半寸。深半寸，不拔，先活。

坛腹微温，应「忍」。忍字丑，丑翁专精。

他低声道：「克扣记着。刘婆粥记着。嗅渣香记着。必还记着。」

半月，饭少三口，三口记着——还自己的命。命还清了，才还得起别人的恩。

刘婆门外咳：「还喝不喝？」

韩泥「嗯」一声。碗递进来，稀，热。热粥稀，也胜冷秤。

他喝尽。赖福门外骂：「疯婆子多事！」

刘婆回骂：「多事你也滚远点！秤在你手，良心在谁肚？」

韩泥摸坛，坛温应粥温。温一线，不烫人，烫仇。

他低声道：「半月短三口，三口记着。记着，熬到还。」

邻床少年梦呓：「饭……」

韩泥睁眼，替少年盖破袄：「排着。管事盯秤了。」

少年哼唧睡了。睡，也是小胜——今晚丙九还有人睡得着。

管事巡夜，敲棚柱，声沉：「赖福，麸皮按册。再克，记名。」

赖福应了一声，声虚。虚，也是小胜——胜在明日还能喝一口稀粥，胜在秤上少短半寸。

同批有人低声：「韩泥，克扣半月，你胃还暖？」

「暖。」韩泥说，「暖在记里。记着，才还得上。」

「还得上什么？」

「还得上叶丫头的汤，还得上刘婆的粥，还得上泥岗六笔恩，还得上宗门第七笔。」

那人闭嘴，翻身睡了。

韩泥合眼，三口饭少在心里刻着。刻完，再睁眼摸坛沿，沿温应粥温，温得浅，浅也真。

他低声道：「赖福的秤，管事的册，刘婆的粥，叶丫头的汤——分列，不混。混了，还错。」

外头风息，风息了，稀粥的热还在肚里。热在，人活；人活，明日盯秤才有戏。

编筐补单，指渗血，血记着，瓮醒才推。推对了，克扣那笔，一笔一笔还。

他低声道：「半月短三口，三口记着。记着，熬到瓮醒，熬到还。还不了，也记全仇。」

邻床少年问：「你真能还七笔？」

「还不了，也记。」韩泥说，「记全了，才活得久。活得久，才还得上。」

少年闭嘴，睡了。

他低声道：「克扣记着，刘婆粥记着，嗅渣香记着，必还记着。记着，熬到瓮醒。」

管事门外敲柱：「明日丙九转渣场。韩泥，酉时到。误者，记过。」

渣场——鼻下那条路，明日要走实。

坛壁一线水汽，应粥温，不应骂。骂记着，不泄；泄，害全丙九。

半月短三口，三口刻心。忍到秤实，忍到备检过——过了，才近瓮醒，才近还。还，先还热的；热的，先还叶丫头汤，汤烫手，烫手是规矩。

半月短三口，三口刻心。忍到秤实，忍到备检过——过了，才近瓮醒，才近还，才近叶汤烫手。

半月短三口，刻心不是怨，是存锤。锤在备检，备检过了，饭才实。

他低声道：「手稳，恩不断；恩不断，汤就不凉。」

坛沿他仍擦。擦时指节白，白像证：辱在册，恩在胸，分列不混。
