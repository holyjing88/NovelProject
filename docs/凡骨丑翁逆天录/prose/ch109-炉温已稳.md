# 第一百零九章 炉温已稳

雨后。

兽潮散，药山界解了夜禁。韩泥回废窑——不为看景，为稳炉。窑口石壁还潮着，潮里混一点药焦味，味正，他心里就松半寸。松半寸不是松劲，是敢把火再稳住一寸。

稳炉这一章，不在名，在温。温一寸，不多不少，像呼吸。窑口红退到掌纹能控。他不再急。急过那阵子，急在潮至；潮过后，要稳。

铁无言送来一包灰渣，声低：「清瘴剩的料，别浪费。浪费，像逃。」

韩泥应声：「不逃。」

不逃，就炼余韵。余韵不是丹，是手。手熟了，路才正。

第一炉走培元路，无丹，只有灰里一点白。白像未成——未成不要紧，未成正是余韵。第二炉走清瘴路，丸不成，成灰香。灰香正，正才是这一章的稳。

他记炉壁烫半寸，烫像坛腹应温。应温，不应喜。喜了手飘，手飘散就不正。赖福克扣又复了半分，盯梢却少了半息。少盯他不谢。谢轻，只记心里：盯少了，窑可多待半刻。半刻够稳，稳里鼻下一嗅，嗅出投序将变——变向培元散。

培元散是散，不是丹。丹在赠叶那夜；散在下一程，自用，是丹道起点。起点不在嘴，在炉。炉温稳了，才像首丹余韵留在掌里。

刘婆端粥来，粥稀：「兽潮过了，手还稳？」

「稳。」韩泥说，「饭稳，恩不断。」

叶青禾托沈枯芽递话：「别再来门缝。」

他接话，不辩。辩费舌，舌要留来稳炉。心里明白：不来是因清，清是因匿；匿着，恩还在脉里温。

赖福堵在坡道上，鼻动：「又上山？成什么？」

「倒渣。」韩泥声平，「手稳。」

「稳你个头！兽潮过了，还装？」

「装的是火。」韩泥抬眼，「火听手，不听你。」

赖福噎住，骂半句走开。没追。没追也好——散还没炼，追也追不出。

夜，席下按坛，低声道：「炉温稳了。余韵在掌。」

坛腹温贴掌，像应「稳」字。稳是钩，钩向培元散。邻床问：「你真要炼散？」

「炼。」韩泥说，「炼了，才还得了下一笔。」

下一笔在省亲前，前头先得把散炼成。

丑时静半刻。香落齐，掌纹温而匀。匀像六层实，实而不飘。经页边他添「散」字小划，划完即抹，抹得像无字。无字才匿得住下一程首炼。

铁无言又探头：「料够不够？」

「够试两炉。」韩泥道，「试完才投真。」

「你倒沉得住。」

「沉不住，炉先炸。」韩泥说，「炸了，并肩白并。」

白织月远远一瞥，不近：「废窑烟少，少出事。」

「少出事最好。」韩泥应一声，继续控火。

第三日炉壁仍烫半寸。他控着呼吸，像跟人说话一样跟火说话：慢一点，匀一点。火听了，白气也不乱。同棚杂役笑他：「兽潮英雄回窑煮灰？」

「煮灰。」韩泥说，「灰煮正了，人也正。」

笑的人噎住，不再闹。闹不出滋味——他手里那包剩料，比闲话实。

管事弟子路过窑口，看一眼就走：「别烧穿瓦。烧穿，你赔。」

「不穿。」韩泥应。

赔不起瓦，更赔不起手。手要留给明日丑时那一炉真火。

第四日雨又细了一阵。他揭开窑口盖布，闻潮气，闻炉灰。潮气淡了，灰香还正。正得让他把投序再背一遍。背完，指腹在褐渣上搓三下——搓熟了，明日才不手生。

赖福偏又堵来：「打听清楚了，你夜里上窑？」

「上。」韩泥说，「上窑倒渣。」

「倒一夜？」

「一夜够稳。」韩泥道，「稳完你再来闻。」

赖福冷笑：「闻出邪味，我揭你。」

「揭。」韩泥独眼平，「揭错了，你自己吃麸皮。」

夜里铁无言送来两滴灵泉：「最后一点。散要成，就靠它压色。」

「记着。」韩泥接，「不浪费。」

「成了告诉我声。」

「成了再说正。」韩泥道，「不正，不吭。」

更鼓尽，废窑温退，退像收敛。收敛不为散场，为明天丑时开炉。他低声道：「稳了。下一程，培元散。」

培元散，丹道起点；起点在丑手，不在名。名还远，掌先在。掌在，炉就不欺人。

坡下石阶还湿，他数息下坡，心里只留三样：温一寸、手不飘、散字匿着。匿着等到丑时星晦，晦才好走窑路。柱上旧条被雨打歪，他没扶。扶条不如稳火。火稳了，条子爱歪爱正，都不碍他袖里那一程。

回棚前，刘婆又塞半块饼：「夜里别空肚。空肚，火就飘。」

「吃。」韩泥应。

沈枯芽在暗处问：「韩爷爷，明日真炼？」

「炼。」韩泥说，「炼完，路才像路。」

「叶姐呢？」

「叶姐歇着。」韩泥道，「我不去吵。吵了恩就吵。」

芽儿缩回头。他进棚，把坛按实，把料码齐。码齐像把明日的火先排好队。队排好了，人才能睡。睡前他再听一声窑风——风静，像等他。

稳炉最忌人围观。废窑本来偏，偏得连野猫都少走；可人一听说「兽潮英雄回窑」，闲嘴就爱往山上飘。韩泥听见门外脚步，立刻把盖布放下，火压半寸，压成倒渣该有的闷。闷味出来，闲嘴就骂一句「晦气」走开。走开正好——正好把火再抬回那一寸温。

他自言自语似的跟炉说话：「别抢。抢了白。白了，散就邪。」

炉当然不答，可白气听懂了似的，聚得更听话。听话的火，才像把昨日清瘴那口正味接着往下传。传到掌纹里，掌纹就不贪炸响，只贪稳。

第三日午后，铁无言带了半筐粗糠来：「垫窑底。垫了，温不跳。」

「谢。」韩泥洒糠，洒得很匀。

「又谢。」铁无言皱眉，「谢多了像生分。生分了，并肩白并。」

韩泥闭嘴，只点头。点头比谢实。实完之后，两人并排蹲着看炉口，看了一炷香，谁也没再提兽潮。不提，不是忘，是把仗收进手里——手里有余韵，才配谈散。

夜里他回棚，听见邻床嘀咕：「听说韩泥在废窑煮神仙。」

另一个嗤：「煮神仙？煮屎还差不多。赖福都说了，倒渣。」

韩泥躺着不动，不动就是承认「倒渣」。承认得好——好让明日丑时那包真货，少遇一双眼睛。眼睛少了，炉才敢红；炉红了，灵泉那两滴才敢滴。滴早了浪费，滴晚了色不正；不正，铁无言闻得出，自己的鼻子更骗不过自己。

丑时前一个时辰，他把料袋按序列排成一排：青芯、褐渣、灵泉。排完又拆开重排一遍，像编筐最后一道勒藤——勒不匀，筐散；投不匀，散败。他勒得很慢，慢到汗都凉了，才听见远处更鼓要转。鼓声一近，他披衣出门。

出门前对坛只留半句：「守着。我去取起点。」

坛腹温一线，像回他：取来，别浪。

第四日傍晚他特意绕远路回棚，避开赖福常蹲的坡口。远路泥深，深得鞋子灌水，他也不改道——改道近了，近处有眼。眼里一多，炉温再稳也白稳。

路遇管事弟子，对方瞥他袖：「山上又待多久？」

「倒渣半日。」韩泥说。

「半日别把瓦熏黑。熏黑，照样赔。」

「不熏。」韩泥应完就走。走时袖里料袋轻轻一磕，磕得他心更定：料还在，序还在，火候还在指上。指上有数，丑时就不慌。

回棚熄灯前，他又摸一遍投序，摸到灵泉那两滴停住：「留给真火。试炉不许碰。」对自己说的规矩，比管事条令还硬。

丑时将至，他披衣起身，摸了摸瓮沿温。温在，就像答应：明日首炼，把「还」字从炉里再拽出一寸。拽出了，散才不当玩笑，当起点。
