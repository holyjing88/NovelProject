# 第八章 馒头

陈姑棉袄第三日，张麻婆骂街骂到晦气翁门口，声却落软：

「韩泥！筐编得歪！歪了卖谁！」

韩泥「嗯」一声，手里藤条没停。张麻婆骂是习惯，习惯里藏一点别的东西——真嫌，转身走；骂，是还肯当邻。「张麻婆。」他开门。她叉腰，脸皱，皱得像晒干的橘皮：「开门干啥，晦气散出来？」

「你骂到门口，不开门，不像话。」

张麻婆噎半息，从怀里掏出布包，甩他怀里：「吃！别饿死我门口，晦气！」

布包烫，实，白面馒头一个，硬，像这老太的嘴。热气扑脸，扑得像一小团白日光。韩泥不辩，接稳。「我不是可怜你，是烦你！」她唾沫星子飞，「你饿死，抬棺还得找你，累我！」

「累你就别送。」韩泥说。张麻婆眼一瞪：「送都送了，还嘴硬！吃热的，别噎死！」

吐唾沫，三步远，骂着走了：「王家鸡又拉我门口，缺德哟……」

骂声远，远里还有一句，轻，像怕人听见：「筐编结实点，别砸我名声。」
韩泥掰馒，慢慢吃。面实，噎人，噎得清醒。麦香粗，粗得像这村子的日子——不细，真。馒皮硬，内里软，软得像张麻婆骂完还肯塞过来的手。吃一半，留一半，布包好，搁坛边——穷人家，一顿分两顿。剩半只，热气散得慢，散得像恩还在。心里又添一圈——张麻婆这一个馒，歪，像馒角，实得真。叶汤、芽粥、陈袄、张馒。四恩齐。他低声道：「汤还是热的。」

门外赵绣春过，瞥见韩泥啃馒，鼻哼：「还有人喂晦气翁，谣传不尽。」

韩泥抬头，独眼平，不接话，只抬了抬手里剩的半馒，像晃一下证据：吃着的，不是鬼。赵绣春唇动：「你……」

「吃馒。」韩泥说，「嚼得动，人就没死。」

赵绣春脸一红，红得像被噎住，没再尖，走了——骂刃碰着闷石头，钝了。门外风过，带起一点馒香，香淡，淡得像反驳。反驳不必响，嚼得动，就是活。午后，韩泥担两筐去集尾。筐结实，张麻婆骂出来的结实。藤条勒肩，勒出印，印浅，像手艺的证。集尾风硬，吹得棉袄领子往脖子里钻，钻得他缩了缩，缩完又直腰——直腰像卖筐，也像活。半个时辰，卖出十二文，比昨日多三文——手艺是穷翁的刀，刀不快，却稳。三文不多，够盐，够米，够多活一日。买主掂筐，笑：「晦气翁筐，倒实在。」

韩泥不接「晦气」二字，只道：「结实。」

「下回还来。」买主又多摸一文，「多一文，不亏。」

韩泥点头：「下回还结实。」

另一买主凑近，低声：「赖管事要收你税，你小心。」

韩泥应声：「税重，筐也重。」

他买粗盐一小把，米半升，盐粒粗，米色黄，黄得实在。回村。村道童谣丫念：「晦气翁，吃馒翁……」

念到一半，张麻婆帚子飞来，唬得娃跑：「再念，撕你嘴！」

韩泥路过，不笑，心里松一寸——有人替他说话，骂着护，也是恩，恩他记着，日后还。张麻婆扫他一眼，哼：「吃馒别噎着，噎着还得我抬棺！」

韩泥只答，算是应。「麻婆。」他停半息，「筐我编结实了。」

张麻婆一愣，哼更大：「结实就卖！别饿死我门口！」

娃躲在墙后探头，小声：「麻婆，他真吃馒？」

张麻婆帚子一举：「吃！活人！再念晦气，撕嘴！」

娃一缩，缩得像信了——晦气翁嚼得动，就不是鬼。夜里，米缸有米，灶生火，粥香薄，香得实在。粥里米粒翻，翻得像四口粮在锅里碰。碰不着，心碰着了。韩泥喝粥，想：四恩齐了，像四口粮，粮少，够活。活够了，恩就能一点一点还。他按了按心口，心口硌，硌得稳。「记着，都记着。」

集尾收摊时，有人问他筐价，他开价低，低得稳，买主掂了掂，喜。喜里没刺，刺少一寸，路宽一寸。回村路上，月薄，照见门墩空，空得静。静里他按心口，四恩在，像四口粮齐了。齐了，心稳一截。馒剩半只，布包还潮，潮得像恩还没凉。凉不得，凉了，人就成嚼舌根的嘴。他心里四笔恩，叶汤深，芽粥细，陈袄短，张馒歪。歪得真，真得像这四口人粮不齐，却齐了。齐了，心里像夯实一层土，土薄，够站。门外有人咳，咳得像陈姑。韩泥不开门，只低声：「四恩齐了。记着，都记着。」

「汤还是热的。」他又道，「馒也是实的。」

灶火渐弱，弱得像命还薄，薄也得撑。撑住了，明日还去集尾，还去井台，还去把筐编结实。他摸坛边剩馒，馒凉，凉得慢，慢得像恩还搁着。搁着，不丢。「四恩齐了。」他低声，「够活。够还。」
次日，张麻婆又骂到门口，骂得比昨日响，却从怀里摸出第二个馒，甩他怀里：「吃！昨剩半只，今再给你整的！」

韩泥接稳：「麻婆，够。」

「够什么够！」她瞪眼，「你瘦，风一吹就倒。倒了，抬棺还得我！」

「不倒。」韩泥说，「筐结实，人结实。」

张麻婆唾沫星子飞：「结实就卖！别饿着我！」

骂着走，走远了还嚷：「王家鸡又拉我门口……」

韩泥掰馒，慢慢吃。面实，实得像张麻婆嘴硬心硬。心里那圈恩，又实一分。集尾，有人压价：「晦气翁筐，便宜三文。」

韩泥摇头：「价不变。筐结实，不贱卖。」

买主掂了掂，终是按原价收：「你这人，倒硬。」

「硬不硬，筐硬。」韩泥说。回村，童谣丫又念半句，张麻婆帚子一飞，娃跑。韩泥路过，心里松一寸——有人骂着护，也是恩。赵绣春瞥他怀里馒，嗤：「晦气翁也配吃白面？」

韩泥掰馒，不慢：「面无晦气。晦气在嘴。」

张麻婆远处骂：「她再嚷，明儿馒给她狗吃！」

馒实掌烫，四形齐了。他分半给沈豁口，不喊恩，只记——记到第十回自检，旁人能答「为何还叶」：因为烫，因为真，因为秤还在。公审不远，锤未落，烫先不能凉。

卖筐十二文落袋，赵绣春嘴钝半寸。钝半寸，也是小胜——胜在钱稳，不在嘴赢；嘴赢不了，筐能赢。

张麻婆骂完，门缝又塞半块饼。饼硬，硬得像邻恩——邻恩不甜，真。坛沿温一线，应馒不应谣——应着，四形齐了，还路不断。明日还去集尾，还去井台；井是村的，谣烫不了心。

夜里，韩泥把馒头掰实，实得像张麻婆骂声落下去后仍留在胃里的暖。暖在，手还要编筐；筐卖一文是一文，一文也是还路的肉。他摸坛沿，沿温不退，像瓮听见了「馒恩」二字。听见，不退温，只记。记全了，公审前才站得住「四恩齐」——齐，才配答「为何还叶」：因为烫，因为真，因为秤还在。
