# 第一百二十一章 再炼散

复测前一月，韩泥把门关上了。柴挡严严实实，他要再炼一炉培元散。

风从缝里挤进来，潮得鼻头发凉。他蹲炉边清灰，铲子一下一下，渣细到指腹都能碾开。铁无言昨夜临走只丢下一句：「别贪。贪，手就飘。」他应了：「不贪。沿净。」沿净了，坛才肯听；坛肯听，火才肯听话。

邻床那少年隔着柴挡喊：「韩叔，你真能出丹？」

「能。」韩泥嗓子哑，「不是今日。今日只是把火路记牢。记牢了，粒才肯来。」

窑门关着，外头闻不见大动静，活像凡人在烤薯。管事来巡时，闻见缝里淡香，也不敲门，只往册上记一笔：「培元散试手，允。」话还没落稳，赖福先踹上柴挡：「丑杂役闭门，偷炼！」

管事弟子皱眉：「嗅得正，不是邪火。赖管事再闹，记过。」

赖福脸青了半天，骂着走远。韩泥在门内没动，只把「正」字按进掌茧——正不在嘴上嚷，在粒上闻；闻得正，赖福踹一百脚也踹不开簿。

第一炉按焚火残诀走七七四十九息。息稳了，香起。他鼻下一片静，静得像首炼那日，却更沉——那时是试手，这回是备战。香正的时候，渣灰里会留一点辛，辛得干净，像清过瘴气的余味。他把散收进陶罐，封泥，封得像腌菜坛。坛相凡，凡相才蔽眼；蔽住了，外头骂闲话也不过是风。

白织月立在窑外，袖风冷冷刮过：「闭门炼，倒像做贼。」

韩泥不开辩：「贼偷，我还。还恩炼，丹才正。」

「做贼也做不出骨。」她笑得薄，「试台上一洒，笑话是你的。」

「骨丑，散正。」韩泥声平，「正不正，试台见。」

袖风走得快，快得像怕那句「正」字跟脚。他也不管，开第二炉、第三炉。炉炉都稳，稳到腹里那口气忽然通开——炼气十一层就这么破了。破得静，静得像不想叫任何人知道。一叫，赖福就会拿「偷境」做文章；做文章，册上就有事。他摸了摸怀里玉牌，牌凉：「三年约，记着。十一层，只给十三铺路。」

铺路不要紧声张，一步一步就行。第一笔还了芽儿半块灵石，刘婆这笔仍欠着，叶青禾那碗汤还在更前头。前头的先记着，手先稳着——稳着，才配上台。

刘婆送到柴门外，碗里粥稀却热：「闭门炼？」

「炼。」韩泥开门缝接碗，「炼备复测。」

「备就别空肚。」刘婆哼一声，「空肚手抖。手抖，粒洒。」

他把粥喝尽，谢轻，恩重。恩在粥里，也在后头那衣袖钱里——衣袖钱要在夺魁后交到她手里，才算数。沈枯芽远远站柱角，怯得不敢近，只抬眼看他。韩泥点一下头，点完对方才退，退得像信：信他还，信他炼，信他不拿魁去吓人。

铁无言又送来外门边角渣，渣里夹着细灵材丝：「两炉够不够？」

「够。」韩泥掂了掂，「七粒在手，手就不飘。」

「七粒别示人。」铁无言低声，「示了，风声就说你盗宗门材。」

「盗不得。」韩泥道，「渣里淘了五年。五年淘出来的正，不是运气。」

午后第三炉成，气走圆满，十一层更实。实了，他才敢再开两炉，粒数爬到九。九粒够试台，够把末席那条路垫实——垫实了，后面还刘婆才不虚。丑时里他练鼻下静，粟壳香混着药香，香正，更要紧的是静得住。静住了，十一层才进心里，不进舌头。进舌头，人就来查；进心里，人查也查不穿。

赖福骂得老远：「闭门也闭不出灵根！」

韩泥对炉低声：「灵根末，散正。正，就够上台。」

够上台，才谈得上接钱戾衡那道影。影还不在眼前，玉牌先凉了。手热在坛上，坛应着「炼」字——炼到十二层、炼到帖落地，炼到那日能站稳，不滚。

闭门第六日，管事再嗅缝香，声仍平：「正，别示。」韩泥应一声。他封罐封严，七粒主用，两粒备用，都不给旁人闻。示人一日，赖福就多一日借口；借口多了，除名就近。除名近了，玉牌白凉，叶汤凉，刘婆的新衣也还不上。还不上，先把魁名放在后头等；后头等，眼前就得把门关死。

第七日开窑，七粒仍正。他把罐按进怀里，怀沉得像把债序也按住了。

他把泥封摸了一圈，圈干了才放心。干了像账清；账清了，外头骂闲话也砸不烂罐。罐在怀，怀才沉得住。外头风声又起，起得像试台近了一寸。近一寸，他只多稳一寸手——手稳，粒就不洒；粒不洒，才有资格去听榜、去挨嘲、去接那道金线边的影。

邻床少年又贴柴挡问：「你里头到底在干啥？香一天比一天正。」

「干活人该干的。」韩泥说，「香正，就别稀奇。稀奇的是后头有人要你洒。」

「洒啥？」

「洒手。」韩泥顿了顿，「手洒了，名就没了。名没了，粥也喝不稳。」

少年愣住，不吱声了。韩泥听见外头脚步远去，才把泥封又按一遍。泥干了，像锁；锁住了，心里那口气才肯歇。歇不是躺平，是把刃藏好——藏好了，白织月再嘲、钱戾衡再压，才不至于一碰就断。

闭门第五日午后，他特意出窑半刻挑水。水桶沉，沉得手腕发麻，麻完反而更醒。醒着好，醒着才记得：十一层不是歇脚的亭，是过道。过道里还得再忍赖福眼神，再忍白织月袖风，再把粥喝完。水挑回来，他先漱口再近炉——口净，闻香才准；香准，投序才不歪。

管事弟子路过，撇一眼柴挡：「还炼？」

「还炼。」韩泥说，「炼完才上坡。」

「上坡手别抖。」管事丢下半句就走。半句比骂有用。韩泥把这句话叠进十一层气里：抖不得，抖了泥封也跟着裂。

傍晚铁无言又堵在门缝外，递进一小包干姜：「刘婆让给你的。说空肚手飘。」

「知晓。」韩泥接过，「谢她，也谢你。」

「谢我做什么。」铁无言哼，「我只看手。手在，渣场账才好做。」

他嚼了半片干姜，辛冲鼻，鼻下更静。静里忽然想起泥岗村那碗叶汤——汤烫，谣也烫，他那时只会认恩。如今认正，正是把恩往后还的底子。底子薄，就用七日窑火补；补厚了，才配说「还」。

夜里他坐到丑时，独眼望着坛沿那点微温。温不亮，却在。在，就像说路还通着。通着，就别急着喊赢；赢要在试台上喊，要在刘婆接过布包那天喊。今天只喊一句给自己听：「十一层。七粒。试台，见。」

雨停后他摸黑清了一回炉灰。灰潮，潮得粘指，他却不怕脏——脏在手上，比脏在名上好洗。名一脏，簿上就有刺；刺多了，十一层再实也挡不住除名二字。

邻床睡梦里哼：「韩叔……丹成了没……」

「未熟。」韩泥低应，「熟了也不叫。叫了，香飘。」

少年翻个身又睡。他坐回坛边，把玉牌贴胸。牌凉，凉得像有人在远处盯着他备战。盯就盯，盯不走七粒。七粒在，复测才有底；有底了，嘲来影来，他都能先站住一脚。

见字落地，窑外忽然落了阵小雨。

同棚有人隔门嘀咕：「闭门七天，真炼还是装样子？」他不应。应了像解释；解释费舌头，舌头要留给闻香。雨打柴挡，像有人轻敲门，敲得他说不出推辞。推辞不了，明日就得上坡听榜——榜上若有他名，嘲就跟着来；嘲来了，也不能还嘴，还嘴不如把散护紧。护紧了，才接得住后面那一锤子。

雨密一阵，他听见坡下似有人贴条。条声纸干，干得像规矩又落一层。规矩落了，玩笑就少；玩笑少了，嘲反而尖。尖嘲他不怕，怕的是手先软。手软了，锤子未落，人先跪。跪不得——跪了，三年约就变成别人的笑料。他把罐再按进怀窝，按到肋骨生疼。疼着记：正了再出门，稳了再挨嘲，挨完嘲再接影。影若金线边亮，他也得站直。站直了，才叫备战不是关狗。
