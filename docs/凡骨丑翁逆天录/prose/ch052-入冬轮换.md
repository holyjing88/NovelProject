# 第五十二章 入冬轮换

入冬第三日，丙九轮换兽栏。韩泥分到西栏——栏风硬，粪腥厚，厚得像冬压下来，压得肩沉，沉得住，像末排惯了。管事敲锣：「冬里清粪加半刻。半刻偷懒，克扣麸皮。」

赖福啐地，啐得虚：「晦气专分西栏，克全栏运。」

管事冷眼：「再嗓，西栏你一人清。」

赖福哑了。哑，韩泥肩上的筐却实——实，像七笔压着。清粪时，小兽咳，咳得细。细不要紧，要紧是手稳——手稳，才护得住小兽，护兽，像兽栏那日，像恩还在。韩泥多留半刻，多留，不邀功，只记。记在心里，刘婆粥那笔便又实一分。粪锹沉，沉得像搬柴。搬柴手稳，清粪手也要稳——稳，试手才不滚。西栏角风硬，硬进骨缝。骨缝寒，不要紧，心记着七笔，便不易飘。小兽眼湿，湿像怕。怕不要紧，他多铺一层干草，草旧，旧能暖。西栏最冷处，有只幼兽蜷在干草下，鼻息浅，浅得像要断。韩泥蹲下，掌心贴它额，额凉，凉才要暖。他多铺两层草，草里塞半块省下的麸皮边角——边角少，少也暖。幼兽舔他指背，舔得像谢。谢轻，他记在心里，不出口。管事远处瞧一眼，没骂，只当没看见——没看见，是默许。同批少年偷懒，偷懒，管事记名。记名，像秤上空一格。韩泥不偷懒。偷懒，麸皮克；克的是粮，克不走手稳。清粪毕，他洗手，水冷，指缝腥洗不尽。洗的是袖，不是心——心记着七笔，袖腥，避；胸热，记。管事远处瞧一眼幼兽那堆草，没骂，只当没看见——没看见，是默许；默许，像恩。---

午后，搬柴过宗门外，驿亭褐袍人仍在，仍在，像等冬寒够深。商队余尘已尽，尽后，只剩五脉旗号在风里猎猎——猎猎不是谣，是界。界在门内门外，门内末排，门外商线。佛门灰旗远了，密宗赭还在亭里——还在，像经片挪到宗门外驿亭。韩泥不凑亭，只瞥一眼。一眼，褐袍人袖里似有一张黄纸，纸边毛，毛得像凡物——凡物才好藏锋。凡物，才入凡人手。人手糙，糙得像编筐的掌茧。老耿挑水路过，水洒稳，声哑：「那人等冬。冬深，才赠。赠，不白受。」

韩泥「嗯」一声：「白不受。受，记恩。」

耿不再言，脚步远。远，像密宗线又近半分。第二趟搬柴，风更硬。硬，吹得侧门霜白。霜白，像辨香那日廊下——廊下过，路还在。褐袍人未起身，袖里黄纸像多了一丝轮廓。轮廓丑，真。韩泥绕远三步，像界——界内末排，界外赠缘。老耿挑水第二趟，水洒稳，声哑：「冬深才赠。你席盖严，沿净，才接得住。」韩泥「嗯」一声，不追问。问急，像邪；等，像守瓮人。韩泥不凑。凑，像贪符。贪，恩易断。回棚，坛腹时温时凉。温时，像瓮听柴声；凉时，像瓮眠。眠不要紧，要紧是沿上无血——无血，才守规矩。邻床抱怨：「坛占角，晦气。」

韩泥不辩，只按席角。按实，像编筐扎掌那日——掌茧再裂，裂在冬，不在沿。他低声道：「冬来，栏腥，柴沉。沉得住，才验得过试手。」

邻床问：「试手验啥？」

「验手。」韩泥说，「手抖，丹歪，滚。丹歪不要紧，手不能歪。」

邻床不懂，哼一声，翻身。翻身，鼾声起。起不要紧，坛角静。刘婆端粥来，稀，热：「西栏腥，别空肚。」

「不空。」韩泥喝尽，「嗯」一声。热在，像第七笔又实。赖福远处骂：「晦气清粪，兽也晦气。」

管事路过，声冷：「再嗓，西栏你一人清到年尾。哑，韩泥肩上的粪锹却捆得更实。黄昏，铁无言巡栏边，灯远，声近：「兽栏腥，洗尽再回棚。」

韩泥「嗯」一声：「洗尽。袖腥，避；胸热，记。」

铁无言停半息，又道：「试手前两日，手别僵。僵，藤条不听。」

韩泥独眼平：「手记着，不僵。」

灯远，像门缝又开一线——线窄，够走。夜，韩泥对坛，声低：

「冬第三日，栏换了，商线还在。在，不急。急，像借腥。」

坛腹微温一线，又凉回去。凉回去，像呼吸——吸温，吐凉，不急醒。他是记心翁，恩仇记在心里。心里七笔，分列，不混。外头风硬，风里无栖香全句。全句散了，散成兽栏粪锹声，散成末排鼾声。鼾声里，他摸木牌，牌边磨毛。毛，像心口七笔——笔笔磨过，磨出实。丑时，坛腹又温一线，温短，像瓮听西栏咳声。咳声细，不应人，应冬。他低声道：「席盖严，沿不碰。碰，不是今日。今日，只清栏，只搬柴，只记恩。」

外头更鼓二响，响沉。沉里，试手又近一日。近不要紧，手要稳。他低声道：「冬第三日，栏腥，柴沉，商线在。线在，心不乱。」坛不应字，只温，温得像「嗯」。丑时将尽，坛腹又凉一线，凉回去，像瓮仍眠。眠不要紧，要紧是沿上无血——无血，才守规矩，才接得住后头凡符、血近沿、瓮醒。韩泥「嗯」一声，不辩。辩轻，恩重。同批少年碰他袖：「西栏腥，你咋不躲？」「躲，手飘。」韩泥说，「手飘，试手滚。滚了，恩断。」少年懂「断」，闭嘴。他摸木牌，牌边磨毛，毛得像心口七笔——笔笔磨过，磨出实。实了，试手才站得住。窄路也是路，路通向还，还叶汤，还振东药，还陈姑袄，还刘婆粥。七笔在，手就稳，稳了，恩就不断。试手又近一日，近，手要稳。坛不应字，只温，温得像「嗯」。恩不断，汤就不凉。轮换稳，瓮醒。冬栏腥，手不飘。轮换夜，西栏幼兽咳一声，他多铺草。草暖，兽暖，像小暖还给丙九末排。西栏最冷，幼兽蜷草下，鼻息浅。韩泥多铺干草，草旧，旧能暖——暖兽，像恩还在。韩泥多铺干草，草旧，旧能暖。管事远处瞧一眼，没骂——没骂，像默许；默许，像恩还在。清粪毕，他洗手。水冷，指缝腥洗不尽，腥不要紧，洗的是袖，不是心。搬柴经侧门，风更硬，吹得霜白。他不凑亭，只瞥一眼，褐袍人仍候冬。同批少年碰袖：「西栏腥，你咋不躲？」秦霜远廊一句：「心别邪。凉回去，像瓮仍眠。眠不要紧，要紧是沿上无血。轮换稳，恩不断，汤就不凉。轮换夜，西栏幼兽咳一声，咳细，他多铺草。草暖，兽暖，像小暖还给丙九。管事记名册：「韩泥，清栏稳。」稳字落簿，簿在，路实一寸。温短，像听商队铃远，铃远，界还在。温短，像听商队铃远；铃远，界还在，缘还在等冬深。西栏小兽夜里咳，咳得细。韩泥多铺草，草旧，旧能暖，暖像刘婆粥。管事巡栏，灯晃兽眼，眼湿，湿像怕冬。怕不要紧，手稳，兽才少死。死多了，谣又多——谣借腥，他不沾袖，不沾，像廊下讲规矩那日教的。凉回去，像瓮呼吸；呼吸不急，醒才正。」

韩泥蹲下，掌心贴它额，额凉，凉不要紧，凉才要暖。他多铺两层草，草里塞半块他省下的麸皮边角——边角少，少也暖。管事远处瞧一眼，没骂，只当没看见——没看见，是默许；默许，像恩。清粪毕，他洗手，水冷，指缝腥洗不尽，腥不要紧，洗的是袖，不是心。心记着七笔，袖腥，避；胸热，记——记，才不借商线腥。搬柴第二趟，褐袍人仍坐驿亭，袖里黄纸角露一线。韩泥绕远三步，三步，像界——界内末排，界外赠缘。老耿第二趟挑水，水洒稳，声哑：「冬深才赠。回棚，邻床抱怨坛占角，抱怨不要紧，坛在，命在。他按席角，席贴坛，坛腹时温时凉——温时像听，凉时像眠。刘婆粥热，热像第七笔；赖福骂，骂像仇苗。管事压赖福，压，像秤上空一格；一格空，韩泥路实一寸。铁无言巡栏，嘱洗袖，嘱手别僵——僵，藤条不听；不听，试手抖。**状态**：大境·凡人 · 资质·丑骨末席（伪灵根） · 鸿蒙九劫瓮·眠（瓮温加剧） · 宗门·丙九杂役 · 漏舍凡舍
