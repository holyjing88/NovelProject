# 第二十三章 审前

先还叶次日，里正敲锣。

锣声哑，哑得像旱。哑也得听——听清了，是村议：「明日巳时，祠堂公审叶家牵扯，全村见证，缺席连坐。」

「连坐」二字又出，出得像枷。枷不在韩泥颈上，在敢对他好的人的颈上。韩泥在门内，手指停在藤上。停一息，又绕。绕得像绕命。命要过公审这一关。关不过，叶家关铺；关过了，谣也还在。还在，也得走。走路不辩嘴，辩心。

祠堂前，赵绣春已摆开阵势，妇人围半圈，像看戏台。戏台不用搭，泥岗村从来不缺戏，缺的是真话。

「明日，叶青禾自己说清！」她嗓子尖，尖得实，「送汤送姜补袄，是不是勾引绝户？是不是引血祠旱？是不是害全村榜末？」

周嫂帮腔，棒槌抱在怀里，像随时能砸：「里正说了，不说清，迁铺！迁了铺，谁治病？治不了病，就等死——死也别赖我们！」

众人附和，声浪一层压一层。压里，叶青禾未出，药铺门闭着，闭得像一块青布，布后秤声实。实，是最硬的辩。韩泥担筐路过，不近祠堂。近了成靶。他绕远半步，半步就够——够看，够听，够记。记不是怂，是候还。

祠堂侧，里正与赵家低语，语里「坐实」二字清楚。清楚不要紧，要紧的是叶家秤还在——还在，公审就有理。里正抬眼，瞥一眼韩泥背影，没叫。叫成戏。戏明日才演，今日只敲锣。

马春花瞥他，啐道：「晦气翁，明日你也去！去听听寡妇怎么辩！辩不清，连你一起滚村西！」

韩泥「嗯」一声：「去。听真话。」

马春花一愣，想再尖，赵绣春拽她袖，低声却亮：「让他去！去了，坐实一对！坐实了，公审好收尾！」

韩泥不辩。辩成嘴仗。嘴不如心。心里叶汤已深，深得够扛明日。

午后，沈枯芽在巷口徘徊，手里攥着小纸包，纸包抖，抖得像叶。叶干，仍要送药，送药像还粥。韩泥停步，筐放脚边，不挡路。

「韩翁……」芽儿声怯，怯得像那日狼口，「我娘让送药叶……不敢进门……赖管事说，签字连坐……」

「敢。」韩泥接过包，轻，「药我收。名你别签。签字会连坐。连坐是扒皮的牙，别让它咬你。」

芽儿抬眼，眼红，红得像旱天的沙：「明日……叶丫头要公审……我娘说，要是关铺，我们没处抓药……」

「公审审谣。」韩泥独眼平着，「不审人心。人心若凉了，抓药抓谁？」

就三句。芽儿点头，跑了。跑得像一粒米，落进沈家锅——锅小，米真。真米能熬粥，熬粥也是活。这是暖，暖在压前，暖得像给明日留一口气。

沈枯芽跑远，韩泥望一眼，望得像记——记芽儿还肯送药，记沈家还肯捎话，记公审前不是全村的嘴都毒。毒有，暖也有，暖就是喘气的空。

傍晚，叶家灯亮。亮得比往日久。久，像在缝，像在称重，像在把明日要说的话一句句过秤。秤准，话就实；话实，铺就实。韩泥站在巷尾望灯，不往前。往前，赵绣春要笑；不往前，叶青禾也要站明日台上。台高，高不过一句「趁热」。

他回屋，心口叶汤深，深得像血里泡过。他不增笔，只按，按实，像按叶家脊梁。脊梁不弯，谣就压不垮。

「明日公审。」他低声道，「我坐着，不害她。她说话，我记着。记着，以后还。」

坛角微凉，坛腹微温。温里他候巳时，候赵绣春哑口的那刻——那刻未到，但绳已绷紧。绷紧到今夜，像鼓。鼓面是全村嘴，鼓槌是旱与谣。明日，鼓要响。响完，要么叶家秤还在，要么全村嘴更毒。

韩泥合上袄，塞进怀，独眼闭一息。闭不是睡，是蓄。蓄到明日，蓄一句：汤是热的，我记着。

里正又敲一遍锣，锣尾拖长，像绳结：「明日巳时，祠堂见。赵家作证，周家作证，叶家自辩。韩泥——」

人群「嗡」一声，嗡向他。里正顿了半息，像也怕「晦气」二字：「韩泥也到。到，不许闹。闹，连坐。」

韩泥在屋里应不了，应也在心里：到。不闹。不害她。害她，恩就白记了。应完，他摸心口叶汤，汤深，深得像能扛明日一整场潮。潮是压，深是气——气不在辩，在坐实，在记着以后还。

赵绣春笑声穿过墙，笑里带胜：「明日，看绝户怎么坐！」

周嫂接：「看寡妇怎么辩！」

声远去，远成钩。钩尖是「明日公审」，钩尾是叶家灯——灯还亮，亮得像没认输。韩泥摸袄肘针脚，针脚密，密得像叶青禾明日要说的话。话不必多，多不如实。实，他信她。信，也是还的一种预备。

张麻婆路过，瞥一眼他窗灯，敲两下门：「还不睡？」

韩泥道：「编筐。手实，心实。」

张麻婆道：「心实就好。叶家那秤，你也实着。」

韩泥道：「不害她。」

张麻婆道：「害她，你那些还字全白写。」

韩泥道：「不写白。记着。」

张麻婆走远，远里嘟囔：「老头倔，倔好。倔才能还。」

韩泥应声，藤条又绕膝。绕出一道细棱，棱不尖，尖的是明日——明日巳时，公审见真，真在秤，也在叶汤。他摸袄肘针脚，针脚密，密得像叶青禾明日要说的理——理在针脚里，也在汤里，也在「不害名」三个字里。理在，就不怕末席。

明日公审，村静得像绷紧的绳。他不备词，备心——心实，叶丫头一句证白就立得住；立得住，汤就还热，锤落，才是明爽。爽是后话，话先在「我记着」里。

丑时前，他仍编筐，指实，筐紧——紧，像把还字编进藤里，编进命里，编进明日末席那一坐。赵绣春门内又摔碗，摔得脆，脆里带虚——虚在药铺门仍开；开，就还有病人，还有竖立的理。

韩泥不近祠堂，不近叶阶——近是害，远是护。护到明日巳时，护到叶丫头把秤摆实。沈枯芽隔巷探头，探头又缩，像怕连坐。韩泥隔巷应，声哑：「别送药了。公审前，你娘要活，叶姑姑要实——实了，药才抓得着。」

芽儿点头，跑了。跑远，像把暖又送一遍；送一遍，够撑到锤落。

陈姑夜过巷，棉袄领拢紧，停在门缝外，没进。「韩翁。」她嗓子沙，「明日公审，你坐着。坐着，叶丫头才说得清。」

「我坐着。」韩泥没开门，「不害她名。害她，恩白记。」

陈姑顿一息：「赵绣春备了半日词。词在嘴上，理在秤上。秤在，她尖不过。」

「嗯。」韩泥藤条又绕膝，「我记着，以后还。」

陈姑步远，远里像叹——叹轻，像也给明日留一口气。坛角微温，像瓮替他守着「我记着」——守着，就不怕谣一夜吞村。

里正锣声又远，远成钩，钩尖是「明日公审」，钩尾是叶家灯——灯还亮，亮得像没认输。韩泥合上袄，不睡实，独眼闭一息。蓄到明日巳时，蓄一句：我坐着，她说话，我记着。

沈枯芽门缝塞一片姜，姜薄，薄得像娃也懂「公审」二字重。韩泥拾起，不尝——尝像收谢；不收，只记。记在心里，娃的胆，也算一证。赖扒皮短签在桩上，风撕角，撕不掉名。名在仇苗里，苗在，公审前还要立汤。

明日巳时，祠堂公审。锤未落，心已实。韩泥举目西望，叶家药铺灯一线，像恩还在远处守着。他低声道：「明日公审。我坐着，她说话，我记着。记着，以后还。」坛角微温不退，沿口水汽一线，像瓮在睡，却听见了「公审」二字。

他合眼，手仍按在坛沿上。他望坛，低声道：「手在，路在。」坛不应字，只温。温在，明日那锤就站得住。
