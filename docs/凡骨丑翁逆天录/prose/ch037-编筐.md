# 第三十七章 编筐

管事掷单，单上多一行：编筐二十只，藤条库房自取。韩泥接单，单旧，字丑。念帖让他进门，进门让他编筐——筐是泥岗老本行，本行在宗门也得有饭。藤条库房，条劣，脆。脆条勒指，指痛，痛他习惯。泥岗野藤韧，韧得筐卖过十二文一只。库役倚垛嗤：「丙九也领条？领多了编不完，别糟蹋库房。」

韩泥只取二十捆。捆沉，压肩。「丑杂役，做不完，没饭。」赖福倚门，笑里带秤，秤在心里。韩泥不辩。舌要留来勒条，留来活，留来还。库役斜眼：「条脆别怨，丙九就这待遇。」

「不怨条。」韩泥说，「怨手不够熟。」

库役一愣，嘟囔：「熟了也是丙九。」

「丙九也要吃饭。」韩泥扛捆出门。同批有人笑：「编筐？泥岗老把式干这个？」

「把式不挑活，活挑饭。」韩泥说。笑的人住了嘴。嘴住，饭还在。库房到院角，路长，霜滑。韩泥肩沉，步却匀——匀得像泥岗集尾担筐，一肩二十捆，捆捆不晃。晃了，脆条散地，散地就没饭。同批有人超他，超完筐底松，管事一眼扫来，那人又慢下来。慢，韩泥不笑——笑费气，气要留来勒条。午后，丙九院角，日斜。他编，慢而稳。底先编，编密，密了才立筐壁。管事路过，瞥一眼：「手熟。」

赖福啐：「手熟有屁用，凡骨炼不了丹！」

韩泥抬眼，独眼平：「筐熟，饭熟。饭熟，人活。人活，丹以后再说。」

赖福唇动，没再尖。尖了也没丹炼，尖了只讨骂。邻床少年凑近：「你编得真慢。」

「慢，不塌。」韩泥说，「塌了，没饭。」

「快一点不也行？」

「快，底松。松了，砸人。砸人，记名。记名，滚。」

少年缩脖，走了。编到第五只，筐底松了一线。他拆，重编。赖福晃过来：「拆什么拆，交差就行。丙九讲什么规矩？」

韩泥手指不停：「塌了，砸灵兽。砸灵兽，管事记名。记名，滚。滚了，嗅渣都没得嗅。」

赖福被噎住，甩袖走了。管事再经，瞥筐底纹路：「这只行。」

「行，才交。」韩泥说。编到第七只，他试勒法——泥岗老把式叫「三压一收」，压三遍，收一口，口紧，壁才立。勒法在指，指在茧，茧厚了，劣条也服。同场杂役蹲旁看：「丙九也配学老把式？」

「把式不挑活。」韩泥说，「活挑饭。饭在，指就在。」

编到第十只，藤条炸刺，刺进掌心，血珠小，却疼。他吮掉，血落篓底——记着，等瓮醒那日扎得准。同批少年蹲旁看：「你手流血还编？」

「编了，有饭。」韩泥说，「饭在，手才在。」

少年啧一声：「放着不管，化脓呢？」

「脓记着。记着，瓮醒才推得对。」韩泥把篓底血抹净，不往坛那边蹭。管事又过：「手裂了换藤，别硬撑。」

「不换。」韩泥说，「劣藤也要编完。编完，才有饭。」

管事「哼」一声，却多掷半捆好条在脚边：「好条编底，底要承重。」

韩泥用好条编底，劣条编壁——壁松可拆，底塌要命。他低声道：「记着。疼是疼，活是活。」

坛腹微温一丝，应疼，不应血。编到第十五只，赖福又晃：「编不完，明日兽栏加倍。」

韩泥指不停：「编不完，今日的饭也没。没饭，加倍也白加倍。」

赖福脸色一青，走了。编到第十八只，藤条又炸，裂口一手指宽。刘婆端水过：「手裂了？」

「裂惯。」韩泥说。刘婆放半块布：「裹着编，别逞。」

他布裹掌，继续编。编完，血渗布，布红一线。编到第十九只，院角风硬，吹得筐壁颤。颤，他不急，急则底歪。邻床少年又凑：「还差一只，赖福又要骂。」

「骂他的。」韩泥说，「筐不塌，骂不落底。」

「你掌都裂两回了。」

「裂两回，记着两回。记着，瓮醒才准。」

少年听不懂瓮醒，却看懂他掌布又红，闭嘴了。编到第二十只，他压底，压三遍，三遍齐，齐才掷给管事验。夜，二十只筐齐，码得齐，齐得管事不用尺量。管事验收，挨个掂底，点头：「丙九韩泥，筐结实，记功。」

同批有人嘀咕：「二十只全结实，丙九里头一个。」嘀咕短，短也像小胜——胜不在嘴，胜在底纹。底纹密，灵兽才不砸脚；脚不砸，管事才少骂。骂少一句，饭就多一口余地——余地不大，够活。赖福阴声：「记功有屁用，仍是丙九。」

管事喝：「滚去清粪。」

赖福走了，走挺急——急，也是小胜。管事掷饭牌：记功。多一口饭，多一口活。饭牌木温，温得薄，薄也真——真在册上多一笔，笔在肚里多一口。同批少年问：「你怎么不跟赖福吵？」

「吵赢了，饭少。饭少，恩断。」韩泥说。少年摸筐底：「真结实。掂着都沉。」

「结实，才卖十二文。」韩泥说，「泥岗人认这个。」

他摸坛沿，沿一线水汽，不退，退不了恩。回房，他展掌，掌裂处布沾血。低声对坛：「手裂，筐成。筐成，饭稳。饭稳，恩不断。」

坛腹温一丝，应完又敛。院角哐一声，赖福踹翻空筐：「记功有屁用，仍是晦气！」

韩泥出门拾筐，扶正，扶得筐立，立得住人才能立。管事夜巡：「明日嗅渣，手别抖。」

「不抖。」韩泥说。邻床少年又问：「你手裂了，嗅渣真不抖？」

「抖了，渣漏。漏了，没饭。」韩泥说，「没饭，活不到还。」

少年闭嘴，睡了。刘婆门外咳：「肘别挂粪。挂了，我又补。」

韩泥应：「记着。」刘婆又道：「编筐手裂，明儿嗅渣别逞强。逞强漏渣，管事记名。」

「不漏。」韩泥说，「手裂，鼻不裂。」

刘婆哼一声，脚步远了。他念叶丫头那句「汤是热的」，念刘婆布，念管事「记功」二字，并在心里，不混。他吐纳一口，气沉掌茧。茧醒着，「必还」也醒着。院角风又硬，空筐被赖福踹翻处，藤刺还扎在地上。扎着，他看了一眼，不捡刺——刺要瓮醒那日才用得上。明日嗅渣，鼻下找路；路找到，丹才可学；丹可学，汤才可还。

掌裂血渗，沿前即止。扎掌是计时，时未到，手先稳——稳，才配深秋末单。缘在瓮醒那日，扎掌才推得对——对前，今夜只记功，记功一口饭，够撑明日嗅渣。

「不抖。」韩泥说。邻床少年又问：「你手裂了，嗅渣真不抖？」「鼻下静，手才稳。」「静了，渣不漏。漏了，没饭。」少年闭嘴，睡了。他低声对坛：「手裂，筐成。筐成，饭稳。饭稳，恩不断。」编筐课毕，掌茧又厚一层。厚，疼真；疼真，记着，等瓮醒那日扎得准。库役数筐，哼：「丙九也能编齐？」「齐。

」韩泥说，「齐，饭才齐。」齐，记功一笔；功在册，功也在手茧里。明日嗅渣，鼻下静，苦甘分列——分列，才不混；不混，路才宽。坛腹温一丝，应「记功」，不应血。血在篓底，不在沿；沿净，才接得住后头缘。丑时末，他把筐摞齐，摞得像给明日嗅渣备一条路——路在鼻下，不在舌上。

摞齐了，掌茧贴着筐沿，沿凉，凉不过记功那口饭热。饭热，手才稳；手稳，嗅渣才不漏；不漏，恩才不断。院角风又硬，他把掌裂处用破布缠紧，缠得像给明日鼻下备一道门——门内苦甘分列，门外赖福骂。骂是风，风过，记功那口饭仍在肚里热着。同批少年梦呓翻身，肘碰筐沿，碰出一声闷响。

闷响小，小也记——记功在册，也在手茧；茧厚了，明日嗅渣鼻下才找得到路。坛腹温一丝，应记功，不应血——血在篓底，不在沿；沿净，恩才不断。丑时末，他把筐沿又抹一遍，抹得像给明日鼻下备一道门缝——门缝窄，窄才分得清苦甘。
