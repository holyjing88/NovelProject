# 第五十四章 坛温起伏

连雨一日，棚漏风。风进，坛席潮。韩泥擦沿，沿滑，滑像祖父韩老栓的手。手不在，滑还在，还在，像守瓮血脉的日常。坛腹午温，夜凉，晨又温一线。起伏，像瓮呼吸。吸，不应人；吐，不应灵。邻床笑：「晦气翁拜坛，拜出邪脉。」管事路过，声冷：「再嗓，滚去西栏清粪。」邻床闭嘴。韩泥不辩，只把辱记在心里，与恩分列。列清，心不乱。雨歇，驿道泥浅，小商队过，两驼，铃轻。褐袍人起身，起身，像等雨停后冬寒才够深。韩泥搬柴，不凑亭。凑，像贪符。

老耿挑水，水洒稳：「那人等冬。冬深，才赠。赠，不白受。」韩泥「嗯」一声：「白不受。受，记恩。」

---

午后，藤运到丙九，藤硬，硬像冬规矩。韩泥领末捆，管事验韧：「韧，编得出筐。筐圆，手才不抖。」筐未成，手先稳。稳，比美重要。同批少年偷懒，管事记名。记名，像秤上空一格。韩泥不偷懒。偷懒，麸皮克。克粮，克不走还路。刘婆端姜汤，姜薄，热：「雨潮，别空肚。」「不空。」韩泥喝尽，「嗯」一声。姜热，像叶汤远影。远影在，还路在。赖福讽：「丑手编筐，筐也晦气。」刘婆哼一声：「手稳，筐就圆。圆，能装柴，能装恩。」赖福脸青，像秤空一格。韩泥不辩，只编末藤。

藤刺扎掌，扎旧茧，茧裂，裂口潮，潮不是血。血不碰沿，沿才净。

黄昏，铁无言廊下见他：「坛温起伏，别当醒。醒，要等缘，不等人催。」韩泥独眼平：「不催。催，恩断。」「沿净？」铁无言问。「净。」韩泥说，「血不碰沿，沿才接得住后头缘。」铁无言「嗯」一声，退回廊角。退回，像门缝又开一线。同批少年碰他袖：「坛温起伏，你怕不怕醒？」「不怕。」韩泥说，「怕也守。守着，沿才净。」「净了干啥？」「净了，才接得住赠。」韩泥说，「赠，不白受。」少年不懂，哼一声，走了。

夜，雨又细。细敲棚顶，顶响像更鼓。他低声对坛：「温凉起伏，像你眠。眠着，不急醒。醒，要等血缘碰沿。碰，不是今日。」他摸掌茧，茧裂，裂口无血。无血，沿便安。安，才接凡符祛寒，接血近沿，接瓮醒。坛不应字，只温一线，又凉。凉，像应「等」。外头更鼓三响，响沉。沉里，褐袍人影仍在驿亭。在，像密宗线候冬，不急，急便邪。丑时前，坛腹又温一线，温短，像应雨歇，不应沿。邻床梦呓：「……温……」韩泥盖席，声低：「你看错了。睡。」他按席，席厚，厚像守了六十日的命。

命跟坛，坛跟缘，缘在沿前那一止。雨里小商队铃轻，铃轻，像界。界内末排，界外五脉。他低声道：「起伏不要紧，沿净要紧。净了，才接后头赠。」符未至，怀尚空。空不要紧，褐袍人袖里黄纸边露一线，露，像凡物将出。韩泥绕远，绕，像避腥沾袖。袖腥，洗得掉。心邪，洗不掉。老耿门外咳：「沿净，好。净，才接得住赠。」韩泥「嗯」一声，不追问。问急，像邪。等，像守瓮人。耿脚步远。远，像密宗线又近半分。

第二日，雨又歇，泥路滑。滑不要紧，肩稳，像编筐扎出来的。搬柴经侧门，驿亭褐袍人袖里黄纸边又露一线。露，像凡物将出，不出急。管事盯梢：「坛边别拜邪。拜邪，滚。」韩泥不拜。拜邪，恩断。守沿，缘在。他是记心翁，恩仇都记在心里。心里七笔，分列，不混。混了，手就飘。不混，手就稳。坛壁水汽爬半寸，敛。敛着，像替他守着。手别飘，恩不断。恩不断，叶汤就还在前头。柱上五日条在，试手在望。望不要紧，要紧是坛温时凉时温。温，像瓮听。凉，像瓮眠。

秦霜远廊经过，一句：「心别邪。邪，像借腥。」韩泥「嗯」一声，不辩。辩轻，恩重。他低声道：「你再眠一夜。眠着，我守席，守沿，守不主动滴血。」坛不应字，只温，温得像「嗯」。外头风硬，风里无栖香全句。全句散了，散成等冬那一息。他独眼平，手不抖：「等冬，等赠，等缘。缘，不在急。」第三日，霜白，侧门窄。窄得只容半眼，半眼，够看商线。驼铃远，尘落。落尽，驿亭褐袍人仍坐，坐得像等冬寒再深半分。贪，恩易断。管事验柴：「沉得住，记功在。」记功在，麸皮不克。

克的是粮，克不走还路。同批少年问：「坛温起伏，像醒吗？」「像眠。」韩泥说，「眠着，不急醒。」铁无言廊角，声低：「沿净，别拜邪。」韩泥「嗯」一声：「不拜。守沿，等缘。」刘婆端粥，稀，热：「雨停了，别空肚。」热在，像第七笔又实。坛腹午温，夜凉，起伏像瓮呼吸。呼吸不应人，不应灵。黄纸边露一线，露，像凡物将出。将出，不出急，急像邪。他独眼平，手不抖：「沿净，才接得住赠。」五脉旗号风里猎猎，猎猎不是谣，是界。界内末排，界外商线。小兽眼湿，他多铺干草。草旧，旧能暖。

暖兽，像恩还在。管事验藤韧，韧，才编得出筐。抖，试手滚。赖福偷懒，管事记名。肩稳，像七笔压着。夜雨又细，细敲棚顶。顶响里，坛腹午温，夜凉，起伏像瓮呼吸。邻床抱怨棚潮，潮不要紧，沿净要紧。」韩泥应：「净。净，像守瓮人。」第四日，风又硬。硬，吹得霜白。韩泥搬柴过半，肩稳。稳，像编筐扎出来的。起伏不要紧，沿净要紧。

他低声对坛：「你再眠。眠着，我守席，守沿。」同批少年问：「褐袍人还不走？」「不走。」韩泥说，「走，冬未深。深，才赠。」刘婆端姜汤，姜薄，热：「雨停了，手别僵。」「不僵。」韩泥说，「僵，藤刺偏。」铁无言廊角，声低：「坛温起伏，别当醒。」韩泥「嗯」一声：「不催。」外头更鼓沉，柱上五日条在。五日，试手在望。」一息在，赠在候，缘在沿前那一止。驿亭褐袍人袖里黄纸边又露一线。」息在，赠在候，缘在沿前那一止。

雨歇后，泥岗来的野藤运抵丙九，藤硬，硬像冬规矩。规矩在，筐才圆。韩泥领末捆，捆沉，沉像克扣的麸皮——克粮，克不走手稳。管事验藤，验韧，不验人：「韧，编得出筐；编得出，试手才不抖。」韩泥「嗯」一声，指节白一息，白完又平。平，像七笔压着。午后搬柴，驿亭褐袍人起身，起身，像雨停后冬寒才够深。人未近，袖里黄纸边又露一线。露，像凡符将出，不出急，急像邪。韩泥绕远，绕，像避腥沾袖。袖腥，洗得掉；心邪，洗不掉。黄昏，铁无言廊下见他一眼：「坛温起伏，别当醒。醒，要等缘，不等人催。」韩泥独眼平：「不催。催，恩断。」夜雨又细，细敲棚顶，顶响像更鼓。鼓里，坛腹午温夜凉，起伏如息。他低声对坛：「你眠，我守。守席，守沿，守不主动滴血。」坛不应字，只温一线，又凉。凉，像答应「等」。
