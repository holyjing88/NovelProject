# 第五十一章 先验商队

三日后，卯时，廊下霜白。霜薄，像给先验铺了一层素纸——纸上不写灵根，只写鼻子稳不稳。

丙九列队，末排先验。韩泥立最前。前不是荣，是险：险在「滚」字。滚了，麸皮停，记功抹，心里那七笔恩也断得慌。他闭了闭眼，把气息按到鼻下，不急。急了，香就飘；飘了，半个月杂役白干。

管事敲锣两声，廊静了下来。秦霜立廊角，木鱼不敲，只托在掌里：「辨香半刻，报香一字不落。漏一字，记过；记过两次，滚。」

香囊三只递到他面前。他闭息，先把别处的味压住，再嗅。第一只淡，像晒干的粟壳；第二只苦底里藏丹烟；第三只甜腻湿湿的，底下却有一丝腥——腥不在面，在底，像旧年栖香谣散了，散成今日规矩。

「粟壳淡，丹烟苦底，甜腻藏腥。」他嗓子哑，却一字不抖。

秦霜「嗯」一声，不夸，只记簿。铁无言立柱边，验他鼻下气，声低：「静着，别贪快。贪快，香飘。」

半刻尽，韩泥一字未漏。末排先验过了——过的不是灵根，是那鼻子沉得住。廊下有人偷吸气，吸完又装没听见。秦霜目光扫到队尾赖福，声冷：「再嗓，记名。」赖福哑了，脸上青一阵白一阵。

管事贴新条：「辨香过，候凝丹试手。试手在三日后。这三日，仍搬柴。」韩泥接条，条薄，真：真在仍是末排，真在路没断。他折好塞进袖里，肩上柴筐又沉回去。袖里木牌边磨毛了，毛得像心口那七笔——笔笔磨过，磨出一点实。

午后，他担柴过宗门外侧门。侧门窄，窄得只容半眼，半眼却够看驿道。

尘起。西驿商队过，驼铃远，车辙深。五脉旗号杂：佛门灰、密宗赭、西教白、密院黑、沉丹青——青是主场，杂是客。齐不齐不要紧，要紧是界：门内是末排，门外是商线，袖里别沾腥。驼队里有人咳，咳得寒，像北风贴骨。商队却不停——停不得，停了货腐，命也跟着腐。

佛门灰旗前有人施粥，粥稀，却热。热在客路，不在丙九棚。密宗赭旗后，褐袍人坐驿亭，坐得不急。急便像邪；不急，像等冬寒够深。西教白旗风里猎猎，密院黑旗压车辕，车沉得像克扣过的麸皮。韩泥肩上柴沉，沉得住，便不停看——看一眼界，够了。

老耿挑水路过，水溅半勺，洒得实：「五脉齐，冬里货紧。紧，别凑秤。」

「不凑。」韩泥说，「凑秤，手就飘。」

老耿又指驿亭：「那人等冬。冬深了才赠。赠，不是白受的。」

韩泥瞥见褐袍人袖里黄纸角露一线，露得像凡物藏锋。密宗那条线不在村庙了，挪到了宗门外。他绕开三步，不凑亭——凑，像贪；贪了，恩易断。

第二趟搬柴，侧门霜更白。褐袍人抬眼，眼淡，淡得不像看仙缘，像看人骨寒不寒。韩泥独眼平过去，平过去就是守。他想起叶丫头那碗汤：烫，辣，暖进胃里。暖进胃里的东西，不能叫外面腥味冲散。

商队领队验秤，秤杆落盘，落声脆得像叶家那杆秤。韩泥旁站，不凑，只记：商线在，心别乱。领队扫他一眼，像看尘，不看人。他倒松了口气——被人看尘，总好过被人当邪盯。

黄昏，丹房外，铁无言翻簿，见他一眼：「鼻下稳，三日后再验手。手抖，仍滚。」

韩泥只答：「手记着七笔，不抖。」

「七笔？」铁无言声更低，「笔在心里就行，别挂嘴上。嘴上一响，赖福那一伙便有由头。」

「不挂。」韩泥说。

秦霜声从远廊来：「五脉商队在，宗门戒躁。躁的人爱借商线生谣——谣借腥，你别沾袖。」

同批少年碰他袖：「你真过了鼻子？」

「过了。」韩泥说，「过鼻子，不过命。命在坛上呢。」

「坛又能咋？」少年挤眉，「又不是灵根。」

「坛是爷辈留下的。」韩泥道，「留下的，就得守。守着，手才不乱。」

赖福远处讽：「看商队也能看出仙缘？」管事声冷：「再嗓，克半月麸皮。」赖福哑了，哑得像吞了苍蝇。韩泥不再看他，只把柴垛摞齐——齐了，像给后头三日留一条正路。

夜里韩泥对坛，声低：「第一关过了。试手前，冬来了。」

坛腹温一线，像应「香」字，不应慌。他把七笔在心里挨个按实：叶汤、振东药、陈姑袄、刘婆粥、耿恩、僧那头还未到的影子，再加远村里几口暖——混了，手就飘；不混，手就正。外头驼铃余音散尽，褐袍人影还在驿亭。在着，像密宗线从村庙挪到宗门，挪得不急。

刘婆门外咳：「试手前，别空肚。空肚，手抖。」

「不空。」韩泥应，「粥在，姜还在记里烫着。」

「烫着就好。」刘婆又道，「烫着，鼻子才不虚。鼻子虚了，下次验手也虚。」

「记着。」韩泥说。

丑时他又近坛，不拜，只验席严不严。席严了，坛才肯应一点温；应温，不该应商线腥。腥若在袖，洗三遍；心若邪了，洗不掉。他袖上已无尘味，才把胸口贴住坛沿——沿净，后头缘才接得住。木牌与试手帖并放枕下，像命与规矩挨着。挨着，才验得过。

同批少年夜里又问：「五脉商队跟你啥关系？」

「没关系。」韩泥说，「关系在界。界内末排，界外商线。线过了，袖洗净；袖净了，才挨得近坛。」

少年似懂非懂，闭嘴睡了。管事夜巡过来嗅袖：「袖净，好。」韩泥不辩，只按席。按实了，像把「不借腥」按进肩里。铁无言第三趟巡廊，只扔半句：「商线过了，心别跟着跑。跟着跑，手就飘。」

「不跟。」韩泥应。

柱上试手条在风里贴得更牢，像倒计时。倒计时在柱，七笔在心，木牌在怀——怀在，路在；路在，恩才不断。他独眼平，低声道：「商队验完了，铃远了。线在界外，心在界内。三日后，试手。」

夜雪细，落肩即化。肩沉得住，三日后手才正。驿亭那人仍未走，黄纸角还在袖里候着——候不要紧，要紧是手别飘。飘了，窄路就断；断了，汤就凉。

卯时前他还去廊下闭息半刻。粟壳香早散了，散不要紧；要紧是鼻下那股静还在。静在，手才听使唤。同棚有人打哈欠问：「先验完了还练啥？」

「练不飘。」韩泥说，「飘一次，试手就凉一半。」

那人嘿嘿笑：「末排也讲究？」

「讲究活命。」韩泥道，「活命才还得上汤。」

棚角坛席又潮一线，他按平。按平了，像把商线腥挡在席外。挡得住腥，才挡得住谣；挡得住谣，七日里那几笔恩才不散。他把试手帖摸了一摸，帖边起毛，毛得像柱上新漆未干——未干也算钉住了：三日后验手，手正，丹才不歪。

丑时过尽，他还摸了一次木牌。牌薄，薄里刻痕浅，浅得像心口记恩那几道。浅不要紧，浅也能按实。按实了，商队再热闹，也热闹不到坛沿上。棚外风硬了一寸，硬得像催他睡觉——他却把席角又压严半分，严了才歇。歇着也把「试手」两字按在腕里。

天边白一线，白像试手前三日也白了半分。分不算荣，只算路还在。路在，他就站得住。站得住，驿亭那张黄纸才肯在冬深时露面；露面之前，他只守界内那条窄路。守住了，三日后验手才有底气——底气不在灵根，在鼻下那口静气还沉不沉。
