# 第一百一十七章 九层压顶

除名试炼，赖福押在坡下石坪。

九层压顶，气聚掌纹，冲关隘。隘裂一线，九层开了——开像内视里阶影又亮一阶，亮在暗，不在册。册仍写仍末，掌见九层，见给自己。

试炼石前，他掌按石，石凉，凉进骨。凉不要紧，九层稳在掌——稳了，118还芽石才接得住，119十层才稳。

钱戾衡影在廊下，影硬，手更硬。手硬，玉牌不虚——虚了，三年约白接。

试炼石风硬，钱戾衡立试炼石侧，声低：「九层破关，别飘。飘了，试炼石就洒。」韩泥应声：「不飘。试炼石在，手在。」手在，就不滚；不滚，还芽石才接得住。

辰时，全丙九列队。赖福扬木牌，牌面新字：「末席试炼，手抖除名。韩泥，先验。」

韩泥上前，掌按试炼石，石凉，凉进骨。石光起，灰线仍短——短不要紧，管事看手。手不抖，光稳，稳里气走九周，周周实，炼气九层。九层破时，他独眼半阖，鼻下静——静一字不落，落齐了，层才记进心里，不进舌。

层破，不露喜。喜像认晦气；认不得，只收掌，声平：「验毕。」

赖福眼赤：「九层也末！末，仍滚！」

管事弟子冷声：「试炼过。手稳，不除名。赖管事，再闹，记过。」

赖福噎，噎完换招：「不除名，永排末。末，永吃最后，永清最脏栏。」

「清。」韩泥只一字。

一字，手稳。手稳，是小胜——胜在活，胜在九层，胜在赖福压顶到头。到头不要紧，仇苗已深——深在玉牌，深在三年约，深在钱戾衡那一脚反面。反面烫，正面手稳；手稳，就不滚。

白织月立廊角，袖风冷：「九层灰线，仍像桩。」

韩泥不辩。辩，费舌；舌要留来记仇，留来备复测。复测在远，赖福压顶在近——近，忍过去了；过去，路宽一寸。宽一寸，够还沈枯芽半块灵石，够炼气十层。

铁无言抱臂，声低：「九层，别飘。」

「不飘。」韩泥说，「飘了，丹不正。」

「正，才配还。」铁无言说，「还谁，心里排序。」

韩泥不答，答在心里——心里芽儿半块灵石在前，刘婆新衣在远，叶汤在省亲前。排序不乱，还才正；还正，粒才正。粒正，魁才配，还刘婆才配。

散队后，赖福堵窑道，嗓哑，哑得像压尽：「韩泥，你赢了今日，赢不了三年。三年后，钱少主让你滚，你滚不滚？」

韩泥摸玉牌，牌凉：「三年约，记着。滚要手抖，我不抖。」

赖福退，退时像被「三年」二字噎住——噎住不要紧，压顶到此为止。到此为止，韩泥记——记赖福七年克扣，记公示三日，记除名试炼。记在心里，刃磨亮一寸。亮一寸，够还芽儿，够备战复测。

刘婆端粥，仍多半勺，半勺像奖：「九层了？」

「九层。」韩泥说。

「层在腹，不在嘴。」刘婆哼，「嘴响，饭稀。稀了，恩也不断。」

韩泥点头。谢轻，恩重。恩在粥里，也在忍里——忍赖福压顶，换九层实，换明日还芽儿资格。资格在，半块灵石在怀，怀沉，沉的是第五笔恩终于要落。

午后，他数怀内灵石碎。碎凑半块，半块旧，旧得像泥岗沈家罚银那夜——那夜芽儿被罚，罚因粥；粥恩真，还也要真。真还，首笔正式报恩；报恩，静着还，还到芽儿手里，才算数。算数，坛才温；温了，路才正。

半块灵石温，温得像烫手——烫手不要紧，该还。该还，夜行去外门杂院；杂院远，远得像徒线。徒线清了，刘婆恩另笔；另笔，在130还新衣养老钱。

他按坛，坛腹温一线，应「九」——九层，是赖福压顶的最后一关；关过了，徒恩在前。在前，半块灵石在怀，怀沉，沉的是第五笔恩终于要落。落了，心里空半寸，空得舒服——舒服不是无债，是债还了一笔，还能还下一笔。

韩泥应声。邪了，香飘；飘了，手滚。不滚，才配还沈枯芽半块灵石；还了芽儿，才配炼气十层，才配复测备战，才配130夺魁还刘婆。

丑时，练鼻下静。粟壳香淡，香落齐，九层记进心里。记进心里，不进舌——舌响，赖福虽退，眼仍盯。盯不要紧，手稳，就不滚。

邻床少年悄问：「你真要去还沈枯芽？」

「半块石，欠五年。」韩泥说，「欠着，丹不正。」

少年闭嘴。闭嘴，韩泥望外门方向——方向远，心近。近，明日还；还了，刘婆粥仍多半勺，钱戾衡玉牌仍凉，三年约仍烫。烫在仇线，不在还线；还线，静，不张扬。

赖福脚步廊外远，远得像压顶尽了。尽了，韩泥低声道：「赖扒皮，记着。压顶到此，刃亮一寸。」

坛不应字，只温。温像推他出门——出门，还沈枯芽半块灵石；还了芽儿，炼气十层才稳，复测备战才开。备战开，两月后上台；上台，钱戾衡眼风再来——来，不抖。

玉牌在怀，半块石在怀，恩在序。序不乱，步不飘——步不飘，才接得住118那一还，才接得住129末那一帖。帖在远，还在近；近，首笔徒恩，落得不张扬；不张扬，坛才温，丹才正。除名试炼前夜，韩泥理怀内灵石碎。碎凑半块，半块旧，旧得像泥岗沈家罚银。罚银因粥，粥恩真；恩真，还也要真。真还，在118，不在今夜。今夜，只九层，只试炼，只赖福压顶最后一关。

邻床少年悄问：「明日试炼，怕不怕？」「怕，手抖。」「不抖就不怕。」「不抖，记玉牌。」韩泥摸玉牌，牌凉，仇烫。烫在三年约，烫在钱戾衡「炼气十三，你才有资格不滚」。

丑时前，他廊下练鼻下静。粟壳香淡，淡不要紧；要紧是香一字不落。落齐，试炼才验得过手，验得过「不除名」。

试炼日，坡下石坪，测石青，青得钝。韩泥掌按，灰线仍短——短不要紧，管事看手。手不抖，光稳，气走九周，九层破。破，不露喜；喜像认晦气。

赖福眼赤，嗓哑：「九层也末！」管事冷声：「试炼过。再闹，逐管事位。」赖福噤，噤像压顶尽。尽了，韩泥记——记七年克扣，记诬告，记公示，记试炼。记在心里，刃亮一寸。

散队，刘婆端粥，仍多半勺，半勺像奖：「九层了，别飘。」「不飘，明早还芽儿。还了，饭更稳。恩在粥里，也在半块石里——石将还，还了，徒恩落笔。铁无言声低：「九层，别示。示了，赖福指偷境。应声，像应坛。

坛腹温一线，应「九」——九层，是赖福压顶尽；尽了，118在前，119十层，129备战帖，130还刘婆夺魁。他低声道：「赖扒皮，压顶到此。记着。」记着，就不白忍七年。白忍，恩断；恩断，叶汤凉。不凉，手在，路在，半块石在怀，玉牌在怀，三年约在怀。外头风软，软得像徒恩将还。

还将还不要紧，先睡一刻，先养手，先稳九层。稳了，明日夜行外门杂院，明日还芽儿，明日炼气十层开。九层破，赖福压顶尽。尽了，韩泥数灵石碎，碎凑半块，半块旧，旧像泥岗罚银。罚银因粥，粥恩真；恩真，明日还芽儿。还芽儿，静着还，还到手里才算。

算数了，心里空半寸；空得舒服，舒服不是无债，是债还了一笔，还能还刘婆，还能还叶汤。他摸玉牌，牌凉，低声道：「三年约，记着。手稳，粒正，试台见。」见字落，坛温一线，像应。应了，路又明一寸——明向还恩，明向不滚，明向刘婆新衣养老钱。」」

更鼓沉，沉里，韩泥按木牌，牌凉贴胸。117章手在，名在册，坛沿净。舌不留，鼻下静。恩辱分列，不混。活长了，才还得了叶汤，才接得住门缝灯。

刘婆粥仍多半勺，姜仍多一片。片暖，恩不断。恩不断，手才稳；手稳，粒才正；粒正，试台才配说话。说话了，白织月袖风尖不过「正」，钱戾衡影硬不过手。

封窑第七夜，窑温如秤，秤平——平不要紧，粒不杂；杂了，试台手就飘。
