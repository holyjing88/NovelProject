# 第八十三章 温听雨洒扫初识

废渣路后第五日，丹阁廊新扫。

丹阁侧廊水痕干一线，干像各做各的界。界在，韩泥不越；不越，像不求。不求，只记帚丝匀，记冷香淡，记她眉蹙那一息——蹙不像情，像规。

丹阁廊扫痕直，直像秤杆。韩泥担渣过侧廊，肩稳，渣不漏。廊角帚丝匀，匀在步，匀在腕，匀在鼻下那一点冷香——冷香淡，淡也要记。

同批少年窃语温氏名，韩泥不插话。插话，像凑热闹；凑热闹要还人情，他还不起。他只答：「不近。记着。」记着，恩线不添，仇线不添，像多了一线空。

丹阁廊让半尺，让得像避光。光在温氏门，不在他。不在他，才安全；安全，才接得住84再遇。

铁无言立栏边，声低：「温氏眼毒，毒里三分真——真在末席。末不要紧，渣里活，活长了，她眼才改。」眼才改，在200，远不要紧，韩泥不盼。

丹阁廊风硬，温听雨立丹阁廊侧，声低：「洒扫初识，别飘。飘了，竹帚就洒。」韩泥应声：「不飘。竹帚在，手在。」手在，就不滚；不滚，温听雨再遇才接得住。

扫痕直，直得像秤杆。秤杆在管事手，不在杂役帚上——杂役帚歪，歪也要扫；扫净了，才不被记过。

漏了，赖扒皮记过；记过，恩也断半寸。

廊角有人洒扫，裙素，帚细，帚丝匀。匀不要紧，人更匀——匀在步，匀在腕，匀在鼻下那一点冷香。

冷香不像白织月，不像刘婆粥，像丹阁里常年浸的药洗。药洗淡，淡不要紧；淡也要记。

那人抬眼，眼清，清里带傲。傲不要紧，傲在皮，不在帚。

她瞥韩泥筐沿，筐丑，人更丑。丑不要紧，她眉仍蹙一息：「丙九也走丹阁廊？廊刚扫，别踩灰。」

韩泥不辩。辩，费舌；舌要留来稳肩，留来守坛。他停半息，让出整尺，让得像避光。

「让就好。」她声不高，高在耳里仍尖，「末席担渣，像污廊。」

韩泥只答：「渣在筐，不在廊。」

答完，走。走稳，肩不晃，筐不洒。洒了，她更嘲；不洒，小胜。

胜在肩，不在嘴。嘴赢不得丹阁风，肩赢得过记过。

她未再嗓，帚丝又匀。匀，像丹阁规矩——规矩在，韩泥不近；不近，像不求。

不求，只记。记丹阁廊，记帚丝匀，记那一点冷香，记她眉蹙那一息。

名他不知道，名不要紧；名在远，人在帚。帚在，像钩——钩温听雨再遇，钩200章愣那一眼。

午后，同批少年窃语：「丹阁廊扫的是温听雨，温氏的外门，灵根不差。」

「温氏？」少年咂舌，「那咱别近，近了就晦气。」

韩泥不插话。

记着——恩线不添，仇线不添，像多了一线空。空不要紧，空才装得下将来的缘；缘在200，远不要紧，眼在近。

兽栏仍腥。腥不要紧，铲下稳，稳里鼻下仍静。静里，丹阁那一点冷香回半息，回像药山深灰的辛——辛淡，淡也是记。

管事路过，记一笔：「丙九韩泥，过廊不洒渣。」

不洒渣，是小胜。赖福讽远：「温氏也嫌你？嫌对了，末等污廊。」

韩泥不接茬。接茬，全丙九饿。他低声道：「渣在筐，不在廊。」

刘婆端粥，仍多半勺：「丹阁廊挨眼了？」

「挨了。」韩泥说，「不辩。」

「不辩怕什么。」刘婆哼，「手稳，饭稳。饭稳，恩不断。」

韩泥点头。恩在粥里，不在丹阁傲里——傲在皮，恩在粥，分列，不混。

铁无言抱臂立栏边，声低：「温氏眼毒，毒里三分真——真在末席。」

眼才改，在远，在200。远不要紧，韩泥不盼——盼像求；不求，只记。

黄昏，药渣场。他分拣，鼻下有根，手稳。稳里，丹阁冷香又回半息，回进沉腐浮里——浮是假香，假香剔净，真香才留得住。

真香淡，淡不要紧；淡也是路。

丑时前，他廊下练半刻闭息。息在鼻，不在嗓。嗓响，香飘；香飘，温氏就真嫌。

他按席角，席严，坛不露。坛腹温一线，贴掌，温不应傲，应「记」。记初识，不记名；名在远，人在帚。

邻床问：「温听雨漂亮，你怕不怕？」

「怕在污廊。」韩泥说，「怕不在貌。貌在皮，污在筐。」

少年不懂「污廊」，却懂「不辩」，闭嘴睡了。

秦霜木鱼轻敲：「识人别识皮。识皮，心邪。」

韩泥不答，答在心里——心里恩辱分列。初识是空线，空线通向再遇，通向嗅药小成。

他独眼平，手不抖：「初识了。帚丝匀，冷香淡。不近，记着。」

刃不亮，先记。记完了，才配再遇，才配辨性那一验。

更鼓沉，沉里，他摸暗袋，袋沉，坛温一线。温应记，不应盼。

玉牌在怀，愧字在怀，丹阁冷香在鼻——鼻响，心不响；心响，手就飘。

韩泥只答：「不恋。初识，记着。」

坛腹应温，不应字。温，像丹阁廊扫痕那一点直——直在规，不在傲。

他低声对坛：「温听雨初识了。不近，不辩。坛在，筐在。筐在，就不滚。」

坛不应字，只温。温，像把「初识」摁进空线，也摁进掌纹那路里。

白织月远廊过，袖风冷：「杂役也过丹阁？过出温氏眼？」

韩泥不抬头：「过廊，不进门。渣在筐。」

白织月唇线绷一息，走了。少一句嘲，小胜。

韩泥只答：「在。沿净。」

在，就不滚。不滚，初识才扎得实——扎实了，才接得住再遇，才接得住200那一愣。

柱上条角白一线，三年约在怀。怀凉，掌温——温在诀，不在丹阁傲里。

他低声道：「温听雨初识。舌不留，手要稳。稳了，污不了廊。」

污不了廊，是小胜。胜在筐，不在帚；帚匀，路仍长。

管事夜巡验棚，灯照席角，沿净。韩泥不辩，只按木牌——牌在，名在，路在。

同批少年悄问：「温氏会不会记你晦气？」

「记不要紧。」韩泥说，「手稳，筐不洒，廊就不污。廊不污，记也白记。」

少年不懂「白记」，却懂「筐不洒」，闭嘴睡了。

廊下风硬，他把木牌系紧一线，系得像把仍末摁进掌心。

同批少年远避，避像避晦气，晦气遮瓮，遮住了才稳。

编筐角藤刺仍在，他偏手，刺浅些，沿才净，坛才藏得住。

管事夜巡验棚，灯照席角，沿净，像腌菜坛，不像邪。

老耿挑水过，声哑，境别拜邪，拜了路断，他只记着。

秦霜木鱼远，一声，心别邪，邪了鼻下香飘，香飘手就滚。

柱上条角白一线，三年约在怀，愧字在怀，手仍不抖。

白织月远廊过，袖风冷，没停，胜在鼻，不在嘴。

铁无言脚步远，停半息，声低，别漏香，别叫境，只记。

刘婆粥稀，勺沉，饭稳恩不断，他喝尽，不谢，只记。

洒扫第三日，外堂阶尘更少。更少不要紧，帚仍稳，稳像药渣场分堆，稳像踹台按实那一掌。

温听雨擦栏，巾湿，水净，二人仍各做各的。她瞥帚痕，瞥完，眉松一线，没再多说。没多说，小胜。

管事验阶栏，皆净，各记一笔。赖福立堂外，眼细，讽晦气。韩泥不接茬，舌要留来记，留来守界。

铁无言声低：「温氏傲，敬才，不敬貌。你别求敬，手稳就行。」韩泥应声：「手稳。」

午后药渣场，他分拣，指稳。温听雨远廊过，袖风温一线，温在记里，不在舌上。他低声对坛：「初识了。再遇在后，不叫。」

洒扫毕，阶尘少，帚仍稳。温听雨巾湿，水净，二人不交舌多余——不交，像规。规在，恩苗才不邪。邪了，秦霜木鱼就敲第二声。

更鼓沉，沉里，韩泥按木牌，牌凉贴胸。83章手在，名在册，坛沿净。舌不留，鼻下静。恩辱分列，不混。活长了，才还得了叶汤，才接得住门缝灯。

丹阁侧廊水痕干一线，界在，韩泥不越——不越，像不求；不求，只记帚丝匀。
