# 第八十七章 暗修三层

小成续后第三夜，丑时将到。韩泥没睡实。

睡实了不敢冲；不冲，三层那关就卡着。他盘坐床沿，席盖严，坛不露。油灯豆大一点光，稳得像渣场记簿那一笔「准」——准在暗处，不在众目。他把玉牌压在膝下，压住三年约那点凉，才敢把气往上推。

掌贴坛沿，沿温一线。二层气转一周，两周；转到第三周，鼻下芽根忽然一紧。紧才真。真了，才敢冲。

关隘在骨里裂开一线，裂声闷，闷得屋里听不见。开不像打雷，像瓮唇吐出一口白气——气冷，冷里带药香，香里又回一丝焦，像废渣路上走过。韩泥屏息，独眼不眨。眨了，境散；散了，白开。

邻床杂役翻身，眯眼骂：「你……身上咋更热？起邪火啦？」

「编筐扎的。」韩泥声平，「看错了。睡。」

「编筐能热成这样？」

「能。」韩泥说，「藤条勒腕，血往上涌。你醒着就冷，睡着就热。睡。」

那人哼一声，又睡死。睡死了，三层才藏得住。韩泥收功像盖席：盖严，沿净，坛仍像腌菜坛。沿净了，赖福掀也掀不出名堂。

掌纹那路比昨夜厚半分，厚在暗里，不写进册。册上他仍是末席；掌里他却见着三层了——见给自己看，不给别人看。他把那半分厚度按回去，按得像把火盖进灰里。

更鼓将尽，天边泛白。他廊下站半刻，鼻下只闻辛淡，不闻辱。铁无言脚步远停，声压得更低：「三层别叫。叫了，像求境。求境的人，赖福最爱听。」

韩泥只答：「灯豆爆了，爆完即灭。没别人看见。」

「看见也别认。」铁无言道，「认了，渣场这口饭就吐。」

「不认。」韩泥说。

「还有。」铁无言补一句，「内视能见四层影，别急着追。追急了，香飘。」

「不追。」韩泥应，「影在就行。脚仍站稳。」

卯时兽栏仍腥。腥得冲鼻，铲下却比昨日更稳。管事弟子巡栏，记一笔：「丙九韩泥，清细。」写完丢开，像记一只扫帚。韩泥不争那一笔——争像讨赏，讨赏会把暗修讨漏。

刘婆端粥来，勺口照旧多半勺：「昨夜又热？」

「豆爆。」韩泥说，「沿净。」

「沿净怕什么。」刘婆瞪他，「怕你空肚冲关。喝完再逞能。」

「不逞能。」他把粥喝尽，「先吃饭。」

刘婆眼梢软一瞬：「知道就好。饭稳，恩不断。」

他点头。粥稀，稀也暖；暖在腹里，也在那一层新境里。境轻，记重——记重了，才接得住往后更难的一破。

午后药渣场灰起。他分拣，鼻下小成仍清，指梢却多半分温——温得辨性更细。铁无言走近半步：「三层别漏。漏了，场上不收你这号。」

「不漏。」韩泥应，「沉腐浮，记着。」

铁无言又指暗袋方向：「袋别示。示了，赖福记私藏，管事也记邪。」

「不示。」韩泥声音更低，「示了，全棚挨饿。」

「懂就好。」铁无言走开，像没来过。

白织月远廊经过，袖风一凉：「杂役也境升？升出邪法？」

韩泥不抬头：「守沿。沿净，不叫。」

「不叫就干净？」她冷笑。

「不叫至少不脏别人。」韩泥说，「脏自己的，自己受。」

她唇线绷一息，走了。少一句嘲，就算小胜——胜在鼻子，不在嘴上赢她。秦霜木鱼轻敲：「境升心别贪。贪，邪近。」

韩泥心里应：不贪。三层是路，路通向更上面那阶，通向丹路，不是通向嘴上风光。他把木鱼声当尺子，量一量心——心不烫，境才稳。

赖福傍晚在门外讽：「末等也热？热出几层了？滚不滚啊？」

韩泥不接。接了要辨，辨了要露。他只把门缝夹紧，席角压实。赖福盯半息，终是没掀——掀不着净沿，掀了也像自己丢脸。

「装蒜。」赖福啐一口，「蒜装久了，也臭。」

韩泥仍不接。臭在别人嘴里，净在自己沿上。

温听雨外堂过，远远一瞥他肩线，声清半分：「肩比洒扫那日更稳。」

更稳，他不谢。谢轻记心。她袖风一转走了，走得干净，界还在。韩泥把「更稳」两字按在心口——按浅，不攀；按住，不忘。

夜里他低声对坛：「三层开了。不叫。坛在，人在。」

坛不应字，只温。温像爹那只手又搭上来：「坛别扔。」扔不得；扔了无根，三层白开。他掌心贴沿，像跟老人对暗号。

邻床忽然又醒半息：「你跟谁说话？」

「跟瞌睡。」韩泥道，「你醒着，它跑了。睡。」

「你最近真怪。」

「怪才活得长。」韩泥说，「正常的，早被克扣饿瘦了。睡。」

少年愣愣睡下。韩泥掌纹里气又转半周，不敢转多——转多了热气会喊人醒。他独眼平着：「三层在掌。四层那盏远灯还在。远也不盲。」

丑时前他内视一息，第四阶影近了半寸。近是催，催人别停。可停也不行，停像把刃入鞘忘拔。他把影记下，不追——追是明日的事，今夜先把三层捂热。

柱上条角白一线，风硬。他摸怀中玉牌，牌凉进骨，凉提醒三年约还摁着命。命在，就得藏；藏住了，才有后面的丹，才有还叶丫头那一碗热汤的日子。

老耿挑水路过廊下，嗓子哑：「境别拜邪。拜了，路就断。」

「不拜。」韩泥说，「只记着。」

「记着比拜强。」老耿走了，水桶晃一下，像点头。

更鼓深处，韩泥按木牌贴胸：「手在，名在册末，沿净。舌不响，鼻子静。」

同批少年远处避他，像避晦气。晦气遮瓮——遮住了，反而好。好让魔教嗅不准，好让破境不进告示。

天快亮时，他仍坐着。三层稳了，稳得像粪铲落栏要实。飘了，管事记「抖」；不飘，四层那影才等得住。

铁无言临走又丢一句：「明日丑时再试内视。影近了就冲。冲时——」

「不叫。」韩泥抢答。

铁无言哼一声，算认可。韩泥合眼，鼻下清一息。清里他听见远处木鱼收尾，像警告，也像准许：境可以进，心不能邪。

他最后低声道：「三层在了。四层在后头。丹路还没开，手先别飘。」

风从门缝过，带一点药渣的焦回。焦回提醒：路从渣起，也从藏起。藏得住今夜，才验得住明日那双眼——管事的眼，赖福的眼，还有他自己不叫出声的眼。

兽栏鸡叫早，他起来洗手。水凉，掌温。温的是境，凉的是规矩。规矩在，境才不像邪火。他把堤坝一样的「不叫」又夯实一寸，等下一夜远灯再近。

同批少年在棚口偷瞄他：「夜里真没叫？」

「没叫。」韩泥系袖口，「叫的人，管事喜欢记；记了，饭就少。」

「那你身上真没邪？」

「邪在嘴。」韩泥说，「不在沿。沿净，你摸也摸不着。」

少年缩手，讪讪去铲栏。韩泥回看席角，坛沿仍严。严得像爹当年说的那句：坛别扔。扔不得，根就在这儿——根在，三层才不算白开。

午后药渣场又加了一小堆混灰试手。他鼻下一落，指头一拨，沉腐浮各归位。铁无言远远点头，没走近：「手还在，就行。别找我夸。」

「不要夸。」韩泥应，「要日子。」

日子往前挪一寸，四层影也往前挪一寸。挪够了，明日丑时才冲得开。他把暗袋按紧，按得像把种子捂进土。土湿，芽才肯抬头。

夜里他再内视半息，第四阶影近得更实。实了也别喊。喊了，瓮也暖不住这张嘴。

他又摸一次席角，角严，缝不露。严着，才像腌菜坛；像腌菜坛，明日冲四层时才不至于露馅。
