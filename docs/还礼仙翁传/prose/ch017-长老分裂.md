# 第十七章 长老分裂

七日之期里，青岚门像一锅将沸未沸的药。

长老会厅，烛高，影长。周德海展折子，声和气缓，句句主和：「东脉三成，非失地，是换时。七年太平，够弟子修炼，够宗门喘息。」

霍镇山拍案，震得茶盏跳：「喘息？赤焰今日要三成，明日要五成！妖潮是他们驱的，韩铁山亲口威胁，你们听不见？」

柳青鸢坐主位，未言，指节叩案，一声一声，像把战意敲实。

莫长春拄杖立窗下，望外头云，像看一张还没落字的契。

周德海转向他，笑里针：「莫长老九月寿元，最懂惜命。该劝宗主三思。」

莫长春回头，淡道：「老夫惜命，所以更懂命不值钱的时候，地更值钱。」

周德海一噎。

霍镇山抱拳：「宗主，主战！镇妖司可列阵，东脉民可迁不可散，散则根拔！」

柳青鸢抬眼：「若战，代价？」

霍镇山声沉：「死伤，灵田一季不收，赤焰再犯。」

「若和？」

周德海接话：「割三成，七年约，赤焰退旗。」

殿内长老分列，像一道无声的河，河水分成两股。柳青鸢目光扫过，在莫长春身上停一停。

莫长春不举手，不站队，只一句，像给河定岸：

「地可以给，根不能拔。」

柳青鸢闭眸一瞬，开眸时已有决：「根在，战亦可；根断，和亦败。」

她未拍板主战，却令周德海知——宗主心向霍，不向割地。

周德海揖手，笑收，退席时袖风冷，像把主和折子藏回怀里，藏成另一把刀。

会后，莫长春独留廊下。

柳青鸢出厅，低声：「师兄，你不表态，我却懂。」

「你懂就好。」莫长春道，「老夫若喊战，周德海会说九月之人煽动宗门；你喊战，才是宗主担。」

柳青鸢看他，眼眶微红又压下：「你总是把重的推给我。」

「重的才还得起。」莫长春笑，「轻的，才送得出去。」

她递茶，他没接，只指杖：「茶留给七日后的客人。赤焰少谷主，口味估计烈。」

苏念慈在丹堂听见长老会消息，炉火调得极稳，像把战意炼进丹里。

弟子问：「首座，若战，丹堂要备伤药否？」

苏念慈道：「备。不止伤药，备培元，备回春，备……让人活着回来的丹。」

她未说莫长春，却把九月二字压在炉盖下，像压一枚将爆的符。

赵玄机在执法堂抄门规，抄到「私囊不入公堂」，手停，墨晕开，像看见自己袖中那三颗培元丹。

他忽然想把丹还库，又拉不下脸——还了，像认自己怕莫长春那句「心在谁那边」。

夜，簿无大字，只寿元行微微一跳，像有人用指甲在脉案上划了一道：

八月。

莫长春瞧见，笑：「少一月，多一月紧迫感。好。」

他抚杖，念：「下一阵，客人要进门了。」

杖中竹纹急游，气音二字，极轻：

备好。

大因果还在厉无殇亲至那日，像帖，像手帕，像全场寂然的三息。

会后，莫长春独留廊下。柳青鸢出厅，低声：「师兄，你不表态，我却懂。」「你懂就好。」莫长春道，「老夫若喊战，周德海会说九月之人煽动宗门；你喊战，才是宗主担。」

苏念慈在丹堂听见长老会消息，炉火调得极稳。弟子问：「首座，若战，丹堂要备伤药否？」苏念慈道：「备。不止伤药，备培元，备回春，备……让人活着回来的丹。」

赵玄机在执法堂抄门规，抄到「私囊不入公堂」，手停，墨晕开，像看见自己袖中那三颗培元丹。夜，簿无大字，只寿元行微微一跳：八月。

霍镇山拍案，震得茶盏跳：「喘息？赤焰今日要三成，明日要五成！妖潮是他们驱的，韩铁山亲口威胁，你们听不见？」柳青鸢坐主位，未言，指节叩案，一声一声，像把战意敲实。莫长春拄杖立窗下，望外头云，像看一张还没落字的契。

周德海转向他，笑里针：「莫长老九月寿元，最懂惜命。该劝宗主三思。」莫长春回头，淡道：「老夫惜命，所以更懂命不值钱的时候，地更值钱。」殿内长老分列，像一道无声的河，河水分成两股。

围观弟子在长老会厅外踮脚，踮得像等主和主战分晓——分晓不在拍桌，在「根不能拔」四字。有人低声：「莫长老不站队？」另一人答：「不站队，才站得住岸。」

柳青鸢收折，折角微皱，像把主战决心也折进袖里。莫长春拄杖出院，低声：「七日之期，够备一份礼。」

大因果还在厉无殇亲至那日，像血旗，像手帕，像全场寂然的三息。
