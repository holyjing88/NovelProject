# 第一百二十四章 无言挡

试台风声紧前，赖福余党嚼「盗渣炼散」。

无言挡，铁无言立左侧三步，抱臂，像108并肩。并肩不要紧，今日挡话——挡粪话，不挡手。手稳，我掷簿；簿清，盗谣虚。

赖福廊外晃木牌：「试台滚，省我工夫！」韩泥只答：「在。」在，就不滚。不滚，125窑炉稳才接得住。

兄弟线，记——记124挡话，记108并肩。并肩不替炼，替证手稳；证了，试台净。

嚼声在丙九灶边，嚼得像粪臭扩散。韩泥端碗，碗糙，粥稀，稀里刘婆仍舀满。满，像挡话——挡话不要紧，铁无言挡更真。真挡，舌净，手稳；手稳，试台才配夺魁，夺魁才配还刘婆。

铁无言立灶侧，声沉：「盗不盗，渣场账在我这。韩泥账清，手稳。再嚼，我去管事处。」

嚼声止一息，又起，起在角落：「铁师兄护晦气，晦气传染！」

铁无言不辩，只掷簿，簿上朱印：「昨夜三袋渣，运丹房，有单。谁盗，谁指印。」

指印无人敢指。不敢，赖福脸青，青完走。走，韩泥记——记兄弟线，记挡话这一胜。胜小，胜在活，胜在试台前口净。口净，舌留来按罐，留来守坛；坛在，粒在，就不滚。

韩泥声平，只谢一字：「谢。」

「谢轻。」铁无言说，「试台，我站坡侧。不替你炼，替你挡粪话。」

「挡就好。」韩泥说，「舌要留来按罐，留来守坛。」

坛在角，罐在怀，九粒培元散沉——沉不要紧，炼气十二层稳，稳得像今日挡话稳。稳了，试台手才不飘；不飘，钱戾衡「看你抖不抖」就虚半分。

白织月廊下过，袖风冷：「铁师兄，护久了，也沾晦气。」

铁无言抱臂：「晦气在嘴，不在手。」

韩泥不插嘴。嘴要留来守静，守鼻下香一字不落。落齐，挡话这一日记进心里——记进心里，不进舌。舌响，赖福记「盗谣」；谣虚，试台净。

刘婆端姜汤，姜薄，热：「铁哑巴护你？」

「护。」韩泥说。

「护就好。」刘婆哼，「有人护舌，你护手。手稳，饭稳。饭稳，恩不断。」

韩泥喝尽。谢轻，恩重。恩在姜里，也在铁无言那一掷簿里——簿清，盗谣就虚；虚了，试台净半分。净半分，够夺魁预备，够130还刘婆新衣养老钱。

午后废窑，他开最后一炉备战炉。炉温稳，稳得像123那日十二层实；实了，再炼两粒，粒成，成九粒在手。九粒够，够试台，够夺魁预备——预备在130，预备在还刘婆。还刘婆，芽儿还了沈枯芽半块灵石，刘婆待还，叶汤在省亲前。

赖福闯窑，被铁无言拦门外：「管事区，验账再来。」

赖福悻悻退。退，韩泥记——记余毒尽，记挡话，记试台近。近不要紧，粒正，手稳，钱戾衡影硬不过「正」。正了，刘婆敢认恩，芽儿敢收石，叶汤还在前头烫。

沈枯芽送一小束干药草，草凡，凡得像泥岗：「叶姑姑说，试台手别抖。」

韩泥收下，收得像接徒线余温——温浅，够记。记着，芽儿恩还了，叶汤还在前头。前头那碗烫，还得到手，才配称一部圆满；圆满不在魁名，在债序。

韩泥应声。傲不得，傲像赢；赢不要紧，试台赢。试台赢，才还刘婆，才接大比帖。帖至，大比影至，钱戾衡名在帖后——后在不要紧，刘婆这笔先还。

丑时，练鼻下静。粟壳香淡，香落齐，无言挡这一日记进心里。记进心里，刃磨亮半分——亮在试台，亮在还恩，不在嚼声。嚼声远，远不过九粒「正」。

邻床少年悄问：「铁无言为何帮你？」

「108并肩。」韩泥说，「并肩，不替炼，替挡话。」

少年闭嘴。闭嘴，韩泥摸玉牌：「钱戾衡，记着。试台见。」

试台见，像钩——钩向125窑炉稳，钩向127债序清，钩向129备战帖。帖至，魁至，还刘婆至。还刘婆，新衣棉实，养老钱数过三遍，布包在怀等130递出。

外头风声又紧一分，紧不要紧，舌净一分，手稳一分。一分一分熬，熬到十二层圆满，熬到炼气十三在130破。破了，大比帖在怀沉，沉的是债序，不是惧。

坛腹微温，应「挡」字。挡了嚼声，挡不了仇——仇在玉牌，在三年约，在钱戾衡「记住今日」。记着，不抖；不抖，就配上台，就配还刘婆新衣养老钱。养老钱不多不少，够补衣婆活到老，不必躲赖福眼。

赖福骂远，骂像风。风过，不退铁无言那一掷簿；簿在，盗谣虚；虚了，韩泥端碗仍稳，窑温仍稳，七粒变九粒仍正。正了，试台见；见了，白织月说「运气」也尖不过「正」。

他低声道：「挡话，记着。试台，见。」

见字落，路又明一寸——明向125封窑，明向128十二层圆，明向130还刘婆。还刘婆，恩偿静着还，还到该还的人手里，才算数；算数，坛认「还」字，腹里微温应。

挡话后第二日，韩泥开最后一炉，炉温稳如秤。秤平，粒正；正了，九粒在手。手不飘，飘了，试台洒；洒了，除名。除名，三年约白接。白接，恩断。恩断，叶汤凉。凉不得，先夺魁；魁在130，不在舌。舌要留来守罐，留来守坛，留来记玉牌。玉牌凉，仇烫，烫在试台，烫在大比，烫在不滚。

铁无言挡话这一日，丙九灶边嚼声止了又起，起在角落。铁无言掷簿，朱印：「渣场账清。」谢轻，兄弟线重——重在不替炼，在挡粪话。挡了，舌净，手稳，试台前口净半分。

试台风声紧前，他再开备战炉。炉温稳，第九粒成，成最后一粒；粒丑，泥纹浅，嗅之正。正，九粒在手，手才不飘。近不要紧，粒正，手稳。」韩泥收下，记——记徒线余温，记叶汤头恩在省亲前。省亲前，先刘婆，先夺魁，先炼气十三破。」「护就好。有人护舌，你护手。」韩泥喝尽，记——记第七笔恩，记130还。还了，芽儿清了，序不乱。

夜，十二层稳，稳得像九粒在怀沉。沉像债序——债序在，步不飘。步不飘，才还得到刘婆新衣，才还得到叶汤烫手。

坛腹微温，应「挡」——挡了嚼声，挡不了仇。仇在玉牌，在三年约，在钱戾衡「记住今日」。记着，不抖；不抖，就配上台。」见字落，接125窑稳，接126刘婆补袖，接129帖至。破了，钱戾衡台前，才不滚。

挡话胜，舌净，手稳。稳了，开最后一炉，九粒在手。粒正，试台不洒。不洒，钱戾衡「上台别洒」白说。白说不要紧，试台见。见了，正字落，嘲哑，影虚，玉牌不虚。不虚，就配125封窑七日，配126刘婆补袖，配130还刘婆新衣养老钱。备战末段，粒在怀，怀沉，沉像债序。序在，步不飘。飘了，试台洒；洒了，刘婆恩还迟。迟了，丹不正；不正，叶汤还凉。凉不得，汤是头的。

九粒在怀，怀沉，沉像债序。序在，步不飘，飘了粒洒。洒了，刘婆恩还迟。

九粒齐，韩泥理怀，理得像理债序。序清，步不飘。飘了，试台洒；洒了，除名。除名，恩断，玉牌白凉。不凉，手在，粒在，坛在。在，就配上台，就配末项，就配130魁，就配刘婆捏新衣袖口终于敢信。敢信，就够还。还了，才正；正了，才配叶汤省亲前还，才配钱戾衡台前不滚。

他摸玉牌，牌凉，低声道：「三年约，记着。手稳，粒正，试台见。」见字落，坛温一线，像应。应了，路又明一寸——明向还恩，明向不滚，明向刘婆新衣养老钱。

挡话胜，舌净，手稳，试台见。

更鼓沉，沉里，韩泥按木牌，牌凉贴胸。124章手在，名在册，坛沿净。舌不留，鼻下静。恩辱分列，不混。活长了，才还得了叶汤，才接得住门缝灯。

刘婆粥仍多半勺，姜仍多一片。片暖，恩不断。恩不断，手才稳；手稳，粒才正；粒正，试台才配说话。说话了，白织月袖风尖不过「正」，钱戾衡影硬不过手。

铁无言立坡侧，声低：末项前，别辩。不辩，恩偿静着还，还到刘婆手里才算数。算数了，试台见正，见还，见帖至。帖至，大比影至，仇路明，恩路也明。明路，手不抖；不抖，就还刘婆，就还叶汤烫手，就钱戾衡台前不滚。
