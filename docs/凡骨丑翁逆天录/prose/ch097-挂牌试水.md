# 第九十七章 挂牌试水

晨雾。

坊市南脚，韩泥挂木牌，牌歪，字丑，三字：「丑丹师」。师字大，大不要紧；大，是试，不是认。

牌下仍卖渣，渣三包，包旁多一小签：「可嗅，不包治。」

不包治，才像杂役该写的。该写的，管事少问。

第一个问价的，是外门瘦弟子：「嗅渣多少钱？」

「半文。」韩泥说，「嗅一次，不加价。」

瘦弟子递渣，渣灰。韩泥鼻下一动，动半息：「青芯过多，火路会焦。减一撮。」

瘦弟子愣：「你真嗅？」

「嗅。」韩泥说，「嗅完，手稳。」

瘦弟子付钱，付钱像试——试了，不传谣；谣在远，远不要紧。

第二个是药摊婆，婆眼浊：「帮嗅这包，嗅不出，不赖你。」

韩泥嗅，嗅出褐多青少，少不要紧；少，他指节在包上点三下：「加青，减褐，先温后投。」

药摊婆喜半分，喜不要紧；喜，不谢——谢轻，记在心里。

白织月路过，瞥牌，瞥完，声冷：「丑丹师？也配？」

韩泥不抬眼：「配试。试不过，撤牌。」

白织月袖风过，过不要紧；过，是风，不是判。判在丹，丹还未炼散。

铁无言立远，抱臂，不近摊。不近，像不认；不认不要紧，眼在，眼是证。

午后，管事弟子来：「谁许你挂师字？」

韩泥声平：「试。试不过，撤。」

管事记簿，笔停半息：「试三日。三日无投诉，再议。有投诉，连坐丙九。」

韩泥应声：「三日。」

三日，够了；够了，才知坊市认不认丑手。

赖福闻讯，晃秤杆：「丑丹师？晦气师！」

韩泥只答：「牌歪，人丑，渣真。」

真，不在嘴，在鼻。鼻下静，手不飘。

收摊，他数半文，文少，少不要紧；少，是试水，不是养家。

夜，按坛，低声道：「试过了。坊市没赶。」

坛腹温，像应「试」字。试，不是成；成在110，在远。

邻床问：「你真敢挂师字？」

「敢试。」韩泥说，「试不过，撤。撤了，仍清栏。」

丑时，静半刻。香落齐，掌纹温一线。温，像培元路又熟半分。

他低声道：「三日。三日后，再看风声。」

风声里，兽潮二字又近。近不要紧，近，要备。

晨雾里，木牌挂出，牌歪，字丑：「丑丹师」。

瘦弟子递渣，韩泥鼻下一动：「青芯过多，火路会焦。减一撮。」

管事弟子记簿：「试三日。」

赖福晃秤杆：「丑丹师？晦气师！」

「牌歪，人丑，渣真。」韩泥只答。

不近，像不认；眼在，眼是证。

收摊，半文少，少不要紧；少，是试水，不是养家。

第三日午后，有个外门弟子再来，再来，像认手。认手，不要紧；认名，才惹眼。韩泥仍半文，仍不加价。加价，像营生；营生，赖扒皮秤。

试毕，他撤「师」字，改「嗅」字，字小，小才藏得住。藏得住，才接得住98续摊——续摊，听兽潮风声更近。更近，柱上贴条，条丑，字真，真在「夜禁」。

第二日，牌仍歪。歪不要紧；歪，像他人。人丑，牌丑，手稳，坊市才肯续。

老客穿灰袍，不语，只递渣五包，包上系红绳——绳旧，像试人。韩泥一包包嗅，嗅到第三包，鼻下停半息：「这包有霉，霉在角，别投炉心。」

老客眼一细：「你怎么知角霉？」

「嗅。」韩泥说，「嗅完，不取钱。」

老客收包，收时塞一块下品灵石碎片——碎片小，小不要紧。小，不能收；收，像受贿。

韩泥推回：「半文就够。够，是试，不是营生。」

老客看他半息，走了，走像记下一笔。记不要紧，记的是手，不是名。

第三日，管事弟子来，簿上无投诉，便记：「试三日，暂留。」

暂留，够了。够了，韩泥撤「师」字，改「嗅」字——字小，小不要紧；小，才藏得住。

散队后，他编筐半刻，藤刺扎掌，偏手，刺浅些。编筐钱买米，米在，恩才还得上。

柱上条角白一线，三年约在怀。怀凉，掌温——温在试，不在众目。

备手，备渣，备不飘。
