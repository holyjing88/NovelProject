# 第四十一章 备检

入门两月余，雪意落了半寸，半寸薄，薄也冷。丙九棚外贴条：凡骨杂役，测灵备检。备检不是正测，是筛——筛手抖的，筛嘴硬的，筛活不下去的。韩泥读条，独眼平。冬测备检过了，测灵备检仍排末。末，他熟。邻床少年抖：「听说正测在冬后，备检不过，正测没名。」

「没名就滚。」另一人接，「滚了，恩断。」

韩泥不插嘴。嘴要留来按石，不留来吵。管事弟子喝：「丙九集合！测灵石前站直，抖的，滚！」

测场在侧门石坪，坪白，石冷。灵石竖中央，石青，青得钝。钝，不像仙，像秤——秤灵根，秤命，秤你配不配留。第一个少年按掌，石光一闪，绿线三寸。三寸，外门边缘。「过。下一。」

韩泥看，不看光，看手——手按石前，气先稳，肩后沉。记法，不记色。轮到丙九，赖福先推人：「丑杂役最后，别沾石晦气。」

管事冷声：「按序。韩泥，末。」

韩泥上前，掌茧裂处疼，疼不露。掌按石，石凉，凉进骨。光起，灰线，线短，短不过指节。众人嗤。嗤声薄，薄也真。管事瞥一眼：「凡骨末等。备检过——手稳，石前不抖。」

赖福阴笑：「末等也过？丙九丢脸。」

韩泥收掌，不辩。舌要留来嗅渣，留来活。管事掷木牌：「韩泥，备检记名。正测冬后，仍末排。」

木牌薄，薄也真。真在「记名」二字——有名，不被滚。邻床少年按石，绿线两寸，抖了半寸。管事喝：「抖！滚！」

少年哭腔。韩泥不回头——回头，手易软。同批有人低声：「灰线也过？」

「过的是手。」韩泥说，「石前不抖，才滚不了。」

那人点头。点头像学稳，学稳，也是小胜。白织月经坪外，袖风冷：「末等灰线，也配站石前？」

韩泥「嗯」一声：「管事看手，不看嘴。」

白织月唇线绷一息，走了。少一句嘲，小胜。铁无言抱臂立栏边，声低：「灵根末，手稳，渣场仍收你。别漏。」

韩泥应声：「不漏。」

管事收石，喝：「备检毕。记名者，正测冬后。未记名者，明日滚。」

丙九静一息。一息里，有人看韩泥木牌——牌薄，牌真。赖福阴笑，笑浅：「末等记名，正测仍是末。末，滚得快。」

韩泥不答。答，费舌；舌要留来记木牌。笑里藏着冬后滚字——滚字离韩泥三尺远，远一寸，多活一日。午后，兽栏仍清。雪混粪，粪铲下稳。小兽角里咳，咳声细。韩泥多清半刻，湿粪换干草。草干，味淡一寸。管事路过：「备检过了，兽栏别抖。冬角仍你，清细，记着。」

韩泥应声：「记着。」赖福晃秤：「末等灵根，多吃一口也是浪费。」

韩泥不接茬。接茬，全丙九饿。刘婆舀粥，仍多半勺。半勺，热在。她低声：「备检过了？」

「过了。」韩泥说，「末等，手不抖。」

「末等怕什么。」刘婆说，「手稳，饭稳，命稳。命稳，丹以后才稳。」

韩泥点头。谢轻，恩重。同批少年凑近：「你怎么石前不抖？」

「搬石练肩，编筐练指，清粪练铲。」韩泥说，「练到稳，石前才稳。」

「可你是灰线……」

「灰线在石上，不在手上。」韩泥说，「手稳，饭稳。饭稳，才熬得到正测。」

少年闭嘴，像被「稳」字堵住。管事扫赖福一眼：「你也去按石。抖了，你滚。」

赖福脸白，闭嘴。白，小胜。韩泥收木牌，收入怀。怀暖，牌更真。酉时，渣场。韩泥分三堆，堆齐，香渣单放。鼻下苦甘分列，甘短，苦长。铁无言再瞥：「备检过了，仍别漏。」

「不漏。」韩泥说。街角赖福喝：「末等也配分堆？」

韩泥不回头。辱记着，手仍分——分堆，路才宽。管事验场：「丙九韩泥，渣清完，香篓齐。明日仍渣场。」

韩泥拈香渣，篓满。满而不泄，泄则路窄。夜，韩泥对坛低声：「灵末，手稳。」

坛壁水汽一线，爬半寸，敛。敛着，应「稳」字，不应灵色。邻床问：「末等，你还熬什么？」

「熬还。」韩泥说，「手稳能活。活了，才还得上。」

「还得上谁？」

「叶丫头汤，刘婆粥。」韩泥说，「泥岗六笔，宗门一笔。分列，不混。」

邻床又问：「正测冬后，你仍末排，图什么？」

「图活。」韩泥说，「活一日，记一日。记满了，才还得上。」

邻床懂「活」，翻身睡了。外头雪落，落声细，细得像秤盘轻响。响在耳，记在心——备检过了，正测还远，远不要紧，手稳一日，多活一日。木牌在枕下，薄，真。真在记名，不在灵色。管事巡夜，敲柱：「备检记名者，正测前别生事。生事，饭少。」

韩泥只答。一声平，平得像木牌落掌——落掌，才真。他低声道：「灵根末，命未末。命未末，就接着做粪，做渣，做筐。做到了，路才宽。」

坛沿温一丝，应手，不应灵。灵是石上的灰线，命是手上的茧。他合眼，不睡实，是记名。记名在枕下，命在手上。丑时，油灯豆，豆光照坛角。他擦沿，沿净。擦到第三遍，坛壁里忽然浮出极淡一道影，像篆，像半个「瓮」字——一闪，灭。灭得他以为眼花。

可掌心跟着微烫，烫得像血脉里有什么，与坛底沉睡了不知多少年的东西，隔了一层腌菜坛壳，对上了半息。韩泥独眼一紧，没喊，没拜，只把席盖压严。邻床翻身，梦呓半句：「……啥光……」

韩泥声低：「你看错了。睡。」

他低声对坛：「你眠着。我守。守到沿净，守到缘满。」

坛不应字，只沿口一线温，温短，像应他「守」字，不像醒。末席过，灵仍末。灵末手不末——手在，廊下嗅路才长，长向辨香关，长向还烫。木牌在枕下，与备检帖并放。并放，像命与规矩挨着——挨着，才验得过正测。丑时符影半闪后，他仍不喊醒。喊醒像拜邪；守沿，才配等缘满。

备检记名在怀，怀暖，牌更真。真在还能站，站得住，才配廊下嗅路。

翌日卯前，雪又落薄一层。韩泥先去兽栏，铲下稳，小兽咳声轻了半分。管事弟子路过，只瞥他木牌：「末等，兽栏别散。」

「不散。」韩泥说，「散兽栏，先散手；手散，石前就抖。」

弟子无话，走了。白织月袖风远过，仍不近——不近，像备检过了，嘲也省半句。韩泥不追嘲，追活：搬干草，换湿粪，铲磨掌茧，茧裂处疼，疼不露。

午后渣场，铁无言掷香篓。篓口松，香泄一丝，一丝就够认。韩泥分堆，苦甘分列，分列得像辨恩辱——恩苦短，辱腥长，长记着，不碰。铁无言「嗯」一声：「手稳，篓齐。明日仍你来。」

韩泥点头，不谢。谢轻，记在木牌边。木牌薄，真在还能站；站得住，才配廊下嗅路，才配冬后正测末排仍到。

夜归丙九，邻床梦呓问灰线。韩泥声低：「灰线在石上，手在拢柴、清粪、按石。手在，命未末。末等，手不抖。」

丑时符影半闪，篆影一掠即灭。他不喊醒，只把席盖压严——喊醒像拜邪，守沿才配等缘满。册上有印，印在，克字又虚一寸；虚一寸，够活，够问香帖在册。路通向栖香谣，谣要剥，剥一寸，手稳一寸。雪又落薄一层，落声细，细得像秤盘轻响——响在耳，记在心，备检过了，正测还远，远不要紧，手稳一日，多活一日。

邻床梦呓半句「灰线」，他声低：「灰线在石上，手在铲上。手在，命未末。」刘婆门外咳：「备检过了，肘别挂粪。」韩泥应：「记着。」坛沿一线水汽，应手，不应灵。灵是石上的灰线，命是手上的茧。备检记名在怀，怀暖，牌更真。稳。等正测。冬后，手仍不抖。
