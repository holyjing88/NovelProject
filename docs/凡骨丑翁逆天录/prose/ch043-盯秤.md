# 第四十三章 盯秤

备检后第五日，麸皮又浅。浅在秤盘，盘在赖福手里。

他晃得欢：「晦气杂役克宗门粮，粮都臭了！」

丙九有人顶：「臭的是你手！」

赖福举鞭：「再顶，明日没麸皮！」

韩泥低头清粪，不接茬。接茬，全丙九饿——饿比辱快，快就先活。他把铲贴地推，推得匀，匀了才不像心里炸。

同批有人压着嗓子骂：「半月短三成，再短就啃风！」

「啃风也别顶鞭。」韩泥低声道，「鞭落了，明天连风也没有。」

那人咬牙，到底没冲出去。冲出去爽一口，全棚饿半月，不值。

管事路过，秤盘一落：「册上三两，你发几两？」

赖福脸僵：「风干了……」

「风没干你的良心。」管事冷声，「今日起，发粮我盯。少一粒，记名。」

丙九静了一息。有人看韩泥——看他仍低头，看他手紧。他不抬头。抬头像邀功；功在饭，不在脸。

发粮场，丙九排队。管事亲执秤，秤杆实得像规矩钉死。前面人接粮时手还抖，抖完却笑——笑不是高兴，是活过来了。

韩泥接粮，三两，沉进掌心。沉得真，真得像把半月虚称一下子钉翻。赖福旁侧嘴唇动了动，没敢尖——管事在。

韩泥眼角扫过秤底，底缝沾着麸，薄薄一层。薄也记。记在心里，不声张；声张，赖福反咬，全棚跟着饿。他只把袋口一勒，勒紧了，才像把命绳拴住。

同批少年低声：「忍出来的？」

「忍半月，等这一盯。」韩泥说，「盯了，秤才实。」

「能实多久？」

「实到管事腻，或者实到我滚。」韩泥说，「滚前，先记仇。」

少年闭嘴。有人数粒，数完脸松一寸。松一寸，丙九都沾光。韩泥把粮袋系紧，系法跟泥岗编筐一样——底密，口收，一粒不漏。管事喝：「数完干活。杵着，粮也臭。」

众人散。有人碰他肩：「谢你忍那一回。」

「谢轻。」韩泥道，「饭实，是真。」

铁无言路过栏外，声低：「手紧的人，秤上才站得住。」

韩泥不抬头。夸轻，记重。

午后发粮场空了，秤盘还温。韩泥收粪路过，见赖福蹲在秤底，指刮盘缝，刮得轻，轻得像「风干」。他没喊，立了三息，等管事折返。管事来了，他只道：「盘缝有麸。」

管事掀盘，麸落掌心，脸色沉：「赖福，明日你发，我盯。再刮，记名。」

发粮场霎时静了。静里有人眼睛亮一下，又赶紧垂下——亮了像偷欢，垂下才像要命。

赖福退时眼里阴着。韩泥记住那阴，不顶。顶，全丙九饿；饿急了，往后的汤都凉。同批少年碰肩：「你早看见？」

「看见，不等管事，没用。」韩泥说，「有用的是饭实。实了，才还得上。」

刘婆端粥来，粥仍稀，勺却立得住半息——比昨日强。她瞪赖福：「再克粥，盆扣你头上。」

赖福退半步，骂「疯婆」，脚真退。韩泥喝粥，不道谢。谢轻，恩重——宗门这半勺，他另记。

午后赖福报复，掷单加倍：「兽栏冬角，韩泥再加半刻。」

韩泥只答：「加。」

管事折返，声沉：「加倍？册上没这一条。」

赖福脸白：「冬角脏……」

「脏，全丙九轮。」管事说，「单上加字，先问我。」

赖福收单，收得哑。韩泥铲下仍实。实，让管事肯盯第二回。同批又有人低声：「韩泥，要是管事不盯了呢？」

「不盯，仍忍。」韩泥说，「忍到他又盯，或者忍到滚。滚前，记全仇。」

那人闭嘴干活去了。酉时韩泥仍去渣场半刻。渣分三堆，香篓齐——粮实一日，手也不能松；手松，明日秤又浅。

同批少年跟来捏鼻：「你还做？粮都实了。」

「粮实一日，手不能松。」韩泥说，「手松，秤又浅。」

渣场灰热，苦甘各归各堆。归对了，鼻下才准；准了，正测辨香才站得住末排。少年不学，走了。刘婆端晚粥，稀却热：「盯秤这日，多喝一口。一口，顶三口克扣。」

韩泥喝尽。赖福门外骂：「疯婆子偏晦气！」

刘婆回骂：「偏的是人，不是秤！」

韩泥不掺。舌要留来记三两落袋那声沉。夜里赖福踹棚柱，骂：「盯秤盯到老子头上！」

邻床问：「你真不闹？」

「闹，饭少。」韩泥说，「饭少，汤还不了热。」

「还什么汤？」

「叶丫头那碗。」韩泥说，「热的。凉了，还乱。」

邻床又问：「你记的那些，到底几笔？」

「泥岗六笔，刘婆一笔。」韩泥说，「各记各的，别搅成一锅。」

邻床懂「热」，睡了。韩泥摸坛沿，沿温一线，温得像灶边半勺粥。管事巡夜敲柱：「明日仍我盯秤。赖福，少玩秤。」

赖福应声虚。虚，韩泥记——记第二回盯秤。同批有人梦呓：「饭……」

他睁眼，替盖袄：「排着。秤实了。」

梦呓止。外头风硬，吹得空秤盘响一声。响在耳，记在心——响的是赖福的虚，不是韩泥的滚。同批又低声问：「盯秤这日，你掌心不抖？」

「抖。」韩泥说，「抖在袖里。秤盘实了，袖里才紧。」

「紧了呢？」

「紧了，明日仍盯。」韩泥说，「仍盯，赖福才虚。虚了，丙九才多一口实粮。」

他低声道：「秤实不是闹出来的，是半月忍出来的。忍到管事肯盯，饭才实。」

翌日卯前，发粮场仍排丙九。管事再来执秤，杆子还实。韩泥接粮，仍是三两。眼角又扫秤底——底缝净了，净得像告诉全棚：盯过一日，虚字短半寸。

丑时刘婆门外咳：「盯秤这日，明儿仍多半勺。」

韩泥点头：「心不少。」

酉时他把三堆再理一遍。粮实一日，手不能松。理齐了，鼻下才静；静了，正测辨香才站得住。同批梦呓又问粮，他替盖袄：「排着。秤实了，梦就紧。」

第二日发粮，赖福那侧仍浅半分——浅在他，不在全棚。全棚有实粮一口，便多活一口气。有人凑近问：「你为什么不谢管事？」

「谢在粮里。」韩泥只答，「粮实了，丙九都少饿半口。」

赖福旁侧唇又动，没尖——管事在。韩泥不抬头，只记三两落袋那声沉。记沉了，才配正测末排仍到，才还得了叶丫头那碗烫。

酉后他回棚，把粮袋塞到床板下最深处。深了，才不叫眼红的人顺手摸走。邻床看着：「你藏这么严？」

「严。」韩泥说，「粮是命。命散了，盯秤白盯。」

「那仇呢？」

「仇另记。」他道，「饭先实，仇才还得动。」

夜里棚柱又响。响不是赖福踹，是风。风硬，秤杆却还实着。韩泥按住怀中木牌，牌凉贴胸。牌在，名在；名在，辨香那日才站得住末排。

他又把今日三两落到掌心的那一下，在心里又过一遍。过一遍，才不像做梦。梦醒了，秤会浅；记实了，仇才有根。

外头骂声远了，丙九才睡得着——睡得着，也是盯秤这日挣来的一口气。勺在灶边轻轻碰了一下，像提醒：实粮到手，还要手稳，才配正测辨香。

坛沿温一线，温得像灶边粥。他低声道：「秤实了，仇记着。记着，才还得动那碗烫。」

邻床睡熟前又闷问一句：「管事会不会明天就不盯了？」

「会。」韩泥说，「不盯也得忍。忍着，手才紧；手紧，渣不漏，辨香才有底。」

「那你图什么？」

「图活着还。」他声音哑，「还不上，秤再实也像别人的饭。」

棚里忽然静了半息。静里只有风擦屋檐。他按住木牌，牌边磨毛了，毛得像半月忍出来的茧——茧在，秤才有今日这一实。实了，正测辨香才不白熬。不白熬，叶丫头那碗烫才还得上。还得上，丑翁这一趟才不算白忍半月。
