# 第一百一十九章 十层稳境

还沈枯芽半块灵石后第三日，韩泥炼气十层实了。

实像粪铲落栏——落得要实，实了才不飘。飘了，管事记「邪」；邪了，复测风一吹就散。他把气按在掌纹里，按得像把门缝那盏灯先在心里钉住。

铁无言立棚侧，声低：「十层，别叫。叫了，像破外门境。」

「不叫。」韩泥应，「手正。」

「手正就好。」铁无言又道，「气别往外冒。冒了，赖福虽退，仍能嗅出你身上那点异样。」

「不冒。」韩泥说，「手在，就不滚。」

卯时丙九列队。赖福眼细，细得还想找由头，找不着，阴笑：「十层？末席十层，仍像灰线三尺。」

队里有人小声：「十层了还末？」

「嘘！」旁边立刻压住，「末也轮不到你提。」

韩泥不辩。辩费舌头；舌头留给窑温，留给再炼培元散。散为复测，复测为三年约——约上写滚，滚要手抖；不抖，才配站钱戾衡台前。台前要炼气十三才「有资格不滚」；十三在远，十在腹。

管事弟子念风声：「外门复测，两月后。杂役可报丹试，末席亦报，手稳者上。」

丙九嗡一声。嗡里有人拿眼扫他——扫，也像扫笑话。笑话不要紧，手稳；稳了，笑话变魁，变在远，变在夺魁那日。那日还刘婆新衣养老钱；还了，叶汤还在省亲前等着。

韩泥摸玉牌，牌凉。凉是钱戾衡，凉是那句「炼气十三」。他把凉按进掌心：「报。不报，三年约白接。」

午后废窑，他再炼培元散。渣掺灵砂，砂细，细像捡漏那粒。粒不要紧，效正；正了藏怀，不示人。示人，赖福克，白织月嘲「运气」。运气她说；正不正，试台见。

铁片嵌口，焚火残诀走息，息稳，窑温稳。散成三粒，粒丑，泥纹浅，嗅之却正。正，记一笔，备复测。复测过了，往十三走，魁名才落得下，刘婆那笔才还得动。

白织月过窑外，袖风冷：「丑杂役也炼散？」

「炼备试台。」韩泥捧渣袋，不抬头。

「炼了也是末。」她笑。

「末，看复测榜。」韩泥声平，「榜上手稳，嘴上末不重要。」

她唇动，没再尖。尖不过「报」字——报了，手稳，就上得了比试台；上了台，钱戾衡眼风再来也不怕。怕，手抖；不抖，玉牌才不虚。

刘婆送姜汤至窑道，姜薄，热：「十层稳了？」

「稳。」韩泥喝尽。

「稳就别飘。」刘婆哼，「飘了，粥洒。洒了，恩断——老婆子可不伺候断恩的人。」

「不飘。」韩泥道，「恩在你勺里，也在我掌里。掌稳，勺才不算白舀。」

刘婆嗯一声，像勉强满意。他把姜汤热意记进心里：层稳，饭稳；饭稳，恩不断；恩不断，还序才不乱；还不乱，丹才正。

邻床少年悄问：「你真报丹试？」

「报。」韩泥说，「不报，三年约白接。」

「钱戾衡……会来？」

「记着。」韩泥摸牌，「记着，刃才磨亮。亮了，才接得住他影。」

「你不怕人笑？」少年眨眨眼，「末席报丹试，棚里都在说。」

「笑在耳。」韩泥道，「手在掌。掌稳，耳就聋半分。」

少年闭嘴。韩泥望山门：山门远，复测帖在远，大比帖在更远。远不要紧，先十层实，先粒正，先刘婆粥多半勺。多半勺够顶一口饥，够顶备战两月。

丑时练鼻下静。粟壳香淡，混一丝培元药香——香正不要紧；要紧是静一字不落。落齐，十层记进心里，不进舌。记进心里才实；实了，才配迎复测风。

外头柱上条角贴新纸，纸边字淡：「外门复测，丹试报名。」

他读条，独眼平，不近廊下笑。笑在耳，不在心——心在坛，坛醒了，备战才真。真备战，两月后上台；上台，眼风来，接住。

报名册前，他押一字。字丑，像人；人稳，字稳。赖福阴笑：「报名也白报！」

管事冷声：「再聒，逐出坡。」

旁人低语：「末席也押字？脸皮厚。」

又有人接：「厚什么厚，他手不抖……嘘，赖福回头了。」

赖福噤了。韩泥把这一噤也记着——记末席风声，记报名这一笔，进备战，不进舌。舌响，赖福就记「报名晦气」；晦气不要紧，手稳夺魁预备。

铁无言掷来薄簿，簿角有印：「报名册有印。印在，手别飘。飘了粒洒；洒了丹不正；不正，刘婆恩还迟。」

「不飘。」韩泥应，「粒在怀，印在册。册在，就不滚。」

簿收回时，他又听见一句：「押了字就要滚得明白。」韩泥当没听见。听见也罢——厚的不是脸，是掌茧。茧厚了，粒才捏得住；捏得住，台上才说得上话。他说不上笑话，只说得上「正」。

午后兽栏一轮，铲子更稳一分。稳里他心里过数：三粒培元散，半袋边角渣，一块凉玉牌，一笔待还的刘婆。数清了，步才不乱；不乱，复测风才接得住。

管事弟子验栏，随口：「报名的是你？」

「是。」韩泥应。

「末席报，杂。」管事道，「杂也行——杂里出净手，宗门要。」

「净手我有。」韩泥说，「手净，粒净。粒净，榜上才不糊。」

夜里再炼一炉，罐封泥，泥封严。严了，盗谣虚；虚了，白织月袖风尖不过「正」。正在粒，不在嘲。嘲在耳，不在心。心在坛，坛温一线，应「稳」——稳十层，稳粒，稳手。

他摸玉牌，低声道：「三年约，记着。钱戾衡，记着。赖福克扣，记着。十层稳——复测，报。」

报字落，像钩。钩向风声，钩向备战两月，钩向炼气十一十二，钩向帖至那天。帖在远，刘婆在远，叶汤更远——前头那碗烫还得到手，才配称一路圆。

外头风起了。起得像试台在等。等在远，备战在近；近，炼粒，稳境，喝刘婆粥。粥稀，热，热得像第七笔恩又温半分。温半分，够熬到夺魁，够熬到新衣递进刘婆手里。

韩泥按木牌贴胸：「手稳，粒正，试台见。见了，袖风尖不过正，影硬不过手。」

见字落下，坛温一线。温像答：答应他不飘，答应他把「末」字当踏脚石——踏过去，才是魁，才是还，才是大比帖进山门那天。那天未到，手先稳着；手稳着，复测风再大，也抽不到掌心里那三粒正散。

夜里刘婆又来窑道口，碗沿磕一下席角：「字押了我就问一句——饿不饿？」

「饿。」韩泥接碗。

「饿就喝。」她哼，「空肚上阵，阵上飘。飘了，老婆子新衣还到猴年。」

「记三遍。」韩泥边喝边道，「衣袖要耐磨，钱数不多不少。」

「少了别进我屋。」刘婆哼着走了。走了，他笑一下，很快收。笑轻，记重。

他回窑角把三粒散又挨个过鼻。过鼻不是炫，是防杂——杂了，榜上笑你末；末不要紧，笑你丹不正才要命。要命的事他见过：公示、诬告、除名试炼都挨过。挨过了还报，报了就要像个人样备战。

清栏、炼粒、喝粥、守坛。四样齐，十层才叫稳。稳了，才配迎那阵复测风。

裂罐泥的事他不敢想。裂了散味飘，飘了赖福又来咬「盗材」。他把铁片嵌死，席角压严——严了，气闷在暗处发热；热不叫，叫了像求人看境。

更鼓沉。韩泥守窑角，掌按坛沿。沿净，气实，纸上丑字还在——字在，备战才真。真了，风再起，他也先立住十层这道坎。

邻床少年梦里含糊念：「滚……滚……」韩泥睁眼听了半息，低声道：「醒了再说。梦里滚，不算。」少年翻个身，没再响。

他把藤筐底敲实两下，筐紧，管事明日少问；少问，窑温才属于他。温属于他，十层才不像偷来的名声。名声他不要；他要手稳，要粒正，要复测台上那一见。

风从窑口进，带着一点报名册墨味。味淡，却像钩：明日闭门再炼，粒要再齐一寸——齐了，风声再嘲，也嘲不过掌心里的正。正字一立，末席也配上台。

坛腹又温一线。温像说：芽儿清了，刘婆在后，叶汤更后——后不要紧，先把十层按实，按实了，复测那条路才走得通。
