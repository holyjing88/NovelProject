# 第四十五章 谣言浪

翌日卯时未到，外门已沸。

浴房昨夜止步三息，今日却传成「徘徊半炷香」。莫长春拄杖送药渣的本事，一夜之间被嚼成「老不正经达峰」——达峰的不是修为，是脏嘴。

丹堂最先炸。苏念慈正验一批培元丹，门外弟子压着嗓议论，议论里「浴房」二字像药渣里的砂，硌得她耳根发烫。她摔杯，瓷碎，声脆，满堂一静：「师兄岂是这种人！戌时止步不入，执法堂有巡者记，你们嚼的是哪一版？」

摔杯的手在抖，抖里不是怯，是怒。她慕师兄半生，慕的是丹在谁手里、心在谁那边，不是慕一张被赵玄机写好的脏稿。弟子噤声，噤里仍有人交换眼神——爱听脏的，脏的比三品金丹还香，香在「首座也护」这层戏。

执法堂，柳青鸢令禁谣。铁牌光冷，拍在案上，声洪：「再传浴房谣，按诬告论。」可铁牌能禁嘴，难禁心。人心爱听丑，丑传得比除魔符还快，快在赵玄机昨夜那句「够了」——够了，便好，明日全宗都会替他们嚼。

赵玄机立堂上，面色平，平里烧。他预想中的轰响没来：莫长春止步不入，戏缺了高潮，谣言只好靠舌补。妒火点着纸，纸边焦了，火却烧不进长老院门。他指节叩铁牌，一声，像把不甘敲实。

林挽月躲廊角，不敢看莫长春方向，只敢看自己的鞋尖。鞋尖泥未干，泥像罪——她昨夜捏着传讯符未眠，今日全宗嚼的，却是她「偶然路过」该看见的丑。她想近，又止，止得像怕自己一近，就要在堂上把话说尽。

杂役堂，顾小满浆洗，低头，指节白。半块灵石在怀，像又被人嚼成脏字。她抬头，冷眼扫过传谣者：「宗主裁过，再传，去执法堂。凭记，你也能记。」

传话者噤声，却嘟囔：「杂役也管门风？」

顾小满把册子往案上一放，心里记着时辰、地点、谁去过林苑外径——记比猜有用，猜比传有用。孙福旧党在角隅嘀咕：「老不正经达峰了……」顾小满冷眼：「达峰的是脏嘴，不是长老。」

莫长春小院，门闭，茶仍煎，煎得淡。门外弟子嚼舌，故意让他听见：「九月老头，浴房外……」

他开门，捧茶出，坐廊凳——不坐主位，借门外风声，笑：「他们说得对，老夫靠的就是这些人——靠他们传话快。传得快，才晓得谁真急，谁假巡。」

弟子一愣，骂也不是，退也不是，像一拳打在棉上，棉里却有杖，杖不点人，点得人心虚。茶淡，淡得像把浪当茶晾——晾透了，才好还。

苏念慈闻讯赶来，眼眶红，未哭，只道：「师兄……」

「不急。」莫长春道，「浪要起，也得起完。起完了，才好还。」

柳青鸢在远处听见「还」字，眸光冷，像把字记进宗主令里。霍镇山自镇妖司赶来，骂声洪：「脏嘴！夜袭在前，还嚼浴房！谁再传，先过我这一关！」

魏无涯在檐角，扇开一线，观气运，观谣言浪，低笑一声，像看戏看到最脏处，反而更静。他自语：「脏字达峰，堂上才肯翻。翻的不止浴房，还有赵执事袖里的妒。」

外门膳堂，谣言已分七版：有说莫长春在径外坐了半炷香，有说林挽月惊呼是故意的，有说执法堂巡者「恰好」路过是赵执事安排的。七版合成一种信——信的不是浴房，信的是局。局缺高潮，浪便替局补高潮。

苏念慈回丹堂，炉火旺，旺得像要把脏字炼净。她遣弟子去长老院送汤，汤温，温得像替师兄也担。弟子回禀：「长老不辩，只喝茶，还说靠我们传话快。」苏念慈闭眸，低声：「不辩，才是还。」

赵玄机午后召心腹，面色仍平：「止步三息，够嚼。嚼到堂上，才算成。」心腹问：「若苏首座摔杯……」赵玄机淡道：「摔杯是给脏嘴看，不是给长老看。脏嘴达峰，才好翻。」

林挽月午后罚役途经外门，听了一路脏字，唇白。她忽然懂：慕错可改，证字更重，重得像恩。恩还压在喉里，时未到，可脏字已贴在每个人眼里——贴在堂上，证人在后头，还字在袖里。

顾小满傍晚把今日传谣者名册抄进杂役簿，抄得像抄自己的胆。她不知戌时发生了什么，只知心口那跳，跳得像半块灵石在警。警的不是浴房，是局。

外门演武场边，两拨弟子争执：一拨咬定长老「老不正经」，一拨搬出执法堂巡者记录，说止步三息、未入门。争执里，脏字与真字绞在一起，绞得像绳——绳一头拴着赵执事的铁牌，一头拴着莫长春的杖。

周德海在长老院听闻，面色平，平里冷，像早就备好的稿。他低声对心腹：「谣言达峰，达的是脏，不是罪。罪要等堂上翻。」心腹问：「若翻的是赵执事……」周德海淡道：「翻谁，看证人肯不肯开口。」

霍镇山骂罢外门，又骂执法堂：「夜袭在前，你们还有闲心嚼浴房？嚼完了，去东脉磨刀！」骂声洪，洪得像把脏字也砍轻一寸。

苏念慈在丹堂廊下听见霍镇山骂，炉火旺，旺得像替全宗也担。她低声对弟子：「脏字炼不净，炉就熄不了。熄不了，丹堂先脏。」

赵玄机傍晚再召执法堂弟子，问：「今日传得最凶的是哪一版？」弟子答：「徘徊半炷香。」赵玄机冷笑：「三息变半炷香，好。传得越凶，堂上越好翻。」

林挽月夜里回林苑，未眠。她反复想莫长老那句「起完了，才好还」——还的是谁？还的是浴房谣，还是她手里那枚未碎的传讯符？符贴枕下，烫得像一块未落的铁牌。

顾小满夜里把杂役簿又翻一遍，添了一行：戌时，浴房外径，止步三息，巡者在。行细，细得像半块石硌深——硌深了，堂上才肯翻。

莫长春独坐廊柱旁，不坐主位，袖空，杖横膝。药渣本在袖，一念出塔，再落掌，渣温，温得像昨夜未送完的礼。他不辩，不追，只把谣言浪当成一笔将落的恩：浪起得凶，退得才快。

魏无涯夜来传讯，只六字：「脏达峰，证将出。」莫长春阅讯，点头，像谢提醒。他自语：「证字不在老夫嘴里，在肯翻的人喉里。」

柳青鸢批宗主令至夜深，笔尖一顿，落笔：「谣言达峰之日，执法堂加巡。」巡是真的，翻也是真的，真假要等证人开口才算。

丹堂夜深，苏念慈仍未熄炉。她反复想师兄那句「起完了，才好还」——还的不是辩白，是证人。证人还在林苑，还在廊角，还在罚役路上。她低声对弟子：「脏字达峰之日，丹堂更要稳。稳了，证人才肯开口。」

赵玄机夜回偏厅，指节叩铁牌，一声比一声轻。心腹问：「若莫长老明日仍不辩……」赵玄机淡道：「不辩，更好。不辩，像心虚；心虚，浪才高。浪高了，堂上才肯翻。」

顾小满把杂役簿收进怀，怀温，温得像半块灵石。她不知戌时三息与半炷香差在哪，只懂记——记清了，脏字日后见真。她低声：「达峰的是脏嘴，不是长老。」

霍镇山夜巡外门，刀未出鞘，声先出：「谁再传浴房谣，先过我这一关！」一句话，廊静半息，半息里像听见铁牌将要落案——落案不在廊上，在堂上，在证人喉里。

风大，脏字达峰，达峰像帖在每个人眼里。识海塔影一荡，塔意沉而不显，像万物可藏，藏得住药渣，也藏得住将起的还。他念：「浪退，恕要来。恕轻，换心。」

大因果还在堂上，像挽月反戈，像夜袭子时——谣言达峰只是小浪，宽恕才是大还礼的前奏。

浪起得凶，退得才快；凶浪尽头，挽月喉里那字，将熟未熟。
