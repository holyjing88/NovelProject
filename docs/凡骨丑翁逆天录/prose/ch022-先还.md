# 第二十二章 先还

新谣第三夜，雨意无，风硬。

风硬如刀背，刮茅檐，刮得漏雨处更漏。漏不要紧，要紧的是人别漏——别漏恩，别漏还，别漏「先还谁」。

韩泥臂结痂，袄肘叶线密。密得像心事的针脚——针脚不亮，亮了就招眼。招眼，叶家更危。

他独坐灶前，米汤稀，稀得见底。底空，人不空。空的是还不起，不是不想还。不想还，心不会热；想还，叶汤才会深。

心口四恩在：叶汤、芽粥、陈袄、张馒，衣纹一捺。赵辱在，仇苗叉。乱而有序，像穷家粮柜——柜少，粒粒真。真粒饿不死人，假粒撑不过冬。

他摸心口，摸得久。久，像摸一碗汤的边。边烫过，边还在，在就欠，欠就得还。
「先还叶。」

他低声，像对自己说，也像对坛说。

「芽儿粥是点，陈姑袄是撇，张麻婆馒是圈。都真，都暖，都该还。」

「可汤是头的。」

就这些。没有豪言，没有天誓。老头口吻，够还，不怨。怨是少年，他不是。

为何先还叶？

因五十八年，无人送汤进门缝。因谣起，她仍送姜，仍补肘，仍塞「别念」。因恩不是可怜，是人该送；人该送，就不该被血祠谣咬碎。

因还字比谢字重。重了，就得活到达得到还的那日。那日也许远，远也要走。走路靠筐，靠手，靠叶汤上那一点深。

他想起张麻婆双馒，想起陈姑棉袄，想起芽儿狼口后那包药叶。四恩都在，序不能乱。乱序，还起来就薄——薄了，不如不还。不还，心就冷；心冷，叶汤就浅。

他低声道，像对空坛：「坛凉不要紧。心热就行。」

坛不答。答的是腹里一线温，温得像应「先还」二字——应半分，够刻进骨。

他想起门墩纸角「趁热」，想起姜辣冲鼻，想起袄肘针脚。想起隔窗那夜，他一句没说——不说，是护名；护名，也是还的一种，还她清白，清白比丹难炼。
他低声道：「先还叶。」

不深，狠。狠得像把「趁热」刻进心骨。心薄，骨要硬，硬了，才压得住赵辱，压得住榜末名，压得住明日公审的潮。

心口一紧，紧得指腹发热——比十月初那夜更烫，烫得像叶汤在心里醒。叶汤旁那点旧恩，被新念浸，浸成深红，像一枚印，印的是「先还」。

叶汤加深，深得像灯芯又捻紧一分。紧一分，亮一寸，亮给自己，不亮全村。全村明日才看。

坛角仍凉。坛腹更温。温贴胸，胸不跳快，跳得稳。稳，是老头能给的勇。勇不炸，勇记，勇候。

门外，叶家方向狗吠一声，吠急，像有人经过。韩泥不追。追，害。他只合袄，塞怀，腰上硌，硌成一句：

「我记着，以后还。」
天将亮，村道有人咳，咳得远，像沈家。像芽儿。咳声弱，弱得人心紧，紧完又松——松因人还活着。

韩泥开门缝望，雾薄，旱气仍在。仍在，也要活。活要粮，粮要筐，筐要藤，藤要手稳。

他绕膝勒藤，勒指，指疼，疼得清醒。清醒里，他知：榜末能忍，谣能忍，公审将至，不能忍的是叶家秤被人砸了。

「先还叶。」他又道，声更低，「还了，再还别的。别乱了序，序乱了，恩就薄了。」

心里叶汤加深。加深到今夜，像灯，灯不亮全村，亮他一条路。路通向明日祠堂，通向叶青禾开口的那刻。

他想起赵绣春泼的脏水，想起赖扒皮短签，想起榜末七两。想起越多，叶汤越稳——稳不是忘辱，是辱候着，恩先还。

他想起贴胸「趁热」，像又要滚一回。滚不得，滚就成血祠谣里的「送汤引旱」。他不滚，只按，按得心口微潮——潮是坛应，应半分，够他扛过公审前夜。

潮是释。释不在喊，释在叶汤深一分，深得像灯芯又捻紧——捻紧给自己，不亮全村。全村明日才看。

「汤是头的。」他第三回道，像盖章。

指腹贴心，贴得烫。烫从心入腹，腹不燃，腹蓄。蓄到明日，只一句够：我记着，以后还。

叶汤再深一分，深得像丑时未到，坛先应半分。半分够，够扛公审前的夜。夜长，长正好把「先还叶」刻实——刻实了，就不怕赵绣春明日尖嗓。

门外风硬，硬得像有人贴墙念：血祠开，雨下来。念罢又散，散成空。空不怕，怕的是人信。人不全信，公审就还有机会。

机会在叶青禾的秤上，也在他叶汤上。两处都稳，韩泥便合袄，贴胸。贴胸，像贴一张不会退的契。

他又想张麻婆双馒，想陈姑棉袄，想芽儿狼口后那包药叶。想一遍，不是动摇序，是记全——全了，还时才不亏心。亏心还恩，恩就薄；薄恩，不如不还。

「还，就还足。」他低声道，「足不了，就先还头一笔。」

头一笔是叶。叶在心上，心在瓮边——瓮仍凉，凉不妨，坛已温。温是路，路通向炼丹，通向省亲，通向门缝再送一碗不烫的汤。

序定了，心就不乱。乱序，四恩就薄；薄了，还起来像施舍。施舍不是还，还是记，记到能还足。

那路太远。远不要紧，五十八岁，远也走。

门外风硬，硬里有人咳，咳得像沈家。韩泥不开门，只隔缝问：「谁？」

张麻婆嗓粗：「我。送半块饼。别饿死在还字前。」

韩泥：「不饿。还字在前头，饿不着。」

张麻婆：「先还叶，对。叶那碗汤，头一笔。」

韩泥：「嗯。」

张麻婆顿半息：「明日公审，你少吭声。吭声，赵家高兴。」

韩泥：「知道。坐着听。」

张麻婆走远，远里又喊：「还字刻实了，再还别的！」

韩泥低声应：「刻实了。」

应完，按心口，叶汤深一分，深得像释——压是谣，释是先还；先还定了，明日公审就不乱。

他再道第四回，声几不可闻：「先还叶。」

四回，像钉，钉实了，就不拔。拔了，序乱；序乱，恩薄。

韩泥独白「先还叶」，声低，字实。实，像立誓——誓不挂嘴边，挂心口。心口一紧，紧像七笔里第一笔抬头。抬头，不见人，见烫——烫是规矩。他不求灵根，不求名，求还。还，先还热。热凉了，恩就假。外头谣还在，在远，像风。风硬，心更硬——硬不是怒，是记。坛沿温，像爹的手。手在，根在。根在，才配说「先还叶」。说了，就不改。改，像欺恩。先还叶，三字落心，像立轻誓。誓不挂嘴边，挂心口——心口一紧，紧像七笔抬头。抬头不见人，见烫；烫是规矩，规矩是还的路。外头谣还在，在远。远，像公审前的潮——潮退之前，最后还要蓄一记辱，蓄一记恩。恩辱分列，不混；混了，这三字就轻，轻了，叶丫头白送汤。先还叶，誓落心。心紧像绳——绳捆手，也捆舌；舌留来还，不在辩。誓落心口，像把「先还叶」三字钉实——钉实，就不改；改了，像欺恩。

先还叶，誓落心口。根在，誓就不改——改了像欺恩。欺恩不做，只做记心翁；等锤，等热，等还，还先还烫，烫凉了，恩就假。

先还叶，誓落心口。根在，誓就不改——改了像欺恩；欺恩不做，只做记心翁。

誓落心口，根在，誓就不改。改了像欺恩；欺恩不做，只做记心翁。

他低声道：「手稳，恩不断；恩不断，汤就不凉。」

先还叶，誓落心口；韩泥按坛：根在，誓就不改——改了像欺恩；欺恩不做，只做记心翁，等锤，等热，等还，还先还烫。
