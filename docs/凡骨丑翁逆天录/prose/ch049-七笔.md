# 第四十九章 七笔

四日后，卯时前，丙九棚静。静，像栖香谣讲完后，谣少了半，半留着，留着等正测规矩。

韩泥未睡实。实睡，像认输；他是记心翁，恩仇都记在心里；夜里在心里盘点——七笔，分列，不混。泥岗六笔，他低声念，声哑，只对自己：

「一，叶丫头热汤，藏姜。热，记着，以后还。」

「二，叶振东赠药，辞村暗送。药在篓，路在省亲那日。」

「三，陈姑棉袄，冬不冻。穿活了，心里记着。」

「四，张麻婆馒头，饥时顶一口。骂是邻，馒头是真。」

「五，沈枯芽稀粥，芽儿怯，恩也真。真，不混邪。」

「六，老耿修瓮补漏，屋不漏，坛在。坛在，根在。」

念完，他停半息，按心口。心口六笔沉，沉得像泥岗村那口井——井深，恩在底。念到第一笔，他眼前浮一线热气，像门墩粗瓷碗沿。碗沿烫过指，烫得真；真，才记得住。叶丫头没问灵根，没问丑，只问烫不烫。问烫，像人该问。该问的恩，不能还凉。

念到第五笔，眼前又浮半张怯脸，像沈枯芽递粥时耳根红那一线——红浅，恩不浅。怯真，记着，不混邪。

他又念辱线，只对自己，声更低：「赵绣春泼，周嫂谣，赖扒皮克——辱记着，不混恩。混了，心就乱。」

宗门第七笔，另念：

「刘婆，灶边粥，肘上布。粥稀，热在。布丑，挡风。」

七笔齐，齐在心，不在册。册废了，心还在——还在，才还得上。还得上，先活四日，先听规矩。

他低声道：「泥岗六笔，宗门一笔。分列，不混。混了，心就凉；心凉，汤就还不了热。」

邻床问：「你嘀咕什么？」

「心里盘点。」韩泥说，「恩在心里，人才能活。活了，才炼丹。炼丹，先还热的。」

「你还不会炼丹。」

「不会，学。」韩泥独眼平，「学一日，记在心里一日。记满了，就还。先还叶丫头汤，热着还。热着还，才配七笔。」

邻床懂「热」，闭嘴。外头风硬，柱上条旧，字丑。丑字他熟——熟像木牌，像记功，像坡下那声锣还没响。他摸枕下帖，帖角粟壳香淡，淡得像栖字合全。合全了，谣少半；谣少半，手仍要紧。

天微亮，刘婆端粥，稀，热：「记心这日，多喝一口。一口，顶一日克扣。」

韩泥喝尽，粥热在肚。她多舀半勺，半勺，热在。赖福远处骂：「疯婆子又偏晦气！」

刘婆回骂：「晦气在你秤上！秤歪，粥才稀！」

韩泥不掺。掺，费舌；舌要留来辨香。

辰时，丹房外廊，辨香半刻。香囊三只，韩泥闭息，鼻先静，报香一字不落。铁无言点头，不重复问，只翻簿——簿上有印，印在，路在。有一只囊口微腥，腥在底，不在温。韩泥眉心一沉，报得慢半息，仍一字不落。落字，像把栖香谣又剥一寸。

秦霜立廊角，声不高：「坡下规矩讲清后，仍廊下辨香。香在，手在。」

韩泥应声，独眼平。铁无言收簿前停半息，声低：「七笔，你数得清？」

「清。」韩泥说，「泥岗六笔，宗门一笔。分列，不混。」

铁无言不夸，只指廊角：「四日后坡下讲规矩。规矩开，验香才开第二道。第二道开了，才谈凝丹。」

韩泥只答：「先还热的。热在叶汤，不在舌。」

铁无言点头，灯远。远，像把「先还」二字钉在验簿边——钉在册，也在心；心在，四日条才验得过。

午后，韩泥仍搬柴。同批少年递绳：「我帮你捆。正测前，手别抖。」

韩泥只答：「捆。」

绳结实，实得像七笔——笔笔在，命在。赖福路过，瞥他一眼，哼：「末排晦气，捆柴也捆不出丹。」

韩泥肩不动：「捆得出命。命在，丹才学得。」

赖福骂一句，走了。骂不重，重在韩泥没回头——回头像认谣，谣借腥，他不碰。

同批有人低声：「七笔，你真数得清？」

「清。混了，坡下站不住。」

那人闭嘴，像被「分列」堵住。

黄昏，韩泥对坛，声低：

「等我能炼丹了，先还叶丫头。」

坛腹微温一丝，应「还」字，不应急。急像少年，他是记心翁——翁可以慢，恩不能凉。他按心口，六笔沉，一笔热。热在刘婆粥里，也在叶汤记忆里——记忆里姜还在，就得还烫。烫，是规矩；规矩，是还的路。

邻床问：「坡下讲规矩，你怕不怕？」

「怕。」韩泥说，「怕也站。站紧了，手就不飘。」

「飘了怎么办？」

「飘了，恩断。」韩泥说，「恩断，汤就凉。」

邻床闭嘴，睡了。

夜，管事贴新条，条压木牌边：

「四日后卯时，坡下讲正测规矩。丙九末排，韩泥到。误者，滚。」

韩泥摸帖，又摸心口。心口七笔，帖上四日，木牌在怀。他低声道：「四日，听规矩。规矩懂了，末排先验才站得住。」

秦霜声从远廊来，不高：「四日后，坡下见。见完规矩，仍廊下辨香。」

丑时，坛腹又温一线，温短，应「烫」，不应慌。慌像少年，他不慌。他低声道：「趁热。记着。等能炼丹，先还叶丫头。碗要烫手。烫手，才配七笔。」

管事门外咳：「四日条贴柱了。末排先验，别误卯时。」

韩泥只答：「不误。误了，七笔断。断不得。」

刘婆门外又咳一声：「记在心里别记凉。凉了，手抖。」

韩泥只点头：「不凉。凉的是谣，谣不贴胸，贴不得。」

外头风硬，风里再无栖香全句。全句散了，散成坡下一声锣——锣未响，响在心里。心里响，手就不飘。他摸破布角上三字，字旧，真。真在「必还」，真在「不混」。不混，七笔才站得住四日。

邻床梦呓：「热……」

韩泥不答，只按心口。心口七笔，六笔沉，一笔烫。烫像叶汤，像刘婆粥，像坡下即将落下的那声锣。

丑时将至，他念第七笔又止——止在「肘上布」，布丑，挡风。挡风了，臂能抬粪、能捆柴、能按石；能按石，正测前才不滚。坛腹微温一线，应「七」，不应慌。

卯前，他摸柱上四日条，条角起霜，霜化字更真。真在「坡下讲规矩」，真在「末排到」——到了，先听，不抢；听了，验香才开第二道。

同批少年滑半步，韩泥伸手扶肘，肘糙，扶紧就收——收，像把第七笔按实。

四更将尽，柱上风歇。风歇，他心里七笔又齐一回——齐在胸，不在嘴。帖在枕下，温短在鼻。都在，四日后坡下见。

辰时廊下，第三只囊口微腥，他报得慢半息，仍一字不落。落字，像把栖香谣又剥一寸——剥在鼻，不在舌。

他想起老耿补瓮那夜屋不漏；不漏，坛在，根在。根在，四日后才听得进规矩——规矩懂了，验香才开第二道。

丑时将尽，坛腹又温一线，温短，应「烫」，不应慌。七笔齐在胸，四日后坡下见——见完规矩，仍廊下辨香。

外头风硬，风里再无栖香全句——全句散了，散成坡下一声未落的锣。心里响，手就不飘；手不飘，恩就不凉。

邻床梦呓一声「热……」，韩泥不答，只按心口。必还在，还不急；急在别凉——凉了，坡下站不住末排。

破布角「必还」三字，旧，真。真在还不急，急在别凉——凉了，七笔断；断了，四日后坡下站不住末排。

丑时，坛腹微温一线，应「七」，不应慌——七笔齐在胸，帖在枕下，四日后坡下见。

辰时廊下，第三只囊口微腥，他报得慢半息，仍一字不落——落字，像把栖香谣又剥一寸；剥在鼻，不在舌。

四更将尽，柱上风歇。风歇，七笔齐在胸，帖在枕下——都在，四日后坡下见，见完规矩，仍廊下辨香。手不飘，恩就不凉，路就不绝，命在。
