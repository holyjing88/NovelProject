# 第五十八章 血线将至

试手日，卯时前。韩泥未睡实。夜里在心里盘点七笔。分列，不混。

符怀暖一丝，坛腹温一线。一线，像瓮仍眠。

坡下，秦霜最后一遍：「手抖，丹歪，滚。丹歪不要紧，手不能歪。」

试手凝丹，只验手，不验灵根色。色末，手抖才滚。

卯时前，坡下霜白。霜贴石阶，石阶滑，滑不要紧，站要正。正，才验得过手。

韩泥立末排，独眼平。铁无言验鼻：「静带到手上。」

同批少年抖肩，抖像怕「滚」字。韩泥不抖。抖是心飘，心不飘，手不飘。

丹炉前，他手不抖。手正，丹形丑，丑而正。正，像辨香那日鼻下静。

执事记功，不记灵。管事只点头：「麸皮不克。」

赖福脸青，退半步。韩泥不喜，不喜像少年。他只记：手正了，路又宽半分。

同批少年碰他袖：「你真过了？」

「过了。」韩泥说，「过的是手，不是灵。」

「灵根末，手正，正就够。」铁无言声低，只对他。

秦霜目光扫过队尾，声冷：「再嗓，记名。」

队散。赖福啐地，啐得虚。韩泥不回头。回头像认谣，谣借腥，他不碰。

坡下霜白，柱上无新条。他摸木牌，与记功在并放。并放，像命与规矩挨着。

秦霜问心时，他声哑，声平：「为还。还，先还烫。」

秦霜不夸，只点头。

丹形丑，丑而正。脸丑，手正，正就够。试手过了，路宽半分。宽，不是灵根好，是七笔没断。

午后，清西栏半刻，粪锹沉，掌裂不僵。符暖护骨，骨不寒，手才正。正，编筐才不偏。

编筐补单，藤刺深扎，扎透旧茧，血珠饱满。

他甩手，血甩向席盖，席吸了，血未至沿。未至，像血仍不渗沿那几日。

他盯掌，盯席，盯坛角。角静，沿净，净，像瓮仍眠。

他低声道：「血来了，沿不接。不接，不是怕，是等缘。」

刘婆端姜汤，姜薄，热：「试手过了，别傲。」

「不傲。」韩泥说，「傲，符自灭。」

管事路过，验手：「手抖否？」

「不抖。」韩泥说，「抖是心飘，心不飘，手不飘。」

他盯席边，席边湿一线，湿不到沿。不到，像瓮仍眠。

铁无言路过，声低：「手正，沿净。净，像等缘的人。」

韩泥只答：「等。」

等，不是拜血祠，不是主动滴血。是活计扎，血找席，席挡沿，沿等韩氏守瓮血那一误。

血到了席，没到沿。沿在眠，血不主动碰，这是铁规。

止在沿前，止得像道线。线内席，线外沿。

管事贴条于柱：「试手记功在。记功，麸皮不克。」条薄，真。真在「手正」，不在「灵色」。

试手后，他仍末排。编筐补刺，藤刺浅些。浅些，血线可控。

管事验手：「试手过了，手还抖否？」

「不抖。」

他独眼平，手不抖：「血线将至，沿不接。」

搬柴过半，肩正。正，像坡下练出来的。练出来，才配接后头血线。

老耿挑水路过，水洒实：「试手过了，沿还净？」

韩泥只答：「净。血在席，不在沿。」

编筐第二单，藤刺再扎，血珠又起。他甩手，血落破席，席厚，吸得尽。吸尽，沿边仍无尘。

他按席，席厚，沿净。净，才接得住瓮醒那一误。

管事验沿，烛照一线：「脏一指，像拜邪。」韩泥用旧布再擦，擦得沿滑，沿尘，沿上无血。

赖福远处骂：「试手过了也晦气。」管事声冷：「再嗓，克半月麸皮。」赖福哑了。

邻床问：「你手咋了？」

「扎藤。」韩泥说，「血在席，不在沿。」

「沿是啥？」

「沿是坛边。」韩泥说，「沿净，才接得住缘。」

他盖席，盖严，严像藏私。私是坛，坛是命。

秦霜远廊经过，一句：「手正，心别邪。邪，符灭。」韩泥心记着七笔，不邪。

夜，符怀暖，药涂裂口，血止。止在沿前，止得像规矩。

坛温加剧，像将醒未醒。醒在缘，不在人催。催，像拜邪。

丑时前，沿仍净，手正，心记着七笔。他按席，席厚，厚像守了六十日的席。

他低声对坛：「血在席，不在沿。沿在眠，我守席，守沿，守不主动滴血。」

坛腹温一线，又凉。凉，像应「等」。

他低声道：「血线将至，沿不接。不接，才接得住瓮醒那一误。」

同批少年蹭袖角，声小：「血在席，会脏坛吗？」

「席挡着。」韩泥说，「挡到沿前，沿不接。接，像主动滴。」

少年不懂，却走了。

符怀暖，药涂裂口，血止沿前。止，像道线。沿在眠，等韩氏守瓮血。

窄路也是路。路通向瓮醒，瓮醒通向还。还，先还烫。

小兽栏角，小兽咳得细。他多铺干草，草旧，旧能暖。暖兽，像冬里少死一人。少死一人，恩便多记一笔。

他撕旧袄角，补席边破处。补罢，席更厚，厚像后头那一夜。席厚，血好找席；席好找，沿才好接沿前那一止。

他摸记功帖，帖边磨毛。毛得像心口七笔。笔笔在，手才不飘。

试手过了，他仍末排。末排不要紧，要紧是沿净，手正。

血线将至那日，他独眼平，手不抖。抖，缘便偏。偏，席挡不住。挡不住，沿便脏。

沿在眠，血不主动碰。

沿不接，缘在等。

他低声：「等。」

血线将至，手仍正。沿仍不接。不接，缘才等得久。久，不靠催，靠席挡。挡在沿前，不在舌上。舌上急，手就飘。飘，像借腥。

五更前，他数更鼓。更鼓沉，沉里掌贴席边湿痕，湿痕止于沿前三指。三指外，沿尘，沿滑，沿上无血。无血，才配等瓮醒那一误。误在活计，不在舌。

丑时末，他掀席角看湿痕。湿痕止于沿前三指，三指外沿尘仍干——像血已至席边，却在沿前认了规矩，不肯再进半步。

同批少年翻身，梦中喊「滚」。韩泥不扰，只低声：「滚在舌上，不在沿上。」沿在，手才正。

坛腹温比白日沉半分，沉完又浮回来。浮回来不要紧，要紧是席边那一线湿痕，湿在沿前，不在沿上——像瓮在眠里试他守不守得住。守得住，缘才近。试手记功帖边，他指节白一息，白完又平——平，像坡下那炉丹形丑而正，正在掌，不在灵色。

执事收帖，末排名在尾。尾不要紧，帖上有「手正」二字，才真。真在记功，不在灵色。

他补席边第三处破角，旧袄角厚，厚像又多挡半分血路。挡到沿前，沿不接；不接，误才来。坡下炉灰冷，冷像试手毕了。毕了，活计才开始——藤刺还会扎，血还会找席。席在，缘在；缘在，七笔就在。

试手后第一夜，他多铺一层干草在席下。草旧，旧能吸潮；吸潮，裂口才不渗。渗了，明日藤刺偏；偏了，血落错地方，缘就薄。薄不要紧，手正才要紧——正，是坡下那炉丹形丑而正教出来的。

刘婆又端半勺姜汤，热：「试手过了，别空肚。」韩泥喝尽：「不空。姜在记里。」姜在记里，裂口才不僵；不僵，编筐第三单藤刺才不偏。偏了，血找错席，沿先脏。脏沿，像拜邪；拜邪，路断。他不拜，只盖席，盖严，沿净。

试手帖收进管事册，册边多一行「手正」。行丑，真在印——印在，麸皮不克；克粮，克不走还路。还路，先还烫。烫在叶汤，不在舌上。

铁无言夜巡，灯停坛角半息：「血在席，不在沿。沿上若有痕，像主动滴。」韩泥只答：「痕在席边，止沿前三指。」

他独眼平，低声道：「血线将至，沿不接。」

管事夜巡验沿，烛照一线：「席边湿，沿仍净。」韩泥只答：「净。痕在席边，止沿前三指。」沿净，才配等瓮醒那一误——误在活计，不在舌；舌若急，沿先脏。

编筐半刻，藤刺扎掌，刺浅，浅些沿才净。沿净，坛才藏得住。
