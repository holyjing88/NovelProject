# 第六十一章 丑时坛温

丑时，丙九棚静得能听见自己咽口水。邻床鼾声忽然断了一息，又接上。韩泥没睡，盘坐床沿，油灯豆光照坛角。

坛角盖着破席，席严，怀底那张凡符却暖了一丝。暖得像半闪，不燃——燃了冒烟，管事夜巡要查棚。查棚查出邪，晦气二字又会贴回脸上；贴回不要紧，席掀了才要紧。

他掌按席面，觉得坛腹那一线温比前几夜又沉了一点，像心跳，像有什么在瓮里翻了半个身，又缩回去继续眠。眠着就安眠。安眠着，他才少惹查。

「你再眠着。」他对坛声极低，「席盖严，旁人少来烦。烦了像邪，守罐的人手正。」

坛只颤极轻，轻得像远处有人应了一声。他不揭席炫宝，也不拜邪求什么。求像赖福，守才像韩家擦坛的人。

白日编筐半刻，末藤硬，他偏了下手，刺浅些。浅归浅，血还是出来了。血落席上，席吸了，沿不接——跟沿前即止那日一样，止在席边，不在沿上。

邻床翻身，梦里含糊念：「温……」

韩泥盖紧席，声低：「你看错了。睡。」

掌茧上还涂着凡药，药凡，手却正。手正了，明日藤刺才不漂；不漂，血才找席，席才挡到沿前。符在怀底，暖一丝，够扛丑时寒，却不燃。燃，烟起，管事查；查了，就容易被人说成借腥。

铁无言的脚步在门外远了又停，停半息，只对他一人说：「符影半闪，别燃。燃了，管事查。」

「不燃。」韩泥应，「席严，才不像邪。」

脚步再远。门缝透进一线风，窄，却够人守这一更。刘婆在门外咳了咳：「丑时寒，别空肚。」

「不空。」韩泥道，「姜在记里。」

鸡未鸣，狗未吠，夜禁靴声远去。三年约、钱戾衡那些名字都还在更远的时辰——时辰未到，瓮先温，温而不醒。丑时将尽，温一线又敛回去，像瓮又眠实了。眠回去不要紧，要紧是下一夜席盖更严将至。

他按掌，掌贴席，席贴坛角。角温一线，短，像廊下「香」字、像凡符「暖」字，更像灶下那个「热」字。热在恩里，不在舌头上。外头风硬，柱上无新条；条没有，他心里却有一条：沿要净。

「丑时坛温加剧了。」他低声道，「加剧，我也不拜。拜了像邪。」

坛腹温短，像应了个「候」字。六十日席盖严实。他不多嘴——嘴急手飘，飘了赖福就有话说。

秦霜远廊经过，声轻得像怕惊醒人：「心别邪。邪了，符灭。」

符在怀底，温一线，记着七笔不误。油灯豆缩了，沿上却浮出半瞬符影，又敛。半瞬过去，他只当眼花。眼花不要紧，席要严。

同批少年咳嗽醒了，眯着问：「你还不睡？」

「丑时守棚。」韩泥说，「守着，不招邪。手正就行。」

少年翻个身又睡了。天边白了一线。他低声对坛：「丑时尽了，温还在。席仍严着。」

坛颤极轻。木牌塞枕下，跟席严并放。

老耿门口挑水，水洒实，声哑：「前夜别拜。拜了，路断。」

「不拜。」韩泥应，「席严着。」

耿不再言，脚步远了，像冬深处那一份赠还在候着。候不要紧，要紧是席不露。不露，符影半闪才说得过去。符影再浮一息，又敛——像提醒他下一夜席盖更严。

管事夜巡经过棚外，靴声沉得像钉钉子：「丑时守坛，别生事。生事，连坐丙九。」

「不生事。」韩泥应声，「生事在嘴上，不在沿上。」

靴声远了。铁无言远处咳一声，像催手稳：「手别僵，符才暖。」

棚外雪粒打瓦，细响像有人远处候着。邻床又梦呓「温」，他盖席，声更低：「温在符，不在坛。」说完，怀符暖一丝，坛腹却仍只温一线——各守各线，不混。同批少年又梦呓「红」，他盖席：「红在符，不在坛。睡。」

刘婆第三声咳：「丑时尽，席还严？符还暖？」

「净。」韩泥应，「暖一丝，不燃。」

铁无言提灯远过棚角，灯晃席角一下，嘱半句：「席盖严。严了，夜巡少烦。」

韩泥应声，不追问。问急了像借腥；守，才像韩家擦坛的人。他又加一层破布，布旧，更能挡眼。刘婆第四声咳得更低：「别拜邪。拜了，粥也救不了你。」

「不拜。」韩泥说。

他把木牌翻过来，背面糙得像心口七笔——笔笔按实。正了，才配接席盖更严那一夜。

棚柱外有人磨刀，磨声短，短得像一夜还没过完。他把心口按实：叶汤要热还，耿恩要冷记，坛温要守，三件不能混。混了手飘，手飘符灭，符灭沿污——污了，下一夜席严也白加。

沈枯芽娘在远处咳两声，咳轻，像怕扰丑时守坛。咳不要紧，芽在，叶家秤在，汤就不凉。他按心口，低声道：「丫头那碗汤，热着还。还，要等日子到，路才长。」

路长之前，先得过席盖更严那一夜。耿恩在底，叶汤在胸，分列，不混——混了，心就乱；不混，手才正。

同批少年又翻身，嘟囔：「你油灯晃，晃得人烦……」

「灭了大半。」韩泥说，「剩豆光，够守席。」

少年不答，鼾声接上。韩泥独眼盯席角半息：席严，罐藏实。油灯豆爆一粒，粒灭，他只当眼花。眼花不要紧，拜了路断；不拜，路才长。

坛腹温仍加剧，他不当一回事——当了就容易拜。他偏不拜。心躁了，手就飘；手飘了，符影半闪也会被看作邪。席盖严着就够。

白日那掌茧涂过的药，夜里仍涩在裂口。涩着也好——提醒他手要正。手正了，藤刺才正；藤刺正了，血才往席上落。落在席上，旁人少说话。

他把第七层破布角又掖紧一分，掖得像给坛戴冬帽。帽丑，厚才真。真在能吸活计血，不在藏宝。邻床第三声梦呓「温」，他盖席，声低到几乎听不见：「温在守，不在嚷。睡。」

这一更，只守席，只守席盖严。风息半更，席盖贴实。下一夜席盖更严；严了，赖福少掀。

油灯豆灭了。灭不要紧。席严，符暖，手正。天边白一线更深。他望坛，独眼平，低声道：「手在，路在。下一夜，席盖更严。」

坛不应字，只温。他守完这一更也就够了。更鼓尽处，他仍按着席角——按住了，明日的严席才有得加。
