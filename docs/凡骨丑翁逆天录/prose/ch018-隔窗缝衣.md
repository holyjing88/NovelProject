# 第十八章 隔窗缝衣

村会后第三夜，风硬。

韩泥早起编筐，藤条绕膝，臂布勒得紧，一勒就疼——疼着也好，至少提醒他，村会那日不是做梦。野藤刮皮、泡软，再绕；结松了筐就塌，他结打得实。

叶青禾夜里穿针，针尖对着自己指腹，不对着人。对着人，就成了勾引；对着自己，只是补肘。肘破在袄上，袄在村西最漏那户——户里人不喊她名，只喊晦气翁。晦气翁肘破了，风像针一样往里钻；她补，不是可怜，是人该补。

线过去，针脚密，密得像药铺的秤——秤药也秤恩。恩轻，秤出来不重；不重，才真。她把灯芯捻紧一分。捻紧不是怕黑，是怕谣。谣说寡妇送汤就是邪，邪字贴门，贴得比旱天还干。她偏不贴字。她贴的是针脚：针脚不写字，只挡风。

袄是旧的，陈姑走了，袄还在。还在，就该暖人。暖不必进门，进门就坐实「一对」。一对是脏名，脏名比破肘更漏风——她懂，所以隔窗送，搁门墩送，送完就走，走得不回头，像从没发生过。

韩泥编着筐，独眼偶尔往巷口瞥——不是望，是怕。怕有人骂叶家，骂到灯灭，骂到铺关。门缝外脚步轻，停得稳，像叶青禾一贯的站法：不进门，不越线，线外也能送恩。

他没开门。开，坐实牵扯；不开，恩也能落下。布包搁上门墩，轻响一声。叶青禾嗓子很低：「袄破了。我补了肘。」

就一句。没有「你冷」，没有「别冻着」。她懂他怕什么——怕的不是冷，是名。名一脏，汤也脏，姜也脏，针脚也脏。门内韩泥应一声：「够了。」

够了，不是谢。谢字轻，还字重。门外她停半息，没再问。问多了像求。她不求。只把针收进布包——布包旧，角磨毛，像她这些年的名声：磨毛了，仍包得住针，包得住药，包得住「人该补」三个字。

门内他不开，她也不催。催像逼。逼恩成债，债就薄了；薄了，不如不送。

布包在门墩上停着，像一块不肯进门的暖。韩泥隔着门板听见针线匣合上的轻响，响完就静。静得他指节发白——白不是怕赵家，是怕这份暖被人踩脏。他低声只给自己听：「肘补了。名我护。」

巷口忽然尖起来，尖得像盆泼剩水。赵绣春嗓子先到：「半夜缝绝户翁的袄？寡妇还要脸不要？」

周嫂帮腔，棒槌没拿，嘴比棒槌狠：「药铺不开，倒有闲针！晦气沾衣，洗都洗不掉！」

叶青禾在门外，没辩。辩，赵绣春更来劲。韩泥隔窗缝看见她侧影——青布素净，肩绷着，手里还捏针，针尖仍对着自己指腹。

他喉结动了动，想开口。想说：汤是热的，袄也是热的，热不是邪。开口，叶家就多一口「一对」的脏名。他咽回去。咽得像咽刺。刺不进谣，进心里。

赵绣春逼前两步，绣花鞋踩上门墩尘：「说！图什么？图他绝户给你送终？」

叶青禾抬眼，一眼很平：「图肘不漏风。」

「漏风？」赵绣春冷笑，「你漏的是德！德漏了，药也卖鬼！」

叶青禾没退：「德在秤上。秤准，药就不鬼。」

赵绣春噎了半息，转尖：「秤准？你秤的是绝户翁的袄！袄晦气，秤也晦气！」

周嫂啐地：「明日公审，看你怎么辩！」

叶青禾收针，针尖仍对自己：「公审审谣，不审针脚。针脚挡风，不挡人嘴。」

就一句。不多。多了她更来劲。叶青禾走，走得稳，稳得像秤砣落盘——盘沉，声轻，轻里不让步。

赵绣春还想跟，被周嫂扯了一把。周嫂看四周人影散了，胆也缺了：「明日公审再说！今夜嚷白了，里正记名！」赵绣春啐一口，绣花鞋跺尘：「记就记！绝户翁的袄，迟早脏她一手！」

门里韩泥听见了，牙关咬紧，还是不开口。开口像替她认「一对」。他宁可把那句骂当辱记下来——记清了，锤落下才不瞎。

门墩上布包还在，像没人敢碰。韩泥开门一尺，取包。肘里细密针脚，密得像叶家秤——秤人，也秤恩。恩轻，才真。

他指尖抚过针脚，热。热不是袄，是人手留下的。五十八岁见过冷，仍有人肯深夜穿针，针脚就是话。隔窗那刻他一句没说。不说不是认辱，是护她名。名比嘴贵，贵就得忍。忍不是怂，是老头能选的勇——勇不炸，勇记，勇候还。

进屋后，心里叶汤旁又添了一捺衣纹——不另立一笔，只让汤旁多一线布面。布面能挡风，挡不了全村嘴；嘴明日再说。「肘不漏。」他低声道，「心也别漏。」

他把袄在灯下照看一眼。针脚细密，没有花俏，像叶家抓药时的手——准，不废话。准比漂亮管用。他摩挲肘弯那一道线，线温还在：「这热，明日公审上我不讲。讲了脏你。」

他穿上补袄，暖进骨里。暖是情，情不出口，出口就浅。浅了像施舍，深了像欠。他宁可欠。他把袄勒紧，像把「先还」勒进肘里。勒完，又编筐。

筐稳，手稳，心也得实——实到明日公审，实到叶家秤还在。巷尾赵绣春声又起，起得远，像不甘心：「补袄？明日看里正怎么判！」

韩泥独眼不抬，藤条绕膝。谣尖，他就不尖——尖了，叶更苦。小胜落在肘上：肘不漏，风不进；风不进，人还能编筐。筐卖出，米买来，命活着，恩才能还。

张麻婆路过，瞥他窗缝灯，嗓子粗，像故意让巷里听见：「补了？谁补的？」

韩泥在门内：「叶家。」

张麻婆：「叶家胆肥。胆肥好，胆肥的人，药才真。」

韩泥只答：「在。」

张麻婆顿半息，又道：「明日公审，你少吭声。吭声，害她。害她，恩就白记。」

韩泥：「知道。坐着听。」

张麻婆啐地向赵家方向：「针脚挡风，嘴挡不了。嘴挡不住，手还得实。」

韩泥不谢。谢字欠着，欠到能还。还字比谢字重，重就得活；活得久，才还得上。

张麻婆又顿了顿，像把话压到人听得见的分量：「叶丫头补肘，补的是人。人能补，谣就补不成铁案。你坐着，别把她拽进你那张嘴。」

「嗯。」韩泥道，「我坐。帖上有她名，我也只听，不抢。」

「这才像记恩的。」张麻婆扫帚一顿，走了。

窗外风硬，吹得门墩尘起又落，像谣也落不进肘里半寸。针脚密，骂在帘外。他隔窗不语：语成牵扯，不语成护——护叶名节，护到公审；公审清了，这缝衣恩才还到实处，实处是还热。

灯豆将尽，针脚在肘上像一行不会写字的信。信说：名可脏，肘不能漏。漏了，公审前叶家秤先乱；秤乱，汤就凉。他穿上补袄，肘贴坛沿擦过——沿凉，袄暖；暖在肘，不在嘴。嘴明日说，肘今夜挡。

补袄记在心里，不必声张。声张害叶。害不到，就是今夜的情——情在窗缝，在针脚，在忍住的半句。

丑时前，叶家药铺灯弱着，弱也是亮。亮着，叶汤就不倒。韩泥举目西望，灯一线，像恩还在远处守着。他低声道：「肘不漏。名我护。公审上，你说话，我坐着。」

明日公审，他坐着听，不害她名。名护住了，缝衣恩才还得上。他合上眼，手按坛沿。瓮没醒，他不喊。五十八岁绝户晦气翁，还住在村西最漏屋里——袄补好了，风短一寸；规矩还在，锤子也就不远了。
