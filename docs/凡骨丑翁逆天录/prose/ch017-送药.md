# 第十七章 送药

公审风声紧，沈家更紧。韩泥臂伤结痂，出门编筐，门墩多了一小包，布旧，结打得怯——像娃的手。里三层，外两层，最里是几株干药：止咳的，清热的，量不多，够熬三帖。纸角二字，歪，小：

别病。不是叶家笔，是芽儿笔。芽儿送药，像送粥，像做贼——贼在泥岗村，是对晦气翁好。韩泥收药，不入腹，先闻。闻得苦，苦里一点甘，像叶家铺子的气。他低声道：「娃胆大。」
晌午，马春花嗓尖，找沈家麻烦：「芽儿又送！送药送晦气，你娘还要不要活！」

沈枯芽哭，闷在喉：「我……我没有……」

「没有？包上泥印，全村都看见！」

韩泥在巷角，没出。出，坐实。他只把药包分出一半，搁沈家豁口，放下便走——走快，像从没来过。墙内静一息，像有人拾起，没谢。谢字惹祸，不谢，恩也在。马春花骂两声，见无人应，啐远，像一拳打空——打空，也是小胜：沈家还接得住，芽儿还肯送。沈枯芽娘在墙内咳，咳轻，像知道药回来了。韩泥不叩墙，只把剩药包搁豁口——还一半，留一半，留给自己臂上未愈的肿，他不配全收。墙内芽儿声细，像对娘：「药……回来了……」

娘咳里夹半句：「别……再送……」

芽儿哽：「不送……娘咳……」

静。静里，韩泥已远——远，像没听见，听见了也只记：娃还肯送，点就不灭。夜里，叶家药铺灯亮一瞬。帘动，像有人把「趁热」那碗端稳，稳在公审前，稳在谣与恩之间。韩泥在门缝闻见药气，苦，淡，像芽儿的胆。胆小，真；真，就得记在心里。午后，赖扒皮的人路过，棍尖点门扇：「晦气翁，公审前别生事。沈家那包药，别藏你灶角！」

韩泥开门，独眼平：「我生事？我编筐。」

来人瞄他腰，又瞄坛角：「腰里别藏东西。灶角也别藏。」

韩泥摊手：「藏心。心里恩辱分列。你要看，看脸。」

来人愣半息，骂「老疯」，走了——怕心，不如怕赖扒皮的令；令在，苗在。韩泥合袄，这是暗爽：仇在，他也不怕示心——示的是恩，不是邪。傍晚，韩泥熬一帖药，不自己喝，装粗碗，搁叶家门阶。他不叩门，只留字，丑，二字：

趁热。叶家与沈家，同一词。词不同人，意同一脉——热，是情，不是奸。叶青禾开门，见碗，见字，站一息，端碗回帘内。帘动，像叹，叹轻，像谢，谢不出声。韩泥远观，不近。近，害她。他回屋，心口芽儿那粒米旁，又添药叶一笔——浅，真。「粥稀，药苦。」他低声道，「情不稀，不苦。」

坛腹微温，温得像恩又记在心里。记在心里，不是多欠，是多证：证芽儿没退，证沈家还活，证公审来之前，叶汤还立。坛角微温一丝，像吸了药气。韩泥按坛，按心。「记着。」

外头风硬，吹不凉那二字：趁热。药气淡，飘不进他的漏屋，飘进的是雨意——雨未落，意先至，像公审前的潮。韩泥合袄，塞怀，低声再道：「汤是头的。药是点的。头在，点在，公审来，我不怕。」

他自留那半包药，熬一碗，苦，入喉，臂肿似退半分——退半分够，够他明日还去集尾，还去卖筐，还去等公审定日。芽儿的药叶在心，浅，真；真，就不白收。白收是施舍，他不做施舍翁，只做记心翁——恩仇都记在心里，灵根记性好，漏不得。「趁热」二字，叶家收过，沈家送过，如今又从翁手回叶阶——回，不是还情，是还礼的形，形先立，情后存。公审风声紧，紧里这一点暖，像姜，辣，小，够他再撑几日。心里芽儿药叶，浅，真，像芽儿的手。手怯，情不怯——怯还送来，就得记在心里，记得正，将来还，还到芽儿身上，不亏。心贴腹，腹微温，温得像又多一口粮。粮是药，药是恩，恩不谢，只记——记到公审后，一并算清。外头「趁热」二字，叶家收过，如今又从翁手回叶阶；回，是形，形立，情就稳，稳到公审，芽儿不必再做贼送。药叶记在心里，点成一点一挑，像芽儿的手在心里留印。印浅，真；真，韩泥便按实，按到腹温，温得像又多一口还命的粮。「别病。」他念纸角二字，念给芽儿，也念给自己——病的是沈家，别的是怕；怕不退，恩就不退。巷口，张麻婆骂马春花，骂得响，却不去沈家闹：「闹娃，你算啥好汉！」

马春花尖：「送药送晦气，不该管？」

张麻婆啐：「管嘴不如管咳。咳死了，你赔？」

马春花噎，尖收——收，像送药这章的小胜：沈家还接得住，芽儿还肯送，嘴再尖，也尖不进豁口。韩泥门缝听，不答。答，成牵扯。他只低声道：「药苦，恩不苦。记着了。」

坛腹温不退，像又吸一口药气——吸得淡，淡就够。够撑到公审，够撑到还。叶铺帘动，似有人端碗，碗热，气苦里带一点姜辣——辣小，真，像「趁热」又立住一回。公审前夜，风更紧。紧里，芽儿又摸豁口，摸空，摸得怯。韩泥早把药送尽，门缝只留一句，哑：「别病。病在娘，不在包。」

芽儿声细：「翁……药苦……」

「苦才治病。」韩泥道，「怕苦，别怕送。送是点，点在，你娘就活。」

芽儿「嗯」一声，步远，远像做贼——贼在泥岗村，是对晦气翁好，好得真，就得记在心里。沈家墙内，娘咳轻，轻半分，够撑一日。一日一日攒，攒到公审，攒到还。马春花远处还尖，尖到叶铺阶下，见碗里「趁热」二字，尖收——收，像风又断一截。韩泥不辩。辩在公审。今夜只按心：药叶浅，真；叶汤深，真；恩辱分列，不混。坛腹微温，像又多一口粮。他低声道：「送药这章，记下了。下一章，等锤。」

药叶浅，公审风紧。第四笔记心，不示人——示人是邪。明日锤近，他只备一句「我记着」，备热，备还，不备词，词多了像求。第四笔记心，不示人——示人是邪；明日锤近，他只备一句「我记着」。第四笔记心，不示人——示人是邪；明日锤近，他只备一句「我记着」，备热，不备词。药叶浅，公审风紧；韩泥合袄：第四笔记心，不示人——示人是邪；明日锤近，他只备一句「我记着」，备热，备还，不备词，词多了像求。**状态**：大境·凡人 · 资质·丑骨末席（伪灵根） · 鸿蒙九劫瓮·眠 · 洞府·漏舍凡舍
