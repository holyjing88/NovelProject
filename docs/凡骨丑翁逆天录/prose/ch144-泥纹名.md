# 第一百四十四章 泥纹名

坊市尘起，起里混「泥纹」二字。

有人递渣，渣里正，正像144该证的——证不在名，在鼻；鼻准，名才传。

韩泥立末摊，牌仍歪：「丑丹师·嗅渣辨性·续留。」

管事贴条：「泥纹丑丹师，名号可续。」

条丑，字真。真，才像144该起的名——名丑，丹正；正，是手嗅出来的。

白织月袖过，声冷：「名号也丑。」

韩泥只答：「骨丑，渣正。」

刘婆买线过，声低：「名丑不要紧，手别飘。」

「不飘。」韩泥说。

回丙九，他按坛。坛腹温一线，像应「名」——名在远，在手；手稳，名就不虚。

更鼓沉，他望坊市棚影：「泥纹名起了。记着。」

坊市开市，尘起，尘里混一点「泥纹」二字——二字在舌，不在掌；掌里粒丑，丑而正，正才传名。

韩泥立末摊，摊仍破筐反扣，牌仍歪：「丑丹师·嗅渣辨性·续留。」

第一日，有人递渣，渣灰，灰里杂。杂不要紧，鼻下静一息，辛淡正——正了，人退时多留半句：「泥纹丑，效不丑。」

半句传，传像144该起的风——风起在末席，起在泥纹浅，不起在嘴。

白织月袖过，声冷：「泥纹也配名？」韩泥只答：「配不配，渣正不正。」

管事弟子贴条：「丑丹师续留，税照旧。」照旧，才像144，不像夺魁显摆。

铁无言不近摊，近了三步外，声低：「名起了，别飘。」「不飘。」韩泥说，「名在牌，活在手。」

刘婆买盐过，盐粗：「少听嘲。」「听。」韩泥说，「听，才避；避，才活。」

老散修再递渣，渣正，正像142后那口气——气匀，嗅才准；准了，泥纹小名才像证，不像谣。

他低声对坛：「名号起了。记着。」坛不应字，只温。

更鼓沉，他收摊，收得像石。石在，牌在，风过，风过留半句「泥纹丑，效不丑」。

第三日收摊，摊空，囊净——净像没来过，来过不要紧，牌在，名在。

管事弟子验牌，牌歪，字丑：「续留，税照旧。」

韩泥交半文，声平。声平，才像末席该交的。

老散修再递渣，渣正，正像142后那口气——气匀，嗅才准；准了，半句「泥纹丑，效不丑」才留得住。

回丙九，疤鬃驴尚未在，在不要紧——145才配畜；144只名，不畜。

他低声道：「名起了。记着。」

更鼓沉，坊市木棚色浅，浅像96听风那日——日过，风改一半。

名号不在榜，在坊市尾巷半句「泥纹丑，效不丑」。半句传，传像133挂牌试水风改一半，又像97三日过耳。过耳不要紧，手在，牌在，渣正，名才真。真，不是丑丹师三个字响，是泥纹浅、嗅之正、沉腐浮落齐那套证。证在鼻，不在舌；舌要留来收半文税，留来听白织月袖风再冷——冷也尖不过「正」字。

正字在掌，144才像丑骨末席也能起小名，不起在大比台，起在末摊。末摊破筐反扣，牌歪字丑，人更丑；丑不要紧，144只传泥纹，不传人。传人，像显摆；显摆，赖扒皮嗅。嗅不到，144才续；续到145驴驮药，146层实，147再续。再续在远，144只起。起法是老散修递渣，渣正，人留半句；半句像证，不像谣。

谣在舌，证在鼻；鼻下静一息，辛淡正，正了，管事贴条续留。续留，够了；够了不在荣，在听。听了，144才像第二部坊市线活着，不像140势收束就断。断不断，看手稳不稳；稳，铁无言立三步外，不近摊，眼在；眼在，是145前驮药，是580从绶远钩。钩在，不叫。不叫，144这名才像线，不像戏。

更鼓沉，收摊如石，石在，风过，风过留半句泥纹丑效不丑——丑效二字，144该落的钉。

丑时末，他再摸木牌，牌稳，稳像泥纹浅——浅不要紧，效正，名才真。真，145驴才驮得住。

回丙九，按坛。坛腹温一线，像应「名」——名起在渣，不在舌；舌要留来候145驴，候146层实。更鼓沉，他低声道：「泥纹在。记着。」风过坊市尾，尾巷留半句丑效不丑。

帖在怀，怀沉，沉的是序，不是惧。

白织月袖风过，韩泥不辩。辩，费舌；舌要留来守坛。

赖福廊下过，韩泥只答：「在。」在，就不滚。

柱上条角白一线，三年约在怀。怀凉，掌温。

坡下石阶湿，他数息，数帖，数债序。

同棚杂役笑，笑在耳，不在心。

废渣巷日斜，他验渣，渣正，手才静。

侧门驿亭风硬，他听风，风过，记在心里。

散队后，他过外堂阶下，分拣半刻，指稳——指稳，路就不飘。

编筐半刻，藤刺扎掌，刺浅，浅些沿才净。沿净，坛才藏得住。

丑时，练鼻下静。粟壳香淡，香一字不落。

管事记簿，记细，是小胜。胜在忍，不在辩。

他望坛，低声道：「手在，路在。更鼓沉，天边白一线。白里，他按木牌，牌凉贴胸。坊市尾巷风硬，他收摊如石，石在，就不滚。」

验息后，他理怀内册页，页黄，字丑，丑而真。

铁无言声低：「别飘。」韩泥应声：「不飘。」

刘婆端粥，仍多半勺。韩泥点头，谢轻，恩重。

药峰分拣照旧，指稳，鼻下静一息。
