# 第五十七章 掌茧再裂

试手前一日，搬柴加半刻。柴沉，肩稳，掌茧裂口加深，仍无血。无血，是席盖严，是符怀暖，是心记着七笔。铁无言路过，看一眼他手：「裂深，别硬扛。扛裂，试手抖。」韩泥「嗯」一声：「不硬扛。扛，恩断。」铁无言递一小瓶药膏，瓶丑，药凡：「凡药，涂裂，不涂沿。沿，不是你该碰的。」韩泥接瓶，记恩在心里。涂药，裂口凉一丝，凉后稳。稳，像鼻下静气移到掌上。赖福见药膏，眼红：「晦气也配？」管事声冷：「药是铁师兄私赠，再嗓，克你半月麸皮。」赖福哑了。哑，韩泥手更稳。

---

午后，坡下秦霜讲最后一遍：「手抖，丹歪，滚。丹歪不要紧，手不能歪。」韩泥立末排，独眼平。铁无言验鼻，声低：「静带到手上。手上静，丹才正。」同批少年抖肩，抖像怕「滚」字。韩泥不抖。抖是恩断，恩不断，手不飘。秦霜目光扫过末排，停在韩泥身上，停半息：「明日试手，仍末。误者，滚。」韩泥「嗯」一声。嗯，像心口七笔压着。管事贴条：「明日卯时，试手凝丹。丙九韩泥到。」条薄，真。真在「仍末」，真在「明日」。刘婆端粥来，稀，热：「明日试手，别空肚。」「不空。」韩泥喝尽，「嗯」一声。

热在，像第七笔又实。

编筐补刺，藤刺再扎，扎透旧茧，裂口渗潮。潮不是血，血不碰沿。他停半息，涂药，药凡，稳手。手稳，明日藤刺才正。正刺，血才找席，席才挡到沿前。赖福讽：「涂药也试手？试手验脸吧。」管事路过，声冷：「验手，不验脸。脸丑，手正，正就够。」赖福脸青，像秤空一格。韩泥不辩，只把辱记在心里，与恩分列。列清，心不乱。搬柴过半，肩勒旧印，印深，像五年杂役磨出来的。符怀暖一丝，护骨寒，掌裂不僵。僵，明日手抖。不僵，丹才正。

铁无言廊下见他：「沿净？」「净。」韩泥说，「血不碰沿，沿才接得住后头缘。」铁无言「嗯」一声，退回廊角。

夜，雪细，细粉肩。肩沉，沉得住。他低声对坛：「沿在前，血不先。先血，是邪。后血，是缘。缘，等我扎那一刺。」坛腹温比白日多一丝，像瓮听裂口稳了，仍眠。外头更鼓沉，柱上「明日卯时」。明日，试手凝丹。凝丹不是醒瓮。醒瓮在更后，在误触那一滴。邻床问：「你怕不怕滚？」「怕。」韩泥说，「怕也记。记着，手就不飘。」「滚了咋办？」「滚了，恩断。」韩泥说，「恩断，汤就凉。」邻床懂「凉」，闭嘴。他是记心翁，恩仇都记在心里。药是铁无言私赠，赠记在心里，不记舌，不记外。

管事盯梢，盯赖福偷懒。偷懒，记名。记名，秤上空一格。韩泥不偷懒。偷懒，麸皮克。克粮，克不走还路。

丑时，坛腹温加剧一丝，像瓮听手稳，仍眠。他按席，席严，沿净。净，才接明日试手，接血近沿，接瓮醒。他低声道：「手稳了，路又宽半分。宽，不是灵根好，是恩没断。」坛不应字，只温，温得像「嗯」。外头风硬，柱上「明日卯时」更真。真在试手，真在末排，真在窄路。窄路也是路。路通向凝丹，凝丹通向还。还叶汤，还振东药。还，先还烫。同批少年碰他袖：「明日你真去试手？」「去。」韩泥说，「末排也去。去，验手，不验灵。」「灵根末，手正，正就够。」铁无言声低，只对他。

他摸木牌，牌边磨毛，毛得像心口七笔。笔笔磨过，磨出实。实了，明日才站得住。卯时前，他又编筐半刻。半刻，藤刺浅些。浅些，裂口不深太快。快不要紧，沿净要紧。刘婆端姜汤，姜薄，热：「涂了药，别傲。」「不傲。」韩泥说，「傲，符自灭。」坛壁水汽爬半寸，敛。敛着，像替他守着。手别飘，恩不断。恩不断，叶汤就还在前头。坡下试手前，秦霜又道：「问心只问一句：为谁炼。答邪，滚。答空，滚。答热，记。」韩泥声哑，声平：「为还。」秦霜「嗯」一声，不夸，只点头。点头，像记功在。

卯时前，坡下又起一层霜。霜贴石阶，石阶滑，滑不要紧，站要稳。秦霜收木鱼，木鱼落掌，声轻：「规矩讲清了，验在鼻，不在嗓。」嗓轻，鼻重。重，才配末排。管事验帖：「明日卯时，丙九韩泥到。」帖薄，真。同批少年碰他袖：「你真不怕滚？」平，像辨香那日。编筐补刺，藤刺浅些。药膏涂裂口，凉后稳。刘婆端粥，稀，热：「明日试手，别空肚。」管事声冷：「验手，不验脸。」外头更鼓沉，柱上「明日卯时」更真。管事又嘱：「明日试手，仍搬柴。搬柴手抖，试手也滚。」韩泥「嗯」一声，肩稳。

同批少年问：「明日你真去试手？」铁无言廊角，声低：「沿净，别拜邪。拜邪，恩断。」韩泥「嗯」一声：「不拜。守沿，等缘。」刘婆又端半勺粥，热：「抖的是心，不是手。心记着，手就稳。半勺，热在，像第七笔粥又实一分。坡下霜白，柱上「明日卯时」在。在，像窄路也亮了半分。他摸木牌，与试手帖并放。并放，像命与规矩挨着。挨着，才验得过。秦霜收木鱼，木鱼落掌，声轻：「验在鼻，不在嗓。」邻床问：「明日验啥？」「验手。」韩泥说，「手抖，丹歪，滚。」夜，雪细粉肩，肩沉，沉得住。黄昏，坡下风硬，柱上「明日卯时」在。韩泥又搬柴半刻。半刻，肩稳，像试手前练出来的。铁无言廊下见他：「明日试手，静带到手上。」韩泥「嗯」一声：「带上。带上，丹才正。」管事验手：「裂深，别硬扛。」同批少年碰他袖：「明日你真去？」韩泥说，「去，验手，不验灵。」秦霜又道：「问心只问一句：为谁炼。」试手前夜，他未睡实。未睡实，像辨香前夜。净，才接明日试手。真在试手，真在末排。

试手前夜，药涂裂口，凉后稳，稳像鼻下静移到掌上。外头更鼓二响，响沉。明日卯时，坡下见。

药膏凡品，涂裂口，凉后稳。稳，像鼻下静气移到掌上。搬柴过半，肩勒旧印，印深，深像五年杂役——杂役久，手才稳。管事盯梢，盯的不是他，是赖福偷懒。偷懒，记名；记名，秤上空一格。韩泥不偷懒。偷懒，麸皮克；克粮，克不走还路。黄昏，秦霜远廊一句：「试手过，手未抖，记功在。记功，不是灵根好。」韩泥「嗯」一声，独眼平。平，像末排惯了。夜，雪细，细粉肩。肩沉，沉得住，像瓮温加剧前最后一夜安稳。他低声对坛：「沿在前，血不先。先血，是邪；后血，是缘。」坛温一线，贴掌，像听「缘」字，不急应。
