# 第三十二章 一盒茶饼

秦商言将离那日，莫长春留她半刻。

客厢茶桌，无珍馐，只有一壶粗茶，两只杯，杯口缺一角，像执法堂寻常待客——不隆重，反而让秦商言这种算惯了灵石的人，更摸不准深浅。

她坐得直，像随时准备讨价还价。莫长春从袖中取出一盒茶饼，纸包旧，边起毛，角磨圆，像揣了很多年，揣在袖里与簿挨着，挨着挨着，就成了礼。

「家乡茶。」他推过去，「不值钱。请你尝。」

秦商言一愣，目光在纸包与莫长春杖尖之间游移：「莫长老……不求价？」

「不求。」

「商道讲公平。」她目光微顿，第一次像被「无价」绊住，「无价的礼，最难还。商言做了十年买卖，最怕难还。」

莫长春笑：「难还就好。还得起的，就不叫礼。」

秦商言打开纸包，茶香扑出，苦后回甘，像凡间日子，不像百宝阁货架上的灵茶。她拈起一块，细嚼，细嚼出涩，也细嚼出暖——暖得像有人把「赠出不计价」五个字，放在她舌上。

袖中什么一跳。她无塔，却像站近这老头时，能觉出「记在心里」的重量。

下一瞬，案上多一卷薄简，墨迹新干，像茶气里突然长出字来：

**赤焰东脉调动：三日后佯攻，七日后夜袭。**

秦商言抬眼，声低：「这笔买卖，商言还没算清。」

「茶饼算清了。」莫长春摆手，「简是回赠。你问赤焰在哪花钱，老夫答在妖与田之间——茶，是谢你肯答。」

她沉默两息，忽然问：「若商言把简卖给别人呢？」

「卖就卖。」莫长春道，「情报值钱，人情更值钱。你卖简，是商道；你记恩，是礼。各算各的恩。」

秦商言起身，郑重揖手：「秦商言记下了。」

出厢时，柳青鸢在廊下路过，目光相触，各自点头，像宗门与商会之间，多了一根看不见的线，线那头拴着东脉，这头拴着茶饼。

山下驿站，秦商言对下属道：「莫长老送的茶，比灵石贵。记入恩心，不记入买卖价。」

下属问：「若赤焰夜袭成真，商言赚不赚？」

秦商言望青岚方向，笑，笑里第一次像亏本也乐意：「赚。赚在青岚若赢，百宝阁有路走；若输，商言也有恩可还——恩，是另一条商路。」

夜，莫长春独坐廊柱旁，不坐主位，塔壁浮字：

**茶轻意重，情报记在心里。**

他念：「秦氏记下了。下一阵，东脉事本。」

杖中竹纹一游，像应。风大，像把茶香送回丹堂，送回东脉，送回三月后的天试。

赵玄机在执法堂听闻「情报记在心里」，指节叩铁牌，一声，像把妒意敲实，又敲松一寸。

一盒茶饼成交那日，普通得不像礼，像故意气赵氏眼。

莫长春袖空，一念出塔，饼在塔中，不示，只记。弟子围看，围看像怕轻礼也烫手。

柳青鸢声低：「师兄，饼轻，线重。」莫长春道：「线重，才用轻换。重了，换不动。」

赵玄机验饼盒，盒空，空得像把妒火也定住。他低声：「饼换线，门里该查。」

苏念慈闻饼香，闭炉，闭得像怕茶饼香进丹堂乱了药。她低声：「茶饼茶饼，别传成贿。」

顾小满册上又记一行，行细，细得像半块石硌手。她低声：「饼记下了。」

赵玄机冷声：「饼换线，当查——」莫长春截话：「查实价，别查心。」

茶饼递出时，秦商言指节微颤，颤得像十年买卖里第一次怕「无价」。她问：「莫长老，若商言还不起，怎么办？」莫长春答：「还不起，就记着。记着，比还快。」

简上墨迹新干那一瞬，厢内烛焰直，直得像战意也直。秦商言把简收入袖，袖中另有百宝阁买卖价，买卖价与恩心分开，分得像把青岚也分成了「可卖」与「不可卖」。

柳青鸢遣人问：「简可真？」莫长春回：「真。宗主主战，简只是让战打得明白。」来人退，退得像把「夜袭」二字也退进全门心里。

赵玄机听闻「情报记在心里」，在思过崖指节叩石，一声，像把妒意敲实，又敲松一寸。他忽然想：若莫长春手松给茶，手紧给东脉——紧的地方，才是该守的门。

秦商言上马前，忽从袖中又取出一纸，纸薄：「东脉侧径草图，商言附赠。非卖，是恩。」莫长春推回：「恩在茶里，图在简里，够了。」秦商言笑：「莫长老手紧，紧得让商言更想记。」

弟子围驿道，围「茶饼换情报」，围得像看一场亏本却乐意的买卖。买卖里敬，也疑——疑茶饼真能换情报。疑着疑着，简已进柳青鸢案，案上像多了一柄看不见的刀。

魏无涯在檐角观简传讯，扇合，低笑：「茶比灵石贵，商言说得对。」

孙福在角隅听简讯，面色白，像听见「第七年」三字，也听见自己的旧恩仇将被翻。他嘀咕：「周长老守田……」旁人瞪：「你看简，别看嘴。」

外门弟子把简抄成七种版本，版本里都有「夜袭」二字，像把怕字写进每个人眼里。弟子问苏念慈：「首座，夜袭丹堂危不危？」苏念慈冷声：「危。所以炉不能熄。」

秦商言驿中写信，信末加一句：「商言欠莫长老半块茶，欠青岚一条暗线。此债，商言认。」

霍镇山在廊下听「茶饼换简」，骂：「好！打得明白，死也明白！」

周德海在远处听，笑僵，笑里像袖中折子湿了又干。

林挽月罚役归途，听「茶饼换情报」，唇动，像想说赵师兄与此有关，终未近。她把一枚新制符纸塞门缝，符薄，意厚。

同一夜，长老会急议，柳青鸢指节叩案：「简既至，主战不必再辩。」周德海笑僵，笑里像袖中折子湿了又干。

茶饼入驿那夜，简上「佯攻」二字像鼓，「夜袭」二字像刀——刀还没落，鼓先响，响得全宗睡不着。

顾小满在杂役堂浆洗，听「茶饼换简」，指节白，像半块灵石在怀，温了一路，温成警。她不懂天试，却懂「记」字——记下了，就不退。

秦商言上马前，忽从袖中取出一纸，纸薄：「东脉侧径草图，商言附赠。非卖，是恩。」莫长春推回：「恩在茶里，图在简里，够了。」

秦商言望青岚方向，笑里第一次像亏本也乐意：「赚在青岚若赢，百宝阁有路走；若输，商言也有恩可还——恩，是另一条商路。」

夜，塔壁浮字：茶轻意重，情报记在心里。赵玄机听闻，指节叩铁牌，一声，像把妒意敲实，又敲松一寸。

弟子围问：「长老又送礼？」莫长春笑：「礼送了，简收了。接下来看宗主怎么战。」

茶饼入怀，情报出袖，袖空的人，反而把线送远了。

弟子围驿道，围「茶饼换情报」，围得像看一场亏本却乐意的买卖——买卖里敬，也疑，疑着疑着，简已进柳青鸢案。

赵玄机验饼盒，盒空，空得像把妒火也定住。他低声：「饼换线，门里该查——」莫长春截话：「查实价，别查心。」

魏无涯传讯厉无殇，只四字：「简已至。」厉无殇阅讯，焚，笑冷：「准不准，莫长春都要接。接，才是戏。」

夜，塔壁浮字：茶轻意重，情报记在心里——风大，像把茶香送回东脉。

一盒茶饼成交那日，普通得不像礼，像故意气赵氏眼——气归气，简已进案，案上像多了一柄看不见的刀。

赵玄机听闻「情报记在心里」，指节叩铁牌，一声，像把妒意敲实，又敲松一寸。

简是刀，刀还没落，鼓先响。
