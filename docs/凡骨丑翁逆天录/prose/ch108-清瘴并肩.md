# 第一百零八章 清瘴并肩

深窝前。

角草黑，黑里甜，甜得像毒眼盯人。韩泥在障后掌炉，障挡眼，眼挡开了，火才炼得正。他把经页摊开半寸，指腹按住投序：青芯三撮，褐渣一把，石灰半指，灵泉五滴——一毫不乱。乱一毫，丸就邪。

炉温稳住一寸。白气聚，聚成丸。丸青灰，像药。药成三粒，正不正，全在鼻下，不在炫耀的眼。他嗅半息，半息够判：正。

第一粒掷进深窝。窝嘶一声，甜退半丈。半丈够同袍往里进。

第二粒铁无言吞下。臂上肿退，退得像活回来。活，就是并肩。

「跟我进。」铁无言说。

韩泥进三步，第三粒撒成灰雾，雾里清出一条路。路刚出，爪风再至。铁无言挡，韩泥不躲——不躲，才叫并肩。

并肩不是喊口号，是手。他铲障改道，同袍拖倒兽。兽毒还在，清瘴丸正克。克尽，深窝平了，平得像清完。

管事弟子记：「清瘴有功，丙九韩泥，铁无言。」

铁无言拍他肩：「兄弟，手在。」

韩泥应声：「在。」

在，就是契。契还远，远不要紧；这一夜，果已经结了。

赖福瞧簿，脸青，没嗓。簿上「有功」两字，像把「晦气」按远半寸。白织月递过凉水，声低：「……正。」

韩泥接了，不多谢。谢轻，记在心里：并肩、清瘴、正。

叶青禾在坡下，门缝灯灭了。灭像睡；脉里培元丹还温着，恩还记着。他仍不近。近像连累。清瘴是还另一笔——还栏活，还同袍活。

回棚，掌裂渗潮。他按坛，低声道：「清过了。并肩过了。」

坛腹温，像应「正」字。

丑时静半刻。香落齐，掌纹烫——烫得厉害，五层圆满，微微一开，开到六层。开得不飘，像从兽潮里实出来的。邻床问：「你真成丹师了？」

「不成。」韩泥说，「成清。清完，仍倒渣。」

秦霜木鱼远敲，声低：「清瘴别骄。」

他不应口，心里只记：不骄，才接得住往后炉温。刘婆远送粥，不敢近窑：「活着，别骄。」

「不骄。」韩泥谢轻，「手还在。」

铁无言又堵门口：「明天我送剩料。浪费，像逃。」

「不逃。」韩泥说，「料留下，炉还要稳。」

「稳了呢？」

「稳了再炼散。」韩泥独眼平，「散不是丹，是路。」

同袍路过，捅他：「管事把你写进簿了，赖扒皮脸都绿了。」

「绿他的。」韩泥道，「我绿我的手茧。」

障后炉灰未冷，他再嗅半息。正味还在，像把「兄弟」两字钉进掌纹。钉住了，就不怕下回夜禁解了有人翻旧账。

有人问管事：「这功算外门还是杂役？」

管事哼：「算活。活下来的，才配算功。」

赖福忍不住：「一个臭倒渣，也能并肩？」

铁无言回身：「能。他手在，你嘴在。手活，嘴死。」

赖福退半步，不再吭。白织月在远处似笑非笑：「……丑是丑，正是正。」

韩泥听进耳朵，不接梗。接梗像讨喜，讨喜不值钱。值钱的是窝平了、人没倒。

夜里同袍们分剩下的干饼，饼硬，咬得响。有人递他一块：「并肩那步，我看着腿软。」

「腿软也进。」韩泥说，「进了才有路。」

「你开六层了？」

「开一线。」韩泥道，「一线够用，不够浪。」

不够浪，所以他把经页收得更紧。「清瘴」二字实了，下一笔就轮到「散」。散还远半寸，半寸要靠炉温稳住，不能手飘。手不飘，明日废窑门口那口炉，才肯听他。

更鼓尽，界外腥退，退像终。终后有余韵——余韵在炉，在温，在下一寸火候。他低声道：「清瘴丸，正。铁无言，在。」

在，就不滚。不滚，才接得住首丹余韵，也才接得住培元散那一程。

白织月远远丢下一句：「六层了？别浪。」

「不浪。」韩泥说，「倒渣。」

「倒渣」落地，像把自己按回末席。末席也行——末席手在，并肩就在。

铁无言临走扔下一袋渣料：「雨停了就回窑。别歇骄。」

「不歇。」韩泥接袋，「歇了，炉冷。炉冷，散不来。」

「你心里全是炉？」

「炉，还有账。」韩泥说，「账比炉沉。」

账里有刘婆粥、叶家汤、同袍活、并肩肩。肩还热着，热提醒他：这一夜不是终点，是丹道再往前挪一步。

棚外夜风干净了些，干净里仍有一点药焦味。味正，他便放心躺半息。半息里掌纹还烫，烫着六层那线微开——开着，却不让它飘。飘了，明日稳炉就稳不住；稳不住，散就白炼。

他睁眼摸摸坛沿，沿温一线，像瓮也听见「正」字。听见了，就不催他逞能，只催他稳。

清完深窝那阵，界外腥气真退了。退得人不大习惯——习惯了夜里听爪、听吼，忽然静下来，耳朵反倒空。韩泥坐在障后收拾炉具，灰磕得干脆，干脆里仍剩一点正焦味。味在，他就安心：这一炉没白烧，这一夜并肩也没白并。

同袍里有个嘴碎的凑近：「韩泥，你刚才第三粒撒雾时，腿到底软没软？」

「软。」韩泥老实说，「软着也进。软着站住，比硬着逃强。」

「你这老头……」那人啧一声，又把剩饼掰给他一半，「吃。并肩的人不能空肚。」

韩泥接了。接饼像接契，契字不写纸上，写在嘴里那口热里。热过之后，他才去看铁无言臂上的绷带——绷带松了半圈，肿确实退了。退了，就证明丸正；正了，才配被叫一声兄弟。

铁无言看见他看，哼道：「看什么？看完记得留料。你不是还要稳炉？」

「记得。」韩泥说。

「记就行。」铁无言顿了顿，声音更低，「六层别让赖福闻出来。闻出来，他会把折上折。」

「藏着。」韩泥应。

藏境界比藏丹难。丹能包纸，境界会从香落匀不匀里漏。他把香捻得更慢，慢成普通杂役那点余力，才敢闭眼歇。歇时掌纹还烫，烫得像有人在里边轻轻敲门——门开一线是六层，开大了就飘。他按住：先开一线。一线够走路，不够卖弄。

清窝之后管事又来核人，点到丙九两名字，笔锋难得不刻薄：「活着的，记。死了的，另册。」

有人小声笑：「这话也太冷。」

韩泥不笑：「冷实话比热假话好。假话烫嘴，烫完还是死。」

铁无言听着，拿肘撞他一下：「你少讲老理。讲多了像说书。」

「说书不顶障。」韩泥道，「顶障靠手。」

手还黑着，黑是灰，灰里正味还没散。他把指缝搓净，搓到能摸坛沿不脏。坛沿温，像替他收住今夜那口「兄弟」。收下了，明天回窑才不飘。

白织月后来又远远丢来一瓶很淡的伤药：「臂裂的，抹。别感染，脏死内墙。」

韩泥接过，转手给了铁无言。铁无言愣：「给你的。」

「你挡得多。」韩泥说，「我抹手就行。」

铁无言骂了句土的，还是抹了。抹完抬眼：「你这人，真够呛。」

「够呛才活。」韩泥道。

活着回棚的路上，他看见簿页在灯下反光。「有功」二字像钉子，钉得赖福那天晚上都没来堵门。没来也好——好让他把「散」字在经页边偷偷划一下，划完又抹掉。抹掉不是忘，是藏。藏到炉温真稳的那一夜，再拿出来当起点。

离开深窝前，他把障根石灰又补了一指：「平了也别松。松了，余毒会装死。」

铁无言应了一声：「听见了。你回窑，我盯两日。」

坡下石阶还湿。他站到棚口，望了望灭灯的门缝，又望了望夜禁将解的山路。路通向废窑。明日回窑稳炉——稳住了，才把「还」字从并肩里接到散上，接到更远的那笔恩。
