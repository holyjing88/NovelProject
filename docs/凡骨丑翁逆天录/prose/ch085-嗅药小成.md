# 第八十五章 嗅药小成

再遇后第三夜，丑时。

角柜六包样摊席前。席严，坛不露，样在膝，膝稳。韩泥鼻下静，沉腐浮并三包，再并三包，并出一线清——清里药性归位，像秤上三堆灰各归各斗。斗不混，味才说话。

第一样，辛沉，沉底一线白，白像引火。引火轻，记重。第二样，苦浮，浮少，浮也要剔。第三样，腐底，底真，真在毒苗——毒苗剔净，筐才净。净了，才敢说小成两个字。

并息到极，颅骨里又一声轻，像瓮唇吐息。吐息里三字加长半分：

「沉腐浮，各归位。」

加长才是小成。小成不在舌，在鼻；鼻里药性各归各位，各位才说话。他屏息，独眼不眨，低声只复给自己听：「辛沉引火，苦浮叶家，腐底毒苗。」叶家二字落心口，像远路药香回了半息——回半息，手更稳。

复诵完，指到未知灰：「沉。」第二堆：「腐。」第三堆：「浮假。」三次连中，不像炫，像活——活长了，嗅药才小成。小成了也不叫；叫了，像把诀名挂脖子上给人摸。

邻床翻身，眯眼：「啥味儿这么清……像屋里起了雾。」

「坛味。」韩泥声平，「睡吧。雾不是邪，是样渣并出来的清。」

小杂役哼一声，又睡实。睡实了，诀才藏得住。他收样，包实，锁回角柜。柜锁丑，像规矩。更鼓将尽，天边白一线。他廊下立半刻，任风硬，鼻下仍清。清得怕人——怕人处，正要藏。

卯时，药渣场验手。管事弟子指五堆混灰：「分。错一堆，记名。」韩泥分拣，指稳，五堆未错一堆。管事眉松：「丙九韩泥，辨性准。」记簿，笔丑，真。真了，才顶得住赖福舌头。

赖福立远处，眼细：「辨性准也末等，准了也炼不出丹。炼不出，准也是臭。」

「准在鼻，不在舌。」韩泥不接茬——接茬，全丙九饿。「臭不臭，管事写过了。」

铁无言走近，瞥分堆：「小成了。成在鼻，别叫。叫了，像求诀名。求名的，丹房先嫌。」

「不叫。沉腐浮，各归位。」

铁无言点头，走了。走也像认可——认可不重，重的是渣场仍收他。收着，才有口粮；有口粮，才谈得上后头续练。

午后温听雨过风口，裙素，袖香温。瞥五堆，眉松：「昨日叶渣另堆，今日五堆未错。错一堆，全丙九记名。你没让他们跟着脏。」

未错，她没夸，只留半句：「鼻下有根，根像真的。真了，别拿来骗人。」

「不骗。」韩泥应，「根给手用，不给嘴用。」

根像真的，他不谢。谢轻，记在心里——记小成，记缘苗温一线，记恩、愧、仇要分开放。放开了，心才不搅；不搅，鼻才继续清。

刘婆端粥，仍多半勺：「辨性准了？」

「准了。」韩泥说，「小成在鼻。鼻成，人不骄。」

「鼻成怕什么。」刘婆哼，「饭稳，恩不断。恩断了，小成也喂不饱。」

韩泥点头。成轻，记重。记重了，才接得住续练，才接得住三层破。破境在丑时，不在嘴里嚷。

秦霜木鱼一声：「成小，心别骄。骄了，香飘。香飘了，温氏一眼就能验出你空。」

他不答，只按席角。坛腹温一线，贴掌，像应「小成」。低声对坛：「嗅药小成了。不叫。坛在，人在。人在，根才在。」

邻床问：「五堆全对，厉害？厉害了能多领麸皮不？」

「不厉害。」韩泥说，「鼻下有根。根在，手才准。麸皮看册，不看嘴。」

少年闭嘴，像也学会忍。丑时前再验五包未知样，连中，连中稳。稳不像飘——飘了，管事记「邪」。记邪一次，小成白成。

赖福讽远：「末等也辨性？辨出邪。邪该报门规。」

韩泥不接。舌留给鼻子，留给坛。柱上条角白一线，春风硬，硬不要紧，鼻清——清在，就不滚。不滚，叶汤还字还挂着。

铁无言夜过渣场，瞥角柜：「小成了，别示样。示了，赖福记过。记过了，样渣全废。」

「不示。沿净。净了才锁。」

温听雨名在记里，不碰舌。舌碰，缘线就邪；邪了，叶愧线也乱。乱，手就飘；飘了，小成白来。他廊下再练半刻，五包未知样连中。稳了，才接得住往后十堆加验。加验更狠，狠才像路。

小成后第二日，未知五堆。鼻下清，五堆连中。管事记「辨性准」。铁无言再取灰一撮：「再闻。闻错，今日白废。」

韩泥鼻下一息，指落：「沉四，腐一，浮零。」

铁无言分秤，对，点头：「小成实了，续在后。续的时候别飘，飘了就假。」

温听雨风口立半息，瞥分堆，眉松：「全对不容易。不容易处，才像手稳。」

他不谢。记她第三回验手，记缘苗温一线。午后外堂洒扫，阶净，栏净，各干各的。她没多问，巾一收，走了。走了好——好在不攀，不攀线不断。

黄昏角柜样十一包，包实，锁柜。邻床问：「小成能炼丹了？炼了能卖钱？」

「鼻成了。」韩泥说，「丹在远。近在槽。槽清了，丹才有日。」

丑时廊下练半刻。五包并息，并出一线清。清里掌纹影颤，颤像三层将破。他对坛：「小成了。续在后，不叫。叫破了，境也破不好。」坛不应字，只温。

老耿挑水过，声哑：「成小别拜诀。拜，路断。断了，十一包样锁着也没用。」

「不拜。记着。记还，记续，记丑时。」

管事夜巡验棚，灯照席角，沿净，记「手稳」。刘婆粥仍多半勺：「手稳，饭稳。饭稳，恩不断。」韩泥点头，喝尽，不谢。

铁无言抱臂立侧：「灵根末，手稳，渣场仍收你。别漏。漏一包，小成变话柄。」

「不漏。」

白织月远廊过，袖风冷，没停：「小成了也末等。末等，别吵。」没停，小胜。他把「各归位」摁进掌纹那路——摁实了，丹路才接得住。三线各归各位：辛沉引火，苦浮叶家，腐底毒苗。各位说话，才不是热闹，是活。活长了，才还得了那碗烫。

更鼓沉。他合眼不睡实，鼻下仍清一息。清里不叫，不骄，不示样。明日续练，十堆加验；加验过后，丑时才谈破三层。刃不亮，先记——记完了，才配嗅药小成续那一段。续得住，远灯才不只是影。

坛腹最后一线温回来，像验。验过了，人在，根在。根在，就不滚。不滚的人，才配把「还」字从今日背到下一回过验。

他摸了摸怀中玉牌，牌边磨毛了。毛边像杂役三年手茧——茧在，诀才不被闲话刮走。刮不走，十堆加验来了也不慌。不慌的人，丑时破三层才站得住；站得住，远灯才不只是内视里晃的影子。影子能见，脚还得一步步踩实。踩实了，嗅药小成才算落进日子里，而不是落进嘴上。

邻床翻了个身，含糊问：「你还不睡？鼻还闻啊？」

「闻完了。」韩泥低声，「睡。明日加验，醒着才准。」

少年哼一声，真睡了。韩泥却多坐片刻，把十一包样在心里排成一列：沉的沉位，腐的腐位，浮的剔干净。排齐了，才像把「各归位」从颅骨里那声轻，落进指尖。指尖稳，下一回过验才不虚；不虚，叶家那碗烫与门缝那盏灯，才一点点挨得近。

风从窗纸缝里钻进来，带着渣场隔夜的苦。苦不冲，他反倒踏实——苦说明样还在，诀还醒着。醒着，才配迎明日十堆；迎住了，嗅药小成续才不是空话。空话喂不饱丙九，也还不了恩。恩在，根在；根在，下一回过验就不虚。不虚，才算小成落进日子——落进日子，远灯才跟得上脚步。
