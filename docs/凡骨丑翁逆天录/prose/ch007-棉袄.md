# 第七章 棉袄

天又冷了。

入冬前夜，泥岗村风硬，硬里夹着雨意。韩泥屋漏，风钻缝，像针，一针一针扎麻脸。藤条散了，编不成，指僵，僵得握不住针。棉袄若不来，这一夜难挨。

他呵气暖手，气白，白得像短命的热。灶上无柴，只剩半捆野藤当引火——引火也舍不得全烧，烧尽了，明日编筐手更僵。他先把散了的藤条理齐，理齐了一截，用草绳扎好，扎结是爹教的：结在，筐就不散；人不散，冬就不绝。

门蹭了一下。蹭得轻，像怕惊动晦气。

陈姑七十出头，背驼，眼浊，抱一件棉袄，旧，洗得发白，絮从袖口钻出，钻得像白发。袄上有皂角味，皂角味旧，旧得像洗过许多冬。

「穿上。」嗓沙，像砂纸磨锅。

韩泥没接。陈姑瞪他：「嫌旧？你身上补丁摞补丁，算新的？」

「不是嫌。」韩泥嗓哑，「是怕。」

「怕什么？」

「怕连累你。」

陈姑哼一声，把袄往他怀里一塞，塞得实：「连累？我七十了，还怕连累？你绝户，我孤寡，凑个伴。你死了我给你烧纸，我死了你给我摔盆！」

她没说完后半句，韩泥懂。他接了。棉袄沉，暖，像一捧实在棉花。指尖碰她手背，皮糙骨硬，硬得像老树根。

「陈姑……」

「别谢。」她摆手像赶鸡，「我不是为你，为我自己。你穿活了，我死后才有人记得给我点一炷。」

「棉袄我收。记在心里。」

陈姑哼一声：「记什么，穿活了再说。」

她停一步，又回头，眼浊里亮一下：「别冻死。冻死了，我白送。」

「不冻。」韩泥说，「穿活了，心里记着。」

陈姑哼一声，转身走，风一刮，晃一晃，像一片旧布在村道上飘。

韩泥穿上，棉絮味旧，暖进骨，骨缝里那阵僵，松了半分。袄袖长，长出一截，他卷了卷，卷得像把寒挡在门外。屋里暗，暗里他抬臂，臂弯棉絮窸窣，窸窣得像有人陪。

心里又添一撇——叶汤、赵辱、芽粥、陈袄。四恩里的第三笔，未到齐，心已齐一截。

他出门打水——冷天夜里去井台，头一遭。棉袄领子竖着，竖得像一层薄盾。村人见棉袄，让了半步。半步够走，够打，够让陈姑那袄在村道上有用。

有人低声：「晦气翁也配穿袄？」

另一人哼：「你配嚼？」

声息散了，散得像风咬不动这半步。

井台边周嫂在，嘴动，想说什么，见棉袄，闭了。闭得紧，像把话咽回肚里。

「晦气翁也穿衣了？」周嫂旁有人嘀咕。

韩泥拎绳，放桶，水起，浑，稳稳落地：「穿衣，也打水。井是村的。」

周嫂棒槌悬着，终是没砸响。旁边妇人低声：「陈姑送的吧？」

周嫂脸一沉：「少嚼。」

韩泥打水，桶满，拎回。水晃，晃在桶沿，不洒。洒了，更像晦气。他不辩，不炫，只穿。穿本身就是答：晦气翁没冻死，有人把他当人。

周嫂终是开口，声硬：「陈姑糊涂，你也糊涂？」

韩泥停半息：「糊涂的人，不打井。」

「你打井，你穿衣，你还真当自己是村人？」

「井是村的。」韩泥说，「衣是陈姑的。我打水，不偷不抢。」

周嫂棒槌一落，落回盆里，溅起水花：「算你识相。」

说罢走，水在桶里晃，晃出稳声。稳声像小胜：井还能打，路还能走，袄还在肩。

井台边有人窃语：「陈姑老糊涂了。」

另一人哼：「糊涂？人家清醒。绝户没人管，她管。」

韩泥听见了，不回头。回头像认。认不得，认恩就行。

回屋路上，冷风刮脸，麻脸疼，疼得清醒。清醒里，他想起张麻婆骂街尾那句「筐编得结实」——结实，就能卖，能卖，就能活。他先把散藤续编，续编手仍僵，僵不要紧，袄袖卷了，卷出一道挡风棱，棱在，指节才暖半分。

编到一半，油灯豆，豆光昏。昏里他抬臂试袄，袄沉，沉得像把第三笔恩压进肩窝。肩窝暖了，藤条才绕得紧；绕得紧，筐沿才齐；沿齐了，明日集尾才换得出铜板。

他低声道：「陈姑，袄我穿着。穿活了，才编得动。」

坛角仍凉，凉不要紧——第三笔恩在人身上，不在坛上。人在，坛在，冬就不绝。

后半夜雨漏。雨打棉袄肩，重些，仍比单衣强。灶边坐，姜与粥的余温似还在胃，棉袄加一层，像把恩捂实。漏雨滴在灶沿，滴成细线，线断，断在坛边。坛沿微凉，凉得手指一触，沉。

他低声道：「冬来了，人没冻死。」

外头陈姑家咳两声，像确认她还活着，他还活着。活着，恩就能还；能还，就不白收。

次日，韩泥担水过陈姑门口。门开一线，陈姑探半张脸，眼浊，却亮：「还暖？」

「暖。」韩泥停一息，「袄在，人没冻。」

「别谢。」陈姑摆手，「谢了，马春花又有一嘴。」

韩泥应声：「不谢。记着。」

陈姑哼一声，门又合。合得轻，轻得像怕人看见她管绝户翁。

韩泥担水回屋，棉袄肩被桶绳勒出一道浅印，印浅，暖还在——暖在絮里，絮在恩里。路过叶铺帘外，药香淡，淡里夹着姜辛，辛像提醒他：叶汤是第一笔，陈袄是第三笔，中间辱与粥都没白挨。

他没近帘，近像牵扯。牵扯了，谣又多一嘴。只把桶放稳，水不洒，洒了，晦气又添一分。

路过赵绣春墙根，有人窃语：「陈姑老糊涂，绝户也送袄？」

赵绣春嗓尖，尖不过桶底碰石台的闷响：「送袄？送晦气！」

韩泥不辩，只把桶放稳，水不洒。

里正巡巷，咳一声：「井是村的，袄是人送的。谁再嚼，记名。」

赵绣春门内一滞，声哑半寸。哑半寸，够他肩上的暖再实一分——实了，第三笔恩就不白收。

里正又补一句，声远：「冬里井水也是命。袄也是命。嚼多了，连坐。」

韩泥应一声，应得短。短像丑骨末席——末席也占席，占席才走得到井台，才穿得住陈姑这一件旧袄。

井台边，张麻婆骂街尾瞥他一眼：「穿活了？别冻死，冻死抬棺累我。」

韩泥只答：「活。袄在肩。」

张麻婆哼更大，却没再骂。骂是邻，邻还在，冬就不算绝。

晌午他去集尾，担一只新筐。买主是老熟人，摸沿：「还紧？」

「紧。」韩泥说，「紧才装药。」

买主数十二文，一文一文落掌心，落实了，第三笔恩像又多了一文重量——重量在肩，不在嘴。他收进腰布，不辩价，辩价像讨。讨轻，袄暖重。

马春花巷口嘟囔：「穿袄卖筐，晦气也卖？」

韩泥肩不动：「筐在身，晦气甩不掉；甩不掉，也得卖。」

丑时前，叶家药铺灯弱着，弱也是亮；亮着，叶汤就不倒。坛沿一线温，贴掌不退，像替他守着陈姑这一件袄，守到还得上的那日。

他按袄袖絮，絮旧，旧得更暖；暖实了，编筐手才不僵。僵手编不出结实筐，结实筐才换得出活路——活路一条，冬就不绝。

丑时末，村犬哑叫一声，像谣也哑半寸。韩泥在门内，棉袄还潮，潮得暖。暖进骨，骨里像多了一根撑，撑住第三笔恩，撑到还得上的那日。

他把桶水倒进水缸，缸底一声闷响，响得像冬也被挡在门外半寸。半寸够活，够了，就不怕下一阵风。

明日还要去集尾卖筐，还要去井台打水。冬来人不冻——不冻，才走得到叶家那道门缝，才接得住下一碗趁热。袄在肩，路就不绝，路不绝，恩就不散。
