# 第一章 晦气翁

五十八岁的韩泥蹲在漏雨的泥屋门口编筐，麻脸冲着日头，独眼好的一只眯着，另一只蒙着白翳，背驼，腿瘸。村童隔着老远走：

「晦气翁来了。」
「别挨他，晦气。」

一个穿开裆裤的娃捡起土块，想扔，被娘一把揪住耳朵：「晦气沾上了，洗三天都洗不掉！」娃哇一声哭了，娘拽着跑远，还回头啐了一口。韩泥眼皮都没抬，手指粗，节骨大，藤条在膝上绕了一圈又一圈，勒得紧。麻脸是胎里带的，跛的是早年抬棺摔的，驼背是常年弯腰编的。村里户籍册上写着「绝户」二字，他一人，无儿无女，死了连摔盆的人都没有。抬棺、扫庙、编筐、采药，泥岗村西头最漏的屋，最晦气的人，五十八年，就这么活。村西泥岗土薄雨多，别人家漏一点补一点，他补不起，茅草旧了，就让它旧。「韩泥！」

马春花的声音从井台那边砸过来，尖，脆：「编你的筐！少往村道中间杵，挡了运！挡了运，你赔得起？」

韩泥「嗯」了一声，藤条没停：「赔不起。所以挪了。」

马春花哼一声，没再近，井台那边棒槌又砸了一下盆沿，砸得脆，像替他收住这阵骂。他不辩，五十八年辩过几回，没用；晦气贴上了，撕不下来。马春花嘴最利，骂归骂，去年他抬棺路过她家，她没拦——韩泥记下了，不说谢，谢字欠着，得等有还的时候。近也近不出钱来，骂也骂不出钱来，她骂她的，他编他的，各过各的。村道上货担铃铛响两声又远了，卖盐老汉冲他点点头，很快把脸别开，怕晦气从点头里传过去。韩泥习惯了，习惯不等于不痛，痛久了，就不往外说。日头偏西，筐编到一半。韩泥起身，腿麻，拄了一下门框。泥屋里黑潮，霉气贴着脸，灶上无米，瓮里无水，墙角堆着昨日采来的苦蒿和半捆野藤——能换几个铜板的留着，换不来的晒干当柴。穷归穷，泥岗村西头这片他住了五十八年，哪块砖松、哪根椽朽，身子比眼记得牢。他还会扫庙，小庙香火淡，扫庙换一顿斋饭，有时换半袋粗面。昨儿扫过，尘土沾裤腿，他没拍——拍着也像邀功。庙祝老眼昏花，常说：「韩泥，你手稳。」

韩泥不答，手稳不稳，筐知道，棺木也知道。隔壁张麻婆骂街，骂得凶，尾声却落一句：「韩泥那筐，编得倒是结实。」

他听见了，没接话——接话像讨夸，夸轻，筐重。午后他担两只筐赴集尾。筐沉，肩勒出旧印，集上人躲他半步，躲完又瞥筐，瞥筐不瞥脸，怕晦气从脸上爬进筐里。买主是个挑盐的，打量筐沿：「编得紧。不松。」

「紧才装得住药。」

买主捏了捏底，底不塌，数了两遍钱才递过来：「十二文两只。晦气翁的手艺，倒实在。给你。」

「下回还来。还编这种底。」

「还来，底还紧。」

铜钱落掌心，十二文一文不少，韩泥收进腰布，布沉了一截，沉，活路坠了坠，坠实了。这是今日小胜——辱在村道，钱在掌心；钱不暖，能买米，米能活命。他不回嘴，担起空筐走稳，把十二文也编进步子。买主在身后又嚷一句：「底别塌！」

韩泥没回头：「塌不了。」

集尾人散得快，他绕远半步出集，半步够活，够躲唾沫。回屋，屋角坛旧，蒙尘无光，内息尽敛，原是腌酸菜用的。韩泥穷，酸菜早吃完了，坛空着，他就当摆设，摆设也是家当，穷人家，坛在，屋在，人在。村里人笑：「绝户翁连坛都蒙尘，命也灰。」

他不答，只每日擦坛沿，用旧布，布洗得发白，也是祖传。擦坛不费劲，费劲的是压住心里那口气——户籍册上「绝户」二字擦不掉，坛沿能擦亮一层。这习惯不知传了几代，祖父韩老栓从不卖、从不扔。老栓对爹韩铁山说过：「这是你爷爷传我的，我也这么传你。别问为啥，问也问不出。坛在，韩家不散。」

铁山咽气那夜手搭坛沿：「坛别扔。韩家就剩你，坛在，根在。」

韩泥不知真名，只唤「腌菜坛」。丑时坛壁微凉，午时又温，他当天候作怪，从不多想。今日擦坛，指尖顿了顿，坛沿仍像凡物；他按了按心口，空了几十年，近来梦里常梦见欠着什么，醒来手心发热。急什么，五十八年都过来了。外头落雨。泥岗村的雨细绵，专找漏瓦钻，一滴两滴砸在坛沿，溅开。韩泥拿缺口陶碗接屋顶漏水，碗小接得慢，水漏进碗又漏进泥地，转眼没了。他忽然想，这辈子是不是也这样接雨——接一碗，漏一碗，到头来还是干的。他望着门口半筐没编完的野藤，又望坛角，雨一滴一停，极有节奏。村道那头赵绣春跟几个妇人笑，笑里带刺，听不清字，刺却扎人；赵绣春夫家做皮货，走路带风，利起来比刀还快。韩泥与她无冤也无恩，只晓得那人嘴利。他把碗搁下，坐到坛边，低声道：「不漏人就行。」

话音落下，坛壁轻轻一颤，极轻，像呼吸。雨珠落沿，蒙尘里浮出一线极淡水汽，热汤刚揭盖那一瞬，一闪又敛。韩泥以为手抖，没理会。雨大了，漏进屋里，打湿半边衣襟，雨丝打在他麻脸上，针扎，也洗，他没躲。泥岗村入夜早，远处喊娃吃饭，喊声拖长，断在雨里。灶冷，他不喊饿，饿惯了。「明日还得编。编了筐，换钱，买米，活。」

村童嚷「晦气翁，别近」，张麻婆咳一声：「少嚷。嚷多了嘴烂。」村童散了，张麻婆没近这间漏屋——近了，也沾晦气。韩泥嘴角动了一下，不知是笑，还是咳。他合上眼，手仍按坛沿，五十八岁，绝户，晦气翁，还在泥岗村西头这间最漏的屋里。掌心轻颤，数到三，颤停；沿口一线温，贴住虎口，脆，短，够撑一日，像十二文落袋那声。雨还在漏，一滴砸在他膝上，凉。他把门口半筐野藤往里拢了拢，免得雨泡烂，明日天不亮还得上山，苦蒿蛇藤都得备，篓绳旧了也得系紧，系紧了才装得住药。腰布贴着肚皮，十二文硌着，踏实。村嘲在远，坛在角，钱在腰。钱在，人还在。雨歇后，村道泥印里映出半轮月。韩泥把十二文又数一遍，数完收进腰布最里层——层在，明日上山采药才不虚。虚了，坛沿白擦，根也白擦。坛在角，雨在漏，人在——三角齐了，第一章就立住了。
