# 第一百一十六章 捡漏得

残页贴窑壁，韩泥按「焚火」二字行息。

息走三遍，窑温升，升得像首炼那日炉心未冷。冷不要紧，温才要紧——温稳，渣里香才正；正了，培元散才配再炼。再炼，为复测；复测，为三年约反面——钱戾衡说末席滚，滚要手抖；不抖，才有资格不滚。不滚，手在，粒在，坛在。

铁片嵌口，嵌稳，稳得像炉角归来。归来不要紧，露不得；露了，坊市捡漏成「盗器」。盗器，除名；除名，叶汤恩断，芽儿半块灵石也还不上。

赖福公示第三日，仍让韩泥吃饭最后。最后，粥薄，薄里刘婆仍舀满：「吃。空肚，手飘。」

韩泥吃尽，不辩。辩，费舌；舌要留来按诀，留来守窑温。守稳半日，气冲关，炼气八层实——实了，仍末席；末不要紧，手稳。手稳，白织月袖风也尖不过「正」字——「正」在远，先炼出来。

午后，他用捡来的铁片剔渣，剔出一粒未化灵砂，砂细，细得像漏网之鱼。鱼小，够掺一炉培元散——散不必多，多惹眼；一炉够验手，够备复测。复测在远，除名试炼在近；近，粒正，手稳，就不滚。

白织月过药山道，袖风带香：「丑杂役也玩火？」

韩泥不辩。火在窑，不在舌；舌响，香飘。飘了，赖福指「偷火诀」。指不要紧，手稳，管事嗅香正，诬就虚。

「玩火玩死。」白织月笑，笑薄。

韩泥按铁片，片温，温得像答——答在稳，不在嘴。嘴硬，窑温更稳；稳了，白织月袖风也尖不过「正」字——「正」在远，先炼出来。炼出来，才配九层，才配还沈枯芽半块灵石。

铁无言傍晚来，掷一小袋渣：「外门边角，你用。」

「谢。」韩泥只一字。

「谢轻。」铁无言说，「手稳，别漏。漏了，赖福笑真。」

韩泥应声，掺灵砂入渣，渣灰，灰里一点香。香起时，他鼻下静，静得像108兽潮那日——那日并肩，这日独炼；独炼不要紧，丹正才要紧。正了，藏怀，不示；示了，赖福克，白织月嘲。

炉心未起，窑温代炉。代炉，是丑翁路——路在渣，路在捡漏，路在五年淘。淘出来了，才配炼气九层，才配徒恩首还。首还，静着还，还到芽儿手里，才算数。

赖福带人闯窑道，嗓尖：「偷诀偷火，在此！」

管事弟子跟来，瞥残页，页碎：「坊市散页，非禁物。韩泥，炼什么？」

「培元散试手。」韩泥声平，「渣里炼，不耗宗门灵材。」

管事嗅香，香淡，淡而正：「正，记一笔。赖管事，捡漏非盗。」

赖福脸青，青完阴笑：「正也晦气。晦气，明日除名试炼。」

「试炼看手。」韩泥说。

试炼字落，像钩——钩向117压顶，钩向炼气九层。九层在远，八层在腹；腹里温一线，坛应「捡」——捡来的铁，捡来的诀，捡来的命。命在忍里，忍在活里，活在还恩前头。

捡漏得，尾摊角，一撮灰，灰里辛正。辛正不要紧，鼻下有根，根细，细才辨得出。辨得出，117九层才压得住。

他指到即捡，捡完，并入暗袋。袋沉半分，沉不要紧，沉才像路。路通向117，118还芽石，119十层稳。

赖福远眼细，细得像秤。扫完，没追——追不要紧，沿净，坛不露，晦气遮瓮。

夜，他再炼。散成，成一粒，粒丑，泥纹浅——泥纹不要紧，效正才要紧。效正，藏怀，不示；示了，赖福克，白织月嘲。嘲不要紧，试台见；见了，正字自现。

邻床少年悄问：「你真炼成了？」

「粒在怀，不在嘴。

少年闭嘴。闭嘴，韩泥摸玉牌——牌凉，三年约烫。烫在钱戾衡，烫在除名试炼前头。前头，赖福压顶最后一关；关过了，还芽儿石在前。

他低声记：「捡漏得诀，得铁，得砂。得了，才配九层，才配还沈枯芽半块灵石。」

坛腹温一线，应「正」——正一粒，够备复测；复测够了，才配130夺魁，才配还刘婆新衣养老钱。刘婆恩在远，先还芽儿——芽儿半块灵石，徒恩首笔。首笔，静着还，不张扬；张扬，像施舍；施舍不是，是还。

秦霜远木鱼远敲：「心别邪。」

韩泥不答，答在心里。心里七笔分列：叶汤①，芽儿⑤，刘婆⑦，分列，不混。混了，还乱；还乱，粒就不正。不正，刘婆恩还迟，叶汤还凉。

丑时，练鼻下静。香落齐，捡漏这一炉记进心里。记进心里，刃磨亮一分——亮在还芽儿，亮在赖福试炼，不在白织月袖风。袖风冷，忍过去；过去，试台见正。

外头公示榜仍挂他名，名末，真末。真末不要紧，腹里八层实，手里一粒正——正，是五年淘出来的，不是运气。运气白织月会说，会说不要紧，试台见正。

赖福廊外骂：「明日除名试炼，看你滚不滚！」

韩泥低声道：「手在，就不滚。」

手在，路在，坛在，粒在怀。怀沉，沉的是债序——债序在，步不飘；步不飘，才还得到芽儿那一半石。一半石，欠五年；欠着，丹不正；还了，才配炼气十层，才配复测备战。捡漏第二日，韩泥用铁片剔窑口余渣，剔出两粒未化灵砂，砂细，像116那粒姊妹。姊妹不要紧，够再炼两炉培元散——炉不必多，多惹眼；两炉够备复测。

赖福带人闯窑道，被管事拦：「坊市散页，非禁。再闹，记过。」赖福悻悻退，退时眼赤，赤得像117试炼在等。等在坡下，等在除名边缘。

白织月立窑外，袖风冷：「焚火诀残页，也配你？配不配，窑温知——温稳，散正；正了，试台见。见了，运气二字尖不过「正」。

铁无言并肩立，声低：「两粒砂，别示。示了，诬告又来。」韩泥应声，砂入罐，罐封泥。封泥像腌菜坛——坛相凡，蔽眼；蔽眼，手才稳。

午后炼第一炉，散成三粒，粒丑，泥纹浅，嗅之正。正，藏怀，不示。示了，赖福克，白织月嘲。嘲不要紧，手稳。

第二炉，气冲关，八层圆满，向九层迈步。迈步在117，不在今日。今日稳窑，稳粒，稳手——手稳，试炼才过，试炼过，赖福压顶尽。

刘婆端姜汤，姜薄，热：「窑里闷？」「闷。」「闷出正丹，老婆子等你试台。」韩泥喝尽，记——记刘婆恩，记将来130新衣养老钱。钱在远，粥在近；近，顶饥，饥顶命。

夜，他抚铁片，片温，温得像炉角归来。归来不要紧，露不得。露了，坊市捡漏成盗。盗不得，淘了五年，淘出正，淘出还芽儿资格。

坛腹温一线，应「炼」——炼粒，炼手，炼忍。忍了公示，忍了诬告，忍了赖福查席——查了，席严，坛在，手在。记进心里，不进舌。舌响，赖福说「晦气炼丹」；晦气不要紧，粒正。

他低声道：「捡漏得诀，得铁，得砂。得了，才配九层，才配还芽儿。」还芽儿，半块灵石在怀渐凑齐——凑齐，118首笔正式报恩；报恩，静着还，还到芽儿手里，才算数。

两炉成，九粒在手，韩泥封罐封泥。封严，不示；示了，白织月嘲运气，赖福说盗。盗不得，淘五年，淘出正，淘出还恩资格。资格在，步不飘。飘了，粒洒；洒了，除名；除名，玉牌白凉，钱戾衡笑真。笑真不要紧，试台见正，见还，见刘婆新衣养老钱。

他摸玉牌，牌凉，低声道：「三年约，记着。手稳，粒正，试台见。」见字落，坛温一线，像应。应了，路又明一寸——明向还恩，明向不滚，明向刘婆新衣养老钱。

更鼓沉，沉里，韩泥按木牌，牌凉贴胸。116章手在，名在册，坛沿净。舌不留，鼻下静。恩辱分列，不混。活长了，才还得了叶汤，才接得住门缝灯。

刘婆粥仍多半勺，姜仍多一片。片暖，恩不断。恩不断，手才稳；手稳，粒才正；粒正，试台才配说话。说话了，白织月袖风尖不过「正」，钱戾衡影硬不过手。

赖福廊外晃木牌——晃不要紧，韩泥不辩；辩，费舌；舌要留来守温，守窑。
