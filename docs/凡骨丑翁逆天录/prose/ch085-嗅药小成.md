# 第八十五章 嗅药辨性小成

再遇后第三夜，丑时。

六包样并息，并出一线清。清里，药性归位：辛沉引火，苦浮叶家，腐底毒苗。三线各归各位，各位才说话——说话，才是小成。

药渣场五堆混灰，他分拣，指稳，五堆未错一堆。未错，管事眉松，记「辨性准」。赖福讽：「准了也炼不出丹。」韩泥低声道：「准在鼻，不在舌。」

温听雨过风口，瞥五堆，眉松：「鼻下有根，根像真的。」根像真的，韩泥不谢。谢轻，记在心里——记小成，记缘苗温一线，记86续练在前。

铁无言摆场在渣场角，角风硬，硬里灰热。热罩脸，脸罩心，心仍要分堆——分堆，才是小成前最后一验。

第七样未知灰，他鼻下一息，指落：「沉。」第八样：「腐。」第九样：「浮假。」连中，连中稳，稳不像飘。

小成在鼻，不在舌。舌响，香飘；香飘，温氏就验出错。验不出错，86十堆加验才接得住。

他收样入柜，柜锁丑，锁像规。锁住了，才不示人；示人，谣又来。谣不要紧，晦气遮瓮。

更鼓沉，沉里，他摸怀玉牌，凉进骨。凉不要紧，鼻暖——暖在诀，暖在「各归位」三字。

角柜六包样，再摊席前。席严，坛不露，样在膝，膝稳，稳像外堂阶净那一帚。

韩泥鼻下静，沉腐浮并三包，再并三包，并出一线清——清里，药性归位，归得像秤上三堆灰各归各斗。

第一样，辛沉，沉底一线白——白像引火，引火轻，记重。第二样，苦浮，浮少，浮也要剔。第三样，腐底，底真，真在毒苗，毒苗剔净，筐才净。

并息到极，颅骨里又一声轻，像瓮唇吐息。吐息里，三字加长半分：

「沉腐浮，各归位。」

加长不要紧，加长才是小成——小成不在舌，在鼻；鼻里，药性各归各位，各位才说话。

韩泥屏息，独眼不眨。他低声复诵，只复给自己听：「辛沉引火，苦浮叶家，腐底毒苗。」

复诵完，指到一堆未知灰，不需细想，鼻先落：「沉。」指到第二堆：「腐。」第三堆：「浮假。」

三次连中，连中不像炫，像活——活长了，嗅药才小成。

小成起，渣场风硬那日铁无言指灰，药山深行界碑，废渣路焦回，温听雨验手——一线串起，串成像鼻下那条根。

邻床翻身，眯眼半息：「啥味儿这么清……」

「坛味。」韩泥声平，「睡吧。」

小杂役哼一声，又睡实。睡实了，诀才藏得住。

他收样，包实，塞回角柜。柜锁丑，锁像规——锁住了，才不示人；示人，谣又来。

更鼓将尽，天边白一线。白里，他廊下立半刻，任风硬，鼻下仍清——清像嗅药小成那口息，息不落舌，落鼻。

卯时，药渣场验手。

管事弟子指五堆灰，灰混，混要分。韩泥分拣，指稳，五堆未错一堆——未错，管事眉松一线。

「丙九韩泥，辨性准。」管事记簿，笔丑，真。

记准，是小胜。赖福立远处，眼细：「辨性准也末等，准了也炼不出丹。」

韩泥不接茬。接茬，全丙九饿。他低声道：「准在鼻，不在舌。」

铁无言走近，瞥分堆：「小成了。成在鼻，别叫。叫了，像求诀名。」

韩泥应声：「不叫。沉腐浮，各归位。」

铁无言点头，走了。走，也像认可——认可不重，重的是渣场仍收他，重的是温听雨若来，也验不出错。

午后，温听雨果然过风口，裙素，袖香温。她瞥五堆，瞥完，眉松：「昨日叶渣另堆，今日五堆未错。错一堆，全丙九记名。」

未错，她没夸，只留半句：「鼻下有根，根像真的。」

根像真的，韩泥不谢。谢轻，记在心里——心里嗅药小成，心里缘苗温一线，心里恩愧仇三分列。

刘婆端粥，仍多半勺：「辨性准了？」

「准了。」韩泥说，「小成在鼻。」

「鼻成怕什么。」刘婆哼，「饭稳，恩不断。」

韩泥点头。恩在粥里，也在小成里——成轻，记重；记重了，才接得住小成续，才接得住三层破。

秦霜木鱼一声：「成小，心别骄。骄了，香飘。」

韩泥不答，答在心里——心里不骄，只记。记小成，记四层影仍在，记三层将破在丑时。

他按席角，席严，坛不露。坛腹温一线，贴掌，像应「小成」二字。应完，他低声对坛：「嗅药小成了。不叫。坛在，人在。」

坛不应字，只温。温，像把「各归位」摁进掌纹那路里。

邻床问：「五堆全对，厉害？」

「不厉害。」韩泥说，「鼻下有根。根在，手才准。」

少年闭嘴，像也学会忍。

丑时前，他再验五包未知样，连中，连中稳，稳不像飘——飘了，管事记「邪」。

他独眼平，手不抖：「小成了。续在后。三层将破，四层影在等。」

刃不亮，先记。记完了，才配嗅药小成续，才配暗修三层。

玉牌在怀，愧字在怀，温氏名在记——记里，不混。混了，心就乱；心乱，手就抖。

更鼓沉，沉里，坛腹应温，像应「辨性小成」最后一笔。应完，他不叫。不叫，才守得住诀，守得住95那盏门缝灯。

他低声道：「小成了。坛应了。明日续练，丑时破三层。」

赖福讽远：「末等也辨性？辨出邪。」

韩泥不接。舌要留来鼻，留来守坛。守坛，守小成，守各归位这一条根。

柱上条角白一线，春风硬，硬不要紧，鼻清——清在，就不滚。

他合眼不睡实，鼻下仍清一息。清里，辛沉引火，苦浮叶家，腐底毒苗——三线各归各位，各位才接得住丹路开。

铁无言夜过渣场，瞥他角柜：「小成了，别示样。示了，赖福记过。」韩泥应声：「不示。沿净。」

温听雨名在记里，不碰舌。舌碰，缘线就邪；邪了，叶愧线也乱。乱，手就飘；飘了，小成白来。

他廊下再练半刻，五包未知样连中，连中稳。稳了，86十堆加验才接得住——接得住，三层丑时才破。

小成后第二日，药渣场未知五堆。韩泥分拣，鼻下清，五堆连中，连中稳。

管事记「辨性准」。准，是小胜。铁无言再取灰一撮：「再闻。」

韩泥鼻下一息，指落：「沉四，腐一，浮零。」铁无言分秤，对，点头：「小成实了，续在后。」

温听雨风口立半息，裙素，袖香温，她瞥分堆，瞥完，眉松：「全对不容易。」

不容易，韩泥不谢。谢轻，记在心里——记她第三回验手，记缘苗温一线。

午后外堂洒扫，阶净，栏净，各做各的。温听雨没多问，巾一收，走了。

黄昏，角柜样十一包，包实，锁柜。

邻床问：「小成能炼丹了？」

「鼻成了。」韩泥说，「丹在远。远在95，近在槽。」

丑时前，廊下练半刻。五包并息，并出一线清。清里，掌纹影颤，颤像三层将破。

他低声对坛：「小成了。续在后，不叫。」坛不应字，只温。

老耿挑水过，声哑：「成小别拜诀。拜，路断。」韩泥只答：「不拜。记着。」

更鼓沉，独眼平，手不抖：「嗅药小成了。十堆加验在后，三层丑时破。」

活长了，才还得了叶汤，才接得住门缝灯。

舌响，香飘；香飘，手就滚。滚了，管事记邪。不滚，五堆全对，缘苗温一线，三层将破在丑时。

管事夜巡验棚，灯照席角，沿净，记「手稳」。手稳，是小胜；胜在活，不在嘴。

刘婆端粥，仍多半勺：「手稳，饭稳。饭稳，恩不断。」韩泥点头，谢轻，恩重。

铁无言抱臂立侧，声低：「灵根末，手稳，渣场仍收你。别漏。」韩泥应声：「不漏。」

秦霜远廊经过，木鱼不敲：「心别邪。邪了，鼻下香飘。」韩泥不答，答在心里。

白织月远廊经过，袖风冷，没停。没停，小胜——胜在活，不在嘴。

铁无言夜过渣场，瞥角柜：「小成了，样别示。沿净。」

他低声对坛：「嗅药小成了。不叫。坛在，人在。」瓮不应字，只温，像应验。
