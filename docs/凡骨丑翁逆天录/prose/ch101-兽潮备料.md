# 第一百零一章 兽潮备料

兽还没撞到界上，备倒先到了。备落在栏里、窑里，也落在袖口那几只丑囊上。

天亮，韩泥清栏又多清一趟。粪铲下去稳，稳得像编筐勒藤——藤勒匀了，筐才卖得出去；栏清净了，甜腥才少往里钻。湿草他换了两遍，草潮着，潮着才咬得住那股又甜又腥的味。铁无言丢来一袋石灰，嗓子硬邦邦的：

「撒角，别撒眼。石灰进眼，比瘴还疼。」

「嗯。」韩泥应一声，撒得匀。匀不是为好看，是为下一更真撞上来时，角还能站住。

夜里丑时落雨，他扛着满筐渣上山。不是试火，是炼备——清瘴那点料，得靠渣和臭灰藏着做。经页上「清瘴」只剩残诀，字浅，浅归浅，真劲还在。青芯两撮，褐渣一把，石灰半指，炉温起半寸。半寸稳，稳里出不来丹，只出一团灰香。他凑近嗅了半息：香苦，苦得对味，像真能把甜毒顶退半寸。

灰香分成三只丑囊，缝脚歪歪的，看着就像杂役随身带的烂药包。歪也好，歪才不像偷火。管事弟子在夜禁口验筐：「丙九韩泥？」

「倒渣。」韩泥立在队里，把筐口摊开，「满了。」

验完，放行。放行他也不喘大气，只把袖口扎紧——香在袖里，人在队里，一个都不能露。

赖福查夜嗓子尖得刮人：「谁上山？夜禁加严，偷溜记过！」

「倒渣。」韩泥又报一遍，「筐满，条上写着的。」

赖福火折照他脸照了半息，啐一口走开。走开也好，走开才说明今晚那点灰香还没被人闻破。

回棚，刘婆坐在灯影里补他肘上那块布，针脚粗，粗得挡风：「备够了？」

「够了。」韩泥道，「手稳。」

「稳就好。」刘婆把线咬断，「别到潮来了手飘。手飘，栏先倒。」

沈枯芽送来半瓢凉水，眼睛瞄他袖口：「叶姐说……别逞能。逞能死得快。」

韩泥接水，不接这句话。话接在心里：逞能不在嘴上显，显在潮真来时手还不抖。白织月立在廊那头，袖风一带香，声音冷：

「丑杂役也备？备了别拖后腿。」

「不拖。」韩泥只回二字。二字够了——拖不拖，要看角漏不漏，不看她袖子飘不飘。

同袍凑过来问：「你袖里鼓鼓囊囊，塞啥？」

「干草。」韩泥扯开给看一眼，真有草，「蒙鼻用的。」

那人哼一声走开。走开正好，灰香便藏得更实。实了，才配叫备。

夜里兽吼又近了一线。栏里小兽齐齐嘶起来，嘶得像潮头先咬了一口。韩泥立在东角，刘婆给的布条蒙上鼻，三囊在袖，经页贴怀，掌心那道纹也温着。一眼看去，像个只会铲粪的丑翁；他自己清楚：备齐了，下一更才接得住近处那股压。

丑时廊下静了半刻。香落齐了，灰香拿到鼻下一试，仍苦而正。正够备，不够战——战得等潮真至。他掌心轻轻按了按怀中坛沿，低声道：

「备齐了。下一更，潮就近。」

坛腹微温一线，像听懂了「备」字。备不是退。退字他册子上没有。

更鼓尽时，界柱上的条又换了。字更急，急得像锤子要落：

「明日黄昏，兽潮或至。」

「或至」俩字看着软，韩泥当作「必至」读。至了，手得在；手不在，备白做。

第二天白日他又多清半趟栏。铲齿磕石上，火星不溅，只溅一点潮湿土腥。铁无言分障草，草湿，顺手又塞他两包干灰：「东角多备一份。近了别跟人抢墙心。」

「不抢。」韩泥道，「角够。」

角够，就够他活。墙心是内门的体面，体面跟他无关；他只守栏不倒，栏不倒，丙九才少一份连坐。连坐一落，粥也薄，恩也难还。

废窑那条路他熟。熟不是炫耀，是丑时雨还能摸到炉口不摔。炉口边他蹲着理料，青芯、褐渣、石灰各归各堆。堆乱了，香也就邪；邪了，秦执事闻一声，这备就变「偷炼」。

「韩叔，窑里烟大不大？」沈枯芽在山下探头，嗓子压着。

「不大。」韩泥往外递半句，「你别上。上了记你过。」

沈枯芽缩回去。缩得好——娃的名别往夜禁簿上蹭。

管事弟子路过废窑坡下，远远喝一声：「倒渣的快点！夜禁前归栏！」

「归。」韩泥应着，把最后一囊灰香塞进袖底。囊缝磨手，手一刺就醒——醒着，备才不散。

第三备其实在心上。心记清楚：兽至，先清瘴，再清栏；清栏，人不逃。逃了名上记逃，活路也断；不逃，才有往后并肩那一步的资格。资格不在今日，今日只把料备实。

夜里他在东角踱两回。湿草踩得吱呀响，响里他听远岭吼声，吼一声，近一声。近不是吓他，是催他。催得袖里三囊贴腕更紧。

同袍低声骂：「这破日子，备到何时是个头？」

「头到潮走。」韩泥道，「潮不走，备不停。」

那人骂完又问：「你真备齐了？齐了分我一口？」

「齐是齐。」韩泥摇头，「分不得。分薄了，角上先破。」

破在角上，活该他自己扛——他选东角，就是认了这块苦。

丑时雨停。他回棚，布条晒半干，又往袖里塞。刘婆盯着他胳膊肘：「疼不疼？」

「不疼。」韩泥道，「疼着才记着还活着。」

刘婆哼一声，算听懂了。

柱上条被夜风刮得扑扑响。韩泥路过再读一遍「明日黄昏」。黄昏不远，备也不能再远。他把灰香、石灰、布条、木牌一件件按进怀里，按完才觉踏实。踏实不是胆子大，是活儿做完了。

更鼓又沉下去时，他贴着坛沿坐了一息。息里不求境动，只求手不飘。手不飘，明日条上的字落下来，他还能接着。

夜里最后一趟查角，铁无言从暗处抽出半边肩：「备齐了？」

「齐了。」韩泥说，「三囊，诀在。」

「诀别出声。」铁无言叮一句，「出声像邪。邪了，比兽潮先死人。」

「嗯。」韩泥把话吞进肚里。吞住了，备才干净。

远岭又吼一声。吼完，栏里小兽轻轻一颤。韩泥按住栏木，木凉，心却热着。热不是火，是明日黄昏那条字已经贴进骨头里——字在，手就得在；手在，潮近了才不至于空袖。

备齐最后半日，他仍没闲着。石灰再匀一遍，湿草再压一层，袖囊的绳结重打。重打不是手贱，是怕撒时散口。铁无言过来踢了踢障脚：

「角泥软了，垫石。石垫稳，人站着才不打晃。」

「垫。」韩泥搬石埋实，「晃了就漏。漏了喊你。」

「喊得响。」铁无言哼一声，「别学内门那套闷气。闷气死得快。」

有个新来的外门弟子路过，掩鼻嫌臭：「守栏的也配谈兽潮？兽潮是阵上的事。」

韩泥不抬眼：「阵上断了，臭先淹你们。淹到了，再嫌晚。」

那弟子脸一僵，想骂，见铁无言站着，骂又咽回去。咽回去也好，口水救不了栏。

沈枯芽又来送线头，线头是刘婆给的：「缝布条。条破了，甜腥直往鼻子里钻。」

「替我回一句。」韩泥接线，「布条恩我记着。记着，往后还。」

「还啥时候？」

「潮过了，还。」韩泥道，「潮不过，人先在。」

午后他抽空上废窑，把炉灰扫净，净了才像没事人倒过渣。管事弟子冷不丁从坡下抬头：「丙九，夜禁簿上你名多。名多不是功。」

「不是功。」韩泥扛筐，「是活儿。活儿做完，名爱怎么写怎么写。」

「嘴硬。」

「硬比飘强。」韩泥走下坡，脚下稳。稳着，备才算落进日子里，不是落在嘴上。

夜里最后听条，他站在界柱下又念一遍「明日黄昏」。念完，把灰香、布条、木牌并成一摞按进怀。按完对坛沿说半句：「听着。备齐了。近处一来，你别乱温——温多了，惹眼。」

坛仍只温一线。一线够他踏实。踏实着回东角，角上夜风大，大也得站。站到更鼓再沉，潮在岭后磨牙——磨着，他也磨着袖囊绳结。绳结发亮时，他再看一眼柱上那条「明日黄昏」：条在，备才算钉死；明日一到，近处便来讨账。
