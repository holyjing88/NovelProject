# 第十二章 培元被扣

月初发丹，丹堂库房后巷最热闹，也最脏。

弟子排队领培元丹，瓷瓶碰瓷瓶，叮当像雨。顾小满排在末，怀里揣着半块灵石磨成的薄片，怕丢，更怕领不到——杂役例份少，一颗培元丹够她半月苦力不垮。

赵玄机立在库房内侧，执法堂铁牌在腰，笑不挂脸，权挂腰。

他接过司库册，指尖在「莫长春」三字上停了一停，像点一枚将熄的灯，然后提笔，把长老例份划进另一栏：「赵玄机·执法堂候补。」

司库迟疑：「赵执事，这……」

「将死之人，吃了也是浪费。」赵玄机淡淡道，「省下来，给还能为宗门出力的人。」

话落，他袖中落瓶，动作快，像风过无痕。

顾小满在巷口看见。

她本要躲——杂役撞见执法堂私事，历来没好下场。可那划册一笔，像把半块灵石也划走了：长老若连例份都保不住，杂役更保不住什么？

她攥拳，指节白，却未出声。

赵玄机出巷，与她擦肩，停步，俯视：「看什么？」

「看赵执事手稳。」顾小满低声。

赵玄机眯眼，忽笑：「手稳才好抓贼。你若乱说，执法堂也稳。」

他走了，铁牌反光掠过她眼底，像一刀未落的影。

午后，顾小满在杂役堂浆洗，孙福来查账，算盘打得脆。

「小满，上月灵石去向，再报一遍。」

「半块购粥，余下缴堂。」她答得平。

孙福哼：「缴堂？你当老夫瞎？」他甩出一纸，上头记着克扣条目，条条属实，却缺最关键一行——赵玄机私扣培元丹，无纸无凭，只有她一双眼。

顾小满盯着纸，两难像两块石头压住喉：

告，赵执事有执法堂；不告，长老例份已失，九月老头连丹都吃不上一口。

她想起莫长春赠符时那句：「恩不用嘴还，用命还的时候，别还错了人。」

夜，她绕到执法堂侧墙，未进门，先至莫长春小院外，跪坐阶下，像跪一块半石前。

莫长春开门，见她，不讶：「小满？」

顾小满捧出抄本，是孙福历年克扣杂役的私账，她早抄了一份藏怀里：「长老，培元丹……赵执事私扣了您的例份。」

小院静。

莫长春接册，未翻，先看她：「你看见了。」

「看见了。」

「怕不怕？」

顾小满喉头紧：「怕。可更怕长老连丹都没有，外头还说您靠女人、靠杂役——小满不想靠长老丢人。」

莫长春须眉一动，像笑，又像叹。他扶她起：「起来。跪着，像杂役堂训话；站着，才像还恩。」

他翻册，孙福账目清楚，赵那一笔却空，空得像执法堂铁牌里的冷。

「你打算怎么办？」莫长春问。

顾小满咬唇：「小满……想请长老定。」

莫长春把册还她：「册子你留着。明日去执法堂，不必跪，把话说明白。老夫也去。」

顾小满抬眼：「长老会护小满？」

「老夫护的不是你一个人。」莫长春道，「护的是丹在谁手里不要紧，心在谁那边要紧。」

她不懂全句，却懂后半：心在她这边，她就不怕了。

更深，赵玄机在房中数瓶。

培元丹三颗，瓶身温，像偷来的寿。他忽觉背脊一凉，像有人隔墙看他，不是莫长春，是更远的柳氏剑意，冷而未出鞘。

他自语：「将死之人……我何必要怕。」

却把瓶藏进暗格，藏得比罪还深。

簿在莫长春袖中无字，却沉了一分，像账将起。

他望执法堂方向，念：「明日堂上，看谁的心稳。」

大因果还在远处，像妖潮，像赤焰——培元被扣只是小账，堂上辩才是大还礼的前奏。

更深，赵玄机在房中数瓶。培元丹三颗，瓶身温，像偷来的寿。他忽觉背脊一凉，像有人隔墙看他，不是莫长春，是更远的柳氏剑意，冷而未出鞘。

顾小满回通铺，把抄本贴身藏好，藏得像藏半石。她闭眼，簿她看不见，却觉青岚峰夜风软了一寸——软得像有人替她定了明日堂上的路。

库房后巷，围观弟子不知私扣，只知赵执事手稳，稳得像执法堂铁牌反光。有人低声：「将死之人，例份也保不住？」另一人答：「保不保，堂上见。」

赵玄机出巷，与她擦肩，停步，俯视：「看什么？」「看赵执事手稳。」顾小满低声。赵玄机眯眼，忽笑：「手稳才好抓贼。你若乱说，执法堂也稳。」

莫长春立廊柱旁，望库房方向，低声：「培元被扣，是试赵小子。试过了，堂上见。」簿无新字，袖却沉，沉得像忠奸考验才开头。

大因果还在孙福告状那日，像谣言，像堂辩——小扣是饵，大还礼还在后头。
