# 第九十一章 废窑试炉

药山风硬。

冬后第三月，韩泥清栏归来，掌茧裂处渗潮。潮不要紧，要紧是鼻下嗅诀已成——成在渣，成在废，成在赖扒皮克扣里淘出的那一点香。

卯时前，他清栏第一趟。栏里小兽角咳一声，咳轻，像昨夜没睡实。韩泥换干草，草干，味淡一寸。管事路过，记一笔：「丙九韩泥，清细。」

同批少年掩鼻：「你怎不怕腥？」

「怕。」韩泥说，「怕也清。清完，才有饭。」

少年退半步，没再问——问，像沾晦气。记细，是小胜。胜在活，不在嘴。

铁无言抱臂立栏边，声低：「药山废渣场，午后有人倒炉灰。灰里若有青芯，别当泥。」

韩泥应声：「记。」

记，不为捡便宜，为试炉。试炉在废窑，不在舌。舌响，手就飘；飘了，坛也藏不住。

午后，他借送药渣之名上山。药山半腰，旧窑塌了半边，窑口红，红像烧过又凉。凉不要紧，灵脉浅一线，浅才没人占。占，要手稳，要丑，要末席——末席占穴，管事少问。

韩泥入窑，窑矮，他驼背，头几乎顶梁。梁上挂蛛网，网旧，网里落半片药渣。渣灰，灰里一点青。青芯，他鼻下一动，香淡，淡而正。

他从怀里取出瓮底泥炉——炉小，像碗，像爹留的陶片拼的。拼不要紧，拼得稳。稳，才配试头一炉。

废渣分拣，指稳。青芯三撮，褐渣一把，灵泉水半葫芦——水少，少不要紧；少，才试得出火路。火从掌纹那路引出，引到炉底，炉温起半寸，起得像坛腹应温。

第一炉，火猛。猛了，渣焦，焦味苦，苦里带腥。腥不要紧，试炉本就要焦几次。他熄半息，掌按炉壁，壁烫，烫进茧里——烫，像记。

第二炉，火弱。弱了，青芯不发，发不出正香。正香在鼻下，不在眼。眼昏，像赖扒皮盯梢；盯不要紧，窑深，盯不到。

第三炉，火匀半寸。匀了，渣里浮一点白气，白气散，散前他嗅到一丝培元路——路细，细像叶汤那日碗沿的热。热在记里，不在炉上；炉上只要稳。

管事弟子巡山，声远：「杂役别乱钻，钻了记过！」

韩泥应声：「送渣。」

管事瞥他筐，筐满，满像常活。常活，才藏得住试炉。管事走了，走了也像没瞧见窑口红那一瞬亮。

韩泥收炉，炉温退，退得像坛腹敛温。敛不要紧，试过了，路开了。路开，不在嘴，在掌——掌纹那路，又多一线细脉。

下山路上，沈枯芽探头又缩，缩像怕赖扒皮。韩泥不近，不求，只记——记沈枯芽怯，怯是叶家线，线在远。

黄昏回兽栏，赖福堵侧门：「上山了？上山干什么？」

「倒渣。」韩泥声平，「筐满，手稳。」

赖福眼细，细得像秤：「秤过，少二斤青芯。」

「看错了。」韩泥说，「青芯在灰里，灰在筐底。」

赖福掀筐，掀见灰，灰里无青——青在袖袋，袖袋扎紧。扎紧，沿才净；净了，坛才藏得住。

刘婆端粥，仍多半勺：「又上山？」

「倒渣。」韩泥说，「手稳，饭稳。」

刘婆哼：「饭稳，恩不断。」

老耿挑水路过，水洒稳，声哑：「试炉别拜邪。拜，路断。」

韩泥只答：「不拜。坛在，人在。」

夜，席盖严。韩泥按坛，低声道：「试过了。窑在，路在。」

坛腹温一线，贴掌，像应「试」字。试，不是成；成在远，远不要紧，手在。

邻床问：「你上山，怕不怕记过？」

「怕。」韩泥说，「怕也试。试过了，才还得了恩。」

韩泥编筐半刻，藤刺扎掌，刺浅，浅些，沿才净。筐勒匀，匀像火路——勒不匀，渣散，路断。

赖福脚步在廊外停半息，停，像盯。盯不要紧，窑红已凉，凉得像没亮过。韩泥独眼平，手不抖，低声道：「试炉一回，不叫。」

丑时前，他廊下练半刻鼻下静。粟壳香淡，淡不要紧；要紧是香一字不落。落齐了，废窑才站得住第二日。

更鼓将尽，药山风硬，硬里带一点窑温余韵。余韵散，散进掌纹，散进心记——心记多了一行：废窑可试，火要匀。

瓮腹应温时，韩泥独眼半阖，内视里阶影稳在五层。嗅诀小成，沉腐浮三字在鼻下一字不落。

根在，试炉就不白试。下一回，火要更匀。
