# 第六十七章 冬测风声

冬风硬了一分。

硬里柱上条角起霜。霜化，字更显——「冬后正测，凡骨杂役，卯时坡下见。」

韩泥读条，独眼平。正测不是备检，是公示——公示灵根，公示末席，公示谁滚谁留。滚字在条上，不在心上；心在坛，坛醒了，滚也要手稳才滚。

同批少年抖：「听说内门弟子也来。来，看笑话。」

「看笑话不要紧。」韩泥说，「手抖才滚。」

少年愣半息，没再问。赖福立条下，笑薄：「丑杂役，正测公示，你灰线三尺，滚定了。」

韩泥不辩。辩，费舌；舌要留来按石，留来守坛。管事弟子喝：「丙九集合！听训！正测前，凡骨杂役，手抖的，先滚！」

训声沉，沉里，白织月过廊角，袖风带香，香冷。她瞥丙九队列，瞥到韩泥末席，唇线绷一息：「凡骨也配听正测训？配站坡下？」

韩泥「嗯」一声：「管事看手，不看嘴。」

白织月走远，走远还留一句：「手稳有屁用。灵根末，丹路断。」

话冷，冷不过叶汤。叶汤热过，热在胸，胸热，手便稳。这是小胜——胜在少辩一句，胜在舌留住了。

铁无言抱臂立侧，声低：「测灵验色，也验手。手抖，色再正也滚。」

韩泥应声：「手不抖。」

秦霜远廊经过，木鱼在指间，不敲：「心别邪。邪了，鼻下香飘。」

韩泥不答，答在心里——心里七笔不混，正测是秤，不是拜。拜邪，路断；不拜，各守各线。

管事弟子读条三遍，条字像钉：「三日后卯时，公开测灵。末席公示，滚者除名。」

三遍，钉进丙九每人耳里。耳里响，手不能抖。抖了，钉就真。韩泥摸木牌，牌凉，膝热——热在坛，热在路，热在「仍末」二字还没写滚的那一日。

他按心口，七笔分列，不混。泥岗六笔沉，宗门刘婆一笔烫——烫在肘，烫在粥，烫在正测前这三天。烫在，就不怕白织月的嘴；嘴冷，手不能冷。

午后，坡下空，空得像等一场雨。雨没来，风先来，风里有人练剑，剑光一闪，闪得像内门的眼。眼在，笑话在；笑话不要紧，手稳才要紧。

邻床少年悄问：「内门弟子真来？」

「来。」韩泥说，「来也不近。近晦气，远笑话。」

少年懂「笑话」，闭嘴。闭嘴，像也学会忍。赖福传话，话毒：「钱戾衡师兄要来看正测。来看，专看末席滚。」

韩泥指节白一息，白完，又平。钱戾衡——名在仇线里，名在将来，不在今夜。今夜，他只清栏，只嗅渣，只守坛。

小杂役悄问：「你怕不怕？」

「怕。」韩泥说，「怕也记。记着，手就不飘。」

飘了，路断；路断，粥就凉。他低声对坛：「正测将至。至了，仍末，也不滚。滚要手抖，我不抖。」

坛不应字，只温。刘婆端姜汤，姜薄，热：「正测前，别空肚。空肚，手飘。」

韩泥喝尽：「不空。姜在记里。」

铁无言路过，声低：「坡下见，验手，不验嗓。嗓响，香飘。」

韩泥只答：「鼻下静。」

午后，他过坡下石阶，阶滑，霜白。他立末排，独眼平，像试站——站要稳，稳了，三日后才验得过。阶上风硬，硬不要紧，手在，帖在，名在册。

管事贴公示榜，榜在坡下柱，柱在风硬处。榜字丑，真在「滚」字。丙九少年挤看，挤看不要紧，挤末才滚。韩泥立末，独眼平，像先验站次——站次定了，手才定；手定了，滚字才不好随便抹。

廊角有人练剑，剑光两闪，闪完，一句笑：「凡骨也配站这阶？」

韩泥不答。答，费舌；舌要留来按石。那人走远，走远也像笑话——笑话不要紧，手稳才要紧。

管事又添半行小字：「公示末席，滚者除名；手抖者，先滚。」小字丑，真；真在「手」字，不在「灵」字。韩泥读半行，不近，不躲：「验手。手在，名在。」

管事弟子另发半张薄册，册丑，字密：「正测规矩，丙九每人抄一遍。抄错，先滚。」丙九席地抄，席地不要紧，席末也是席。韩泥抄末排，笔拙，字丑，丑不要紧——丑在纸，不在手。抄到「末席公示，滚者除名」，笔停半息，又落：落齐了，才像把条钉进掌里。

同批少年抄到手抖，抖半息，管事喝：「抖，先滚！」少年立刻按住腕，腕稳，字才稳。韩泥瞧一眼，不教——教了，像显摆；显摆，近晦气。

廊下执事抬测灵石过，石温，光淡。光淡不要紧，光要照手——手抖，光照出抖；抖了，滚字就真。韩泥立侧，独眼平，像试站末排：站要稳，稳了，三日后才验得过。

石光扫过他掌茧，茧裂旧，光落不上灵色，落得上手稳。手稳，石光淡也照不出抖——照不出抖，滚字就还没钉实。执事瞥他一眼，没嗓——没嗓，也像小胜：胜在活，不在嘴。

钱戾衡名从廊下飘过，飘冷，冷在将来，不在今夜。老耿挑水过坡，水洒稳，声哑：「坡下见，站末不要紧。末，不滚；滚，看手。」韩泥应：「手在。」

秦霜远廊又过，木鱼轻敲一下，又止：「心别求胜。求胜，像借香。」韩泥心里七笔分列，不混——正测是秤，秤在执事手，手在他掌上；掌上凉后稳，稳，才验得过公开测灵。

更鼓沉，柱上条角白一线，条被风吹得猎猎。猎猎不是催滚，是催手稳。他摸怀木牌，牌边起霜，霜化，字更显——「仍末」二字，像被冬风又摁实一摁。

忍不是怂，是刃。刃不亮，先磨；磨在丙九，亮在还恩那日。坛腹温一线，贴掌，像爹的手又搭上来。他低声对坛：「三日后卯时，坡下见。见在末排，不在舌。」坛不应字，只温一线，温完又敛。

夜前，他又清栏半刻，铲下稳，稳得像把「手」字先写进掌里。邻床少年悄问：「你真不滚？」

「滚看手。」韩泥说，「手在，就不滚。」

少年闭嘴，像也学会忍。忍不是怂，是活路——活路通向坡下，通向公开测灵，通向还恩那日。

管事弟子夜巡，灯照丙九名册，照到韩泥「仍末」二字，笔尖停半息：「正测前，别空肚。空肚，手飘。」韩泥应：「不空。」应完，不近——近，像求情；求情像拜，拜了，路断。

铁无言远廊半句，声低：「公开测灵，验手先。手抖，色再正也滚。」韩泥应：「手不抖。」

秦霜远廊风过，木鱼不敲，只留一句：「忍刃不亮，先磨。磨在丙九，亮在还恩。」韩泥不答，答在心里——心里七笔分列，不混；混则还错人，汤就凉。

坡下石阶将晚又起霜。霜白，像给三日后先铺一层素净。素净不要紧，手在，路才直；路直，才还得了叶丫头那碗烫。

天边白一线，条在柱，柱在坡下。坡下空，空得像白纸等那一笔——笔在执事手里，手在他掌上；掌上凉后稳，稳，才验得过公开测灵。

赖福立条下，笑薄：「丑杂役，三日后专看你滚。」韩泥只答一字：「在。」在，接得住那一验——验手，也验忍。

他摸怀木牌，牌凉，膝热——热在坛，热在路，热在正测前这三日。三日后卯时，坡下见；见在末排，不在舌。舌响，香飘；香飘，手就滚。

更鼓将尽，丙九棚静。静里，韩泥独眼平，手不抖——手不抖，才接得住公开测灵，接得住羞辱，接得住内门的眼。

柱上条在风里猎猎，猎猎不是催滚，是催手在。他低声对坛：「三日后，坡下见。」坛不应字，只温一线，温完又敛。敛着，才守得住秘密，守得住仍末，守得住公开测灵那一日。

钱戾衡要来，要来专看末席滚——滚看手，手在，就不滚。

柱上条换，字真：「冬测备检。」备检在坡下，验手不验灵色，仍末也要沿净。
