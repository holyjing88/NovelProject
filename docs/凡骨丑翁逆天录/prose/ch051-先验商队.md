# 第五十一章 先验商队

三日后，卯时，廊下霜白。丙九列队，末排先验。韩泥立最前——前，不是荣，是险；险在「滚」字，滚了，麸皮停，记功抹，七笔恩断。

管事敲锣两声，廊静。秦霜立廊角，木鱼不敲，只托：「辨香半刻，报香一字不落。漏一字，记过；记过两次，滚。」

赖福在尾，尾里窃语。语轻，像等晦气滚。韩泥木牌在怀，帖在袖——袖里旧条作废，作废，像昨日数完。数完，今日只验鼻。

香囊三只。韩泥闭息，鼻先静。粟壳淡，丹烟苦底，第三囊甜腻藏腥——腥在底，不在面，像栖香谣散了，散成规矩。「粟壳淡，丹烟苦底，甜腻藏腥。」他声哑，声平。

秦霜「嗯」一声，不夸，只记簿。铁无言立柱边，验他鼻下静气，声低：「静，别贪快。贪快，香飘。」

半刻尽，韩泥一字未漏。未漏，末排先验过——过了，不是灵根好，是鼻下稳。秦霜目光扫到队尾赖福，声冷：「再嗓，记名。」

赖福哑了。哑，像秤空一格——一格空，韩泥肩上的路却实一寸。

管事贴新条：「辨香过，候凝丹试手。试手在三日后。这三日，仍搬柴。」韩泥接条，条薄，真——真在「仍末」，真在路未断。

同批少年窃语：「末排也配先验？」管事冷眼：「再嗓，滚。」窃语散了。

午后，宗门外驿道尘起。西驿商队过，驼铃远，车辙深。五脉旗号杂：佛门灰、密宗赭、西教白、密院黑、沉丹青——青是主场，杂是客。齐不要紧，要紧是界；界在门内门外，门内末排，门外商线。

韩泥搬柴经侧门，窄得只容半眼。半眼，够看商线。驼队里有人咳，咳得寒，像北风贴骨。商队却不停——停不得，停，货腐，命也腐。

佛门灰旗前，有人施粥，粥稀，热。热在客路，不在丙九棚。密宗赭旗后，褐袍人坐驿亭，坐得不急。急便邪，不急，像等冬寒够深。西教白旗风里猎猎，猎猎不是谣，是界。密院黑旗压车辕，车沉，沉得像克扣的麸皮。沉丹青旗最末——五脉都过这一道门。

老耿挑水路过，水洒半勺，洒得稳：「五脉齐，冬里货紧。紧，别凑秤。」

「不凑。」韩泥说，「凑秤，手飘。」

老耿又指驿亭：「那人等冬。冬深，才赠。赠，不白受。」

韩泥瞥褐袍人袖里黄纸角露一线，露，像凡物藏锋。密宗线不在村庙了，线在宗门外。他不凑亭。凑，像贪；贪，恩易断。

侧门风硬，吹得他袖腥一线。腥在商线，不在胸——胸记着七笔，袖腥，避。管事嘱过：「商线过，袖洗三遍。」他记在心里，不出口。

第二趟搬柴，侧门霜白，霜白像辨香那日廊下。褐袍人抬眼，眼淡，淡得像凡物看人——不像看仙缘，像看骨寒。韩泥独眼平，平过去。平，是守。

商队领队验秤，秤杆稳，稳得像叶家那杆。韩泥旁站，不凑，只记：商线在，心不乱。领队扫他一眼，扫过，像看尘，不看人。尘不要紧，界在——界内末排，界外五脉。

黄昏，丹房外，铁无言翻簿，见他一眼：「鼻下稳，三日后再验手。手抖，仍滚。」

韩泥只答：「手记着七笔，不抖。」

秦霜声从远廊来：「五脉商队在，宗门戒躁。躁者，借商线生谣——谣借腥，你别沾袖。」

同批少年碰他袖：「你真过了鼻子？」

「过了。」韩泥说，「过鼻子，不过命。命在坛。」

「坛？」

「坛在，才验得过第二关。关过，才凝丹。凝丹，先还热的。」

少年不懂「热」，闭嘴。韩泥不解释——解释轻，恩重。

赖福远处讽：「看商队能看出仙缘？」管事声冷：「再嗓，克半月麸皮。」赖福哑了。

夜，韩泥对坛，声低：「第一关过了。试手前，冬来了。」

坛腹温一线，应「香」字，不应慌。心里七笔分列，不混。混了，手就飘；不混，手就稳。

外头驼铃余音散尽，褐袍人影还在驿亭——在，像密宗线从村庙挪到宗门，挪得不急。柱上旧条作废，新条写「三日后试手」。路通向凝丹，通向还。还，先还烫。

刘婆门外咳：「试手前，别空肚。空肚，手抖。」

韩泥应：「不空。粥在，姜在记里。」

丑时，坛腹又温一线，温短，像瓮听辨香过耳。过耳，不应灵，应规矩。外头风硬，柱上试手条却更真——真在末排，真在验手，真在「滚」字后那一条窄路。

他低声道：「三日，试手。试得过，关才开第二道。关开了，才学凝丹。学了，先还烫。」

坛壁水汽爬半寸，敛。敛着，像替他守着：香已过，手别飘；手不飘，恩不断。

老耿门外挑水，水洒稳：「僧不入门，符不入丹房。入门，像邪。」韩泥点头。点头，像把界记住——记住，才不借商线腥。

木牌在枕下，与试手帖并放。并放，像命与规矩挨着。挨着，才验得过。

管事嘱：「商线腥，袖洗三遍。洗不净，别近坛。」他近坛，不近亭。袖洗三遍，三遍净了，胸才贴坛；坛应一线温，不应慌。

铁无言低声：「五脉商线，腥在袖，不在心。心在，符才接得住。」韩泥应声，不追问。问急，像借腥。

第二趟末，侧门风硬，他肩柴停半息——褐袍人抬眼，眼不看他脸，看他鼻下。鼻下静，像辨香那日廊角。褐袍人袖里黄纸角又露一线，露完即收，收得像试：凑不凑亭。韩泥绕远三步，三步是界；界内末排，界外赠缘。驿亭柱影斜压他独眼，他独眼平过去，平，像把「不借腥」按进肩里。

夜雪粒细，落肩即化。化不要紧，肩沉，沉得住，冬深才赠。商队铃远去，五脉线在界外；韩泥不借腥，心在界内。界内，手别飘——飘了，试手滚；不滚，路还在。

邻床梦呓：「香……」韩泥不答。答轻，恩重。重，记在心里，不在舌。

天边白一线，白像试手前三日也白了半分。白不要紧，手要紧。驿亭褐袍人未走，黄纸角仍在袖里——在，像冬深那一赠还在等。等不要紧，要紧是三日后手正。手正，才验得过凝丹；凝丹，先还烫。

他独眼平，手不抖，低声道：「商队验讫，铃远去。线在界外，心在界内。三日后，试手。」

柱上试手条在风里贴实，贴实，像倒计时。倒计时在柱，七笔在心，木牌在怀——怀在，路在；路在，恩不断。

风息了，符未至，怀仍空。空不要紧，沿净，席严，才接得住后头缘。缘在坛，坛在角，角在末排——末排不要紧，手正才要紧。

辨香已过，商线现，试手在三日。三日在柱，七笔在心——心不乱，手就不飘。褐袍人影仍在驿亭，像密宗线从村庙挪到宗门侧门，挪得不急；不急，才试得出末排守界守不借腥。
