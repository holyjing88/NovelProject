# 第九十六章 坊市风声

坊市人稠。

外门坊在药山南脚，棚杂，价乱，乱里藏真，也藏谣。谣起第三日，有人嚼：「听说丙九有个丑杂役，夜里嗅渣，嗅得像丹。」

韩泥不近棚心。近心，像显摆；显摆，赖扒皮来。他立末摊，摊是破筐反扣，扣上三包药渣，渣灰，标价最低。

标最低，才像末席该标的。该标的，才不惹眼。

铁无言过，脚停半息：「你要挂牌，先听风声。」

「听。」韩泥说。

「风声有两条。」铁无言声低，「一条说你能嗅；一条说你是晦气，碰谁谁倒霉。」

韩泥独眼平：「晦气我熟。」

铁无言没笑：「熟不要紧，别接骂。接骂，坊市不收你。」

韩泥点头。收，不在这一日；这一日，只听。

听里，白织月袖过，声冷：「丑杂役也摆摊？摊什么，卖腥？」

韩泥只答：「卖渣。渣净，栏清。」

白织月嗤，嗤声远。远不要紧，远，才听得全。

全里，有人议兽潮——议在远，在下一月，在「南荒妖兽聚，或冲药山界」。

韩泥耳尖一动，动不要紧；动，记在心里。记兽潮，不在舌；舌要留来守摊，守坛。

刘婆买线过，线粗，声低：「少摆摊，多吃饭。」

「饭稳。」韩泥说，「摊是听风。」

刘婆哼：「风硬，别吹歪路。」

韩泥应声：「不歪。」

不歪，是七笔分列；列在心，不在牌。

午后，管事弟子巡坊：「杂役摊，税半文。」

韩泥交半文，文旧，旧不要紧。旧，像常杂役。

巡毕，他收摊，收得像没来过。没来过不要紧，风声已入耳——入耳两条：嗅渣像丹；兽潮将至。

入夜，席下按坛。坛腹温一线，贴掌，像应「听」字。听，不是应；应是下一回挂牌试水。

赖福堵门：「坊市去了？去干什么？」

「听风。」韩泥声平，「风硬，手稳。」

赖福眼细：「风里有你的晦气？」

「有。」韩泥说，「我熟。」

赖福啐，啐完走。走不要紧，风已听过。

邻床问：「兽潮真来？」

「来不来，都要备。」韩泥说，「备手，备渣，备不飘。」

丑时，廊下静半刻。粟壳香淡，香里混一点坊市尘——尘淡，淡不要紧；淡，才像去过。

他低声道：「风声过了。下一回，挂牌。」

挂牌，不在名，在试；试坊市认不认丑手，认不认末席。

坛在，根在。根在，风声就不白听。

坊市南脚，棚杂，价乱。乱里藏真，也藏谣。

韩泥立末摊，摊是破筐反扣。标价最低，最低才像末席该标的。

铁无言抱臂立远：「风声有两条。一条说你能嗅；一条说你是晦气。」

「晦气我熟。」韩泥独眼平。

白织月袖过，声冷：「丑杂役也摆摊？」

「卖渣。」韩泥只答，「渣净，栏清。」

刘婆买线过，声低：「少摆摊，多吃饭。」

管事弟子巡坊：「杂役摊，税半文。」韩泥交半文，文旧，旧像常杂役。

收摊，风声已入耳——嗅渣像丹；兽潮将至。

坊市尘里，他听第三条风声：有人赌兽潮来，赌杂役先逃。赌不要紧，他不接赌——接赌，像认晦气。认不得，只认手在。手在，栏在，坛在。

收摊后，他清栏第二趟，铲下稳，稳里带腥。腥不要紧，要紧是鼻下仍静。静，才听得全。全里，兽潮二字又近，近到刘婆缝布条，条粗，说「蒙鼻」。蒙鼻，是备，备在99，备在101，备在108清瘴。

坊市尾巷，风硬，硬里带一点药回。回不要紧，他鼻下静一息，静里辨出谣与真——谣在舌，真在耳。

铁无言立十步外，声低：「风硬，别辩。辩，费舌；舌要留来守坛。」

韩泥应声：「听风，不叫。」

同批少年悄问：「你真敢挂牌？」

「敢试。」韩泥说，「试不过，撤。撤了，仍清栏。」

少年不懂「试」，却懂「撤」，闭嘴睡了。

管事弟子贴条：「南荒兽聚，或冲药山界。」条丑，字真。真在耳，不在舌。

韩泥读条，不近，不躲。躲，像怕；怕不要紧，怕也备。备，首在耳，次在鼻，末在手。

散队后，他过药渣场角，风硬，硬里鼻下静一息——静，是把风声摁进路里。

赖福堵坊口，晃木牌：「丑杂役，风里有你的晦气？」

「「」韩泥应声：「听。」听里，白织月袖过，声冷：「丑杂役也摆摊？摊什么，卖腥？渣净，栏清。」记，不在舌；舌要留来守摊，留来守坛。赖福晃木牌：「丑杂役也摆摊？晦气沾全坊。」韩泥不辩。辩，费舌；舌要留来嗅渣，留来听风。风里有仇线，仇线在钱戾衡名上；名在远，手在摊。管事弟子记簿，笔停半息：「末摊，三日看。

」韩泥应声：「三日。」下一回，挂牌。」挂牌，不在名，在试；试坊市认不认丑手。更鼓沉，韩泥按木牌，牌凉贴胸。仇烫，约凉，分列才走得远。手在，摊在，路在。坊市风声起，起在尘，起在谣。谣说「丑杂役嗅渣，嗅出邪」，邪不要紧，邪在嘴，不在鼻。鼻下嗅诀，沉腐浮三字一字不落。落齐，才听风。

韩泥立末摊，摊破筐反扣，扣上三包渣，渣灰，标价最低。最低，才像末席。末席不惹眼，不惹眼，赖扒皮才少查。少查不要紧，查在耳——耳要听兽潮，听仇，听白织月袖风。

钱戾衡名在仇线，玉牌在怀，烫在还路前头。前头有三年，有十三层，有让他滚那一日。日未到，手先要稳。稳，不在摊名，在嗅准。

铁无言立远，声低：「挂牌前先听三日。」听里，药摊婆议霉渣，议青芯，议「南荒聚兽，或冲药山界」。界字入耳，入耳不要紧，入耳才备——备在99，备在远，备在手。

赖福过，鼻动：「晦气摊，离远些。」韩泥不挪。不挪，末席就在末。末，才试坊市认不认丑手。认不认，不在这一日，在97挂牌试水。

丑时，廊下静半刻，粟壳香淡，香里混坊市尘。尘淡，淡不要紧，淡像去过。去过，风声才算听过。听过，下一回，挂牌——挂牌，试三日，试过才续。

坊市尘起，谣起，谣说嗅渣邪。邪在嘴，不在鼻——鼻下嗅诀，沉腐浮落齐。落齐，才听风。风里有兽潮，有仇，有白织月袖风。袖风冷，冷不要紧，冷才试手稳不稳。

末摊破筐，渣三包，价最低。最低像末席，末席不惹眼。不惹眼，赖扒皮少查；少查，耳才听全——听全「南荒聚兽，或冲药山界」。界字入耳，99才接得住。

钱戾衡名在仇线，玉牌在怀。怀贴身，不示人；示人，谣又来。谣不要紧，谣在远，手在摊。摊在，97才挂牌试水——试水，试三日，试过才98续。

铁无言声低：「挂牌前先听风。」听过了，风声算过。风声过了，下一回挂牌——挂牌不在名，在鼻。鼻准，坊市才认丑手。认不认，97见分晓。

坊市第三日，尘起，尘里混一点兽潮二字——二字在远，在条，在99。99在前，96只闻风；风闻过了，97才挂牌。挂牌试水，试三日，试过才98续。续不在荣，在听——听兽潮，听障，听栏先倒。倒不要紧，倒前先备；备在耳，备在鼻，备在手。手在末摊，摊破筐，价最低，最低像末席。末席不惹眼，不惹眼，赖扒皮少查。

少查，耳才听全。听全，风声算过；算过，下一回挂牌。挂牌不在名，在鼻——鼻准，沉腐浮落齐，落齐，手才不飘。飘了，仇线也乱；不乱，玉牌烫，约凉，分列才走得远。更鼓沉，按木牌，牌凉贴胸。风声过了，97在前。手在，路就不绝。

末摊旁，老客问：「嗅准吗？」「嗅。」韩泥说，「嗅完，手稳。」手稳，97挂牌；挂牌，试三日。三日无投诉，98续；续，99条至。条至不要紧，条在远，手在摊。摊在，路就不绝。

坊市开市，尘起，尘里混谣。谣说丑杂役嗅渣邪，邪在嘴，不在鼻——鼻下嗅诀，沉腐浮落齐。落齐，才听风。风里有兽潮，有仇，有白织月袖风；袖风冷，冷不要紧，冷才试手稳不稳。手稳，97才挂牌试水；试水，试三日，试过才98续。

韩泥立末摊，摊破筐反扣，渣三包，价最低。最低像末席，末席不惹眼；不惹眼，赖扒皮少查。少查，耳才听全——听全「南荒聚兽，或冲药山界」。接得住，100才前夜，101才备，102才撒，103才压，104才至。至在远，96只闻风。风闻过，下一回挂牌——挂牌不在名，在鼻。鼻准，仇烫，约凉，分列才走得远。更鼓沉，按木牌，风声过了，97在前。
