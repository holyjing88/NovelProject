# 第一百三十七章 坊市续

坊市续摊，续在听，不在热闹。

韩泥仍立末摊，牌仍歪，字仍丑。丑不要紧，渣正、鼻准，比牌子好看管用。

摊前人堆里，议论大比的嗓门抬得高：「言伯钧预选铁定进，内门少主还监场呢。」

「少主监场，丑骨也敢去？」

「敢不敢问他自己。」有人瞥韩泥一眼，「他牌挂着呢，耳朵也挂着呢。」

韩泥不接。接了像争名；名在远，耳要留来辨谣与真——谣在舌尖转，真在渣里藏。

一老散修递过来一包渣，笑得褶子全开：「丑丹师，这包废料，给口价？」

韩泥鼻下一息，辛里出涩。涩是石屑。石屑得剔。

「石掺了。」他把包推回去，「掺石的渣，我不收。」

老散修脸白：「你……你真辨？」

「辨。」韩泥声平，「辨了，才续摊。续摊不靠名，靠手。」

人走时还嘀咕：「丑丹师，有点真……」

「真就麻烦。」旁人笑，「真了，盯梢的人也来。」

管事弟子贴条：「坊市丑丹师，续留三日。」字丑，条真。真了，才像外门积势，不像一天热闹。

刘婆买盐过摊边，盐粗，手却稳：「少听仇话。仇话灌耳，手就飘。」

「听。」韩泥说，「听，才避；避，才活。」

「活着比赢嘴重要。」刘婆哼一声，「赢嘴的人多，还债的人少。」

铁无言从坊尾拐过来，抱臂，声贴地面：「坊尾有人盯你。」

「盯不要紧。」韩泥说，「盯在皮，不在手。」

「皮破了也疼。」铁无言道，「别近。近像挑衅。」

「不近。」韩泥应，「近，费舌。舌要留着守坛。」

白织月远廊过，袖风冷，脚步不停。不停，是小胜——胜在渣正，不在跟她辩骨贵骨贱。

有弟子追一句：「白师姐，那丑丹师续留三日，您看不看？」

「不看。」白织月声冷，「看渣的人多，看台的人少。台见。」

韩泥听见「台见」，把字按进心里。台见是约，约不在坊尾骂里兑现。

第二日，老散修又来，这回渣正。正了，像续摊的回响——回响在掌，不在嘴。

「昨儿那石……」老散修摸鼻子，「被人塞的。今儿干净。」

「干净就行。」韩泥收了，「被塞一次，往后自己筛。」

「筛？」老散修苦笑，「咱们这种人，筛得过仇家？」

「筛不过仇家，也筛得过自己的手。」韩泥道，「手脏，摊先死。」

赖福来查税，税半文，文旧。旧不要紧，旧像常杂役该交的。

「续留三日？」赖福冷笑，「丑摊也风光？风光完了交钱。」

「交。」韩泥把铜递上，「税在，摊在。」

「摊在又能怎样。」赖福哼，「钱戾衡那边风紧，你这牌迟早摘。」

「摘不摘台见。」韩泥说，「台前见，不在您火折前见。」

赖福甩袖走。韩泥收摊如石——石在，风过不留痕。

回丙九，他按坛。坛腹温一线，像应「续」字。

邻床悄问：「你真打算上大比台？」

「上。」韩泥说，「不上，三年约白接。」

「白接咋了？」少年抖一下，「接不住钱戾衡，接约更吓人。」

「吓人比丢人强。」韩泥道，「丢人像滚。滚了，恩断。」

丑时练鼻下静。粟壳香淡，香落齐，这一日的风声记进心里。

编筐半刻，藤刺扎掌，偏手，刺浅。编筐钱买米，米在，恩才还得上。刘婆那笔还了，叶汤还在前——前在远，今儿只把耳练尖。

管事弟子记一笔，笔落「续留」。真了，才像收风，不像摆摊热闹。

第三日收摊前，坊尾那道盯梢影又晃。晃一下就远，远得像故意留痕。

铁无言隔三步说：「影在，别追。」

「不追。」韩泥说，「追像急。急了，手飘。」

「手飘，丹不正。」铁无言点头，「正了，才配上预选。」

韩泥把歪牌擦了擦。擦不亮字，擦得心里清醒：续摊续的是听，不是荣。荣在大比，在胜钱，在筑基后——那都远，今日只听。

废渣巷日斜，他又验两包，包正，手才静。静了，回屋才睡得着。

侧门驿亭风硬，他路过时把木牌按实。牌凉贴胸，凉提醒他：风紧，下一夜要守坛，坛露了，续留白续。

续留第二日午后，摊前来了个瘦袍客，说话转着弯：「丑丹师，听说你闻石甚准。门里有人想请你瞧瞧私货。」

「私货不看。」韩泥摇头，「看了像沾线。沾线，摊先死。」

「给你三倍灵石呢？」

「灵石买得动我手，买不动我坛。」韩泥声平，「坛要净。净了，才续得下去。」

瘦袍客笑意一僵，甩袖走。坊尾那道盯梢影跟着挪了半寸，像在记账。

铁无言买了包粗盐路过，盐袋往摊沿一顿：「方才那人，邪。」

「邪我晓得。」韩泥道，「邪处在急。急着掏钱，多半套绳。」

「绳你不套就成。」铁无言眼神扫坊尾，「影还在。别抬头找。」

「不找。」韩泥低着鼻子验渣，「找像怕。怕了，手飘。」

有个买丹的小妮子捏着半文铜，怯生生问：「爷爷，这包渣……我娘咳嗽，能不能入药引？」

韩泥闻了闻，眉心一松：「能。辛里带甜，不冲肺。煮前再晒一刻。」

「真能？」小妮子眼睛亮。

「真能。」他把包递回去，「骗人渣，我这摊先砸。」

小妮子千恩万谢跑了。旁人嘀咕：「丑丹师也讲情？」

「讲的不是情。」韩泥听见了，抬眼，「讲的是正。正了，人少死。」

第三日清晨，管事弟子又来贴条角：「续留最后一日。一日内无事，簿上记嘉。」

「嘉不嘉不要紧。」韩泥说，「渣正就行。」

「你这人真木。」管事笑骂一句，「木也好，少惹祸。」

午时风忽然紧，紧封摊帘。帘后有人低骂：「预选丑骨，台上必要丢人。」

韩泥掀帘一角，只见背影远。他不追。追像接骂，接骂像滚。

刘婆偏在这时送姜汤过来，汤沿烫手：「喝。听见骂也喝。空着肚听骂，骂进骨。」

「进不了骨。」韩泥喝一口，「骨丑，骂嫌弃还来不及。」

「贫。」刘婆瞪他，「贫完了去睡会儿。夜里不是还有事？」

「有。」他点头，「守坛。坛比摊贵。」

收摊装箱时，又来个挎剑青年，剑穗新，嗓子也新：「末摊丑丹师？把你鼻下那点本事卖我半个时辰。」

「不卖。」韩泥盖箱，「本事挂在摊上，不挂在雇工契上。」

「怕钱少？」青年扬眉，「内门价。」

「怕线脏。」韩泥抬头，独眼平，「脏线一沾，预选帖先臭。帖臭了，台上更臭。」

青年哼笑走开。铁无言不知何时又立到三步外：「内门价都拒？」

「拒。」韩泥道，「拒得干净，夜里才守得着坛。」

「行。」铁无言点一下下巴，「你这棵丑树，根系倒深。」

回丙九路上，药香混着土腥。韩泥把今日听到的字在心里归堆：大比、言伯钧、少主监场、坊尾盯梢、钱戾衡风紧。堆好了，不乱；不乱，睡前才敢闭眼。

路过柴门，刘婆正在晾布条，抬头吆喝：「摊上顺不顺？」

「顺。」韩泥停半步，「顺在渣正。人不顺，我不管。」

「不管也好。」刘婆把布扯直，「管多了，鼻子就闻人情去了。人情一杂，丹就邪。」

「记着。」韩泥应一声，继续走。布条在风里啪啪响，像给他打拍子：拍子稳，步子才不抢。

邻床少年踢他板床：「韩叔，你今儿又得罪人了吧？」

「得罪就得罪。」韩泥说，「讨好更脏。脏了，鼻先废。」

「那明日还去吗？」

「去。」他拉被角，「续留最后一日。一日听完，再把风声听紧。」

「听完能怎样？」少年闷声，「听完钱家还不盯着？」

「盯着让他盯。」韩泥闭眼，「盯进坛里，我再睁眼。盯在皮上，皮厚。」

更鼓沉，他望坛，低声道：「坊市续了。听风紧。记着。」

坛不应字，只温。温像压着巷口鼓——鼓一响，钱戾衡的影就该从字后头探出来。他今夜不出去，出去手飘；手不飘，守坛才守得住那一线实。实了，明日巷鼓再响也不慌。
