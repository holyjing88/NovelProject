# 第二十二章 先还

新谣第三夜，雨意无，风硬。

风硬得像刀背，刮茅檐，刮得漏雨处更漏。漏不要紧，要紧的是人别漏——别漏恩，别漏还，别漏「先还谁」。韩泥臂上结了痂，袄肘叶线密。密得像心事的针脚——针脚不亮，亮了就招眼。招眼，叶家更危。

他独坐灶前，米汤稀得见底。底空，人不空。空的是还不起，不是不想还。不想还，心不会热；想还，叶汤才会深。心口四恩在：叶汤、芽粥、陈袄、张馒，衣纹一捺。赵辱在，仇苗也叉着。乱而有序，像穷家粮柜——柜少，粒粒真。

他摸心口，摸得久。久，像摸一碗汤的边。边烫过，边还在；在就欠，欠就得还。

「先还叶。」

他低声，像对自己说，也像对坛说。「芽儿粥是点，陈姑袄是撇，张麻婆馒是圈。都真，都暖，都该还。」

「可汤是头的。」

就这些。没有豪言，没有天誓。老头口吻，够还，不怨。怨是少年，他不是。为何先还叶？因五十八年，无人送汤进门缝。因谣起，她仍送姜，仍补肘，仍塞「别念」。因恩不是可怜，是人该送；人该送，就不该被血祠谣咬碎。因还字比谢字重。重了，就得活到达得到还的那日。

他想起张麻婆双馒，想起陈姑棉袄，想起芽儿狼口后那包药叶。四恩都在，序不能乱。乱序，还起来就薄——薄了，不如不还。不还，心就冷；心冷，叶汤就浅。

他低声道，像对空坛：「坛凉不要紧。心热就行。」

坛不答。答的是腹里一线温，温得像应了「先还」二字——应半分，够刻进骨。他想起门墩纸角「趁热」，想起姜辣冲鼻，想起袄肘针脚。想起隔窗那夜，他一句没说——不说，是护名；护名，也是还的一种，还她清白。清白比丹难炼。

不深，却狠。狠得像把「趁热」刻进心骨。心薄，骨要硬，硬了才压得住赵辱，压得住榜末名，压得住明日公审的潮。心口一紧，紧得指腹发热——比十月初那夜更烫，烫得像叶汤在心里醒。叶汤旁那点旧恩，被新念浸成深红，像一枚印，印的是「先还」。

叶汤加深，深得像灯芯又捻紧一分。紧一分，亮一寸，亮给自己，不亮全村。全村明日才看。坛角仍凉，坛腹更温。温贴胸，胸不跳快，跳得实。实，是老头能给的勇。勇不炸，勇记，勇候。

门外，叶家方向狗吠一声，吠急，像有人经过。韩泥不追。追，害。他只合上袄，塞进怀，腰上硌一下，硌成一句：

「我记着，以后还。」

天将亮，村道有人咳，咳得远，像沈家，像芽儿。咳声弱，弱得人心紧，紧完又松——松因人还活着。韩泥开门缝望，雾薄，旱气仍在。仍在，也要活。活要粮，粮要筐，筐要藤，藤要手实。

他绕膝勒藤，勒指，指疼，疼得清醒。清醒里他知：榜末能忍，谣能忍，公审将至；不能忍的是叶家秤被人砸了。

「先还叶。」他又道，声更低，「还了，再还别的。别乱了序，序乱了，恩就薄了。」

心里叶汤加深。加深到今夜，像灯。灯不亮全村，亮他一条路。路通向明日祠堂，通向叶青禾开口的那刻。他想起赵绣春泼的脏水，想起赖扒皮短签，想起榜末七两。辱候着，恩先还。他不滚，只按，按得心口微潮——潮是坛应，应半分，够他扛过公审前夜。

「汤是头的。」他第三回道，像盖章。指腹贴心，贴得烫。

门外风硬，硬得像有人贴墙念：血祠开，雨下来。念罢又散，散成空。空不怕，怕的是人信。人不全信，公审就还有机会。机会在叶青禾的秤上，也在他叶汤上。两处都实，韩泥便合上袄，贴胸。贴胸，像贴一张不会退的契。

头一笔是叶。叶在心上，心在瓮边——瓮仍凉，凉不妨，坛已温。温是路，路通向炼丹，通向省亲，通向门缝再送一碗不烫的汤。序定了，心就不乱。乱序，四恩就薄；薄了，还起来像施舍。施舍不是还，还是记，记到能还足。

门外风硬，硬里有人咳，咳得像沈家。韩泥不开门，只隔缝问：「谁？」

张麻婆嗓子粗：「我。送半块饼。别饿死在还字前。」

韩泥道：「不饿。还字在前头，饿不着。」

张麻婆道：「先还叶，对。叶那碗汤，头一笔。」

韩泥道：「嗯。」

张麻婆顿了半息：「明日公审，你少吭声。吭声，赵家高兴。」

韩泥道：「知道。坐着听。」

张麻婆走远，远里又喊：「还字刻实了，再还别的！」

韩泥低声应：「刻实了。」

应完，按心口，叶汤深一分，深得像喘过气来——压是谣，松是先还；先还定了，明日公审就不乱。他再道第四回，声几不可闻：「先还叶。」

四回，像钉，钉实了就不拔。拔了，序乱；序乱，恩薄。根在，誓就不改——改了像欺恩；欺恩不做，只做记心翁。

布条「先还叶」钉在心口，钉实，像还路起点。起点丑，路才真。隔巷沈家一灯灭，灭得像芽儿睡了。睡了就好，省亲二字明日不让挂公审柱——挂了，叶家更窄。

韩泥不走近，只按布条。布条粗，粗得像还路的绳。绳丑，能拴住心，拴住序。序定了，四恩不混：叶汤头一笔，芽粥是点，陈袄是撇，张馒是圈。圈在，冬不愁粮；粮不愁，手才不僵；手不僵，绳才编得实。

祠前有人贴第二道告示，墨迹黑，黑得像锤。锤形未落，声先在——先在风里，先在旱里。旱里有人念血祠，念到半句散。散了好，散在舌上，不进心。不进心，公审就能听叶青禾开口；开口，秤在，还字才站得住。

他隔半巷望柱影，影长，长像末席仍有一尺地。一尺地，够坐，够听，够记一句「我记着」。

丑时前，灶里余灰尚温。温不见人，见坛腹——腹应「先还」，应半分，半分像针扎心口，扎实了就不拔。拔了是少年意气，他不拔。只把藤条绕腕，绕出勒痕，勒痕疼，疼得清醒。

公审告示在祠堂柱上贴实了，墨迹未干。韩泥不近柱，只隔巷望一眼——望一眼，记住锤落时辰，记住末席还有一席。席在，还字就不悬空。他把藤条收进筐底，筐底实了，手才实；手实了，明日坐末席才像坐，不像躲。

坛腹应「先还」半分，半分够扛一夜风硬。风硬里，叶家方向仍有一盏灯弱着；弱灯在，烫便在。烫在叶汤，不在舌上。公审前夜，赵绣春墙根又贴半张谣条，条角被风吹卷。卷了，他不去撕——撕像辩；不辩，只记。记够了，锤落时末席仍有一席。席在，叶青禾开口，秤在，还字才站得住。

他把半块饼掰碎，碎粒落掌心，像把四恩再数一遍——数清了，序就不乱。序不乱，明日公审只备一句「我记着」，够了。坛沿一线温贴掌不退——退不得，退像欺恩。布条「先还叶」贴胸，贴实，像还路契。心口叶汤印痕浮着，浮得像字在呼吸。呼吸轻，恩重——重，便活长些，好还。

他合眼，手仍按在坛沿上。先还叶，立誓在心，不喊天，不喊地，只喊自己——自己听见了，还字就不飘。心口一紧，紧得像叶汤在心里醒。醒半分，够扛明日公审一整场潮。沿口一线温贴住虎口，够撑明日公审。瓮未醒，他不喊醒；醒在缘，不在这一更。明日巳时，祠堂公审，叶汤得立给全村看。
