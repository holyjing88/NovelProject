# 第十一章 牵连

谣起半月，叶家药铺门冷。韩泥担筐过巷，听见赵绣春在铺前嚷：「晦气沾了铺，谁还敢抓药？寡妇不要脸，连绝户翁都喂！」

叶青禾在帘后，没出。只听见秤砣落盘，一声，沉。韩泥脚步没停。停，坐实「牵扯」；不停，耳已入耳，得记在心里——记的不是谣，是叶家被拖下水。村童绕远，绕得像绕瘟疫。绕的不是他，是「叶家也沾了」的新词。童谣丫墙根改词，念得脆：「叶家铺，晦气铺，寡妇送汤全家倒——」

念到「倒」字，自己卡壳，像怕叶家秤砣真砸下来。韩泥路过，不停，只把筐担直——担直，路还长。晌午，集尾有人拒收他的筐：「晦气筐，压价。」

韩泥「嗯」一声，换摊，半个时辰，仍卖十二文。集尾另一摊，有人窃语：「叶家秤砣，听说夜里也落。落给谁听？」

「给寡妇听。听她还不还汤。」

韩泥独眼不抬，只道：「秤砣落，药还抓。抓药的人，不怕秤。」

两摊静一息，买主掂筐，仍付十二文——小胜，在钱还在手里，不在嘴上。午后，周嫂聚妇人在井台，棒槌砸得响：

「叶青禾送姜送汤，药铺生意淡，活该！」

「沈家芽儿被罚，也是她带坏风气！」

「赖管事说了，再牵扯，叶家迁出村！」

韩泥打水，桶满，拎稳。「井是村的。」他嗓哑，「药铺也是村的。村人生病，不抓药，抓谁？」

周嫂棒槌悬半空：「你替寡妇说话？」

「我替活路说话。」韩泥转身，「谣在嘴上，病在身上。嘴能停，病不能停。」

周嫂脸青，棒槌落下，砸衣上，水花溅自己脚。旁人窃笑，笑轻——笑的是周嫂噎，不是笑韩泥。马春花远处帮腔：「韩泥，你少装。装善人，叶家就能开？」

韩泥拎桶走：「开不开，病说了算。你嗓子尖，尖不治咳。」

马春花噎半息，没再追——追，像怕井边人多，人多，理还沾一点边。这是小胜：井还能打，话还能落一句。落一句够，不多落，落多了成靶。周嫂棒槌又悬，悬在衣角上，像还想砸，砸不出声——声砸在自己脚上，溅起的水花凉，凉得像公审前的潮。韩泥不回头，只把桶担直。担直了，叶家药铺方向的灯，弱，仍亮；亮着，牵连就不算死局。傍晚，叶家药铺门板半掩。韩泥不进门，把编筐剩的三根细藤搁门阶，转身走。走三步，门内出一声，低：

「韩翁。」

叶青禾。「藤补筐用。」韩泥没回头，「我不欠藤。欠汤。」

「汤……」她停一息，「村人说，再送，铺子开不下去。」

韩泥停步，侧半脸，独眼平：「铺子开不开，是病的事。送不送，是你心的事。心的事，别听嘴。」

帘后又一声秤砣，轻，像应他。叶青禾低声：「赵绣春说，再牵扯，要报官。」

「报。」韩泥道，「官来，也抓药。药铺关，全村咳。」

就五句。叶青禾没再追。他回屋编筐，藤条绕膝，勒指。指疼，疼得清醒——清醒里，叶汤那笔恩，像一根梁。梁若被谣咬断，整间泥屋都塌。他按了按心口，摸到贴胸那张「趁热」纸角。纸角旧，边毛，像叶青禾没退的证明。韩泥低声道：「牵连你，是我的债。债我担，不推你。」

说给自己，也说给帘后那声秤砣响——响在夜里，像叶家还开着门。夜里，心口叶汤那笔恩，压实，像一枚烫印。韩泥低声道记着：叶家被谣咬了，咬在叶汤的根上——危记心，心等公审。坛角微凉。坛腹微温。温里他低声道：「汤是头的。头不被谣咬断，我就还。」

坛壁尘里，水汽一线，极淡，像瓮在睡，却听见了「牵连」二字——听见，不退温，只不退。村东忽起狗吠，吠急，像有人砸门。韩泥推门望，叶家药铺方向，灯乱晃，妇人声尖：「滚！再闹，报官！」

赵绣春嗓更高：「报啊！看谁晦气！」

里正声插进来，沉：「再闹，记名！记名，连坐！」

赵绣春一窒，骂声收半寸，尖还在，后力没了。灯灭一息，又亮。亮得弱，像叶家撑。韩泥合袄，塞怀。他不往前冲——冲是害。害叶家更深。他只记：叶家危，危在谣，谣根在赵，根在赖，根在全村的怕。「要还。」他摸心口，「得活到达得到还的那日。」

狗吠渐远。远，不等于安。叶家灯弱着，弱到更深，仍弱。药铺秤砣再落一声，沉，像给全村听：叶家还在，还没被谣压死。韩泥关门，漏雨泥屋又漏一阵，雨丝细，细得像针。针扎肩，他不躲——躲像亏心。心不虚，雨扎就扎。他摸坛，坛凉，心热，中间夹着五十八年的命。命薄，还没断；断不了，就因为叶汤还在，叶家灯还弱着，弱也是亮。赵绣春在远处骂，骂声尖，尖里带虚——虚在药铺门仍开，开，就还有病人，还有竖立的理。韩泥按心口，像把叶家危，也塞进叶汤里：危不记辱，辱候着；危记心，心等公审。村狗再吠两声，吠向叶铺，吠完又哑。哑，像全村悬着一口气——气在叶家灯上，灯弱，未灭，未灭就还有救。巷口，陈姑咳着路过，停门缝外，没进。「韩翁。」她嗓沙，「叶家的事，你别往前凑。凑，害她。」

「我不凑。」韩泥没开门，「我记。记着，公审还。」

陈姑顿一息：「公审……真开？」

「里正锣未敲，心先悬。」韩泥道，「悬着，叶家就不算死。」

陈姑「嗯」一声，脚步远，远里像叹——叹轻，像也给叶家留一口气。韩泥低声道：「牵连，是谣的刀。刀落叶家，我记心，不记逃。」

坛腹温不退，像应他这句——应得淡，淡就够。够撑到公审，够撑到还汤。叶铺方向，秤砣又落一声，远，沉，沉得像给全村钉一颗钉：钉名「还开着」。沈枯芽门缝问：「韩爷爷，叶姑姑会关铺吗？」

韩泥应声：「秤在，就不关。」

他低声道：「秤在，铺就不关。铺不关，叶家的恩就不断。」

叶铺帘静，谣又近一步。他不近阶前——近是害，远是护。护到公审，护到叶丫头还能开秤；秤在，这恩才还得了烫，烫不能凉。

秤杆弯而不断，不断，韩泥就不辩，只记：牵连不牵连，汤不能凉。

叶帘内秤又落一声，落脆，脆响压过马春花嗓尖——尖不过秤，秤在，铺就不关；铺不关，叶家的恩就不断。

夜里，韩泥编筐至丑时，藤勒指，指疼，疼得清醒——清醒里，叶家灯仍弱，弱也是亮；亮着，牵连就不算死局。坛腹微温一丝，像应他那句「记心，不记逃」。逃了，叶汤就浅；浅了，公审前心就空。心不空，只候锤——锤来，证白才爽；爽是后话，话先在「我记着」里。
