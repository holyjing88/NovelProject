# 多道侣后宫体系（主次分明 · 家族入戏 · 有情有义）

> **定位**：韩泥 **收纳多位道侣**，非「正绶唯一」；倒贴有因、纳后负责、家族并进主线。  
> **硬规**：写 **情与义**（共患难、共担、共还账），不写无脑花痴；**允许**媚术/采补当主角爽点（`01` 五不准③）。栖香 **邪脉** 可写破媚、收服、采补爽段；228 毁媚丹为先例，非全书禁媚。
> **关联**：`11` §二、`08` 情感浪、`17` 馈缘表、`18` 正文主准。

## 五、后院有情有义（写法硬规）

1. **纳前**：倒贴必有因（救命、丹惠、护家、并肩）；正绶 **知情并点头** 后方可立副/剑契。  
2. **纳后**：各侣有 **职司**（叶当家、温丹阁、萧外事、乐盟约），不争宠互撕。  
3. **家族**：纳绶必写 **对方父辈/宗门反应**（温砚章、萧凛、叶振东至少各 2 场戏）。  
4. **恩仇分立**：纳妃章不杀人；报仇章不温存收人。  
5. **男欢女爱**：272/420/505/535 可加密度；写担当与立契，不猎奇丑化韩泥。  
6. **栖香邪脉**：杜烟萝仍拒；**非后宫**。

## 十三、平台敏感替代写法（v16）

| 敏感倾向 | 替代写法 |
|----------|----------|
| 采补/双修爽 | 合修写「丹气入脉」「成功率↑」；采补可写爽点，一笔带过床戏 |
| 媚术控制 | 写「破媚」「毁丹」「驱符」或「反采/收服」，韩泥可占上风 |
| 争宠互撕 | 写「职司分工」「叶点头」「共同护家」 |
| 逼嫁/强纳 | 写「立契」「见证」「家族盟礼」 |
| 少年后宫向 | 韩泥 **五十八岁口吻**：「我欠的债，纳得起就得还」 |
| 猎奇丑化 | 丑在外貌，不在人格；写担当不写窝囊 |
