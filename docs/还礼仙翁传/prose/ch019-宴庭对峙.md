# 第十九章 宴庭对峙

和谈席面摆在大殿前庭，灵桌玉盏，酒未温，话先冷。

厉无殇主位，韩铁山按刀立侧，周德海陪末席，笑得像和已成；霍镇山不入席，盾立廊下，像随时要把桌子掀了。

柳青鸢出席，礼数足，眼不软。莫长春袖空，杖在塔中，立她身后半步——不是护，是送；送一份全场都看得见、却没人猜得着的礼。

弟子围庭外三层，外门内门皆至，像等一场将载入青岚门闲谈的戏。

赵玄机立人群前，铁牌在腰，冷眼观莫：「赠魔头礼？找死。」

林挽月挤在后排，踮脚，竟也怕错过。

苏念慈未入席，只在远廊，丹香与酒气隔一条风，她像怕酒气进丹堂，更怕莫长春又赠出一份「只有别人收得下」的东西。

魏无涯在梁上，扇开一线，观气运，观莫袖。

席中，厉无殇举杯未饮，先刺：「莫长老，方才说送我礼。九月寿元，别只动嘴。」

满堂哄意微起，像潮头。

莫长春出列，不急，袖空，一念出塔，素帕落掌——洁白，无绣，折得方正，像凡间老翁随身之物，绝不像法宝。

「厉少谷主，」他递出，声平，「擦擦汗。将死之人，见不得血，也见不得汗糊眼。」

韩铁山变色，跨半步：「少谷主，帕来路不明——」

厉无殇扬手止，笑里傲：「一张帕，还能是定身符不成？莫长老，你若下作，青岚门脸先碎。」

他接帕，指触帕角的一瞬，莫长春杖尖轻点地，竹纹在袖中一跳，像应。

厉无殇未觉，拿帕拭额，动作随意，像接一份轻蔑的羞辱——九月老头赠帕，传出去，魔道也能笑青岚门老糊涂。

拭到第二下，他笑意定住。

不是汗停，是身停。

指还举在额前，帕还贴皮，唇还弯着笑，眼还睁着——人却像被无形的线穿骨，穿脉，穿神魂，钉在原地。

一息。

两息。

三息。

全场先呆，像没听懂发生了什么；继而哗然，像听懂却不敢信。

「少谷主？」韩铁山伸手推肩，推不动，像推一尊铁铸的像。

「定身……符？！」周德海杯洒，主和折子湿了一片。

霍镇山盾光一亮，又收，像怕一动就坏了喜剧。

柳青鸢眸光剧震，看向莫长春——师兄仍立原地，灰袍素，须眉白，像什么都没做，又像做尽了。

赵玄机指节咔一声，铁牌撞腰，妒与骇绞：「他……真敢？」

林挽月捂嘴，眼里竟有光，像第一次觉得「老」不等于「弱」。

苏念慈在远廊，指尖掐进木栏，不妒了——她只觉那方帕重，重得像七夜丹成，重得像药渣里熬出的一整片心。

魏无涯扇合，气运线颤，低笑一声，像看一枚子落对了格：「有趣。」

三息将尽，厉无殇眼睫微颤，像要从钉里拔出魂。

莫长春却先一步收帕——袖空，一念出塔，素帕归塔，符意已尽，像什么都没发生过。

「礼轻，」他淡淡道，「意重。」

韩铁山拔刀半寸，厉无殇却吼出一声闷吼，能动了，踉跄一步，额汗更冷，眼里杀意与羞怒搅成一片：「莫长春——！」

「少谷主，」莫长春截话，仍笑，「够活到你后悔今天说的话——说到这一步了。下一步，看你肯不肯还礼。」

庭外弟子炸开，炸成浪：

「定身！莫长老定住魔少！」

「一张手帕？！」

「九月老头……疯了还是神了？」

浪声里，簿在莫长春袖中烫，烫得像有人用指甲在他脉门上连划数道，不跳字，只记在心里。

他合袖，杖点庭石，声轻，却压过全场：「围观的，看清楚了。礼，可以轻；还，未必轻。」

韩铁山退后，刀仍在抖，抖里怒：「少谷主，为何不接帕时拔刀？」

厉无殇不答，答不出——三息里，刀意也被定住半寸，半寸够他记一辈子。

周德海扶湿折子，折子废，废得像七年和平一起废。他望柳青鸢，柳目光冷，冷里无和字。

霍镇山盾收，声洪：「今日之后，主战！镇妖司前移！」

弟子声浪涌至山门，涌里怕，也燃——怕赤焰记恨，燃的是青岚终于不割地先割脸。

林挽月挤出人群，唇动，像想说赵师兄也在人群里，终未说。她只记：定身三息，比门规抄三百遍还响。

苏念慈回丹堂，不开炉，先洗手，洗得像洗帕上看不见的符灰。弟子问：「首座，帕……」

「闭嘴。」她声硬，硬里掺敬，「凡物定魔少，才叫丹堂本事。本事不在问，在炼。」

赵玄机立远处，铁牌撞腰，妒与骇绞，绞里像还库之后又挪一寸。

他低声：「真敢。敢的人，不像演。」

魏无涯梁上退，退前扇合，气运线颤，低笑：「帕落格，子活。」

莫长春立庭心，袖空，一念出塔，杖点庭石，声轻，却压过全场余音：「围观的，看清楚了。礼，可以轻；还，未必轻。还轻了，恨才记得住。」

塔影在识海烫，不跳字，只记在心里。寿元行仍示八月，像钟慢一格，够他把下一阵熬浓。

他望厉无殇退去的方向，笑：「恨记在心里。记在心里了，才好还。」

宴散后，弟子三层仍未散，散得像怕错过下一息——下一息没来，来的是谣言，谣言比酒快。

有人传「莫长老袖里藏塔」，传者被执法弟子瞪回去：「宗主令，不许传帕，不许传定身。」

传的人更多，罚不及谣快，谣快里掺敬，敬里掺怕。

柳青鸢收剑穗，穗无风，低声：「师兄，今日之后，更险。」

莫长春道：「险才好送。送出去，才收得回。」

延迟的爽点不在定身那一瞬，在三息之后全场仍不敢信——不敢信，才嚼得久，嚼得久，大因果还在后头。

执法堂弟子执柳青鸢令，巡庭喝止：「不许传帕，不许传定身，传者罚。」传的人更多，罚不及谣快，谣快里掺敬，敬里掺怕。

韩铁山退后，刀仍在抖，抖里怒：「少谷主，为何不接帕时拔刀？」厉无殇不答，答不出——三息里，刀意也被定住半寸，半寸够他记一辈子。

霍镇山盾收，声洪：「今日之后，主战！镇妖司前移！」弟子声浪涌至山门，涌里怕，也燃。

林挽月挤出人群，眼里有光，光像第一次觉得「老」不等于「弱」。她唇动，像想说赵师兄也在人群里，终未说。

苏念慈回丹堂，不开炉，先洗手，洗得像洗帕上看不见的符灰。她声硬：「凡物定魔少，才叫丹堂本事。本事不在问，在炼。」

赵玄机立远处，铁牌撞腰，妒与骇绞，绞里像还库之后又挪一寸。他低声：「真敢。敢的人，不像演。」

魏无涯梁上退，扇合，气运线颤，低笑：「帕落格，子活。」

他望厉无殇退去的方向，笑：「恨记在心里。记在心里了，才好还。下一阵，看他拿什么还。」

宴散后，外门把「莫长老不入席」讲成清高，清高越传越像装。莫长春笑：「老夫若入席，你们又说老不正经。不如不入，入你们的心。」弟子哄笑，笑里敬惧参半。

厉无殇离席，离席像把脸带走，脸带走，恨留下。韩铁山按刀，刀仍抖。柳青鸢收剑穗，穗无风，低声：「师兄，今日之后，更险。」

莫长春念：「席完，帕已出。下一回，看他拿什么还。」塔中沉，沉得像庭心杖还在。

延迟的爽点不在定身那一瞬，在三息之后全场仍不敢信——不敢信，才嚼得久，大因果还在后头。

柳青鸢低声：「师兄，今日之后，赤焰必记恨。」莫长春道：「记恨才好。记恨，才记得住谁送的礼。」

席散，帕角仍温，温得像把「和还是战」四个字，先揣进每个人袖里。
