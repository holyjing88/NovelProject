# 第二十六章 坊市闲逛

备战间隙，莫长春下山，青岚坊市。

未乘轿，未带弟子，灰袍素，袖空，杖在塔中，杖点青石板，像寻常老翁赶集。摊贩认得他，却不敢大声招呼——「莫长老」三字在坊间比灵石还硬，硬到砍价都像冒犯。

可今日偏偏有人敢围——围得不近，围在巷口、茶楼下、药铺檐下，像看一场不像长老的戏。

莫长春在棉衣摊前停，摸布料，厚，糙，暖，问价，摊主伸五指，他伸三指，摊主苦脸，他笑：「三指，送杂役堂十件，记青岚门里实价。」

摊主一愣：「长老……真记门里实价？」

「记。」莫长春道，「杂役堂人冻，手抖，搬药洒；洒了药，丹堂骂；丹堂骂，你少卖一瓶伤药。一环扣一环，暖的是手，赚的是命。」

摊主终点头，像被说服，也像被九月老头一种说不清的气压服。隔壁摊主见状，凑过来：「长老，俺这伤药也记门里实价？」

「记。」莫长春道，「但先砍价。门里实价不是冤价。」

围观人群笑，笑轻，像松一口气——定身魔少的老神仙，下山却像抠门老翁，反差比戏还好看。

他又至低阶符纸摊，同样砍价，同样嘱送杂役堂，灵石从袖中出，出得慢，像真穷，数得细，像真抠。一张黄纸扯半炷香，扯得认真，掌柜额头见汗，最后仍让了半指。

路人瞧见，窃语：「莫长老定身魔少，下山却像抠门老翁？」

另一人答：「抠门才真，灵石撒给杂役，不撒给面子。」

第三人低声：「听说他九月将尽，还这么省……省给谁看？」

第四人答：「给咱们看。杂役手暖了，咱们伤药好卖。」

有人嚷：「莫长老，也给俺记一件棉衣！」

莫长春笑：「排队。一笔一笔记在心里，乱不得。」

茶楼二层，白漱玉凭窗，远观莫长春在符纸摊前与掌柜扯半炷香，扯得认真，像不是渡劫败后的长老，是凡间会为一张黄纸计较的人。

她心口微动，慕意初起，像茶烟，轻，却缠。

同伴笑：「漱玉，你看什么？」

白漱玉收目，淡道：「看一个人，把钱花在别人看不见的地方。」

同伴撇嘴：「看不见的地方，怎么赚名声？」

白漱玉指楼下：「楼下都在看他砍价。名声不在他花多少，在他花给谁。」

同伴一愣，不语，却忍不住又瞧一眼楼下——九月老头数灵石，数得像数命，数得围观者都静了半息。

坊市第二摊，莫长春买一根草绳，绳贱，贱里像藏阵线。摊主笑：「长老也买绳？」莫长春道：「买实价，不买脸。」

赵玄机的人跟半条街，跟丢，丢得像妒也丢。柳青鸢遣银，银被推回：「买茶买实，不买情。」

顾小满远处记：坊市，棉衣，草绳，实价。记完，声平：「线可能在这里。」

茶摊前，莫长春停半息，不问价先问：「可还实价？」摊主笑道：「实价。」他买两饼粗茶，饼普通，普通里却像藏线——线不在茶，在谁来买、买后给谁看。

围观弟子坊市口停步，停步里敬，也疑。有人嚷：「莫长老，也给俺记门里实价！」他答：「排队，一笔一笔记在心里。」嚷者笑退，笑退里像第一次觉得长老不是天价，是实价。

苏念慈闻谣，在丹堂闭炉半炷香，谣里掺笑：「九月老头买绳买菜像凡人。」她低声：「闲逛别传成私。私了，丹堂先乱。」

赵玄机遣人跟第三次，跟至茶摊，掌柜认得是青岚长老，跟者反被人群挤散。赵闻讯，铁牌在腰，竟未再遣——遣少了，像还库之后的一寸克制。

日暮前，符纸摊掌柜送半捆边角黄纸至杂役堂，附条：「莫长老砍价让的，记门里实价。」管事一愣，顾小满记下：黄纸，赠，实价余。

坊市雨意将至，摊贩收摊匆忙，匆忙里仍给莫长春让半指价，让得像怕长老记门里实价记成仇心。他摆手：「让价可以，别让良心。」

归山途遇执法堂巡城，巡者想问长老为何不乘剑，莫长春答：「剑载人，不载人看不见处。看不见处才要数三指。」巡者一愣，旋即笑，笑里敬意多过疑。

杂役堂夜，十件棉衣到，新棉衣糙暖，弟子试穿，袖宽，宽得像终于被看见是活手。顾小满灯下记在心里：棉衣十，符纸一批，草绳一根，茶二饼，皆实价。

柳青鸢闻坊市归来，未遣银第四次，只令：「师兄买实，宗门记门里实价，不许再传成长老私逛。」令下，谣声少半分。

白漱玉客栈夜，粗茶沏开，映窗月，月里像九月老头的袖仍空。她提笔欲写名帖，又搁，搁得像终于懂得：慕一个人之前，先慕他把钱花给看不见的手。

日暮，莫长春归山，袖中少灵石，心中多烟火。

途经杂役堂，把棉衣单掷给管事：「十件，明日穿。穿了手稳，别洒药。」

管事接单，手抖，抖得像接旨。顾小满在廊下瞧见，眼里亮，亮得像又多记在心里。

簿无字，他却觉善缘又涨一分，涨在凡人摊位的三指砍价里，涨在杂役堂将到的棉衣里。

坊市弟子尾随半里，议论门里实价，议论里敬，也疑——疑长老真穷，疑长老真抠，疑得越久，越像看懂：抠的是面子，松的是手。

符纸掌柜私下对伙计说：「莫长老砍价像砍价，记门里实价像记命。」伙计问：「咱赚吗？」掌柜答：「赚在青岚门还在。」

白漱玉下楼，欲递名帖，又止，止得像怕买卖玷了慕。她只买一包粗茶，茶苦，苦后回甘，像莫长春那日递茶饼。

归山路上，孩童追问：「老爷爷，你真定住魔少？」莫长春笑：「定住的是戏。你们别学戏，学记门里实价。」

司库认门里实价，册上棉衣十件、符纸一批，备注皆「莫长春三指砍价，记门里实价」。执事低声：「长老真穷，真抠？」司库答：「抠的是面子，松的是杂役手。手稳了，丹堂少骂，咱们少赔。」

他念，像钩，钩向远山：

「坊市烟火，是根的另一面。妖讯再来时，这双手，还得暖着握杖。」

杖答，气音轻，像等。

莫长春笑：「不等。」

归山路口，孩童再问：「老爷爷，你真抠？」莫长春把草绳绕杖底结了一下，结得结实：「抠给看，不是给虚。虚了，阵脚先摇。」

第二摊买的绳，贱里真藏了阵脚线。顾小满远处又补一笔：绳，实价，塔在袖。

赵家人跟丢第四次，跟到巷口不见人，只好折返。赵玄机得知，铁牌在腰，冷笑半声又收住——收住像还库之后，不再派人盯九月老头的每一步。

白漱玉未递名帖，把粗茶带回客栈，苦后回甘，甘里第一次懂得：名声不在银子，在手暖了几双搬药的手。

司库记在心里当夜，执事怕长老真穷。司库答：「抠的是门面，松的是杂役手。手稳了，丹堂少骂，咱们少赔。」

莫长春归院，簿无字，袖暖。他念：「坊市烟火，是根的另一面。茶凉，讯要来。」杖答，气音轻。他笑：「不等。」

大因果还在远处，像妖潮——坊市小暖，第二次妖讯才烫。

归院前，莫长春在山门石阶停半息，袖空，杖在塔中，念：「烟火买了，线藏了，手暖了。下一帖红，比棉衣烫。」

司库记在心里次日，执事复查门里实价，见棉衣十件旁注「杂役手稳」，叹道：「长老抠门，抠的是咱们的脸面，松的是搬药的手。这门里实价，得记一辈子。」

名帖搁在茶摊，帖薄，意重——重得像白漱玉在远处，先记在心里未见的缘。
