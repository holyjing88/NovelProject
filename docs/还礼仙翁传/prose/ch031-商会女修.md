# 第三十一章 商会女修

缘箓二转后半月，青岚门比秋末更忙。

天试风声在，东脉战备在，丹堂炉火也在——忙得像一锅将沸的药。百宝阁管事入山这天，执法堂客厢早早收拾出来，茶未沏，先摆了货册，像摆一副算盘，等秦商言来拨珠。

弟子围在客厢外三层，不敢噪，却忍不住偷瞧——百宝阁女修入青岚，比赤焰来使还稀奇，稀奇在买卖，买卖里也许有情报。

秦商言入厢，青衣利落，笑挂在唇，像把买卖也穿成衣裳。她目光在莫长春杖与柳青鸢剑穗间游移，游得准，像数灵石也数人情。

「秦管事。」柳青鸢公事公办，「灵石周转、情报互换，按宗门例。」

「宗主言重。」秦商言拱手，「商言此行，不只谈灵石。」

她看向莫长春，目光停得久一分：「听闻莫长老将死之人，手却松——坊市送棉衣，东脉定魔少，风声都传到百宝阁了。」

廊外弟子听见「定魔少」，窃语起，像潮。

莫长春拄杖笑：「手松？老夫手紧。紧在只紧该紧的地方。」

秦商言挑眉：「何处该紧？」

「赤焰最近在哪花钱。」莫长春道，「老夫想知道这个。」

厢内一静。柳青鸢眸光微动，未拦——师兄要情报，宗主也要。霍镇山立在廊下，粗眉一扬，像听见「花钱」二字就精神。

秦商言沉吟，笑收半分，像终于认这是一场不对等的开价：「在妖与田之间。买路，买命，买你们青岚门犹豫的那七日。」

「很好。」莫长春点头，「茶还没沏，先记在心里。」

塔壁无新铭，塔影却一烫，像等人把下一礼送出去。

秦商言目光扫他袖，扫得细，像商人的眼能看灵石，也能看「记在心里」的重量：「莫长老，百宝阁做买卖，也做情报。情报有价，莫长老打算拿什么付？」

莫长春袖空，一念出塔，饼在塔中，再出，推案。纸包旧，边起毛，角磨圆：「家乡茶。不值钱。请你尝。」

秦商言一愣，厢内弟子也愣——谈赤焰夜袭，付的是茶饼？

莫长春笑：「难还才好。还得起的，就不叫礼。」

秦商言打开纸包，茶香扑出，苦后回甘。她拈起一块，细嚼，细嚼出涩，也细嚼出暖，像第一次被人把「无价」放在舌上。

案上多一卷薄简，墨迹新干，像茶气里突然长出字来：

**赤焰东脉调动：三日后佯攻，七日后夜袭。**

秦商言抬眼，声低：「这笔买卖，商言还没算清。」

「茶饼算清了。」莫长春摆手，「简是回赠。你问赤焰在哪花钱，老夫答在妖与田之间——茶，是谢你肯答。」

她沉默两息，忽然问：「若商言把简卖给别人呢？」

「卖就卖。」莫长春道，「情报值钱，人情更值钱。你卖简，是商道；你记恩，是礼。各算各的恩。」

秦商言起身，郑重揖手：「秦商言记下了。」

出厢时，柳青鸢在廊下路过，目光相触，各自点头，像宗门与商会之间，多了一根看不见的线，线那头拴着东脉，这头拴着茶饼。

弟子炸开一层议论：「茶饼换情报？」「莫长老又送礼！」「百宝阁管事竟真吃了！」

秦商言听在耳里，笑，笑里像亏本也乐意。

霍镇山在廊下听完全程，骂：「茶饼换夜袭？好！」周德海在远处听，笑僵，笑里像袖中有什么湿了。

秦商言下属问：「管事，这茶真比灵石贵？」秦商言答：「贵在人肯记恩。恩是另一条商路。」

莫长春出厢，弟子围问：「长老又送礼？」莫长春笑：「礼送了，简收了。接下来看宗主怎么战。」

秦商言出厢前，忽又回身，从袖中取出一枚小印，印轻，意重：「百宝阁实价印。莫长老若再入坊市，凭此印，不插队，不抬价。」莫长春推回：「老夫不插队，也不占商道便宜。」秦商言笑：「占的不是便宜，是信。信比印硬。」

霍镇山在廊下等秦商言过，粗声问：「管事，赤焰买路，买不买青岚的盾？」秦商言答：「盾在霍长老腰，不在商言心。商言只卖看得见的路。」霍镇山咧嘴，像终于肯信：「好，看得见就行。」

周德海在远处听「茶饼换简」，笑僵，笑里像袖中折子湿了又干。孙福旧党嘀咕：「长老又送礼……」顾小满冷眼：「礼在席面，不在嘴上。你看简，别看嘴。」

莫长春出厢，袖中简已交柳青鸢亲兵转送，转送像把战意也送进长老会。他杖点阶石，阶石凉，凉里像东脉七日后那一刀，已在磨刀。

客厢散后，柳青鸢独留秦商言半步，声稳：「简若真，青岚记恩；简若假，百宝阁记祸。」秦商言答：「商言不卖假，也不卖恩——恩在茶里，祸在别处。」

弟子围驿道，围「茶饼换情报」，围得像看一场亏本却乐意的买卖。有人怕，有人燃，燃的是终于明白赤焰花钱在哪——钱花在田，花在命，花在子时那一刀。

莫长春回院展简副本，烛火映字，像映一张将到的刀。他低声：「公平二字，商言说得对。老夫偏不爱公平，爱还礼。」杖中竹纹一游，气音轻。

苏念慈闭窗，闭得像怕商香进丹堂乱了苦甘。她低声：「买卖买卖，别传成私。」

女修离山前，留一句：「实价成交，下回还来。」还来像记在心里，笔在商会，不在私。

赵玄机铁牌在腰，笑浅：「商会价硬，门里该统。」顾小满远处记：客堂，实价，不赊。

延迟的爽点不在茶香，在简上墨迹新干那一瞬——新干，像茶气里突然长出字来，字是夜袭，夜袭是东脉，东脉是根。

商会女修进山门那日，香比丹堂还细，细里带价。

莫长春袖空，一念出塔，杖立客堂，不砍价，只问实价。弟子围半圈，围半圈像怕买卖烫手。

柳青鸢礼数足，声稳：「商会远来，青岚备茶。」女修笑：「茶要喝，价也要实。」

围观弟子客堂外踮脚，踮脚等成交——成交了，成交出一盒普通茶饼。

赵玄机验印记，印记硬，硬里仍想统，统到塔意，塔意统不住。

柳青鸢礼数足，声稳：「买卖公平，门才稳。」莫长春推银，推像第三遍证据：不买情。

顾小满远处记：客堂，实价。苏念慈闭窗，闭像怕商香进丹堂。

莫长春念：「修走，饼要来。饼轻，线重。」

客堂散后，秦商言将茶饼盒递顾小满，声平：「盒轻，线重。线别断在三月前。」顾小满接过，像接过半块石的另一块。

女修离山前，在驿馆留一句：「茶比灵石贵，贵在人还在。」像商言也懂——根在，价才实。

柳青鸢礼数足，声稳：「买卖公平，门才稳。」莫长春推银，推像第三遍证据：不买情，只买实。

客堂外，围观弟子踮脚，踮脚等成交——成交了，成交出一盒普通茶饼，饼轻，线重，重得像三月后的路。

赵玄机验印记，印记硬，硬里仍想统，统到塔意，塔意统不住——统得住一时价，统不住实。

魏无涯在檐角观成交，扇合，低笑：「实价成交，成交出一盒茶饼——饼轻，线重，重得像三月后的路。」

秦商言合价，笑浅，笑里价已涨——涨的不是茶，是青岚门三月后的路，路在根上。

弟子渐散，散里仍回头瞧一眼灰袍袖空——瞧得像瞧一眼九月还能走多久。

秦商言合价，笑浅，笑里价已涨——涨的不是茶，是青岚门三月后的路。路在礼上，不在价上。
