# 第六十章 血止沿前

第六十日，雪细得像粉。粉落肩上，肩沉，沉得住。

韩泥清西栏，粪锹沉，掌茧裂口只渗一丝潮。潮不是血，血不碰沿——沿上干着，像六十日守出来的底。

卯时前他又去搬柴半刻。霜厚，柴湿，斧声闷得像更鼓。管事验垛：「沉得住，记功在。」

记功在，麸皮不克。克粮克得走麸皮，克不走他那口正手。

午后编筐，末藤刺深，血珠再饱满。他甩手，血甩破席，席吸尽，沿仍净。净，像瓮守最后一夜前的预演——预演归预演，仍不主动滴。

坛腹温加剧，比廊下讲规矩那日又多一线。又多，像将醒未醒。醒在缘，不在人催。

秦霜远廊经过，一句：「手正，心别邪。邪，符灭。」

符温贴腕，七笔都记着，不往邪路偏。铁无言立柱边，声低：「沿净，好。净，像等缘的人。」

「等。」韩泥答。

等，不是拜血祠，也不是主动滴血。是活计扎，血找席，席挡沿，沿等韩氏守瓮血那一误。

同批少年问：「坛温加剧，像醒吗？」

「像眠。」韩泥说，「眠着，不急醒。」

「急醒咋办？」

「急，像拜邪。」韩泥说，「拜邪，路断。」

少年懂「断」，闭嘴。老耿挑水路过，水洒实：「六十日了，还守沿？」

韩泥点头：「守。守席，守缘。」

老耿不再言。言少，水正，像褐袍人当年嘱托的余音。管事查棚，灯晃坛角：「杂物收起。」

韩泥席盖不动。动像心虚；不虚，是守。管事盯半息走了——只见破席，不见仙缘。

邻床笑：「坛角盖那么严，藏宝？」

韩泥不辩，只把席盖再按实。按实了像编筐扎掌那日。掌茧再裂，裂在冬，不在沿。

刘婆端姜汤，姜薄，热：「六十日了，沿还净？」

「净。」韩泥说，「净，才接得住后头缘。」

他喝尽，只点头。姜热，像灶下还着的一星火。赖福讽：「守席守出仙缘？」

管事路过，声冷：「再嗓，克半月麸皮。」

赖福哑了。哑了，韩泥按席的手更正。夜雪停，棚静。他揭席一角看沿——沿有尘，有滑，没有血。无血，盖得更严，像给席盖更严那一夜做预演。

他对坛低声：「血止沿前，瓮温加剧。加剧，我不喊醒。」

掌正，心正，符暖，席严。外头更鼓二响，沉。明日仍搬柴，藤刺还会扎，血还会到席，席还会挡沿。挡到那一夜，缘才满；满到误触那一滴，瓮才醒。

邻床鼾如雷，雷里坛角静。他再对坛说：「你再眠半夜。半夜后缘满不满，看活计，不看心急。」

坛腹温一线，又凉。凉像应「等」。六十日擦沿，六十日不主动滴血——滴，要等误那一甩。

同批少年又碰他袖：「你守坛六十日了？」

「守。」韩泥说，「守席，守沿，守不主动滴血。」

「滴血干啥？」

「主动滴，像借腥。」韩泥说，「腥借来，路易断。」

他按心口，低声：「丫头那碗汤，热着还。还，要等瓮醒后，路才长。」

编筐末藤再扎，血珠饱满，甩落破席，席厚吸尽，沿边仍无尘。管事查沿，烛晃一线：「六十日，沿要更净。净，才接得住瓮温加剧。」

韩泥用旧布再擦一圈。擦罢沿滑、有尘、无血。少年蹭过来：「坛温加剧，你怕不怕？」

「怕。」韩泥说，「怕也记。记着，手就不飘。」

少年一愣，退半步。退像懂「怕也记」。管事验垛，斧痕新，柴纹直——直像手正留下的印。肩上旧印更深，沉得住，瓮温加剧才好接。

夜里少年又问：「六十日，你图啥？」

「图沿净。」韩泥说，「净了，才接得住误那一甩。」

少年不懂「误」，却懂「净」，闭嘴。他撕旧袄角补席边，补罢席更厚，厚像后头那一夜。席厚，血好找席；席好找，沿才好停在沿前。

扫罢又搬柴一担。柴沉，肩勒旧印，印深像丙九五年。五年磨出的肩，才扛得住瓮温加剧。符怀暖一线，掌裂不僵——僵了下一刺便偏；不偏，缘才正。

他摸木牌，七笔在心：叶丫头汤，振东药，陈姑袄，刘婆粥，符恩，耿恩，僧恩。笔笔在，手才不飘。卯时前斧声闷，闷得像把七笔一下一下钉实。

黄昏雪停，他扫棚前雪半刻。雪薄，帚短，扫出一路净。净不要紧，沿要净。铁无言路过，看一眼肩：「六十日了，肩还正？」

「正。」韩泥说，「正，才接得住沿前那一止。」

铁无言不再言。他盖席，破布又加一层。层厚，厚像后头那一夜预演。对坛低声：「六十日了。六十日沿净，才配说你暖加剧。加剧，我仍不喊醒。」

坛腹颤极轻，轻像应「等」。二更揭席看席边，湿痕止在沿前三指。三指外，沿净。

「血止沿前，席严，符暖，手正。」他说，「正，才配接席盖更严那一夜。」

三更将尽，雪又细一丝。他把破席再加一层角，角压住沿边湿痕，像给后头那一夜垫坎。邻床梦中喊「滚」，他不扰，只道：「滚在舌上，不在沿上。」

坛腹温加剧，他掌心贴坛底一瞬，贴完即离——离像敬，也像守：守规矩，不借温拜邪。四更铁无言巡廊而过，灯晃坛角一下便远，像说「沿净，手正，够了」。

五更天边白一线，白像席盖更严的前奏。前奏里更鼓尽，尽处一声鸦叫，叫短，叫得像缘将满未满。秦霜远廊一句：「瓮温加剧，别拜。拜，符灭。」

他低声：「加剧是缘，不是命。缘在，七笔就在。」

雪细停，棚静。静里更鼓远，远像瓮将醒未醒——醒在缘，不在人催。席盖更严那一夜已在眼前；眼前不要紧，沿要紧。沿净，才配接；接得住，才配醒。

五更末他按席，席厚，沿净。沿净处坛腹温一线又凉回去——像六十日守出来的界：界内席严，界外缘候。候的是活计误那一甩，不是人喊醒。

祖父嘱过：「席厚，沿净，韩家不散。」嘱在耳，他加第四层破布。布旧，旧更能吸；吸的不是仙缘，是活计扎出的血路。管事查棚第二趟，灯照席角，只见厚席不见坛。不见不要紧，沿要净。沿净了，管事走了，像默认守瓮也是杂役活计里的一种。

六十日午后，藤刺偏深半分，血珠更饱满，仍甩向破席，席吸尽，沿仍净。净像给后头那一夜预演——预演不主动滴血；滴血，要等活计误那一甩。甩在缘里，不在舌上。

管事查棚第三趟，灯再照，册上多记「六十日」一行，行丑，真在印。他独眼平，低声：「席严一线，才接得住下一误。血止沿前了——下一夜席盖更严，瓮温还要再沉一沉，沉完才醒。」

六十日黄昏，他把棚前雪扫成一条窄道。窄道通向西栏，也通向坛角。道净了，不等于沿很干净；沿干净，要靠手、靠席、靠不催。

铁无言第二次过廊，站停半息：「六十日，难为你。」

「不难。」韩泥说，「难的是急。急了会伸手滴。滴了，像偷醒。」

「懂就行。」铁无言灯影一晃远了。

刘婆收碗时多看他一眼：「沿净的人，夜里睡得着吗？」

「睡不实。」韩泥说，「睡不实也守。守到误那一甩，才算到站。」

她叹息一声，没再劝。劝不动的翁，粥却能热——热进肚，七笔里粥那一笔又实半分。坛腹温加剧时他不喊，只加第四层破布。布旧吸潮，吸的是血路，不是仙气。仙气还早；早的人爱拜，拜了符灭。

五更鸦叫后再敲一记更鼓远音。远音里他覆席盖严，严到管事第三次查棚也只看见厚席。厚席底下瓮温沉了又浮，浮完仍眠。眠着好——眠着，才把自己逼成一句落地的话：血止沿前了，下一夜席盖还要更严；更严之后，缘满不满，看活计误不误，不看舌头急不急。
