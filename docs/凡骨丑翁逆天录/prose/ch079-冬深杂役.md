# 第七十九章 冬深杂役

嗅诀萌芽后，冬更深一寸。柱上条换新，字丑，真：「冬深轮换，丙九韩泥，兽栏、药山、编筐，三日一轮。」

韩泥看完，应一声：「在。」

条墨还湿。湿墨像警告：冬深不是歇，是加压。加在兽栏臭里，加在药山霜里，加在编筐刺里。他倒不怕加。怕的是鼻下那寸根被冻僵——僵了，沉腐浮就退回哑巴。所以他袖里藏了半块刘婆饼，冷的时候啃一口，暖胃，也暖那寸根。

轮换不要紧，手在才要紧。手在，名才在册；名在册，才不给赖福晃秤晃没了。卯时丙九列队，霜白，呵气成线。赖福晃木牌，眼细：「冬深也轮换？轮换专给末席晦气活。」

韩泥不辩。辩费舌头；舌头得留着清栏、守坛、守鼻下那寸根。他只应：「在。」

第一日，兽栏。腥重，粪硬，铲下仍稳。稳里鼻下有根，根细，细才辨得出粪里一点药回——回像兽栏夜那日，回进掌纹，回进二层境里。小兽角里咳一声，咳轻。他换干草，草干，味淡一寸。管事路过，记一笔：「丙九韩泥，冬深仍清细。」

同批少年掩鼻：「冬深还兽栏，臭死你。」

「臭不要紧。」韩泥说，「臭也清。清完，才有饭。」

少年退半步，没再问。问像沾晦气；晦气不要紧，晦气遮瓮。他把角里浊草拖出去时，靴子陷进冻泥，泥咬踝骨。咬不要紧，二层气沉足底，足还抬得起。抬不起，轮换第一天就趴；趴了，名就从册上滑掉。

第二日，药山。边门开一线，他背筐上坡，坡滑，霜白。二层气沉足底，足底稳半分——稳像界碑前那一步：不过界，不贪活药，只捡渣。深雾起一寸，鼻下静，沉腐浮三字不落舌。沉者入左堆，腐者入右堆，假香上浮，指到即剔。

铁无言立雾外，声低：「冬深别怠。怠了，渣混；混了，诀白芽。」

「不怠。」韩泥应声，「沉腐浮，记着。」

筐满，交管事，记「深捡净」。赖福讽远：「冬深也深捡？捡出邪，连坐丙九。」

韩泥低声道：「只捡渣，沿净。」

坡下有人喳喳：「末席还能深捡，管事眼瞎了？」韩泥当没听见。听见要回嘴，回嘴像把芽吐给风。风一刮，三字就散。

第三日，编筐。藤刺扎掌，掌茧旧裂处又渗潮。潮不是血——血不碰沿。他偏手，刺浅些：浅些，沿才净；净了，坛才藏得住。邻床少年瞧：「冬深还编？」

「编。」韩泥说，「筐卖钱，钱买米。米在，愧字才还得上。」

刘婆端粥，仍多半勺，肘袄针脚又补一线：「冬深轮换累不累？」

「累。」韩泥说，「累也转。转了，名才在册。」

「名在怕什么。」刘婆哼，「手稳，饭稳。饭稳，恩不断。」

韩泥点头。恩在粥里，也在针脚里——针脚丑，挡风；风不进，臂才抬得起筐。午后药渣场角风硬，他分拣，鼻下有根，手快半分，快也要稳。铁无言瞥分堆：「冬深仍对。对，才配废渣收集。」

「废渣收集」四字落耳，像钩。钩先记——记冬深轮换，记三日一轮，记手不怠。秦霜远廊经过，木鱼轻敲：「冬深心易惰。惰了，鼻下香飘。」

韩泥不答。答在心里：冬深是活，活是路；路通向四层将显，通向废渣路。黄昏回棚，席严，坛不露。他按席角，坛腹温一线，像应「续」字。丑时前廊下练半刻闭息。息在鼻，不在嗓。嗓响，香飘；香飘，赖福就记「邪」。

邻床问：「冬深还守丑时？」

「守。」韩泥说，「沿净，不叫。」

「不拜灯？」

「不拜。」他声音平，「坛在，人在。」

更鼓沉，他摸怀玉牌，又摸愧字条，凉进骨。凉不要紧，鼻暖——暖在诀，不在众目。独眼平，手不抖：「冬深续了。轮换三轮，手不怠。四层那影子，在丑时。」

轮换第二轮，编筐与兽栏对调。筐成，管事验沿，记「手稳」。兽栏午后雪混粪，铲下仍稳。稳里鼻下有根，药回才辨得出。管事弟子验角，角净，又记「冬深仍清细」。赖福克扣未反弹，眼却细：「冬深也清？清出晦气。」

下雪时他袖里饼啃得更碎。碎屑掉进掌茧缝，他吹也不吹——吹了像馋。馋给人看，赖福就专扣你的粥。他把碎汤饼就着雪水咽，咽完继续铲。铲声稳，才像人还活着。活着，才谈得上后面见境。

韩泥不顶。刘婆顶风端粥，勺沉：「冬深别僵。」

「不僵。」韩泥说，「肘针脚挡风。」

铁无言立风里，声低：「冬深寒咬骨，鼻下更静。静了，四层影才不散。」

「静在鼻。影在掌。」韩泥应声。

白织月远廊过，没停。秦霜木鱼一声：「冬深心易缩。缩了，恩就凉。」

他心里恩不凉，愧不混，仇不拜。冬深是压，压后仍要活。黄昏分拣，沉腐浮并一息，管事记「辨性准」。准，是小胜。邻床又问：「冬深啥时候完？」

「完在春。」韩泥说，「春未至，手要在。」

「春前你都靠粥撑？」

「靠手。」他把茧掌摊开给少年看一眼，「手在，粥才进肚；手掉了，粥也救不了。」

编筐第三日夜里，棚顶霜厚，他哈气暖指再穿藤。穿到一半，藤条突然崩开，弹到颧骨，火辣辣的。少年笑：「丑叔也有闪手？」

「手会闪。」韩泥抹一下，「人别闪。人闪了，条上的名就空。」

他重新穿。穿实了，筐才卖得出价钱。价钱在，米在；米在，愧条才不干成废纸。

丑时前，他内视半息。第四阶影淡，淡不要紧；淡也是可见。可见是钩，钩着三层四层将破。他低声对坛：「冬深续了。四层影见了。不叫。」

坛环温一线。温像霜白里那一点掌烫——烫在暗，不在册。册写仍末，掌写二层，鼻写沉腐浮。北栏轮换第十日，雪深一寸，铲下仍稳。稳像踹台按实：掌在，境在，路在。

同批少年远避，避像避晦气。他偏手编筐，刺浅些，沿净。管事夜巡验棚，沿净，像腌菜坛。老耿挑水过，声哑：「境别拜邪。」

「不拜。」韩泥道，「只记着。」

老耿嗯了一声，水桶挂霜，像认可，又不像。韩泥不追问。追问像求指路；指路要自己走。更鼓又沉。他按木牌，牌凉贴胸。手在，名在册，坛沿净。活长了，才还得了叶汤，才接得住门缝灯。

夜里棚里有人冻得砸牙：「这日子啥时到头？」

「到春。」韩泥睡前只留一句，「到春前，别让手先停。」

他说完把自己那半片饼掰给邻床一点。给了不解释。解释像买好。买好没用，春前手别停才有用。

药渣场他又多留半刻，把冬深三日样各取一撮并闻。并出一线清，清得像铁无言点头那下。他不跟任何人说快。说快像求赏；赏来了，芽就贵，贵了就飘。飘不得。飘了，轮换条上三个字全变枷。

同批有人冻哭：「丑骨头命硬，我们可撑不住。」

「撑不住就少说话。」韩泥把干草塞进那人手里，「手有活，牙就咬得住。咬住了，春才近。」

那人接草，骂了一句，到底没撂挑子。撂了，全棚更冷。韩泥看他走开，才把自己手插回袖里。袖里暖一分，鼻下根就稳一分。稳一分，四层影就不散。

四层那盏远灯已在掌前晃——晃着不叫；不叫，明日丑时才配真去见它一眼。见了也不吭；吭破了，冬深这一轮脏活就白熬。白熬不起。熬不起，三年约就只剩凉牌，凉牌还不起叶青禾那碗烫。烫还在心里，心里有烫，手才抬得起明日那一铲。
