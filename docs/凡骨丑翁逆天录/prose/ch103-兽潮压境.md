# 第一百零三章 兽潮压境

气压沉得像暴雨前。雨还没下，兽先在界外聚成潮。潮压药山，压得人胸口发闷，闷得像一口痰卡在喉里。

韩泥仍钉在东角。角上的草已经黑了一片，黑是爪刮出来的。刮在障上，障还没破——没破不等于轻松，甜腥那一线细得像蜜，蜜便是毒。毒一渗，栏里倒了两只小兽。倒了就得清，不清，整栏都要染。

他把第三囊灰香尽数撒开。撒完指间空了，空里却还咬着经页残诀。诀在口中打转，不敢出声——出声像邪，邪了秦执事一到，比兽潮先收拾人。雾从齿缝里慢慢漾开，白而细，细着往角里钻。甜毒退，小兽喘息缓下来，缓得像又捞回一口气。

铁无言拍他肩，掌沉：「角稳了。跟我挪三步，填障。」

韩泥挪三步，步步实。实里看见铁无言臂上被爪风划开一道口子，口子浅，浅也渗血。他扯出刘婆那条粗布，给他包上。布扎紧，血止住。铁无言嗓子低：

「谢不用。手在就行。」

「在。」韩泥应。

在，就是并肩的开头。开头不响，响了反像邀功。

赖福躲在障后，声音发抖：「怎么还不退！退啊！压上来了！」

管事弟子喝得更硬：「退不退，听令！守到卯时！提前撤，记过连坐！」

卯时还远。远归远，压在眼前。压在眼前，心就不能先跑。白织月立在内墙，这回没拿冷话甩人——不嘲也不是温柔，是她也闻见那股甜，甜得心里发毛。发毛归发毛，角上仍要人顶着。

韩泥蒙条又湿了一层。湿着才挡得住往肺里钻的味。他独眼平着扫障外，兽影一挤一挤，像要把界压弯。界弯一线，栏里就多倒一只。

「还能守吗？」同袍嗓子发干。

「能。」韩泥说，「手稳。」

「稳……我腿在抖。」

「抖也踩实。」韩泥道，「踩实了，障才不塌。」

铁无言扛来两根湿木，往缺口一卡：「填！别看内墙。内墙看我们，我们看障。」

韩泥跟他并着肩把木顶牢。木吱呀响，响里他肩头发麻。麻不是逃，是活计做到这份上该有的沉。

夜渐尽时，兽吼忽然矮一线，矮得像退。退是假的。假退最毒——人一松气，真压会再上来。韩泥不松。他把空囊塞回袖底，经页按回怀中，像把「压」字也按住。

回棚半刻，掌茧新裂，裂口涩疼。疼是活计，不是血糊上坛沿。沿得净，坛才藏得住。他按住坛，低声道：

「压过了。角还在。」

坛腹温着，像应「压」字。压不是终。终在「至」——至了，才算潮真咬进嘴里。

邻床喘着问：「还会再来？」

「会。」韩泥说，「来了，仍守。」

「守到啥时候算完？」

「守到角还在。」韩泥顿了顿，「角在，人就有地方站。」

更鼓尽，柱上条换了字，字硬得刮眼：「兽潮压界，全员备战。」

备战他其实已在战外缘。外缘不大，够丑翁把自己钉住。钉住了，内墙的香袖、赖福的尖嗓，都暂且靠边。

第二更障木又嘎吱响。韩泥肩顶上去，顶得汗往眼里爬。铁无言喘着骂：「压界，别退。退一步，界破；界破，内门就会拿丙九祭旗。」

「不退角。」韩泥说，「只挪步填障。」

管事弟子远处喊：「退！退至内墙线！外线危险！」

喊归喊，韩泥脚仍踩着角泥。退角等于让栏先倒；栏倒，赖福乐得记过，连坐就落到丙九每个人碗里。碗里没粥，恩就还不起。

刘婆托沈枯芽送来姜汤，汤已凉。沈枯芽怯声：「叶姐说，别逞能。逞能……回不来。」

韩泥不接碗，只把话接进心里。逞能不在角上站着显，显在清路能不能接住下一更。路清了，至时才有掌炉的空。

「姜汤你拿回去。」韩泥道，「喝了手软。手软别上角。」

沈枯芽眼圈红了，还是走了。走时回头看他，看一眼，像确认他还立着。

白织月忽然朝这边望过来，望完又扭开。扭开像怕被当成关心。韩泥也不去认那一眼——认了像讨，讨不如把障再砸结实些。

雾散后角内甜味薄了。薄得能听见小兽肺叶轻微起伏。铁无言活动一下包布的手臂：「还疼。疼着能挡。」

「挡就行。」韩泥说，「疼留着，记账。」

「账记谁头上？」

「记潮上头。」韩泥独眼一眯，「也记自己——自己要是飘了，先打自己。」

赖福又缩出来喊：「丙九别送死！退后面来！」

韩泥不应。应了就是跟怕一起跑。他只把木桩再楔深一寸。寸有寸的用处。

第三更小兽又倒了三只。倒要清，清仍靠诀。诀不出大声，只在齿间一碾，雾再起。雾里甜退半丈，半丈够活人换气。换气的人别嚷「胜利」——胜利还不配，只能说还喘着。

「韩泥！」铁无言喊，「左边缝！」

「看见了。」韩泥铲粪叉一别，把湿草石灰糊进缝里，糊得丑，丑能挡一阵也行。

管事弟子过来记簿，笔尖顿了顿：「丙九韩泥，角稳。」

赖福脸瞬间发青。青就青，簿上字真，真在「稳」，不在「丑」。丑他认了半辈子，稳是近半年来一铲一铲练出来的。

第四更末，气压更低，低像界破前最后一口气。气里韩泥空袖发凉——囊尽了。尽了还有掌上那点炉火埋伏着，埋伏不能现在亮。亮早了，至时反而没后手。

铁无言逼近一步：「还能守？」

「能。」

「手稳？」

「手稳。」韩泥说得短。短才实。

柱上「全员备战」的条在夜风里啪啪作响。响像催命，也像提醒：压住了不等于完，下一更若「至」字落地，才真正咬进肉里。他按住木牌，牌凉贴胸，胸口那股闷不散——不散也好，闷着人清醒。清醒着立到更尽，角还黑着、还湿着、还在。

近处吼声忽然贴到障外皮上。贴着磨，磨得木屑落。韩泥低声道：压在了。下一更，至。至时手要在，铁无言要在，经页要在——在，才谈得上还把人带出潮口。潮口之外，那块「明日」的条早被新条盖住，盖住不等于忘；忘了规矩，压完照样死。

压界将尽那一刻，雾散得更快。散完，甜味还会回潮。回潮时韩泥咬牙再碾半口残诀，雾浅浅罩住角内。罩住，小兽才不至于接二连三地软倒。铁无言喘息着问：

「囊真空了？」

「空了。」韩泥摊指，「空了还有嘴。嘴里这点诀，不能天天用尽。」

「留着至时？」

「留着。」韩泥点头，「至时比压更咬人。」

同袍忽地抓住他袖：「我看见……看见爪从缝里伸进来了！」

「看见就糊！」韩泥把石灰糊一把塞他手里，「糊完别看。看久了腿软。」

那人糊得手抖，抖完缝居然堵住。堵住就值。值不在漂亮，在多撑一息。

白织月在内墙忽然掷下一句话：「东角别塌。塌了，我这线也难看。」

韩泥听着，像没听。难看是她的事；角不塌是他的活。活做完，脸自然在。

刘婆那边又派人递来粗布两条。递的人喘着：「刘婆说——血流多了包不住命。命在，粥才有人喝。」

「替我应声。」韩泥接布，「应了：人在。」

人在俩字轻，轻得像风。风里兽潮还在压。压到木桩咯吱响，韩泥肩膀顶上去，顶到骨头发酸。酸着他仍盯裂口：裂口一线，一线里若涌出「至」，今晚这钉子就不能拔。拔了，备也白、撒也白、雾也白。

他按着木牌，对铁无言只剩半句：「下一更若至，你挡，我炉。」

「成。」铁无言牙关一咬，「至了就别省。」

「不省。」韩泥望向岭色最黑处。黑里有脚步，脚步很快会踩进「至」字里——踩进来之前，他先把心按进角泥。泥凉，规矩不凉。
