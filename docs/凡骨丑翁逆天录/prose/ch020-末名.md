# 第二十章 末名

旱第五日，里正贴榜。

榜在祠堂门，墨新，字丑，却硬：「十月采药额，按斤计，末名罚月例，连坐户口。」

「连坐」二字小，小得像故意让人先看见名，后看见罚。罚在后面，像刀背；名在前，像刀口。韩泥挤进人群，独眼辨字。名字从上往下，像刀排队：赵家亲戚、沈家药账、周家汉子……每一个名字旁都有斤两，斤两像肉，割的是活路。

最末，三个字，浅，浅得像被故意磨过：

韩泥。斤两：七两。倒数第一。

人群让开缝，缝像看戏。戏免费，戏刀却真。马春花笑，笑短：「绝户翁采药，采到榜尾，晦气连山都嫌！山都枯，你活啥？」

赵绣春扇风，扇得慢，慢得像体面：「赖管事早说了，晦气翁沾谁，谁倒霉。山倒霉，榜也倒霉。」

周嫂念榜尾，念得响，响给全祠堂听：「七两！够买几文咸菜？药铺寡妇还送他姜，送姜送成榜末！」

韩泥不辩。辩，成嘴仗；嘴仗救不了榜。榜是村规，规后有人——赖扒皮短签还在村头柜，像牙，牙等咬。他看榜，看自己的名在末。末，不是手笨。后山旱，药枯；他臂伤未全，坡滑；更有人传，跟他同坡的，会沾晦气，村娃不肯指路，指路要挨娘打。压，压得实。实里，他反而静。静得像井绳，绳上没水，绳还在。

里正念榜时，嗓哑，像也怕旱：「末名罚月例，不得推诿。」

有人插嘴：「韩泥臂伤，坡滑，七两……也算数？」

里正顿：「榜按斤，不按伤。伤，自己熬。辩，成求。求字轻，手字重。手还实，七两就七两。」

韩泥转身，担筐去集尾。筐里还有昨夜编剩的两捆细藤，藤韧，能卖。藤不晦气，晦气的是人嘴。嘴能贴榜，贴不了藤的价。半个时辰，卖出九文——比旱前少，少也入袋。九文，够买盐，够熬稀粥，够让命再撑几日。几日就够，够到还，够到公审后。

回村路上，他路过沈家豁口，停半息，没叩门——叩门，成牵扯。隔巷听见芽儿咳，咳轻，像药叶还管用。管用，榜末也不绝恩。恩在，手就还实。

张麻婆路过，瞥他，嗓劈：「榜末咋了？筐实，手就实！手实的人，老天爷瞎眼！」

张麻婆啐：「山不实，扒皮实！实吃人！吃沈家，吃芽儿，吃你，吃全村！」

骂是暖，暖里带刀。刀向外，不向他。韩泥记着，不谢。谢字欠着，欠到能还。张麻婆又道：「榜末怕啥？筐在，手在，命在。命在，扒皮吃不着你骨头。」

韩泥：「骨头硬，给他硌牙。」

张麻婆笑，笑短，笑里没嘲：「对，硌牙好。硌牙，他就不爱咬你。」

她又补一句，嗓压低：「榜末咋了？末席也能坐。坐着，听真话，比站着骂强。」

韩泥：「明日去。坐着听。」

张麻婆「嗯」一声，担筐继续走。走，像释——释不在榜，释在九文，释在张麻婆那句「命在」。

回村，他路过榜，用指尖蘸唾，不抹名，只抹「罚」字。抹不净，净的是心：心不被末名压弯。弯了，就真成晦气。榜前有人偷指他名，指给娃看，像指鬼。娃躲，躲得像躲榜末的斤两。

韩泥路过，不近，只把担筐放直——筐直，人直，直不是傲，是手还实。实，就是末名章里的小胜：榜能压名，压不住藤，压不住九文，压不住叶汤那点热。热在，人就还不是绝户——绝户是户籍，不是命。

门外有人敲，敲得轻：「韩翁，沈家让捎句话。」

韩泥开门缝：「说。」

来人低声：「芽儿咳，药叶不够。你榜末，别往心里去。沈家还记着你狼口那回。」

韩泥：「记着了。药叶我收过。咳，让娃多喝热水。」

来人：「里正说明日公审，沈家不站赵家。」

韩泥顿半息：「不站，别惹。惹了，连坐。」

来人点头，走。走远了，韩泥合门，心里又沉一分——沉不是怕，是压里多了个暖：末名在榜，沈家还肯捎话，话就是释。

屋里，心口四恩在，仇苗在。他在末名旁又记一辱——辱，候着；不急着报。报要活，活要粮。粮在缸，缸里米是九文换来的，少，够三口。三口不是一家三口，是一家一口、一家命、一家还。

他舀米，慢洗，水浑，仍洗。洗米像洗榜，洗不掉末，洗得掉慌。慌洗了，手就实，实就能编筐，编筐就能再有九文。榜末是压，洗米是释——释不在名，释在手还实。

「末名在榜。」他低声道，「恩不在榜。恩在心里。恩在叶汤，在衣捺，在门墩的布包，在叶家没灭的灯。」

心沉一分，沉得像肩水。肩水实，路还长。长不怕，怕的是不走。门外，祠堂方向又聚声，声里提「明日公审叶家」。韩泥合袄，塞怀。腰上硌，硌得清醒。末名是压。清醒是释。释不够，够活。活到明日巳时，活到叶青禾把秤摆实。

赵绣春派人路过，瞥榜末，瞥他屋，瞥完啐：「末名配绝户，绝配！」

韩泥在门内「嗯」一声，不出门。出门，成戏。戏让赵绣春赢嘴，他不让嘴赢，让手赢——手实，筐实，命实。命实，明日公审才坐得住末席。

编筐时，他摸袄肘针脚，针脚密，密得像叶青禾的手还在——手在，汤就还能热；汤热，榜末也不冷命。榜末名在，九文在掌。掌沉，沉得像告诉全村：末不是绝，是候——候公审锤，候叶汤还热。他用指节敲了敲筐底，声闷，闷而实——实了，才配装药，配卖钱。

夜里，他仍编筐。筐卖出，榜末也能活。活，就是压里的释。独眼瞥榜墨，墨新，名旧——旧名在五十八年，五十八年都在末，末惯了，不怕再末一笔。

丑时，榜前墨香散进漏屋，散得像催他早睡——早睡不实，实的是手。手还在编，藤条勒指，勒出红印。红印浅，像末名也能勒出一道活路。坛角微凉，腹里一线温，像瓮听见了「末席」——听见，不醒，只记。

他低声道：「榜末七两，手字重。公审照样去。」

编到丑时，藤条断一根，断得像把「七两」也编进筐里——编进，背着走，不辩。缸里米少，少，仍煮粥。粥稀，稀得像榜末的命——命稀，还在。

明日巳时，祠堂锣未敲，心先悬。悬着，就不算绝户绝尽。绝尽的是户籍，不是手。手在，末席在，锤来也坐得住。

粥落碗，碗烫，烫得像叶汤还在——还在，榜末也不绝命。坛腹微温一丝，像瓮听见了「末席」——听见，不醒，只候明日锤。九文在掌，掌温，温得像沈家捎来的话还在——话在，榜压不住手。

韩泥举目西望，叶家药铺灯一线，像恩还在远处守着。榜能压名，压不住叶汤那点热。热在，人就还不是绝户。叶家药铺灯，终夜弱亮，弱亮像叶汤的影——影在，榜末也不冷命。

赵绣春门内又骂半句，骂到「末名」便住——住，像也怕明日锤。韩泥合门，漏雨泥屋又漏一阵，雨丝细，他不躲——躲像亏心。手实，就能带坛上路；路还长，长不怕，怕的是不走。

他合眼，手仍按坛沿。五十八岁，绝户，晦气翁，还在泥岗村西头这间最漏的屋里。末名在榜，恩在心里——心里热，手就还实。怕的是叶汤被谣咬断；叶汤在，末也在末席，末席也能听真话。公审明日见。沿口一线温，贴住虎口，脆，短，够撑明日公审。瓮未醒，他不喊醒；醒在缘，不在这一更。活路在，坛在。
