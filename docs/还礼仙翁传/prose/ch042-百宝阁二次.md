# 第四十二章 百宝阁二次

秦商言二次入山，驻点设在坊市驿，人未到，货先到——隐匿符材料一小匣，附字条：「夜袭需暗线。茶饼之恩，商言还一半。」

莫长春至驻点，未带弟子，灰袍素，像老客串门。驿外弟子偷瞧，偷瞧里好奇——百宝阁女修二次来，送的不是灵石，是符材，像又一出「茶比灵石贵」的戏。

秦商言斟茶，茶比上次好，仍非灵品，像刻意让他记得「家乡」与「买卖」的差别：「赤焰夜袭时，东脉侧径最险。暗线若断，丹堂先危。」

「所以你来送材料。」莫长春道。

「来换线索。」秦商言笑，「公平。」

他取茶饼余半块，推过去：「上次你说无价最难还。老夫再给你半块，把难还再推一推。」

秦商言怔，继而笑出声，笑里真了几分：「莫长老，商言亏本乐意。」

「亏本好。」莫长春道，「亏本的人，记得恩。」

她低声：「公平二字，我常说。遇见你，才知公平不是等价，是肯不肯还。」

莫长春不接情话，只问：「材料谁手？」

「百宝阁最好的裁纸人。」秦商言道，「裁符纸如裁命，贵，值。」

他收匣，塔影一烫。秦商言望他袖，像觉出那里有什么，终未问，只道：「商言还有一句。周德海近月与赤焰信使频见，驿馆后巷，三次。商言不卖此条，记恩心。」

莫长春点头，像记在心里：「记下了。恩心比买卖价难还，你还得起。」

秦商言起身，郑重揖手。出驿时，弟子窃语：「长老又送半块茶？」「百宝阁管事又笑了。」「这次笑得像真亏。」

霍镇山闻讯，骂：「好！暗线要活，丹堂才稳！」

周德海在长老院听闻，面色平，平里冷，像早就备好的稿。

顾小满在杂役堂分班，眼里亮，亮得像雨夜金光还在。她问：「隐匿符……小满能学吗？」杂役师兄答：「学不学，看长老肯不肯给路。」

秦商言出驿，望青岚山门，低声对下属：「莫长老的难还，商言认。认恩的人，不走绝路。」下属问：「若周德海真是奸……」秦商言截话：「奸不奸，商言不卖。商言只记：茶比灵石贵，暗线比货贵。」

赵玄机在思过崖，听坊市驿消息，指节白，像妒里又掺进一丝说不清的东西——百宝阁女修二次来，不为卖，为还。

茶饼又少半块，符材多一匣——还礼仙翁的恩，从来不算公平，只算肯不肯还。

魏无涯在檐角观，扇合，低笑：「周德海三次见信使，线深了。」

百宝阁二次开那日，价硬，硬得像把三十年也标了数。

莫长春袖空，一念出塔，杖立阁心，不赊，不欠。弟子围半圈，围半圈像怕价烫手。

柳青鸢遣银，银被推回：「买实，不买情。」顾小满远处记：百宝阁，二次，实价。

赵玄机铁牌在腰，笑浅，笑里空一线——价若硬，私情就软。

苏念慈闻阁香，闭窗，闭得像怕价进丹堂乱了炉。她低声：「买卖买卖，别传成私。」

二次进阁，莫长春买一根阵钉，钉贱，贱里像藏东脉线。阁主笑：「长老也买钉？」莫长春道：「买实价。」

赵玄机的人跟丢第四次，丢里像妒也丢——丢的是跟，跟不住的是塔意。

驻点茶未凉，秦商言已把符材匣推过：「裁纸如裁命，贵，值。莫长老若问价，商言仍说实价。」莫长春答：「价在买卖，恩在茶。茶少半块，恩多一半。」

霍镇山在廊下等秦商言过，粗声问：「管事，符材到了，东脉侧径谁走？」秦商言答：「路在商言心外，恩在茶里——走不走，看青岚敢不敢。」霍镇山咧嘴：「好，敢走就行。」

弟子围驿外，围「半块茶饼再送」，围得像看亏本乐意——乐意的人，记得恩。

莫长春不辩，杖点青石板，像把路点向东脉侧径：「材料到了。暗线能不能活，看杂役堂敢不敢走夜路。」

孙福在角隅听「周德海三次见信使」，面色白，像听见自己的旧恩仇将被翻。顾小满冷眼：「你看路，别看嘴。」

秦商言驿中写信，信末加一句：「商言欠莫长老半块茶，欠青岚一条暗线。此债，商言认。」

魏无涯传讯厉无殇，只四字：「线已深。」厉无殇阅讯，焚，笑冷：「奸线深了，戏才好看。」

同一夜，长老会急议，柳青鸢指节叩案：「暗线若活，丹堂才稳。」周德海笑僵，笑里像袖中折子湿了又干。

林挽月罚役归途，听「半块茶饼再送」，唇动，像想说赵师兄与此有关，终未近。她把一枚新制符纸塞门缝，符薄，意厚。

苏念慈闻阁香，闭窗，闭得像怕价进丹堂乱了炉。她低声：「符材到了，炉不能熄。子时丹堂最危，炉不能熄。」

顾小满在杂役堂分班，眼里亮，亮得像雨夜金光还在。她问：「隐匿符……小满能学吗？」杂役师兄答：「学不学，看长老肯不肯给路。路在子时，在侧径，在还恩。」

外门弟子围驿道，围「半块茶饼再送」，围得像看一场亏本却乐意的买卖。有人怕，有人燃，燃的是终于明白暗线比货贵——贵在人肯记恩，恩是另一条商路。

霍镇山在廊下听完全程，骂：「好！暗线要活，丹堂才稳！杂役堂敢不敢走，看小满敢不敢学。」

周德海在长老院听闻「隐匿符材」，令旗在指间转，转得像犹豫，犹豫得像信使已在后巷候过三次。

驻点散后，秦商言对下属又补半句：「莫长老的难还，商言认。认恩的人，不走绝路——走绝路的人，记不了恩。恩在茶里，线在侧径，线在子时。」

苏念慈闻阁香，闭窗，闭得像怕价进丹堂乱了炉。她低声：「符材到了，炉不能熄。子时丹堂最危，炉不能熄——熄了，全门都苦。」

魏无涯在檐角观，扇合，低笑：「周德海三次见信使，线深了。材料到了，暗线能不能活，看杂役堂敢不敢走夜路。」

弟子围驿外，围「半块茶饼再送」，围得像看亏本乐意——乐意的人，记得恩；记得恩的人，不走绝路。

莫长春回院，塔壁无新铭，袖却暖，暖得像符材的重量已进袖里，进东脉侧径，进子时。他低声：「钉小，谋要来。谋比钉毒，毒在身边——毒在周公，不在秦商。」

顾小满远处记：百宝阁，二次，实价；隐匿符材，一次，不赊。她低声：「线记下了。线活，丹堂才稳。」

秦商言望青岚方向，笑，笑里第一次像亏本也乐意：「赚在青岚若赢，百宝阁有路走；若输，商言也有恩可还——恩，是另一条商路。」

孙福在角隅听「隐匿符材」，面色白，像听见自己的旧恩仇将被翻。顾小满冷眼：「你看路，别看嘴。线活，丹堂才稳。」

驻点灯未灭，茶未凉，符材匣在案，案上像多了一条将到的路——路在侧径，路在子时，路在还恩。

霍镇山在廊下等秦商言过，粗声问：「管事，赤焰买路，买不买青岚的盾？」秦商言答：「盾在霍长老腰，不在商言心。商言只卖看得见的路。」

风腥，云低，子时像已贴在靴底。隐匿符材在匣，匣在袖，袖里有网，网在子时，时在刀前。

莫长春袖空，一念收塔，杖落掌，声轻：「钉小，谋要来。谋比钉毒，毒在身边。」

百宝阁价又敲了一遍，敲得像告诉全门——礼在价上，价在人心。

弟子渐散，散里仍回头瞧一眼灰袍袖空——瞧得像瞧一眼九月还能走多久。
