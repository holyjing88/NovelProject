# 坐门口梗 · 变奏表

> **目的**：保留莫长春「不抢戏、装穷、坐门外」人设记忆点，**避免因果复读**导致读者疲劳。  
> **铁律**：同一姿势，**每场戏的功能必须不同**。

## 四、写作禁忌

- 禁止连续两章「门外石阶+不接茶+跟坐四椅」全套复刻。  
- 禁止弟子笑料同一台词（「又坐门口」）连用三章；第三次起改弟子反应为**习惯成传统**。  
- 莫长春进门 = 稀有事件，进门必有**剧情级转折**（纳绶夜、生死战）。
