# 第一百零五章 兽潮中

潮中。

没有「中间」好看，只有第二更、第三更、第四更。更尽，障破又补，补又破，破不要紧；破在障，不在心。

心要稳，稳在角。角在，韩泥在。

铁无言臂伤重一线，重不要紧；重，仍挡。挡时，韩泥以灰香清路，清路像清栏——清栏他熟，熟才稳。

「左漏！」铁无言喊。

韩泥左挪三步，三步撒灰，灰尽，尽不要紧；尽，还有经诀。诀出，雾起，雾里甜毒退。

退半寸，半寸像活。活，是栏里小兽少倒两只。

两只，够了；够了，不算功，算活。

赖福被派运石灰，运过韩泥角，瞥一眼，瞥像不认——不认不要紧，瞥是记：丑杂役没逃。

管事弟子传令：「杂役退二步，内门上！」

退二步，韩泥退，退到仍触障角。触角，角就不散。

白织月内墙，这次递来一包伤药，药低，低像意外：「不是谢你。是别死在内墙前，晦气。」

韩泥收，收不谢。谢轻，药用在铁无言臂上。

铁无言声低：「药正。你嗅的？」

「嗅。」韩泥说，「嗅完，手稳。」

臂包好了，好不要紧；好，才配继续并肩。

兽影再贴，贴，爪风带毒。毒来，他含清瘴丸第二粒——粒尽，尽不要紧；尽，还有第三粒，第三粒留至续。

第三更，他掌炉再炼，炼在角背，背障，障挡目。挡目，才匿炼。

丸成两粒，一粒自服，一粒掷邻角同袍。同袍接，接而活，活像链——链在108，在远。

链不要紧，这一章是中：中没有高潮，只有守。

守到卯时，兽吼低，低像退潮。退潮不要紧，潮中，人还立。

他立，像桩，桩裂，裂不要紧；裂，是活计。

回棚，刘婆端粥，粥稀：「还活着？」

「活着。」韩泥说，「手稳。」

刘婆哼：「饭稳，恩不断。」

按坛，低声道：「潮中过了。还没清。」

坛腹温，像应「中」字。中，不是终。

丑时，静半刻。香落齐，掌纹又烫一线——烫，像五层实极，极而不破。

邻床睡实，睡实了，像信他角稳。

更鼓尽，界外仍腥，腥像续。续在106，在远。

他低声道：「潮中。下一程，续。」

续，手在，铁无言在，清瘴路在。

潮中，第二更、第三更、第四更。更尽，障破又补。

铁无言臂伤重一线，仍挡。韩泥以灰香清路，清路像清栏。

韩泥左挪三步，撒灰，灰尽，诀出，雾起，甜毒退。

白织月递伤药，声低：「别死在内墙前，晦气。」

韩泥收，药用在铁无言臂上。

管事传令：「杂役退二步！」

韩泥退，退到仍触障角。

刘婆端粥，粥稀：「还活着？」

第三更掌炉，炉在角背，背障，障挡目。挡目，才匿炼。匿炼，像95，像110，像丑翁的路——路在暗处，路在渣，路在手稳里。

白织月递药，药低，低像意外。意外不要紧，药用在铁无言臂上，臂包好了，才好继续挡。继续挡，106续，107钩，108清——清，才终这一段。

同袍接，接而吞，声低：「……谢。」韩泥不答谢，答：「手稳。」

「左漏！」铁无言喊。退半丈，半丈够同袍进。进，才并肩；并肩，才像108。药用在铁无言臂上。臂肿退一线，退，像活。活，是小胜；胜在活，不在嘴。

赖福记过簿，笔停：「丙九韩泥，东角在。」在，就不逃。不逃，铁无言拍他肩，肩重：「还能守？」「能。」

守到卯时，兽吼低，低像退潮。退潮不要紧，退潮是假，106才是真。真在更尽，更尽，角还在。更鼓沉，韩泥按木牌，牌凉贴胸。下一程，续。」

兽潮中，中没有「好看」，只有第二更、第三更、第四更。更尽，障破又补，补又破，破不要紧，破在障，不在心。

铁无言臂伤重一线，重仍挡。稳，才第三更掌炉再炼，炼在角背，背障，障挡目。目挡，才成丸两粒，一粒自服，一粒掷邻角。

「左漏！」铁无言喊。韩泥左挪三步，三步撒灰，灰尽，诀出，雾里甜毒退。退半丈，够同袍进。进，才并肩；并肩，108才像真。

白织月递伤药，低：「别死内墙前，晦气。药用在铁无言臂，臂肿退，退像活。活，小胜；胜在活，不在嘴。嘴要留来喊，喊角漏，漏比谢大。

赖福记簿：「东角在。」在，就不逃。不逃，106才续——续在更尽，更尽，角还在。还在，107才钩余瘴，108才炼清瘴丸，109才稳炉，110才培元散。退潮假，106真。下一程，续。」续，手在，铁无言在，经页在，坛温一线在怀。

兽潮中，第二更、第三更、第四更——更尽，障破又补，补又破，破在障，不在心。心稳，角稳；角稳，第三更掌炉再炼，炼在角背，背障，障挡目。

「左漏！」铁无言喊。韩泥左挪三步，撒灰，诀出，雾里甜毒退。退半丈，够同袍进——进，才并肩；并肩，108才真。真在107钩，钩在108炼。

白织月递伤药：「别死内墙前，晦气。」韩泥收，收不谢，药用在铁无言臂。臂肿退，退像活——活，小胜；胜在活，不在嘴。嘴要留来喊，喊角漏。更鼓沉，按木牌，潮中过了，下一程续。续，手在，铁无言在，经页在，坛温在怀。

潮中第三更，障破又补，补又破，破在障，不在心。心稳，角稳；角稳，掌炉再炼，炼在角背，背障，障挡目。目挡，成丸两粒，一粒自服，一粒掷邻角——掷，像活；活，才106续。续在远，105只守更尽。更尽，铁无言臂伤，白织月递伤药，药低，低像意外：「别死内墙前，晦气。」收，收不谢，药用在臂上，臂肿退，退像活。

赖福记「东角在」，在，就不逃。不逃，107才钩，108才清瘴。清瘴在远，105只清路——清路像清栏，清栏他熟，熟才稳。在，就不绝。

潮中第四更，障又破，破不要紧，破在障，不在心。心稳，掌炉再炼，炼在角背。背障，障挡目，目挡，成丸两粒。两粒，一粒自服，一粒掷邻角——掷，像活。活，106才续；续在远，105只守更尽。更尽，手在，就不滚。

目挡，成丸两粒，一粒自服，一粒掷邻角——掷，像活。

「左漏！」铁无言喊。还在，107才钩，108才清瘴。清瘴在远，105只守更尽。在，就不滚。

第五更，障又破，破不要紧，破在障，不在心。心稳，掌炉再炼，炼在角背，背障，障挡目。目挡，成丸两粒，自服一粒，掷邻角一粒——掷，像活。活，106才续；续，退时扶铁无言，扶是并肩之续。更鼓沉，按木牌，手在，就不滚。

潮中第六更，人换角不换，不换，角黑，黑是汗，是爪，是毒。毒浓，浓像最后一扑——扑时，掌炉再炼，炼在角背，背障，障挡目。目挡，成丸两粒，自服一粒，掷邻角一粒。掷，像活；活，106才续。续，退时扶铁无言，扶不要紧，扶是并肩之续。扶了，107才钩，108才清瘴。更尽，兽吼低，低像退潮。还在，就不滚。

滚在钱戾衡脚，脚在远，约在怀。怀贴身，像守坛沿。更鼓沉，按木牌，手在，路在，就不绝。

更鼓尽，兽吼低一线，低像退潮。还在，手在，铁无言在，经页在，就不滚。滚在远，约在怀，怀贴身，像守坛沿。

潮中第六更，障破又补，补又破，破在障，不在心。心稳，角稳；角稳，掌炉再炼，成丸两粒，自服一粒，掷邻角一粒。更鼓沉，按木牌，潮中过了，手在，铁无言在，经页在，就不滚。

第七更，兽影稀，毒更浓——浓像最后一扑。扑时，咬诀，雾浓，清半丈。半丈，够退而不逃；不逃，106才续。更鼓沉，按木牌，角还在，掌在，路在，就不滚。

在，就不滚。

更鼓沉，按木牌，潮中过了，续在远，105只守更尽。角还在，掌在，铁无言在，就不滚。

更鼓沉，按木牌，角还在，铁无言在，经页在，坛温在怀。在，就不滚。

潮中过了，下一程续。续，手在，就不滚。
