## 二、获得剧情（ch001～003 定稿）



### 触发三因（缺一不可）



1. **尘念之缘**：百年前荒野，莫长春葬一濒死散修。此人乃馈元天尊一缕**尘念**入世，临终只留一句：「肯赠人者，天道记。」葬仪即与天道订下记号。  

2. **赠念先于求生**：**大乘巅峰**渡**第九重**雷劫，第九道雷落时神魂将散，他最后一念不是「活」，而是「若有一息，先把师尊的恩还了」——与塔训「赠出不计价」完全同频。  

3. **劫雷贯魂**：青岚护阵一角被执法堂提前松动（赵玄机妒线伏笔），雷劫直贯神魂，逼出**天道秤阵**；秤上「一生赠人不索」之重压过劫雷一息。



### 眷顾一刻



魂飞魄散临界点，混沌光裂，馈元天尊残念与天道规则共振一瞬，**鸿蒙万缘塔**化流光破劫云，直入莫长春**识海**，一座塔影顶天立地，镇住三魂七魄。塔鸣一声，雷劫余威退半寸——人未死透，修为跌至筑基，寿元九月。



### 首赠兑现（ch003）



还续命丹于师尊，心真不计价 → 塔鸣大作 → 塔内骤亮 → 青竹杖**先入塔**、再**一念落掌**认主。塔壁浮铭：**初缘铭定**。

## 四、塔中收纳（v16 · 万物皆可收存取）



馈元天尊铸塔，本为**万缘有处藏**；回赠机制与储物并生，**塔内不分层格，万物皆可收、皆可取**。



| 动作 | 规则 | 爽点写法 |

|------|------|----------|

| **回赠入塔** | 赠出真心礼 → 塔鸣 → 物即时凝于塔中 | 塔内一亮，回赠已至 |

| **意取** | 心念一动 → 物现于掌/案/庭心 | **袖空出杖**、**空掌落丹**、**一念出符** |

| **意还** | 心念一动 → 物没入塔，袖又空 | 战后收妖核、宴后收手帕、剑归塔 |

| **储物** | 战利品、信物、自购物皆可收 | 茶饼、简帖、抄本、培元丹 |

| **禁则** | 假赠、交易、比价之物**塔不收** | 塔壁裂纹反噬 |



**围观铁律**：旁观者只见「袖空却有物」，**不见识海、不面板**；弟子窃语：「袖里明明什么都没有？」



### 与赠礼链衔接



```

真心赠出 → 塔鸣 → 回赠即时入塔

         → 意取：当场取用 / 意还：先存后取

         → 自用或再赠（再赠仍走赠礼链）

```



**首赠范本**（ch003）：还续命丹→塔内亮→杖入塔→一念落掌。
