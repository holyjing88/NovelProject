# 第一百一十九章 十层稳

还沈枯芽半块灵石後第三日，韩泥炼气十层实。

十层稳，气在掌，稳像粪铲落栏——落得要实，实了，才不飘。飘了，管事记「邪」；邪了，120复测风就吹不散。

铁无言声低：「十层，别叫。叫了，像破外门境。」韩泥应声：「不叫。沿净。」沿净，守坛，守120复测。

坛腹温一线，贴掌，像应「稳」字。应完，他不叫。不叫，才守得住十层，守得住121再炼散。

杂役棚风硬，铁无言立杂役棚侧，声低：「十层稳境，别飘。飘了，席沿就洒。」韩泥应声：「不飘。席沿在，手在。」手在，就不滚；不滚，复测风才接得住。

实不要紧，不露——露了，赖福虽压顶尽，仍嗅得出生机；嗅不要紧，管事看手，不看腹。腹里气走周，周周实，像坛腹应「稳」——稳，是丑翁专精；专精在渣，专精在窑，专精在五年忍。

卯时散队，丙九队列。赖福眼细，细得像还想找由头，找不着，阴笑：「十层？末席十层，仍像灰线三尺。」

韩泥不辩。辩，费舌；舌要留来守窑温，留来再炼培元散。散为复测，复测为三年约——约上写滚，滚要手抖；不抖，才配站钱戾衡台前。台前，炼气十三，你才有资格不滚；十三在远，十在腹。

管事弟子念风声：「外门复测，两月后。杂役可报丹试，末席亦报，手稳者上。」

丙九嗡一声。嗡里，有人看韩泥——看，也像看笑话。笑话不要紧，手稳；稳了，笑话变魁，变在远，变在130。130夺魁，还刘婆新衣养老钱；还了，叶汤还在省亲前。

韩泥摸玉牌，牌凉。凉是钱戾衡，凉是「炼气十三，你才有资格不滚」。十三在远，十在腹；腹里气走周，周周实，像坛腹应「稳」。稳了，才报名；报了，风声才真。

午后废窑，他再炼培元散。渣掺灵砂，砂细，细得像116捡漏那粒。粒不要紧，效正；正了，藏怀，不示。示了，赖福克，白织月嘲「运气」。运气不要紧，试台见正；见了，正字自现。

铁片嵌口，焚火残诀走息，息稳，窑温稳。稳了，散成，成三粒，粒丑，泥纹浅——浅不要紧，嗅之正。正，记一笔，备复测。复测过，炼气十三破，魁名落，还刘婆序才落地。

白织月过，袖风冷：「丑杂役也炼散？辩，费舌；舌要留来数粒，留来排序还恩——芽儿还了，刘婆在复测后，叶汤在省亲前。序不乱，粒才正。粒正，魁才配，还刘婆才配。

「炼了也是末。」白织月笑。

「末，看复测榜。」韩泥声平。

白织月唇动，没再尖。尖不过「报」字——报了，手稳，就上了比试台；上了台，钱戾衡眼风再来，再来也不怕——怕，手抖；不抖，玉牌才不虚。

刘婆送姜汤至窑道，姜薄，热：「十层稳了？」

「稳。」韩泥说。

「稳就别飘。」刘婆哼，「飘了，粥洒。洒了，恩断。」

韩泥喝尽。谢轻，恩重。恩在姜里，也在十层里——层稳，饭稳，饭稳，恩不断。恩不断，还序不乱；序不乱，丹才正。

韩泥应声。邪了，香飘；飘了，滚字就真。不滚，记仇苗——苗对钱戾衡，苗对赖福，苗对末席公示。公示过了，压顶尽了，徒恩还了；还了，备战才开。

邻床少年悄问：「你真报丹试？」

「报。」韩泥说，「不报，三年约白接。」

「钱戾衡……」

「记着。」韩泥摸玉牌，「记着，刃才磨亮。」

少年闭嘴。闭嘴，韩泥望山门——山门远，复测帖在远，大比帖在更远。远不要紧，先十层实，先粒正，先刘婆粥多半勺。多半勺，够顶一口饥，够顶备战两月。

丑时，练鼻下静。粟壳香淡，混一丝培元药香——香正，正不要紧；要紧是静一字不落。落齐，十层记进心里，不进舌。记进心里，才实；实了，才配120复测风声起。

坛腹微温，应「稳」字。稳，是丑翁专精；专精在渣，专精在窑，专精在五年忍。忍了，才还了沈枯芽半块灵石；还了，才配备战。备战，两月后验手；验手，为130魁，为还刘婆。

外头柱上条角贴新纸，纸边字淡：「外门复测，丹试报名。」

韩泥读条，独眼平，不近廊下笑。笑在耳，不在心——心在坛，坛醒了，备战才真。真备战，两月后上台；上台，钱戾衡眼风再来——来，不抖。

他低声道：「十层稳。复测，报。」

报字落，像钩——钩向120风声，钩向钱戾衡影，钩向三年约反面。反面烫，正面手稳；手稳，就不滚。不滚，炼气十一十二在121至123破，破在试台前，破在还刘婆前。

赖福脚步远，远得像压顶成了往事。往事不要紧，记着；记着，刃亮，丹正，路不断。路不断，大比帖在远，还刘婆在130，叶汤在省亲前——前头那碗烫，还得到手，才配称一部圆满。

十层实，报名册押字。字丑，人稳。稳了，赖福阴笑白笑。白笑不要紧，手稳夺魁预备。预备两月，炼粒，稳境，喝刘婆粥。粥稀，热，热顶饥，饥顶命，命顶还路。还路长，先复测，先魁，先还刘婆，先大比帖，先钱戾衡台前不滚。不滚，记着68眼风，70踹台，123廊下影。

十层实后，韩泥报名册押字。字丑，丑得像人；人稳，字稳。稳了，赖福阴笑：「报名也白报！」管事冷声：「再聒，逐出坡。」赖福噤，噤，韩泥记——记末席风声，记报名这一笔，进备战，不进舌。

午后废窑，再炼培元散。渣掺灵砂，砂细，窑温稳，散成三粒，粒丑，泥纹浅，嗅之正。正，藏怀，不示。示了，白织月说运气，赖福说盗材。

白织月过，袖风冷：「十层也炼散？」韩泥不辩：「炼备试台。」袖风过，过不要紧，腹里十层实，粒在怀沉。沉像债序——芽儿还了，刘婆待还，叶汤在省亲前。

铁无言掷簿：「报名册有印。印在，手别飘。飘了，粒洒；洒了，丹不正；不正，刘婆恩还迟。

刘婆送粥，仍多半勺：「十层稳了？」「稳就别飘。飘了，老婆子粥洒。」韩泥喝尽，记——记第七笔恩，记130新衣养老钱。钱数过三遍，不多不少，够补衣婆活到老。

夜，练鼻下静。香落齐，十层记进心里。记进心里，不进舌。舌响，赖福记「报名晦气」；晦气不要紧，手稳夺魁预备。

坛腹微温，应「稳」——稳十层，稳粒，稳手。稳了，接120复测风，接121再炼，接钱戾衡影。影硬，手更硬；手硬，玉牌不虚，三年约不白接。」报字落，像钩——钩向备战两月，钩向炼气十一十二，钩向129帖至，钩向130还刘婆。还刘婆，芽儿清了，序不乱，步不飘。

外头风声起，起得像试台在等。等在远，备战在近；近，炼粒，稳十层，喝刘婆粥。粥稀，热，热得像恩又温半分。温半分，够熬到夺魁，够熬到新衣递刘婆手里。

他摸玉牌，牌凉，低声道：「三年约，记着。钱戾衡，记着。赖福克扣，记着。」记着，刃磨亮。亮在还刘婆，亮在夺魁，亮在大比帖至山门。

刘婆粥仍多半勺，姜仍多一片。片暖，恩不断。恩不断，手才稳；手稳，粒才正；粒正，试台才配说话。说话了，白织月袖风尖不过「正」，钱戾衡影硬不过手。

手稳，粒正，试台见。」见字落，坛温一线，像应。应了，路又明一寸——明向还恩，明向不滚，明向刘婆新衣养老钱。

更鼓沉，沉里，韩泥按木牌，牌凉贴胸。119章手在，名在册，坛沿净。舌不留，鼻下静。恩辱分列，不混。活长了，才还得了叶汤，才接得住门缝灯。稳不是飘，是丑翁专精；专精在渣，专精在窑，专精在五年忍。忍了赖福压顶，还了沈枯芽半块灵石，备战才真；真备战，夺魁，还刘婆，大比帖在怀。

韩泥不辩。辩，费舌；舌要留来守坛，留来按罐。手稳，粒正，就不滚；不滚，才配还刘婆，才配接大比帖，才配叶汤省亲前还尽。
