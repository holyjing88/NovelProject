# 第六十八章 公开测灵

三日后卯时，坡下霜白。

柱上条已干，字更硬：「凡骨杂役，公开测灵。末席公示，滚者除名。」

韩泥读完，独眼平。备检是过筛，今日是公示——公示给内门看，公示给全宗看，公示谁滚谁留。滚字在条上，他掌心却先蓄一分力：滚也要手抖才配滚。

丙九列队，队末是他。赖福立队首，眼细得像秤，秤人，秤灰线，秤谁多喘一口气。管事喝：「按序！手抖的，先滚！」

测灵石抬到坡下中央，石青，青得钝，像一口不说话的秤。内门弟子立廊下，袖风冷，有人笑：「凡骨也配公开测？」

「配不配，看手。」韩泥声平，「手抖才滚。」

第一个少年上场，绿线半寸，光跳一下又灭。管事记：「浅。下。」第二个按石，绿线一寸半，光稳，手也稳：「过。」第三个绿线两寸半，肩沉半分，沉不要紧，光正了就收。韩泥不盯色，盯手——手怎么按，腕怎么稳。

第四个按石，绿线两寸，抖了半寸。管事冷声：「抖！滚！」少年哭腔被架走。丙九静一息，有人拿眼看韩泥，像怕末席会传染晦气。

赖福阴笑：「丑杂役最后。最后，专看滚。」

「按序。」管事喝，「韩泥，末。」

他上前。掌茧裂处疼，疼不露。掌按石，石凉进骨。光起，灰线短——短不过指节，却钉在众目下，像把「末」字当众烙进坡面。

有人嗤：「灰线三尺，也配留？」

嗤声薄，也扎人。白织月立廊角，香冷：「凡骨末等，丹路断。断了的站石前，像笑话。」

韩泥不辩。管事瞥他掌：「手抖没？」

「没抖。」

管事再看灰线，灰线三尺，真在公示。簿上落笔，声沉：「韩泥，凡骨末等。手稳，石前不抖——仍末排，公示。」

执事念榜：「公示末席，韩泥。灵根凡骨末等，灰线三尺。手稳者，冬后仍杂役；手抖者，除名。」念完，有人松气，有人看笑话。

仍末排，公示。比备检更辱——辱在众目，辱在内门眼，辱在赖福那声笑。

赖福笑出声：「公示也末等！丙九的脸，让你丢尽了！」

「脸在饭里。」同批有人闷哼半句，又缩回去。

韩泥收掌，不接茬。廊下有人踏前一步，靴底沉，金线边亮——衣鲜，眉锐，瞥一眼灰线，唇角一挑：「钱戾衡。」

名过，风冷。冷不是冬风，是眼风，眼风踩在灰线上。

钱戾衡声不高，众耳却听见：「灰线三尺，也配站这坡？配站，就配滚。滚了，省得污内门眼。」

内门哄笑起来。笑薄，真。瞬短，辱长；辱长，才扎成仇苗。

韩泥指节白一息，又平。他只答：「管事看手，不看嘴。」

钱戾衡笑一声：「嘴硬。正测后，看你还硬不硬。」

笑里藏着踹字——踹不在今日落地，却已经落进耳朵。今日他只记名，不近，不拜。拜邪断路；不拜，各守各线。

铁无言抱臂立侧，声低：「灵根末，手稳，渣场仍收你。别漏。」

「不漏。」韩泥应。

秦霜远廊经过，木鱼不敲：「心别邪。邪了，鼻下香飘。」

他心里按住七笔：泥岗那六笔沉着，刘婆那一笔烫着——烫在肘、在粥、在今日这口辱。辱在众目，手不能抖；抖了，滚字成真。

管事收石：「正测毕。公示末席：韩泥。滚者已滚。仍末排者，冬后仍杂役，手抖再滚。」

榜上新添一行，行丑，字真：「韩泥·仍末·手稳。」手稳是小胜。胜在活，不在嘴。

邻床悄问：「你真不滚？」

「滚看手。」韩泥说，「手在，就不滚。」

少年又挤一句：「钱师兄那眼，你接住了？」

「接住了。」韩泥声低，「接住才记得清。」

赖福晃木牌：「公示了，更滚得快。」

韩泥摸怀中牌。牌凉，膝热——热在坛，热在路，热在钱戾衡那一眼。钱戾衡走远，还留一句：「丑杂役，记住今日。」

「记着。」韩泥低声。

仇苗又深半寸。深半寸不拔，先活。活长了，才还得了叶丫头那碗烫，才接得住将来那一踹。

午后兽栏仍腥。他铲下稳，稳得像坡下按石那一掌。管事记一笔：「丙九韩泥，正测后仍清细。」小兽咳轻，他多换一把干草——干草淡一寸，像日子也淡一寸。淡不要紧，只要名还在册。

刘婆端粥，仍多半勺：「公示了？」

「公示了。」韩泥说，「仍末排。」

「末排怕什么。」刘婆哼，「手稳，饭稳。饭稳，恩不断。」

他点头。谢轻，恩重。散队后又过坡下石阶，阶上有人吐唾，唾落阶边，不落靴——落靴像沾晦气。他不辩，只按木牌：牌在，名在，路在。

老耿挑水路过，声哑：「公示不要紧。末，不滚；滚，看手。」

「手在。」韩泥答。

更鼓将起，柱上条角又白一线，像仇苗出土，根扎坡下。他按席角，席严，坛不露。坛腹温一线贴掌，像爹的手又搭上来。爹说过：「坛别扔。」扔了就无根；无根，辱就白挨。

廊下仍有人练剑，笑远：「灰线三尺，也配留？」

他不答。答费舌，舌要留来记仇苗。记不在册，在心——心在坛，坛醒了，刃才磨得亮。

药渣场角，铁无言又丢半句：「鼻下别跟他们吵。吵香，手飘。」

「不吵。」韩泥分拣灰，「吵的人没粮。」

赖福廊外停半息：「丑杂役，我盯着你。」

「在。」韩泥说。

在，就不滚。不滚，仇苗才扎得实。实了，才接得住踹台前夜，接得住玉牌风声，接得住三年约还没落地的那一脚。

他廊下练半刻鼻下静，粟壳香淡，一字不落。落齐了，钱戾衡那句「记住今日」才记进心，不进嘴。

棚里少年又问：「仇记在心里，会不会手反抖？」

「记清了不抖。」韩泥说，「记糊了才抖。」

他独眼平，掌不散，低声道：「钱戾衡，记着。记着，才配等踹台那夜。」

散队后药渣场空了一角。他蹲着分灰，分到一半，管事远远道：「公示了还清细？也好。别让人说末席只会挨嘲。」

「清细是活。」韩泥不抬头，「嘲是嘴。嘴管不了粪。」

管事顿一息，没骂，走了。走了像默许：默许他再占丙九这口窄饭。饭窄，仇苗却不窄——钱戾衡那眼风，已经在心里生了根。

夜里棚门响，赖福又来踢柱：「公示出丑了，还装稳？」

「装稳？」韩泥应声，「稳是掌上的。掌上抖了，你才乐。」

赖福噎回一句脏骂，到底没冲进棚。棚里少年捂嘴笑半息，又怕，笑熄了。熄了也好——笑太响，仇记不深；记不深，踹台前夜就软。

帖在怀，怀沉。沉的是序，不是惧。惧可以有，手不能先散——手散了，仇苗白扎，叶汤也凉。踹台在前头，风已经硬了；硬风里，名还在榜末那一行「手稳」两字上。两字丑，够他撑到下一夜。下一夜若真见玉牌，牌凉，仇更烫。

他回到席角，把公示时掌上凉意再摸一遍。凉意退去，茧还在。茧在，灰线就钉不死命——钉得死的是抖，钉不死的是人。他忽然问坛：「爹，仍末也能活？」坛只温一线。一线够了。够了，就把钱戾衡那句「记住今日」再往下按半寸。按实了，踹台前夜才不像慌梦。慌梦醒了还是手，手里有茧，就比笑话硬。

刘婆卯时又来送粥，粥仍稀，她声却硬：「公示了也喝。喝完铲粪。铲完，人还在丙九，他们的嘴才能饿着。」

「在。」韩泥应。

「钱那个名字，记牢。」刘婆压低嗓，「记牢别半夜喊。喊了，棚里人吓，赖福听了又折腾。」

「不喊。」韩泥说，「记在骨里，不记在嘴上。嘴上一喊，仇就轻了。」

粥喝尽，他把空碗扣好。扣响一声，像给今日公示收口。口收了，路还在——路通向踹台前夜。夜未到，仇苗先扎稳；扎稳了，明日石前那口辱才不白受。
