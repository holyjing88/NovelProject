# 第一百二十五章 窑炉稳

备战最后七日，韩泥只守窑温。

窑门柴挡，挡严，严得像62席盖。席严坛不露，窑严粒才正；正了，试台才配夺魁，夺魁才配还刘婆。

焚火残诀走息，息稳，铁片嵌口温一线。一线升，升得像110首炼余温未散——散不要紧，稳才要紧。稳了，第九粒成，成最后一粒备战粒；粒丑，泥纹浅，嗅之正。

赖福已无新招，只廊外晃木牌：「七日后试台，看你滚。」韩泥不辩。辩，费舌；舌要留来数息，留来守温。

铁无言送炭渣，渣里一点灵火屑，屑细，够稳一夜窑温：「七日，别出门。」「不出。」韩泥说，「出门，手飘。」

刘婆柴门外咳：「窑里闷，别闷坏。」「闷不坏。」韩泥说，「闷，炉才稳。」

白织月过药山道，袖风远：「七日后，看丑骨出丑。」韩泥不辩。出丑在脸，不出在手；手稳，丑也能「正」。

第一夜，窑温如秤，秤平，粒不杂。第二夜，邻床少年悄问：「七日不出门，饿不饿？」「不饿。」韩泥说，「饿，手飘。」

第三夜，窑温忽跳，跳像赖福余党使坏——使坏不要紧，铁片稳口，残诀压温，压回一线。

第四夜，刘婆送半块饼，饼糙，糙得像泥岗麦麸：「试台前，别空肚。」韩泥收下，谢轻，恩重。

第五夜，气走圆满，十二层实到极致，极致像弓拉满——拉满不要紧，箭在130射。

第六夜，他再验九粒，粒粒正，正得像五年淘的。淘在渣，淘在克扣，淘在忍。

第七夜，他封窑，封得像封坛。封坛，坛温一线不退；不退，心稳，步稳，债序稳。

丑时末，练鼻下静。粟壳香混药香，药香正，正不要紧；要紧是静一字不落。

开窑晨，天色白，白得像刘婆补衣线。线旧，线暖，线记着，等130还。

铁无言来窑外，声低：「七日毕，手别飘。」「不飘。」韩泥应声。

刘婆柴门外咳：「窑稳了？」「稳了。」「稳了，试台别抖。」韩泥只答：「不抖。」

他望坛，独眼平：「稳。记着。」七日封窑毕，九粒仍正。正，试台在手。

散队后，他数息，数粒，数债序。债序在，步不飘；步不飘，才接得住128验正，129帖至，130魁与还刘婆。

更鼓沉，沉里，他望坛，独眼平：「稳。记着。」记着，是丑翁本事；本事在忍七年赖福，在还沈枯芽半块灵石，在守七日窑温。
