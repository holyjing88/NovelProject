# 第八章 青狼袭村

东脉李家村的妖讯，是在顾小满喝完热粥的翌日清晨到的。

报讯弟子满身泥，喘着撞进山门：「青狼妖围村，伤三人，请长老援！」

霍镇山已在石道拦人，盾未亮，嗓门先亮：「莫长老若去，镇妖司记恩；若不去，老夫自己去送死！」

莫长春拄杖出院：「霍老头，别喊。去。」

村外炊烟断，犬吠绝，只剩妖风腥。

青狼妖伏在塌了一半的院墙上，眼绿，爪利，像把夜撕剩的布挂在牙上。村民缩在灶下，孩子哭，被母亲死死捂住。

顾小满扶老妇转移，脚下一滑，险些栽进妖口——她本不该来，却跟着杂役队送药，像半块灵石还在掌心里推她往前。

妖扑。

莫长春杖至。

不快，像老人散步；却准，杖尖点地，竹影暴起，如索如鞭，缠妖腰，一提一摔，青狼妖砸在晒谷场，尘起，骨断。

村民呆住，继而跪谢，谢得乱，像一场迟来的雨。

霍镇山抱拳，声洪：「莫长老，镇妖司记一笔！」

莫长春不居功，只从袖中取符一张，黄纸朱纹，灵品除魔符，递顾小满：「下次别一个人挡。」

顾小满指尖抖，接符，像接第二块半石：「长老……我……」

「记着就行。」莫长春道，「恩不用嘴还，用命还的时候，别还错了人。」

回山路上，弟子群情振奋，外门终于不再只说「九月将死」。

有人说莫长老杖里有灵，有人说青岚门还有底，赵玄机在石道旁冷听，指节叩铁牌，一声一声，像把妒意敲实。

林挽月望莫长春背影，忽然觉得「老」字不一定等于「弱」。

她没敢说出口，怕赵师兄笑她动摇。

苏念慈在丹堂廊下听讯，炉火正旺，她低声对弟子：「除魔符给了杂役……师兄总把重的给别人。」

弟子不敢接话。苏念慈望东脉方向，妒里掺一点敬，敬得像药渣里的甘。

回山路上，霍镇山声洪，像要把救村功劳喊实：「莫长老除魔，镇妖司记恩！」

弟子群情振奋，外门终于不再只说「九月将死」。有人说莫长老杖里有灵，有人说青岚门还有底。

赵玄机在石道旁冷听，指节叩铁牌，一声一声，像把妒意敲实。

孙福在杂役堂阴脸，对弟子道：「半块石算什么，救村是长老本分！」

弟子不敢大声应，却有人小声：「孙管事，你克扣月例时，莫长老怎么算？」

孙福脸色一变，拂袖走，走得像怕下一阵，账算到他头上。

夜，簿上浮字，淡金：

**除魔一念，善缘入账。**

莫长春合簿，望东脉方向，村灯重亮，像有人把掰开的灵石又拼回一半。

他念：「小满，符收好。下次别挡前面，挡侧面也行。」

风过，无人答，像话进了土里，等着长。

他望杖，笑：「霍老头记了一笔，赵执事妒了一笔。好，人多，账才厚。」

杖中竹纹一游，像应。

大因果还在远处，像妖潮，像赤焰——小善缘，却先入了簿。

回山路上，林挽月望莫长春背影，忽然觉得「老」字不一定等于「弱」。她没敢说出口，怕赵师兄笑她动摇，却把除魔符的事记进心里——记得像半石，记得像恩。

村民扶老弱回屋，屋灯重亮，亮得像有人把掰开的灵石又拼回一半。霍镇山声洪，像要把救村功劳喊实。孙福在杂役堂阴脸，对弟子道：「半块石算什么，救村是长老本分！」弟子不敢大声应，却有人小声：「孙管事，你克扣月例时，莫长老怎么算？」孙福脸色一变，拂袖走。

莫长春立廊柱旁，望东脉方向，村灯重亮，轻声：「小满，符收好。下次别挡前面，挡侧面也行。」风过，无人答，像话进了土里，等着长。他望杖，笑：「霍老头记了一笔，赵执事妒了一笔。好，人多，账才厚。」杖中竹纹一游，像应。

围观弟子窃语：「莫长老杖扫青狼，又赠除魔符给杂役——老不正经？」另一人答：「不正经的是孙福克扣，不是长老赠符。」窃语轻，轻不过村灯重亮那一寸。莫长春不辩，只望东脉，像望一条还将起妖讯的河。

苏念慈在丹堂廊下听讯，炉火正旺，她低声对弟子：「除魔符给了杂役……师兄总把重的给别人。」弟子不敢接话。她望东脉方向，妒里掺一点敬，敬得像药渣里的甘。

莫长春立廊柱旁，轻声：「小满，符收好。恩，慢慢还。」大因果还在远处，像妖潮，像赤焰——小善缘，却先入了簿。
