## 二、每章开工模板（复制即用）

### 2.1 章头状态行（八要素 · 强制）

```
【第 X 章】标题
陈寻 · 境界 · 恩施余 X · 因果阳 X/阴 X · 道侣【正/侧/缘】 · 绑缘 X/7 · 馈缘【对象·档】 · 符录【品·页】
```

**示例（第 188 章）**：
```
【第 188 章】雨夜拒契
陈寻 · 炼气十二层 · 恩施余 6 · 因果阳 18/阴 5 · 道侣【正·沈意】 · 绑缘 0/7 · 馈缘【沈·半】 · 符录【玄·2页】
```

### 2.2 章内四块（建议结构）

| 块 | 字数约 | 内容 |
|----|--------|------|
| 起 | 400～600 | 承接上章钩；携带伤势/时间 |
| 事 | 1800～2400 | **一章一事**（一场戏/一次赠礼/一次对决） |
| 系统 | 200～400 | 翻册、划账、升维、破境节点（融入动作，不写面板） |
| 钩 | 300～500 | 末 500 字内必留悬念或下一锚点 |

高潮章（63/70/188/235/272/440/510/720 等）：**5000～7000 字**，仍一章一事。

### 2.3 章后自检（打勾再定稿）

```
[ ] 八要素与上章衔接
[ ] 本类章节遵守恩仇分立（报恩不杀 / 报仇不吻）
[ ] 无自刎/自尽/无意义赴死
[ ] 若有人陨→是否拉垫背或寿尽坐化（720 特例）
[ ] 272 前无绑缘/合击；420 前无契缘合击
[ ] 命名全用 15 本作名
[ ] 章末有钩
[ ] 赠礼章：距上一赠 ≤8 章（或 L 章豁免已补）
```

## 三、主线链 · 写正文时怎么穿插

```
低谷受恩(1～50) ──→ 无力亏欠(51～130) ──→ 择机还恩(130～720)
        │                                      │
        └────────── 馈缘赠出(95+) ──────────────┘
                              │
        赠礼 → 羁绊/亲密/忠诚/馈缘↑ → 倒贴 → 报恩 → 纳绶 → 鸡犬升天
```

| 阶段 | 正文写法 | 禁止 |
|------|----------|------|
| 赠礼 | 陈寻弱时先送（汤改灯油、半石、护心符） | 假赠、转手倒卖 |
| 四维↑ | 簿上划「正/半/满」，不写阿拉伯数字 | UI 面板 |
| 倒贴 | 写选择写动作（留灯、吻拒、护家） | 同章杀人、低俗直写 |
| 报恩 | 翻册、划账、安置、养老 | 同章温存 |
| 纳绶 | 献绶、血印、一章写完 | 272 前纳主绶 |
| 升天 | 272 得簿；580 4/7；1555 封册 | 无上限带人 |

**汇合章（扩写重点）**：272（恩施+我娶+得簿，全书唯一合章特例）、580（还铁恩+录入）。

## 六、活着优先 · 正文标准句

**440（必考样章）**：
- ❌ 裘横秋自知罪孽自刎谢罪。
- ✅ 裘横秋断臂仍要遁，反手一掌拍向岳含章天灵——要拉垫背。
- ✅ 岳含章不退，袖中地符炸开，将裘横秋拖入火网，同陨于廊下；裘先死。

**通用**：
- 败局 → 逃、诈降、拖、反咬
- 必死 → 拉仇人垫背（先确保仇死）
- 陈寻 → 符/阵/队友换命，不算计赴死
