# 第一百二十二章 织月嘲

白织月再嘲，选在丹试榜前。

织月嘲，声细，细里带刃：「丑杂役也再炼？炼了也末等。」韩泥不抬头：「再炼，不叫。沿净，不叫。」

她袖风过，带过一点冷香。冷香不要紧，韩泥鼻下仍静。静里，他记仇苗又深半寸——深在嘲，深在123戾衡影。

白织月唇线绷一息，走了。少一句嘲，小胜——胜在掌，不在嘴；嘴稳，试台认「正」。

辰时，外门小榜贴坡下，榜上有名，名末行「韩泥，丑骨，报培元散」。末行丑，丑得像他脸——脸丑不要紧，手稳；稳了，榜末也能上移，上移在试台，不在嘴。嘴硬，手飘；手飘，除名。

白织月立榜侧，袖风带香，香冷：「丑名也上榜？上榜，像污榜。」

韩泥不辩。辩，费舌；舌要留来护罐，护七粒培元散。护不要紧，不近榜前吵——吵，手飘；飘了，除名。除名，叶汤恩断，刘婆恩还迟，芽儿那半块石白还。

「污榜不要紧。」铁无言声沉，立韩泥侧，「榜看试台，不看脸。」

白织月瞥铁无言，瞥完仍尖：「铁师兄也护晦气？护了，丹道跌价。」

「跌价试台见。」铁无言说。

韩泥独眼平，只答：「试台见。」

三字，手稳。手稳，白织月袖风也尖不过「见」——见了，散正不正，钱戾衡眼风硬不硬，一并见。见了，刘婆敢穿新衣，芽儿敢收石，叶汤还在前头烫。

赖福挤人群，嗓尖：「丑杂役上榜，丙九晦气满坡！」

管事弟子喝：「聒噪！再聒，逐出坡！」

赖福噤，噤，韩泥记——记末席风声，记织月嘲，记赖福余毒。余毒不要紧，压顶尽了；尽了，备战才净。净了，舌留来按罐，留来守坛；坛在，手在，就不滚。

刘婆立人群末，蓝布包头，眼浊，手稳端空碗——碗空，像来占位。占位不要紧，韩泥望她一眼，一眼像还恩预告：预告在130，预告在复测后新衣养老钱。养老钱数过三遍，新衣棉实，袖口多缝一层，耐磨。

刘婆摇头，摇得像怕沾魁晦气；怕不要紧，恩不断。不断，粥仍多半勺，姜仍多一片。一片姜暖，手别僵；僵了，粒洒；洒了，丹不正。

散队后，药山道，白织月再拦，袖风冷：「你七粒散，怕是杂的。」

「杂不杂，鼻下知。」韩泥声平。

「鼻下？丑鼻也配嗅丹？鼻丑，嗅正；正，是五年渣场淘出来的。淘出来了，白织月嘲就尖不过一粒「正」——正，在试台，不在舌。舌响，香飘；飘了，滚字就真。

沈枯芽远站，怯笑浅。笑浅，像芽；芽在，徒恩还了沈枯芽半块灵石，还了更怕沾——沾不要紧，韩泥不招手，招手像讨。讨轻，恩重；重了，还刘婆才配，还叶汤才配。

他回窑，盘坐。气走周，周里十一层稳，稳得像七粒在怀沉。沉不要紧，炼气十二门槛开一线——开一线，不露；露了，白织月说「偷境」。偷境不是，是忍；忍在活，活在还恩前头。

坛腹温一线，应「见」——试台见，仇见，钱戾衡见。见不要紧，手不抖；不抖，玉牌才不虚，三年约才不白接。不白接，钱戾衡「让你滚，你才有资格不滚」才接得住。

秦霜远木鱼远敲：「嘲声大，心别邪。」

韩泥不答，答在心里——心里排序：芽儿还了，刘婆待还，叶汤头恩，钱戾衡仇。仇恩分列，不混；混了，粒不正。不正，刘婆恩还迟，叶汤还凉。

丑时，练鼻下静。粟壳香淡，香落齐，织月嘲这一日记进心里。记进心里，刃磨亮半分——亮在试台，亮在还刘婆，不在袖风。袖风冷，忍过去；过去，试台见正。

邻床少年悄问：「白织月为何总盯你？」

「末席丑，好嘲。」韩泥说，「嘲不要紧，手稳。」

少年闭嘴。闭嘴，韩泥摸玉牌：「钱戾衡，也记着。记着，试台见。」

试台见，像钩——钩向123戾衡影，钩向十二层，钩向复测备战末段。末段，帖将至，魁将至，还刘婆将至。将至不要紧，先正粒，先稳手，先债序清。

外头榜风掀角，角白一线，白得像大比影。影在远，嘲在近；近，忍过去了；过去，粒正手稳，就接得住帖。帖至，大比影至，还刘婆在130落地；落地，叶汤还在省亲前。

赖福廊外吐唾，唾落阶边，不落他靴。不落靴，像内门仍不肯认——认不要紧，试台认「正」。正了，刘婆敢穿新衣，芽儿敢收石，叶汤还在前头烫。烫着，省亲路才明，大比帖後才接得住。

他低声道：「织月嘲，记着。试台见。」

见字落，坛温一线，像应。应了，备战又实一分——实一分，够走到十二层，够走到124铁无言挡话，够走到130还刘婆新衣养老钱。养老钱不多不少，新衣袖口多缝一层，耐磨，像126刘婆顶针那夜。

榜贴第三日，韩泥仍吃饭最后。最后不要紧，刘婆等——等像补衣婆等恩还。还不要紧，先试台；试台验正，末项夺魁在130。130未到，先稳罐，先稳手，先稳十一层。稳了，白织月嘲像风；风过，不退粒正。粒正，试台见；见了，嘲哑，影虚，玉牌不虚。不虚，就配123影，配124挡话，配129帖至。帖至，大比影至，仇路明，恩路也明。明路，手不抖；不抖，就还刘婆，就还叶汤烫手。

丹试榜贴坡下，末行「韩泥，丑骨，报培元散」。末行丑，丑得像脸；脸丑不要紧，手稳；稳了，榜末也能上移——上移在试台，不在嘴。

白织月立榜侧，袖风冷：「丑名也上榜？」铁无言声沉：「榜看试台，不看脸。」韩泥只答：「试台见。」三字，手稳；手稳，袖风尖不过「见」。」管事喝：「聒噪！」赖福噤，噤，韩泥记——记织月嘲，记榜末行，进备战，不进舌。

刘婆立人群末，蓝布包头，眼浊，手抖端空碗——碗空，像来占位。占位不要紧，韩泥望她一眼，一眼像130还恩预告：预告在试台后，新衣养老钱在怀。

散队后，药山道，白织月再拦：「你七粒散，怕是杂的。」韩泥声平：「杂不杂，鼻下知。」鼻丑，嗅正；正，五年渣场淘出来的。淘出来了，嘲就尖不过一粒「正」。笑浅，像芽；芽在，徒恩还了。还了，更怕沾魁；沾不要紧，韩泥不招手。讨轻，恩重。

夜，十一层稳，稳得像七粒在怀沉。沉不要紧，十二层门槛开一线——开一线，不露；露了，白织月说偷境。见不要紧，手不抖；不抖，玉牌才不虚。香落齐，织月嘲这一日记进心里。」见字落，接123戾衡影，接124挡话，接125窑稳。稳了，粒正，手稳，备战尽。帖至，魁至，还刘婆至。

榜末行丑，韩泥不辩。辩，费舌；舌要留来守罐。罐在怀，怀沉，沉像债序。序在，步不飘。飘了，试台洒。洒了，除名。除名，恩断。恩断，叶汤凉。凉不得，汤是头的。头的先记着，刘婆130还，芽儿已还，钱戾衡仇在玉牌记着。

榜末行在，手在稳。稳，试台见正，见还，见刘婆。见刘婆，在130，不在舌。

榜风止一息，韩泥捧罐，罐稳。稳，像捧热汤——汤是头的，粒是今日的，刘婆新衣是130的。130未到，先稳十一层，先挡嘲，先验正。验正了，末项夺魁；夺魁了，炼气十三破；破了，还刘婆，帖至，大比在路。路长，仇在玉牌，恩在序，手在稳。稳，就不滚；不滚，钱戾衡台前，才接得住三年约反面。

他摸玉牌，牌凉，低声道：「三年约，记着。手稳，粒正，试台见。应了，路又明一寸——明向还恩，明向不滚，明向刘婆新衣养老钱。

更鼓沉，沉里，韩泥按木牌，牌凉贴胸。122章手在，名在册，坛沿净。舌不留，鼻下静。恩辱分列，不混。活长了，才还得了叶汤，才接得住门缝灯。

候末项时，风硬，吹得灰线名像刀——刀在皮上划，划不出血，划得出忍。
