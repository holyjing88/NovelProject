## 2025–2026 流行趋势

| 趋势 | 本作用法 |
|------|----------|
| 凡人流回归 | 四灵根、无系统、靠算计 |
| 情感复合 | 迟恋+妒火+背叛+担当 |
| 章末强钩 | 每章末 500 字内爆点 |
| 恩仇分立 | 报恩章与报仇章不同章 |
| 老头/少年双镜像 | 岳含章迟暮 vs 陈寻少年（非同一主角） |
| 短剧友好 | 每 20 章一个单元爽点 |
