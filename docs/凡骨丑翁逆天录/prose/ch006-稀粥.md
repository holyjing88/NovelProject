# 第六章 稀粥

谣起第三日。

门墩多了一只粗碗。不是叶家那只——沿口缺在不同处，缺得像月牙，缺在碗东。碗里稀粥，稀得照见底，米粒几粒，浮着，像可怜人的良心：少，却有。粥面凝一层薄皮，揭开，底下更稀，稀得像娃的眼泪。

碗底蹭着门墩石，蹭出一道湿印。印浅，像有人放下时手在抖。

韩泥开门，门外无人。泥地小脚印浅、乱，像娃的，鞋尖还沾着沈家院墙根的青苔。村里喊「芽儿」——沈枯芽。怯，瘦，爹早死，娘病。送粥像做贼。在泥岗村，对晦气翁好，就是贼。

他端碗进屋，捧着，先吹一口气。粥气淡，淡得几乎没有米香，只有灶灰味。喝。淡，腥，米少，胃却暖一线。暖细，比叶家那碗姜汤弱，更难得——姜是胆里暖，粥是娃里暖。

喝完，舌尖舔碗沿，舔到一粒米，含住，嚼了，嚼得慢，像嚼恩。粥腥里有灶烟味，烟味淡，淡得像沈家灶小、柴少、火怯。碗洗尽，搁回门墩。洗时水凉，指节红，红得像记。

他站在门边又看一眼脚印。脚印进门方向不正，像娃放下碗就跑，跑时还回头。回头会摔——果然泥里多一处膝盖印，浅，灰扑扑。

天未亮透，他开门再看门墩。脚印边多一粒米，米粘泥，像娃慌时掉的。韩泥拾起，含进嘴里：「芽儿，翁看见了。」

晌午，村道炸了。

马春花嗓尖，尖得像剪子：「沈家芽儿学谁不好！往晦气翁门缝塞粥，晦气沾上了，你娘药钱谁出！」

沈枯芽哭，闷在喉里：「我……我没有……」

「没有？碗人赃并获！」周嫂帮腔，棒槌还攥着，「赖管事说了，沈家再牵扯，连坐全族！」

人群呼啦围上来，像看杀猪。有人吮牙花：「绝户翁连娃的粥也抢？」有人小声：「抢什么抢，人家送的……」话没说完，被旁人肘了肘。

韩泥在屋里听清了，声不高，却够穿出门缝：「碗在门墩，粥在我肚。要罚，罚我。」

「你倒敢认！」马春花更尖，「认了就坐实，沈家跑不了！」

「跑不了，我认。」韩泥说，「粥在我肚，不在芽儿手里。」

马春花一把撕开沈枯芽衣角，布声刺耳，像撕一面赃旗：「咳？咳就是晦气！你爹早死，你娘病，再沾这翁，沈家绝不绝？」

芽儿哭得肩抖，衣角挂丝，丝在风里颤。赖扒皮的爪牙路过，腕上缠着沉丹宗的粗绳，专给管事跑腿。芽儿一碗粥，惹了眼。哭声远，像被拖回院。

韩泥没立刻出门——出门，坐实「牵扯」；不出门，粥已入腹，情已入心。他握着门闩，指节发白，白完又松开。松开才算忍。

屋里静，静得听见屋顶存水一滴一滴磕在坛沿上。他低声道：「记着。粥稀，情不稀。」

午后，那爪牙路过泥屋，一脚踹在门墩上，墩响，木屑飞：「晦气翁！再收沈家东西，罚银加倍！」

韩泥开门，独眼平：「我收的是粥，不是沈家。粥已入腹，你要罚，罚我。」

来人愣半息，上下打量，像打量一块啃不动的骨头：「你倒硬。」

「硬不硬，粥已硬进肚。」韩泥说，「回去告诉赖管事，要连坐，坐我。莫坐娃。」

靴跟又碾门墩，碾出一声闷响：「罚银呢？别装死！」

「有。」韩泥拍腰间空袋，「命在，银慢慢凑。你要现拿，拿不走。」

来人啐一口，扬长而去。门墩裂了一道细缝，缝里还嵌着半粒米。韩泥蹲下，用指甲把米抠出，埋进泥地角落：「米归地，情归心。」

傍晚，沈家院墙后头响起轻扣。韩泥开条门缝，芽儿缩着肩，手里捧半块冷饼，饼边有指甲掐痕，掐得深，像用力忍哭。

「韩爷爷……娘让我……让我别再送……」芽儿嗓子哑，「可我……粥还在你肚里。」

「在。」韩泥接过半饼，掰开，大的递回去，「饼给你。粥我喝了，情我记着。往后别偷偷来——被人撕一次，够了。」

芽儿咬着唇：「马春花说，罚银要算到沈家账上。」

「账算到我。」韩泥说，「谁的粥，谁的罚。你娘病着呢，别拿她的药钱顶。」

「可你……你也穷。」

「穷不怕，怕忘。」韩泥把小半块饼塞回芽儿袖里，「吃。手抖，编筐就编不圆。编不圆，银更没有。」

芽儿抹了一把脸，点头，像贼一样悄然退走。门缝外只剩泥印和那道湿痕。韩泥把门合上，合到一半又停，对外头道：「芽儿，听见锣别跑太近。跑近，又撕你。」

墙外极轻地应了一声：「……嗯。」

夜里，村道上又起碎嘴。马春花把铜盆敲得山响：「芽儿那碗粥，就是认主！晦气认主，沈家完了！」

有人小声：「粥……也稀啊。」

「稀也是粥！」马春花骂，「稀也沾手！沾了手，赖管事眼睛不瞎！」

周嫂哼着帮腔，棒槌墩地。墩了两下，看见里正影子远过，忙把声音收半寸。收归收，刺还在。

韩泥坐在漏屋里编筐，藤条绕膝。筐口还没收，他忽然停住，把半块饼搁在坛沿上——不是供，是记。记着：谁撕过芽儿的衣角，谁踹过门墩，谁把稀粥说成罪过。

坛沿凉。凉也不妨，心里有火就行。火不大，够熬过这一夜。

第二日一早，他照旧上集卖筐。过沈家墙根，听见芽儿娘咳嗽，听见芽儿压低声：「娘，别咳那么响……马婶又来了。」

韩泥没停步。停了像招。他把嗓子沉下去，对墙根说一句：「粥我认了。罚也认。」

墙里一瞬静了，随后芽儿哽了一下：「韩爷爷……」

「少喊。」韩泥道，「喊多了，又给她撕。衣角撕一次疼，心撕一次更深。」

集上，有人盯着他空碗走的影子嘀咕。赵家那边的嘴还没张全，里正远远看一眼，只咳一声，没当场接茬。咳完，韩泥明白：今日能忍一日，是一日。

卖方汉子摸他的筐：「藤条干了点。给你十一文？」

「十二。」韩泥不让。

「你今儿硬。」

「粥硬的。」韩泥说，「人跟着硬半寸。」

汉子嘿一声，还是给了十二。钱落袋，袋底终于有响。响虽小，也比空袋强。

傍晚回来，门墩上竟又悄悄搁了半块饼——不是芽儿那种冷的，是粘着灶灰的热半边。他纳罕片刻，抬眼，巷尽头张麻婆扫帚一闪，像什么也没干。

他没追。追了恩就浅。他把饼掰开，一半留下，一半仍悄悄搁回沈家墙根缝里。墙上立刻传来芽儿吸鼻子的声音。

韩泥低声：「别哭。粥稀，情不稀。情在，迟早比罚银硬。」

屋里，他洗手洗脸，洗掉一日的集尘。手洗净了，心里那笔没洗掉——赖家要罚银，银可以欠；芽儿的粥，不能欠。欠银是账，欠粥是心。心账不能抹。

夜里风从门缝钻进，坛腹一线微温，像听懂了那句「罚我」。韩泥靠墙坐，独眼望黑：「赖管事，你要罚，罚在外头。情，记在里头。」

外头罚银还悬着，里头那碗稀粥却已经沉底。沉了，就会长苗——仇的苗，也是还的苗。苗不急着冒尖，先把根扎住。

天快亮时，他又摸到门墩那道细缝。缝里半粒米影还在。他用指腹按了按，按实：「芽儿，翁不吃你第二碗。第一碗够了。」

角落湿了，像又落了一点粥。

他关门，编筐，天亮还要去卖。卖得动，银才有影子；有影子，赖家嘴才堵不全。堵不全也不怕——情记着，比银先硬。

「明日，」他对空屋说，「还走集。罚银在外，情记着。」
