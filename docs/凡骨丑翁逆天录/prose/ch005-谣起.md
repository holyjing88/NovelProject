# 第五章 谣起

脏水的事，半日传遍泥岗村。

风一过，话比风快——快在井台，快在集尾，快在叶家药铺帘外。韩泥再去井台，人散得比往日快。躲的不是他，是晦气二字。赵绣春那盆水，泼成了由头；落在周嫂嘴里，最响。

周嫂洗衣，棒槌砸石台，咚咚，像砸人心：「叶家寡妇往晦气翁门缝送东西……」「药铺生意淡，沾上了嘛……」「绝户翁克邻克亲，谁近谁倒霉……」

韩泥拎桶走过，水不洒。洒了，更像晦气。童谣丫墙根排石子，念得脆：「晦气翁，脏水翁，叶家寡妇不要命，送汤送粥晦气重——」

新词。毒在把叶青禾扯进来。韩泥脚步没停。谣是风，风无姓，不入心；人有姓，叶青禾的恩，他记了，不能让谣记脏。

井台边，周嫂棒槌正欢，见韩泥，故意敲得更响：「晦气翁，你敢打水？」

韩泥放绳，提桶，水起，浑，稳稳落地：「敢。井是村的，不是赵家的。」

周嫂棒槌悬半空，像没料到他会回。回得平，不尖，像石头落地，砸不出火星：「你……你顶嘴？」

「打水。」韩泥拎桶转身，「谣烫不了心，棒槌也烫不了。」

周嫂脸青一阵白一阵，棒槌落下，砸衣上，水花溅自己一脚。旁人窃笑，笑很轻，像怕周嫂听见。马春花凑过来，嗓尖：「你还敢回嘴？叶家寡妇都晦气了，你更晦气！」

韩泥拎桶，水不洒：「晦气在嘴，不在井。井是村的。」

马春花啐一口，啐远，像没讨着好。没讨着，也是小胜——胜在路还能走，井还能打。

午后，赵绣春家门口聚妇人，传她的话：「见一次泼一次。」韩泥屋里编筐，藤条勒指。门缝外飘过一句：「叶青禾也不嫌丢人！」

韩泥指尖顿了顿，低声道：「牵扯的是汤，不是人。」

他不说给外人听，说给自己听——汤是情，不是奸；情脏不了，脏的是嚼舌根的嘴。

里正巡巷，咳：「谣少嚼。嚼多了，连坐。」赵绣春门内一滞，声哑半寸。里正不看她，只看全村：「井是村的，铺是叶家的。谁再聚众辱人，记名。」

记名二字，妇人噤了。噤了，韩泥指节松半分——松不是喜，是路还能走。

傍晚倒泔水，娃们嚷：「晦气翁出来啦！叶家寡妇也要倒霉啦！」

韩泥直腰，嗓哑，不高：「倒霉的是嘴贱的。娃小，别学。」

娃们一散。散不全，散一半——够他回屋。

夜里雨意来，茅草沙沙响。门墩一响。韩泥开门，无人。地上两片姜，切得薄，像叶家药铺刀工。姜旁一张粗纸，二字，丑却正：趁热。

不是汤，是姜。汤喝过，姜仍来——谣起，她没退。韩泥拾姜，掌心热，热得像反驳全村嘴。他进屋，姜搁窗台，把趁热那张纸折好，塞入袄内贴胸——不是血证，是证：叶青禾没怕谣。

门外叶青禾声轻，轻得像怕人听见：「姜别省。省是穷，不是怕。」

韩泥嗯一声：「不怕。记着。」

脚步远去，远得像把谣甩在巷口。巷口谣还在，屋里姜在，坛在——在三样，人就散不了。

叶家药铺灯亮到夜深。帘内秤声不绝，一声接一声，稳得像钉：铺子没关，谣就没赢。韩泥在门缝外听一息，不近——近像牵扯。听就够，够知叶丫头没退，够知自己该活长些，好还。

赵绣春门内骂：「关什么铺！晦气！」

骂声尖，尖不过秤杆落盘的脆。脆响又起，赵声哑半寸。

韩泥回屋，姜切片，投进热水碗。姜辣冲鼻，他喝一口，辣得眼热，热得像把全村谣烫退半寸。里正巡夜，咳：「谣少嚼。」赵绣春门内一滞，声哑。哑，也是小胜——胜在还能喝一口姜汤，胜在叶家秤还在。

他摸坛沿，沿温不退，像瓮替他守着趁热——守着，就不怕谣一夜吞村。贴胸那纸印痕浮，浮得像字在呼吸。沈枯芽隔门问，声小：「韩爷爷，姜辣不辣？」

「辣。」韩泥说，「辣好。辣，谣退半寸。」

巷口周嫂还洗衣，见韩泥倒泔，故意嚷：「晦气翁，叶寡妇还给你送姜呐？」

韩泥盆稳，泔水不洒：「送的是姜，不是谣。谣你洗不掉，姜我能收。」

周嫂棒槌一滞，脸青半寸，没再接。叶铺秤声落，落脆，摁得赵家骂声又哑半寸。

他低声道：「谣在舌上，汤在心里。心里不凉，路就不断。」

次日晌午，他还去井台。周嫂仍在，见他便把衣往盆里按，按得猛，水又溅自己脚背。韩泥放绳提桶，水起浑，稳着落地：「谣还在，井也在。」周嫂棒槌停半息，没接话。旁妇窃语：「晦气翁倒硬。」窃语轻，轻也是胜——胜在绳还能放，桶还能满。

路过叶铺，帘内药香混着姜辛。叶青禾没露面，只秤杆落一声，落脆，像在答：铺没关，谣没赢。韩泥不近帘，只把贴胸趁热按平。按平了，脚下的泥路就能再走一程。

陈姑路过，领拢紧，低声：「嘴贱的，烫不死心。心热，才行。」

韩泥应声：「心热。靠姜，靠秤，靠活。」

里正巡夜又咳，咳过巷口：「叶家铺子开着，谁再关，记名。」赵绣春门内骂声咽半寸。叶青禾帘内一句：「秤在，谣进不来。」

韩泥没抬头：「进不来，也得防。」

谣在巷尾拖长，他编筐不辍。新词借旧调，调在井台——台边嘴多，理少，少理就多记。记谁推谣，记叶帘还动。姜在心口发辣，秤在叶帘里落脆。脆响压过赵骂，坛在角上应着趁热。

他编筐至更鼓沉，藤条勒肩，勒出印。筐编到一半，巷尾谣又拖长。拖长不要紧，要紧是叶铺灯还亮一线。亮着，他就把筐底再密一针，像替叶丫头把秤杆摁实。

沈枯芽隔墙咳一声，咳轻，像告诉全村：沈家还醒着，醒着，谣就吞不完全村。韩泥不接咳，只低声道：「姜辣，心热。心热，筐就硬。」

他数筐篾，数到第十二根，篾韧，韧得像心口那张纸折出的角。角不尖，却扎谣——扎不紧谣舌，扎得住自己的手。手不停，活就不停；活不停，叶丫头那碗烫才候得着还。

韩泥对门缝不应，只对窗台上姜——不对姜，对自己说：「她没退，我不散。」

谣在舌上，姜在心里，坛在角上——三样都在，人就散不了。丑时雨歇，井台空，空得像全村嘴也歇了半寸。他把贴胸趁热又按平，按平了，辛还在。还在，路就不断。

他摞好筐，摞齐，像把谣压下去半寸。压不下去舌，压得住手——手有活，人就不散。散不了，叶铺那盏灯才候得着他还。风从门缝钻，带着姜辛，辛里夹着药香。香不远，远也够知叶丫头没睡实。

纸上二字贴着胸口，睡觉翻身也硌一下。硌一下正好——正好提醒他：谣再响，叶家没关；姜再薄，情没断。断的是闲话，不是人。

他把明天要担的水桶绳又理顺。绳毛了，毛的地方他绕了半圈布。绕布不是讲究，是防打滑。打滑了桶晃，晃了水洒，洒了又像晦气。晦气的名头够多，他不添新的。

周嫂砸自己脚那一下，村里人当笑话；他当记号。记号不用写，写在心里就行：谁嘴硬，谁心虚，谁敢在井台把叶家寡妇编进童谣。编进去的，将来一个个还得掰开看。

夜深了，坛角又温一线，像把「趁热」二字听进去了。听进去就好，好过满村耳朵只听闲话。

她没退，我不散；明日井台。
