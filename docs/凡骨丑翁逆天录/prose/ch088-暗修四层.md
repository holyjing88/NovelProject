# 第八十八章 暗修四层

三层后第三夜，丑时。远灯近了半寸。近了，不是到，是灯芯更清。清像废窑余温里那一点亮，亮在渣，不在舌。韩泥盘坐床沿，席严，坛不露。掌贴沿，沿温一线。三层气转，转一周，转两周，转第三周时，远灯忽然定住。定住，像碑。碑钝，钝像测灵石——石秤命，灯秤境。气再转一周，掌纹烫，烫得踏实。

四层破关，内视里第四阶亮起，亮里一线白像引火。引火像废渣路尽处，像丹路将开——开在89，远不要紧，记她「像能炼丹」那一句。

温听雨洒扫，巾湿，水净。她瞥韩泥指稳，声清半分：「指比洒扫那日更稳。」更稳，韩泥不谢。谢轻，记在心里——记缘苗，记89丹路开。

角柜十包样，再锁。锁住了，才不示人。示人，谣又来；谣不要紧，晦气遮瓮。遮瓮，守四层，守三年约这一条命。

温听雨评指稳，像能炼丹的人。不是真炼——真炼在89丹路开，在95匿丹。远不要紧，记她这一句。

踏实了，关隘裂一线，裂声闷，闷在骨里，不在屋里。四层开了。开不像雷，像药山深雾散半寸——散出界碑影，影外一步无印，印内是境。韩泥屏息，独眼不眨。眨了，境就散。散不得，散了，四层白开。气稳在四层，稳像筐底那撮辛——沉在底，不浮。浮了，像求；不求，只记。他收功，收得像盖席。盖严，沿净，坛不露。

掌纹那路比昨夜厚半分，厚不要紧，厚在暗，不在册。册仍写仍末，掌写四层——写给自己，不写给别人。邻床鼾声如雷。雷不要紧，他鼻下静一息。沉腐浮三字不落舌，落鼻。更鼓将尽，天边白一线。白里，他廊下立半刻，任风硬，鼻下仍静。静里，四层境像掌里多了一尺暗线——暗线不在公示，在掌。

铁无言脚步远，停半息，声更低：「四层别叫。叫了，煞骨门嗅得到。」

韩泥不答，答在心里——心里记一笔：暗修四层，见境已到。到在暗，不在众目。坛底轻颤，臾墟残念弱一息：「……四层在掌，丹路将开。开在渣，别叫丹。」

韩泥低声道：「记着。」

韩泥只答：「豆爆。沿净。」

赖福盯半息，终是没掀席。掀不得，沿净，坛不露。卯时，药山边门。他背筐上坡，四层气沉足底，足底稳一分，稳像界碑前那一步——不过界，只捡渣。深雾起，鼻下小成，手快而稳。稳里四层境配嗅诀，配废渣路，配暗袋十包。管事验筐，记「深捡净」。赖福讽远：「四层也捡？捡出邪丹。」

韩泥不接茬。接茬，全丙九饿。他低声道：「只捡渣，不叫境。」

刘婆端粥，仍多半勺：「昨夜又亮？」

「豆爆。」韩泥说，「沿净。」

「沿净怕什么。」刘婆哼，「饭稳，恩不断。」

韩泥点头。恩在粥里，也在四层境里——境轻，记重；记重了，才开丹路。午后，废窑门槛外。他再试温，不占窑。温应掌，应得像坛腹温一线——温不应字，应「路」。渣灰里亮多一丝，多不要紧，多像丹路将开——开在渣，不开在舌。

铁无言立径外，声低：「四层到了，别贪炉。贪炉，除名。」

韩泥应声：「不贪。试温。」

白织月主道过，袖风冷：「杂役也四层？四层也炼不出丹。」

韩泥不辩。辩，费舌；舌要留来守境，留来守坛。他只答：「炼渣，不炼名。名在册，渣在袋。」

白织月鼻下哼，走了。少一句嘲，小胜。丑时前，他再练半刻。三层在忆，四层在掌，五层在远——远在95，远在110，远不要紧，脚不盲。秦霜木鱼轻敲：「到境心别骄。骄，邪。」

韩泥不答，答在心里——心里恩辱分列。四层是境，境通向丹路开，通向炼气五层。他独眼平，手不抖：「四层到了。丹路在明日。五层在远。」

韩泥只答：「不恋。记着。」

坛腹应温，像应「四层」最后一笔。应完，他不叫。不叫，才守得住暗修，守得住门缝灯。他低声对坛：「四层到了。不叫。坛在，袋在。袋在，丹路才开。」

坛不应字，只温。温，像把远灯那一点清，摁进掌纹那路里。邻床悄问：「你身上热，真不怕？」

「怕。」韩泥说，「怕也叫。不叫，沿净，才藏得住。」

少年不懂「藏境」，闭嘴睡了。更鼓沉，沉里，他摸暗袋，袋沉，渣像种。种轻，记重；记重了，明日丹路开。他低声道：「四层到了。舌不留，手要稳。稳了，丹路自来开。」

韩泥只答：「沿净。睡吧。」

在，就不滚。不滚，四层才扎得实——扎实了，才开丹路，才冲五层，才钩95匿丹。柱上条角白一线，三年约在怀。怀凉，掌烫——烫在四层，不在廊下笑里。四层破后第二日，药渣场验指。韩泥分拣，指梢温半分，辨性更快，快也要稳。十堆全对，管事记「春验仍准」。准，是小胜。铁无言声低：「四层别示人。示人，像破外门境。」

韩泥应声：「不示。境在席下。」

温听雨声清：「指更稳了。稳，像能守炉的人。」

像能守炉，韩泥不谢。谢轻，记在心里——记498在远，记各做各的界仍在。赖福讽远：「末等四层？四层也滚。」

韩泥不接。舌要留来鼻，留来守坛。刘婆端粥，仍多半勺：「四层别飘。」

「不飘。」韩泥说，「飘了，废窑不收。」

午后角柜十包，锁实。秦霜木鱼一声：「境升别贪名。贪名，路断。」

韩泥不答，答在心里——心里不贪名，只记四层，记丹路将开。黄昏，废渣巷口，他立半息，不进。进是收，今日只记四层配槽。邻床问：「四层能赢钱戾衡？」

「赢不在四层。」韩泥说，「赢在活长。活长了，刃才亮。」

丑时前，内视半息，第四阶亮，阶上白线像炉门缝。门缝像95灯，远着呢。他低声对坛：「四层在掌。丹路明日开，不叫。温，像四阶亮摁进槽里。更鼓沉，独眼平：「四层稳了。丹路开在后，五层在丑时。」

更鼓沉，沉里，韩泥按木牌，牌凉，掌温。掌温在88，不在舌上。滚了，恩断。他不滚，手在，名在，路在。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。

滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。

滚了，恩断。滚了，恩断。

廊下风硬，他把木牌系紧一线，系得像把仍末摁进掌心。

同批少年远避，避像避晦气，晦气遮瓮，遮住了才稳。

编筐角藤刺仍在，他偏手，刺浅些，沿才净，坛才藏得住。

管事夜巡验棚，灯照席角，沿净，像腌菜坛，不像邪。

老耿挑水过，声哑，境别拜邪，拜了路断，他只记着。

秦霜木鱼远，一声，心别邪，邪了鼻下香飘，香飘手就滚。

柱上条角白一线，三年约在怀，愧字在怀，手仍不抖。

白织月远廊过，袖风冷，没停，胜在鼻，不在嘴。

铁无言脚步远，停半息，声低，别漏香，别叫境，只记。

废窑凡火试温，火匀一寸。一寸够了，再多，窑壁裂。裂不要紧，旧窑才没人争；没人争，91试炉才藏得住。

更鼓沉，沉里，韩泥按木牌，牌凉贴胸。88章手在，名在册，坛沿净。舌不留，鼻下静。恩辱分列，不混。活长了，才还得了叶汤，才接得住门缝灯。

柱上条角白一线，三年约在怀——怀凉，掌温；温在诀，不在众目，明日仍末。
