# 第十七章 送药

村会前一日，风更紧了。

韩泥胳膊上的伤结了痂，天亮照旧编筐。门墩上多了一小包，布旧，结打得怯——像娃的手。里三层外两层，最里是几株干药：止咳的，清热的，量不多，够熬三帖。纸角上二字歪歪扭扭：

别病。

不是叶家笔迹，是芽儿的。芽儿送药，像送粥，也像做贼——在泥岗村，对晦气翁好，就得偷偷摸摸。韩泥收药，先闻，不入口。苦里带一点甘，像叶家铺子的气。

「娃胆大。」他低声道。

药包布角磨毛了，像娃手心出的汗。他把药叶摊在膝上，一叶一叶看脉——看芽儿有没有退缩。藤条绕着膝，编筐仍不停；手不停，心才静。

药叶干了，味还在。他拈起一片对着门缝光，光很淡，照得出叶脉细纹。细纹像娃写「别病」时手抖的那一下——抖不是假，假不会送上门。韩泥把药叶按回布里，布结重勒一扣：「送了就记。记了就不白。」

编到晌午，筐底圆了。他搁下藤条，朝沈家方向望了一眼，确认豁口还在。

晌午马春花嗓子尖着找沈家麻烦：「芽儿又送！送药送晦气，你娘还要不要活！」

沈枯芽哭声闷在喉里：「我……我没有……」

「没有？包上泥印，全村都看见！」

韩泥在巷角，没出来。出来就坐实。他把药包分出一半，搁到沈家豁口，放下便走——走得快，像从没来过。墙内静了一息，像有人拾起来，没谢。谢字惹祸；不谢，恩也在。

马春花骂了两声，见无人应，啐远了，像一拳打空。打空也是小胜：沈家还接得住，芽儿还肯送。墙内芽儿娘咳得轻，像知道药回来了。

韩泥不叩墙，只把剩的那半包留下给自己胳膊上未愈的肿——他不配全收。自熬半碗，苦得皱眉，皱眉像把村会前的夜也熬进去。臂肿退了一线，心里那笔记得更深。

墙内芽儿细声对娘：「药……回来了……」

娘咳里夹半句：「别……再送……」

芽儿哽着：「不送……娘咳……」

静了。静里韩泥已经走远——远得像没听见；听见了也只记一件事：娃还肯送，这一点就不灭。

巷角有人贴条，条旧，写着「晦气勿近」。贴条的人手抖，像自己也怕那几个字。韩泥瞥一眼，不撕——撕成闹；不撕，当风吹过去。

夜里叶青禾隔巷低声问：「韩翁，村会上……我怎么说？」

「说汤。」韩泥道，「热，送，该送。别辩奸，奸在谣嘴里。」

「你呢？」

「我认恩。」他独眼平，「恩我担。担了，正好连。」

她「嗯」一声走远。药铺灯仍弱弱亮着，像村会前的烛——烛救不了嘴，救抓药的人。

午后，赖扒皮的人路过，棍尖点门扇：「晦气翁，村会前别生事。沈家那包药，别藏你灶角！」

韩泥开门，独眼平：「我生事？我编筐。」

来人瞄他腰，又瞄坛角：「腰里别藏东西。灶角也别藏。」

韩泥摊开手：「藏心里。恩是恩，辱是辱，各归各的。你要看，看脸。」

来人愣半息，骂句「老疯」走了——怕心，不如怕赖扒皮的令；令在，仇苗就在。韩泥合上袄，心里暗松一口气：仇在，他也不怕把心亮出来——亮的是恩，不是邪。

傍晚他熬一帖药，不自己喝，装进粗碗，搁到叶家门阶。不叩门，只留两字，写得丑：

趁热。

叶家和沈家，同一个词。词对不同人，意同一脉——热是情，不是奸。叶青禾开门见碗见字，站了一息，端碗回帘内。帘动，像轻叹，像谢，谢不出声。韩泥远远看着，不凑近。近，害她。

回屋后，心口芽儿那粒米旁，又添药叶一笔——浅，真。「粥稀，药苦。」他低声道，「情不稀，也不苦。」

他在灶边坐了片刻，听叶家门帘动了一下，像有人把碗里的热气接进屋里。接住了就好。接住了，「趁热」两字才不算他一个人自说自话。韩泥不开门致谢。谢字出门，赵绣春就能拿去编成「一对」。他宁可欠着，欠到还得起的那天。

外头风硬，吹不凉那两个字：趁热。药气淡，飘不进漏屋，飘进的是雨意——雨还没落，潮先到了，像村会前的气。

他自留的那半包药熬成一碗，苦入喉，臂肿似退半分。退半分够明日再去集尾卖筐，够他等村会锤落。芽儿的药叶在心里，浅而真；真就不白收。白收是讨施舍，他不做讨施舍的翁，只做记在心里的翁。

巷口张麻婆骂马春花，骂得响，却不去沈家闹：「闹娃，你算啥好汉！」

马春花尖着：「送药送晦气，不该管？」

张麻婆啐：「管嘴不如管咳。咳死了，你赔？」

马春花噎住，尖收回去——像今日送药又赢了一寸：沈家接得住，芽儿肯送，嘴再尖也尖不进豁口。

韩泥门缝里听着，不答。答成牵扯。他只低声道：「药苦，恩不苦。记着了。」

集尾卖筐，有人闻见袖里药香，退半步，又上前：「筐还卖？」

「卖。药香在袖，不在筐。」

买主掂了掂，仍付十二文。十二文落掌，像村会前还能活一日。回村，童谣丫远远看他袖子，不敢近——近像沾药香也沾晦气。韩泥路过不停：「药香不传染。传染的是嘴。」

井台边周嫂看见他袖口，棒槌顿了顿，到底没骂出口。骂不出口不是认了，是怕里正帖上「记名」二字还烫着。韩泥不看她，只顾走路。走路像编筐：一圈绕实，下一圈才不松。

盐担老汉迎面碰上，嗅了嗅：「你袖子苦。」

「苦药。」韩泥道，「苦的治咳，不治嘴。」

老汉笑一下，笑得很短：「明日听锤。锤落下，嘴才知道疼。」

公审前夜风更紧。紧里芽儿又摸豁口，摸空了，手怯。韩泥早把药送尽，门缝只留一句哑话：「别病。病在娘，不在包。」

芽儿声细：「翁……药苦……」

「苦才治病。」韩泥道，「怕苦，别怕送。送是点，点在，你娘就活。」

芽儿「嗯」一声远了，远得像做贼——在泥岗村，对晦气翁好，就得这样好；好得真，就得记在心里。

沈家墙内娘咳得轻了半分，够撑一日。一日一日攒，攒到村会，攒到还。马春花远处还尖，尖到叶铺阶下，见碗边「趁热」二字，尖又收了——风又断一截。

丑时他把药叶夹进旧纸，纸角再写二字：别病。写给自己，也写给娃——娃若再来，看见字，就知道翁记着，不必再做贼。药气散尽，屋里只剩坛腹一线温。温像第四笔记妥了：落妥不示人；示人，成邪。

天将亮，沈家又咳一声，像给村会前添一证：娃肯送，娘肯活，药叶记心，不虚。韩泥合上袄，独眼闭半息再睁——明日坐末席，备一句「我记着」，够了。

门缝又塞进一片薄姜。他拾起不尝，只记：娃的胆，也算一证。药泥味冲鼻，冲得像恩不必甜——甜在叶汤里，冲在芽儿手里；手里有药，村会就不空。

叶铺帘动，似有人端碗，苦气里带一点姜辣。韩泥隔巷不去闻。闻像牵扯；不牵扯，也是护。护到村会，护到叶丫头还能开秤。

他低声道：「药叶记心了。这一笔恩，落妥。」手按坛沿，瓮没醒，他不喊醒。沿口那线温贴着虎口，短，却够撑到明日村会。藤条靠墙立着，像今晚编筐也编够了——够了就不贪多；多了像讨好，讨好会把恩说薄。巳时他就坐末席，一句「我记着」，把芽儿的药、叶家的热，一并验给全村听。
