# 第五十章 赵罚

判决当日，执法堂钟响三声，声落时，妒火线像被收进一只铜匣，匣盖未合，余温尚在。

柳青鸢坐主位，眸光冷而直：「赵玄机，思过崖半月，戴罪立功。林挽月，罚役三月，逐出外门林苑。」

赵玄机跪谢，额上仍有思过崖磕出的痕，痕浅，却洗不掉。他抬头一瞬，看林挽月背影，背影瘦，步稳，像把「私」字留在堂上，把「证」字带走。堂下弟子屏息，有人低声：「半月思过，比加重轻。」另一人答：「轻的是罚，重的是后面那句——戴罪立功。」

莫长春自侧廊出列，拱手，声平：「宗主，夜袭将至，赵执事熟执法堂阵，熟丹堂侧翼。思过崖半月，可否改十日，守丹堂门？」

柳青鸢看他，一瞬，点头：「准。十日之内，不得离门三步。离一步，按逃罪论。」

赵玄机猛地抬头，像没听懂，又像听懂了太多：「长老……」

「守门。」莫长春道，「你说过这条命押在门口。提前押，比崖上空想有用。守住了，恩还一半；守不住，半月变整月，规矩不变。」

赵玄机重重磕头，声哑：「弟子……记下了。」

林挽月退堂途中，听见「守门」二字，唇白，白得像终于看懂：恩对路，不能改；慕错人，还能改。她不敢回头，只把罚役灰边在袖里攥紧，攥得像攥一枚更轻的「记」。路过侧廊，顾小满递她一张薄简，简上无求情，只有细则：「想走武试路，先消罚役记。路要直，字要清。」林挽月指尖一颤，颤得像终于摸到一点「改」的可能。

堂散，弟子三三两两往廊下走，议论比往日少一层油，多一层实。有人道：「莫长老替赵执事求情，求的是门，不是脸。」有人道：「守门比思过难，难在刀要向外，心不能向内。」孙福旧党在角隅交换眼色，眼色像还想把脏字捞回去，却被霍镇山从镇妖司方向投来一眼，眼如刀背，压得他们噤声。

赵玄机披甲出堂，甲新，眼亮，亮得像妒火终于换成守门的硬。他路过莫长春身侧，抱拳，礼深，却不多说——多说像私，少说是路。莫长春只一点头，像把昨夜不入浴房的半息，也点进这一礼里。

赵玄机抬头，眼里妒未灭，却薄了：「莫长老……昨夜不入浴房，是等我今日跪？」

「等你跪，也等你站，」莫长春道，「站不站得起，看你还库之后，还剩多少私。今夜守门，就是站给我看。」

赵玄机喉头一紧，颔首，转身赴丹堂。甲叶相碰，碰出闷响，闷响像把「守门」二字刻进铁里。

周德海在长老院听闻判决，面色平，平里冷，像早就备好的稿。他令旗在指间转半圈，迟迟不落，落得像犹豫，犹豫得像在等子时称一称谁的忠。副将问：「赵执事改守丹堂，周长老可增后阵人手？」他淡道：「后阵自有后阵的数。数在令旗，不在嘴。」

厉无殇在赤焰帐内听报，笑冷：「子时？赵玄机守门？好。本少谷主先破门，再破人。莫长春若出袖，正好一并称一称。」帐外妖骑蹄声震地，像给子时敲鼓。

莫长春不辩这些远话，至执法堂廊下，袖空，杖在掌，像把全宗的议论都隔在袖外。秦商言飞符至，符上八字，墨迹急：

**今夜子时，赤焰动东脉。**

柳青鸢收符，指节发白，未言，像把「子时」二字记进宗主令里。霍镇山抱拳，声洪：「镇妖司前移！东脉灵田、丹堂、侧径，全线布防！」苏念慈遣人报：丹堂地火阵已温，炉不灭——炉不灭，门才守得住。

魏无涯在檐角，扇合，低笑一声，像看因果满到不得不兑：「罚从轻，兆从急。急的是符，不是嘴。戈是刀背，罚是绳。绳小，门大。」

莫长春拄杖，望东脉，笑，像笑给九月将尽的夜看：「礼还没送完呢。他们倒先来了。」杖中竹纹急游，气音二字，像应：

备好。

寿元在塔壁，三月余。因果沉，像七十五斤人情压在胸口，不压垮，只催战。他不数息，只数门——丹堂门、东脉门、心门，哪一扇先响，哪一扇先还。

苏念慈自丹堂来报第二遍，声稳：「地火阵纹已红一线，弟子搬材搬药，不敢噪。赵执事若至，请长老示下，门洞阵眼与侧翼如何分。」

「阵眼她守，门洞他守，老夫门外坐着。」莫长春道，「坐着，不抢戏；不抢，戏才真。七夜前守炉，今夜守人。人守住了，炉才守得住。」

赵玄机闻言，刀光出鞘半寸，像把妒火也出鞘，换成守门的硬。他低声，仅自己：「门比剑难守。难，才像还。」

顾小满在杂役堂分班，路线图在怀，像第二张半石。她抬头看东脉云低，云低得像帖已贴在门上，门上贴的是命，命贴的是还礼。她不问「何时动」，只问「哪条路还活着」——路在秦商言简上，简在长老袖中隔层，隔层烫，烫得像夜袭已提前写在每个人眼皮上。她把三班人按三条侧径拆开，拆得像网眼，网眼小，容不下英雄，只容得下报信的人。

殿外弟子踮脚交换眼神，一人低声：「守门提前，思过缩短？」另一人答：「缩短的是恨，拉长的是夜。」话音未落，秦商言第二道飞符掠过檐角，符尾尚湿，湿得像子时提前落在眉间。

柳青鸢立宗主殿阶上，望东脉，声低：「符线要直。直了，夜袭才像试炼，不像乱。」霍镇山甲甲相碰，碰出闷响，闷响把「子时」二字撞进众人耳里——耳里先腥，后紧，紧得像弓将张满。

苏念慈闭炉一线窗，又开，让药香散出去，散得像给全门一颗定心丸。她低声对弟子：「炉不灭，门才守得住。守不住门，金丹也白修。」

霍镇山自镇妖司赶来，刀未出鞘，声先出：「谁再传浴房谣，先过我这一关。」一句话，廊下浪又退一寸。他抱拳对柳青鸢：「镇妖司前移，东脉三线，子时之前布完。赵执事守丹堂，我守田，宗主守令——令在，门才像门。」

秦商言第三道飞符至，符上六字：**先锋佯攻，实断东脉。**柳青鸢眸光更冷，令旗微抬：「佯攻丹堂，实断东脉。断脉先断耳，耳在杂役。」她望莫长春，莫长春只点头，像把「耳」字也点进簿上。

魏无涯在檐角收扇，扇骨合拢，像合拢一局妒棋。他笑浅：「罚是绳，兆是符。绳小，符烫。烫的不是手，是胆。」

莫长春袖空，一念出塔，杖点地砖，声轻：「罚完，兆要来。夜袭预兆，快里见血。」塔影在识海一荡，荡完又静，静里像全门都听见：赵罚从轻，不是纵私，是留人守门；留人守门，不是信赵，是信门规还能还恩。

子时之前，全宗都在等——等赵玄机是妒是守，等莫长春袖里还有什么。等，不是看戏，是称胆；胆在守门，也在还礼。赵玄机立丹堂门洞，甲新，眼亮，像终于把「心在谁那边」的问题，留给自己在门前答。风过执法堂檐角，檐角铃不响，响的是远处东脉一线腥。

罚轻，规立，立完东脉云低一线——低得像夜袭在门前先嗅了一口。

弟子渐散，散里仍回头瞧一眼灰袍袖空——瞧得像瞧一眼九月还能走多久。

赵罚那日，执事堂贴条，条红，红得像给全门看规。

莫长春袖空，一念出塔，杖点条角，不撕，只记。弟子围看，围看像怕罚传染，传染了就得还。

柳青鸢声稳：「罚为规，不为私。」赵玄机铁牌在腰，笑浅，笑里仍像藏下一招。

顾小满声平：「罚记下了。库还了，路还长。」

苏念慈开炉，炉旺，旺得像把罚炼成丹香。她低声：「罚罚别传成辱。」

围观弟子屏息，屏息等下一讯——讯来，来像兽吼，吼从东脉起。

罚贴出后，库门响，响像规立一寸，立里赵玄机仍挺，挺里妒薄不等于灭。

莫长春不撕条，只记，记像记礼——礼还，路还长。

柳青鸢声稳：「罚为规，不为私。」顾小满声平：「库还了，阵在。」

苏念慈开炉，炉旺，旺像把罚炼成丹香。赵玄机铁牌在腰，笑浅，笑里敬中带谋。
