# 第三十二章 西驿验骨

西驿蹲在坡下，人挤人，像把官道上的风都塞进棚里。

韩泥背篓入驿，肩稳着。驿门吏查路引——泥岗没引，他捧出里正荐帖抄本。吏展开，瞥见「言伯钧」三字，眉梢一抬：「凡骨？五十八？排队。别插队。插队，滚。」

他排进队里。队里少年多，骨嫩眼亮，看他像看一块碍事的旧木头。老，他不辩——辩骨像求人，他不求，只等。

验骨棚外有人卖测骨符，纸黄，嗓子响：「贴符保过！」

韩泥不掏三文。掏了像求，求轻；他信掌心，不信黄纸。

卖符人瞥他：「凡骨？贴符也白贴。」

「白贴，不贴。」韩泥说，「掌在，命在。」

队首少年测灵根，木牌青白闪过，执事弟子赞：「中品！入外门！」

少年挺胸，从韩泥身边绕远，绕得像避晦气。韩泥贴边让半步——让半步，够活。

候到午后才轮到验骨。木牌凉，贴掌，掌糙，牌滑。执事弟子瞥他麻脸，鼻下一哼：「凡骨，下品里的下品。去杂役处登记。别挡测灵根的人。」

旁侧又有少年测灵，青白一闪，众人「哟」起来。哟声落，韩泥已转身，转得像避光。比不过就不比。这辱跟泥岗同味，只是换了衣裳。他不辩，记在心里，不写在脸上。

队里有人嘀咕：「五十八也排队？占名额。」

他照旧不辩，把木牌抄本收好——绳拴命，命拴门缝。

又一轮，少年测骨只得「下品」。少年脸白，执事冷声：「杂役处登记。别挡道。」

少年哭半声，硬生生收住。收住的人，才听得懂「滚」字有多重。

轮到韩泥，执事不耐烦：「贴掌。别磨蹭。」

韩泥贴掌。弟子看结果，又哼：「凡骨，下品里的下品。下一位！」

旁侧有人笑：「这种也进宗？晦气。」

他转身走开，不跟人争长短。争是少年的事。

验骨棚角，一个测不过的老者被架走，哭腔远远的。韩泥不回头——回头像软，软了，西驿白来。

登记处簿上落下字：韩泥，五十八岁，凡骨，泥岗村，言伯钧荐，杂役试录。吏掷一枚木牌过来：丙九。

「丙九杂役房。」吏说，「明日卯时，山门侧门入。今夜宿驿西棚。棚费三文。」

韩泥付三文，三文从泥岗带来，像根。小胜就在这一下：没被拒在门外。门缝窄，窄也挤进去了。

登记处前又有人拦：「言伯钧荐的？云游去了，名额作数？」

吏翻簿：「作数。抬棺手稳，记过。」

韩泥应一声，不谢言伯钧。谢轻，记重——门缝是那名字撬开的，名字在末，缝却在。

登记吏又问：「带坛？」

「祖辈传的。」

「杂役房不许摆明面。」

「席下盖。」韩泥说，「盖了，不碍眼。」

吏哼一声，掷册：「丙九韩泥，入册。误卯时，除名。」

旁人叹：「丙九，末席中的末席。」

韩泥收牌，不辩末。末不是绝，是缝。缝在，门在；门在，饭在。

棚里潮，霉气重。邻铺少年嘀咕：「五十八做杂役，占棚位，晦气。」

韩泥铺席角，从叶振东药包里拣一叶嚼，苦得清醒。摸坛沿——一线温，像应「丙九」二字。

邻铺少年忍不住问：「你真抬棺？」

「抬过。」韩泥说，「抬棺手稳，清粪也稳。」

「稳有屁用，凡骨炼不了丹！」

「丹以后再说。」韩泥说，「先说粪。粪清完，才有饭。」

少年闭嘴，像被「饭」字堵住。

棚外有人卖热饼，香得让他想起张麻婆的馒。馒恩在，饼他不买——买饼像馋，馋轻，恩重。

邻铺又嘀咕：「听说言伯钧长老年少云游，留名额给抬棺稳的人……这老丑，抬棺？」

另一人笑：「抬棺手稳，炼丹手未必稳。」

韩泥闭眼，不辩。辩费舌。舌要留来活命，活长了才能把泥岗六笔还尽。

驿吏夜巡，敲棚柱：「丙九韩泥，明日卯时集合。迟到，除名。」

「不迟。」

夜深，驿外马蹄乱。有人喝：「让路！内门弟子过！」棚里少年涌出去看，韩泥不起——起像凑热闹，他还凑不起。

独坐听马蹄远去，远成钩：钩向山门，钩向兽栏，钩向那日瓮醒。

邻铺少年梦呓：「灵根……青白……」

他不答。答像比。比不过，就不比。

丑时前，他把木牌绳拴腕，拴实，像拴命。命拴在门缝上，缝是言伯钧名开的。

天将亮，邻铺少年催：「老头，起！卯时点名！」

韩泥早已醒——醒在丑时，像泥岗擦坛，习惯在，根就在。

卯时集合，执事弟子点名，点到丙九，韩泥应「在」。弟子瞥他：「凡骨，侧门入。别走正门，丢人。」

邻铺少年凑近：「你真五十八？不怕兽栏臭？」

「臭过泥岗井台。」韩泥说，「臭不算事。事是活。」

弟子掷单：「丙九韩泥，兽栏清粪，入门先练手稳。」

韩泥接单。篓底坛闷响一声，极轻，像应「兽栏」。他随队出棚，雾厚，丹香一线细得可怜，细也够引凡骨往前钻。

有人悄问：「五十八，兽栏臭，你图什么？」

「图活。」韩泥说，「活长了，还恩。」

「还什么恩？」

「泥岗六笔。」韩泥独眼平，「宗门新债，另算。」

问的人闭嘴。执事又喝：「凡骨记住，侧门入，别闯正门。闯了，打断腿。」

韩泥贴边再让半步——够活，够看，够记。记西驿这一日：骨下品，册丙九，门缝窄，窄也进。

棚外风硬，榜角翻飞。「凡骨可试」四字旧得像他的五十八年。年旧，命要新。新命从丙九起，末席也是席。

他把坛沿再按一下，温应「入册」，不应名。名在末，温在沿；沿在，根在；根在，韩家就不散。

散的是棚里霉气，不散的是肩上六笔恩。恩背着，西驿算过了；过了，明日卯时，山门侧门见。侧门见之前，先把手按实——按实了，兽栏清粪才不抖；不抖，才有饭；有饭，叶汤还不至于凉。

驿吏查棚时又瞥他独眼：「丙九韩泥，明日卯时，别迟到。」

「不迟。」韩泥说，「迟了，除名；除名，恩断。」

驿吏一滞，没再尖。尖收回去，路还直。

棚角有人赌：「这老丑，测骨过不了。」

「过了，我输半袋麸皮。」

韩泥不辩。辩费舌。舌要留着活，活长了，泥岗那几笔才还得完。

邻铺少年临睡前又忍不住：「老头，你进了山门，第一天干啥？」

「听吩咐。」韩泥说，「叫清粪就清粪，叫劈柴就劈柴。不挑。」

「不挑？我们都想挨着丹房。」

「挨得近，摔得也近。」韩泥独眼平，「我先挨着兽栏。兽栏臭，人清醒。」

少年啧一声，倒也不再嘲。韩泥把这条答也记了：清醒比香气值钱。香气能迷少年，迷不了他这把岁数。

驿西棚外忽然有人摔了碗，骂「晦气」。骂声远一点，他也不探头。探头像认账；不认账，晦气就还在别人嘴里转，转不到他丙九木牌上。

天光一线挤进棚缝。沉丹宗路牌已在心里，西驿这一关算过去了。山门还在前头。前头之前，他把丙九木牌按进掌纹——按实了，兽栏那铲才接得住。雾里丹香仍细，细得像线，线头却指向侧门。侧门窄，窄也要进；进了，才谈得上把热的一点点还回去。

卯时的鼓还没响，棚里已有人鞋底打地，像催命。韩泥早收拾停当：篓绳勒紧，坛在篓底，荐帖抄本贴怀。他起身时对邻铺少年只丢一句：「别睡过卯时。过了，除名。」少年迷糊应「哦」。他不再多话——话多浪费气，气要留给侧门那一脚跨出去。跨出去，西驿就变成后头的影；影在后，命在前。他摸了摸腕上丙九木牌，凉，却按得住。
