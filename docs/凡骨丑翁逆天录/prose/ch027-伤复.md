# 第二十七章 伤复

狼口那道伤，公审后才化脓。化脓，像迟来的嘴，咬自己。韩泥臂肿，肿得编筐慢半拍。慢，不停。停，像认输。沈枯芽怯来，拎小陶罐，罐里是药泥，泥黑，味冲：「叶姑姑让送的。她说，别化脓，化脓，抬不了棺。」

韩泥接罐，独眼平：「跟你叶姑姑说，我记着。」

芽儿点头，点得像米粒：「韩爷爷，狼那夜……我不怕你了。」

「怕也不用。」韩泥说，「活着就行。」

娃跑远，远得像恩又一笔，不必再记，已在心里。心里芽儿那粒米旁，早有一挑，挑浅，像药叶，像芽儿手。午后，赵绣春路过，见他臂缠布，鼻哼：「晦气连自己也咬？」

韩泥不辩，只举筐，筐沉，沉得赵绣春退半步——退，像公审余韵还在。还在，就够小胜。张麻婆骂赵：「人家救娃，你泼汤，谁晦气，心里清！」

赵脸青，走快。走快，像怕再当众哑口。哑口一次，够她记一辈子。周嫂洗衣，棒槌顿，顿得像也想退半步，退不出，便轻敲。轻敲，也是变。韩泥上药，药辣，辣得清醒。清醒里，他望后山狼道，道静，静得像那夜没发生过——发生过，记在心里，记在芽儿那句「不怕」。不怕，不是胆大，是恩厚。厚，他得还，还到芽儿长大，还到她能自立。夜里，他不增叶汤，只在芽儿那粒米旁又添一小点，像药泥落心，落得像印。印浅，真。坛角微温一丝，像吸药气。吸了，瓮未醒，人心先稳半分。稳半分，够他明日还去集尾，还去卖筐，还去听沉丹宗风声。外头谣又起，起得低：「晦气翁被狼咬，天罚！」

韩泥低声道：「天罚在嘴，不在臂。臂好了，嘴还在。」

嘴在，记也在。记，比辩狠。狠，不急，急便乱，乱便邪。他合袄，塞怀，臂肿似退半分——退半分够，够撑到荐帖贴榜那日。叶青禾隔帘问，声平：「臂还化脓？」

韩泥在门外：「上了药。能抬筐。」

「能抬，别逞。」她顿一息，「逞进宗门，死半路，我爹药白送。」

帘内药杵轻响一声，响得像叶家仍在，仍在，公审才不算白审。韩泥「嗯」一声：「不逞。命要还恩，不贱卖。」

帘静，静里秤响一声，像盖章。章盖「还活着」，盖得硬。沈枯芽又探头，探头，像芽出土：「韩爷爷，狼若再来……」

「再来，棍还在。」韩泥说，「你别上山，山给采药的人走。你走巷，巷安全。」

芽儿点头，跑回。跑回，像把「不怕」又送一遍。送一遍，够记在心里，不必再划。夜里，赵绣春门内摔碗，摔得脆，像不服公审。不服，也不出街骂了——骂累了，或怕了再哑口。韩泥听声，不笑，只按心口。按叶汤，按芽儿那粒米，按臂上布结。结紧，命紧；命紧，恩才还得上。坛角微温退，退得像伤退。退半分，是实；实，就不说满话。满话惹祸，实话记在心里。他低声道：「臂好了，心不能软。软，谣又进。」

赖扒皮的人路过，瞥臂布：「狼咬晦气翁，天意。」

韩泥举筐：「天意在天，筐在我手。手稳，天咬不死。」

来人骂「老疯」，走了。走，也是退半步——退，够。张麻婆送一贴膏药，膏药黑，黑得像馒皮：「贴上，别流脓。流脓，抬棺手抖，耿恩笑话你。」

韩泥贴，贴实，低声道：「耿恩记了，你的馒也记了。记全，手不抖。」

沈枯芽又探头，探头，像芽出土：「韩爷爷，公审那日叶姑姑说汤是热的……我记住了。」

韩泥应声：「记住就好。谣会旧，汤不能旧。」

臂肿退半分，退，够抬筐，够听荐帖风声。第三日换布，叶青禾隔帘递新布，布净，净里带药气。韩泥独臂解旧结，布揭时皮肉一紧，紧里泛白，白得像狼牙留的印。药气冲鼻，冲得独眼微涩——涩不要紧，要紧是脓退，退到抬筐不抖。「自己换？」她问。

「能。」韩泥缠布，缠实，「换不了，抬不了筐。」

「狼那夜，芽儿说不怕你了。」她声平，「不怕，不是胆大，是恩厚。恩厚，你得更惜命——别逞进宗门，死在半路，六笔恩谁还？」

韩泥只答：「不逞。命要还恩，还到芽儿长大，还到叶家秤还在。」

沈枯芽隔巷递来小陶罐，罐里是清水，水凉，凉得像娃手抖过：「叶姑姑让洗的。洗布，别化脓。」

韩泥接罐，独眼平：「跟你叶姑姑说，我记着。」

芽儿跑远，跑远里回头一眼——一眼，像徒线又实一分。张麻婆路过，骂赵绣春门缝：「人家臂伤还编筐，你嘴伤还泼汤，谁晦气，天知地知！」

赵门内摔碗，脆里没声——没声，像怕再当众哑口。陈姑送半块干饼，饼硬，硬得像冬恩：「臂好了，手别停。停手，像认输；认输，赵绣春又笑。」

韩泥收饼，掰半给沈家豁口：「不停。手停，筐空；筐空，谣说天罚应验。」

臂肿退尽，试抬筐，筐沉得稳。集尾有人贴榜角，角上「沉丹宗」三字丑，丑而真——真在路开，路开前，臂须好，须稳，须抬得动坛。卖筐时，买主掂他臂布，问：「狼咬的？」

韩泥独眼平：「咬过。筐还紧。」

买主愣半息，仍付十二文——文稳，像告诉全村：天罚在嘴，不在臂。叶青禾帘内秤落一声，落脆，像给臂伤摁了章。章盖「还活着」，章盖他该活，该还，该带坛走到宗门那日。韩泥望榜角，独眼眯一息：臂伤化脓像迟来的嘴，嘴咬过了，路还在——还在，就不白忍这一回。他低声道：「徒恩记着——还，先活；活，才配半块灵石。灵石远，臂先不能废。」坛角微温，像吸药气退了，人心却稳——稳，够接荐帖风声；风声来，宗门篇才开得了。

回屋，他洗布，水凉，凉里带药气。洗时独眼瞥坛，坛腹一线温，温得像应「还活着」那章——章盖在臂上，也盖在心里。心里芽儿那粒米旁，药叶又实一分；实了，将来还沈家，还到芽儿长大，还到她能自立，不必再做贼送药。外头赵绣春仍嗓，嗓尖，尖不过叶家秤落的那一声脆。

试抬空筐，筐起，起得稳——稳，像告诉全村：狼咬过，人还在；还在，就不白忍这一回。里正巡巷，咳：「臂好了，别生事。生事，连坐。」

韩泥举筐：「不生事。生事在嘴，不在臂。」

里正瞥他臂布，瞥完步远——远里，像也给公审留一寸缝：臂好了，手还在；手在，路还在。夜来，他仍擦坛沿，沿凉，心热，中间夹着五十八年的命。命薄，还没断；断不了，就因为芽儿那句「不怕」，因为叶家秤还在。

沈枯芽隔门问，声小：「韩爷爷，臂好了，还能救娃吗？」

韩泥应声：「能。救过一回，还能救第二回——第二回，别上山。」

芽儿「嗯」一声，步远。步远，像徒线又送一遍；送一遍，够记在心里，不必再划。
