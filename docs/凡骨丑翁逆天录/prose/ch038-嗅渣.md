# 第三十八章 嗅渣

药渣场，灰热，热气罩脸，罩得人喘。韩泥分拣，慢里鼻尖一动——苦蒿、蛇藤、叶振东那包药，都在鼻下排着。渣里一点香，香淡，淡得真，真就不乱。同场杂役掩鼻：「臭！」

韩泥不掩。掩了，闻不出路。路在香里，香在渣里，渣在手里，手不能抖。同场另一人呕了一声：「这灰吃进去，肺都要黑！」

「黑肺仍要清。清完，才有饭。」韩泥说。那人骂骂咧咧，仍拨渣。拨乱了两堆，管事一眼扫来，他赶紧学韩泥分堆。管事喝：「丙九的，今日渣场清到酉时。清不完，没饭。」

韩泥指不停，把灰渣拨成三堆：药渣、炭渣、废渣。他拈渣，灰里香不散。白织月再经，袖风冷：「闻什么？闻你也闻不出灵根。」

「闻药性。药性在渣，不在根。」韩泥说。白织月一滞，唇线绷了一息，走了。少一句嘲，也是小胜。同场杂役笑：「白师姐都懒得骂你了。」

韩泥不答。舌要留来辨香，留来记，留来还。铁无言抱臂立桩，声低：「渣别乱丢，丹房要回收。一粒不漏，回收才收你。」

韩泥把香渣单放小篓，篓小，香聚，聚才不泄。铁无言瞥他：「鼻灵？」

「鼻笨。」韩泥说，「笨久，笨出记性。记药性，记恩一样——记错了，还错了。」

铁无言盯着他分拣的手，手稳：「笨人守规，比灵人漏渣强。」

「不漏。」韩泥说。铁无言应一声，走了。同场杂役嘀咕：「丑杂役也配跟铁师兄说话？」

铁无言扫一眼，声不高：「渣不漏，话就少。话多，往往是手乱。」

嘀咕断了。断，韩泥记——记铁无言这一眼，记将来丹道门口规矩。酉时近，渣场渐空。香还在篓里，还在他鼻下，鼻下便是路。管事来验：「丙九韩泥，渣清完了？」

「完了。」韩泥说，「香渣单放，没乱丢。」

管事瞥篓，又瞥三堆：「分堆也对。谁教你的？」

「没谁教。」韩泥说，「鼻笨，笨出记性。」

管事「哼」一声，却记了一笔：「明日仍渣场。」

同场杂役凑近：「你分三堆，多费工夫，饭又不加。」

「加不加，管事看。」韩泥说，「堆乱了，丹房不收。不收，记名。记名，滚。」

杂役啧一声，学他分堆，手乱，灰扬一脸。管事骂：「学也学像点！灰进眼，明日你顶韩泥的酉时。」

杂役揉眼，不学了。韩泥指不停，香渣篓又满一层。午后换场，丹房外，药烟浓，浓得人咳。韩泥隔街站，闭息，辨出一丝甘——叶家药铺，叶振东那包药，「必还」未竟的余味。会，不声张。声张，赖福又来克扣。管事路过，喝：「别堵道。」

韩泥挪半步，鼻仍辨。辨出第二味苦，苦里带燥，燥是蛇藤渣余气。丹房弟子过，捏鼻：「杂役也闻？」

「闻路。」韩泥说。弟子一愣：「路在哪？」

「在丹房门里。」韩泥说，「门里我进不去，门外我记。」

弟子摇摇头，走了。他记下一味苦，一味甘，甘苦分列，不混——混了，还时必错。街角赖福瞥见他伫立，喝：「丑杂役，站岗？站岗也闻不出灵根！」

韩泥挪回渣场方向。赖福啐：「闻也没用，丙九一辈子闻粪！」

韩泥不回头。辱记着，鼻仍辨——辨出第三味涩，涩是废渣里残药，残药不能丢，丢则路断。回场，他把涩渣并入香篓，篓满，满而不泄。夜，丙九房，油灯暗。韩泥对坛低声：「渣里有香，香是路。路通向炼丹，炼丹通向还。」

坛壁水汽一线，爬半寸，敛。敛着，不是拒，是等。他翻刘婆补过的肘，肘暖。恩在，辱在，仇在，分列，不混。邻床问：「你天天对坛嘀咕什么？」

「记。」韩泥说，「记香，记恩，记饭。」

「记那么多，睡得着？」

「睡不着，就别睡。醒着记，记错了少一口。」

邻床翻身睡了。同场另一杂役隔床问：「白师姐都说你闻不出灵根，你还嗅什么？」

「嗅饭。」韩泥说，「饭在，才嗅得出药性。药性在，才还得上恩。」

那人愣了愣，也睡了。刘婆门外咳：「明儿渣场，别漏。」

韩泥应：「不漏。」

同场杂役凑门边：「你鼻真灵？白师姐都说你闻不出根。」

「灵不灵，渣不漏。」韩泥说，「不漏，就有饭。有饭，才活得久，才还得上。」

杂役闭嘴，走了。他把香渣篓藏床底。路不能张扬，张扬来克扣，克扣来滚。他低声道：「鼻下香，坛上温，心里恩。恩不断，人不死。人不死，汤还能还热。」

管事夜巡路过，敲棚柱：「丙九韩泥，明日仍渣场。漏一粒，记名。」

外头风硬，吹得棚顶灰落。灰落肩，他不拍——拍了，手上有香，香散了，路就窄。他摸篓沿，沿上余香未散。散着，便是丹道门缝里的第一道光——光不进丹房，光在心里，心里记着铁无言那句「渣别漏」。邻床鼾起，鼾粗，粗不影响他辨香。辨香在日间，夜间只记——记苦、记甘、记涩，记叶振东那包药的余味，余味淡，淡得真。

他低声道：「铁无言要渣不漏，管事要渣场清。都要，我就都要做到。做到，路才宽。」

刘婆门外咳：「别漏。漏了，我扣你盆。」

他闭眼，鼻下仍有一丝甘。甘短，够走；走，够熬到瓮醒那日。丑时，渣场味淡半寸。淡半寸，像铁无言白日那一瞥——瞥见手稳，明日少踹一脚。坛壁一线水汽，应「不漏」字，不应灵。灵在瓮醒那日，醒前，鼻下这一缕甘就够走。铁无言远处又咳一声，像确认他还在：「丙九韩泥，明日仍渣场。」

韩泥应声，鼻下路开。不报灵根，报稳——稳，才配嗅渣，才配将来辨香。辨香过，才还恩不飘；飘了，恩断，汤就凉，凉不得。渣香里一线正，正短，像汤温——温短记着还，邪腥长记着避。他摸床底香篓，篓沿余温未散。散着，像丹房门缝透出的第一缕光——光不进身，进心；心有了路，瓮醒才推得对。

丑时末，外头赖福骂远，骂不过「不漏」二字。叶振东那包药的甘还在鼻下，甘短，像叶丫头那句「趁热」——趁热记着，还时才不会错。明日卯前仍渣场。首场清堆，他先辨掠过的风——风里若有腥，不进篓；若无，甘苦涩才入鼻。辨清一次，铁无言才肯再翻簿；簿上有印，路才向嗅诀、向编筐、向冬后正测末排。

他把香篓推深半寸，像把路藏好——藏好了，克扣克不到，管事却看得见。天将亮，渣场雾起，雾薄，薄得像丹房廊下那半刻辨香的预演。预演在鼻，不在嘴；嘴稳不算，鼻下静才算。静了，甘短苦长才分得清——分得清，恩才还得了热，邪才碰不得。他覆席盖坛，沿净，不漏，路在。

鼻下甘短仍在，不许混进腥——混了，辨香路断；断了，恩还不了热。错不得，丹房门缝里的光才进得了心；心有了路，瓮醒才推得对。铁无言立桩，声低，不夸。不夸，眼却看堆——堆齐，名在末，也在路。路通向嗅诀，通向编筐，通向冬后正测末排。渣场灰热未散，他贴席卧，息匀才睡。睡浅，鼻却醒——醒着记腥甘分界。

分界清，明日不漏；漏一粒，管事记名，名压路。路在末，末在忍；忍过冬，正测才轮到编筐前那一关。舌要留来，还热先还鼻下这一缕甘。天光一线时，他披麻衣起身，篓不撒手——撒手气散，辨不准；准一次，名往后挪半寸，半寸虽小，末席活大。管事夜巡靴声又远，远，他仍应声一声——声轻，路在鼻下。
