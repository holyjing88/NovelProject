# 第五十章 赵罚

判决当日，执法堂钟响，像给妒火线收束。

柳青鸢宣：「赵玄机，思过崖半月，戴罪立功。林挽月，罚役三月。」

赵玄机跪谢，额仍有思过崖磕出的痕。堂下弟子窃语，窃语里反转——反转赵执事不是忠奸定死，反转莫长老仍肯留路。

莫长春出列，拱手：「宗主，夜袭将至，赵执事熟执法堂阵，可否半月改十日，守丹堂门？」

柳青鸢看他，一瞬，点头：「准。」

赵玄机猛地抬头，像没听懂，又像听懂了太多：「长老……」

「守门。」莫长春道，「你说过这条命押在门口。提前押。」

赵玄机重重磕头，声哑：「弟子……记下了。」

堂散，弟子炸开议论：「莫长老替赵执事求情？」「守门比思过难。」「九月老头……又要送礼？」

议论里，围观层爽点起——起在赵从妒到守，起在莫不罚而留路，起在全宗尚不知夜袭已在门外。

莫长春不辩，至廊下，秦商言飞符至，符上八字，墨迹急：

**今夜子时，赤焰动东脉。**

柳青鸢收符，指节发白。霍镇山抱拳：「镇妖司列阵！」

苏念慈遣人报：丹堂地火阵已温，炉不灭。

莫长春拄杖，望东脉，笑，像笑给九月将尽的夜看：

「礼还没送完呢。他们倒先来了。」

杖中竹纹急游，气音二字：

备好。

寿元在簿上，三月余。因果沉，像七十五斤人情压在胸口，不压垮，只催战。

弟子议「莫长老又送礼」，礼是路，路是守门。围观里怕，也燃——怕夜袭，燃的是赵终于有路还恩。

他低声，像钩：

「子时。青霜在袖，杖在手。看谁先抖。」

火把明灭，东脉方向杀意已腥。

有人低声：「罚从轻，恩从重。」另一人答：「恩是守门。门比思过难。」孙福旧党嘀咕：「赵执事还能翻身？」顾小满冷眼：「翻身看门，不看嘴。」

林挽月罚役途中，听「守门」二字，唇白，白得像终于看懂：恩对路，不能改。慕错人，还能改。

周德海在长老院听闻，面色平，平里冷，像早就备好的稿。他令旗在指间，迟迟不落，落得像犹豫，犹豫得像奸。

厉无殇在帐内听报，笑冷：「子时？赵玄机守门？好。本少谷主先破门，再破人。」

赵玄机披甲出堂，甲新，眼亮，亮得像妒火终于换成守门的硬。

罚已宣，恩已留，符已至——子时之前，全宗都在等，等赵玄机是妒是守，等莫长春袖里还有什么。

寿元在簿上，三月余。因果沉，像七十五斤人情压在胸口，不压垮，只催战。弟子议「莫长老又送礼」，礼是路，路是守门。火把明灭，东脉方向杀意已腥——腥在子时，腥在赵终于有路还恩。

堂散，弟子炸开议论：「莫长老替赵执事求情？」「守门比思过难。」「九月老头……又要送礼？」议论里，围观层爽点起——起在赵从妒到守，起在莫不罚而留路，起在全宗尚不知夜袭已在门外。秦商言飞符至，符上八字，墨迹急：**今夜子时，赤焰动东脉。**

赵玄机重重磕头，声哑：「弟子……记下了。」他低声，像钩：「子时。青霜在袖，杖在手。看谁先抖。」

赵玄机披甲出堂，甲新，眼亮，亮得像妒火终于换成守门的硬。罚已宣，恩已留，符已至——子时之前，全宗都在等。

莫长春望东脉，笑：「礼还没送完呢。他们倒先来了。」杖中竹纹急游，气音二字：备好。子时。青霜在袖，杖在手。看谁先抖。

围观弟子在殿外踮脚，踮得像等飞符落字——字落，夜袭已在门外。有人低声：「莫长老又送礼？」另一人答：「礼是路，路是守门。」

殿外弟子踮脚交换眼神，一人低声：「守门提前，思过缩短？」另一人答：「缩短的是恨，拉长的是夜。」话音刚落，秦商言飞符掠过檐角，符尾尚湿，湿得像子时提前写在每个人眼皮上。

柳青鸢收符，指节发白，未言，像把「子时」二字记进宗主令里。霍镇山抱拳，声洪：「镇妖司前移！」苏念慈遣人报：丹堂地火阵已温，炉不灭——炉不灭，门才守得住。

罚已宣，恩已留，符已至。子时之前，全宗都在等——等赵玄机是妒是守，等莫长春袖里还有什么。火把明灭，东脉方向杀意已腥，腥得像帖已贴在门上，门上贴的是命，命贴的是还礼。
