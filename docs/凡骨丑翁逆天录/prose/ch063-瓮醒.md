# 第六十三章 瓮醒

丑时。

沉丹宗丙九杂役房，风从窗缝钻进来，钻得像针。五十九岁的韩泥盘坐床沿，油灯豆光照坛角——那是泥岗村带来的腌菜坛，破席盖着，席厚，厚得挡得住眼，却挡不住丑时这一更。

外头赖福骂得远：「丙九别嚷，嚷了加倍克！」

韩泥不答。答费舌，舌要留给守坛。同屋小杂役翻身，眯眼半息：「丑老头，还不睡？」

「在。」韩泥声平，「编筐扎的，手还疼。」

小杂役哼一声，又睡实。睡实了，秘密才藏得住。

丑时前他没睡。清栏归来，指缝腥洗不净——洗不净不要紧，要紧是手不能去揭坛。他先换干草，再铲湿粪，粪铲沉得像压着六十日守沿的规矩；规在，席在，坛在。六十日里席一层层加厚，厚不为保暖，为挡管事的目，也挡自己揭坛的冲动。坛腹这几个月微温过几回，温完又敛，总像候人来碰这一碰。

他摸坛沿，沿厚，承得住三代人擦过的手。席下陶腹凉，凉得凡，凡才藏得住。今夜丑时，他把这一更掐在掌里，不准邻床梦呓把它掐散。

半只补单筐搁梁下。白日管事点过末席他的名字，名在末，活不能末。藤条干硬，他先浸软，绕膝两圈，第三圈勒紧，勒得疤边发白发痒。编筐末藤硬得像冬里的规矩；规矩在，刺才来。茧磨藤，藤找疤，疤在食指第二道旧裂上——冬测备检后就没合透。今夜藤刺一扎，血冒出来。

他甩手。

血甩到破席上。席旧，吸了血，血却不老实：渗过布纹，往坛沿走。不是有意滴，是活计扎的。血自己找路，找的是沿，不是喊出来的。

破席布纹竖起，吃住大半片血，挡出巴掌大的暗印，却挡不住那条细线——细线找沿，找沿不认理。一线朱红落到坛沿上。

血一落沿，坛壁颤了。

颤极轻，却是对他的血的颤。不是随便谁的血都能碰。

外头巡夜执事敲柱：「丙九，丑时了，熄灯。」

韩泥低声：「灭了。」

执事脚步远去。韩泥独眼一紧，心口突跳。

爹临终那夜忽然浮上来——铁山手搭这坛沿，嗓子哑得厉害：「坛别扔。韩家就剩你，坛在，根在。」

韩泥低声应了一声，应给自己听。爹的手凉，沿也凉；凉了三十八年，今夜才暖回来。

他又翻出祖父老栓擦沿的冬夜，翻出祖辈口传的「守瓮」二字——字不传外，传血，传擦沿的手。老耿经片里提过，祖上守过一只瓮，他那时只当故事听。三十八年了，他以为「根」是穷人家安慰绝户的话。

坛沿蒙尘里浮出极淡一道影，半个「韩」字，一闪即灭。掌心跟着烫，血脉里有什么，与坛底沉睡多年的东西对上了。

不是随便谁的血。是韩氏一代代擦过的沿，是爹搭过手的沿，是他五十九年每日再擦的沿——守瓮人的血，碰守瓮人的瓮。

掌纹里细线亮起，像灶下灰里一点红，沿脉走到腕，走到心口。心口不燃，只温，深过叶汤那一点。心记里「先还」二字，坛应了一声。

「嗡——」

一声闷响，非雷非钟，在颅骨里响。韩泥独眼一花，见坛底水光聚成一线，爬壁，爬出符形——丑，歪，像孩童乱画，却亮。符不是贴上去的，是从坛壁里长出来的；半个「韩」影被血喂饱，喂出第二笔、第三笔，钝得真，真得像丑翁自己。

符形沿壁游走半圈，圈口如瓮唇，一线白光吞吐。吐的不是仙气，是认——认他掌按处，认他五十九年擦沿的手，认韩家最后一脉守瓮血。

韩泥屏息，独眼不眨，怕眨一下，这认就散了。亮一闪，灭。灭前符形投在窗纸上，影丑影真，像有人把「瓮」字拆开给他看——不全不要紧，认主要紧。

坛口吐出一口白气，冷里带药香、泥腥、腌菜坛旧日的酸，还有一丝说不清的古气。很远的仙界有人争过的那只瓮，在凡间腌菜坛的壳里，睁了半只眼。

韩泥换指再按。未破皮的指腹贴沿，符不亮，光不应——应了，仍是那道血线。换指不应，他才知道这认，认的是沿，不是喊出来的声势。血线是钥，钥配了，瓮才肯醒。血脉认主，不是喊名，是沿上擦出来的认。

手背起皮，火舔冰咬交替。他没躲。

瓮醒了——不是梦，不是丑时幻觉。坛壁符形隐去，余温灼掌，掌纹里多了一道路，通向还恩，通向省亲，通向泥岗那碗还没还尽的叶汤。

韩泥独眼盯住窗纸，符影已灭，灭处仍留一点暖，暖得像叶青禾送汤那日碗沿的热。「醒了？」他低声问，「沿在，手在。」

无人答。坛沿一线温光，像爹的手又搭上来。韩泥按实，按到掌烫，烫得踏实。

里屋兽栏有夜鸟叫一声，哑。臾墟子是否在坛底、是否在将来开口，此刻不知——此刻他只要坛醒，命牵到了掌上。

外头鸡未鸣，沉丹宗夜禁脚步远。钱戾衡在玉牌里，在三年约里——约还在，瓮先醒，醒在血缘碰沿那一瞬，不急在这一夜。

韩泥把席盖回，盖严，像盖一个刚出生的秘密。秘密不能叫；叫了，赖福来，管事来，煞骨门旧债也来。

外头管事又喝一声：「丙九，还不静？」

韩泥只答：「在。」

他擦掌血，血入布，布黑，布上印痕丑，丑而真。「记着。」他对坛说，「瓮醒了。爹说的根，原来在这儿。路还长。汤还要还。」

他又低声补一句：「不认名，认沿。」

邻屋又有人嘟囔：「丑老头，又不睡……」

「行。」韩泥只答。

窗上符影余亮未散，隔壁小杂役眯眼半醒：「你那边……啥在亮？」

「油灯豆。」

「不像油灯……你手还滴血……」

韩泥盖席，席严，声更低：「编筐扎的。看错了。睡。」

小杂役哼一声，又嘟囔：「晦气……啥味儿……」

「坛味。睡吧。」

窗上余亮散尽，屋里又只剩鼾声，一切照旧。至宝仍肯叫他「腌菜坛」，不吓他——真名什么，仙界谁争，他不懂，也不追问。少年才追问。

鼾雷里，他按掌不松。克扣是压，瓮醒是释——释不在喊，释在认。

「爹，根在这儿。」他低声对坛，「我活长些，还恩。」

坛底极轻一颤，像应。掌纹那路还在，在就不怕明日麸皮再克——克的是粮，克不走瓮。

更鼓将尽，天边白一线。丑时尽了，坛腹温一线不退——不退，像瓮从眠里抬了半寸眼。路才长，长得够还。

他摸席角，席边湿在布纹里，沿上无血——无血，才像守住了六十日的规。规守住了，瓮才肯醒；醒了，不叫，不叫才藏得住。

明日清栏还得干，干完手还得编筐。编筐扎掌，血找席，席挡沿——沿认了，瓮醒了，登记还在后头等着。醒在缘，不在舌。

坊市尾巷风硬，他心里却稳，稳在掌纹那路还在。路在，就不怕末排；末排是名册上的字，根是坛沿上的血。

他盖严席，掌按坛腹那一线温。温得清楚：血触了沿，爹的遗言不再是空话，根贴在掌心。梦散得开，这一温散不开——明日名册仍写末，坛腹却已经醒了半寸眼。眼抬半寸，登记在后，还恩的路，总算开了口。口一开，路才真会长远。
