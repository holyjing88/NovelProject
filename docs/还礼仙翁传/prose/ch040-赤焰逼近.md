# 第四十章 赤焰逼近

赤焰使者二次上门，旗红，马嘶，像把七日之期又插回山门。

韩铁山抱拳，礼数足，刀意不足：「宗主，三日之内，东脉划界。否则夜袭，妖骑先行。」

柳青鸢沉脸：「东脉不让。」

「那便战。」韩铁山笑，笑里刀，「莫长老九月……哦，听说还剩四月？将死之人，也配谈战？」

全场一寂，寂里有人偷瞧莫长春。

弟子围山门三层，像等一场戏——上回手帕定身，这回韩铁山当众辱九月，看长老如何接。

莫长春拄杖，声轻却清：「配不配，试过才知。礼还没送完，你们倒急了。」

韩铁山扬眉，未料九月老头接话接得稳，稳得像不接辱，只接刀柄。他驱妖骑耀武，蹄声震阶，像给三日之期敲鼓。妖骑在阶下踏圈，腥风扑面，弟子后退，后退里又有人咬牙往前，像半块灵石撑着不退。

霍镇山盾光微亮，骂声起：「韩铁山，耀武给谁看？给田看，还是给死人看？」

柳青鸢指节发白，未拔剑，剑意已冷。莫长春却只侧脸对柳，低声：「宗主，该送的礼，还差几件。」

「什么礼？」

「给丹堂的，给杂役的，给赵小子的。」他笑，「还有给赤焰的——他们急，我们不必急。」

韩铁山听见，冷笑：「莫长春，夜袭子时，看你还急不急。」

莫长春点头，像谢提醒：「子时好。省得老夫敲门。」

全场哗然，哗然里，韩铁山笑意一僵，像被反刺了一刀，刀不深，却痒，痒在弟子眼里，痒在「将死之人竟敢约子时」的反差里。

魏无涯在檐角，扇合，低笑一声，像看戏看到高潮前一刻。

韩铁山耀武，妖骑踏圈，弟子后退又前，像半石撑胆。霍镇山骂声洪，骂得韩铁山笑意一僵。

周德海立后阵棚，令旗在指间，迟迟不落，落得像犹豫，犹豫得像奸。

柳青鸢剑意冷，未拔剑，剑在鞘，威在眸。莫长春侧脸对柳，声轻，却全场能听：「他们急，我们不必急。」

弟子哗然，哗在「省得老夫敲门」，哗得像将死之人反约刀。

韩铁山拂袖去，旗转，像血拖过山门。

寿元在簿上跳：四月。莫长春瞧见，不慌，像瞧见季节换牌。廊下弟子炸开议论，议论里怕，也燃——怕夜袭，燃的是长老敢接子时。

他低声，像钩：

「四月够送。子时到了，看谁先露怯。」

风腥，旗远，像刀已提名。

外门弟子比划定身三息，内门弟子摇头：「你们没瞧见，韩铁山这句话，比妖骑还腥。」「像捅刀。」「看长老接不接。」

顾小满在杂役堂分班，听「子时」二字，眼里亮，亮得像雨夜那道光还在等。她不知自己将走暗线，只知东脉不让，杂役也不能退。

林挽月罚役途中，听山门哗然，唇白，像终于看懂：韩铁山辱九月，辱不出莫长春退；退的只能是赤焰旗，不是青岚根。

苏念慈在丹堂听讯，炉火旺，旺得像替山门也担。她低声：「子时丹堂最危，炉不能熄。」

厉无殇在帐内听韩铁山回报，笑冷：「将死之人约子时？好。本少谷主也省得敲门。」魏无涯传讯，只八字：「莫长春，不急，急的是你们。」韩铁山阅讯，指节白，像被反刺一刀，刀不深，却痒在弟子眼里，痒在「赤焰竟先急」的反差里。

三日之期缩短成一夜，一夜缩短成子时——子时之前，全宗都在等，等韩铁山笑还能挂多久。

韩铁山驱妖骑耀武，蹄声震阶，像给三日之期敲鼓。弟子围山门三层，像等一场戏——上回手帕定身，这回韩铁山当众辱九月，看长老如何接。莫长春拄杖，声轻却清，稳得像不接辱，只接刀柄。全场哗然，哗然里，韩铁山笑意一僵，像被反刺一刀，刀不深，却痒在「将死之人竟敢约子时」的反差里。

山门风腥，旗红拖血。弟子窃语：「四月够吗？」「够不够，子时见。」莫长春袖中簿无新字，却沉，像又记一笔：辱九月者，子时还。

苏念慈在丹堂听讯，炉火旺，旺得像替山门也担。她低声：「子时丹堂最危，炉不能熄。」厉无殇在帐内听报，笑冷：「将死之人约子时？好。」

韩铁山听见，冷笑：「莫长春，夜袭子时，看你还急不急。」莫长春点头，像谢提醒：「子时好。省得老夫敲门。」全场哗然，哗然里，韩铁山笑意一僵，像被反刺了一刀。
