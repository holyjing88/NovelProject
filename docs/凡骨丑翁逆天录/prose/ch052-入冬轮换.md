# 第五十二章 入冬轮换

入冬第三日，丙九轮换兽栏。锣声一响，韩泥分到西栏——栏风硬，粪腥厚，厚得像冬压下来。

赖福啐地：「晦气专分西栏，克全栏运。」

管事冷眼：「再嗓，西栏你一人清到年尾。」

赖福哑了。韩泥肩上的筐却实，实得像七笔压着。

清粪时，西栏角风硬，硬进骨缝。粪锹沉，沉得像搬柴。韩泥不偷懒——偷懒，麸皮克；克的是粮，克不走手稳。

栏里最冷处，有只幼兽蜷在干草下，鼻息浅，浅得像要断。它咳了一声，咳得细。

韩泥蹲下，掌心贴它额。额凉，凉才要暖。他多铺两层草，草里塞半块省下的麸皮边角——边角少，少也暖。

幼兽舔他指背，舔得像谢。谢轻，他记在心里，不出口。

管事远处瞧一眼，没骂，只当没看见。没看见，是默许；默许，像恩还在。

管事走近，烛照幼兽鼻息，鼻息匀了半分。他低声：「多铺草，记功在。」

韩泥只点头，不谢。谢轻，记在第七笔里。

同批少年偷懒，管事记名。记名，像秤上空一格。韩泥多留半刻，多留，不邀功，只记。记在心里，刘婆粥那笔便又实一分。

清粪毕，他洗手。水冷，指缝腥洗不尽。洗的是袖，不是心——心记着七笔，袖腥，避；胸热，记。

管事验栏，烛晃兽眼，眼湿，湿像怕冬。怕不要紧，手稳，兽才少死。死多了，谣又多——谣借腥，他不沾袖。

午后，搬柴过宗门外。驿亭褐袍人仍在，仍在，像等冬寒够深。商队余尘已尽，只剩五脉旗号在风里猎猎——猎猎不是谣，是界。

佛门灰旗远了，密宗赭还在亭里——还在，像经片挪到宗门外驿亭。

韩泥不凑亭，只瞥一眼。一眼，褐袍人袖里似有一张黄纸，纸边毛，毛得像凡物。凡物，才入凡人手。

侧门窄，窄得只容半眼。半眼，够看商线还在不在。还在，心不乱；乱了，手就飘。

老耿挑水路过，水洒稳，声哑：「那人等冬。冬深，才赠。赠，不白受。」

韩泥应声：「白不受。受，记恩。」

耿不再言，脚步远。

第二趟搬柴，风更硬，吹得侧门霜白。霜白，像辨香那日廊下——廊下过，路还在。

褐袍人未起身，袖里黄纸像多了一丝轮廓。轮廓丑，真。韩泥绕远三步，三步，像界——界内末排，界外赠缘。

管事验垛，声硬：「冬里柴沉，沉得住，记功在。」

韩泥只点头。记功在，窄路又实一寸。

回棚，坛腹时温时凉。温时，像瓮听柴声；凉时，像瓮眠。眠不要紧，要紧是沿上无血。

他揭席一角，看沿。沿尘，沿滑，沿上无血。无血，才守不主动滴血的规矩。

邻床抱怨：「坛占角，晦气。」

韩泥不辩，只按席角。按实，像编筐扎掌那日——掌茧再裂，裂在冬，不在沿。

邻床问：「试手验啥？」

「验手。」韩泥说，「手抖，丹歪，滚。丹歪不要紧，手不能歪。」

邻床不懂，哼一声，翻身。

刘婆端粥来，稀，热：「西栏腥，别空肚。」

「不空。」韩泥喝尽。热在，像第七笔又实。

赖福远处骂：「晦气清粪，兽也晦气。」

管事路过，声冷：「再嗓，克半月麸皮。」

黄昏，铁无言巡栏边，灯远，声近：「兽栏腥，洗尽再回棚。」

韩泥应声：「洗尽。袖腥，避；胸热，记。」

铁无言停半息，又道：「试手前两日，手别僵。僵，藤条不听。」

韩泥独眼平：「手记着，不僵。」

铁无言又补一句：「西栏腥，洗袖再睡。袖腥借商线，路易断。」

韩泥只答：「记下了。」

灯远，像门缝又开一线。线窄，够走。

夜，韩泥对坛，声低：「冬第三日，栏换了，商线还在。在，不急。急，像借腥。」

坛腹微温一线，又凉回去。凉回去，像呼吸——吸温，吐凉，不急醒。

外头更鼓二响，响沉。沉里，试手又近一日。

他低声道：「席盖严，沿不碰。碰，不是今日。今日，只清栏，只搬柴，只记恩。」

同批少年碰他袖：「西栏腥，你咋不躲？」

「躲，手飘。」韩泥说，「手飘，试手滚。滚了，恩断。」

少年懂「断」，闭嘴。

丑时，坛腹又温一线，温短，像瓮听西栏咳声。咳声细，不应人，应冬。

他摸木牌，牌边磨毛，毛得像心口七笔。笔笔磨过，磨出实。实了，试手才站得住。

秦霜远廊一句，声轻：「心别邪。邪，符灭。」

韩泥怀符暖一丝，心记着七笔，不邪。

三更，西栏幼兽又咳一声。咳细，他想起白日那舌舔指背——轻，却真。真，像第七笔里添了一粒粟。他不奔栏；奔，像邀功。功在记，不在舌。棚角坛腹却温一线，温短，像瓮听那咳声——听，不应人，应冬。温完又凉，时温时凉，像呼吸。

四更，铁无言提灯路过棚角，灯照坛席，照出一圈净沿。他低声：「沿净，好。好，才接得住冬深那一赠。」

韩泥应声，不追问。问急，像借腥；等，像守瓮人。

侧门风硬，风里驼铃远了一声，远得像商线还在门外。还在，界就还在；界在，心不乱。

他摸木牌，七笔分列。叶丫头汤，振东药，陈姑袄，刘婆粥，符恩，耿恩，僧恩。笔笔在，手才不飘。

五更将尽，天边白一线。白，像试手前两日也白了半分。白不要紧，手要紧。

他按席，席贴坛，坛腹时温时凉——温时像听，凉时像眠。眠着，不急醒。醒，要等缘，不等人催。

轮换稳，手不飘。手不飘，恩不断；恩不断，汤就不凉。

搬柴第三趟，霜厚，柴湿，斧声闷。闷不要紧，腰记着；腰记着，试手才不滚。

同批少年夜間问：「你图啥守坛？」

「图沿净。」韩泥说，「净了，才接得住误那一甩。」

少年不懂「误」，却懂「净」，闭嘴。

试手又近一日。近，他仍末排；末排不要紧，手正才要紧。

轮换夜将尽，西栏幼兽咳声又细一声。细不要紧，草还暖着；草暖，像小暖还给丙九末排。

他低声对坛：「你再眠。眠着，我守沿，守席，守不主动滴血。」

坛腹温一线，又凉。凉，像应「等」。

试手前两日，已在眼前。眼前不要紧，沿要紧。

管事夜巡名册，笔尖停他名上，停半息：「西栏清栏稳，记功在。」

韩泥不谢。谢轻，记在木牌边。木牌薄，真在还能站；站得住，才配试手前两日那一坡。

西栏风硬，风息了，兽栏却更静。静里，幼兽鼻息匀，匀得像第七笔添了实。

轮换夜尽，沿净，手稳，缘在等。幼兽鼻息匀，匀得像西栏那一角草还留着他的麸皮角——角少，少也暖；暖了，试手前两日那条帖才贴得住柱。

天将亮，侧门霜白一线。白里，驿亭已空，空像冬深那一赠还候在门外——候不要紧，沿净，手稳，缘才接得住。
