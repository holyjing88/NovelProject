# 第二十八章 赵之初见

回山主道，赵玄机带弟子拦路。

不是战，是问，问得像审，审得像妒。

「莫长老，」赵玄机拱手，礼足，声冷，「除妖为何不用青霜剑？定身为何不用化神符？破境为何靠培元，不靠闭关？」

弟子围一圈，像等标准答案，好回去讲戏本。有人踮脚，有人举符纸想记，像要把长老的答，抄成门规外的第三本册。

莫长春停步，杖点落叶，叶让路，像当年赵讽杖灵那日，叶也让过：「剑是师妹的，杖是老夫的。符在礼里，不在嘴里。培元是堂里正份，闭关是年轻人的奢侈。」

弟子哄笑，笑轻，像松一口气——长老答得巧，像相声，像戏，像九月老头又露了一手。

赵玄机却笑不出，笑里更妒：他怕长老深藏，又恨自己看不透。看不透，像执法堂铁牌照不亮的影。

他再问：「手帕定身，帕从何来？」

莫长春看他，像看一块终于肯开口的铁：「你问帕，不如问心。心还了库，帕从心来。」

赵玄机一窒，当众被点「还库」，脸热，却未怒，只拱手退，退得硬，像保留最后一点执法堂面子。

围观弟子炸开一层声浪：

「还库？赵执事还库了？」

「手帕从心而来？什么意思？」

「听不懂，但好听。」

「赵执事脸热了，罕见。」

林挽月远处瞧见，忽然觉得赵师兄不亮，莫长老不弱，亮弱之间，像一条她还没走完的路。

午后，丹堂传话：苏首座请莫长老评新丹样。

莫长春未去，只回四字：「信首座。」

苏念慈瞧字，笑，笑又像哭——他信丹堂，信首座，却不把私密话点破在席面，像丹堂夜语那夜一样，把重意留在门外。

她把丹样封匣，匣上贴签：「筑基后期培元·莫长老例份加倍。」

弟子问：「首座，为何加倍？」

苏念慈道：「因为他收得下苦，也该收得下甜。」

弟子不懂，懂的人已在廊下窃语：「苏首座替长老争例份，像替谁争一口气。」

柳青鸢路过丹堂，听见，未停，眸光微动，像风过剑鞘，不起声，却有声。

赵玄机在执法堂抄门规，听见「例份加倍」，指节叩铁牌，一声，像把妒与服绞在一起，绞成更硬的墨。

暮时，莫长春立院中，簿无新字，袖却暖。

他望主道，像望一条还将闹的路。弟子门外仍议论「帕从心而来」，议论被风扯散，扯不散赵玄机眼里那一寸疑。

弟子回去讲：「赵执事问帕，长老答从心！」讲成戏本，戏本里赵执事成了反角，反角却不再尖牙。

林挽月夜里写信，信未成，成的是犹豫——犹豫赵师兄，犹豫莫长老，犹豫自己该慕该证。

魏无涯在梁上听完全场，扇合，低笑：「问得响，答得巧。巧比剑难。」

霍镇山路过，骂：「问那么多干嘛？杖扫过妖就行！」弟子笑，笑里敬霍，也敬莫——敬的是答巧，像相声，也像刀。

顾小满在杂役堂听见主道喧闹，抬头，像又记了一笔：赵执事问，长老答，答得像刀，刀不点人，点得人心虚。

孙福旧党在角隅嘀咕：「还库了也问不出帕来路。」顾小满冷眼：「来路在堂上见，不在嘴上嚼。」

他低声，像钩：

「问完了？问完了，才好把百年旧账，翻出一页给人看。」

秋虫起，细，像簿翻页。缘字在袖中微亮，亮一息，又敛。

大因果还在远处，像天试，像赤焰——赵之初见只是小问，闪回一瞬才是大还礼的前奏。

莫长春立院中，簿无新字，袖却暖。他望主道，像望一条还将闹的路。弟子门外仍议论「帕从心而来」，议论被风扯散，扯不散赵玄机眼里那一寸疑。

他低声，像钩：「问完了？问完了，才好把百年旧账，翻出一页给人看。」秋虫起，细，像簿翻页。缘字在袖中微亮，亮一息，又敛。

苏念慈瞧字，笑，笑又像哭——他信丹堂，信首座，却不把私密话点破在席面，像丹堂夜语那夜一样，把重意留在门外。她把丹样封匣，匣上贴签：「筑基后期培元·莫长老例份加倍。」

赵玄机在执法堂抄门规，听见「例份加倍」，指节叩铁牌，一声，像把妒与服绞在一起。

围观弟子在主道围一圈，围得像等标准答案——答案来得巧，巧得像相声，也像刀。有人低声：「帕从心而来，什么意思？」另一人答：「意思是你还库了，心才松一寸。」
