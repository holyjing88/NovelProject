# 第八十一章 废渣收集

四层那盏远灯看见后，药山差事换了字。柱上新贴条，字硬得像钉子：

「丙九韩泥，废渣收集，日半。只收渣，不采药。私藏活药，除名。」

韩泥盯着看了两遍。只收渣，他做得惯；惯不等于轻——手飘了，鼻也跟着飘。暗袋里早攒了几包样渣，包丑，包实，全塞床角，绝不示人。示人，赖福那双细眼就来了。细眼一来，丙九都跟着饿。

铁无言在廊角停脚，下巴朝废窑方向一抬：「窑在远，渣在先。先攒够了，窑才开得。」

「攒多少够？」韩泥问。

「够你手稳。」铁无言声低，「混了堆，废窑不收。收错了，倒不如不收。」

韩泥「嗯」了一声。够手稳，比够斤两难。斤两靠秤，手稳靠鼻子——鼻子一花，活药混进渣里，除名条就落到头上。

卯时边门一开，他背两只旧筐上坡。筐底垫粗布，布丑，不漏。药山三处渣场：坡下灰黑，坡腰更腐，深堆边苦得扑鼻。他轮着转，鼻下一静，指尖才动——辛沉的入筐，腐浮的另捏，假香上头那层先剔净。真东西沉在底，不在那一层哄人的香里。剔的时候手慢半拍，慢比快稳；快了，假香混进真渣，午间过秤就露馅。

坡腰风湿，湿里腐气贴脸。同批有人掩鼻走过，骂：「晦气活，晦气人干。」

韩泥不抬眼。骂不顶饿，顶饿的是斗不满、册不对。他把一撮深灰再捻一下，捻出一点辛，入筐；上层浮黄，剔除，甩进弃堆。弃堆上也有人眼红——眼红没用，弃的是假，不用贪。

铁无言立在坡角，补一句：「收集别混。混了，废窑不收。」

韩泥应道：「三堆，不混。」

午间管事验筐，倒秤，半斤对册。对上了，记「丙九韩泥，收净」。赖福贴在秤边，眼细得像缝：「收渣也收？收出私藏，连坐丙九。」

韩泥不接茬。接茬，全棚饿。他只低声：「只渣，无活药。」

管事冷声：「验过了。赖福，少嗓，多秤。」

赖福脸一青，青完还扫韩泥袖口——袖里无香，无烟，像从未摸过活药。扫完，笑薄得刺人：「验过也得盯着。丑杂役手长，长了就夹带。」

管事鞭梢一指：「盯秤可以。诬人，记你名。」

赖福嘴噎住，退半步。韩泥接过空筐，肩一沉，像把「收净」两字压进骨头里。压实了，才配回棚分装。

午后他把白日收的渣分成七纸包，包实，塞进床角暗袋。暗袋是编筐时留下的夹层，丑，却严。邻床少年掩着鼻：「你床角啥味儿？」

「渣味。」韩泥声平，「药山活，活就有味。味在角，不在席。」

「能换钱不？」

「换不了。」韩泥说，「换了，明日除名。」

少年没再问。问多了像沾晦气。韩泥按席角，席严，坛不露。坛腹温一线，贴掌，像应了一个「收」字。他数纸包——七包，包轻，记却重。七十包还在远，远就先攒；先攒，才谈得上开窑。开窑在远，远也不急——急了手飘，手飘袋漏，漏了全白干。

刘婆提篮路过，篮里饼硬：「收渣饿了？」

「饿了。」韩泥说，「饼就行。」

刘婆往他手里塞饼，糙得掌印深：「别谢。记着。私藏别沾，沾了婆也救不了你。」

「记着。」韩泥点头。谢轻，恩重。「别沾」二字跟铁无言那句一样，像界碑。碑在心口，脚才不迈错。

黄昏他在渣场角多留半刻，把七包样再辨一遍，辨完并入暗袋。袋沉半分，沉得踏实。铁无言折返，声压得更低：「袋别示人。示人，赖福记过。」

「不示。记着。」

铁无言又指废窑方向：「窑在远，渣在先。你先把坡下坡腰走熟，熟了再碰窑门。」

「走熟。」韩泥应。

白织月从主道过，袖风带香，香冷：「杂役也收渣？收了，也炼不出丹。」

韩泥不辩。辩费舌。他侧身让半尺，让得干净。白织月鼻下哼一声：「让路倒利索。利索也洗不掉晦气。」说完走了。少挨一句嘲，他也当小胜——胜不大，够过今日。

丑时前，他廊下练半刻闭息。息在鼻，不在嗓。嗓响，香飘；香飘，袋就藏不住。掌纹那路转一周，二层稳着，远灯不熄——不熄像催他攒渣，他脚仍稳，稳了袋才不漏。漏一包，等于把命交给赖福秤。

秦霜木鱼轻敲，声远：「藏别藏邪。邪了，鼻下先飘香。」

韩泥不答，答在心里。收是活，活是路；路通向废窑，通向往后那些要过的门。他对暗袋哑声道：「七包在角。废渣路，明日走。」

邻床翻身，嘟囔：「啥味儿……像药……」

「渣味。」韩泥说，「睡吧。药价贵，这味不值钱。」

坛腹应温，不应字。温得像七包渣里那一点辛——辛在活里。活在，才收得长。

收渣第二日，坡腰斗车辙深，辙里积水冻了一线。他足底气沉，斗车推得稳。第一斗满，过磅，记册；第二斗满，再记。记功轻，路重。管事验到巷尽，验足迹不扒砖，又记「收净」。

铁无言声低：「斗里别露活草。露了，煞骨门的人也要闻。闻见了，你袋里那些样全废。」

「不露。」

刘婆提水过，水洒稳：「废渣臭，别空肚。空肚耳鸣，耳鸣手就花。」

韩泥喝一口，水凉，凉也能静鼻。赖福立巷口，又尖：「两斗也收？收出丹味。」

「只收，不炼。」韩泥道，「炼字在条上，不在掌上。」

午后回棚，七包样并到九包，包实，锁角柜。邻床问：「九包能炼丹？」

「能。」韩泥说，「不在今日。今日只收。收不齐，炉门不开。」

白织月远廊过，袖风冷，没停。没停也好，他鼻下干净就行。老耿挑水路过，嗓子哑：「袋别当宝示人。示，谣来。谣来，日子就短。」

「记着。」韩泥应。

夜里管事灯照席角，沿净，像腌菜坛，不像邪。秦霜木鱼又远一声，他心里仍把恩和辱分开——恩在饼里，辱在赖福那条缝眼里，两本账，各写各的。混写了，手就乱；手乱了，明日斗车先翻。

收渣第三日，斗车轧霜进巷。管事验三斗，对册，再记「收净」。刘婆又塞半块饼：「收长了，腰才不软。」

韩泥嚼着：「腰软，袋就漏。漏了，饼也白吃。」

铁无言最后一句落在他耳边：「收长了，废渣路才走得通。路通了，才谈得上后头那些人、那些窑。」

韩泥不答，心里记三斗，记焦回，记不叫。十包废渣锁进角柜，柜丑，锁像规矩。锁住了，赖福记不得；记不得，恩就不轻易断。

廊下风硬，他把木牌系紧。同批少年远避，避像避晦气——晦气遮瓮，遮住了反而稳。藤刺扎掌，他偏手，刺浅些，沿才净，坛才藏得住。净了，夜巡灯来也不怕；怕的是自己先松手。

更鼓沉下来。他按住木牌，牌凉，掌温。低声对坛：「渣收了。袋在角。明日走废渣路——窑门在坡背，先过门槛，别贪深。深一步，除名条比霜还冷。」

棚外有人脚步乱，像巡夜换班。他不动，只把暗袋口再勒一勒。勒紧了，才像把三日收净都拴住。邻床梦呓半句「麸皮……」，他伸手替人掖了掖袄角——掖完，手又缩回自己席下，按住坛沿。沿温，人不慌。

坛腹温一线，像听懂了「明日」二字。二字落地，坡背那扇塌门才不慌。不慌，试温才试得准；试得准，十包渣才不算白攒。白攒不如不攒；不攒，窑门永远是别人的影。影可以看，脚还得自己迈——迈出去，才叫废渣路。
