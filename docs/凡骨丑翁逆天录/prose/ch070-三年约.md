# 第七十章 三年约

卯时，踹台。

坡下霜白，台新，木色浅，浅像专门给末席准备的辱。辱在众目，辱在内门的眼，辱在赖福的笑——笑薄，薄也真。

丙九列队末排，韩泥立最前。前不是荣，是险；险在「滚」字，滚了，麸皮停，记功抹，七笔恩断。管事弟子敲锣，声沉：「踹台验末，手抖除名。」

内门弟子列廊下，衣鲜，眉锐。钱戾衡立台侧，靴底声沉，沉里带金线边。他瞥韩泥灰线，瞥完，唇角一挑：「丑杂役，上来。」

踹台散后，韩泥独留坡下。坡下测灵石凉，线灰，灰在众目下公示过——公示不要紧，玉牌在怀，烫在还路前头。前头有三年，有十三层，有让他滚的那一日。

韩泥上前，掌茧裂处疼，疼不露。掌按台沿，沿凉，凉进骨。台不高，高在众目——众目下，末席像笑话。

钱戾衡声不高，高在众耳都听见：「灰线三尺，也配站这坡？配站，就配滚。滚了，省得污内门眼。」

内门有人哄笑，笑薄，薄也真。真在众目，真在灰线三尺公示那一瞬——瞬短，辱长；辱长，才扎成仇苗。

韩泥指节白一息，白完，又平。他不辩。辩，费舌；舌要留来按台，留来守坛。管事冷声：「踹台验末，手抖除名。」

钱戾衡抬脚，脚落台沿，落得像踩。踩不要紧，踩实了，才记得住——韩泥独眼平，只答：「管事看手，不看嘴。」

脚落半寸，台木吱呀一声。吱呀不要紧，手要紧——韩泥手不抖，掌按实，实得像坡下按石那一掌。

有人嗤：「末席也硬？」

嗤声薄，薄也真。钱戾衡收脚，收得像意未尽。未尽不要紧，辱已够——够扎仇苗，够记三年。

他从袖里取出玉牌，牌小，牌青，青里刻三字：「三年约。」

牌掷台心，掷得像契。契在，仇在，路在。

钱戾衡声冷：「三年。三年后外门大比，你若还是末席，滚出沉丹宗，永不再入。若你炼气十三，站我台前——我让你滚，你才有资格不滚。」

哄笑又起，起得像潮。潮不要紧，韩泥不接笑——接笑，像认晦气；认不得，只认牌在台心那行字：三年约。

管事记簿，声沉：「韩泥，踹台手稳，不除名。三年约，记册。」

簿上又添一行小字：「契在怀，示人当记过。」示人不要紧，怀要紧——怀贴身，贴像守瓮人守沿。

廊角有人影立，立得远，像看客——影瘦，衣暗，暗里带一点煞。韩泥不近，只记：记影在，像魔教线远观，远观不要紧，手要紧。

铁无言抱臂立侧，声低：「灵根末，手稳，渣场仍收你。别漏。」

韩泥应声：「不漏。」

秦霜远廊经过，木鱼不敲：「心别邪。邪了，鼻下香飘。」

韩泥不答，答在心里——心里七笔分列，不混。仇苗又深半寸，深在玉牌，深在三年，深在钱戾衡这一脚。

他拾玉牌，牌凉，凉进掌纹。凉不要紧，仇烫——烫在还路前头，不在这一日。

钱戾衡已走远，走远还留一句：「丑杂役，记住今日。」

韩泥低声道：「记着。」

记着——仇因又实一分。实了，才配活三年。活三年，才还得了叶丫头那碗烫，才接得住235那一胜，才接得住510那一诛。

散队后，赖福晃木牌：「三年？丑杂役活不了三年。」

韩泥不接茬。接茬，全丙九饿。他把玉牌藏怀，贴身，不示人。示人，谣又来。

刘婆端粥，仍多半勺：「踹了？」

玉牌掷台心，台面震半息。震不要紧，韩泥掌仍按实——实了，钱戾衡那一脚才记进骨。骨记仇，不记滚。滚在远，约在怀。

散队后，同批少年围上来，问玉牌凉不凉。凉，像冬测那日灰线。韩泥只答：「凉进掌，烫在还路前头。」前头有三年，有十三层，有让他滚的那一日——日未到，手先要稳。

铁无言夜过渣场，瞥他怀：「契贴身，别示。示了，赖福记过；记过，恩也断半寸。」韩泥应声：「贴身。」贴身，像守瓮人守沿。

「踹了。」韩泥说，「手稳，不除名。」

「不除名怕什么。」刘婆哼，「手稳，饭稳。饭稳，恩不断。」

韩泥点头。谢轻，恩重。恩在粥里，也在忍里——忍里藏着刃，刃对着钱戾衡，对着三年约，对着玉牌上那三字。

他回棚，不近廊下笑。笑在耳，不在心——心在坛，坛腹温一线，温不应辱，应「记」。记仇，记约，记玉牌。

邻床悄问：「你真敢接三年约？」

「不接，除名。」韩泥说，「除名，恩断。」

少年不懂「恩断」，却懂「除名」二字重。重，像麸皮停，像记功抹，像叶丫头那碗烫还不到手。

管事夜巡验棚，灯照席角，照见净沿。沿净，像腌菜坛，不像邪。管事记簿一笔：「丙九韩泥，踹台后仍手稳。」

韩泥只答：「不示。记着。」

散队后，药渣场角风硬。他分拣半刻，指稳，稳里鼻下静一息——静，不是忘仇，是把仇摁进约里。约凉，仇烫，分列才走得远。

踹台后，管事另贴细条：「契在怀，示人当记过。」示人不要紧，怀要紧——韩泥把玉牌系肚兜，兜布粗，粗像泥岗旧袄。粗不要紧，粗才贴肉；贴肉，才像守瓮人守沿。

同批有人窃语：「玉牌是契，契接了，三年后必滚。」

韩泥不接语。接语，像认晦气；认不得，只认手在。手在，三年就不白接——接了，才还得了叶汤，才接得住235那一胜。

午后，他编筐换米，米少半升，少不要紧，少也煮。煮时，他把玉牌按在米缸沿，沿凉，牌凉，凉进骨——骨记仇，皮记约，分列不混。

钱戾衡弟子过廊，靴底声沉，沉里带笑：「丑杂役，三年后记得滚。」

韩泥铲粪，铲下稳，稳像按台：「滚看手。」

就三字，声平。声平，是小胜——胜在忍，不在嘴。弟子笑远，笑远不要紧，玉牌在怀，仇苗在心。

白织月远廊经过，袖风冷，没停。没停，小胜——胜在活，不在嘴。赖福讽远：「三年？丑杂役活不过三年。」

韩泥不接。舌要留来记，留来忍。忍不是怂，是刃——刃对着钱戾衡，对着玉牌，对着还恩那日。

他按席角，席严，坛不露。坛腹温一线，贴掌，像爹的手又搭上来。爹说过：「坛别扔。」扔不得，扔了就无根；无根，辱就白挨。

邻床悄问：「玉牌真接？」

「接了。」韩泥说，「接才有名。名在，就不滚。」

少年不懂「不滚」，却懂「接」，闭嘴睡了。睡了，棚里更静；静里，玉牌凉进怀，凉不要紧，路烫。

他独眼平，手不抖，低声道：「三年约，记着。记着，刃才磨得亮。」

刃不亮，先记。记，是丑翁的本事。记完了，才配活三年——活在末席，活在忍里，活在还恩那日前头。玉牌在怀，仇苗在心；心在，就不滚。手在，路就不绝。

沈枯芽在药圃角弯腰，弯得像没看见。没看见不要紧，韩泥也不近——近，像求恩；不求，只记半块灵石那笔，恩在还，不在这一日。

廊角煞影远，远像看客。看客不要紧，手要紧——手稳了，三年才站得住，才接得住臾墟开口，才接得住药山捡渣。

他廊下再练半刻鼻下静，粟壳香淡，淡不要紧；要紧是香一字不落。落齐了，玉牌才记进心里，不进舌。

踹台后第七日，柱上条角白一线。韩泥仍末排，仍不滚。玉牌贴怀，凉进骨；仇烫在还路前头。管事记簿「三年约，手稳」，手稳，名在册；册在，就不除名。除名，恩断；恩不断，忍里藏刃，刃对着钱戾衡，对着三年，对着130夺魁那日。

更鼓沉，沉里，韩泥按木牌，牌凉贴胸。70章手在，名在册，坛沿净。舌不留，鼻下静。恩辱分列，不混。活长了，才还得了叶汤，才接得住门缝灯。
