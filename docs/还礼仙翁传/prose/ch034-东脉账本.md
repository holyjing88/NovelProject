# 第三十四章 东脉事本

长老会再议东脉，烛比人长。

周德海捧册上前，册厚，厚得像七年和平能称斤卖：「割让三成，七年不战，灵石收益、伤亡减免，皆在此表。」

他指节敲表，敲得自信，像数字能当盾：「第七年，宗门弟子守田，轮换休整，成本可控。」

霍镇山拍案，震得茶杯跳：「割地换安，安在谁枕上？第七年赤焰再要五成，你拿什么填表？」

周德海笑：「霍长老，打仗也要算利。」

「利可以算，」霍镇山吼，「根不能拔！」

殿内低议如潮，年轻长老看表，年老长老看周，柳青鸢不言，目光落莫长春——师兄拄杖立窗下，像一根不表态的秤砣。

周德海趁势推：「莫长老，九月将尽，战起耗命，和谈省命，长老以为呢？」

全场目光齐落，像等九月老头也给「和」字添一笔。

莫长春忽然开口，声轻，却清，像秤砣落盘：

「周长老，第七年，谁守田？」

周德海一噎，像被问住的不是表，是魂：「……宗门弟子。」

「弟子会老，田不会。」莫长春道，「根在民，不在表。表上的灵石，买不回迁走的民，买不回被妖踏过的稻。」

全场一寂。寂里，有人倒吸一口气，像第一次听见「守田」二字还能这么问。

周德海笑容僵，揖手退半步：「莫长老……言重了。」

「不重。」莫长春道，「重的是表。表厚，心薄。」

霍镇山抱拳，声洪：「莫长老这一句，比盾还硬！」

柳青鸢指节叩案，一声，两声，第三声前，周德海已识趣闭嘴，册子仍捧在手里，像捧一块烫手山芋。

廊外弟子听见「谁守田」，窃语炸开：「周长老答不上来？」「莫长老又一句话定场？」「九月老头，专问要命处。」

会后，莫长春出厅，霍镇山抱拳：「莫长老，东脉这一仗，霍某跟你。」

莫长春笑：「硬不过东脉泥。泥在，人才在。」

周德海从侧廊过，目光与莫长春一触，一触即分，像谁都没看谁，又像谁都看见了谁袖中的简。

魏无涯在檐角，扇合，低笑一声，像看一枚子将落。

孙福远闻「谁守田」，嘀咕：「守田？守的是克扣吧。」

柳青鸢眸光微动，未言，像把「谁守田」三字记进宗主令里。

顾小满在杂役堂听见消息，冷眼压回孙福旧党嘀咕，册上又记一行：东脉事·第七年·谁守田——问在殿上，答在风里。

莫长春望东脉方向，像望一场将到的夜。风腥，云低，像秦氏简上的「夜袭」二字，已贴到东脉田埂。

秦商言飞符至驿，符上四字：「问魂好。」像商言也懂——根在，恩才有路走。

东脉田埂上，老农半夜被符光惊起，只当是妖讯，后来才知道殿上有人问「谁守田」。有人骂割让，有人骂衙门，更多人沉默——沉默像在等青岚门给一句能落地的答。

顾小满夜值抄册，笔锋压深，像把半块灵石又硌了一遍。她低声对凳边杂役：「田在，事才真；人散了，表再厚也是纸。」

赵玄机验表，验到妒火也亮，亮里仍想踩，踩到「真」，踩不实。莫长春不夺，只问真，问真像问礼——礼还，恩才平。

魏无涯在檐角，扇合，低笑：「第七年问魂，比三成割让贵。贵在人还在。」

周德海退席后，袖中和折子仍湿，湿得像七年和平泡过汗。他在侧廊停半息，令旗在指间，迟迟不落，落得像犹豫，犹豫得像奸。

年轻长老围抄「谁守田」三字，抄得像抄门规。抄门规也好，规在，根就在。

苏念慈闭炉，闭得像怕表烟进丹堂。她低声：「表别传成逃，逃了丹堂先苦。」

莫长春回望殿内烛影，声更轻：「表可以捧，根不能拔。下一声拍桌，该宗主了。」

柳青鸢在帘后，眸光微动，未言，像已听见下一声将落在哪里。

柳青鸢独自翻表至第三页，指节发白，发白不是因为灵石，是因为第七年轮换旁，注了一行小字：「民散不计」。她低声：「师兄问得对。表可以厚，人不能写成数字。」

霍镇山追到侧廊，声洪却不莽：「周长老，表上第七年轮换，轮换的第一批是谁？」周德海面色一白：「按资历……」霍镇山截话：「资历在山上，稻在东脉。你从山上抽人，田谁耕？」

殿外送粮队队长跪求，哭诉割让草案已传到村口，民想逃。莫长春扶他起身，声轻：「逃不急在这一夜。青岚门不先逃，你们也别先逃。」队长磕头，磕得像把一群人的命，暂时磕在山上。

年轻长老私议至更鼓，议不出第七年答，却议出一口闷气——闷气憋在「谁守田」三字里，憋得像东脉夜风灌进衣领。

秦商言再飞符，符上多两字：「表假。」魏无涯瞧讯，低笑：「商言比周德海先看懂表。」

莫长春回到檐下，袖空，一念出塔，杖点表角，不撕，只记。顾小满远远瞧见，瞧见像瞧见塔意也记在心里：纸可以厚，人不能散。

表摊到第三页时，有人故意传「莫长老要夺东脉」，传得像九月老头要把周德海的表撕了。莫长春不撕，只问：「第七年轮换，第一批名单在何处？」周德海答不出，答不出像答在风里。

霍镇山令东脉巡哨加一倍，哨响过田埂，响得像替「谁守田」先站了一夜岗。年轻弟子问：「守田也要修仙？」霍镇山骂：「修仙先活着，活着先吃饭。饭在田，田在根。」

孙福在角隅嘀咕「守田就是守克扣」，被顾小满一眼压回。她册上又补：东脉事·民散不计——柳青鸢指节白处，已有人看见。

会后第二日，周德海令旗仍不落，落得像犹豫，犹豫得像等宗主拍桌。霍镇山却已将东脉哨册摊在案上，册上空白处写着「谁守田」三字，像等周德海填名。

莫长春路过田埂，与老农对坐半盏茶，茶粗，话更粗：「门里争表，田里争秋。秋到了，表填不满肚子。」老农点头，点头像点在一笔还没认的事上。

柳青鸢遣执事抄录「民散不计」四字，抄得像抄宗主令。她低声：「表可以捧，人不能在纸里消失。」

魏无涯檐下观气，见东脉事气缠周德海袖，缠得像七年和平也缠不住第七年一问。他扇合，低笑：「问魂一问，比三成割让贵。」

莫长春回院前，对顾小满道：「册记下了，就别再添油。添油的人，往往怕真。」顾小满点头，像把半块石又握紧一寸。

周德海夜归长老院，令旗仍不落，落得像等宗主拍桌。霍镇山却在哨册空白处填了「谁守田」三字，像替殿上那一问，先落在纸上。

顾小满夜值抄册，将「民散不计」圈红，圈得像圈住一笔还没认的险。她低声：「田在，事才真；人写成数字，数字填不饱肚子。」

莫长春望东脉，风腥，云低，像秦氏简上的「夜袭」二字，已贴到东脉田埂——贴上了，表再厚，也挡不住田埂上的风。

霍镇山抱拳，声洪：「莫长老这一句，比盾还硬！东脉这一仗，霍某跟你。」莫长春笑：「硬不过东脉泥。泥在，人才在。」

东脉事本摊开那夜，纸厚，厚得像把三十年画在每个人眼里。

弟子渐散，散里仍回头瞧一眼灰袍袖空——瞧得像瞧一眼九月还能走多久。

表册厚，厚里藏火——火不在纸，在谁肯认那一笔民散之险。
