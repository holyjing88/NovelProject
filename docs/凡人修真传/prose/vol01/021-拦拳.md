# 第 21 章 拦拳

> 连载名：《账簿仙途》

泼水后，堂上更乱。

乱不是声，是气——气里水腥，笑腥，赵绣春掌腥，腥混成一锅，锅沸在堂心，沸在陈寻湿衣上。湿衣贴骨，骨不抖，抖的是人群里那些等着看「跪」的眼。

赵绣春要陈寻跪，声尖，尖里带胜：「洗了晦气，还不跪？跪，才像悔！」

陈寻不跪，膝直，直得像桩：「无罪，不跪。有水，无罪。」

「你辱村风！」

「辱在舌，不在行。」

八字落，落得像石，石击水，水花溅到赵绣春脸上，她脸一扭，扭得像被反泼。反泼不在水，在理；理一硬，跪就成了赵家要的台阶，台阶他不给。

萧惊鸿挤前，肩撞人群，撞出一条缝。缝里赵小满退，退半步，半步像怕拳，怕的不是拳，是拳后的案。萧仍追半步，半步追的是水，水泼在陈寻脸上，泼在他眼里——眼里不是泪，是火。

拳起，带风，风硬，硬得像三月前集口粮铺柱上那悬着的拳。

陈寻抓住腕，抓得准，准得像练过千次。低喝，声低，低里烫：「还没到时候！」

萧惊鸿眼红，红里水与怒混，混成一片：「都泼成这样了！你还要忍？」

「泼水是辱，拳是案。」陈寻说，一字一字，字像钉，「案上加拳，测灵名没了。名没了，站什么石？」

萧惊鸿喘，喘得像弓满又被人按住弦。弦在骨，骨响，响而不出。拳落，砸在自家腿上，疼，疼得闷；闷里不吭，像把怒砸进骨头，砸成三月前那夜祭父母时说的：听你的。

人群静了一息，一息里赵村长眼转，转得急——急的不是理，是势；势若成案，除名便有名目，名目要拳，不要水。

他改口，改得像翻页：「以证为据。周嫂，你男人，可是陈寻推下山？」

周嫂哑，哑得像被人掐住喉。她看赵绣春，赵绣春眼刀，刀快，快得她舌乱：「我……我听说……」

「听说，不是见。」沈振东声冷，冷得像霜落药柜，「见者，是我。醉者，是你男人。伤者，已证。推人者，无。」

人群分波，像水遇石，石是证，水是舌；舌退一寸，波就散一寸。

老耿独眼出，眼亮，亮里旧：「瓮我修，与他爹旧恩有关，与沈家无涉。陈寻封门练体，我瞧见，未夜出。」

张麻婆骂，骂得直，直里白：「周家酒虫，关陈寻屁事！我盐白送，送在他门口，未送在沈家帘里！」

方小沅抖，抖得像叶，叶仍立：「粥是我送，与他无夜入。门边，未进。」

赵绣春要拦，拦成尖叫：「小丫头懂什么！册邪！血字——」

钱管事记，笔快，快得像怕漏链：「证词在此。除名票不够。」

陈寻湿衣站直，像钉，钉在堂心。钉住了，赵绣春笑不出来；笑僵，僵里她仍拍掌，掌响空，空得像敲在铁上。

他侧目萧惊鸿，一眼，眼里有谢，谢不在言，在「不出拳」。不出拳，名在；名在，链才能继续。链继续，才是公审；公审未完，反噬未至。

赵村长拍案，案响，响里催：「再听！还有谁见？」

堂前风紧，紧里陈寻任水冷，任笑热，只立。立里，他觉出阴面那痕仍温，温而毒，毒在「当众」——当众的辱已入账，账不收拳，拳只添案。

他低声，只给自己：「审。继续。」

赵绣春脸扭，扭得像被反泼，反泼不在水，在萧那一拳未落。未落，势便泄；泄一寸，链紧一寸。她急，急里换声：「周嫂！你再说！」

周嫂抖，抖得像寒，寒里跪不得，跪了是给赵家台阶。台阶陈寻不给，萧也不给——不给，靠证。

猎户自人群出，声粗，粗里直：「我见刀落，落在他自家臂。陈寻在队尾，未近醉人。」

又一环，环扣伤者。伤者臂抬，抬得像证：「拉我，不是推。我若滚，是他拉。」

赵村长眼沉，沉里仍笑，笑僵：「还有呢？」

还有，是老耿瓮，是张麻婆盐，是方小沅粥——粥小，盐硬，瓮静，静里都是「未夜入」。

陈寻任水顺颌滴，滴在土上，土黑。黑像墨，墨在记；记谁泼，记谁拦拳，记谁把拳砸在自己腿上。

萧惊鸿腿疼，疼而闷，闷里他看陈寻，陈寻不看回，只看堂上——堂上链在齐，齐才公；公，才名在。

赵村长再拍案，声里带虚：「今日，除名票不够……」

不够，是理；理在，审就不能私结。私结是赵家想要的，想要不成，成的是下一章：证齐之后，醉者该上了。

风紧，棚顶绳响，响像催。陈寻湿衣贴骨，骨立，立成堂心那根桩——桩不倒，赵家的水就白泼。
