# 第九章 克扣小胜

告示贴在祠堂门板上，墨未干，风一吹，纸边就翘。翘着像一只随时要咬人的嘴。

「凡私助晦气翁者，连坐本家；集市货税，从严。」落款歪扭，却盖着赖管事那枚泥印。印色发紫，紫得像瘀。

马春花第一个念，念得声高，像敲盆：「听见没有！再给绝户送东西，全家倒霉！汤粥袄馒，一件也别再递！」

人群里有人「嗯」，有人低头。低头的人里有昨儿还接过半句闲话的。沈枯芽躲在娘身后，手指掐进掌心，掐出月牙白。

韩泥站在末排听完，嗓子不大，却钉进场子：「坐我。」

「你说什么？」赖管事的爪牙眯眼，手按着腰间短棍。

「告示写连坐。」韩泥独眼平，「谁帮我，坐我。别坐别人。坐别人，冤。」

爪牙哼笑：「你倒会捡便宜。税呢？税照收！少一文，封你筐！封了你喝风！」

周围有人窃笑，有人不敢笑。里正在侧，脸色平静，平静里压着火。

集尾，税摊支起木杆。往常卖筐抽两成多，今日加码，加得明目张胆。前头卖户被刮得脸色发白，有人把货又抱回去，嘴里骂天天不应。轮到韩泥，税吏秤也不秤，伸手：「十三文。少一分，扣筐。」

他袋里刚好十四文。交出十三，日子就剩一文喘气。喘气不够过两日。

「十文。」韩泥把钱拍在案上，不多不少，铜声脆。

税吏一愣：「你聋？告示说从严！从严就是加！」

「从严也得按秤。」韩泥指筐，「筐在，货在。十文是我平日税。多的，你拿告示问天。天不答，别问我。」

场子一静。静得听见旁边鸡叫。税吏还想吼，里正从侧面走过，咳了一声：「秤在不在？」

税吏脸色变，变完仍硬：「在……在。可管事说了……」

「管事说归管事。」里正眼神斜着，「秤在村里。克扣太狠，人也记。记多了，日子难过的不止绝户。」

税吏恨恨收走十文，少收三文像骨头卡在喉。他拿眼睛剜韩泥：「你等着。」

「等。」韩泥把剩下的钱按回袄底，「我学会等。」

小胜一场，胜得薄，却真。薄得像刀口风，风过，皮还疼。

回村路过沈家豁口，墙缝裂得更宽，宽得能塞进半只手。芽儿探头，眼圈红，红得像昨晚没睡。韩泥没说话，把袖里半个馒头——昨夜留下的冷边——塞进去。

「别吭。」他低声，「吭了，告示就咬人。咬了，比撕衣角狠。」

芽儿点头，眼泪却先掉。娘在里头压着咳：「韩……韩翁……连坐……」

「坐我。」韩泥又说一遍，「你们别坐。你们坐不起，我坐得起。」

「你也坐不起。」娘声哑。

「坐不起也坐。」韩泥道，「坐完再攒银。攒不动，就熬。」

傍晚，周癞子醉醺醺砸门。门板「砰」地响，泥灰掉，掉进领口里痒。

「晦气翁！开门！有浑水给你喝！」周癞子舌头大，「赖管事说啦……你害人……连坐……嘻嘻……告示贴了你还敢给沈家塞东西……」

韩泥开门，拦在门槛：「酒醒了再骂。醉着砸门，砸你自己脸。脸砸肿了，明日更像赖家狗。」

周癞子酒气扑人，一瓶浑水往他身上泼，半泼空，半泼门槛。水腥，腥得像污水沟，沟里还漂着草屑。

「泼吧。」韩泥不退，「泼完记账。账本不嫌湿。」

「记什么账！」周癞子笑歪，「告示说了，克扣你活该！活该你绝户！」

里正正好路过，看见门槛湿痕，眉一拧：「周癞子，酒壮屁。再砸，记我！」

「记你？」周癞子懵，酒瓶晃，「你……你护绝户？」

「记我名下。」里正板脸，「醉砸民居，明日祠堂问。你赖管事罩不住村里门板。门板是泥岗的，不是宗门的。」

周癞子酒醒一半，骂骂咧咧退去，退去时还踢飞一块碎瓦。门槛湿着，韩泥用破布擦干，擦时手上用力——用力把那三文少交的税按进心里：克扣是刀，刀落下也有痕。痕在，就不算白挨。

夜里，马春花还在敲盆：「少三文也是反！反了就该抄屋！抄屋看他还神气！」

张麻婆扫帚远远杵地：「你抄啊！抄完谁给你补屋顶？补屋顶的手抖了，雨漏进你锅里！」

有人笑，有人不敢笑。韩泥听着，不开口。开口像逞。逞了，明日税更狠。更狠也不是不能抗，只是今日先把小胜捂住。

他回屋，把告示内容默念一遍，念完对坛沿说：「连坐找我。克扣，我也记下。记下不是为吵，是为活。」

默念时，他把「从严」两字掰开看：严可以严在秤上，不能严成明抢。明抢了，村里人看在眼里，日子也会寒。寒了，里正那声咳才有地方落。

坛腹微温一线。温得像提醒：忍，不是忘。忘了，刀口就白挨。白挨一次，第二次更狠——所以要把第一次的痕按住。

他把少交的三文在心里翻过：不是赚，是没被多吃。没被多吃，算小胜；小胜要捂着，别拿到村道上喊。喊了，明日十三文就变成十五文，抠得更深。

门口湿痕干成白印，白印像一条浅疤。疤不哭，疤会提醒：浑水泼过，人也还站着。站着，十日税才交得响。响一响，心里才不慌。不慌，才忍得住。忍得住，才活得过。

第二日集上，税吏见他还来，眼瞪圆：「你还有货？」

「有。」韩泥放下筐，「人还喘气，货就还在。」

税吏仍想加码到十三。韩泥仍交十文。税吏脖子红，刚要拍案，里正影子在远处一晃，拍案落不下去，只好收钱，收得像咬牙。

「你仗谁？」税吏咬牙。

「仗秤。」韩泥说，「秤在，人就不跪空话。空话能贴门板，定不了斤两。」

买方多看他两眼，小声：「绝户翁今儿硬。」

硬不是脾气，是饭还得吃。他摸袄底的钱，薄，却够。够就活。活着，才有往后。

回村，沈枯芽偷偷跟到井台后头：「韩爷爷，告示……会不会把娘抓走？」

「抓我。」韩泥说，「你娘病着，抓不住。你少出门，粥别再送。馒也别伸手讨——我塞你吃，别你来要。」

「那馒头……我吃了。」芽儿抹眼，「你会不会……被罚银？」

「会。」韩泥不骗他，「罚银在外，日子在内。日子忍着，银慢慢凑。凑不够，命先顶着。」

芽儿使劲点头，点得像要把害怕点碎。

夜里周癞子又路过，没砸门，只隔墙啐一声。啐完自己也矮了半截——里正「记我」还压着。压着就好，压着村里门板才像门板。

韩泥躺下，棉袄扣严，回想今日：告示压人，连坐吓人，税要十三只交十。小胜像针尖，扎一下，提醒他还活着。活着就把克扣二字按进枕边。

陈姑敲门，只递一句：「别为三文去硬碰硬。硬够用就行。」

「够用了。」韩泥说。

「那就忍着活。」陈姑道，「活到他们眼神发虚。」

他「嗯」一声。门外脚步远了。

门外风把祠堂方向的纸边吹得猎猎响，像告示还在夜里示威。示威归示威，门槛上的湿痕已经干了。干了不是忘，是他把湿换成记。

「克扣记下了，」他对黑屋说，「忍着活。」

忍着活，不是认栽。是把仇苗埋深一点，等它长成能还的枝。枝不急着冒，根要先扎稳——扎在秤上，也扎在「坐我」两个字上。

明日他还出村，规矩还在秤上；秤上能站住，人就不先倒。先不倒，告示那张纸，就咬不满他整个人。咬不满，他就还有气把筐编圆，把十文税交得响。
