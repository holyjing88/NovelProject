# 第五十六章 双妖帅

金丹坠田，泥还没沉，妖雾已起。

灵田东麓，腥风卷过未收的稻茬，雾色由灰转青，像一条活鳞从地里翻出来。青鳞妖君破雾而出——鳞青，眼绿，爪如镰，身高过丈，每一步都把田埂踩成浅坑。非野生妖帅，是赤焰驱策的刀，颈后符纹亮赤，亮得像有人把遥控握在掌里；刀尖直指青岚喉。

厉无殇立高坡，未近，笑里恨与快绞在一起：「韩铁山，一个金丹都守不住，还要本少谷主擦屁股？」

韩铁山面色铁青，旗挥，妖骑再冲，冲得像用妖帅的怒，盖住自己刚才的骇。他不敢再让九月老头近身，只驱妖帅先行，自己在后阵压阵，压得像把最后一枚棋子，押在青鳞身上。

霍镇山盾阵已裂一角，盾面纹路如老树皮，裂口处血渗进来，渗得像阵也要流血。他仍骂：「畜生！往莫长老那边去啊——」骂声洪，洪里却有一线虚，虚得像盾再挨一记，整排弟子都要跪。

盾后弟子有人退，有人跪，有人仍咬牙往前，像半块灵石撑着他们不退。妖帅一爪扫来，爪风带毒，毒里混着焦稻的味，近处三名外门弟子当场呕血，血溅盾面，溅得像阵纹也要红。

柳青鸢剑意再起，却被妖帅灵压压得剑穗不起风。她未退，只把剑尖压低半寸，像等师兄再出那一杖一剑。剑在鞘，意在刃，刃指妖帅颈后赤纹——她亦看出驱策所在，却不抢功，功要留给还礼的人。

莫长春不躲。

杖剑再起，这一次杖与剑合，合得像还礼还到刃上。他不抢柳青鸢的宗主功，不抢霍镇山的镇妖司名，只把剑意送进妖君七寸一线，送进驱策符纹的弱处——秦氏简上写过，顾小满暗线证实过，茶饼换的情报，今夜兑命。

青鳞妖君巨爪扫下，爪风带毒，毒里混着未收稻茬的焦味。莫长春杖点地，身形不动，剑意先至，霜色一线没入符纹，像针扎进赤纹的节骨眼。妖君嘶吼，吼到一半，颈后赤光炸碎，碎成无数细屑，细屑落进泥里，像一场迟来的雪。

剑气余波荡开，妖雾碎，青鳞妖君庞大的身躯僵在半空，僵了一息，断。断处无血，只有妖核滚落，滚进莫长春脚边半尺的泥里，温凉，像一颗未炼的丹，像一颗将来要送的礼。

全场再哗，哗势比斩金丹更甚——妖帅也斩？

弟子声音叠浪：「两连斩！」

「莫长老……还是九月？」

「九月能斩妖帅？驱策符纹都碎……」

「茶饼换的情报，兑的是弱处，不是名！」顾小满从侧径奔至，嗓子哑，却亮，亮得像雨夜那道光还在。

林挽月立人群后，唇白，白得像终于看懂：慕错人，还能改；眼盲，改不了。她望赵师兄方向，赵师兄在丹堂门洞，离得远，却站得直，直得像也在这浪里，改了一寸眼。

厉无殇面色骤变，骂声更厉：「撤！」

赤焰旗转，马蹄乱。韩铁山抱伤属退，退前狠狠瞪莫长春，瞪得像把必还的血仇记在心里——却不敢再上前半步。手帕定身，杖扫妖骑，一剑斩金丹，再斩妖帅，像三连击打在他脸上，打得他恨与惧绞在一起。

魏无涯在远处梁上，扇合，悄然退，像看一枚子落盘，落满，仍未终局。他低声，仅风听：「符纹弱处，茶饼换的。秦商言这一手，比价贵。」

莫长春弯腰拾核，不炫。妖核入手，温凉转烫，烫得像活物最后一口息。他心念一动，妖核没入袖中——袖空，塔中已藏。围观只见袖空，不知核已在塔格；一念收放，万物可藏，这是还礼之外的第二层爽——空袖，意满。

他望厉无殇退处，低声，风送过去，像送一句旧约：

「够活到你后悔——又近一步。」

丹堂方向，地火阵鸣收半档，赵玄机仍立门洞，血透甲，人如钉。甲上刀痕新添，肩塌处用布条勒紧，勒紧里仍站。苏念慈在阵眼，单膝微屈，灵力见底，却笑，笑很轻，像三月前整理衣襟那一下：师兄在灵田斩妖帅，她在丹堂守炉，各算各的戏，各还各的礼。

最后一波赤焰先锋扑门，赵玄机刀卷，卷缺第三口，缺了仍挡。苏念慈袖风送出两枚疗伤丹，丹丸穿门洞，落赵甲上，落得不浪费一颗——抛丹作战，金丹境的基本功，今夜用在守门人身上，像把私怨也炼进火里。

莫长春杖点地，往丹堂走，不快，像老人散步，像把全场哗声留在身后。路过盾阵裂口，霍镇山抱拳，声哑了一度：「莫长老……青岚还在。」

「在不在，看下一夜。」莫长春答，答完已远。

灵田边，弟子扶伤退，退时仍回头看莫长春背影，像看一块旧符，符旧，意满。有人想追上去问，被同伴拉住：「问什么？问九月？问塔？不如问自己的眼。」

周德海在后阵棚下，令旗举起又落，落得像侧门阵闸仍不开。副将急声：「周长老，赤焰已退，开闸包抄残敌——」

周德海淡道：「后阵不稳，妄动者军法从事。」

顾小满冲至，浑身泥血，嗓子哑：「周长老！战时三回求开，闸纹未动。赤焰已退，再不开，残敌走脱，丹堂白守——」

周德海看她，目光平，平里冷：「若丹堂都守不住，开侧门不过是送头。」

顾小满咬牙，转身再奔，奔回灵田，像把「侧门不开」四个字奔成一把刀，递给宗主，递给每一个肯听的人。她奔时册子仍在怀，册子上记了三回求开，记了周长老每一次「不开」。

柳青鸢收剑，剑意未散，望向师兄，只一字：「够。」

莫长春点头：「够一夜。不够一生。礼，还没送完。」

塔壁无大字，塔影却烫，像提醒：小因果今夜兑尽，大因果还在——周德海侧门，厉无殇恨意，天试三月。

韩铁山退至三里外，令旗再挥，挥得无力，像一夜把胆也丢在田里。副将问：「还攻否？」他未答，只望青岚方向，望见丹堂地火仍亮，亮得像告诉他：再攻，还要再丢一颗头。

厉无殇在帐内摔杯，杯碎，碎声像咬碎牙：「周德海不开侧门，韩铁山送妖帅，本少谷主倒成了看客。」看客也不甘，不甘里仍记莫长春三字：还礼。

柳青鸢命人收金丹储物袋与妖帅残鳞，令下平静，平静里是把「两连斩」先记进宗门档，档在，盟会有的辩。霍镇山补盾阵，补得像给东脉再缝一道口；缝完，仍骂赤焰，骂里敬，敬的是莫长老在，青岚还在。

莫长春至丹堂阶下，赵玄机抱拳，未跪，像终于懂：跪是旧妒，抱拳是还。苏念慈出阵眼，炉温未降，降的是她眉间一线紧——紧了一夜，松半寸。

「师兄。」她只二字，二字里丹香与血腥缠在一起。

「守得好。」莫长春答，答完入堂，不入阵眼，不抢戏，像仍做那个敲鼓的人，鼓点落在赵铁牌上，落在苏炉盖上，落在东脉第三夜的收束处。

弟子扶伤，扶里仍传两连斩，传得九月老头的名，与莫长老的名，终于叠在一起。传的人越多，柳青鸢剑越稳——稳得像给围观者看：斩的不是境，是恩；恩在，门在。

识海塔影一荡，一念收塔，杖落掌，他念：「帅狂，豫要来。周犹豫，小里见和战。」

妖帅退半步，半步里妖核已入手——入手之前，杖剑还在袖里候着。
