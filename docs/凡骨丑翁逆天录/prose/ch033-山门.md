# 第三十三章 山门

卯时雾厚。沉丹宗山门立在雾上头，石阶一层层青苔，滑不过人心滑。人心滑，韩泥见过——泥岗井台见过，祠堂阶上见过。他走侧门。侧门窄，窄给杂役，也窄给凡骨。

守门管事瞥木牌：「丙九？韩泥？」

他点头。

「抬棺的？」

「抬过。」

管事鼻下一哼，像不信，又懒得查：「进去。兽栏、药渣、劈柴，哪叫去哪。别闯内门，闯了，打断腿。」

韩泥「嗯」一声，迈进去。山门内丹香扑面，浓得像能把泥岗霉气洗掉半寸——洗不全，够活。道旁弟子衣裳鲜亮，瞥他一眼便绕远。绕远他习惯。习惯里他望见远处丹塔影，高得像还不起的恩——恩要还，塔要爬，爬不急，急便摔。

侧门石阶旁堆着废筐，筐破得像泥岗村西头。西头他走了五十八年，才识得：破筐能补，破命也能活。破骨呢？先活着看。

有人喝：「杂役！避让！」

韩泥贴墙让半步——够活，够看，够记。山门进了，腿没断，谣没跟进来；跟进来的是他自己，和篓底那只坛。

管事领他到丙九杂役房。房小，四床，乱里带着霉。他选床角，角暗，暗好藏坛。坛从篓取出，破席盖严，像泥岗旧习。

邻床少年嫌：「晦气东西别摆明面。克修炼。」

「我盖。」韩泥不辩克不克，只道这一句。

盖住的像是泥岗，像是五十八年，像是「晦气翁」三字。三字还在门外舌尖，门里却换了称呼：丑杂役。名换，人不换——还是记恩、记辱、记仇那一个。

入门册上还有一行小字：凡骨试录，三年为期。期像绳子，拴着杂役日子，拴着将来的测灵，拴着还恩的长路。路长，先活到能干活。

午后分活，管事掷单：兽栏清粪，药渣分拣，劈柴二十捆。韩泥接了兽栏。粪臭像泥岗井台脏水，臭不过要命。命在这臭里，先活下来。

清粪时独眼眯着，看见栏角一滩旧血，旧得像有人伤过灵兽，又像故意留下吓人。他不碰血，只把地面清净到见石。石凉，人醒。栏深处小兽抬眼，眼怯得像沈枯芽。他投半把草过去，草落，小兽啃——啃就是活。他懂这个，所以清粪不图谢，图手稳。

管事路过，瞥一眼：「老丑，手稳。」

三字，像庙祝那句「手稳」。韩泥应声，不谢。谢轻，记重。

清粪毕，同组杂役歇在栏外。有人嫌：「丙九老丑，分活总多。」

「多，手稳。」韩泥只答，「稳，才有饭。」

另一人问：「你抬棺真不抖？」

「抖一次，掉一次。」韩泥说，「掉一次，记一次。记多了，滚。」

两人噤声，像被「滚」字堵住。管事再经，扔一把草镞：「栏角别留血。血留，灵兽躁。」

韩泥点头，把地面再净一遍。

午后劈柴二十捆。斧沉，肩驼，斧落却匀——匀得像泥岗编筐勒藤条，一下一下，不飘。同批少年劈到第十捆，手抖，柴飞半尺。管事喝：「抖！滚！」

韩泥不笑。他劈完二十捆，捆齐，齐得管事不用数。管事又瞥他：「老丑，斧也稳。」

他把「手稳」二字再记深一点：记侧门，记丙九末席。坛腹微温一丝，应丹香，也应这夸。

夜，兽栏远吠。韩泥回房擦手，手糙里带腥。腥洗不尽，心却要净——心净了才能熬，熬得住才能慢慢摸丹，摸得着才谈得上还叶。他低声道：「山门进了。下一步，活。活到能还的那日。」

二更后邻床少年梦呓醒来，看见他还盘坐：「你不睡？」

「睡不实。」韩泥说，「睡实了，像认输。」

「怕什么？」

「怕恩断。」韩泥独眼平，「恩不断，命才肯活。活，才有还。」

少年不懂，翻身又睡。韩泥摸丙九木牌，牌凉得像榜末名。末不是绝，是缝。外头更鼓三响，沉得像瓮在睡，睡里有脚步，一步步往瓮醒走近。他对坛低声：「山门进了，兽栏清了。」

坛不应字，只沿口一线水汽，应完又敛。敛像睡，睡像等——等醒，等还。

三更管事巡房，敲柱：「丙九，明日药渣分拣。手别抖。」

「不抖。」

管事瞥席下陶色：「坛？」

「祖辈传的。」

「别摆明面。」管事鼻哼，「晦气杂役，少惹眼。」

韩泥低声应，不辩晦气。辩费舌；舌要留来分拣，留来活，留来还。管事走了，邻床少年又悄问：「你真带坛？」

「带。」韩泥说，「坛在，根在。根在，人不在绝路。」

「赖福说扔。」

「扔不得。」韩泥独眼平，「扔坛像扔命。命扔不得。」

少年不懂，又睡。韩泥按坛沿——一线水汽应「侧门」，应「手稳」，应完又敛。

管事门外贴条：「入门第七日，凡骨测骨登记。丙九末排，韩泥到。误者，滚。」

七日测骨——比瓮醒近，比还汤远。远不要紧，帖在柱上，手就不能抖。丹香淡下去，兽栏活臭上来；他铲下仍稳。帖在，恩就不能断；恩不断，汤才不凉。

他默念入门册上「三年为期」四字——三年是绳，绳拴杂役，拴将来测灵，拴还恩的路。路长，先活三年。三年里手稳一日，多活一日；多活一日，多记一日恩。恩记满了，才还得上热的。

管事门外又贴细条：「药渣分拣，酉时到。鼻笨没事，手别抖。」

酉时渣场——鼻下那条路，明日要走实。走实了，丹香才引得深一寸；深一寸，还汤才近一步。

同批少年又梦呓：「灵根……」

韩泥声低：「灵根在石上，手在铲上。手在，命就没落到末。」

侧门风硬，兽栏臭扑面。臭不要紧，要紧是铲下稳——稳，管事才肯认「手稳」；手稳，七日后那帖才验得过。他把木牌再按实一遍，像把丙九名钉进掌纹。名在末，步先稳。

丑时将近，坛沿一线水汽一闪便灭。他不追问，只按掌。按实了，才配七日后那验。稳像泥岗抬棺，稳像公审末席。末席也是席，席在，饭在；饭在，帖才验得过。

山门丹香又淡一寸，淡不过兽栏活臭。活臭里铲声仍匀。匀着匀着，明日酉时渣场的路便清楚了——路不清也不怕，手别先软。软了，侧门白进；硬着，丙九末席也能站住。

同房另一个少年翻身问：「老丑，你真五十八？兽栏那臭，你闻着不难受？」

「难受。」韩泥如实说，「难受也铲。铲完了才有粥。」

「你就为口粥？」

「粥是眼下。」韩泥道，「还是以后。眼下不断地，以后才接得上。」

少年嘀咕「唠嗑的老头」，又睡了。韩泥不恼。恼费气，气留给明日药渣场。渣场脏，脏里偶有一点香；那一点香，才是凡骨摸丹的缝。缝细，细也得先把手伸进去。

更鼓又闷响一声。他起身去角落洗手，水凉，洗不尽腥，却洗得开一点瞌睡。洗完，把破席角再掖严，坛沿只露出一线，像泥岗夜里盖瓮。盖严不是怕人偷，是怕人嚼舌头——舌头一动，坛就成「晦气玩意」。晦气他不怕，乱嚼坏的是还恩的路。

他低声把今日三件事钉住：侧门进了，手稳落下了，药渣还在后头。后头那一关没过，夜里就不能睡死。雾从窗缝钻进来，钻不进他按着木牌的掌心。掌心有温，温从坛来，也从「还」字来。

还字远，远也得先把明日酉时活干完。干完了，七日测骨的帖才不像吓人；帖不吓人，人才能在丙九末席站住。站住了，山门这脚才算真踏实。他闭眼一息，再睁——睁开，就是药渣场的路。
