# 第二十四章 小满情报

雨后，青岚石道洗得发亮，像刚擦过的剑。

顾小满登门，未跪，立，怀里捧册，册厚，厚得像她这些年在杂役堂攒下的胆。册边磨毛，角起卷，像被掌心的汗浸过许多夜——记比告状难，她记了三年，才敢捧到长老门前。

莫长春在书房接册，翻页，一页一页。孙福历年克扣杂役灵石、口粮、例份，条目清楚，旁注日期，有的旁注还画小小半块灵石，像画给自己看，提醒恩从哪来。

「长老不查，小满也会记。」顾小满低声。

莫长春合册，抬眼：「记住比告状难。你做到了。」

顾小满脸红：「小满……怕长老嫌琐事。」

「琐事堆成山，才是宗门根。」莫长春袖空，一念出塔，册抄本落掌，收入隔层，「册子我留抄本，原本你还拿着。哪日堂上需要，你拿得出来，不必跪。」

顾小满点头，像接第二张除魔符。她退到门边，停住，忽然道：「长老，手帕定身那日……小满在人群里。小满不怕魔少，小满怕长老九月不够。」

莫长春笑：「九月不够，够送；不够还，塔会记。」

顾小满不懂塔，却懂「记」字，像记半石，记除魔符，记今夜册子。

她走后，塔影在识海微烫，字成：

**记住难，告状易；忠在此册。**

莫长春望雨停后的云，云缝里露出一线青岚峰影，像根还在天上挂着。

杂役堂，孙福苦役未归，堂里换管事，新管事姓李，说话客气，客气里仍怕——怕莫长老，怕那本册子，怕半块灵石的故事再被翻出来。

弟子私下仍传「莫长老勾引杂役」，传一句，便被顾小满冷眼压回去：「宗主裁过，再传，去执法堂。」

传话者噤声，像谣言终于碰上一块半石，磕碎牙。有人不服，嘟囔：「一个杂役，凭什么压人？」

顾小满把册子往案上一放，册子声不重，却响：「凭这个。你也能记，记完了，再来压我。」

堂里静了。静里，有人第一次觉得杂役堂不只有浆洗与苦役，还有一本能上台面的证。

孙福旧党暗中嘀咕，嘀咕被顾小满听见，她只一句：「孙福在苦役，册子在堂上。谁克扣，谁记在心里。」

嘀咕者噤声，像克扣也怕「记」字。

赵玄机路过，听见，未讽，只拂袖走，袖中已无培元私囊，走得轻了一分。他走过顾小满身侧，停半息，低声：「记得好。」

顾小满一愣，赵已走远，背影硬，像不愿人看见他软了这一句。

林挽月远立门外，瞧见，心口复杂，复杂得像一根线松了，又不敢全松。

夜，东脉方向钟响比往日急。

莫长春立院中，杖点地，像给远方敲节拍。塔壁无新铭，袖中册抄本却沉，沉得像把孙福的克扣、杂役的胆、青岚的根，一并收进袖里。

柳青鸢传讯来，简上二字：「备战。」

莫长春回简二字：「已备。」

简下，他望东脉，稻浪声远，像半块灵石拼回的人间，正在等一场更大的阵。

司库听闻册子，面色变，变得像怕半块灵石的故事再演。赵玄机路过司库，低声：「堂上见。」

夜雨又至，顾小满在杂役堂抄册，抄得像抄自己的胆，抄到指节白，白得像半石又长出一块。

霍镇山闻讯，骂：「孙福那厮，早该记在心里！」弟子围杂役堂，围得像等一场小堂辩，辩的是克扣，不是谣言。

魏无涯檐角观气运，观册子入袖，低笑：「记字，比杀字难。」

顾小满翻开册中未呈之页，夜雨妖讯一栏写得密：某夜东脉稻浪异动，赤焰旗角一闪，狼嗥与小妖嘶交织，距百里不足。她未敢擅传，只记，记得像在等与长老问一句「记真了」。

杂役堂外，弟子踮脚，踮得像等册子上堂——册子未上堂，先上了胆。有人低声：「杂役也敢记？」另一人答：「记字，比杀字难。」

林挽月隔远听见「记得好」三字，未近，只回房，把对赵师兄的慕与对册子的敬拽在同一根弦上，弦紧，不敢全松。

执法堂小吏欲抄册条目送赵家，被柳青鸢令止：「册入宗主殿，赵家若要看，堂上见。」小吏噤声，噤得像克扣遇上了半块石。

丹堂外，苏念慈隔窗望雨，望见莫长春院灯长明，灯亮里像册子又厚一寸。她对炉低语：「报真报假，炉只认药。可药也要人搬，人也要册记。」

册递上后，有人偷抄，抄到一半被顾小满发现，发现不骂，只声平：「抄可以，别改。改了就假。」

赵玄机想验册，册不让验，让验像让私，私验不到，验到妒更亮。

柳青鸢接册抄本，声稳：「情报入宗主殿，不外泄。外传者，与泄密同罪。」

霍镇山声洪：「妖讯既至，战字就不收回！」

苏念慈送汤至廊外，汤温，温得像怕情报进丹堂乱了炉。她低声：「报真报假，丹堂只认药。」

孙福在苦役地面色白，嘀咕「杂役也敢记」，被旁人瞪回去。

新管事姓李，令：「堂里例份，按册记。谁克扣，谁记在心里。」

天未亮，顾小满又在册末添一行空白，留白是给下一声妖讯。她对着空白立半炷香，像等对长老问「记真了」，问到了，才敢递第二份。

莫长春览册抄本时，见夜雨妖讯条下压半片枯叶，叶从东脉来，缘顾小满雨后拾，压在条上，像证。

执法堂门口，两名外门弟子争辨「杂役也敢记」与「记字比杀字难」，争到第三句同时噤声——赵玄机自内出，衣带系紧，未斥，只掷二字：「记去。」掷完，径自往司库，像还库之后，连脚步都肯记实。

柳青鸢收册后，令各脉主事各抄「备战」二字悬堂，悬字不及册厚，却先把心拽紧。霍镇山闻讯，远方马蹄更急，急得像妖讯已在百里外磨刀。

新管事李姓当夜清查例份，查出两处短斤缺两，记在心里册旁，短处画小小半块灵石，像学顾小满笔法。

魏无涯观气运转折，观册入袖那一瞬，扇骨轻敲栏，低笑：「记字难，难才救命。命续着，卷二才开。」

他低声，像钩：

「塔壁金铭又亮一线。下一声钟响，该把人都赶到田上去。」

东脉钟再响两声，像催阵，也像催命——催的不是九月，是根。

宗主殿侧，柳青鸢亲接册抄本，阅毕，只问顾小满一句：「记真了？」顾小满声平：「真。日期旁注，谁克扣谁记在心里，不敢改一字。」柳青鸢点头，册收入殿，不外泄。

赵玄机立在殿外廊，未进，未抢册，只拱手一礼——礼里仍有妒，妒里多了一寸克制：「堂上见。」

孙福在苦役地闻塔壁金铭又亮，面色白，嘀咕「杂役也敢记」，被同伴瞪：「你克扣那些，早该记在心里。」孙福闭嘴，像鞭子终会赶上。

莫长春立院中，袖空，一念出塔，杖点册角，点得像在识海再记在心里。塔鸣一瞬，轻得像翻页。他低声：「报真，集要来。集比报响，响里见人心。」

风雨将至，稻浪声远，像半块灵石拼回的人间，正在等一场更大的阵。

顾小满夜抄册至最后一页，页角画一枚小铃，铃不响，响在心头：铃一响，便是堂上见。她把册抱紧，抱得像抱第二张除魔符。

莫长春收袖，塔鸣又轻一瞬，轻得像册翻页。他望东脉，低声：「报真了。下一声钟，不是催命，是催人齐。」

册上枯叶证妖讯，证真了，卷末集结的鼓，就该响了。
