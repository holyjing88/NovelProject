# 第四十八章 隐匿符

雨后，顾小满登门，未跪，立。

双手捧符，黄纸朱纹，比除魔符更深一重，像把夜色裁进符里——她裁了七夜，指节割破，血染纸边，染得像半块灵石的延续，染得像杂役堂里那些年的苦，终于肯换一种还法。

「长老，小满只能做到这些。」

莫长春接符，符温，温得像活命。他不看符纹先看她：「眼睛还亮？」

「亮了。」顾小满道，「雨夜那道金光之后，小满看得更清——赤焰探子换岗，田埂火把，侧径空隙，都进眼里，像地图展开。」

「地图展开，」莫长春点头，「比杀一人有用。」

「可小满……」她摇头，「比起半石、除魔符，这符太轻。」

「轻？」他笑，「宝品隐匿符，裁纸如裁命。你以血裁，还叫轻？」

顾小满怔。她不懂宝品不宝品，只懂长老递半石时，她跪过；递除魔符时，她抖过；今日捧符，她竟敢不跪，像被那道金光洗直了腰。

莫长春袖空一收，一念出塔，符入袖中隔层，与簿相贴，簿一烫，烫得像因果涨了一截，涨得沉，却不吓人。万物可藏，藏得住符，也藏得住将成的网。

「够了。」他道，「礼不在重，在心真不真。心真，簿才记。」

顾小满抬眼，忽然道：「小满能走暗线了。赤焰探子，小满能跟；周长老侧门不开，小满也能报。」

「跟可以。」莫长春道，「别跟丢命。暗线走侧面，别走正面。正面是霍老头和宗主的戏，侧面才是杂役的戏。」

她点头，泪落，笑也落：「小满记恩。」

门外，赵玄机路过，停半息，听「宝品」二字，指节叩铁牌，一声。他没进门，只拂袖走，走前目光扫过顾小满背影——妒吗？少一点了；怕吗？多一点了：怕九月老头连杂役都能养成暗线，怕自己的心还在私囊里。

林挽月远立廊角，唇动，像想说对不起，终未近。她腕上还有罚役的灰，灰像三月之罚已提前上身。

莫长春关门，塔壁浮字：

**隐匿有时，情报成形。**

他念，像延迟层记在心里：「小因果够了。大因果——夜袭子时，看这张符能不能送命回来。」

廊下，弟子窃语：「顾小满献宝品符……杂役也能回赠？」

一人道：「七夜血染纸边，也叫轻？」另一人答：「轻的是杂役的身分，不是符的纹。」

赵玄机拂袖走远，走远两步，却停，停得像终于听懂——懂的是暗线在侧面，妒在正面，正面留给他自己咽。

周德海在长老院听闻，面色平，平里冷。他低声：「杂役能裁宝品符？裁得了，也得看谁肯让她进门报。」

苏念慈在丹堂听讯，炉火旺，旺得像替情报网也担。她低声：「符轻，恩重。重的是心真。」

霍镇山闻讯，骂声洪：「好！暗线有符，丹堂才稳！」

顾小满回杂役堂，把符贴身藏好，藏得像藏半块灵石——恩在掌心，还不了嘴，还得了命。她把七夜裁符的时辰、用料、纹路，一条条补进册子，补得像补自己的胆。

她裁符那七夜，杂役堂灯未灭。黄纸朱纹，她不懂宝品，只懂血染纸边染得像半石——半石是恩，血是还。还不了嘴，还得了形。

第一夜，裁歪了，纸裂，她咬牙重裁。第二夜，指节割破，血入纹，纹亮一线，亮得像雨夜金光残影。第三夜至第七夜，她一条条记赤焰探子换岗，记完再裁，裁得像把路也裁进符里。

莫长春独坐廊凳上，不坐主位，望东脉方向云低，轻声：「小满以血裁符，裁的是命，不是纸。裁得了，才走得进暗线。」

秦商言飞符至长老院，符上八字：「隐匿符成，网眼将开。」莫长春阅后，袖空一收，一念出塔，符与秦飞符并放，并放得像并两笔恩心。

魏无涯在檐角，扇合，低笑，像看网将成。他自语：「符比眼轻，轻里藏形。藏得住，夜袭才有人送命回来。」

霍镇山至杂役堂，看顾小满册子，眸光一亮：「七夜血符，纹路准！暗线有符，丹堂才稳！」骂完，又骂周德海：「侧门闭不闭，看时辰，别看私！」

周德海在远处听闻，面色平，平里冷，像早就备好的稿。他低声对心腹：「杂役能裁宝品符，裁的是命。命裁进了符，侧门就得有人闭——闭谁，看夜袭。」

林挽月远立廊角，唇动，像想说对不起，终未近。她看见顾小满捧符出门，看见赵玄机拂袖走远，看见莫长春袖空收符。她忽然懂：回赠不是攀，是还；还不了嘴，还得了形。

苏念慈遣药至杂役堂，药温，温得像替顾小满指节也担。她低声：「血符血符，别传成妖。妖的是赤焰，不是还。」

顾小满把册子又厚一寸，厚得像把「情报网」三字压实。她低声问杂役师兄：「暗线能走吗？」师兄答：「走不走，看长老肯不肯给路。」路在子时，在侧径，在还恩。

赵玄机回思过崖，跪，膝磕石，声哑，像把「宝品」二字也磕进石里。他低声：「暗线在侧面，妒在正面。正面留给我，侧面留给门。」

莫长春接符，先看她指节。指节割破，血痕未全干，干得像七夜未眠。他道：「裁纸如裁命。你以血裁，还叫轻？」顾小满摇头，摇得像终于敢不跪。

符入袖，一念出塔，再归袖中隔层，与簿相贴。簿一烫，烫得像因果涨了一截。他低声：「够了。礼不在重，在心真不真。」

赵玄机路过门外，听「宝品」二字，指节叩铁牌，一声。他没进门，只拂袖走，走前目光扫过顾小满背影。妒吗？少一点了。怕吗？多一点了：怕九月老头连杂役都能养成暗线。

林挽月远立廊角，唇动，像想说对不起，终未近。她看见顾小满捧符出门，看见血染纸边，看见莫长春袖空收符。她忽然懂：回赠不是攀，是还；还不了嘴，还得了形。形有了，证才肯开口。

霍镇山至长老院，骂声洪：「好！暗线有符，丹堂才稳！谁再笑杂役，先过我这一关！」骂完，又看东脉：「符有了，网有了，夜袭才像试炼，不像乱。」

周德海在远处观望，观得像观一门「私」字还能藏多久。他低声：「杂役能裁宝品符，裁的是命。命裁进了符，侧门就得有人闭——闭谁，看夜袭。」

苏念慈遣药至杂役堂，药温，温得像替顾小满指节也担。她低声：「血符血符，别传成妖。妖的是赤焰，不是还。还了形，恩才长。」

顾小满回杂役堂，把七夜裁符的时辰、用料、纹路，一条条补进册子。册厚一寸，厚得像把「情报网」三字压实。她低声：「小满记恩。」

莫长春望东脉，望得像望符将递出的那一息。识海塔影一荡，一念收塔，杖落掌，他念：「符完，戈要来。挽月反戈，响里见心。」

风停雨住，东脉方向云低，像夜还未来，牙已露。隐匿符入了袖，情报成形——成形了，挽月反戈才肯在堂上把脏字翻过来。网眼小，容得下报信，容不下英雄。

大因果还在夜袭子时，像妖骑，像杖剑——隐匿符只是小回赠，挽月反戈才是大还礼的前奏。

符在袖，形在暗，暗线将走——走的不是正面，是子时那一息。

弟子渐散，散里仍回头瞧一眼灰袍袖空——瞧得像瞧一眼九月还能走多久。
