# 第六章 霜华回赠

三日后，柳青鸢登门，未带随从，只捧一只长匣。

匣开，剑鸣如霜，一室生寒——不是杀意，是洗过的锋芒，像冬晨第一口空气。青霜剑卧在绸上，剑身比旧剑窄一分，剑穗新系，色如柳芽。

「师兄赠的旧剑，」柳青鸢道，「我炼进本命了。这一柄，师兄拄杖也能防身。」

莫长春不急着接：「本命炼进去，还回赠？」

「回赠不是还债。」她学他口吻，罕见地笑，「是你教我的。」

莫长春接剑，剑入掌，寒意沿脉而走，却不伤，像柳青鸢刻意留下的剑意护体。他拄杖试了一式，杖剑并提，动作老，气势却不老。

「好剑。」他道。

「好师兄。」她低声，又立刻补一句，「好长老。」

二人对坐小院，试剑不过三招，招招慢，像怕惊动九月。柳青鸢收剑时，目光落他袖上：「簿……又记了？」

「记了。」莫长春道，「记的是你会还。」

柳青鸢起身告辞，走到门边，停：「苏师姐今日仍关炉。」

莫长春嗯了一声：「让她关。关一日，省一日药渣。」

「师兄知道她在妒？」

「知道。」莫长春望丹堂，「妒不黑化，便还是火在炉里。」

丹堂石阶上，苏念慈果然立远。

她望见柳青鸢从莫长春小院出来，青衫袖风仍带剑意，像把门外的秋也削凉了一寸。苏念慈对身边弟子道：「回丹堂，开火。再关下去，弟子以为首座死了。」

弟子笑，不敢大声。苏念慈不笑，袖中回春丹瓶一紧，像握着一句没说出口的「我也能还」。

她转身时，听见外门有人嚼舌：「宗主又进莫长老院了。」

「九月将死，还谈情？」

「谈剑呢。」

苏念慈指节发白，最终只道：「丹堂做事。」

她进门，炉火一点即旺，旺得像把妒火也炼进去——炼成丹，还是炼成渣，她还没想好，却知道不能再关炉了。

莫长春关门，青霜剑挂壁，杖倚案。

廊外弟子窃语未歇：「宗主回赠青霜剑，连环戏啊。」

赵玄机在石道旁冷听，指节叩铁牌，一声比一声轻——他妒，却开始疑：疑自己看不透九月老头。

林挽月路过，低声：「赵师兄，门规抄完了。」

赵玄机看她，沉默，像吞下一句话，话里曾有妒，如今剩疑。

外门，弟子把「宗主回赠青霜剑」讲成戏本，添油加醋，添到莫长春成了靠女人上位的老头。

莫长春关门，青霜剑挂壁，杖倚案，簿沉了一分，像人情账上又添一笔未兑的暖。

他念：「剑还了，债还在。好。」

窗外，柳氏剑意远，苏氏丹香近，都不浓，都够九月里慢慢品。

他望剑，低声：「霜华在壁，旧剑在宗主手。下一回赠，该轮到药渣了。」

杖答：等。

风过，像把剑鸣送进丹堂，送进外门嚼舌的人耳里——大因果还在远处，小回赠却已在路上，一步比一步近。

廊外，弟子把「宗主回赠青霜剑」讲成戏本，添到莫长春成了靠女人上位的老头。有人添一句：「霜华剑在壁，旧剑在宗主手，连环戏。」

苏念慈在丹堂听见，炉火一跳，跳得像妒里掺敬。她低声：「剑不剑，别传。传多了，传成辱。」弟子噤声，却忍不住偷瞧丹堂方向——瞧首座会不会再关炉。

莫长春在廊柱旁又坐半炷香，不进门，像给回赠留一点门外的礼。弟子路过，窃语：「宗主回赠青霜剑，莫长老又坐门外——老不正经？」莫长春笑，不辩，只抚杖。

柳青鸢走后，莫长春在廊柱旁又坐半炷香，不进门，像给回赠留一点门外的礼。弟子路过，窃语：「宗主回赠青霜剑，莫长老又坐门外——老不正经？」莫长春笑，不辩，只抚杖，抚得像抚一条还礼的路。

外门，弟子把「宗主回赠青霜剑」讲成七种版本，版本里总有「老不正经」四字。苏念慈在丹堂听见，炉火一跳，跳得像妒里掺敬。她低声对弟子：「剑不剑，别传。传多了，传成情。」弟子噤声，苏念慈望小院方向，袖中回春丹瓶一紧，像握着一句没说出口的「我也能还」。

他念：「剑还了，债还在。下一份，半块灵石该进杂役堂了。」杖中竹纹游动，气音仍是一字：等。大因果还在远处，像妖潮，像赤焰——霜华回赠只是小回赠，药渣之心才是大还礼的前奏。

廊下弟子窃语：「宗主回赠青霜剑，莫长老挂壁不佩——是弱，还是礼？」苏念慈在丹堂听见，炉火一跳，低声：「别传。传多了，传成辱。」
