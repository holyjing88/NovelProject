# 第一百零一章 兽潮备

天亮。

兽未至，备先至。备至在栏，在窑，在袖。

韩泥白日清栏加趟，栏净，净不要紧；净，少瘴入。入夜仍守东角，角湿草换两遍，两遍不要紧；两遍，才挡甜腥。

铁无言送石灰一包：「撒角，别撒眼。」

韩泥撒，撒匀，匀像编筐勒藤——勒匀，才稳。

第二备，在废窑。丑时雨，他上山，不为试，为炼备——备清瘴丸料。料从渣来，来路丑，丑不要紧；丑，才匿。

经页有「清瘴」残诀，诀浅，浅而真。真，他投青芯两撮，褐渣一把，加石灰半指——指少，少才祛邪，不伤人。

炉温起，起半寸，半寸稳。稳里，无丹，只有一团灰香——灰香正，正像能清甜毒。

灰香入囊，囊三只，囊缝丑。丑，才像杂役随身药。

赖福查夜禁：「谁上山？」

韩泥立队：「倒渣。筐满。」

管事弟子验筐，验完，放行。放行不要紧，灰香在袖，袖扎紧。

第三备，在心。心记：兽至，先清瘴，再清栏；清栏，不逃。

刘婆补肘布，布厚：「备够了？」

「够了。」韩泥说，「手稳。」

沈枯芽送水，水凉：「叶姐说……别逞能。」

韩泥接水，不接话。话在还，不在嘴。

白织月立远，声冷：「丑杂役也备？备了别拖后腿。」

韩泥只答：「不拖。」

不拖，是证；证在108，在远。

夜，兽吼又近。近里，栏中小兽齐嘶，嘶像潮头。潮头不要紧，头过，才是身。

他东角立，蒙条，袖囊在，经页在，掌纹在。在，就不滚。

丑时，静半刻。香落齐，灰香在鼻下一试，试而正。正，够备，不够战——战在至，不在备。

他低声道：「备齐了。下一更，潮近。」

坛腹温，贴掌，像应「备」字。备，不是退；退字不在他册。

更鼓尽，柱上条又换，换字更急：「明日黄昏，兽潮或至。」

或至，就当至。至，手在。

韩泥白日清栏加趟，栏净，少瘴入。湿草换两遍，才挡甜腥。

铁无言送石灰：「撒角，别撒眼。」

韩泥撒匀，匀像编筐勒藤。

丑时雨，他上山备清瘴料。经页残诀浅，浅而真。

管事验筐，验完，放行。灰香在袖，袖扎紧。

刘婆补肘布：「备够了？」

柱上条换：「明日黄昏，兽潮或至。」

废窑里，灰香成时，他嗅半息，半息够辨甜毒退不退。退半寸，够；不够，再加石灰半指。半指，少才祛邪，不伤人。伤人，像邪；邪了，秦执事来。

回栏，赖扒皮查夜禁，查完，没上山追。没追不要紧，灰香在袖，袖在，备在。备在，102近，103压，104至——至，手在，铁无言在。

备第二日，韩泥白日清栏加趟，栏净，净不要紧；净，少瘴入。

废窑丑时雨，他上山，不为试，为炼备——备清瘴丸料。

管事记簿：「丙九韩泥，备障仍清细。」记细，是小胜；胜在活，不在嘴。
