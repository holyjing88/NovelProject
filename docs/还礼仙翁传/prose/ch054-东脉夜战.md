# 第五十四章 东脉夜战

子时，青岚峰的钟只响一声，像被人掐住喉咙——随后，东脉灵田那边杀声炸了。

火光冲天，赤焰先锋踏田而入，灵稻成灰，灰里混着血。韩铁山立旗冷笑，旗红映甲，像把七日之期提前点着：「青岚宗主，今夜之后，东脉改姓赤！」

柳青鸢剑出，青霜色洗夜，洗出一道冷路：「田在，姓在。人散，姓才改。」

霍镇山盾阵横移，声洪得像雷滚进田里：「镇妖司在！东脉根在！」盾面扛住第一波冲击，弟子惨呼，血溅田埂，血被脚碾进泥——泥仍算东脉的根，根在，战就有意义。

同一时刻，丹堂方向地火阵鸣，淡红阵纹撑起半座堂。苏念慈立阵眼，金丹灵力灌入地砖，炉不熄。赵玄机率执法堂二十人杀到侧翼，刀卷血，嗓子哑了仍清：「苏首座，赵玄机今夜守门！」

「守。」苏念慈头也不回，袖风送出三颗疗伤丹，丹丸穿门洞，落盾后，落倒地处，落得不浪费一颗——抛丹作战，金丹境的基本功，今夜头一回用在活人身上。

莫长春坐丹堂门外石阶，杖点节拍，不进阵。进阵，阵眼易乱；不乱，也抢苏念慈的戏。他像给这场夜战敲鼓，鼓点不快，却稳。袖空，霜在塔中，一念可出，不出，也像还礼——还的是门，不是名。

后阵指挥棚下，周德海令旗举起，又落。副将急声：「周长老，侧门阵闸已开，只等你令——包抄赤焰侧翼，丹堂可解！」

「不开。」周德海目光阴沉，「防中伏。后阵不稳，妄动者军法从事。」

顾小满从侧径冲至后阵边缘，浑身泥血，嗓子哑：「周长老！赤焰绕田埂，再不开闸，丹堂阵眼必破——」

周德海看她一眼，淡道：「杂役弟子，回你的路。」

顾小满咬牙，转身再奔，奔回灵田，像要把「侧门不开」四个字奔成一把刀，递给宗主，递给每一个肯听的人。柳青鸢在高台听见，眸光更冷，令旗微抬，仍未直落周棚——战未半，称忠太早；救门要紧，门在，忠才称得准。

灵田中央，灵压暴起。赤焰金丹修士破阵一角，护体灵光如炉盖掀开，压得弟子跪地，压得柳青鸢剑意一滞——

韩铁山远观，笑深：「青岚门底牌，不过如此。」

弟子群情骇然，有人退，有人跪，有人仍咬牙往前，像半块灵石撑着他们不退。退一寸，田就失；失一寸，丹堂先危。

莫长春站起身。

不快，像老人散步；却准，杖点地，袖中青霜剑鸣，剑未全出，剑意先出，霜落刃上，像秋晨的露。灰袍素，须眉白，在火光里像一张旧符，贴在最要命处。

「将死之人……」韩铁山瞳孔一缩，「也配出剑？」

全场一寂。寂里，柳青鸢剑意复起，像师兄替她挣回一息。霍镇山盾阵再挪半步，骂声又洪：「莫长老在，青岚还在！」

赵玄机在丹堂门洞听见，刀锋更硬，硬得像终于把「心在谁那边」刻进铁里。他低声，仅自己：「恩还一半，剩一半，看门。」

莫长春拄杖前行，一步，两步，像把九月走成剑距。风腥，田燃，杀声未绝——下一息，剑要落。塔影在识海沉，沉得像七十五斤人情，同时了结恩仇。

韩铁山远观，笑深，笑里却有一线骇——九月老头杖点节拍，像给这场夜战敲鼓，鼓点不快，却稳，稳得让人不信他只有九月。他忽然明白：辱九月，辱的是名；出剑，出的是门。门在，名就不散。

莫长春望丹堂方向，地火阵仍亮，仍响，像苏念慈在阵眼里把金丹灵力榨到最后一滴。他低声，仅自己：「苏丫头守得住。赵小子……该还人情了。」

大因果还在下一息，像杖剑出，像全场寂然——夜战只是小开幕，一剑斩金丹才是大还礼的前奏。小开幕里，三线并燃：灵田燃，丹堂鸣，侧径耳还在。耳在，周德海闭闸才更像刀背，不像忠。

田埂上有人屏息，屏息里问：「将死之人，也配出剑？」答：「配不配，下一息见。」答完，全场更寂，寂得像等霜落。

赤焰金丹修士再进半寸，灵光压得更重，压得霍镇山盾阵吱呀作响。柳青鸢剑意再滞，滞得像被一只无形的手按住腕——按住的不是剑，是东脉的胆。

莫长春杖点第三下，节拍仍稳。稳里，袖中青霜剑鸣更清，清得像秋晨的露将要坠地。他不急，急像演；不急，才像还礼。

苏念慈在丹堂阵眼，地火阵纹又红一线，红得像替灵田也担。她抛丹不停，丹丸落处，倒者能爬，爬者能战——战不是为了名，是为了门还在。

赵玄机在门洞挡下两拨佯攻，刀卷血，血溅门楣，楣上「丹堂」二字未被污尽。污不尽，门就还在；门还在，恩就还能还。

魏无涯在檐角观战，扇合，低笑一声，像看因果满到不得不兑：「出意不出剑，像还礼。还礼还到门前，剑就得见见血。」

顾小满从侧径奔回，报柳青鸢：「侧门仍闭，赤焰绕后！」柳青鸢眸光更冷，令旗微抬，仍未直落周棚——救门要紧，称忠要等战后。霍镇山盾阵再挪，骂声又洪：「莫长老在，青岚还在！」

厉无殇在帐内听报，声哑：「青霜在袖，剑未全出？」他砸杯，杯碎，像砸不碎袖中记忆：手帕三息，杖扫妖骑，如今霜入袖——礼还没送完，恨还没记够。

韩铁山立旗冷笑：「将死之人，也配出剑？」莫长春不答，答像演；不答，才像还礼。他一步再进，进得像把九月走成剑距，距满，霜就要见血。

弟子有人跪，有人退，有人仍往前，像半块灵石撑着。霍镇山骂：「退半步，田就失！莫长老在，青岚还在！」骂声滚进田里，滚得像替跪地的弟子也挣回半寸胆。

识海塔影一荡，莫长春袖空，一念出塔，杖在掌，剑意在刃，声轻：「战半，帅要来。双妖帅，狂里见杖剑。」塔鸣荡完，他未全出剑，只出意——意够了，柳青鸢剑路还能直，赵玄机门还能守，顾小满奔回的四字才不像白奔。

田埂上屏息更寂，寂得像等霜落。下一息，剑要落，帅要至——东脉夜战只是小开幕，三线并燃，燃的是门还在，燃的是恩还能还。

风腥更重，田燃更烈，杀声未绝。柳青鸢剑意复起，复起像师兄替她挣回一息，她剑出，青霜色洗夜：「田在，姓在。人散，姓才改。」

莫长春望韩铁山，望厉无殇帐方向，望周德海后阵棚，一眼扫过，扫得像把三线并燃都铭于塔壁。簿沉，沉得像七十五斤人情，同时了结恩仇。

全宗都在等——等的不是戏，是还礼能不能在门前了结恩仇。下一息，剑落，还是帅至，霜要不要见血，袖空的人，说了算。

赤焰先锋再扑丹堂，赵玄机刀卷血，嗓子哑，仍清：「苏首座，赵玄机今夜守门！」苏念慈不回话，只抛丹，丹丸落盾后，落得不浪费一颗。

莫长春一步再进，进得像把九月走成剑距。韩铁山笑深，笑里骇：「将死之人，也配出剑？」全场更寂，寂得像等霜落——霜未落，意先至，至在最要命处。风腥，田燃，杀声未绝，像双妖帅已在门外，已在下一息。

夜战开幕，火起一线，线那头，杖剑尚未出，出则全场寂。
