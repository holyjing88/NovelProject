## 二、细纲两层结构（已定稿）

| 层级 | 覆盖 | 数量 | 说明 |
|------|------|------|------|
| **章段细纲** | 全书 | **1560 章**（`05`+`16`） | 高密度段合并为 20～90 章簇 |
| **逐章硬锚** | 高密度段 only | **~27 章段 / ~200 章** | `16` **附录E** + `05` §63～130 |

**结论**：**不必全书1560章全拆逐章**。余下低密度段保持章段簇，段内按锚点扩写即可。

## 三、逐章拆分规则（后续迭代沿用）

| 条件 | 处理 |
|------|------|
| 章段内 **≥2 个 S/A 丹阵锚**（`22` §一） | **必须拆** → 附录E 增表 |
| 章段内 **≥3 个情感/道侣硬锚**（`19`/`08`） | **必须拆** → 附录E 增表 |
| 章段内仅 **1 个 S** + 日常铺垫 | **保持 10 章段** |
| 连续大战簇（如十二部1431～1510） | **保持章段**，段内按战拍写 |
| 新增锚点导致段内超标 | 先改 `22`/`04`，再补附录E，最后改 `16` 章段指针 |

**禁止**：为「更细」而无规则地全库拆逐章；禁止后移已锁章号。
