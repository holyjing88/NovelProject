# 第一百一十二章 查棚席

赖福查棚，选在丑时。

韩泥早醒，席角压石，石沉，沉得像把坛护在席下。护不要紧，露不得——露了，腌菜坛成「邪器」，邪器，除名。除名，叶汤恩断，芽儿半块灵石也还不上；还不上，丹不正。

门外靴声碎，碎里赖福嗓尖：「丙九漏舍，查席！查晦气，查偷器，查暗修！」

丙九少年慌，慌得像怕连坐。韩泥独眼平，不起身，不起身像睡——睡，才不像心虚。心虚，手抖；手抖，滚字就真。

赖福踹门，门吱呀。吱呀不要紧，他仍躺——躺到靴底近床，近到能闻见赖福身上皂角味，才慢吞吞坐起：「管事查什么？」

「查坛！」赖福眼亮，亮得像逮住，「你屋角那坛，什么坛？」

「腌菜坛。」韩泥声平，「爹传的。腌菜，不腌气。」

赖福掀席角，席下空——空得只剩一块压席石。石凉，坛已挪到邻床底缝，缝窄，窄得像凡人杂物。挪不要紧，挪在丑时前；丑时前挪完，席盖更严。严了，六层才稳；稳了，才接得住下一诬。

「坛呢？」

「灶边。」韩泥指外廊，「腌菜空了，坛洗过。管事要看，去看。」

赖福奔灶边，坛在，坛沿有菜渍，渍旧，旧得像五年杂役。五年杂役，像凡人；凡人，才不像偷器。赖福嗅坛沿，嗅完脸青：「没气。」

「凡人坛，有什么气。」韩泥说，「气在腹，不在坛。」

赖福没法，没法也要留辱：「丑杂役，席盖不严，罚你明日兽栏多铲半刻。」

「铲。」韩泥只一字。

一字，手稳。手稳，赖福找不到滚字——滚字在管事册，不在赖福嘴。管事弟子远处咳：「查完没？查完，歇。明日还要清粪。」

赖福悻悻退。退，韩泥摸邻床底缝，坛腹微温，温得像夸「严」——席严，坛不露；不露，六层才稳。稳了，查席这一夜才记进心里，不进舌。

邻床少年抖声：「你真把坛挪了？」

「坛在，人活。」韩泥低声，「活，别声张。」

少年噤。噤，韩泥从缝摸出坛，挪回角，角稳，心才稳。坛沿一线水汽，贴掌，像爹说：「坛别扔。」扔不得，扔了就无根。

秦霜远木鱼远敲一声，声钝：「心别邪。邪了，鼻下香飘。」

韩泥不答，答在心里——心里七笔分列，不混。仇苗对着赖福，对着查席这一夜，对着钱戾衡眼风。眼风在远，查席在近；近，忍过去，刃才磨亮。

天亮，兽栏多铲半刻。铲下稳，稳得像查席那夜没来过。粪腥灌鼻，灌鼻不要紧，鼻下静——静，才不飘；不飘，气才凝。管事记簿：「韩泥，手稳。」

刘婆端粥，仍多半勺：「查棚？」

查棚席，管事灯照席角，照见厚席不见坛。坛不露不要紧，沿要净。沿净，赖福才掀不了席。

赖福眼细，细得像秤：「席下啥？啥味儿？」「渣味。」韩泥声平，「药山活，活有味。味在角，不在席。」

管事记簿一笔，笔丑，真在「仍末」——仍末，也是名；有名，就不易被滚。滚要手抖，韩泥不抖。

「查了。」韩泥说，「坛在，席严。」

「严就好。」刘婆哼，「手稳，饭稳。饭稳，恩不断。」

韩泥喝粥，稀，热。热顶饥，饥顶一日活。活一日，炼气六层实一日——实了，才接得住下一秤。下一秤，在渣场；渣场，赖福诬告等着。

午后药山废窑，他再盘坐。气走周，周里坛温应，应得像护。护不是躲，是活；活长了，才还恩，才接复测。复测在远，诬告在近；近，手稳，账清，就不滚。

白织月廊下过，袖风带香，香冷：「丑杂役，也配坐窑？」

韩泥不辩。辩，费舌；舌要留来守温，留来按炉。炉在远，窑在近；近，够炼渣，够嗅诀，够把末席熬成手稳。手稳，白织月袖风也尖不过「正」字——「正」在远，先炼出来。

赖福傍晚又晃木牌：「查棚无果，明日查筐。筐里若藏渣，除名。」

「筐编给宗门，不藏私。」韩泥说。

「私在腹，谁知道。」赖福阴笑。

韩泥不接笑。接笑，像认晦气；认不得，只认手稳。手稳，筐空，腹有气——气不露，露了，赖福笑就真。

他回棚，编筐至沿齐。藤条浸软，绕两圈勒紧，指节敲底，声闷，闷而实。实了，赖福查也查不出私——私在坛，坛在角，角在席严后。席严，是丑翁本事；本事在忍，忍在活，活在还芽儿半块灵石前头。

丑时，他练鼻下静。香一字不落，落齐，席严这一夜才记进心里。记进心里，不进舌——舌响，赖福又来。又来不要紧，手稳，坛在，席严，就不滚。

他按玉牌，牌凉：「三年约，记着。赖福查席，也记着。」

记着，刃才磨亮。亮在还恩那日，不在今夜。今夜，只席严，只坛在，只手在。手在，路在；路在，才接得住113诬告那一压。

外头更鼓沉，沉得像顶针顶心。顶一下，又一下，顶出明日查筐——查筐不要紧，筐实，手稳，查不出晦气。查不出，赖福就要换招；换招，诬告药渣场等着。

坛腹微温，应「严」字。严，是丑翁的本事；本事在忍，忍在活，活在还芽儿半块灵石前头。前头路长，席严一步，路稳一步。一步稳，才配炼气七层，才配坊市尾摊潜行。查棚次日，赖福果然查筐。筐空，只有编筐藤屑，屑旧，旧得像五年杂役。韩泥立侧，独眼平，让赖福翻——翻三遍，三遍无渣无符，无偷藏。赖福阴笑：「筐空，腹不空。」韩泥不接，只捧筐交差。交差，手稳，管事记「筐实」一笔。

午后兽栏，他多铲半刻粪。半刻不要紧，手稳才要紧。铲下稳，稳得像昨夜席严——严了，坛不露；不露，赖福嗅不出气。管事路过，瞥掌茧：「丙九韩泥，查棚后仍清细。」

刘婆第二趟端姜汤，姜薄，热得钝：「席严了？」「严了。老婆子粥稀，恩不稀。」韩泥喝尽，记——记刘婆恩，记叶丫头针脚，分列，不混。混了，还乱；还乱，六层白实。

铁无言渣场边低语：「赖福盯你，像盯瓮。瓮是凡人坛，别露真温。」韩泥应声：「露不了。露了，除名。」除名，恩断；恩断，叶汤凉，芽儿半块灵石也还不上。

夜，他再挪坛一回，挪得像凡人腌菜。腌菜坛洗过，沿有渍，渍旧，旧得像泥岗带来的。泥岗坛在，韩家不散——不散，才配活宗门五年，才配将来省亲。

丑时三刻，练鼻下静。粟壳香淡，混一丝兽栏腥——腥不要紧，要紧是香一字不落。落齐，查席这一夜才记进心里。记进心里，不进舌。舌响，赖福又来；来不要紧，席严，坛在，手在。

外头赖福骂：「丑杂役，明日仍两折！」韩泥低声道：「在。」在，就不滚。不滚，查席才不虚——虚了，像怕；怕了，手抖；抖了，滚字就真。真了，三年约白接，玉牌白凉。

他望药山方向，方向像113诬告在远。远不要紧，先守席，先守坛，先守六层实。实了，诬告也戳不破账；戳不破，刃又亮半分。

赖福查筐毕，韩泥编筐至沿齐，藤条浸软勒紧，敲底声闷。闷而实，实像席严——严了，坛不露；不露，诬告戳不破。戳不破，路宽半寸。半寸，够接113诬告，够接114公示。公示辱，手不能抖；抖了，滚字就真。不滚，玉牌在怀，三年约在胸，仇苗在刃。刃不亮，先磨；磨在丙九，亮在还恩那日。

他摸玉牌，牌凉，低声道：「三年约，记着。手稳，粒正，试台见。」见字落，坛温一线，像应。应了，路又明一寸——明向还恩，明向不滚，明向刘婆新衣养老钱。

更鼓沉，沉里，韩泥按木牌，牌凉贴胸。112章手在，名在册，坛沿净。舌不留，鼻下静。恩辱分列，不混。活长了，才还得了叶汤，才接得住门缝灯。

刘婆粥仍多半勺，姜仍多一片。片暖，恩不断。恩不断，手才稳；手稳，粒才正；粒正，试台才配说话。说话了，白织月袖风尖不过「正」，钱戾衡影硬不过手。

管事扬声：「外门复测，丹试开！」——开不要紧，手稳；稳了，末席也能验正。
