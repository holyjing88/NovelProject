# 第七十五章 药山捡渣

卯时前，药山边门开一线。边门窄，窄给杂役，不给内门。内门弟子走主道，主道香，香冷；杂役走渣道，渣道苦，苦真。管事弟子验牌：「丙九韩泥，捡渣半日。只捡，不采。采了，记名。」

药山捡渣，坡滑，霜白。霜下碎石响半声，响不要紧，脚轻才要紧。脚重了，界碑那一步就踏错；踏错，贪活药，除名。

筐满半，他鼻下一静，沉腐浮三字不落舌。沉者入左堆，腐者入右堆，假香上浮，指到即剔。剔净，铁无言点头，走了——走，也像认。

赖福立坡下，眼细：「捡渣也捡？捡出私藏，连坐丙九。」韩泥低声道：「只渣，沿净。」沿净，像界碑，像坛不露，像守瓮人这一条规。

韩泥应声：「只捡。」

牌在，名在，路在。赖福立门外，眼细：「药山也给你？末席专捡晦气渣。」

韩泥不辩。辩，费舌；舌要留来分拣，留来守坛。药山坡下，废渣堆三层。三层高，高里灰黑，黑里偶有一点亮——亮不是宝，是未烬的药性。风过，灰起一线。韩泥鼻下静，静到只闻辛淡，不闻辱。辛淡正，他指稳，拣一撮入筐，筐底垫粗布，布丑，不漏。第二层渣堆更腐，腐沉，沉底才真。

真在底，不在浮；浮的少，少也要记——记错了，鼻下就飘。铁无言立坡角，声低：「捡渣别贪。贪，像求。不求，只记。」

韩泥分拣，分出三堆：辛、腐、假香。假香浮，浮不要紧，浮也要剔——剔净了，手才稳。铁无言又取半片枯叶，叶上沾灰：「苦浅，像叶家药。药在远，恩在胸。」

韩泥鼻下一静，静里辨出苦与辛。苦沉，辛浮。浮少，记牢；记牢了，嗅诀才生根。午后，日斜，药山深雾起一线。雾不要紧，他仍只捡边堆——边堆许可，深堆不许；不许，就不进。筐满半寸，半寸不要紧，半寸也是路。路通向废渣炼丹，通向95匿丹，通向还叶丫头那一碗烫。白织月主道过，袖风带香，香冷：「杂役也碰药山？碰了，也炼不出丹。」

韩泥不抬头：「碰渣，不碰丹。丹在远，渣在近。」

白织月唇线绷一息，走了。少一句嘲，小胜。管事验筐，筐底无活草，无整药，只有渣。渣对，记一笔：「丙九韩泥，捡净。」

捡净，是小胜。胜在筐，不在嘴。赖福讽远：「捡渣也捡，末等也末等，三年仍滚。」

韩泥不接茬。接茬，全丙九饿。他低声道：「滚字在柱上，不在筐上。」

筐在，渣在，路在。二层气走指梢，指梢温半分，温像辨渣更细——细了，辛腐才分得清。回棚前，他在药渣场多留半刻，把白日样再辨一遍。样少，少不要紧，要紧是记。记辛，记腐，记假香浮。样纸包实，塞怀。怀暖半分，暖不应符，应「记」。记牢了，药山深行才走得稳。邻床问：「药山累不累？」

「累。」韩泥说，「累也捡。捡了，才有渣练。」

少年不懂「渣练」，却懂「累也捡」，闭嘴睡了。他按席角，席严，坛不露。坛腹温一线，贴掌，像应「捡」字。应完，他合眼不睡实，鼻下仍静一息。

丑时前，他廊下练半刻闭息。息在鼻，不在嗓。嗓响，香飘；香飘，手就滚。

掌纹那路转半周，二层境稳，稳像筐底那撮辛——沉在底，不浮。浮了，像求；不求，只记。他低声对坛：「渣捡了。辛记了。深行在明日，别贪。」

坛不应字，只温。温，像把「渣」字摁进掌纹那路里——路在，就不滚。刘婆端粥，仍多半勺：「药山去了？」

「去了。」韩泥说，「只捡渣。」

「只捡怕什么。」刘婆哼，「手稳，饭稳。饭稳，恩不断。」

韩泥点头。恩在粥里，也在筐底那撮灰里——灰轻，记重；记重了，才接得住深行。秦霜远廊经过，木鱼不敲：「心别邪。邪了，鼻下香飘。」

韩泥只答：「不贪。只捡。」

更鼓沉，沉里，他摸怀木牌，牌凉，筐渣凉，坛温一线。温应活，不应慌——慌像少年，他是记心翁。独眼平，手不抖：「渣捡了。深行在明日。嗅诀在鼻下，不在舌上。」

刃不亮，先记。记完了，才配药山深行，才配白织月再嘲那一日。柱上条角白一线，三年约在怀，愧字在怀，渣筐在角——角藏，不示人。示人，谣又来。坛腹应温，不应字。温，像药山边堆那一点辛，辛在活里。活在，才嗅得到；嗅得到，丹路才开。他低声道：「捡净了。坛应了。明日往深一步，仍只捡。」

仍只捡，不采。采了，记名；记名，恩断。恩不断，才还得了叶汤，才接得住门缝灯。沈枯芽在药圃角弯腰，弯得像没看见。没看见不要紧，韩泥也不近——近，像求；不求，只记。记沈枯芽半块灵石那笔恩，恩在还，不在这一日。散队后，兽栏仍腥。腥不要紧，二层在掌，铲下更稳。稳，像踹台那一掌——掌在，渣在，路在。

管事弟子另记一栏「药山」，笔尖停半息：「捡净——写上。」写上，像把晦气也记成一笔活；活记在册，滚字才不好随便抹。同批少年掩鼻：「药山渣也臭。」

「臭不要紧。」韩泥说，「臭也捡。捡了，鼻下才有根。」

韩泥只答：「渣味。睡吧。」

在，就不滚。不滚，渣才捡得长——捡长了，废渣路才走得通。药山第二趟，坡角风硬，硬里苦辛。韩泥筐沿压布，布厚，厚了灰才不漏。漏了，像求药性；不求，只捡。管事弟子立界碑旁，声沉：「再深半步，除名。」

韩泥应声：「不越。」

不越，是规。规在，手在。手在，就不滚。第二筐满，他歇半息，歇不像懒，像鼻下静。静里，辛沉底，假香浮，浮少也要剔。剔净，铁无言点头：「捡渣如辨性。性在底，不在嘴。」

韩泥不答，答在心里——心里记铁无言这一句，记药山边堆，记二层境配指稳。白织月主道过，裙素，袖香冷，她瞥筐，瞥完，走了。没嘲，小胜。赖福立门外，眼细：「捡渣也捡，末等也末等。」

韩泥不接茬。舌要留来鼻，留来守坛。午后回坡，筐交管事，记「丙九韩泥，药山捡净」。记净，是小胜。刘婆提篮过，篮里饼硬：「药山饿不饿？」

「饿。」韩泥说，「饼就行。」

刘婆塞饼，饼糙：「别谢。记着。药山别贪活药。」

韩泥点头。恩在饼里，也在「别贪」二字里。回棚，样纸包实，塞怀。邻床问：「药山香不香？」

「香在灰里。」韩泥说，「灰在活里。活在，手才稳。」

丑时前，廊下练半刻闭息。低声道：「渣捡了。深行在明日。界碑不越，恩不断。」

更鼓沉，掌纹那路转半周，二层稳。稳了，才接得住药山深行，才配白织月再嘲。

坡下归途，药渣场角风硬。他多留半刻，把样纸并入白日样。样轻，记重；记重了，深行才走得稳。

同批少年掩鼻：「你身上咋有药味？」

「渣味。」韩泥说，「味在灰里，不在身上。」

秦霜远廊过，木鱼轻敲：「捡渣心别贪。贪，鼻下香飘。」

韩泥不答，答在心里——心里不贪，只记。记药山边堆，记二层境配指稳。

韩泥只答：「不恋。只捡。」

坛腹应温，像应「捡」最后一笔。应完，他不叫。不叫，才守得住深行那一日。
