# 第四十四章 误闯

戌时，外门浴房外径，水汽混草木气，暧昧得像一场设计好的误会。

赵玄机在暗处弹指，一线灵息引路，引莫长春至岔口——莫长春本往丹堂送渣，脚步一顿，嗅出不对，像嗅出谣言的味道。灵息里无杀，只有「引」字，引得像戏台拉幕。

他止步，不入门，只立径外三息，杖点地，声轻：「浴房？老夫还没老糊涂。」

转身欲走，林挽月「恰好」路过，惊呼，袖风带起水汽，像一场戏拉到高潮。

暗处，执法堂弟子「恰好」巡至，目光一扫，扫出明日嚼舌的料。弟子眼里亮，亮得像终于逮住九月老头的丑，亮得像赵执事许诺的门规功。

莫长春知是局，却不点破，只叹，像叹给风听：「年轻人，急什么。」

他拄杖离去，背影稳，稳得像谣言砸不碎的一块旧玉。林挽月立原地，指尖凉，凉得像第一次觉出赵师兄的礼，也要还。

围观弟子其实未到——局要的是「恰好」的巡者，不是满堂闹，闹大了，反而不像谣言，像陷害。可远处廊角仍有人影，像顾小满，像魏无涯，像将来要翻的堂。

赵玄机在暗处收指，指节白，白得像妒火终于点着纸，却烧不出预想中的轰响——莫长春止步不入，像把火掐在纸边。

林挽月捏袖，唇抖，终未喊住。她忽然明白：赵师兄要的不是真，是「像真」；莫长老给的不是辩，是「不进门」。

执法堂弟子低声问：「赵执事，这就够了？」

赵玄机淡道：「够了。明日，全宗都会替我们嚼。」

弟子笑，笑里脏，像已看见明日浪。

莫长春出院门，簿无字，袖却沉。廊下有人窃语「浴房」，窃语尚小，小得像浪还没起。

顾小满远影在廊角，像将证。魏无涯扇开一线，像看子将落。

戌时之前，莫长春本往丹堂送药渣，渣在袖，袖空，渣在塔中，一念出塔，再落掌。他嗅灵息不对，停步，不追，不怒，只立径外三息——三息够巡者看，够林挽月惊，不够他进门。进门，像坐实；不进门，像还礼。

林挽月「恰好」路过时，水汽扑脸，扑得像一场设计好的 blush。她惊呼，声却抖，抖里像第一次觉出：赵师兄要的不是她的名节，是长老的丑。丑若成，她成刀；丑若败，她成证。

赵玄机暗处收指，指节白，白里空一线——空的是预想中的轰响。执法堂弟子回堂报「止步三息」，他仍淡道：「够了。」够了，便好。明日，全宗都会替他们嚼。

顾小满在廊角远影，像将证。她不知戌时发生了什么，只知心口那跳，跳得像半块灵石在警。她记下一行：戌时，浴房外径，长老止步，未入门；赵执事引路，林挽月惊呼，执法堂巡至——记比猜有用，猜比传有用。

苏念慈在丹堂值夜，听外径风声，炉火旺，旺得像替林苑也担。她遣弟子去外径探，探回一句：「长老止步三息，未入门。」苏念慈闭眸，低声：「不进门，才是还。」

柳青鸢在宗主殿批文书，听执法堂回禀，笔尖一顿，却不点破，只落笔：「戌时之事，记档，不辩。」——记是真的，辩要等浪起完。

霍镇山自镇妖司闻讯，骂声洪：「浴房局？赵小子若敢借体统咬人，老子先剁他铁牌！」骂完仍列阵，列阵像给明日嚼舌列阵——嚼的是脏，守的是门。

周德海在长老院听闻「止步三息」，抚须不语，像一盆浇了一半的油。他令心腹：「明日若起浪，后阵巡一遍，别让人说长老院见死不救。」

秦商言飞符至杂役堂，符上六字：「止步即还，浪在后。」像商言也懂：局缺高潮，谣言要靠嚼补；嚼够了，堂上才肯翻。

林挽月回林苑，未眠。她反复想那三息——三息里，赵师兄要的是「像真」；三息外，莫长春给的是「不进门」。她忽然觉得，自己若再在堂上作证，证的不是长老，是赵师兄的谋。

赵玄机夜回偏厅，指节叩铁牌，一声比一声轻。心腹问：「若莫长老明日仍不辩……」赵玄机淡道：「不辩，更好。不辩，像心虚；心虚，浪才高。浪高了，堂上才肯翻。」

孙福旧党聚在角隅，低声：「赵执事这回要成。」顾小满自后经过，册子一翻，只记谁去过浴房外径、谁去过执法堂偏厅——记比猜有用，猜比传有用。

苏念慈夜回丹堂，炉仍开，像把妒与担都炼进火里。她低声对弟子：「止步三息，未入门——这六字，比浴房二字更真。真字炼进炉，脏字才退。」

魏无涯夜来传讯，只六字：「局成半，浪将起。」莫长春阅讯，点头，像谢提醒。他自语：「浪起得凶，退得才快。起完了，才好还。」

霍镇山夜巡外门，刀未出鞘，声先出：「谁再传浴房谣，先过我这一关！」一句话，廊静半息，半息里像听见铁牌将要落案——落案不在廊上，在堂上，在证人喉里。

执法堂偏厅，巡者把「止步三息，未入门」六字写入档。赵玄机阅档，指节白，白里却笑——笑里脏，像已看见明日浪。

莫长春回院，茶仍煎，煎得淡。他不辩，不追，只把浴房局当成一笔将落的恩：局成了半截，另半截，让浪自己起。起完了，才好还。

外门膳堂，已有弟子压着嗓议论：「浴房外径……长老真去了？」另一人答：「听说是止步，未入门。」议论轻，轻不过水汽，水汽像谣言已湿，只等日晒。

顾小满夜里把杂役簿又翻一遍，添一行：戌时，浴房外径，止步三息，巡者在。行细，细得像半块石硌深——硌深了，堂上才肯翻。

林挽月林苑独坐，未眠。她反复想那三息——三息里，赵师兄要的是「像真」；三息外，莫长春给的是「不进门」。她忽然懂：证字还压在喉里，时未到，可脏字已在每个人眼里。

弟子日后传此夜，传成七种版本，版本里都有「止步三息」——有版说三息太短，不够定罪；有版说三息太长，够嚼半年；有版说莫长老若真进过门，全宗今日已在骂。七种版本合成一种信：信的不是浴房，信的是止步。

莫长春不辩，杖点地，像点住余音。识海塔影一荡，一念收塔，杖落掌，他念：「误清，浪要来。谣言浪，比阵快。」

风带水气，像谣言已湿，只等日晒。日晒之前，全宗尚不知——止步不入，才是还礼。

赵玄机引路，莫长春止步，反差起——浴房局缺高潮，谣言要靠嚼补。他低声，像钩：「局成了半截。另半截，让浪自己起。起完了，才好还。」

柳青鸢遣执事传话：「师兄，止步不入，门在。」莫长春答：「门在，浪也在。浪起了，才好裁。」

翌日卯时，外门膳堂已沸。赵玄机铁牌在腰，笑浅，笑里像已看见浪——浪将湿，嚼将起，嚼够了，堂上才肯翻。苏念慈遣人送汤至廊外，汤温，温得像怕浪进丹堂乱了炉。霍镇山在营里骂：「浴房？放屁！」骂完仍磨刀，像给明日嚼舌也磨一刀。

止步三息，未入门，门在，浪也在——浪将湿，嚼将起。

弟子渐散，散里仍回头瞧一眼灰袍袖空——瞧得像瞧一眼九月还能走多久。

误闯禁阵那日，阵光起，起得像给全门看一次错。

莫长春袖空，一念出塔，杖点阵角，不破阵，只问：「误谁？」顾小满声平：「误我。我认。」

柳青鸢声稳：「误闯当查，查真了再罚。」赵玄机铁牌在腰，冷声：「误闯禁阵，当罚——」莫长春截话：「罚前先问，谁开的阵门？」

苏念慈闭炉，闭得像怕阵光进丹堂。她低声：「误误别传成陷。」

顾小满册上又记一行，行细，细得像半块石硌手。她低声：「误记下了。」

围观弟子围阵，围阵像怕误传染，传染了就得还。

误闯后，阵光未散，有人传「莫长老设陷」，传者被柳青鸢令查，查像查谣，谣查比阵快。

莫长春不破阵，只问：「误谁？」顾小满声平：「误我。我认。」

赵玄机冷声：「误闯当罚——」莫长春截话：「罚前先问，谁开的阵门？」

苏念慈闭炉，闭像怕阵光进丹堂。顾小满册上又记一行。

莫长春念：「误清，浪要来。谣言浪，比阵快。」

塔鸣一瞬，轻得像阵光落。围观弟子围阵，围像怕误传染——传染了，就得还。
