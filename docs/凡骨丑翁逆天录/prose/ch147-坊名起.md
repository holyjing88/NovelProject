# 第一百四十七章 坊名起

坊市续，续在名，不在荣。

老客递渣，渣正，正像147该续的——续在掌，不在舌。

管事记：「泥纹丑丹师，续留。」

韩泥收条，条丑，字真。

白织月远廊过，没停。没停，小胜——胜在渣正，不在辩。

刘婆哼：「名渐稳，饭也要稳。」

韩泥点头。

疤鬃驴立摊侧，不打响，响不要紧；响，像显摆。

回丙九，他按坛：「坊名起了。记着。」

坊市续，续在泥纹二字后——二字传了，传不要紧，手更要稳；稳了，名才是真名，不是谣。

韩泥立末摊，疤鬃驴拴柱后，柱旧，旧而稳。稳，才像147该过的日——日里辨渣，日里收半文税，日里听风。

第三日，老散修再递渣，渣正，正像144那半句回响：「泥纹丑，效不丑。」

韩泥鼻下静一息，息里辨出辛淡——辛淡正，正了，人留下一块下品灵石，石小，小不要紧，小像证。

证不在荣，在听；听了，147才像名号续，不像一夜风。

管事弟子记一笔，笔丑，真在「续留再续」。再续，够了；够了不在名，在手。没停，小胜——胜在层十七，胜在驴稳，胜在渣正。

铁无言声低：「名起了，坊尾别近。」「不近。」韩泥说，「近，费舌。」

刘婆哼：「名是名，饭是饭。」韩泥喝粥，粥稳，稳像146那层实。

赖福盯梢，盯不要紧，盯在皮，不在手；手稳，盯也白盯。

他低声对坛：「名号续了。记着。」坛不应字，只温。

更鼓沉，收摊，收得像石。石在，牌在，泥纹在，风过，风过不留痕。

续摊第三日，条丑，字真。真了，147才像名号续，不像一夜风。

坊尾盯梢，盯不要紧，盯在皮，不在手；手稳，盯也白盯。

老散修留石，石小，小不要紧，小像证——证泥纹，证手，证十七层也辨得准。

韩泥收摊，收得像石。石在，牌在，泥纹在。

回丙九，疤鬃在柱后，柱稳，稳像147——稳名，不稳嘴。

他按坛。坛腹温一线，像应「续」。

更鼓沉，他望帖：「听风紧。148叶线在前。记着。」

续不在榜，在耳。耳听老散修再递渣，渣正，人留一块下品灵石——石小，小不要紧，小像证，不像赏。赏在远，在大比，在235胜钱；147只听，不接赏。接赏，像显摆；显摆，白织月袖风就来。来不要紧，手稳，渣正，泥纹二字才续得住。续得住，144起的半句才不飘；飘了，145驴白驮，146层白实。

实虚在远，147只续。续法同144：末摊，破筐，牌歪，税半文，管事记簿「续留再续」。手不停，名才真；真，不是丑丹师三字响，是泥纹浅、嗅之正那套重复证。证重复，才像147，不像一夜风。风过不留痕，留痕的是手稳；稳，铁无言说坊尾别近，刘婆说名是名饭是饭，赖福盯梢白盯——盯在皮，不在手。

手稳，147才收；收摊如石，石在，牌在，泥纹在。在，148叶信才接得住；接叶线，不缠情，情在还，不在舌。舌要留来候149周家风——风在远，147只续名。名续完，更鼓沉，按坛，坛腹温一线，像应「续」——续在耳，不在舌；舌若辩，144白起，145白驮。不辩，147真；真，150簇才足。

足在远，147只续半寸——半寸在耳，半寸在掌，半寸在疤鬃驴拴柱那一线稳。

坊市续，泥纹在，名在，手更要稳；稳了，148叶线才接得住。接得住，149风才只听不架。

风过坊市尾，尾巷留半句泥纹丑效不丑——半句在，147才像续，不像一夜风。他收摊：「名号续了。148在前。记着。」石在，牌在，手在，就不滚；不滚，叶线才接得住。续在耳，不在舌。

柱上条角白一线，三年约在怀。怀凉，掌温。

坡下石阶湿，他数息，数帖，数债序。

同棚杂役笑，笑在耳，不在心。

废渣巷日斜，他验渣，渣正，手才静。

侧门驿亭风硬，他听风，风过，记在心里。

散队后，他过外堂阶下，分拣半刻，指稳——指稳，路就不飘。

编筐半刻，藤刺扎掌，刺浅，浅些沿才净。沿净，坛才藏得住。

丑时，练鼻下静。粟壳香淡，香一字不落。

管事记簿，记细，是小胜。胜在忍，不在辩。

他望坛，低声道：「手在，路在。更鼓沉，天边白一线。白里，他按木牌，牌凉贴胸。坊市尾巷风硬，他收摊如石，石在，就不滚。」

验息后，他理怀内册页，页黄，字丑，丑而真。

铁无言声低：「别飘。」韩泥应声：「不飘。」

刘婆端粥，仍多半勺。韩泥点头，谢轻，恩重。

药峰分拣照旧，指稳，鼻下静一息。

帖在怀，怀沉，沉的是序，不是惧。

白织月袖风过，韩泥不辩。辩，费舌；舌要留来守坛。

赖福廊下过，韩泥只答：「在。」在，就不滚。
