# 第九十一章 废窑试炉

冬后第三月，药山风硬得像刮铁皮。

韩泥清完兽栏，掌茧裂口还渗一点潮。潮不算大事。大事是鼻下那套嗅诀已经认得活了——沉、腐、浮三字落齐，废里也能淘出一丝真香。铁无言午时在栏边抱臂，声压得很低：「药山废渣场，午后有人倒炉灰。灰里若见青芯，别当泥踢走。」

「记着。」韩泥应声。

记不是为捡便宜，是为试炉。试炉得去没人争的地方——半腰那口塌了半边的废窑。路绕远些，远也好：远了，赖福那双盯秤的眼不容易跟上来。

他借「送渣」的条子上山。筐满，灰盖面，像平常杂役倒的那些。窑口残留一点旧红，红得发淡，淡得像烧过又凉了多年。灵脉浅，浅才没内门弟子来占；末席杂役占一口凉窑，管事一般懒得问。

窑矮。他驼着背进去，额头几乎顶梁。梁上蛛网旧得发灰，网里卡着半片渣屑。鼻下一动，香淡而正——正香才配试第一炉。

怀里摸出瓮底泥炉。炉小得像碗，是碎陶拼的，拼得却稳。稳了，火才听他的。青芯三撮、褐渣一把、灵泉水半葫芦，按嗅诀里那点「投序」一一落下。火从掌纹那路引出，炉底先温半寸，温得像坛腹贴掌的感觉。

第一炉火猛了。渣焦，苦里带腥。他熄半息，掌按炉壁，烫进茧里，烫像刻字：「记火。别逞。」

第二炉又弱。青芯发不出正香，白气起得稀。他皱眉，改掌上力度，像编筐时最后一匝硬藤——勒紧半寸，路才不断。

第三炉终于匀。渣里浮一线白气，散前鼻下掠过一丝培元路：细，细得像叶汤那日碗沿的热。热在记里，不在炉上。炉上只要稳，先把路摸熟。

山外忽然有巡山声：「杂役别乱钻！钻了记过！」

「送渣。」韩泥应，把筐往前挪一寸，盖住窑口红焰。

管事弟子瞥一眼筐，筐满，满得像常活。常活藏得住秘密。那人骂两句走了，像根本没见窑里闪过的那点亮。

赖福傍晚堵侧门：「上山了？干什么去了？」

「倒渣。」韩泥声平，「筐满，手稳。」

「少二斤青芯。」赖福眼细成缝，「秤过了。」

「看错了。」韩泥说，「青芯在灰里，灰在筐底。」

赖福掀筐，灰扑一脸，青芯却无——真的青芯扎在袖袋里，扎得紧。扎紧了，坛沿才净；沿净了，席下那坛才藏得住人。

夜里席盖严。韩泥按坛，低声道：「试过了。窑在，路在。」

坛腹温一线，贴掌应着。邻床少年眯眼：「你上山，不怕记过？」

「怕。」韩泥说，「怕也试。不试，往后拿什么还人。」

「还谁？」

「你懂不懂都睡。」韩泥道，「明早还要清栏。」

少年哼一声睡了。赖福脚步在廊外停半息，又走。韩泥独眼平，手不抖——窑红已凉，像从没亮过。不叫，才守得住；守得住，下一回火才能更匀。

卯时仍清栏。铲下稳，稳里偶尔拾到褐渣，他都不贪。贪会晃手。铁无言在药渣场角验灰，点头极短：「正。别忙成丹。先记火路。」

「不忙。」韩泥说，「先记。」

刘婆端粥，多半勺：「脸黑成炭？夜里偷火了？」

「倒渣。」韩泥喝粥，「手稳，饭稳。」

「饭稳好。」刘婆哼，「恩不断。你要真偷火，连坐丙九，别把我粥也连坐进去。」

「不偷。」韩泥道，「试的是废窑，不算偷。」

「算不算由你说了不算。」刘婆白他一眼，「管事眼里，看见亮就是祸。祸藏紧点。」

白织月远廊过，袖风一扫：「末席杂役，药山也敢钻？」

「倒渣。」韩泥不抬头，「手稳，不脏内门路。」

她嗤一声走远。嗤得远，试得才更安静。秦霜木鱼从墙外轻敲两下：「试炉别拜邪。拜，路断。」

韩泥心里应：不拜。坛在，人在。拜的是命，不是邪香。

午后又有同批少年掩鼻路过：「韩泥，你怎天天往山上跑？」

「渣多。」韩泥说，「筐沉，多跑两趟。」

「少装。」有人挤眼，「装得像真干活。」

「就是干活。」韩泥应，「干活的人命长。」

管事夜巡灯扫棚角，照见厚席，看不见坛。韩泥编筐半刻，藤刺浅扎掌心——浅疼提醒他：手别飘。飘了，正测那日没滚也白滚。他把木牌贴胸，牌凉，凉的是名；坛温，温的是根。

沈枯芽在药田边探头又缩：「韩……韩叔，叶姐咳轻点了。」

「好。」韩泥只一词。

「你还……还不去见她？」

「不去。」韩泥说，「她说过别来丑。丑能离远一点，恩也能还近一点。」

芽儿似懂非懂，抱着干饼跑了。韩泥把那句「别来丑」按进胸口，按得像按石。按实了，手才敢继续摸火。

药渣场午后，他分拣得更慢一分。慢不是懒，是怕火路散。旁人催：「韩泥，你磨蹭啥？管事要收筐！」

「磨蹭一天，省得焦一炉。」韩泥说，「焦了，灰也骂人。」

催的人噎住，甩袖走了。铁无言远远递一眼，像说：记着就好。韩泥把青芯边角的细末另包，包进袖袋最里层——里层贴牌，牌凉，末香才不外冒。

夜里邻床又问：「你袖里咋总响？」

「藤条。」韩泥应，「编筐剩下的。睡。」

「藤条会香？」

「你鼻子醒着，身子睡。」韩泥道，「醒多了记过。睡。」

少年不再问。韩泥掌贴坛沿，沿温一线，像把第一炉的匀火又温回掌茧。温回了，明日再上山才有底。底不是胆子，是手记得住轻重要紧。

廊外雨丝细，细得像筛灰。他听雨声判断窑壁潮不潮——潮则先温炉半息，半息够把湿气逼走。逼走了，第三炉那种匀才接得住第二回。

赖福远处骂人：「丙九那晦气，上山次数记着！」骂完摔门。摔门也好，摔门说明他还没掀到席下。席下瓮在，瓮在就不慌。

韩泥低声把明日再试那半寸火路按进掌茧。按实了，还人的路才续得上。风过药山，窑口更黑。黑里没有第二个人的脚步。他睡得浅——浅着好，浅着才听得见明日该不该再上山。明日若雨小，炉壁更听话；炉壁听话了，控火才算入门。

翌日清栏，幼兽踩翻食槽，槽里麸皮撒了一地。旁边少年骂：「晦气！又是你扫？」

「我扫。」韩泥蹲下拢麸，「扫干净，牛才不踢人。」

「你上山倒渣，倒得手都黑，还装干净？」

「手黑是灰。」韩泥说，「心别黑。心黑了，栏会臭。」

少年噎住，扔来一把扫帚：「扫你的。」韩泥接帚，帚柄旧，旧得合掌。合掌了他才安心——安心不是怕人骂，是怕手飘。手飘，废窑那点匀火就白记。

午间药渣场起风，灰扑面。铁无言把三堆拨开，指尖一点：「青芯边缘那撮，焦过。焦过也能闻。闻见焦，你就知道火猛到哪一步。」

「闻见了。」韩泥应，「猛在第一炉。第二炉弱，第三炉才匀。」

「匀就收。」铁无言道，「收了别跟人说匀。说匀像夸自己。夸自己，管事当偷火办。」

「不说。」韩泥把灰筐压实，「只记在茧里。」

傍晚编筐，藤条缺一根。小杂役借他半根硬藤：「明天还。」

「不用还藤。」韩泥说，「还饭就行。饭稳，人稳。」

「你说话老像欠谁似的。」

「我是欠。」韩泥声平，「欠的是热。热得还完，才能睡实。」

夜深，他再摸木牌。牌凉贴胸，胸下瓮温一线。一线勾着明日：雨小则再上山，雨大则编筐等。等也算路。路上有窑，窑里有火，火旁有还人的空位——空位今未填，先把火路站稳。站稳了，坡下那碗汤才不至于永远烫在他自己一个人的记忆里。

瓮腹温一线，像应了「试」字。入门不叫人知，只要他自己心里有数：废窑占住了，还人的路就多了一寸。
