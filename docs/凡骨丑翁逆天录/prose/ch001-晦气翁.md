# 第一章 晦气翁

冬前。

五十八岁的韩泥蹲在泥岗村西头最漏的泥屋门口编筐，村里叫他「晦气翁」——麻脸，独眼好的一只常年眯着，另一只蒙着白翳，背驼，腿瘸。村童隔着老远走，声音脆，像石子丢进水洼：

「晦气翁来了。」
「别挨他，晦气。」

一个穿开裆裤的娃捡起土块，想扔，被娘一把揪住耳朵：「晦气沾上了，洗三天都洗不掉！」娃哇一声哭了，娘拽着跑远，还回头啐了一口。

韩泥眼皮都没抬。户籍册上写着「绝户」二字，他一人，无儿无女，死了连摔盆的人都没有。抬棺、扫庙、编筐、采药，五十八年，就这么活。

土岗西坡土薄，雨多，风也多。风一过，茅草屋顶簌簌作响，响得人心发空——别人家漏一点补一点，他补不起，旧了，就让它旧。

藤条是前日自后山采来的野藤，先拿刀刮去老皮，再入水浸软，浸软了才编。藤条在膝上绕一圈，勒紧，再绕一圈——绕两圈，勒紧，留半寸尾，尾塞结里，跑不了。麻脸是胎里带的，跛的是早年抬棺摔的，驼背是常年弯腰编的。晦气贴上了，撕不下来；撕不下来，就不撕。辩，费舌；舌要留来编筐，留来活。

「韩泥！」

马春花的声音从井台那边砸过来，尖，脆，棒槌跟着在盆沿上敲了一下：「编你的筐！少往村道中间杵，挡了运！挡了运，你赔得起？」

韩泥「嗯」了一声，藤条没停：「赔不起。所以挪了。」

马春花哼一声，没再近，棒槌又砸了一下盆沿，砸得脆，替他收住这阵骂。近也近不出钱来，骂也骂不出钱来，她骂她的，他编他的，各过各的。她不近，韩泥也记——去年他抬棺路过她家，她没拦。恩轻，记在心里，谢字欠着，得等有还的时候。

藤条勒指，指节起皮，起皮不慌——慌了结就松，筐就塌。塌不得，塌了，十二文也没处落。

日头偏西，筐底圆了，沿齐了。韩泥用指节敲了敲底，声闷，闷而实——实了，才配装药，配卖钱。他起身，腿麻，拄了一下门框。门框朽，朽得跟指节一样，起皮，裂口。

泥屋里黑潮，霉气贴着脸。灶上无米，瓮里无水，墙角堆着苦蒿和半捆野藤——能换铜板的留着，换不来的晒干当柴。穷归穷，哪块砖松、哪根椽朽，他身子比眼记得牢。

他还会扫庙。村东小庙香火淡，灰厚，扫庙换一顿斋饭，有时换半袋粗面。昨儿扫过，尘土沾裤腿，他没拍——拍着也嫌邀功。庙祝老眼昏花，常说：「韩泥，你手稳。」

韩泥不答，手稳不稳，筐知道，棺木也知道。

隔壁张麻婆骂街，骂得凶，尾声却落一句：「韩泥那筐，编得倒是结实。」他听见了，没接话——接话像讨夸，夸轻，筐重。

他把两只筐叠好，用草绳捆了，绳结是爹教的。爹咽气那夜手搭坛沿，嗓哑：「坛别扔。韩家就剩你，坛在，根在。」沿凉，摸上去跟冬夜一样。祖父老栓从不卖、从不扔，只留这句话——坛在，韩家不散。韩泥不知真名，只唤「腌菜坛」，每日用旧布擦沿，布洗得发白，擦得惯，习惯成了根。

屋角那只坛，腌酸菜用的，蒙尘无光，内息尽敛，瞧不出半点仙气。酸菜早吃完了，坛空着，他便当摆设。摆设也是家当，穷人家，坛在，屋在，人在。村里人笑：「绝户翁连坛都蒙尘，命也灰。」他不答。擦坛不费劲，费劲的是压住心里那口气——册上「绝户」二字擦不掉，坛沿能擦亮一层。

午后，他担两只筐赴集尾。

泥岗村集尾在村道口，道窄，人杂，货担铃铛响两声又远了。卖盐老汉冲他点点头，很快把脸别开，怕晦气从点头里传过去。韩泥不近不躲，担筐走稳，走稳了，辱也追不上步。

肩勒出旧印，印深，沉得住。集上人躲半步，躲完又瞥筐，瞥筐不瞥脸。韩泥习惯了，习惯不等于不痛，痛久了，就不往外说。

买主是个挑盐的，老熟人，打量筐沿：「编得紧。不松。」

「紧才装得住药。」

买主捏了捏底，底不塌，从腰里摸出铜钱，数了两遍才递过来：「十二文两只。晦气翁的手艺，倒实在。给你。」

「下回还来。还编这种底。」

「还来，底还紧。」

铜钱落掌心，十二文一文不少。韩泥收进腰布，布沉了一截——这是今日小胜，辱在村道，钱在掌心；钱不暖，能买米，米能活命。他不回嘴，担起空筐走稳，把十二文也编进步子。

买主在身后嚷：「底别塌！」

韩泥没回头：「塌不了。」

巷口沈枯芽娘拽娃让开。芽儿仰脸问：「韩爷爷，筐紧不紧？」

娘捂嘴拖到暗处：「紧不紧，别问晦气。」

韩泥脚步没停：「紧。装药不紧，人倒霉。」

话落，自己也觉晦，晦归晦，手不停。他绕远半步出集，风把货摊上的草屑卷起来，又落下，像村道上的话，起得快，散得也快。

回屋时，天阴了。泥岗村的雨细绵，专找漏瓦钻。韩泥拿缺口陶碗接屋顶漏水，碗小，接得慢，水漏进碗又漏进泥地，转眼没了。他忽然想，这辈子是不是也这样接雨——接一碗，漏一碗，到头来还是干的。

他把门口半筐野藤往里拢了拢，免得雨泡烂。明日天不亮还得上山，苦蒿蛇藤都得备，篓绳旧了也得系紧——系法仍是爹教的：绕两圈，勒紧，留半寸尾。跑得了结，跑不了命；命在，筐在，坛在。

雨大了，漏进屋里，打湿半边衣襟。雨丝打在他麻脸上，针扎，也洗，他没躲。

村道那头，赵绣春跟几个妇人笑，笑里带刺，听不清字，刺却扎人。赵绣春夫家做皮货，走路带风，利起来比刀还快。韩泥与她无冤也无恩，只晓得那人嘴利——嘴利不要紧，记不要紧，恩辱分列，不混。

他把碗搁下，坐到坛边，低声道：「不漏人就行。」

话音落下，坛壁轻轻一颤，极轻，像呼吸。雨珠落沿，蒙尘里浮出一线极淡水汽，热汤刚揭盖那一瞬，一闪又敛。韩泥以为手抖，没理会。丑时坛壁微凉，午时又温，他当天候作怪，从不多想。

今日擦坛，指尖顿了顿，沿仍跟凡物无二；他按了按心口，空了几十年，近来梦里常梦见欠着什么，醒来手心发热。急什么，五十八年都过来了。

泥岗村入夜早，远处喊娃吃饭，喊声拖长，断在雨里。灶冷，他不喊饿，饿惯了。村童嚷「晦气翁，别近」，张麻婆咳一声：「少嚷。嚷多了嘴烂。」村童散了，张麻婆没近这间漏屋——近了，也沾晦气。

韩泥嘴角动了一下，不知是笑，还是咳。他摸腰布，十二文硌着，踏实。村嘲在远，坛在角，钱在腰。沿在，根不散；人不问，坛也不答。

他合上眼，手仍按坛沿。五十八岁，绝户，晦气翁，还在泥岗村西头这间最漏的屋里。雨声里，沿口一线温，贴住虎口，脆，短，够撑一日，脆如十二文落袋。沿温罢又凉，凉也罢——瓮未醒，他不喊醒；醒在缘，不在雨夜这一更。

明日还得编。编了筐，换钱，买米，活。

雨声里，远处井台棒槌又砸了一下盆沿，砸得短，短而脆，给明日留半寸活路。活路在，坛在，钱在腰，手还得稳。
