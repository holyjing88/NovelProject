# 第一百三十章 还刘婆

外门复测，丹香起。

韩泥挂牌「丑丹师」，牌歪，字丑，人更丑。独眼半阖，鼻下嗅诀轻动，嗅药辨性已成——成在废渣里淘，淘在赖扒皮克扣里淘，淘在五年杂役里淘，淘在兽栏腥气里淘。

药渣摊前，他捻渣，渣灰，灰里一点香。香正，丹方正。

一品培元散，成。

丹成时，天色白，白得像刘婆补衣那线——线旧，线暖，线他记了五年。记从泥岗村起，记到沉丹宗，记到今日。

「韩泥，成丹！」

执事验丹，两指捏丹，对着日头看色：「正。」

正得不像丑杂役的手笔。围观外门弟子嗡一声，嗡里带嘲带惊。嘲还在，惊也在了——丑，能成丹；老，能夺魁。

白织月袖过，袖风带香，声冷：「运气。」

韩泥不辩，只捧丹，捧去比试台。捧得稳，像捧一碗不洒的热汤。

比试末项，培元散夺魁。

没有惊天动地的雷，没有少年天才的吼。只有执事扬声：「韩泥，魁。」

魁字落，落他六十三岁脸上——入门五年，炼气十三，魁。

铁无言在人群侧，抱臂，没喝采，只点了一下头。点得像认可，又像等你还下一笔。

他声低，低得只有韩泥听见：「刘婆这笔清了。下一笔，别迟。」

韩泥「嗯」一声，不辩。辩，像显摆恩偿；恩偿，静着还，还到该还的人手里，才算数。

刘婆在人群末，蓝布包头，眼浊，手抖，抖得像当年灶边补肘那夜。那夜冷，冷得像今日霜，霜贴在宗门石阶上。

有人推她：「去啊，你补衣的翁成丹师了！」

刘婆慌，退半步：「别……别沾晦气……」

推她那人啐：「晦气？魁也是晦气？」

刘婆更慌，手把旧补衣角攥皱——皱的是恩，恩旧，不敢认。

韩泥走过去。

不走快，怕摔了怀里两只布包。一包新衣，棉实，不艳，耐穿，袖口多缝一层，耐磨；一包铜钱，养老钱，数过三遍，不多不少，够一个补衣婆活到老，不必再偷偷摸摸，不必再怕赖扒皮眼。

「刘婆。」

刘婆抬眼，慌，像怕晦气沾身，又像怕梦醒：「你……你成丹了？」

「成了。」韩泥把布包塞她手里，托住她抖的手，「衣，换新的。钱，养老。当年补衣，我记着。」

刘婆眼湿，没哭出声，唇颤：「我……我不是图这个……」

「我知道。」韩泥说，「你图我活着。我活了，还你。」

她抖着手摸新衣袖口，袖口多缝一层，耐磨，像当年她偷偷给韩泥补的那件：「这衣……你记得我补过？」

「记得。」韩泥说，「袖口磨破，你顶针顶到半夜。半夜冷，你手抖，抖也没停。」

刘婆唇颤，颤得像终于敢信：「我以为你忘了。杂役五年，谁还记得补衣婆……」

「杂役五年，恩没忘。」韩泥声平，「忘恩的人，炼不出正丹。」

刘婆抬眼，眼浊里清了一线：「你……你当真还我？不是可怜我？」

韩泥：「不是可怜。是还。还了，才正。」

刘婆点头，点得像补衣时顶针顶布，一下，又一下。新衣布糙，糙得真，真得像恩。

围观者静。静里有人啐，有人不语。不语的多——恩还了，还得丑，还得静，静得像一碗热汤终于递到了该递的人手里。

沈枯芽在远，怯笑一下，笑浅，像芽。铁无言仍抱臂，声低：「下一笔，还谁？」

韩泥不答，答在心里。心里排序：刘婆还了，叶汤还在前头，芽儿还在前头，泥岗还在前头。

白织月袖风又过，过得像不服：「魁是运气，丹道看骨。」

韩泥捧丹盒，盒稳：「骨丑，丹正。正，是五年淘出来的，不是运气。」

白织月唇动，没再尖。尖不过「正」字。

韩泥回杂役屋，独坐坛边。

刘婆那一笔，恩还了，记在心里——减得他胸空半寸，空得舒服——舒服不是无债，是债还了一笔，还能还下一笔。

他低声道记着，深：

筑基后，必省亲。

六字，像路标，标向泥岗，标向省亲，标向叶丫头那碗还没还尽的汤。汤是头的，刘婆是还了的，叶还在前头。

坛在角。坛腹微温，人稳。温里一线光，光沿壁爬半寸，爬得像应「还」字——还应刘婆，还应叶汤，还应泥岗。

光爬时，他看得分明：不是异象吓人，是坛认「还」——还到了该还的人手里，瓮便温一线，温得像爹搭沿，像泥岗丑时那口白气又吐半分。

韩泥按坛，不声张。声张，赖扒皮来，管事来，煞骨门旧债也来。来不要紧，要紧的是刘婆那一笔，恩债减了一笔，减得胸空半寸，空得舒服——舒服不是无债，是还能还下一笔。

外门大比帖，已贴山门。帖上言伯钧名，上钱戾衡影，影在字后，像仇在路后。

韩泥望帖，独眼平：「三年约，快够了。」

魁，是丹道一步；省亲，是恩偿一步。两步不抢，一步一步来。

他按心口，低声道：

「汤还是热的。」

窗外丹香散，散进风，风进袖，袖里像还留着刘婆新衣的棉味。

刘婆走远，走远还回头一眼，眼浊，浊里清了一寸。清，像恩还了一半，另一半是她终于敢穿新衣出门，不必再躲赖扒皮的眼。

她把新衣往怀里拢，拢得像拢当年那件补衣——补衣旧，恩新；新衣新，命也新。

够。

铁无言在门外，没进来，只隔窗问：「还了？」

韩泥：「还了。」

铁无言：「下一笔？」

韩泥：「叶汤。在省亲前。」

铁无言：「别迟。」

韩泥：「不迟。迟了，丹不正。」

韩泥望坛，低声道：「还了刘婆，记在心里。叶还在前头。」

坛不答。答的是腹里微温，温得像应「还」字第二笔——笔未落，路已明。

培元散瓶温，温像叶汤远影。远影在，还路在；还路在，筑基后必省亲。

还刘婆，新衣养老钱落袋，袋沉，沉得像第七笔宗门恩落了地。

刘婆捏新衣袖口，袖口厚，厚得像当年肘布——布丑，肘暖了，才走得动还路。韩泥把养老钱推回去半份。刘婆骂：「少啰嗦，收下。」他「嗯」一声，收下，记在心里。

大比散场，培元散夺魁，魁声远，远不要紧，要紧是叶丫头在廊下等他。他低声道：「筑基后必省亲。省亲路长，七笔旧恩，还热着。」

坛内光渐强，符纹游走，贴腹微震——震像应还，不应名。名未醒，步先稳。丑骨不改，还路在——在叶汤，在刘婆灶，在省亲那日。

廊下风硬，他按心口七笔。笔在，手就稳；稳了，才配接262省亲那一程。夺魁是小胜，胜在还，不在傲。傲，邪；还，热——热字不写，按在胃里。

培元散瓶温，温像叶汤远影。远影在，还路在，省亲路也在。

刘婆新衣叠齐，叠得像当年肘布再缝一回。缝一回，恩又落一地。

他摸坛，坛内光一线，光不应名，应「还」字——还，先还热的。

刘婆新衣叠齐，叠得像肘布再缝。缝一回，恩落一地；落地，记在心里，不在舌。

还刘婆新衣养老钱落袋，炼气十三培元散夺魁；韩泥记筑基后必省亲——省亲路长，七笔旧恩还热着，热着就不怕丑骨。

他望山门帖，帖上言伯钧名，上钱戾衡影。影在字后，像仇在路后；路后不要紧，刘婆这笔已还，还了，丹才正。正，是五年淘出来的，不是运气——运气救不了丑骨，手稳才能。
