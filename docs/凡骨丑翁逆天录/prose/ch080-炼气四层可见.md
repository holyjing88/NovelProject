# 第八十章 炼气四层可见

冬深续后第七夜，丑时。

内视见境，灰像药山深雾，雾里碑影钝。气再转一周，灰散开半寸，四层境像远灯，灯不亮，可见。可见，不是到，是见——见到，脚才不盲。

邻床问身上咋亮，韩泥声平：「油灯豆。看错了。睡。」睡实了，内视才藏得住。藏得住，赖福才掀不了席。

样纸三包，纸糙，糙得像路。路轻，记重——记四层可见，记三层在远，记废渣收集在81。81收集了，路才宽。

内视灰散，四层远灯可见。脚不盲，81废渣收集才有着落。

收功像盖席，盖严，沿净，境敛。敛了，四层在掌，三层在忆，分列不混。混了，赖福就记「邪」。

韩泥未睡实。睡实了，内视才藏得住；藏得住，才看得见远境。

他盘坐床沿，席严，坛不露。油灯豆，豆光晃一下，又稳。稳，像踹台按实——掌在，名在，路在。

掌贴沿，沿温一线。温进掌纹，二层气转，转一周，转两周，转第三周时，鼻下忽静到极。

极静里，眼前不是黑，是灰——灰像药山深雾，雾里有一点碑影。碑影钝，钝像测灵石，石秤命，碑秤规矩。

气再转一周，灰散开半寸。散开处，四层境像远灯，灯不亮，可见。

可见，不是到，是见。见到，手才知往哪走；知往哪走，暗修才不盲。

韩泥屏息，独眼不眨。眨了，灯就灭。灭不得，灭了，四层白见。

他低声道：「四层可见。未到，先见。」

不在舌上，在掌里。舌响，香飘；香飘，赖福来掀席。

坛底极轻一颤，像臾墟残念应一息：「……见境，不叫境。叫境，煞骨门嗅得到。」

韩泥不答，答在心里——心里记一笔：四层可见，见在暗，不在公示。公示写仍末，掌见四层——见给自己，不见给别人。

气稳在二层，稳里那盏远灯不熄。不熄，像指路；指路不要紧，脚仍停——停，不是怂，是规。规在，坛在；坛在，根在。

邻床翻身，眯眼半息：「你……身上咋亮？」

「油灯豆。」韩泥声平，「看错了。睡。」

小杂役哼一声，又睡实。睡实了，内视才藏得住。

他收功，收得像盖席。盖严，沿净，坛不露。不露，像腌菜坛，不像邪。

更鼓将尽，天边白一线。白里，他廊下立半刻，鼻下仍静。静里，四层那盏远灯仍在——仍在，像钩，钩三层暗修，钩四层将到。

铁无言脚步远，停半息，声更低：「见境别叫。叫了，像求。」

韩泥不答，答在心里——心里恩辱分列。见是路，路通向废渣收集，通向丹路开。

卯时，兽栏腥重。腥不要紧，远灯在掌，铲下更稳。稳，像踹台那一掌——掌在，境在，路在。

管事弟子巡栏，记一笔：「丙九韩泥，冬深仍清细。」

刘婆端粥，仍多半勺：「昨夜又亮？」

「豆爆。」韩泥说，「沿净。」

「沿净怕什么。」刘婆哼，「饭稳，恩不断。」

韩泥点头。恩在粥里，也在远灯里——灯不亮，可见；可见，记重。

午后，药渣场。灰起，他分拣，鼻下有根，手快半分。快不要紧，快也要稳——稳了，四层那盏远灯才不晃。

铁无言走近，瞥分堆，又瞥他掌：「见境了？」

韩泥应声：「见了。未到。」

铁无言点头：「见好。到，在远。远不要紧，渣里活，活长了，才到得了。」

到得了，韩泥不声张。声张像求境；不求，只记。记四层可见，记三层在远，记二层在掌。

白织月远廊过，袖风冷：「杂役也守丑时？守出四层？」

韩泥不抬头：「守沿。沿净，不叫。」

白织月唇线绷一息，走了。少一句嘲，小胜。

秦霜木鱼轻敲：「见境心别贪。贪，邪。」

韩泥不答，答在心里——心里七笔分列。见境是路，路通向废渣路，通向温听雨初识。

他按席角，席严，坛不露。坛腹温一线，贴掌，像应「见」字。应完，他低声对坛：「四层见了。未到，先记。不叫。」

坛不应字，只温。温，像把远灯那一点灰，摁进掌纹那路里。

邻床悄问：「你身上有时热，怕不怕？」

「怕。」韩泥说，「怕也叫。不叫，沿净，才藏得住。」

少年不懂「藏境」，却懂「不叫」，闭嘴睡了。

丑时前，他再练半刻。二层气转，转时远灯仍在——仍在，像四层在等他。等不要紧，脚仍停；停稳了，三层才开得了。

掌纹那路比昨夜厚半分，厚不要紧，厚在暗，不在册。册仍写仍末，掌见四层——见给自己，不写给别人。

他独眼平，手不抖：「四层可见。三层在远。废渣收集在明日。」

刃不亮，先记。记完了，才配攒渣，才配废渣路那一袋。

玉牌在怀，凉进骨。凉不要紧，掌里远灯不熄——不熄在暗，不在众目。众目看灰线，掌里见四层。

韩泥只答：「不恋。记着。」

更鼓沉，沉里，他摸样纸三包，纸糙，糙得像路。路轻，记重；记重了，才接得住110首丹。

坛腹应温，像应「可见」最后一笔。应完，他不叫。不叫，才守得住内视，守得住暗修。

他低声道：「四层见了。叔未到。未到不要紧，见着了，脚就不盲。」

脚不盲，在渣里，在冬深，在三年约前头。前头有废渣路，有丹路开，有95那盏门缝灯。

韩泥只答：「豆爆。睡吧。」

在，就不滚。不滚，远灯才照得长——照长了，才到得了四层，才接得住五层那一境。

柱上条角白一线，冬深，风硬。风硬不要紧，掌里灯不熄——灯在，就不滚。

内视后第三日，药渣场春风硬。他分拣，指稳，忽觉掌纹影里第四阶边多一丝辛——辛像深灰焦回，辛像废渣将收。辛不要紧，记牢；记牢了，81废渣收集才有着落。

管事弟子验五堆，五堆全对，记「春验仍准」。铁无言声低：「影见了，别求破。稳二层，破三层四层在后。」韩泥应声：「不求。影记着。」

管事夜巡验棚，灯照席角，沿净，记「手稳」。手稳，是小胜；胜在活，不在嘴。

刘婆端粥，仍多半勺：「手稳，饭稳。饭稳，恩不断。」韩泥点头，谢轻，恩重。

铁无言抱臂立侧，声低：「灵根末，手稳，渣场仍收你。别漏。」韩泥应声：「不漏。」

秦霜远廊经过，木鱼不敲：「心别邪。邪了，鼻下香飘。」韩泥不答，答在心里。

白织月远廊经过，袖风冷，没停。没停，小胜——胜在活，不在嘴。

赖福脚步在廊外停半息：「第80章，我盯着你。」韩泥只答：「在。」在，就不滚。

坛腹温一线，贴掌，温不应慌，应「记」。

丑时前，他廊下练半刻鼻下静。粟壳香淡，要紧是香一字不落。落齐了，才记进心里。

柱上条角白一线，三年约在怀，愧字在怀，境在掌——分列，不混。

邻床悄问：「你又守丑时？」「守沿。」韩泥说，「沿净，不叫。」少年闭嘴，像也学会忍。

药渣场角风硬，他分拣半刻，指稳，稳里鼻下静一息——静，是把恩摁进路里。午后废渣巷口，他立半息，不进巷。进巷是收，今日只记影——影在掌，不在众目。众目看灰线三尺，掌里见四层远灯。

刘婆端粥，仍多半勺：「见着了别贪。贪了，舌就响；舌响，香飘。」韩泥点头：「不贪。记着。」

温听雨外堂过，裙素，没停。没停不要紧；要紧是手稳，境不露。露了，像求；不求，只记四层可见这一钩。

黄昏回棚，按席角，席严，坛不露。坛腹温一线，像应「见」字余韵。废渣将收，温氏将识，不叫。」

丑时前，内视再半息，远灯仍在，仍在就不慌——慌像少年，他是记心翁，翁慢，恩不凉。

同批少年悄问：「你身上有时热，邪不邪？」「沿净，不邪。」韩泥说，「邪在舌，不在坛。」

秦霜木鱼一声：「见境心别贪。贪，路断。」韩泥不答，答在心里——心里不贪，只记影，记废渣，记95灯。

老耿挑水过，声哑：「见境别拜灯。拜，路断。」韩泥只答：「不拜。记着。」

更鼓沉，独眼平，手不抖：「四层可见。三层将破，五层在远，丹路将开，不叫。」

柱上条角白一线，冬深将尽，春风声远。远不要紧，掌里灯不熄——灯在，就不滚。活长了，才还得了叶汤，才接得住门缝灯。
