> **效力**：后续撰写全书正文（第 1～1560 章）时，**以本目录策划文档为准**；正文不得与文档冲突。若文档之间有歧义，按下文 **优先级** 裁决。  
> **标准版本**：500 万字 · **1560 章** · 十二部 · AUDIT **v37**  
> **最后修订**：2026-07-14（**v50 笔法**：后续正文 → **`31` 通俗白话小说笔锋**）

## 一、铁律三条

1. **文档优先**：章号、锚点、人名、地名、道具名、系统规则，以策划文档为准；临时灵感须先改文档再写正文。  
2. **章号不可改**：硬锚分四级——**S 锚**（`22` §一 丹阵互斥）· **KPI 锚**（`16` 附录 A + `07` 锚点表）· **逐章硬锚**（`16` 附录 E / `05` §63～130）· **战役簇**（`04` 事件段）；增删情节只能在章内扩写，不得后移里程碑。  
3. **写后必查**：每部写完对照 `09` v37 清单 + 本部对应 `05`/`16`/`17`/`19`/`22` 表。

## 三、写一章的标准流程

```
① 查部别 → ② 查章段（05/16 v2）→ ③ 查 22（丹/阵互斥）→ ④ 查 17（馈缘）
→ ⑤ 查系统（02/**27**/**28**/**29**/EXPANSION/11/12/14/20/21）→ ⑥ 查情感恩仇（03/08/19）
→ ⑦ 查命名（15）→ ⑧ 按 **`31` 通俗白话** 落笔（`07` 节奏/字闸）→ ⑨ 章末对照锚点
```

### 3.1 单章自检（写完后勾选）

- [ ] **笔法**：通俗白话、可朗读；章末脚注含 **通俗笔锋**（见 `31`）  
- [ ] 章号与 `16` 附录 A / 附录 E（须拆章段）/ 本部表一致  
- [ ] 人物名符合 `15`（禁还礼同名）  
- [ ] 若有赠礼：对口、有四维或心记变化（`17`）  
- [ ] 查 `22`：**炼丹与布阵分章**；本章标「丹」则不写阵高潮，标「阵」则不写丹高潮  
- [ ] 恩仇分立：本章是报恩还是报仇还是日常，不混双高潮  
- [ ] 死亡戏：无自刎；若战死须拖垫背（`07` §世界观）  
- [ ] 章末标 **大境+小境**（炼气层数/渡劫重数；`28` §10.0）  
- [ ] 道具出场标 **名称+品阶+攻/防**；丹药标 **X品+子类**（`28` §十一）  
- [ ] 洞府/升府标 **位阶·品阶·种类**；升府同步六区（`12` §4.5）  
- [ ] 破境章对照 **破境·核心丹** 链（`29` §四·4.0 · EXPANSION §四）  
- [ ] 每5章≥1具名七品类；部末 KPI 对照 EXPANSION §六  
- [ ] 符录/道具/丹道/阵法不跳阶（`14`/`02`/`20`/`21`/`28` §1.2）  
- [ ] 简洁爽快：一章一事，章末有钩（`07`）  
- [ ] **字数硬闸**：本章 **2000～2500 字**（`07` §v48 · `prose_utils`）；超 2500 删冗，不足 2000 补钩
