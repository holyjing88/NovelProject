# 第三十五章 宗主拍桌

争论至夜，长老会厅空气像绷紧的弦。

周德海最后一推，像把全部和谈希望押在这一推上：「宗主，七年和平，够年轻一辈成长。割让三成，换七日缓冲，换丹堂七夜，换……」

「换什么？」柳青鸢抬眼，眸光冷而清，「换东脉民散？换霍镇山盾碎？换莫师兄九月，再送七夜？」

周德海笑僵：「宗主言重……」

「不重。」柳青鸢起身，袖风带起册页，册页落案，像雪落火里，「七年之后呢？把成长交给割让？」

殿内低议又起，起得像潮，潮里有人看周，有人看柳，有人看门边——莫长春仍拄杖立原处，不帮腔，像一根秤，秤砣落在「不让」那边。

周德海咬牙：「宗主，赤焰夜袭若至，青岚门扛得住吗？」

柳青鸢拍桌，板声落，震得烛焰直，像定东脉命：

「东脉不让。」

四字，无多余修辞，却比任何剑意都硬。殿内应声起，霍镇山第一个吼「不让」，年轻长老拍膝，连平日主和者也沉默——秦氏简在，手帕定在，谁还把「让」字当路？

周德海冷笑，拂袖退席，袖风带起册页一角，像一扇门悄悄合上，门外不知谁候着，候得像风，像信，像奸。

霍镇山抱拳，声洪：「宗主英明！」

莫长春拄杖立原处，不鼓掌，不喝彩，只一句，像落账：

「地可以给一点，根不能拔。宗主不给，是对的。」

柳青鸢看他，眼底微红又压下：「师兄，奸线加深了。」

「加深才好。」莫长春道，「线深了，才晓得谁剪谁。」

他出厅，夜风扑面，像秦氏简上「夜袭」二字迎面而来。廊下弟子听见「东脉不让」，炸开欢呼，欢呼里燃，也惧——燃的是终于主战，惧的是夜袭近了。

魏无涯在远处檐角，扇开一线，观气运，观柳青鸢背影，观周德海退席的方向，低笑一声，像看一枚子落对了格。

寿元在簿上，五月余，浅淡，够他把下一礼送给丹堂。

苏念慈在丹堂听讯，炉火旺，旺得像替宗主也担。她低声：「不让好。丹堂七夜，不为让。」

霍镇山拾也不拾周德海落廊的册页，骂：「让字滚远点！」

弟子围殿外，听见「东脉不让」，炸开欢呼，欢呼里惧夜袭，燃的是终于主战。

莫长春低声，像钩：

「宗主拍桌了。丹堂的火，该烧七天七夜。」

风大，像把「不让」二字，吹进每个弟子眼里。

廊外弟子听见拍桌声，炸开一层声浪：「宗主拍桌了！」「东脉不让！」「周长老退席了？」有人低声：「不让好，让了田也没了。」另一人答：「夜袭近了，不让也得扛。」

他出厅，廊下弟子围他，问：「莫长老，不让之后呢？」莫长春笑：「之后该烧炉，该备符，该让该守门的人守门。」弟子一愣，猜成七种版本，版本里都有赵玄机三个字，却无人敢大声说。

孙福在角隅，面色白，像听见「不让」二字，也听见自己的旧账将被翻。林挽月立廊外，听「东脉不让」，唇动，像想说赵师兄的局与此无关，终未近。

赵玄机在思过崖，听拍桌声远传，指节叩石，像把「不让」二字，叩进将来要守的门里。

周德海退至长老院，令旗在案，迟迟不落，落得像犹豫，犹豫得像信使已在后巷候过三次。

顾小满在杂役堂值夜，听「东脉不让」，指节白，像半块灵石在怀，温了一路。她不知暗线将走，只知不让字一落，杂役也不能退。

秦商言飞符至驿，符上四字：「不让好。」像商言也懂——根在，恩才有路走。

拍桌声远，远得像战鼓第一响——响完了，没人再谈让字。

风大，像把「不让」二字，吹进每个弟子眼里。秦商言飞符至驿，符上四字：「不让好。」像商言也懂——根在，恩才有路走。拍桌声远，远得像战鼓第一响。

殿内年轻长老拍膝，连平日主和者也沉默——秦氏简在，手帕定在，谁还把「让」字当路？莫长春拄杖立原处，不帮腔，像一根秤，秤砣落在「不让」那边。柳青鸢看他，眼底微红又压下：「师兄，奸线加深了。」「加深才好。」莫长春道，「线深了，才晓得谁剪谁。」

魏无涯在远处檐角，扇开一线，观气运，观柳青鸢背影，观周德海退席的方向，低笑一声，像看一枚子落对了格。莫长春低声，像钩：「宗主拍桌了。丹堂的火，该烧七天七夜。」风大，像把「不让」二字，吹进每个弟子眼里。
