# 第三十七章 藤条编筐

管事一扔工单，单上多一行：编筐二十只，藤条库房自取。韩泥接过，字丑单旧——念帖让他进门，进门又让他编筐。筐是泥岗老本行，本行到了宗门，也得换饭吃。

藤条库房的条劣而脆。脆条勒指，指痛，痛他习惯。泥岗野藤韧，韧得筐卖过十二文一只；这儿的条像故意刁难人。库役倚着垛嗤笑：「丙九也领条？领多了编不完，别糟蹋库房。」

韩泥只取二十捆。捆沉，压肩。

「丑杂役，做不完，没饭。」赖福倚门，笑里带秤。

韩泥不辩。舌要留来勒条，留来活。库役斜眼：「条脆别怨，丙九就这待遇。」

「不怨条。」韩泥说，「怨手不够熟。」

库役一愣，嘟囔：「熟了也是丙九。」

「丙九也要吃饭。」他扛捆出门。同批有人笑：「编筐？泥岗老把式干这个？」

「把式不挑活，活挑饭。」韩泥道。笑的人住了嘴。

库房到院角路长，霜滑。肩沉，步却匀，匀得像村集尾担筐——捆捆不晃。晃了脆条散地，散地就没饭。同批有人超他，超完筐底松，管事一眼扫来，那人又慢下来。韩泥不笑：笑费气，气要留来勒条。

午后日斜，他坐院角编。底先编密，密了才立壁。管事路过，瞥一眼：「手熟。」

赖福啐：「手熟有屁用，凡骨炼不了丹！」

韩泥抬眼，独眼平：「筐熟，饭熟。饭熟，人活。人活，丹以后再说。」

赖福唇动，没再尖。邻床少年凑近：「你编得真慢。」

「慢，不塌。」韩泥说，「塌了，没饭。」

「快一点不也行？」

「快，底松。松了砸人，砸人记名，记名就滚。」

少年缩脖走开。编到第五只，筐底松了一线，他拆开重编。赖福晃过来：「拆什么拆，交差就行。丙九讲什么规矩？」

手指不停：「塌了，砸灵兽。砸灵兽，管事记名。记名滚。滚了，嗅渣都没得嗅。」

赖福被噎住，甩袖走了。管事再经，瞥筐底纹路：「这只行。」

「行，才交。」韩泥说。编到第七只，他试泥岗老把式的「三压一收」——压三遍，收一口，口紧，壁才立。勒法在茧里，劣条也渐渐服。同场杂役蹲旁看：「丙九也配学老把式？」

「把式不挑活。」他说，「活挑饭。饭在，指就在。」

编到第十只，藤条炸刺进掌心。血珠小，却疼。他吮掉，血落篓底——记着，等将来那日子扎得准。同批少年蹲着问：「你手流血还编？」

「编了，有饭。饭在，手才在。」

「放着不管，化脓呢？」

「脓记着。」他把篓底血抹净，不往坛那边蹭。「记着，才推得对。」

管事又过：「手裂了换藤，别硬撑。」

「不换。劣藤也要编完。编完，才有饭。」

管事哼一声，却多掷半捆好条在脚边：「好条编底，底要承重。」

他用好条编底，劣条编壁——壁松可拆，底塌要命。编到第十五只，赖福又晃：「编不完，明日兽栏加倍。」

「编不完，今日的饭也没。没饭，加倍也白加倍。」

赖福脸色一青，走了。编到第十八只，藤条又炸，裂口一指宽。刘婆端水过来：「手裂了？」

「裂惯。」

她放下半块布：「裹着编，别逞。」

布裹掌，继续编。血渗布，红一线。编到第十九只，院角风硬，吹得筐壁颤。他不急——急则底歪。邻床少年又凑：「还差一只，赖福又要骂。」

「骂他的。筐不塌，骂不落底。」

「你掌都裂两回了。」

「裂两回，记着两回。」

少年听不大懂，却看懂掌布又红，闭嘴了。第二十只压底三遍齐了，他才掷给管事验。夜色里二十只筐码齐，管事挨个掂底，点头：「丙九韩泥，筐结实，记功。」

同批有人嘀咕：「二十只全结实，丙九里头一个。」嘀咕短，也像小胜——胜在底纹，不在嘴。赖福阴声：「记功有屁用，仍是丙九。」

管事喝：「滚去清粪。」

赖福走得挺急。管事掷饭牌：记功。多一口饭，多一口活。饭牌木温薄，却真写在册上。同批少年问：「你怎么不跟赖福吵？」

「吵赢了，饭少。饭少，恩断。」韩泥说。少年摸筐底：「真结实。掂着都沉。」

「结实，才卖十二文。泥岗人认这个。」

回房，他展掌，裂处布沾血，低声对坛：「手裂，筐成。筐成，饭稳。饭稳，恩不断。」

坛腹温一丝，应完又敛，不应血。院角哐一声，赖福踹翻空筐：「记功有屁用，仍是晦气！」

韩泥出门拾筐，扶正，扶得筐立——立得住人，人才能立。管事夜巡过：「明日嗅渣，手别抖。」

「不抖。」

邻床少年又问：「你手裂了，嗅渣真不抖？」

「抖了，渣漏。漏了，没饭。没饭，活不到还。」

少年睡了。刘婆门外咳：「编筐手裂，明儿嗅渣别逞强。逞强漏渣，管事记名。」

「不漏。手裂，鼻不裂。」

她哼一声，脚步远了。他心里并着叶丫头那句「汤是热的」、刘婆的布、管事「记功」二字——并着，却不搅成一锅。吐纳一口，气沉掌茧。茧醒着，「必还」也醒着。

院角风又硬，空筐被踹翻处还扎着藤刺。他看一眼，不捡——刺要等对的日子。掌裂血渗到沿前即止。今夜只记这一口饭功：功在册，也在手茧里。摞齐的筐像给明日备了一条路——路在鼻下，不在舌上。

库役夜里来数筐，数完哼：「丙九也能编齐？」

「齐。」韩泥说，「齐，饭才齐。」

「别得意。明日嗅渣，鼻要比手聪明。手结实没用，闻错了照样记过。」

「闻不错。」他展了展裹布的掌，「手裂，鼻还在。鼻在，渣就不乱。」

库役盯他一眼，像想再刺两句，终究甩册走了。册角沾了点藤屑，屑小，却证明二十只筐从这双手过过。过过，就有底。

他把掌裂处用破布缠紧，缠得像给明日鼻下备一道门。门内苦甘要分清，门外赖福骂。骂是风，风过，记功那口饭仍在肚里热着。同批少年梦呓翻身，肘碰筐沿，碰出一声闷响。

闷响小，他也听见了。「别压塌。」他低声说，伸手把那只筐扶正。少年迷糊应：「塌不了……你编的……」

「塌得了。」韩泥道，「人一懒，底就松。松了砸脚，砸脚记名。」

少年含糊睡去。他把筐摞口对齐，齐得像给嗅渣课铺台阶——阶不高，却要踩实。灶间那边刘婆又咳，隔门问：「手裹了？」

「裹了。」

「明日别逞。逞着漏渣，记功那口饭吐回去。」

「不逞。」他说，「静着闻。」

她脚步远了。院角空筐被踹歪处，他再扶一遍，扶到能立。立着的东西，像他自己：丑，跛，却不趴。丑时末，掌茧贴着筐沿，沿凉，凉不过那口饭热。饭热，手才稳；手稳，明日嗅渣才不漏；不漏，管事才肯再看他一眼。一眼不多，够他把鼻下那条窄路踩开。

他临睡前把二十只筐数了两遍，数到末一只，掌心刺还隐隐跳。跳着记：藤刺进肉不是倒霉，是提醒——提醒血别淌到坛沿，提醒缘未到、手先稳。稳到嗅渣场灰热扑面时，鼻子还能从苦里抠出那一线甘。

门外更鼓沉。他低声对掌心那道裂口说：「裂着记，别裂着哭。哭费气，气留给明日鼻下静。」静够了，苦甘才分得开；分开了，记功那口饭才不只进肚，还进路。路一开，赖福再踹空筐，也踹不倒他明日要闻的那缕香。香在鼻下，功在册上，血在掌茧——三样都真，才配得上「丙九也能编齐」那一句刺。刺着也好，刺醒手。手醒了，明日闻渣才不懵。
