# 第十六章 跟风

村会定了日，风先紧。

韩泥早起编筐，臂布未拆，藤条绕膝，绕得像把明日要说的话先编实。野藤刮老皮，入水浸软，浸软了才编。藤条勒指，指节起皮，起皮不慌——慌了结松，筐就塌。

筐底圆了，沿齐了。他用指节敲了敲底，声闷，闷而实——实了，才配装药，配卖钱。里正锣余音还在耳里，锣声短，短而清，像给全村钉一颗钉：钉名「辨诬」，钉日「明日巳时」。

编到晌午，他担两只筐赴集尾。肩勒出旧印，印深，沉得住。买主是个挑盐的，老熟人，打量筐沿：「编得紧。不松。」

「紧才装得住药。」

买主捏了捏底，底不塌，从腰里摸出铜钱，数了两遍才递过来：「十二文两只。晦气翁的手艺，倒实在。」

韩泥收进腰布，布沉了一截——钱不暖，能买米，米能活命。他不回嘴，担起空筐走实，把十二文也编进步子。

午后担筐过井台，周癞子醉，醉到红眼，晃过来推他肩。推不动——筐肩练出来的劲，不是醉劲。

「晦气翁！」周癞子嗓哑，「赖管事说，你克全村！克到村会都要开！」

韩泥拎桶，水不洒：「村会开，开的是谣，不是我。」

周癞子愣半息，骂：「叶寡妇送你汤，睡没睡，谁知道！」

这话毒，毒到连醉鬼都信自己的嘴。韩泥不辩睡没睡。辩，成戏。他只道：「睡没睡，天知道。汤热不热，我知道。」

周癞子手悬半空，像没料到他不躲。不躲，像心记着。记着压人，压到醉鬼手落不下，落不下，就骂一句「疯子」，晃远。

井台边，周嫂递眼色，周癞子又来推他肩，推不动。「推我，不如推井绳。」韩泥道，「绳断，你渴。」

周癞子啐他脚边，啐不到人，像唾沫自辱。周嫂帮腔：「韩泥，你硬，硬到全村跟着受罪！」

韩泥拎桶：「受罪的是病。病等着抓药，不等你的棒槌。」

周嫂棒槌一紧，没砸——砸，里正记名。记名，村会前，胆先缺一截。村人见，见晦气翁不倒，倒的是周癞子的步。步乱，像跟风踩空。

马春花远处看，看两息，没帮腔——帮腔要力，力要胆，胆被村会钩住，钩住便少一截。赵绣春却近前，嗓尖：「村会辨了，你也跑不了！」

韩泥拎桶，水不洒：「跑不了，就不跑。站给全村看。」

赵绣春唇动，没再尖——尖需要众，众散一半，尖便短。井台边，童谣丫被娘拽住，仍探头。探头里，念半句，又卡壳：

「叶家铺，晦气铺——」

娘捂她嘴：「再念，回家挨揍！」

童谣丫眼圆，圆里怕，也像好奇：好奇晦气翁怎么还不倒。韩泥路过，不停，只道：「念完再念，嘴累。累的是嘴，不是铺。」

童谣丫挣出半声：「翁……狼……真打？」

「真打。」韩泥道，「牙怕棍，人不怕谣。」

娘拽紧，拽远。韩泥回屋，心口四恩在，赵辱旁，他又记一辱——辱增，不写名，写风。风无姓，风有痕。痕在心，不在口。

「周癞子，记着。」他低声道，「跟风的人，风停，先摔。」

坛沿微凉，心口微热。热里，他想起村会就在明日——未定锤，谣更急；急，就更要记恩，不能乱拔仇苗。夜里，沈家咳两声，咳得轻。轻，像芽儿娘还活。韩泥合袄，塞怀，望叶家药铺方向，灯弱，仍亮。

亮着，叶汤就不倒。风过村道，童谣丫不念了——不是怕，是娘拽回。拽回，风仍吹，吹到告示桩「连坐」二字，字黑，像等下一阵更醉的嘴。

韩泥不追风。追风像少年。他只记辱，候村会。辱二，比赵辱短，却更硬——硬在跟风者名虽不写，形已在。形在，将来村会上，一并算。

陈姑远处咳，咳两声，像提醒：还活着的人，别被风带倒。韩泥「嗯」一声，应给自己。井台边，周癞子又晃来，想再推，见韩泥桶满，推不动，自骂「晦气」，晦气落在自己脚面。

风停一瞬，停里，叶家灯仍亮。亮，叶汤还在。还在，跟风的人，先摔一半胆。夜里，叶铺秤又落一声。落声远，远过周癞子骂街，远过赵绣春门内摔碗。韩泥在屋听，不近，近像牵扯。听，就够——够知叶家秤还在，够知村会前叶汤不倒。

马春花远处嚷：「村会开，看你还硬！」

韩泥开门缝一线：「硬的不是我，是汤。汤热，你嗓尖也尖不凉。」

马春花噎，嚷声收——收，像风又断一截。坛沿微温一丝，温应秤声，不应骂声。骂声在舌，秤声在理。理在，翁便候村会。

他低声道：「辱二记着。恩四笔，一笔不能浅。浅了，村会白开。」

风再起时，村人分三拨：一拨跟赵绣春骂，一拨跟里正等，一拨只绕叶铺走。绕铺者停门阶外，低声：「叶掌柜，明儿还开？」

叶青禾帘内：「开。病不等村会。」

问者「嗯」一声，步快，快里像松半口气——气松，风就短一寸。周癞子醉醒又晃，晃到告示桩，读「连坐」，读得舌打结：「连……连我？」

韩泥路过，独眼平：「连不连，村会判。你先醒。」

周癞子啐自己脚面，啐像自辱——辱二记在心里，形在，名不写，写风。赵绣春想借风再尖，尖到一半，见里正锣在巷口，咽回去。咽回去，也是钩：锤未定，人心已定一半。

叶铺秤又落，落声压风。压得住风，压不住村会——村会来，恩辱一并算。韩泥低声道：「风停，摔的是跟风的。站住的，是记恩的。」

巷口，昨日跟风嚷「晦气铺」的汉子低头，放一竹篮鸡蛋在叶铺阶前，声小：「昨嘴贱……叶掌柜，还开铺不？」

汉子步快逃走——逃，像跟风先摔一截胆。叶青禾隔帘对韩泥方向，声低：「蛋我收。铺还开。村会上，少一张邪嘴。」

韩泥路过，不停，只道：「胆摔了，汤别砸。砸秤的，帖上见。」

张麻婆帚过巷，骂：「再跟风，我先打腿！」

周癞子醒半分，听见，脸涨，涨得像昨日推不动的桶——推不动，就摔自己的胆。韩泥不笑。笑轻。他只低声道：「明日巳时。帖在，脚就站住。」

丑时末，沈家灯芯又跳一下，跳得像芽儿娘还醒着。韩泥在屋听叶铺秤声，不近，近像牵扯——听，就够知叶家秤还在，够知村会前叶汤不倒。倒不得，翁才站得住。跟风汉子赔蛋，胆先摔；赔蛋的胆摔了，翁的脚还站——站着，等锤。

坛角又温一线，温应秤声，不应骂声。辱二记在心里，形在，名不写，写风。风停，摔的是跟风的；站住的，是记恩的。

夜里，里正叩门，三下，沉。韩泥开门一尺：「里正。」

「村会要开。」里正嗓低，「你少往前冲。冲，害叶家。」

「我不冲。」韩泥道，「我站。站末席，听锤。」

里正顿一息：「锤落谁身上，未定。定的是，谣不能糊。」

「糊不了。」韩泥独眼平，「汤烫着，糊不了。」

里正「嗯」一声，步远。韩泥回屋编筐，藤条绕膝，绕得像把「辨诬」编进去——编进去，等那日，叶汤能立，芽儿能活，叶家能开门。

明日巳时，祠堂村会，叶汤得立给全村看。

他合眼，手仍按坛沿。五十八岁，绝户，晦气翁，还在泥岗村西头这间最漏的屋里。明日巳时，祠堂见。村狗叫一声，又哑。哑，像全村等日——日定了，钩更紧。韩泥不拜日，只拜手——手实，筐实，汤才能立。坛角微温不退，沿口水汽一线，像瓮在睡，却听见了「辨诬」二字。瓮未醒，他不喊醒；醒在缘，不在这一更。

同棚杂役笑，笑在耳，不在心。
