# 第三十五章 叶挡飞石

入门第五日，韩泥扛着麸皮袋从侧门出来，袋沉得像赖福咬着牙不肯松秤。他本想快回丙九，驿亭下却站住了——亭角布鞋稳，蓝布袄补过一道，正是叶青禾。

「叶丫头。」他嗓子哑。

「韩翁。」她声平，「我爹那包药，你收全了？」

「收全了。」

「别死半路上。」她顿了顿，「死了，汤白送。」

「汤是热的。」韩泥道，「我记着。」

就这几句。不多，却像药铺秤砣落了——落下便实。亭外坡道忽然马蹄乱，赵绣春带着周嫂、马春花，从泥岗一路追到驿亭，嗓子比蹄声还尖：

「叶青禾！追到宗门门口送汉子！你还要脸不！」

叶青禾看她，不躲：「送药，不送人。人已在宗门，药是我爹的。」

「送药送情！全村谣都听着——」

马春花弯腰拾道旁石卵，扬手便掷，冲着韩泥脸来。叶青禾侧身一步，挡在飞石前。石砸肩头，闷响短，衣肩裂开一线。血没涌，韩泥独眼却猛地一紧。

山门侧门处杂役静了一瞬。赖福想笑，笑不出来。管事沉着脸喝：「驿亭闹事，记名！」

赵绣春见管事，嗓子收半寸，仍尖：「她自找！寡妇追男人——」

叶青禾捂肩，声音仍不高：「药是叶振东的。情是恩。恩不脏，脏的是嘴。」

韩泥放下麸皮袋，独眼先看叶肩，再落到赵绣春脸上。不落泪，也不吼，只吐四个字：

「这恩，必还。」

四字平稳，像钉子敲进木里。叶青禾看他，眼不媚也不怯，像看着一碗还温的汤：「我不图你还。图你活着。」

「活着，就还。」韩泥说，「等我能炼丹了，先还叶丫头。」

她嘴唇动了动，想再说，他抬手止住，止得像公审那天：「别谢。谢字轻。」

赵绣春见势不妙，拽周嫂就走。马春花远远啐一口，啐像败了——石已落在叶肩，韩泥的誓却立住了。管事记完赵名，对叶青禾道：「叶氏，伤不重，去医。再闹，报县。」

「嗯。」叶青禾退半步，半步近而不缠，「药在袋。肩在痛。痛换你一句实誓，值。」

韩泥不接「值」字，心里却把叶汤那笔恩钉死——钉名「必还」，不跟辱搅，不跟仇搅。管事又对他说：「你，搬麸皮回房。别杵亭下惹眼。」

他应声扛袋。叶青禾退到亭外十步，仍不闯侧门，只扬声：「药袋在栏边石下。别忘。」

晌午，他在栏边石下揭开药袋。绳松，药皮干，仍带着叶家药铺那股苦辛。苦辛钻进鼻里，像辞村那篓药债又压回肩上——债要还，先还肩上一石，再还袋里药。

「不忘。」韩泥低声说，「药是热的，肩也是热的。」

骂声已远，远得像泥岗那摊谣。谣还在村里，腿进了宗门，谣追不上侧门。夜里回丙九房，他摸坛沿，沿烫一线，像听见了「必还」。坛壁爬起半寸水汽，蒙尘里符影半闪，像公审前夜，也像爹的遗言还在底。

「必还。」他声音很低，「先还汤。先还肩上一石。」

肩上一石不是血债，是情债。情债用丹还，用命还，用将来省亲那日还。邻床少年听了半句，侧头问：「你跟那寡妇……」

「恩。」韩泥独眼平，「不是奸。奸在嘴，恩在心。」

少年闭嘴。外头更鼓响了三声，像给这句话盖了印。瓮还没醒，人心先醒半分——半分够忍，够走到醒的那天。刘婆夜里送粥，粥稀却热：「喝了。别空肚立誓，空肚誓轻。」

他喝尽，不谢。轻的是谢字，重的是还。少年又问：「必还，还什么？」

「还汤，还肩上一石，还叶丫头哪包药。」

「你还不会炼丹。」

「不会，学。」韩泥道，「学一日，记一日。记满了，就还。」

门外管事巡夜，敲柱：「驿亭闹事记过了。别再生事。」

「不生。」韩泥答，「生事，饭少。」

管事瞥他一眼走了。刘婆在门外咳了声：「肩上一石，别扛在肘上。肘漏风，粥白送。」

「不扛肘。」他说，「扛心。心在，誓不轻。」

柱边新贴了一张条，字利：「入门满旬，凡骨测骨。五日后卯时，丙九末排到。」

五日测骨——比还汤近，比滚更近。近得手就不能抖。未时杂役分班，有人嚼「寡妇挡石」，嚼得短，短不过管事记名那一笔。韩泥铲粪仍稳，像告诉整条驿道：石落了，誓更实；誓实了，脸不脏。

兽栏角里小兽又咳，咳声细。他多投一把干草，草落，小兽啃。啃着，他心里那句「活着就还」又沉半分。邻床少年路过栏外，愣问：「你还喂它？喂它管事也不加饭。」

「加不加，是册上的事。」韩泥说，「咳轻半分，栏稳半分。栏稳，我少挨骂。」

「你净想这些。」

「想这些，才活得下去。」他把铲贴稳，「活下去，才还得起肩上一石。」

少年咂舌走开。赖福远处骂了两句「晦气」，骂完见管事影子，骂声噎住。噎住也好——噎住才不惹记名。韩泥不看他，只把湿粪换净，净到石缝露出来。石缝凉，凉得人清醒：清醒着，才记得亭下那一声闷响不是梦。

夜里他又摸药袋。袋口绳他重新系紧，系得像把叶家那股苦辛拴住。苦辛在，爹的药在；药在，恩就不空。邻床翻身问：「那寡妇肩伤重不重？」

「衣裂一线。」韩泥道，「一线够疼。疼换一句实话，她不亏——她心里清楚。」

「你真要炼丹还她？」

「不会炼，就学。学不会，就熬。熬到会。」他独眼平，「熬不起大话，熬得起日子。」

天将亮时，叶肩那一线裂仍疼在他眼里。疼在心里，不在嘴上。嘴上轻，心重——才配得上叶丫头那句「我不图你还」。图不图，他都要还；还，先活过这五日，再活到能炼丹。

卯时前，他在柱下把木牌按实，像把「必还」按进掌纹。搬麸皮过侧门时，袋沉，肩不塌。石印那裂在袄里疼着，他不喊。喊像少年，他是记心翁。刘婆远处瞥一眼，只把粥勺在灶边顿了顿——顿得像说：肩疼不要紧，肘别漏风。

灶边有人悄问刘婆：「驿亭那寡妇是他什么人？」

刘婆不抬头：「恩人。恩人不是谣里的人。再嚼，扣你盆。」

问话的人缩脖。韩泥端粥路过，听见了，也不接。接了像炫耀，炫耀污恩。他只低头喝净，稀粥烫喉，烫得像提醒：热的东西，别让别人泼脏。

午后又过驿亭。亭空了，石子还在道旁滚。他看一眼那颗石卵，不捡——捡不起石，捡得起誓。管事弟子远远喝：「丙九的，别杵！测骨帖看过就滚干活！」

「看过了。」韩泥应，「五日卯时，末排。」

「末排也别迟到。迟到当弃测。」

「不弃。」他说，「弃了，誓白立。」

弟子愣半息，挥手赶他走。他扛袋回栏，步子慢，却一步不虚。虚不起——虚了，叶青禾挡的那一下就轻了；那一下不轻，他就得把它驮到能还的那天。

驿亭柱下，测骨帖在风里不动。不动像钉，钉住了，必还的誓就不飘。栏边药袋仍干，苦辛催着：先活五日，再还肩上一石。五日后的卯时贴在柱上，帖没落，锤子却已经悬着——悬着的日子，才是他往后每一口气的去处。

丑时，他再按一次木牌，牌凉贴胸。凉着也好，凉提醒他：热的是汤，热的是肩，冷的是测骨帖——冷的东西不能怕，怕了手抖，抖了誓就空。空不得。五日卯时一到，末排也好，他得站直了应名——应完名，才配得上亭下那声「必还」。
