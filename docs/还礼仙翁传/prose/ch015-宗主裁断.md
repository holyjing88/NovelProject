# 第十五章 宗主裁断

执法堂正殿，柳青鸢升座，未佩剑，威在眉睫。

孙福跪地，顾小满立侧廊——莫长春嘱她勿进，她便在柱后听，指节把袖角拧成绳。

周德海立殿左，抚须不语，像一盆浇了一半的油，等风。

莫长春拄杖立殿心，不跪不辩，灰袍素，像一块不会被嚼舌嚼碎的旧玉。

柳青鸢开口，声冷而清：「孙福，你说莫长老勾引杂役，证据何在？」

孙福叩头：「外门皆言……」

「外门皆言，」柳青鸢截话，「便是证据？」

孙福一窒。

柳青鸢目光扫过廊下弟子：「谣言止于宗主。信谁，我来担。」

殿内一静，静得能听见铁牌光落在地砖上的冷。

她续道：「莫师兄不是那种人。半石之事，是师兄赠恩，顾小满受恩，杂役堂孙福克扣在前，顾小满举证在后——账在堂，不在床。」

孙福脸白，膝软：「宗主……孙福一时糊涂……」

「糊涂？」柳青鸢道，「糊涂到敢拿宗主师兄名节换你三月罚役的轻？」

她拍案，声不大，意重：「孙福，罚役三年，逐出杂役堂，发往灵田苦役。再敢诬长老，逐出山门。」

孙福瘫软，被执法弟子拖出，拖出一串求饶，求到廊外，像风扯碎纸。

周德海揖手：「宗主英明。只是外门流言……」

「流言我压。」柳青鸢看他，「周长老，东脉妖潮将起，赤焰使者已在路上。你若还有闲心拱火，不如去长老会备折子。」

周德海笑容终于收尽，退席。

莫长春抬眼，与柳青鸢对视一瞬，像二十年同门，一句话不必说全。

柳青鸢低声，仅他听见：「师兄，我不解释你，只信你。」

莫长春点头：「够了。」

柱后，苏念慈听见「莫师兄」三字，与心里那声「师兄」叠在一起，妒与忠绞成一股，勒得她喉紧。

她看见柳青鸢维护时，眼底没有妒，只有宗主的担——那一担，比她丹堂首座的火更烫。

苏念慈退半步，未现形，像怕自己一露面，就把私心里那点不该有的酸，溅到堂上。

廊外，弟子群情反转：「宗主都说了，还传什么！」

赵玄机立远处，抄门规的手停住，墨滴在纸，晕开一字「心」，像莫长春那句定场词又浮上来。

他忽然觉得，妒一个人，若妒到连宗主都信他，自己便显得很小。

午后，莫长春回院，顾小满仍在阶下，未跪，立。

「长老清白了。」她低声。

「清白本来在。」莫长春道，「只是世人爱听脏的。」

他递她一张薄符：「收好。不是除魔符，是护身。孙福虽逐，周德海还在。」

顾小满接符，泪落，却笑：「小满记恩。」

簿在袖中一烫，字成：

**宗主一担，忠诚入账。**

莫长春望青岚峰顶云，念：「柳氏信我，苏氏妒我，赵氏疑我——好，人多，礼才好送。」

杖中竹纹游动，气音仍是一字：

等。

他笑：「不等了。下一阵，赤焰旗要来。」大因果还在七日之期，像帖，像手帕，像全场寂然的三息。霍镇山在营帐外磨刀，刀声霍霍，像给东脉磨刀，也像给七日后的宴磨刀。

柱后，苏念慈听见「莫师兄」三字，与心里那声「师兄」叠在一起，妒与忠绞成一股，勒得她喉紧。她退半步，未现形，像怕自己一露面，就把私心里那点不该有的酸，溅到堂上。

廊外，弟子群情反转：「宗主都说了，还传什么！」赵玄机立远处，抄门规的手停住，墨滴在纸，晕开一字「心」。他忽然觉得，妒一个人，若妒到连宗主都信他，自己便显得很小。

午后，莫长春回院，顾小满仍在阶下，未跪，立。「长老清白了。」「清白本来在。」他递她一张薄符：「收好。不是除魔符，是护身。」

柳青鸢开口，声冷而清：「孙福，你说莫长老勾引杂役，证据何在？」孙福叩头：「外门皆言……」「外门皆言，」柳青鸢截话，「便是证据？」她续道：「莫师兄不是那种人。半石之事，是师兄赠恩，顾小满受恩——账在堂，不在床。」

孙福瘫软，被执法弟子拖出，拖出一串求饶，求到廊外，像风扯碎纸。周德海揖手：「宗主英明。只是外门流言……」「流言我压。」柳青鸢看他，「周长老，东脉妖潮将起，赤焰使者已在路上。」

围观弟子在殿外踮脚，踮得像等「账在堂，不在床」落进耳朵里——字一落，脏字退潮一寸。有人低声：「宗主担了，还传什么？」另一人答：「传的人，怕的不是脏，怕的是担。」
