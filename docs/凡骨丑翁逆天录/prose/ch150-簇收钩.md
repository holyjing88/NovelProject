# 第一百五十章 簇收钩

141～150，册页翻尽。

管事收册，声平：「此簇齐。下一簇，165青木药火，废窑探得。」

韩泥不辩。辩像急；急，手飘。

册上写：142耿恩、144泥纹名、145疤鬃驴、十七层实——写齐，像150该收。

沈枯芽口信，叶脉稳；耿爷聋，心明；刘婆粥稳，铁兄站侧。

赖福最后哼，哼声远，远不过坛腹那一线温。

丑时，练鼻下静。香落齐，150这一簇记进心里。

他望坛，低声道：「势够了。165，青木药火，在前。」

坛腹温，贴掌，像应「火」——火在远，手在；手在，就不滚。

更鼓将尽，天边白一线。他按心口：「汤还是热的。」

册页翻尽，尽像150收束——收束不越165，不越175，不越188；越界，手飘。

管事弟子收簿，声平：「141～150记齐。下一簇，151二品丹路，165青木药火。辩，像急；急，手飘。药峰、坊市、泥岗、兽栏——四处在，手才稳；手稳，141～149才像簇，不像散章。142耿恩送出，145驴在，146十七实，147名号续，149周家风——风在远，势在足。」

赖福最后骂一句，骂声远，远不过坛腹那一线温。

沈枯芽口信无字，只有半片枯叶——叶黄，像叶汤还在前，像165药火在远。

铁无言声低：「165在前。别躁。」「不躁。」韩泥说，「躁，火不稳。」

刘婆端粥，仍多半勺：「150了，别空肚。」韩泥点头。粟壳香淡，香落齐。

他望坛，低声道：「簇够了。165，青木药火。记着。」

坛腹温，贴掌，像应「势」——势够，才探得废窑火；火在远，手在近。白里，大比帖在怀，怀沉，沉的是路，不是惧。

簇收，势足，足像150该收的——收，不越165。

管事扬声：「下一簇，151二品丹路，165青木药火。」

韩泥记在心里，心里排序：165、175、188——三笔在远，一笔142已还半。

药峰、坊市、兽栏、验息——四处在，手才稳；手稳，簇才像簇。

帖在怀，怀沉，沉的是路。路通向大比，通向筑基，通向省亲。

他按心口，低声道：「火在远。手在近。」

六字落，像路标——标向165，标向废窑，标向第二部中段。

收141备材，收142送粒，收143耿波，收144～147泥纹名，收145疤鬃，收146十七实，收148叶线，收149周家风；收而不越165，不越175，不越188。越界，手飘；飘了，青木药火探不稳，筑基丹不正，雨夜解不了架。架在远，150只势。

势足，管事收簿，声平：下一簇151二品丹路，165青木药火——火是150钩，钩像140钩142。142已送，150钩165；165得火，200才融，260才筑基。筑基在远，150只记齐。记齐像册页翻尽：药峰、坊市、泥岗、兽栏、验息堂——五处在，手才稳；手稳，帖才压得住，坛才温一线。

温不应喜，应「势」——势够，才配称第二部141～150簇走完，不像140首簇收束就断。断不断，看142耿恩落没落半笔，看145驴在不在，看146十七实不实，看149风只听不架。只听不架，151才接得住周家线，165才探火不躁。躁，手飘；飘，火不稳。

稳，铁无言说165别躁，刘婆说150别空肚，沈枯芽枯叶像叶汤还在前——还在，262省亲还在远；远不要紧，150只钩。钩法更鼓将尽，天边白一线，韩泥按心口：「簇够了。记着。」坛腹温，贴掌，像应「势」——势够，能还才送；送142半笔，送势一整簇，送第二部在走。

簇势足，165在前。在前，手稳，火才探得出；探得出，才像第二部在走。走，不在快，在势。

外门弟子渐散，散声远，远不过坛腹那一线温。温在，就不滚；不滚，165才近。近在165，不在150；150只收势，势够，才探火。更鼓将尽，天边白一线，白里，第二部中段在走。

废渣巷日斜，他验渣，渣正，手才静。

侧门驿亭风硬，他听风，风过，记在心里。

散队后，他过外堂阶下，分拣半刻，指稳——指稳，路就不飘。

编筐半刻，藤刺扎掌，刺浅，浅些沿才净。沿净，坛才藏得住。

粟壳香淡，香一字不落。

管事记簿，记细，是小胜。胜在忍，不在辩。

他望坛，低声道：「手在，路在。」坛不应字，只温。

更鼓沉，天边白一线。白里，他按木牌，牌凉贴胸。

坊市尾巷风硬，他收摊如石，石在，就不滚。

验息后，他理怀内册页，页黄，字丑，丑而真。

铁无言声低：「别飘。」韩泥应声：「不飘。」

刘婆端粥，仍多半勺。韩泥点头，谢轻，恩重。

药峰分拣照旧，指稳，鼻下静一息。

帖在怀，怀沉，沉的是序，不是惧。

白织月袖风过，韩泥不辩。辩，费舌；舌要留来守坛。

赖福廊下过，韩泥只答：「在。」在，就不滚。

柱上条角白一线，三年约在怀。怀凉，掌温。

坡下石阶湿，他数息，数帖，数债序。

同棚杂役笑，笑在耳，不在心。
