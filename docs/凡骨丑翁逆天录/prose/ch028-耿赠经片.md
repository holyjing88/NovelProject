# 第二十八章 耿赠经片

村外来个跛僧，袍旧，鞋破，眉白。

韩泥早起扫庙换斋，帚过碑肩，尘起又落，落得像替前人收一句。庙前僧已摆摊，不卖香，卖经片——铜片，薄，刻字，字是祛寒纹，纹丑，却真。真不在美，在用。

僧看他独眼，不躲：「绝户翁，祖上守过一只瓮。瓮在眠，人别睡死。」

韩泥停：「你说什么？」

僧笑得浅：「你屋角那只坛，蒙尘如凡物，是韩家祖辈口传下来的，不是腌菜。命也沉。沉要补，补先补心。心暖，寒不侵。」

韩泥指节一紧，紧得像爹那句「坛在，根在」。根在口传，口传在擦沿，擦沿在命——命沉，片来补缝。

他不辩信不信，只问：「经片何价？」

「不要钱。」僧推一片过来，「老耿让我捎的。他说，你抬棺稳，坛该稳。」

老耿。韩泥愣半息，接片。片凉，贴掌，凉后一丝温，温得像凡符未燃先暖。暖浅，够记。他低声道：「记耿恩。」

僧已走，走得不回头，像一条线从片上起了头。线在片，在坛，在以后——以前，先有这一片。

午后村人围看，围得像看戏。戏是僧，戏也是片。片在韩泥掌心，掌糙，片不滑。马春花尖嗓：「和尚也找晦气翁？晦气传染！」

跛僧远了，像没听见。听见也不辩——辩费舌，舌要留命用。韩泥回屋，片压坛底，压严，像压一道外来的缝。坛腹微温退半分，像应。应，不是醒，是稳。稳坛，稳人。

僧走远十步忽然停，没回头，只嗓沙：「片别示人。示了，寒又进骨。」

韩泥应：「压坛底。」

「压实。」僧说，「坛眠，人别睡死。睡死了，片也凉。」

声落，人没影。没影，像那条线只起一线，却真起了。

韩泥把片纹对着灯看。灯昏，纹丑，丑得像抬棺绳上的结——结牢，棺不歪；坛稳，人不飘。

外头西边商队铃远，远得像有什么还没到村，声先到。声先到，心先动。动，不急；急了便邪。

夜里韩泥摸坛，坛温，温得像耿恩在底。底稳，顶才稳。顶是命，命还长，长得够还。他低声道：「坛稳，人不漏。漏的，是谣嘴。」

嘴漏，用心堵。恩厚，经片才肯来。村里老人忆：「耿抬棺，绳不抖，棺不歪。」

韩泥听，不插嘴。插嘴像攀恩。恩在，记便够。

里正来问经片，问得警惕：「和尚的东西，别惹邪。」

韩泥摊片，片铜，纹丑：「祛寒纹。我压坛底，不拜血祠。」

里正「哼」一声：「压好。压不好，连坐。」

「压实了。」韩泥说，「实，坛稳；人稳，谣就漏不进屋。」

里正无话，走了。走，也是默许——默许丑翁多一层防，防的是旱里人心邪。

隔日叶青禾送来一小包盐姜，不放声。声惹祸，姜暖胃——胃暖，心便不易拜邪。韩泥收姜，不入腹，先贴坛边一息，取回，像借温。温借过，还记在叶汤旁。不欠叶家盐，欠的是情，情将来还。

经片在底，姜温在边，坛稳半分，人稳半分。半分累加，够走到风声贴榜。他低声道：「线起了，起得静。静，才不邪。」

韩泥夜看片纹，纹丑，真，真便记。记耿恩，不拜血祠——拜邪，片会凉；不拜，片温一丝，温得像路。路通向以后，通向寒不侵。阶在远，片在近。近，先稳坛。

马春花终是远问一句：「片能祛晦气？」

韩泥不答晦气，只答：「片能祛寒。寒在骨，不在名。」

名在嘴，骨在命。命稳，名自哑。

沈枯芽路过，怯问：「韩爷爷，和尚说的瓮……是你屋角那只？」

韩泥蹲低：「是坛，不是神。神在血祠，坛在屋角。角稳，人不拜邪。」

芽儿半懂，点头。点得像米粒——米粒旁他心里又记：静，才不教孩口学邪。耿恩在底，叶汤在胸，两笔记清，莫搅到一处。搅了，片会凉；不搅，片温一丝，路不歪。

第三夜，韩泥摸坛底，片温一丝，温得像耿恩还在。外头西边铃远，远得像东西还没到村。声先到，心先动——动，不急。

他对片纹低声念，念不成咒，念成记——记耿恩，记跛僧，记「瓮在眠，人别睡死」。睡死，是拜邪；眠，是守坛。守坛，不是信血祠，是等。等，不是懒，是活。

里正路过，瞥窗纸影，没近。近晦气翁，像沾边。他只扬声：「和尚走了，片别乱贴。贴邪了，连坐。」

韩泥开门，独眼平：「片压坛底，贴的是寒，不是邪。邪在血祠，寒在骨。」

「骨？」

「五十八了，骨寒。」韩泥说，「寒不进心，心便不拜邪。不拜邪，里正省事。」

里正愣半息，「哼」一声，走了。

叶青禾隔帘问，声平：「片可稳坛？」

「稳半分。」

「半分够？」

「不够，再加叶家姜。」韩泥说，「姜暖胃，胃暖，心不易飘。心飘，便信血祠；心定，便信秤。」

帘内药杵轻响，像盖章。章盖「还在」，盖得硬。叶青禾又补一句：「片别示人。示人，马春花又有话说；话在，谣在。」

韩泥只答：「片在底，人在门。门内门外，各守各的。」

老耿门外咳：「片别乱放，乱放碎了。」

韩泥应：「放稳了。底稳，心才稳。」

老耿脚步远，远里像叹——叹轻，像也信半分。夜里霜落，经片在坛边，古字像远钟闷响。韩泥不示人，只识沿温——沿温在，西驿路就不飘；飘了，恩断，帖便接不住。

辞村前他还去扫庙，帚过碑肩，尘起又落，落得像替跛僧收一句。碑上「路」字残，残得认不全，认得就好。收完了，片在底，人在路；路在，坛眠着等。

庙后枯井沿结霜，霜白，白得像新帖将至。他蹲井台，捧一掌冷水洗脸，脸凉，心不退温。洗罢，把姜包揣怀，姜辣一丝，辣得像叶家秤还在。秤在，寒才不进心；心不进寒，片才肯温。

第四夜，马春花远远问：「片能治丑？」

韩泥不答丑，答寒：「寒在骨。骨暖，才走得远。」答完，仍不示片。示人，片凉；不示，沿温一丝不退。

沈枯芽路过，塞一颗干枣，枣小，甜短：「韩爷爷，和尚说的瓮……别拜血祠。」

韩泥蹲下：「不拜。坛在角，角稳，人便不虚。」

枣核吐掌心，像吐一粒邪念——吐净，心定。

日头偏西，里正差人绕村贴告示：沉丹宗十月验骨，西驿见。韩泥瞥一眼，指节一紧——紧得像片在底应了一声。应了一声，辞村尺又近一寸；近一寸，经片古字便多认一笔：认的是路，不是神。

里正差人敲锣三遍，锣哑，哑得像催命，也像开路。开路细，细也得走——走先验骨，先站末席，先活长。

经片贴怀一瞬又压回坛底，怀暖一丝，暖得像那条线从村庙挪到将来。挪得不急，急便断；断不得，还路在。

集尾有个卖绳的老头瞅他一眼：「听说和尚给了你片？能祛晦气不？」

韩泥摇头：「祛寒。寒在骨里，跟晦气两回事。」

「那你还收？」

「老耿让捎的。」他说，「耿恩，我记。片压坛底，不示人。」

老头嘟囔着走开，像半信半疑。半信半疑也好，总比全村跪血祠强。

丑时，他再摸坛底，片不示人，只识纹——纹丑，丑里真。真在，帖便接得住。接得住，辞村才带得走恩。沿温不退，像耿恩还在底替他守着寒。守住了，西驿才接得住。

老耿门外又咳一声，咳轻：「片在底，人别睡死。」

韩泥应：「不睡死。片在底，人在路。」

声落，脚步远，远里像抬棺绳不抖——抖了，恩也歪；不抖，路才直。坛腹微温一丝不退，像替他守着这一夜。夜过了，帖便近。验骨那日不远，片在底，人才能站得住。
