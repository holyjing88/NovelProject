# 第四十六章 丙九内谣

编筐记功后第三日，管事又扔轮换单。单子旧，字密：丙九韩泥，药库搬柴，冬日起。搬柴不比编筐，柴沉，肩要实；肩一抖，柴就滚，滚了记名。

韩泥接单。单上没写加饭，只写「记过滚」——滚字丑，他熟。赖福倚门框，嗓子尖：「编筐克运，换搬柴，柴更克！」

同批少年退半步，像躲那个「克」字。韩泥不躲——躲了，像认。

药库在丹房后坡，坡陡，霜滑。柴捆压肩，他走得实。实，是编筐练出来的：底密，步才密；步密，柴不滚。库役嗤：「丙九也搬？搬不动，别占道。」

「搬得动。」韩泥说，「搬不动，手赔。手赔不起，饭赔。」

库役愣一下，扔绳过来：「捆紧。松了，砸脚。」

韩泥捆紧，绳结跟编筐收口一个理——口收住，捆才实。

午后，同批搬柴的少年嘀咕，嘀咕像谣起前的风：

「听说编筐那几日，麸皮又浅了。」

「浅了赖福，关韩泥什么事？」

「关。」另一人嗓子压低，「赖福说，晦气杂役编的筐漏粮气。粮气漏了，秤才浅。」

韩泥听见了，不辩。一辩，谣跑得更快；跑快了，全丙九跟着饿。他把柴捆摞齐，摞得直。直，像堵墙——墙堵不死谣，至少能把手稳住。库役傍晚点数，数到他那堆，哼声轻了：「丙九也摞齐？」

「齐。」韩泥说。

库役嘟囔：「齐，少骂一句。少骂，也是饭。」

他不谢。谢轻了，功就飘；功在摞齐，功在少挨骂。

坡下霜更厚，柴捆勒肩。同批少年喘着问：「你肩不疼？」

「疼。」韩泥说，「疼也摞齐。摞齐，少扣半月麸皮。少扣，也是饭。」

「谣说你克粮……」

「谣是嘴。」韩泥说，「柴是肩。肩实，嘴就虚半寸。」

少年闭嘴，学他把绳结收紧——收紧了，柴不滚，克字也少一点借口。

酉时回丙九棚，谣已经长成句，句在灶边、兽栏门、渣场角乱窜：

「晦气杂役，编筐克粮，搬柴克运。」

「冬角死兽，也是他沾的。」

「备检灰线，克丙九灵根。」

韩泥喝粥，粥稀，却热。热在，谣也在。邻床少年挪远半尺，挪完又偷看他一眼，像怕晦气从粥碗里爬出来。韩泥不恼。恼像少年；他是记心翁——谁躲，谁近，谁开口，全记着。

刘婆端盆过，盆沿碰着赖福脚背。赖福骂：「疯婆子！」

刘婆回嘴：「谣从你嘴里长，长到你秤上，秤才歪！」

赖福脸白了，骂不敢骂狠——狠了，盆真可能扣下来。她给韩泥多舀半勺，半勺热在：「搬柴手别抖。抖了，柴滚；滚了，他们又说克运。」

「不抖。」

「谣呢？」

「记着。」韩泥说，「记着，熬到管事查。查清了，饭才实。」

刘婆哼一声走了。走得像给丙九留了一口暖——暖在谣线外头。

夜里赖福在廊下唱谣，唱得油：

「晦气翁，编筐筐，漏粮气，克丙九；搬柴柴，滚坡坡，克冬运，滚得快！」

同批有人笑，笑得浅，像怕管事。笑浅，谣却真在传。韩泥在棚内擦手，手裂处布丑，是刘婆缝的，丑却能挡风。他不出去辩。出去像求洗；求洗，又像认晦气。

邻床低声：「你真不吭？」

「吭，谣更大。」韩泥说，「大了，克的不止我，克全丙九。全丙九饿，赖福才笑。」

邻床闭嘴，翻身，身子仍离他半尺。外头谣声又起，起得像泥岗井台——井台谣他听了四十年，宗门谣才三日，三日也熟。

他低声道：「谣在嘴，活在我。我活一日，手紧一日。手紧，柴不滚，粮不漏，克字就虚。」

坛腹微温一丝，像应「记」字，不应辩。辩费气，气要留来搬明日柴。

丑时前，管事带人巡棚，靴声沉。赖福收嗓，收不干净，尾音还挂着「克运」。管事冷声：「谁造的谣？」

丙九静了。静里有人指韩泥，指完又缩手——缩手像后悔。管事扫一眼：「指人，要有据。据呢？」

那人哑了。管事掷木牌，牌薄：「明日查库。柴捆松一只，扣全丙九半月麸皮。谣再唱，记名。」

赖福脸青。青，算小胜——胜在管事肯查，胜在谣被按下一寸。韩泥不接话。接话像邀功；功在查，不在舌。

次日午后，管事真来查。韩泥柴摞齐，绳结实，一只不松。赖福那堆却塌了一只——塌处绳结松，松得像被人解过。管事冷声：「谁松的？」

赖福脸白：「风……」

「风不解结。」管事说，「赖福，半月麸皮扣你。韩泥，记着紧。」

同批少年碰他肩：「你早摞好了。」

「早好一日，少扣一日。」韩泥说，「少扣，也是饭。」

廊外忽然有人咳，咳得清，像执事。女子声不高，却冷：「丙九谣，污耳。沉丹宗不收晦气，也不收长舌。」

韩泥抬眼，见青衣执事立在灯下。名帖没看清，只看清袖线细，细得像符。赖福躬身：「秦执事，谣是杂役嘴碎……」

秦霜看他，不笑：「嘴碎，先割舌。割不了，就记名。」

她目光掠过韩泥，停半息，像看柴捆，不像看人：「搬柴的，手紧。手紧，柴不滚。滚了，才克运。」

「不滚。」韩泥应。

秦霜走了，步子实，像把谣切开一道缝。缝里夜风硬，硬里带一丝极淡的香——香短，不远，像南方来的。

同批少年低声：「她怎么帮丑杂役说话？」

「帮的是柴。」韩泥说，「柴不滚，全丙九少扣半月麸皮。少扣，也是饭。」

少年似懂非懂，闭嘴。韩泥鼻下一动，动完即敛。敛，像嗅诀里「鼻先静」。那香不像丹房药烟，不像渣场苦蒿，像谣里将要长出、还没长全的字。

他低声道：「谣记着。香记着。记着，别掺一处。」

邻床悄问：「秦执事谁？」

「执事。」韩泥说，「执事查谣，比赖福唱谣真。真了，克字就虚一日。」

外头赖福啐地，啐得不甘。不甘不要紧，要紧是谣尾被人截断——截断处又飘来半句新谣，不知谁传的，传得轻：

「晦气沾不得，沾了像栖香邪脉……」

句没传全，风就掐了。掐了，更钩人。韩泥独眼平，按心口。泥岗那几笔沉着，宗门刘婆一笔另热——沉的归沉，热的归热，不拿谣去糊汤。

刘婆端盆再过，瞥他一眼：「搬柴手别抖。抖了，他们又说克运。」

「不抖。」韩泥说，「明日问香，鼻到就行。舌不到，沿不到。」

她哼一声：「嗅着鼻，别嗅着心。心乱，香也乱。」

邻床又问：「问香要是验不出呢？」

「验不出，舌还在。」韩泥说，「舌在，谣还在。谣在，手更要紧。」

丑时末，丙九柱上又贴条。条新，字利，压一粒粟壳——壳里带极淡香，香短，像「栖」字落地那一半：

「卯时，药库坡。秦执事问香。韩泥，到。」

邻床嘀咕：「执事找你，晦气沾大了。」

「找香，不找人。」韩泥应，「人到，鼻到就行。」

他不躲——躲了，像认。认不得人，只认柱上那行字：卯时，坡见。

他收绳，绳结与筐口同理。明日卯时药库坡，秦执事问香——问到了，才分得出哪句是赖福造的，哪句有人在后头推。

查库小胜是真，秦执事截谣也是真。邪腥长，正温短；短像汤，七日后正测还远，明日卯时先验鼻下香。他合眼，掌心按着账——账在，人不虚。虚的是谣，不是他明天要走上坡的那半寸脚。

铁无言路过棚外，声低：「别飘。」

「不飘。」韩泥应。香在帖上，谣在廊外，手在绳结里。结实了，坡见才站得住。

他揭席看沿一眼，沿净。净，跟编筐那夜一样规矩。谣再尖，也锐不过明日坡上那半缕栖香——香短，真。真了，克字才虚；虚了，全丙九少饿半口。他合眼，掌心按绳结余温，像按住账本边。账在，人不虚。虚的是廊外那句没唱全的谣，不是他卯时要走上坡的脚。
