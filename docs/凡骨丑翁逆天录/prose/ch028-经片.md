# 第二十八章 经片

村外来个跛僧，袍旧，鞋破，眉白。

在庙前摆摊，不卖香，卖经片——铜片，薄，刻字，字是密宗祛寒纹，纹丑，真。真，不在美，在用。

韩泥扫庙换斋，路过，僧看他独眼，不躲：「绝户翁，坛漏，人别漏。」

韩泥停：「你说什么？」

僧笑，笑浅：「你屋角那只坛，底漏，命也漏。漏要补，补先补心。心暖，寒不侵。」

韩泥不辩信不信，只问：「经片何价？」

「不要钱。」僧推一片过来，「老耿让我捎的。他说，你抬棺稳，坛该稳。」

老耿。

韩泥愣半息，接片。片凉，贴掌，凉后一丝温，温得像凡符未燃先暖。暖浅，够记。

他低声道：「记耿恩。」

僧已走，走得不回头，像密宗线起，起在片，起在坛，起在55章凡符之前。之前，先有一片。

---

午后，村人围看，围得像看戏。戏是僧，戏也是片。片在韩泥掌，掌糙，糙得片不滑。

马春花尖：「和尚也找晦气翁？晦气传染！」

跛僧远，远得像没听见。听见，也不辩。辩，费舌，舌要留命用。

韩泥回屋，片压坛底，压严，像压一道外来的缝。坛底潮退半分，退得像应。应，不是醒，是稳。稳坛，稳人。

册上，又多一小竖，细，像经纹。恩在，物在，路在。路通向密宗，通向寒不侵，通向将来符录不越阶。

外头西教商队铃远，远得像五脉之一，尚未到村，声先到。声先到，心先动。动，不急，急便邪。

夜里，韩泥摸坛，坛温，温得像耿恩在底。底稳，顶才稳。顶是命，命还长，长得够还。

他蘸血，在竖旁划一细撇，像僧袍角。浅，真。真，便按实，按到册腹热。

「坛不漏，人不漏。」他低声道，「漏的，是谣嘴。」

嘴漏，用册堵。册厚，厚不过恩。恩厚，经片才肯来。

---

老耿名出，村里老人忆：「耿抬棺，绳不抖，棺不歪。」

韩泥听，不插嘴。插嘴，像攀恩。恩在，记便够。

里正来问经片，问得警惕：「和尚的东西，别惹邪。」

韩泥摊片，片铜，纹丑：「祛寒纹。我压坛底，不拜血祠。」

里正「哼」一声：「压好。压不好，连坐。」

「压实了。」韩泥说，「实，坛不漏；人不漏，谣就漏不进屋。」

里正无话，走了。走，也是默许——默许丑翁多一层防，防的是旱里人心邪。

叶青禾隔日送一小包盐姜，不放声。声惹祸，姜暖胃——胃暖，心便不易拜邪。

韩泥收姜，不入腹，先贴坛边，贴一息，取回，像借温。温借过，还记在竖旁，不欠叶家盐，欠的是情，情将来还。

经片在底，姜温在边，坛稳半分，人稳半分。半分累加，够走到沉丹宗风声贴榜。

他低声道：「密宗线起，起得静。静，才不邪。」

韩泥夜读片纹，纹丑，真，真便记。记耿恩，不拜血祠——拜邪，片会凉；不拜，片温一丝，温得像路。

路通向55章凡符，通向寒不侵，通向将来符录不越阶。阶在远，片在近。近，先稳坛。

马春花终是远问一句：「片能祛晦气？」

韩泥不答晦气，只答：「片能祛寒。寒在骨，不在名。」

名在嘴，骨在命。命稳，名自哑。
