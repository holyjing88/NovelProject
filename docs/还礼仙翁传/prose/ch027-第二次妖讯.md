# 第二十七章 第二次妖讯

坊市归途，镇妖司急帖至。

霍镇山亲来传帖，甲未卸，喘粗，马汗溅土：「东脉边村，小妖群再袭，苦战半日，请长老援！」

莫长春瞧帖，笑：「霍老头，你又喊。」

「不喊不行！」霍镇山道，「这一回妖小，却多，像赤焰试探阵脚。弟子伤七人，民屋焚三间，再拖，根就摇了！」

二人至边村，炊烟乱，犬吠绝，小妖群伏墙头，眼绿，爪利，比青狼弱，却像蝗，多。墙头、树梢、草垛后，绿眼连成线，线像赤焰投过来的一根针，专扎青岚门刚稳的阵脚。

村民缩在井边，孩子哭，哭被捂在嘴里。霍镇山盾亮，苦战，吼声震墙，墙震，妖不散，反扑更急。

弟子惊呼：「霍长老，妖太多了！」

霍镇山骂：「多就杀！镇妖司怕多，还是怕赤焰笑？」

围观村民呆住，像又看见那日边村得救，却不敢信——九月老头何时到？

莫长春袖空，一念出塔，杖落掌，杖至，不急，杖点地，竹影起，扫、缠、摔，像青狼袭村一幕复刻，却更快，像老人赶鸡赶多了，手熟。小妖被竹影卷起，摔进田里，摔进沟里，绿眼碎成一片慌。

村民跪谢，谢又被霍镇山吼回去：「谢镇妖司！功归司！」

莫长春摇头：「妖给你们，功给你们，人情铭塔壁。」

霍镇山愣，继而大笑：「就爱这句话！」

案结，执法堂补发曾被扣的培元丹例份，司库亲送莫长春院，瓶温，像迟来的正。

赵玄机立司库侧，面色平，低声：「长老例份，堂上已批。私扣之罪，弟子还库，不敢再犯。」

莫长春看他，笑：「还库好。库在，心在。」

赵玄机拱手退，退得硬，像把一句软话藏进袖里。廊下弟子瞧见，窃语：「赵执事给长老送丹？」「是堂上正份。」「还库之后，果然不一样。」

莫长春服丹，闭关三日。

第一日，脉沉；第二日，脉跳；第三日，脉开，像一扇门在筑基中期里推开一线——**筑基后期**。

出关时，须眉仍白，眼神却亮了一寸，亮得像九月里多了一盏灯。廊下弟子瞧见，窃语炸开：「破境了？」「靠培元？」「九月老头还能破境？」

柳青鸢亲来，递茶：「师兄破境？」

「丹是堂里正份，境是妖是霍老头逼的。」莫长春笑，「别高看老夫，高看了，又像演。」

柳青鸢松一口气，像东脉阵前又多一根能点的杖。

苏念慈在丹堂听见，炉火旺，旺得像替师兄喜，又酸，酸得像当年远处捏盏——她喜破境，酸破境因堂里丹，不因她炉边夜语。她咬牙，把酸炼进火里，火更稳。

夜，塔壁浮字：

**除魔再一念，境开一线。**

莫长春收念，望东脉，风腥。

边村老者跪谢，谢被霍镇山扶起：「谢镇妖司！」莫长春补一句：「谢自己没跑。跑了，根就断。」

执法堂贴告：补发培元例份，因除妖有功。弟子围告，围得像看迟来的正，正里仍有疑：九月老头靠丹破境，算不算本事？

苏念慈亲检丹瓶，瓶温，温得像堂里终于肯给长老一份不酸的甜。她低声对炉：「破境好。别死在丹前，别死在战前。」

赵玄机在告前立一停，未讽，像把「还库」之后的路，又走了一寸。

魏无涯观莫长春闭关，低笑：「境开一线，好让赵当众来问。」

顾小满册上又记：边村，小妖，二讯。记完，声平：「两回一致，真。」

柳青鸢令：「心要加，阵要加。」霍镇山拍刀：「战字既出，就不收回！」

出关时，廊下弟子炸开窃语，窃语里慕，也怕——怕九月老头还能破境，怕青岚门底又厚一寸。

边村战后，霍镇山验伤，伤七人，屋焚三间，焚得像赤焰试探的针，针扎在阵脚，阵脚未破。

村民扶老携幼，扶得像又信青岚一次——信在杖扫之后，不在嘴上。

他低声，像钩：

「境开一线，正好让某些人当众来问——问得越响，答得越巧。」

杖中竹纹一游，气音二字，像笑：

好。

大因果还在远处，像妖潮，像赤焰——破境小境，赵之初见才是大还礼前奏。

急帖又抄一份，贴至布告柱，红边如血。围观弟子围柱，挤而不乱，像怕讯传染——传染了，就得还。

莫长春袖空，一念出塔，至柱前，杖点贴角，不撕帖，只问：「两回一致？」

顾小满自人群后出，册捧怀，声平：「真。第一次边村狼群，第二次小妖群。回回写道，距东脉不足百里，旗动赤焰。」

赵玄机验讯符，符烫，烫里妒火也亮一寸，亮而不煽，低声：「讯至，门里该统。统在符，不在嘴。」

苏念慈开炉，炉旺，旺得像把讯炼成苦甘，低声：「讯别传成慌，慌乱了丹。」

有人故意喊「妖已至东脉」，慌起半圈。霍镇山眼刀一刮，刮得像再听一遍兽吼：「再传假，按军法！」慌声立哑。

边村归来弟子讲述战况：七伤、三焚、妖多如蝗。霍镇山验伤口，骂赤焰「针扎阵脚」，骂得凶，却给伤弟子按了军中伤药——药免费，记在镇妖司。

柳青鸢亲研阵图，阵脚又多一线竹纹，像莫师兄杖影落在图上。她低声：「师兄破境一线，东脉可多撑半日。」

周德海在商会得讯，讯说青岚柱前红贴又急，他抚须笑：「讯急，价先稳。稳不住，先卖慌。」笑里像等赵氏脸到，好抬一手价。

夜深，莫长春收念，望东脉，风腥。他低声：「讯急，见要来。初见，刀藏袖。」

延迟的爽点不在杖扫一瞬，在出关廊下窃语与柱前红贴同日——同日里，弟子又怕又慕：九月老头还能破境，还能扫妖，青岚底又厚一寸。

井边老者拉莫长春袖，袖空却不寒：「长老，下回还来？」莫长春道：「来。不来，根断。」

杖中竹纹再游，气音轻，像应：好。他望柱前红贴，贴红，红得像第二遍催命，却催得门里心齐了一寸。

柱前围观，有外门弟子壮胆问霍镇山：「霍长老，妖小还是妖大，哪个更险？」霍镇山瞪眼：「小得像针，扎脚才深！再问，回营跑十圈！」

莫长春听完顾小满报册，不看人群脸，只对柳青鸢道：「两讯一致，东脉阵脚可点两处。点一处防狼，点一处防蝗。」柳青鸢颔首，袖中剑穗无风自动，像阵图上又多一笔。

布告柱下，孩童踮脚指红贴问娘：「妖要来吗？」娘按他肩：「来了，有杖。」孩童听不见杖，只看见红，红得像陆上的第二道霞。

顾小满回册上添第三行：柱前，红贴，二讯验真。写毕，指节白，白得像除魔符后又添一张。

苏念慈连夜炼一批稳脉散，散给边村伤弟子托霍镇山带回去，药不责功，只防脱战。她对炉：「破境好，战前别乱。」

赵玄机验符毕，收符入袖，袖里已无培元私囊，却仍有妒。妒不对莫长春嚷，对己道：「统在符。符齐，心下一步。」

魏无涯远立檐角，扇面映柱前红贴，低笑：「境开一线，讯至二回，赵氏脸就要到。到之前，先让门里心齐。」

霍镇山收刀归营，刀鞘震地，震得像给边村伤者压一口气。他回头望柱前灰袍，灰袍袖空，却像比刀还稳。

莫长春袖空，一念出塔，杖点红贴下角，低声：「二讯验真，初见将至。将至之前，先让人心齐一寸，再让刀藏袖。」杖中竹纹应了一声，像笑。

柱前红贴验真，真了，赵氏收妒，霍氏收刀——刀在等下一阵妖风。
