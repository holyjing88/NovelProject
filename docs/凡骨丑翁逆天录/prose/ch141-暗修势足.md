# 第一百四十一章 暗修势足

夺魁后的日子，不像锣鼓，像一锅慢慢炖的粥。

韩泥天不亮就醒，先数息，再摸怀里册页。页上字丑，丑里写着真活：分拣、编筐、筛材。筛材是为了下一炉培元散——散不自己吃，要送给泥岗那位老耿。

外门鼓刚响，管事弟子便在廊下扬声：「夺魁者三日后验息登记。炼气十六，层实才记！」

「在。」韩泥只应一个字。应了，脚就不飘；脚稳，材才筛得齐。

同棚少年挤过来：「韩叔，十六层了？显摆一下呗。」

「没验，就不显。」韩泥把筐扛上肩，「显空的，像骗子。」

「你真闷。」少年撇嘴，「闷着也能成丹师？」

「闷着成的。」韩泥说，「嘴响成不了。」

药山坡上风硬，他按旧规矩把渣摊开。指尖捻细，鼻下静一息，只辨辛淡，不辨闲话。闲话有的是——丑丹师、运气魁、灰线三尺——他都听进耳朵，不往心里塞。心里这会儿只留三笔：散材、老耿、叶家那条递手的缝。

铁无言立坡下，抱臂，声低：「备材别躁。躁，手就飘。」

「不躁。」韩泥说，「飘了，散不正；不正，送不出去。」

「送出去再说。」铁无言顿了顿，「经叶家那条路，别自己撞门。」

「不撞。」韩泥应，「撞像攀。攀，恩就脏。」

日上三竿，刘婆端着稀粥过来，勺子在碗沿多顿一下，粥就高半寸：「登记前，别空肚。空肚，息虚。」

「谢。」韩泥接碗，「半勺我记下。」

「记啥。」刘婆哼，「记着别飘就行。飘了，手拿不住炉。」

赖福突然从廊角钻出来，火折一晃：「夺魁了，税还交不交？」

「交。」韩泥掏半文旧钱，「末席该交的，一个不少。」

「少哄我。」赖福鼻翼动了动，「你屋里香。」

「药山活，活有味。」韩泥声平，「味在渣，不在偷。」

赖福噎半息，甩袖走了。韩泥把半文钱的影按进心里——交了，才像积势；积势不是摆谱，是活稳。

未时，沈枯芽从坡下气喘吁吁奔来，手里捏半片枯叶，叶边焦黄：「韩叔……叶姐口信。」

「字呢？」

「没字。」芽儿脸红，「她说，谢老耿那条路，她接。她接缝，你别进门。」

韩泥接过枯叶，叶凉，像把叶汤温又绕过他一圈：「告诉她，谢了。我不近铺。」

「她说你记恩就行。」芽儿小声，「别让人看笑话。」

「笑话让他们看。」韩泥道，「恩我自己还。」

芽儿走后，邻床少年又探头：「真还耿老头？他耳聋。」

「耳聋心不聋。」韩泥说，「抬棺那天绳不抖，我记着。」

「一粒散换一根绳？」少年不懂。

「不是换。」韩泥望坛，「是还。能还才送。」

坊市末摊他仍去半日。牌歪着挂「丑丹师·嗅渣辨性」，人不多，风却杂。有人盯着他笑：「魁了还蹲尾摊？」

「尾摊能闻正。」韩泥辨完一捧灰渣，「正了，才配炼下一炉。」

「下一炉给谁？」

「该还的人。」他不多讲。

傍晚回丙九，他把灵砂、药渣、沉腐各按投序分格，格格写小字，丑而清。铁无言路过门缝，丢一句：「今晚睡实点。材料齐了，火才听话。」

「睡。」韩泥说，「睡不实也睡。」

「别通宵把劲耗光。」铁无言警告，「耗光了，炉前手软。」

「知道。」

丑时，棚里只剩粟壳香淡淡一线。韩泥坐坛边，掌贴坛腹。腹里温，温得像应「备」字——备齐了，才叫能炼；炼了，才叫能送。

「势足了。」他对瓮低声，「明儿起炉。起炉不为人看，为人手里那笔恩。」

坛不应话，只把那线温贴得更实。

廊下忽然有脚步，管事夜巡举灯一照：「韩泥？备培元散？」

「备。」韩泥答。

「册上有你名。」管事把灯移开，「别闹出岔子。岔了，十六层登记也记不稳。」

「记稳。」韩泥说，「手在，就不岔。」

灯远了。他编筐半刻，藤刺扎掌，刺浅，疼得清醒。清醒好——清醒才记得：刘婆半勺粥是一笔，叶家接缝是一笔，老耿抬棺是更早那笔。笔笔分开记，手才不乱。

夜里风硬，验息堂门还关着。关着不要紧，三日后开。开之前，他把散材再过一遍筛。筛齐了，才敢对瓮说第二句。

「材在。」他按木牌贴胸，「十六层在后头。谢老耿，在眼前。」

木牌凉，心却热。热不是喜，是担——担着一炉要正的散，担着一条不进门的路。路窄，够递一只灰布囊；递得出去，这一日积的势才不算白攒。

更鼓沉落时，邻床翻身迷糊问：「韩叔，你真不去泥岗？自己送？」

「不去。」韩泥说，「去了像显摆。显摆，恩就轻。」

「那你图啥？」

「图老耿收下。」韩泥合眼，「收下，我才睡得实。」

天边白一线爬上来时，他已经站在坛边。掌下温，像把「还」字提前焐热半分——焐热了，明日废窑里的火才肯认他。

午前他把培元散料称过三遍。灵砂轻重差半指，他就停手重筛。邻筐杂役探头：「韩叔，夺魁了还计较这点？」

「计较。」韩泥说，「差半指，炉里就差一气；差一气，粒就不正。」

「你累不累？」

「累。」韩泥擦手，「累也要把料办齐。齐了，夜里才有底。」

铁无言再送一筐细炭：「窑里缺这个。缺了，火漂。」

「记下了。」韩泥接炭，「漂了，散香就假。」

「假香骗得过人，骗不过鼻子。」铁无言道，「你鼻子还在，就别省炭。」

下午坊市尾又起一阵笑。有人指他牌：「魁师蹲尾，怪样子。」

韩泥辨完一封试渣，把钱掂一掂：「样子随人骂。渣正，我就收。」

「真不恼？」

「恼费劲。」韩泥收布，「劲留给炉。」

回袍棚路上，沈枯芽又跟人撞了个满怀，怀里多半块硬馍：「叶姐让带着。说筛材手抖，馍垫肚。」

「她人呢？」

「铺里。」芽儿摇头，「不让您去。去了又传闲话。」

「不去。」韩泥掰馍，「闲话她们爱传，馍我收下。」

夜里他独坐，把木牌、枯叶、料格一一摆开，像摆一桌不能错的席。席上没有酒，只有「还」字悬着。

「十六层登记在后。」他对自己说，「谢老耿在前。前后别乱。」

乱了手就飘。飘了，五年鼻子白练。

同棚有人梦里嘟囔「魁啊魁」，韩泥听着，只把炭屑又掸净一层。净了，才像外门日子该有的样子——苦，实，一步一步挪。

散队后他绕到验息堂外看了一眼。门还关，门环锈着一点绿。绿不要紧，三日后会亮。亮之前，他把门外石阶上的湿痕数清楚——数清了，心就不慌。

「看门干啥？」巡夜杂役喝问。

「看它三日后开。」韩泥说，「开了记十六。记之前，我把手里的散办完。」

「办给你自己？」

「办给该收的人。」韩泥转身，「收的人耳聋，心不聋。」

回棚他又磨了一遍投序口诀。口里念得低，低得像怕把「还」字念漏。漏了，炉就不认。

刘婆隔窗听了一会儿：「念啥呢，像念经。」

「念料序。」韩泥应，「念熟了，火前不懵。」

「懵了怎么办？」

「懵了就停。」韩泥说，「停比毁强。毁了，恩更还不起。」

窗纸一暗，刘婆走了。他知道她不信邪，只信人还活着、饭还热着。热着，路就还在。

临睡前他把灰布囊翻出来晾干，囊里还没有粒，却要先净。净了，明天粒才肯住。

「空囊也擦？」少年偷瞄。

「空囊更要擦。」韩泥说，「空着脏，装满也脏。」

擦完，他按心口，把三日后登记、明日废窑、叶家缝三条线又排一遍。排完，才敢闭眼。

闭眼前窗外有人低声赌：「丑翁明日炼不炼？」另一人道：「炼就炼，别吵。吵醒了，他记仇。」脚步远了。韩泥在暗里应了一声只有自己听见的：「仇后记。恩先还。」

火一认，粒才成；粒一成，这一身丑骨才配把抬棺那笔恩，一点点还回去。
