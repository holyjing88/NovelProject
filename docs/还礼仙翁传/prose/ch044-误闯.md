# 第四十四章 误闯

戌时，外门浴房外径，水汽混草木气，暧昧得像一场设计好的误会。

赵玄机在暗处弹指，一线灵息引路，引莫长春至岔口——莫长春本往丹堂送渣，脚步一顿，嗅出不对，像嗅出谣言的味道。灵息里无杀，只有「引」字，引得像戏台拉幕。

他止步，不入门，只立径外三息，杖点地，声轻：「浴房？老夫还没老糊涂。」

转身欲走，林挽月「恰好」路过，惊呼，袖风带起水汽，像一场戏拉到高潮。

暗处，执法堂弟子「恰好」巡至，目光一扫，扫出明日嚼舌的料。弟子眼里亮，亮得像终于逮住九月老头的丑，亮得像赵执事许诺的门规功。

莫长春知是局，却不点破，只叹，像叹给风听：「年轻人，急什么。」

他拄杖离去，背影稳，稳得像谣言砸不碎的一块旧玉。林挽月立原地，指尖凉，凉得像第一次觉出赵师兄的礼，也要还。

围观弟子其实未到——局要的是「恰好」的巡者，不是满堂闹，闹大了，反而不像谣言，像陷害。可远处廊角仍有人影，像顾小满，像魏无涯，像风，像将来要翻的堂。

赵玄机在暗处收指，指节白，白得像妒火终于点着纸，却烧不出预想中的轰响——莫长春止步不入，像把火掐在纸边。

林挽月捏袖，唇抖，终未喊住。她忽然明白：赵师兄要的不是真，是「像真」；莫长老给的不是辩，是「不进门」。

执法堂弟子低声问：「赵执事，这就够了？」

赵玄机淡道：「够了。明日，全宗都会替我们嚼。」

弟子笑，笑里脏，像已看见明日浪。

莫长春出院门，簿无字，袖却沉。廊下有人窃语「浴房」，窃语尚小，小得像浪还没起。

顾小满远影在廊角，像将证。魏无涯扇开一线，像看子将落。

赵玄机引路，莫长春止步，反差起——浴房局缺高潮，谣言要靠嚼补。

他低声，像钩：

「局成了半截。另半截，让浪自己起。起完了，才好还。」

风带水气，像谣言已湿，只等日晒。

弟子日后传此夜，传成七种版本，版本里都有「止步三息」，却无人说清：三息里，长老看见了什么。有人说看见了局，有人说看见了恩，有人说看见了九月将死前的清醒——清醒比糊涂更辱人，辱在赵玄机袖里。

林挽月立径外，指尖凉，凉得像第一次觉出：赵师兄要的不是真，是「像真」；莫长老给的不是辩，是「不进门」。

执法堂弟子回堂，低声报：「赵执事，长老止步三息，未入门。」赵玄机指节白，白得像妒火点着纸，却烧不出轰响。他淡道：「够了。明日，全宗都会替我们嚼。」

顾小满在廊角远影，像将证。她不知戌时发生了什么，只知心口那跳，跳得像半块灵石在警。

日晒之前，全宗尚不知——止步不入，才是还礼。

赵玄机引路，莫长春止步，反差起——浴房局缺高潮，谣言要靠嚼补。魏无涯扇开一线，像看子将落。风带水气，像谣言已湿，只等日晒；日晒之后，堂上要翻，翻的不只是浴房，还有赵执事袖里的妒。

执法堂弟子低声问：「赵执事，这就够了？」赵玄机淡道：「够了。明日，全宗都会替我们嚼。」弟子笑，笑里脏，像已看见明日浪。莫长春知是局，却不点破，只叹，像叹给风听：「年轻人，急什么。」他拄杖离去，背影稳，稳得像谣言砸不碎的一块旧玉。

风带水气，像谣言已湿，只等日晒。日晒之前，全宗尚不知——止步不入，才是还礼。廊角顾小满远影，像将证，像半块灵石在警。

林挽月立原地，指尖凉，凉得像第一次觉出赵师兄的礼，也要还。弟子日后传此夜，传成七种版本，版本里都有「止步三息」。

赵玄机在暗处收指，指节白，白得像妒火终于点着纸，却烧不出预想中的轰响——莫长春止步不入，像把火掐在纸边。反差起——浴房局缺高潮，谣言要靠嚼补。

围观弟子其实未到——局要的是「恰好」的巡者，不是满堂闹，闹大了，反而不像谣言，像陷害。可远处廊角仍有人影，像顾小满，像魏无涯，像风。

魏无涯扇开一线，像看子将落。莫长春叹「急什么」，叹给风，风带水气，像浪将湿。局成了半截，另半截，让浪自己起。

日晒之前，全宗尚不知——止步不入，才是还礼。谣言要靠嚼补，嚼够了，才好还。
