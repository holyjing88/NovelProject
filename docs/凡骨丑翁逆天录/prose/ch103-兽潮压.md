# 第一百零三章 兽潮压

气压低。

低像暴雨前，暴雨不要紧，爆的是兽。兽在界外聚，聚成潮，潮压药山，压得人喘。

韩泥仍东角，角草已黑，黑是爪刮，刮不要紧；刮在障上，障未破。

破，在一线。一线甜腥，甜像蜜，蜜是毒。毒入，小兽倒两只，倒不要紧；倒，要清。

清，他第三囊灰香尽撒，撒完，指间剩经页残诀——诀在口，口不出声；声出，像邪。

邪了，秦执事来。来不要紧，诀出，灰香起白雾，雾细，细清角内毒。

小兽缓，缓像活。活，是小胜；胜在活，不在嘴。

铁无言拍他肩，肩重，重不要紧：「角稳了。跟我挪三步，填障。」

韩泥挪，挪三步，三步稳。稳里，他见铁无言臂被爪风划一线，线浅，浅不要紧；浅，也要包。

包，用刘婆布条，条粗，粗才止血。血止，铁无言声低：「谢不用。手在就行。」

韩泥应声：「在。」

在，就是并肩的序——序在108全，这一章是压。

赖福躲障后，声抖：「怎么还不退！」

管事弟子喝：「退不退，守到卯时！」

卯时，远；远不要紧，压在前。压在前，心别飘。

白织月内墙，这回不嘲，不嘲不要紧；不嘲，像也怕。怕不要紧，角仍要人。

韩泥独眼平，蒙条湿，湿不要紧；湿，才挡甜毒。

夜尽，兽吼低一线，低像退，退是假；假不要紧，真压还在。

他回棚，掌茧新裂，裂不要紧；裂，是活计，不是血碰沿。沿净，坛才藏得住。

按坛，低声道：「压过了。角还在。」

坛腹温，像应「压」字。压，不是终；终在至。

丑时，静半刻。香落齐，境微动——动像五层圆满，圆满不要紧；圆满，仍不破，破在远。

邻床问：「还会来？」

「来。」韩泥说，「来了，仍守。」

更鼓尽，柱上条换：「兽潮压界，全员备战。」

备战，他已在战外缘。外缘，够丑翁站。

他低声道：「压在了。下一更，至。」

至，手在，囊在，经页在。

气压低，低像暴雨前。

韩泥仍东角，角草已黑。黑是爪，是毒，是汗。

铁无言臂被爪风划一线，韩泥用刘婆布条包。手在就行。」

「在。」韩泥应声。

管事弟子喝：「守到卯时！」

白织月内墙，这回不嘲。不嘲不要紧；角仍要人。

韩泥蒙条湿，湿才挡甜毒。

管事仍记：「丙九韩泥，角稳。」

记，是小胜；胜在活，不在嘴。

铁无言臂伤，伤不要紧，伤是并肩的证。证不在嘴，在布条，在血止，在「手在就行」。就行，够了；够了，108还要进深窝——进，不退；退不在路。

管事记「角稳」，记，赖扒皮脸青。脸青不要紧，簿上字真，真在「稳」，不在「丑」。丑，他熟；稳，他练。练在栏，练在角，练在清瘴路。

压界第二更，障木吱呀。吱呀不要紧，韩泥肩顶木，顶得要实。实了，丙九才不连坐。

铁无言并肩，声低：「压界，别退。退了，界破；界破，内门来。」
