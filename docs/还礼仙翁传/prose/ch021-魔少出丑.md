# 第二十一章 魔少出丑

赤焰退旗的第三日，青岚门主道比集市还闹。

赵玄机立于碑阴，听弟子讲，指节叩铁牌，一声比一声轻。他妒，却不再当众讽「演给谁看」——手帕定身那日他在人群里，妖骑散时他也在，演不出那种扫法，演不出那种稳。

林挽月路过，低声：「赵师兄，门规抄完了。」

赵玄机看她：「你觉得莫长老在演？」

林挽月犹豫，终道：「若演，演不出妖骑散。若演，演不出韩铁山退时那眼神。」

赵玄机沉默，像吞下一句话。话里曾有妒，如今剩疑：疑自己看不透九月老头，疑自己还库那日，心挪的一寸，到底挪向哪边。

碑阴外又聚一层人，外门弟子比划定身三息，内门弟子摇头：「你们没瞧见，厉无殇拭汗那下，笑还挂着，人已经钉住了。」

「像木偶。」

「像被人当众剥了皮。」

围观声浪一层推一层，推得赵玄机指节更白。他忽然转身，往执法堂走，走得快，像怕多听一句，多听一句，妒火又要烧回嘴上。

丹堂廊下，苏念慈听弟子复述定身三息，忽然问：「帕是谁做的？」

弟子愣：「莫长老袖里掏的……」

「符纹呢？」

苏念慈不再问，回炉，炉火一点即旺，像把一句话炼进丹里：轻礼可以重意，重意不必点破在廊上。

她想起丹堂夜语那夜，师兄说「药渣给你，是只有你收得下」——帕，大概也是。收得下的人炼得出，炼得出的人还得起。她指尖触炉盖，炉盖温，温得像那方素帕边角磨出的毛。

廊外弟子窃语：「苏首座又开炉了，定身跟炼丹有什么关系？」

另一人答：「你懂什么，莫长老送的东西，最后都进丹堂。」

苏念慈冷声：「闭嘴。开炉。」

执法堂偏厅，柳青鸢与霍镇山对图。

东脉灵田、妖潮路线、赤焰再犯窗口，一条条标红。柳青鸢指节敲图：「主战已定，三日之内，镇妖司前移。」

霍镇山抱拳：「莫长老杖扫妖骑，镇妖司愿记恩。恩怎么还？」

柳青鸢淡道：「记在青岚门根上。根在，恩在。」

霍镇山粗眉一扬：「霍某不爱欠人情。欠了，还；还不了，战。」

廊外，莫长春拄杖过，未入门，只留一句：「恩不用还我，还东脉民。」

霍镇山一愣，继而大笑，笑洪，震得图上红标都跳：「好！就爱莫长老这句话！民在，田在，盾才站得住！」

柳青鸢望师兄背影，眸光微动，不拦，不追，像二十年同门，终于不必问「师兄又要赠什么」。

山门外，赤焰营地，厉无殇闭门不见客。

帐内烛灭，人未睡。韩铁山跪报：「少谷主，青岚门主战，东脉增兵，镇妖司前移……」

「闭嘴。」厉无殇声哑，像牙还咬着那三息，「莫长春……一张帕。」

他砸杯，杯碎，瓷渣割掌，血渗，像砸不碎袖中记忆：定身时，全场笑，笑在弟子眼里，笑在柳青鸢眼里，笑在魏无涯眼里。三息不长，够一个人把颜面碎成渣，渣比药渣还轻，却比金丹还难炼回。

韩铁山低声：「少谷主，妖骑之耻，韩某记下了。」

「记。」厉无殇抬眼，眼里血丝与杀意绞，「记他九月，记他青岚，记他……敢用凡帕定魔少。」

魏无涯传讯来，只八字：「莫长春，气运异数，勿轻敌。」

厉无殇阅毕，焚讯，火起，像烧帕，烧不出。灰烬落案，他忽然笑，笑冷：「异数？好。异数才值得本少谷主亲自来取。」

夜，莫长春院中，桂香淡，秋虫细。

塔壁无新铭，袖却余温，像定身符意散尽后，还留着一点「礼已送出」的烫。他独坐廊凳，不坐石阶，不坐主位，像习惯把重话放在门外说。

杖中竹纹微动，气音轻，像问。

莫长春笑：「魔少出丑，恨意记在心里。恨比忘好，忘的人，还不了礼。」

他望东脉方向，云低，三日之期像帖，帖在天上。风里带腥，带远，像赤焰还没走远，像赵家妒火还没进门。

丑名传出后，赤焰随从不肯再提「定身」二字，提者被韩铁山眼刀一刮，刮得像再定三息。

莫长春不扬丑，只记丑，记丑像记礼——礼还了，脸也还了。

柳青鸢声稳：「三日之限，备要加。」霍镇山声洪：「备战，不备逃！」

赵玄机立侧，像等莫长春错，错没来，来的是顾小满一句：「丑是赤焰的，不是青岚的。」

苏念慈送茶至廊外，茶凉，凉得像怕席火进丹堂。

院外脚步声轻，三下，不重，像铁牌敲在人心上——来人了。

莫长春不回头，只低声，像章末钩落进夜里：

「来便来。妒火未灭，正好温一壶真话。」

卖符小贩被罚抄门规，抄边偷笑，笑里又怕——怕长老真从袖里再掏一张帕，帕若再出，黄纸更卖不动。

丹堂里，苏念慈守炉，炉火旺，旺得像把「凡物定魔少」炼进丹里。她不问帕谁做，问也没用——有用的是收得下的人，收得下的人还得起。

执法堂偏厅，霍镇山指地图，指节粗，粗得像东脉田埂：「三日之内，镇妖司前移。前移不是逃，是钉阵。」

山门外赤焰营地，厉无殇闭门，韩铁山跪报，报得像刀背刮喉。魏无涯传讯：「莫长春，气运异数，勿轻敌。」厉无殇焚讯，笑冷：「异数才值得本少谷主亲自来取。」

赵玄机转身往执法堂走，走得快，像怕多听一句，妒火又要烧回嘴上。他回想定身三息，回想妖骑散，演不出那种稳——稳的人，不像演。

林挽月把慕字写纸上，又撕，撕得像撕一根将断的线。她忽然明白：赵师兄的礼也要还，还得比慕更难。

顾小满在杂役院记：魔少出丑，三日之限，主战。记完，声平：「丑是赤焰的，不是青岚的。」

莫长春不扬丑，只记丑，记丑像记礼——礼还了，脸也还了。他抚杖，念：「丑出，和战要辩。辩比丑响，响里定调。」

塔中沉，沉得像丑脸还在识海。风过院墙，桂香与东脉稻浪气缠在一起，缠得像战字定了，妒火也到了门口。

碑阴外，弟子比划定身三息，比得像比命，不像比戏。赵玄机指节叩铁牌，一声比一声轻，轻里像还库之后又挪一寸。

丹堂炉火旺，旺得像把「轻礼可以重意」炼进看不见的丹。夜深，桂香淡，院外脚步声轻，三下——赵家妒火，果然上门。

执法堂偏厅，柳青鸢与霍镇山对图，红标跳，跳得像东脉脉门被钉实。厉无殇帐内烛灭，人未睡，砸杯声闷，闷得像砸不碎三息记忆。

莫长春院中，桂香淡，塔壁无新铭，袖余温，像礼已送出，恨已记在心里。他抚杖，念：「丑出，和战要辩。辩比丑响，响里定调。」

碑阴外，外门弟子比划定身三息，比得像比命，不像比戏。卖符小贩被罚抄门规，抄边偷笑，笑里又怕。

丑名出圈，圈外却有人笑——笑的是青岚门终于肯把礼，送到席面上。

弟子渐散，散里仍回头瞧一眼灰袍袖空——瞧得像瞧一眼九月还能走多久。

魔少出丑那日，赤焰旗还在，旗红像笑，笑青岚没割地，先割了脸。

莫长春袖空，杖在塔中，不追丑，只记丑。弟子围看，围看像怕丑传染，传染了就得还。

柳青鸢低声：「师兄，三日之限——」莫长春截话：「限是限，礼是礼。先还礼，再谈限。」

赵玄机立侧，铁牌在腰，笑浅：「魔少出丑，门里该统。」顾小满声平：「丑是赤焰的，不是长老的。」

苏念慈送茶至廊外，茶凉，凉得像怕席火进丹堂。她低声：「出丑出丑，别传成胜。」

围观弟子屏息，屏息等长老再辩——辩来，辩成两种风，两种刀。

莫长春念：「丑出，和战要辩。辩比丑响，响里定调。」

塔中沉，沉得像丑脸还在识海。围观弟子传谣，传谣像传戏——戏到下一场，和还是战。
