# 第七十七章 白织月再嘲

深行后第三日，内门小比余波未散。廊下还有人练剑，剑光两闪，笑也两声。笑声薄，薄也真——真在凡骨杂役仍末席，真在灰线三尺还挂着。

韩泥天亮先去清过兽栏半角。角净了，心里才空得出地方挨骂。挨骂也要腾地方——地方不腾，骂进来就跟恩缠一块，缠乱了，手最容易飘。他宁愿先闻一夜粪腥，再去丹房阶前挨那口香冷的嘲。

韩泥过廊，独眼平，不近。近像凑热闹；凑热闹要还人情，他还不起。白织月立在丹房阶前，裙素，袖香冷。她瞥他筐沿，筐空，空里还沾一点渣灰：「丙九也走丹房阶？阶上沾灰，像老鼠尾巴。」

韩泥肩稳，渣不漏，侧身让半尺：「走阶，不进门。灰在筐，不在阶。」

白织月声不高，高在众耳都听见：「凡骨末等，丹路断。断了还嗅药山？嗅出邪味，污我丹房风。」

内门有人哄笑。笑薄不要紧，辱在众目，手就不能抖；抖了，「滚」字就真。韩泥指节白了一息，白完又平。平，像把那一口嘲按进怀里玉牌——牌凉，仇烫。烫在还路前头，不在这一日。

白织月眉一挑：「嘴硬。三年后钱师兄让你滚，你滚不滚？」

「滚看手。」韩泥低声道，「手在，就不滚。」

「手稳有屁用。」她笑冷，「灵根末，稳也炼不出丹。」

笑里藏着断字：断丹路，断名，断幻想。韩泥不接断——接断像认晦气。他只认鼻下静。秦霜远廊经过，木鱼敲一声：「嘲归嘲，心别邪。」

白织月收笑，袖风一转，走了。走，也像给他留一口辱——辱长，才扎仇苗。韩泥不追。追像求；不求，只记。记「丹路断」，记众目下那一瞬，记柱上三年约还凉着。

阶下有个杂役少年咬牙：「她凭啥骂人炼不出丹？」

「凭灵根。」韩泥说，「凭灵根骂，咱就凭手活。手活在，骂就短一分。」

少年噎住。内门剑光又闪一下，闪完仍笑。笑像风，风刮过，韩泥筐绳勒得更紧。紧了，肩才稳；肩稳，才不像被她袖风刮跑。

午后兽栏仍腥。他铲下稳，稳得像廊下那句「手在」。管事路过，记一笔：「丙九韩泥，正测后仍清细。」赖福晃秤过来，眼细：「白织月也嘲你？嘲对了，末等炼什么丹。」

韩泥不接茬：「嘲在舌，不在掌。」

掌里有二层，掌里有嗅芽将开——开在鼻，不在舌。舌响，香飘；香飘，手就滚。刘婆端粥来，仍多半勺：「丹房阶前挨嘲了？」

「挨了。」韩泥说，「不辩。」

「不辩怕什么。」刘婆哼，「手稳，饭稳。饭稳，恩不断。」

韩泥点头。恩在粥里，也在不辩里——不辩不是怂，是刃；刃不亮，先记。铁无言抱臂立栏边，声低：「嘲别贴脸。贴了，鼻下飘。白织月话毒，毒里七分真——真在灵根末。末不要紧，渣里活，活长了，丹路自开。」

「不贴。静在鼻。」韩泥应声。

铁无言又道：「三年约是秤。秤在怀，别被一句断字压塌。」

「压不塌。」韩泥摸了摸牌边，「牌凉，手热。热的在干活。」

黄昏药渣场，灰起一线。他分拣，鼻下静，辨出假香上浮，指到即剔。十堆连对。管事眉松，记「辨性准」。准，是小胜。胜在指，不在嘴。白织月若知，大概仍说丹路断——断在舌，不断在渣。

分完堆，他袖口蹭灰，蹭得像普通脏手。脏手好看——好看在不像「嗅得出药性的人」。内门弟子远远瞥一眼，骂句晦气，走开了。走开正好。走开会盯、会抄、会往钱戾衡耳朵里塞话。他宁可被当晦气，不当靶子。

邻床少年悄问：「白织月长得好看，你心里堵不堵？」

「堵在辱。」韩泥说，「不堵在脸。脸在皮，辱在众目。」

「那你咋不回嘴？」

「回嘴像跪。」他声音哑，「跪了，路断。不跪，只记。」

少年不懂「众目」，却懂「不跪」，闭嘴睡了。丑时前，韩泥廊下练半刻闭息。息在鼻，不在嗓。嗓响，香飘；香飘，她就笑得更真。他按席角，席严，坛不露，低声对坛：「她嘲丹路断。叔记着。断不断，鼻下见。」

坛不应字，只温。温像把「嘲」字摁进仇线，不摁进恩线。玉牌在怀，愧字在怀，耳边还回着那句「炼不出丹」。耳响，心不响；心一响，手就飘。

第二日外堂阶前，他扫阶，帚稳。管事验阶，阶净，又记「洒扫净」。记净两笔，饭才能多撑几日。午后又逢药渣场，管事加点验：「错一堆，记名。」

韩泥指稳，鼻先落，十二堆全对。对完不抬头。抬头像邀功；功在饭，不在脸。赖福立远处砸嘴：「准也末等。」

韩泥只当风过。风里忽然飘来丹房炉息，他鼻下一滞，立刻退半步——退不是怕香，是怕闻太清。闻太清，人就想追；一追，嘲里那句「丹路断」反倒坐实了。他宁愿慢，慢到渣堆肯认他的手指。傍晚白织月远廊又过，袖风冷，没停——没停也是小胜：胜在他没抬眼去追辱。沈枯芽怯生生跟半步：「韩叔，心里还堵吗？」

「堵过。」韩泥道，「堵完干活。干活，手才在。」

芽儿退了。他编筐半刻，藤刺扎掌，偏手，刺浅些——浅些，沿才净；净了，坛才藏得住。编筐钱买米，米在，恩才还得上；还得上，才对得起这一口嘲。筐沿磨出一道新痕，痕浅，像仇苗刚拱土。

夜里管事验棚，灯照席角，沿净。韩泥合眼不睡实。掌纹那路转半周，二层稳，稳里有一点芽意——芽意淡，淡也是路。他摸怀木牌，牌凉，膝热：「嘲了。记着。芽要开，别在她眼前开。」

远处丹房又传来一声轻笑，不知道是不是白织月。他不去辨。辨笑声费神，神要留给明日分拣。分拣准了，饭才实；饭实了，骂才砸不穿骨头。骨头还在，三年约就还没被人笑崩。

更鼓沉下去。廊下练剑的笑声还偶尔飘一下，薄得像纸。纸不怕，怕的是心里跟纸一起薄。韩泥把粥恩、渣准、三年约分开放——混了，手就抖。手不抖，嗅诀那一夜才接得住。接住了，她断的丹路，才轮得到他在渣里自己开。

散队后他路过丹阁侧廊，廊扫痕直，直得像秤。他不踩灰，肩稳，筐空——筐空，像只清栏，不像嗅药。风里一点冷香擦过鼻尖，他鼻下仍静。静住了，嘲才算白挨；白挨不要紧，记仇苗，也记明日芽。

棚后编筐角，藤条还沾着霜。他坐下再编半升筐，藤刺扎指，他偏手让开。刺浅些，血才不碰坛沿。邻床凑过来看：「你挨骂了还编？」

「骂在耳。」韩泥说，「筐在手里。手里的能卖钱，耳里的只能记。」

「记了有啥用？」

「用来还。」他把筐沿按实，「还汤，还针脚，还那口辱。还不起大的，先把手磨利。」

少年缩回去。远处内门又笑两声，笑薄，刮不过他膝上那圈藤。藤在，人在。人在，丹路她断得了嘴，断不了他鼻下那寸将芽的根。

根藏好，明日丑时才配让它冒一线。冒一线也不喊；喊了，阶前那口嘲就成真锁。锁开开不在嘴硬，在手稳、鼻静、饭还进得了肚。进肚了，才配把「丹路断」三个字慢慢顶回去。顶回去的夜，不在阶前，在渣场，在丑时，在没人听见的鼻息里。鼻息一稳，嘲就短。短了，仇苗才扎得深。深了，才配挨下一口。
