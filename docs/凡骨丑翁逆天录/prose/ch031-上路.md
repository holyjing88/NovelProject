# 第三十一章 上路

辞村那天，霜白得刺眼。

韩泥背篓出泥岗，篓里沉着药、馒、袄，还有坛底那片旧经。六笔恩捆在肩上，瘸足一步一步踩实。村口老槐已落在身后，槐下无人，风过，像把「晦气翁」三字吹散半寸——散不全，够他往前挪。

官道窄，车辙深。挑担老汉超他半步，瞥一眼篓，不看脸：「去镇？」

「去宗。」

老汉脚顿了半息：「沉丹宗？」

韩泥点头。老汉把脸猛地别开：「五十八岁，凡骨，去那地方做杂役？」

「做。」韩泥只回一字，「活。」

老汉摇头快走，走得像躲瘟。韩泥看惯了这种躲——习惯不等于退，退字他心里早就挖掉了。

午前歇进茶棚，棚里三五人。有人笑得又短又尖：「晦气翁也赶考？考什么，考克宗门？」

韩泥坐棚角，掏出张麻婆的馒，掰一半留篓，一半慢慢嚼：「克不克，宗门说了算。我说不算。」

笑的人一滞，茶棚伙计敲桌：「喝茶的少嚼舌头。嚼多了，滚。」

棚角另有人低声：「五十八去宗门……倒有种。」

种字轻轻落进耳里。韩泥当夸不接，当骂也不回，只管把馒吃实。这是上路第一场小胜：笑还在，嘴先闭了。

老板娘端来一碗粗瓷：「老头，喝口热茶。茶钱我出。」

韩泥摇头：「不欠。」

「不欠也喝。」老板娘说，「晦气翁走远路，渴死半路，晦气更大。」

他接碗喝一口，烫得像叶汤送行。放下碗只说：「记着。」

午后坡陡，膝上旧伤顶了一下。他停住，扶着担，独眼望坡顶。方才那挑担老汉喘得厉害，汗里带苦：「你……还走？」

「走。」韩泥放下篓，从叶振东那包药里拣两叶苦蒿递过去，「嚼。止喘。」

老汉愣住：「你不记我躲你？」

「躲是你的事。」韩泥说，「喘是你的命。命在前，躲在后。」

老汉嚼叶，喘平半分，挑担走远，还回头看了他一眼——那一眼真，像泥岗卖盐老汉那日点头。韩泥把这一眼也记了：路还长，人还能帮一口，就不算绝户绝尽。

官道中段，小石桥下有婆子淘米。婆子瞥他篓：「去宗门？带坛了？」

韩泥脚停半息——这句像从泥岗追了三十里。婆子扁嘴：「听说的。晦气翁带坛，克一方。」

「克不克，宗门说了算。」韩泥独眼平。

「宗门也怕晦气。」

「怕不怕，门开了再说。」他抬担，「水凉，快淘米。米沉，别浪。」

婆子一滞，尖气收了回去。收一半，路就还直。

坡尽处驿亭在望。亭里贴着旧榜，字是沉丹宗外门杂役招募。韩泥站榜前细看：「凡骨可试，年五十上限六十」——五十八岁，两年窗，够钻。

亭吏嗑瓜子：「报名？带荐帖？」

韩泥摸怀：「在。」

亭吏展帖愣半息：「言伯钧那纸？……晦气翁还真去？」

「去。活。」

亭吏指驿道：「西三十里到西驿。验骨在明日卯时。别死半路。死半路，连坐泥岗。」

连坐二字他不入心。六笔恩在肩上，死不起半路。

驿亭旁卖绳老汉问：「去西驿？绳要一根？担沉，绳断，坛碎。」

韩泥摸篓绳，绳还韧：「绳够。坛在，绳在。」

老汉啧啧：「带坛进宗，头一回见。」

「头一回，也是最后一回这么带。」韩泥说，「进了门，坛藏席下。藏了，不碍眼。」

日偏西，路牌立出来，木牌旧，字却深：

沉丹宗 · 西驿三十里

他望着字，心里那根钉子从泥岗拔出，钉进宗门这一篇。掌按在腹前，温从篓底坛沿传来，应的是脚步，不是名声。

夜宿破庙，风穿瓦缝。韩泥盘坐篓边，摸坛沿，一线温像耿老的送行，也像叶汤的余热。邻村乞丐塞他半块饼：「路上吃。别死半路。」

「活一日，还一日。」韩泥收饼，谢字不吐。

乞丐又道：「听说杂役活过三年，才能摸丹炉边角。你五十八，三年后六十一。」

「六十一，也是活。」

天将亮，挑脚夫瞥见荐帖抄本：「言伯钧那老儿还收人？」

「收。」

「你什么骨？」

「凡骨。」

挑脚夫嗤一声：「凡骨走西驿，测骨，登记。测不过，半路滚。」

「测过才知道。」韩泥点头，「没测，不算滚。」

挑脚夫看他独眼，笑意收了：「有种。别死半路。」

「死不了。」韩泥说，「恩未还，命不肯死。」

破庙檐下，茶棚里笑过他的人也歇脚，笑没了，嗓子沙：「晦气翁，还真走？」

「走。」

「宗门收你，我输半块饼。」

「饼记着。」韩泥应，「不收谢。谢轻。」

那人别过脸去，别脸像茶棚闭嘴——嘴闭了，路才直。

庙顶漏下一点雨，落在坛沿，水汽一闪便敛。韩泥不追问，只把手按实，按到丑时将近。他把六笔又在心里过一遍，清楚：叶汤、芽粥、陈袄、张馒、耿经、叶父药。辱线另搁，仇线另记，今晚不拿来搅路。

路牌旁有弃卒石，石凉。他踩实，像把泥岗最后一口谣踩进土。官道尽头驿亭旗猎猎，旗旧字深，深得像里正那枚出门印。门开了，恩还在身后钉着。他望旗半息，不拜，只记：记路，记恩，记西驿验骨。

沈枯芽娘在岔路口远站，娃没追上来，只喊：「韩爷爷，六笔我记住了！」

韩泥停半息，不回头：「记住就好。」

娃在远处「嗯」一声，像米粒又落秤盘。他这才继续走。

官道旁一辆弃车，车轴断了，车主蹲着骂：「晦气！轴断在这！」又骂：「带坛的别靠近！靠近克人！」

韩泥绕远半步：「克不克，宗门量。你轴断，是你路。」

车主骂声收半寸。收半寸，也算小胜。

午后再歇，他数篓里物。药须缠实，叶脉还湿；馒布硬，陈袄补过，经片在底，「别病」二字仍清楚。老耿那片经在坛底微温，温得像送行。他复述爹的话：「这是你爷爷传我的，我也这么传你。」又补：「坛在，韩家不散。」

散的是村口槐影，不散的是肩上篓。篓在，六笔在；六笔在，西驿才验得过骨。

挑脚夫又道：「西驿验骨，丑的也收？」

「收不收，测过才知。」韩泥说，「没测，不算滚。」

「滚了，回泥岗丢人。」

「丢人另算。」韩泥独眼平，「活命先算。活不了，恩成空。」

挑脚夫闭嘴，像被「空」字堵住。

夜过丑时，他摸荐帖抄本，纸边磨毛，像五十八年摸出来的命。命在纸，纸在怀，怀对着宗门那条窄缝。缝窄，也要钻。钻过去，才有丙九末席，才有兽栏练手，才有将来把那碗烫汤还回去。

路旁有个赶驴的汉子路过，驴铃乱响。汉子瞥荐帖角：「真去西驿验骨？验不过，丢脸丢回村。」

「丢脸另算。」韩泥说，「先把骨量完。量完了，人才知自己站哪。」

「站末席呗。」汉子嗤笑。

「末席也是席。」韩泥抬担，「有席就有饭，有饭就能还。」

汉子一噎，驴铃晃着远了。韩泥把这话也按进心里：席可以丑，饭不能断。断了，叶家那碗烫就白烫了。

破庙后院忽然窜出一只瘦狗，冲他篓叫了两声。他掰半块馒丢过去，狗叼走，不咬人。他把这一幕也记了——路上能分一口就分一口，分一口不是散财，是不准自己先变成村里人嘴里的「绝户」。

天光一线挤开雾。风过官道，极轻一声，仍不是名字。韩泥按心口，继续走。沉丹宗路牌在前，雾白得像关口。关口那边是西驿验骨——骨在，命在；命在，叶汤便还不至于凉。他望着雾白再停半息，瘸足踩实，像把辞村那一夜霜踩进土里。土里埋的不是怕，是约：约验骨，约侧门，约活长。活长了，才有资格把「还」字从心里掏出来当众立住。
