# 第七十八章 嗅诀萌芽

嘲后第五夜，丑时。韩泥未睡实。睡实了，芽就藏不住；藏得住，才配开诀。

白织月那句「炼不出丹」还挂在耳廓边上，像冰碴。他不去抠。抠了费神，神要留给鼻子。鼻子这几日越来越刁：假香一浮，指就痒。痒了还不能当众快，快了惹眼。只好把快劲儿憋到夜里，憋到豆灯光稳、邻床鼾起的时候，再放给自己。

他盘坐床沿，席严，坛不露。油灯豆，豆光稳，稳像筐底那撮深灰——沉在底，不浮。掌贴坛沿，沿温一线。温进掌纹，二层气转一周、两周；转到第三周，鼻下忽然静到极。极静不是没气，是气只肯走鼻。

走鼻，不走舌。舌一响，香就飘；香一飘，白织月的笑就更真。他取出白日样纸三包，摊开。灰里辛淡，淡不要紧，辛在，芽就在。

第一包，鼻下一吸，辛沉。第二包，苦浮。第三包，腐底一线，底真。三包并息，息落在鼻端，掌纹微颤。颤不像惧，像认——认渣在说话，认药性归位。忽然颅骨里一声轻，非钟非雷，像瓮唇吐息。

吐息里浮出三字，浮在鼻端，不入口：

「沉腐浮。」

三字丑，丑而真。真在诀，不在册；册上没这行，诀在他自己鼻子里。他屏息，独眼不眨。眨了，诀就散。低声复诵，只复给自己听：「沉腐浮。沉者药，腐者毒，浮者假。」

复诵不在舌，在鼻。鼻里辛苦腐三线并，并出一线清——清像渣场风硬那日，铁无言指过的三堆灰。芽起了。起不像光，像鼻下多了一寸根。根细，细才藏得住；藏得住，煞骨门才嗅不准。

邻床翻身，眯眼半息：「啥味儿……」

「坛味。」韩泥声平，「睡吧。」

小杂役哼一声，又睡实。睡实了，诀才藏得住。他收功，收得像盖席。盖严，沿净，样纸仍包，包实。实了，芽才不离鼻。更鼓将尽，他廊下立半刻，任风硬，鼻下仍静。静里，三字不落舌，只落鼻。

风从药山那边来，还带着一点焦回。他故意不追那味——追像贪；贪了，芽就飘。他把三字在鼻里滚一遍，滚完按进掌纹：沉者入底，腐者靠边，浮者先剔。滚完，才像真有钥匙，不是做梦。

卯时兽栏腥重。腥不要紧，鼻下有根，铲下更稳。管事弟子巡栏，记一笔：「丙九韩泥，清细。」刘婆端粥，仍多半勺：「昨夜又静？」

「静了。」韩泥说，「鼻下有根。」

刘婆不懂「根」，懂「静」：「静好。饭稳，恩不断。」

韩泥点头。恩在粥里，也在诀芽里——芽轻，记重。同批少年盯他碗：「你鼻下长啥了？说话谜。」

「长记性。」韩泥喝完，「记性在，手才快。」

午后药渣场，灰起，他分拣：不需细想，鼻先落。辛左，腐右，假香上浮，指到即剔。分堆快半分，快也要稳——稳了，管事才不记「飘」。管事看得眉跳：「今日手快了。」

「手跟着鼻子。」韩泥低声道，「鼻子不飘，手就不飘。」

旁边有人试着学他快，快完三堆全错，被记名骂出去。韩泥装没看见。看见要教，教了露馅。露馅比记名可怕——记名不过饿饭，露馅能惹煞骨门那一嗅。他宁可让人骂独食，不让人闻出他鼻下多了根。

铁无言走近，瞥分堆：「对。芽起了，手才快。别叫诀名。」

「不叫。」韩泥应声，「沉腐浮，记着。」

铁无言点头，走了。走也像认可——认可不重，重的是渣场仍收他。白织月远廊过，袖风冷，没停。没停是小胜：胜在鼻，不在嘴。秦霜木鱼轻敲：「诀在鼻，心别贪。贪，邪。」

韩泥不答。答在心里：诀是路，路通向废渣收集，通向温听雨初识，通向丹路开。他按席角，低声对坛：「诀起了。三字记着。不叫。」

坛不应字，只温。温像把「沉腐浮」摁进掌纹那路——路在，就不滚。邻床又问：「你分拣咋快了？」

「鼻下有根。」韩泥说，「根在，手才稳。」

少年不懂，闭嘴。赖福讽远：「末等也分拣快？快出邪。」

韩泥不接。舌头要留给鼻子，留给守坛。守坛，守诀，守三字不落舌。黄昏样纸再验，连中。连中稳，稳不像炫——炫像求，不求，只记。他回棚把三包装进夹层，夹层布旧，旧得像泥岗那些年——旧布藏得住新芽。

第二日渣场加验五堆。他鼻先落，五堆全对。管事记「芽后仍准」。铁无言取未知灰一撮：「再闻。」

韩泥鼻下一息，指落：「沉三，腐一，浮一。」

铁无言分秤，对了，点头：「芽起了，别叫。」

「不叫。」韩泥道，「叫了，像求诀。」

铁无言临走丢下一句：「冬深轮换要来。芽起了也别懒。懒了，三字退成空响。」

韩泥记住「空响」两字。空响比挨骂可怕——骂还能记仇，空响是自己把根掐死。他当晚把样纸又摸了一遍，摸到纸角起毛才停。

温听雨远廊过，裙素，袖香温，她没近，只一眼。一眼也像验手稳。刘婆又多半勺：「芽起了别飘。」

「不飘。」韩泥说，「飘了，手就滚。」

「滚了婆的粥也喂不饱三年。」刘婆瞪他一眼，「手按着。」

夜里他再练半刻。三包并息，并出一线清。清像嗅诀小成前那口息。柱上条角白一线，冬深风硬。风硬不要紧，鼻下有根——根在，就不滚。他摸样纸，纸糙得像路，低声道：「芽在近。开丹路还远。远不要紧，近先守住。」

更鼓二响。散队后，他过坡下测灵石。石凉，灰线还公示着——公示不要紧，鼻子里有根，手要紧。铁无言脚步远，停半息：「手稳就行。稳了，冬深那些活才接得住。」

「稳。」韩泥应声。

稳了，冬深脏活才接得住；接住了，四层影才肯露面。他合眼不睡实，鼻下仍静一息。静里，沉腐浮三字像根，根细，细才藏得住下一轮杂役与更远的废窑火。火还远，芽已在——在就别喊；喊破了，白织月那句「丹路断」才真要成真。

棚外更鼓又沉一分。他把木牌系紧，系得像把仍末摁进掌心。末在册上，芽在鼻下——两件事并排，不打架。不打架，才配进冬深轮换；进了轮换，才配把明日那些臭活一件件扛过去。

丑时将尽，他又摊开一包未知灰，自测半息。鼻下一落：沉二，腐半，浮一。浮的剔掉，指尖不抖。不抖才像诀真长在肉里，不是夜里发烧。样纸收回夹层时，布边磨破一线，他拿针——刘婆针脚教的那种歪针——缝了两针。缝完低声道：「芽在布里，不在嘴上。」

邻床迷迷糊糊问：「你跟谁说话呢？」

「跟手。」韩泥道，「手听话，冬深才熬得过。」

天亮前他去兽栏换草，草干味里仍能辨出昨夜焦回那一点。一点就够。够了，柱上新条「三日一轮」才不吓人。吓人的是手软；手软了，芽白长，名也白在。他贴了贴柜角三纸包，包实，像给自己立规矩：冬深轮换来了，三字仍不叫，手仍要稳。稳着进杂役续，才配谈后面那盏门缝灯。

风雪从门缝灌进来，灌到脚背。他跺一下，跺醒冻麻的趾。趾醒了，人才能跟上轮换锣。锣一响，就没有「昨夜萌芽」的浪漫，只剩畜栏臭、山坡霜、藤条刺。刺得过，芽才算真。真了，才配进冬深杂役续那一串脏日子。脏日子不怕，怕的是芽飘成空响。空响一出，路就断。断不得。
