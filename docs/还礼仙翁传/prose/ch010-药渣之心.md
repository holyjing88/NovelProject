# 第十章 药渣之心

苏念慈送药上门时，脸色比药汤还苦。

「师兄，外头说你老不正经。」她站在门槛，不进，像怕进门就坐实谣言，「我听了一上午。」

「听累了？」莫长春让座。

「累了。」她进门，把药壶放下，壶盖微开，热气扑脸，「更累的是，他们还说你赠剑赠符，赠到杂役头上，唯独不赠丹堂——好像丹堂不算恩，算应当。」

莫长春看她：「念慈也觉得老夫该赠丹？」

苏念慈一窒，随即硬声：「我想赠师兄活路。」

「活路在塔里。」他指袖，「不在丹。丹堂给你，是应当；药渣给你，是只有你收得下。」

她沉默，忽然从篓中取出一包药渣，正是她最熟悉的东西，苦里带甘，甘里带炭：「那师兄为何总赠渣？外头说你在羞辱丹堂首座。」

莫长春接过药渣，嗅了嗅，笑：「渣里有你熬夜的气。羞辱？羞辱是给人看的；赠渣是给你看的。」

苏念慈眼眶一热，别脸：「……我送培元汤来，师兄回赠渣，像骂人。」

「好丹给你，是应当。」他把渣递回她掌心，「药渣给你，是只有你收得下。丹堂首座，收得下苦，才炼得出甜。」

她指尖颤，袖中无簿，心口却一跳，像回赠将至——她不懂塔，却懂那一跳与回春丹有关。

下一瞬，案上落两瓷瓶，回春丹，瓶身温，像刚从炉里出炉，温得像有人把她熬夜的气，炼成甜，又送回来。

苏愣，继而红着眼笑，笑又像哭：「师兄……你使诈。」

「不使诈，」莫长春道，「你肯收渣，天道肯给丹。各算各的恩。」

苏念慈握瓶，低声：「回赠……不是还债。」

「对。」莫长春点头，「你终于会了。」

她忽然问：「师兄，若我明日仍送药渣来，你还收吗？」

「收。」他答得干脆，「渣是渣，你是你。渣里若有你熬夜的气，老夫就记得你哪一夜守炉。」

苏念慈别脸，别脸像把红意也别进袖里：「……那我明日多带一包。包重，你记得久一点。」

廊外，弟子窃语未歇，这一回却多一句：「苏首座红着眼出来……像哭又像笑。」

赵玄机在石道远处听，冷哼，哼里空了一线——若药渣也能换丹，他那三颗私藏培元丹，算什么？

柳青鸢遣人送帖，六字：「师兄，别听风。」莫长春回帖四字：「听风也好。」

她走时已近黄昏，丹堂重开炉火，烟起，柳线剑意远，顾小满除魔符贴身，外门谣言未歇，却像被这缕丹香盖去一半。

莫长春独坐廊柱旁，不坐主位，识海塔壁金铭浮起一行：

**药渣亦礼，回赠有时。**

杖中竹纹游动，气音仍是一字：等。

莫长春笑：「等。」

九月还剩八月余，够送，够还，够把青岚门这锅药，熬浓一点——浓到连外门嚼舌的人，都闻得出丹香压过流言。

苏念慈握瓶回丹堂，炉火一跳，跳得像妒里掺敬。她低声对弟子：「渣不渣，别传。传多了，传成辱。」

弟子噤声。苏念慈望小院方向，袖中回春丹瓶一紧，像握着一句没说出口的「我也能还」。

围观弟子在院外踮脚，踮得像等苏首座再哭再笑——哭笑没来，来的是丹香，香压过流言半寸。有人低声：「药渣换丹，九月老头使诈？」另一人答：「使诈的是天道，不是长老。」

苏念慈在炉边把药渣重新分拣，分拣像把师兄赠的渣，与自己的渣，一并炼进下一炉。她低声，仅自己：「渣好，丹更好。各算各的恩。」

霍镇山路过丹堂，声洪：「苏首座回春丹成，镇妖司也记在心里——药渣换甜，换得正。」周德海抚须，笑和气缓：「回赠，不是私。」

莫长春不入席，只立廊柱旁，袖空，杖在塔中，塔中却沉，沉得像药渣之心已记进格子里。他不追，不扑，只等——等礼自己走到该走的人面前。

苏念慈真听见弟子私下比谁的渣更苦，不骂，只道：「苦比完，去炼甜。比渣，不如比功。」弟子噤声，噤声里掺敬。

赵玄机在执法堂抄门规，抄到「私赠丹药」一条，停住，像想踩，又踩不实——回春丹是苏氏回赠，名分上挑不出刺。他最终只低声：「回赠有渣为证，私赠无渣为耻。」

莫长春不入院，只留帖四字：「渣好，丹更好。」帖贴丹堂门，门像被温了一下。

更深，苏念慈在炉前把回春丹入匣，匣温，温得像回赠终于落了地。她望小院方向，一线夜灯已灭，丹堂灯仍亮——亮得像替师兄守一句「别误炼丹」背后的真意。

林挽月路过丹堂，低声问弟子：「苏首座……真哭了？」弟子摇头：「没哭，是笑。笑得像哭。」林挽月一怔，怔完仍往执法堂走，走前目光扫过丹堂窗影——影暖，暖得像挑不出刺。

顾小满在通铺角，把除魔符与半块灵石并藏，藏得像并藏两条还不清的恩。她低声：「药渣换甜，记下了。小满不会还错人。」

丹堂后院，苏念慈把回春丹分装入匣，匣分三等：一等给伤重弟子，二等给杂役例份，三等留丹堂备急。她指尖抚过匣盖，抚过像抚一句「我也能还」。弟子问：「首座，这丹算私赠吗？」苏念慈答：「算回赠。回赠有渣为证，私赠无渣为耻。」

赵玄机夜至执法堂，翻门规至「回赠」条，条下空白，空白像故意留给他踩。他提笔，欲批「药渣换丹当查」，笔落，却想起苏念慈红着眼出丹堂那一幕，幕里无私，只有炉温。他把笔搁下，搁下像承认自己踩不实。

顾小满在通铺角，把除魔符与半块灵石并藏，藏得像并藏两条还不清的恩。她不懂塔，却懂「甜」字要写进册里——册厚一寸，厚得像把路也写直了。

小院门外，有弟子故意提高声调：「九月老头靠渣换丹，羞辱丹堂！」莫长春不辩，只捧茶，抿一口，笑：「他们说得对。老夫靠的就是渣——渣里有念慈熬夜的气，气炼成甜，甜再送回去，才叫还礼。」弟子一愣，骂也不是，退也不是。

林挽月路过丹堂，低声问弟子：「苏首座……真哭了？」弟子摇头：「没哭，是笑。笑得像哭。」她怔完仍往执法堂走，走前目光扫过窗影——影暖，暖得像一条还没走的路。

丹堂后院，苏念慈把回春丹分三等：一等给伤重弟子，二等给杂役例份，三等留丹堂备急。分完，她望小院，一线夜灯已灭，丹堂灯仍亮——亮得像替师兄守一句「别误炼丹」背后的真意。

外门仍传「药渣换丹是羞辱」，传的人越多，丹香越浓——浓得像给围观者看：不是羞辱，是还。

魏无涯在檐角观丹堂烟，低笑：「渣轻丹重，重的是心。」弟子渐散，散里仍回头瞧丹堂窗影——影暖，暖得像挑不出刺。

丹堂门外，有弟子比谁的流言更脏，比到第三句，被丹香压回去，压得像被炉温烫了舌头。脏话少了，敬惧多了——敬惧参半，才是回赠落地的声。

塔壁金铭浮起，淡金字稳：药渣亦礼，回赠有时。莫长春念出声，像念给风听——九月还剩八月，八月够把一盏丹香，熬成一盏胆。

药渣入炉，炉温起，起得像把「苦里藏甜」四字先炼给全门看。

塔壁浮半句：**寿元八月**。莫长春瞧见，笑：「八月够甜，够渣，够把流言炼淡一寸。」
