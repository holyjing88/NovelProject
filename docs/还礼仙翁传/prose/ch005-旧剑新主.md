# 第五章 旧剑新主

柳青鸢在偏厅收剑时，指节发白，不是因为剑沉，是因为话沉。

莫长春说「剑跟你有用」时，语气像把一件旧衣递给她：随意，却合身。她当了宗主三年，听惯了恭维与试探，唯独师兄这种「把重物推给你」的理所当然，让她一时不知如何回礼。

「师兄。」她终是开口，「此剑若在我手，外人会说宗主贪长老旧物。」

「让他们说。」莫长春拄杖，「老夫将死之人，贪不贪，都一样难听。你年轻，该贪的是剑，不是名。」

柳青鸢抬眼：「我不贪剑。」

「那你贪什么？」

她唇动，像有字要出，终是咽下，换一句硬的：「贪青岚门不断档。」

莫长春点头：「那更该拿剑。门断不断，在剑，不在杖。」

他取旧剑，剑出鞘半寸，霜光一掠，像二十年前那个还会在月下练剑的青年长老。柳青鸢忽然明白，师兄赠的不是铁，是「你还可以往前」。

她双手接剑，簿意再动——她仍无簿，却觉山门风一紧，像天道在师兄袖中记了一笔。

「我会还。」她低声。

「不必还剑。」莫长春道，「还别的。」

「什么？」

「到时候你就懂了。」他笑，「可能是剑穗，可能是名分，可能是一句替天下说的话。」

柳青鸢心口一撞，像剑穗无风自动。

她忽然问：「师兄，旧剑给了青岚，你拿什么防身？」

莫长春指杖：「杖在。剑在青岚，青岚在，杖便在。」

柳青鸢沉默，像听懂一句没说全的话——师兄赠的不是铁，是把「你还可以往前」四个字，推回她手里。

偏厅外，丹堂石阶上，苏念慈捧药篓止步。

她听见「剑」字，听见柳青鸢低声「还是你懂我」，听见莫长春笑。药篓里草药被攥断一根，汁渗掌心，苦得她皱眉。她本要送新采的柴胡，却转身往丹堂走，对弟子冷声：「今日不开炉，别围观宗主厅。」

弟子问：「首座，谁围观？」

苏念慈道：「围观自己的人。」

消息半日传遍外门：莫长老将佩剑赠宗主。

说法各异。有人说长老疯，有人说宗主夺产，有人说这是青岚门内部一场体面交接。赵玄机在执法堂冷笑：「九月寿元，还送剑？送完了送命？」

林挽月小声问：「赵师兄，莫长老……是不是对宗主……」

「别乱说。」赵玄机截话，眼神却亮，像抓到可踩的影，「不过老不正经，倒是真的。」

他抄门规，抄到「私赠宗门器物」一条，停住，像想踩，又踩不实——剑是师兄私物，赠给宗主，名分上挑不出刺。

林挽月小声道：「赵师兄，若宗主真拿了剑……」

赵玄机截话：「拿了便拿了。剑在谁手，心还在谁那边——莫长春那句，比门规难抄。」

午后，莫长春回小院，不进门，在廊柱旁坐下——不坐主位，只让半边灰袍被秋光洗过。

弟子在院外嚼舌，嚼得故意大声：「宗主又进莫长老院了，赠剑赠符，赠到杂役头上，唯独不赠丹堂。」

莫长春捧茶，抿一口，笑：「他们说得对。老夫靠的就是这些人——靠他们传话快，靠他们眼睛尖。」

弟子一愣，骂也不是，退也不是。

袖中簿烫一息。烫完，并无新物落地，只有一行淡字浮在膝上薄册，转瞬即逝：

**赠出旧缘，回赠在路上。**

他合簿，笑：「路上就路上。老夫等得起。」

夜更深，丹堂方向仍无火光。苏念慈关炉一日，外门不知，只道首座闭关。莫长春望那一片黑，轻声：「念慈，妒火别烧坏了丹。」

风过，无人应答。

外门，弟子把「旧剑赠宗主」讲成七种版本，版本里总有「老不正经」四字。

苏念慈在丹堂听见，炉火一跳，跳得像妒里掺敬。她低声对弟子：「剑不剑，别传。传多了，传成情。」

弟子噤声。苏念慈望小院方向，袖中回春丹瓶一紧，像握着一句没说出口的「我也能还」。

赵玄机在石道旁对弟子道：「旧剑赠宗主，名分上挑不出刺，可心里挑得出妒。」弟子不敢接话。林挽月远处瞧见柳青鸢捧匣出偏厅，匣轻，剑意却重，重得像把青岚门的锋，又送回该锋在的人手里。

偏厅廊下，还有弟子踮脚偷瞧，窃语成浪：「宗主拿旧剑，莫长老拿什么？」「杖啊，九月老头，杖里还有竹灵呢。」「竹灵能斩妖吗？」「青狼那日，杖扫过。」浪声轻，轻不过柳青鸢捧匣出偏厅那一瞬——匣轻，剑意却重。

莫长春独坐廊柱旁，簿上浮半句，像章末钩：

**赠出旧缘，回赠在路上。**

他念：「柳氏拿剑，霜华剑还在后头。下一份，该轮到半块灵石了。」

杖中竹纹游动，气音仍是一字：等。

风过，无人应答，像话进了土里，等着长。
