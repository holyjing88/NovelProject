# 第 22 章 公审

公审续，证词如链，一环扣一环。

链不是绳，是眼——眼见过，舌才不能乱；舌一乱，链就勒紧，勒在「听说」二字上，勒成虚。

伤者先指，指得稳：「他拉我退，不是推。」

猎户接，接得像补环：「醉刀，自伤。我瞧见刀落，落在他自家臂上。」

沈振东指路线，指得像画在土上：「我夺刀，陈寻断后，队先下。山下路口，我指，他未推人。」

老耿指瓮，瓮在堂边摆着，像一件静物：「修瓮，旧恩，与谣无关。封门练体，我瞧见，未夜出。」

张麻婆指盐，盐字硬，硬得像她人：「我送，他记，与沈家无夜入。门边，未帘内。」

方小沅指粥，声小，小里稳：「我送，门边，未进。粥冷，心不乱。」

六环齐，齐得像扣锁，锁的是谣，不是陈寻。

赵绣春只剩嘴，嘴快，快里空：「册邪！血字谁信！沈家女夜入，谁瞧见？」

「我信我自己记。」陈寻说，声平，平里不辩辱，「阳面只记恩，不记辱。辱，等时辰。」

钱管事合册，合得响，响里像关门：「测灵名，陈寻在。除名，需三分二，今日，票不够。」

赵村长脸青，青里笑不出，笑僵在「礼」上——礼已泼，审已链，链齐，票不够，不够便是理在。

他拍案，案响，响里带急：「再议！周嫂，你还有话？」

周嫂哭，哭已乱，乱像被链勒住喉，喉里漏风，漏「听说」，漏「赵婶说」，漏「沈家」，每漏一句，赵绣春脸白一分，白里仍嘴硬：「以证为据！以证为据！」

「以证为据。」陈寻重复，重复像钉，钉回正题。

堂前静，静里闷，闷像雷雨前。雷雨前不是雨，是气压，气压落在赵家，落在周嫂，落在还没拖上堂的那张嘴。

陈寻不擦脸上的水，水干，留下寒，寒里，他反而更清醒——公审不是赢嘴，是赢证；证齐，谣自断。断不在他拳，在链；链齐，名在，名在，石才有机会。

他摸册，册在怀，阳面六笔温，温像六盏小灯；阴面痕淡，淡而毒，毒在「当众」——辱已入账，账不收辩，辩是给口水加盐。

萧惊鸿立人群，拳硬，硬而不出。不出，是陈寻那一眼；一眼，够堂上，也够三月前。他看链齐，眼里火退半寸，退成闷，闷在骨，骨里响，响而不爆。

赵村长看周嫂，周嫂看地，地看得像坑。坑里，或还有一个人——醉的，胡话的，被赵绣春挑过的。

陈寻抬眼，眼干：「审。未到尽。」

未到尽，便是钩——钩不在胜，在反噬；反噬不在他，在谣口；谣口若反，赵家才疼。

风紧一线，棚顶灰天低。低处，他低声，只给自己：「链齐。等反。」

钱管事翻册，翻得像念条：「张麻婆，盐，门边。方小沅，粥，门边。老耿，瓮，旧恩。沈氏，汤衣，未夜入。」

每念一条，赵绣春脸白一分。白，是谣在退；退不退，看最后一扣——扣在醉者嘴上。

陈寻不辩「沈家女夜入」，辩在此时，是给谣加盐。盐在证，不在舌；舌乱，证不乱，乱的是周嫂，是赵绣春，是还没上场的那张嘴。

方母拽女，拽得更紧，紧里像护，护的不是陈寻，是女儿的名。名若污，粥也污；粥不污，门边二字就够。

张麻婆骂，骂成风，风过堂，堂上竟静半息。半息里，陈寻觉出阳面六笔齐温，温像六盏灯照湿脸——照不乾水，照得清：恩在阳，辱在阴；阴已收过峰，峰后，等反。

赵村长看票册，册上名多，多不等于三分二。不等于，他急；急里拍案，案响空，空得像敲铁。

「再议！」他嗓硬，硬里虚，「周嫂，你还有话？」

周嫂哭，哭成碎，碎里漏「赵婶让我说」，漏半句，像链扣回赵绣春。赵绣春嘴硬，硬里退半步：「胡说！」

胡说二字，像自证虚。虚到顶，或反——反不在陈寻拳，在谣口；谣口若反咬，赵家才疼。

萧惊鸿立人群，拳松了又紧，紧而不出。不出，是信链；链齐，名在；名在，三月前那夜祭父母时立的约，才不算白立。

陈寻抬眼，看棚外天，天低，低得像压。压不垮桩，压垮的是心虚者——心虚者，下一息该拖上堂了。

堂前有人低声，低声像沸前闷：「周家男人呢？」

赵村长眼亮，亮得像抓住缝：「拖！」

拖字未落，陈寻不拦——拦在此时，像怕；不怕，等反。反是链的最后一扣，扣在醉嘴上，不在他拳上。

他任水寒在颌，寒像清醒。清醒里，他重复钱管事那句：「以证为据。」

四字落，落得像钉，钉住公审的骨——骨在证，不在辱；辱已过峰，峰后，该平了。
