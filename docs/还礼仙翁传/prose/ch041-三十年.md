# 第四十一章 三十年

限期第二夜，丹堂后院，炉声细，像有人在远处哼歌。

苏念慈绕炉走了一圈，终是止步莫长春面前，月白衫子被火光映得透明，像把心事也映透明。后院无弟子，只有风与药香，像特意把热闹留在前堂，把真话留在后炉。

「师兄。」她声稳，眼不稳，「我等了三十年。」

莫长春看她，须眉白，笑温：「念慈啊，老夫剩四月了。你想清楚。」

她竟笑，笑里倔，倔里酸：「那正好，不用等太久。」

莫长春一愣，随即也笑，像被一句反话撞中软处：「你这丫头，倒会算恩。」

「丹堂算恩，」她道，「我比你熟。四年也好，四十年也好，回赠不是还债——是你教的。」

风过，药香扑怀，像昨夜重现，却不再有扑的动作，只并排立，像两盏灯。远处前堂仍有弟子窃语「体统」，窃语被风切断，断在后院这一瞬。

莫长春低声：「想清楚了就好。清楚的人，才收得下苦，炼得出甜。」

苏念慈点头，望炉：「炉还开着。」

「开着。」他应，「别误了明日验丹。」

她转身回炉，背影挺直，像把三十年折进一炉火里，烧给青岚门看。

廊角，柳青鸢立过一停，未入后院，像刻意把空间留给别人，又像怕自己一入，就要说体统。她指节叩栏，一声，转身，宗主令在腰，威在回。

苏念慈瞧见那影，眸光微动，不妒了——妒轻不过命，命在东脉，在子时，在夜袭。

霍镇山在远处骂阵，骂声洪，像把战意敲进每个人心里。他粗声对亲兵道：「首座守炉，长老守东脉，你们守田——田在，人才在。」

赵玄机思过崖方向，风硬，像妒火还在冷，冷到要上场守门。

弟子在前堂窃语「三十年」，窃语轻，轻不过后院这一瞬真话。有人猜苏首座表白，有人猜莫长老拒，猜成七种版本，版本里都没有后院炉边并排立的两盏灯。

前堂有人猜：「苏首座是不是表白了？」「莫长老拒了？」「拒什么，你看首座回炉那背影，像战，不像拒。」猜成七种版本，版本里都没有后院并排立的两盏灯。

魏无涯在檐角，扇开一线，观气运，低笑：「三十年，四月，好记。」

顾小满在杂役堂值夜，听后院炉声细，无端心稳，像半块灵石在怀，温了一路。她不知三十年，只知首座守炉，守得像战。

赵玄机在思过崖，听丹堂炉声远，妒火又冷一寸。他忽然想：若守门能还恩，妒人还妒什么？想完，他自己也愣，像没料到思过崖风能吹出这一句。

三十年落在炉里，四月落在簿上——等字收束，战字开门。苏念慈不回看，莫长春不追问，像多年同门，终于不必问「你还要等什么」。

后院两盏灯并排，前堂七种版本——版本里都没有真话，真话在炉边，在四月里，在等字收束、战字开门那一瞬。

莫长春望东脉，低声：「验丹明日。今夜之后，丹堂只有战，没有等。」

风大，炉声细，像一句未说完的话，留给子时。

林挽月罚役归途，望见丹堂后院灯影，未近，只把一枚新制符纸塞门缝——符纸薄，意厚，厚得像「别死在天试前」之外，还有一句未写的「别先算」。

魏无涯传讯厉无殇，只四字：「简已至。」厉无殇阅讯，焚，笑冷：「准不准，莫长春都要接。接，才是戏。」

孙福在角隅听「三十年」，面色白，像听见自己的旧恩仇将被翻。顾小满冷眼：「你看炉，别看嘴。」

柳青鸢隔窗望后院灯影，不进门，像怕进门就惊破这一瞬。她低声：「师兄，旧伤别撕。」

苏念慈在后院拨火，火舌舔炉壁，舔得像把三十年也炼进去。她忽然懂：师兄不追问，不是不懂，是把等字收束，留给战字开门。

苏念慈在后院封丹样，手稳，稳得像把羞炼进丹。她望东脉方向，低声：「验丹明日。今夜之后，丹堂只有战，没有等。」

莫长春袖空，一念出塔，杖在塔中，不追旧，只记旧。

柳青鸢声低：「师兄，旧恩仇别撕。」莫长春道：「不撕，才还得清。清了，才送下一礼。」

霍镇山声洪：「三十年不让！」周德海赔笑，笑洒，洒得像怕话也洒。

苏念慈闭炉，闭得像怕旧烟进丹堂。她低声：「旧别传成咒。」

顾小满册上又记一行，行细，细得像半块石硌深。她低声：「三十年记下了。不往外传。」

莫长春不撕，只记，记像记礼——礼还，恩才平。

验丹钟将响，响之前，后院真话在前堂七种版本里——版本里都没有灯，灯在炉边，在四月里，在等字收束、战字开门那一瞬。

周德海在长老院听闻「三十年」，令旗在指间转，转得像犹豫，犹豫得像信使已在后巷候过三次。

魏无涯在檐角，扇开一线，观气运，低笑：「三十年，四月，好记——恩在炉里，也在战前。」

赵玄机在思过崖，听丹堂炉声远，妒火又冷一寸——若守门能还恩，妒人还妒什么？

弟子在前堂猜七种版本，版本里都没有后院并排的两盏灯——灯在，话在，战在近，等在收。

莫长春望东脉，风腥，云低，像秦氏简上的「夜袭」二字，已贴到东脉田埂。风大，炉声细，像一句未说完的话，留给子时。

验丹明日。今夜之后，丹堂只有战，没有等——等字收束，战字开门，开门之前，验丹钟先响。

苏念慈在后院又停半息，像还有话，终是收住，收得像首座。她低声：「师兄，验丹文书半个时辰后到。到了，全宗又要围一层——围金丹，不围嘴。」

莫长春笑：「围嘴的人，围不过验丹钟。钟响，嘴就哑。」

霍镇山在远处骂阵，骂声洪，像把战意敲进每个人心里。他粗声对亲兵道：「验丹钟响之前，别噪。噪了，首座手抖，丹堂先苦。」

周德海在长老院听闻「三十年」，令旗在指间转，转得像犹豫，犹豫得像信使已在后巷候过三次。孙福在角隅听「三十年」，面色白，像听见自己的旧恩仇将被翻。

魏无涯在檐角，扇开一线，观气运，低笑：「三十年，四月，好记——恩在炉里，也在战前。验丹钟响，响的是门，不是私。」

顾小满册上又记一行，行细，细得像半块石硌深。她低声：「三十年记下了。不往外传——传了，就轻了。」

赵玄机在思过崖，听丹堂炉声远，妒火又冷一寸——若守门能还恩，妒人还妒什么？他摸铁牌，牌凉，凉得像提醒：思过半月，只是开始。

弟子在前堂猜七种版本，版本里都没有后院并排的两盏灯——灯在，话在，战在近，等在收。有人低声：「首座守炉，守得像战——战在近，等在收。」

苏念慈在后院封丹样，手稳，稳得像把羞炼进丹。她望东脉方向，低声：「师兄，验丹文书半个时辰后到。到了，全宗又要围一层——围金丹，不围嘴。」

莫长春笑：「围嘴的人，围不过验丹钟。钟响，嘴就哑。验丹明日，丹堂只有战，没有等。」

莫长春袖空，一念收塔，杖落掌，声轻：「话完，阁要开。百宝阁二次，轻里藏价。」

三十年话才开个头，头在莫长春须眉上，眉上像又落了一层旧雪。

弟子渐散，散里仍回头瞧一眼灰袍袖空——瞧得像瞧一眼九月还能走多久。
