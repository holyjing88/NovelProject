# 第九十八章 坊市续

第二日。

「丑丹师」牌仍歪，歪不要紧；歪，像他人。人丑，牌丑，手稳，坊市才肯续。

续在客。客里，有个穿灰袍的老客，老客不语，只递渣五包，包上系红绳——绳旧，像试人。

韩泥一包包嗅，嗅到第三包，鼻下停半息：「这包有霉，霉在角，别投炉心。」

老客眼一细：「你怎么知角霉？」

「嗅。」韩泥说，「嗅完，不取钱。」

老客收包，收时塞一块下品灵石碎片——碎片小，小不要紧。小，不能收；收，像受贿。

韩泥推回：「半文就够。够，是试，不是营生。」

老客看他半息，走了，走像记下一笔。记不要紧，记的是手，不是名。

铁无言午后过，声低：「那老客是药峰管事的人。你推石，推对了。」

韩泥应声：「推。」

推，不为清名，为续摊。续摊，才能听兽潮风声更近。

白织月第二回来，这回带同门，同门笑：「丑丹师，嗅嗅我这块是不是假灵石？」

韩泥不接：「不嗅石。只嗅渣。」

同门无趣，无趣不要紧；无趣，谣就少半分。

管事弟子第三日来，簿上无投诉，无投诉，便记：「试三日，暂留。」

暂留，够了。够了，韩泥撤「师」字，改「嗅」字——字小，小不要紧；小，才藏得住。

赖福来坊，坊边立，立像盯：「暂留？留你晦气？」

韩泥收牌：「留嗅。嗅完，仍倒渣。」

赖福没掀摊，没掀不要紧；没掀，是风变了——风里兽潮更近，近到柱上贴条。

条丑，字真：「南荒兽聚，药山戒严，杂役夜禁。」

夜禁，不要紧；禁，他本就在丑时炼。炼在窑，不在坊。

回棚，刘婆端粥：「坊市还去？」

「去。」韩泥说，「听风。风里有兽。」

刘婆脸白半分：「兽来了，别逞能。」

「不逞。」韩泥说，「手稳，栏清。」

邻床慌：「夜禁了，清栏怎么办？」

「白天清。」韩泥说，「夜里守坛。」

守坛，也守坊市续来的那一线认——认丑手，认末席，认嗅而不飘。

丑时，廊下静。粟壳香淡，香里混兽栏腥与坊市尘，混不要紧；混，才是他的日。

他低声道：「坊市续了。下一程，兽潮。」

坛腹温，像应「续」字。续，不是名；名在远，远在130。

更鼓尽，柱上条在风里猎猎。猎猎，像催备——备手，备渣，备清瘴路。

第二日，老客递渣五包，包上系红绳。

韩泥一包包嗅，嗅到第三包：「角霉，别投炉心。」

老客推灵石碎片，韩泥推回：「半文就够。」

铁无言午后过：「那老客是药峰管事的人。」

第三日，簿上无投诉，管事记：「试三日，暂留。」

韩泥撤「师」字，改「嗅」字——字小，小才藏得住。

柱上贴条：「南荒兽聚，药山戒严，杂役夜禁。」

刘婆端粥：「风里有兽。」

老客那笔，记的是手，不是名。名在远，在130「丑丹师」。远不要紧，110先起点——起点在散，散在渣，渣在栏，栏在手稳里。

夜禁后，他丑时仍炼，炼在窑，不在坊。坊市续来的认，认丑手，认末席，认嗅而不飘。飘了，三年约白接；白接不要紧，玉牌在怀，怀沉，沉的是约，不是名。

第二日，「丑丹师」改「丑丹嗅」，字小，小才藏得住。

「「」「嗅。」管事弟子第三日来，簿上无投诉，便记：「试三日，暂留。」韩泥撤「师」字，改「嗅」字——字小，小不要紧；小，才藏得住。白织月袖风过，不近摊，只远瞥一眼。瞥，像验；验不要紧，手稳才验得过。赖福堵路：「三日没投诉？运气。」韩泥只答：「手稳。」手稳，不在运，在鼻。鼻下嗅诀，沉腐浮三字一字不落。

落齐，坊市才续得住。更鼓沉，韩泥按木牌，牌凉贴胸。柱上条在风里猎猎。猎猎，像催备。下一程，兽潮。」兽潮在远，备在近。备，首在耳，次在鼻。坊市续，续在第三日。第三日，老客红绳五包，包是试，试不要紧，试过才续。试过，推石推对，石在脚，脚稳，摊才稳。韩泥改「师」字为「嗅」字，字小，小不要紧，小才藏得住。

藏得住，98才续，99才闻兽潮条至。条至在柱，柱在界，界在药山——药山，是丙九命。

白织月远瞥，不近，像验。验不要紧，簿上无投诉，管事记「试三日，暂留」。暂留，够了。够了，不在荣，在听——听兽潮，听障，听栏先倒。

赖福堵路：「暂留也晦气。」手稳，续摊；续摊，才听风更近。风近，备在近。备，首在耳，次在鼻，再次在手。

同袍问：「你真嗅出角霉？」「嗅。」不取钱，像傻，像末席该傻。傻不要紧，傻才试人——试老客，试管事，试铁无言眼。

铁无言午后过，声低：「老客是药峰管事的人。」推对，不为清名，为活。活长了，才还叶汤，才接兽潮。兽潮在远，柱上条已贴——贴，是99。99在前，98收摊，收摊不要紧，收的是师字，不是嗅路。嗅路在鼻，鼻在，路就不绝。

坊市第三日，老客红绳五包，试人。试不要紧，试过才续——续，改「师」为「嗅」，字小，小才藏得住。藏得住，99才备兽潮。兽潮在柱，柱在界，界在命。

管事记「试三日，暂留」。暂留，够了——够了不在荣，在听。听兽潮，听障，听栏先倒。倒不要紧，倒前先备；备在99，备在100前夜，备在101清瘴料。」手稳，嗅路在；嗅路在，鼻在。鼻在，沉腐浮三字才落齐。落齐，更鼓沉，按木牌，牌凉贴胸。铁无言午后过：「老客是药峰管事的人。推石推对。兽潮风声已在条上——条猎猎，像催备。催备，99在前。

老客红绳五包，试人，试不要紧，试过才续——续，暂留，够了。够了不在荣，在听兽潮。兽潮条已贴柱，柱在界，界在命；命在丙九，丙九守栏，守在100前夜，守在102撒线，守在103压，守在104至。至在远，98只续摊；续摊，才听风更近。风近，备在近；备，首在耳，次在鼻。鼻在，嗅路在；嗅路在，丑手才认。

认不认，不在98，在130远。远不要紧，98收师字，留嗅字——字小，小像末席该写的。写小，赖扒皮少盯；少盯，耳才听全。听全，更鼓沉，按木牌，坊市续了。续了，兽潮风声已到耳。已到耳，99在前。手在，路就不绝。

收摊时，柱上条新贴，条丑，字真——真在「杂役备障」。备障，99在前。98只续嗅字，嗅字小，小藏得住。藏得住，耳才听全。听全，更鼓沉，按木牌，坊市续了，兽潮条到了。到了，手在，路就不绝。

坊市续在第三日，老客红绳五包，试人。试过，推石推对，推对不为清名，为活。兽潮条已贴柱，柱在界，界在药山——药山，是丙九命。命在，手在，就不滚。

管事记「试三日，暂留」，暂留，够了。够了不在荣，在听——听兽潮，听障，听栏先倒。倒不要紧，倒前先备；备在99，备在100，备在101清瘴料。料在经页残诀，诀浅，浅而真；真，102才撒，103才压，104才至。至在远，98只续摊。续摊，改师为嗅，字小，小像末席该写的。听全，更鼓沉，按木牌，坊市续了，兽潮风声已到耳。

收摊时，同袍问：「你真嗅出角霉？」「嗅。眼在，98才续；续了，99条至。条至，手在，路就不绝。柱上兽潮条贴实了，贴不要紧，贴才催备。催备，99在前；在前，韩泥收摊，收「师」留「嗅」，字小，小藏得住。藏得住，耳才听全——听全栏先倒，听全甜毒，听全不许逃。不许逃，不是勇，是活；活，才还恩。

恩在七笔，笔分列，不混。混，路断。路不断，100前夜等在东角。角在，栏在，坛在怀。更鼓沉，按木牌，坊市续了，风声到了。到了，手在，就不绝。手在，摊在，路在，就不滚。」

管事记簿，记细，是小胜。胜在忍，不在辩。

他望坛，低声道：「手在，路在。」坛不应字，只温。
