# 第九十二章 废窑再试

雨歇。

药山废窑，窑口红湿，湿像昨夜火没尽。没尽不要紧，韩泥二回上山，筐里仍是药渣，渣上盖灰，灰像常倒。

这一回，他不急火。急火焦渣，焦渣苦，苦不是路。路在匀，匀在掌，掌在茧。

铁无言昨夜说过一句：「再试，先温炉，后投芯。」

韩泥记在心里。记，不声张。声张像求丹；不求，只试。

入窑，先以掌温炉壁。壁凉，凉退半寸，退得像坛腹应温。温够了，投青芯一撮，褐渣半把，灵泉三滴——滴少，少才试得出分寸。

火起，火不猛，不弱，像呼吸。呼吸匀，渣里白气起一线，起而复散。散前，他鼻下捕捉到培元路第二丝——丝细，细而正。

正，他不敢喜。喜了，手飘；飘了，赖扒皮来。来不要紧，窑深，声不出。

第二炉，他换投序：先褐后青。序错，香杂；序对，香单。单香在鼻下，像铁无言验渣时说的「正」。

管事弟子过路，问：「韩泥，你倒渣倒两趟？」

「渣多。」韩泥说，「筐沉，分两次。」

管事记一笔，笔丑，真在「分两次」。真，才像常活。

第三炉，火匀一寸。一寸够了，再多，窑壁裂。裂不要紧，裂是旧窑，旧窑才没人争。争穴，要末席；末席，要手稳。

炉开，无丹，只有一团凝灰。凝灰里，一点白，白像未成之珠。未成不要紧，再试；试炉本就不成丹。

韩泥收灰，灰入袖袋，袋扎紧。扎紧，沿净。沿净，坛藏得住。

下山遇白织月，袖风带香，声冷：「丑杂役，药山是你扫的？」

「倒渣。」韩泥不抬眼，「手稳，不脏内门路。」

白织月嗤：「末席也配碰药山？」

韩泥只答：「配倒渣。」

嗤声远，远不要紧。远，才试得稳。

回棚，赖福秤筐：「仍少二斤。」

「看错了。」韩泥说，「渣在筐底，底沉。」

赖福掀底，底有灰，无青。青在袖，袖在，路在。

夜，他按席角，席严。坛腹温一线，贴掌，像应「再试」二字。再试，火路熟半分；熟半分，不够成丹，够准备。

刘婆端粥过，粥稀：「又上山？」

「倒渣。」韩泥说，「手稳，饭稳。」

「饭稳，恩不断。」刘婆哼，「别偷火，偷了连坐丙九。」

韩泥点头。偷火不在他路上；路上只有试炉，试到匀，试到能匿炼。

邻床悄问：「你真只倒渣？」

「倒渣里试火。」韩泥声平，「火试过了，才藏得住。」

少年闭嘴。闭嘴，像丙九也学会忍。

丑时，廊下鼻下静半刻。粟壳香淡，香落齐，掌纹那路又温一线。温不应喜，应「稳」——稳，是再试的回响。

赖福脚步远，远里一句：「丑杂役，我盯着你。」

韩泥只答：「在。」

在，就不滚。不滚，废窑第二回就不白来。下一回，要备匿丹之料——备在暗处，备在席下，备在还恩前头。

他低声对坛：「再试过了。火匀半寸，路熟半寸。」

坛不应字，只温。温，像爹搭沿，像泥岗丑时那口白气。

更鼓尽，雨歇，药山湿，湿里藏一点窑温余韵。余韵散，散进掌，散进记——记：再试，火可匿。

卯时，他先清栏。栏腥，腥不要紧，手稳才要紧。铲下稳，稳里拾褐渣半把，把入袖，袖扎紧。扎紧，沿才净。

药渣场角，铁无言再嘱：「再试，先温炉，后投芯。芯多一分，焦；少一分，不发。」

编筐季未了，他编半刻，藤刺扎掌，刺浅，浅些，沿才净。邻床瞧：「又编？」

「编。」韩泥说，「筐卖钱，钱买米。」

午后二回上山，筐仍满，满像常活。

下山时，他故意绕经药渣场，场角风硬，硬里带苦辛。铁无言不再嘱，只点头，点得像验过第二回。验过，才信火路熟半分。熟半分，不够匿炼；够再备，备在93，备在席下，备在经页黄。

赖扒皮傍晚查筐，筐底灰厚，厚像常倒。常倒，才藏得住袖里青芯。青芯干，干而香，香在鼻下，不在舌。舌要留来守坛，守坛里坛腹温一线，温不应喜，应「稳」。
