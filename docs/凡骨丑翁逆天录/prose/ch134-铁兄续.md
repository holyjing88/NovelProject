# 第一百三十四章 铁兄续

兽潮散后月余，东角栏修好了。

修好不要紧，修像108那日并肩的回响——回响在栏，也在人。

铁无言立栏边，臂上有旧伤，伤浅，浅像清瘴并肩留的。留不要紧，留是证；证在，兄弟线就不断。

韩泥递灰囊，囊里清瘴余灰，灰细：「栏角还缺半袋。」

「缺。」铁无言说，「缺，你补。补了，我站台侧。」

韩泥不辩。辩，像显摆108；显摆不要紧，灰在，栏在，就在。

撒灰半刻，风硬，灰线像刀——刀划皮，不划手；手稳，栏才稳。

赖福远处骂：「丑杂役又跟铁哑巴混！」

韩泥不应。不应，像石；石在，路在。

铁无言声低：「赖福余党，还会使坏。使坏不要紧，栏稳，坛稳，就不滚。」

「不滚。」韩泥说，「滚了，刘婆新衣白还。」

两人立栏边，立像108那日，但不并肩献粒——粒在远，在142，在175。

并肩续，续在守，不在嘴。嘴要留来数息，留来守坛。

沈枯芽递口信，信仍无字，只有一片枯叶：「叶姐说，别来丑。」

韩泥不接话，接在心里。愧线烫半分，分列——愧是愧，恩是恩。

刘婆端粥，仍多半勺：「又跟铁无言？」

「修栏。」韩泥说，「栏稳，兽潮才不白守。」

刘婆哼：「守栏别守邪。」

秦霜远木鱼敲一声：「兄弟线，心别邪。」

韩泥不答，答在心里。心里记：108并肩，134续守——守到580，才配从绶；绶在远，线在今。

日暮，铁无言收工，声低：「下一回，我还来。」

「来。」韩泥说，「来，栏在，就在。」

丑时，练鼻下静。粟壳香淡，香一字不落。

他望栏影，影长，长像108那日余瘴未散——散不要紧，栏正，路正。

邻床悄问：「铁无言跟你很熟？」

「熟。」韩泥声平，「熟在108，不在嘴。」

更鼓沉，他按坛，低声道：「兄弟线续了。记着。」

坛腹温，贴掌，像应「栏」——栏稳，恩稳，就不滚。

东角栏边，灰线名如刀——刀划皮，不划手；手稳，栏才稳。

管事弟子验栏，栏正，正像108清瘴后那半分余温。余温在，兄弟线就不断。

废渣巷风硬，他倒渣半筐，渣灰，灰里正。正不要紧，正像守栏回响。

赖福廊外骂，骂声远，远不过栏木那一线稳。稳在，就不滚。

刘婆柴门外咳：「修栏别沾晦气。」韩泥只答：「不沾。栏稳，恩稳。」

秦霜远木鱼敲一声：「兄弟线，分列才稳。更鼓沉，他按玉牌，牌凉：「108并肩，134续守。记着。」邻床悄问：「你跟铁无言，像兄弟？」「像。」韩泥说，「像，不在嘴，在站。」」

栏木新，新里带旧伤——伤是108，新是134。新旧并，像兄弟线。

撒灰半刻，灰细，细像清瘴余韵。韵在栏，不在舌。

赖福远处盯，盯不要紧，栏正，盯也白盯。

韩泥望铁无言，一眼像约：「580前，只站。」

铁无言点一下头，点得像134那日修栏——栏在，就在。

丑时末，他再验栏角，角稳，稳像108并肩那半步。

108兽潮并肩，134东角修栏——栏在，就在；在，就不滚。栏木新，新里带旧伤，伤是清瘴，新是续。续不在嘴，在撒灰，在守，在站。站侧，像580从绶前先站；站，不叫，才像铁无言。铁无言臂上旧伤浅，浅像108那日挡的半步——半步韩泥记在心里，心里排序：108、134、580；580纳从绶，纳前，只站。

站不要紧，栏稳，兽潮才不白守。守了，134才像第二部兄弟线，不像一部尾韵。赖福骂「跟铁哑巴混」，骂在舌，不在栏；栏正，骂就远。远不要紧，远，才守得住坛。坛在席下，席严，坛不露；不露，134修栏才像常活里藏线。线藏，才配135白织月袖风嘲——嘲在，手更要稳。

稳了，136层十五才实；实了，140积势才足。足在远，134只修栏。修完，铁无言声低：「下一回，我还来。」来，栏在，就在。丑时，练鼻下静，粟壳香淡，香一字不落。落齐，134这一日记进心里——记108，记栏，记兄弟线续，不叫从绶，只站。

栏稳，兄弟线续，续在守，不在嘴。

栏角再验，角稳，稳像108那半步——半步续到134，580前，只站，不叫。

更鼓沉，他望栏影：「108并肩，134续守。记着。」

落齐，134这一日记进心里——记栏，记108，记兄弟线续。续到580前，只站，不叫。

编筐半刻，藤刺扎掌，刺浅，浅些沿才净。沿净，坛才藏得住。

管事记簿，记细，是小胜。胜在忍，不在辩。

他望坛，低声道：「手在，路在。」坛不应字，只温。

更鼓沉，天边白一线。白里，他按木牌，牌凉贴胸。

坊市尾巷风硬，他收摊如石，石在，就不滚。

验息后，他理怀内册页，页黄，字丑，丑而真。

铁无言声低：「别飘。」韩泥应声：「不飘。」

刘婆端粥，仍多半勺。韩泥点头，谢轻，恩重。

药峰分拣照旧，指稳，鼻下静一息。

帖在怀，怀沉，沉的是序，不是惧。

白织月袖风过，韩泥不辩。辩，费舌；舌要留来守坛。

赖福廊下过，韩泥只答：「在。」在，就不滚。

柱上条角白一线，三年约在怀。怀凉，掌温。

坡下石阶湿，他数息，数帖，数债序。

同棚杂役笑，笑在耳，不在心。

废渣巷日斜，他验渣，渣正，手才静。

侧门驿亭风硬，他听风，风过，记在心里。

散队后，他过外堂阶下，分拣半刻，指稳——指稳，路就不飘。
