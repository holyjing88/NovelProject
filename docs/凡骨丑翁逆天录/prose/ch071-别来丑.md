# 第七十一章 别来丑

三年约后第七日，驿卒过侧门。驿卒肩湿，湿里夹泥岗方向的风。风旧，旧得像叶家灶烟。他喊：「丙九韩泥！泥岗来字！」

韩泥正清栏，铲下停半息。停，不为字，为恩——恩在远，字在近；近字烫，远恩更烫。管事弟子验封，封粗，蜡印浅。浅不要紧，印真——真在「叶」字一角，真在泥岗里正那枚印。「念。」韩泥声平。驿卒念得急：「韩叔，别来丑。青禾。」

四字落耳，像碗沿热退了一寸。退不要紧，字在；字在，恩不断，愧也不断。同批少年窃笑：「叶家丫头嫌你丑？」

韩泥不辩。辩，费舌；舌要留来记愧，留来守坛。他把字条收怀，贴身，不示人——示人，谣又来。赖福晃秤，眼细：「泥岗来字？来字也晦气。」

韩泥不接茬。接茬，全丙九饿。他摸怀字条，纸糙，糙得像叶丫头当年送汤那碗边。「别来丑。」他低声重复，重复给自己听——不来，是不连累；丑，是别污她名。名在，他更不敢叫。午后，兽栏仍腥。腥不要紧，手稳才要紧。他铲下稳，稳得像踹台那一掌——掌在，字在，路在。刘婆端粥，仍多半勺：「泥岗来字了？」

「来了。」韩泥说，「四字。」

「四字怕什么。」刘婆哼，「手稳，饭稳。饭稳，恩不断。」

韩泥点头。谢轻，恩重。恩在粥里，也在字条里——字条无礼，有礼；有礼是愧，愧是叶丫头替他挡的谣，替他受的辱。铁无言抱臂立栏边，声低：「字别贴脸上。贴了，像求。」

韩泥应声：「不贴。记着。」

秦霜远廊经过，木鱼不敲：「心别邪。邪了，鼻下香飘。」

韩泥不答，答在心里——心里七笔分列，不混。泥岗六笔沉，宗门刘婆一笔烫；第七笔，叶愧线新开一线。愧不重礼，重名。名是叶青禾，名是「别来丑」——丑在脸，不在心；心在还，还就要活长。他回棚，揭席半寸，不揭坛，只让字条在膝上摊一息。纸黄，字稚，稚得真——真在「青禾」二字，真在「别来」二字割手。他低声道：「丫头，叔记着。丑着活，活长了才还得上。你别来，叔去。」

不去舌上，去心里。心里记一笔：愧线71，恩线仍热。热不退，像坛腹温一线。邻床悄问：「你真不恼？」

「恼在愧。」韩泥说，「恼不在字。字是护我，不是辱我。」

叶青禾名在愧线，愧线不碰舌。舌碰，名就污；名污，叶丫头那碗烫就还凉。韩泥编筐时，藤条绕指，绕得像把「别来丑」三字绕进掌纹——绕进掌，不进廊下笑。赖福堵棚口，晃木牌：「丑杂役，叶家丫头与你啥？啥关系，记过。」韩泥不辩关系，只答：「恩。恩在还，不在舌。」舌留来稳筐，稳筐才卖钱，卖钱才买米，买米恩才不断。

刘婆提水过，水洒半滴，滴在筐沿。沿湿一线，湿像叶汤那日碗沿热——热在记里，不在众目。众目问丑，韩泥只记：记别来，记别污名，记手稳。玉牌在怀，三年约在怀，字条也在怀——三件凉，一件烫。烫的是恩，凉的是仇与愧；分列，不混。

钱戾衡名在仇线，叶青禾名在恩线，「别来丑」在愧线——三线各走各的，走长了，才接得住95那盏门缝灯。他独眼平，手不抖：「别来丑。叔记着。记着，刃才磨得亮，恩才还得热。」

刃不亮，先记。记，是丑翁的本事。记完了，才配清栏，才配嗅渣，才配等坛里那魂开口。外头风硬，柱上条角白一线。白，像愧苗出土，根扎泥岗，拔不出。拔不出不要紧，先活——活在末席，活在忍里，活在还恩那日前头。赖福脚步在廊外停半息：「丑杂役，字条也晦气？」

韩泥只答：「在。」

在，就不滚。不滚，愧才扎得实——扎实了，才接得住臾墟开口，接得住煞骨门旧债那一问。他低声对坛：「叶丫头护名，叔护坛。坛在，名在，路就不绝。」

坛不应字，只温。温，像应「别来丑」的反面——不来丑，就来还；还迟，不还不算。更鼓将尽，天边白一线。白里，他再把字条塞怀深处，塞得像把愧摁进心记第七笔。第七笔不亮，亮在将来；将来还热汤，还省亲，还叶家那一席不丢人的面。管事夜巡验棚，灯照席角，照见净沿不见宝。不见不要紧，沿要净。

韩泥只答：「不拜。记着。」

散场后，药渣场角风硬。他分拣半刻，指稳，稳里鼻下静一息——静，不是忘愧，是把愧摁进恩里。恩烫，愧凉，分列才走得远。字条无署名，却有一角泥岗灶灰——灰淡，淡像叶青禾在灶前写四字时，袖口蹭的。蹭不要紧，真在「青禾」二字，真在「别来」二字割手。韩泥把灰拂去，拂，像拂辱；辱在字外，恩在字里。

字里「丑」是护，护名，护路，护她不必因叔沾谣——谣在沉丹宗，谣也在泥岗，丑叔不来，她才能正。他托沈枯芽回一句口信，信仍无字，只有半块干饼，饼边刻浅痕：「好。」

沈枯芽怯，点头：「叶姐……收下了。她说，叔丑着活，别来。」

韩泥只答：「不来。」

不来，是还；还来，是连累。连累不要紧，恩不断——不断，才配臾墟开口，才配坛底那一声哑。午后过坡下，叶家小舍远，门缝未亮——未亮不要紧，未亮是因「别来」。别来，他记；记，是把愧线71写进心记第七笔，笔不亮，亮在将来还热汤。白织月远廊经过，袖风冷，没停。没停，小胜——胜在活，不在嘴。赖福讽远：「叶家也嫌你晦气。」

韩泥不接。舌要留来记，留来忍。忍不是怂，是刃——刃对着三年约，对着「别来丑」，对着还恩那日。他编筐末单，藤刺扎掌，偏手，刺浅些。浅些，沿才净；净了，坛才藏得住。编筐钱买米，米在，恩才还得上——还得上，才对得起叶丫头这一刀字。更鼓二响，响沉。他合眼不睡实，怀三字并怀：三年约、别来丑、仍末。

仍末不要紧，手在；手在，就不滚。他低声道：「别来丑。叔丑着活。活长了，门缝灯才照得到你。」

门缝灯在远，钩在95；远不要紧，愧在近。近在，就不叫。不叫，才守得住坛，守得住恩，守得住心记第七笔。坛腹应温，不应字。温，像叶汤碗沿又热回半分——热在记里，不在舌上；舌响，香飘，愧就白挨。韩泥按实，按到掌烫，烫得踏实。踏实了，才配明日兽栏，才配等丑时那一声开口。

散队后，韩泥过侧门驿亭，亭风硬，硬里泥岗方向一线。他摸字条，纸糙，字稚，稚得像叶丫头当年握笔不稳。不稳不要紧，字真。真在「别来」，真在「丑」——丑是叔的脸，不是她的名。名要护，护就不去连累。他驻足半息，不折返泥岗。折返，像求；不求，只还。还在宗门，还在末席，还在忍里磨刃。

驿卒又喊下一姓，韩泥挪步，挪得像避光。避光，不是怕，是手还要清栏。清栏完，才配守丑时，才配听坛开口。坡下石阶霜化一线，化不要紧，仇苗仍在。钱戾衡眼风仍在，玉牌凉仍在——凉烫分列，不混。叔丑着活。活长了，汤才热，门缝灯才照得到。」

灯在95，远着呢。远不要紧，愧在近。愧是动力，不是跪；不跪，才配当丑翁。编筐角里藤刺仍在，刺他偏手。偏手像偏舌——舌不偏，费来记愧；手不偏，费来清栏。管事弟子验编筐，筐沿稳，记「手稳」。手稳，是小胜；胜在活，不在辩。辩轻，恩重。同批少年远避，避像避晦气。晦气不要紧，晦气遮瓮。

遮住了，叶字才接得住，坛才守得住。他摸怀，玉牌、字条、木牌三凉，坛温一线。温应恩，不应愧凉——愧凉也要记，记完了才还热。更鼓将起，柱上条角白，白像冬后已过，像三年才走了一小段。小段不要紧，根在；根在，就不滚。
