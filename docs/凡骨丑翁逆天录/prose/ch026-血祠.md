# 第二十六章 血祠

旱到第三月，井更深，水更浑。浑水照脸，脸瘦，瘦得像全村。全村瘦，瘦不过一张榜——泥岗村采药榜末，韩泥名在末，末得稳定，像旱里一滴水，滴少，也是水。村东废祠，无人祭，偏有人拜。拜的是泥坛，坛口红，像血干。干成壳，壳上有人新刻字，刻得歪：天罚晦气，血祠开，雨自来。韩泥过祠，不拜，只看。

独眼眯，见坛底湿，湿得不正常——旱成这样，坛底仍潮，潮得像与自家腌菜坛同脉。同脉，他不说，记。童谣丫念新谣，念得小声，被娘捂嘴：「血祠翁，晦气翁，送汤送姜天不容……」

谣新，旧骨还在：晦气、牵扯、绝户。骨上新肉：血祠、天罚。肉贴骨，贴得像真，真里带邪。周嫂传，传得像真：「叶家公审赢了，天仍不雨，是血祠示警！」

赵绣春门开半寸，声尖：「看吧！我就说牵扯晦气，连老天都罚！」

声尖，尖不过旱风。旱风刮祠，刮得泥粉落，落像灰，灰像祭。韩泥没辩。辩，坐实他懂邪。他懂坛，不懂拜。拜是别人的心，心是坑，坑要绕。里正带人过祠，踢倒新刻字，骂：「再刻，抓人。」拜的人散，散得快，像没拜过。快，心还在——心在，邪苗便还在。韩泥在暗处看，不看脸，看脚。脚虚，心虚；心虚，易拜。

易拜，便易被人推去血祠当枪。夜里，他低声道记着：心里又多一小圈，像旱，像坛。圈旁不写「血祠」，写仇苗外又一苗——邪苗，不急着拔，先记。坛在屋角，坛腹微温，温得他掌心微温。温不是醒，是应——应外头邪祠，应旱。应，不是信，是防。沈枯芽娘咳，咳得夜长。芽儿点灯，灯豆，像怕黑。

黑里，韩泥送半帖药到豁口，不放声。声惹祸，药不惹。叶家药铺灯也亮一瞬，亮得像秤还在。秤在，谣便不能全吞村。叶青禾隔帘咳一声，咳短，咳得像告诉全村：药铺开门，便是不拜血祠。不拜，秤才稳；秤稳，村才不全沉进旱里。韩泥低声道：「邪在祠，恩在心。心不乱，人不乱。」

风过废祠方向，像有人叹气。叹的是旱，不是韩泥。韩泥坐坛前，独眼闭一息，闭不是睡，是记——记邪苗，记旱，记公审后仍不散的嘴。坛腹微温，温得像叶汤应圈。圈浅，真；真，将来拔苗有据。他回屋，对坛低声：「你应你的，我记我的。记清了，将来不冤。」

坛腹微温一丝，像答。答无声，够。叶家药铺夜深仍亮一瞬，亮得像叶青禾在配药，配给咳的人，配给怕的人。怕血祠的人，也怕绝户谣——谣与祠，一嘴一坛，都是压。韩泥不往叶铺送字，不送，是护。护她刚松的气。气松，秤才能稳到天亮。天亮，童谣又起，起得低。低，比高声更毒——毒在孩口，孩口不懂，只学。韩泥路过，停，对丫说：「谣会旧，人会老。老的不是你，是嚼谣的嘴。」

丫眨眼，半懂。半懂，够。赵绣春门内又摔碗，摔得脆，脆得像不服公审，也不服血祠。不服，嘴还在；嘴在，邪苗还在。他走，走稳，像把邪苗圈在心里，圈住，不急拔。急，手乱，乱便中招。陈姑夜送一块干饼，饼硬，硬得像冬恩：「别信血祠。信坛，信心，别信嘴。」

韩泥收饼，掰半，半给沈家豁口，半自留。留，不是饿，是分恩。分清了，心不乱。叶铺帘内药杵轻响，响得像公审余波还在——余波不是松气，是秤还在，秤在，邪嘴便不能一口吞村。韩泥不跟里正辩，辩像同流。他只回屋，按心，按坛。心定，邪苗便不能立苗为命。叶青禾隔帘咳，咳短：「别拜血祠。拜邪，秤就停。秤停，全村沉。」

韩泥「嗯」一声，不近帘。不近，是护。护她刚松的气，护公审后那一寸安。里正又踢倒新刻字，骂：「再刻，抓人。」拜的人散，散得快，心还在——心在，邪苗便还在。他低声道：「旱在，邪在，恩也在。恩在，人不拜邪，人就能活。」

坛腹微温，应旱，应邪祠，不应拜。不拜，片温一丝，路不歪。陈姑又送干饼，饼硬：「信坛，信心。别信血祠嘴。」

韩泥收饼，分沈家半，半自留。分清了，心不乱，邪苗圈得住。第三夜，废祠方向有人低拜，拜声碎，碎像旱里求雨。里正带人灭灯，骂：「再拜，抓！」

拜的人散，散时嘟囔：「公审赢了，天仍旱，不拜拜谁？」

韩泥在暗处听，不辩。辩，像同流。他只回屋，对坛低声：「旱在天，拜解决不了。」

沈枯芽娘咳，咳得窗纸颤。芽儿点灯，灯豆，豆光里韩泥送半帖药到豁口。芽儿怯问：「韩爷爷，血祠真能让雨来？」

韩泥蹲低，声平：「雨在天，不在坛。坛是别人的心，心是坑。你别跳坑，跳了，叶姑姑药白配。」

「那他们为什么拜？」

「嘴怕旱，怕便找坛。」韩泥说，「你怕黑，点灯；他们怕旱，点血。血点错了，村就沉。」

芽儿点头，点得像米粒。米粒旁，韩泥心里又稳半分——稳，不是信血祠，是不让孩口学邪。叶青禾隔帘咳一声，咳短：「别往祠去。去了，里正连坐，连坐连到你编筐的手。」

韩泥在门外：「不去。我看脚，不拜坛。」

「看脚？」

「脚虚的人，易被人推去当枪。」他说，「枪不是我。我是编筐的。」

帘内药杵轻响，响得像秤还在量活路。量给全村，也量给怕血祠的人——怕归怕，别拜邪；不拜邪，公审那日松的一寸气，才不算白松。韩泥低声道：「邪苗圈住了。圈住，不急拔。」

坛腹微温一丝，应旱，不应拜。不应拜，路不歪；路不歪，将来拔苗有据。他低声道：「血祠在谣里，不在坛上。坛在，根在，公审在前。」

血祠谣落地，邪苗在影。韩泥扫庙换面——面实，比邪实。实，才走得到西驿，才进得丙九，才还七笔；七笔分列，不混，混则还乱。

旱在，邪在，恩也在。人不拜邪，人就能活——活长了，才还得起叶汤，还得上泥岗这几笔恩。

里正又贴柱角告示，告示末行「八日后上路，误者不许」。韩泥路过，独眼辨字，字丑，真——真在尺，尺量还路，量到西驿，量到丙九，量到带坛进宗门那日。他低声道：「八日，够辞村，够记实六笔恩，够不拜血祠。」

丑时，他把坛沿又擦一遍，擦得像把血祠谣从手上洗掉。洗不掉谣，洗得掉拜意——拜意不进心，心就不乱。心不乱，八日后上路，肩才不沉。沉的是篓，不是邪；篓里是恩，邪在祠里，各走各路。

叶家药铺夜深仍亮一瞬，亮得像叶青禾在配药，配给咳的人，配给怕血祠的人。怕归怕，别拜邪；不拜邪，秤才能稳到天亮。韩泥不往叶铺送字，不送，是护。护她刚松的气，护公审后那一寸安。沈枯芽隔巷探头，探头又缩，像怕牵扯。韩泥隔巷应，声哑：「别拜血祠。拜邪，叶姑姑药白配。」芽儿点头，跑远。

丑时末，村狗哑叫一声，像血祠谣也哑了半寸。哑半寸，够他合眼——合眼浅，坛沿仍净；沿净，八日后上路肩才不沉邪。里正柱角又添一行小字：「误期者，除名。」除名像滚，他不滚，只记尺——尺在还路，尺到西驿，尺到丙九。面实了，八日够辞村，够记实六笔恩，够不拜血祠上路。叶铺灯还亮一线，亮着，叶汤就不倒。稳。等帖。
