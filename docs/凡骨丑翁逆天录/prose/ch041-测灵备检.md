# 第四十一章 测灵备检

入门两月余，雪落了半寸，丙九棚外贴了新条。

条上写：凡骨杂役，测灵备检。备检不是正测，是筛——筛手抖的，筛嘴硬的，筛活不下去的。韩泥读完，独眼仍平。冬测过了，测灵仍排末。末他熟，熟就不慌。

邻床少年抖着嗓子：「听说正测在冬后。备检不过，正测连名都没有。」

「没名就滚。」另一人接，「滚了回家，比在这儿冻强？」

韩泥不插嘴。嘴要留给按石，不留给吵架。他把木牌摸了一下——冬测过的字还在，字在，心就有个垫。

棚里有人骂娘，有人穿衣服抖成筛子。抖成筛子不要紧，石前别抖成滚。他把破靴系紧，系紧了，步子才不飘。

刘婆从灶边递过一碗温水：「漱口。别空嗓子上去，空嗓子也像抖。」

「谢。」韩泥喝了一口，水淡，淡得醒神。

管事弟子喝开了：「丙九集合！测灵石前站直，抖的，滚！」

测场在侧门石坪，坪白石冷。灵石竖在中央，石青钝，像一杆秤——秤的是灵根，也是命。第一个少年按掌，石光一闪，绿线三寸。「过。下一。」

又有人按出黄线两寸半，脸却笑得僵。韩泥看着别人的手，不盯别人的光。光是石上的事，手是自己的事。

轮到丙九，赖福先推人：「丑杂役最后，别沾石晦气。」

管事冷声：「按序。韩泥，末。」

韩泥上前，掌茧裂处疼，疼也不露。他先呼一口气，气沉到腰，肩才不晃。晃了，别人看见的不是灵，是抖；抖了一寸，滚就近一尺。

掌按上去，石凉进骨。光起，一条灰线，短得不过指节。众人嗤笑，嗤声薄，也够刻薄。有人故意咳一声：「灰线？扔了得了。」

管事瞥一眼：「凡骨末等。备检过——手紧，石前不抖。」

赖福阴笑：「末等也过？丙九丢脸。」

韩泥收掌，不辩。舌留给嗅渣，留给活命。管事掷木牌：「韩泥，备检记名。正测冬后，仍末排。」

木牌薄，真在「记名」两字——有名就不被滚。邻床少年按石，绿线两寸，手却抖了半寸。管事喝：「抖！滚！」

少年哭腔起。韩泥不回头，回头容易手软。

同批有人低声：「灰线也过？」

「过的是手。」韩泥说，「石前不抖，才滚不掉。」

那人点点头，像把「紧」字学进肩里。白织月经坪外走，袖风冷：「末等灰线，也配站石前？」

韩泥「嗯」一声：「管事看手，不看嘴。」

白织月唇线绷了息，走了。少一句嘲，便是省一口气。铁无言抱臂立在栏边，声低：「灵根末，手紧，渣场仍收你。别漏。」

「不漏。」

管事收石，扬声：「备检毕。记名者，正测冬后。未记名者，明日滚。」

丙九静一息。有人盯着韩泥怀里的薄牌，眼神像盯半个命。赖福笑得浅：「末等记名，正测仍是末。末，滚得快。」

韩泥不答。答费气，气留给干活。散场路上，同批有人故意落半步，拿眼刮他木牌：「末等也配挂牌？」

「配不配，管事说了算。」韩泥道，「你挂上了吗？」

那人噎住，骂声矮下去。矮下去也好——骂赢不了灰线，也骂不来麸皮。

午后兽栏仍要清，雪混着粪，铲下要实。小兽角里咳，咳声细。他多清半刻，湿粪换干草，味淡了些。角里潮气贴脸，贴着也不躲。躲了味重，味重管事骂全棚。

管事路过：「备检过了，兽栏别松。冬角仍你，清细。」

「记着。」

赖福晃秤：「末等灵根，多吃一口也是浪费。」

韩泥不接。接了，全棚跟着吵，吵完仍饿。刘婆舀粥，仍多半勺，低声问：「备检过了？」

「过了。末等，手不抖。」

「末等怕什么。」刘婆说，「手紧，饭在，命在。命在，丹以后才学得上。」

韩泥应了声。同批少年凑近：「你怎么石前不抖？」

「搬石练肩，编筐练指，清粪练铲。」韩泥说，「练到紧，石前才紧。」

「可你是灰线……」

「灰线在石上，不在手上。」韩泥道，「手紧，饭在。饭在，才熬得到正测。」

管事扫赖福一眼：「你也去按石。抖了，你滚。」

赖福脸白，闭嘴。酉时渣场，韩泥分三堆，香渣单放。苦的、甘的，鼻下一辨就开。铁无言再瞥：「备检过了，仍别漏。」

街角赖福喝：「末等也配分堆？」

韩泥不回头，手仍分。分齐了，路才宽一点。管事验场：「丙九韩泥，渣清完，香篓齐。明日仍渣场。」

夜里他对坛低声：「灵末，手不末。」

坛壁爬上一线水汽，半寸便敛。邻床问：「末等，你还熬什么？」

「熬还。」韩泥说，「手紧能活。活了，才还得上。」

「还得上谁？」

「叶丫头汤，刘婆粥。」韩泥说，「泥岗那些，宗门这笔，各记各的，别搅浑。」

邻床又问：「正测冬后你仍末排，图什么？」

「图活。」韩泥说，「活一日，记一日。记满了，才还得上。」

邻床懂「活」，翻身睡了。外头雪细得像秤盘轻响。管事巡夜敲柱：「备检记名者，正测前别生事。生事，饭少。」

韩泥应一声，平得像木牌落掌。丑时油灯豆大，他擦坛沿三遍。坛壁忽然浮出极淡的影，像半个字——一闪就灭。掌心微烫，烫得像有什么隔着坛壳对上了半息。他没喊，没拜，把席盖压严。

邻床梦呓：「……啥光……」

「你看错了。睡。」

他低声对坛：「你眠着。我守。守到沿净，守到缘够。」

坛不应字，只沿口温一线，短得像应「守」，不像醒。翌日卯前，他先去兽栏，铲下实，小兽咳声轻了半分。管事弟子瞥他木牌：「末等，兽栏别散。」

「不散。」韩泥说，「散了栏，先散手；手散，石前就抖。」

弟子无话走了。午后渣场，铁无言掷香篓，篓口松，香泄一丝。韩泥分堆分得齐，铁无言「嗯」一声：「手紧，篓齐。明日仍你来。」

夜归丙九，邻床又问灰线。韩泥声低：「灰线在石上，手在铲上。手在，命就还在。」

柱上又贴细条：记名者渣场勿漏。他看条，看木牌，看坛沿——沿净，牌在，名在，漏不得。漏了，备检白过；白过，冬后正测就没排。

同批有人夜里咬牙：「灰线也能记名，规矩是不是瞎了？」

「规矩看手。」另一人哑声接，「你抖了，喊瞎也没用。」

韩泥听见，不插。插了像炫耀。炫耀不如明早把铲磨利一点。磨利了，粪少粘，手少抖，抖少了，冬后正测才有站场。

他又摸一次坛沿。沿温一线，温得像告诉他：灵可以末，命不能末。命不末，叶丫头那碗汤才还得上热。

第二天清晨，他把备检木牌又擦了一遍。擦不是拜，是怕字糊。字糊了，查起来容易被赖福咬成「没名」。没名，就滚。

赖福远处甩闲话：「擦牌有屁用，正测照样末。」

「末也站。」韩泥说，「站着，才有下一刀活。」

午后他把香篓口勒紧，勒到不泄。泄一丝，铁无言就看一眼；看一眼，日子就窄一点。窄不得——窄了，备检记名也白记。

刘婆路过棚口，又瞥他一眼：「手别抖。抖了，牌再真也没用。」

「不抖。」韩泥应着，把铲柄握紧。握紧不是用力，是催自己别忘：灵末可以，手末不行。手末了，冬后正测连站场都没有。站场都没有，还谈什么还恩还仇？

雪又落薄一层。木牌在枕下，与备检帖并放，像命和规矩挨着睡。贴着，才验得过后面。韩泥合眼，不睡死，是记着名。记着名，手就不许飘。正测还远，远不要紧——手紧一日，就多活一日。
