# 符录系统（符箓谱 · 全品质）

> **定位**：鸿蒙万缘塔**战修/除魔**支线的标准化载体；与丹药、灵器、法宝并列，主攻 **①赠礼 ②忠诚 ④感恩**。  
> **原则**：符为消耗品，真心赠出不计价；假符（邪教）触发簿预警。  
> **关联**：[`08-道具灵宠洞府系统.md`](./08-道具灵宠洞府系统.md) · [`04-修仙系统速查表.md`](./04-修仙系统速查表.md) · [`09-外道魔邪势力图.md`](./09-外道魔邪势力图.md) · [`11-阵法丹道系统.md`](./11-阵法丹道系统.md)（符阵丹分工）

## 一、符录定义

| 项 | 内容 |
|----|------|
| **符录** | 鸿蒙万缘塔内**符箓分册**；每赠一符、每用一符，塔铭记品阶与善缘 |
| **符箓** | 一次性或有限次消耗；战斗、遁走、传讯、镇邪 |
| **炼制** | 苏念慈丹堂（主）、清虚观授谱（道）、莫长春以缘点符（辅） |
| **品阶** | 与全书道具统一：**凡→灵→宝→仙→神** |

## 三、与鸿蒙万缘塔绑定

```
赠符（①）→ 塔铭品阶+对象 → 用符除魔/破局 → 善缘/因果↑（②④）
                ↓
         回赠专属符（T05/T15）→ 馈缘+、倒贴铺垫（③）
```

| 规则 | 说明 |
|------|------|
| 赠符记账 | 正文须写「Txx·品阶」或等价（如「灵品除魔符」） |
| 对症赠 | 赠赵战符→忠诚+；赠顾遁符→亲密+ |
| 当众赠符 | 柳/苏见赠符→亲密+5，吃醋钩子 |
| 符尽回找 | 「符用完了回来找我」→ 赵/顾忠诚链回收 |
| 假符 | 邪教 T17 类；簿示「假缘」预警，秦揭穿 |
