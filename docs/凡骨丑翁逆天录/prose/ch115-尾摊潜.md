# 第一百一十五章 尾摊潜

公示第二日，韩泥要坊市一行。

尾摊潜，不挂牌，只拣漏。拣漏不要紧，手稳才要紧。手稳，沉腐浮不落舌，漏才捡得到。

铁无言立十步外，声低：「潜，别贪。贪，赖福记过。」韩泥应声：「不贪。沿净。」沿净，守坛，守116捡漏得。

坊市尾摊，人少，少不要紧；少，才像潜，不像求名。求名像攀，他不攀——攀要还人情，他还不起。

坊市尾风硬，铁无言立坊市尾侧，声低：「尾摊潜拣，别飘。飘了，漏灰就洒。」韩泥应声：「不飘。漏灰在，手在。」手在，就不滚；不滚，捡漏得才接得住。

药山废渣多了半袋，半袋能换三文铜，三文够买捆旧藤——藤旧，能编筐，筐能抵赖福查。查不要紧，手稳才要紧。要紧是坊市尾摊，尾摊有捡漏传闻；传闻不要紧，去一趟，才知真假。真假在鼻下，鼻下嗅得出一丝灵——灵淡，淡得像渣里香。

丑时末，他披破袄，揣玉牌，独眼辨路。路滑，霜白，白得像仇苗出土。出土不要紧，先活；活，才捡漏，才还恩。还恩序不乱，叶汤头恩在前，芽儿半块灵石在后——分列，不混。

侧门守卫打盹，韩泥贴墙过，过得像风。风过，不惹眼；惹眼，赖福知，知了就除名借口。借口在腹，腹有气，气不露；露了，公示第三日就成真滚。

坊市尾摊，天未亮，摊主张灯如豆。豆灯下，残器、残页、药槽边铁，堆得像垃圾。垃圾不要紧，鼻下嗅得出一丝灵——灵淡，淡得像渣里香。香正，手才稳；手稳，一文两文才值。

韩泥蹲摊，指节拈铁片，铁锈厚，厚里一点温。温正，像炉边角。摊主睡眼：「一文，拿走。」

一文拿铁，铁入怀，怀沉，沉得像捡着半条命。命不要紧，不露；露了，白织月说「偷捡」。偷捡不是，是淘；淘在尾摊，淘在五年杂役，淘在丑手里。

他又翻残页，页碎，碎里三字可辨：「焚火诀」。诀残，残不要紧，够稳窑温——温稳，培元散才正；正了，复测才配夺魁。魁在远，捡漏在近；近，手稳，就不白挨公示。

摊主哼：「两文。爱要不要。」

韩泥摸出三文，三文换铁加页，剩零，零藏袖。藏袖，像藏路——路向炼气八层，路向还沈枯芽半块灵石。石在远，尾摊在近；近，捡了，路宽一寸。

天边白一线，早市人稠。稠里，白织月带婢过，婢指他：「丑杂役也逛坊？」

白织月袖风冷：「末等捡垃圾，像配。」

韩泥不辩。辩，费舌；舌要留来藏铁藏页，留来贴墙回宗。回宗，赶在卯时散队前——散队前到，赖福不知夜行。不知不要紧，铁页已藏窑缝——缝窄，窄得像席严那夜。

卯时散队，赖福点人，点韩泥：「在。晦气还在。」

「在。」韩泥只一字。

一字，手稳。手稳，赖福找不到夜行由头——由头在腹，腹有气，气不露。不露，八层门槛开一线；开一线，不声张，声张，赖福指「偷境」。

午后废窑，他展残页，页碎，碎里「焚火」二字像钩。钩进眼，眼进手，手进炉——炉无，窑有；窑温一线，按诀走，走半周，温升半分。升半分，炼气八层门槛开一线。开一线，不声张；声张，赖福来；赖福来，公示末席又加辱。

铁片嵌窑口，嵌得像炉角。角稳，渣炼时香正一分——正一分，够记，够熬，够接赖福最后一压。最后一压，在除名试炼；试炼过，九层开，还芽儿在前。

铁无言过窑外，声低：「坊市回来的？」

「买藤。」韩泥说，「藤编筐。」

「筐实，手稳。」铁无言说，「别漏。」

韩泥应声。应声，像应坛。坛在角，角稳，捡漏才不白捡——白捡，辱就白挨；不白挨，路宽一寸。宽一寸，够炼116那一炉，够稳窑温。

刘婆送粥至窑道，粥稀，热：「夜去哪？」韩泥说。

「买藤别漏名。」刘婆哼，「漏名，赖福克。克了，饭更稀。」

韩泥点头。谢轻，恩重。恩在粥里，也在忍里——忍夜行险，换一文铁两文诀；换得值，值在窑温，值在八层。八层实了，才配除名试炼，才配还沈枯芽半块灵石。

公示第二日尽，坡下仍指他末席。指不要紧，腹里气涨半寸——半寸，八层。八层，仍立末；立末，手稳，就不滚。不滚，三年约就不绝；不绝，钱戾衡「让你滚，你才有资格不滚」才接得住。

他按玉牌，牌凉：「三年约，记着。坊市一行，也记着。」

记着，捡漏才不白捡。白捡，仇苗浅；不浅，刃才磨亮。亮在还芽儿，亮在复测，不在尾摊。尾摊潜行，像风；风过，不惹眼，惹眼就除名。

丑时，练鼻下静。粟壳香淡，混一丝炉角铁锈味——味淡，淡不要紧；要紧是香一字不落。落齐，八层记进心里，不进舌。记进心里，才实；实了，才配116捡漏得诀，才配炼气九层。

外头赖福骂：「公示二日，明日仍末！」

韩泥低声道：「在。」

在，就不滚。不滚，尾摊潜行才值——值了，接赖福压顶；压顶后，是还沈枯芽半块灵石。石在远，八层在腹；腹里温一线，坛应「捡」——捡，是丑翁的本事；本事在渣里淘，淘在尾摊里淘，淘在五年杂役里淘。

坛腹微温，应「捡」字。捡，是丑翁的本事；本事在渣里淘，淘在尾摊里淘，淘在五年杂役里淘。淘出来了，才配炼气九层，才配徒恩首还。首还，静着还，还到芽儿手里，才算数。坊市归来，天已微亮。韩泥贴墙进侧门，靴底泥少，少得像没夜行。少泥不要紧，铁页藏窑缝，铁片嵌口，露不得。

卯时散队，赖福点人，点韩泥「在」。在，就不滚。在，夜行无痕——痕在腹，腹有八层气，气不露。露了，赖福指「偷气」。

管事弟子验筐，筐实，藤屑新，新得像刚编。赖福阴笑：「实也晦气。」韩泥不辩，捧筐交差。交差，手稳，记功一笔。

午后废窑，展焚火残页，页碎，碎里二字「焚火」像钩。钩进息，息进温，温升半分。半分，八层实；实了，仍末席，末不要紧，手稳。

铁无言送边角渣，渣里一丝灵材，灵材细，够掺一炉。掺不要紧，效正；正了，备复测。复测在远，赖福公示在尽——尽在即，尾摊值。

刘婆送粥至窑道：「夜行别跌。」「没跌。」「没跌就好。跌跤，粥洒；洒了，恩断。」韩泥喝尽，记——记刘婆恩，记第七笔烫，烫在粥，烫在姜，烫在肘布。

白织月药山道再嘲：「丑杂役也逛坊？坊市垃圾配你。」韩泥不辩：「垃圾里有时有温。」温在铁片，温在残页；温不要紧，不露；露了，成盗。

夜，八层气走周，周周实。实像坛腹应「捡」——捡来的命，捡来的诀，捡来的路。路向116稳窑，路向117试炼，路向118还芽儿。

他低声道：「尾摊潜行，记着。八层，记着。」记着，就不白险。白险，辱白挨；不白挨，捡漏才值——值了，接赖福除名试炼，接沈枯芽半块灵石。

八层实后，韩泥夜守窑缝，铁页不示，铁片不露。不露，坊市潜行才值。值了，116稳窑炼粒，117试炼九层，118还芽儿半块灵石。石还了，徒恩首笔；首笔不张扬，坛温应「还」。还应刘婆，还应叶汤，还应泥岗。还应，路才正；路正，粒才正；粒正，试台才说话。

他摸玉牌，牌凉，低声道：「三年约，记着。手稳，粒正，试台见。」见字落，坛温一线，像应。应了，路又明一寸——明向还恩，明向不滚，明向刘婆新衣养老钱。

更鼓沉，沉里，韩泥按木牌，牌凉贴胸。115章手在，名在册，坛沿净。舌不留，鼻下静。恩辱分列，不混。活长了，才还得了叶汤，才接得住门缝灯。
