# 第六十七章 冬测风声

冬风又硬一分。

柱上条角起霜，霜化了，字更显：「冬后正测，凡骨杂役，卯时坡下见。」

韩泥读条，独眼平。正测不是备检那点私下过关，是公示——公示灵根，公示末席，公示谁滚谁留。滚字写在条上，他心里先写另一个字：手。手稳了，滚也难随随便便抹掉。

同批少年嗓子发紧：「听说内门弟子也来。来，看笑话。」

「看笑话不要紧。」韩泥说，「手抖才滚。」

少年愣半息，没再问。赖福立条下，笑薄：「丑杂役，正测一公示，你灰线三尺，滚定了。」

「定不定，看执事。」韩泥不跟他抬杠，「抬杠饿肚子。」

管事弟子喝：「丙九集合！听训！正测前，手抖的，先滚！」

训声沉。白织月过廊角，袖风带一点冷香，瞥到韩泥末席，唇线绷紧：「凡骨也配听正测训？配站坡下？」

「管事看手，不看嘴。」韩泥「嗯」了一声。

她走远，还撂一句：「手稳有屁用。灵根末，丹路断。」

话冷。冷不过叶家那碗汤——汤热过，热在胸，胸热，手指就不飘。少辩一句，就是小胜。

铁无言抱臂立侧，声低：「测灵验色，也验手。手抖，色再正也滚。」

「掌上稳。」韩泥应。

秦霜远廊经过，木鱼在指间，不敲：「心别邪。邪了，鼻下香飘。」

韩泥心里记下：正测是秤，不是拜。拜邪断路；不拜，各干各的。

管事读条三遍，字像钉子：「三日后卯时，公开测灵。末席公示，滚者除名。」

三遍钉进耳朵。耳里响，手却不能抖。他摸木牌，牌凉，膝下却像有一点温——温在坛，温在「仍末」还没写成「滚」的那一日。

同批有人嘀咕：「条念三遍，是要把人念滚吗？」

「念滚的是抖。」韩泥低声，「抖别在石前露。」

午后坡下空着，空得像等一场雨。雨没来，风先来，风里有人练剑，剑光一闪，闪得像内门的眼。眼在，笑话就在；笑话不要紧，手稳才要紧。

邻床悄问：「内门真来？」

「来。」韩泥说，「来也不近。近了沾晦气，远了正好看笑话。」

少年懂「笑话」，闭嘴。赖福又传话，毒：「钱戾衡师兄要来看正测。专看末席滚。」

韩泥指节白一息，又平。钱戾衡这名字他听见了，记在仇线里——记今晚不必碰，碰也碰不到。今夜他清栏，嗅渣，守坛。

小杂役蹭过来：「你怕不怕？」

「怕。」韩泥说，「怕也记着。记着，手就不飘。」

刘婆端姜汤来，姜薄，热：「正测前，别空肚。空肚，手飘。」

他喝尽：「不空。姜在记里。」

「记里有什么用？」小杂役嘀咕。

「有用。」刘婆瞪他，「记里热，人就不飘。」

铁无言又路过，只剩一句：「坡下见，验手，不验嗓。嗓响，香飘。」

「鼻下静。」韩泥答。

他上坡下石阶试站。阶滑，霜白，末排风更硬。站半息，像把三日后的脚位先踩实。管事贴公示榜，榜字丑，丑里有「滚」。少年们挤着看，韩泥立末不动：「站次定了，手才定。」

廊角有人练剑，笑一句：「凡骨也配站这阶？」

他不答。答费舌，舌要留来按石。那人骂得没趣，走了。

管事又添半行小字：「公示末席，滚者除名；手抖者，先滚。」韩泥读完，不近，也不躲：「验手。手在，名在。」

丙九每人发薄册抄规矩。席地抄，韩泥笔拙，字丑，抄到「末席公示，滚者除名」时笔停半息，又落下——落齐了，才像把条钉进掌心里。

同批少年抄到手抖，管事喝：「抖，先滚！」少年立刻按腕，腕稳，字才稳。韩泥瞧一眼，不教——教像显摆，显摆近晦气。

廊下执事抬测灵石过，石温，光淡。光要照手：手抖，光照出抖；抖了，滚字成真。石光扫过他掌茧，茧裂旧，光落不上灵色，却落得上手稳。执事瞥他一眼，没嗓——没嗓，也像允许他再活几日。

有人悄悄问执事：「灰线末的，还抄规矩？」

执事头也不抬：「抄。抄了才知滚字怎么写。」

钱戾衡的名字从廊下飘过，飘冷。老耿挑水过坡，水洒稳，声哑：「坡下见，站末不要紧。末，不滚；滚，看手。」

「手还在。」韩泥应。

秦霜再过，木鱼轻敲一下又止：「心别求胜。求胜，像借香。」

韩泥把七笔记在心里：泥岗六笔沉，刘婆一笔烫。烫在肘、在粥、在正测前这三天。烫在，就不怕白织月的嘴。

夜前他又清栏半刻，铲下稳，稳得像先把「手」字写进掌纹。邻床问：「你真不滚？」

「滚看手。」韩泥说，「手在，就不滚。」

管事夜巡照名册，灯停在「仍末」两字上，笔尖顿半息：「正测前，别空肚。」

「肚不空。」韩泥应完就退半步。近了像求情，求情像拜——拜了，路断。

铁无言远廊半句：「公开测灵，验手先。」

「验手我认得。」韩泥道。

坡下石阶夜里又起霜，霜白得像给三日后垫一层素布。素布不是体面，是冷。冷不要紧，路还得直——直了，才还得了叶丫头那碗烫。

赖福立条下再笑：「三日后专看你滚。」

韩泥只答：「在。」

在，就接得住那一验。验手，也验忍。天边白一线，柱上条在风里猎猎响，响不像催他滚，像催他把手再按实一分。

棚里有人翻身，问得发虚：「要是内门真当面笑呢？」

「笑就笑。」韩泥说，「笑进耳，别笑进手。手一进笑，石前就散。」

那人噤声。韩泥摸怀木牌，牌边起霜，霜化，字更显——「仍末」两字像被冬风又摁实一摁。忍不是怂，是刃。刃不亮，先磨；磨在丙九，亮在还恩那日。

夜半他又起身清了半撮兽栏湿草。湿草沉，铲却不偏。邻棚有人骂梦话：「灰线……滚……」骂完又鼾。韩泥停下听一息，听完继续铲——梦里滚，梦外只能把手练稳。

小杂役披袄探头：「你还不睡？」

「睡不实。」韩泥说，「实了像忘条。条在柱上，我忘不起。」

「那你怕钱师兄？」

「怕。」韩泥直说，「怕也去。不去，滚来得更快。」

小杂役缩回去。风从棚缝钻进来，冷得像条上霜。他把木牌攥紧一瞬，又松开——攥太死，明早钟响手倒发木。木了，才真滚。

他低声对坛：「三日后卯时，坡下见。见在末排，不在舌上。舌响香飘，香飘手就散。」

坛不应字，只温一线。温完又敛。敛着，才守得住秘密，守得住仍末，守得住公开测灵那一日——钱戾衡若真要来看笑话，笑话也得先过他掌上那一点稳。稳住了，条上「滚」字才不好随便抹到他名上。条未换，风先换——风硬了，正测就不会软着来。

卯时将亮时，他再把规矩簿翻到末页。末页「滚者除名」四字被他指腹摩过，摩得纸起毛。起毛不要紧，字还在。字在，人就得按字站。刘婆隔棚送半块冷姜：「含着。含着，手热。」

「谢。」韩泥含进嘴里，姜辛冲鼻。辛得真，像三日后石前那口凉也冲得开。他咽下姜汁，低声道：「三日后见。」姜还烫舌尖，舌尖烫，话说出口才不飘。

赖福一大清早又蹲条下，冲他扬下巴：「丑杂役，今日再练站？站到石前也是灰。」

「灰也站。」韩泥走过，不停，「灰站稳了，比绿抖强。」

赖福骂脏字。骂声落在背后，像石子打井——咚一声，沉底，掀不起他掌上那点稳。稳字丑，丑翁专啃。啃到正测那天，才拿给人看。给人看之前，先把这三日过完。过完，才算正测入场验手。
