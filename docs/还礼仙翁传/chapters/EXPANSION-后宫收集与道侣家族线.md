## 一、后宫收集铁律（不收一位）

### 1.1 设计目标

| 项 | 要求 |
|----|------|
| **收集向** | 鼓励收满 **正绶1 + 侧绶3**（最多4名道侣），非「只娶一人」单线 |
| **主次分明** | 正绶主位；侧绶按 **综合羁绊排序** 分副位一/二/三 |
| **有情有义** | 对每位道侣**本人+其家族**真心赠礼，家族回赠入主线 |
| **真结局** | **E14 鸡犬升天** 优先满编道侣登舟；E13 单携为次结局 |

### 1.2 位阶与名分（主次）

```
┌─────────────────────────────────────────────────────────┐
│  正绶（大绶·主位）  ×1   综合羁绊最高 · 飞升主携        │
├─────────────────────────────────────────────────────────┤
│  侧绶一（副位）     ×1   侧绶内羁绊最高 · 副携排序第1   │
│  侧绶二（副位）     ×1   副携排序第2                     │
│  侧绶三（副位）     ×1   副携排序第3 · 含厉隐藏线可选    │
└─────────────────────────────────────────────────────────┘
```

| 位阶 | 人数 | 名分 | 别院 D5 | 飞升 |
|------|------|------|---------|------|
| **正绶** | 1 | 主位；正室名分 | 正室·主卧连会客亭 | **主携席** |
| **侧绶一** | 1 | 副位排序第一 | 东厢·近会客亭 | 副携席① |
| **侧绶二** | 1 | 副位排序第二 | 西厢 | 副携席② |
| **侧绶三** | 1 | 副位排序第三 | 南厢 | 副携席③ |

**排序公示**：ch898 预纳绶立意向 → ch944～945 纳绶夜 → ch1113 **鸡犬升天榜（F19）** 公示综合羁绊 → 玩家/剧情可对话让位（柳苏分差＜3 喜剧）。

**铁律**：
- 嫉妒须先化解方可纳下一位（修罗场喜剧，非黑化害人）
- 每纳一人消耗 **合契丹×1**（D09），苏开炉喜剧
- 纳绶后仍可持续赠礼其**家族**，馈缘与家族善缘独立累计

## 四、分人家族剧情（详）

### 4.1 柳青鸢 · 青岚柳氏

| 维度 | 内容 |
|------|------|
| **家族处境** | 柳氏旁支掌执法堂；宗主位年轻，长老会主和派拱火割地 |
| **情义赠礼** | 赠柳氏祖坟护阵符；东脉战赠灵阵盘耗材；战后赠《柳氏剑谱》残页予旁支子弟 |
| **家族回赠** | 柳氏旁支战时充先锋；名分定后旁支立「赠缘碑」侧碑 |

## 五、家族线 × 六步穿插

| 步 | 家族玩法 |
|----|----------|
| ① 赠礼 | 赠家族信物（寿元丹、剑谱、商路帖、赎债灵石） |
| ② 四维 | 道侣忠诚+；家族「善缘」独立条 |
| ③ 倒贴 | 美人代家族出面邀茶/求援（有情有义，非卖女） |
| ④ 感恩 | 家族援军、商路、丹师团回赠 |
| ⑤ 纳绶 | 纳绶典礼**家族观礼席**；合契丹由苏氏炼、秦氏供材 |
| ⑥ E14 | 家族代表可选眷升（非道侣）；道侣全家登舟台词 |

**铁律**：家族赠礼须**真心**；交易式联姻（魏无涯割脉式）走奸线，莫长春拒签。
