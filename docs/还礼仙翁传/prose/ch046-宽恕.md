# 第四十六章 宽恕

思过崖口，风硬，石冷。

赵玄机跪，膝磕石，声哑：「弟子……陷害长老，罪该万死。」

莫长春拄杖立他面前，不扶，不骂，只一句，像落账：

「去吧。恩是我留的，你得还人情——以后。」

赵玄机抬头，愣，愣得像听不懂「以后」二字：「长老……不罚？」

「罚了，」莫长春道，「你就只记着恨。记着还，才记得住守门。」

赵玄机眼眶红，重重磕头，额血沾石，像把妒火磕裂一缝。

远处，林挽月瞧见，心摇，摇得像廊边竹。她想近，又止，止得像怕自己一近，就要在堂上把话说尽。

执法堂弟子在崖下窃语：「赵执事思过，莫长老不罚？」「恩字……什么意思？」「听说夜袭将至，罚轻了？」

窃语里，慕与怕交织，像全宗都在等赵玄机是忠是奸的最后一笔。

柳青鸢立崖下远处，未近，眸光冷，冷里却有一线松——松的不是奸，是师兄又把一条路留出来了。

霍镇山骂：「赵小子，思过别白思！夜袭要守门！」

赵玄机跪姿更直，直得像终于把「还库」之后的那一寸，跪进了石里。

莫长春转身下山，杖点每一步，袖中簿微烫，字淡：

**宽恕有时，忠诚入账。**

他念：「赵小子，别浪费半月思过。半月后，东脉要人守门。」

风过崖口，血痕干，干得像妒火也干了一寸。

弟子围观下山背影，围观里窃语反转：「莫长老不罚赵执事？」「恩要还？」「还什么？」

魏无涯观崖口，低笑，像看妒火线收束一截，收束里夜袭已在门外。

赵立起，望丹堂，眼亮，亮得像等还恩夜。

他低声，像钩：

「思过崖风硬，正好把妒火吹冷。冷透了，才知道门往哪守。」

崖上，赵玄机立起，望丹堂方向，眼亮，亮得像终于等到还人情的夜。

有人低声：「不罚？莫长老疯？」另一人答：「疯的是妒，不是恩。你看赵执事额血。」「额血像还库之后的路。」窃语反转，反转成疑，疑里又燃——燃的是夜袭要人守门，守门的人，会不会是赵？

孙福旧党嘀咕：「宽恕？宽恕谁？」顾小满冷眼：「宽恕守门的人。你守什么？」

苏念慈在丹堂听讯，炉火旺，旺得像替思过崖也担。她低声：「宽恕不是纵，是留路。」

林挽月远立，心摇，摇得像廊边竹。她想近，又止，止得像怕自己一近，就要在堂上把话说尽——证字还压在喉里，时未到。

霍镇山骂罢思过崖，又骂丹堂：「赵小子，半月思过别白思！夜袭要守门，门比妒人难！」

崖上风硬，血痕干，妒火冷一寸——冷透了，才是守门。全宗尚不知：宽恕不是纵，是留路；留路的人，要在子时还恩。

魏无涯观崖口，低笑，像看妒火线收束一截，收束里夜袭已在门外。赵玄机立起，望丹堂方向，眼亮，亮得像终于等到还人情的夜——夜在子时，恩在门，门在还。

「罚了，」莫长春道，「你就只记着恨。记着还，才记得住守门。」赵玄机眼眶红，重重磕头，额血沾石，像把妒火磕裂一缝。

围观弟子在崖下踮脚，踮得像等赵执事再辩——辩没来，来的是额血，血像还库之后的路。有人低声：「恩要还，还什么？」另一人答：「还守门。门比剑难。」

崖下角落，两弟子压着嗓：「思过还能改十日？莫长老年事糊涂了吧。」另一人却摇头：「你糊涂。门一开，恩不开。」话没落，赵玄机膝再磕石，声哑，围观里那一瞬静了——静得像妒火被风按住，按住片刻，才又窃起。

莫长春下山时，弟子让路，让路里仍有人低声：「不罚像纵奸。」霍镇山自后喝：「纵的是路，不是奸！路开了，夜袭才有人守。」喝声洪，洪得像把「以后」二字也砸进每个人耳里。

莫长春独坐廊柱旁，不坐主位，望思过崖方向，杖横膝，簿无新字，袖却沉，沉得像把赵玄机那一缝妒火，暂存待后兑。

弟子在崖下仍窃语，窃语里反转未止——反转不在不罚，在「以后还」三字，还的是门，不是嘴。风过，崖口血痕干，干得像妒火也干了一寸。思过崖风硬，正好把妒火吹冷——冷透了，才知道门往哪守。大因果还在夜袭子时，像妖骑，像杖剑；宽恕只是小留路，赵死守才是大还礼的前奏。
