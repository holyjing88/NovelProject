# 第十五章 铺垫公审

村议辱叶后两日，里正又敲锣。

锣声沉，沉得像把全村的嘴先按进泥里。韩泥胳膊上的布还没拆，天亮照旧编筐。野藤刮去老皮，泡软了，才往膝上绕。绕一圈勒紧，再绕一圈，尾头塞进结里——编筐靠实，不靠嗓门。藤条勒得指节起皮，他也不慌；慌了结松，筐一塌，集尾就没人买。

编到晌午，筐底圆了，沿也齐了。他用指节敲了敲底，声闷而实。锣又响，他搁下藤条，往祠堂去。

「叶家事，不能糊。」里正扬声，「明日巳时，祠堂开村会，全村到场，辨诬还清白。缺席，记名。」

祠堂前人又挤满了。赵绣春那一拨没散，散的只是胆子——胆怕里正报官，也怕上头查连坐。

马春花嗓子还尖：「辨什么？寡妇送汤，事实！」

里正板着脸：「事实，当众说。说不服，再罚。」

韩泥坐在末席，独眼低着。低不是怂，是等——等叶家开口，也等锤子有个准日子。

议到午后，争出三条。一是叶家关铺谢罪七日；二是韩泥搬离村西，去住破庙；三是开村会，村人作证，辨清楚「送汤是恩还是邪」。

赵绣春抢着要一、二，不要三：「三有什么好辨？辨了倒给晦气翁长脸？」

帘后叶青禾的声音平平传出来：「要三。不要一、二。」

「不要关铺？」周嫂叫，「铺子晦气，谁还敢抓药？」

「铺子开。」叶青禾道，「抓药的是病，不是嘴。」

里正拍板：「三留。一、二缓。村会要开。」

场子里有人倒抽气，有人抠衣角。抠衣角的人昨夜还跟着喊「关铺」，如今嗓子短了半截——短不是心软，是怕日子落了字，字落了就赖不掉。韩泥坐着不动。动了像抢锤；不动，锤子才听得见他自己的心跳。

「何日？」有人问。

里正望了望天，天阴着，像旱前兆：「明日巳时，祠堂。帖今日贴。」他补一句，「日子定了。谁再辱叶、砸秤，先记名，再报县。」

里正贴帖，浆糊沾在指上，指红得吓人，却无害。村人围着帖念到「巳时」，有人拱手，像在拜日子——拜也是怕。韩泥不拜。他回屋摸了摸坛沿，沿冷，心却热着。热不是火，是那碗汤还压在心口——压住了，村会才不像听戏，像还账。

告示桩边有人抄帖，抄得歪，嘴里念「巳时」。念完又四下看看，像怕被赵家听见自己念认真了。韩泥担筐从旁边过，不停脚：「念清楚点。念糊了，锤子也糊。」

那人一缩脖子，笔尖却顿了一顿，顿完再写，写得慢了些，也实了些。

马春花瞥他一眼，啐：「晦气翁，明日你也去！去听听寡妇怎么辩！」

韩泥「嗯」一声：「去。听真话。」

马春花一愣，还想尖，赵绣春拽她袖口，嗓子压低却故意让人听见：「让他去！去了，正好坐实一对！」

韩泥不辩。辩就是嘴仗，嘴仗赢不了叶汤那碗温。心里那碗热汤已经压实了，够他扛到明日村会。

日子一定，锤才有期。赵绣春退了半步，退得不甘，像牙咬进空气。周嫂还想嚷，被里正一眼压住：「嚷到村会上嚷。今日散。」

散场时，村人三三两两，嘴里仍转「辨诬」二字。二字新，新得像门缝透进来的光——光不大，却照见：叶家未必输，谣未必赢。

祠堂阶下，有人扯住里正袖子，低声问：「村会，赖管事来不来？」

里正沉声：「来不来，村都要辨。辨不清，连坐先落在嘴里。」

侧巷里，叶青禾等着，隔开三步，没近身。「韩翁。」

「嗯。」

「村会……」她顿了顿，「会连你。」

「该连。」韩泥独眼平，「汤是你送的。恩是我记的。连上了，正好辨。」

她嘴唇动了动，还想说谢，他抬手止住：「别谢。谢字轻。等辨完，我还。」

叶青禾低声：「赵绣春说，会前还要闹。闹到关铺。」

「闹。」韩泥道，「铺开着，闹就空。空的是嘴，不是秤。」

她「嗯」一声，布鞋踩着泥走了，步子轻，却稳。韩泥摸了摸心口——叶汤那笔恩深深钉着，钉没落下，洞却先凿好了。

傍晚他担筐回村，路过告示桩，见短签角又被风撕开一分，撕不掉底下的名。名在仇里生了苗，村会前这苗还得压住，别让它盖过汤。

药铺外忽然「哐」一声。秤杆被人砸中，杆弯了，没断。

叶青禾扶住杆，指节发白，嗓子仍平：「砸断了，全村没秤。你们砸谁？」

巷口人影退得快，像怕里正记名。韩泥隔巷听见，没凑近。凑近就成「牵扯」。他只低声道：「秤在，铺就在。砸秤的，村会上见。」

叶青禾把秤杆扶正，秤砣再落一声——落给全村听：铺还在，脊还直。

巷里有个买药的汉子刚退半步，又站住了。他低声骂句「贼折腾」，却还是把药方捏紧，没走。药方在手里，比谣言实——实的东西，才撑得住明日祠堂。

韩泥看见了，心里那笔恩又沉一分。沉不是累，是稳：铺没关，人还敢来抓药，辨诬就不空谈。

里正闻声赶来，锣举着没敲，先问：「谁砸的？」

巷里静了。静里，赵绣春门内啐了一声，啐得碎，像心虚。

里正板脸：「再砸秤，先记名。村会日子定了，砸秤也是辱。」

叶青禾扶着杆道：「秤杆弯，弯不断。断的是谣，不是铺。」

夜里，里正来叩门，三下，沉。韩泥开一条门缝：「里正。」

「村会要开。」里正嗓子压低，「你少往前冲。冲，害叶家。」

「我不冲。」韩泥道，「我站。站末席，等锤。」

里正顿了顿：「锤落谁身上，还没定。定的是，谣不能糊。」

「糊不了。」韩泥独眼平，「汤烫着，糊不了。」

里正「嗯」一声，脚步远了。韩泥关上门，低声对自己说：「日定了，理也定了。定的是辨诬，不是关铺。」

门缝里塞进来一片薄姜，像芽儿的手。他拾起来，没尝——尝像收谢；他只记在心里：娃的胆，也算一证。

夜里，叶青禾隔巷问：「韩翁，村会上……我怎么说？」

「说汤。」韩泥道，「热，送，该送。别辩奸，奸在谣嘴里。」

「你呢？」

「我认恩。」他道，「恩我担。担了，正好连。」

她「嗯」一声走远。远处药铺灯仍弱弱亮着——亮不大，像村会前一盏烛：烛救不了嘴，却救得了抓药的人。

赵绣春过巷，瞥他一眼，尖声涌到一半，见巷口里正的锣还在，又咽回去。日子定了，赵党也想借锤；人心却已稳了一半。

张麻婆从井台提水回来，嗓子粗，故意说给巷里听：「帖贴了就得认。认完再吵，吵也有边。」她看见韩泥门缝灯，顿半息，「明日巳时，你坐末席就行。站起来害她，我不认你这笔恩。」

韩泥隔门应：「知道。末席。听锤。」

「嗯。」张麻婆啐地，啐向赵家那边，「谁再砸秤，先砸他自己的脸。」

韩泥收筐，藤条在手里很平。明日巳时祠堂见，叶汤得立住——立住了，才走得向公审那日。羊圈边村狗叫一声，又哑了，像全村都在等这个日子。他不拜日，只把筐编实：手平了，汤才能立。

丑时前，叶家药铺灯还亮。亮着，叶汤就不倒。韩泥举目西望，灯一线，像恩还在远处守着。他低声道：「明日村会。叶汤得立。立住了，才走得向公审那日。」

沈家豁口里咳了一声，咳得轻，像在作证：日子定了，芽儿娘还能喘，村会就能辨。韩泥不开门，只把藤条再绕紧一圈——实了，才撑得到辨诬那日。秤杆弯着没断，铺还在，汤就不凉。

瓮还没醒，他不喊醒。醒不在这一更。他合上眼，手按在坛沿上：明日巳时，祠堂村会，叶汤得立给全村看。活路在，坛就在；帖贴着，规矩也就钉住了。
