# 第九十二章 废窑再试

雨歇了。药山湿，湿里还留一点昨夜窑温的余味。

韩泥第二次上山，筐里仍是药渣，灰盖面，像常倒。这一回他不急火。急火焦渣，焦渣只苦，不是路。铁无言昨夜一句说得死死的：「再试，先温炉，后投芯。芯多一分焦，少一分不发。」

「记着。」韩泥当时应。现在入窑，先用掌温炉壁。壁凉退半寸，退得像坛腹应温。温够了，才投青芯一撮、褐渣半把、灵泉三滴——滴少，少才试得出分寸。

火起，不猛不弱，像呼吸。渣里白气起一线，起而复散。散前他捉住培元路第二丝：细，正，贴着鼻下不散。

正香让心里热一下。热一下就立刻压住——喜了，手飘；飘了，赖福就来。窑深，声出不去，可他仍把呼吸按平。

第二炉他故意换投序：先褐后青。序错，香杂；序对，香单。单香落地，像铁无言验渣时那个「正」字。他把对的序用指甲在袖袋边轻轻刻一下，刻完又抹平，抹得像没字。

管事弟子路过问：「韩泥，倒渣倒两趟？」

「渣多。」韩泥说，「筐沉，分两次。」

「分两次？」那人皱眉。

「分两次。」韩泥重复，「一次倒不完，压栏口。」

管事记一笔，笔丑，却真：「分两次。」真记成常活，常活最好藏事。

第三炉火匀一寸。再多，旧窑壁怕裂。裂也无所谓——旧窑本来就没人争。炉开，无丹，只有一团凝灰，灰心一点白，白得像未成之珠。

「未成也好。」韩泥低声，「成路比成丹急。」

他把凝灰收进袖袋，袋口扎死。下山遇白织月，袖风带着香，声冷：「丑杂役，药山是你扫的？」

「倒渣。」韩泥不抬眼，「手稳，不脏内门路。」

「末席也配碰药山？」

「配倒渣。」韩泥只这一句。

她嗤远。嗤远了，窑才仍是他的。回棚赖福又秤筐：「仍少二斤。」

「看错了。」韩泥说，「渣在筐底，底沉。」

赖福掀底，底里灰沉，青芯仍无。青芯在袖，袖在怀里，怀里还有三年约玉牌贴着凉。

刘婆端粥：「又上山？」

「倒渣。」韩泥说，「手稳，饭稳。」

「饭稳，恩不断。」刘婆哼，「别偷火。偷了连坐丙九，我兜粥给你也白兜。」

「不偷火。」韩泥道，「试的是废灰里的温。」

「温不温你自己清楚。」刘婆白他，「清楚就别叫我问第二遍。」

邻床悄声：「你真只倒渣？」

「倒渣里试火。」韩泥声平，「火试过了，才藏得住后头的事。」

「后头啥事？」

「你睡。」韩泥说，「问多了记过罚到你头上。」

少年闭嘴。闭嘴像丙九开始学会忍。

丑时廊下静半刻。粟壳香淡，香落齐，掌纹那路又温一线。温不该喜，该稳——稳是再试的回响。赖福远处嗓子尖：「丑杂役，我盯着你！」

「在。」韩泥应。

在就不滚。不滚，第二回就不白跑。他按坛低声：「再试过了。火匀半寸，路熟半寸。」

坛不应字，只温。温像泥岗丑时那口白气，又像爹搭沿时的手劲。

卯时清栏，栏腥照旧。他铲得稳，稳里把褐渣半把又拢进袖——半把够下一回备料。铁无言在渣场分三堆，堆边立牌：青、褐、废。韩泥只取废堆边的灰。

「取废？」旁人笑。

「废里也有命。」韩泥说，「命脏一点，我认得。」

铁无言脚停半息，声更低：「控火半寸够了。下一回别急炼，先备料。料散三处，像杂役藏粮。」

「散三处。」韩泥记。

「序别写纸上。」铁无言补，「写了，赖扒皮翻得到。」

「不写纸。」韩泥道，「写在掌茧里。」

掌茧裂处疼，疼不露脸。刘婆缝顶针路过，针亮：「你掌又裂？」

「编筐。」韩泥说，「藤硬。」

「硬好。」刘婆哼，「硬手，饭稳。软手的人，汤都端不稳。」

汤字一落，他心里那碗叶汤又烫半分。烫着提醒他：再试不是为炫技，是为还。还以前，得先把备料藏妥。

雨线斜进半塌窑口的记忆还粘着。他第二回分两次投——先青后褐，中间掌按炉壁半息。半息像控火，也像把匿炼的序先预演一遍。预演不成丹，成路。路熟半寸，够接备料那一夜。

更鼓尽，药山湿气不散。他摸怀木牌，牌凉贴胸：「下一回备。备齐了再炼。炼不叫人知，只叫门缝那边夜里亮一点。」

坡下方向风硬，硬里像有人咳。咳轻，他不近。近了像连累。连累不配「别来丑」那三个字。他只把「备」字按进坛温里，按实了，手就不抖。

同棚有人打赌：「再焦，管事记过，他还有胆上？」

韩泥走过，只丢一句：「倒渣的人有胆，骂人的人没筐。」

笑声噎住。噎住也好。噎住了，他才能把第二回的火路听进骨头里：匀、稳、不喜。不喜，才还得上往后那一粒。

夜里他把凝灰摊开看了又看。灰心那点白，像珠未圆，却已经有了形。形在，路就在。他把灰重新扎进袖袋最里，袋口再勒一匝，勒得像把秘密勒死。

沈枯芽白天又来探头：「韩叔，你袖里咋总沉？」

「沉才能压风。」韩泥说，「风硬的日子，轻东西先飞。」

芽儿似懂非懂，小声道：「叶姐又咳了两声。不重，就是夜里怕黑。」

「知道了。」韩泥应，「你别乱传。传了害她。」

「我不传。」芽儿抱饼跑了。韩泥看着他背影，把「怕黑」两字按进心里——按实了，下一回备料才知道该给谁留光。

廊外更鼓一下一下敲。他编筐末了，藤条勒指，指疼，疼正好提醒投序：先温、后芯、再褐，中间一息。一息记死了，雨夜炼真丹时，手才不会飘。

赖福半夜又过廊，脚步停又走。停是嗅，走是没嗅到。没嗅到最好——窑里那半寸匀火，只该进骨头，不该进旁人鼻子。

韩泥枕臂低声：「再试成了。下一程备料，备齐了才炼。炼给门缝那边，不给嘴上的人。」

第二天兽栏值班，管事点名：「韩泥，倒渣次数过多，改去清外围沟。沟臭，配你。」

「清沟。」韩泥应。

沟里果然臭。臭里却有一点焦香，像废窑带出来的灰。他用鼻下分：沟臭是腐，窑灰是浮。分清了，手上铲子不抖。不抖，才像常活。

清沟到一半，赖福路过踢石子：「丑杂役，沟清不完，别想再上山。」

「清完再上。」韩泥说，「上山倒渣。倒渣是条子上的活。」

「条子我能撕。」

「撕了管事再开。」韩泥应，「开了还是倒。」

赖福脸青一阵，骂「活腻了」走开。韩泥把沟泥铲稳，稳里想：再试那半寸火，已经写进掌茧。掌茧疼时提醒他——备料要散，散三处，处不连，连则翻。

午后铁无言扔来半截废炉条：「热传导你会不会？」

「会一点。」韩泥接，「掌温壁，壁温芯。」

「会一点就够。」铁无言道，「够到匿炼。匿炼别求亮。亮了，全山都是你仇人。」

「不求亮。」韩泥说，「只求匀。」

夜里雨又斜。斜进棚顶漏处，滴在他席边。他挪席半寸，护住坛沿。沿净，料才藏得住。藏得住，下一步才能把手伸进「备」字里。备字重，重过试炉——试炉败了还能再试；备露了，叶家线断，断了汤就还不成。

邻床梦话里喊「挂牌换钱」。韩泥听着，不接。换钱远，还人近。近的先做。做完了，耳里再听坊市风也不迟。风未起，手先稳；手稳，凝灰那点白才不白揣着。

丑时他起身喝冷水。水凉，凉进牙，牙紧，紧着把「备三处」默数一遍：袖、筐、草。数完躺下，闭眼前低声：「再试过了。下一回，为还而备。」瓮腹应温，像盖了章。

坛温一线，像应了「备」字。应完仍不叫，像替他藏着这口窑。窑藏住了，还叶的路才真正起头——起头就够，够他把明天的干草坑挖浅一点，挖得像藏粮。
