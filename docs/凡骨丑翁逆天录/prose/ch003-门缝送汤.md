# 第三章 门缝送汤

天又冷了。

泥岗村入秋，风硬，先从村西漏屋尝寒。风从门缝灌进来，灌得藤条更僵，绕两圈又松。韩泥早起编筐，指尖僵，骂藤没用，骂完了还得绕。编筐手不能抖——抖了，筐松，药散，活路断。

张麻婆门外路过，瞥一眼：「编慢些，手别裂。」

「裂不了。」

心口还空着。不是没人恩他，是恩还没记实。韩泥不知该怎么记，只晓得恩不能乱记——记了就得还。还，得等手上有东西；手上有东西，才还得不寒酸。

晌午前，门缝才透进一点光。有人影顿在门口，没敲也没喊。韩泥抬头，见一双布鞋，洗得发白，鞋尖沾泥，站得稳。

「叶丫头。」

他开口，声音哑。叶青禾应一声。村里都说她命苦，男人走得早，一个人撑药铺，清瘦耐看，不张扬。她与韩泥隔门站着——晦气翁不配让人进门，她也不进，怕闲话。闲话在泥岗村比风快，风过无痕，闲话能留很久。

「韩泥，开门缝就行，我不进。」

叶青禾顿一息，又道：「汤趁热，莫挡门口风。」

韩泥把门缝推开一指：「一指够。」

碗递进来，粗瓷沿口缺了一角。汤热气还在，香味淡，是姜。热气贴着门缝往外散了一线，又敛进碗里。

「趁热。」叶青禾说。

就两字，没有可怜，没有嫌弃，没有你怎么活成这样的叹息。韩泥最怕叹息——叹息比骂还轻贱。骂是明的，叹息是暗的，暗的扎心。

他伸手接碗，指尖碰到碗底，一物硌手——薄片切得细，藏在汤下，姜。他没抬头：「够了。」

叶青禾在门外道：「天冷，门缝大，别冻着。」

「冻不死。」

「我知道，所以我送热的。姜别扔，晒干还能煮一回。暖着喝，凉了别硬咽。」

韩泥捧着碗，没让叶进门。门缝这一指宽，够递碗，够记恩，不够容闲话。

「够暖。不硬咽。」

叶青禾声更轻：「记得住，比谢字要紧。」

韩泥没接谢字。巷口周嫂咳一声，没近，只甩一句：「叶丫头，药铺还开不开？」

叶青禾应：「开。秤在，铺就在。」

风硬，汤热，热抵风半寸。他捧碗站门内喝得很慢——烫，辣一点，胃暖。热从喉落到胃，落进心里那道空处。空处填了一指，不多，够记。

汤里无肉，只有姜辛顶肉。姜切得薄，薄而匀，匀才入味。走岔的恩是施舍，不走岔的是实。碗沿烫手，他换了个握法，指节贴着粗瓷——烫着才记得牢。汤尽，姜片挑出窗台晾干。穷人家姜用两回。

叶青禾脚步远去，布鞋踩泥，轻而稳，把趁热二字留在门缝里。韩泥不会写字，也不把恩划在纸上。这一碗汤是头一笔恩，深，烙在心里。

他低声道：「汤还是热的。」

不是对她说，是说给自己——记着，有一天还。热汤能凉，恩不能凉。凉了，人就成了嚼舌根的嘴，轻，薄，一吹就散。

傍晚，村童远处嚷晦气翁。韩泥没出门，坐到坛边。腌菜坛白日凉，入夜却温了一指，温应心口那碗汤，不应谣。

赵绣春门外尖声：「叶丫头倒是心善，善到晦气翁门里去了？」

叶秤落一声，脆而短。赵绣春哼一声噤了——尖不过那声稳。

卖盐老汉停一息：「叶丫头心善，善在实。实，不是邪。」

韩泥不谢。谢轻，他记。老汉走了，脚步沉——公道不常来，来一次，够撑明日。

叶青禾为何送汤？不为名声——要名声不会沾晦气翁；不为可怜——送法不像可怜。人该送时，不图回报，图心里过得去。韩泥五十八年，少见这种过得去。

她男人走后，铺子独自开着。开着就懂：心实的人，看不得门缝里灌风。灌风灌不死人，灌久了，人就没劲还手。她还手的方式是送汤——不进门，不张扬，就留一碗热的在门槛。

还的时候，不能寒酸，不能敷衍，不能学村头那些算了算了。碗底缺沿不碍用，穷人家碗缺沿，汤照样烫。

童谣丫远处嚷一句，嚷完被娘捂嘴。捂了，够他今夜安一息。卖盐老汉远处又补一句：「实人送实汤，别叫谣搅浑。」韩泥不接，只把窗台姜片按平。外头风硬，门缝灌进来，他拿空碗堵住缝——碗空，还能挡半寸风。

姜片按平了，辛气还在。还在，就还能煮第二回。穷人家一物两用，恩也两用：记着，暖着，还的时候才不空。

夜更深，井台桶绳吱一声。布鞋踩泥，轻而稳，没停——叶丫头又去铺里守秤。叶铺灯还亮一线。韩泥举目西望，药铺灯稳：明日去井台，还得听那声秤，也得见那双洗白的布鞋。

他按坛沿，又按心口。坛壁一线水汽一闪又敛。坛不拒热，拒的是脏水。坛凉，心热，人夹在中间，五十八岁，仍活着。

他轻声道：「这恩，我记了。」

心里那一笔，丑，却真。他洗尽粗瓷碗搁回门墩，夜雨敲瓦，粗瓷碰门墩，叮一声。

叶家药铺灯亮到夜深，秤声又起，脆稳，钉住铺子没关，也钉住心口那笔恩。张麻婆骂童谣丫：「少学周嫂那嘴！」骂完塞门缝半块饼：「吃。编筐手不能抖。」

沈枯芽隔门问：「韩爷爷，汤烫不烫？」

「烫。辣好。趁热，才记得住。」

叶青禾在巷那头应：「记得住就好。」

韩泥没接。接轻了是讨。他藤条绕膝，勒出闷响，心口那笔恩更实。实了，手就不抖；手不抖，才还得上那一碗汤。

门外赵绣春又骂一句，骂给风听。风过，叶铺秤声又起，谣就矮了半寸。他低声又道：「趁热。记着。」

风从门缝过，带一点姜辛，辛得短，短得够记一夜。合眼，碗要烫手，姜要还在——还在，叶丫头才晓得晦气翁没忘。忘不得，也还不得急。急了就还寒酸。

张麻婆折返：「叶丫头心实，你别多想。」

「不多想。想还。」

「还什么还，先把筐编完。」

「筐编完，人还在，才还得上。」

他藤条绕膝，勒出闷响。恩在，明日打水的手就不会抖。抖了，叶汤就还不了热。风硬了一点，屋里更冷，可心口还暖着——暖着就够活过今夜。

门外暮色沉下来，远处赵绣春家亮起灯，亮得张扬。他在像瓮底似的屋里，不抬头，只低头看窗台姜片还在不在。辛在，叶丫头才晓得他没忘。

合眼，手仍按坛沿。沿在，根就不散。瓮未醒，他不喊醒。恩在，手就不抖；手不抖，还的时候才不寒酸。明日去井台，还得听叶家那声秤——秤在，汤就不凉。

窗台姜片晾了一夜，边沿微卷，辛味却没走。他用指腹按了按，按平了才踏实。踏实不是因为姜贵，是因为有人肯在闲话堆里递一碗热的过来——递完还不进门，把门缝留给体面，把热留给胃。

村里若问他谢不谢，他答不上谢字。谢轻，像风；记重，像石。石头沉在心口，挪不走，也砸不碎。砸不碎，将来才能搬得动还字。

他洗过的粗瓷碗仍搁门墩，缺角对着屋外。缺角不丢人，丢人的是碗空了还假客气。他不作假。作假对不起烫，也对不起那句趁热。

巷那头叶铺的灯终于弱了一线，却没灭。没灭就好——灭了像退，亮着像人还站着。韩泥隔着雨听了一息秤砣，没有声，也当听见了。听见了，心口那笔账就又盖了一章红泥。

汤还是热的，要还。
