# 第一百四十章 积势暗记

积势记，记在一册薄册里。

册是管事所发，发得像例行。例行不要紧，例行里藏路——路通向药峰、坊市、大比预选、层数登记。登记齐了，首簇才像收束，不像散沙。

韩泥一页页翻，独眼平。页上没有恩字，恩在他心里：刘婆已还，叶汤在前，老耿要谢。

铁无言立门外，声低：「积势够了，别躁。」

「不躁。」韩泥说，「躁，手飘；飘了，谢不出。」

「谢不出就白熬这些年？」

「熬不白。」韩泥道，「熬在手里。手里有散，才配送人。」

赖福最后查席，查完哼一声：「丑杂役，滚远点。」

韩泥不滚。不滚是约；约在玉牌，不在赖福舌头上。

「听见没有？」赖福嗓尖，「滚！」

「席上有名。」韩泥说，「名在，人就在。在，就不滚。」

赖福甩袖，骂声远了。远不过坛腹那一线温。温在，骂就短。

白织月远廊过，袖风冷，脚步不停。不停，像默认——默认丑骨还在走，走就还有戏看。

有弟子踮脚问：「白师姐，他册都记齐了，真能谢谁？」

「谢谁是他的债。」白织月声淡，「债还不清，上台也软。」

刘婆补扣，过来拍他衣襟：「册都齐了，别空肚。」

「不空。」韩泥说，「空肚，手飘。」

「就会飘。」刘婆塞来一碗粥，仍多半勺，「喝完再翻册。翻册不当饭。」

「嗯。」他接碗，「记着。」

沈枯芽口信到，仍无字，只有半片枯叶，叶背细针脚画了两道痕，像脉稳的记号。旁边人传话：「叶姐脉稳些。」

稳不要紧，稳像培元余温——温在脉，不在嘴。嘴一响，恩就吵。

韩泥把枯叶按进袖：「回去说，稳就好。别来外门送口信。」

「芽儿不敢多说。」传话的杂役捂嘴，「说多了怕惹白姑娘。」

「怕就对。」韩泥道，「怕才活得久。」

丑时练鼻下静。粟壳香淡，香落齐，这一簇积势记进心里。

他望坛，低声道：「谢老耿，经叶家。记着。」

坛腹温，贴掌，像应「势」——势够了，恩才送得出；送得出，才配称能还才送。

更鼓将尽，天边白一线。白里他按心口：「汤还是热的。」

六字落地，像路标——标向培元散，标向叶汤，标向雨夜以后的路。路远，脚步得稳。

大比帖在怀，怀沉。沉的是债序，不是惧。惧在，手也要稳；稳了，第二部才像第二部，不像还停在杂役棚口头。

外门弟子渐散，散声远。远不过坛温。温在，就不滚。

管事弟子收册，声平：「积势记齐。下一簇，培元散谢老耿，经叶家。」

场边有人咋舌：「谢老耿？丑骨还能送散？」

「送不送看丹。」旁人回，「丹正，谁收谁磕头；丹歪，送也打脸。」

韩泥不辩。辩像急；急了手飘。他只把排序按实：谢老耿、筑基丹、雨夜——三笔在远，一笔刘婆已还。

铁无言又靠近半步：「听清了？下一簇。」

「听清了。」韩泥说，「听清了才不越界。越界，手飘。」

「不越就好。」铁无言抱臂，「兄弟线站着，不替你炼丹。丹你自己正。」

「正。」韩泥应，「正了才送。送了才像还。」

药峰、坊市、栏边、坛角——四处在，手才稳；手稳，帖才压得住。压得住，钱戾衡那道风才接在台上，不接在巷骂里。

管事扬声复述一遍：「下一簇，培元散谢老耿，经叶家。有恩先还，无恩别蹭。」

「经叶家。」韩泥把三字咬实。叶家在前，汤就在前；汤在前，人就不能飘到仇里去撒野。

赖福最后骂一句：「谢来谢去，谢得掉丑骨吗？」

「谢的不是骨。」韩泥说，「谢的是人。人救过命，散就该送。」

赖福噎住。噎住也好，省得再喷脏。

韩泥回屋，把薄册合上。合上不是完，是首簇收束——收束不抢后面的谢、不抢筑基、不抢省亲。抢了，恩送不出；送不出，就不配谈还。

他按坛，掌温：「势够了。下一粒，给该给的人。」

坛腹温一线，像应「送」。送在后头，锣还没响，叶铺那碗热却先在心里烫了一下。

管事散场前把他叫住，把薄册封角按实：「册你可留抄页。抄页丢了，下一簇说不清。」

「不丢。」韩泥把抄页贴胸，「丢了像忘恩。忘恩，散炼不正。」

「你倒会归到丹上。」管事哑然失笑，「去吧。炼散前，把炉借条写明白。」

「写。」韩泥应，「写明白，才不给人卡脖子。」

回棚路上，同批少年追上来打听：「韩叔，真要谢一个老头？谢完有用吗？」

「有用。」韩泥说，「用处不在旁人眼，在我心里。心里通，气才通。」

「心通气通……」少年摸下巴，「这话像庙里和尚。」

「和尚不炼渣。」韩泥脚步不停，「我炼。炼完，路才像路。」

夜里他摊开抄页，一笔笔勾：药峰活满；坊市续留三日无事；守坛夜沿净；风声紧未接骂；层十五实登。勾完，笔尖停在空白处——空白留给培元散，留给叶家院门，留给老耿那一声或许轻、却重的咳。

铁无言推门缝看一眼：「勾这么工整，怕忘？」

「怕乱。」韩泥说，「乱了，送错人。送错人，比不送还脏。」

「脏倒不像你。」铁无言丢来一块擦炉布，「布先收着。散炉烫，别怂。」

「不怂。」韩泥按布，「怂的时候在台上才露。台上之前，先把散炼正。」

刘婆夜深又咳着送来棉线：「扣又松了。松着出去，像没人疼。」

「有人疼。」韩泥低头由她缝，「疼在还里。还到了，才是真疼。」

「贫死了。」刘婆线头一咬，「明天灶上给你留粥。炼散别空肚，空肚炸炉。」

「不空。」他笑了笑，笑很浅，「粥在，手在，散就在。」

笑完他收线，掌再贴坛。坛温像催：催不是躁，是提醒——首簇完了，刃要出鞘，鞘里那粒必须正。

临出门前，管事又派人送来一张炉借条草稿：「名字、时辰、药材去向，三项别空。空一项，炉房敢卡你。」

韩泥当场补字，字丑，行距却清：「去向——培元散一剂，谢老耿，经叶家转。」

送条的弟子愣住：「写这么白？不怕人做文章？」

「做文章让他做。」韩泥按印，「藏着送，才像贼。贼的恩，我不认。」

「……您挺横。」

「横给债看。」韩泥把草稿叠好，「债看明白了，人就少绕弯。」

药峰下坡处，白织月恰从上层梯下来，二人错身。她目光扫过他怀中抄页，声冷：「积势记齐，别以为能换骨。」

「不换骨。」韩泥不停步，「换债。债清一笔，骨还是丑骨，路更好走。」

「油。」她袖风一扫而过，「台上见真章。」

「台上见。」韩泥应得短，短得像钉子。

回到坛边，他把借条、抄页、玉牌三样并排放好。三样凉热不同，却都压着同一个方向——方向上写着老耿的名字，名字旁边留着叶家院门的空位。

邻床探头：「韩叔，你真写经叶家？不怕村里旧谣再冒泡？」

「怕。」韩泥说，「怕也走。走恩人的门，比走仇人的嘴干净。」

「干净……」少年咂舌，「你们老头说话总绕。」

「不绕。」韩泥吹灭豆灯，「绕的人，散会炼歪。歪了，莫说谢，连自己都对不起。」

豆灯灭后，坛温更显。他在暗里按心口，把「汤还是热的」又默念一遍。念完，手心像握着那只旧碗——碗沿毛刺还在，刺着他别骄、别躁、别把首簇收成吹嘘。

天边更白。白里像有人已经支起炉火，等他炼那颗该送出的培元散——散一成，经叶家，谢老耿，这一簇才真正揭开下一页。页角微微翘着，翘得人心口发紧：紧不是怕，是终于要动那笔旧恩了。旧恩要还，明日炉前就不准手抖。
