# 第五十六章 双妖帅

金丹坠田，泥还没沉，妖雾已起。

灵田东麓，腥风卷过未收的稻茬，青鳞妖君破雾而出——鳞青，眼绿，爪如镰。非野生妖帅，是赤焰驱策的刀，刀尖直指青岚喉。

厉无殇立高坡，未近，笑里恨与快绞在一起：「韩铁山，一个金丹都守不住，还要本少谷主擦屁股？」

韩铁山面色铁青，旗挥，妖骑再冲，冲得像用妖帅的怒，盖住自己刚才的骇。

霍镇山盾阵已裂一角，盾面纹路如老树皮，仍骂：「畜生！往莫长老那边去啊——」

莫长春不躲。

杖剑再起，这一次杖与剑合，合得像还礼还到刃上。他不抢柳青鸢的宗主功，不抢霍镇山的镇妖司名，只把剑意送进妖君七寸一线，送进驱策符纹的弱处——秦氏简上写过，顾小满暗线证实过，茶饼换的情报，今夜兑命。

剑气余波荡开，妖雾碎，青鳞妖君嘶吼，吼到一半，断。

妖核入手，温凉，像一颗未炼的丹，像一颗将来要送的礼。

全场再哗，哗势比斩金丹更甚——妖帅也斩？

弟子声音叠浪：「两连斩！」

「莫长老……还是九月？」

林挽月立人群后，唇白，白得像终于看懂：慕错人，还能改；眼盲，改不了。

厉无殇面色骤变，骂声更厉：「撤！」

赤焰旗转，马蹄乱。韩铁山抱伤属退，退前狠狠瞪莫长春，瞪得像记一笔必还的血账——却不敢再上前半步。

魏无涯在远处梁上，扇合，悄然退，像看一枚子落盘，落满，仍未终局。

莫长春握妖核，不炫，收入袖中。他望厉无殇退处，低声，风送过去，像送一句旧约：

「够活到你后悔——又近一步。」

丹堂方向，地火阵鸣收半档，赵玄机仍立门洞，血透甲，人如钉。苏念慈在阵眼，单膝微屈，灵力见底，却笑，笑很轻，像三月前整理衣襟那一下。

莫长春杖点地，往丹堂走，不快，像老人散步，像把全场哗声留在身后。

灵田边，弟子扶伤退，退时仍回头看莫长春背影，像看一块旧符，符旧，意满。

周德海在后阵棚下，令旗举起又落，落得像侧门阵闸仍不开——顾小满冲至，浑身泥血，嗓子哑：「周长老！赤焰已退，开闸包抄残敌——」

周德海淡道：「后阵不稳，妄动者军法从事。」

顾小满咬牙，转身再奔，奔回灵田，像把「侧门不开」四个字奔成一把刀，递给宗主，递给每一个肯听的人。

柳青鸢收剑，剑意未散，望向师兄，只一字：「够。」

莫长春点头：「够一夜。不够一生。礼，还没送完。」

簿无大字，袖却烫，像提醒：小因果今夜兑尽，大因果还在——周德海侧门，厉无殇恨意，天试三月。

他念，像章末钩，也像给读者留的延迟层：

「礼还没送完。妖核，也算礼的一种。」

风大，妖雾在东方再起一线，像下一章的牙，已露。

大因果还在周德海犹豫里，像侧门，像盟会——双妖帅只是小斩，战后抚伤才是大还礼的前奏。

剑气余波荡开，妖雾碎，青鳞妖君嘶吼，吼到一半，断。妖核入手，温凉，像一颗未炼的丹。弟子声音叠浪：「两连斩！」「莫长老……还是九月？」林挽月立人群后，唇白，白得像终于看懂：慕错人，还能改。

厉无殇面色骤变，骂声更厉：「撤！」韩铁山抱伤属退，退前狠狠瞪莫长春，却不敢再上前半步。魏无涯在远处梁上，扇合，悄然退，像看一枚子落盘，落满，仍未终局。

周德海在后阵棚下，令旗举起又落。顾小满冲至，浑身泥血，嗓子哑：「周长老！赤焰已退，开闸包抄残敌——」周德海淡道：「后阵不稳，妄动者军法从事。」柳青鸢收剑，望向师兄，只一字：「够。」

莫长春不躲。杖剑再起，这一次杖与剑合，合得像还礼还到刃上。他不抢柳青鸢的宗主功，不抢霍镇山的镇妖司名，只把剑意送进妖君七寸一线，送进驱策符纹的弱处——秦氏简上写过，顾小满暗线证实过，茶饼换的情报，今夜兑命。

莫长春握妖核，不炫，收入袖中。他望厉无殇退处，低声，风送过去，像送一句旧约：「够活到你后悔——又近一步。」莫长春杖点地，往丹堂走，不快，像老人散步，像把全场哗声留在身后。簿无大字，袖却烫，像提醒：小因果今夜兑尽，大因果还在。

围观弟子在灵田边踮脚，踮得像等两连斩落账——斩金丹之后，妖帅也断，哗势更甚。有人低声：「茶饼换的情报，今夜兑命。」另一人答：「兑的是弱处，不是名。」
