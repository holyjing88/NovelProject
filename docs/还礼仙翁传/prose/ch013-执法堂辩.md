# 第十三章 执法堂辩

执法堂正殿，晨光斜切，铁牌悬壁，冷光如尺。

柳青鸢未至，周德海先至。他抚须笑，笑里拱火：「听说杂役要告执事？青岚门多少年没这般热闹了。」

顾小满立殿心，背挺直，像半块灵石撑着她。

赵玄机在侧，礼数足，眼神仍冷：「顾师妹，话出口，要有凭。」

「凭在此。」顾小满道，「我亲眼见赵执事划册，将莫长老例份改入己名。」

赵玄机一笑：「眼见的，未必是真。库房人多手杂，或许看岔。」

「没看岔。」顾小满不退，「赵执事还说，将死之人，吃了也是浪费。」

殿内一寂。

周德海插话，声和气缓：「赵执事也是为宗门惜丹。莫长老年事已高，培元丹给他，不如给年轻弟子——这话糙，理不糙吧？」

莫长春拄杖入殿，灰袍素，杖点地砖，声轻却清：「理糙，心更糙。」

周德海揖手：「莫长老也来了。琐事也劳您亲自走一趟，看来九月之后，精神倒好。」

「精神好，才好记账。」莫长春落座，不看周，只看赵，「赵执事，丹在谁手里？」

赵玄机下颌线一紧：「若在我处，也是暂代保管。」

「保管给谁？」

「给……还能为宗门出力的人。」

莫长春点头，像认了一句实话，却补刀：「那老夫出力除妖、杖扫青狼，算不算出力？」

殿外弟子窃语骤起，被铁牌光一压，又噤。

柳青鸢入殿，青衫带风，剑未佩，威先在。她目光扫过赵玄机袖侧，淡声：「赵执事，例份丹何在？」

赵玄机沉默两息，从袖中取出瓷瓶，放案上，瓶底磕木，声脆，像认罪。

柳青鸢不看瓶，只看人：「为何私扣？」

赵玄机唇白，硬撑：「弟子以为……长老用不上。」

「用不上，由司库回库，不由执事入囊。」柳青鸢道，「执法堂铁牌，是正法，不是私法。」

她转向莫长春，语气略缓：「师兄，例份还你。赵执事记过一次，扣月俸，当众罚抄门规三遍。」

赵玄机跪，膝磕地砖，响得实。他抬头，眼里有妒，也有什么东西被敲裂了一缝：「弟子……认罚。」

莫长春却未笑，只一句，像给全殿定调：

「丹在谁手里不要紧，心在谁那边要紧。」

他指顾小满：「心在杂役这边，堂上便不能当没看见。」

周德海笑容微僵，揖手退半步，像把拱出去的火收回来一半。

罚抄毕，赵玄机立执法堂外廊，手抄门规，墨污指节。

林挽月送水，不敢近，只放盏即走。赵玄机看她，忽然问：「你觉得莫长老在演？」

林挽月一颤：「赵师兄……门规写三遍，别说话。」

赵玄机低笑，笑里苦：「他一句『心在谁那边』，比门规还难抄。」

殿内，柳青鸢留莫长春片刻。

「师兄，」她低声，「赵玄机妒你，不是一日。今日罚了，他更恨，也可能更醒。」

「醒就好。」莫长春道，「恨不怕，怕的是心全在私囊里。」

柳青鸢欲言又止，终只道：「孙福那边，也在动。」

莫长春抬眼：「让他动。动大了，才好裁。」

夜，簿上浮字：

**心在何处，堂上自明。**

莫长春合簿，望外门方向，像望一条将起谣言的河。

他念：「下一份，是告状，还是栽赃？」

杖中竹纹一游，无答，像堂上铁光还挂在眼里。

大因果还在远处，像妖潮，像赤焰——堂上辩只是小定场，孙福告状才是大还礼的前奏。

罚抄毕，赵玄机立执法堂外廊，手抄门规，墨污指节。林挽月送水，不敢近，只放盏即走。赵玄机看她，忽然问：「你觉得莫长老在演？」林挽月一颤：「赵师兄……门规写三遍，别说话。」

殿内，柳青鸢留莫长春片刻：「师兄，赵玄机妒你，不是一日。今日罚了，他更恨，也可能更醒。」「醒就好。」莫长春道，「恨不怕，怕的是心全在私囊里。」

周德海插话，声和气缓：「赵执事也是为宗门惜丹。这话糙，理不糙吧？」莫长春拄杖入殿，声轻却清：「理糙，心更糙。」殿外弟子窃语骤起，被铁牌光一压，又噤。柳青鸢入殿，青衫带风，剑未佩，威先在。

赵玄机跪，膝磕地砖，响得实。他抬头，眼里有妒，也有什么东西被敲裂了一缝：「弟子……认罚。」莫长春却未笑，只一句，像给全殿定调：「丹在谁手里不要紧，心在谁那边要紧。」

围观弟子在殿外踮脚，踮得像等赵执事再辩——辩没来，来的是瓶底磕木那一声脆，脆得像认罪。有人低声：「杂役告执事，青岚多少年没这般热闹？」另一人答：「热闹在堂上，不在嘴上。」
