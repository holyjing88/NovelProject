# 第一百三十一章 帖在怀

复测散后第三日，山门晨鼓迟半刻。

鼓迟不要紧，帖要在——帖在，路才像第二部，不像一部尾韵。韩泥立丙九队末，怀帖，帖厚，厚得像刘婆新衣那包棉。棉旧，恩新；帖新，债序又落一笔。

执事验牌，牌歪，字丑：「韩泥，炼气十三，夺魁。帖，收好。」

「收好。」韩泥应声。应声短，短才像常活。

验息册上，又添一行：炼气十四，登记。十四，不是喜，是实；实了，大比路才踩得稳。

钱戾衡名在帖背，背字淡，淡像眼风。风过，韩泥不抬眼——眼要留来数息，留来守坛。

赖福廊下过，脚步乱，乱像嚼不出晦气：「丑杂役也接帖？帖，太阳西边出！」

韩泥不辩。辩，费舌；舌要留来听药峰派活，留来守丑丹招牌。

铁无言立阶下三步，声低：「帖在怀，手别飘。」

「不飘。」韩泥说，「飘了，刘婆那笔白还。」

铁无言没再多话。话在108那日并肩，并肩续到今日——续不在嘴，在站。站侧，像还半分仇谣。

白织月远廊过，袖风冷：「帖是预选，预选仍是运气。」

韩泥捧帖，帖稳：「运气大比见。见在台，不在舌。」

沈枯芽口信无字，只有半片枯叶——叶黄，像提醒：叶汤还在前，省亲还在远。远不要紧，刘婆已还；还了一笔，帖才配在怀。

秦霜木鱼轻敲：「帖至，心别邪。邪了，鼻下香飘。」

韩泥不答，答在心里。心里排序：帖、药峰、坊市、叶汤——四笔分列，不混。

管事弟子扬声：「夺魁者，明日药峰派活。丑丹师，别躲。」

韩泥只答：「不躲。」躲，像邪；邪了，管事记簿，记「丙九韩泥，躲活」，记躲，路断。

回杂役屋，他按坛。坛腹温一线，贴掌，像应「帖」字。应，不应喜；喜了，手飘。

丑时，练鼻下静。粟壳香淡，香一字不落。落齐，十四层这一日记进心里。

邻床少年悄问：「你真接帖了？」「接了。」「接了，大比不怕？」

韩泥不答，答在玉牌——牌凉，烫着三年约。约说站钱戾衡台前，才不滚。

更鼓沉，他望坛，独眼平：「帖在怀。记着。」

记着，是丑翁本事；本事在忍，在还，在叶汤那碗还没还尽的烫。

外门风硬，帖角掀半分，掀出「言伯钧」三字影。影在字后，像仇在路后。路后不要紧，药峰在前。

他低声对坛：「明日药峰。手在，路在。」

坛不应字，只温。温像应「十四」——层在，帖在，就不滚。

管事夜巡记簿，记「丙九韩泥，帖在怀」，记细，是小胜。胜在忍，不在辩；辩，费舌。

柱上条角白一线，三年约在怀。怀凉，掌温——温在还，不在众目。众目笑，笑在耳，不在心。

编筐棚借半刻，藤刺扎掌，偏手，刺浅些。编筐钱买米，米在，恩才还得上。

药渣场角风硬，他把剩渣摊匀，匀像秤平——秤平，手才静；静了，帖才压得住。

同棚杂役赌大比：「丑骨上台，太阳西边出。」韩泥不接茬，接在玉牌上——牌在，名在，路在。

老耿挑水路过，水洒稳，声哑：「帖在，别拜暖。拜暖，路断。」韩泥只答：「不拜。坛在，人在。」

叶青禾门缝灯未亮。未亮，像避；脉里培元仍温——温是95；散是110，两笔不混。

他低声对坛：「十四登记了。记着。下一程，药峰。散队后，他过外堂阶下，分拣半刻，指稳——指稳，帖才像活，不像摆设。坛边席角，十四在。帖在。人不滚。。」

邻床翻身，眯眼半息：「大比远不远？」「远。」韩泥说，「远不要紧，手在。」

管事记簿再翻，翻见「丑丹师」三字——三字丑，真在登记。

他摸怀帖，帖厚，厚像债序第二笔——笔是帖，笔还在走。

帖在怀这一日，外门执事堂人声稠，稠里藏眼——眼在帖，眼在层，眼在丑骨能不能上大比台。韩泥不近堂心，近心，像显摆；显摆，赖扒皮来。他立队末，怀帖，帖厚，厚得像130刘婆那包棉还温在袖里——温是恩，帖是债；债序在，步就不飘。验息册翻到新页，页黄，字丑，丑而真：炼气十四，登记。

登记不是贺，是钉——钉住大比路，钉住三年约，钉住钱戾衡那眼风。眼风在帖背，在廊下，在坊尾；风在，手更要稳。稳了，132药峰才像活，不像罚。罚不要紧，活才像积势——积势不在快，在忍。忍五年赖福，忍108并肩，忍130还刘婆；还了一笔，帖才配在怀。怀沉，沉的是序，不是惧。

惧在，也要守；守完，才有132药峰那一日。更鼓沉，按木牌，牌凉贴胸，坛腹温一线，像应「十四」——十四在，帖在，就不滚。

更鼓沉，他望山门帖影，影在字后，像仇在路后。在前，就不滚。

散队后，他过坡下石阶，阶湿，湿里数息——数帖，数层，数债序。序在，就不滚。

管事弟子再验帖，帖角齐，齐像债序落笔。落笔在怀，怀沉，沉的不是惧，是路。更鼓沉，按木牌，他低声对坛：「记着。」

外门风硬，帖角再掀半分，掀不要紧；掀在远，手在近。在近，就不滚。滚了，132药峰白去；去了，帖像摆设。摆设不要紧，手活，帖才活。活，才像第二部开笔。他望坛，独眼平：「帖在怀。十四在。下一程，药峰。」

废渣巷日斜，他验渣，渣正，手才静。

侧门驿亭风硬，他听风，风过，记在心里。

散队后，他过外堂阶下，分拣半刻，指稳——指稳，路就不飘。

编筐半刻，藤刺扎掌，刺浅，浅些沿才净。沿净，坛才藏得住。

管事记簿，记细，是小胜。胜在忍，不在辩。

他望坛，低声道：「手在，路在。」坛不应字，只温。

更鼓沉，天边白一线。白里，他按木牌，牌凉贴胸。

坊市尾巷风硬，他收摊如石，石在，就不滚。

验息后，他理怀内册页，页黄，字丑，丑而真。

铁无言声低：「别飘。」韩泥应声：「不飘。」

刘婆端粥，仍多半勺。韩泥点头，谢轻，恩重。

药峰分拣照旧，指稳，鼻下静一息。

帖在怀，怀沉，沉的是序，不是惧。

白织月袖风过，韩泥不辩。辩，费舌；舌要留来守坛。

赖福廊下过，韩泥只答：「在。」在，就不滚。

怀凉，掌温。

坡下石阶湿，他数息，数帖，数债序。

同棚杂役笑，笑在耳，不在心。
