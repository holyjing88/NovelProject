# 第七十三章 兽栏杂务夜

冬夜更深。丙九排了夜值，值在兽栏，值在腥与粪之间。

韩泥接牌。牌是旧木，字浅：「丙九韩泥，戌至丑，清栏换草。」

赖福瞥牌，眼细：「夜值也给你？末席专干晦气活。」

韩泥不辩。辩费嘴，嘴要留着清栏，也留着守坛。他应一声：「在。」

戌时初，栏里两盏灯豆，黄得像丙九的稀饭。饭稀不要紧，活要细——细了，管事才不记名。小兽角里咳一声，咳得很轻。韩泥换干草，草干了，味淡一寸。他鼻下静一息，静里竟闻见粪中一点药回——回得像白日渣场那撮辛。

第二趟清栏，雪混着粪，铲下仍稳。稳像当年踹台按实那一掌——掌在，名在，路也在。同值少年掩鼻：「丑杂役，夜值你也接？」

「接。」韩泥说，「接才有饭。饭在，恩才还得上。」

少年退半步，没再问。问了像沾晦气。晦气遮瓮也好，遮住了才活得长。管事弟子巡栏，灯照角里：「湿粪换净。换不净，记名。」

「换净。」韩泥应。

换净，是小胜。胜在活，不在嘴。管事又验水桶，桶沿净，水浊却稳。稳了，才记「丙九韩泥，夜清细」。子时风硬，灌进栏角。肩上旧伤印勒着，沉得住。沉得住，掌纹那路才不飘——飘了，粪铲就偏。

栏外铁无言抱臂立着，嗓子压低：「夜值别漏香。漏香，像求。」

「不漏。」韩泥指稳，「鼻下静。」

铁无言点头，走了。走，也像认——认不重，重在手不抖。刘婆提灯过，灯昏，照见韩泥肘袄针脚：「夜值也漏风？」

「漏一线。」韩泥说，「清栏挂袋，挂的。」

刘婆哼：「挂完回去补。补完，别谢。记着。」

韩泥点头。谢轻，恩重。恩在针脚里，也在这碗稀粥前头。丑时前，栏角最静。小兽齐睡，铲声才显得沉。他趁静，席下拽半寸坛沿，掌贴半息——沿温一线，应夜，不应字。他不揭坛，只让鼻息贴近沿边。沿边无香，有凡气；凡气里药香回涌，像一层境还活着。

邻值少年打盹，头一点一点。韩泥不喊——喊了，全丙九记名。他多铲半刻，替少年角里清干净。净了，像还刘婆一勺粥恩。更鼓二响，管事来验：「丑时末，角里仍湿？」

「干了。」韩泥说，「草换两遍。」

管事记簿，声平：「丙九韩泥，夜值细。」

赖福立栏外，像嗅：「夜值也嗅？嗅晦气？」

韩泥不接茬。接茬会饿一整排。他低声道：「鼻下静，手在。」

兽栏夜里，粪铲落栏要实。实了，气才稳；气稳了，鼻下才静。静里那一点药回，进掌纹，也进往后暗修那条路。小兽角里又咳两声，咳得重些。韩泥添草，又添水。水少，少才显勤；勤记在册，不记在嘴。嘴一响，赖福就记「邪」。

丑时前他到廊下立半刻，任腥风刮。掌贴沿，沿温不应腥，应「忍」——忍里藏刃，刃对着三年约，对着钱戾衡，对着内门那些眼。赖福盯半息，终没掀席。掀不得：沿净，坛不露；不露，就像腌菜坛，不像邪物。

丑时尽，天边白一线。韩泥收铲，肩酸，手不抖。不抖，夜值才算完。回棚路上过药渣场角，风硬，他多留半刻，分拣一撮灰，灰里一点辛。辛正，鼻下辨出苦与辛——苦沉，辛浮。样纸包实，塞怀。怀暖半分，暖不应符，应「记」。记牢了，嗅诀才生根；根在，药山才走得深。

邻床问：「夜值累不累？」

「累。」韩泥说，「累也清。清完，才有饭。」

少年闭嘴，像学会了一点忍。忍不是怂，是活路。他按席角，席严，坛不露。坛腹温一线，贴掌，像应「夜清」二字。外头鸡未鸣，夜禁脚步远。钱戾衡在玉牌里，在三年约里——约还在，夜值先做完。做完，才配下一步暗修。

他对坛低声：「兽栏夜清了。一层在掌。二层在远，手在近。」

更鼓沉。他摸怀木牌，牌凉，膝热。热在坛，热在路，热在夜值记下的那一笔「细」。独眼平：「夜值了。嗅到了。丑时暗修，二层在眼前。」

白织月远廊过，袖风冷，没停。没停小胜。秦霜木鱼远着一声，像提醒心别邪。韩泥把恩和辱分开搁：夜值是活，活是路——通向药山捡渣，通向废渣炼丹，通向以后还得上的日子。

同值少年打盹醒来，愣半息：「我角里……干了？」

「干了。」韩泥说，「你睡，我铲。」

少年脸热，没谢。谢像欠。韩泥不记欠，只记夜值那一笔「细」。沈枯芽远廊经过，步子轻，像没看见。韩泥也不近——近像求；不求，只记半块灵石恩还在前头等还。

戌时末栏角水桶翻半寸，水泼了。他舀净，舀得像还刘婆粥恩。舀净了，管事才记「角净」。角里有只独眼兔，眼浊，舔他靴边，舔得轻，像认人。认人不要紧，韩泥换草，草干，栏角才像能睡。

铁无言又过，脚步停半息：「夜值别省水。省了，腥更重；重了，瘴近。」

「不省。」韩泥应。

不省不为显勤，为栏活。栏活，丙九才不被连坐。子时二更风灌栏缝，缝响着。肩勒旧印，沉住了，粪铲才不偏——偏了，管事就记「飘」。

午后药渣场风更硬，他仍多留半刻。留不为赏，为鼻下那点根——根粗了，夜值才不虚。铁无言指一堆灰：「夜值后鼻易飘。飘了，腐当辛。腐当辛，手就滚。」

韩泥分拣，指稳，分出腐与辛。分出，铁无言点头走了。刘婆端粥，仍多半勺：「夜值饿不饿？」

「饿。」韩泥说，「饿也清。清完，才有饭。」

「饭稳，恩不断。」刘婆哼着转身走了。韩泥喝尽粥，粥稀也能暖腹。暖腹，鼻下才静，静才接得住二层暗修。赖福秤杆落盘轻，眼仍细。韩泥不辩，只按木牌。牌上「仍末」，他熟。熟不等于认命——命在坛，坛醒了，夜值也是路。

戌时末栏角水桶差点再翻，他用靴尖抵住：「稳。」水没泼。同值少年擦擦眼睛：「韩叔，你不困？」

「困。」韩泥说，「困也铲。铲完，才睡得起。」

「睡得起？」少年愣。

「睡得起，秘密才不散。」韩泥把干草铺平，「散了，坛在也像不在。」

少年似懂非懂，闷头舀粪。韩泥看他手生，伸指点一点：「从角里头往外推。推顺了，粪不溅鞋。」少年照做，溅少了，脸红：「谢……」

「别谢。」韩泥说，「谢像欠。欠不如把角清干。」

铁无言三更又过一次，只丢一句：「夜值的鼻，天亮最飘。飘了别碰灰堆。」

「记下了。」韩泥应。

记下了就真不碰——他把样纸先塞怀，灰堆只在天亮后再辨。辨之前，先把栏里最后一角湿粪换干。干了，管事簿上那笔「细」才站得住。站得住，不是为讨好管事，是为丙九一整排别被连坐饿饭。

回棚路上路过编筐房，门缝里透出一点灯。刘婆在里头纳鞋底，头也不抬：「夜里风大，领口扣上。」

「扣了。」韩泥说。

「扣了就好。」刘婆线一扯，「人丑不要紧，冻出病才丑。」

韩泥「嗯」一声。嗯里有暖。暖进心口，跟坛温并着走。并着走，他才不怕夜值后的乏。乏可以有，手不能飘；手不飘，二层那一步才接得住。

路通向药山，通向废渣，通向还汤那一日。灯在远，手在近；近在，就不滚。丑时前他再清栏半刻，替同值少年角里湿处。替不为名，为恩——恩在刘婆粥，也在丙九不连坐。管事夜巡记「夜清细」。他回棚按席角，席严坛不露。低声道：「夜值了。明日丑时，暗修二层，仍不叫。」
