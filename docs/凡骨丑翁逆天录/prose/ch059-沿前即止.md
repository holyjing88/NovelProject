# 第五十九章 沿前即止

冬更深，丙九静。静里，坛温却醒半分。醒半分，不是醒全，像瓮翻身，翻完又眠。

韩泥编筐末藤，藤刺再扎，扎得深，血珠再饱满。他仍甩向破席。席旧，吸血，血渗至席边，沿前止。

止，像一道看不见的线。线内是席，线外是沿。沿上无尘，沿上无血。

邻床梦呓：「……红……」

韩泥盖席，声低：「你看错了。睡。」

血到了席，没到沿。沿在眠，血不主动碰，这是铁规。

符怀暖一丝，护手不僵。僵，下一刺便偏。不偏，缘才正。

老耿门外咳：「沿净，好。净，才接得住赠。」

韩泥只答「好」，不追问。问急，像邪。守，像守瓮人。耿脚步远。

褐袍僧影已散，像凡符使命已了。余下瓮与血缘，余下席与沿。

午后，清西栏，粪锹沉，掌茧裂口渗潮。潮不是血，血不碰沿。

管事盯梢：「沿净，记在心里。脏沿，像拜邪。」

韩泥只答：「不拜邪。拜邪，路断。」

赖福讽：「守坛守出仙缘？」

管事声冷：「再嗓，西栏你一人清到年尾。」

赖福哑了。哑，韩泥肩上的粪锹却捆得更实。

刘婆端粥来，稀，热：「冬深，别空肚。」

「不空。」韩泥喝尽，只点头。热在，像第七笔又实。

铁无言巡栏边，声低：「沿净？」

「净。」韩泥说，「血在席，不在沿。」

铁无言灯远。

他低声对坛：「你再眠一夜。眠着，我席盖严。严，不主动滴你。滴，要等扎那一刺，误那一甩。」

坛腹起伏一线，又沉。外头风息，柱上无新条。

明日仍搬柴，仍清栏，仍末排。末排不要紧，要紧是沿净，手正，心记着七笔。

他按席，席厚，厚像守了六十日的席。命跟坛，坛跟缘，缘在沿前那一止。

血是活计扎出的，不是主动滴坛的。滴坛，要等误那一甩。

秦霜远廊一句：「手正，心别邪。邪，符灭。」

韩泥怀符暖一丝，心记着七笔，不邪。

同批少年碰他袖：「你席盖那么严，藏啥？」

「坛。」韩泥说，「坛在眠，不在醒。」

「醒了呢？」

「醒了，缘满。」韩泥说，「缘满，不在急。」

少年不懂，哼一声，走了。

他独眼平：「沿在眠，血不主动碰。碰，要等活计误那一甩。」

外头更鼓三响，响沉。沉里，瓮温醒半分，又眠回去。眠回去不要紧，要紧是沿净。

他低声道：「沿前即止，不接沿。不接，才接得住瓮醒。」

夜尽，外头更鼓又两响。响沉，坛温浮一线，又沉回去。

邻床又梦呓「红」。韩泥盖席，声低：「你看错了。红在席，不在沿。」

血止在席边，沿上仍干。仍干，像缘还差半寸。半寸，靠等，不靠催。

沿滑，像祖父韩老栓的手。手不在，滑还在。滑不要紧，沿上无血，无血才净。

清西栏第三趟，粪锹沉，肩骨吱呀一声。沿要紧，管事盯梢半息，声冷：「沿脏，像拜邪。」

韩泥只答：「沿脏，路断。」

霜贴丙九檐角，檐角白，白得像给沿描了一道边。他拿旧席角蘸水，水凉，沿滑，滑里仍干。干，才接得住后头那一甩。

编筐末藤，藤刺偏深半分。深半分，血珠更饱满。饱满，仍甩向破席，席吸尽，沿仍净。净，像瓮眠里翻身那半分，翻完又沉回去。

卯时前，他搬柴半刻。柴沉，肩勒旧印。印深，像丙九五年磨出来的。磨出来的肩，才扛得住冬深。

编筐午后，藤刺浅半分再扎。浅半分，血珠仍饱满。饱满，甩向破席，席厚，吸得尽。

清栏时，赖福故意撞他肩。撞，像试沿净不净。韩泥让半步，不让席。席在，缘在。

管事瞧见，声冷：「撞也没用。沿净的人，撞不动。」

赖福气青，走了。走了，沿更净。

铁无言巡廊，灯远前一句：「沿前即止，是止给瓮看，不是止给牙看。」

韩泥只点头。点头，像懂「牙」字指谁。

刘婆门外咳：「沿净，好。净，才少病。」

韩泥应：「净。净，像守瓮人。」

他擦沿，沿净，沿滑。净，像六十日擦出来的规矩。规矩在沿，不在舌。

赖福路过，故意踢起沿尘。尘落，韩泥不辩，只拿旧席角按住尘，再擦沿一圈。擦净，管事瞧见，声冷：「赖福，再搅，你清栏加倍。」

赖福退，退得像不甘心，却不敢再嗓。

管事验沿：「沿净，记心里。心里记着，舌上不辩。」

韩泥不辩。辩，像赖福。

铁无言夜里提灯来，灯照席边湿痕，照出一线止于前三指。他低声：「止得好。止，不是怕，是等。」

韩泥应一声，像把「等」字按进掌上。

同批少年蹭袖角，声小：「沿前三指，啥意思？」

「三指外是沿。」韩泥说，「沿不接血，缘才等得久。」

少年不懂，却走了。

黄昏，他掀席看湿痕。湿痕像一线墨，墨止于沿前三指，不洇沿。

老耿挑水路过，水洒实：「沿净，好。」

韩泥不追问。问急，像邪。

他对坛低声：「血不渗沿。渗沿，是主动滴。主动滴，像借腥。」

坛腹起伏，仍眠。

秦霜远廊木鱼不敲，只一句：「沿前止，舌也止。舌止，符才暖。」韩泥怀符，符温在肋下一线。暖，像第七笔在冬里添柴。

他用草绳量沿前三指，绳短，指尖到沿，刚好。刚好，像界已画好——界内席，界外沿；沿不接，缘才等得久。

老耿挑水又过，桶沿碰栏边，碰出一声闷响。韩泥抬眼，只道：「栏边尘，我擦。」耿不答，却多留半息，目光落席边湿痕，湿痕止于绳量处，止得像契。契在，赠才接得住。

夜第四更，邻床翻身碰他席角，碰出一声闷响。韩泥不恼，只把席角又掖实一寸。掖实，像给沿前那一止加了一道坎。坎在席，不在舌；舌若辩，沿先脏。

赖福踢尘那日，管事罚他清栏加倍。加倍，沿更净——净，像赖福的尘反而替丙九擦了界。

他拿草绳在沿边再打第二道结，结丑，丑在能记——记的是界，不是舌。界内席，界外沿；沿不接，缘才等得久。

管事夜巡验沿，烛照一线：「席边湿，沿仍净。」韩泥只答：「净。痕在席边，止于绳量处。」管事记心里，走了。

他掀席角再看一眼湿痕。湿痕像一线墨，墨止于绳结处，不洇沿。不洇，像瓮眠里翻身那半分——翻完又沉，沉不要紧，沿前止住了。

夜第四更，席边湿痕扩了半线，仍止于沿前三指。止住了，他加旧布角压上去。压实，像给瓮醒那夜垫坎。

丑时将近，檐角霜白，白里映出沿边那道界。韩泥独眼平，低声道：「沿前即止，席严符暖——止住了，才接得住瓮醒那一误。误在活计，不在舌；舌若急，沿先脏。脏沿，像拜邪。」

同批少年蹭袖角，声小：「沿前三指，你量那么准，像画符。」韩泥只答：「不是符，是界。界在，缘才等得久。等得久，才配瓮醒。」少年似懂，收声走了。

刘婆第二趟端姜汤来，姜薄，热得钝：「沿净，好。好，才少病。冬深，手别飘。」韩泥喝尽，不空，手才不飘。热在，像第七笔在冬里又实一分。

铁无言巡栏最后一趟，灯照沿边，声低：「沿前即止，记住了？」韩泥只答：「记住了。止在席，不在舌。」铁无言点头，灯远。灯远不要紧，界在，缘在，等瓮醒。

风息一刻，柱上无新条。条无不要紧，席边湿痕仍止于绳量处——止住了，才接得住瓮醒那一误。误在活计，不在舌；符怀暖一线，沿净，席严，等丑时前瓮翻那一息。翻那一息，缘才满；缘满，不在急，急像拜邪路断。
