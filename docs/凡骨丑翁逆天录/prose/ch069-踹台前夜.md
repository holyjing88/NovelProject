# 第六十九章 踹台前夜

正测后第三日，柱上又贴新条。

字利，像刀背刮纸：「明日卯时，踹台验末。凡骨杂役韩泥，必到。不到，除名。」

管事另补细规：末席验手须赤掌按台，不许戴布。布厚，手抖藏不住；藏不住，滚字成真。韩泥解腕布，布边是刘婆补过的线——线丑，线暖，暖在肘，不在台。台凉，他先把掌心里的茧按实：实了，明日才不抖。

赖福晃木牌，笑薄：「踹台专踢末席。踢完你滚，丙九脸也干净。」

「脸干净不干净，管事说了算。」韩泥按牌，「牌在，我就去。」

同批少年悄问：「你真去？」

「去。」韩泥说，「不去，除名。除名，恩断。」

少年不懂「恩断」，却懂「除名」重——重像麸皮停，像记功抹，像叶丫头那碗烫永远到不了手。

卯时前丙九列队。管事敲柱两声：「踹台专验末席手稳。手抖，除名；除名，麸皮停，记功抹。」

末排少年脸白。韩泥只摸木牌：牌在，名在，路在。散队后他过坡下测灵石。石凉，线灰，灰已公示过——公示不要紧，手要紧。

午后兽栏仍腥。他铲下稳，小兽角里咳一声，他换干草，草干味淡一寸。管事路过，记一笔：「丙九韩泥，踹台前仍清细。」

刘婆端粥，仍多半勺：「踹台？」

「踹台。」韩泥说，「末席，必到。」

「末席怕什么。」刘婆哼，「手稳，饭稳。饭稳，恩不断。」

他点头。谢轻，恩重。

药山远路，他担废渣过侧阶。白织月立阶上，声细：「丑杂役，踹台也配去？配去，别脏台沿。」

肩稳，渣不漏。漏了，赖福记过；记过，恩也断半寸。「配不配，管事贴了条。」韩泥不辩，只走。

沈枯芽在药圃角弯腰，弯得像没看见。韩泥也不近——近像求，不求，只记半块灵石那笔恩。恩在还，不在这一日。

丹房外铁无言抱臂：「灵根末，手稳，渣场仍收你。别漏。」

「不漏。」

铁无言又指坡：「踹台看手，不看嘴。嘴响，手飘；手飘，滚。」

「七笔按在掌里。」韩泥说，「掌里有数，就不飘。」

秦霜远廊经过，木鱼不敲：「心别邪。邪了，鼻下香飘。」

廊下有人练剑，笑远：「丑杂役，明日踹台见。」

另一内门接话，声更薄：「听说钱师兄要掷玉牌——三年约那种？」

笑又起，起得像潮。韩泥不接笑。接笑像认晦气；认不得，只记。仇苗不在册，在心；心在坛，坛腹温一线，应「忍」不应「慌」。

有少年追着笑问：「三年约是啥？契吗？」

「是契。」老杂役压嗓，「接了就钉死。钉死三年，活不活看命。」

韩泥听见了，不插嘴。插嘴像认，认了像怕。怕可以咽，不能露到掌上。

邻床问：「踹台怕不怕？」

「怕。」韩泥说，「怕恩断。恩不断，怕也要去——去，手才不抖。」

少年闭嘴睡了。棚里更静，更鼓沉，沉得像踹台蹲在前头。

管事夜巡验棚，灯照坛角，照见净沿不见宝。韩泥不辩，只擦沿——擦沿，像守瓮人守口。

小杂役蹭袖，声更小：「听说玉牌是契。契接了，活不过三年。」

「活不过，也要接。」韩泥说，「不接，除名。除名，叶丫头那碗烫就还不了。」

小杂役懂「还不了」三字重，收声走了。玉牌还在远，手在近——近，才验得过踹台。

外头风硬，柱上条角又白一线。他按席角，席严，坛不露。坛腹温一线贴掌，像爹的手又搭上来。爹说过：「坛别扔。」扔了就无根；无根，辱就白挨。

丑时末，他鼻下再静半刻。粟壳香里多一丝药苦，苦像药山渣。渣不要紧，鼻下稳才要紧——稳了，明日卯时掌才不抖。

赖福廊外停半息：「丑杂役，我盯着你。盯出邪，连坐丙九。」

「在。」韩泥答。

天边白一线时，内门廊下有人影立着，袖线细，金边隐约——像钱戾衡。韩泥不近。近像求；不求，只记。记完了，才配上台。

戌时他编筐末单，藤刺扎掌，偏手扎浅些。茧仍裂，裂口潮，潮不是血——血要等明日台沿那一脚。编筐钱换米，米在，恩才还得上；还得上，才对得起刘婆多半勺，才对得起叶丫头那碗烫还在前头。

管事又贴细条：「踹台只验手。手抖，除名；除名，连坐丙九。」

少年脸更白。韩泥独眼平：「连坐不要紧。手在，就不连坐。」

沈枯芽收工路过坡下，停半息，像想问，又不敢问。终于低声：「韩……韩叔，明日……手别抖。」

「记着。」韩泥只两字。无怕，无谢。谢轻，恩重；重不在舌，在手。

戌时末他过粪渠，渠冰裂一线。蹲渠边冷水洗掌，茧裂处刺疼，疼不露。洗完掌按席沿，沿净，像预演明日按台——按实，脚落也不抖。

邻值少年问：「你真不躲？」

「躲，除名。」韩泥说，「除名，恩断。恩不断，就不躲。」

更鼓二响。他合眼不睡实，怀里并着三样字：三年约、仍末、踹台。仍末不要紧，手在就不滚。

药渣场角风硬，他多留半刻，分拣一撮灰，灰里一点辛。辛正，鼻下静，静里辨出苦沉辛浮。样纸包实塞怀，怀暖半分——暖不应符，应忍。

管事又嘱半句：「踹台只验手。」

「七笔记得清。」韩泥应，「记清就不飘。」

同批又有人夜半起尿，路过他棚口，嘀咕：「听说玉牌掷台心，接了就像卖命。」

韩泥睁眼又合：「卖命也比除名强。除名，恩先断。」

那人吓一跳，脚赶忙挪开。棚里只剩坛温一线，像把「明日」二字慢慢焐热。

丑时将尽，他再把腕布叠好塞进袖底。布上刘婆针脚还在，在得像一句没说出口的话：肘暖了，台前别给脸。他把掌心里茧按了又按，按到裂口微疼——疼着也好，疼提醒他：明日不是梦。

外头更鼓又响一记。棚外有人小声赌：「赌他抖。」「赌他滚。」「赌钱师兄直接一脚踹飞。」

韩泥听着，像听风。风进棚，进不了掌。掌只答自己一句：「不抖。」

他低声道：「踹台前夜。玉牌在远，仇苗在近。近了，才磨得亮。」

刃不亮，先记。记完，才配接玉牌那一夜。台在前头，牌在更前头——前头有一夜硬霜，也有一口还要咽下去的气。气咽住，手不散；手不散，明日卯时才站得住。站得住，三年约才有地方落；落不住，叶汤凉了，仇苗也白扎。扎了白扎，就对不起公示那口辱，也对不起腕上那道丑线。

东方将白一线，他最后擦一遍坛沿。沿净，像腌菜坛，不像邪物。净给管事看，也给自己看——看清楚了，上天才知道自己为何挨踹：不为滚，为留；不为硬嘴，为还。他把玉牌二字在心里念了一遍，念完收声。声收住，手松开；手松开，明日才按得实。台在坡下，人在棚里，一夜硬霜把两边钉成同一件事：卯时见。

铁无言丑时路过渣场口，丢下一句就走：「台前别闻香。闻香，分心；分心，脚落你身上你还以为风。」

「不闻。」韩泥答。

「七笔呢？」

「在。」韩泥拍胸口偏下，「在，就不空。」

铁无言「嗯」一声，影没入廊暗。廊暗里剑笑还在远处飘，飘不进他掌茧。茧裂着疼，疼着也好——疼把人钉在今夜，钉在明日，钉在那块还没掷下的玉牌前头。前头再近，也得先挨过这一夜霜。霜过了，台才真；台真了，玉牌才掷得响。响不响由钱戾衡，稳不稳由他自己——自己稳了，明日卯时坡下验手，才接得住那一脚。
