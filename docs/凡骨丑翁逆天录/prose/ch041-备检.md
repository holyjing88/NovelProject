# 第四十一章 备检

入门两月余，雪意落了半寸，半寸薄，薄也冷。

丙九棚外贴条：凡骨杂役，测灵备检。备检不是正测，是筛——筛手抖的，筛嘴硬的，筛活不下去的。

韩泥读条，独眼平。冬测备检过了，测灵备检仍排末。末，他熟。

邻床少年抖：「听说正测在冬后，备检不过，正测没名。」

「没名就滚。」另一人接，「滚了，恩断。」

韩泥不插嘴。嘴要留来按石，不留来吵。

管事弟子喝：「丙九集合！测灵石前站直，抖的，滚！」
测场在侧门石坪，坪白，石冷。

灵石竖中央，石青，青得钝。钝，不像仙，像秤——秤灵根，秤命，秤你配不配留。

第一个少年按掌，石光一闪，绿线三寸。三寸，外门边缘。

「过。下一。」

韩泥看，不看光，看手——手按石前，气先稳，肩后沉。记法，不记色。

轮到丙九，赖福先推人：「丑杂役最后，别沾石晦气。」

管事冷声：「按序。韩泥，末。」

韩泥上前，掌茧裂处疼，疼不露。掌按石，石凉，凉进骨。

光起，灰线，线短，短不过指节。

众人嗤。嗤声薄，薄也真。

管事瞥一眼：「凡骨末等。备检过——手稳，石前不抖。」

赖福阴笑：「末等也过？丙九丢脸。」

韩泥收掌，不辩。舌要留来嗅渣，留来活。

管事掷木牌：「韩泥，备检记名。正测冬后，仍末排。」

木牌薄，薄也真。真在「记名」二字——有名，不被滚。

邻床少年按石，绿线两寸，抖了半寸。管事喝：「抖！滚！」

少年哭腔。韩泥不回头——回头，手易软。

同批有人低声：「灰线也过？」

「过的是手。」韩泥说，「石前不抖，才滚不了。」

那人点头。点头像学稳，学稳，也是小胜。

白织月经坪外，袖风冷：「末等灰线，也配站石前？」

韩泥「嗯」一声：「管事看手，不看嘴。」

白织月唇线绷一息，走了。少一句嘲，小胜。

铁无言抱臂立栏边，声低：「灵根末，手稳，渣场仍收你。别漏。」

韩泥「嗯」一声：「不漏。」

管事收石，喝：「备检毕。记名者，正测冬后。未记名者，明日滚。」

丙九静一息。一息里，有人看韩泥木牌——牌薄，牌真。

赖福阴笑，笑浅：「末等记名，正测仍是末。末，滚得快。」

韩泥不答。答，费舌；舌要留来记木牌。
午后，兽栏仍清。雪混粪，粪铲下稳。

小兽角里咳，咳声细。韩泥多清半刻，湿粪换干草。草干，味淡一寸。

管事路过：「备检过了，兽栏别抖。冬角仍你，清细，记着。」

「嗯。」韩泥说。

赖福晃秤：「末等灵根，多吃一口也是浪费。」

韩泥不接茬。接茬，全丙九饿。

刘婆舀粥，仍多半勺。半勺，热在。

她低声：「备检过了？」

「过了。」韩泥说，「末等，手不抖。」

「末等怕什么。」刘婆说，「手稳，饭稳，命稳。命稳，丹以后才稳。」

韩泥点头。谢轻，恩重。

同批少年凑近：「你怎么石前不抖？」

「搬石练肩，编筐练指，清粪练铲。」韩泥说，「练到稳，石前才稳。」

「可你是灰线……」

「灰线在石上，不在手上。」韩泥说，「手稳，饭稳。饭稳，才熬得到正测。」

少年闭嘴，像被「稳」字堵住。

管事扫赖福一眼：「你也去按石。抖了，你滚。」

赖福脸白，闭嘴。白，小胜。

赖福旁侧少年按石，绿线一寸，抖。管事喝：「滚！」

少年哭。赖福唇动，没敢再笑——笑，管事在。

韩泥收木牌，收入怀。怀暖，牌更真。
酉时，渣场。

韩泥分三堆，堆齐，香渣单放。鼻下苦甘分列，甘短，苦长。

铁无言再瞥：「备检过了，仍别漏。」

「不漏。」韩泥说。

街角赖福喝：「末等也配分堆？」

韩泥不回头。辱记着，手仍分——分堆，路才宽。

管事验场：「丙九韩泥，渣清完，香篓齐。明日仍渣场。」

「嗯。」韩泥说。

同场杂役嘀咕：「末等也配明日仍渣场？」

管事冷声：「漏了，你顶。顶不了，滚。」

嘀咕断了。

韩泥拈香渣，篓满。满而不泄，泄则路窄。

邻场杂役问：「你分三堆，费工夫，饭又不加。」

「加不加，管事看。」韩泥说，「堆乱了，丹房不收。不收，记名。记名，滚。」

杂役啧一声，仍乱拨。乱，灰扬一脸，管事骂，他揉眼走了。

韩泥指不停。不停，是备检给的底气——手稳了，渣也不乱。
夜，韩泥对坛低声：「灵末，手稳。」

坛壁水汽一线，爬半寸，敛。敛着，应「稳」字，不应灵色。

邻床问：「末等，你还熬什么？」

「熬还。」韩泥说，「手稳能活。活了，才还得上。」

「还得上谁？」

「叶丫头汤，刘婆粥。」韩泥说，「泥岗六笔，宗门一笔。分列，不混。」

邻床又问：「正测冬后，你仍末排，图什么？」

「图活。」韩泥说，「活一日，记一日。记满了，才还得上。」

邻床懂「活」，翻身睡了。

外头雪落，落声细，细得像秤盘轻响。响在耳，记在心——备检过了，正测还远，远不要紧，手稳一日，多活一日。

木牌在枕下，薄，真。真在记名，不在灵色。

管事巡夜，敲柱：「备检记名者，正测前别生事。生事，饭少。」

韩泥「嗯」一声。一声平，平得像木牌落掌——落掌，才真。

他低声道：「灵根末，命未末。命未末，就接着做粪，做渣，做筐。做到了，路才宽。」

邻床又问：「你枕下藏什么？」

「木牌。」韩泥说，「记名。有名，不被滚。」

邻床懂「滚」，翻身睡了。

坛沿温一丝，应手，不应灵。灵是石上的灰线，命是手上的茧。韩泥「嗯」一声，合眼。合眼，不是睡实，是记名。记名在枕下，命在手上。正测还远，远不要紧，手稳一日，多活一日。

测灵备检，预筛，非羞辱。羞辱在远，远像68那日——日未至，手先稳。灵根仍末，末，不谎。谎，还乱；真，才配记名。灰线过石，手稳，管事「嗯」。嗯，像小胜——胜在还能站在廊下，胜在鼻下还有路。末席过，不是荣，是门缝。缝通丹房，通嗅渣，通将来凝丹。他低声道：「灵末，手不末。」手不末，恩不末。恩在，叶汤还在前头烫。烫，是规矩；规矩，先过备检这一关。测灵备检，预筛，非羞辱——羞辱在远，像六十八那日，日未至，手先稳。灵根仍末，末，不谎；谎，还乱——真，才配记名，才配廊下嗅路。灰线过石，管事「嗯」——嗯，像小胜；胜在还能站，胜在鼻下还有路。路通凝丹，通还；还，先还烫，烫在叶汤，在刘婆粥，在七笔里。末席过，灵仍末。末不谎——谎乱还；真，才配廊下嗅路，路通凝丹。灵末手不末——手在，廊下嗅路才长，长向凝丹，长向还叶汤。末席过是门缝，缝通凝丹——凝丹为了还。

末席过，灵仍末。灵末手不末——手末，恩断；恩不断，廊下嗅路才长。长向凝丹，长向还烫；还，先过辨香关，关过，才近问香帖。

末席过，灵仍末。灵末手不末——手末，恩断；恩不断，廊下嗅路才长，长向辨香关。

灵仍末，手不末。手稳，廊下嗅路才长，长向辨香关。

他低声道：「手稳，恩不断；恩不断，汤就不凉。」

坛沿他仍擦。擦时指节白，白像证：辱在册，恩在胸，分列不混。

还远不怕，怕的是汤凉——汤在叶铺，在刘婆灶，在心里。

泥岗根在，坛沿在，还路在——还路在，手就在；手在，恩就不断。

独眼平，像看秤——秤在，铺就不关；铺不关，汤就能还热。

末席也是席，席在，手就稳——稳了，才配听帖，才配上路。

末席过，灵仍末；韩泥不谎：灵末手不末——手末，恩断；恩不断，廊下嗅路才长，长向凝丹，长向还烫，还，先过辨香关。
