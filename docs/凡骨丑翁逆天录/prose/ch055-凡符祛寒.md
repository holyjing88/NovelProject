# 第五十五章 凡符祛寒

冬深，霜厚，宗门外驿亭终有人起身。韩泥搬柴过侧门，老耿在旁挑水，水洒半勺，声哑：「去。不白拿。拿了，记耿恩，记僧恩，不拜血祠。」褐袍僧不诵长经，袖里取出黄符，符边毛，毛得像凡物。凡物，才入凡人手。「寒侵骨，骨寒侵手。」僧声低，「符不入丹房，不越阶。阶在远，寒在近。近，先祛。」韩泥不接，先问：「为何是我？」僧指他掌茧裂口：「你鼻下静，手下稳，坛边席严。严者，符不辱主。」老耿哼一声：「僧是我引的。引，不是邪，是冬里少死人。」韩泥这才接符。符薄，贴掌，掌暖一丝，暖得像经片贴怀，暖得像凡符未燃先温。「凡符祛寒。」僧道，「燃则一日暖，不燃则贴怀。贴怀，护骨寒，不护心邪。心邪，符自灭。」韩泥「嗯」一声：「心记着七笔，不邪。」僧又嘱：「符凡品，祛寒，防。勿示人，示人，谣又来。」僧合十，不拜坛，只拜冬：「瓮在眠，人别睡死。睡死，是拜邪。眠，是等。」

---

回棚，符藏怀，贴身，不燃。燃，烟起，管事查。不燃，暖一丝，够扛冬栏腥、柴房霜。赖福嗅他袖，嗅像找由头：「藏啥？」「寒。」韩泥说，「寒在骨，不在袖。」赖福不信，管事路过，声冷：「宗门外的事，少议论。议论，像借商线腥。」赖福哑了。哑，韩泥袖腥淡去一丝。管事又道：「符凡物，别傲。傲，邪。」韩泥「嗯」一声：「不傲。傲，符自灭。」刘婆端姜汤，姜薄，热：「怀里有符，别傲。」韩泥喝尽，「嗯」一声。姜热，像叶汤远影。远影在，还路在。邻床问：「啥符？」「凡符。」韩泥说，「祛寒，防。不燃，贴怀。」「贴怀干啥？」「贴怀，手不僵。」韩泥说，「手僵，藤刺偏。偏刺，血找错路。」邻床不懂，哼一声，翻身。午后，西栏清粪，风硬，符暖护骨，手不僵。粪锹沉，沉得像搬柴。搬柴手稳，清粪手也要稳。稳，试手才不滚。铁无言巡栏边，声低：「符在怀，别示。示，谣来。」韩泥「嗯」一声：「不示。示，像借腥。」「沿净？」「净。」韩泥说，「符护人，瓮眠等缘。缘，不在符。」铁无言不再言，灯远。小兽咳，咳得细。细不要紧，他多铺一层干草，草旧，旧能暖。暖兽，像兽栏那日，像恩还在。管事盯梢：「清粪加半刻。半刻偷懒，克扣麸皮。」韩泥不偷懒。偷懒，麸皮克。克粮，克不走还路。夜，他对坛低声：「凡符起，仍凡。凡，够活命。活命，才还恩。」坛腹温一线，不与符抢。符护人，瓮眠等缘。外头风硬，风里无栖香全句。全句散了，散成怀符一丝暖，散成席盖将严。他低声道：「符录起了。起了，仍凡。凡，够还恩前活命。」他是记心翁，恩仇都记在心里。符是物，物记耿恩，记僧恩，不记舌，不记外。秦霜远廊经过，一句：「心别邪。邪，符灭。」韩泥怀符暖一丝，心记着七笔，不邪。丑时，坛腹又温一线，温短，像瓮听符暖，不抢暖。他按席，席严，沿净。净，才接后头血近沿，接瓮醒那一误。外头更鼓沉，柱上五日条在。五日，试手凝丹。凝丹前，符不燃，沿不血，席盖严。严，才接瓮醒。他低声道：「叶丫头汤，热着还。还，要等凝丹。等，先过试手。」坛壁水汽爬半寸，敛。敛着，像替他守着。符护骨，手别飘。手不飘，恩不断。同批少年碰他袖：「你真得了符？」「得了。」韩泥说，「凡符，祛寒。不示，不傲。」「示了咋办？」「示了，谣来。」韩泥说，「谣来，恩易断。」少年懂「断」，闭嘴。僧离亭，脚步远。远，像凡符使命落了地。落了，仍凡，不越阶。韩泥第二趟搬柴，仍不示符。不示，像廊下避腥。腥沾袖，不贴胸。他独眼平，手不抖：「符在怀，沿在净。净了，才接后头缘。」铁无言又嘱：「符凡品，祛寒，防。记在心里，不在舌。」韩泥「嗯」一声。舌轻，恩重。西教白旗风里猎猎，猎猎不是谣，是界。界远了，谣才借不到腥。密院黑旗压车辕，车沉，沉得像克扣的麸皮。克的是粮，克不走商线。管事验袖：「宗门外的事，少议论。」韩泥不议。不议，恩在，仇也在，都在心里。夜禁脚步远，远，像密宗线又收半分。收了，仍候冬，不急。小兽眼湿，他多铺干草。草旧，旧能暖。暖兽，像恩还在。刘婆又端姜汤，姜薄，热：「符在怀，别傲。」同批少年问：「僧为啥赠你？」「鼻下静，手下稳。」韩泥说，「稳，符不辱主。」「辱主咋办？」「辱主，符自灭。」韩泥说，「灭，像心邪。」少年不懂，哼一声。第三日，搬柴经侧门，风更硬。硬，吹得霜白。韩泥不凑亭，只瞥一眼。一眼，褐袍人袖里黄符边毛，毛得像凡物。人手糙，糙得像编筐的掌茧。铁无言巡栏边，声低：「符在怀，手别僵。僵，藤刺偏。」韩泥「嗯」一声：「不僵。僵，血找错路。」刘婆端粥，稀，热：「符在怀，别空肚。」「不空。热在，像第七笔又实。邻床问：「符能燃吗？」「能。」韩泥说，「不燃，贴怀。贴怀，够扛冬栏腥。」

他独眼平，手不抖：「符护骨，沿在净。」

同批少年碰他袖：「符真暖？」「暖一丝。」韩泥说，「一丝，够扛冬寒。」「燃了更暖吧？」「燃了，管事查。」韩泥说，「查，像示人。示人，谣来。」少年懂「谣」，闭嘴。夜，坛腹温一线，不与符抢。净，才接后头血近沿。全句散了，散成怀符一丝暖。坛壁水汽爬半寸，敛。僧离亭那日，他符藏怀，不示，不燃，像经片挪了地方。缘，不在符，在瓮眠等的那一日。外头更鼓沉，沉里，符暖仍在，手仍稳。僧离亭前，又嘱一句：「符凡品，祛寒，防。」韩泥「嗯」一声，符贴怀，怀暖，暖像第七笔里又多一丝白善。回棚，赖福嗅他袖，嗅像找由头。由头无，只有骨寒淡去一丝。管事路过，声冷：「宗门外的事，少议论。不议，像廊下避腥——腥沾袖，不贴胸；贴胸的，记在心里。午后清栏，西栏风硬，符暖护骨，手不僵。僵，藤刺偏；不偏，缘才正。刘婆端姜汤，姜薄，热：「怀里有符，别傲。姜热，像叶汤远影，远影在，还路在。凡，够活命；活命，才还恩。」坛腹温一线，不与符抢，各守各的——符护人，瓮眠等缘。

符贴怀第三日，他不燃不示。示了，谣来；谣来，恩易断，汤就凉。
