# 第四十二章 兽栏

冬深，霜咬棚顶，咬得丙九人缩肩。

兽栏更深，深在角。角里小兽卧干草，卧不稳，喘细，喘得像漏风的箱。韩泥清粪，粪铲下实。实，是备检后第三日练出来的——练在雪里，练在腥里，练在赖福的秤旁。管事掷单：「丙九韩泥，冬角仍你。清细，别漏。」

「清。」

赖福阴笑：「晦气杂役，兽栏专收晦气兽。」

韩泥只应一声，舌要留来哄兽——兽不懂灵根，懂手温。

午后，小兽咳重，重一声，轻一声。韩泥换干草，草旧，潮。潮草撤，新草铺，铺得平，平才卧得实。同批杂役掩鼻：「病兽，别沾身！」

「沾了，洗手。」韩泥说，「手净，粪才净。」

那人退半步，仍嘟：「晦气。」

小兽眼半阖，半阖里还有一点活气。活气薄，薄也真。韩泥蹲，掌轻按兽背，背烫，烫得浅。浅烫，比冷强。「活一日，清一日。」他低声道，「清一日，少挨一句骂。」

兽不懂，咳轻半声。轻半声，也是小胜。

同批杂役又凑近半步：「你按它，传染你怎么办？」

「传染不了饿。」韩泥说，「饿传染全丙九。你躲，栏臭；栏臭，管事骂你。」

那人噎住，退远，仍嘟：「晦气。」

韩泥从破袄内摸出半块麸皮，掰指甲大，搁干草边。兽鼻动，舔不到，舔，也是活气。管事路过，瞥一眼：「还管病兽？」

「管。」韩泥说，「角清不净，全栏臭。臭了，你骂丙九。」

管事「哼」一声，走前却道：「清细，记着。」

赖福远处啐：「装善。」

韩泥低头换草。善不善，兽不管；兽只管草干不湿。

黄昏，雪又落。落进栏，落进角，落进小兽鼻下。韩泥加半刻，把湿粪清尽，尽到角外三尺。三尺净，味淡一寸。小兽喘更细，细得像线将断。他低声道：「断不得。你断，我少一笔记功；你活，我多一口饭。」

兽眼动一下，动，像应。

夜前，管事来查，靴底踩雪，声沉：「角清了？」

「清了。」

管事蹲，看兽：「这畜，活不过今夜。」

「知。」韩泥说，「活不过，仍清。」

管事看他一眼，眼里没有笑，也没有嘲：「丙九里，你话少，手紧。」

「话多，手抖。」

管事起身，掷半块麸皮：「赏你的。别给赖福看见。」

韩泥接住，麸皮糙，糙也香。香在冬里，冬里一口，够撑丑时。

同批杂役嘀咕：「病兽死了，你也捞赏？」

「兽死栏净。」韩泥说，「净了，少骂丙九。少骂，也是饭。」

嘀咕断了。刘婆提粥来，粥稀，热：「守角别抖手。抖了，备检白过。」

「不抖。」

她哼一声，多舀半勺。半勺，热在。赖福远处骂：「疯婆子又偏晦气！」

刘婆回骂：「晦气在你秤上！秤歪，粥才稀！」

韩泥不掺。掺，费舌；舌要留来清兽栏。

卯时将至，同批杂役路过栏外，仍掩鼻。掩鼻挡不住味淡的事实——事实在铲下，不在嘴上。韩泥把消毒水淋角三遍，水凉，凉不过手紧。手紧，管事才肯在簿上添「冬角细」一笔。细字小，小也真；真在备检前站得住脚。

他抬眼望发粮场方向，场棚矮，矮里藏着秤。秤虚一日，丙九颤一日；颤久了，坛沿也会凉。凉不得，他把目光收回栏角，角净，才配盯秤那日三两实沉。

同批少年路过栏外，仍掩鼻。韩泥不劝，只把铲柄磨光滑一寸——磨光滑了，明日第一铲才落得快。快不是躁，是活。

雪停时，栏外脚印乱。乱脚印里有一只是管事的，一只赖福的，一只他自己的。他自己的最实，实得像在雪里刻字——字丑，刻的是「净」。净了，病兽罪不至丙九；不至丙九，就少挨一脚。

同批杂役躲远：「死兽角，晦气！」

「晦气在嘴，净在铲。」韩泥说。管事路过，鼻下动一息：「味淡了。淡一寸，丙九少骂一句。」

赖福阴笑：「味淡有屁用，凡骨仍是凡骨。」

韩泥铲下不停。凡骨在石上，不在栏上。栏净，饭在；饭在，命在。

丑时，小兽咳止。止，不是好，是尽。韩泥立半息，不拜，不嚎。他清最后一铲，铲下实，实得像替兽收尽。移兽时，同批少年躲远。

韩泥独力抬兽，抬得实。赖福啐：「抬晦气畜，晦气上加晦气！」

韩泥抬实，不辩。管事复查角：「死兽移走，角消毒。韩泥，明日仍你。」

邻床少年悄问：「你真不怕晦气？」

「怕晦气，怕不过饿。」韩泥说，「饿狠了，手抖；手抖，备检白过。」

他把半块麸皮收进破袄内袋。赏不重，重在管事看见了手紧——看见了，明日少踹一脚。

他摸坛沿，沿温应「净」字，不应哭。哭费气，气要留来清明日粪。外头风息，栏味淡一寸。他低声道：「冬深兽栏，死一畜，净一角。角净，人少骂一句。」

刘婆门外咳一声：「栏净了，睡。睡实，手才不抖。」

韩泥应，不回头。恩在粥里，不在栏哭。丑时末，同批少年梦呓问粮，他盖袄，声哑：「实粮落袋前，梦里也别抖手。」

天将亮，第一铲落下，雪混粪，铲下仍实。管事卯时验角：「味淡了。备检前仍你。」

韩泥应一声，把目光留在栏角——角净，才配盯秤那日三两沉进袋的那一声。刘婆提粥来，粥稀，热，多舀半勺。他不谢，只记粥热，记栏净，分列，不混。

丑时前，他把湿草垛挪开三尺，露净地。净地白，白得像备检的验单——验单手在，手抖验单就白。他不抖，把铲柄磨光滑一寸，光滑了，明日第一铲才落得快。活，才等得到三两落袋那声沉。

丑时末，栏味又淡半寸。淡半寸，够他清明日仍站这角。冬角在，命在。命在，瓮才醒得正。同批少年问：「兽死了，你难过？」「难过像少年。」韩泥说，「恩仇都记在心里。兽命尽，心里记着清了。」少年不懂，懂「栏净」，点头睡了。

邻床少年梦呓：「冷……」韩泥替盖袄，袄破，破也暖。暖一线，够活。管事巡夜，靴声沉：「冬角清了，明日仍你。别抖。」韩泥应，不回头。回头像认恩，恩在粥里，不在栏哭。他低声道：「栏净一日，少骂一句。少骂一句，丙九多喘一口气。」赖福远处骂：「晦气扫栏，扫出妖！」韩泥收铲入棚，铲柄凉，凉不过沿一线温。温应净，不应绝户——绝户在册，净在栏；栏净了，备检记名才站得住。

兽栏夜风硬，硬里栏味淡一寸。淡一寸，够丙九喘一口气；一口气，够活到盯秤那日。

管事卯时验角，靴立三尺外：「味淡了。备检前仍你。」韩泥应一声，目光留在栏角。

他把麸皮掰半，半留己，半塞邻床手里。塞了，不声张。声张，赖福来抢。

兽栏夜，他多清半刻粪。半刻暖丙九，暖了，手才紧，才配盯秤那日三两落袋。

他不抖，把铲柄磨光滑一寸。

淡一寸，够丙九喘一口气。

他把铲柄磨光滑一寸，光滑了，明日第一铲才落得快。

沉了，恩才不断。

栏净了，病兽罪不至丙九；不至丙九，就少挨一脚。少挨一脚，多喘一口气。

雪又落细，落进栏角，落不进他眼里——眼里只有净字，净到底，备检才验得过。

谣在秤上，不在栏净；栏净了，手紧，命在，坛沿应温一线。

丑时末，他收铲入棚，铲柄凉，凉不过沿一线温。温应净，不应哭。哭费气，气要留来清明日仍站这角。

角净了，盯秤那日三两才沉进袋。沉了，恩才不断；恩不断，叶丫头那碗烫才还得了。

雪落栏角，落不进他眼里。
