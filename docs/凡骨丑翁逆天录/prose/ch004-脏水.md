# 第四章 脏水

天刚亮。

泥岗村井台还罩着一层薄雾，雾浑，浑得像没醒透的水。韩泥去井台打水，桶旧，绳磨毛，勒手心。他独眼眯着，避开井沿青苔，放绳。水面咕一声，桶沉了。泥岗村井浅，水浑，打上来要澄半日，澄不清也喝，喝了不死。

身后脚步脆，像碎瓷踩泥。

「让开。」

韩泥侧身。赵绣春穿半新棉袄，袖口绣花，在泥岗村就算体面。她身后两妇人，一个捂嘴笑，一个拿眼斜他——斜得像看地上的脏。

赵绣春不看他脸，看他桶：「晦气翁也配打水？莫把井打浊了。」

韩泥没应，把桶提出半寸，水晃，不洒。洒了，更像晦气。

「装聋？」赵绣春嗓拔高。井台边早起的人停了，停就看戏。泥岗村不缺戏，缺角儿，赵绣春最爱当角。

她扬手。半盆脏水泼过来。

韩泥没躲全，半边麻脸和肩湿透。水臭，皂角混泥，像刷盆剩水。村人哟一声——同情在眼里，热闹在嘴上。

赵绣春拍手，笑：「冲冲晦气，也算帮你。绝户翁，别怨。」

韩泥低头，水滴淌进衣领。他独眼不抬，像认罚，也像把辱记在心里。他不辩——辩了，赵绣春更来劲；辩了，村人记住的是晦气翁顶嘴，不是脏水。五十八年，他懂：赢嘴，输路。

桶绳勒手心，勒出红印。红印浅，浅得像今日这盆脏水——泼得凶，也泼不散井。井是村的，水浑，人打浑水，也得活。

妇人里有人低声：「过分了……」

赵绣春回头一瞪，那声就灭。韩泥不指望谁出头——出头要还人情，他还不起。里正巡巷路过，瞥井台水渍，瞥韩泥湿衣，没停，只扬声：「井是村的。谁再泼，记名。」

赵绣春脸一白，甩袖走快。走快，像怕记名——记名，戏就不好看了。

韩泥等人群散尽，才把桶打满。水浑，照不出脸。他也不需要照。他低声道：「水脏了，人没脏。」

回村路上，童谣丫蹲墙根，见他湿半边，拍掌念新词：「晦气翁，脏水翁，靠近要倒霉——」

韩泥停步，看她一眼。就一眼。独眼平，不凶，像看一粒石子。童谣丫声音卡壳，卡半句，抱着石子跑了。

韩泥脚步没停。吓娃不是本意，本意是：你念你的，我走我的，念不死我，就走不死路。泥屋门口，坛在角。湿衣搭绳上，绳弯。他坐灶前，灶冷，等干。心里叶丫头的汤还在，是头一笔恩，最深——赵绣春这一泼，是头一笔辱。

韩泥低声道：「记着。」

湿衣贴背，背凉，凉不过心口那碗叶汤。汤温还在，赵盆再脏，泼不散那股热。外头马春花嗓又响，骂鸡，啐一口晦气，不知骂鸡还是骂谁。韩泥不应，拾藤条，绕膝，勒指。指疼，疼让人清醒。明日还打水——水浑，也得打。

湿衣干了，他擦坛沿。擦过一遍，沿口比白日温一线——温应恩，不应辱。辱泼得再凶，泼不进坛；坛在，根在。

赵绣春门内骂：「关什么铺！晦气！」

骂声尖，尖不过远处叶家秤杆落盘的脆。脆响起，赵声哑半寸——哑半寸，也是小胜。

张麻婆傍晚送半个馒，馒热，塞门缝：「吃。编筐手不能抖。」

韩泥掰半，半留门墩，半自食。食完，嗓稳：「记着。」

卖盐老汉路过，瞥他湿衣，没躲，只点头：「赵家那盆，过。」

过字轻，轻过脏水。坛角蒙尘，沿口温一丝不退，像替他守着心里那本账：恩归恩，辱归辱。

夜里他编筐至三更，指稳，稳得像告诉全村：水泼得脏，手不脏。手不脏，筐就结实；筐结实，路就还在。

陈姑路过，棉袄领拢紧，低声：「水脏，人心别脏。心脏，才真晦气。」

韩泥应声，不谢。谢轻，记重。翌日晨起，井台边多了一截新绳——里正叫人换的。绳新，勒手力道却旧。韩泥仍独眼低垂打水。赵绣春远远路过，未近，只鼻哼一声，剩半盆空。

他提桶回，水浑依旧，肩不湿——不湿，辱就洗过一层。沈枯芽娘在巷口咳，咳得像问：「人……没事吧？」

韩泥嗯一声：「没事。水脏，人没脏。」

芽娘点头，拽娃走远。走远，像收了一句公道——公道轻，够撑明日井台。

集尾风硬，他把湿衣搭绳上，绳弯，弯得像全村嘴。嘴弯不了心。集尾只看筐，不看脸——脸丑，筐紧，紧就能活。丑时，叶铺帘内秤杆轻响一声，响得像叶青禾没睡实。没睡实不要紧，秤在，铺就不关。

沈枯芽隔门问：「韩叔，水还打？」

韩泥应声：「打。井是村的。」

他隔巷望叶铺灯影，不近，只低声道：「水脏，人没脏。」

肩干了，衣上还留皂角腥，腥像旧辱贴着。贴着不要紧，要紧是桶还能满，满就能活。里正晨巡又咳：「再泼，记名报县。」咳声沉，沉得像给全村缝一道边——边内过日，边外惹祸。

韩泥提桶，水仍浑，却稳：「记他的名。泼我的水。水浑，路不浑。」

童谣丫远处偷看，没念词，只把石子排成一串，串像路。韩泥瞥一眼，不骂娃：「排好就排好。路长，慢慢走。」石子串排得歪，歪也行，路本就不直。娃不念词，比念词好——不念，谣就还没生根。

赵绣春泼来的脏水，韩泥不接，只擦坛沿。沿净，脏水便留不进心。心留着空位，先装恩，辱这笔候着，不急着报。

翌日井台，赵绣春远，不敢近。里正咳：「昨日那盆，记下了。」韩泥提桶，水仍浑，却稳。里正点头，走两步又停：「叶家那边，别牵扯谣。」

韩泥：「牵扯的是盆，不是叶。」

卖盐老汉蹲沿，没躲他湿肩：「赵家过。过，心里记着就行。」

韩泥只答，桶提直。张麻婆又塞门缝半块饼：「吃。手别抖。」韩泥点头，饼掰半，半留门墩——留，像还恩，也像告诉赵家：泼不散手。

衣拧干，他仍去集尾。锤没落前，手不能浑，筐不能松。沿口温一丝不退，像替他守着那本账。他望坛，低声道：「手在，路在。」

坛不应字，只温。更鼓沉，天边白一线。白里，他按了按心口，硌得踏实——叶汤在，赵辱在。

他夜里又摸了一次坛沿。沿温一线，像站他这边。站不说话，也比满村吆喝实在。吆喝能泼脏水，泼不进坛腹；坛腹空着，空着才装得下往后的日子。

洗过的衣裳还留皂角腥，腥提醒他：辱不是梦。梦会醒，辱要记。记不是为了天天翻出来磨牙，是为了将来一报报得准——报早了像小孩子赌气，报晚了别散了账。

井台新绳摸着手涩。涩得好，涩说明刚换过。里正那声「记名」还挂着，挂着就多半寸规矩。规矩不大，大不过赵绣春的脸；脸再大，也大不过一村的井。

他编到半夜，筐沿齐了，指节却被脏水渍过的地方发紧。发紧提醒他：今天不是白挨。挨过就要长记性——记赵绣春怎么扬手，记村人怎么看戏，记童谣卡在半句时那点慌。慌过的嘴，以后未必再硬。

半块馒还留着门墩上那一半。他没全吃光。留着，像留一口气给明天：气在，手就不空；手不空，才配去井台把桶打满。馒热散了也没关系，热过就行——跟那碗汤一样，热过的，心里留得住。

辱记着，不急报。报要时机，时机靠活。明日井台还去——还去，像告诉赵绣春：泼得脏，泼不散命。命在，坛在，规矩还在村里那一句话：再泼，记名。
