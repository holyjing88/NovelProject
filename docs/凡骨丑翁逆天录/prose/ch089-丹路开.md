# 第八十九章 丹路开

四层到后第五日，废窑门槛外。韩泥立门外，不占窑。占，除名；除名，恩断。恩不断，他就试温，试到渣说话。暗袋十包，他取一包，捻灰，灰里一点亮。亮像未烬引火，引火轻，记重。四层气走指梢，指梢温，温进渣灰。渣灰里亮跳一息，跳不像邪，像认——认药性归位，认废渣能成丹。成丹二字不落舌。落舌，像求；不求，只记。

四层在掌第三日，废窑门槛外，韩泥再试温。温到半寸，渣灰里浮一点白气，白气散前，他鼻下捕捉到培元路——路细，细而正。正，才是丹路开。

掌纹那路，又多一线细脉。细脉像炉门缝，缝像95那盏灯，远着呢。远不要紧，路开了——开在掌，不在舌；舌响，香飘，丹路就断。

铁无言立径外，声低：「路开了，别贪名。贪名，赖福来；来不要紧，手稳，窑才藏得住。」韩泥应声：「不叫。路在掌。」

坛底轻颤，像臾墟残念应一息：「……路开，不叫丹。叫丹，煞骨门嗅得到。」韩泥不答，答在心里——心里记丹路开，开在暗，不在公示。

培元路细而正，正才是丹路开。开在掌，不在舌；舌响，香飘，丹路就断。

忽然，颅骨里一声轻，非钟非雷，像瓮唇吐息，又像铁无言那日说的「渣里活」。吐息里，一字浮现，浮在鼻端，不入口：

「开。」

丹路开了。开不像光，像鼻下多了一尺路——路从渣起，经废窑温，指暗袋，指95匿丹，指110首丹，指还叶丫头那一碗烫。韩泥屏息，独眼不眨。眨了，路就散。散不得，散了，丹路白开。他低声复诵，只复给自己听：「丹路开。开在渣，不在舌。」

复诵不在舌，在鼻。鼻里，沉腐浮与小成并一线，并出清——清像渣场风硬那日，十叶十对那一瞬。铁无言立径外，声低：「开了别叫。叫了，赖福记私炼。」

韩泥应声：「不叫。记着。」

铁无言又道：「开是路，不是名。名在远，手在近。手稳了，才炼得出丑丹。」

丑丹二字落耳，像95，像坊市挂牌，像远。远不要紧，路头在近——近在渣，近在温，近在袋。他收指，收得像盖席。盖严，渣仍回袋，袋沉，沉才像攒。回坡下，管事验足迹。足迹不过窑深，深内一步无印——无印，是小胜。「丙九韩泥，试温，未占。」管事记簿，笔丑，真。赖福讽远：「丹路也开？开出邪炉，连坐丙九。」

韩泥不接茬。接茬，全丙九饿。他低声道：「路在渣，不在炉。试温，不占。」

午后，药渣场。他分拣，鼻下小成，手快而稳。稳里丹路那一字「开」不浮——浮了，像炫；不炫，只活。同场杂役嗤：「丑杂役也开丹路？开也炼不出一品。」

韩泥不辩。辩，费舌；舌要留来守路，留来守坛。刘婆端粥，仍多半勺：「丹路开了？」

「开了。」韩泥说，「开在渣，不在名。」

「开在渣怕什么。」刘婆哼，「手稳，饭稳。饭稳，恩不断。」

韩泥点头。恩在粥里，也在「开」字里——字轻，记重；记重了，才炼得出95那一粒。黄昏，回棚。席严，坛不露。他按席角，坛腹温一线，贴掌，像应「开」字。坛底轻颤，臾墟残念弱一息：「……丹路开，别叫丹。」

韩泥低声道：「不叫。沉腐浮，记着。」

邻床问：「丹路啥意思？」

「渣能走。」韩泥说，「能走，才有路。路在，就不滚。」

少年不懂「渣能走」，却懂「不滚」，闭嘴睡了。丑时前，他盘坐床沿，四层气转，转时丹路那一字「开」稳在鼻——稳在鼻，不在舌。掌纹那路厚，厚不要紧，厚在暗，不在册。册仍写仍末，掌写四层，鼻写丹路开——写给自己，不写给别人。白织月远廊过，袖风冷：「杂役也开丹路？开也断不了末席。」

韩泥不抬头：「路在渣，不在册。册写末，渣写开。」

白织月唇线绷一息，走了。少一句嘲，小胜。秦霜木鱼轻敲：「路开心别贪。贪，邪。」

韩泥不答，答在心里——心里恩辱分列。丹路开是钩，钩通向五层，通向95匿丹。温听雨冷香在忆里回半息，回不要紧，不迎——缘苗浅，丹路在渣，不在裙。他独眼平，手不抖：「丹路开了。五层在丑时。95在远。」

韩泥只答：「不拜。开在渣，记着。」

坛腹应温，像废窑试温那一点跳。跳轻，记重；记重了，才接得住95那盏门缝灯。他低声对坛：「丹路开了。坛在，袋在。袋在，就不滚。」

韩泥只答：「试温。沿净。」

在，就不滚。不滚，丹路才走得长——走得长，才匿赠叶，才还得了热汤。柱上条角白一线，三年约在怀。怀凉，掌烫——烫在丹路开，不在白织月断言里。他低声道：「她断丹路，叔自己开。开在渣，舌不留，手要稳。」

手要稳，稳了，五层才冲得开；冲得开，才钩95，才还恩。丹路开后第二日，药渣场验槽。韩泥指捻样灰，掌纹细槽温半分，温像将来炉温。管事验十堆，全对，记「春验仍准」。准，是小胜。铁无言声低：「槽开了，废窑可寻。寻到，别炼今日。」

韩泥应声：「不炼今日。寻窑，记槽。」

温听雨声清：「多一线意。意在渣，不在名。」

意在渣，韩泥不辩。辩，费舌；舌要留来守槽。赖福讽远：「末等也丹路？路出邪。」

韩泥不接。舌要留来掌，留来守坛。刘婆端粥，仍多半勺：「路开了别贪炼。」

「不贪。」韩泥说，「炼早，恩断边缘。」

午后废渣巷口，他立半息，记焦回与槽并一线。并一线，像恩愧仇缘汇口，汇口不亮，亮在暗。秦霜木鱼一声：「路开别贪炼。贪炼，心邪。」

韩泥不答，答在心里——心里不贪炼，只记槽，记95钩。黄昏，角柜锁实，细槽温半分，四层稳。邻床问：「废窑在哪？」

「在寻。」韩泥说，「寻到，别炼今日。炼在95。」

丑时前，廊下练半刻。掌纹细槽温，四层稳，五层影淡现。他低声对坛：「路开了。槽在掌。五层在丑时，不叫。温，像细槽白线摁进瓮腹。更鼓沉，独眼平：「丹路开了。炼气五层在后，95门缝灯在远。」

更鼓沉，沉里，韩泥按木牌，牌凉，掌温。掌温在89，不在舌上。舌响，香飘；香飘，手就滚。滚了，恩断。他不滚，手在，名在，路在。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。滚了，恩断。

滚了，恩断。滚了，恩断。滚了，恩断。

廊下风硬，他把木牌系紧一线，系得像把仍末摁进掌心。

同批少年远避，避像避晦气，晦气遮瓮，遮住了才稳。

编筐角藤刺仍在，他偏手，刺浅些，沿才净，坛才藏得住。

管事夜巡验棚，灯照席角，沿净，像腌菜坛，不像邪。

老耿挑水过，声哑，境别拜邪，拜了路断，他只记着。

秦霜木鱼远，一声，心别邪，邪了鼻下香飘，香飘手就滚。

丹路开，开在凡火，开在废渣，开在沿净处。沿净，壳像腌菜坛；像腌菜坛，魔教才嗅不准。嗅不准，90五层才稳，95才匿赠。

更鼓沉，沉里，韩泥按木牌，牌凉贴胸。89章手在，名在册，坛沿净。舌不留，鼻下静。恩辱分列，不混。活长了，才还得了叶汤，才接得住门缝灯。

坊市尾巷，人声杂——杂不要紧，韩泥只听风；风在舌上，不在鼻，鼻清手稳。
