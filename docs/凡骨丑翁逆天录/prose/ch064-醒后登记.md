# 第六十四章 醒后登记

天白一线。

更鼓已尽，韩泥睁眼，席下坛腹温一线不退——不退，像昨夜不是梦。梦不留掌纹，掌纹里那路还在，还在就不敢松。

邻床鼾声如雷。小杂役翻身，眯眼半息：「你昨夜……啥味儿？像药，又像泥。」

「编筐扎的。」韩泥声平，「看错了。睡。」

小杂役哼一声，又睡实。韩泥擦掌，布黑，布上印痕浅——浅不要紧，沿上那一线朱红他记得，记得就不叫。

他按席角，席严，坛不露。露了，赖福来，管事来，晦气二字又贴回脸上。贴回不要紧，瓮醒不能贴——醒在沿，沿在席，席在命。

窗纸灰白，符影早散，散处仍留一点暖，暖得像叶青禾送汤那日碗沿的热。热在记里，不在舌上；舌响，手就飘。

卯时前，丙九集合。管事弟子敲柱：「名册更！凡骨杂役，冬后测灵前，再登记！」

名册旧，纸黄，黄得像泥岗里正那枚印。韩泥排队末席，独眼平。赖福站队首，眼细，细得像秤——秤人，秤麸皮，秤谁多喘一口气。

「韩泥。」管事弟子念名。

「在。」

「入门半年，兽栏包清，备检记名，灵根末等——仍末？」

「仍末。」韩泥说，「手稳，石前不抖。」

管事弟子笔尖停半息，瞥他掌茧：「掌裂未合，编筐扎的？」

「扎的。」韩泥说，「活计扎，不是邪。」

「邪不邪，冬后验。」管事弟子记簿，「丙九韩泥，冬后正测，仍末排。」

赖福阴笑：「末排也记？记了，正测滚更快。」

管事冷声：「滚不滚，看手。你少嗓，多秤。」

赖福脸一青，青完，眼却扫韩泥袖——袖里无香，无烟，像昨夜那一点亮从未有过。扫完，笑更薄：「丑杂役，登记也沾晦气？」

韩泥不辩。辩，费舌；舌要留来清栏，留来守坛。管事掷木牌，木牌旧，字深：「仍末。」

仍末，他熟。熟不等于认命——命在坛，坛醒了，末排也能活。活长了，才还得了叶丫头那碗烫。

散队后，赖福堵侧门：「昨夜你屋里有光。光什么？偷练邪法？」

韩泥独眼抬一下，又落：「油灯豆。豆爆，爆完即灭。」

「豆爆有符影？」

「看错了。」韩泥说，「眼昏，像符；符在经片，不在坛。」

赖福盯半息，盯得像要把席盖掀开。韩泥不躲，不迎，只按木牌——牌在，名在，路在。赖福终是没掀，啐一口：「晦气。盯着你。盯出邪，连坐丙九。」

「盯。」韩泥只答一字。一字，像把「滚」字按远半寸。

兽栏仍腥。腥不要紧，手稳才要紧。他清栏第一趟，铲下稳，稳得像昨夜血没出过——出过，只在席，只在沿，只在心。心记里多了一路，路通向还恩，通向省亲，通向泥岗那碗还没还尽的汤。

小兽角里咳一声，咳轻。韩泥换干草，草干，味淡一寸。管事路过，记一笔：「丙九韩泥，清细。」

同批少年掩鼻：「你怎不怕腥？」

「怕。」韩泥说，「怕也清。清完，才有饭。」

少年退半步，没再问——问，像沾晦气。记细，是小胜。胜在活，不在嘴。

管事弟子另翻一栏「特长」，笔尖停半息：「兽栏清细——写上。」韩泥不谢。写上，像把晦气也记成一笔活；活记在册，滚字才不好随便抹。赖福瞥见那栏字，脸青半分，没再嗓——嗓要留来秤麸皮，秤不到韩泥掌茧上。

午后，药渣场角风硬，硬里带苦辛。韩泥分拣，指稳，稳里掌纹那路微温一线——温不应渣，应「醒」字还新。铁无言抱臂立栏边，声低：「登记了，手别飘。飘了，渣场不收。」

韩泥应声：「不飘。」

铁无言走了，走了也像认可——认可不重，重的是手不抖。他不声张。声张像求丹；不求，只记。记瓮醒，记仍末，记赖福盯梢——盯梢是仇苗，苗不深，深在将来。

刘婆端粥过，粥稀，勺沉：「登记了？」

「记了。」韩泥说，「仍末排。」

「末排怕什么。」刘婆哼，「手稳，饭稳。饭稳，恩不断。」

韩泥点头。谢轻，恩重。恩在粥里，也在肘上新针脚里——针脚丑，挡风；风不进，臂才抬得起粪。

老耿挑水路过，水洒稳，声哑：「登记别拜邪。拜，路断。」

韩泥只答：「不拜。坛在，人在。」

老耿点头，水晃稳，稳得像告诉他：沿净，路就不绝。

黄昏，韩泥编筐半刻，藤刺扎掌，掌茧旧裂处又渗潮。潮不是血，血不碰沿。他偏手，刺浅些——浅些，沿才净；净了，坛才藏得住。邻床少年瞧：「登记了还编？」

「编。」韩泥说，「筐卖钱，钱买米。米在，恩才还得上。」

管事夜巡，灯照席角，照见厚席不见坛。坛不露不要紧，沿要净。管事记簿一笔，笔丑，真在「仍末」——仍末，也是名；有名，就不易被滚。

赖福傍晚又来，秤杆落盘，落轻半分。轻半分，韩泥不接——接，像谢。谢轻，记在心里：赖福盯梢，盯的是坛，盯的是光，盯的是「晦气杂役怎么还不滚」。

他低声对坛：「醒过了，仍像腌菜坛。坛在，人不叫。不叫，才守得住。」

坛不应字，只温，温得像应。更鼓将起，柱上贴新条一角——角未干，像有人刚刷冬后正测的字。字丑，真；真在「仍末」，真在「卯时」。

赖福立条下，眼细，像等韩泥去读。韩泥读条，不近，不躲：「仍末。卯时。手稳。」

赖福笑：「手稳有屁用。正测滚了，坛也扔。」

「扔不得。」韩泥说，「祖辈传的。」

赖福想再尖，管事喝：「赖福，再嗓，克半月麸皮！」

赖福哑了。哑，韩泥掌按木牌，牌凉，心热——热在坛，热在路，热在「仍末」二字还没写滚的那一日。

夜，席盖又严一层。严，不为保暖，为挡目。邻床问：「你盖那么严，热不热？」

「热不要紧。」韩泥说，「沿净要紧。」

他合眼，不睡实。赖福脚步在廊外停半息——停，像盯。盯不要紧，醒过了，不怕盯；怕的是叫，叫出来了，路断。

他低声道：「盯吧。坛在，根在。根在，正测才站得住。」

坛腹温一线，贴掌，像爹的手又搭上来。爹说过：「坛别扔。」扔不得，扔了就无根；无根，叶汤就还不了热。

赖福的脚步远一息，又近一息，像故意让韩泥听见：「丑杂役，我盯着你。」

韩泥只答：「在。」

在，就不滚。不滚，才接得住正测，接得住瓮醒后的每一日。三日后条在柱，冬后在远；远不要紧，手在，帖在，卯时在。

他廊下练半刻鼻下静，粟壳香淡，淡不要紧；要紧是香一字不落。落齐了，坡下才站得住。坛腹温一线，应「仍末」，不应慌——慌像少年，他是记心翁，翁慢，恩不凉。

同批少年悄问：「赖扒皮盯你，怕不怕？」

「怕。」韩泥说，「怕也记。记着，手就不飘。」

少年懂「飘」，闭嘴。闭嘴，像丙九也学会忍——忍不是怂，是活路。活路通向坡下，通向正测，通向还恩那日。

他摸怀木牌，牌边起霜，霜化，字更显——「仍末」二字，像被登记又摁实一摁。摁实，不惧脸丑，只惧手抖。抖了，正测白来；白来不要紧，坛在，根在。

执事远廊练嗓，嗓沉：「公示时，末席站末。末，不滚；滚，看手。」韩泥听半句，不近——近，像凑热闹。凑热闹要还人情，他还不起。

廊下风硬，他把木牌系紧一线，系得像把「仍末」摁进掌心。掌心里有瓮醒那路，路不走舌，走手——手稳了，登记才不是白登记。

柱上条角白一线，冬后正测的字在风里猎猎。猎猎不是催滚，是催手在——手在，名在册，册在管事手，不在赖福秤上。登记过了，仍末，也不叫。

柱上条角白一线，登记未完，明日验沿——验手，不验名；瓮温在内，醒字还远。
