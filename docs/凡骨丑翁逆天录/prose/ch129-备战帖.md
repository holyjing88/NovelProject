# 第一百二十九章 备战帖

试台末项前夜，备战帖至。

山门执事传话，话像鼓余音：「明日末项，培元散比试夺魁。魁者，挂牌丑丹师，递大比预选帖。」

话落丙九，丙九嗡一声。嗡里，赖福脸白，白得像嚼不出晦气：「丑杂役夺魁？魁，太阳西边出！」

韩泥不辩。辩，费舌；舌要留来养息，留来守最后一粒——粒在怀，怀沉，沉得像129末这夜。

玉牌贴胸，凉，凉里烫着三年约——约说炼气十三，站钱戾衡台前，才不滚。

刘婆端粥，仍多半勺，半勺像送行：「明日末项？」「末项。」韩泥说。

「末项别空肚。」刘婆哼，「空肚，手飘。飘了，新衣还不上。」韩泥点头。谢轻，恩重。

铁无言立门外，声低：「明日我站台侧。魁名落，别辩。辩像显摆恩。」

「不辩。」韩泥说，「恩偿，静着还，还到刘婆手里，才算数。」

钱戾衡影在山门帖背面，背面字淡，淡得像眼风：「言伯钧预选，内门少主监。」

白织月袖风过，过得冷：「明日魁，仍是运气。」韩泥捧陶罐，罐稳：「运气试台见。见正，见还，见刘婆。」

沈枯芽远远鞠半躬，不近。不近不要紧，徒恩清了半块灵石；清了，韩泥心里排序再落一笔。

秦霜远木鱼敲一声：「帖至，心别邪。邪了，鼻下香飘。」韩泥不答，答在心里。

丑时，练鼻下静。粟壳香淡，香落齐，备战帖这一夜记进心里。

邻床少年悄问：「你真敢夺魁？」「敢。」韩泥说，「不夺，三年约白接。白接，恩断。」

他望坛，低声道：「明日魁，后日还刘婆。还了，帖在怀，大比在路，钱戾衡在台前。」

外头山门风硬，帖角掀，掀出「大比」二字影。影在字后，像仇在路后；路后不要紧，刘婆这笔先还。

更鼓将尽，天边白一线。白里，试台木色浅，浅像70踹台——踹台接了约，约接到明日魁。

他按心口，低声道：「汤还是热的。」

六字落，像路标——标向130还刘婆，标向大比帖，标向钱戾衡台前不滚。

帖至当夜，丙九廊下嗡一声。嗡里，赖福脸白，白像嚼不出晦气：「丑杂役夺魁？」韩泥不辩。辩，费舌；舌要留来养息，留来守最后一粒——粒在怀，怀沉，沉像债序最后一笔待落。滚了，恩断；不滚，刘婆新衣才还得上。」「末项。」韩泥说。「末项别空肚。」韩泥点头。谢轻，恩重。魁名落，别辩。辩像显摆恩。」「不辩。」

钱戾衡影在山门帖背面，背面字淡，淡像眼风：「言伯钧预选，内门少主监。」监不要紧，手稳；稳了，丑也能挂牌——牌歪，字丑，人更丑；丑不要紧，丹正。

」见，不在嘴，在掌；掌稳，路就不绝。

不近不要紧，徒恩清了半块灵石；清了，叶汤还在前头。前头远，远不要紧，先还刘婆这笔。心里排序再落一笔：帖、魁、还、大比——四笔分列，不混。

丑时末，他理布包——新衣叠实，钱串齐，陶罐封泥，九粒培元散在怀。怀沉，沉像债序最后一笔待落。落不要紧，落要正；正了，才配夺魁。

外门弟子渐少，韩泥不近廊下笑。笑在耳，不在心——心在坛，坛腹温一线，温像应「帖」。应，不应喜；喜了，手飘。」「敢。白接，恩断。」恩断，路断；路断，坛温也断。

」台前不要紧，手稳；稳了，就不滚。」

叶青禾门缝灯未亮。未亮，像避；脉里培元仍温——温是95；散是110，两笔不混。混，四笔分列；列在心，不在舌。

老杂役问：「你真敢上台夺魁？」「敢。」韩泥声平，「不夺，约白接。白接，恩断。

柱上条角白一线，三年约在怀。怀凉，掌温——温在还刘婆，不在众目。众目笑，笑在耳，不在心。

药渣场角，他备料半刻，料细，细像末项前最后一筛。筛齐，手才静；静了，夺魁才像活，不像赌。

编筐棚里，藤刺扎掌，刺浅，浅些沿才净。沿净，坛才藏得住；藏得住，备战帖这一夜才不算空。

废渣巷风硬，硬像山门帖角。角掀，掀出「大比」二字影——影在字后，像仇在路后。路后不要紧，刘婆这笔先还。

坡下石阶，他数息，数粒，数债序。债序四笔：帖、魁、还、大比——四笔分列，不混。混了，鼻下香飘；不混，手才稳。

外堂阶下，管事夜巡记簿，记「丙九韩泥，末项前夜」。记细，是小胜。胜在忍，不在辩；辩，费舌。

侧门驿亭，风紧，紧像试台前那口气。那口气要匀，匀了，手才不飘；飘了，夺魁就虚，还刘婆就迟。

老耿挑水路过，水洒稳，声哑：「末项别拜暖。拜暖，路断。」韩泥只答：「不拜。坛在，人在。」人在，就不滚。

同棚杂役赌明日魁名：「丑骨夺魁，太阳西边出。」韩泥不接茬，接在陶罐上——罐稳，粒正，就不像赌。

他低声对坛：「明日魁，后日还。还了，帖在怀。」坛腹温，贴掌，像应「热」字。热在汤，汤在130还刘婆前头。

丙九廊下灯灭两盏，灭不要紧，坛腹温一线还在。还在，就不滚；不滚，末项才像还恩，不像赌名。

管事弟子传话第二回，话硬：「末项卯时，别迟到。」韩泥只答：「不迟。」迟了，手飘；飘了，刘婆新衣还不上。

散队后，他过药山半腰，风硬，分拣半刻，指稳，鼻下静一息——静，是把守角摁进路里。

更鼓沉，沉里，他过药渣场角，清栏，独眼平，手不抖——抖了，沿净就白挨。

午后兽栏侧，风硬，他编筐，指稳，稳里带一点不叫的回响。

邻床悄问一句，韩泥只答：「坛在。」答短，短才像常活。

怀凉，掌温——温在记恩，不在众目。

赖福脚步在廊外停半息，韩泥只答：「在。沿净。」在，就不滚，才守得住守角。

老耿挑水路过，水洒稳，声哑：「别拜暖。拜暖，路断。坛在，人在。」

秦霜远廊过，木鱼轻敲：「心别邪。」韩泥不答，答在心里——心里不叫，分列，不混。

铁无言声低：「手在就行。」韩泥应声：「在。」在，就是证——证坛在，不在嘴。

刘婆端粥，仍多半勺：「手稳，饭稳。饭稳，恩不断。」韩泥点头，谢轻，恩重。

沈枯芽怯，悄问：「韩叔，怕不怕？」「怕。」韩泥说，「怕也守。守完，才有饭。」

白织月远廊过，袖风冷，没停。没停，小胜——胜在沿净，不在嘴。

他低声对坛：「坛在，人在。人在，就不滚。」坛不应字，只温。

刃不亮，先记。记完了，才配坛在，才配下一程。

编筐半刻，藤刺扎掌，刺浅，浅些沿才净。筐卖钱，米在，恩才还得上。

入夜废渣巷，风沉，他备料半刻，指节白一息——白完，守角才落稳。

辰初外堂阶下，雾未散，他数息，独眼平，掌茧裂处疼——疼不露，沿净在忍里。

午后廊下暗处，日斜，他验渣半刻，鼻下静一息——静，是把不叫摁进路里。

管事夜巡过坛边席角，灯照席角，照见厚席不见坛——坛不露，坛在才藏得住。

散队后，他过侧门驿亭，风硬，练诀半刻，指稳，鼻下静一息——静，是把记恩摁进路里。

更鼓沉，沉里，他过药山半腰，分拣，独眼平，手不抖——抖了，守角就白挨。

丑时末，他再按陶罐，罐底温一线，温像应「末项」——项在卯时，还在刘婆针脚前头。

外门风硬，帖角再掀半分，掀不要紧；掀在远，手在近。在近，就不滚。
