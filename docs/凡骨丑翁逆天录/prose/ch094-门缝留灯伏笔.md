# 第九十四章 门缝留灯

冬夜。

沉丹宗外门侧，叶青禾住的小舍在坡下。舍小，门旧，门缝宽一线——宽一线，够透光，不够窥人。

韩泥不去敲门。敲门，像连累；连累，她说过「别来丑」。丑不要紧，恩要还——还在缝，还在灯，还在她怕黑那几年。

沈枯芽先过，怯声：「叶姐，韩……韩叔让带话：冬夜别灭灯。」

叶青禾愣半息，眼低：「他……他还记着？」

「记着。」沈枯芽说，「说怕黑。」

叶青禾指节白一息，白完，又松：「我……我不怕。别让人看见。」

「无字。」沈枯芽说，「只有饼。」

饼干，干不要紧；干，像常送，不像情。情在还，不在嘴。

韩泥立远廊，独眼不往门里探。探，像邪；邪了，秦执事来。来不要紧，他只备灯油半壶——壶小，壶口封泥，泥上无字。

油是凡油，豆油耗，耗慢，慢才留得住长夜。长夜，她怕黑；怕黑，是恩因，恩因要还。

他把油壶搁门侧石下，石下垫草，草湿，湿像雨夜常有人放物。常有人，才不惹眼。

第二夜，他仍不近。近，赖扒皮盯梢；盯不要紧，他换法：悬细线于门楣，线尾系油豆一粒，豆小，爆即灭，灭前够她取壶。

第三夜，叶青禾开门缝，缝里瞧见石下油壶，壶旁又多一小囊——囊空，囊上缝丑，缝像刘婆顶针路。她取壶，手抖，抖不要紧；抖，像终于敢接恩。

她低声对缝外：「……还不起。别再来。」

缝外无人应。无人应，才是韩泥。韩泥在远，在兽栏，在席下按坛。

他低声道：「灯在，就不黑。」

不黑，还一半；另一半，在丹。丹未成，灯先留——留缝，留温，留她冬夜敢合眼。

赖福查夜，火折照坡：「谁放油？」

韩泥立队中，声平：「不知。丙九在栏。」

赖福眼细，扫石下，石下空——空，因叶已取。取不要紧，取在恩，不在证。

管事弟子记：「外门失窃？无。散。」

散队，韩泥回棚。席下坛温一线，贴掌，像应「缝」字。缝留灯，是伏笔；伏笔，不是戏，是还。

邻床问：「你夜归迟，干什么？」

「清栏。」韩泥说，「栏腥，手稳。」

丑时，他廊下静半刻。粟壳香淡，香里多一线豆油气——气淡，淡不要紧；淡，才像凡灯，不像灵火。

他摸怀经页，页上「培元」浅。浅，下一回要炼实；炼实，匿赠门缝，赠不示名。

沈枯芽白天悄问：「叶姐……收下了。」

韩泥只答：「好。」

好，不多字。多字，像情；情在还，不在舌。

更鼓尽，坡下小舍门缝，一线灯黄。黄不要紧，黄是凡光；凡光，才守得住秘密。

韩泥独眼平，低声道：「灯留了。下一回，丹。」

丹在暗处，暗处还恩，才配「别来丑」三个字后的那一碗烫。

坛在，根在。根在，门缝就不白留。

坡下小舍，门缝宽一线。一线光，够透，不够窥。

第二夜，他悬细线于门楣，线尾系油豆一粒。豆小，爆即灭，灭前够她取壶。

第三夜，叶青禾开门缝，手抖，抖不要紧；抖，像终于敢接恩。

「「」好，不多字。近，赖扒皮盯梢；盯不要紧，他换法：悬细线于门楣，线尾系油豆一粒。细线是他编筐剩的藤，藤细，细像虫丝。丝在暗处，暗处才看不见人。别再来。」韩泥摸怀经页，页上「培元」浅。沈枯芽在暗处探头，探头又缩：「叶姐怕黑。」韩泥只答：「灯留了。」留了，不在近；在近，像连累。

刘婆缝布条过，声低：「门缝那丫头，脉还弱？」「弱。」韩泥说，「弱，才留灯。」刘婆哼：「留灯别留祸。祸连丙九。」韩泥不接茬，接在木牌上——牌在，名在，路在。更鼓沉，韩泥按木牌，牌凉贴胸。他独眼平，低声道：「灯留了。下一回，丹。」丹在暗处，暗处还恩，才配叶丫头那碗烫。坛沿净，席严，坛不露。

门缝留灯，不在明，在细。韩泥用编筐剩藤搓细线，线细，细像虫丝。丝系门楣，楣上悬油豆，豆小，爆即灭，灭前够她取壶。取壶，她就不怕黑——怕黑，是叶青禾；怕黑不要紧，灯在，怕退半寸。第一夜，他只放石下油壶，壶小，壶嘴封蜡，蜡封，封住味。味泄，赖扒皮来；来不要紧，味在蜡里，蜡在石下，石下无人。

第二夜，他不近，近像连累。叶青禾开门缝，缝里空，空不要紧，空才敢探。探手，摸见油壶，指节白一息，白完，收回去。收回去，像仍说「别来丑」——丑在脸，不在恩；恩在还，不在嘴。

刘婆顶针缝囊，囊丑，囊空，空不要紧，空才像常物。常物藏枕下，枕下她敢接。接，是95将赠之预演；预演不要紧，预演不在丹，在敢。

浅，下一回要炼实。炼实，匿赠门缝，赠不示名。名示，赖扒皮记过；记过，丙九连坐。连坐不要紧，匿才不连。

第三夜，油壶旁多小囊，囊缝丑，丑像刘婆路。叶青禾取壶，手抖，抖不要紧；抖，像终于敢接恩。她低声：「……还不起。别再来。」声轻，轻进缝，缝外无人。无人，灯却留了——留了，下一回，丹。

门缝留灯，油豆悬楣，细线如虫丝。丝在暗处，暗处看不见人。看不见人，叶青禾才敢取壶——取壶，怕黑退半寸。怕黑不要紧，怕在幼，幼在恩未还尽。

韩泥不近，不近像冷，冷不要紧，冷才像「别来丑」。丑在脸，不在灯；灯在还，不在嘴。嘴要留来守坛，留来备95暗炼。暗炼在远，灯在近。近，是伏笔。

刘婆缝囊，囊丑，囊空，空才像常物。接，是恩；恩不示名，名示则连累。连累不要紧，匿才不连——不连，95囊才过缝。

第三夜，她取壶，手抖，抖像终于敢接。低声「别再来」，声轻，缝外无人。无人，灯却留了——留了，下一回丹。丹在暗处，暗处还恩，才配叶丫头那碗烫。烫在记，不在舌。

坡下小舍，门缝一线，一线像叶青禾的怕——怕黑，怕丑，怕恩还不完。还不完不要紧，还一半是一半；一半在灯，一半在95将赠之丹。丹在远，灯在近；近，不现人，现油豆。油豆爆，爆完即灭，灭前够她取壶。取壶，怕退半寸；退半寸，像敢接恩。敢接，是刘婆缝囊之路；囊丑，囊空，空才像常物。

韩泥摸怀经页，页上「培元」浅，浅要炼实，炼实才过缝。过缝不示名，名示则赖扒皮记过；记过，丙九连坐，叶家线断。线不断，95才匿赠；匿赠，温而不灼，灼了脉受不住。受得住，脉稳，恩还一半。更鼓沉，按木牌，灯留了，缝在，坛沿净。下一回，丹。

坡下风硬，硬里门缝一线。一线像等，等油豆，等囊，等95之丹。丹未炼，灯先留——留灯不现人，现人赖扒皮记过。记过不要紧，匿才不连。连了，叶家线断；线不断，恩还在缝里。

门缝留灯，灯不在明，在细。细线悬楣，楣上油豆，豆小，爆即灭，灭前够取壶。取壶，叶青禾指节白一息，白完，收手，像仍说「别来丑」——丑在脸，不在灯；灯在还，不在嘴。韩泥不近，不近像冷，冷不要紧，冷才像恩不示名。名示，赖扒皮记过；记过，丙九连坐，叶家线断。常物藏枕下，枕下她敢接；敢接，是95预演。

预演不要紧，预演不在丹，在敢。敢接，脉才稳半分；稳半分，省亲前才还一半。一半在灯，一半在丹；丹在远，灯在近。近，不现人，现油豆；豆爆，爆完即灭，灭像93无字投序。更鼓沉，按木牌，牌凉贴胸，坛沿净，席严。灯留了，下一回，丹。

手在，路就不绝。灯留了，缝在，坛沿净。

侧门驿亭风硬，他听风，风过，记在心里。
