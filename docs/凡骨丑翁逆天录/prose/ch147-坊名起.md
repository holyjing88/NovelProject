# 第一百四十七章 坊名起

坊市尾摊的旗还是旧的，字丑，歪在风里。

韩泥立末摊，疤鬃驴拴在柱后。柱旧，蹄却稳。有人路过笑：「泥纹丑丹师？听着像诨名。」

「诨名也行。」韩泥不抬眼，「效不丑就行。」

第三日，老散修又递来一把药渣。渣灰扑鼻。韩泥鼻下静一息，辨出辛淡——辛淡正，正了，对方才肯把下品灵石往桌上推。

「就这块。」老散修嗓子哑，「小，当个证。」

「够。」韩泥收下，「证效，不证名。」

管事弟子过来记簿，笔顿了顿：「泥纹丑丹师，续留再续。」

「记。」韩泥说。

白织月从远廊过，袖风带香，脚不停。不停，像不愿跟丑名抢路。铁无言站在巷口，抱臂：「名起了，坊尾别近。」

「不近。」韩泥道，「近了费舌。」

「舌留给验渣。」铁无言点头，「有人盯梢，你看不见？」

「看见。」韩泥说，「盯皮上。手上稳，他就白盯。」

赖福果然在摊对面转了两圈，火折晃着，像要找偷税缝。韩泥把半文税推过去，文旧，旧得老实。

赖福冷声：「十七层了，税还半文？」

「半文规矩。」韩泥道，「层高不是架子。」

「哼。」赖福走了，走前丢一句，「名再响，也是杂役。」

杂役就杂役。杂役能把渣辨正，也能把驴拴稳。刘婆拎着粥罐路过，哼一声：「名是名，饭是饭。别饿着撑体面。」

「不撑。」韩泥喝粥，「体面在效里。」

「效在别处说就行。」刘婆压低嗓，「这巷子听风的人多。听多了，闲话就长腿。」

「我听。」韩泥说，「听完记，不接架。」

午后有少年故意起哄：「泥纹丑，效不丑——谁教你这半句？」

韩泥掸渣灰：「渣教的。渣不正，半句也飞。」

少年噎住。旁边老客补一句：「飞了也值。至少他闻得出。」

韩泥不接谢。谢像讨。他只把破筐摆正，牌歪着仍挂——歪牌也要歪得清楚：末摊续留，不是榜首风光。

沈枯芽远远探头，没敢近，只举手比了个「叶」的影。韩泥略一点头。点头不说话。说话容易混线；线要分清，叶汤归叶汤，坊名归坊名。

傍晚收摊时，管事又来：「续留再续，三日了。你还要挂？」

「挂。」韩泥说，「名号是风传的，手不停，才像真。」

「真就行。」管事看一眼驴，「驴别进主道。进了，税加倍。」

「不进。」韩泥牵绳，「驮药走尾巷。」

回丙九，他把条记塞进怀：「坊名起了。记着。」

坛腹温一线，像应「续」。续不在榜，在耳——耳里听过「泥纹丑、效不丑」，掌里还得再验十次。十次都正，名才不是一夜风。

邻床少年问：「韩叔，你这名听着怪，怪里还带硬？」

「硬在渣。」韩泥说，「嘴上硬没用。」

「那白织月为何不停？」

「停不住她。」韩泥道，「她不停，我正好省一口气。口气留给青木火。」

「青木火？」少年懵。

「远着。」韩泥按木牌，「远着才不躁。今日只续名。」

夜深有人在门外学那半句，学得走调。韩泥不开门。走调的是热闹，热闹熬不过鼻下那一口静。静到只闻辛淡，才把名按实。

铁无言后半夜递进半块干饼：「吃。明天药峰还你早班。」

「谢。」韩泥接过。

「别谢。」铁无言说，「名起了容易飘。飘一天，驴也会嫌你。」

「不飘。」韩泥道，「驴嫌我，我就背。」

次日清晨，药峰上照旧分拣。指稳，鼻下静一息。同事笑：「丑丹师来背渣啦？」

「背渣。」韩泥说，「背完再去尾摊。」

「尾摊热闹吗？」

「热闹归风。」韩泥道，「我只要渣正。」

渣正了，管事才肯把「续留」二字再写一笔。写一笔，名就多一分实。实了，叶家那头的信才接得住；接不住，名白起，十七也白实。

下午又来个试探的摊客，故作随意：「听说你跟叶家有来往？」

「有谢粒。」韩泥不抬眼，「来往是谣。谣买不走我的税单。」

摊客笑两声走了。笑不要紧。要紧的是手不停、税照交、驴不进主道。

夜里他望坛，低声：「名号续了。叶线在前。记着。」

记着就好。记着，收摊才像收石，石在，就不滚。滚了，风会把「泥纹」二字吹扁；不滚，二字才在尾巷站住，像一颗不漂亮却认得准的钉。

清晨摊未开，先有个老头蹲着等。老头递来一袋发霉边角渣：「闻闻。我听说你鼻子毒。」

韩泥蹲下，鼻下静一息：「边坏，中心还能用。剔边，别整袋扔。」

「真能剔？」

「能。」韩泥说，「扔了浪费。浪费不是俭。」

老头笑出声：「丑丹师这话真。我买你的效，不买你的相。」

「相本来也不卖。」韩泥收半文修理费，像收税一样老实。

白织月午前再过一次，终于停了半步：「泥纹二字，也配上嘴？」

「不上嘴也上鼻。」韩泥不抬眼，「鼻准，二字才活。」

「活几天？」

「活到效败为止。」韩泥道，「效不败，闲话先败。」

袖风冷冷甩开。甩开好。甩开不缠，名才干净。干净不是漂亮，是不欠舌债。

铁无言买了两截粗绳来：「拴驴用。名起了，驴别给人顺走。」

「谢。」韩泥接绳。

「又谢。」铁无言皱眉，「谢多像欠。欠多手脚软。」

「那记。」韩泥改口，「绳在，蹄稳。」

下午续摊时，来了个少年故意把渣袋扔远：「捡啊，丑丹师！」

韩泥不捡。捡像跪。他只说：「袋自己拿近。拿近我才闻。闻正，价算。不正，你拿走。」

少年哑火。旁边老客鼓掌：「这就对。末摊不是下人。」

管事远远看着，往簿上又添半行：「续留稳。」

稳字比名好听。好听不如管用。管用在明日还能挂牌；挂得住，叶线才有地方接。接不住，名就是一场热闹雨。

夜里他清点条记，条丑，字真。真了，才敢对坛说：「坊名起了，不是起疯。起疯会烧手。」

坛温一线。温像提醒：烧手的事先别碰。碰周家更早。周家风还没紧到门口，门口先把自己站稳。

站稳方法简单——税交、渣正、驴不进主道、舌不接架。四样齐，名才像石头，不像纸。纸一湿就破；石头能压住后头那封枯叶。

清晨续摊前，他先拴驴、摆破筐、歪挂木牌。一套动作慢，慢得像仪式。仪式不是拜神，是告诉自己：名从手上起，不从嘴上起。

老散修第三条摊子就来了，这回不递渣，递半袋晒干的边皮：「闻闻成色。」

韩泥鼻下静：「干得过度，香跑了。能配底料，不能当主材。」

「你不糊弄人。」老散修点头，「糊弄过的鼻子我闻得出。你这鼻子还算干净。」

「干净才能活。」韩泥说。

旁头有个妇人悄悄问：「泥纹丹师……真能续留？」

「续留看效。」韩泥道，「效在，管事才写字。字不在我嘴上。」

妇人笑着走了。笑比嘲好听。好听也不骄——骄一天，明日白织月袖风会更尖。

铁无言午后路过，丢下一句：「坊尾盯梢散了半个。半个还在。你别去惹那半个。」

「不惹。」韩泥应，「惹了费规矩。」

夜里他按坛：「名号续了。续在耳，也续在条。条在怀，明日还挂。」

挂着，才像石头。石头压得住闲话，也压得住后头那片枯叶。

散场前他最后摸一次歪牌。牌凉，字丑，丑得老实。老实好。老实的名能过夜；过夜的名，明日才接得住叶线，也接得住更紧的风。

邻床忽然问：「你这名要是传歪了怎么办？」

「歪了就再闻正。」韩泥说，「鼻还在，名就能纠。鼻若不在，挂一百块牌也是纸。」

「那你今晚还编筐？」

「编。」韩泥道，「编筐钱买米。米在，人才不像只靠名活。」

他把藤条绕上膝盖，刺浅浅扎一下，疼得清醒。清醒里只留一句：续留再续，续在手上。

钉住了，下一封枯叶才有地方落。落下来，他便收——收字，不收热闹。

收完，他低声道：「明日还去尾摊。摊在，名才不散。」
