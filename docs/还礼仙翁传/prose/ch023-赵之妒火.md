# 第二十三章 赵之妒火

当夜，赵玄机叩门。

三下，不重，像执法堂铁牌敲在人心上。

莫长春开门，未请入，只让出廊下凳——不坐石阶，不坐主位，借门外夜风说话，像老习惯：重话不进屋，进屋就成了私恩，私恩难还。

赵玄机立而不坐，甲未卸，铁牌在腰，月光下亮得冷。他拱手，礼足，声却平，平里烫：「莫长老，手帕定身，全场皆见。弟子有一问。」

「问。」

「为何偏帮杂役、赠剑赠符、又当众辱魔少？」赵玄机目光不移，「弟子妒你，弟子认。可弟子更不懂——你若真靠女人上位，为何又救我于微末？若不靠，为何把礼散到每个人头上，唯独不散到执法堂？」

廊下远处，有脚步停了一停，又远去——林挽月在廊角，没近，像怕惊动一场不该围观的私话。

莫长春看他，像看一块被妒火炼过的铁，铁还硬，硬里已有裂：「你今夜来，不是问礼散不散，是问你自己心在哪边。」

赵玄机一窒。

莫长春续道：「老夫赠人，不为收。你若也赠，便懂——赠出去那一刻，最怕的不是对方不还，是自己先算计还什么。」

赵玄机指节白：「弟子……私扣培元丹，是算计。」

「知是算计，便还有救。」莫长春道，「丹还库了？」

赵玄机一愣：「长老怎知？」

「心在谁那边，堂上见过。」莫长春笑，「还库，是心往那边挪了一寸。一寸够了，别贪多，贪多又像私囊。」

赵玄机沉默良久，忽然问：「长老……手帕从何来？」

莫长春看他，像看一块终于肯开口的铁：「你问帕，不如问心。心还了库，帕从心来。」

赵玄机喉紧：「弟子仍妒。妒你当众定魔少，妒霍镇山记你的恩，妒顾小满也敢记你的册……」

「妒着吧。」莫长春道，「妒可以，嚼舌不行。嚼舌伤的是你的铁牌，不是老夫的九月。」

赵玄机忽然拱手，礼比往日深，深得像把脊梁弯出一寸：「弟子……不嚼舌了。」

莫长春点头：「不嚼舌，是还库之后第二寸。两寸够了，别贪多。」

赵玄机退，退到月影里，像把妒火收进袖，收得紧，却未灭。他走过廊角时，与林挽月目光一触，一触即分，像谁都没看见谁。

---

林挽月在远处廊角，心口复杂——她慕赵师兄，却也看见赵师兄在莫长老面前矮了一寸，矮得像终于承认：有些人不是演，是活得真。

她想起赵师兄白日里对弟子仍冷，夜里却来叩门，忽然觉得慕字轻了，轻得像一张纸，风一吹就破。她转身，未近，像怕惊动一场不该围观的私话，也像怕自己一近，就要在两个人之间选边。

廊下又有杂役路过，窃语：「赵执事从莫长老院出来，脸不凶了？」

「还库之后，人都软一寸。」

「软的是嘴，不一定软心。」

窃语远去，像围观一层，又退一层，留给院里一夜静。

韩铁山在赤焰营地听闻赵夜访，冷笑对厉无殇：「青岚门内斗，正好。」厉无殇不笑，眼里血丝：「内斗？莫长春送的是恩，不是斗。你不懂。」

赵玄机回执法堂，弟子问：「赵执事，长老说什么？」赵玄机只一字：「问心。」弟子面面相觑，面觑里，赵执事铁牌仍亮，亮得像终于肯照自己的影。

孙福在苦役地听说赵夜访，嘀咕：「赵家也软了？」顾小满冷眼：「软的是嘴，不是册。」

林挽月回房，把慕字写在纸上，又撕，撕得像撕一根将断的线。她忽然明白：赵师兄的礼也要还，还得比慕更难。

莫长春独坐，簿无字，却觉赵氏那一寸挪动，像善缘微涨，不涨在名上，涨在心上。

杖中竹纹一游，气音轻。

他望月，月白，像半块灵石洗过的光。簿边无新字，袖却暖，暖得像赵家妒火还没灭，却肯进门说话。

他低声，像钩：

「妒火未灭，够暖。小满那册子，该上台面了。」

魏无涯在远处檐角，扇开一线，观赵退，观林挽月立廊角，低笑一声，像看一枚子将落：「问心二字，比问帕难。」

院外秋雨意起，起得像东脉的云又低了，低得像赵家妒火还没灭，却肯收进袖里，收得紧，紧得像铁牌终于肯照影。

风过，桂香里带雨意，像东脉的云又低了。

魏无涯檐角观，扇合，低笑：「问心二字，比问帕难。难，才像还礼。」

莫长春独坐，簿无字，却觉赵氏那一寸挪动，像善缘微涨，不涨在名上，涨在心上。杖中竹纹一游，气音轻。他望月，月白，像半块灵石洗过的光。

风过，桂香里带雨意，像东脉的云又低了。妒火未灭，够暖——小满那册子，该上台面了。
