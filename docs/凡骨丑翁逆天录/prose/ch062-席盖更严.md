# 第六十二章 席盖更严

瓮醒前夜，风硬。硬像最后一道冬关。关不过，缘不满。缘满，才误触那一滴。

韩泥清栏归来，掌茧新裂，裂口渗血，血珠饱满。他甩手，血甩破席，席吸尽，沿仍净。

他不主动滴血，不揭布诱沿。只用活计扎出的血喂席，席挡沿，沿等误触那一瞬。

邻床笑：「坛角盖那么严，藏宝？」

韩泥不辩，只把席盖又加一层。层厚，厚像秘密。秘密是坛，坛是命，命跟了五十八岁。

管事查棚，灯晃坛角：「杂物收起。」

韩泥「嗯」一声，席盖不动。动像心虚。不虚，是守。

管事盯半息，走了。只见破席，不见仙缘。

夜，符怀暖一丝，护骨寒，手不僵。僵，明日藤刺偏。不偏，缘才正。

他低声对坛：「明日或后日，藤刺还会扎。扎了，血还会来。来席，不来沿。沿上，等你认的血。」

坛腹温一线，贴掌，贴到心口也暖半分。暖，不像火，像认亲前的试探。

更鼓三响，响沉。丑时将至，像丑时坛温那一线又要走一圈。

走一圈，走到编筐扎掌，走到血误触席，走到席渗沿，走到沿颤，走到瓮醒。

他合眼，不睡实。符在怀，坛在角，席盖严。

他低声道：「席盖严了。严了，才接得住那一误。误了，根在。根在，路长。」

坛不应字，只温，温得像答应，像等。

刘婆端姜汤，姜薄，热：「前夜别空肚。空肚，手飘。」

「不空。」韩泥喝尽，「嗯」一声。

赖福讽：「盖席盖出仙缘？」

管事声冷：「再嗓，克半月麸皮。」

赖福哑了。哑，韩泥加席层的手却更稳。

铁无言门外脚步停半息，声低：「沿净？」

「净。」韩泥说，「血在席，不在沿。」

「不主动滴？」

「不滴。」韩泥说，「滴，要等活计误那一甩。」

铁无言「嗯」一声，脚步远。

韩泥又加一层破布，布旧，旧更能吸。吸血，挡沿，沿等韩氏守瓮血那一误。

邻床问：「盖那么严，热不热？」

「热不要紧。」韩泥说，「沿净要紧。净了，才接得住瓮醒。」

席盖严，严像守了五十八年的命。命跟坛，坛跟缘。

他按心口，低声道：「丫头那碗汤，热着还。还，要等瓮醒后，路才长。」

坛腹温一线加剧，加剧，像明日就在沿边等那一滴守瓮血。

他低声对坛：「席盖严了。守得严，韩家不散。」

坛不应字，只温，温得像答应。祖父韩老栓嘱过：「坛在，韩家不散。」

误触那一滴，缘才满。满，不在心急，在活计，在席，在沿。

他独眼平，手不抖：「守住了，才配醒。醒在缘，不在舌。」

老耿挑水路过，声哑：「前夜别拜。拜，路断。」

韩泥「嗯」一声：「不拜。席盖严，不主动滴血。」

他低声对坛：「钩瓮醒，钩在沿。沿净，钩才牢。牢了，才误那一滴。」

坛颤极轻，轻像应钩。

他独眼平，手不抖，低声道：「最后一夜，席盖严。不主动滴血，活计喂席，席挡沿。」

席严，沿净，符暖，手稳。稳，才配明日藤刺再扎那一误。明日，或丑时，瓮醒。

白日编筐半刻，末藤硬，藤刺偏，他偏手，刺浅些。浅些，血仍来。血落席，沿不接。

管事验筐：「季末记功在。手稳，沿还净？」「席边湿，沿仍净。」管事「嗯」一声，走了。

清栏归来，掌茧新裂，裂口渗血，血珠饱满。甩手，血甩破席，席吸尽，沿仍净。

白日第二趟编筐，刺浅些，血仍来。不接，像等缘。

他加第四层破布，布更旧，旧更能吸。层厚，厚像钩瓮醒。

管事第二趟查棚，只见破席，不见仙缘。不见，不要紧，沿要净。

刘婆第二趟姜汤，热。韩泥喝尽，「嗯」一声。不空，手才不飘。

铁无言第二趟：「不主动滴？」「不滴。」韩泥说，「滴，要等误那一甩。」

邻床第二问：「误不了咋办？」「误不了，就等。」韩泥说，「等，像守瓮人。」

坛腹温加剧第二遍，贴掌，贴到心口也暖半分。暖，像认亲前的试探。

他低声第二遍：「钩瓮醒，钩在沿。沿净，钩才牢。」

外头更鼓又三响，响沉。
