# 第二十八章 经片

村外来个跛僧，袍旧，鞋破，眉白。

在庙前摆摊，不卖香，卖经片——铜片，薄，刻字，字是密宗祛寒纹，纹丑，真。真，不在美，在用。

韩泥扫庙换斋，路过，僧看他独眼，不躲：「绝户翁，祖上守过一只瓮。瓮在眠，人别睡死。」

韩泥停：「你说什么？」

僧笑，笑浅：「你屋角那只坛，蒙尘如凡物，是韩家祖辈口传下来的，不是腌菜。命也沉。沉要补，补先补心。心暖，寒不侵。」

韩泥指节一紧，紧得像爹那句「坛在，根在」。根在口传，口传在擦沿，擦沿在命——命沉，片来补缝。

韩泥不辩信不信，只问：「经片何价？」

「不要钱。」僧推一片过来，「老耿让我捎的。他说，你抬棺稳，坛该稳。」

老耿。

韩泥愣半息，接片。片凉，贴掌，凉后一丝温，温得像凡符未燃先暖。暖浅，够记。

他低声道：「记耿恩。」

僧已走，走得不回头，像密宗线起，起在片，起在坛，起在55章凡符之前。之前，先有一片。午后，村人围看，围得像看戏。戏是僧，戏也是片。片在韩泥掌，掌糙，糙得片不滑。

马春花尖：「和尚也找晦气翁？晦气传染！」

跛僧远，远得像没听见。听见，也不辩。辩，费舌，舌要留命用。

韩泥回屋，片压坛底，压严，像压一道外来的缝。坛腹微温退半分，退得像应。应，不是醒，是稳。稳坛，稳人。

心里，又多一笔耿恩，细，像经纹。恩在，物在，路在。路通向密宗，通向寒不侵，通向将来符录不越阶。

外头西教商队铃远，远得像五脉之一，尚未到村，声先到。声先到，心先动。动，不急，急便邪。

夜里，韩泥摸坛，坛温，温得像耿恩在底。底稳，顶才稳。顶是命，命还长，长得够还。

他低声道记着，像僧袍角。浅，真。真，便按实，按到坛腹微温。

「坛稳，人不漏。」他低声道，「漏的，是谣嘴。」

嘴漏，用心堵。恩厚，经片才肯来。老耿名出，村里老人忆：「耿抬棺，绳不抖，棺不歪。」

韩泥听，不插嘴。插嘴，像攀恩。恩在，记便够。

里正来问经片，问得警惕：「和尚的东西，别惹邪。」

韩泥摊片，片铜，纹丑：「祛寒纹。我压坛底，不拜血祠。」

里正「哼」一声：「压好。压不好，连坐。」

「压实了。」韩泥说，「实，坛稳；人稳，谣就漏不进屋。」

里正无话，走了。走，也是默许——默许丑翁多一层防，防的是旱里人心邪。

叶青禾隔日送一小包盐姜，不放声。声惹祸，姜暖胃——胃暖，心便不易拜邪。

韩泥收姜，不入腹，先贴坛边，贴一息，取回，像借温。温借过，还记在叶汤旁，不欠叶家盐，欠的是情，情将来还。

经片在底，姜温在边，坛稳半分，人稳半分。半分累加，够走到沉丹宗风声贴榜。

他低声道：「密宗线起，起得静。静，才不邪。」

韩泥夜读片纹，纹丑，真，真便记。记耿恩，不拜血祠——拜邪，片会凉；不拜，片温一丝，温得像路。

路通向55章凡符，通向寒不侵，通向将来符录不越阶。阶在远，片在近。近，先稳坛。

村里老人又忆：「耿抬棺，绳不抖，棺不歪。歪了，恩也歪。」

韩泥听，不插嘴。插嘴，像攀恩。恩在，记便够；够，片才肯温。

马春花终是远问一句：「片能祛晦气？」

韩泥不答晦气，只答：「片能祛寒。寒在骨，不在名。」

名在嘴，骨在命。命稳，名自哑。

沈枯芽路过，怯问：「韩爷爷，和尚说的瓮……是你屋角那只？」

韩泥蹲低：「是坛，不是神。神在血祠，坛在屋角。角稳，人不拜邪。」

芽儿半懂，点头。点得像米粒——米粒旁，韩泥又记在心里：密宗线起，起得静；静，才不教孩口学邪。

耿恩在底，叶汤在胸，分列，不混。混了，片会凉；不混，片温一丝，路不歪。第三夜，韩泥摸坛底，片温一丝，温得像耿恩还在。

外头西教铃远，远得像五脉之一尚未到村。声先到，心先动——动，不急，急便邪。

韩泥对片纹低声念，念不成咒，念成记——记耿恩，记跛僧，记「瓮在眠，人别睡死」。睡死，是拜邪；眠，是等。等，不是信血祠，是守坛。

片纹丑，丑里真。真，不在美，在底。底稳，顶才稳；顶是命，命还长，长得够还。

里正路过，瞥窗纸影，没近。近晦气翁，像沾边。他只扬声：「和尚走了，片别乱贴。贴邪了，连坐。」

韩泥开门，独眼平：「片压坛底，贴的是寒，不是邪。邪在血祠，寒在骨。」

「骨？」

「五十八了，骨寒。」韩泥说，「寒不进心，心便不拜邪。不拜邪，里正省事。」

里正愣半息，「哼」一声，走了。走，也是默许——默许丑翁多一层防，防的是旱里人心邪。

叶青禾隔帘问，声平：「片可稳坛？」

「稳半分。」

「半分够？」

「不够，再加叶家姜。」韩泥说，「姜暖胃，胃暖，心不易飘。心飘，便信血祠；心定，便信秤。」

帘内药杵轻响，响得像盖章。章盖「还在」，盖得硬。

叶青禾又补一句：「片别示人。示人，马春花又有话说；话在，谣在。」

韩泥「嗯」一声：「片在底，人在门。门内门外，各守各的。」

韩泥对坛低声：「密宗线起，起得静。静，才不邪。邪在祠，片在底；底稳，顶才稳。顶是命，命还长，长得够还。」

坛腹微温一丝，像答。答无声，够他明日还去扫庙换斋，还去集尾卖筐，还去等荐帖贴榜。榜来，路开一线，线够钻。

老耿门外咳：「片别乱放，乱放碎坛。」

韩泥「嗯」：「放坛底。底稳，坛才眠。」

老耿脚步远，远里像叹——叹轻，像也信半分。

经片古字不识，识的是沿温。沿温在，西驿路就不飘。

他低声道：「手稳，恩不断；恩不断，汤就不凉。」

坛沿他仍擦。擦时指节白，白像证：辱在册，恩在胸，分列不混。

还远不怕，怕的是汤凉——汤在叶铺，在刘婆灶，在心里。
