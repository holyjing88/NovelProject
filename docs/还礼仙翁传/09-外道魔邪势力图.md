# 外道 · 魔邪势力图

> **定位**：在东南**五宗天试**格局之上，扩展**合欢宗、道门、佛门、密宗、西方教派**（正面/中性）与**魔教、邪教**（负面），服务六步穿插与情感线。  
> **原则**：外道不替代鸿蒙万缘塔；魔教可化敌（厉线）；邪教必灭（夺缘悖道）。
