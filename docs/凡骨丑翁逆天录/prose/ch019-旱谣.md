# 第十九章 旱谣

十月尽，井浅。

韩泥早起上山采药，篓绳旧了，系紧——系法仍是爹教的：绕两圈，勒紧，留半寸尾。后山旱，苦蒿枯了大半，蛇藤也蔫。他臂伤未全，坡滑，起根慢，一株一株拔，泥沾指，指红，红得像记。

浅得桶底蹭泥，泥腥混水腥，喝进喉，喉里也像旱。韩泥打水，桶绳放长，水起，浑，少半指。少，也得打。不打，米也煮不成；不煮，臂伤也好得慢。

井台辘轳吱一声，吱得像骨节在叫。韩泥臂用力，绳勒掌心，勒出红印——红印浅，手还实。慢不要紧，要紧的是村还要活。村道龟裂，娃不跑，跑起尘，尘像灰雪。

马春花骂天，骂到一半咽了——天骂不动，骂韩泥更动不得，动不得就憋。憋出谣，谣比旱更快。井台边，老汉蹲着，桶排成队，队里低语，低语像旱虫爬：「再不下，秋种全废。」

「废就废，废也比祭血祠强。」

「强不强，嘴说了不算，里正说了才算。」

韩泥路过，脚步没停。停，成靶。他拎桶，桶沉，沉得像榜末名——名在末，手还实。

午后，告示桩前聚人，不贴赖扒皮的黑纸，贴的是口述。口述最毒，毒在不用墨，用嘴就能杀人。周嫂嗓门大，像替天传话：「后山土地干裂，里正求雨，求不动！龙王爷不管泥岗！」

「有个过路货郎说，西边旱了三年，有人祭血祠，雨就落！落得像瓢泼！」

「血祠要什么？要晦气！要绝户！要寡妇送汤！汤一送，旱就缠！」

人群「嘶」一声，像牙酸。酸里带怕，怕里带贪——贪雨，贪替罪羊，贪一个能骂能打的出口。

赵绣春站荫下，扇风，扇得闲，像看戏不花钱：「祭什么血祠？叶家药铺不就是小血祠？抓药抓晦气，送汤送灾星！」

众人笑，笑短，笑里带怕。怕旱，怕死，怕比死更慢的渴。渴让人信邪，邪让人找近的人下手。

韩泥停步，独眼平，筐不晃：「旱是天。祠是人编的。人编来吓人，不是救雨。」

赵绣春眉一挑：「你懂祠？绝户翁成半仙了？」

「我懂渴。」韩泥拎筐转身，「渴了，井还要打。打了，村才活。活不靠祭，靠手。」

就三句。不辩邪，只辩活。活字落，像石进浑水，浑水仍浑，石在。石在，就有人噎。周嫂噎半息，脸青，啐他背影：「晦气翁装神仙！神仙怎在榜末？」

有人低声接：「榜末……也是人打的井。」

周嫂瞪眼：「你替他说话？」

那人缩，缩回队列，队列仍浑，浑里韩泥背影已远。远，像井绳，绳沉，仍拎得动桶。

村头老槐下，货郎早走，走前却留下一句像咒：「血祠不开，雨不开；开了血祠，先死晦气。」

咒落在旱土里，土干，咒不干。他低声道：「记着。封的不是雨，是谣的根。根在怕，怕在旱，旱在天，天不许人赖，人偏要赖。」

沈枯芽在巷口，手里空篮，眼望天，望得像等雨。韩泥停半息：「娃，别信血祠。」

芽儿声小：「我不信……可娘咳，赵婶说叶丫头邪……」

韩泥：「叶丫头不邪。邪在嘴。嘴信谣，药就抓不着。」

芽儿点头，跑。跑得像一粒米，落进沈家锅——锅小，米真，真米能活，活就不靠祭。

傍晚，叶家药铺门开一线，搁一只陶罐，罐口湿，湿的是凉茶。无人声，像叶青禾也知道：旱天，送汤不如送水；水也能救命，命在，铺就在。韩泥不取罐。取，坐实血祠谣——谣会说「寡妇送水也牵扯」。他只隔门道：「心领了。」

门内静一息，叶青禾声低，像从秤后传来：「旱天，汤烫不得。水凉些，别渴着。」

韩泥：「嗯。你也别熬太晚。」

叶青禾：「秤还要摆。摆实了，明日才好说话。」

韩泥：「说话的是你。我坐着听。」

门内秤砣落，一声，沉。沉后，再无声。无声，像应。应完，铺还在，理还在。

夜来，风干，像磨刀。磨刀不砍人，砍人心。村童不敢念谣，怕。怕里，有人低声学货郎句：「血祠开，雨下来……」

学的人自己先住，住得像咬了舌——舌疼，比血祠轻。韩泥独坐灶前，听一句，记一句。记不进心的，记进眼。眼记着，候还，候破。破不是杀人，是破谣，破到明日公审。

旱加重。谣也加重。加重到今夜，像绳，绳往明日公审勒，勒得紧，紧出钩。韩泥摸坛沿，坛腹微温回一丝，像吸了「血祠」二字又吐不出。

吐不出不要紧，瓮未醒，醒不在今夜。今夜只一件事：井还要打，心还要按，叶家灯还不能灭。他舀半桶浑水，浇门口一小畦枯姜苗——苗是叶家曾送姜的根，根活，「趁热」就不死。苗颤，颤得像在旱里点头。点头，就是释。

井台边，有人窃语血祠，语罢又闭嘴——闭嘴不是信，是怕。怕正好给公审留缝。井壁裂一线，裂得像村口干渴的唇——唇裂，还要打。打，才活。

马春花路过，瞥他门口那畦姜苗，嗓尖又收：「还浇？浇了也长不活。」

韩泥：「根在，就活。」

马春花：「根在，旱更大！」

韩泥不辩，只舀水，水浑，仍浇。浇完，苗颤，颤得像点头。马春花愣半息，啐地：「晦气翁跟苗说话！」

韩泥：「苗不骂人。」

马春花语塞，走。走远了，远里仍骂天，骂天骂不动，骂人也骂不出雨——雨没来，苗还在，就是释。

里正巡夜，敲锣一声，哑：「血祠再拜，抓人。抓人，连坐。」

韩泥担水路过，脚步没停：「井是村的。拜邪，关不了井。」

里正灯影晃过他肩，晃得像照见榜末名——名在末，肩还直。直，就是旱夜里的小胜。

天将亮，姜苗叶尖凝一粒夜露，露少，真——真，像「趁热」在旱里也没死绝。露滚落，落进土，土仍裂，裂不怕，根还在。

他多打半桶，半桶给邻舍寡老婆，放门阶，不叩门——叩门，成牵扯；放水，成活路。寡老婆开门缝，见桶，没出声，只放还一只小陶罐，罐空，像说「我接住了」。韩泥收罐，不收谢——谢轻，他记。记的是旱天不拜邪，也给人一口活路。

旱风硬，血祠词出街。韩泥不碰邪——碰像求，求轻。他多担一担水，水给活，不给谣。血祠鼓远，远得像催命；要紧是井台还开着，叶铺秤还在。

赖扒皮短签在桩，风撕角，撕不撕名。名在仇苗里，苗在，公审前还要立汤。韩泥擦坛沿三遍，沿净，心定。旱谣在街，姜苗在田。田旱，心不能旱——心热，秤还在，坛还在，公审前汤就不凉。井台排队仍长，长，像全村还信手，不信嘴。韩泥担水回屋，水浑，仍煮稀粥。粥稀，稀得像旱天的命——命稀，还在。还在，血祠谣就短半寸。

他合眼，手仍按坛沿。五十八岁，绝户，晦气翁，还在泥岗村西头这间最漏的屋里。

锣声远，远里姜苗仍颤，颤得像点头——点头，就是释。天将亮，云层压极低，低得像一口扣在村上的锅。锅不开，雨不落；不落，手还得打。

他舀半桶浑水，浇门口一小畦枯姜苗——苗颤，颤得像在旱里点头。点头，就是释。释完，他仍去打水。桶沉，沉得像榜末名，名在末，手还实。

他低声道：「旱还在，手还实。手实，公审前汤就不凉。血祠谣在街，姜苗在田——田旱，心不能旱。」坛角微温不退，沿口水汽一线，像瓮在睡，却听见了「血祠」二字。瓮未醒，他不喊醒；醒在缘，不在这一更。

散队后，他过外堂阶下，分拣半刻，指稳——指稳，路就不飘。
