# 第五十七章 掌茧再裂

试手前一日，搬柴加半刻。柴沉，肩稳，掌茧裂口加深，仍无血。

无血，是席盖严，是符怀暖，是心记着七笔。

铁无言路过，看一眼他手：「裂深，别硬扛。扛裂，试手抖。」

韩泥只答：「不硬扛。扛，路断。」

铁无言递一小瓶药膏，瓶丑，药凡：「凡药，涂裂，不涂沿。沿，不是你该碰的。」

韩泥接瓶，记恩在心里。涂药，裂口凉一丝，凉后稳。稳，像鼻下静气移到掌上。

赖福见药膏，眼红：「晦气也配？」

管事声冷：「药是铁师兄私赠，再嗓，克你半月麸皮。」

赖福哑了。哑，韩泥手更稳。

午后，坡下秦霜讲最后一遍：「手抖，丹歪，滚。丹歪不要紧，手不能歪。」

韩泥立末排，独眼平。铁无言验鼻，声低：「静带到手上。手上静，丹才正。」

同批少年抖肩，抖像怕「滚」字。韩泥不抖。抖是心飘，心不飘，手不飘。

秦霜目光扫过末排，停在韩泥身上，停半息：「明日试手，仍末。误者，滚。」

韩泥只点头。点头，像心口七笔压着。

管事贴条：「明日卯时，试手凝丹。丙九韩泥到。」

条薄，真。真在「仍末」，真在「明日」。

刘婆端粥来，稀，热：「明日试手，别空肚。」

「不空。」韩泥喝尽，只点头。热在，像第七笔又实。

编筐补刺，藤刺再扎，扎透旧茧，裂口渗潮。潮不是血，血不碰沿。

他停半息，涂药，药凡，稳手。手稳，明日藤刺才正。正刺，血才找席，席才挡到沿前。

赖福讽：「涂药也试手？试手验脸吧。」

管事路过，声冷：「验手，不验脸。脸丑，手正，正就够。」

赖福脸青，退半步。韩泥不辩，辱记心里，与恩分列。列清，心不乱。

搬柴过半，肩勒旧印，印深，像五年杂役磨出来的。符怀暖一丝，护骨寒，掌裂不僵。僵，明日手抖。不僵，丹才正。

铁无言廊下见他：「沿净？」

「净。」韩泥说，「血不碰沿，沿才接得住后头缘。」

铁无言退回廊角。

午后，编筐补刺，藤刺再深半分。深半分，裂口渗潮，潮不是血。

独眼瞥邻床试手帖，帖上有名，名不在末。韩泥不妒，只记自己的「仍末」。仍末不要紧，手正要紧。

铁无言折返验掌，掌心贴裂口上方一寸，不碰裂：「热不退，稳。稳，明日坡下站得住。」

韩泥应一声，像把「站得住」按进膝里。

管事嘟囔：「末席也配药膏？师兄偏心。」

铁无言声冷：「偏心偏给手稳的人。舌不稳，滚。」

管事退，退得像被「滚」字烫了一下。

坡下石阶将晚又起霜。霜白，像给明日试手铺一层白纸。纸上写字，字是「验手」。

他立坡下末排，看前排少年抖肩。抖，像怕丹歪。他不慌，掌才正。

夜，雪细，细粉肩。肩沉，沉得住。

他低声对坛：「沿在前，血不先。先血，是邪。后血，是缘。缘，等我扎那一刺。」

坛腹温比白日多一丝，像瓮听裂口稳了，仍眠。

外头更鼓沉，柱上「明日卯时」。明日，试手凝丹。凝丹不是醒瓮。醒瓮在更后，在误触那一滴。

邻床问：「你怕不怕滚？」

「怕。」韩泥说，「怕也记。记着，手就不飘。」

「滚了咋办？」

「滚了，路断。」韩泥说，「路断，粥就凉。」

邻床懂「凉」，闭嘴。

他按席，席严，沿净。净，才接明日试手，接血近沿，接瓮醒。

他低声道：「手稳了，路又宽半分。宽，不是灵根好，是七笔没断。」

坛腹温一线，又凉。凉，像应「等」。

秦霜收木鱼，木鱼落掌，声轻：「问心只问一句：为谁炼。答邪，滚。答空，滚。答热，记。」

韩泥声哑，声平：「为还。」

秦霜不夸，只点头。点头，像记功在。

他摸木牌，与试手帖并放。并放，像命与规矩挨着。挨着，才验得过。

外头风息一刻，柱上条子在风里晃。晃不要紧，字真。真在「仍末」，真在「卯时」。

他数更鼓，更鼓沉。沉里，裂口渗潮，潮不是血。无血，沿便安。

同批有人鼾如雷，雷里，他掌贴膝，膝稳，掌才稳。稳，才验得过试手。

刘婆夜来过一趟，留下半块姜。姜薄，他含在舌下，舌热，鼻便静。鼻静，手上才静。

铁无言廊下脚步声又停半息，停完即远。远，像说「药涂够了，睡仍要浅」。

五更将尽，坡下霜白，石阶滑。滑不要紧，站要稳。稳，才站得住坡下。

四更将尽，他掀帘望坡。坡下空，空得像白纸上等明日那一笔。笔在执事手里，手在他掌上。

邻床鼾声起，起得像提醒：睡浅，明早起。起得起，才配卯时。

他掌贴裂口，裂口凉，凉后稳。稳，像把「仍末」两个字摁实了。摁实，不惧验脸，只惧手抖。

刘婆留话：「试手凝丹，丹丑不要紧，手正要紧。」

韩泥记在心里，不记舌。舌记，像傲；心记，像还。

赖福临睡前嘟囔一句，嘟囔像酸。韩泥不接，只按裂口。按实，像按明日坡下那一步。

坡下风硬，吹得帖角起。起不要紧，帖在，名在，明日卯时在。

同批有人翻身，翻身像梦试手。梦不要紧，掌要紧。掌稳，梦就不乱。

铁无言远廊又嘱半句，声轻：「凝丹验手，不验灵。灵末，手不能末。」

韩泥只点头。点头，像把嘱语与帖语并在一处。并处，心不乱。

三更，坛腹温一线。一线，像瓮旁听裂口。旁听，不应，仍眠。

管事验手，声低：「裂口渗潮，别硬练。硬练，明日手抖。」

韩泥药膏收怀。怀里有药，心里有帖，帖里有「仍末」——末席不要紧，手正才要紧。

试手前夜，他合眼，不睡实。鼻下静，手上才静。静，才配末排。

药膏涂裂口最后一遍，凉后稳。掌上静，丹炉前手才不抖。

铁无言前夜最后一嘱：「沿净，别涂药上沿。」

韩泥只答：「沿净。药只涂裂，不涂沿。」

坛腹温一线，像瓮听裂口稳了。仍眠，不等醒。

秦霜远廊一句，声轻：「问心只问一句。答热，记。」

他声平：「为还。」

木牌与试手帖并放枕下。卯时，坡下见。见完试手，血线才近。

他低声道：「明日试手，验手，不验灵。灵末，手不能末。」

试手前夜，手仍稳。稳，才配接住后头血线。血线将至，将至在沿前，不在舌上。舌上急，手就飘。

五更前，他掌贴膝，膝稳，掌才稳。稳里，更鼓落底一声，像给明日坡下那一步定了调——调在掌，不在灵。药膏凉后，掌纹里「仍末」二字像被摁实了，摁实，不惧验脸，只惧手抖。

四更，坡下空，空像白纸等执事落笔。笔在执事，手在他掌。掌贴膝，膝贴地，地凉，凉不要紧，帖真。

刘婆深夜又来，塞半块姜。姜薄，他含在舌下，舌热，鼻便静。鼻静，明日坡下余韵才还在——还在，手才正。

管事夜巡，见他仍醒，声低：「明日卯时，别迟到。」韩泥只答：「在。」在，像末排也配早起；早起，不为脸，为手。
