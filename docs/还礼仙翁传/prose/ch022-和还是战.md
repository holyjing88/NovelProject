# 第二十二章 和还是战

长老会再开，烛比上次更亮，像战意也亮。

殿内坐了十七人，空了两席——莫长春未入席，只袖空，杖在塔中，立门边，像把「战」字立在殿口；魏无涯也不在，据说云游去了，据说在梁上，据说在看谁先开口。

周德海仍捧和折子，却不敢先开口，折子角皱，像被手汗浸过。他目光扫霍镇山，扫柳青鸢，扫众长老，扫到门边那根杖，像扫到一块秤砣，秤砣不表态，却比表态更沉。

霍镇山先声，声洪，震得烛焰直：「赤焰已退，退得不甘，必再犯。东脉三成若给，明日要五成！割地换安，安在谁枕上？」

周德海赔笑：「霍长老，打仗也要算利……」

「利可以算，」霍镇山吼，「根不能拔！田给了，民散了，青岚门还剩什么？」

殿内低议起，像潮头。有人附和霍，有人看周，有人看柳，有人偷瞧门边莫长春——九月老头立那儿，像什么都没听见，又像什么都听见了。

年轻长老起身：「霍长老说得对！手帕定身那日，赤焰驱妖骑袭庭，这还算盟？」

年老长老抚须：「可三日之期已至，东脉民户……」

「民户在田上！」霍镇山截话，「民散，田还在？妖骑踏过稻，你拿什么填表？」

周德海被顶得退半步，折子抖，抖得像七年的和平在风里碎。他强笑：「诸位，赤焰少谷主虽羞，韩铁山却稳。稳人主和，我们主和，不丢人……」

「丢人的是妖骑踏庭！」霍镇山拍膝，「莫长老杖扫妖骑，扫给谁看？扫给『和』字看！」

全场一寂，寂里，目光齐落门边。莫长春不抬眼，杖点地，一声，像给「和」字敲丧钟。

柳青鸢坐主位，目光扫过众长老，停在一处空席，再落回周德海脸上，眸光冷而清：「和谈已破。赤焰驱妖骑袭庭，非盟，是战书。」

她开口，声清，压过殿内潮声：「青岚门主战。」

殿内应「战」者众，声汇成一股，像风过剑林。年轻长老起身抱拳，年老长老拍膝，连平日主和的几位也沉默——手帕定身与杖扫妖骑两件事，把「和」字压成了笑话。

周德海揖手，退半步：「宗主，三思……七年和平，够年轻一辈成长……」

「三思已过。」柳青鸢道，「根在，战亦可；根断，和亦败。七年之后，把成长交给割让？」

她拍板，板声落，像定东脉命。周德海笑容彻底没了，袖中和折子像一块废铁，沉得拖腕。

莫长春杖点门槛，接一字：

「战。」

只一字，柳眼眶微红又压下，像二十年同门，终于不必独自担。霍镇山抱拳：「镇妖司今夜前移！」

孙福在殿外洒扫，听见「战」字，手抖，水洒，洒得像七年克扣的恩仇，也要被这一字洗。

众长老散议，声浪涌出殿门，廊下弟子听见「主战」二字，炸开一层欢呼，欢呼里怕，也燃——怕三日之期，燃的是青岚门终于不再主和。

有人喊：「莫长老也说了战！」

有人答：「一字就够，像定身三息。」

围观声浪推至山门，推得周德海袖中和折子更沉，沉得像废铁。

会后，柳青鸢留莫长春廊下。

月色白，风凉，像把殿内战意洗出一层静。她递茶，这一次他接了，抿一口，淡：「茶好。比和谈酒好。」

「师兄，因果已满四十，塔可小结算。」她低声，「你若收手，无人怪你。」

莫长春摇头：「收手，像把礼送一半。老夫九月，最忌送一半。」

「厉无殇记恨你。」

「记恨才好。」莫长春道，「恨比忘强。忘的人，还不了礼。」

柳青鸢望他须眉，忽然问：「帕……谁做的？」

莫长春笑，不答，只指丹堂方向：「有人收得下药渣，就有人收得下符灰。别问谁，问也没用。」

苏念慈在丹堂远处听见「符灰」二字，心口一跳，像炉盖被风掀起一线。她没出门，只把炉火拨旺，旺得像替谁守一个不能说的秘密。

廊下又有弟子经过，窃语：「宗主留莫长老说话，说什么？」

「说战。」

「九月老头也要战？」

「杖扫过妖骑的人，不战，谁战？」

围观声远去。柳青鸢收茶盏，低声：「赵家妒火，今夜怕要上门。」

莫长春点头：「上门好。上门，才送得出真话。」

夜，塔壁浮字，淡金：

**主战已定，再赠在即。**

莫长春收念，寿元行仍示八月，像钟慢了一格，够他把东脉这锅药熬浓。他望杖，杖中竹纹游动，气音轻，像应。

风过院墙，桂香与远处东脉的稻浪气缠在一起。院外脚步声轻，三下，不重——赵家妒火，果然上门。

他低声，像钩，钩在夜风尽头：

「战字定了。妒火也到了。正好，两件事一起记在心里。」

孙福在灵田苦役，听见「战」字，手抖，水洒。他嘀咕：「战了，谁还管杂役册？」

旁人瞪：「宗主裁过，莫长老定过，你管你的苦役。」

孙福噤声，像半块石又硌进膝里。

周德海袖中和折子沉得拖腕，遣心腹探赵家夜访，探得像探缝——缝在妒，妒在问心。

年轻长老抱拳：「宗主，镇妖司今夜可前移？」

柳青鸢点头：「移。移的不是盾，是胆。」

和战辩到后半夜，外门仍分两派，派别像两股风，风碰头，头破的是谣言。莫长春不站队，只递一句：「和是缓，战是急，礼是根。根在，缓急都不死。」

柳青鸢佩剑，剑穗无风，声低：「师兄，根在塔。」莫长春道：「万缘塔镇识海，识海在人心。」

赵玄机验和战帖，帖乱，乱里仍想统，统到塔意，塔意统不住。他立门边远处，像听，又像等下一句能踩——踩不到「战」字，踩不实。

苏念慈不进门，只炼丹，丹香浓，浓得像给和战盖一层苦甘。她听见「符灰」二字，心口一跳，没出门，只拨旺火。

魏无涯檐角观，扇开一线，低笑：「战字定了，妒火到了。两件事一起记在心里，才像还礼。」

莫长春立门边，袖空，杖在塔中，像秤砣——秤砣不表态，却比表态更沉。他杖点门槛，只一字：「战。」

柳眼眶微红又压下，像二十年同门，终于不必独自担。风过廊下，桂香与东脉稻浪气缠在一起。

他低声：「战字定了。妒火也到了。正好，两件事一起记在心里。」

塔壁浮字，淡金：**主战已定，再赠在即。**寿元仍示八月，像钟慢一格，够他把东脉这锅药熬浓。

孙福在灵田苦役，听见「战」字，手抖，水洒。他嘀咕：「战了，谁还管杂役册？」旁人瞪：「宗主裁过，莫长老定过，你管你的苦役。」

周德海袖中和折子沉得拖腕，遣心腹探赵家夜访，探得像探缝——缝在妒，妒在问心。年轻长老抱拳：「宗主，镇妖司今夜可前移？」

院外脚步声轻，三下——赵家妒火，果然上门。莫长春点头：「上门好。上门，才送得出真话。」

会后，柳青鸢留莫长春廊下，月色白，风凉，茶盏温。她问帕谁做，他不答，只指丹堂：「有人收得下药渣，就有人收得下符灰。」

莫长春立门边，袖空，杖在塔中，像秤砣。他低声：「战字定了。妒火也到了。正好，两件事一起记在心里。」

苏念慈不进门，只炼丹，丹香浓，浓得像给和战盖一层苦甘。她听见「符灰」，心口一跳，没出门，只拨旺火。

孙福在灵田苦役，听见「战」字，手抖，水洒。旁人瞪：「宗主裁过，莫长老定过，你管你的苦役。」

莫长春收念，寿元仍示八月，低声：「战字定了。妒火也到了。正好，两件事一起记在心里。」

和战二字悬在殿上，悬得像一把还没落下去的刀；落下去之前，赵氏妒火先烫了一步。
