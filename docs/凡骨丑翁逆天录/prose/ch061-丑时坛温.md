# 第六十一章 丑时坛温

丑时，棚静，鼾雷断了一息。韩泥未睡，盘坐，油灯豆，豆光照坛角。

角在，席严，符怀暖一丝。暖像半闪，不燃，光太亮，管事查。

坛腹温一线加剧，像心跳，像瓮里翻身半次，又眠。眠，不认主。认主，要等血缘碰沿。

他不主动滴血，不揭席诱沿，不拜邪求醒。求，像赖福。等，像守瓮人。

掌茧涂药，药凡，稳手。手稳，明日藤刺才正。正刺，血才找席，席才挡到沿前。

邻床翻身，梦呓「温」。韩泥盖席，声低：「你看错了。睡。」

他低声对坛：「你再眠半夜。缘满不满，看活计，不看心急。心急，路断。」

坛不应字，只颤极轻，轻像「嗯」，像祖父远处应。

沿上符影浮半瞬，又敛。韩泥独眼盯沿，沿净，沿滑，沿上无血。

无血，符影才像半闪，不像邪。符在怀，暖一丝，不燃。燃，烟起，管事查。不燃，暖够扛丑时寒。

铁无言门外脚步远，远，像夜禁巡过。巡过不要紧，要紧是席严，沿净。

刘婆门外咳：「丑时寒，别空肚。」

韩泥应：「不空。姜在记里。」

鸡未鸣，狗未吠，夜禁脚步远。三年约、钱戾衡，都在另一时辰。

时辰未到，瓮先温，温而不醒。

丑时将尽，温一线敛，敛，像瓮又眠回去。眠回去不要紧，要紧是席盖更严那一夜将至。

他按掌，掌贴席，席贴坛角。角温一线，温短，像廊下香字，又像凡符暖字，又像灶下那个「热」字。热在恩里，不在舌上。

外头风硬，柱上无新条。无条，有条在坛。

他低声道：「丑时坛温，一线加剧。加剧，不急喊醒。喊醒，像拜邪。」

坛不应字，只温，温得像答应，像等，像后半夜。

油灯豆将尽，豆光缩一线，缩了，沿上符影却浮半瞬。半瞬，像韩氏守瓮血还未到，符先探路。探路不要紧，沿要净。

同批少年咳嗽，咳醒半息，问：「你还不睡？」

「丑时守坛。」韩泥说，「守，不等醒。等误。」

少年不懂「误」，又睡。

天边白一线，白像瓮醒前夜。前夜，席盖更严。更严，他要加破布，加旧席，加压柴角。

他低声对坛：「丑时尽，坛温仍加剧。加剧，我仍不揭席诱你。」

坛颤极轻，轻像守瓮人远处「嗯」。

外头风息，符影却似在沿上再浮一息，又敛。敛，像钩，钩瓮醒，钩正测，钩还。

他摸怀符，符暖一丝，贴骨。骨暖，手稳，沿净，席严。严了，才配接后头席盖更严那一夜，才配接瓮醒那一误。

误在缘，不在心急。心急，血偏。偏，席挡不住。

他独眼平，手不抖，低声道：「丑时坛温，符影半闪。半闪罢，仍等活计那一刺。」

刘婆门外咳：「丑时尽，沿还净？」

韩泥应：「净。符影半闪，不燃。」

木牌在枕下，与席严并放。并放，像命与坛挨着。挨着，才接得住瓮醒那一误。

外头更鼓尽，尽处，天边白一线。白一线，像瓮醒前夜将至。将至，接席盖更严，接瓮醒，接鸿蒙九劫瓮认主。

子时尽，油灯豆，豆光照坛角。

丑时正，坛腹温一线加剧，像心跳，像瓮里翻身半次，又眠。眠，不认主。

丑时前半，油灯豆晃坛角。晃，像符影将浮。浮，沿要净。净，才像半闪。

坛腹温加剧前半，像心跳。跳，不认主。

沿上符影浮第二瞬，又敛。敛，像钩瓮醒。钩，在缘，不在急。

铁无言脚步第二趟远，远，像夜禁巡过。巡过，席仍严。

刘婆第二声咳：「丑时寒，别空肚。」

鸡未鸣，狗未吠，时辰未到。未到，瓮先温。温而不醒。

丑时后半，温一线敛，敛，像瓮又眠回去。眠回去，席盖更严将至。

他按掌第二遍，掌贴席，席贴坛角。角温一线，温短。

油灯豆将尽，符影又浮半瞬。半瞬，像符先探路。探路，沿要净。

同批少年第二问：「等啥？」「等误。」韩泥说，「误，在活计，不在心急。」

天边白一线第二遍，白像瓮醒前夜。

木牌与席严并放，像命与坛挨着。

丑时第三刻，符影浮又敛。敛，像半闪，不像邪。

坛腹温加剧第三刻，像心跳。跳，不认主。

油灯豆第三回将尽，豆光缩一线。缩了，沿要净。

他低声第三遍：「丑时坛温，符影半闪。半闪罢，仍等。」

同批少年第三回又睡。睡，像不懂「误」。

天边白一线第三回，瓮醒前夜将至。

木牌第三回与席严并放。

外头更鼓尽第三回，接席盖更严，接瓮醒。

鸡未鸣第三回，时辰未到。未到，瓮先温。

他按席第三遍，席严，沿净。净，才配符影半闪。

丑时正中，他独眼盯沿。沿净，沿滑，沿上无血。无血，符影浮半瞬，才像探路，不像邪脉。

油灯豆光缩一线，沿上符影又浮。浮罢，敛。敛，像瓮翻身半次，又眠。眠，不认主。认主，等血缘碰沿。

他低声对坛：「丑时坛温加剧，符影半闪。半闪，不燃。燃，管事查。查，像借腥。」

更鼓尽时，天边白一线。白一线，席盖更严将至。将至，接瓮醒，接鸿蒙九劫瓮认主那一误。

掌贴席，席贴坛角。角温一线，温短，温得像应「等」。等，像守瓮人。守，不等醒，等误。

温而不醒，醒在缘，不在舌。

丑时坛温加剧，符影半闪又敛。他不揭席，不滴血，不拜邪。只守，守到活计误那一甩。符影半闪，沿仍净。净，才配瓮醒那一误。外头鸡未鸣，时辰未到，瓮先温，温而不醒。

丑时后段，符影沿上再浮一息。一息，像凡符探路，像韩氏血未到。

油灯尽时，他独眼盯沿。沿净，符影半闪，不像邪脉。

天边白一线里，席盖更严将至。将至，接瓮醒，接沿边那一误。

掌纹贴席，席贴坛角。角温加剧，仍不认主。

丑时守坛，他不睡实。不睡实，像辨香前夜。鼻下静，手上才静。

符怀暖一丝，不燃。

丑时尽，沿仍净。净，才配符影半闪，才配瓮醒那一误。

他低声：「符影半闪，仍等误。」

更鼓尽时，席盖更严将至。

丑时坛温，一线加剧。

丑时坛温，符影半闪。
