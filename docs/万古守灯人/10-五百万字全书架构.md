# 《万古守灯人》500 万字 · 全书架构

> **篇幅目标**：约 **500 万字**（汉字）· **约 1250 章** · 均章 **3800～4200 字**（高潮章 5000～8000 字）  
> **现状**（2026-07-11）：220 章锚点稿约 **51 万字**（阶段一目标 150 万，完成约 **34%**）  
> **缺口**：约 **468 万字**  
> **笔锋参照**：《斗破苍穹》迟暮之约、闯塔、大会、震场；《凡人修仙传》坊市、秘境、宗门政治、谨慎生存；《斗罗大陆》测定、战队、大赛、师徒传承  
> **情感承诺**：有血有肉的迟暮之恋、忠诚与背叛、责任与担当；文字**简洁爽快**，章末必留钩  
> **与现有文档**：220 章为**不可删锚点**；100 万→500 万为逐级加厚；百万方案见 [`07`](./07-百万字扩充方案.md)

## 一、篇幅公式

```
500 万 ÷ 1250 章 ≈ 4000 字/章

锚点 220 章 × 5～6 倍加厚 + 插章 ≈ 900 章人间主线
+ 200 章情感深描 / 单元案 / 副本
+ 150 章域外 / 上界余波 / 百年番外
≈ 1250 章 → 终稿删冗余压至 500 万 ±5%
```

## 五、写法铁律（简洁爽快）

1. **一章一事**：一个冲突 + 一个小胜/小败 + 章末钩子
2. **对话≤三行**：超三行拆动作或围观
3. **阶位一句标**：「二阶烛火起」不解释半页
4. **情感当场给**：吻、抱、抵额在冲突后立即，不拖十章
5. **忠诚一句证**：「灯在，人在」类台词每人专属一句
6. **背叛当场揭**：烛火/灯影照账，不隔夜
7. **禁止**：英文夹杂、摘要重复段、同一钩子写三遍

## 六、衔接与系统铁律（500 万不可破）

1. **化灯仅部十一一次**（原第五卷 216 章锚点）
2. **陆承安仅 185 章战死**，不复活
3. **阶位递进表**以 [`06`](./06-衔接检查与修订说明.md) 为准
4. **承平年号**：37→38→39→40+
5. **无筑基**：一律「灯影稳了」「灯盏成」
6. **守灯十诫**：触犯必写反噬现场
7. **禁自害剧情**：禁止自刎/自尽/服毒/咬毒等主动自害；活不成须拉敌人垫背（见 `02` 守灯十诫下「剧情铁律」）
