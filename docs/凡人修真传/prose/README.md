## 约定

1. **一文件一章**：`volXX/NNN-章节名.md`（三位章号）
2. **文件头**：八要素状态行 + 纲来源（vol + 扩写密度）
3. **破境/天试/合阵章**：追加 `丹道【X品】 · 阵道【X阶·阵名】`（见章锚表 §六）
4. **字数**：**常章 2000～3000 字**（硬上限 3000；锚章亦同，不够则拆章）
5. **定稿前**：对照 `16` §2.2～§2.3 自检 + `18` 黄金三章 KPI
