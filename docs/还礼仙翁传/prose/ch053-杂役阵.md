# 第五十三章 杂役阵

子时将至，东脉侧径比主道更黑。

顾小满率杂役分班潜伏，路线是秦商言简上标的三条，交错，像网眼。网眼小，容不下英雄，只容得下报信的人。

第一班出发，第二班接隙，第三班备换。顾小满走在最前，隐匿符贴胸，符温，像长老的手按在心上。

田埂方向火把骤起——赤焰先锋贴地行，果然佯攻侧径，果然要断丹堂耳。

「退，不战。」顾小满低声，「消息比命贵。」

一队人缩进土坎，呼吸压到极轻。先锋过，过出一阵腥风，风后，她抬手，三短一长，信号哨，极细，像虫鸣。

丹堂方向，有人接应，接应成一线。

第二路，第三路，险象环生。有人跌，顾小满扶，扶完仍跑，跑得像半块灵石在掌里跳，跳成一颗心。有人吓软，她冷眼：「跪着送不出消息。站起来。」

赤焰先锋回头搜，搜不到人，搜得到记号——止血散撒路口，像给后来者留字：活着送到。

子时钟声远传，一声，像弓弦满。

顾小满在第三处拐点停下，取出莫长春备的止血散小包，撒路口，像给后来者留记号：「活着送到。」

她抬头，东脉灵田方向杀声已起，杀声里丹堂一线地火阵鸣，淡红，像另一颗心也在跳。

杂役阵动，动得无声，动得比剑有用。

后阵边缘，她冲至周德海指挥棚，嗓子哑：「周长老，侧门阵闸，开一次！」

周德海淡道：「杂役弟子，回你的路。」

顾小满咬牙，转身再奔，奔回灵田，像要把「侧门不开」四个字奔成一把刀，递给宗主，递给每一个肯听的人。

围观阵后弟子不知暗线，只知侧径静得反常，反常得像一根耳还在。

她低声，像钩，说给夜风：

「消息送到了。接下来，看谁还装瞎。」

风腥，杀声起，像东脉夜战开幕，缺不得这一根耳。

有人低声：「侧径怎么没动静？」另一人答：「没动静，才是动静。」

赵玄机在丹堂门洞听见信号哨，极细，像虫鸣，却懂：侧径耳还在，耳在杂役，不在长老。

苏念慈在阵眼石，地火阵纹淡红，炉中火舌细稳，像替暗线也担。她低声：「消息比命贵，命在门。」霍镇山在前阵骂：「赤焰先锋贴地行？老子骂醒你们！」

网眼小，容不下英雄，容得下报信——报信送到了，侧门仍闭，闭的人，尚要战后称一称。

围观阵后弟子不知暗线，只知侧径静得反常，反常得像一根耳还在。风腥，杀声起，像东脉夜战开幕，缺不得这一根耳——耳在杂役，不在长老；在顾小满眼里，在雨夜那道光还亮着的那一处。

顾小满在第三处拐点停下，取出莫长春备的止血散小包，撒路口，像给后来者留记号：「活着送到。」她抬头，东脉灵田方向杀声已起，杀声里丹堂一线地火阵鸣，淡红，像另一颗心也在跳。后阵边缘，她冲至周德海指挥棚，嗓子哑：「周长老，侧门阵闸，开一次！」周德海淡道：「杂役弟子，回你的路。」

杂役阵动，动得无声，动得比剑有用。她咬牙，转身再奔，奔回灵田，像要把「侧门不开」四个字奔成一把刀，递给宗主。

子时钟声远传，一声，像弓弦满。她低声，像钩，说给夜风：「消息送到了。接下来，看谁还装瞎。」风腥，杀声起，像东脉夜战开幕。

丹堂方向，有人接应，接应成一线。赤焰先锋回头搜，搜不到人，搜得到记号——止血散撒路口，像给后来者留字：活着送到。网眼小，容不下英雄，容得下报信。

第二路，第三路，险象环生。有人跌，顾小满扶，扶完仍跑，跑得像半块灵石在掌里跳，跳成一颗心。有人吓软，她冷眼：「跪着送不出消息。站起来。」

围观阵后弟子踮脚，踮得像等侧径有动静——没动静，才是动静。有人低声：「杂役暗线，比剑有用？」另一人答：「有用在耳，不在刀。」

阵后有人悄问：「杂役去哪了？」答：「去耳还回来了。」答完，侧径仍静，静得反常，反常得像一根针扎在每个人后颈。
