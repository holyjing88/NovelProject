# 第八十九章 渣中开丹

四层到后第五日，废窑门槛外。韩泥立在门外，不踏进去。

占窑除名；除名，恩也跟着断。恩不能断，他就试温——试到渣肯说话。暗袋取一包，捻灰，灰里一点亮，像未烬引火。引火轻，他记重。风从窑缝钻出来，带一点陈年焦味，像有人多年前也在这儿栽过跟头。

四层气走指梢，指梢温进渣灰。亮跳一息，跳不像邪，像认：药性归位了，废渣也能成丹。成丹二字不落舌。落舌像求；不求，只记。

温到半寸，白气浮起又散。散前，他鼻下忽然捉住一条细路——培元路，细而正。正，才叫丹路开。

颅骨里轻响一声，非钟非雷，像瓮唇吐息，又像铁无言说过的「渣里有活」。吐息里一字浮到鼻端，不进嘴：

「开。」

丹路开了。开不像光，像鼻下多了一尺路——从渣起，经废窑温，指暗袋，指往后匿赠，指还叶丫头那碗烫。韩泥屏息，独眼不眨。眨了，路散；散了，白开。

他只给自己复一句：「丹路开。开在渣，不在舌。」

铁无言立径外，声低：「开了别叫。叫了，赖福记私炼。」

「不叫。」韩泥应，「记着。」

「开是路，不是名。」铁无言又道，「名在远，手在近。手稳了，才炼得出丑丹。」

丑丹落耳，像坊市挂牌，还远。路头却近——近在渣，近在温，近在袋。他收指像盖席，渣回袋，袋沉，沉才像攒。铁无言又补：「今日只记住『开』，别想炉。想炉就贪，贪就败。」

「不想炉。」韩泥说，「先走路。」

回坡下，管事验足迹：窑深内一步无印。「丙九韩泥，试温，未占。」记簿。笔丑，字真。赖福远讽：「丹路也开？开出邪炉，连坐丙九！」

韩泥不接：「路在渣，不在炉。试温，不进窑。」

「你倒会绕！」赖福瞪眼。

「绕着活。」韩泥说，「直着死。你爱直，你去占。」

赖福脸色青一阵，终没敢踏那门槛——除名的条他还认得。午后药渣场，分拣仍稳。「开」字不浮——浮了像炫；不炫，只活。同场杂役嗤：「丑杂役也开丹路？开也炼不出一品。」

「一品不急。」韩泥说，「先走出渣。」

「走出渣算本事？」

「算出气。」韩泥道，「出气比出一口废嘴强。」

刘婆端粥，仍多半勺：「丹路开了？」

「开了。」韩泥接碗，「开在渣，不在名。」

「开在渣怕什么。」刘婆哼，「手稳，饭稳。饭稳，恩不断。你要炼，也等肚子不闹。」

「今日不炼。」他说，「先寻窑口，记槽温。」

「寻就寻，别跟赖福硬顶。」刘婆叮嘱，「硬顶饿全棚。」

「不顶。」韩泥喝尽粥，「嘴留给鼻子。」

黄昏回棚，席严，坛不露。掌贴坛腹，温一线，像应「开」字。坛底轻颤，臾墟残念弱：「丹路开，别叫丹。」

「不叫。」韩泥低声，「沉腐浮，记着。」

邻床问：「丹路啥意思？」

「渣能走。」韩泥说，「能走，才有路。路在，就不滚。」

少年不懂「渣能走」，懂「不滚」，闭嘴睡了。丑时前他盘坐，四层气转，「开」字稳在鼻——稳在鼻，不在舌。册仍末席，掌写四层，鼻写丹路开。写给自己，不写给别人。

白织月远廊过，袖风冷：「杂役也开丹路？开也断不了末席。」

韩泥不抬头：「路在渣，不在册。册写末，渣写开。」

「写开有用？」她冷声。

「有用在还。」韩泥说，「还不在你眼前。」

她唇线绷一息，走了。少一句嘲。秦霜木鱼轻敲：「路开心别贪。贪，邪。」

韩泥心里记：丹路开是钩，钩通向五层，通向匿赠，通向还恩。温听雨冷香在忆里回半息，他不迎——缘苗浅，丹路在渣，不在裙。

温听雨午间远廊一眼，声清：「多一线意。意在渣，不在名。」

「在渣。」韩泥应，「各做各的。」

「各做各的最好。」她点一点头，「你走路，我看炉。看炉不是给你开炉。」

「不开也行。」韩泥说，「有意就够。」

界拍在两边，苗也不死。次日药渣场验槽，他指捻样灰，掌纹细槽温半分，像将来炉温。十堆全对，管事记「春验仍准」。铁无言说：「槽开了，废窑可寻。寻到，今日仍不许炼。」

「知道。」韩泥应，「只寻窑，只记槽。」

赖福讽：「末等也丹路？路出邪！」

他不接。刘婆又嘱：「路开了别贪着往炉里跳。」

「不跳。」韩泥说，「跳早了，恩断边缘。」

废渣巷口他立半息，记焦回与槽并一线。并一线不亮，亮在暗里。黄昏角柜锁实，细槽温着，四层稳着。邻床问：「废窑在哪？」

「还在找。」韩泥道，「找到也不炼今日。炼，等还恩那日近了。」

「还谁的恩？」

「还该还的。」韩泥说，「你别问名。」

「不问。」少年嘀咕，「问了也害怕。」

丑时廊下练半刻，掌纹细槽温，五层影淡现。他对坛：「路开了。槽在掌。五层在后，不叫。」

坛温一线，像细槽白线摁进瓮腹。他按木牌贴胸：「丹路开了。炼气五层还在后头。门缝那盏灯还远，手要先稳。」

老耿挑水过，哑声：「路开别拜邪。」

「不拜。」韩泥说，「开在渣，记着。」

管事夜巡，灯照席角，沿净，像腌菜坛。像腌菜坛，魔教才嗅不准。嗅不准，五层才冲得稳，匿赠才藏得住。

坊市方向偶有人声杂，他只听风。风在舌上，不进鼻——鼻清，手才稳。他低声道：「她断丹路，叔自己开。开在渣，舌不留。」

更鼓尽时，雨歇，药山湿。湿里藏一点窑温余韵。余韵散了，他还记着那个「开」字——字轻，路重；路重了，明日破五层才有地可站。

他最后摸一下暗袋：「袋在，就不滚。滚了，路断；不滚，五层才接得住。」

坡下规矩还是那条：试温可，占窑不可。不可里藏着可——可把渣走成路，把路走成还。他还把「未占」那一笔记实：记实了，告示才咬不到他；咬不到，丹路才走得长。

次日兽栏铲粪，铲下比昨日更实。实不是炫境，是路开了以后手更肯听话。管事弟子巡过，随口道：「手稳。」

韩泥「嗯」一声，不多接。多接像邀功。邀功的路，开不长。

温听雨远廊再过，巾上水珠未干，声清：「路开了也别进门。门里是炉，炉不认缘。」

「不进门。」韩泥说，「只走路。」

「走路好。」她看他指头一眼，「指还稳。稳着，将来才有丑丹可闻。」

丑丹二字又落耳，他记下，不追。追像求炉；求炉，铁无言那句「不许炼」就白说了。

角柜锁又检查一遍，十二包样仍实。实得像把「开」字锁在柜子里——锁在柜子里，舌上才落不下来。舌上落了，赖福就写成私炼；写成私炼，丙九全棚遭殃。

夜里邻床又问：「你那天说渣能走，走到哪？」

「走到还能还人。」韩泥道，「还之前，先把命捂热。睡。」

少年似懂非懂，翻身睡去。韩泥内视一息，五层影淡，仍远。远不怕，路既开了，影总会近。他把木牌贴胸：「明日再试鼻下清。清着，五层才冲得稳。」

瓮腹温一线，像应了「还」字。应完仍不叫丹——不叫，路才属于他。

廊下一阵夜风，把药香送远又送回。送回的那点焦，正合他鼻下那条细路。路细不怕，细而正才长。他把玉牌贴胸，牌凉掌烫：「凉的是约，烫的是路。路开了，五层在后，还恩在更后——一步一步来。」

远处有人哼坊市小调，调子轻浮。他不跟着哼。哼嘴的，走不远；闷着走的，才还得上恩，才摸得着门缝灯。灯还远，路已开——开了，就别停。停了，五层也进不来，还恩更谈不上半句真实话。
