# 第四十九章 挽月反戈

执法堂，雨后天青，堂上却冷。

林挽月跪心，声颤却清：「谣言是赵师兄与弟子所设，浴房局陷害莫长老。顾小满无辜，弟子愿证。」

堂内一寂。赵玄机面色骤白，霍然看她，像看背叛，又像看镜子。

柳青鸢坐主位，眸光冷：「林挽月，你可知诬告反坐？」

「知。」林挽月磕头，「弟子愿坐。」

顾小满立侧廊，腕上还有罚跪雨痕，红，肿，却挺直，像半块灵石撑着她。

赵玄机咬牙，声哑：「挽月……」

「赵师兄，」林挽月不看他，「慕错了人，还能改。恩错了人，改不了。」

堂下弟子屏息，屏息里炸开一层浪——浪不是欢呼，是惊，惊林挽月敢反戈，惊赵执事脸色白，惊莫长老立堂侧不言，只点头，像把「以后还人情」又说了一遍。

苏念慈在丹堂廊下听见消息，炉火旺，旺得像替谁把脏字炼净。霍镇山在镇妖司骂：「执法堂终于肯洗脸了！」

魏无涯在檐角，扇合，低笑一声，像看一枚子落对了格，又像看妒火线终于收束一截。

柳青鸢拍案：「赵玄机，思过崖半月，加重。林挽月，罚役三月，逐出外门林苑。」

赵玄机跪，膝磕地砖，响得实。他抬头，看莫长春——莫长春立堂侧，不言，只一点头。

林挽月退堂，与顾小满擦肩，低声：「对不起。」

顾小满摇头：「记着就行。」

围观弟子炸开反转，反转成浪：

「林挽月反戈了！」

「赵执事陷害长老？」

「莫长老昨夜止步不入，原来早知局！」

浪声里，脏字退潮，退出一寸清。赵玄机跪姿更直，直得像终于把「还库」之后的路，跪进了地砖。

莫长春出堂，杖点阶石，低声，像钩：

「证完了。夜袭的符，该递出去了。」

风腥，云低，子时像已贴在门上。

有人低声：「慕错人，还能改。」另一人答：「恩对路，不能改。」孙福旧党面色白，像听见「陷害」二字，也听见自己的旧恩仇将被翻。

苏念慈遣人送药至侧廊，药温，像替顾小满腕上雨痕也担。她低声：「脏字炼净，炉不能熄。」

堂上冷，堂外浪——浪退一寸，符将递出。全宗尚不知：挽月反戈，不是毁赵，是留路；留路之后，夜袭已在门外。

堂外忽有人嚷：「挽月疯了！」立刻被旁人拽袖：「疯的是造谣那张嘴，不是证字。」林挽月膝仍跪，跪得像把脏字一寸寸坐实；赵玄机咬牙，牙咬得响，响不过堂上那半息——半息拉长，拉长得像把「以后」二字也审了一遍，审完才落锤。

堂外围观渐成两拨，一拨惊，一拨疑。惊者道：「赵执事脸色白，白得像被镜子照见。」疑者道：「照见也好，照见了，夜袭才有人守门。」

林挽月声颤却稳：「我反的不是赵师兄，是私。」赵玄机铁牌在腰，笑浅，笑里空一线——戈若真，谋就浅。

柳青鸢声洪：「反戈为门，不为私。」顾小满册上又记一行，行细，细得像半块石硌深。

苏念慈闭炉，闭得像怕堂冷进丹堂。她低声：「反戈反戈，别传成叛。」

夜前，执法堂灯未灭，灭的是浴房谣最后一缕灰。莫长春不坐主位，只袖空立柱旁，像把「还礼」从堂上挪到堂外——堂外风大，风大正好把脏字吹远。

苏念慈遣药至侧廊，药温，温得像替顾小满腕上雨痕也担。她低声：「脏字炼净，炉不能熄。」

霍镇山骂罢执法堂，又骂东脉防务松，松得像等符来补。周德海在远处观望，观得像观一门「私」字还能藏多久——藏不久了，戈已响。

赵玄机跪后，孙福旧党在廊角交换眼色，眼色像还想把脏字捞回去。霍镇山从镇妖司赶来，刀未出鞘，声先出：「谁再传浴房谣，先过我这一关。」

一句话，堂外浪又退一寸。

魏无涯在檐角对风自语：「戈是刀背，不是刀刃。刀刃留给夜袭，刀背留给门内。」

林挽月退至阶下，罚役灰边在袖，像三月之罚已提前上身。她不敢看赵玄机，只看地砖，地砖凉，凉得像把「私」字刻进缝里。

顾小满递她一张薄简，简上无求情，只有细则：「想走武试路，先消罚役记。路要直，字要清。」

林挽月指尖一颤，颤得像终于摸到一点「改」的可能。

莫长春立堂侧，袖空，一念出塔，杖点地砖，声轻：「证字落了，符该递了。递符的人，不必是我，心要是门的就行。」

赵玄机抬头，眼里妒未灭，却薄了：「莫长老……昨夜不入浴房，是等我今日跪？」

「等你跪，也等你站，」莫长春道，「站不站得起，看你还库之后，还剩多少私。」

柳青鸢拂袖，执法堂灯亮，亮得像把「规」字也点上。她声洪：「反戈为门，不为私。林挽月罚役三月，赵玄机思过半月——半月后，东脉符线，谁守，谁记。」

苏念慈在丹堂听见「符线」二字，闭炉，闭得像怕堂冷进丹堂，又开一线窗，让药香散出去，散得像给全门一颗定心丸。

风腥更重，云更低，子时像已贴在门上。

顾小满收册，册厚一寸，厚得像把「证路」二字压实。她望林挽月背影，背影瘦，却直，直得像终于从「慕」里抽出一根脊梁。

孙福旧党散去，散得像潮退，退后滩上还留着旧印，旧印等下一阵浪来洗——浪不来浴房谣，来的是夜袭符。

莫长春望东脉，望得像望符将递出的那一息。顾小满在心里记下「挽月反戈」四字，又添两字：「证路」。

她低声问：「长老，符真递吗？」

「真递，」莫长春道，「递符如递茶，茶烫，符更烫。烫的不是手，是胆。」

赵玄机被执事扶起，扶得像扶一面倒下的牌——牌面写着执法，牌背却藏着私。他走过林挽月身侧，停半息，终是哑声：「路……你自己走直。」

林挽月点头，点得像把慕字从颈上摘下，换了一枚更轻的「记」。

围观弟子里，有人开始改口，改口比反戈慢半拍，慢半拍也是进步：「昨夜莫长老不入浴房，不是怯，是等今日这一跪。」

「等跪，也等戈，」另一人道，「戈响了，门才像门。」

魏无涯在檐角收扇，扇骨合拢，像合拢一局妒棋。他笑浅：「戈是刀背，罚是绳。绳小，门大。」

柳青鸢立执法堂阶上，望东脉，声低：「符线要直。直了，夜袭才像试炼，不像乱。」

莫长春袖空，杖在塔中，念：「戈响，罚至，符出——三步连一步，一步都不许歪。」

顾小满收册入怀，怀温，温得像又揣回半块灵石——石未送完，路已更直。

挽月反戈那日，执法堂冷，冷里像有慕字碎成清。赵玄机走过林挽月身侧，停半息，终是哑声：「路……你自己走直。」

莫长春不坐，只记，记像记礼——礼还，堂才静。风腥更重，云更低，子时像已贴在门上。

柳青鸢声洪：「反戈为门，不为私。」顾小满册厚，声平：「戈记下了。证路也记下了。」

苏念慈闭炉，闭像怕堂冷进丹堂。风腥更重，云更低，子时像已贴在门上。

风腥更重，云更低，子时像已贴在门上——门上贴着下一纸罚。

围观弟子里，有人开始改口：「戈响了，门才像门。」孙福旧党散去，散得像潮退，退后滩上还留着旧印，旧印等下一阵浪来洗。

挽月反戈那日，执法堂冷，冷里像有慕字碎成清——清完，赵罚将至。
