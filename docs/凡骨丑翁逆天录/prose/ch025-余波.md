# 第二十五章 余波

公审第二日，泥岗村像退潮。

潮退，滩上露的是旧泥，泥里还嵌着昨日嚼过的字：晦气、牵扯、绝户。字嵌着，嚼不动了——嚼累了，嘴也酸。

赵绣春门闭，闭得紧，妇人路过也不围了。围了半辈子戏台，戏台塌一角，塌在叶青禾那句「谁没受过一碗饭」上。饭字轻，压得人哑。

周嫂洗衣，棒槌轻了半分，轻得像怕再砸出「汤是邪」来。水仍浑，浑里她骂自己小声：「砸多了，手也疼。」

叶家药铺开门。

秤声稳，药气苦，苦里带甘——甘是活路。叶青禾在柜后，不抬头，不辩，只称药。称得准，比辩十句硬。硬，公审才真落了地。

韩泥担筐过巷，筐空，去集尾。村人让路，让得别扭，像让晦气，又像让「公审赢了一半」的人。让半步，也是松气。

马春花啐远，啐给自己听：「疯子……说得倒也在理。」在理二字，她不肯吞全，吞一半，够她今日不堵叶铺门。午后，沈枯芽送一碗药汤到叶铺，汤不进门缝，进门。进门，公审才真落了地。

张麻婆骂街，骂到一半改口：「叶丫头硬，硬得像我这馒。馒热，心热，热不过诬人嘴。」

陈姑路过，棉袄边还新，新得像冬恩还在。她看韩泥，不看脸，看筐：「筐编得稳，人便稳。稳了，谣追不上。」

韩泥「嗯」一声，不谢。谢轻，恩重。

集尾，筐十二文，价稳。稳，像叶家秤。买筐人嘟囔：「晦气翁筐，倒结实。」结实二字，韩泥入耳，当夸，不当骂。回村，他站巷尾，望叶铺灯。灯白日也亮，亮得像松气——松的不是他，是全村绷了十几日的绳。绳松，人不松；人松，恩就散。

他低声道：「气松了，恩没松。」

心口叶汤深，深得像公审那日记的。他不增笔，只按。按实，像按叶家脊梁还在。脊梁在，省亲那日才有路。

坛角微凉。坛腹微温。温里，他听见远处赵家门内一声摔碗——摔得脆，像认输，像不服，像两者兼有。兼有，才真。

里正巡巷，咳一声：「少嚼。嚼多了，连坐。」连坐二字，仍悬，悬得像旱。旱在，谣在，恩也在。

里正又补一句，声沉，沉得像钉：「公审过了，谁再聚众辱人，先记名，再报县。县吏来，不比今日温和。」

村人噤，噤得像退潮。潮退，滩上露旧泥，泥里嵌字，字嚼不动了。

韩泥回屋，编筐，指稳。稳，是给明日活路。活路通向沉丹宗风声，通向辞村，通向还。

外头风软，软得像公审余韵。余韵里，叶家秤声又起，起一声，村心便安一寸。安一寸，够暖。傍晚，里正又巡，巡到叶铺，咳：「公审过了，秤别停。停秤，比哑口还毒。」

叶青禾在帘内：「不停。秤在，人在。」

里正「嗯」一声，转向韩泥屋方向，却未近。近晦气翁，像沾边。他只扬声：「韩泥，少生事。」

韩泥开门，独眼平：「我生事？我编筐。」

里正瞥他心口，瞥见叶汤那笔恩深，愣半息：「心收好。收不好，也是祸。」

「恩在心里，祸在嘴。」韩泥说，「嘴我管不了，心我管得住。」

里正无话，走了。走，也是松气——松给全村看，公审不是结局，是缝；缝上了，日子还得过。

韩泥回屋，坛角微凉，坛腹微温。温里，他听叶铺最后一声秤落，落得像钉，钉住「叶家还在」。

他低声道：「松气是松气，叶汤不能浅。浅了，省亲那日便无根。」

根在泥岗，枝在宗门。枝要长，根不能断。断，汤就凉。

赵绣春门缝探头，探头，像不甘：「松什么气？天还旱！」

张麻婆啐她：「旱是天，汤是人。人还在，你少泼。」

赵缩回，像又哑半寸。半寸，够韩泥记，不增笔，只按叶汤。

叶铺帘内，药杵又响一声，响得像秤还在量活路。活路量给全村，也量给末席那个编筐的绝户翁——翁丑，路不丑。

沈枯芽送一碗粥到叶铺，粥稀，热，热得像公审后第一碗「正常」——正常不重，重的是叶家还在。

韩泥路过，不近，近像牵扯。他只望秤杆，杆稳，心便安一寸。安一寸，够编筐，够活，够走到荐帖那日。

他低声道：「公审过了，谣没死，嘴还在。嘴在，心别软。心软，恩就浅。」

坛腹微温，温得像叶汤还在。还在，省亲那日才有根。根在泥岗，枝在宗门。入夜，叶铺帘内药杵停一息。

叶青禾隔帘问，声平，平里带倦：「公审过了，你筐还编？」

韩泥在门外，不近帘：「编。不编，集尾空，空像认输。」

「认输给赵绣春？」

「认输给谣。」韩泥说，「谣没死，手不能停。手停，秤也停；秤停，你爹药白送。」

帘静半息。静里，秤杆轻碰一声，像盖章。

叶青禾又道：「沈枯芽那帖药，你用了？」

「用了。臂能抬筐。」

「能抬，别逞进宗门。」她顿一息，「宗门路远，远里也有嘴。嘴比泥岗尖，尖了你别硬顶。」

韩泥「嗯」一声：「不顶。顶是活，不是辩。辩赢了公审，活路还在末。」

里正巡至巷口，咳：「叶丫头，帘内少议宗门。议多了，像牵扯。」

叶青禾在帘内：「里正叔，我只说药。药不断，人不断；人不断，村不断。」

里正无话，转向韩泥：「你，少往叶铺门缝凑。凑了，赵绣春又有话说。」

韩泥独眼平：「我凑集尾，不凑门缝。门缝是她的，筐是我的。」

里正「哼」一声，走了。走，也是给全村看——公审过了，规矩还在；规矩在，谣便不能借规矩再咬一轮。

韩泥回屋，对坛低声：「公审是缝，不是结。结死了，恩也死；缝活着，省亲那日才有缝可钻。」

坛腹微温一丝，像答。答无声，够他明日还去编筐，还去集尾，还去等沉丹宗风声贴榜。

里正差人送抄本：「沉丹宗验骨，十日后卯时，西驿见。」韩泥瞥一眼：「十日，够辞村，够还一路恩。」叶家松气，谣还悬。不近帘——近惹闲话，远成护，护到荐帖到，护到辞村那日。汤仍记烫，烫是余波里唯一要守的字；守住了，才配上路，才配西驿验骨。

十日抄本在柱，像辞村的尺——尺在，手就不能抖，抖，路就飘。

他低声道：「手稳，恩不断；恩不断，汤就不凉。」

坛沿他仍擦。擦时指节白，白像证：辱在册，恩在胸，分列不混。
