# 第一百一十章 首炼培元

丑时。

药山废窑，窑口红得稳，稳得像昨日余韵落地。落地了，才配首炼——炼的是培元散。散在一品，品在路，路藏在倒渣里。韩泥把门帘压严，外面听不见里面，里面也看不见外面。看不见最好——看不见的东西，赖扒皮才难揭。

他分拣，指稳。青芯五撮，褐渣三把，灵泉七滴，投序照经页。页深，深的是散，不是丹。丹在赠叶那一夜；散在这一夜，自用，才是丹道起点。

炉温起一寸，一寸稳。白气聚，聚而不散——散了就败。败过，这一回要成。成不成，在鼻下。嗅诀一动，像秤：秤香、秤色、秤「正」。正来得慢，慢才正。

正来一瞬，窑里无雷无光，只有掌纹那一路烫。烫像叶汤碗沿，像门缝灯，像清瘴并肩——手稳，才成散。

散成灰白，细得像雪，雪像药。药入粗纸包，包丑——丑才像杂役该得的。得不在名，在正。他嗅半息，半息够自用，够备后头更远的挂牌，够还下一笔恩。

他不自服。服像抢破境；破境不在这一章，远着。这一章只起点——起点在散，散在废窑浅穴。浅穴他占住了，不声张。声张，赖扒皮就来。

铁无言立窑外，不进来，声低：「成了？」

「成了。」韩泥说，「散，不是丹。」

「正？」

「正。」

正一字，铁无言点头，点像证。证不在嘴，在鼻。

下山遇管事弟子。弟子验包，包灰，灰里白细。管事不验名——名还远，「丑丹师」三个字还没到日子。赖福堵路：「又上山？上山成什么？」

「倒渣。」韩泥声平，「手稳。」

赖福鼻动半息，没再追。散在袖里，袖扎紧。

回棚，席下按坛，低声道：「首炼过了。散在，路在。」

坛腹温贴掌，像应「起点」二字。起点不是终，终还远。刘婆端粥：「兽潮后，你还炼？」

「炼。」韩泥说，「炼了，才还得了恩。」

叶青禾在坡下，门缝灯未亮。未亮像避；脉里培元丹仍温——那是赠叶一笔；这一笔是散，两笔不搅。搅乱了，账就乱。

邻床问：「你真成丹师了？」

「不成。」韩泥说，「成路。路在散，散在渣。」

秦霜木鱼轻敲：「首炼别骄。骄了，手飘。」

他不应口，心里只记：不骄，只记正、记起点、记往后还有压。沈枯芽悄悄问：「散能破境吗？」

「不能。」韩泥说，「散是路，不是境。」

「那炼它干什么？」

「走路。」他声平，「路走稳了，境才跟。」

同棚有人笑：「丑翁又从山上背灰回来了？」

「灰正。」韩泥道，「正灰也能养人。」

笑停了。正字一出口，闲话就短。

铁无言后来又堵在棚口：「真正？」

「真。」韩泥把纸包在袖里按了按，「你闻不到，是我扎得紧。」

「紧得好。松了，赖扒皮抢着当粪。」

「不当粪。」韩泥说，「当下一笔。」

管事弟子傍晚再路过，随口问：「山上可有异香？」

「无。」韩泥说，「只有倒渣味。」

「有异，报。」管事丢下三个字就走。三个字像刀背——没砍他，却提醒他：藏住。

夜里他再摸袖里纸包，包丑，味正。正得像把一整条丹道从渣灰里拽出半寸。半寸够了——够把他从「倒渣」托到「有散」，又还不至于张扬到惹人夺。

白织月袖风过廊，冷冷一句：「有东西别晃。」

「不晃。」韩泥说，「藏着。」

「藏住了才像活。」她去得快，像不愿多认。

柱角雨痕还在，他走过时数息。数息里债务仍在：刘婆、叶汤、并肩、三年后的账。账多不要紧，散在袖，手就有底。底有了，明天赖扒皮再堵路，他也只回三个字：倒渣、手稳。手稳背后，多了一包正味，正味不给人闻，给人还恩。

更鼓尽，废窑温退，退像收敛。收敛为下一程——赖扒皮压、炼气六层往上、坊市再挂牌。他低声道：「培元散，正。丹道，起点。」

起点在丑手，不在名；名在远，路在掌。掌在，就不滚。

邻床又嘀咕：「你今晚香落怎么匀？」

「匀才活。」韩泥说，「飘了，散白炼。」

「你真不吃那包？」

「不吃。」韩泥道，「吃早了，路断。路不断，才还得完。」

他还不起大的，先把小的按实：散正、手在、账不混。不混，七笔才排得开。

天将亮时，他出棚望了望坡下。坡下雾白，门缝仍黑。黑得正好——正好让他把喜按住，把「起点」按进掌茧，不按到人面前。

散成那一刻，窑里其实静得可怕。可怕不是邪，是太正——正到他反而不敢出声。出声像把喜抖出去；喜一抖，味会跟着飘。味一飘，半山都能闻见。他便只做三件事：熄火、包纸、扎袖。三件事做完，人才重新变成倒渣杂役该有的灰脸。

下山小路湿，湿得脚印深。铁无言跟在后头两丈远，远得像不认识。不认识正好。正好撞见赖福时，赖福只看见一个人，看不见一笔。

「袖里鼓什么？」赖福突然伸手。

韩泥早有防备，袖里转的是空渣袋：「鼓灰。要不要闻？」

赖福真凑近闻了，闻得一脸鄙夷：「臭。滚。」

「滚。」韩泥顺着他的字走，走得干脆。真货在另一只袖，另一只袖贴着肋骨，贴得像第二层骨。骨在，就不怕骂。

回棚后他把纸包塞进席下与坛之间的缝。缝窄，包丑，正好卡死。卡死了，他才敢去喝粥。粥入口，忽然觉得这条路终于从「闻」走到了「炼」——闻是杂役本事，炼是丹道门缝。门缝开一线就够；开大了，会有人抢着把门拆了。

沈枯芽夜里又来，眼睛亮：「真成了？我闻着……」

「别闻。」韩泥按住他脑门，「闻多了嘴碎，嘴碎了账乱。」

「我不通报。」芽儿急。

「通不通报不是嘴。」韩泥道，「是腿。腿往热闹地方跑，热闹就来咬你。」

芽儿乖乖缩回去。缩回去才像徒弟该有的样——样不在喊师父，在肯藏。

韩泥独自坐到将近天亮，把「起点」两个字在心里写了又擦，擦了又写。写不是为骄，是怕忘。忘了会把散当饭吃掉，吃掉就断后手。后手要留给压、留给还、留给更远的挂牌。挂牌还早，压却可能明天就到。到了也不慌——袖里有正味，脸上仍是倒渣。倒渣脸好用，好用到能把一整条丹道驮过赖福的眼皮。

天亮前他还做了一件小事：把废窑门枢的吱声抹了点糠油。门不响，进出就不招耳。招耳的东西最怕深夜——深夜一响，山猫人嘴都来。他摸完门枢，低声道：「静着。静着才成路。」

路成了，也不许脚飘。脚飘会把纸包蹭松；纸包一松，正味就漏。他把袖绳又勒紧一扣，勒到指尖发木，才往棚里走。走时经过柱角，雨痕还在，像昨夜没走远。没走远也好——好提醒他：丹道刚起头，头要低，肩要沉，账要一分一分还。

棚口风一吹，他嗅到袖里正味纹丝未漏，才真正放下半口气：「成了就藏。藏住，才算真成。」

远处更鼓尾音还在，他数完三息才迈进棚门——三息够把脸上那点喜按灭，按成倒渣该有的平。平了，明天被人骂，也骂不破袖里那包真的。

雾散半寸，他摸到瓮沿。沿温，像替他留一句：今日散成了，明日还要验压、还要藏、还要还。还字拉得长，他也不慌——手在，散在，路就在。可下一笔压要从赖福册页里砸下来，砸下来之前，袖里这包灰白，得先藏成石头一样沉。
