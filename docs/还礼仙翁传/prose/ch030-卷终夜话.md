# 第三十章 卷终夜话

秋末，灵息复苏风声起。

非正式帖，是口耳相传，像风先于人到了青岚峰：五宗天试，或将重启；各宗需提前物色护法长老，年轻一辈试剑试丹，老一辈押阵护道。

柳青鸢约莫长春峰顶夜话。

月白，云薄，峰顶风硬，吹得青衫猎猎。她递茶，茶热，像把战后的松，重新系回宗主腕上。

「师兄，」她低声，「天试若开，我需护法。你能赠，不能打。」

莫长春接茶，抿一口：「我能赠，不能打。」

柳青鸢笑，笑里疲，疲里坚：「正好。你赠，我打。青岚门不缺一柄剑，缺的是还礼的路。」

她递**风声简**，简上无多字，只列五宗名、试期约、护法名额。

莫长春接简，塔影一烫，像等这一刻。

简与塔意相合，淡金字浮，缓缓成句：

**缘箓二转·善赠——师尊、柳氏、苏氏/顾氏，首赠已满三人。**

因果沉，再涨五分，累计约四十五，像东脉一季的雨，终于落进心里。

寿元行跳：六月。

莫长春瞧见，不慌：「六月，第二卷够开场。」

柳青鸢望他：「师兄，礼还没送完。」

「没送完。」他抚杖，竹纹游动，气音温柔：够。

莫长春点头：「够送，够还，够到天试门前。」

峰顶远瞧，东脉灯火像坊市烟火升上山；丹堂一线夜灯，像苏氏丹香未歇；杂役堂窗下人影，像顾小满仍在记在心里。

柳青鸢忽然道：「苏师姐今日炼培元，炉火到天亮。她没问天试，只问……师兄九月够不够。」

莫长春笑：「够。不够也会装够。」

「赵执事还了三颗私丹。」柳青鸢续道，「他说还库，不是还你。心却松了一寸。」

「松就好。」莫长春望执法堂方向，「妒松了，才接得住下一礼。」

柳青鸢沉默片刻，低声：「厉无殇三日之期，已过两日。赤焰旗还在东脉外。」

「旗在，礼就在。」莫长春道，「手帕定身是部一的名场面，部二该有人接场了。」

月下，风更大。

柳青鸢收简，起身：「师兄，回吧。峰顶风硬，九月经不得久吹。」

莫长春拄杖起立，灰袍在月下像一张旧符，符纹淡，意却满。

他念，像卷终一句，也像开卷一句：

「礼还没送完呢。」

风大，正好送礼。

塔影在识海沉，沉得像四十五斤人情。杖答：够。

他笑：「够。」

峰顶下，外门弟子尚议论天试风声，议论里怕，也燃——怕五宗试剑，燃的是青岚门终于不再只守东脉。

苏念慈在丹堂封炉，封得稳，像把卷一余悸也封进瓷瓶。她望峰顶方向，低声：「师兄，六月……够炼几炉了。」

赵玄机在执法堂抄门规，抄到「护法长老」一条，停住，墨滴成字，像莫长春那句「我能赠，不能打」又浮上来。

顾小满在杂役堂记在心里，册上多一行：东脉妖讯，赤焰旗，七日之期——她不懂天试，却懂「记」字，记得像还恩。

卷终夜话散时，月过屋脊，全门像被同一句话洗过。

柳青鸢佩旧剑，声低：「师兄，卷二更险。」莫长春道：「险才好送。送出去，才收得回。」

苏念慈开炉，炉旺，旺得像把夜话炼成丹香。她低声：「夜话夜话，别传成私。」

赵玄机立远，铁牌在腰，像听，又像等下一句能踩。顾小满册上合页，页合，合得像卷一终于硌对。

魏无涯在檐角收扇，收得像收一局棋。他自语：「够送，够还，够到天试门前——好记。」

下山途中，莫长春在岔口停半息，望丹堂、望执法堂、望杂役堂，像望三条还将收礼的线。峰顶风硬，硬到把「六月」二字吹进每个人袖里——袖里各有怕，也各有燃。

外门弟子围在峰下，问：「天试风声，是真的？」莫长春不答真假，只答：「真不真，六月见。见之前，先把东脉守住。」弟子一愣，愣里像把「守住」二字记进比风声更实的册。

苏念慈开炉至天明，炉火映窗，窗纸红，红得像把卷一最后一夜也炼进去。她低声对火：「卷二更险，险在礼还没送完——送不完，就继续炼。」

赵玄机抄到「护法长老不得私离阵」一条，墨停，停得像看见自己将来要守的门。顾小满合册，册厚，厚得像卷一终于硌对，硌得半块石也温。

回院途中，莫长春在丹堂外停半息，隔门听炉细响，响像苏氏把「六月够不够」炼进火里。他不进门，像卷一收束：重意留在门外，轻话留在峰顶。

赵玄机立执法堂阶上，听「缘箓二转」风传，指节叩铁牌，一声，像把妒与服又绞在一起。绞成更硬的墨，墨里仍留一寸空——空处像还库那日挪出的，挪向哪边，尚说不清。

外门有人把「善赠已满三人」抄成册，册厚，厚得像把卷一也抄进去了。抄进去也好，抄的是路，不是秘籍。

霍镇山在峰下等候，粗声问：「天试风声，莫长老咋说？」莫长春答：「说够。够送，够还，够到门前。」霍镇山咧嘴：「好！够字比剑硬。」

林挽月罚役途中，听「缘箓二转」，唇动，像想说赵师兄与此有关，终未近。她把一枚新制符纸塞门缝，符薄，意厚，厚得像「别先算」三字未写，却已贴进纸里。

孙福在角隅听「善赠已满三人」，面色白，像听见自己的旧恩仇将被翻。顾小满冷眼：「你看路，别看嘴。」

周德海在长老院听闻「缘箓二转」，令旗在指间转，转得像犹豫，犹豫得像信使已在后巷候过三次。

莫长春袖空，一念出塔，杖点石阶，不辩，只留一句：「老夫还没死，别急着写遗言。」

回院，莫长春在院中立半息，塔壁无新铭，袖却暖，暖得像四十五斤人情刚落袋。他念：「礼还没送完呢。」风大，正好送礼。

大因果还在天试门前，像帖，像雷劫——卷一记在心里，部二才刚开场。

魏无涯在檐角收扇，扇合，低笑：「卷一收，卷二起——起在商会，起在茶饼，起在还礼。好记。」

赵玄机立执法堂阶上，听「缘箓二转」风传，指节叩铁牌，声像把妒与服又绞在一起。

秦商言山脚驿中，已闻风声简，笑浅：「青岚卷一收，卷二该有茶。茶未到，恩先记在心里。」

夜话后，外门把「卷一收卷二开」讲成九月老头遗言，邪到莫长春自己也要笑。

莫长春袖空，一念收塔，杖落掌，声轻：「卷二起，商会女修至。修亮，价硬。」

卷一灯灭，塔影却亮，亮得像在卷二门口，先贴了一张请帖。

苏念慈开炉至天明，炉火映窗，窗纸红，红得像把卷一最后一夜也炼进去。她低声对火：「六月够不够？够炼几炉，就够送几礼。」

顾小满册上合页，页合，合得像卷一终于硌对，硌得半块石也温。

魏无涯在檐角收扇，收得像收一局棋：「下一转，要看谁还肯收难还。」莫长春回院，塔影在识海沉，沉得像四十五斤人情——够送，够还，够到天试门前。

赵玄机立远，铁牌在腰，像听，又像等下一句能踩——踩不实，实的是缘箓二转已落。

下山途中，莫长春在岔口停半息，望丹堂、望执法堂、望杂役堂，像望三条还将收礼的线。峰顶风硬，硬到把「六月」二字吹进每个人袖里。

林挽月罚役途中，听「缘箓二转」，唇动，像想说赵师兄与此有关，终未近。她把一枚新制符纸塞门缝，符薄，意厚。
