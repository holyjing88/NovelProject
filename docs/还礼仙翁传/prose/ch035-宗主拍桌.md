# 第三十五章 宗主拍桌

争论至夜，长老会厅空气像绷紧的弦。

周德海最后一推，像把全部和谈希望押在这一推上：「宗主，七年和平，够年轻一辈成长。割让三成，换七日缓冲，换丹堂七夜，换……」

「换什么？」柳青鸢抬眼，眸光冷而清，「换东脉民散？换霍镇山盾碎？换莫师兄九月，再送七夜？」

周德海笑僵：「宗主言重……」

「不重。」柳青鸢起身，袖风带起册页，册页落案，像雪落火里，「七年之后呢？把成长交给割让？」

殿内低议又起，起得像潮，潮里有人看周，有人看柳，有人看门边——莫长春仍拄杖立原处，不帮腔，像一根秤，秤砣落在「不让」那边。

周德海咬牙：「宗主，赤焰夜袭若至，青岚门扛得住吗？」

柳青鸢拍桌，板声落，震得烛焰直，像定东脉命：

「东脉不让。」

四字，无多余修辞，却比任何剑意都硬。殿内应声起，霍镇山第一个吼「不让」，年轻长老拍膝，连平日主和者也沉默——秦氏简在，手帕定在，谁还把「让」字当路？

周德海冷笑，拂袖退席，袖风带起册页一角，像一扇门悄悄合上，门外不知谁候着，候得像风，像信，像奸。

霍镇山抱拳，声洪：「宗主英明！」

莫长春拄杖立原处，不鼓掌，不喝彩，只一句，像记在心里：

「地可以给一点，根不能拔。宗主不给，是对的。」

柳青鸢看他，眼底微红又压下：「师兄，奸线加深了。」

「加深才好。」莫长春道，「线深了，才晓得谁剪谁。」

他出厅，夜风扑面，像秦氏简上「夜袭」二字迎面而来。廊下弟子听见「东脉不让」，炸开欢呼，欢呼里燃，也惧——燃的是终于主战，惧的是夜袭近了。

魏无涯在远处檐角，扇开一线，观气运，观柳青鸢背影，观周德海退席的方向，低笑一声，像看一枚子落对了格。

寿元在塔壁，五月余，浅淡，够他把下一礼送给丹堂。

苏念慈在丹堂听讯，炉火旺，旺得像替宗主也担。她低声：「不让好。丹堂七夜，不为让。」

霍镇山拾也不拾周德海落廊的册页，骂：「让字滚远点！」

孙福在角隅，面色白，像听见「不让」二字，也听见自己的旧恩仇将被翻。林挽月立廊外，听「东脉不让」，唇动，像想说赵师兄的局与此无关，终未近。

赵玄机在思过崖，听拍桌声远传，指节叩石，像把「不让」二字，叩进将来要守的门里。

周德海退至长老院，令旗在案，迟迟不落，落得像犹豫，犹豫得像信使已在后巷候过三次。

顾小满在杂役堂值夜，听「东脉不让」，指节白，像半块灵石在怀，温了一路。她不知暗线将走，只知不让字一落，杂役也不能退。

秦商言飞符至驿，符上四字：「不让好。」像商言也懂——根在，恩才有路走。

拍桌声远，远得像战鼓第一响——响完了，没人再谈让字。

他出厅，廊下弟子围他，问：「莫长老，不让之后呢？」莫长春笑：「之后该烧炉，该备符，该让该守门的人守门。」

宗主拍桌那声，比表册还响，响得像把东脉钉进每个人心里。

丹堂方向，苏念慈隔窗听闻拍桌，炉火星子一跳，像应了一声「不让」。她将丹方残卷重新压进匣底，匣盖落响，响里像藏了七夜炉火的命。

赵玄机在思过崖，指节叩石三下，像在表上划了一道：东脉不让，便是赵氏该守的门。

周德海退至长老院，指节发白。廊外信使脚步极轻，轻得像怕惊动「东脉不让」四个字——字已落，信该出了。

莫长春念及秦氏简上「夜袭」二字，望东脉，风腥扑面，却笑：「让字退了，礼字才进得去。」

柳青鸢佩旧剑，声洪：「事真，阵真，人不假。」霍镇山按刀：「不让，就战！」

莫长春袖空，一念出塔，杖点案角，不撕，只记。声轻：「拍响，页要来。丹方残页，轻里藏丹。」

拍桌声落，烛焰直，直得像把「不让」二字也钉进梁上。周德海拂袖出殿，册页落阶，像和字滚远。

夜袭符影在妖讯柱上晃了一下，又稳，稳得像提醒「不让」之后的第一仗不在纸上。

霍镇山连夜点兵东脉，盾阵试演第三遍，演得像给拍桌声擦刃。年轻弟子问阵眼，霍镇山答：「阵眼在田埂，不在议事厅。」

孙福缩在角隅，听「不让」后杂役清单翻倍，面色更白——白得像听见旧克扣要被宗主令追着算。

林挽月在廊外站了许久，终是不敢进殿。她把罚役时捡的一片枯叶塞进门缝，叶薄，像替赵玄机留一条退最深的退路的警示：别让字退尽，门里无赵。

魏无涯观柳青鸢背影，扇尖一点，像落子：「让字退了，奸线该现形。」

柳青鸢独自回殿，将「东脉不让」四字写入宗主令，笔锋压深，像把拍桌声也压进纸里。她低声对执事：「丹堂七夜，杂役加倍，东脉巡哨，今夜起。」

周德海在后巷接信使，信使袖湿，湿得像汗也像夜露。周德海只问一句：「赤焰那边，可知道不让？」信使点头，点头像把奸线先递出谷外。

赵玄机在思过崖，听拍桌余音，忽然觉得妒火里掺进一丝空——空不是败，是门里终于有一声齐。他指节叩石，叩得像把「不让」先存进将来要守的门。

苏念慈隔窗听闻拍桌，将丹方残卷压匣，匣盖落响，响里像七夜炉火已先应了一声「不让」。她低声：「不让好。炼不为让。」

拍桌后第一更鼓，杂役堂灯火比平日亮一倍，亮得像东脉不让，杂役也不能暗。

长老院外，年轻弟子自发列队往东脉方向拱手，拱手像把「不让」先走成阵。霍镇山瞧见，骂：「阵在田，不在摆样子！」骂完，仍令盾试第三遍。

周德海后巷信使出谷时，袖中多了一封无署名的简，简薄，薄得像奸线先探了路。魏无涯在远处瞧见，扇合，低笑：「让字退了，信该出了。」

莫长春望丹堂方向，声轻：「拍响了，页该来。七夜炉火，等残页。」

执事捧宗主令草案至丹堂，苏念慈接令，不接话，只将丹方残卷又压深一寸——压得像把「不让」也压进炉火里。她低声对丹童：「七夜起，火不能熄，熄了，残页也活不了。」

魏无涯檐下观气，见周德海袖气缠奸，缠得像信已出谷。他扇合，低笑：「拍桌定了调，下一页，该丹方了。」

赵玄机在思过崖，听拍桌余音，忽然觉得妒火里掺进一丝空——空不是败，是门里终于有一声齐，齐得像赵氏该守的门，也在「不让」里。

顾小满在杂役堂值夜，将「东脉不让」抄进册，笔锋压深，像把半块灵石又硌了一遍。她低声：「不让字一落，杂役也不能退。」

霍镇山连夜点兵东脉，盾阵试演第四遍，演得像给拍桌声擦最后一道刃。年轻弟子问：「阵眼在哪？」霍镇山答：「在田埂，不在嘴上，更不在让字上。」

桌响，堂静，静里像把丹方残页四个字，先钉进每个人眼里——眼里有火，火等七夜，夜等残页。

弟子渐散，散里仍回头瞧一眼灰袍袖空——瞧得像瞧一眼九月还能走多久。
