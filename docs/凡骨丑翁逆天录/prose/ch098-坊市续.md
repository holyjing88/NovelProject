# 第九十八章 坊市续

第二日，「丑丹师」那牌仍歪着。

歪不要紧，歪得像别人家的物件。人丑，牌丑，手稳，坊市才肯续。续不在名，在客。

客里来了个穿灰袍的老客。老客不说话，只递上五包渣，包上系着红绳——绳旧，像在试人。

韩泥一包包嗅。嗅到第三包，鼻下忽然停半息：「这包有霉，霉在角上，别投炉心。」

老客眼睛一细：「你怎么知角上有霉？」

「嗅。」韩泥说，「嗅完，这包不取钱。」

老客收包，收时塞过来一块下品灵石碎片。碎片小，小也不能收；收了像受贿。

韩泥推回去：「半文就够。够，是试，不是营生。」

老客看他半息，走了。走像记下了一笔。记不要紧，记的是手，不是名。

铁无言午后过来，声低：「那老客是药峰管事身边的人。你推石，推对了。」

「推。」韩泥应。

推不为清名，为续摊。摊续住了，才能听见兽潮风声一天比一天近。

白织月第二回来，这回带着同门。同门笑：「丑丹师，嗅嗅我这块是不是假灵石？」

韩泥不接：「不嗅石。只嗅渣。」

同门没趣，没趣就少半分谣。白织月袖风扫过摊面，冷冷道：「渣师傅规矩倒不少。」

「规矩半文。」韩泥说，「半文买鼻，不卖戏。」

她唇线一绷，带着人走了。

管事弟子第三日来，簿上无投诉，便记：「试三日，暂留。」

暂留够了。韩泥当晚就撤下「师」字，改成「嗅」字——字小，小才藏得住。

赖福立在坊边盯着：「暂留？留你晦气？」

韩泥收牌：「留嗅。嗅完，仍倒渣。」

赖福没掀摊。没掀也是风变了——风里兽潮更近，近到柱上新帖一条。

条丑，字真：「南荒兽聚，药山戒严，杂役夜禁。」

夜禁他本不怕；他丑时本就在窑边炼，不在坊里晃。晃的是心：心要记条，记障，记栏先倒。

回棚，刘婆端粥：「坊市还去？」

「去。」韩泥说，「听风。风里有兽。」

刘婆脸白半分：「兽来了，别逞能。」

「不逞。」韩泥说，「手稳，栏清。」

邻床慌：「夜禁了，清栏怎么办？」

「白天清。」韩泥道，「夜里守坛。」

守坛，也守住坊市续来的那一线认——认丑手，认末席，认嗅了不飘。

同袍问：「你真嗅出角霉？」

「嗅。」韩泥说，「不取钱。」

「傻？」

「末席该傻的地方，得傻。」韩泥道，「傻完手还在。」

丑时廊下静。粟壳香淡，香里混兽栏腥和坊市尘。混着，才是他的日子。

他低声对坛：「坊市续了。下一程，兽潮。」

坛腹温，像应「续」字。续不是名；名在远，活在近。

柱上条在风里猎猎响。响像催备——备手，备渣，备清瘴那条路。

铁无言临走又丢半句：「推石人对，别得意。得意，手飘。」

「不飘。」韩泥应，「飘了，暂留也飞。」

「知道就好。」铁无言道，「条到了。条到，耳要比嘴勤。」

韩泥记下。夜里他仍去窑口看火色，只看，不炼。炼早了惹腥；腥了，赖福鼻子尖。

沈枯芽悄声：「叶姐灯还亮……」

「亮就好。」韩泥说，「亮归亮，别近。近了连累。」

「我懂。」芽儿缩回去。

韩泥摸玉牌，牌凉贴胸。凉着提醒他：三年约还远，兽潮却近。近的先挡；远的先记。挡和记分开，才走得长。

第三日傍晚，有个老客远远又看了一眼摊。没买，也没笑。没笑比笑沉——沉得像认手。认手轻，轻不要紧；轻到簿上「暂留」二字，就够他把「嗅」字挂稳。

白织月走廊上远远一句：「丑丹嗅？越改越小。」

「小才藏得住。」韩泥隔空应了一句，不知她听见没有。听见最好；听见也只当风。

赖福夜里查棚：「夜禁了还炼？」

「不炼。」韩泥说，「守坛。」

「守你晦气。」赖福骂完走。走时火折一晃，晃见他席角净，骂声便瘪半分。

更鼓尽，他把「嗅」字牌擦干净，挂回筐侧备着。明日还去。去不是图半文，是把耳朵钉在风上——风一变，潮就到；潮未到，手先不能软。

管事弟子夜里巡棚，灯照席角，照见木牌旁只剩「嗅」字，便哼一声：「改小了？识相。」

「识相。」韩泥应。

「识相也别歇。」管事道，「夜禁起，杂役白日照样清栏。偷懒，记过。」

「清。」韩泥说。

灯移走后，邻床小声：「暂留算过关吗？」

「算没被撵。」韩泥道，「撵不走，才能听条。条在，潮就不远。」

「你就不想多赚几文？」

「多赚招秤。」韩泥说，「秤一来，鼻子先毁。鼻子毁了，半文也赚不光。」

少年叹口气睡了。韩泥却睡不实——不实不是慌，是耳朵尖。尖着听廊外脚步：赖福的散，铁无言的稳，远处偶尔一串跑动，像有人在搬障草。

他起身到廊下站片刻，风硬，布条在袖里发涩。涩得提醒他：坊市续上了，不代表活宽了；宽的是名，紧的是命。名小一点好，命紧一点也认——认了，才把障草那条线接到前头。

老耿挑水路过，低声：「听说你推了灵石？」

「推了。」韩泥说。

「推对。」老耿道，「石进袖，祸进门。你这丑脸，装不起那种亮。」

「装不起。」韩泥应，「半文亮就够。」

老耿咳着走了。咳声远，话却留：亮不是福，稳才是。

回席后他把红绳那五包里记过的霉味再默一遍。默不是恋功，是防忘——忘了角霉，下次炉心一焦，就是别人的命和他的簿。簿上「暂留」才写上，别自己墨未干就糊掉。

白织月方面的嘲他不再排场里接。接一次，添一分戏；戏多了，摊就不像试水，像唱戏。唱戏的人坛不温。他要坛温，不要掌声。

他把「嗅」字牌又擦了一遍，擦到木纹露出来。露着丑，丑得诚实。诚实挂出去，客人才肯递包；递包了他才闻得见真假。真假闻清，续才续得值。

铁无言临午夜又现一瞬：「柱上条，看见了？」

「看见。」韩泥说。

「看见就备。」铁无言道，「备不是喊。喊给赖福听，听了更讹你逃。」

「不喊。」韩泥应，「叠草，运灰，守角。」

「角的事到前夜再说。」铁无言顿了顿，「今夜先睡。睡足，鼻子才灵。」

他这回应了声，倒头真睡了半息。半息够把心按平：坊市这一截，算是续上了。

回棚路上，药香混着兽腥，像两股绳拧一起。他不躲腥，只把「嗅」字牌抱紧。抱紧了才晓得：续摊续的不是面子，是一条还能站到潮前的路。

管事簿上「暂留」两字他不敢看第二遍。看多了像求。求字一出，手就飘；手飘，角霉那种准头先丢。丢了准头，半文规矩也守不住。

刘婆丑时又嘱一句：「夜禁了就别野。野出去，潮没咬你，巡夜先咬你。」

「不野。」韩泥说，「守席，守坛，守明日再去的摊。」

他摸暗袋，袋里渣种沉，沉得像还有用。有用就别扔——扔了，潮中清栏缺一把灰，缺一把就可能缺一条命。他把袋口勒死，勒得指节发白。白着提醒他：续摊不是歇脚，是换气——换过气，才接得住柱上那条兽潮令。令在，耳就得比嘴勤；嘴勤了，半文规矩先毁。规矩一毁，暂留两字也就站不住脚根骨。

他独眼望界柱，条角猎猎。猎猎像锤子尚未落下，声音先到。声音到了，他就知道：坊市这一截算续上了；续上了，下一槌便是兽潮。槌落之前，他还要把鼻子养稳，把半文规矩守死，把恩和潮分开搁——分开搁，才还得起该还的，也守得住该守的栏。
