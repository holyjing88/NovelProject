# 第三十六章 丹方残页

莫长春从旧藏取出一页残纸，纸黄，边焦，像劫火里逃出来的命。

丹堂，苏念慈正在理药材，见他入，手一顿：「师兄？」

「给你。」他把残页递过去，「上古丹方残页。能不能破金丹，看丹堂，看你。」

苏念慈接过，指尖抖，抖得像捧的不是纸，是一炉命：「师兄……此物可破金丹。宗门谱上，标为禁品残卷……」

「禁品，」莫长春道，「禁的是乱用。你炼，是稳丹堂，稳青岚。」

她抬眼，目光里不止师姐看师兄，还有丹师看火，还有女人看一个把重礼推给她的人：「你……你又把重的推给我。」

「重的才还得起。」他笑，像说过千百遍的话，「轻的，才送得出去。」

苏念慈把残页贴胸，一瞬，像贴炉心：「我炼。」

「七夜。」莫长春道，「老夫送渣，你守火。渣是心意，不是废料。」

她点头，点头像立约。

第一夜子时，丹堂地火映窗，窗外影成双——一个坐门外，一个在门内拨火。外门弟子问：「长老真不进？」莫长春答：「进去了，戏就像抢。抢戏的人，还得了回赠？」

第二夜，药烟苦，苦里掺一点回春丹香，像昨夜药渣的余温还在。苏念慈隔门问：「师兄为何不睡？」「睡了，谁听炸炉？」他笑，「老夫耳朵灵。」

老农托顾小满捎一袋粗盐，盐粒硌手，像东脉土。莫长春接过，撒门槛外，低声：「盐洁路，路洁丹。」

廊外弟子窃语：「莫长老又把好东西塞丹堂？」「禁品残卷也敢给？」「苏首座敢接？」

苏念慈冷声：「闭嘴。开炉。」

火起，药香起，像一条线，从残页接到炉盖。弟子不敢噪，噪也被药香压住，像全宗都在等七夜后的那一声「成」。

柳青鸢遣人来问：「宗主问，残页可有谱？」

莫长春回：「有。在丹堂，不在嘴上。」

来人退。苏念慈听见，眸光微动，不谢，只把火拨旺，旺得像替宗主也担一分。

赵玄机路过丹堂，停半息，望门内火光，未讽，只拂袖走，走前低声对守门弟子：「执法堂轮值，闲人退三十步。」

弟子窃语：「赵执事替丹堂清场？」「还库之后，果然不一样。」

夜，莫长春立丹堂门外廊柱旁，不进门，像守门，也像陪炉。

苏念慈在门内扬声：「师兄，为何不进门？」

「进门，」莫长春道，「像抢你的戏。老夫在门口坐着，像等回赠。」

苏念慈一窒，继而笑，笑里倔：「回赠不是还债。」

「对。」他应，「所以老夫等。」

第四夜，地火阵脚微裂，苏念慈以灵力补纹，补得指尖见血。莫长春隔门问：「撑得住？」她答：「页是你送的，炉是我守的。守不住，页就真成灰。」

第五夜，苏念慈缺一味「寒髓」，寒髓在库房锁着，锁像体统也在等。她隔门问：「师兄，缺药怎办？」莫长春从塔中取一小包渣，渣苦，苦里竟有凉：「渣里有过霜华剑意，你试试。」苏念慈愣，愣完入炉，炉温一敛，敛得像缺口被补上。

霍镇山派人送盾粉至丹堂门外，粉细，像把镇妖司的硬气磨成护炉的一层皮。他不进门，只骂：「七夜谁噪，谁滚！」

塔壁无新铭，袖却暖，像残页的重量已进丹堂，进苏念慈掌心，进七夜后的那一颗金丹。

第三夜炸炉险象，莫长春门外递渣，渣稳火，稳得像心意不是废料。炉盖跳起半寸，苏念慈喝声：「别进门！」他在门外答：「不进。进了，你分神。」炸声落，药烟苦，苦里却稳，稳得像全宗都听见丹堂还撑得住。

第六夜，赤焰方向腥风一线，像夜袭在提前嗅。莫长春门外杖点地，声轻：「你守炉，老夫守门口。妖来了，也是霍老头先叫。」阶下弟子窃语里怕，也敬——怕夜袭，敬的是长老七夜不进门，像把成丹的戏，全留给丹堂。

赵玄机在远处听见，摸铁牌，牌凉，凉得像提醒：思过半月，只是开始。

孙福在角隅听「丹方残页」，面色白，像听见自己的旧恩仇将被翻。顾小满冷眼：「你看炉，别看嘴。」

林挽月罚役归途，望见丹堂七夜火，未近，只把一枚新制符纸塞门缝——符薄，意厚，厚得像「别先算」三字未写，却已贴进纸里。

第七夜前，全宗熄灯，独丹堂亮。阶下弟子屏息，屏息等延迟的那一声「成」——成不在噪，在炉；不在门内，在廊外那一杖点地的声。莫长春杖点廊石，一声，像给第七夜敲鼓。

顾小满送粥时多留半句：「杂役堂听见谣言，说莫长老要进丹堂夺功。」苏念慈哼：「让他来夺，看炉答不答。」莫长春在廊下笑：「功不在夺，在成。」

第七夜，雷意远方一线，像小丹劫将至。苏念慈隔门问：「师兄，雷来了，你还不进？」莫长春答：「进了，雷就像夺。夺雷的人，还得了你的成？」

雷落，轻，像天公肯给丹堂留面子。药香先于人声到了山门，温甜里掺雷息，像七月暑气被滤过一遍。阶下弟子踮脚，踮得酸，仍不肯退：「苏首座……快成了吧？」

顾小满送最后一盆水，水温，温得像杂役也押七夜到底。她抬头看门外长老，目光一触，像说：记着，比跪有用。

柳青鸢遣执事送阵材，材轻，意重，重得像全宗都在等七夜后的那一声「成」。霍镇山派人送盾粉，粉细，像把镇妖司的硬气磨成护炉的一层皮。

丹方残页在炉边，页边焦，焦得像从劫火里逃出来的命。苏念慈指尖抚过缺字，隔门扬声：「师兄，若成丹，我第一件事，是把回赠举给你看。」莫长春答：「举给全门看。全门看了，才记得住丹堂七夜。」

塔壁浮半句，淡金：**寿元五月**。莫长春瞧见，不慌：「五月够炼，够守，够等一声成。」

残页在怀，页轻，意重——重得像七夜炉火，已在远处先亮一线。

魏无涯观丹堂火，低笑：「重的推给她，轻的才送得出。」霍镇山闻讯，骂声洪：「丹堂七夜，谁噪谁滚！」

弟子轮值，顾小满送水，水温，温得像杂役也押七夜。她抬头看门外长老，目光一触，像说：记着，比跪有用。

阶下弟子听见七夜不进门，窃语里敬多过疑：「苏首座守炉，长老守门口——守的不是功，是成。」有人赌第七夜成与不成，赌的不是灵石，是青岚门还有没有底。

苏念慈在门内拨火，火舌舔炉壁，舔得像把禁品残卷炼成胆。她忽然懂：师兄不进门，是把成丹的戏，全留给她唱——唱成了，才是还礼。

魏无涯观丹堂火，低笑：「重的推给她，轻的才送得出。送得出，才还得起。」霍镇山闻讯，骂声洪：「丹堂七夜，谁噪谁滚！」

第三夜炸炉险象，莫长春门外递渣，渣稳火，稳得像心意不是废料。阶下弟子屏息，全宗都在等延迟的那一声「成」——成不在噪，在炉；不在门内，在廊外那一杖点地的声。

莫长春低声：「七夜。成不成，都在这一炉。成之前，谁也别替丹堂喊体统。」阶下弟子听见，窃语里怕，也敬。

魏无涯观丹堂火，低笑：「重的推给她，轻的才送得出。」霍镇山闻讯，骂声洪：「丹堂七夜，谁噪谁滚！」

柳青鸢遣人来问第七夜进度，莫长春只答四字：「等成。」全宗屏息，等一声成——成不在噪，在炉。

残页在怀，页轻，意重——重得像七夜炉火，已在远处先亮一线，亮得像等一声成。全宗皆屏息。
