# 第一百五十章 簇收钩

管事收簿时，晨鼓刚过。

册页翻开，声平：「一百四十一到一百五十，此簇记齐。下一簇——二品丹路，废窑探得青木药火。记。」

韩泥立队末，不辩。辩像急；急了手飘，飘了火不稳。

册上写得丑：耿恩送出、泥纹名起、疤鬃驴在、十七层实、坊名续、叶信收、周家风只听——写齐了，像把这一截路按进掌茧。

铁无言声低：「青木药火在前。别躁。」

「不躁。」韩泥说，「躁，火不稳。」

「火要稳温。」铁无言道，「温在你鼻子上。鼻子一飘，窑就废。」

「不飘。」韩泥应。

刘婆端粥，仍多半勺：「此簇收了，别空肚。空肚逞能，人先散。」

「不空。」韩泥接粥，「势够才吃得下。」

「势够？」刘婆斜他一眼，「够了别嚷。嚷了，外头人当你吹。」

「不嚷。」韩泥说。

沈枯芽在远处举起半片枯叶，叶黄，像提醒：叶汤还在前，药火在远。韩泥略一点头。点头不喊谢。谢过了脉，喊嘴就多余。

赖福最后哼一声：「收簇？收完你还不是杂役！」

「杂役能背药。」韩泥道，「背到窑前再说。」

「油嘴。」赖福骂远了。骂声远，远不过坛腹那一线温。

丑时他练鼻下静。香落齐。他把这一簇按进心里：备材、送粒、耿波、泥纹、驴稳、层实、名续、叶线、风紧——九笔落地，一笔钩火。

「簇够了。」他望坛，低声道，「青木药火，在前。」

坛腹温，贴掌，像应「势」。势够，才探得废窑；火在远，手在近。近的才要紧。

管事弟子又扬声：「下一簇开：二品丹路铺展。有耳的记住——火未到手，别自称丹师满贯。」

「记住。」韩泥说。

邻床少年挤过来：「韩叔，青木药火是啥？听着烫。」

「烫。」韩泥道，「烫才有用。用在控火，不在嘴上吹。」

「你怕不怕？」

「怕。」韩泥老实答，「怕也去。怕跑了，势白积。」

少年摸摸下巴：「那周家风呢？还听？」

「还听。」韩泥说，「听不架。架留给该解的夜。」

白织月远廊过，袖风冷：「收簇也像演戏。」

「演戏费妆。」韩泥声平，「我只走路。路走到火边，才算第二部真起来。」

袖风走远。韩泥不追。追闲话像扔势。

夜里他编筐，藤刺浅扎，疼得清醒。清醒里排序：药火、筑基丹、雨夜解围——三笔在远，谢老耿那半笔已还。还半笔的人更要稳：稳了，才能送；送得出，才敢谈还尽。

铁无言推门：「明日药峰照旧？」

「照旧。」韩泥说。

「照旧里藏火。」铁无言丢下炭渣，「炭你留着。留着，窑门好开。」

「留。」韩泥接过。

赖福廊下过，故意撞门框：「别做梦探火！」

「梦不探。」韩泥道，「手探。手在就行。」

更鼓将尽，天边白一线。他按心口，六个字再落一遍：「汤还是热的。」

热着，帖在怀；怀沉，沉的是路，不是惧。路通向废窑，通向控火，通向这一簇走完之后更硬的日子。

药峰散队后，管事又私下问一句：「你真准备探废窑？」

「准备。」韩泥说，「势够了，不去浪费。」

「浪费什么？」

「浪费这一簇攒下的劲。」韩泥道，「劲攒了，不点在火上，就会散进闲话。」

管事沉默半息：「那你记得——火不是威风，是活计。」

「记得。」韩泥应，「活计在鼻下。鼻准，火才肯听话。」

他把疤鬃驴绳收紧，对坛也对远处的窑影：「势够了。火在前。记着。」

收簿散队后，坡下有人起哄：「下一簇探火？丑翁也配沾青木味？」

韩泥牵驴走过，声平：「配不配，鼻子说了算。鼻子没到窑前，先别抢味道。」

「口气好大。」

「口气小过火。」韩泥道，「火大，人更要小。小了才灌得进窑。」

铁无言把一块焦炭塞他袖里：「留着闻。闻熟了，真火来时不慌。」

「领了。」韩泥说。

「领了就记：二品丹路不是神话。神话你不会炼。」铁无言道，「炼的是手上死温度。」

「死温度我懂。」韩泥应，「活温度还得摸。」

刘婆追到廊下，塞半块饼：「探火前先把肚子填实。空肚人爱听见响就跳。」

「填了。」韩泥吃饼。

「填了还记着还。」刘婆戳他胸口，「汤热着。别让火把汤煮干。」

「不干。」韩泥道，「汤在前，火也在前。前头分先后——先势，后火，再还。」

管事夜里来棚外敲门：「册收齐了。你心里有数吗？」

「有。」韩泥开门，「耿恩半笔、泥纹、驴、十七、坊名、叶信、周家风——齐了。齐了钩火。」

「钩火别钩命。」管事警告，「废窑死人年年有。」

「死人多因躁。」韩泥说，「我不躁。我不去得早，也不去得飘。」

「那你何时去？」

「下一簇开了，排到就去。」韩泥道，「排到以前，照旧分拣、续摊、听风。」

沈枯芽远处举叶影，像问他还记不记叶线。韩泥比个「记」的手势。记着。记着不等于现在跑去叶府。跑去，像用火去燎人情——燎歪了，两头焦。

夜里他清点这一簇：备材起势、送粒还半、驴蹄稳、层钉实、名续住、信收稳、风只听——七样落地，只欠一口真火。欠火不是穷，是时机。时机未到，硬探会烫手；烫手了，大比帖也会跟着抖。

他按帖角，石镇仍沉：「势够了。明日起，二品路开。开了，才谈窑。」

谈窑以前，先睡。睡够了，鼻子才准；鼻准了，青木药火才肯露脸。露脸那天，再把「汤还是热的」六个字带进窑门——带进去，火才不是野火，是还路上的灯。

簇收那天傍晚，外门灯亮得早。亮得像要把这一截账照清。韩泥在灯下又翻了一遍心里的册：备材、送粒、驴稳、层实、名续、叶信、风听——翻完，掌茧发热。

发热不是喜，是势。势够了，才配听管事说下一簇。下一簇四个字：青木药火。火字落地，他胸口那块玉牌轻轻一凉，凉得像提醒：约还在，仇还远，火在近处的远——近处因为下一簇，远因为还不能躁。

铁无言站灯影里：「你要是今晚睡不着，就数驴蹄。数到睡着。」

「数。」韩泥说。

「数完别做梦娶火。」铁无言又补，「火是活计，不是媳妇。」

「不当媳妇。」韩泥道，「当还路上的灯。」

刘婆在门外骂：「少一文绉绉！睡！」

「睡。」韩泥应。

睡前他仍把六个字按实：汤还是热的。热着，帖在；帖在，明日才走得动二品路。路一开，废窑影就近一分；近一分不要紧，手稳一分才要紧。

他灭灯，听更鼓。鼓声里像有钩子轻轻一勾——勾住一百六十五，勾住青木药火，勾住他还没说出口的「去」。

去，是势够了的人才能走的步。步未迈，先把今天的簇收干净。收干净了，锤子才悬得住。

门外炭味淡淡飘过，像废窑提前递来的名片。名片不接也行，先闻着。闻熟了，势才稳。

邻床小声：「韩叔，你真去探火？」

「真去。」韩泥说，「势够了不去，白攒。」

「吓人吗？」

「吓人。」韩泥老实，「吓人才能醒。醒着探，火才听鼻子。」

他把大比帖按进怀，怀沉，沉得踏实。踏实了，明日二品路一开，他便抬脚去接那口青木药火——接下时，仍把「汤还是热的」带着。带上，火才像灯，不像灾。灯一亮，这一簇才算真收完。

坛温一线，像点头。点头不响，钩已清楚——钩住青木药火，钩住下一簇还没掀开的册页。册页未掀，人不能停。停了，汤会凉；凉了，势就散。

他推门出去时，脊背仍直。直着去接火。火未到手，锤子却已悬在半空——悬着，就是明天。
