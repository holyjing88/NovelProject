# 第一百三十章 还刘婆

外门复测，丹香起。

韩泥挂牌「丑丹师」，牌歪，字丑，人更丑。独眼半阖，鼻下嗅诀轻动，嗅药辨性已成——成在废渣里淘，淘在赖扒皮克扣里淘，淘在五年杂役里淘，淘在兽栏腥气里淘。

药渣摊前，他捻渣，渣灰，灰里一点香。香正，丹方正。

一品培元散，成。

丹成时，天色白，白得像刘婆补衣那线——线旧，线暖，线他记了五年。记从泥岗村起，记到沉丹宗，记到今日。

「韩泥，成丹！」

执事验丹，两指捏丹，对着日头看色：「正。」

正得不像丑杂役的手笔。围观外门弟子嗡一声，嗡里带嘲带惊。嘲还在，惊也在了——丑，能成丹；老，能夺魁。

白织月袖过，袖风带香，声冷：「运气。」

韩泥不辩，只捧丹，捧去比试台。捧得稳，像捧一碗不洒的热汤。

比试末项，培元散夺魁。

没有惊天动地的雷，没有少年天才的吼。只有执事扬声：「韩泥，魁。」

魁字落，落他五十三岁脸上——入门五年，炼气十三，魁。

铁无言在人群侧，抱臂，没喝采，只点了一下头。点得像认可，又像等你还下一笔。

---

刘婆在人群末，蓝布包头，眼浊，手抖，抖得像当年偷偷塞他补衣那夜。那夜冷，冷得像今日霜，霜贴在宗门石阶上。

韩泥走过去。

不走快，怕摔了怀里两只布包。一包新衣，棉实，不艳，耐穿，袖口多缝一层，耐磨；一包铜钱，养老钱，数过三遍，不多不少，够一个补衣婆活到老，不必再偷偷摸摸，不必再怕赖扒皮眼。

「刘婆。」

刘婆抬眼，慌，像怕晦气沾身，又像怕梦醒：「你……你成丹了？」

「成了。」韩泥把布包塞她手里，托住她抖的手，「衣，换新的。钱，养老。当年补衣，我记着。」

刘婆眼湿，没哭出声，唇颤：「我……我不是图这个……」

「我知道。」韩泥说，「你图我活着。我活了，还你。」

刘婆点头，点得像补衣时顶针顶布，一下，又一下。新衣布糙，糙得真，真得像恩。

围观者静。静里有人啐，有人不语。不语的多——恩还了，还得丑，还得静，静得像一碗热汤终于递到了该递的人手里。

沈枯芽在远，怯笑一下，笑浅，像芽。铁无言仍抱臂，声低：「下一笔，还谁？」

韩泥不答，答在册上。

---

韩泥回杂役屋，册摊膝。

丑翁账上，刘婆那一笔，圈旁多一捺，像新衣折角。恩因减一，减得他胸空半寸，空得舒服——舒服不是无债，是债还了一笔，还能还下一笔。

他蘸血，在册末划字，丑，深：

筑基后，必省亲。

六字，像路标，标向泥岗，标向262，标向叶丫头那碗还没还尽的汤。汤是头的，刘婆是还了的，叶还在前头。

坛在角。泥瓮簿之名，尚未全升，升在260。此刻坛温，册热，人稳。

外门大比帖，已贴山门。帖上言伯钧名，上钱戾衡影，影在字后，像仇在路后。

韩泥望帖，独眼平：「三年约，快够了。」

魁，是丹道一步；省亲，是恩偿一步。两步不抢，一步一步来。

他合册，塞腰，低声道：

「汤还是热的。」

窗外丹香散，散进风，风进袖，袖里像还留着刘婆新衣的棉味。

够。
