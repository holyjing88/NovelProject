# 第一百零二章 兽潮渐近

黄昏落下来时，兽吼不再远了。远变成近，近变成栏外草一阵一阵乱摇——摇得像有手，又像有爪，潮头先来探界。

韩泥立在东角。火把照他那张丑脸，照得疤痕发亮。丑也好，丑才没人把他派去墙心充样子。墙心要脸面，角要钉子。他宁愿当钉子。

甜腥扑面而来。他蒙紧布条，布条湿了，湿着反而咬味。袖里灰香摸到指间，指稳，手才撒得匀。铁无言一吼：

「撒！」

韩泥撒出去。东角划出一线白，白得像雪，雪里却苦。苦对了——甜毒一退半寸，栏里小兽嘶声立刻矮下去一线。矮一线，像还能喘。

赖福退了半步，退完又装作查人：「丙九！别乱撒！撒多了谁赔？」

「不乱。」韩泥声平，「撒在角上，角漏了才赔。」

赖福噎住，火折一甩走了。走时眼仍斜着看谁稳、谁飘——怕也记人，记人比骂人阴。

墙外人影一动，动得像兽。兽影贴着界蹭，蹭下来的不是脚，是瘴。瘴厚，厚就得清。韩泥第二囊再撒，囊空了也不慌，第三囊他还捏着——捏着留给压得更狠的时候，留给潮进「中」字里那几更。

铁无言侧过脸，嗓子压低：「近处了。别省。」

「不省。」韩泥道。

不省不为逞勇，为栏。栏在，丙九少一份连坐；连坐一落，活计断，恩也还不起。白织月在内墙探身，声音远远扔过来：

「外门挡！杂役别乱！乱了挡路！」

韩泥不答。答费舌，舌要留给角漏时喊人。角此刻未漏。未漏就像他手还稳——稳是丑翁半年来扫栏、背渣、忍骂攒下来的本事，不是嘴里吹的。

同袍哆嗦着问：「还要撒吗？囊不太够了吧？」

「够。」韩泥说，「够到压住。」

够不在多，在准。他鼻下一动，便知毒浓还是淡——浓，多半指；淡，省着留。省不是抠搜，是给下一更留命。

兽影退了又进，进像试界。试不要紧，试实了，明日便会真至。韩泥回棚半刻，席下按住坛沿，低声道：

「近了。撒过了。」

坛腹温了一线，像应了「近」字。近不是退的理由。退不在他路上。

邻床抖得草席响：「我听见爪刮石头……刮得刺耳。」

「听见也别乱。」韩泥说，「手稳，栏就在。栏在，人就还在。」

丑时静了半刻。香落齐了，掌纹又温一线。温像境微微动了一下，动完仍是炼气五层。五层够守角，不够充英雄——他也没想充。

赖福半夜又来，火折照他袖口：「又偷上山了？」

「守角。」韩泥平声，「角在，人在。」

「嘴硬。」赖福啐完走开。走开也好，近处已到眼前，他骂也骂不退兽。

更鼓尽时，远岭忽然起火，火一窜一窜，像信号。信号真，真在四个字上：兽潮近了。丙九棚里有人腿软往下蹲，有人骂天。韩泥独眼平着，只把第三囊又按实一分。

管事弟子沿着障巡过来，簿子一翻：「东角韩泥？囊还剩几只？」

「一只。」韩泥答，「留着。」

「留着干啥？先撒光才算尽忠！」

铁无言在侧喝住：「尽忠不是撒光。撒光了潮中谁清？簿子上写『在』就行，别教他们空袖！」

管事噎半息，记下「角稳」二字便走。二字轻，对韩泥却像钉子——钉住了，才不至于被人一口咬成「拖后腿」。

第二更草动更狠。有头小兽扑上障沿，扑得轻，矛尖还是得稳。韩泥刺退它，刺完肩酸，酸里他仍记着：第三囊不能动。动早了，压界那晚手就空。

「丑脸别挡道！」赖福又过，肘往外撞。

韩泥让半步，让完仍立回原位。立角不靠脸，靠桩。桩歪了，内墙再香的袖风也救不了栏。

白织月那头又送来一声：「听见没有？外门挡，杂役少凑热闹！」

同袍气得嘟囔：「她倒会说……」

「说就说。」韩泥道，「咱干活。活干完，谣自己短。」

夜深，甜腥一阵浓一阵淡。浓时他蒙条更紧，淡时他听草。草里有爪，爪探一下又缩。缩不是走，是量药山这道界软不软。软了，明日就灌；硬了，还得再压。

他在棚口编了几匝藤，藤刺扎掌，扎得浅。浅好——浅才醒神，醒着才不把「近」当成「已经退」。退是假的，假潮最会骗人。

沈枯芽摸黑递来半块硬饼：「刘婆说吃口。饿了手抖。」

「谢。」韩泥咀两口，饼噎嗓子，噎着也咽，「回去睡。别站这儿。」

「叶姐灯……还亮着吗？」沈枯芽小声。

「亮不亮，你别管。」韩泥盯东角，「亮着也别近。近了连累她。」

沈枯芽应了一声，脚步轻得像怕踩破夜。韩泥把饼渣拍净，手回到囊上。囊在，近处就接得住。

远岭火又亮一截。亮得人心跳。邻床又问：「明日真会冲进来？」

「冲不冲都当冲。」韩泥说，「当冲，手才不飘。」

铁无言巡到角上，看见他仍立着，点一点头：「还能撑？」

「能。」韩泥道，「下一更若压过来，你喊，我撒。」

「好。」铁无言短促应一声，像钉子对钉子。

更鼓沉到骨头里时，韩泥按了按胸口木牌。牌凉，心不凉。他低声对自己说：近了。下一更，压。压是气顶上来，气得在胸里顶着，不能先泄到嘴里——嘴闭住，手才能张开。

界外草忽然一静。静得假。假静之后往往是更近的一扑。他独眼眯起，袖里第三囊的绳结咬在指上，咬紧，像把「明日」俩字也勒住了。条上写过或至，今夜近处已经验过——验过了，退半寸甜毒不是赢家，是换下一更喘气的缝。缝还在，角就还在。

近处第三更，瘴气忽然淡一会儿。淡不是退，是换气。换气时有人松肩，韩泥反而把布条勒紧。铁无言低喝：

「别信淡。淡完更浓。」

「信手。」韩泥应，「手在，淡浓都能撒。」

同袍凑近问：「第三囊到底啥时用？现在撒完多痛快？」

「痛快死得快。」韩泥盯草动，「压过来再开。开早了，到中段你只有鼻子。」

「中段？」

「潮咬进人堆里那几更。」韩泥说，「那时候哭都没人听。」

墙心方向鼓声乱了一阵。乱完，白织月的声音又飘过来：「外门别倒！倒一个堵一条路！」

堵路的从来不是杂役，是慌。慌从脚上爬，爬到嗓子就变成嚷。韩泥不嚷，只把矛柄在掌心里转半圈，转完仍立回钉住的位置。

赖福提着记过簿转了一圈，看见东角没人逃，笔尖顿住又抬：「韩泥，近处你立着——立着就算。别以为记功。」

「不记功。」韩泥道，「记角在就行。」

「油嘴。」

「油不了障。」韩泥把湿草再压一层，「障油了，你簿子也写不出字。」

丑时末，他回棚换布条。旧条湿臭，臭里仍能闻出毒细。细要记。记进心里，比记进簿强。沈枯芽蹲门口，眼睛红：「岭上火……越来越近。」

「近就近。」韩泥换条，「近了才知道备没白做。」

「你怕吗？」

「怕。」韩泥实话，「怕也站。站完，才有明日。」

他把第三囊绳子咬在齿间试了试拉力，拉力够，才塞回袖。够了，近处这一夜就算咬住了边——边咬住，压字才接得上。火把炸了一下，炸得独眼一缩，缩完仍平。平着看草，草再动就再撒；不该撒时，只当自己是木桩。桩外那条旧预警还在柱上晃，晃着像催明日的压——压来了，第三囊才能开，开早了明日也救不回角。
