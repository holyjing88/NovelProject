# 第五十九章 赵死守

丹堂战报汇总，墨迹新干，血味未干。

赵玄机守门一夜，刀缺三口，肩甲塌，人仍立，立得像执法堂铁牌终于亮在该亮处。

柳青鸢宣功：「赵执事戴罪立功，守丹堂门，斩先锋三人，援阵眼两次。」

赵玄机跪谢，膝磕地砖，声哑：「分内之事。」

莫长春出列，拱手，当众，像当众还他一份体面：

「赵执事，人情还了。以后，继续还。」

赵玄机抬头，愣，眼眶红，不跪谢，只重重抱拳，拳硬，像把妒火敲进铁里，敲成守门。

弟子窃语反转，反转成浪：

「莫长老一剑斩金丹……」

「赵师兄血战丹堂门……」

「九月老头，深藏不露？」

林挽月立人群后，唇白，白得像终于看懂：慕错人，还能改；恩对路，不能改。

苏念慈在丹堂廊下，听见「深藏不露」，冷笑，笑里酸，酸里傲：「师兄从不藏，你们眼盲。」

霍镇山抱拳，声洪：「赵小子，今夜像个人！」

弟子笑，笑里敬，敬赵，也敬莫——敬的是恩能还，恨能收，门能守。

赵玄机立门洞，刀缺，甲塌，人直，直得像执法堂终于有一块铁，亮在该亮处。他望莫长春，目光一触，一触即分，像把「继续还」三字，刻进刀鞘。

莫长春不辩，杖点地，像点住全宗嚼舌的余音。他低声，仅赵听见：

「守门比妒人难。你守住了，比还库更难。」

赵玄机喉紧，嗯了一声，重，像剑入鞘。

顾小满在人群后，眼里亮，亮得像雨夜那道光还在。她低声：「赵师兄……也记恩了。」

魏无涯在檐角，扇合，观气运，观赵玄机背影，低笑一声，像看一枚子落对了格。

莫长春望宗主殿方向，天试风声像帖在云上。

他低声，像钩：

「守门功入账了。下一帖，该天试了。」

风过丹堂，药香与血腥缠在一起，像战收束，帖将至。

浪声里，敬与疑交织，像全宗都在改口——改口赵，改口莫，改口九月。

林挽月立人群后，望赵玄机背影，目光一触，一触即分，像把「继续还」三字，也刻进自己心里。

孙福旧党面色白，像听见「记恩」二字，也听见自己的旧账将被翻。霍镇山抱拳，声洪：「赵小子，今夜像个人！莫长老从不藏，你们眼盲！」

刀缺甲塌人仍立——立的是门，立的是恩，立的是下一帖天试。全宗改口，改的是眼；眼改了，帖还在云上写名。

魏无涯在檐角，扇合，观气运，观赵玄机背影，低笑一声，像看一枚子落对了格。风过丹堂，药香与血腥缠在一起，像战收束，帖将至——帖是天试，试在眼改之后，眼改在赵守门之后。

弟子窃语反转，反转成浪：「莫长老一剑斩金丹……」「赵师兄血战丹堂门……」「九月老头，深藏不露？」林挽月立人群后，唇白，白得像终于看懂：慕错人，还能改；恩对路，不能改。赵玄机抬头，愣，眼眶红，不跪谢，只重重抱拳，拳硬，像把妒火敲进铁里，敲成守门。莫长春低声，仅赵听见：「守门比妒人难。你守住了，比还库更难。」

顾小满在人群后，眼里亮，亮得像雨夜那道光还在。她低声：「赵师兄……也记恩了。」风过丹堂，药香与血腥缠在一起，像战收束，帖将至。

苏念慈在丹堂廊下，听见「深藏不露」，冷笑，笑里酸，酸里傲：「师兄从不藏，你们眼盲。」弟子笑，笑里敬，敬赵，也敬莫——敬的是恩能还，恨能收，门能守。

赵玄机喉紧，嗯了一声，重，像剑入鞘。魏无涯在檐角，扇合，观气运，观赵玄机背影，低笑一声，像看一枚子落对了格。

围观弟子在丹堂外踮脚，踮得像等全宗改口——改口赵，改口莫，改口九月。有人低声：「赵师兄血战丹堂门……」另一人答：「门比剑难守，守住了，比还库更难。」

堂外有人还嘀咕「九月老头深藏」，立刻被旁人喝止：「深藏的是你们的眼，不是长老的剑。」赵玄机立门洞，刀缺，甲塌，围观里那口改口气姗姗来迟——迟得像恩入账，账到账才肯改口；风过丹堂，药香压住血腥一寸，压住一寸，天试帖的影又近了一线。

林挽月立人群后，听全宗改口，唇白，白得像终于看懂：慕错可改，恩对路不能改。她低声，像说给自己：「证完了，守也完了，下一程，各自还。」
