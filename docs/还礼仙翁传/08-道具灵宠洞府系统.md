## 二、道具体系（丹药 · 灵器 · 法宝）

### 2.1 三大品类

| 品类 | 定义 | 赠礼四维偏好 | 回赠类型 | 流行参照 |
|------|------|--------------|----------|----------|
| **丹药** | 服炼即用，消耗品 | 亲密+、忠诚+ | 修为/疗伤/破境 | 凡人炼丹、斗破丹会 |
| **灵器** | 认主兵器、饰物 | 羁绊+、亲密+ | 战力/合击/剑意 | 本命剑、杖 |
| **法宝** | 镇脉/防御/一次性大杀 | 羁绊+、馈缘+ | 剧情大回赠/因果爆 | 阵盘、金钟、玉佩 |

**品阶（统一）**：**凡品·基础**(灰) → 灵品(绿) → 宝品(蓝) → 仙品(紫) → 神品(金)

### 2.0.1 攻防子类总则（灵器·法宝·符箓·阵法·灵宠·坐骑）

> **丹药**按 **功能八类**（培元/破境/疗伤/解毒/续命/护关/悟道/飞升），不拆攻防。下列 **六类** 统一增 **攻击 / 防守** 子类。

| 大类 | 攻:防 | 归类说明 | 莫长青倾向 |
|------|-------|----------|------------|
| **灵器** | **6:4** | 剑杖戈→攻；镜甲钗→防 | 赠剑多，纳绶首饰偏防 |
| **法宝** | **4:6** | 玉佩一击→攻；金钟阵盘→防 | 护关守脉多 |
| **符箓** | **4:6** | 除魔镇妖→攻；护身遁隐→防 | 保命符叠用 |
| **阵法** | **5:5** | 幻阵杀阵→攻；护山养阵→防 | 东脉守关 |
| **灵宠** | **5:5** | 雕蛟狐→攻；龟鹤叶灵→防 | 缘契预警多 |
| **坐骑** | **3:7** | 剑御雕背→攻；驴车舟→防 | 前期步行驴，后期舟 |

**硬规**：每境 **攻≥4名、防≥2名**（灵器/阵/宠）；法宝/符 **攻≥3、防≥3**；坐骑 **攻≥2、防≥2**。低谷章（练气～筑基）符/阵/骑 **防侧≥60%**。攻方名可与其他修仙作品重名（凡人/诛仙/斗破常见器名），**粗体**仍为本作主线锚点。

### 4.3 洞府升级总则（境界可选 · 赠礼驱动）

> **与破境关系**：大境界提升后，主角 **可择时升府** 至该境对应的洞府位阶——**非强制**，可暂缓继续赠礼/游历；但 **不升府则八区效果停留旧阶**，闭关与丹符器效率落后同境修士。  
> **流行参照**：凡人换灵脉洞府、斗破开修炼室、诛仙换峰——本作强调 **赠礼养阵** 而非纯砸灵石。

#### 4.3.0 升级流程（写作/程序通用）

```
大境界突破 → 【可选】升府窗口（3～7 日叙事期）
    ↓
检：主角境界 ≥ 目标府最低境 ＋ 善缘/赠礼达标 ＋ 人物线（若有）
    ↓
择：立即升府 / 暂缓（保留旧府） / 旧府赠徒·侧绶掌事
    ↓
升府成功 → 洞府品阶↑ → 已解锁八区 **同步对齐**（§4.2.2）→ 触发②③④事件
```

| 铁规 | 说明 |
|------|------|
| **可选** | 破境后不升府合法；正文须写主角 **主动选择**（「府不急，礼先送完」） |
| **不跳两阶** | 单次升府最多 Dn→Dn+1（D0→D1 例外） |
| **八区联动** | 升府后已开区品阶 = 新洞府品阶；丹房可叠丹师档 |
| **旧府处置** | 可赠顾/沈徒/侧绶，馈缘+；旧府阵眼降一档仍可用 |
| **阵眼上限** | 核心阵品阶 ≤ 洞府品阶 |
