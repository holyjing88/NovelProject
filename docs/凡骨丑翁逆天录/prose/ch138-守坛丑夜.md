# 第一百三十八章 守坛丑夜

守坛夜，坛在角，角在席下。

席压得严，坛不露——不露，才像丙九末席该藏的东西。藏不是怯，是活；活了，后面仇风才接得住。

韩泥按坛沿，沿净。净像那年席盖严实的夜：赖福眼睛进不来，嗅诀才练得稳。

鼻下静半刻，沉、腐、浮三字在息里起落。浮不要紧，沉才准；准了，往后谢老耿那一笔才送得出。

邻床翻身，眯眼：「你丑时还不睡？修仙修成夜游神了？」

「守坛。」韩泥声平，「坛在，人在。」

「坛？啥坛？」邻床顿住，「唉，别说。说像沾晦气。」

人问半句，没再问。韩泥省了舌——舌要留来数息，留来守夜。

瓮腹应温，温一线。像还刘婆那夜：那夜还恩，这夜守路。路在，就不滚；不滚，大比帖才压得住。

铁无言窗外过，脚步停半息，声低：「守夜，别出门。」

「不出。」韩泥说，「出门，手飘。」

「飘了呢？」

「飘了，渣不正。不正，挂牌白挂。」

脚步远了，像修栏那天：日里续活，夜里守根，兄弟线在，不在嘴上喊。

刘婆柴门外咳两声：「夜里冷，别冻手。手冻了，明儿闻不准。」

「不冻。」韩泥应，「冻了手飘；飘了，丹路歪。」

「歪了谁给你补？」刘婆哼，「我补衣不补命。命自己守。」

门缝塞进一小罐姜汤，汤烫，仍多半勺：「喝。守夜别空肚。」

「不空。」韩泥接罐，「空肚手飘。」

「就会这一句。」刘婆嘴硬心软，「喝完睡半息也成。睡死了才叫真蠢。」

秦霜木鱼远远轻敲：「守夜，心别邪。」

韩泥不答。答在心里。心里只静，静到只闻辛淡，不闻外头催命似的风。

管事夜巡，灯照席角，照见厚席不见坛。灯一晃又走。

「丙九末席，醒着？」管事问。

「在。」韩泥说。

「在就好。夜巡记账，别给赖管事抓空子。」

「嗯。」

赖福火折后到，照栏，照见韩泥立像桩：「大半夜站着？装仙人？」

「守夜。」韩泥说，「管事巡过，记账了。」

「记账就了不起？」赖福冷笑，「席下味道不对，我回头拆！」

「拆可得有令。」韩泥声平，「没令拆席，您担。」

赖福噎住，火折一抖，走了。桩稳，栏稳，人就不滚。

丑时末，粟壳香淡，香一字不落。他望坛，独眼平：「守过了。记着。」

坛不应字，只温。温像应「夜」——夜守，日活，根才不断。

天边白一线，药峰派活影将至。将至不要紧，手在，路在。

坡下石阶，他数息、数粒、数债序。债序在，步不飘。刘婆已还，叶汤在前；前在远，今夜只守。

内视里阶影仍稳在十五。根在，守坛就不白守——白守像熬，熬出虚火；实守像温，温住那一线。

铁无言又从窗外回一次，只丢俩字：「还在？」

「在。」韩泥应。

「在就成。」窗外人远了，像怕进门沾情。情在还，不在夜里来叙旧。

同棚少年梦里嘟囔：「钱戾衡……台……」嘟完又睡死。韩泥听见，把仇字按实一分：风声要紧了，紧在外面；里面这坛不能晃。

他再验嗅诀半刻。诀在鼻，不在舌。舌响香飘，香飘坛露；坛露，赖福就真要拆席。

沈枯芽夜里送来半片枯叶，不敢进门，只从窗缝塞：「叶姐说……脉稳些。您别回信，回了惹眼。」

韩泥把枯叶按进袖：「回去告诉她，稳就好。别来外门。」

「是。」芽儿脚步轻，轻得像怕踩醒仇。

韩泥望坛，低声道：「叶汤在前。守完，才走得动。」

坛腹温贴掌。温不应喜，应稳。稳了，明日风声再紧，他也不接骂。

更鼓沉尽。天边更白。白里他按木牌，牌凉贴胸，像把三年约重新压一遍。

门缝风硬，硬得像坊尾盯梢那道影。影还不进门，进门是后头的事——今宵只守，守到心静、沿净、人像桩。

丑时过半，邻床忽然坐起：「韩叔，外头脚步乱，是不是查夜加班？」

「加就加。」韩泥不动，「加到席上，再睁眼。加不到席上，当雨声。」

「你真定得住……」少年咽口水，「我心口咚咚的。」

「咚咚让它咚。」韩泥道，「咚进鼻子，嗅诀就废。废了，明天渣场当众丢脸。」

少年乖乖躺下。韩泥重新数息，一息一息把沉腐浮拣齐。拣齐了，掌心那点温才肯贴回来。

窗外忽然有人小声招呼：「丙九……有没有多余姜汤？」

是巡夜小役，嗓子发颤。韩泥把刘婆罐里余温推到窗台：「喝。别进门。」

「谢……谢韩叔。」小役灌两口，脚又远了，「赖管事今儿火气大，见人就骂。」

「骂让他骂。」韩泥说，「骂进门再说。门里是席，席下是命。」

小役去后，铁无言的声又贴窗纸：「姜汤分人了？」

「分了。」韩泥应，「分不走人。人还在桩上。」

「桩好。」铁无言低笑半声，「桩歪了，我可懒得扶第二回。」

「不劳扶。」韩泥掌按坛，「扶恩像讨。讨了，还不清。」

坛腹又温一线。温里像回到清瘴并肩那年——那年挡半步，今夜隔一窗，窗不进，线不断。

药峰那边隐隐有驴鸣，鸣一声又静。静得过分时，韩泥反而更醒：醒着守，比睡着赌运气强。

他低声把债又理一遍：刘婆衣钱清了；芽儿那半块灵石清了半边；叶汤热着；老耿的散还没炼。没炼就不逞能出门——出门手飘，飘了炉也歪。

「韩叔。」邻床又吱一声，「钱戾衡真有那么狠？」

「狠。」韩泥说，「狠也不接巷骂。接在台，才算数。」

「台……还远吧？」

「远近不看嘴。」韩泥摸玉牌，「看牌。牌烫，就近了。」

子时将尽，外面又起一阵靴响，响得像队列。韩泥脊背仍直，鼻下不乱。乱了，沉腐浮就搅成一团，一团就废。

「停。」门外管事喝住谁，「丙九已巡过。别扰人清修。」

「清修？」赖福嗓子远远飘来，「丑杂役也会清修？」

「账上写守夜。」管事冷回，「您有令再拆席。没令，回。」

靴声撤了。韩泥才慢慢吐出半口气。半口浊，半口清——清的留给坛，浊的吐给夜。

他摸出袖里枯叶，叶缘毛刺扎指：「脉稳些……」四个字像叶青禾隔山递来的手温。温不进门，却够他把夜守完。

丑时后段，他把嗅诀又过一遍。过得慢，慢到耳里能听见自己筋脉里那点细响。细响不乱，才敢说小成；小成不是名，是夜深还能闻清辛淡。

棚外猫叫一声，尖。邻床吓得一抖：「什么东西？」

「猫。」韩泥道，「猫比人老实。人绕圈子，猫只叫饿。」

「你还能开玩笑……」

「开不起。」韩泥掌不离坛，「玩笑漏气，气漏了，沿就不净。」

「韩叔，」邻床梦呓突然清楚一句，「明儿风紧怎么办？」

「紧就听。」韩泥说，「听完不接。不接，人还在桩上。」

「桩……好沉啊。」

「沉才不倒。」韩泥收枯叶，「倒了，坛先露。坛露，什么层都白涨。」

他把姜汤罐底刮净，刮得金属轻响。响过之后，夜又静。静里他数债：衣钱一笔、汤一笔、散一笔——笔笔分开，分开才还得动。

远处更鼓换班，鼓点沉。韩泥跟着点在指节上轻轻敲了三下，三下够了——够记今夜没露坛、没出门、没接赖福的火。

「够了吗？」他问自己。

「够守这一夜。」他又答，「守完才谈外头的风。风没进门，人先别乱。」

他闭眼前对坛丢下一句：「下一回，风声紧。台见以前，先把夜守牢。」

坛仍不说话，只留那一线温。温在，锣还没响，他已经听见远处有人抄条、传谣、等着看丑骨出丑。等着的人多，守得住的人少；他得先当那少数。
