# 第二十七章 伤复

狼口那道伤，公审后才化脓。化脓，像迟来的嘴，咬自己。

韩泥臂肿，编筐慢半拍。慢，不停。停了，像认输。他早起浸藤条，浸软了绕膝，指节勒出闷响，臂骨一紧，紧里泛白——泛白不要紧，要紧是手不能停。

沈枯芽怯来，拎小陶罐，罐里药泥黑，味冲：「叶姑姑让送的。她说，别化脓，化脓，抬不了筐。」

韩泥接罐，独眼平：「跟你叶姑姑说，我记着。」

芽儿点头，点得像米粒：「韩爷爷，狼那夜……我不怕你了。」

「怕也不用。」韩泥说，「活着就行。」

娃跑远，像又送一笔恩过来。他心里芽儿那粒米旁早有一挑，挑浅，像药叶，像芽儿手。

晌午他担半只筐去集尾。臂布勒紧，编得慢，慢也得卖。卖十二文，文落掌心，像告诉全村：天罚在嘴，不在臂。

赵绣春路过，见他臂缠布，鼻哼：「晦气连自己也咬？」

韩泥不辩，只举筐。筐沉，沉得赵绣春退半步——退，像公审余韵还在。还在，就够小胜。

张麻婆骂赵：「人家救娃，你泼汤，谁晦气，心里清！」

赵脸青，走得快。走快，像怕再当众哑口。哑口一次，够她记一辈子。

周嫂洗衣，棒槌顿了一顿，像也想退半步。退不出，便轻敲。轻敲，也是变。

韩泥上药，药辣，辣得清醒。清醒里他望后山狼道，道静，静得像那夜没发生过——发生过，记在心里，记在芽儿那句「不怕」。厚，他得还，还到芽儿长大，还到她能自立。

夜里他不增叶汤，只在芽儿那粒米旁又添一小点，像药泥落心，落得像印。印浅，真。坛角微温一丝，像吸药气。吸了，瓮未醒，人心先定半分。

外头谣又起，起得低：「晦气翁被狼咬，天罚！」

韩泥低声道：「天罚在嘴，不在臂。臂好了，嘴还在。」

嘴在，记也在。记，比辩狠。狠，不急；急了便乱，乱了便邪。

叶青禾隔帘问，声平：「臂还化脓？」

韩泥在门外：「上了药。能抬筐。」

「能抬，别逞。」她顿一息，「逞着往外冲，死半路，我爹药白送。」

帘内药杵轻响一声，像叶家仍在。仍在，公审才不算白审。韩泥「嗯」一声：「不逞。命要还恩，不贱卖。」

帘静，静里秤响一声，像盖章。章盖「还活着」，盖得硬。

沈枯芽又探头，像芽出土：「韩爷爷，狼若再来……」

「再来，棍还在。」韩泥说，「你别上山，山给采药的人走。你走巷，巷安全。」

芽儿又补一句，声小：「公审那日叶姑姑说汤是热的……我记住了。」

韩泥应：「记住就好。谣会旧，汤不能旧。」

娃跑回，像把「不怕」又送一遍。送一遍，够记在心里，不必再划。

赖扒皮的人路过，瞥臂布：「狼咬晦气翁，天意。」

韩泥举筐：「天意在天，筐在我手。手紧，天咬不死。」

来人骂「老疯」，走了。走，也是退半步——退，够。

张麻婆送一贴膏药，膏黑得像馒皮：「贴上，别流脓。流脓，手抖，耿恩笑话你。」

韩泥贴实，低声道：「耿恩记了，你的馒也记了。记全，手不抖。」

夜里，赵绣春门内摔碗，摔得脆，像不服公审。不服，也不出街骂了——骂累了，或怕了再哑口。韩泥听声，不笑，只按心口。按叶汤，按芽儿那粒米，按臂上布结。结紧，命紧；命紧，恩才还得上。

坛角微温退，退得像伤退。退半分，是实；实，就不说满话。满话惹祸，实话记在心里。

第三日换布，叶青禾隔帘递新布，布净，净里带药气。韩泥独臂解旧结，布揭时皮肉一紧，紧里泛白，白得像狼牙留的印。药气冲鼻，独眼微涩——涩不要紧，要紧是脓退，退到抬筐不抖。

「自己换？」她问。

「能。」韩泥缠布，缠实，「换不了，抬不了筐。」

「狼那夜，芽儿说不怕你了。」她声平，「不怕，不是胆大，是恩厚。恩厚，你得更惜命——别逞着往外冲，死在半路，心里那些恩谁还？」

韩泥只答：「不逞。命要还恩，还到芽儿长大，还到叶家秤还在。」

叶青禾又补一句，声低：「布三日一换。换不净，脓又进。进脓，抬不了筐，也带不了坛。」

「带坛。」韩泥缠布，缠实，「臂好了，坛在，路在。」

沈枯芽隔巷递来小陶罐，罐里清水，凉得像娃手抖过：「叶姑姑让洗的。洗布，别化脓。」

韩泥接罐：「洗布，我记着。跟你叶姑姑说，药泥用了。」

陈姑送半块干饼，饼硬，硬得像冬恩：「臂好了，手别停。停手，像认输；认输，赵绣春又笑。」

韩泥收饼，掰半给沈家豁口：「不停。手停，筐空；筐空，谣说天罚应验。」

集尾有人贴榜角，「沉丹宗」三字丑，却真——真在路开。路开前，臂须好，须抬得动坛。卖筐时，买主掂他臂布，问：「狼咬的？」

韩泥独眼平：「咬过。筐还紧。」

买主愣半息，仍付十二文——文落掌，像告诉全村：天罚在嘴，不在臂。

叶青禾帘内秤落一声，落脆，像给臂伤摁了章。章盖「还活着」，章盖他该活，该还。

回屋，他洗布，水凉里带药气。洗时独眼瞥坛，坛腹一线温，温得像应「还活着」——章盖在臂上，也盖在心里。

试抬空筐，筐起，起得紧——紧，像告诉全村：狼咬过，人还在；还在，就不白忍这一回。

试抬满筐，筐起得沉，沉得像将来要带坛上路。臂骨一紧，紧里不疼，疼得清醒——清醒里，脓退了，路还在。

沈枯芽隔门问，声小：「韩爷爷，臂好了，还能救娃吗？」

韩泥应：「能。救过一回，还能救第二回——第二回，别上山。」

芽儿「嗯」一声，步远。步远，像徒线又送一遍；送一遍，够记在心里。

里正柱角告示边，有人小声议：「八日上路，臂肿的也能去？」

韩泥经过，只抬了抬空筐。筐起得稳，稳得像答话：能去的，不是嘴，是手。

丑时末，他把膏药又按实一遍，按得像把「还活着」钉进臂骨。臂骨疼，疼得清醒——清醒里，帖的风声在远。远不要紧，手在，筐在，坛在。

他低声道：「徒恩记着——还，先活；活，才配往后还。还路远，臂先不能废。」臂好了，心不能软；软了，谣又进。

里正巡巷，咳：「臂好了，别生事。生事，连坐。」

韩泥举筐：「不生事。生事在嘴，不在臂。」

里正瞥他臂布，瞥完步远——远里，像也给公审留一寸缝：臂好了，手还在；手在，路还在。

夜来，他仍擦坛沿。沿凉，心热，中间夹着五十八年的命。命薄，还没断；断不了，就因为芽儿那句「不怕」，因为叶家秤还在。他合袄，塞怀，臂肿似退半分——退半分够，够撑到帖贴榜那日。

窗外有人嘀咕：「狼咬的翁，臂好了？好了也晦气。」另一人啐：「公审那天你怎么不吱声？现在说有什么用。」嘀咕散了。散了也好，手还得编——编稳了，帖到了才接得住。接得住，才配往后还。

沈枯芽隔门又问：「韩爷爷，臂好了，还能编筐吗？」

韩泥扳扳手指，声哑：「编。编筐手不能抖，抖了，还的时候才不寒酸。」

娃「嗯」一声，步远。步远，像把「不怕」又送一遍。

叶青禾帘内药杵轻响，像给臂伤送行——行不送，送的是记。记着，就不怕半路死。臂好了，能抬筐，能带坛，能接柱上那尺。尺在，手就不飘；手不飘，还的时候才不寒酸。

坛角微温一丝，应「活」字，不应谣。夜里风从门缝过，他按住臂布结，低声说：「稳。等帖。」
