# 第三十四章 杂役

入门第三日，赖扒皮来了。

他本名赖福，皮笑肉不笑，笑里带秤——秤用来克扣：克半袋麸皮，克一把柴刀，克一句「晦气杂役」。他甩单过来：「韩泥，丙九末席，兽栏你包。」

韩泥接单，单旧字丑：「包。」

赖福眼一眯：「就一个字？老丑，嘴硬。硬没用，宗门看手。」

韩泥举起宗门发的破篓——篓底补过，补得像他的脸：丑，牢。赖福瞥筐底，瞥见席角一截陶色，暗得像腌菜坛沿。眼停半息，啐一口：「还带村里晦气坛？扔了！」

「扔不得。」韩泥独眼平，「祖辈传的。」

「传？传给你克宗门？」

「克不克，天知道。坛在，人在。」

赖福还想尖，管事路过，咳了一声：「赖福，麸皮按册发，少一粒，我记你名。」

赖福脸一收，收得像刀入鞘。鞘在，刀还在——刀是克扣，克扣是仇苗。韩泥记在心里，不拔，先活。他低声道：「坛在，人在。人活着，克扣迟早还。」

这是小胜：坛没扔，麸皮也没少发。少发可以是明日的事，今日赖福先退了半步。管事记簿时瞥他一眼，在册边添半行：「丙九韩泥，兽栏。」字丑，丑也真——名上了册，饭路才在。

赖福出门又折返，笑里仍带秤：「丙九，柴刀钝了，自己磨。磨钝，没饭。」

「钝，手稳。」韩泥说。

「稳？」赖福眯眼，「稳有屁用，凡骨炼不了丹！」

「丹以后再说。」韩泥说，「先说粪。粪清完，才有饭。」

赖福嘴唇动了动，没再尖——尖不过「饭」字。饭在，路在；路在，克扣便总能等到还的时候。

午后首清兽栏，栏深，腥冲鼻。韩泥铲下稳，稳得像泥岗抬棺：棺沉，手不抖；粪沉，手也不抖。同批有人掩鼻退远，退远还嘟「晦气」。他不辩，只把湿粪换干草，换得角里小兽咳轻半声。管事路过，记一笔：「丙九韩泥，清细。」

晨起杂役灶。刘婆端粥桶，粥稀，勺却沉。她瞥韩泥丙九牌，又瞥他肘上破袄——补过，针脚密，密得有点像叶家手艺，又不完全是：针脚歪一点，真一点，真像补衣婆自己的手。

「新来的，丑脸那个？」她嗓子沙。

韩泥点头。

刘婆舀粥，舀得满，满得像多给半勺。半勺恩轻，轻也记。她低声道：「肘破了，找我。别找外门裁缝，裁缝嘴贱，嘴贱还收缝钱，钱贵。」

韩泥不多接话。话多像欠。他只道：「记着。」

刘婆愣半息，哼一声：「记着就记着。老婆子不图你还，图你活。」

活，韩泥懂。懂，才把这句暖收稳——暖在粥里，暖在仇线外头，暖得像将来还刘婆那一笔，已有了根。

午后去药渣场。他分拣渣，灰里偶然一点香。香淡，淡得像将来摸丹的命。命在渣里淘，淘是丑杂役的活，活脏，脏也能活。白织月打从旁过，袖风带香，瞥他像看灰：「丙九也分拣？浪费香。」

韩泥不辩，低头，指稳得像抬棺。白织月走远，还丢一句：「凡骨，炼什么丹。」

话冷，冷不过叶汤。叶汤热过，热在胸里，胸热，手便稳。药渣场角，铁无言路过，抱臂，没正眼看他，只瞥分拣堆：「渣里别漏香。」

「不漏。」韩泥应一声。

铁无言走了，走得也像认了一下手——认可不重，重的是手不抖。管事记簿，记他分拣量足。足，像泥岗卖筐十二文：钱不多，够活。沈枯芽若知他入门，大概会怯笑一下。笑浅，像芽。芽在泥岗，他在宗门——门隔着，恩不隔。

夜，杂役房。韩泥盘坐床角，擦坛沿。沿厚，厚得像爹的手，像祖父的手，像祖辈不知几代人的手。赖福在门外骂邻床，骂完踢门：「韩泥，明日兽栏加一倍。做不完，没饭。」

门内他不怒。怒像少年。他近六十了，记恩记辱记仇，记到能还那日。坛腹微温，像应赖福那一脚——脚狠，坛不应狠，应一个「忍」。忍不是怂，是活路。他低声道：「仇苗在。恩也在。忍得住，才醒得了瓮，还得了叶。」

邻床少年悄问：「你跟谁有仇？」

韩泥独眼闭一息：「跟克扣的人。跟嚼谣的人。跟欠恩未还的人——欠恩未还，先把命还稳。」

少年不懂，不懂便睡。韩泥睡不实。睡实像认输。丑时将近，坛沿一线温光一闪便灭，灭得像公审前夜那闪——闪的是血渗沿，闪的是爹那句「坛在，根在」。他不追问。追问像少年。他只按掌，按实。

赖福门外又骂一句，骂远，骂像风。风过，坛温不退——退的是怕，不退的是命。刘婆门外咳：「肘别挂粪，挂了，我又补。」

「记着。」韩泥应。

白织月白日那句「凡骨炼什么丹」，他记着，不辩。辩费舌。舌要留来清粪，留来活，留来还。邻床少年又醒，悄问：「赖福为何总盯你？」

「盯末席。」韩泥说，「末席好克。克了，他秤满。」

「你不告管事？」

「告了，饭少。」韩泥独眼平，「饭少，恩断。恩不断，先忍。」

少年似懂，闭嘴。韩泥再按坛沿——沿温不退。外头赖福踢门又嚷：「韩泥，明日兽栏再加半倍！」

「加。」韩泥应，「手稳，粪不洒。」

管事门外贴条，条压门柱：「三日后，凡骨测骨。丙九末排，韩泥到。误者，滚。」

三日测骨——比瓮醒近，比还叶汤远。远不要紧，要紧是帖钉在柱上。刘婆门外又咳：「明早肘别挂粪。挂了，老婆子又得顶针。」

白织月廊下冷声：「末席别碍眼。」

韩泥不接：「眼碍不住。碍的是屎，不是你的贵。」

同批杂役凑近：「白师姐都懒得骂你了。」

他不答，只低声道：「三日测骨。帖在柱，手别抖。」外头赖福骂远了，今晚丙九仍要活——活，才谈得上醒瓮，才谈得上还叶。他咽稀粥，粥稀，热却在。热在，心里又像添了半分暖；半分暖，够熬到丑时前那一闪温光。

同批有人低声：「韩泥，忍得住？」

「忍得住。」韩泥说，「忍不住，坛就扔。坛扔了，根就断。根断，恩就凉。」

那人闭嘴，像被「凉」字堵住。午后又清一轮兽栏，湿粪换干草，角里小兽咳声再轻半分。管事记簿又添：「丙九韩泥，冬角细。」细字丑，丑也真——真在名上，名在，三日测骨帖才验得过。同批嘟囔：「末席也配记细？」他不辩，铲下仍稳。稳，像告诉赖福：克扣克的是粮，克不住手；手在，坛在，恩在。

丑时将近，符影半闪，沿上一线水汽又灭。他不追问，只按掌。坛腹应忍，不应怒；怒像少年，忍才像丑翁。他把恩、辱、仇各自搁开，不搅成一锅——搅了，手会软；软了，赖福那秤就更虚。

赖福的秤虚着，他便把锤子先存着。锤未落，心先稳。稳到能立「必还」；还，先还热的。外头骂声远了，远像风；风过之后，不退的是命。命在，恩在，坛在。白织月廊下冷声远逝，他不比——比是少年。丑时末，他把破袄角又掖实一遍，掖得像把「忍」字钉进骨里。

掖实了，三日测骨那日，手才不软；手不软，末席也是席。席上了，恩才不断。廊下衣裳鲜亮的闲话起又落，落不进他掌茧——掌茧裂，裂得真；真裂，才配忍赖福虚秤。虚秤克粮，克不住坛在怀底那一丝温。

坡下石阶潮，他数帖，数债，数明日兽栏还要加的那一倍。加就加，手稳就行。帖在柱上三日，三日里哪一铲都不能抖。抖了，滚字成真；硬着，末席也能把热的一点点攒回来。攒回来那日，叶汤才不算白烫过他。
