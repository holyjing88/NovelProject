# 第一百零一章 兽潮备

天亮。

兽未至，备先至。备至在栏，在窑，在袖。

韩泥白日清栏加趟，栏净，净不要紧；净，少瘴入。入夜仍守东角，角湿草换两遍，两遍不要紧；两遍，才挡甜腥。

铁无言送石灰一包：「撒角，别撒眼。」

韩泥撒，撒匀，匀像编筐勒藤——勒匀，才稳。

第二备，在废窑。丑时雨，他上山，不为试，为炼备——备清瘴丸料。料从渣来，来路丑，丑不要紧；丑，才匿。

经页有「清瘴」残诀，诀浅，浅而真。真，他投青芯两撮，褐渣一把，加石灰半指——指少，少才祛邪，不伤人。

炉温起，起半寸，半寸稳。稳里，无丹，只有一团灰香——灰香正，正像能清甜毒。

灰香入囊，囊三只，囊缝丑。丑，才像杂役随身药。

赖福查夜禁：「谁上山？」

韩泥立队：「倒渣。筐满。」

管事弟子验筐，验完，放行。放行不要紧，灰香在袖，袖扎紧。

第三备，在心。心记：兽至，先清瘴，再清栏；清栏，不逃。

刘婆补肘布，布厚：「备够了？」

「够了。」韩泥说，「手稳。」

沈枯芽送水，水凉：「叶姐说……别逞能。」

韩泥接水，不接话。话在还，不在嘴。

白织月立远，声冷：「丑杂役也备？备了别拖后腿。」

韩泥只答：「不拖。」

不拖，是证；证在108，在远。

夜，兽吼又近。近里，栏中小兽齐嘶，嘶像潮头。潮头不要紧，头过，才是身。

他东角立，蒙条，袖囊在，经页在，掌纹在。在，就不滚。

丑时，静半刻。香落齐，灰香在鼻下一试，试而正。正，够备，不够战——战在至，不在备。

他低声道：「备齐了。下一更，潮近。」

坛腹温，贴掌，像应「备」字。备，不是退；退字不在他册。

更鼓尽，柱上条又换，换字更急：「明日黄昏，兽潮或至。」

或至，就当至。至，手在。

韩泥白日清栏加趟，栏净，少瘴入。湿草换两遍，才挡甜腥。

铁无言送石灰：「撒角，别撒眼。」

韩泥撒匀，匀像编筐勒藤。

丑时雨，他上山备清瘴料。经页残诀浅，浅而真。

管事验筐，验完，放行。灰香在袖，袖扎紧。

刘婆补肘布：「备够了？」

柱上条换：「明日黄昏，兽潮或至。」

废窑里，灰香成时，他嗅半息，半息够辨甜毒退不退。退半寸，够；不够，再加石灰半指。半指，少才祛邪，不伤人。伤人，像邪；邪了，秦执事来。

回栏，赖扒皮查夜禁，查完，没上山追。没追不要紧，灰香在袖，袖在，备在。备在，102近，103压，104至——至，手在，铁无言在。

备第二日，韩泥白日清栏加趟，栏净，净不要紧；净，少瘴入。

废窑丑时雨，他上山，不为试，为炼备——备清瘴丸料。

管事记簿：「丙九韩泥，备障仍清细。」记细，是小胜；胜在活，不在嘴。

正不要紧，正不在成，在备。备齐了，前夜才接得住102。筐满。放行不要紧，窑深，炼备藏得住。藏得住，才不算偷火。

刘婆缝布条，又塞袖一口：「蒙鼻，别嗅太过。嗅太过，甜毒入肺。」韩泥收，收不谢。谢轻，恩在布条里。布条粗，粗不要紧；粗，才挡得住风。

铁无言分障草，草湿，声低：「撒角，别撒眼。眼进了石灰，比瘴还疼。」韩泥应声：「撒角。」撒角，他熟——熟像清栏，清栏他做了半年。半年不要紧，半年才稳。

更鼓沉，韩泥按木牌，牌凉贴胸。备齐了，下一更，潮近。柱上条又换，换字更急：「明日黄昏，兽潮或至。至，手在。兽潮备，备在三处：栏、角、窑。栏净，净少瘴入；角湿草换两遍，两遍挡甜腥；窑里炼备，备清瘴丸料，料在经页残诀，诀浅，浅而真。韩泥白日清栏加趟，铲下稳，稳像编筐勒藤。勒匀，筐才卖；卖钱，买米，买静。

静，在丑时上山——上山倒渣，渣满，筐满，像常活。常活藏炼备，炼备不成丹，成灰香。灰香正，正像能清甜毒。」撒角，他熟。熟，像清栏半年——半年不要紧，半年才稳。稳，才接102近。近在黄昏，黄昏兽吼不再远，远变草动，草动像爪。

」「倒渣。」韩泥立队，筐满，管事验完放行。放行不要紧，窑深，声不出。声不出，101备才藏得住——藏得住，102才撒得开。刘婆又塞布条，条粗：「蒙鼻，别嗅太过。」嗅太过，甜毒入肺；入肺，手飘。手飘，栏先倒——倒，99记过，100前夜白守。白守不要紧，白守因备不足；备足，在101。

柱上条换字更急：「明日黄昏，兽潮或至。至，手在。近，不是退，是试界——试界实了，103才压，104才至。

兽潮备，备在栏、角、窑。栏净，角湿草换，窑里炼清瘴料——料依经页残诀，诀浅，浅而真。真，投青芯两撮，褐渣一把，石灰半指，炉温半寸，成灰香。

赖福查夜：「谁上山？」「倒渣。筐满。」验筐放行，放行不要紧，窑深，炼备藏得住。藏得住，102才撒得开——撒东角一线，一线白，白退甜毒半寸。」撒角他熟，熟像清栏半年。稳，才接102近、103压、104至。至在远，101只备齐。柱上条换字：「明日黄昏，兽潮或至。」或至就当至，至，手在。

更鼓沉，按木牌，备齐了，下一更潮近。潮近，不是退，是试界——试界实了，角还在，就不滚。备齐在三处：栏净，角湿草换，窑里灰香正。正，像能清甜毒；甜毒，是99记的，100前夜蒙条挡的，102要撒的，103要压的——压在远，101只备。备了，柱上条换字更急：「明日黄昏，兽潮或至。

手在，赖福查夜「谁上山」，答「倒渣」，筐满，验完放行。放行不要紧，窑深，声不出；声不出，炼备藏得住。

藏得住，102撒得开；撒得开，103压得住；压得住，104接得住。接得住，105潮中，106续，107钩，108清瘴。清瘴在远，101只备料。料齐，更鼓沉，按木牌，备齐了，下一更潮近。潮近，试界实了，角还在，就不滚。

备障第二日，铁无言分石灰，石灰白，白像102将撒之线。线在远，101只备齐。齐，柱上条又换，换字急，急不要紧，急才真。真在「或至」，或至就当至。至，手在，角在，坛在怀。怀贴坛，像应「备」字。

兽潮备在三处：栏净，角湿草换，窑里炼清瘴料。料依经页残诀，诀浅，浅而真；真，投青芯两撮，褐渣一把，石灰半指，炉温半寸，成灰香。灰香正，正像能清甜毒——甜毒是99记的，100前夜蒙条挡的，102要撒的，103要压的。压在远，101只备齐。」「倒渣。筐满。

「「」验筐放行，放行不要紧，窑深，声不出；声不出，炼备藏得住。藏得住，102撒得开。」撒角他熟，熟像清栏半年——半年不要紧，半年才稳。手飘，栏先倒。潮近，角还在，就不滚。第三备，在同袍袖里——袖里多一包干草，草干，味淡，淡少瘴入。入不要紧，入前先备；备齐，102才撒，103才压。

压，铁无言喊，韩泥挪，挪不移角。角在，名在，路在。路在，就不滚。滚在钱戾衡脚，脚在远，约在怀。怀贴身，像守坛沿。备齐第四日，同袍借韩泥编筐，借完还藤，藤硬，硬像规矩。规矩在，手才不飘——飘了，栏先倒；倒，99白备。白备不要紧，101只备实。备实，栏净，角湿草换，窑里灰香正。

正，像能清甜毒；甜毒，102要撒，103要压。压在远，101只齐。至，手在，角在，铁无言在侧。侧在，就不散。散，赖福喊；喊，韩泥不应，应的是备。备足了，下一更潮近。丑时，廊下静半刻，香落齐，掌纹温一线。温像培元路稳半分——稳，为清瘴备。备足了，102才撒，103才压。齐，手在，就不滚。

备齐最后一夜，柱上条字更急，急不要紧，急才真。至，手在，角在，窑里灰香正。正，像能清甜毒；102才撒，103才压。齐，更鼓沉，按木牌，掌在，路在，就不滚。
