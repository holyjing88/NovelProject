# 第一百零四章 兽潮至

至。

卯时前，界破一线，线细，细不要紧；细，毒先涌。涌像潮，潮头是瘴，瘴后是爪。

爪在障外，障内，韩泥东角立。立，像钉；钉稳，角不散。

「至了！」铁无言喊。

韩泥撒灰香，撒尽，尽不要紧；尽，就炼——炼在角，不在窑。窑远，远不要紧，袖里还有废渣青芯，芯少，少够一炉小丸。

炉是掌炉，掌纹火起，起半寸，投料，投序乱不得。乱，败；败不要紧，败过，这一回要成。

成在急，急而不飘。飘了，栏倒；栏倒，丙九连坐。

丸成，成三粒，粒小，小不要紧。小，塞入口，口含半息，半息清体内甜毒；另两粒，掷铁无言。

铁无言接，接而吞，声低：「正。」

正，一字，够并肩。

兽影贴障，贴，爪破障角。破，铁无言挡，挡时韩泥铲粪叉改障，改粗，粗不要紧；粗，能挡一爪。

一爪，够了；够了，内墙援至。援至，兽影退一线。

退，不是终；终在续。续在105，在远。

赖福瘫障后，瘫不要紧；瘫，仍记名——名在谁稳。

管事弟子记：「丙九韩泥，角稳；铁无言，挡爪。」

记，是小胜；胜在册，不在嘴。

白织月远观，声低，低像意外：「丑杂役……没逃。」

韩泥不答。答，费舌；舌要留来守第二更。

第二更，兽又近。近，他丸尽，尽不要紧；尽，还有经页诀，诀出，灰雾再起。

再起，角又清。清，像活。

夜尽，他掌裂，裂渗潮，潮不是血碰沿。沿净，坛在怀温。

回棚半刻，按坛，低声道：「至了。守过了。」

坛腹温，像应「至」字。至，不是清；清在108。

丑时，静半刻。香落齐，掌纹烫——烫像境动，动半分，仍五层，五层实了。

邻床哭：「我差点跑。」

「跑不要紧。」韩泥说，「怕也守。守了，才活。」

更鼓尽，界外仍吼，吼像未终。未终不要紧，至，已至眼。

他低声道：「至了。下一程，潮中。」

潮中，手在，铁无言在，清瘴路在。

界破一线，毒先涌。涌像潮，潮头是瘴。

韩泥东角立，立像钉。钉稳，角不散。

掌炉起，投序乱不得。丸成三粒，粒小，小不要紧。

铁无言接丸，吞，声低：「正。」

兽影贴障，爪破障角。铁无言挡，韩泥改障，改粗，能挡一爪。

内墙援至，兽影退一线。退，不是终。

管事记：「丙九韩泥，角稳；铁无言，挡爪。」

白织月远观，声低：「……没逃。」

内墙援至，援不要紧，援前那一爪，是铁无言挡，是韩泥改障。改障，像编筐勒藤——勒匀，才稳。稳，才并肩；并肩，不是喊，是手。

第二更，诀出，灰雾再起，再起，角又清。清，像活。活，是小胜；胜在栏里小兽少倒，胜在同袍不退。不退，108还要清深窝——窝深，深不要紧，深才要丸。

至后第二更，兽又近。

一兽爪落障，落重，重不要紧，他侧身，身稳，矛再刺。刺退，记功一笔——笔记功，不进舌。

刘婆远立，篮里饼硬，没递——递不要紧，恩在粥里，也在「手稳」二字里。
