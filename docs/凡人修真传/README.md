## 写作顺序

1. **`16` 正文写作参考圣经**（扩写前必读）  
2. `00` → `01` → `02` → **`11`**  
3. 对照 `chapters/volXX` 章纲 + `15` 命名  
4. 正文写入 **`prose/volXX/`**（用 `_模板-章正文.md`）  
5. `03` + `08` 对照情感 · 每部完成用 `09` 复查
