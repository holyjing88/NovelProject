# 第三十六章 刘婆补肘

入门第七日，丙九灶上粥更稀。稀得勺子几乎立不住，刘婆仍一勺一勺舀满——满得像多给了半勺。

「丑脸那个，来。」

韩泥端碗。碗糙，掌印深，深得像编筐抬棺养出来的老茧。刘婆瞥他肘上那截袄，瞥见针脚——密，却歪一线，歪得真，像补衣婆自己的手，又像别人补过：「肘谁补的？」

「叶丫头。」

刘婆鼻子里哼一声，像不信寡妇好心：「寡妇心善，善到晦气？」

「恩。」韩泥独眼平，「不是晦气。」

三句落完。刘婆愣了半息，从袖底摸出针线包，包旧，顶针亮：「坐下。肘我看看。」

灶边少年窃语：「刘婆也沾晦气？」

刘婆眼一瞪：「晦气在你们嘴，不在他肘。肘漏风，风进肺，肺坏了谁清粪？你们清？」

少年噤住。韩泥听见了，却不坐。坐像讨，讨轻，恩重。刘婆把针线搁桌上，嗓子沙：「不坐也行。站着风仍钻。钻久了臂僵，僵了抬不了粪，抬不了筐，赖福正好克你饭。」

他这才坐到凳角。凳歪，歪得像丙九末席——席末，也是席。刘婆剪线穿针，眼浊，手却稳，稳得像顶针顶了一辈子布。针脚比叶家还糙，糙得却牢。

「老婆子补，不收钱。」她低声道，「钱要，你们绝户拿不出。拿恩，恩太重，老婆子扛不动。」

韩泥不接「扛不动」，只讲事实：「肘破，风进。」

「进就补。」刘婆咬紧线头，「补完，别谢。谢轻。记着就行。」

「记着」两字他最会。指节白了一息，又平下去。门外赖福路过，瞥一眼便笑：「刘婆，你也沾晦气？沾了，粥更稀！」

刘婆头也不抬：「粥稀是你克的，不是他克的。他克你，你早倒了。」

赖福脸一青，青完便走。走了，灶边便像赢了半寸——赢在有人肯给丑杂役顶针顶布。韩泥低声道：「记着。」

「记着记着，」刘婆哼，「老婆子又不少块肉。」

肉不少，恩却多了一笔。补肘时，他肘伸得稳，像当年抬棺不抖。刘婆针走动，边走边说：「老婆子顶针顶一辈子，顶的是活，不是脸。脸丑，肘不能漏风。」

「不漏。」韩泥说，「漏了，清不了粪。」

「知道就好。」她咬线，「别挂粪栏。挂了，我又补。」

夜里他回丙九房，抚肘上新针脚。脚糙，却暖进骨。骨里想起叶丫头那针，又想起刘婆这针——两针不同，同一意：不漏风，不丢路。坛腹微温一线，像应了「刘婆」二字。

「恩又多一笔。」他低声说，「笔多，命要活长。活长了，才还得起。」

邻床少年梦呓，呓得像嫌粪臭。臭他不怕，怕的是恩断——恩断了，瓮醒也没用。外头更鼓沉，鼓点像顶针一下一下顶在心上：顶出将来还刘婆一身新衣。

翌日清晨，赖福又在门外骂：「刘婆，你也沾晦气？」

刘婆门外应：「粥稀是你克的，不是他克的。再骂，明早你也没粥！」

赖福哑了，像秤空了一格。韩泥端碗喝尽那稀粥，只点头一声，像盖了印。灶边少年悄问：「刘婆为何帮你？」

「帮肘，不帮脸。」韩泥说，「脸丑，肘不能漏风。漏风，清不了粪，编不了筐，活不了。」

「她真不收钱？」

「不收。钱贵，恩重。重，先记着。」

「你怎么还？」

「活长了，才还得起。」他独眼平，「活一日，记在心里一日。记满了，就还。」

少年似懂非懂，闭嘴干活。韩泥抚针脚，暖还在。他心里把刘婆恩与叶丫头恩分开摆——泥岗那几笔归泥岗，宗门这一笔归宗门；混了，将来还错人，汤就凉。

午后搬麸皮，肘新针脚不挂袋角。不挂，风不进；风不进，臂能抬。能抬，才抬得了将来省亲路上的篓。编筐课上，管事弟子验筐松紧，他筐紧，弟子无话。无话，赖福脸又青半分——青半分，也够丙九多喘一口气。

夜里他摸坛沿，沿一线水汽，应完又敛。敛着，瓮仍眠；眠着，步先稳。字纸只记饭，饭稀，恩不稀——稀粥顶一口饥，就多近还刘婆那日一步。

清粪时肘布蹭到栏角一次，他立刻停铲，退半步，把袄袖往上捋稳。管事弟子远瞥，没骂，只在册边点了一笔「手稳」。点一笔比骂轻，轻里却是真：肘不漏，粪才清得细；清细了，赖福想扣饭，也找不着由头。

药渣场那边风过，苦辛一线。线细，像钩——钩向分拣，钩向鼻下静，钩向将来还她一身新衣。他闻一息，不声张。声张像求丹；不求，只记。

编筐课间隙，他坐廊角检查肘布。针脚糙，蹭到藤条也不开线。同场少年凑近：「刘婆手艺真糙。」

「糙得牢。」韩泥说，「牢比好看重要。好看挡不住风。」

「她真不图你什么？」

「图我活。」他抚针脚，「活着，饭才吃得下去；饭吃得下去，粪才清得完。她算的是这个账。」

少年愣了愣：「你们老头子说话都绕。」

「不绕。」韩泥道，「绕的是嘴里客气。老实话直。」

赖福路过廊角，见他抚肘，阴阳怪气：「丑脸疼了？疼了找医房啊，找厨娘做什么？」

「厨娘缝得住风。」韩泥抬眼，「医房缝不住你的秤。」

赖福噎半息，骂着走远。廊上几个杂役偷偷笑，笑完又怕，散得快。刘婆从灶间探头，朝韩泥扬扬勺：「少跟秤吵架。吵架饿肚子，饿肚子针脚白补。」

「不吵。」他应，「记。」

她哼一声缩回去。勺碰盆沿叮一声，像给这一日盖戳：肘暖了，粥稀着，仇苗却没拔——没拔不要紧，先把风堵住。

入夜雨丝细。丙九房潮，潮里坛沿更润。他按沿，沿温一线，像应「补肘」两字。邻床少年忽然问：「你泥岗那寡妇，跟刘婆，哪个恩重？」

「都重。」韩泥说，「重法不一样。一个挡石送药，一个顶针补肘。不一样，就得分着记。混成一笔，将来还乱。」

「你会还到什么时候？」

「还到她们不挨饿、不漏风。」他闭眼，「还不到那日，就不停。」

少年翻身，像被这句话压住，呼噜声反而匀了。韩泥覆席盖坛，沿净。雨打棚顶，点点像顶针——顶一下，记一下。

丑时，少年又醒，瞥他抚肘：「明儿还要编筐？」

「要。」韩泥道，「筐熟，饭熟。饭熟，人活。」

「刘婆那粥……」

「稀也是热的。」他说，「热的，记着。」

少年又睡了。门外刘婆咳了一声：「肘别挂粪。挂了，明早又来。」

「记着。」

「记就记，」她沙着嗓子回，「老婆子又不少块肉。」

他不谢。谢轻，记重——记顶针，记半勺，记赖福那张青过又哑过的脸。丹香从远处飘来一线，细得像门缝。门缝里若有课，他会站到课前；课前还没开门，肘先别漏风。漏了风，粥白送，补也白补。明日药渣场若分班，他鼻子先到位——味认清了，还衣那笔才有底。

寅时前他又摸一次肘布。布糙，针脚仍勾着肉，勾得他清醒。清醒着，把第七笔在心里默念一遍：刘婆顶针，不收人钱，只收人活。活着去认药渣场那点苦辛，苦辛认清了，将来新衣才送得出手，送得不寒酸。

灶间传来盆响，刘婆骂人声音沙却硬：「麸皮再短，短不过我这双眼睛。谁克谁，我看着呢。」韩泥听着，心里那笔恩又实半分。实了，肘上风就攻不进来；攻不进，他才扛得动明日分拣场那一筐灰。灰里若有香，他便记——记香，也记补肘的人。人不求谢，他更不能忘。忘了，肘上那针就白扎了。
