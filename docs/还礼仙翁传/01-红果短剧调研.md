## 二、核心设定

### 主角：徐霄

| 维度 | 设定 |
|------|------|
| 身份 | 穿越者，缥缈宗内门长老，实际千岁 |
| 开局状态 | 渡化神雷劫失败，修为从元婴九重跌至筑基一重 |

> **还礼仙翁传映射**：原著徐霄设定；本项目莫长春为 **大乘巅峰渡第九重失败→筑基中期**，见 [`02-原创剧情.md`](./02-原创剧情.md) §渡劫与跌境。
| 寿元 | 仅剩不足一年 |
| 外表 | 白发苍苍、满脸皱纹、拄杖暮年老人 |
| 性格 | 表面德高望重、高风亮节；内心精明，为系统奖励"赠礼" |
| 反差梗 | 千岁高龄"无力"长老，实则靠系统疯狂变强 |

### 金手指：万倍返还系统

**核心规则**：
- 向「气运之女」赠送物品 → 触发暴击返还
- 首次赠送给某位气运之女 → 万倍暴击
- 气运之女等级越高，返还倍率越高（师尊姬雪为 SSS 级，100 倍基础）

**经典返还案例**：

| 赠送物品 | 返还物品 |
|----------|----------|
| 下品法剑 | 南明离火剑（仙器级） |
| 驻颜丹 | 三千大道丹 |
| 引雷符 | 九天神雷符 |
| 寿元丹（师尊所赠） | 再返还高阶寿元/修为资源 |
| 仙元丹 | 五气朝元丹、五行仙丹（万倍暴击） |

### 世界观

- **宗门**：缥缈宗（正道大宗）
- **势力**：玄魔宗（魔道）、九宗会谈（多方势力博弈）
- **修炼**：化神 → 元婴 → 结丹 → 筑基，渡劫晋升
- **大道**：地火大道、人火大道等

## 四、剧情脉络（短剧 114 集结构）

### 第一幕：绝境开局（约 1-20 集）

**开场**
- 缥缈峰，姬雪闭关前叮嘱灵儿（宗主）和徐霄
- 徐霄渡劫失败，修为跌至筑基，寿元不足一年
- 姬雪赠寿元丹，嘱其五十年内寻道侣、留子嗣
- 徐霄内心狂喜：「终于有值钱东西可以送人了！」
- **万倍返还系统激活**

**系统首秀**
- 检测气运之女：姬雪（SSS）、灵儿等
- 首次赠礼触发万倍暴击
- 获得南明离火剑（赠送下品法剑返还）
- 服用/返还寿元丹，延长寿命

**喜剧反差**
- 同门不信德高望重的徐长老会"撩"女弟子
- 「千岁高龄，就是有心也无力啊！」
- 事实摆在面前，弟子集体破防

### 第二幕：赠礼崛起（约 21-50 集）

**宗门内**
- 向师妹、女弟子赠送低阶物品，暴击高阶法宝丹药
- 修为逐步恢复：筑基 → 结丹 → 元婴
- 领悟地火、人火大道
- 南明离火剑炼化，越级战斗

**宗门外**
- 金石城林家退婚风波
- 徐霄护短，雷霆手段斩杀玄魔宗火旗先锋（元婴三重）
- 李九夜暴怒，魔道上门

**名场面（第 30 集左右）**
- 徐霄表面只有结丹前期灵力
- 南明离火剑一出，元婴修士不可挡
- 众人震惊：暮年老人竟敢杀魔道中人？

### 第三幕：风云四起（约 51-80 集）

**势力博弈**
- 魔道上门，缥缈宗危机
- 九宗会谈，多方势力重新洗牌
- 徐霄以暮年长老身份周旋，暗中底牌渐露

**人物关系**
- 师尊、青梅、商行美人、宗门少女轮番登场
- 外界只见「群芳环绕」
- 实则是系统刷奖励 + 宗门庇护

**修为攀升**
- 重回元婴，向化神迈进
- 三千大道丹、九天神雷符等至宝到手
- 开始布局更大棋局

### 第四幕：底牌揭晓（约 81-114 集）

**冲突升级**
- 玄魔宗、独孤惊云、诸葛青天等天才联手
- 天骄争霸、仙门往来（五灵仙门、万妖仙宗等）
- 徐霄擂台赠丹、拉关系、气死对手

**高潮**
- 众人以为徐霄靠女人上位
- 暮年长老亮出真正底牌——足以撼动大陆的修为与法宝
- 魔道覆灭/退败，九宗重定格局

**结局方向**
- 徐霄恢复巅峰甚至超越
- 赠物得长生——通过赠礼系统实现逆袭长生
- 多女主感情线收束（喜剧向）

## 五、短剧叙事特色

1. **反差喜剧**：德高望重 vs 系统舔狗，年龄梗 + 误会梗
2. **爽点密集**：每次赠礼 = 一次暴击返还 = 一次装逼
3. **多女主群像**：师尊、青梅、师妹、商会、仙门天才
4. **扮猪吃虎**：表面风烛残年，实则南明离火剑斩元婴
5. **节奏快**：114 集，每集一个爽点/反转

## 六、与《万古守灯人》的差异

| 维度 | 万古守灯人（已备份） | 赠物得长生（短剧）/ 还礼仙翁传（本作） |
|------|---------------------|----------------------|
| 主角年龄 | 六十岁凡人书吏 | 千岁暮年长老 |
| 金手指 | 守岁灯（灯油/烟火） | 万倍返还系统（赠礼暴击） |
| 主题 | 人间烟火、守灯人间 | 逆袭爽文、喜剧反差 |
| 节奏 | 慢热放置 | 快节奏爽剧 |
| 感情线 | 含蓄温情 | 多女主喜剧 |

**结论**：两部作品题材方向不同，《还礼仙翁传》应走**老年修仙 + 赠礼逆袭 + 喜剧反差**路线，而非《万古守灯人》的烟火温情路线。

---

## 七、可借鉴的题材方向（非照搬）

- 老年主角修仙（打破少年天才惯例）
- 绝境开局 + 系统逆袭
- 赠礼/返还机制作为核心玩法
- 扮猪吃虎、反差喜剧
- 宗门权谋 + 魔道冲突
- 多角色气运系统
- 短剧节奏：每章一个爽点
