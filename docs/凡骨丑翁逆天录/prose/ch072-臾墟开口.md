# 第七十二章 臾墟开口

丑时。三年约后第九夜，风从窗缝钻进来，钻得像针。韩泥盘坐床沿，油灯豆，豆光照坛角——坛仍腌菜坛壳，席厚，厚得像挡目。外头赖福骂远：「丙九别嚷，嚷了加倍克。」

韩泥不答。答，费舌；舌要留来听坛。邻床翻身，眯眼半息：「丑老头，又不睡？」

「在。」韩泥声平，「守沿。」

小杂役哼一声，又睡实。睡实了，秘密才藏得住。他揭席半寸，不揭坛，只让掌贴沿边。沿温一线不退，不退，像瓮从眠里抬了半寸眼。眼在坛底，声还未出——出不出，今夜等。丑时前，他鼻下静一息。粟壳香淡，淡不要紧；要紧是香一字不落。落齐了，坛底才肯开口。坛腹轻颤，颤不像风，像有人醒。醒在底，醒在沿，醒在五十九年擦沿的手上。一声哑响，非人非兽，在颅骨里：「……韩氏守瓮人。」

韩泥独眼一紧，心口突跳。跳不要紧，手不抖——抖了，声就散。「谁在？」他低声问，问沿，不问邻床。坛底又颤，颤出第二声，更哑：「吾……臾墟子。瓮中残念，非仙，非魔，只认沿。」

名过，风冷。冷不是冬风，是古气——古气里带药香、泥腥、一丝煞意。煞意浅，浅像旧债未清。韩泥屏息，不拜。拜邪，路断；不拜，只听。听恩，也听仇苗。臾墟子声更轻，轻像怕惊动巡夜：「你掌纹有路，路通炼气一层——首境已开，开于血碰沿，不在测灵石。」

韩泥按掌，掌纹里那路微温，温得像坡下灰线三尺外的另一尺——尺在暗，不在公示。「一层？」他低声问。「一层。」臾墟子道，「凡骨壳，瓮穴灵穴。灵穴不露，露了，煞骨门嗅得到。」

煞骨门三字落耳，像钉。钉不要紧，先记——记仇因，记魔教钩，记祖辈口传里没说完的那半句。韩泥不辩，只问：「煞骨门，与瓮何干？」

坛底一寂，寂里古气翻半寸：「昔年夺瓮。韩氏守三代，魔教夜袭泥岗，夺真名未果，留壳腌菜。壳在，根在；根在，他们仍嗅。」

嗅不要紧，沿净才要紧。沿净，壳才像腌菜坛，不像宝。「嗅得到？」韩泥声平。「嗅得到。」臾墟子道，「你测灵末席，好。末席像晦气，晦气遮瓮。遮住了，三年里别叫真名。」

真名什么，韩泥不追问。少年才追问。他只答：「不叫。坛在，人在。」

臾墟子又道：「炼气一层在掌，不在舌。舌响，香飘；香飘，手就滚。滚了，玉牌白接。」

玉牌在怀，凉进骨。凉不要紧，路烫——路烫在掌纹，不在众目。韩泥低声道：「记着。」

记着——恩记瓮醒，仇记煞骨门，境记一层。三层分列，不混。外头巡夜执事敲柱：「丙九，丑时了，熄灯。」

臾墟残念弱一息，弱像瓮唇将醒未醒：「……开口，不叫名。叫名，煞骨门来。」韩泥不答名，答在心里——心里记开口这一条：开口为活，不为求。求像攀，攀要还人情，他还不起。兽栏角，小兽舔他靴边，舔轻，舔像试探。试探不要紧，韩泥换干草，草干，味淡一寸。管事记「仍清细」——细在活，细在忍，细在开口前那一息静。

邻床问：「你咋不跟内门辩？」「辩，费舌。」韩泥说，「舌要留来清栏，留来守坛。坛在，根在；根在，就不滚。」

韩泥低声：「灭了。」

执事脚步远去。坛底臾墟子声又弱一分：「吾念将散，散前只赠一句：鼻下静，渣里活。活长了，丹路才开。」

声散，散得像风过沿。沿仍温，温不退，像应「一层」二字。韩泥盖席，盖严，像盖一个刚出生的秘密。秘密不能叫，叫了，赖福来，管事来，煞骨门旧债也来。他擦掌，掌无汗，汗在独眼里退了一寸。退不要紧，境在——境在暗，不在册；册写仍末，掌写一层。邻床嘟囔：「啥味儿……像药……」

「坛味。」韩泥说，「睡吧。」

更鼓将尽，天边白一线。白里，他廊下练半刻鼻下静，静到只闻辛淡，不闻辱。铁无言脚步远，停半息，没近——没近，像知丑时不宜扰。扰了，香飘。韩泥按心口，七笔分列。第六笔泥岗沉，第七笔愧刚开，今添一笔：臾墟恩。恩轻，记重；记重了，才接得住魔教钩。他低声对坛：「一层在掌。煞骨门记着。真名不叫。」

韩泥只答：「守沿。沿净，不叫。」

赖福哼一声，脚步远。远里仍有一句：「晦气。」

晦气不要紧，晦气遮瓮。遮住了，一层才稳；稳了，才接得住二层暗修。他摸怀玉牌，又摸愧线字条，两件凉，坛温一线。温应境，不应慌——慌像少年，他是记心翁，翁慢，恩不凉。管事卯时前巡棚，灯照席角，照见净沿。沿净，像腌菜坛，不像邪。管事记簿一笔，笔丑，真在「手稳」——手稳，是小胜。散队后，兽栏腥重。

腥不要紧，掌纹那路微温，温像一层在活。活不在公示，活在暗；暗修，才接得住药山捡渣。刘婆端粥，仍多半勺：「昨夜没睡实？」

「守了。」韩泥说，「沿净。」

「沿净怕什么。」刘婆哼，「饭稳，恩不断。」

韩泥点头。恩在粥里，也在坛底那一声哑。哑声轻，记重——记臾墟开口，记煞骨门夺瓮，记炼气一层首境。他独眼平，手不抖：「一层在掌。魔教钩记着。丹路在远，手在近。」

刃不亮，先记。记完了，才配兽栏夜杂务，才配暗修二层那一步。坛腹温一线，贴掌，像应「开口」最后一笔。应完，他不叫。不叫，才守得住三年，守得住95那盏门缝灯。散队后，药渣场角风硬。他分拣半刻，指稳，稳里掌纹那路微温——温不应渣，应「一层」还新。同批少年掩鼻：「你咋不怕腥渣？」

「怕。」韩泥说，「怕也分。分完，才有饭。」

少年退半步，没再问。问，像沾晦气。他摸怀玉牌，又摸愧字条，两件凉，坛温一线。更鼓沉，沉里，他低声道：「一层在掌。煞骨门记着。兽栏夜在明日，暗修在远。」

丑时前，他廊下立半刻，掌贴坛沿，沿温不退。不退，像臾墟子那声哑还在底里应——应一息，又寂。寂不要紧，寂里他记煞骨门夺瓮：夺真名未果，留壳腌菜。秦霜远廊经过，木鱼轻敲一声：「丑时心别躁。躁了，鼻下香飘。」

韩泥不答，答在心里——心里记开口这一条：开口为活，不为求。散队后，他过坡下测灵石。石凉，线灰，灰在众目下公示过——公示不要紧，掌里有另一尺暗线。暗线新，新不要紧；新在丑时，新在席下，新在沿净处。同批少年悄问：「你身上有时热，邪不邪？」

「沿净。」韩泥说，「沿净，不叫。不叫，就不邪。」

韩泥只答：「不拜。坛在，人在。」

夜禁脚步远，风里像煞骨门三字一闪——闪不要紧，沿净，壳才像腌菜坛。像腌菜坛，魔教才嗅不准。柱上条角白一线，三年约在怀，愧字在怀，一层在掌——三件凉，一件烫。烫的是境，凉的是仇与愧；分列，不混。坛腹应温，像应「开口」最后一笔。应完，他不叫。天白后，韩泥清栏第一趟，铲下稳。

稳里掌纹那路比昨夜厚半分，厚像一层境在活。活在暗，不在公示。管事弟子验角，角净，记「丙九韩泥，清细」。记细，是小胜。赖福瞥掌，瞥完，没再嗓——嗓要留来秤麸皮。韩泥不谢。谢轻，记在心里：臾墟开口，煞骨门钩，一层首境。三事分列，不混。药渣场午后，灰冷，他分拣，鼻下静。

静里忽闻一线古气，气浅，浅像坛底残念未散。未散不要紧，沿净就稳。铁无言立栏边，声低：「坛底声，别传舌。传了，像拜器灵。不拜，只记。」

韩泥应声：「不拜。记着。」

秦霜远廊木鱼一声：「魂念易牵心。牵了，鼻下香飘。」

韩泥不答，答在心里——心里臾墟恩轻，仇重；恩轻也要还，还就是守沿，守就是不叫真名。邻床编筐，藤刺扎掌，韩泥偏手。偏手像守规——规在，席在，坛在。坛在，煞骨门才嗅不准。夺瓮旧事记着。鼻下静，渣里活。温，像应「开口」余韵。余韵不散，散就要叫；不叫，才守得住三年。老耿挑水过，声哑：「坛底声当故事。当故事，别当邪。」

韩泥只答：「当恩记。不当邪。」

更鼓沉，沉里，他合眼不睡实。一层气转半周，半周稳，稳像灰线三尺外的暗尺——尺在掌，不在坡下众目。独眼平，手不抖：「煞骨门记着。真名不叫。二层在远，手在近。」

臾墟风硬，硬里一点灵息——息在掌，境在暗；瓮穴灵穴醒一线，试台还远。
