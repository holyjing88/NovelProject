# 第一百三十四章 铁兄续

兽潮散后月余，东角栏修好了。

修好像回响——回响在栏，也在人。韩泥递灰囊，囊里清瘴余灰，灰细：「栏角还缺半袋。」

「缺。」铁无言立栏边，臂上旧伤浅，「缺，你补。补了，我站台侧。」

「补。」韩泥说。辩像显摆并肩；不辩，灰在，栏在，就在。

撒灰半刻，风硬，灰线像刀——刀划皮，不划手；手稳，栏才稳。

赖福远处骂：「丑杂役又跟铁哑巴混！」

有人帮腔：「混一起，晦气成双！」又有人偷笑：「兽潮那日，谁挡的谁还记得？」

韩泥不应。不应像石；石在，路在。

铁无言声低：「赖福余党，还会使坏。使坏不要紧，栏稳，坛稳，就不滚。」

「不滚。」韩泥说，「滚了，刘婆新衣白还。」

「还了的别丢人。」铁无言道，「丢人，我挡半步也白挡。」

「不丢人。只守。」

两人立栏边，立像兽潮那日，却不并肩献粒——粒在远，谢要走叶家那条路。并肩续，续在守，不在嘴。

沈枯芽递口信，信仍无字，只有一片枯叶，叶上印了一句口头：「叶姐说，别来丑。」

韩泥不接话，接在心里。愧线烫半分——愧是愧，恩是恩，分列才稳。

他低声对枯叶：「叶在。我不近。近了像惹。」

刘婆端粥，仍多半勺：「又跟铁无言？」

「修栏。」韩泥说，「栏稳，兽潮才不白守。」

「守栏别守邪。」刘婆哼。

「不守邪。守栏。」

「守栏也吃饭。」她把粥推近，「空肚，灰撒不匀。」

韩泥喝尽，姜辣一口：「匀。」

秦霜远木鱼敲一声：「兄弟线，心别邪。」

韩泥不答，答在心里。心里记：并肩在前，续守在今——守到该纳从绶那天，才配开口；开口前，只站。

日暮，铁无言收工，声低：「下一回，我还来。」

「来。」韩泥说，「来，栏在，就在。」

「别叫。」铁无言补半句，「叫了，像卖。」

「不叫。只站。」

「站侧就行。站近了，像情。」

「不近。近在栏，不在嘴。」韩泥道。

铁无言点一下头，点得像修栏这一日——栏在，就在。

邻床悄问：「铁无言跟你很熟？」

「熟。」韩泥声平，「熟在清瘴，不在嘴。」

「像兄弟？」

「像。」他说，「像，不在嘴，在站。」

「站什么？」

「站侧。不挡路，不抢功，不叫卖。」

管事弟子验栏，栏正：「东角稳了？」

「稳了。」韩泥答。

「稳了，照旧巡。」

「照旧。」

照旧才像常活。常活里藏线，线藏，赖扒皮嗅不到；嗅不到，丙九不连坐。

丑时，练鼻下静。粟壳香淡，一字不落。落齐，修栏这一日记进心里。

他望栏影，影长，长像清瘴余瘴未散。散不要紧，栏正，路正。

巡栏弟子提灯走过，灯火一晃：「东角谁守？」

「丙九韩泥。铁无言帮灰。」韩泥应。

「帮灰别帮出事。」弟子叮一句。

「不出事。」铁无言抱臂，「出事的是栏倒，不是人站。」

弟子哼一声离开。灯远了，栏边只剩灰香淡。韩泥把最后一撮灰按实：「按实了，风才穿不透。」

「穿不透才像兄弟。」铁无言说，「穿透了，就像给人看热闹。」

赖福廊外再骂：「铁哑巴护丑骨，护到何时？」

铁无言路过，只丢一句：「护栏。不护嘴。」

骂声噎住。噎住也好。

刘婆柴门外咳：「修栏别沾晦气。」

「不沾。栏稳，恩稳。」韩泥说。

她递半勺粥渣：「再喝一口。手抖了，灰线断。」

他喝尽，像提醒自己：续不是情，是义；义在挡半步，在撒灰，在站侧。

韩泥望铁无言，一眼像约：「纳绶前天，只站。」

铁无言点头：「只站。不叫。」

「叫了呢？」

「叫了，线脏。」铁无言说，「脏了，日后绶也不香。」

约落栏木上，木新，新里带旧伤——伤是清瘴，新是续。新旧并，像兄弟线。

废渣巷风硬，他倒渣半筐，渣灰，灰里正。正像守栏回响。回响在掌，不在舌。

白织月袖风没近栏。没近也好——近了像沾情；情在还，不在廊。廊上的嘲，留到下一回接。

更鼓沉，他按坛，低声道：「兄弟线续了。记着。」

坛腹温，贴掌，像应「栏」——栏稳，恩稳，就不滚。

编筐半刻，藤刺浅，沿净。沿净，坛才藏得住；藏得住，线才藏得住。

同棚杂役笑：「跟铁哑巴修栏，图什么？」

「图栏不倒。」韩泥说，「栏倒了，兽潮白守。」

「守给人看？」

「守给自己记。」他说，「记下了，日后还才不寒酸。」

笑的人噎住。噎住也好。

他再验栏角，角稳，稳像并肩那半步。半步续到今日，今日不叫从绶，只站。站侧，像把恩按实——按实了，日后纳绶才不寒酸。

沈枯芽远远鞠半躬，不近栏。不近像叶姐那句「别来丑」。丑不要紧，线在；线在，汤还在前。

管事夜巡记簿，记「东角修栏稳」，字细，细得像小胜。胜在忍，不在辩。

老耿挑水过栏边，哑声：「栏正了？」

「正了。」韩泥说。

「正了就守。守到风硬也不滚。」

「不滚。」

天边白一线，他望栏影：「并肩过了。续守过了。记着。」

记着，是丑翁本事。本事在站，在灰，在不把兄弟喊成买卖。

他按心口，帖与约并贴：「只站。不叫。下一回，袖风来了，手别飘。」

山风过栏，灰线微晃，晃出下一道袖香的冷。冷还没到，钩已经到了——钩住白织月阶上那声嘲，也钩住他还没实的下一层。层未满，线先稳；线稳了，嘲才试得动手稳不稳。风一紧，廊就近了；廊近了，人就得把「不辩」按实。

栏下石缝里还嵌着一点兽潮旧泥。泥干了，干得像疤。韩泥蹲下去，用指背抹过，抹得轻：「疤在。人在。线也在。」

铁无言听见了，没回头：「疤别揭。揭了，像卖惨。」

「不揭。只记。」

「记就够。」铁无言道，「够了，日后从绶才不脏。」

从绶二字落得很轻，轻得像怕被廊风听见。听见了像情；情在还，不在廊。廊上只留嘲，嘲来了，就用手稳去接，不用嘴硬去顶。

丙九夜里有人问：「你修栏修到何时？」

「修到栏不倒。」韩泥说，「倒了，旧伤白留。」

「旧伤是谁的？」

「并肩的人的。」他答，「并肩过，就得续。续不在热闹，在灰匀。」

问的人散了。散了，他才听见坛腹那一线温，温得像应「站」——站侧，不叫，不抢，不把兄弟线喊成买卖。

更鼓尽时，东角栏影投进杂役屋门缝，影直。直得像约：约好了下一回廊上那阵袖风，约好了手不飘。飘了，灰线断；断了，续也断。

夜里他复盘三事：灰匀否，角稳否，嘴乱否。匀、稳、不乱，才算续成。续成了，从绶才能远远候着——候着不是盼情，是盼日后断臂那粒丹递得出手，递得不寒酸。

赖福余党窗外丢泥块，泥打在席上，砰一声。韩泥不起身：「泥打席，打不进坛。进不了，就白费。」

铁无言远处咳一声，像应。应够了。够了，明日廊上再尖的袖风，也只是风。风来，手稳就行；手稳了，层十五才接得住，大比台才不塌在自己脚上。

他把清瘴余灰袋口扎紧，扎得死：「袋在，栏在，人在。在，就不滚。」

坛边再坐半息，他把「从绶」两字在心里按下去，按到不冒泡。冒泡像盼，盼像情，情脏义。义要干净：干净到廊上嘲来也不辩，干净到断臂那天丹递得出，递得不是买卖。

铁无言临走丢下一句：「袖风要是尖，你看着我袖——我袖不动，你舌也别动。」

「不动。」韩泥应。

这一应，像把门闩插上。闩上了，风进来也掀不翻坛。

栏外虫鸣一声，声短。短得像提示：续已落成，下出戏在廊，不在栏。廊上袖香冷，冷来便接；接时手稳，舌就省着。省着，明日袖风嘲时才不丢阵。
