# 第四十七章 小满觉醒

夜雨，杂役堂后山，泥滑，路窄。

顾小满抄近路报讯，怀里揣着孙福私例抄本，像揣半块灵石，揣一路，揣到心口猛地一跳——她无塔，却觉天地静了一瞬。

金光冲霄，一瞬即敛，像有人在她头顶记在心里，又迅速藏起。识海塔影在她意识里荡了一下，荡完即无，像塔在远处看她，看她丁级气运里忽然亮了一线。

她跌坐泥中，喘，却看得更清：左径三人，右径火把，赤焰探子换岗的间隙，全进眼里，像地图展开。雨线变成线，线变成路，路变成田埂，田埂变成丹堂侧径——她以前走夜路只懂怕，如今懂「看」。

莫长春立在雨里，不知何时至，杖点地，声轻：「小满，看见了？」

「看见了……」她颤，「小满……以前瞎，现在不瞎了。」

「不瞎就好。」他道，「恩不用嘴还。你记路，就是还。」

顾小满跪又想立，终是立，雨打脸，像洗一双新开的眼睛：「长老，赤焰探子在田埂三处，左径三人，右径火把，换岗间隙半炷香……」

「慢慢说。」莫长春道，「说清了，比跪清了有用。」

她一条条说，莫长春听，听进袖中，像听进一张将成的网。雨里远处，赵玄机思过崖方向，风硬，像另一根妒火还在冷。

顾小满说完，忽然问：「长老，小满这眼睛……能走暗线吗？」

莫长春袖空一拂，一念出塔，一枚薄护身符落掌，非宝品，贴肉可戴：「别跟丢命。暗线走侧面，别走正面。正面是霍老头和宗主的戏，侧面才是杂役的戏。」

她接符，泪落，笑也落：「小满记恩。」

雨歇，云破，一星露，像气运刚抬头，抬头便藏，藏得像丁级将甲，又像将成网的眼。

霍镇山闻讯，骂声洪：「好！暗线要活，丹堂才稳！」

周德海在长老院听闻，面色平，平里冷，像早就备好的稿。他低声对心腹：「杂役开眼？眼开了，也得看谁肯闭闸。」

魏无涯观金光一瞬，扇合，低笑，像看网将成，成在子时。他自语：「眼开了。下一程，看这张网能不能在子时送命回来。」

秦商言飞符至杂役堂，符上六字：「暗线需耳，耳在丁级。」像商言也懂：网眼小，容不下英雄，容得下报信。

孙福私例抄本仍在她怀，像第二张半石。她忽然懂：金光不是显圣，是开眼——开眼的人，走暗线，不走正面。

雨歇后，顾小满把路线一条条画进册子抄本，画得像画自己的胆——胆画了，路才走得稳。莫长春收讯，不夸，只杖点地，声轻：「路记清了，比金光更显圣。」

杂役堂檐下，有人嗤：「金光？丁级也配？」另一人悄道：「配不配，探子换岗准不准说了算。」话出，众人噤声半息，半息里像听见田埂上的火把噼啪——噼啪先远，后近，像夜还未来，眼已至。

顾小满摸护身符，符温，像长老的手还在。眼开了，路记了，网将成——成在子时，成在侧径，成在杂役堂敢不敢走夜路的那一步。

她回杂役堂，把田埂三处、左径三人、右径火把，又画了一遍，画得像画自己的命。画完，册厚一寸，厚得像把「暗线」二字压实。

杂役师兄见她册子，愣：「小满，你夜里去哪了？」顾小满声平：「记路。路记清了，比金光更显圣。」师兄不敢再问，问多了像脏嘴。

霍镇山连夜至杂役堂，骂声洪：「好！暗线要活，丹堂才稳！谁再笑丁级，先过我这一关！」骂完，又看顾小满册子，眸光一亮：「探子换岗半炷香？准！」

周德海在长老院听闻霍镇山夸杂役，面色平，平里冷。他低声：「眼开了，网也开了。网开了，侧门谁守，谁闭，得看时辰。」

秦商言飞符至杂役堂，符上六字：「暗线需耳，耳在丁级。」像商言也懂：网眼小，容不下英雄，容得下报信。她又添四字：「隐匿材备。」眼开了，符要来，符来了，网才藏得住形。

雨夜之前，顾小满只知怕。怕黑，怕赤焰，怕孙福旧党。雨夜之后，她知看——看田埂，看火把，看换岗间隙，看侧径空隙。看比怕有用，怕只会退，看才会记。

林挽月罚役途中，远闻杂役堂方向金光余韵，唇白。她不知那光为谁而亮，只知心口一紧，紧得像证字又重了一分。慕错可改，证字更重，重得像恩。

赵玄机在思过崖听风，风里像也夹着一丝金光余韵。他跪姿更直，直得像终于懂：暗线在侧面，妒在正面，正面留给他自己咽。

苏念慈在丹堂听讯，炉火旺，旺得像替暗线也担。她低声：「眼开眼开，别传成妖。妖的是赤焰，不是杂役。杂役记路，路记清了，丹堂才稳。」

莫长春立在雨后泥径，袖空，杖点地，不夸，只念：「醒比恕响，响里换命。」识海塔影一荡，塔意沉而不显，像万物可藏，藏得住路，也藏得住将成的符。

顾小满把路线册与孙福私例并放，并放得像并两块半石。她忽然问莫长春：「长老，小满这眼睛……能走多久？」莫长春道：「能走多久，看你能记多久。记路，就是还。」

第二日清晨，她把路线册呈霍镇山。霍镇山看罢，骂声洪：「好！准得像亲眼见！暗线要活，丹堂才稳！」顾小满把护身符贴胸，符温，像长老的手还在。眼开了，路记了，网将成——成在子时，成在侧径，成在杂役堂敢不敢走夜路的那一步。

霍镇山看罢册子，又骂周德海：「侧门闭不闭，看时辰，别看私！私字藏久了，夜袭先死门内。」骂声洪，洪得像把暗线也砸进每个人耳里。

莫长春收讯，不夸，只杖点地，声轻：「路记清了，比金光更显圣。显圣不在光，在敢走夜路的那一步。」一念收塔，杖落掌，他念：「醒完，符要来。隐匿符，轻里藏形。」

顾小满立在杂役堂檐下，望东脉，望得像望一条将走的暗线。暗线不在正面，在侧面；侧面才是杂役的戏，戏唱好了，丹堂才稳，夜袭才有人送命回来。

杂役堂师兄见她雨夜归来，衣湿，眼亮，亮得像换了一个人。师兄问：「小满，你去哪了？」顾小满声平：「记路。路记清了，比金光更显圣。」师兄不敢再问，问多了像脏嘴。

送回来了，半块灵石的故事，才算又长了一块。

风停，东脉云低，腥一线，像夜还未来，牙已露。网将成，成在子时，成在还恩，成在暗线，成在记路。

秦商言又飞符，符上添四字：「隐匿材备。」像商言也懂：眼开了，符要来，符来了，网才藏得住形。

魏无涯传讯，只八字：「丁级开眼，网将成。」莫长春袖中阅讯，点头，像记在心里。他自语：「网眼小，容得下报信，容不下英雄——英雄走正面，报信走侧面。」

金光敛去，路在册上，册在怀——怀暖，像半块石又长了一块。

弟子渐散，散里仍回头瞧一眼灰袍袖空——瞧得像瞧一眼九月还能走多久。

小满觉醒那夜，光从杂役院起，起得像半块石终于发光。

莫长春袖空，杖在塔中，不近光，只近人。弟子围半圈，围半圈像怕光烫眼，烫眼就得还。

顾小满声平：「我醒了，第二仍是我挣的，不是藏出来的。」赵玄机眸光微动，动得像妒又起，却起得克制。

柳青鸢声低：「师兄，觉醒在杂役，门眼要亮。」莫长春道：「亮了，才记得住半块石。」

苏念慈开炉，炉旺，旺得像把光炼成丹香。她低声：「觉醒觉醒，别传成妖。」

围观弟子院外跪半圈，跪里掺敬——敬里掺盼：盼下一符，还能不能护阵。

觉醒第二日，杂役院晨课改叫「识塔影」，影淡，淡里像雨夜那道光又近一寸。

莫长春不近光，只近人，近人像问：「各算各的，懂了吗？」顾小满声平：「懂。第二仍要挣。」

赵玄机验光，验到妒火也亮，亮里起得克制。柳青鸢声低：「门眼要亮。」

苏念慈开炉，炉旺，旺像把光炼成丹香。她低声：「觉醒别传成妖。」

莫长春念：「醒完，符要来。隐匿符，轻里藏形。」

塔鸣一瞬，轻像光落。围观弟子跪半圈，跪里掺敬——敬里等下一符，护阵。
