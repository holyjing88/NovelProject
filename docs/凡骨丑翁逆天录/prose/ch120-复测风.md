# 第一百二十章 复测风

外门复测风声，半月内传遍丙九。

复测风至，柱上条换新：「外门复测风声，丙九备粒。」备粒不要紧，手稳才要紧。手稳，127债序才清，128才验正。

他理怀内培元散样，粒粒泥纹浅，浅不要紧，嗅之正；正了，试台才配说话。说话才配钱戾衡影硬不过「正」。

刘婆端粥，仍多半勺：「复测？」「风声。」韩泥说，「备粒，手稳。」手稳，饭稳；饭稳，恩不断。

柱上条日日新，新字像秤：「丹试，培元散；手抖除名，末席可报。」韩泥读条，独眼平，报名册上押一字——字丑，丑得像人，人稳，字也稳。稳了，赖福找不到滚字——滚字在管事册，不在赖福嘴。

赖福晃木牌，笑薄：「丑杂役也报名？报名，省我除名工夫。」

「除名看手。」韩泥声平。

「手稳，灵根末。」赖福说，「末，丹道断。断了，上台像笑话。」

韩泥不辩。辩，费舌；舌要留来炼粒，留来守坛。坛在角，角稳，风声再嘲也不飘——飘了，滚字就真。真了，叶汤恩断，刘婆恩还迟，芽儿那半块石白还。

报名第二日，内门廊下议论嗡成一片。嗡里，钱戾衡名被提起，提起像影：「三年约那丑杂役，也报丹试？」

「报，让他滚得明白。」有人笑。

笑薄，薄也真。韩泥路过，不近，不躲——躲像怕；怕，手抖。手不抖，笑在耳，不在心。心在坛，坛醒了，备战才真；真备战，两月后上台，上台钱戾衡眼风再来。

铁无言立渣场，声低：「风声紧，别漏。」

「不漏。」韩泥说，「漏了，粒不正。」

「正，才配上台。」铁无言说，「上台，我站侧。不替你辩，替你挡粪话。」

韩泥点头。兄弟线，记——记108并肩，记今日挡话。挡话不要紧，手稳才要紧。手稳，九粒在怀，怀沉，沉的是债序；债序在，步不飘。

刘婆端粥，仍多半勺：「复测风？」

「起了。」韩泥说。

「起风别空肚。」刘婆哼，「空肚，手飘。飘了，恩断。」

韩泥吃尽。谢轻，恩重。恩在粥里，也在风声里——风里嘲，粥里热；热顶饥，饥顶备战。备战两月，一日一日熬——熬到121再炼，熬到130魁，熬到还刘婆。

午后，白织月带婢至药山道，袖风冷：「丑杂役报名，像桩也上台？」

韩泥捧渣袋，不辩：「桩稳，台不塌。」

「塌的是丹道骨。」白织月笑，「骨丑，散必杂。」

「杂不杂，试台见。白织月袖过，过得像不服。不服不要紧，试台在远；远，先炼，先稳十层，先藏好三粒培元散。三粒不够，121再炼至七粒九粒；粒正，手稳，风声再嘲也尖不过「正」。废窑里，他再开一炉。炉温稳，稳得像119那日；那日稳，今日更稳——稳了，多炼两粒，粒在怀，怀沉，沉得像债序。债序在，步不飘。

芽儿还了沈枯芽半块灵石，刘婆待还，叶汤在省亲前。序不乱，粒才正。秦霜远木鱼远敲：「风大，心别邪。」韩泥不答，答在心里——心里玉牌凉，三年约烫，钱戾衡影在条后。条后不要紧，手在前；手稳，影就虚半分。虚半分，够走到121闭门再炼，够走到123戾衡影，够走到130还刘婆。邻床少年悄问：「钱戾衡会来吗？」」

「来。」韩泥说，「来，记着。记着，不抖。」

少年闭嘴。闭嘴，韩泥望坡下试台方向——方向像130魁，魁像还刘婆。还刘婆前，先复测；复测过，才炼气十三，才夺魁名。魁名在牌，牌歪，字丑，人更丑；丑不要紧，丹正才要紧。

风声这一半月，柱上条日日新。新字像秤，秤手，不秤脸。脸丑不要紧，手稳；稳了，风声是鞭子，鞭子抽耳，不抽手——手在，粒在，坛在，玉牌在。

赖福廊外骂：「报名也白报！白报，滚得更快！」

韩泥只答：「在。」

在，就不滚。不滚，风声再嘲也尖不过「报」字——报了名，备战才真；真备战，两月后验手；验手，为130魁，为还刘婆新衣养老钱。

丑时，练鼻下静。粟壳香淡，香落齐，风声这一半月记进心里。记进心里，不进舌——舌响，赖福记「报名晦气」。晦气不要紧，手稳夺魁；夺魁，还刘婆，大比帖在怀。

他低声道：「复测风，起。起不要紧，手稳。」

坛腹微温，应「报」字。报了名，风就真；风真，备战才真。真备战，两月后上台；上台，钱戾衡眼风再来——来，接得住。接得住，就不滚；不滚，炼气十一十二在121至123破。

外头更鼓沉，沉得像试台在等。等在远，备战在近；近，炼粒，稳十层，喝刘婆粥。粥稀，热，热得像第七笔恩又温半分。半分够撑到130，够撑到新衣递刘婆手里，够撑到养老钱数过三遍。

他摸玉牌，牌凉：「三年约，记着。钱戾衡，记着。赖福克扣，记着。」

记着，刃磨亮。亮在还刘婆，亮在夺魁，亮在大比帖至山门。山门帖上言伯钧名，钱戾衡影在字后——影在路后，刘婆这笔先还。先还，坛认「还」字，腹里微温应。

风声里，执事又贴条：「丹试限培元散，私带除名。」韩泥捧陶罐，罐封泥，泥封严。严了，私带谣虚；虚了，舌净，手稳，试台见。见了，正字落，嘲哑半分。散队后，他仍渣场分拣，掌茧裂处疼，疼不露。不露，像没听见风声嘲。嘲在耳，备战在心——心在坛，坛醒，粒在怀沉。

第二周，韩泥闭门半日炼粒，半日清粪，日夜不乱。不乱，赖福找不到由头；由头在腹，腹有气，气不露。不露，白织月袖风再冷，也冷不过坛腹一线温。温应备战，不应嘲。嘲在耳，手在稳；手稳，粒在正；粒正，试台说话。说话了，钱戾衡影硬不过正，赖福滚字硬不过册。册在，名在，路在。路在，三年约就不绝。

不绝，就熬到121闭门七粒，熬到123戾衡影，熬到129帖至，熬到130魁与还刘婆。还刘婆，沈枯芽半块石已清，序不乱，步不飘。飘了，叶汤还凉；凉不得，汤是头的。头的先记着，手先稳着，粒先正着。正着，风声再紧，也紧不过鼻下静那一字不落。

复测风声半月，柱上条日日新。新字像秤：「丹试，培元散；手抖除名。」韩泥读条，独眼平，不近廊下笑。笑在耳，不在心——心在坛，坛醒了，备战才真。

钱戾衡名在风声里，提起像影：「三年约那丑杂役，也报丹试？」笑薄，薄也真。

铁无言渣场边：「风声紧，别漏。」「漏了，粒不正。」韩泥应声。上台，铁无言站侧，不替炼，挡粪话——挡了，舌净，手稳。

刘婆端粥，仍多半勺：「起风别空肚。」「空肚，手飘。午后，白织月带婢至药山道，袖风冷：「风声这么大，丑骨也上台？」韩泥捧渣袋，不辩：「上台看手。」袖风过，过不要紧，怀内三粒培元散沉——沉像债序，序在，步不飘。沈枯芽远站，怯笑浅。笑浅，像芽；芽在，徒恩还了；还了，更怕沾魁晦气。

沾不要紧，韩泥不招手，招手像讨。讨轻，恩重。夜，再炼一炉，多两粒，粒成，成五粒在手。五粒够，够试台，够夺魁预备——预备在130，不在今日。今日风声紧，紧不要紧，粒正，手稳。条后不要紧，手在前；手稳，影虚半分。」坛腹微温，应「报」——报了名，风就真；风真，备战才真。

真备战，接121再炼，接122织月嘲，接123戾衡影。赖福骂：「报名白报！不滚，风声是鞭子；鞭子抽耳，不抽手。风声半月，柱上条日日新。脸丑不要紧，手稳。稳了，铁无言掷簿，簿清，盗谣虚。虚了，白织月袖风尖不过正。正在粒，不在嘲。嘲在耳，不在心。心在坛，坛醒，备战真。

真备战，121闭门炼，122榜末行，123戾衡影，129帖至，130还刘婆。他摸玉牌，牌凉，低声道：「三年约，记着。手稳，粒正，试台见。」见字落，坛温一线，像应。应了，路又明一寸——明向还恩，明向不滚，明向刘婆新衣养老钱。」

日暮，他回丙九，不近廊下笑——心在坛，坛腹温一线，温像应「圆」。
