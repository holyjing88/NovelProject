# 第一百零六章 兽潮续

续。

退潮是假，续才是真。真在第四更，第五更，第六更——更尽，人换，角不换。

韩泥不换角，角黑，黑是爪，是毒，是汗。汗不要紧，汗洗得掉；掉不了的是记——记谁挡，谁清，谁没逃。

铁无言换绷，绷粗，声低：「还能守？」

「能。」韩泥说，「手稳。」

能，不为名，为栏。栏倒，丙九连坐；连坐，叶丫头那碗烫还不了。

第七更，兽影稀，稀不要紧；稀，毒更浓。浓，像最后一扑。

最后一扑，贴障，障破，破在内侧一线。内侧，韩泥第三粒清瘴丸尽，尽不要紧；尽，就咬经页诀，诀出，灰雾浓半分。

浓半分，清半丈。半丈，够小兽活，够同袍退而不逃。

管事弟子喊：「退！退至内墙！」

韩泥退，退时扶铁无言，扶不要紧；扶，是并肩的续。

退至内墙根，根稳，稳像终前一息。

终前，兽吼远，远像散。散不要紧，散前，栏里倒的少了——少了，是小胜。

胜在活，不在册。册，管事仍记：「丙九角稳。」

记，赖福瞧见，脸青半分，没嗓——嗓要留来秤麸皮，秤不到韩泥掌茧上。

白织月远，声低：「……还站得住。」

韩泥不答。答，费舌；舌要留来数囊，数经页，数清瘴路还剩几程。

几程，在108全。全不要紧，这一章是续——续，不是清。

夜尽，他坐棚角，坐半刻，掌裂，裂渗潮。潮不要紧，潮洗得掉。

刘婆塞饼，饼干：「吃。别空肚守。」

韩泥吃，吃不多。多不要紧，够活。

按坛，低声道：「续了。还没完。」

坛腹温，像应「续」字。续，指向清。

丑时，静半刻。香落齐，掌纹烫——烫像境将动，动不要紧，动仍五层。

邻床问：「完没完？」

「没完。」韩泥说，「完了，才清。」

更鼓尽，界外安静，安静像钩。钩108，在下一程。

他低声道：「续了。下一程，清瘴。」

清瘴，手在，铁无言在，经页在。

续，才是真。第四更，第五更，第六更——更尽，人换，角不换。

铁无言换绷，声低：「还能守？」

第七更，兽影稀，毒更浓。

韩泥第三粒清瘴丸尽，咬诀，灰雾浓半分。

管事喊：「退至内墙！」

韩泥退，退时扶铁无言。退至内墙根，根稳。

兽吼远，远像散。栏里倒的少了——少了，是小胜。

赖福瞧簿，脸青半分，没嗓。

刘婆塞饼：「吃。别空肚守。」

扶铁无言退，退不要紧，退而不逃。不逃，簿上「角稳」又实一分。实一分，赖扒皮没嗓——嗓要留来秤麸皮，秤不到掌茧上。

饼吃半块，半块够活。活，才清107余瘴。余瘴，钩108；108，炼清瘴丸，丸正，铁无言并肩。并肩，契起，契在580远，远不要紧，108已证。

退至内墙！终前不要紧，终前，107才钩。

刘婆在坡下送粥，粥稀，没近栏：「活着就好。」韩泥点头，谢轻，恩重。恩在粥里，也在扶铁无言那半息里——半息，够并肩。并肩，才像108。

赖福脸青，青完没嗓——嗓要留来记过，记不过韩泥掌茧。掌茧裂，裂不要紧；裂，是活计。活计在，名在，路在。

更鼓沉，韩泥按木牌，牌凉贴胸。兽吼远一线，远不要紧，余瘴还在。还在，107才钩。下一程，清瘴。

兽潮续，续在第四更、第五更、第六更——更尽，人换，角不换。汗不要紧，汗是活计；活计在，就不滚。

第七更，兽影稀，稀不要紧，稀，毒更浓。浓像最后一扑，扑在内侧，内侧韩泥咬诀，诀出，灰雾浓半分，清半丈。

管事喊退，韩泥退时扶铁无言，扶不要紧，扶是并肩之续。终前，107钩——钩余瘴，钩深窝，钩108将炼之「清瘴」二字。

刘婆坡下送粥，没近栏：「活着就好。」活着，才还恩；恩在95，恩在94，两笔不混。混，七笔乱。乱，路断；路不断，108才清瘴，109才稳炉。

「赖福脸青，没嗓——嗓要留来记过，记不过掌茧。掌茧裂，裂是活计，不是滚。滚在钱戾衡脚，脚在远，约在怀。怀贴身，像守坛沿。下一程，清瘴。」清瘴，铁无言在，手在，经页在。在，就不绝。兽潮续，第四更、第五更、第六更——更尽，人换，角不换。汗是活计，活计在，就不滚。第七更，兽影稀，毒更浓，浓像最后一扑。

韩泥咬诀，雾浓半分，清半丈——半丈，够小兽活，够同袍退而不逃。不逃，107才钩，108才清瘴。管事喊退，韩泥退时扶铁无言，扶是并肩之续。终前，107钩余瘴——钩深窝，钩108将实之「清瘴」二字。刘婆坡下：「活着就好。」活着，才还恩。恩在95，恩在94，两笔不混。更鼓沉，按木牌，续了，下一程清瘴。

清瘴，铁无言在，手在，路在。续在第四更、第五更、第六更——更尽，人换，角不换。不换，角黑，黑是爪，是毒，是汗。扑时，咬诀，雾浓半分，清半丈——半丈，够小兽活，够同袍退而不逃。不逃，107才钩；钩，余瘴，钩深窝，钩108将实之「清瘴」。清瘴在远，106只续——续，退时扶铁无言，扶是并肩之续。

扶不要紧，扶，才像108。

路在，109稳炉，110培元散。散在远，远不要紧，106只续命。命在，就不滚。

续第五更，人换角不换，不换，角黑，黑是汗，是爪，是毒。毒浓，浓像最后一扑——扑时，咬诀，雾浓，清半丈。半丈，够退而不逃；不逃，107钩。钩余瘴，钩深窝，钩108清瘴。清瘴在远，106只续命。命在，就不滚。

不换，角黑，黑是汗，是爪，是毒。毒浓，浓像最后一扑——扑时，咬诀，雾浓半分，清半丈。终前，107钩——钩余瘴，钩深窝，钩108将实之「清瘴」。命在，就不滚。混，七笔乱；乱，路断。

第八更，兽影稀尽，稀不要紧，稀，毒最浓。浓，像最后一扑——扑时，咬诀，雾浓，清半丈。半丈，够退而不逃；不逃，107才钩。钩，余瘴，钩深窝，钩108清瘴。命在，就不滚。滚在远，扶铁无言，扶不要紧，扶是并肩。并肩，手在，路在。

续第八更，兽影稀尽，稀不要紧，稀，毒最浓。浓，像界外最后一扑——扑时，咬诀，雾浓，清半丈。命在，就不滚。并肩，手在，铁无言在，经页在怀。怀贴坛，坛温一线，像应「续」字。续字落齐，管事喊退，退至内墙根，根稳，稳像终前一息。终前，107钩108。钩住了，就不白续。白续不要紧，手在，路在，就不绝。

钩在余瘴，钩在深窝，钩在108将实之「清瘴」二字。二字将实，手在，铁无言在，经页在怀。怀贴坛，坛温一线，像应「钩」字。钩字落齐，掌在，路在，就不滚。滚在钱戾衡脚，脚在远，守在角，角还在，就不散。

第八更尽，韩泥仍立角，角黑，黑是汗，是爪，是毒——毒浓，浓像最后一扑。扑时，咬诀，雾浓，清半丈；半丈，够同袍退而不逃。不逃，107才钩；钩，余瘴，钩深窝，钩108清瘴。管事喊退，退时扶铁无言，扶是并肩之续。更鼓沉，按木牌，续了，掌在，路在，就不滚。滚在远，守在角，角还在，就不散。

管事再喊退，韩泥退至内墙根，根稳，稳像终前一息。终前，107钩108——钩余瘴，钩深窝，钩108将实之「清瘴」。钩在远，106只续命。命在，就不滚。更鼓沉，按木牌，扶铁无言，扶是并肩之续。掌在，铁无言在，路在，就不绝。

终前最后一更，兽吼远一线，远不要紧，余瘴还在。还在，107才钩——钩深窝，钩108清瘴。续了，退至内墙根，根稳，扶铁无言，扶不要紧，扶是并肩之续。更鼓沉，按木牌，掌在，铁无言在，路在，就不滚。

续了，下一程清瘴。命在，掌在，铁无言在，路在，就不滚。

更鼓沉，按木牌，掌在，铁无言在，路在，就不绝。

续了，清瘴在远。命在，掌在，就不滚。

同袍递来一碗浑水，水浑，韩泥接，接不谢——谢轻，恩在活里。活里，107才钩，108才清瘴。
