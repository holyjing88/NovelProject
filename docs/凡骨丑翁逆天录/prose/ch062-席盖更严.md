# 第六十二章 席盖更严

瓮醒前夜，风硬得像最后一道冬关。关不过，缘不满；缘满了，才谈得上误触那一滴。

韩泥清栏归来，掌茧新裂，裂口渗血珠。他甩手，血甩上破席，席吸尽，沿仍净。不主动滴血，不揭布诱沿——只用活计扎出的血喂席，席挡沿，沿等误触那一瞬。这一瞬等了六十日，他今晚只加席，不加催。

邻床笑：「坛角盖那么严，藏宝啊？」

韩泥不辩，只把席盖又加一层。层厚，厚得像秘密。秘密是坛，坛是命，命跟了他五十八岁。

管事查棚，灯晃坛角：「杂物收起。」

韩泥只答一声，席盖不动。动了像心虚，他不虚，是守。管事盯半息走了，只见破席，不见仙缘。不见不要紧，沿要净——净了，管事才肯记「季末记功在」；功在，饭才正，恩才不断。

清栏第二趟，掌茧又渗血。血落席，席厚，吸得尽；沿仍净，净得像一道不动的界。他加第三层破布，布更旧，旧更能吸。层厚像钩，钩着瓮醒，钩在沿，钩在误那一甩。诱沿像借腥，腥借来，路易断；他不诱，只守。

夜里符怀暖一丝，护骨寒，手不僵。僵了，明日藤刺会偏；不偏，缘才正。他对坛低声道：「明日或后日，藤还会扎。扎了，血来席，不来沿。沿上，等你认的血。」

坛腹温一线，贴掌，贴到心口也暖半分。暖不像火，像认亲前的试探。更鼓三响，沉沉的，像丑时坛温那一线又要走一圈——走到编筐扎掌，走到血误触席，走到席渗沿，走到沿颤，走到瓮醒。

刘婆端姜汤来，姜薄，却热：「前夜别空肚。空肚，手飘。」

「不空。」韩泥喝尽，只点头。姜辣醒指，指醒了，茧才不白裂。「席盖严了。」他对坛说，「守得严，韩家不散。」

祖父韩老栓嘱过：「坛在，韩家不散。」嘱在耳，不在舌；舌急，血偏，偏了席挡不住。

赖福讽：「盖席盖出仙缘？」

管事声冷：「再嗓，克半月麸皮。」

赖福哑了。哑了，韩泥加席的手却更正。同批少年碰他袖：「盖那么严，坛里真有宝贝？」

「有。」韩泥说，「有命。命跟坛，坛跟缘。」

少年缩手，像被「命」字烫到。他不辩宝贝，只把第七层破布角压实——压实了，沿前多挡半寸风。

秦霜远廊停半息：「前夜别空肚。」她不再言，木鱼在指间，不托，不敲——像规矩还在耳，正测三关还在柱上。

铁无言门外脚步停半息：「沿净？」

「净。」韩泥说，「血在席，不在沿。」

「不主动滴？」

「不滴。滴，要等活计误那一甩。」

铁无言点头，脚步远了。韩泥又加一层破布。邻床问：「盖那么严，热不热？」

「热不要紧。」韩泥说，「沿净要紧。净了，才接得住瓮醒。」

他按心口，低声道：「丫头那碗汤，热着还。还，要等瓮醒后，路才长。」

坛腹温一线加剧，像明日就在沿边等那一滴守瓮血。误触那一滴，缘才满。满不在心急，在活计，在席，在沿。他独眼平，手不抖：「守住了，才配醒。醒在缘，不在舌。」

编筐第三刻，末藤硬，刺偏，他仍偏手，刺浅些。浅些，血仍来。血落席，席边湿，沿不接——不接像候缘。

同批少年夜间又问：「瓮醒了你怕不怕？」

「怕。」韩泥说，「怕恩断。恩不断，怕也要守——守席，守沿，守不主动滴血。」

少年不懂「恩断」，却懂「怕也记」，闭嘴睡了。棚里更静，更鼓沉得像瓮将醒未醒。他摸祖父嘱，嘱短，字丑，丑得还能读：「误了，根在。」根在，路才长；路长，才还得上叶丫头那碗烫。

管事第三趟查棚，灯晃坛角，只见破席。他加第八层角，角旧，旧更能挡——挡到沿前，沿不接，缘才满。秦霜远廊下过，停半息：「手正，心别邪。」他不拜，亦不揭席。揭席像心虚，心虚像借腥。

老耿挑水路过，声哑：「前夜别拜。拜，路断。」

「不拜。」韩泥应，「席盖严，不主动滴血。」

老耿点头，水晃正，像告诉他：沿净，路就不绝。邻床问：「误不了咋办？」

「误不了，就等。」韩泥说，「等，像守瓮人。」

管事验筐：「季末记功在。手正，沿还净？」

「席边湿，沿仍净。」

管事记簿走了。午后搬柴一担，肩勒旧印，印深得像丙九五年。柴担落地，掌心又裂一线，血珠饱满。甩手，血甩破席，席吸尽，沿仍净。净是六十日守出来的；守出来，才配瓮醒前夜这一更。

管事又在棚柱贴条：「季末记功在。手正，沿还净者，少克三日麸皮。」条新，字利。韩泥不谢，只把破席角掖进坛底——掖实了，沿前多挡半寸风。

赖福瞄一眼条，嗓尖：「丑杂役也配少克？」

管事冷声：「再嗓，克你半月。」

赖福退得像吞了苍蝇。韩泥不接话——接话像邀功；功在沿净，不在舌。

外头更鼓又三响。沉里，瓮醒将至。最后一夜，风硬，符暖，手正。正了，才配明日藤刺再扎那一误——扎了，血找席，席挡沿，沿等认主那一滴。

天边白一线，白像瓮醒就在下一更。他对坛低声：「席严，沿净，符暖。正了，才配你醒。」坛不应字，只温一线加剧，像沿边已等那一滴守瓮血。

刘婆第二趟姜汤搁棚柱，他喝尽，不空，手才不飘。第七层破布压上，像给坛角戴了顶冬帽。帽丑，厚才真——真在能吸活计血，不在藏宝。

他独眼盯沿半刻：沿尘，沿滑，沿上无血。无血，才配韩氏守瓮那一滴落对地方。席湿，沿仍净——净是规，规比命长。同棚少年梦呓问「醒了吗」，他不答。答像求；不求，只守。他把席边湿处又按干一遍，按得像把血关在席里，关在缘里。

丑时前，更鼓又沉。铁无言最后一嘱：「沿净，不主动滴。滴，等藤刺误那一甩。」

韩泥只答，手不抖。木牌在怀，七笔分列。还，先还烫——烫在叶丫头，不在这一更。祖父嘱在枕下：「误了，根在。」

邻棚有人烧纸，纸灰飘过一缕，飘到沿边即止。止，像沿也认界——界内席，界外尘；尘不落沿，沿才不脏。他只把第八层角又压实一分。

管事夜巡路过，只问一句：「沿还净？」

「净。」韩泥答，手不抖。

管事记心里走了。走了不要紧，沿在，缘在。坛腹温一线加剧，像沿边已等那一滴守瓮血。候不在舌，在活计；活计误那一甩，缘才满。

他独眼平，手不抖，低声道：「席盖严了，沿还净。丑时前，藤刺还会扎——扎了，血找席，席挡沿，沿等认主那一滴。瓮将醒，醒在缘，不在人催。」

同批少年忽然半醒，嗓子糊：「你一夜加席，手不酸？」

「酸。」韩泥说，「酸也加。加严了，明日才接得住误。」

「误啥？」

「误了就根在。」韩泥不解释。解释多了像求，求像拜邪。少年哼一声，又睡。睡了，棚里只剩更鼓与坛温。

他把木牌与席角并放，并放，像命与缘挨着。挨着，他才敢等那一甩。符在怀，暖一丝，护骨寒；骨不寒，明日藤刺才不偏。偏不得——偏了，血找不着席，席挡不住沿，六十日规矩就散。

更鼓沉，天边白一线。白里，他按木牌，牌凉贴胸。下一更若来误触，席严不严、沿净不净，就全要见真——真在瓮将醒，不在他嘴上催。催不得，等得久；久了这一夜，明日丑时那一甩，才配醒。醒不醒，全看缘满不满两字。
